# 武当山济肾堂强肾七式：07：第二式：养气法 🧘

在本节课中，我们将学习“养气法”的理论核心与具体练习方法。养气，即养护我们身体内推动生命活动的能量。我们将从理解“气”的消耗原理开始，逐步掌握让身心进入最佳修养状态的方法。

## 理论核心：何为养气？

“气”在这里指代人体的能量，即推动我们进行一切生命活动的动力。养气的目标，就是减少不必要的消耗并补充这种能量。

### 能量的五种消耗途径

人体在五种情况下会持续消耗“气”。理解这些途径是学会养气的前提。

1.  **身体活动**：任何肢体运动，如行走、跑步或搬运物品，都会消耗能量。因此，养气时身体需保持静止。
2.  **言语交谈**：开口说话会耗气。养气时应保持缄默。
3.  **用眼观看**：眼睛如同摄像机，观看外界景象会将信息录入大脑进行处理，从而消耗能量。养气时宜闭目。
4.  **用耳聆听**：耳朵接收声音信息同样会耗气。养气时需避免主动去听。
5.  **大脑思虑**：这是消耗能量最大的方式。无论是主动思考（如解题、策划），还是被动杂念，都会持续耗气。养气追求的是大脑停止思虑的状态。

综上所述，理想的养气状态是：**身不动、口不言、目不视、耳不听、心不思**。

## 大脑的三种状态与进阶之路

上一节我们明确了养气需避免大脑思虑，本节我们来看看如何具体实现。我们的大脑在练习时通常经历三种状态。

*   **万念**：这是最常见的状态。身体静止时，头脑中会不由自主地涌现各种杂念，虽比主动思考耗能少，但仍在消耗。
*   **一念**：这是从“万念”通往最佳状态的桥梁。指将注意力高度专注于**一个单一的对象**上，以此收束纷飞的杂念。
*   **无念**：这是养气追求的最佳状态。此时大脑完全停止主动与被动思虑，进入一种“无为”或“空”的深度宁静状态。道家称此状态为“虚极”，佛家称为“空定”。此时身心得到最深层的修复与能量补充。

那么，如何从“万念”经由“一念”抵达“无念”呢？关键在于 **“听呼吸”**。

### 从“万念”到“无念”的实践方法

1.  **进入“一念”（听呼吸）**：当你觉察到自己处于“万念”纷飞的状态时，立即将注意力转向用耳朵去聆听自己的呼吸。仔细感受空气如何通过鼻腔吸入，又如何缓缓呼出。初时可能听不真切，随着练习加深，在安静环境下便能清晰感知。这即是“精神内守，病安从来”的身心合一状态。
2.  **迈向“无念”（忘我）**：在专注“听呼吸”（一念）的过程中，继续深入，逐渐“忘掉”自己在听呼吸这件事，乃至忘掉自我，进入物我两忘的恍惚之境，这便是“无念”状态。
3.  **循环往复**：初学者在“无念”状态中停留时间很短，很快又会滑回“万念”状态。这非常正常。一旦觉察，便再次回到“听呼吸”（一念）的专注中，如此循环。练习功夫的高低，就在于能在“一念”和“无念”状态中保持的时间长短。

## 养气法的三种练习姿势

掌握了核心理论后，我们可以通过不同姿势进行练习。根据“身体不动”的原则，主要有以下三种方式：

1.  **站桩**：站立不动进行练习。这是补充能量、增强体质最快、效果最好的方式，也是中国传统武术和养生功夫的基础。
2.  **打坐**：坐着不动进行练习。
3.  **睡功**：躺着不动进行练习，或利用睡眠时间进行修养。

以下我们将以 **站桩** 为例，讲解养气法的具体操作步骤。

## 实践演练：站桩养气法

### 桩架要领

1.  **起式**：双脚并拢，脚尖、脚跟皆靠紧。然后屈膝下蹲，将左脚轻轻提起，向左侧横开一步，使两脚间距与肩同宽。
2.  **身形调整**：
    *   膝盖微屈，但**膝盖尖不可超过脚尖**。
    *   感觉臀部如同坐在一张高凳上。
    *   裹臀、收腹、含胸。
    *   沉肩：肩膀放松下沉。
    *   坠肘：肘部感觉像被绳子吊在肩上，自然下垂。
    *   双手自然环抱，置于腹部丹田前。
    *   **关键点**：双脚平行朝前，不可内八或外八。全身重心必须**下沉至脚底**，让骨骼支撑身体，而非用肌肉硬扛。可通过轻微前后左右移动身体来感知并调整重心，确保重心落于脚底，如此才能久站不累。
3.  **头部姿态**：颈部微微后靠，使头颈正直。下颌微收，不要上扬。

### 心法练习流程

1.  **外静**：摆好桩架后，保持身体不动、嘴唇轻闭、双眼微合。
2.  **内观**：初始时，头脑通常处于“万念”状态。
3.  **收心（一念）**：一旦觉察杂念纷飞，立即将注意力转向用耳朵“听呼吸”。专注感受气息的进出。
4.  **入静（无念）**：在专注听呼吸的过程中，逐渐淡化这份专注，顺其自然地进入忘我、忘息的“无念”宁静状态。
5.  **循环**：若思绪再度飘散，则温和地再次回到“听呼吸”上。在整个练习期间，如此循环往复。

### 收功方法

收功至关重要，能引导气血归元。

1.  将双腿缓缓站直。
2.  慢慢睁开双眼。
3.  双手相叠（男左手在下，女右手在下），覆盖于腹部。
4.  顺时针揉腹12圈，再逆时针揉腹12圈。
5.  由于练习中口中会产生大量津液（金津玉液），需将其分三次缓缓咽下，意送丹田。此乃滋补佳品。
6.  完成上述步骤后，即可进行日常活动。

本节课中，我们一起学习了“养气法”的精髓。我们首先了解了人体能量（气）的五种消耗途径，从而明确养气需“身、口、目、耳、意”五者俱静。接着，我们剖析了练习中大脑“万念 -> 一念 -> 无念”的三种状态与进阶方法，其核心诀窍在于 **“以耳听息，由一念入无念”**。最后，我们以站桩为例，详细讲解了从桩架、心法到收功的完整练习流程。记住，练习的关键在于持之以恒，在反复的“念起 -> 听息 -> 入静”循环中，逐步增强身心能量的储备。
