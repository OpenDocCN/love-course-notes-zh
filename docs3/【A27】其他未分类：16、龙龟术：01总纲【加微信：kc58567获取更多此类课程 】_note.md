# 龙龟回春术：01：总纲 🐢

在本节课中，我们将学习龙龟回春术的总体纲要。这套功法旨在强身健体，尤其关注提升精力。掌握总纲是后续所有练习的基础，当你在练习具体功法遇到困惑时，应回顾此总纲。

龙龟回春术共包含六个功法：**天龙八部**、**七窍乌龟**、**陡肾功**、**铁板桥**、**拍打**和**付脑功**。虽然各功法练习内容不同，但需遵循一个统一的总纲。

上一节我们介绍了课程概述，本节中我们来看看练习任何功法时都必须把握的四个核心要点。

以下是练习时必须考虑的四点总纲：

1.  **目的**：明确练习该功法的目标，以及期望达到的效果。
2.  **技巧**：严格遵循教学要求，不要自行更改。这是数万名学员实践优化后的方法。
3.  **火候**：需自行体会和控制。核心原则是 **“切忌泄身”**。无论练习到何种程度，一旦有泄精征兆必须立刻停止。功法旨在强身，百日筑基期内禁止泄精。因个人体质差异，火候需自行把握。
4.  **次数与循序渐进**：规定的次数（如 `9次为一组，做9组，共81次` 为一个循环）是目标，但需根据自身身体状况（即火候）调整，不可强求。功夫需日积月累，**循序渐进**，不可急于求成。

除了上述核心要点，练习时还需注意个人卫生与准备。

以下是练习前的必要准备事项：

*   练习前需排清二便，确保全身放松。
*   若使用道具或润滑油，推荐食用植物油（如橄榄油、大豆油）。若觉油腻，可使用唾液替代。
*   无论使用何种润滑，双手必须清洁干净，以防引发炎症。

龙龟回春术主要分为两大板块：第一是**身体基础训练**，旨在恢复精满、气足、神旺的状态；第二是**技巧应用**，学习在具体情境中如何运用。两者相辅相成，缺一不可。

本节课中我们一起学习了龙龟回春术的总纲，明确了练习的四大核心要点（目的、技巧、火候、循序渐进）以及必要的准备事项。牢记总纲是安全、有效进行后续所有练习的关键。
