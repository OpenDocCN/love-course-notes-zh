# 洗髓功：第01节：易筋六式（男）🧘‍♂️

在本节课中，我们将要学习“易筋六式”中的六个核心动作。这套动作旨在拉伸全身主要经络，特别是督脉和任脉，有助于促进气血循环、增强身体柔韧性并改善整体健康。每个动作都简单易学，适合初学者练习。

## 第一式：盘龙探海 🌊

上一节我们介绍了课程概述，本节中我们来看看第一个动作“盘龙探海”。这个动作主要拉伸督脉、腿部经络、腰部脊柱以及膀胱经，对改善大脑供血供氧有良好效果。

**动作要领如下：**

1.  双腿分开，与肩同宽。
2.  舌抵上腭，保持自然呼吸。
3.  两臂向身体两侧平展，反掌（掌心向上），力达指尖，举过头顶。
4.  双手食指交叉，反掌向上推，感受身体向上提拉。
5.  保持脊柱正直，弯腰向下压。
6.  尽力下压，保持姿势30秒。
7.  吸气，身体回正；呼气，双臂向两侧划弧线落下，双膝微屈放松。

这个动作拉伸了腿部、腰部和手臂的经络（手三阴、手三阳、腿三阴、腿三阳），你会感觉胸腔也被提拉起来。它通过拉伸任督二脉，间接活动了全身的经络网络。

## 第二式：左右弯弓射雕 🏹

盘龙探海主要拉伸身体后侧，本节中我们来看看“左右弯弓射雕”。这个动作有一定难度，能有效拉伸侧腰和肋部区域的筋络。

**以下是左侧动作步骤：**

1.  保持自然呼吸，舌抵上腭。
2.  迈出左腿，两脚与肩同宽。
3.  两臂向两侧展开，反掌，掌心朝上，双手食指交叉向上推。
4.  身体向右侧旋转。
5.  呼气，身体向下压，同时翻转手掌使掌心向上。
6.  吸气，重心前移成左弓步（左腿弓，右腿蹬直）。
7.  反掌，双臂向外推出，眼角余光看手背。
8.  保持此姿势30秒。练习久了可尝试加大侧弯幅度。
9.  呼气，收回动作。

**右侧动作与左侧对称：**

1.  双腿向右侧分开，与肩同宽。
2.  重复上述臂部动作，身体向左侧旋转并下压。
3.  迈出右腿成右弓步，反掌外推，保持30秒。
4.  呼气，收回动作。

此动作要求膝、胯、肩尽量保持在一条直线上，身体呈弓形，能充分拉伸侧身筋络。

## 第三式：太虚拔阴 🌌

左右弯弓射雕锻炼了侧身，本节中我们来看看“太虚拔阴”。这个动作简单，能锻炼肾经，起到强肾的作用。中医认为“抬脚养肾，闭目养肝”。

**动作步骤如下：**

1.  双腿分开，与肩同宽，保持自然呼吸，舌抵上腭。
2.  两臂向两侧展开，反掌举过头顶。
3.  双手食指交叉向上推。
4.  在向上推的同时，踮起脚尖，让脚后跟抬起，前脚掌着地。
5.  尽力向上拉伸身体，你会感觉腰部变得空松，小腿可能抽筋，这属正常反应。
6.  手臂可能感到酸、麻、胀、痛，均属正常。
7.  保持此姿势30秒。
8.  呼气，脚后跟落下，同时两臂向两侧划弧线落下。

## 第四式：天圆地达式 🌍

太虚拔阴着重提升，本节中我们来看看“天圆地达式”。此动作主要拉伸身体前侧的任脉和胸腔，并能打开盆腔。盆腔是人体容易积聚浊物的部位，脾胃不适、女性宫寒妇科问题、男性性功能问题等都可能与此区域淤堵有关。现代人常见的“三高”、慢性病伴随的腹部肥胖，也常因带脉淤堵、毒素无法排出所致。

**动作步骤如下：**

1.  双脚分开，略宽于肩，保持自然呼吸。
2.  双手在背后交叉，食指相扣。
3.  反掌，使掌心朝向地面（这样手不会限制身体后弯）。
4.  身体自行向后弯腰。**注意安全**，初学者可能无法下弯太多，避免摔倒。
5.  保持后弯姿势30秒。此时胸腔打开，胯部伸展，腰部有顶起感，可能感到难受，多练习即可改善。
6.  吸气，身体慢慢回正，双手自然放开落下。

这个动作拉伸了胸腔、背部督脉和命门区域，对强肾也有良好效果。

## 第五式：乾空通途一式 💆‍♂️

前四个动作拉伸了躯干和四肢，本节中我们重点看看颈椎。“乾空通途一式”专门拉伸颈部经络和淋巴。现代人常见淋巴淤堵和颈椎问题，会导致记忆力减退、大脑供血供氧不足。

**以下是动作分解：**

1.  双腿分开，与肩同宽，全身放松，舌抵上腭。
2.  **吸气**，头部缓慢向左转，尝试让左耳贴近左肩。身体和肩膀尽量保持不动。如果贴不到，肩膀可微抬协助，找到感觉后固定肩膀。
3.  吸气过程持续15秒，到达最大幅度后**屏息**1秒钟。
4.  **呼气**，持续15秒，同时头部从右侧绕回，使右耳贴近右肩，回到中位。
5.  接着进行另一侧：**吸气**，头向右转（右耳贴右肩），15秒。
6.  **屏息**1秒。
7.  **呼气**，15秒，头部从左侧绕回（左耳贴左肩）。

此动作活动了整个颈椎，重点刺激最下方的大椎穴。大椎是人体气血交汇的“十字路口”，容易淤堵，此式对改善“富贵包”也有帮助。

## 第六式：乾空通途二式 🔄

一式活动了颈椎，本节中我们通过“乾空通途二式”进行更深度的拉伸。你会明显感到颈部侧方淋巴区域被拉紧。

**以下是左侧动作步骤：**

1.  双腿分开，与肩同宽，身体放松直立。
2.  **吸气**，头部向左侧转动，身体和肩膀保持不动。
3.  转动的同时，眼角余光尽力看向尾椎骨方向。
4.  如果看不到尾椎，则改为看**右脚脚后跟**。
5.  看到脚后跟后，视线顺着右腿向上移动，尝试看到臀部的尾椎区域。
6.  保持这个最大限度扭转的姿势30秒。
7.  呼气，头部慢慢转回。

**右侧动作与左侧对称：**

1.  **吸气**，头部向右侧转动。
2.  眼角余光看**左脚脚后跟**。
3.  视线顺左腿向上，看向尾椎区域。
4.  保持30秒。
5.  呼气，头部转回。

**要点**：身体始终保持正直，不要前倾或后仰。练习时会感到颈部一侧被强力拉伸，这有助于淋巴排毒和疏通颈椎气血。坚持练习，你会感到肩颈区域发热，非常舒畅。

## 课程总结 📝

本节课中我们一起学习了“易筋六式”的全部六个动作：
1.  **盘龙探海**：拉伸督脉与全身后侧。
2.  **左右弯弓射雕**：拉伸身体两侧筋络。
3.  **太虚拔阴**：提升阳气，强健肾经。
4.  **天圆地达式**：打开任脉与盆腔，排浊通络。
5.  **乾空通途一式**：活化颈椎，疏通淋巴。
6.  **乾空通途二式**：深度拉伸颈侧，促进排毒。

这套组合能有效拉伸身体主要经络，促进排汗排毒，非常适合现代人用于日常拉筋强身。虽然它涵盖了大部分肌群，但人体内侧的肝、脾、肾三条筋拉伸较少，若想针对性提升，需练习其他专门动作。请大家坚持练习，感受身体的积极变化。
