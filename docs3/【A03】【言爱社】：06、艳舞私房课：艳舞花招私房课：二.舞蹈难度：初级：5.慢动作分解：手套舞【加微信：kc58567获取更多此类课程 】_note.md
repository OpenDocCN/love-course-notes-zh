# 性感舞蹈入门：06：慢动作分解：手套舞 🧤💃

在本节课中，我们将学习一段名为“手套舞”的初级性感舞蹈。课程将通过慢动作分解，详细讲解每一个身体姿态与动作衔接，帮助你掌握基础舞步与表现力。

## 动作分解

以下是整套舞蹈动作的分解步骤。

**起始姿态**
背对观众站立，保持背部挺直，双腿伸直，旋转臀部。

**第一组动作**
1.  右膝微屈，身体稍下沉，同时伸出右臂。
2.  做一个交叉步，转身面向观众。
3.  身体前倾，头部随之上下摆动。
4.  双脚并拢，臀部轻微画圈，同时双手沿身体向上移动。

**第二组动作**
1.  做一个交叉步，向侧方迈步，同时双臂向前伸出。
2.  将臀部向一侧推出并画圈。
3.  双脚并拢，抬起一只手。
4.  将一根手指轻触嘴唇，身体微蹲。

**第三组动作**
1.  向侧方迈步，用牙齿咬住手套指尖。
2.  走几步，同时缓慢地将手套脱下。
3.  向侧方蹲下，完成脱手套的动作。
4.  将手套扔掉。

**第四组动作**
1.  将手沿身体向下移动。
2.  起身，身体旋转90度，同时旋转臀部。
3.  将双臂移到身前，然后放回身后。
4.  头部摆动两次。

**第五组动作**
1.  再次蹲下，身体稍向前倾，手触碰腿部。
2.  起身。
3.  向前迈步，将左手移至嘴边。
4.  开始脱另一只手套，同时转身。
5.  将手套扔掉。

**结束动作**
1.  向后退一步并蹲下，双膝分开。
2.  双手放下，将左腿向后伸直。
3.  右膝跪下，头部摆动。

本节课中，我们一起学习了“手套舞”的完整慢动作分解。从起始的背部姿态到结束的跪姿，我们逐步拆解了臀部的旋转、步伐的移动、手套的脱卸以及头部的配合等核心动作。请跟随步骤反复练习，直至能够流畅衔接。
