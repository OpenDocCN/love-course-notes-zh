# 潮水课程：03：实战操作指南

上一节我们介绍了潮吹的基本概念，本节中我们来看看如何进行具体的实践操作。

## 🔬 核心原理回顾

在实践之前，我们必须理解其背后的生理机制。

上一节我们介绍了斯基恩氏腺（Skenes glands），它位于G点上方。当手指进入阴道并刺激G点时，会连带刺激到斯基恩氏腺。

**公式**：刺激G点 → 刺激斯基恩氏腺 → 腺体膨胀分泌液体（斯基恩氏腺液）→ 液体进入膀胱与少量尿液混合 → 从尿道喷出（即潮吹）。

有时可能感觉不到G点，这通常是因为阴蒂头未得到充分刺激。充分刺激阴蒂头使其勃起后，会连带下方的阴蒂脚、前庭球等组织膨胀，从而使G点变得明显凸出。此时在阴道内约5厘米处、朝向12点（腹部）方向，便能摸到一个略有凹凸不平的区域，那就是G点。

## ✋ 刺激手法详解

理解了原理，接下来我们学习具体的手法操作。

第一步是**刺激外阴以唤起居点**。正如之前所述，充分刺激阴蒂头是找到并唤起G点的前提。对外阴及阴蒂的刺激方法，可参考第二节课中详细的“阴蒂按摩”教程，也可借助小道具。

第二步是**手指刺激G点**。将手指伸入阴道，**务必指腹朝上**，然后做出“come on”的勾动手势。可以用单指、双指，或像走路一样交替勾动来刺激G点。
