# 情趣探索：01：五十度灰入门 🎭

在本节课中，我们将要学习BDSM（常被简称为SM）的基本概念，澄清常见的误解，并理解其核心运作原理。我们将从社会认知、个人体验和学术定义等多个角度，系统地认识这一主题。

---

很多人一提到BDSM，都会联想到性变态、虐恋或受伤等画面。但这实际上是对BDSM的一种普遍误解。

首先，人们常认为BDSM是非常小众的需求，与自己无关。然而事实并非如此，它并不小众，也离我们并不遥远。据不完全统计，有61%的女性观众和37%的男性观众曾在观看SM题材电影《五十度灰》时产生过自慰行为。

在我们的社群中，也有不少女性表示自己存在轻微施虐或受虐的倾向。或许很多人和过去的我一样，在影视作品中看到角色被捆绑或受罚时，会莫名感到兴奋。

我从小就喜欢偷偷观看这类影视作品，甚至童年时的玩具和衣物也常带有相关元素，例如我始终偏爱有绑带的衣服和鞋子。青少年时期，我曾因此感到焦虑，认为这种想法不正常。

但如今我不再为此焦虑。因为我清楚地认识了这件事的本质，并学会了如何控制与运用它。

上一节我们提到了个人体验与普遍性，本节中我们来看看BDSM的正式定义与核心概念。

虐恋在英文中涉及S（施虐者）和M（受虐者）等词汇，涵盖了捆绑、鞭打及其他调教形式。其正式名称是BDSM，即**B**ondage（捆绑）、**D**iscipline（纪律）、**D**ominance（支配）、**S**ubmission（服从）的缩写。

S和M实际上是多个词汇首字母缩略后的混合泛称。中国社会学家李银河在《虐恋亚文化》中描述：**BDSM是一种将快感与痛感联系在一起的性活动，或者说是一种通过痛感获得快感的性活动**。我认同这一描述。

但在实践中，我们通常不会施加难以忍受的巨大痛苦。绝大多数时候，只是通过象征性的方式让参与者获得压迫感，进而转化为快感。这是一种类似感觉转换的行为。

我们通过看似痛苦的方式来换取兴奋与快感。例如，S（攻方）通过捆绑进行控制与管教，看着M（受方）被束缚、扭动和挣扎的表情。而M则通过被S命令、屈服或羞辱来获得内心的满足。这里的“痛苦”不单纯是肉体上的，也包括精神上的。

以下是几种常见的精神层面“痛苦”形式：
*   语言上的羞辱
*   行为上的限制
*   心理上的压迫

一次良好的BDSM体验能让双方都获得满足。对此不理解的人可能会问：为什么会有人喜欢这种痛苦的方式？

目前我无法完全解释这个问题，现有的专业性学著作也仅记录了诸多学者的观点，并未给出确切答案。但事实就是如此，有时确实令人琢磨不透。
