# 丁丁按摩训练课程：17：推肾俞

## 概述
在本节课中，我们将要学习一种名为“推肾俞”的持久训练方法。该方法通过特定的按摩动作配合呼吸与肌肉收缩，旨在达到特定的训练效果。我们将分步解析动作要领与注意事项。

## 教程内容

首先，取几滴精油于手掌，双手互相搓动直至精油微热。

接着，将双手手掌并拢，平放在腰后部区域。

然后，双手同时施加向下压力，并沿着从腰部到尾椎骨的方向，进行上下来回的推按动作。推按速度应保持平缓，不宜过快。

在进行推按的同时，需要配合动作的频率，有节奏地快速收紧肛门肌肉，随后立即放松。这个收缩与放松的节奏应与推按动作同步。

以下是该步骤的关键执行要点列表：

*   在手掌涂抹并搓热少量精油。
*   双手并拢，按压于腰后。
*   沿腰部至尾椎方向，以适中速度上下推按。
*   配合推按节奏，同步进行快速的肛门收缩与放松。

此套连续动作需持续进行两分钟。

## 总结
本节课中，我们一起学习了“推肾俞”训练的核心步骤。该方法要求将手掌搓热后推按腰骶部位，同时有节奏地配合肛门肌肉的收缩与放松，整个练习需持续两分钟。请确保动作准确、节奏稳定以达到预期效果。
