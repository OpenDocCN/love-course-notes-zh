## 核心手爱手法
当前戏将兴奋感带动起来后，就可以开始正式的手爱环节了。先在手上涂抹少量润滑剂，确保覆盖阴茎和睾丸。接下来，我们介绍几种体验较好的核心手法。

2.  **钻木取火法**：用两个手掌夹住阴茎，然后来回搓弄旋转，模仿古人钻木取火的动作。
    *   `动作描述：双掌合十夹住阴茎 -> 前后搓动`
3.  **十指相扣作揖法**：双手十指相扣，用手掌将阴茎夹住，然后上下套弄，像传统作揖行礼的动作。
    *   `动作描述：十指相扣成杯状 -> 包裹阴茎 -> 上下运动`
4.  **反向螺旋法**：一只手握住阴茎体，另一只手握住龟头，然后像拧毛巾一样向相反方向轻轻螺旋拧动。
