# Python入门：01：Python基础语法与变量 🐍

在本节课中，我们将要学习Python编程语言的基础语法，包括如何编写简单的代码、理解注释的作用以及掌握变量的概念和使用方法。这些是开启Python编程之旅的基石。

## 1. 第一个Python程序

学习任何编程语言，通常都从编写一个简单的“Hello, World!”程序开始。这个程序会在屏幕上显示一行文字。

以下是实现这个功能的Python代码：

```python
print("Hello, World!")
```

在这行代码中，`print` 是一个内置函数，它的作用是将括号内的内容输出到屏幕上。我们传递给 `print` 函数一个文本字符串 `"Hello, World!"`，它就会被显示出来。字符串必须用引号（单引号 `'` 或双引号 `"`）包围。

## 2. 代码注释

注释是写在代码中但不会被计算机执行的文字，它的主要目的是向阅读代码的人（包括未来的你自己）解释代码的功能和逻辑。良好的注释习惯能让代码更易于理解和维护。

Python中有两种主要的注释方式：

以下是两种注释的示例：

*   **单行注释**：以 `#` 符号开头，其后的内容在本行内被视为注释。
    ```python
    # 这是一个单行注释，这行代码会输出问候语
    print("Hello, World!")  # 这行代码打印了Hello, World!
    ```

*   **多行注释**：可以使用三个连续的单引号 `'''` 或双引号 `"""` 将多行内容包围起来作为注释。
    ```python
    """
    这是一个多行注释的示例。
    它可以跨越多行，
    常用于为函数或整个文件添加说明文档。
    """
    print("程序开始执行。")
    ```

## 3. 变量与数据类型

变量是编程中用于存储数据的容器。你可以把变量想象成一个带标签的盒子，标签就是变量名，盒子里装的东西就是数据。

### 3.1 变量的命名与赋值

在Python中，使用等号 `=` 为变量赋值。等号左边是变量名，右边是要存储的值。

以下是变量赋值的基本语法：

```python
variable_name = value
```

例如：
```python
message = "Hello, Python!"  # 将字符串“Hello, Python!”赋值给变量message
number = 42                 # 将整数42赋值给变量number
pi_value = 3.14159         # 将浮点数3.14159赋值给变量pi_value
```

变量名可以随后在代码中使用，Python会自动引用其存储的值。

```python
print(message)  # 这行代码会输出：Hello, Python!
```

### 3.2 基本数据类型

Python中每个值都有一种数据类型，它决定了该值可以进行的操作。最常见的基本数据类型有以下几种：

以下是Python基本数据类型的简要介绍：

*   **整数 (`int`)**：表示没有小数部分的数字，如 `-10`, `0`, `100`。
*   **浮点数 (`float`)**：表示带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
*   **字符串 (`str`)**：表示文本数据，由一系列字符组成，需要用单引号或双引号括起来，如 `"hello"`, `'Python'`。
*   **布尔值 (`bool`)**：表示逻辑值，只有两个：`True`（真）和 `False`（假）。它们通常用于条件判断。

你可以使用 `type()` 函数来查看任何值或变量的数据类型。

```python
print(type(10))        # 输出：<class 'int'>
print(type(3.14))      # 输出：<class 'float'>
print(type("Text"))    # 输出：<class 'str'>
print(type(True))      # 输出：<class 'bool'>
```

### 3.3 变量的重新赋值

变量的值是可以改变的。通过再次使用赋值语句，可以为同一个变量名赋予一个新的值。新值的数据类型可以与之前不同。

```python
x = 5      # 最初，x是整数5
print(x)   # 输出：5

x = "现在我是字符串了"  # 现在，x被重新赋值为一个字符串
print(x)               # 输出：现在我是字符串了
```
