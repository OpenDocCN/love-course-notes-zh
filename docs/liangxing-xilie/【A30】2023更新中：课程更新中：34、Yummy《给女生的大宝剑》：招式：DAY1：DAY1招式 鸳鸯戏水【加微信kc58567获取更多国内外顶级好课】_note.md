# 亲密关系课程：34：给女生的大宝剑：DAY1招式 鸳鸯戏水 🛁

在本节课中，我们将学习亲密互动中的一个经典场景——“鸳鸯戏水”的具体操作步骤。这个环节注重细节与体贴，能有效提升双方的体验感。

上一节我们介绍了前期的准备，本节中我们来看看如何在浴室环境中进行“鸳鸯戏水”。

当双方进入浴室后，首先需要调节水温。确保水温适宜，避免过冷或过热。

以下是具体的操作步骤列表：

1.  使用莲蓬头将两人身体打湿。
2.  冲洗时，可用手掌或莲蓬头略微遮挡对方面部方向。
3.  这个动作能防止水流直接溅到对方脸上。

不要小看这些细节。这些细致的举动会让对方感受到你的体贴和照顾。

本节课中我们一起学习了“鸳鸯戏水”的基本步骤，其核心在于通过调节水温和遮挡面部水流等细节，营造体贴、舒适的共享沐浴体验。
