# 李氏养生功法：A30：控精术详解

在本节课中，我们将要学习控制射精的核心技术——控精术。我们将从基础的外控经开始，逐步深入到高级的内控经练习，帮助你掌握延长性行为时间、提升控制能力的方法。

## 概述

控精术是解决早泄与射精过快问题的关键技巧。它分为**外控经**与**内控经**两部分。外控经侧重于利用身体外部动作与肌肉控制来抑制射精冲动，是入门基础。内控经则专注于锻炼小腹内部肌肉（如PC肌）与神经系统，是更高级、更根本的控制方法。两者结合使用，效果最佳。

## 外控经：身体动作控制法

外控经的核心思想是：**通过全身动作来控制射精**。它利用身体外在的动作和肌肉紧张，来分散和转移性兴奋，从而压制射精欲望。这是初级的、借助外力的控制方法。

当通过性交或自慰达到即将射精的临界点时，请立即按以下步骤操作。以下是外控经的六个连贯动作：

1.  **深吸气并闭气**：立即长长地吸一口气，闭住，并将气感引至小腹。
2.  **按压会阴穴**：用左手用力按住会阴穴（肛门与阴囊之间的位置）。
3.  **捏紧阴茎根部**：用右手的拇指、食指和中指平均用力，捏紧阴茎的根部。
4.  **收缩PC肌与全身紧绷**：用力收缩PC肌（提肛），同时瞪大眼睛，使颈部、手部、腿部及腹部肌肉全部紧张僵硬。
5.  **强化收缩**：加强力量，用力收缩肛门、阴部以及整个小腹，随吸气向上提拉，持续不放松，尽力将射精感压制下去。
6.  **上挺阴茎**：伴随每一次吸气，将阴茎向上挺起；呼气时让其回落。

**关键要点**：以上六个动作需几乎在同一时间内完成，并且必须忍住绝不射精。当第一次射精感被压制后，应继续刺激直至再次达到临界点，并重复上述动作将其压下。如此反复，直至阴茎疲软，过程中始终避免射精，以建立信心。

古代典籍对此有精要总结：“缘经欲走，把金愁。守都向纳钳与钩。二阴加紧强忍地。登目缩颈一元猴。” 其意为：当射精感来袭时，立即抽出阴茎，手脚紧张如钩，肛门阴部加紧强忍，瞪眼缩颈，状如猿猴。

## 内控经：腹内肌肉控制法

上一节我们介绍了借助外力的基础方法，本节中我们来看看更高级的内控经。其含义是：**通过腹内动作控制射精**。它只练习小腹内部的收缩，是控制射精最基本且最有效的高级手段，核心在于锻炼PC肌的力量与神经控制能力。

内控经的练习不要求必须在性兴奋时进行，日常生活中随时可练。姿势不限，站立、躺卧或端坐皆可。我们将其分为**基本练习**与**强力练习**。

### 基本练习

以下是内控经的三种基本呼吸与收缩配合方法，请跟随指引练习：

*   **半拍子法**：一吸一呼为一个循环。吸气时，收紧肛门、PC肌及小腹；呼气时，彻底放松。
    *   指令：吸气，收紧——呼气，放松。
*   **一拍子法**：一个完整的呼吸（吸+呼）只做一个动作。吸气并保持收紧贯穿整个吸气和呼气过程，然后在下一个呼吸循环中放松。
    *   指令：吸-呼（收紧）—— 吸-呼（放松）。
*   **多拍子法**：难度提升。用多个呼吸缓慢完成收紧与放松。例如，用三次呼吸逐渐达到最大收紧度，再用三次呼吸逐渐完全放松。
    *   指令：呼吸一（稍紧）—— 呼吸二（更紧）—— 呼吸三（最紧）—— 呼吸四（稍松）—— 呼吸五（更松）—— 呼吸六（全松）。

### 强力练习

在掌握基本练习后，可以进行更强化的训练，以提升控制力。

*   **慢收快放**：用较长时间（如5次呼吸）缓慢地将收缩力度增至最强，然后在第6次呼吸时突然彻底放松。重复此过程3次。练习时，双脚脚趾可抓紧地面。
*   **快收慢放**：在第1次呼吸时迅速收紧至最强并再加一点力保持，随后用较长时间（如第2至第6次呼吸）缓慢地、逐渐地放松。重复此过程3次。

内控经的训练需要耐心与持之以恒，请认真琢磨并坚持练习。

## 总结

本节课中我们一起学习了控精术的两大组成部分：**外控经**与**内控经**。外控经是在射精临界点通过一系列外部身体动作来紧急制精；而内控经是通过日常对PC肌及腹内肌肉的收缩锻炼，从根本上提高神经控制阈值与制精能力。

坚持修炼，将两者辩证结合，方能显著提升控制力。赠诗共勉：“认认真真练养田，性力回到十年前。坚持养完一百天，能力胜过而立年。”
