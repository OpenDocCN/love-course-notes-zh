# 夫妻保健按摩教程：A27：柔式保健按摩入门指南

在本节课中，我们将系统学习柔式保健按摩的基本概念、特点、作用以及一套完整的操作流程。柔式保健按摩是一种源自我国沿海地区、手法独特的保健方法，旨在通过柔和细腻的按摩促进全身放松与健康。

## 概述与基本原理

柔式保健按摩是一种风靡于我国沿海地区的按摩手法，随着按摩保健业的发展而传入内地。

其手法独特，核心原理在于通过按摩一点来带动胰腺，进而以胰腺激活全身，使全身放松，肌肉伸展，达到舒筋活络、气血畅通的效果。

在此过程中，毛细血管得到充分扩张，细胞也能吸收充足营养。这种方法既能辅助医治疾病，又能达到保健强身的效果。

## 柔式按摩的特点与作用

上一节我们介绍了按摩的基本原理，本节中我们来看看这种按摩手法的具体特点和作用。

柔式保健按摩主要特点是以手掌与皮肤的柔和韵律为主，能较快减轻全身疲劳。其手法柔和、细腻、有力且渗透，可使肢体脏腑活性化，促进新陈代谢。

## 仰卧位按摩步骤

了解了按摩的特点后，我们从仰卧位开始，学习具体的操作步骤。仰卧位按摩从头面部开始，逐步向下进行。

### 头面部按摩

以下是头面部按摩的具体手法，旨在放松面部、促进循环。

1.  **抚摸面部**：用双手交替轻轻抚摸被按摩者面部。此动作需轻柔，可使面部肌肉放松，促进血液循环。
2.  **点揉面部穴位**：使用指腹点揉印堂穴、太阳穴、听宫穴、颊车穴、地仓穴、承浆穴、人中穴、迎香穴、睛明穴等穴位。此法能放松面部皮肤，减轻疲劳，促进血液循环。
3.  **推揉面部**：双手推揉被按摩者的鼻翼、口周、耳周、眼周、颧周区域。能促进血液循环，具有美容美肤的效果。
4.  **点揉双眼**：双手四指呈环形，轻柔地按揉被按摩者的眼部。能减轻眼部疲劳，具有明目美目的效果。
5.  **刮运前额**：双手交替，用食指、拇指侧面刮运被按摩者前额。能帮助减少前额皱纹。
6.  **捏揉双耳**：用中指和食指揉捏被按摩者耳部。可调节人体内脏功能。
7.  **夹揉双耳**：用双手手掌夹住被按摩者耳部，进行揉动。夹持力与揉动力需配合得当，揉运速度要适中。能调节耳部血液循环。
8.  **梳理头部**：双手沿被按摩者头部，自上而下往返梳理。动作需流畅、有节奏。能益肾安神，适用于肝肾不足所致的头晕、耳鸣及神经衰弱等。

### 上肢部按摩

完成头面部按摩后，我们转向上肢。上肢按摩旨在通经活血，缓解肩颈与手臂不适。

1.  **点按上肢穴位**：用大拇指指腹依次缓慢点按上肢相关穴位。能通经活血，适用于肩周炎、颈椎病、落枕。
2.  **揉拿上肢肌肉**：双手依次缓慢揉拿被按摩者上肢内外侧肌肉。适用于上肢酸痛、屈伸困难等。
3.  **按揉上肢**：用手掌依次从被按摩者肩部到腕部，反复按揉。能舒筋活血，适用于四肢酸痛麻木。
4.  **揉按肩颈部**：用双手指腹用力揉按被按摩者肩颈部位。具有松解粘连、解除疲劳、放松肌肉、舒筋活络的效果。
5.  **叉摇手腕**：一手四指与被按摩者四指交叉，缓慢摇动其腕关节。适用于腕关节疼痛等，可达到舒筋活血的目的。
6.  **搓揉腕部**：用双手掌跟夹住被按摩者腕关节根部前后侧，做反复搓揉。可以通关利结，适用于腕关节扭挫伤等。
7.  **推按掌心**：两手拇指在被按摩者掌心处，分别向手掌两侧推摩，同时向下点按掌心。能清热化浊，理气止痛。
8.  **摇抖上肢**：一手握住被按摩者的手腕部，另一手握住肘部，顺时针或逆时针方向摇转，并做上下抖动。可疏松脉络，滑利关节，适用于肩臂酸痛、活动不利等。
9.  **理顺上肢**：用手掌依次从肩部至腕部梳理上肢。具有放松手臂肌肉的效果。

### 胸腹部按摩

理顺上肢后，按摩进入胸腹部。这部分按摩侧重于调理脏腑功能，促进消化。

1.  **压放胸部穴位**：用手指指腹沿胸部中线，依次压放天突穴、膻中穴、中脘穴。具有健脾舒肝、温阳益肾的作用。
2.  **揉运胸部穴位**：用手指指腹沿胸部中线，揉运天突穴、膻中穴、中脘穴。适用于食欲不振、大便秘结等。
3.  **震颤胸部穴位**：用手指指腹沿胸部中线，震颤天突穴、膻中穴、中脘穴。具有健脾舒肝、温阳益肾的作用。
4.  **按揉腹部**：用拇指指腹按揉被按摩者腹部周围，用力不宜过大。适用于腹胀、腹痛、食欲不振等。
5.  **掌摩腹部**：用双手扣在被按摩者的腹部，进行环状轮压揉摩。有利于消化吸收，促进新陈代谢。
6.  **震颤腹部**：用双手掌快速搓热，捂在被按摩者腹部，施加震颤手法。能增强肠胃蠕动，健脾和胃。
7.  **点按腹部穴位**：用中指轻轻点按气海穴、关元穴。可增补元气，温阳益肾。

### 下肢部按摩（仰卧位）

最后，在仰卧位进行下肢按摩，以缓解下肢疲劳与不适。

1.  **点揉下肢**：用手指从上到下依次点揉被按摩者下肢。动作需轻柔连贯。具有滑利关节，缓解下肢麻木之功效。
2.  **掌揉下肢**：用手掌按揉被按摩者的下肢，力度要适中。可辅助治疗风湿、腿痛等。
3.  **抱揉下肢**：用双手交替抱住被按摩者下肢部位进行揉运，动作要连贯。对因血流不畅引起的大腿冰冷等症状有缓解作用。
4.  **理顺下肢**：用手掌掌面轻抚被按摩者下肢，然后开始摇运。具有松肌解筋之功效。

## 俯卧位按摩步骤

仰卧位按摩完成后，请被按摩者转为俯卧位。俯卧位按摩主要针对头、背、腰及下肢后侧。

### 头颈肩背部按摩

我们从头部开始俯卧位的按摩。

1.  **点揉头部穴位**：用手指指腹点揉被按摩者头部的百会穴、角孙穴、风池穴、风府穴。具有健脑安神的作用。
2.  **干梳头**：双手五指重复梳理被按摩者头部。有放松醒脑、安神、促进血液循环的作用。
3.  **合揉颈部**：双手以拇指和其他四指合力揉捏其颈部肌肉。具有放松肌肉的作用。
4.  **掌揉肩部**：用双手掌根揉压被按摩者肩部，操作时用力要轻柔和缓。能温经止痛，适用于肩周炎、颈椎病等。
5.  **分推后背**：用双手分推被按摩者背部。能放松肌肉，促进血液循环。
6.  **揉滚脊柱两侧**：用双手反复滚揉被按摩者脊柱两侧肌肉。要利用腕关节活动带动手的滚动，有疏松筋骨、整理脊柱的作用。
7.  **震颤腰部**：双手相叠，用力压在被按摩者腰眼部位进行有节奏的颤动。可以通经活血，温补脾肾，适用于腰背酸痛等。

### 下肢后侧按摩（俯卧位）

最后，在俯卧位完成下肢后侧的按摩。

1.  **肘揉臀部**：用肘部依次揉运被按摩者臀部肌肉。能舒筋活血。
2.  **点按下肢穴位**：点揉被按摩者下肢的承扶穴、委中穴、承山穴、昆仑穴、太溪穴等穴位。具有通经活血的效果，适用于腰椎间盘突出症、偏瘫、膝关节炎等。
3.  **掌揉下肢膀胱经**：让手掌重叠，沿被按摩者下肢后面足太阳膀胱经的循行路线，从上到下进行按压揉动。能通经活血，适用于偏瘫、腰腿痛等。
4.  **揉运腘窝**：使被按摩者下肢屈曲，操作者用大鱼际揉运其腘窝。可以滑利关节，顺理筋经，增强膝关节灵活性。
5.  **理顺下肢**：用单手掌或双手掌抚摸被按摩者下肢后侧，使其肌肉得到放松。能舒筋活血。

## 结束手法：站位反背

所有卧位按摩结束后，可采用一个独特的结束手法。

操作者反方向站立于被按摩者后方，两上肢与其上肢交叉，弯腰将其背起，并进行左右轻轻晃动。此手法有助于进一步舒展脊柱，放松全身。
