# 李氏养生功法：第三课：养肾功核心八式详解

在本节课中，我们将学习养肾功的八个核心动作，从第十二式到第十九式。这些动作旨在通过特定的按摩与呼吸方法，促进肾区及生殖器官的血液循环，以达到养护与强健的目的。请准备好纸笔，认真记录动作要领，并用心体会。

## 概述
本节课程将详细讲解搓骨沟、揉耻骨、碾睾丸、压睾丸、揉睾丸、滑睾丸、吸睾丸、挂睾丸这八个核心动作。每个动作都需配合特定的呼吸方式，操作时务必注意力道，以微有感觉但不造成疼痛为原则，循序渐进。

---

### 第十二式：搓骨沟

上一节我们结束了第十一式，本节开始学习第十二式“搓骨沟”。此动作旨在按摩腹股沟区域，促进该部位的血液循环。

**动作要领**：
1.  用双手搓揉腹股沟。
2.  由于腹股沟与胫骨形成角度，手掌需贴合此角度，确保上下均能搓到。
3.  搓揉时需紧贴阴部，向阴部靠拢。
4.  手指尖应触摸到会阴处，并对会阴进行揉托，以养护前列腺。

**核心原理**：
位于阴茎根部两侧的腹股沟管，是向睾丸输送血液和连接神经的通路。若腹股沟管机能低下，必然引起睾丸功能低下。因此，搓腹股沟是促进血液循环的重要环节。

**操作要求**：
采取腹式呼吸。搓揉约50次，对应6次呼吸。

---

### 第十三式：揉耻骨

完成了对腹股沟的按摩后，本节我们来看看对阴部上方骨骼的护理——揉耻骨。此式是对搓骨沟的补充，旨在使阴部周围血液畅通。

**动作要领**：
1.  耻骨位于阴茎根部的上端。
2.  搓揉时，手按在阴茎根上方，应有意识地让阴茎在搓揉过程中随之跳动晃动。
3.  左右手交替进行。

**操作要求**：
采取腹式呼吸。左右手各做25次，总共6个呼吸。

---

### 第十四式：碾睾丸

在疏通了外部通路后，本节开始直接对睾丸进行养护。第一个方法是“碾睾丸”。

**动作要领**：
1.  将两只睾丸置于两手掌心握住。
2.  用大拇指指腹碾压睾丸。以拇指大关节为轴心，来回摆动拇指进行碾压。
3.  其余四指（主要是食指和中指）在下方托住睾丸。
4.  力度以稍有一点痛感为宜，不宜过重。
5.  碾压时注意把握睾丸，避免其滑脱造成突然疼痛。

**操作要求**：
采取腹式呼吸。碾压约50次，对应6次呼吸。

---

### 第十五式：压睾丸

学习了碾法后，本节我们学习另一种力度控制更精细的方法——压睾丸。

**动作要领**：
1.  食指、中指在下，拇指在上，将睾丸置于其间。
2.  吸气时，拇指指腹下压睾丸，同时整体（睾丸和阴囊）平行向外转动。此时应有轻微的疼痛麻胀感，但不可用力过猛。
3.  缓缓呼气时，放松指压。
4.  下压时间与吸气时间等长。

**操作要求**：
关键在于感受压力的承受范围，由轻到重，切勿超出承受能力造成伤害。吸气下压，呼气松开。

---

### 第十六式：揉睾丸

为了让睾丸得到更全面的按摩，本节我们学习“揉睾丸”，此式可照顾到睾丸的各个侧面。

**动作要领**：
以下是具体步骤：
1.  先用左手从下方握起阴茎与睾丸，使睾丸凸显于阴囊中。
2.  用右手掌心（劳宫穴）画圆按摩右侧睾丸50次（腹式呼吸）。
3.  换右手握持，左手以同样方式按摩左侧睾丸50次。
4.  紧接着，用左手掌心同时按摩两个睾丸的上半部25次。
5.  再按摩两个睾丸的下半部25次。

**操作要点**：
按摩时手掌需紧贴阴囊皮肤，可稍用劲，但避免悬空搓动挫伤皮肤。务必学会保护器官，不可用蛮力。

---

### 第十七式：滑睾丸

对睾丸进行了按压和揉按后，本节我们通过“滑睾丸”来活动睾丸两侧。

**动作要领**：
1.  将两只睾丸分别置于两手的拇指与食指之间。
2.  其余三指蜷回掌心，在后方作为支撑。
3.  通过拇指与食指的挤动，使睾丸在其间上下滑动。
4.  滑动时可稍加用力，以睾丸有隐隐疼痛感为度。

**操作要求**：
初学者万万不可用力过猛。采取腹式呼吸。一般一次呼吸滑动约10次，总共10个呼吸，约滑动100次。

---

### 第十八式：吸睾丸

本节我们将学习一个特别的提升动作“吸睾丸”，此式主要采用胸式呼吸，旨在锻炼提睾肌。

**动作分两步**：
**第一步**：
1.  采取胸式呼吸（本节首次使用）。
2.  吸气时，向上提肛，同时用力将睾丸向上吸提，两手也配合微微上提。此时需瞪大眼睛，脚掌抓地。
3.  呼气时，让睾丸及肛门整体向下坠落。
4.  一呼一吸为一次，共做9次。

**第二步**：
1.  完成9次后，肌力有所消耗，此时用双手辅助。
2.  吸气时，用两手将睾丸托起，并向上塞入腹股沟中，让睾丸暂时脱离阴囊，停留几秒。
3.  呼气时，两手撤离，让睾丸自由坠落。
4.  其他要求（瞪眼、脚掌抓地）与第一步相同。
5.  同样做9次。

**总计**：整个吸睾丸动作为18次呼吸。

---

### 第十九式：挂睾丸

最后，我们来学习模拟传统挂重训练的“挂睾丸”，此式利用自身力量形成拉力，相对安全。

**动作要领**：
1.  左手掌心向下，用食指与拇指箍住阴囊阴茎的根部。
2.  右手以同样手势压在左手之上。
3.  身体呈弓步（马步状）。
4.  双手下压阴部，同时腿部向上蹬起，形成上下对抗的拉力，起到挂拉的作用。
5.  此动作做50次（可采用腹式呼吸，也可不必太在意呼吸）。
6.  然后换右手在下，左手在上，再做50次。

**安全提示**：此动作模拟传统“挂沙袋”，但避免了负重不当造成的拉扯伤害。

---

### 实操要点回顾与总结

在本节课中，我们一起学习了养肾功的八个核心动作。现在，我们对一些关键实操要点进行回顾：

**关于呼吸的纠正**：
在上一课的“勾金锁”动作中，当锁紧至顶部、睾丸被彻底挤到上方时，应在**吸气时提肛**，**呼气时放松**肛部，而非呼气时提肛。

**各动作实操核心**：
以下是每个动作需特别注意的细节：
*   **搓骨沟**：手掌需适应腹股沟角度，一半贴骨沟下方，一半贴上方。尽量靠拢阴部，中指应触及会阴穴按摩前列腺。需直接接触皮肤进行。
*   **揉耻骨**：位置在阴茎根部正上方。揉动时用小指外侧卡住阴茎根，使其随揉动而跳动，从而按摩阴茎平滑肌。
*   **碾睾丸**：用大拇指指腹以关节为轴碾压。初期无需追求强力度，应感到微麻胀，后腰有反应即可。
*   **压睾丸**：吸气下压并外转，呼气放松。务必在自身承受范围内进行，感觉由轻到重。
*   **揉睾丸**：按摩时手掌必须紧贴皮肤，避免悬空搓动损伤阴囊。目的是全方位养护，促进睾丸内部血液循环与激素分泌。
*   **滑睾丸**：通过拇指与食指的挤压，让椭圆形的睾丸在指间上下滑动，以活动其两侧。可稍快进行，完成约100次。
*   **吸睾丸**：第一步用胸式呼吸和气力上提；第二步用手辅助将睾丸推入腹股沟并短暂停留。目的是锻炼PC肌（提肛肌）。
*   **挂睾丸**：利用弓步下压与身体上蹬形成的对抗力来拉伸。注意力度，勿用猛力造成拉伤。

**总结**：
本节课传授的“搓、揉、碾、压、滑、吸、挂”等方法是养肾功的核心功法。请大家耐心领会，细细琢磨。训练中务必以保护器官为首要原则，循序渐进，待有一定承受力后再逐渐加大力度。持之以恒，方见成效。
