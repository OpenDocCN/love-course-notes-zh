# 清肺内功：2.10：清肺内功第二式详解 🫁

在本节课中，我们将要学习清肺内功的第二式。这一式主要涉及特定的运气方法和身体姿势，通过协调呼吸与动作来达到强身健体的效果。我们将分解动作要领，并解释其背后的原理。

## 动作分解与运气原理

上一节我们介绍了基础概念，本节中我们来看看第二式的具体做法。整个练习过程的核心在于“运气”，这与我们日常生活中的用力原理相通。

例如，当生活中需要抱起重物时，我们会先蹲下、双手抓牢，然后瞬间发力。这个功法正是源于此类生活经验。在运气的状态下，我们的力量能达到最大。

以下是第二式的姿势要点：
*   采用弓步。前腿弓，后腿绷直。
*   两脚方向均呈45度角。
*   身体转向正前方。

## 动作流程详解

接下来，我们学习详细的动作流程。一共包含四个部分。

以下是动作的四个步骤：
1.  第一个动作。
2.  第二个动作。
3.  第三个动作（只有这个动作的手掌是向上的，其余都是向下的）。
4.  第四个动作。

动作学会后，开始结合呼吸。吸气，双手抱球、翻掌并向后拉。当拉至身体两侧时，分三次将气息吐出。

然后换另一侧腿进行练习，确保身体两侧都得到锻炼。

## 运气技巧与身体协调

在运气时，有一些关键细节需要注意。抱球后，上半身要微微前倾。回拉时，同样要求脚趾抓地。

此时，前腿与后腿有一个向内合拢的劲力。向后拉到体侧后，分三次将气呼出。整个运气过程中，必须保持头顶悬、脚趾抓地的状态。

## 常见错误与纠正

现在大家开始练习，我来观察并纠正。手的动作位置很重要，不是停在某处，而是要持续向后拉。

手要像抓住东西一样，用力向后拉。手指要勾回来，再用力。关键是要绷紧身体，用力向后拉。

## 练习心得与休息

可以休息一下。我们过去练武时，师兄弟们经常互相突然地点按。这就要求身体随时保持警觉：要么绷紧，要么及时闪开。

这养成了快速反应的习惯。如果绷不紧，就会被点到。而且我们点的不是普通部位，是特定的穴位。

稍微休息一下。这个动作对身体各部位的区分和劲力要求非常细致、严格。
