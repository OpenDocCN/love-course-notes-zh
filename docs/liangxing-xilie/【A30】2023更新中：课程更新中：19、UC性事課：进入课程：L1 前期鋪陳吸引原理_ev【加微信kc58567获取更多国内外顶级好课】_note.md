# 两性关系课程：L1：前期铺垫与吸引原理 🎯

在本节课中，我们将学习如何为建立亲密关系打下基础，核心是理解并运用“吸引力”与“安全感”两大原则。我们将探讨如何找到性观念合适的对象，以及如何通过提升自身价值来增强性吸引力。

## 吸引、安全与升级：关系的基石 🎼

上一节我们介绍了课程的整体框架，本节中我们来看看关系建立的核心要素。我们可以把建立关系想象成爬一座旋转楼梯。楼梯顶端有一份吸引人的礼物，让人好奇并想得到。爬楼者会思考楼梯是否安全，安全才会让人愿意向上爬。爬楼的速度快慢，也与安全感直接相关。

因此，关心的三个核心要素是：吸引、安全和升级。关系的升级建立在吸引力和安全感这两个前提之下。

我们一开始需要做的事情就是吸引。每一段关系都应是两情相悦、互相吸引的结果。通过学习，让自己变成一份吸引人的礼物，去吸引自己想要的人，这才是关系美好的样子。礼物之所以是礼物，是因为在这个世界上，没有人不喜欢礼物。

我们的性事课会将焦点放在性吸引上。如何制造性吸引力，就是本单元的重点。

## 如何找到性观念开放的对象？ 🔍

很多同学曾问过：“怎么找到性观念开放的女生？” 正如前面提到的，我们要去吸引自己想要的人。性爱话题在当今社会逐渐开放，但真要深入讨论价值观时，对大部分人来说仍难以启齿且害羞。正因如此，在性事发展上常会遇到阻碍，人们常常错过合适的对象。

因此，这里将教大家如何找到性观念合适的对象。我们邀请女嘉宾佑佑来分享观点。

找人主要有两种方法：明示和暗示。两种方法后续导向不同，明示做的是筛选，暗示做的是引导。

### 方法一：明示筛选

明示筛选是指在交友软件上明确表示，以筛选出性观念开放的对象。有些人会选择特定平台进行筛选，因为每个平台的规范和属性不同。

以下是选择平台的步骤：
1.  开始时可以设定为男女皆可配对，以观察各交友软件的属性。
2.  根据观察，选择你想要的平台。在合适的平台上，找到观念类似的人会容易很多。

大部分明示的方法会在个人资料页面上花功夫。以Tinder为例，可以放上帅照、肌肉照等展示身材来吸引人，并在自我介绍区用文字明确表示自己的取向，例如“寻找ONS”或“寻找FWB”等。

这么做的好处是可以直接找到性观念开放且需求匹配的对象。但既然是筛选，条件明确自然会过滤掉大部分人，配对率可能降低。因此建议可以先充值会员，提高曝光率，并让每日配对次数无上限。

### 方法二：暗示引导

暗示引导则是在个人页面上使用暗示性字眼引发对方好奇。例如使用“Netflix and chill”这类带有性暗示的词汇，或者使用相关歌词。在聊天中，可以引导对方谈论爱情观和性爱观，以确认彼此是否合适。

当然，个人页面也需要精心设计。与明示相比，暗示需要更多心思去引导，不像明示那样直接，但好处是配对率相对较高。

无论选择哪种方式，都请先诚实地问自己：你想寻找什么样的关系？不要口是心非。如果你想要随意的关系，却在资料里写寻找长期稳定关系，这就是欺骗。请诚实面对自己和他人。

## 提升性吸引力：繁衍价值与情绪价值 💪

如果你想要有性事的交流但不知如何开始，那么性吸引力就非常重要。在亚洲社会观念中，女生相对被动，因此男生的主动引导和吸引就非常必要。无论单身还是处于一段关系中，展现性吸引力都能帮助你顺利推进关系。

在UC系统中，男女吸引力有四大价值：生存价值、社交价值、繁衍价值和情绪价值。本单元我们主要讨论繁衍价值和情绪价值。

### 什么是繁衍价值？

从进化心理学角度看，两人建立关系的主要目的是繁衍后代，并期望后代健康、有竞争力。因此，肉体是性吸引力的重要指标之一。男生的外表很大程度上决定了女生是否会喜欢。

很多人会问：“如果我不高不帅怎么办？” 先天条件确实无法改变，但我们可以通过后天努力提升繁衍价值。简单来说，繁衍价值就是：**好看、有活力**。

以下是几种提升繁衍价值的方法：

**1. 健身**
健身是改善体态和身形最根本的方法。通过运动和训练，可以让体态更健康，提升活力。初期可能难以养成习惯，但只要坚持三个月以上，你一定会发现自己变得更健康、更好看。可以将运动照片发在交友软件上提升吸引力。

**2. 穿搭与造型**
如果体态改善需要时间，穿搭是改善外形最快的方法。合适的穿搭可以提升整体气质和体态。此外，发型也很重要，适合的发型可以衬托五官，让人看起来更有精神。也可以和造型师讨论是否留胡子来修饰脸型，增添男性魅力。

**3. 保养**
光滑的皮肤比较吸引人，因为皮肤的平滑程度会让人联想到健康。男性也可以保养皮肤，至少洗脸后使用化妆水和乳液保持湿润。睡眠和饮食同样重要。

**4. 户外活动**
与室内活动相比，户外活动更能展现健康活力。可以抽空进行户外运动并拍照，拍照时注意人与景的搭配，人最多占画面的三分之一，展现身形即可。

拍照后可以上传到社交软件展现生活，这会大大提升吸引力。拍照可以适当修图提升质感，但不要过度。

### 什么是情绪价值？

情绪价值是指如何让女孩在与你聊天时有情绪起伏的体验，就像坐云霄飞车。最重要的两个能力是同理心和感染力。

以下是在网络聊天和约会中提升情绪价值的简单方法：

**1. 善用语音和视频**
在网络聊天中，除了话题，最简单的情绪渲染方式是使用视频和语音。文字缺乏温度，而视频和语音能通过语调、表情和肢体语言传递更多信息，让对方更直接地感受到情绪流动。可以在文字聊天热络时切入使用，讲话时放慢速度、音调低沉。

**2. 眼神接触**
眼睛是灵魂之窗。双方凝视时会引发强烈的脑内活动，也会让人觉得对方更真诚、更专注。平时可以多练习眼神接触。如果约会时害羞，可以看对方的鼻子或单边眼睛。如果对方移开视线，不必在意，保持轻松自信即可。

**3. 保持正面态度与自在心态**
心态往往最重要但也最难做到。尤其是在聊天升温或邀约时被拒绝，很多人心态会崩溃。应对的准则是：**以朋友心态相处**。

可以想想平时和好朋友聊天打屁时的状态。如果讲笑话对方没笑，你会轻松带过；如果邀约被拒，你会说“下次再说”。与女生互动时最大的差别在于得失心和同理心。请记住，女生拒绝可能有各种原因（感觉没到、真的有事等）。只要保持正面、自在、尊重且同理对方，被拒绝时只需轻松回应，继续像朋友一样聊天即可。

关系升温就像钓鱼。鱼上钩时，应该让鱼游，等它游累了或向你游来时再收线。在关系中，可能一开始女生感觉不够，但通过聊天和相处，感觉会慢慢产生。
