# 肾虚系列课程：A30：你不知道的经络补肾法！💆‍♂️

在本节课中，我们将要学习如何利用人体自身的经络与穴位系统，通过简单的按摩与刮拭方法来补肾、调和阴阳，从而缓解亚健康状态与身体不适。经络是运行气血、联络脏腑的通道，正确刺激相关穴位能有效激发身体的自愈与平衡能力。

## 经络补肾概述

上一节我们介绍了改善生活方式来缓解肾虚，本节中我们来看看如何通过具体的经络操作来达到补肾养生的目的。经络是经脉和络脉的总称，是运行全身气血、联络脏腑形体官窍、沟通上下内外的通路系统。因此，按揉穴位或顺应经络进行养生，对于缓解和预防不适都有较好效果。

## 补肾阳的经络养生法

针对肾阳虚的情况，以下是三种推荐的操作方法，旨在激发和温煦身体的阳气。

**刮拭背部补阳气**
背部在中医理论中属阳。利用刮痧板刮拭背部可以激发背部阳气，进而带动全身阳气。操作时力度需适中，刮拭太过会导致背部毛孔开放过大，阳气容易外泄，反而达不到补阳效果。公式表示为：`适度刮拭背部 → 激发阳气`。

**搓内肾（搓腰）**
将两手手心互相搓热，然后用热手搓腰部两侧各36次。带脉之中为肾脏所在部位，常搓此处能温煦肾阳，畅达气血。中医认为，用掌搓腰眼不仅可以疏通带脉和强壮腰脊，还能起到聪耳明目、固精益肾和延年益寿的作用。

**双掌推腹**
首先仰卧，两手重叠，从腹中线向上压并向下推摩至下腹部，重复20次。然后分别从两侧肋骨下缘，向下推摩至大腿根部，同样重复20次。接着按顺时针方向按摩肚脐周围，并逐渐扩大至整个腹部，按摩2到3分钟。最后按逆时针方向以同样方法按摩腹部2到3分钟。这是一个较为完整的按摩养生过程。

## 补肾阴的经络养生法

介绍过了肾阳虚的按摩养生方式，就来跟大家说一下肾阴虚的按摩养生方式。肾阴虚主要表现为阴虚火旺，因此按摩方式应以平衡阴阳为主，而非单纯进补。

**刮拭肾经与膀胱经**
膀胱经是太阳经，阳气较旺；肾经则能起到平衡阴阳的作用。两经经气互通，通过刮痧可以互相影响，激发阴阳平衡的状态。操作时，只需选择背部，从颈椎一直刮拭至尾骨，刺激到膀胱经和肾经即可。可以分段多次进行。

**揉搓耳垂**
用手指轻捏耳垂，反复揉搓，并适度前后上下拉动，力度以不疼痛为限。每日2到3次，每次一分钟。具体操作：以双手拇指和食指分别提揉双耳垂半分钟，使其发红发热，然后揪住耳垂向下拉，再放手让耳垂弹回。此法有助于缓解肾阴虚火旺引起的耳鸣、耳聋、心神不安、失眠等症状。

**按摩三阴交穴与照海穴**
以下是两个关键的滋阴穴位按摩方法。

**三阴交穴**
此穴位于小腿内侧，胫骨后缘靠近骨边的凹陷处。经常用手按摩三阴交穴，对于补阴调和、增强功能有良好作用。代码式定位：`胫骨后缘，内踝尖上约4横指（3寸）处`。

**照海穴**
此穴被称为“大补阴穴”，位于脚内侧足弓弓背中部靠前的位置，可摸到一个骨节缝隙。用大拇指用力向下按压，然后马上放松。按压时，穴位周围乃至腿部肾经会有酸胀感，放松后消退。如此重复按压10到20次，双脚穴位都需按揉。

## 通用补肾调理法

如果分不清肾阴虚还是肾阳虚，可以使用以下几种通用的经络穴位按摩方式来缓解不适。

**刮涌泉穴**
用热水清洗足部后，用刮痧板从涌泉穴开始刮拭整个足底，以产生热感为宜。

**刮拭/按揉肾俞穴**
肾俞穴具有阴阳同补、激发脏腑精气的作用。按揉时可用手掌心或手指按压该穴10到15分钟，力度适中。也可用刮痧板顺时针从上至下轻轻刮拭，尽量不刮出痧。

**叩齿吞津**
集中精神，上下牙齿轻轻叩击36次。叩齿过程中口腔唾液会增多。中医认为唾液能滋养五脏六腑，现代医学也认可其促进消化、滋阴降火、生津补肾的重要作用。
