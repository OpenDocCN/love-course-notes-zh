# Python入门：01：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是如何工作的，是编写任何Python程序的第一步。

## 什么是变量？📦

变量是计算机内存中用于存储数据的“盒子”或“标签”。你可以把数据放进这个盒子里，然后通过标签（变量名）来找到并使用它。

在Python中，创建一个变量并给它赋值非常简单，只需要使用等号 `=`。等号左边是变量名，右边是你想存储的值。

```python
my_name = "小明"
my_age = 25
```

在上面的例子中，我们创建了两个变量：
*   `my_name` 这个变量里存储了文本 `"小明"`。
*   `my_age` 这个变量里存储了数字 `25`。

现在，每当我们使用 `my_name`，Python就知道我们指的是 `"小明"`。

---

上一节我们介绍了变量的基本概念，本节中我们来看看Python中常见的数据类型。数据类型决定了变量可以存储什么样的数据，以及可以对它进行哪些操作。

## 基本数据类型 🔢

Python有几种内置的基本数据类型，以下是其中最常用的几种：

### 1. 整数 (int)
整数就是没有小数部分的数字，可以是正数、负数或零。
```python
count = 10
temperature = -5
```

### 2. 浮点数 (float)
浮点数就是包含小数点的数字。
```python
price = 19.99
pi = 3.14159
```

### 3. 字符串 (str)
字符串是由一系列字符组成的文本。在Python中，需要用单引号 `' '` 或双引号 `" "` 将文本括起来。
```python
greeting = "你好，世界！"
name = 'Python'
```

### 4. 布尔值 (bool)
布尔值只有两种可能：`True`（真）或 `False`（假）。它们通常用于逻辑判断。
```python
is_learning = True
game_over = False
```

---

了解了基本数据类型后，你可能会想知道如何查看一个变量的类型，或者如何在不同类型之间转换。这就是我们接下来要讨论的内容。

## 检查与转换类型 🔄

### 使用 `type()` 函数检查类型
你可以使用 `type()` 函数来查看任何变量或值的数据类型。
```python
print(type(10))      # 输出：<class 'int'>
print(type(3.14))    # 输出：<class 'float'>
print(type("文本"))  # 输出：<class 'str'>
print(type(True))    # 输出：<class 'bool'>
```

### 类型转换
有时你需要将一种数据类型转换为另一种。Python提供了简单的内置函数来完成这个操作。

以下是常用的类型转换函数：
*   `int()`：将值转换为整数。
*   `float()`：将值转换为浮点数。
*   `str()`：将值转换为字符串。

```python
# 将字符串转换为整数
num_str = "123"
num_int = int(num_str)  # num_int 现在是整数 123

# 将整数转换为字符串
age = 25
age_str = str(age)      # age_str 现在是字符串 "25"

# 将整数转换为浮点数
x = 10
x_float = float(x)      # x_float 现在是浮点数 10.0
```

**注意**：转换必须合理。例如，你不能将 `"你好"` 这样的文本转换成整数。

掌握了类型的转换，我们就可以让不同类型的数据一起工作了。最后，让我们看看变量命名需要遵循哪些规则。

## 变量命名规则 📝

为了写出清晰且可维护的代码，给变量起一个好名字非常重要。以下是Python变量命名的核心规则：

1.  **只能包含**：字母（a-z， A-Z）、数字（0-9）和下划线（_）。
2.  **不能以数字开头**：`1variable` 是无效的，但 `variable1` 是有效的。
3.  **区分大小写**：`age`、`Age` 和 `AGE` 是三个不同的变量。
4.  **不能使用Python关键字**：例如 `if`， `for`， `while`， `True`， `False` 等是Python语言本身使用的单词，不能用作变量名。

良好的命名习惯是使用描述性的名称，并使用下划线分隔单词（这种风格称为“蛇形命名法”）。
```python
# 好的例子
user_name = “张三”
total_price = 99.8
is_logged_in = True

# 不好的例子（不清晰）
a = “张三”
tp = 99.8
ili = True
```
