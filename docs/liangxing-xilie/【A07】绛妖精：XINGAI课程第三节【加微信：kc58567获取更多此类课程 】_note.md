## 历史与心理认知

如果存在心理障碍，我们可以通过道具或手部技巧作为过渡，无需实际口交或仅少量口交就能帮助对方达到高潮。

1.  双手一上一下重叠握住阴茎（靠近睾丸处）。
2.  双手以相反方向（一正一反）边旋转边向上搓揉，类似拧干衣服的动作。
3.  双手力度一定要温柔。
4.  进行此动作前，务必涂抹足够的润滑液，以免弄痛对方。

**第三招：柠檬榨汁机**
1.  摊开手掌（保持放松，不要僵硬）。
2.  轻轻按住龟头部位。
3.  旋转你的手掌。
