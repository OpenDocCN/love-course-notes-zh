# 李氏养生功法：第七节：强肾习惯与户外功法

在本节课中，我们将学习两个核心内容：一是融入日常生活的八大强肾好习惯，二是适合在户外环境中练习的养生功法组合。这些方法旨在帮助您在不占用额外时间的情况下，随时随地养护身体。

## 日常八大强肾好习惯

上一节我们介绍了具体的功法练习，本节中我们来看看如何将养生融入生活点滴。以下是八个简单易行、无需刻意安排时间的好习惯，其精髓可概括为八个字：**断、金、捏、颈、提、摇、捶、搓**。

**断尿法**
小便时，不要一次性将尿液全部排出。应将尿流分几次断开排出。在排尿过程中，有意识地中断尿流，将其“切”成几段即可，无需过度用力。此方法有两种练习方式：
1.  轻轻排出尿液。
2.  用力将尿液射出，使尿液冲击阴茎根部的尿道球部产生胀感，然后用力射出，再蓄力，再射出，共进行6次。

此法既能锻炼PC肌（耻骨尾骨肌），也能锻炼射精力度。每人每日小便通常超过5次，白天均可应用此法，使之成为习惯。同时，排尿时应紧咬牙齿，不要出声。

**金冷法**
洗澡时，应有意识地用冷水冲洗阴部。即使在冬天也应坚持。让阴部彻底冷透后，再用热水冲洗；待阴部热透后，再次用冷水冲洗。此处“阴部”包括阴茎、睾丸、阴囊，以及周边的腹部、腹股沟和耻骨区域的皮肤。通过冷热交替刺激，可以扩张和收缩阴部及周围的血管，使其保持畅通与弹性。若不常洗澡，平时也可准备一盆热水和一盆冷水，交替浸泡整个阴部。夏季水温不够冷时，可在水中加入冰块。

**捏食指**
食指指尖有“商阳穴”，刺激此穴位有强筋壮阳之效。快速摩擦食指指腹直至发热。也可用食指摩擦其他物体，如桌面、公交车拉环等。还可以将双手食指勾在一起相互摩擦。

**紧腹肌**
腹部肌肉与阴部肌肉联系紧密。锻炼腹肌也能间接锻炼阴部肌肉。同时，强健的腹部能为正常的性生活提供有力支持。随着年龄增长和生活水平提高，许多人开始发胖，小腹隆起。这并非好事，应努力使其收缩。一个腹肌强健的男性，其性生活通常也更满意。

**提肛门**
收缩PC肌（提肛运动）。此训练可随时随地进行，例如等电梯、排队或看电视时。

**摇膝盖**
俗话说“人老腿先老”，膝盖是下半身非常重要的关节。摇膝盖分为左摇、右摇、内摇和外摇。同时应将双手盖在膝盖上，上身与下身同步摇动。

**捶大腿**
盘坐在椅子或沙发上，看电视时即可进行捶打。大腿内侧分布着肝经和肾经，特别是靠近膝盖的部位，更是这两条经络的枢纽。捶打此部位通常会感到特别疼痛，应用力捶打至产生麻木感。

**搓脚心**
脚心即“涌泉穴”。古语有云：“富人吃人参，穷人搓脚心。”刺激涌泉穴对身体强健有极大好处。

这八大好习惯都可在日常生活中不经意间完成，无需刻意安排时间。若能自觉将其转化为习惯，将终身受益。

## 好习惯动作演示

下面，我们将其中四个习惯的动作要领进行演示。

**捏食指**
捏住食指两侧，反复捏按60次。也可将双手食指勾在一起，摩擦指腹。

**摇膝盖**
双手放在膝盖上。
1.  向左摇动10次。
2.  向右摇动10次。
3.  向内摇动10次。
4.  向外摇动10次。
也可配合上下轻微晃动。

**紧腹肌（坐姿收腹）**
最好进行仰卧起坐，但上班或居家环境可能不便。这里介绍一个在椅子上即可完成的方法：
1.  坐在椅子前半部，双手撑住身体后方以保持稳定。
2.  利用腹部力量将双腿向上提起，做收腹运动。
**关键**：动作应依靠腹部肌肉收缩完成，而非大腿肌肉发力。可将腿抬得高一些，背部不要完全靠在椅背上，应与椅背保持一定距离。
每日可练习多次，一组50个，累计做5到10组（共250-500个）均可。

**捶大腿**
盘坐，用力捶打大腿内侧，从膝盖上方到大腿根部的区域，尤其是感到又疼又麻的穴位处。用力敲打至该部位产生麻木感。此动作在看电视时即可进行。

## 李氏户外养生功法

由于部分养肾功法不适合在户外练习，而许多朋友又喜欢在空气新鲜、阳光充足、能接地气的户外环境锻炼，因此特精选以下12式功法，供大家户外修炼。整套练习约需30分钟。

以下是户外功法十二式的名称与练习次数，除“通元气”外，均采用腹式呼吸：
1.  **手托天**：9次（配合9次呼吸）。
2.  **通元气**：13次。
3.  **提耳朵**：左提20次，右提20次。
4.  **拨耳朵**：20次。
5.  **弹耳朵（鸣天鼓）**：左右各20次。
6.  **揪耳朵**：20次。
7.  **摩丹田**：50次（注意：户外练习时，双手同时作用于小腹部按摩，不直接按摩阴部）。
8.  **搓后腰**：20次（包含抖动动作）。
9.  **捶后腰**：50次。
10. **捶丹田**：50次。
11. **抱太极**：时间可长可短，短则5分钟左右，长则20-30分钟。
12. **摇膝盖**：左摇10次，右摇10次，内摇10次，外摇10次（此式在八大好习惯中已介绍）。

练习地点宜选择阳光充足、有树木、近水且避风的户外场所。抱太极的时间，短则5分钟，长则不宜超过20分钟。

## 部分招式演示

在十二式户外功中，有几个动作之前未详细学习，现演示如下：

**拨耳朵**
用掌心盖住耳朵，再突然松开，如此反复20次。

**揪耳朵**
用双手手指捏住耳廓，可交叉进行（左手揪右耳，右手揪左耳）。不是揪住不放，而是揪住后顺势滑下，反复20次。

**弹耳朵（鸣天鼓）**
1.  用双手掌心紧紧捂住耳朵。
2.  将左手中指搭在右手中指上，然后用力滑下弹击后脑勺20次，耳中会有“咚咚”如击鼓之声。
3.  换右手中指弹击左手中指20次。
**注意**：弹击时，双手掌心需始终捂紧耳朵。

**摩丹田（户外版）**
与室内功法不同，户外练习时，双手同时重叠置于小腹（丹田）位置，进行顺时针或逆时针按摩，共50次。

**搓后腰（注意事项）**
天气暖和时，可将上衣撩起，双手直接贴在后腰皮肤上搓摩。冬天寒冷时，可将外衣撩起，双手贴住内衣进行搓摩，避免寒气直接侵入身体造成伤害。
