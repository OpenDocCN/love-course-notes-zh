2.  **大腿根部按摩**
    *   左手握住阴茎根部并向上轻提。
    *   用右手手掌根部轻揉大腿根部两侧。
    *   两侧各执行 **20次**。

3.  **会阴穴位按压**

    *   右手四指弯曲握拳，用指关节轻压阴茎根部的会阴穴。

4.  **前列腺按摩（肛门内）**
    *   左手握住阴茎根部并向上轻提。
    *   右手食指涂足润滑剂，轻轻进入肛门内按摩。
