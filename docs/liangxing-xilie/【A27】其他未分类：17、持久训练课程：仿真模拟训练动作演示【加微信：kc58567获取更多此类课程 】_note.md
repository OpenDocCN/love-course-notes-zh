## 核心概念：持久训练

持久训练，在机器学习中通常指模型通过**多次迭代**和**持续优化**来提升性能的过程。其核心在于不断调整模型参数，以最小化预测误差。这个过程可以用一个简单的**公式**来描述：

**参数更新公式：** `θ_new = θ_old - α * ∇J(θ)`

其中：
*   `θ` 代表模型参数。
*   `α` 是学习率，控制每次更新的步长。
*   `∇J(θ)` 是损失函数 `J` 关于参数 `θ` 的梯度。

这个公式是梯度下降法的核心，它指导模型如何朝着减少错误的方向“前进”。

## 仿真模拟训练动作演示

理解了核心公式后，我们通过一个仿真模拟来演示完整的训练流程。以下是训练过程中涉及的关键步骤：

以下是训练循环中的四个主要阶段：

1.  **前向传播**
    *   输入数据通过模型网络，得到预测输出。

2.  **计算损失**
    *   使用损失函数（如均方误差）比较预测输出与真实标签，计算误差值。

3.  **反向传播**
    *   计算损失函数相对于每个模型参数的梯度（即导数），以确定参数调整的方向和幅度。

4.  **参数更新**
    *   根据上一步计算出的梯度，使用优化器（如梯度下降）更新模型参数。

这个过程会循环往复，直到模型性能达到满意标准或达到预设的训练轮次。

## 代码示例

为了让概念更具体，我们来看一个使用Python和PyTorch框架的简化代码示例，它展示了上述训练循环：

```python
import torch
import torch.nn as nn
import torch.optim as optim

# 1. 定义一个简单的线性模型
model = nn.Linear(in_features=1, out_features=1)
# 2. 定义损失函数和优化器
criterion = nn.MSELoss()
optimizer = optim.SGD(model.parameters(), lr=0.01)

# 模拟训练数据
inputs = torch.tensor([[1.0], [2.0], [3.0]])
labels = torch.tensor([[2.0], [4.0], [6.0]])

# 3. 训练循环
for epoch in range(100): # 持久训练100轮
    # 前向传播
    predictions = model(inputs)
    # 计算损失
    loss = criterion(predictions, labels)
    # 反向传播
    optimizer.zero_grad() # 清空上一轮的梯度
    loss.backward()       # 计算当前梯度
    # 参数更新
    optimizer.step()      # 根据梯度更新参数

    if (epoch+1) % 20 == 0:
        print(f'Epoch [{epoch+1}/100], Loss: {loss.item():.4f}')
```

在这段代码中，模型在100个“纪元”内持续学习，每次循环都执行前向传播、计算损失、反向传播和参数更新这四个动作，损失值会随着训练逐渐降低。
