# 亲密关系课程：A22：省力体位指南

在本节课中，我们将学习几种旨在节省体力、缓解疲劳，同时仍能获得愉悦体验的亲密姿势。这些姿势适合在双方感到疲惫或希望进行温和互动时使用。

上一节我们介绍了课程的目的，本节中我们将具体学习三种省力体位。以下是三种推荐姿势的详细步骤与要点。

## 第一招：美人侧卧 😴

首先，请诗诗侧卧，用后背对着柱子。她可以夸出一条腿，方便柱子进入。诗诗也可以轻轻并拢双腿，这能为柱子带来更大的刺激。

柱子可以用钉钉爱抚诗诗的臀部，同时用手爱抚她的乳房和私处，也可以用嘴唇安抚她的后背。当情到深处，柱子用钉钉缓缓进入诗诗的身体进行运动。

**核心要点**：美人侧卧姿势中，双方都不必承受对方的体重。这特别适合在双方都有些疲惫或刚睡醒时使用。其缺点是进入深度可能较浅。

**深入技巧**：若想增加深度，诗诗可将双腿蜷缩成`90度角`，这样柱子进入时就能更加深入。

*   **推荐指数**：★★★★☆
*   **难度系数**：★★★★☆

## 第二招：拉弓射箭 🏹

接下来我们看看拉弓射箭式。柱子坐在床上，伸直双腿。诗诗张开大腿，面对面跨坐在柱子的大腿上，并将双腿伸直到柱子后方。

双方可以紧紧拥抱，实现肌肤的全面接触。诗诗的胸部大约停留在柱子脸部的位置，柱子可以用嘴唇爱抚其胸部，同时用手托住诗诗的臀部，帮助她进行上下或前后的运动。

**核心要点**：此坐姿有助于钉钉触碰G点和摩擦阴蒂。运动幅度通常不会太剧烈，但双方紧贴的肌肤与私处摩擦产生的快感，能带来强烈的亲密感。

**深入技巧**：可以抬高臀部，或在柱子的臀部下面垫一个垫子，以帮助钉钉更加深入。

*   **推荐指数**：★★★★☆
*   **难度系数**：★★★★☆

## 第三招：拉弓射箭变式 🔄

最后，我们来学习拉弓射箭的一个变式。顾名思义，这是上一姿势的变形。柱子躺在床上并伸直双腿。诗诗同样张开大腿面对面跨坐，并将双腿伸直到柱子后方。

此时，柱子的钉钉与诗诗阴道的中心线相交。诗诗在运动过程中可以将重心后移，用双手撑住床面。

**核心效果**：这个变式能给柱子的钉钉带来一种压迫性的刺激，从而有助于在缓解疲劳的同时，获得强烈的快感。

本节课中，我们一起学习了三种侧重于节省体力、适合疲惫时使用的亲密姿势：美人侧卧、拉弓射箭及其变式。掌握这些方法有助于在享受亲密的同时更好地保存体能。
