# 李氏养生功法：1-2：三式基础训练教程 🧘

在本节课中，我们将学习李氏养生功法的前三个基础招式：手托天、通源器和提耳朵。这些动作旨在通过特定的姿势与呼吸配合，帮助初学者调理气息、强健身体。

上一节我们介绍了课程的基本信息，本节中我们将进行实际的操作训练。

以下是三个招式的分步详解。

## 第一式：手托天

接预备式。两脚与肩同宽，双膝微屈。

首先，意守丹田穴。采用腹式呼吸法。
1.  吸气，双手缓缓抬起。
2.  双手托斜，同时踮起脚尖。
3.  呼气，双手下落。
4.  同时吸气，缓一口气，继续上托，踮脚尖。
5.  呼气，意想端起大地精华。
6.  缓一口气，再次上托。
7.  呼气。
8.  吸气，意想搂地气，可以不换气。
9.  上托，呼气。

**动作要点**：吸气时手上托，呼气时手下落或放松。初学者在手托至高处时，可缓一口气再继续。训练有素者，可从底下吸气，一气呵成上托。

## 第二式：通源器

接上式。双手收回，劳宫穴照住丹田穴。

将双手抬起，掌心朝下，如按压皮球，双腿微屈。
1.  脚掌用力抓地，眼睛睁圆。
2.  采用综合呼吸法（结合腹式与提肛）。吸气，同时提肛缩阴。
3.  快速喷气（呼气）。
4.  重复：吸气，提肛，喷气。

**动作要点**：掌心朝向两侧，训练中两眼需睁大，精神集中。提肛缩阴时，务必睁大眼睛，双脚紧抓地面。

## 第三式：提耳朵

此动作较为简单，采用腹式呼吸。
1.  右手上举，搭住头顶。
2.  用食指与拇指夹住左耳上沿。
3.  上提，左手亦可协助。
4.  改为左手上提右耳，此时右手可护住丹田穴。

本节课中我们一起学习了手托天、通源器和提耳朵三个基础养生招式。关键在于姿势准确并与呼吸（腹式呼吸或综合呼吸）紧密配合。
