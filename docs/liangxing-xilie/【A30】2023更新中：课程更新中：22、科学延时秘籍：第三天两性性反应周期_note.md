## 性反应周期简介

性反应周期是指从性欲被唤起，到性活动结束、身心恢复平静的整个过程所遵循的周期性变化规律。这一理论由美国性学家马斯特斯和约翰逊通过大量实验观察总结得出。

通俗地说，男性的周期表现为：**勃起 → 射精 → 疲软**。女性的周期表现为：**唤起 → 获得快感/高潮 → 恢复平静**。两者都遵循一个基本的周期性规律。

## 性反应周期的四个阶段

    *   女性标志：**阴道高度润滑** 或乳头竖起、乳房膨大，通常需 **10到30秒**。

*   **平台期**
    这是兴奋状态持续并逐步向高潮推进的短暂阶段，可理解为积蓄能量、快速升温的过程，通常持续 **半分钟到几分钟**。此阶段生理上没有非常明显的变化特征。

*   **高潮期**

    *   男性标志：以 **射精** 为核心表现。
    *   女性标志：表现为 **全身痉挛、阴道收缩**。

*   **消退期**
    高潮后，身体与心理逐渐松弛，恢复平静状态的阶段，通常需要 **10到15分钟**。
