1.  **“活泼型”和“力量型”看起来都很外向，如何准确区分？**
    *   核心区别在于内在驱动力。活泼型（`外向 + 感性`）的驱动力是**获得快乐与关注**，行为像“蝴蝶”，追求新鲜有趣。力量型（`外向 + 理性`）的驱动力是**达成目标与掌控**，行为像“狮子”，追求效率与结果。

2.  **“和平型”是不是等同于“没主见”？**
    *   不等于。和平型（`内向 + 感性`）的核心需求是**维持和谐与稳定**。她们并非没有主见，而是倾向于回避冲突，其意见常以委婉或迁就的方式表达。在感到安全的环境中，她们会明确表达观点。
