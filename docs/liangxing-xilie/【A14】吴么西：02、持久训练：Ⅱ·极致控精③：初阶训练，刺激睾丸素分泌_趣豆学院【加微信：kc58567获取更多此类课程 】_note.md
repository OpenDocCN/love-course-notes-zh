# 性健康训练：02：持久训练：Ⅱ·极致控精③：初阶训练，刺激睾丸素分泌 🏋️‍♂️

## 概述
在本节课中，我们将学习如何通过特定的PC肌训练和深蹲练习，来有效刺激身体分泌睾丸素。睾丸素是影响男性性功能、肌肉生长和整体健康的关键激素。掌握这些基础训练方法，有助于提升你的激素水平。

## 认识睾丸素
上一节我们介绍了训练的重要性，本节中我们来看看训练的核心目标之一：促进睾丸素分泌。

睾丸素由睾丸分泌。它主要影响性发育和性冲动。其次，它会影响骨骼质量、记忆力、心情。它还能刺激肌肉生长合成，帮助燃烧脂肪，降低心血管疾病风险。

男性睾丸素的分泌通常在20至30岁时达到顶峰。随后，它会随着年龄增长而减少分泌。若想在一定的年龄阶段保持较高的睾丸素水平，则需要依靠运动来激发。

## PC肌训练：仰卧抬身
以下是第一套PC肌训练动作，旨在激活相关肌群。

1.  平躺在瑜伽垫上，小腿自然弯曲。
2.  脚掌贴地，双手翻转，放在大腿两侧。
3.  以臀部为支点，将上半身抬起。
4.  双手掌在大腿间来回滑动，同时收紧PC肌。
5.  呼气，放松PC肌并落下身体。

休息3秒后重复。该组动作共需完成15次（讲师已演示3次，学员需自行完成剩余12次）。整个动作需要持续练习三组。

## PC肌训练：仰卧踏步
完成了静态的抬身练习后，我们接下来进行一个结合腿部动作的PC肌训练。

1.  平躺在瑜伽垫上，双手枕在后脑勺，腰腹以上保持不动。
2.  以臀部和大腿为重心，吸气并夹紧PC肌。
3.  左脚踏出，像走路一样停留半空。
4.  右脚踏出与左脚平齐，随后放下双脚。
5.  呼气，放松PC肌。

休息3秒后重复。该组动作共需完成15次（讲师已演示3次，学员需自行完成剩余12次）。整个动作需要持续练习三组。

## 深蹲训练：标准深蹲
上一部分我们专注于PC肌，现在我们将加入深蹲训练，以全面刺激睾丸素分泌。以下是第一种深蹲姿势。

1.  自然站立，两脚与肩同宽，脚尖朝外。
2.  后背保持挺直，身体下降时，确保膝盖不超过脚尖。
3.  臀部向下坐，如同坐向板凳。
4.  双手可打直前伸，或交叉抱在脑后，维持在胸部高度。
5.  全程保持挺胸、挺直后背、收紧小腹。
6.  臀部绷紧并向上翘起，速度放缓，自然呼吸。

休息3秒后重复。该组动作共需完成15次（讲师已演示2次，学员需自行完成剩余13次）。整个动作需要持续练习三组。

## 深蹲训练：宽距深蹲
最后，我们学习一个能刺激不同肌群的宽距深蹲变式，以巩固训练效果。

1.  自然站立，两脚距离较宽，脚尖向外张开45度。
2.  双手可以叉腰，也可以向前伸直或握拳，维持在胸部高度。
3.  身体垂直下降，注意躯干不要前倾。
4.  起身回到预备姿势。

休息3秒后重复。该组动作共需完成15次（讲师已演示3次，学员需自行完成剩余12次）。整个动作需要持续练习三组。
