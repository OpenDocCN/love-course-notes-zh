# 艳舞私房课：06：地板动作基础篇 🩰

在本节课中，我们将要学习几个简单的地板动作。这些动作能为你的舞蹈增添多样性。课程内容专为初学者设计，每个动作练习时间约为2分钟，易于掌握。

---

地板动作是丰富舞蹈表现力的绝佳方式。根据舞蹈环境的不同，有些动作在光滑地面上更容易完成，而有些则在地毯上更易施展。

为了确保所有动作在实际表演中都能顺利施展，建议你在计划为伴侣跳舞的同一房间内进行练习。这样可以帮助你提前适应环境。

以下是几个基础地板动作的分解步骤，每个动作的核心要领将通过**加粗**或`代码`形式进行强调。

**动作一：地面滑行**
1.  双膝跪地，保持上身挺直。
2.  将身体重心缓慢向一侧转移，同时同侧腿向外伸直。
3.  利用手掌与膝盖支撑，平滑地将重心移回中心。关键点是控制**重心转移的速度** `speed = smooth`。

**动作二：仰卧起旋**
1.  平躺于地面，双腿并拢抬起，与地面呈90度角。
2.  利用腰腹力量，将臀部向一侧扭转，同时头部转向另一侧。
3.  缓慢恢复至起始姿势，再向另一侧重复。核心是**腰腹的扭转力** `torque = core_engagement`。

**动作三：侧身支撑**
1.  侧卧，用下方手臂的前臂和手掌支撑起上半身。
2.  上方手臂可自由伸展以保持平衡或增加美感。
3.  下方腿伸直，上方腿屈膝，脚尖轻点下方腿膝盖前方，形成稳定三角支撑。重点是**身体的直线与平衡** `balance = true`。

---

本节课中我们一起学习了三个基础的地板动作：地面滑行、仰卧起旋与侧身支撑。记住，成功的关键在于在真实环境中练习，并专注于每个动作的核心发力要点。持续练习这些基础动作，能为你的舞蹈打下坚实的根基。
