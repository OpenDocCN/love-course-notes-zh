# 武当山济肾堂强肾清肺功：2.2：清肺内功第一式详解

在本节课中，我们将详细学习武当山济肾堂清肺内功的第一式。此式通过特定的呼吸与动作配合，旨在引导气血运行，达到疏通经络、强健肺腑的效果。我们将分解每一个动作和呼吸要点，确保初学者能够安全、准确地掌握。

## 动作与呼吸分解

上一节我们介绍了本套功法的基本理念，本节中我们来看看第一式的具体练习方法。其核心在于呼吸控制与肢体发力的协调。

以下是第一式的完整动作流程：

1.  **起始姿势**：双脚站立，略宽于肩。双手自然下垂。
2.  **吸气**：缓慢吸气，同时双手回收至腹部前方，但**不呼不吸**，保持气息。
3.  **初次呼气**：身体微微下蹲，同时将刚才吸入的气呼出约**三分之一**。此时体内应保留**三分之二**的气。
4.  **闭气抱球**：将剩余的气闭在体内。双手在腹部前呈抱球状。一旦抱球，全身开始**绷紧**，进入运气状态。
5.  **翻掌与意念**：双掌翻转。想象双手各抓着一个沉重的铁桩。
6.  **发力外分**：在闭气状态下，双手用力向两侧分开，过程持续**5到8秒**。同时，脚趾抓地，膝盖有意识地向**内扣**。
7.  **喷气完成**：最后，以“**嗨**”声将气快速喷出，双手仿佛将重物猛然丢出，动作完成。

## 核心要领详解

理解了基本步骤后，我们需要掌握其中的关键细节，这些是功法生效的重点。

以下是练习时必须注意的几个核心要领：

*   **下肢稳固**：发力时，脚趾要紧紧抓地，膝盖感觉向内侧扣紧，确保下盘稳定，力量从地而生。代码式描述为：`脚趾抓地 && 膝盖内扣`。
*   **上肢发力**：双手向两侧分开时，路线是平行的，不要向上或向下摆动。力量应灌注于手掌，如鹰爪般有力。身体保持中正，**不要**跟随手的动作而晃动。
*   **呼吸配合**：呼吸公式为：`吸气(满) -> 呼气(1/3) -> 闭气(运气) -> 喷气(完)`。闭气运气阶段是气血加速运行的关键。
*   **意念引导**：分掌时想象分开重物，有助于调动全身力量。若练习中出现头晕，需进行意念引导：想象头顶百会穴有一股清泉，沿身体前侧任脉流向下丹田，将上涌的气引导下来即可缓解。

## 常见问题与功效原理

在练习过程中，身体可能会产生一些反应，了解其原理能帮助我们更好地坚持与调整。

练习此式时，身体气血运行加快，一些潜在的健康问题可能会显现出来，例如感到某些部位酸、闷、胀，这属于正常现象。此式疏通经络的原理在于：通过**深吸、闭气、绷紧、爆发**这一系列动作，迫使体内气血在高压下加速循环，从而冲刷、打通淤滞的经络。
