### 第二步：阴囊按摩
目标是让伴侣放松并感到安全。由于睾丸痛觉神经丰富，所有动作务必轻柔。

2.  **揉蛋**：用手掌心托住阴囊，轻轻按揉。
3.  **推蛋**：四指并拢，用手指背面轻轻向上推阴囊。
4.  **扯蛋**：用手指轻轻夹住阴囊根部，向下轻微拉扯。

此时需使用润滑剂，将其涂抹于整个阴茎。

3.  **拧毛巾法**：双手分别握住阴茎上端和下端，手腕放松，向相反方向轻轻旋转。
4.  **虎口套弄法**：双手交叠，利用虎口处的嫩肉上下刮蹭冠状沟。
