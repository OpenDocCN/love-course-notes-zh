# 性福力提升课程：A08：别让性问题演变为婚姻危机

## 概述

在本节课中，我们将探讨如何识别和处理两性关系中的常见性问题，理解这些问题如何从最初的微小困扰演变为重大的婚姻危机，并学习通过提升“性福力”来主动维护关系的健康与和谐。

## 课程内容

我是幸福玛利亚，一名性心理与两性关系咨询师及成人性教育老师。我的微博和微信公众号都叫“幸福玛利亚”。我专注于性心理与婚姻咨询已超过12年。

在咨询工作中，我几乎接触过所有类型的性心理与两性关系问题。

以下是咨询中经常被问及的一些典型问题：

*   结婚仅两年，性爱频率已降至两三个月一次，且总是由一方（通常是女性）提出。提出方常感到自尊受伤，担心自己性冷淡。
*   对伴侣失去了性欲。
*   从未体验过高潮。
*   尝试与伴侣交流性问题时，总被拒绝，对方认为这是过高要求或施加压力。
*   因怕疼，结婚多年仍未成功完成性交，关键时刻会紧张、身体紧绷并本能推开伴侣。
*   伴侣沉迷于找小姐或按摩女，咨询者想知道性工作者有何“绝活”或特色服务。
*   被伴侣认为在床上过于保守，不够风情，但不知如何改变。
*   伴侣宁愿自慰、寻求外部性服务或观看A片，也不愿进行夫妻性爱。
*   因长期性不满足导致身体不适，甚至出现妇科肿瘤。
*   因性需求未被满足，感觉自己控制不住想要出轨。

中国人的习惯往往是，不到身体出现严重疾病（如癌症）时，不舍得花钱保养身体。对待两性关系和婚姻也是如此。

不到关系濒临破裂、伴侣反复外遇或一方坚决提出离婚时，人们通常意识不到关系已恶化至此，也不会舍得花钱维护婚姻。

在最初发现苗头不对时，很少人会第一时间寻求专业咨询师的帮助。很多时候，作为咨询师，我感到“亡羊补牢，为时已晚”。

因此，我意识到仅靠心理咨询进行事后补救远远不够且时常太迟。最好的方式是提前主动学习、调整，以避免问题发生。

于是，我在2013年初创办了全国首个成人性教育成长课程——“提升性福力”两天线下密集强化班。

课程全面教授科学健康的性观念、性态度、性知识、专业性技巧、两性相处方法及亲密关系经营之道。至今已举办48期，引导全国超过1500名女性走上提升幸福之路。

这些学员年龄从20多岁到50多岁不等，人生阶段涵盖恋爱、单身、已婚甚至离异单身等各种状态。

但她们的共同点是，在性知识、性技巧以及对男性性生理、性心理的了解方面，几乎都是空白。

通常经过两天培训，学员们能实现从“幼儿园水平”到“研究生水平”的快速提升。回家后，她们常能成为自己朋友圈中的“性爱大师”。

在我的咨询和教学中，我发现许多人对性爱存在大量盲点、误区和困惑。

当存在这些问题时，人们只是稀里糊涂地过着低质量的性生活，性满意度低且不善于交流。

女性可能转而百般挑剔、抱怨、郁闷烦躁；男性则可能转身寻求其他途径或其他女人来解决。

久而久之，两人的亲密关系越来越疏远，像室友或搭伙过日子的朋友。缺乏性爱和身体亲密作为关系的润滑剂，关系会变得干巴巴且紧张。

任何小事都可能一触即发，矛盾愈演愈烈，也无法通过美好的性爱实现“床头吵架床尾和”。

于是，最初小小的性问题，最终可能发展成大大的婚姻危机事件。

在婚姻咨询中，我发现离婚原因中，性不和谐实际上占有很高比例，并非通常所说的“性格不合”，只是人们不好意思对外直言。

同时，男性和女性发生外遇的原因，很高比率也源于最初的性不和谐、性不满足或性满意度低。

因此，我提出一个观点：我们每个人都需要关注并提升自己的“性福力”。

我理解的“性福力”，是一种你能游刃有余地掌握性爱、灵活自如且放松地享受性爱快乐的能力。

这种游刃有余，必须建立在充分的性知识与技巧、正确的观点与积极的态度之上。

也必须建立在充分了解自身与对方的生理和心理，以及懂得应对男女差异的基础之上。唯有如此，才能做到 **`知己知彼，百战不殆`**。

掌握性爱的能力，还包括你要非常熟悉自己的身体，知道如何快速调动自身与伴侣的身体和情绪。

知道如何扬长避短，创造性地让你们的性爱更轻松、简单、富有乐趣。

因为人的身体本能地需要重复快乐。如果一件事又累又无趣，本能就会让人想要回避和逃避，不愿再多做。

因此，无论是为了身体的健康快乐，还是为了两人的情感交流与持久的亲密关系，你都需要补上“提升性福力”这一课。

## 总结

本节课我们一起学习了常见性问题的表现及其对婚姻关系的潜在危害。我们认识到，许多婚姻危机源于未被妥善处理的微小性问题。通过主动学习科学的性观念、知识与技巧，提升个人的“性福力”，我们可以有效预防问题升级，维护亲密关系的健康与和谐，让性爱成为关系的润滑剂而非矛盾源。
