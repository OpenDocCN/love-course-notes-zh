# 强精壮阳铁裆功：A27：1：功法详解与早泄辅助治疗

在本节课中，我们将学习一套名为“强精壮阳铁裆功”的传统养生功法。这套功法包含十五个具体步骤，并附带针对早泄问题的辅助练习方法。我们将逐一解析每个动作的要领、作用及注意事项，旨在帮助初学者理解并安全练习。

## 概述

阳痿指男性在性交时，阴茎无法勃起或无法维持足够勃起程度以完成性交。经过三十多年的临床治疗，此疗法取得了满意疗效。根据三百例心理性阳痿的临床统计，总有效率在百分之九十五以上。为了有助于治疗，我们先简单介绍男性生殖器官。

男性生殖器官包括阴茎、睾丸、精索、输精管、前列腺、精囊等。

## 功法详解：按摩铁裆功十五节

按摩铁裆功共分为十五节。以下是各节的具体操作方法及其作用。

### 第一节：推腹

两手相叠，从剑突部位推至耻骨联合处，共推三十六次。

**作用**：有健脾和胃、引气归于丹田的作用。

### 第二节：分推腹

两手掌自剑突部分别向腹部两侧推去，共推三十六次。

**作用**：有和胃消食、理气健脾的作用。

### 第三节：揉脐

两手相叠，按于肚脐上，左手在下。先向顺时针方向揉摩三十六次，然后向逆时针方向揉摩三十六次。

**作用**：有健脾益肾、温阳散寒的作用。

### 第四节：捻精索

精索位于阴囊两侧，睾丸之上，由输精管、动静脉等组成，被精索膜包绕。以两手食指、中指与拇指对称捻搓精索五十次。

**作用**：有疏肝理气、激发肾阳的作用。

### 第五节：揉睾丸

右手将阴囊、阴茎一并抓起，使睾丸与阴茎头露在虎口外面。用左手掌心按在左侧睾丸上揉五十次。然后换手，以同样方法用右手揉右侧睾丸五十次。

**作用**：有强身益肾的作用。

### 第六节：搓睾丸

以两手食指、中指分别在阴囊下托住两侧睾丸，拇指压在上面，搓揉五十次。

**作用**：有益肾强精的作用。

### 第七节：顶睾丸

两手食指、中指在阴囊下托住同侧睾丸，再以两手拇指端将睾丸向腹股沟方向顶上去，然后慢慢放下。共做三次。

**作用**：有提气、生阳、益精的作用。

### 第八节：挂档

准备一个纱布袋，宽五十厘米，长八十厘米，缝制成圆筒状。将铁环套在纱布袋上，以便连接纱袋。另缝制一个十五乘二十厘米的布袋，装入一千五百克沙子。将沙袋与铁环连接在一起。在纱布袋上打一个活结，将活结套在阴茎与阴囊的根部并扎紧，注意将阴毛留在外面。将沙袋慢慢放下，悬于两腿之间，并前后摆动五十次。

**作用**：有养肾强筋、充益精气、壮命门之火的作用。摆动完毕后，先放下沙袋，再解开活结。

### 第九节：捶睾丸

两手握空拳，交替捶击两侧睾丸各二十五次，以感觉酸胀但不疼痛为佳。

**作用**：有滋养肾经、引气血上行的作用。

### 第十节：捶茎根

两手握空拳，捶击阴茎根部两侧各五十次，然后捶击其上方的耻骨联合处五十次。

**作用**：有引气血上行、强健宗筋的作用。

### 第十一节：捶肾

肾区位于脊柱两侧，上端平第十一胸椎，下端与第三腰椎相平。两手握空拳，用手背部捶击背部肾区各五十次。

**作用**：有益肾强腰、导引气血归藏于肾的作用。

### 第十二节：通背

两手握空拳。一手捶击胸前锁骨下区域，一手捶击对侧肩胛骨下角。以腰部带动两上肢，前后摆动捶击，各做五十次。

**作用**：有通调周身气血的作用。

### 第十三节：转膝

两脚微开，屈膝，两手分别按在膝关节上。先向左旋转二十五次，再向右旋转二十五次。

**作用**：有通调足三阴经、足三阳经的作用。

### 第十四节：滚棍

准备一根长四十厘米、直径约四厘米的圆木棍。取坐位，穿平底鞋或不穿鞋，两脚踏在木棍上，前后滚动五十次。

**作用**：有导气下行、引火归元的作用。

### 第十五节：收功

练完以上十四节后，静坐三至五分钟，意守丹田。然后自由活动一会儿即可。

## 注意事项

练习本功法需遵循以下注意事项。

1.  练功前应将所有用具准备好。一般需先连续练习一百天，在此期间应禁止同房。
2.  练功中要集中精力体会各项动作的感觉，不可急于求成。
3.  每日练习一至二次。若有酸胀、疼痛等不适，可减少练功次数或改为隔日一次。
4.  有以下疾病者应先治病，例如：严重的精索静脉曲张、包茎不能上翻、包茎嵌顿、尿道炎症、输精管睾丸结核、包皮过长等。

练功后可以配合中药外洗外阴部二十至三十分钟，每日或隔日一次，以增强疗效。

**中药处方**：蛇床子30克、露蜂房30克、当归30克、地骨皮30克、红花酒客、甘草30克。

## 早泄的辅助治疗

上一节我们介绍了铁裆功的主体功法，本节中我们来看看如何将其应用于早泄的辅助治疗。

早泄是指性交时男性射精过早。百分之八十五的患者由精神因素引起，如焦虑、恐惧、过度兴奋等。也有许多人是因为龟头的感觉神经过分敏感，刺激阈值过低而引起。较严重的早泄易于诊断，例如射精在性交前发生或阴茎正常进入阴道时发生。器质性早泄占百分之十五，如阴茎包皮系带过短妨碍充分勃起。前列腺炎引起的早泄占比较大，患者常伴有腹股沟区、耻骨区、阴囊及会阴部疼痛，多有尿频、尿急等症状。

早泄的治疗主要练习按摩铁裆功。但练至第十节“捶茎根”时，往往阴茎勃起，小腹部有一种预射精的压迫感。此时应立即加练“捏挣法”。

**捏挣法**：将食指、中指放在包皮系带处，将拇指压于阴茎头冠背侧，从前向后挤捏。硬度越大，挤捏力量越大。另一手握住阴囊与睾丸，同时向下向外拉。配合呼吸与意念，捏挣时吸气、收腹、提肛，引气从龟头经会阴至督脉，直至阴茎萎缩，无射精压迫感为止。

每当阳亢阴茎勃起时，应练“导阳功”。

**导阳功**：右侧卧位。用右手拇指指甲掐住无名指根处，其余四指握拳压住拇指。左手中指按压尾骨尖处。吸气时握拳，按压尾骨圈，屈脚趾微弓伸，引气自龟头经会阴至督脉。呼气时放松。至阴茎萎缩为止。

然后以两手手背捶击腰骶部三十六次，有激发精气、收敛元阳的作用。再以两掌搓摩腰部两侧三十六次，有补肾益气的作用。最后搓两足心各三十六次，有滋阴降火、引气归元的作用。

练功后可配合以下药物，水煎后外洗外阴部十五至二十分钟。

**中药处方**：蛇床子30克、地骨皮30克、罂粟壳30克、诃子30克、五味子30克、甘草30克、牡蛎60克。
