# 李氏养生功法：第一课：养肾功二十六式完整演练教程 🧘♂️

在本节课中，我们将完整学习并演练“李氏养肾功”的二十六式动作。这套功法旨在通过一系列特定动作来锻炼和养护身体。我们将从第一式开始，连贯地练习至收功。

## 概述
本节教程将逐步演示养肾功的全部二十六式。请跟随指导，连贯地完成所有动作。这是对之前学习内容的综合运用。

## 动作详解

上一节我们介绍了课程目标，本节中我们来看看具体的动作分解。请确保在练习时，动作涉及的身体部位需直接接触皮肤，避免隔著衣物。同时，保持自然的腹式呼吸。

以下是第一式至第六式的动作说明：
1.  **手托天**：双手上托，引导元气。
2.  **通元气**：注意将气息提高。
3.  **提耳朵**：提拉耳朵。
4.  **牡丹海**：此动作需接触腹部皮肤。将衣服撩起，裤子褪下，确保皮肤裸露。不特意控制呼吸，让身体自然预热。
5.  **搓睾丸**：持续搓动，直至该部位产生热感。
6.  **挤睾丸**：此动作旨在运转肾经。

完成了基础的预热和运转动作后，我们进入对关键部位的针对性锻炼。

以下是第七式至第十一式的动作说明：
7.  **转阴茎**：旋转阴茎。
8.  **出阴茎**：向前牵拉阴茎，注意将包皮口向前推至龟头，挤出血液。快速完成，并保持腹式呼吸。
9.  **挤阴茎**：像挤牛奶一样挤压阴茎，然后换左手重复。
10. **勾精锁**：向上勾提，直至顶端。吸气时提肛，呼气时放松肛门并放下。从下方重新开始向上勾提，反复多次。
11. **按阴根**：用中指指腹左右按摩会阴穴，可勾动阴部根处的肌肉。适当时候也可搓动。按摩后注意对前列腺的保护。可用中指左右两侧指节上下戳动。

接下来，我们将学习针对睾丸及周围区域的一系列按摩与运动方法。

以下是第十二式至第十八式的动作说明：
12. **搓腹股沟**：将裤子褪下，手伸入直接接触皮肤。手掌弯曲，用掌心下部接触腹股沟下方，掌心上部接触上方。此过程可按摩靠近阴茎根部的区域。
13. **揉耻骨**：揉动耻骨区域，使阴茎随之跳动。
14. **捻睾丸**：用手指捻动睾丸。
15. **压睾丸**：吸气时下压，呼气时放松。重复13次呼吸循环。
16. **揉睾丸**：此即“养丸十字精华功”。直接接触阴囊皮肤进行揉动（非搓动）。左右各50次，或上下各25次。
17. **滑睾丸**：主要依靠拇指和食指上下滑动睾丸，中指在后辅助。
18. **吸睾丸**：分两步。第一步，仅通过胸式呼吸将睾丸吸提起来，重复9次。之后，因肌肉乏力，可用手辅助：吸气时用手将睾丸上合至腹股沟，停留约两秒；呼气时放手，让睾丸自然坠回阴囊。最少重复50次。

在完成了对睾丸的集中锻炼后，我们继续进行提升和放松的招式。

以下是第十九式至第二十六式的动作说明：
19. **挂睾丸**：将睾丸向上挂提。
20. **摇睾丸**：手朝上，像摇橹一样摇动睾丸，并在摇动中向上拔提。摇、拔结合，重复50次。注意力度，勿伤及睾丸和阴茎。之后换手，做活塞状运动50次，过程中将睾丸向前推挤，超过龟头。
21. **拍睾丸**：先将阴茎用内裤挡起立，以内裤作为保护，然后拍打睾丸，左右各50次。
22. **扣睾丸**：用右手将阴茎扶成勺状，扣击睾丸25次，以示“问候”。
23. **搓后腰**：将手伸到后腰，撩起衣服紧贴皮肤。用力搓后腰9次。之后，上下快速抖动，紧贴后腰，至少200次。
24. **捶后腰**：用手背捶打后腰。
25. **捶大腿**：捶打大腿部位，保持自然呼吸，小腹微凸。重复100次。
26. **抱太极**：脚后跟微微抬起。怀抱太极图（与地面平行），并使其转动。反方向亦转动。收功时吸气，意想将怀中之气吸入丹田穴。呼气，结束功法。最后放松膝盖。
