## 步骤一：锁骨以上区域（脸、颈、肩、锁骨）

这个区域包括面部、颈部、肩膀和锁骨。在接触这些具体部位之前，有一个非常重要的预备步骤。

**核心节奏**：`缓慢且从容 > 快速`

### 面部爱抚

爱抚的区域是连接下巴边缘、脸颊和耳根三点所构成的面。

以下是面部爱抚的要点与常见错误：
*   **错误1：动作幅度太小**。应在这个较大的面上进行大幅度、缓慢的滑动。
*   **错误2：手指翘起**。手部运动必须始终与爱抚面保持平行，避免手指立起导致角度错误。
*   **注意事项**：面部可能因湿润而产生摩擦，不要因此跳过，应带着爱意继续温柔抚触。

### 耳朵爱抚

2.  **耳角**：即耳屏（耳朵前方的突起）。想象在温柔爱抚阴蒂，用最轻柔的触感接触这个部位。

**核心手法**：`压力 = 极轻（似触非触）`
避免用力按压或插入耳道，轻柔的亚当触摸更为有效。

### 下巴爱抚

下巴指从下颌边缘到颈部连接处的狭窄面。

使用一根或两根手指，沿着这个细长区域描绘纵向椭圆。想象在抚摸猫咪的下巴，女性通常也很喜欢这种感觉。

2.  **侧面（胸锁乳突肌）**：颈部侧面的肌肉线条。同样使用中指和无名指，注意手腕要转动，尽量让手掌与这个斜面保持平行，避免手指竖起。

**核心原则**：`手掌接触面 > 指尖接触面`
始终确保是手掌面而非指尖垂直于皮肤，手腕需配合转动以保持平行。

### 肩膀爱抚

肩膀也是敏感部位。通常使用食指、中指、无名指三根手指。根据位置切换手指组合：靠近颈部时可能只用中指和无名指，在肩头时则用三根手指。

*   **中部**：作为起点。
*   **上部与下部**：分别爱抚。

由于面积狭窄，可使用食指和中指两根手指，或仅用一根手指。动作上更接近前后推拉，而非画大椭圆。

## 步骤二：手臂、手掌与躯干侧面

本步骤涵盖从手臂外侧到躯干侧面的连贯爱抚。

2.  大拇指根部肉垫
3.  小指侧边缘
4.  指根部的肉垫
5.  手腕

逐一温柔爱抚这些点，能传达出与随意抚摸不同的用心。

### 前臂、肘部与上臂爱抚
*   **前臂**：将其视为一个椭圆面，保持手掌水平进行爱抚。
*   **肘部**：肘部骨骼处的薄皮肤也是性感带。用两根手指温柔地在小圆圈内画圆爱抚。
