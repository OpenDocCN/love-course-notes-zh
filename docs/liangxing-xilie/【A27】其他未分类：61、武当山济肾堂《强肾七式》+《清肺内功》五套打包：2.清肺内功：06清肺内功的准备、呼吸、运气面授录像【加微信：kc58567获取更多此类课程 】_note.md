# 清肺内功：2.6：清肺内功的准备、呼吸与运气详解 🫁

在本节课中，我们将要学习清肺内功的具体练习方法，包括练习前的准备工作、独特的呼吸技巧以及核心的运气要领。掌握这些基础是有效练习的关键。

## 练习前的准备

在开始练习清肺内功前，需要做好以下三项准备。

以下是三项准备工作：
1.  **选择空气清新的环境**。练习时用口吸气，而非鼻腔吸气。因为鼻腔内有鼻毛和粘膜，可以过滤空气中的灰尘与细菌，口腔则不具备此功能。因此需寻找空气较好的地方进行练习。
2.  **系上腰带**。不系腰带也可以练习，但系上腰带效果更佳。系腰带时需微微勒紧。这有助于初学者更好地体会丹田发力的感觉。
3.  **准备纸巾**。因为此功法采用鼻腔喷气，而非普通呼气。喷气时力道较强，可能将鼻涕喷出，准备好纸巾便于擦拭。

## 独特的呼吸方法

上一节我们介绍了练习前的准备，本节中我们来看看清肺内功独特的呼吸方法，这是功法的核心之一。

清肺内功的呼吸方法很独特，其流程可概括为：**口吸 → 闭气 → 鼻喷**。

以下是呼吸的三个步骤：
1.  **口吸气**。用牙齿缝隙吸气，吸气时发出“嘶”的声音。公式表示为：`吸气 = 口吸（齿缝，“嘶”声）`。
2.  **闭气**。吸气后屏住呼吸，既不吸气也不呼气，持续5到8秒钟。将气息吸到腰部区域。公式表示为：`闭气时长 ∈ [5秒, 8秒]`。
3.  **鼻喷气**。闭气5到8秒后，用鼻腔将刚才吸入的气一下子用力喷出。

**呼吸原理**：我们平常多用鼻腔吸气，因此鼻腔内会积存灰尘和细菌。清肺内功通过口腔吸气、鼻腔猛烈喷气的方式，旨在清理肺部、气管及鼻腔内的浊物。练习后会感到呼吸特别通畅。

## 核心运气要领

掌握了呼吸方法后，本节我们将深入探讨在闭气阶段需要配合的“运气”过程。这与强调放松的“强肾七式”不同，清肺内功的运气要求全身用力。

运气发生在“吸气”之后、“闭气”的这段时间内。其核心要求是：在闭气绷紧的同时，协调调动全身多个部位发力。

以下是运气的六点具体要求：
1.  **脚趾抓地**。十个脚趾如同大树生根，牢牢抓住地面。
2.  **两膝暗劲内扣**。膝盖暗中向内侧用力扣紧，但外观上不明显晃动。
3.  **提肛收肾**。男性需上提肛门，并将睾丸向上收至生殖器根部；女性需收紧外阴。
4.  **气沉丹田**。将气息沉于小腹丹田处。
5.  **闭气绷紧**。在闭气的同时，使整个小腹区域绷紧发力，而非软塌状态。
6.  **全身用力，头部放松**。除头部保持放松外，全身应协调用力。

在练习时，腿部根据不同动作可能有内扣、前后夹等具体要求，但核心是必须用力。手上也同样要使上劲。

## 完整动作配合演示

最后，我们将呼吸与运气配合起来，完成一次完整的练习循环。

动作顺序为：**口吸气 → 闭气并同时运气（按上述六点要求发力）→ 鼻腔喷气**。喷气时要做到通畅、顺畅，将气全部喷出。
