# XO初级训练自学课：02：XO挤+道具训练

在本节课中，我们将学习如何结合道具进行XO挤训练，以增强盆底肌的控制力，并将此技巧融入实际互动中。

上一节我们介绍了基础的XO挤技巧，本节中我们来看看如何借助道具进行进阶练习。

## 🧘‍♀️ 训练准备与目标

本次训练需要使用道具。其核心作用是帮助你在实际互动中，将**XO挤**技巧自然地融入与伴侣的交互过程。

请准备好你的道具。

## 🛠️ 核心流程与技巧

以下是结合道具进行XO挤训练的标准流程。

1.  **涂抹润滑与进入**：在道具上涂抹足量润滑油，找准角度，缓慢准备放入身体。此时，先进行一次**X型挤**（向外推），同时深吸一口气，欢迎道具进入。
2.  **初始夹紧**：当道具进入约两三厘米时，进行**O型夹**（向内收），将玩具夹紧。
3.  **深入与放松**：当道具到达最深处时，再次进行**X型挤**（向外推）。

理解上述流程后，我们正式开始训练。

## 🌀 分步训练指导

请调整呼吸，深吸一口气，然后缓慢呼出，保持呼吸平缓。准备好后，我们开始。

**第一步：向外推 (X型挤)**
保持X型挤的动作，将阴道口张开。用阴道深处的肌肉发力，将玩具轻轻向外推出一点。注意控制力度，确保玩具不脱离阴道，仅停留在阴道口附近。

**第二步：向内收 (O型夹)**
现在，尝试解放双手，仅用盆底肌的力量将玩具向内吸入。初期力量不足是正常的，请尽可能尝试。注意力集中在阴道口，用阴道口的肌肉向内挤压。

**第三步：循环练习**
接着，我们循环进行推与收的练习。
*   再次向外推：注意控制力度，避免将玩具推出。
*   再次向内收：尽力收紧，将玩具吸入。

## ⏩ 节奏与进阶练习

接下来，我们将逐步加快练习节奏。

以下是节奏练习的口令指引：
*   以“12345678”为节拍，在“5678”时向外推。
*   下一个“5678”时向内收。
*   循环进行“向外推”与“向内收”，注意推的幅度不宜过大。
*   持续练习：**X型挤，O型夹；X型挤，O型夹**。
*   最后，放慢速度，进行几次舒缓的**X型挤**与**O型夹**，然后保持收缩状态片刻。

完成一组后，请休息片刻。

## 🎯 高阶目标与练习建议

在后续练习中，你可以尝试更高阶的控制：仅用左侧肌肉发力时右侧放松，仅用右侧发力时左侧放松，从而控制道具在体内顺时针或逆时针转动。目标是完全解放双手，仅通过盆底肌收缩达到高潮。

这对新手而言难度较高，但无需担心。只要跟随我们7天XO挤集训的音频，每天保质保量练习，就一定能够达成目标。
