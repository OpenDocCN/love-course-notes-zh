# 生命能量-命根子智慧和功夫：001：为什么绝大多数人赚不到钱 💰

在本节课中，我们将要学习一个核心问题：为什么绝大多数人都想赚钱，却最终赚不到钱？我们将通过对比分析，揭示其背后的深层原因，并引入一套全新的思维体系。

## 课程概述

我们首先需要认识到，赚钱能力的缺失并非偶然，而是源于一套系统性的教育缺失和思维误区。本节课将对比一个在财富创造上极为成功的民族——犹太民族，来剖析我们自身在金钱观、财富教育以及人性智慧上的不足。

## 为什么绝大多数人赚不到钱

上一节我们概述了课程目标，本节中我们来看看导致“想赚钱却赚不到钱”这一普遍现象的核心原因。

绝大多数人赚不到钱，最重要的原因之一是：**我们严重缺乏一套关于金钱、财富以及如何创造财富的系统性教育**。

这与一个民族形成了鲜明对比：犹太民族。犹太民族人口仅占世界总人口的约0.2%，却据说控制着世界上40%的财富。他们拥有悠久的历史，尽管曾失去国家、颠沛流离2000年，却在1948年复国，并在科学、商业等领域成就斐然。

他们的成功并非偶然，而是源于两套强大的支撑体系：

### 第一套体系：信仰支撑体系（道德价值观教育）

这套体系的核心是做人的根本教育，即道德价值观。犹太民族通过他们的经典（如《旧约》），从小对孩子进行内省、反思、诚信、感恩等“天使一面”品格的塑造，用以约束人性中“魔鬼一面”（如自私、冷漠、不择手段）。这相当于为人生这棵“树”从小扶正，确保其长大后能正直生长。

**公式表示：**
```
健全人格 = 信仰体系（道德价值观教育）
```

反观我们的教育环境，在0至15岁的人格塑造关键期，普遍更重视书本知识（技能教育），而非做人的根本道德教育。这导致许多人进入社会时，在“做人”方面存在短板，难以赢得他人的欣赏与支持。

### 第二套体系：智慧支撑体系（人性与商业法则教育）

这套体系的核心是**智慧**。犹太民族拥有250万字、22卷的《塔木德》，其中汇集了世代传承的做人、做事、经商的法则，尤其是**了解人性、经营人性、满足人性**的具体方法和策略。

**核心概念定义：**
```
智慧 = 了解人性 + 经营人性 + 满足人性
```

他们认为，真正的智慧能让你赢得他人的欣赏、认可、支持与跟随。世界上永远不变的是**人性**，掌握了与人打交道的核心法则，就掌握了成功的王牌。

我们的教育在此方面同样存在巨大缺失：
1.  **缺乏金钱观教育**：我们从小未被系统教育“钱是什么”、“如何正确赚钱”、“什么钱值得赚”。反而常被灌输“追求金钱不高尚”的观念，这与人性追求“名”和“利”的基本需求相悖。
2.  **混淆知识与智慧**：我们极度重视书本知识（学历，可视为“铜牌”或“银牌”），却严重忽视关乎人际成败的“智慧”（真正的“王牌”）。导致许多高学历者进入社会后，不会交朋友、不被同事欣赏、难以经营感情。
3.  **赚钱思维本末倒置**：很多人的思维是“如何把别人口袋的钱装进自己口袋”，而非“如何通过提供更好的产品和服务来创造价值、换取财富”。这是一种索取而非创造的思维。

以下是导致赚不到钱的几个关键点总结：
*   **根源不正**：缺乏正确的道德价值观根基，做人失败。
*   **智慧缺失**：不懂人性，不会经营人际关系，无法与社会有效连接。
*   **教育错位**：重视技能知识，轻视财富教育与人性智慧。
*   **思维颠倒**：只想赚钱，不想如何先让自己“值钱”；只想索取，不愿创造价值。

## 思维决定命运的链条

理解上述原因后，我们需要建立一个正确的认知链条，这是改变的开始。

这个逻辑链条是：
```
思维 → 行为 → 习惯 → 性格 → 命运
```
1.  **思维决定行为**：你拥有什么样的想法，就会采取什么样的行动。
2.  **行为决定习惯**：长期重复的行动会固化为习惯。
3.  **习惯决定性格**：一系列习惯构成了你的性格。
4.  **性格决定命运**：你的性格最终决定了你人生的走向和结果。

因此，若想改变“赚不到钱”的命运，必须从源头——**思维**——开始改变。我们需要用世界级成功者的思维智慧，替换掉我们头脑中普通甚至错误的“软件程序”。

## 本节课总结

本节课中，我们一起学习了“绝大多数人赚不到钱”的深层原因。核心在于我们在**道德价值观教育**和**人性智慧教育**上的系统性缺失，导致我们做人根基不牢、不懂人性、缺乏正确的财富思维。我们通过对比犹太民族的两大支撑体系，明确了学习的方向：要改造我们的思维，学习世界级成功者关于人性与财富的智慧，从让自己“值钱”开始，最终实现赚钱的目标。
