# 武当山济肾堂功法教程：2：清肺内功第一式详解 🫁

在本节课中，我们将要学习武当山济肾堂清肺内功的第一式。清肺内功共有三式，第一式是核心基础，重点在于独特的呼吸控制与肢体动作的配合。我们将分步解析动作要领、呼吸方法及运气技巧。

## 动作分解与起始姿势

上一节我们介绍了课程背景，本节中我们来看看第一式的具体练习方法。首先从身体的起始姿势开始。

练习者需两脚打开站立，宽度略大于肩宽。直接打开即可，无需特殊起势。

以下是关于脚部姿势的具体要求：
*   两脚脚掌朝前。
*   双脚平行向前，不呈外八字，也不向内扣。

## 肢体动作流程

确立好稳定的下盘姿势后，我们开始进行上肢动作。双手伸直，从身体前方直接向上抬起，抬至与肩同高、与肩同宽的位置。

随后，双手回收，同时屈膝下蹲。双手随下落之势自然向下，当手落至腹部时，下蹲动作也正好完成。

接着，双手在腹部前做一个抱球的动作，想象两手之间环抱了一个球。

抱球后，双手手掌向外翻转。翻掌时，所有手指需弯曲。

此时，想象双手各抓住一个极重之物（如两座大山），然后用力将这两个重物向身体两侧分开。

以下是动作的最终环节：
*   双手用力向两侧分开至极限。
*   在动作的最后，想象将重物猛然丢出去。

至此，便完成了第一式的全部肢体动作。

## 呼吸与运气配合详解

掌握了肢体动作后，最关键的是配合独特的呼吸与运气方法。下面我们来配合呼吸方式详细讲解。

整个过程的起始是吸气。当双手从体前向上抬起时，伴随吸气动作。

此时上半身可微微前倾，用口腔将气吸满。**抬手 = 吸气**。

双手回收时，呼吸暂停，不呼不吸。

当双手屈膝下蹲并下落时，将刚才吸满的气呼出去三分之一。**下落 = 呼出1/3**。

这意味着，我们体内仍保留着三分之二的气。

这剩余的气将存于腹部。在双手做抱球动作时，开始闭气，不呼不吸。

气闭住后，开始运气。以下是运气时的身体要领：
*   两脚脚趾用力抓地。
*   两个膝盖微微向内扣。
*   提肛收臀，气沉丹田。
*   全身闭气绷紧。

在绷紧状态下，双手用力向两侧分开。整个运气过程持续5到8秒。

5到8秒后，在双手向外打出的同时，将闭住的气瞬间喷出。**外打 = 喷气**。

## 完整流程回顾与要点总结

为了帮助大家融会贯通，我将关键要点再串联讲解一遍。

首先是吸气抬手，接着回收时呼吸暂停，然后下落时呼出三分之一的气。随后闭气抱球，开始运气，双手用力向两侧分开。

此时需注意身体形成的对撑力：膝盖向内扣，而双手向外分，形成上下相反的张力。

在5到8秒的运气过程后，将气一下子喷发出去。

本节课中我们一起学习了清肺内功第一式的完整练习方法。我们从基础的站立姿势开始，逐步学习了抬手、下蹲、抱球、分掌的肢体动作，并重点掌握了“吸满—呼出部分—闭气运气—瞬间喷发”这一独特的呼吸运气节奏。请牢记动作与呼吸的配合要点，膝盖内扣与手外分的对撑力是运气的关键。坚持练习，有助于增强肺活量与气息控制能力。
