# 艳舞私房课：06：小白必学的艳舞基本姿势

在本节课中，我们将要学习一些适用于舞蹈的基本姿势。这些姿势是构成舞蹈动作的基础单元，掌握它们能帮助你更好地衔接动作，并融入任何舞蹈套路中。

上一节我们介绍了课程的整体框架，本节中我们来看看具体的基础姿势。

首先，观察舞者如何从一个姿势流畅地转换到另一个姿势。这个过程的核心在于对身体的控制和节奏的把握。

以下是几个关键的基础姿势动作描述：

*   **下蹲**：弯曲膝盖，降低身体重心。
*   **后仰**：在保持平衡的前提下，向后弯曲上半身。
*   **举手**：将手臂向上或向侧方伸展，延伸身体线条。

这些基础动作，如下蹲、后仰和举手，可以被灵活运用于任何舞蹈编排中。

在练习这些姿势时，需要配合音乐节奏，并用心感受身体的移动和姿态。动作的翻转与连接是舞蹈流畅性的关键。

本节课中我们一起学习了艳舞的几个基本姿势，包括下蹲、后仰和举手。理解并熟练这些姿势是构建更复杂舞蹈动作的第一步。记住，在练习中要注重姿势的准确性与动作之间的流畅转换。
