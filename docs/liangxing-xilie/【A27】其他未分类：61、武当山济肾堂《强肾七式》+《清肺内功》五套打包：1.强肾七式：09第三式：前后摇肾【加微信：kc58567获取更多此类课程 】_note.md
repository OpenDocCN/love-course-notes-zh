# 武当山济肾堂强肾功法：09：第三式 - 前后摇肾 🧘‍♂️

在本节课中，我们将要学习武当山济肾堂《强肾七式》中的第三式——前后摇肾。此式旨在通过特定的身体伸展与呼吸配合，重点锻炼腰部，以达到强健肾脏、疏通经络的效果。我们将从预备姿势开始，逐步分解动作要领，并阐明练习的核心原则。

## 概述

前后摇肾式是一套结合了呼吸与脊柱运动的养生功法。其核心在于通过缓慢、放松的后仰与前屈动作，充分拉伸腰椎、脊椎、颈椎及肩关节，同时配合深长的呼吸，以激活肾气，滋养身心。

## 预备姿势

以下是动作的起始步骤。

1.  两脚并拢，脚后跟与脚尖完全并拢。
2.  屈膝下蹲。
3.  提起左脚，脚后跟先抬起，随后脚尖慢慢提起。
4.  落脚时，脚尖先落地，然后脚后跟慢慢放下。
5.  双脚打开，宽度与肩同宽。
6.  双腿站直。
7.  站直后，先进行三次深呼吸。采用口呼鼻吸的方式。
8.  呼完第三口气后，在呼第四口气时，双手从身体两侧提到胸前。
9.  此时气已呼完，接着吸气，然后呼气，双手随之下按。

## 动作分解与要点

上一节我们介绍了预备姿势，本节中我们来看看核心动作的具体做法与关键要领。

首先，确保双脚脚尖完全朝前，不要呈外八字或内扣。

当从预备姿势的呼气末开始吸气时，身体向后仰。此过程有几个核心要点：

*   **腰椎受力**：腰为肾之府，练腰即强肾。后仰时，腰部应主动受力，并向斜后方伸展。
*   **脊椎伸展**：整个脊椎应斜着向后方延展。
*   **颈椎后拔**：颈椎需完全向后伸展。
*   **肩关节打开**：两个肩关节务必充分打开。

因此，在后仰时，需同时将腰椎、脊椎、颈椎、肩关节这四个部位舒展拉开。对于初学者，前三遍练习不必追求极限幅度，可先轻柔舒展，再逐步加大幅度。

吸气至后仰极限后，开始呼气，身体向前俯。此过程需注意：

*   **以腰为轴**：前俯时，首先弯曲的是腰部，而非脊椎或颈椎。保持上半身挺直，用腰部力量向下弯。
*   **顺序弯曲**：腰部弯到不动时，再弯曲脊椎，最后是颈椎。
*   **下颌内收**：身体前俯至最低点时，头部应回勾，用下巴去贴近锁骨，达到极限。这对颈椎是很好的锻炼。
*   **膝盖挺直**：在整个后仰与前俯过程中，膝盖应始终保持挺直，不能弯曲。手能下探到哪里就是哪里，切勿通过弯曲膝盖来追求幅度。

## 核心练习原则

在掌握了动作要领后，理解并贯彻以下练习原则至关重要。

*   **放松**：与传统运动不同，道家功法讲究放松。练习时身心皆需放松，越放松，动作越能舒展到位。这体现了性命双修的理念，既锻炼身体，也修养心性。
*   **缓慢**：养生功法追求慢。动作越慢，效果越好。这要求练习者摒弃求快的心态，在缓慢中体会身体的细微变化。
*   **配合秘诀**：练习过程中，需保持“炼精化津”的秘诀，口中会滋生津液，应按照特定方法吞下以补充肾精。（具体秘诀需由老师一对一亲授。）

## 完整演示与收功

以下是前后摇肾式的完整练习流程与收功方法。

**完整练习流程**：
1.  站定，收心，做好练习准备。
2.  执行前述“预备姿势”步骤。
3.  吸气向后仰，充分伸展腰、脊、颈、肩。
4.  呼气向前俯，依次活动腰、脊、颈，下颌内收。
5.  重复练习数次。

**收功动作**：
1.  最后一次前俯后将气呼完，微微吸气。
2.  呼气，双手收回大腿两侧。
3.  屈膝下蹲，将练习中产生的津液吞下。
4.  左脚向左开一小步，双手背于腰后。
5.  仰头，然后回正。
6.  双手打开，弯曲膝盖将腿收回，双手下按，恢复初始站姿。

**收功注意事项**：
*   双手背于腰后时，上半身需保持中正挺直。
*   仰头时，膝盖应微曲，勿挺直。
*   开步与收步时，膝盖始终是弯曲的，动作应轻缓。

## 总结

本节课中我们一起学习了强肾七式的第三式“前后摇肾”。我们从预备姿势开始，详细分解了后仰与前俯的动作要领，强调了以腰为轴、顺序弯曲以及膝直的关键点。更重要的是，我们明确了“放松”与“缓慢”两大核心练习原则，这是本功法见效的根基。最后，通过完整的流程演示与收功方法，为大家呈现了该式功法的全貌。坚持练习，有助于拉伸筋骨、畅通气血，达到强肾健体的目的。
