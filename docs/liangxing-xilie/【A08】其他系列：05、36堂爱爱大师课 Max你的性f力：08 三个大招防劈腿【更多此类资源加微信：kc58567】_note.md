*   **共同探索**：约定一个时间尝试一种新姿势，努力的发掘出更合适你们的体位。
*   **利用恐惧激发**：第一招里提到的营造未知恐惧的氛围，还有一个即学即用的小办法：和爱人一起看一场恐怖片，或者进行刺激的极限运动。危机会通过激发我们对死亡的恐惧而连带激发性欲，让人会想疯狂做爱来找回安全感。

总之，在前几集里说过很多做一个好爱的办法，不要害怕给你的爱人惊喜。
