# 武当山济肾堂《强肾七式》+《清肺内功》：5.2：顺腹式呼吸法 🧘

在本节课中，我们将要学习顺腹式呼吸法，这是一种通过调整呼吸模式来增加氧气摄入、改善身体能量状态的重要方法。我们将从理论原理开始，逐步讲解其具体练习步骤和应用场景。

## 理论原理：认识“气态食物”

上一节我们介绍了调息炼气的概念，本节中我们来看看“气态食物”这一核心理论。

气在这里指我们呼吸的空气。空气中含有人体必需的氧气。

“食气”有两层含义。第一层是将“食”当作“吃”来理解，即摄取空气中的氧气。第二层是将“食”当作“食物”来理解。

人类能够生存在地球上，离不开三类食物。

以下是三类食物的分类：

*   **固态食物**：例如我们吃的米饭、蔬菜、肉类、水果、坚果。
*   **液态食物**：例如我们每天喝的水、果汁、奶、汤。
*   **气态食物**：即我们呼吸的空气。

一般人通常只注重固态食物和液态食物，认为这两类才是食物，对气态食物不够重视。实际上，一个人可以7天不吃饭，3天不喝水，但如果3分钟呼吸不到空气，就无法生存。道家的祖先很早就认识到，气态食物是对人体最重要的一类食物。

然而，平常很多人对气态食物的重视度不够，每次吸入的空气量很少，导致获得的氧气不足，身体长期处于缺氧状态。现代科学已证明，许多慢性疾病都与缺氧有关。所有癌细胞都有一个共同特征：厌氧，即讨厌有氧气的环境。

因此，“食气法”的核心目的，就是通过改变呼吸方式，增加每次吸入的空气量。吸入的空气越多，人体能从中获得的氧气就越多。人体能量的两大来源，一是血液，二就是氧气。

## 练习方法详解

理解了“食气”的重要性后，现在我们来学习顺腹式呼吸法的具体练习方法。

此法的练习姿势不限，行、站、坐、卧皆可。为便于讲解，此处以站立姿势为例。初学者可将双手相叠，置于腹部位置。

呼吸的顺序是**先呼气，再吸气**。

以下是详细的呼吸步骤：

1.  **呼气**：用嘴或鼻腔缓缓呼气，同时有意识地将腹部向内回缩、凹陷。
2.  **吸气**：用鼻腔缓缓吸气，同时有意识地将腹部向外鼓起、膨胀。

每一次呼气，都应尽量将体内的气体全部呼出，腹部微微用力向内收。每一次吸气，则应尽量吸满，直至不能再吸为止。

## 原理与要点

为什么这样做可以增加吸气容量呢？关键在于理解呼吸的生理机制。

我们的呼吸器官是肺，它位于胸腔。吸气时，肺需要膨胀，因此需要空间。当我们吸气并鼓起腹部时，横膈肌会向下移动，挤压五脏六腑下移，从而为肺的膨胀腾出更多空间，使其能容纳更多气体。公式可以简单理解为：**吸气鼓腹 → 横膈肌下移 → 胸腔空间增大 → 肺容量增加**。

反之，呼气时腹部回凹，横膈肌上移，就像一只无形的手将肺里的废气挤压出去。因此，吸气时腹部要鼓起，呼气时腹部要回缩。

呼吸时的核心要求是**均匀、深长**。不能急促呼吸，应尽量将呼吸的节奏放慢、拉长。

## 练习要求与日常应用

顺腹式呼吸法的动作虽然简单，但其练习方式与其他招式不同。

其他招式可以每天固定一个时间练习。但食气法不行，它并非每天练习5-10分钟即可。它要求我们养成一种习惯，让行、站、坐、卧都尽量按照这种深长均匀的呼吸方式来进行。

只要大脑未被特定事务占据，我们就应有意识地调节呼吸，采用这种呼吸方式。

以下是一些可以应用此法的日常场景：

*   等车时。
*   坐车时。
*   坐电梯时。
*   走路时。

只要有空闲，就应采用这种呼吸，因为每一次深长呼吸都能为身体带来更多氧气。如果身处空气清新、有树有草有水的地方，更应抓紧时间，积极采用这种呼吸方式，为身体摄入高质量的氧气。
