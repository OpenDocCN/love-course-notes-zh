# Python编程入门：1：变量与数据类型

在本节课中，我们将要学习Python编程中最基础也最重要的两个概念：**变量**与**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？

上一节我们介绍了课程概述，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，这个盒子用来存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

**核心公式**：`变量名 = 值`

例如，我们创建一个名为 `age` 的变量，并将数字 `20` 放入其中：
```python
age = 20
```
执行这行代码后，Python就在内存中创建了一个名为 `age` 的“盒子”，里面存放了数值 `20`。之后，当我们使用 `age` 这个名字时，Python就知道我们指的是这个值。

以下是使用变量时的几个关键点：
*   **变量名**：是盒子的标签，用于在代码中引用这个值。它由字母、数字和下划线组成，但不能以数字开头。
*   **赋值操作符（`=`）**：不是数学中的“等于”，而是“将右边的值，赋予左边的变量名”。
*   **值**：是存放在盒子里的具体内容，可以是数字、文字等多种形式。

## 基本数据类型

我们已经知道如何用变量存放数据，那么数据本身有哪些种类呢？这就是数据类型要回答的问题。Python中常见的基本数据类型有以下几种。

以下是几种核心的数据类型：

1.  **整数（int）**
    表示没有小数部分的数字，可以是正数、负数或零。
    ```python
    year = 2023
    temperature = -5
    count = 0
    ```

2.  **浮点数（float）**
    表示带有小数点的数字，用于处理更精确的数值。
    ```python
    price = 19.99
    pi = 3.14159
    ```

3.  **字符串（str）**
    表示文本信息，需要用单引号（`‘’`）或双引号（`“”`）括起来。
    ```python
    name = “小明”
    greeting = ‘Hello, World!’
    ```

4.  **布尔值（bool）**
    只有两种取值：`True`（真）或 `False`（假），常用于逻辑判断。
    ```python
    is_learning = True
    is_completed = False
    ```

## 如何查看数据类型？

在编程时，有时我们需要确认一个变量里存放的是什么类型的数据。Python提供了一个非常方便的函数 `type()` 来帮助我们。

**核心代码**：`type(变量名)`

例如：
```python
age = 20
print(type(age))  # 输出：<class ‘int’>

price = 29.9
print(type(price)) # 输出：<class ‘float’>
```
`print()` 函数会将 `type()` 函数的结果显示在屏幕上，这样我们就能清楚地知道 `age` 是整数类型，`price` 是浮点数类型。

## 变量的重新赋值

变量的值并非一成不变。正如你可以把盒子里的旧东西拿出来，再放入新东西一样，变量也可以被重新赋予新的值。

例如：
```python
message = “你好”  # 初始赋值
print(message)    # 输出：你好

message = “世界”  # 重新赋值
print(message)    # 输出：世界
```
请注意，当变量被重新赋值后，它之前存放的值就被新的值替换掉了。

本节课中我们一起学习了Python编程的基石：**变量**与**数据类型**。我们知道了变量是存储数据的命名容器，并通过 `=` 进行赋值。同时，我们认识了四种基本数据类型：整数（`int`）、浮点数（`float`）、字符串（`str`）和布尔值（`bool`），并学会了使用 `type()` 函数来检查数据类型。掌握这些概念，你就已经迈出了编写Python程序的第一步。
