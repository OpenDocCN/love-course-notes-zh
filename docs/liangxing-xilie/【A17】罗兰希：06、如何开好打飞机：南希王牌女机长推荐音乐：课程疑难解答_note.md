上一节我们介绍了音乐在飞行训练中的辅助作用，本节中我们来看看学员们在实践过程中遇到的具体疑问及其解决方案。

以下是关于音乐选择与应用的常见问题列表：

1.  **问题：推荐的音乐类型是什么？**
    *   **解答**：推荐节奏稳定、旋律舒缓的纯音乐或古典乐。避免歌词复杂或节奏激烈的音乐，以免分散注意力。核心原则是音乐应作为背景，帮助建立稳定的操作节律，而非成为注意力的焦点。

2.  **问题：音乐音量应该如何控制？**
    *   **解答**：音量需保持在较低水平。一个实用的参考公式是：`环境噪音 < 音乐音量 < 仪表警报音量`。确保你能清晰听到所有驾驶舱内的重要音频提示。

3.  **问题：在特情处置时应该听音乐吗？**
    *   **解答**：不应该。一旦进入特情处置程序，应立即暂停或关闭音乐。此时需要全神贯注于标准操作程序（SOP），代码如下所示：
        ```plaintext
        if (特情发生 == true) {
            执行处置程序();
            音乐.暂停();
        }
        ```
