# 性福诊疗室：A18：不做爱是不是没有爱 ❓

在本节课中，我们将探讨一个常见的亲密关系疑问：缺乏性行为是否等同于缺乏爱。我们将从不同年龄阶段的角度分析这个问题，并理解亲密关系的多元表现形式。

很多人会询问，如果和伴侣没有性行为，是否意味着彼此之间没有爱。如果将这个观点推向极端，即“不做爱就没有爱”，那么老年人该如何自处？他们通常也没有性行为，难道他们之间就没有爱吗？事实显然并非如此。

也就是说，我们期望两个人之间能达到一种亲密的要求。即使在年纪较大的时候，我们也可以借助身体接触，来完成两人之间的亲密磨合，并不一定需要插入性的行为。

当然，在年轻时，插入性的行为会带来许多视觉或感觉上的满足。我也会期待在年轻时，个体应具备这样的功能，应该能够好好地享受，我们应尽可能去享受身体带来的愉悦。

如果等到真的无法享受时，我们就接受这样的现状，并将我们目前的状态发挥到极致。我认为这就是对伴侣关系最好的方式。

---

上一节我们探讨了爱与性行为并非绝对绑定的观点，本节中我们来看看如何在不同阶段实现亲密。

以下是实现亲密关系的不同方式：

*   **核心在于亲密感**：关系的核心是双方的亲密与连接，其形式可以多样化。
*   **年轻时的享受**：在功能允许的年轻阶段，可以充分享受包括插入性行为在内的性愉悦。公式可表示为：**亲密感 ≈ 情感连接 + 身体愉悦（包括性）**。
*   **年长时的调整**：当身体功能变化时，重点可转向非插入的亲密接触。公式可表示为：**亲密感 ≈ 情感连接 + 非插入性身体接触（如拥抱、抚摸）**。
*   **接纳与极致**：关键在于接纳当前阶段的状态，并致力于将现有的亲密方式做到最好。

---

本节课中我们一起学习了爱与性行为的关系。我们明白了，爱并不仅仅通过插入性的性行为来体现，亲密感才是关系的核心。无论是在年轻时享受性的愉悦，还是在年长时通过其他身体接触维持亲密，最重要的是接纳现状并努力经营当下的关系，将其发挥到极致。
