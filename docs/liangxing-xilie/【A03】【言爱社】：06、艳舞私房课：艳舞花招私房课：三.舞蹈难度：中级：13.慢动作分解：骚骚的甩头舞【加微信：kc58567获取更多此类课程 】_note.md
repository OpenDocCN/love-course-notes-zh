# 舞蹈教程：06：艳舞私房课：中级慢动作分解

## 概述
在本节课中，我们将学习一套中级难度的舞蹈动作，其核心在于通过慢动作分解，展现充满魅力的“甩头舞”。教程将逐步拆解每个动作，确保初学者能够理解并跟随练习。

## 动作分解

上一节我们介绍了舞蹈的整体风格，本节中我们来看看具体的动作步骤。以下是完整的动作流程分解。

**步骤一：接近舞伴**
缓慢走向你的舞伴，运用你所学过的性感步伐之一。

**步骤二：肩部滑动与甩头**
当你接近时，单腿下蹲，同时双手沿对方肩膀滑下，并将头部向后摆动。你可以选择以一个角度面对他站立，以便他能稍微看到你的动作。

**步骤三：转身与接触**
走到他正前方。做一个交叉步，转身，然后将双手放在他的膝盖上。

**步骤四：环绕臀部与对视**
臀部做环绕动作，同时注视着他。

**步骤五：坐下**
转身，坐在他的大腿上。一只手扶住他的膝盖，另一只手扶住椅子。

**步骤六：甩头与腿部动作**
头部猛地向下低，同时伸直一条腿，脚尖绷直。将双膝分开得更宽，并摆动你的头发。完成此动作时，请保持披发状态。

**步骤七：起身**
首先抬起臀部，拱起背部，以此姿势站起来。双手沿腿部向上移动，转身面对他，然后单腿下蹲，另一条腿向侧面伸直。

**步骤八：借助支撑起身**
利用他的膝盖作为支撑，站起来。

**步骤九：面对面坐下**
面对他，再次坐到他的大腿上。双手扶住椅子，摆动你的头发。然后站起来。

**步骤十：跪姿动作**
站到他身旁，面对他，然后双膝跪下。臀部向一侧摆动，双手向后支撑身体，面对他做扇子舞动作。

**步骤十一：结束动作**
背对他，双手放在地板上，然后跳起，抬起臀部并保持头部低垂。最后，以走开的动作结束这套舞蹈。
