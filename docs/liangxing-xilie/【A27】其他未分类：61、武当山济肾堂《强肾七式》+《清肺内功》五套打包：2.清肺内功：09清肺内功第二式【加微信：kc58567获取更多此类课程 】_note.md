# 武当山济肾堂强肾七式与清肺内功：2.清肺内功：09清肺内功第二式 🫁

在本节课中，我们将要学习清肺内功的第二式。这一式在呼吸方法与步伐上，均与第一式有所不同。我们将详细拆解其动作要领、呼吸配合及练习要点，确保初学者能够清晰掌握。

## 概述

清肺内功第二式的核心在于呼吸与动作的协调。其呼吸模式为**一次性完全吸气，闭气后分三次将气呼出**，同时配合特定的弓步姿势与手部动作。掌握此式有助于提升气息控制能力。

## 动作与呼吸详解

上一节我们介绍了第一式的练习方法，本节中我们来看看第二式的具体操作。第二式与第一式的主要区别在于呼吸节奏和步伐。

### 步伐与站姿

首先，其步行为弓步。具体要求如下：
*   前腿弓，后腿绷直。
*   双脚方向均与身体正中线呈 **45度**角。
*   身体正面应对准前腿膝盖的方向。

### 手部动作与吸气配合

手部动作共有四拍，均在吸气过程中完成。以下是四拍动作的分解：
1.  双手从体侧抬起，与肩同高。
2.  双手落下至体前。
3.  双手再次抬起（本次掌心向上）。
4.  双手再次落下（掌心向下）。

**关键点**：只有第三拍动作时掌心向上，其余各拍掌心均向下。

为了使动作与吸气过程完美配合，你需要：
*   要么加快动作速度。
*   要么延长吸气时间。
*   目标是**动作做完时，吸气刚好完成**。

### 闭气、运气与呼气

吸气及四拍动作完成后，进入下一阶段。

1.  **抱球**：双手在膝盖上方呈抱球状。此时身体从正直状态变为**微微前倾**。
2.  **翻掌绷紧**：双手翻掌，手指如鹰爪般用力勾回。想象双手抓握重物，同时全身绷紧，脚趾抓地。前腿与后腿产生一股向中间扣合的对抗力。
3.  **回拉运气**：以身体回正带动双手向体侧后方拉动。此过程为**闭气运气**阶段，需持续 **5到8秒**。要领是提肛收肾，气沉丹田，全身保持绷紧用力。
4.  **分次呼气**：闭气结束后，将气息分**三次**喷出。每次喷气后稍作停顿。呼气公式可理解为：`呼气 = 喷气(停顿) + 喷气(停顿) + 喷气`

## 完整练习流程与要点

以下是完整的单次练习步骤总结：

1.  摆好45度弓步。
2.  开始吸气，同时完成四个手部动作。
3.  吸气结束，双手抱球，身体微前倾。
4.  翻掌绷紧，全身蓄力，脚趾抓地，双腿内扣。
5.  身体回正，带动双手向后拉，闭气保持5-8秒。
6.  分三次将气喷出。

### 练习安排注意事项

练习时需注意左右交替平衡：
*   双脚应**交替练习**（先左腿在前，后右腿在前）。
*   若计划总练习次数为6次，则每侧各练习3次。
*   换脚时可直接转换方向，也可收回后再出另一脚。

## 总结

本节课中我们一起学习了清肺内功第二式。我们重点掌握了**“一次性吸满，闭气后分三次呼出”**的呼吸节奏，以及配合呼吸的**45度弓步**和**四拍手部动作**。请记住协调动作与吸气的关键，并在闭气运气时保持全身绷紧。通过左右交替的均衡练习，你将能更好地掌握此式精髓。
