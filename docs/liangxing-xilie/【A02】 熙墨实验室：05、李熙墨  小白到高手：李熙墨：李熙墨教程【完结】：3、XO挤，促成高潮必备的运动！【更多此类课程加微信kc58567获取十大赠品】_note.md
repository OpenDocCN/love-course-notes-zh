## 核心动作：XO挤

第一项就是我们常说的 **XO挤**。这是最简单、最基础也最必要的动作，可以随时随地练习，不占用额外时间。

XO挤是一组动作，包含以下两个部分：

**动作一：O型夹**

**动作二：X型挤**
现在看第二张图，进行X型挤。
你需要调动阴道内部的肌肉，尝试向外“推挤”。
一个形象的比喻是：想象你在排尿困难时，用力把尿往外挤的那个动作。

## XO挤的练习要点与技巧

初学者可以这样做：将手放在下腹部（小肚子），确保腹部没有在动。如果你感觉到腹部在发力，说明动作效率不高且容易导致肚子疼。

XO挤的重点是锻炼盆底肌和括约肌，发力点应集中在这两个部位。

如果你觉得发力困难，可能是因为过于追求动作幅度。初期幅度小是正常的，只要动作正确，通过持续练习幅度自然会增大。盲目追求幅度而用其他部位代偿，反而不利于目标肌群的锻炼。

如果你觉得O型夹（向内收）容易，而X型挤（向外推）困难，可以从站姿开始练习。站姿时臀部悬空，不受椅子或床的限制。

**站姿练习步骤：**
1.  站立，双腿微张与肩同宽。
2.  大腿放松，仅作支撑，腿部不要发力。
3.  收紧臀部和阴道，深吸一口气。
4.  将臀部和阴道整体向外推，深呼一口气。

> XO挤不能用腹部发力，但新手用臀部带动发力是可以接受的，因为肛门括约肌与盆底肌是相连的。锻炼肛门括约肌的同时也能刺激盆底肌。配合呼吸有助于发力。

男性也可以做这个运动，只需将阴道的收缩换成肛门的收缩。强大的括约肌运动能力有助于预防勃起障碍和控制性爱时长。

一个有趣的现象是：做O型夹时，生殖器会向上弹动；做X型挤时，则会向下耷拉。你可以亲自尝试观察。
