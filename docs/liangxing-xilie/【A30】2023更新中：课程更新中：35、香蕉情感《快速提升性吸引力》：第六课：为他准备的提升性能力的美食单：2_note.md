# Python入门：1：变量与基础数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**基础数据类型**。理解这些概念是编写任何Python程序的基石。

## 概述

变量就像是一个贴有标签的盒子，我们可以把数据放进去，并且通过标签（变量名）来找到它。Python中的基础数据类型则定义了盒子中可以存放什么类型的东西，比如数字、文字等。本节课我们将逐一了解它们。

## 变量：数据的容器 📦

变量是用于存储数据值的名称。你可以把变量想象成一个带标签的盒子，标签就是变量名，盒子里装的东西就是变量的值。

在Python中，创建一个变量并给它赋值非常简单，只需要使用等号 `=`。

```python
my_variable = 10
```

这行代码创建了一个名为 `my_variable` 的变量，并把整数值 `10` 存储在里面。之后，我们就可以通过这个名字来使用这个值。

### 变量的命名规则

以下是给变量命名时需要遵守的规则：

*   变量名只能包含字母、数字和下划线（`_`）。
*   变量名不能以数字开头。
*   变量名不能是Python的保留关键字（如 `if`、`for`、`while` 等）。
*   变量名是区分大小写的（`age` 和 `Age` 是两个不同的变量）。

一个好的习惯是使用描述性的变量名，让代码更容易理解。例如，用 `user_age` 而不是 `a`。

上一节我们介绍了如何创建和使用变量，本节中我们来看看Python中有哪些基本的数据类型可以放入这些“盒子”中。

## 基础数据类型 🔢

Python内置了几种基础数据类型，用于表示不同种类的数据。最常用的有以下几种：

### 1. 整数 (int)

整数是不带小数点的数字，可以是正数、负数或零。

```python
age = 25
temperature = -5
count = 0
```

### 2. 浮点数 (float)

浮点数是带有小数点的数字，用于表示实数。

```python
price = 19.99
pi = 3.14159
height = 1.75
```

### 3. 字符串 (str)

字符串是由零个或多个字符组成的文本，需要用单引号 `‘’` 或双引号 `“”` 括起来。

```python
name = “Alice”
greeting = ‘Hello, World!’
empty_string = “”
```

### 4. 布尔值 (bool)

布尔值只有两个：`True`（真）和 `False`（假）。它们通常用于逻辑判断。

```python
is_student = True
is_logged_in = False
```

了解了这些基础类型后，我们有时需要知道一个变量里到底装的是什么类型的数据。

## 使用 `type()` 函数检查类型 🔍

`type()` 函数可以帮助我们查看任何变量或值的数据类型。

```python
print(type(10))        # 输出：<class ‘int’>
print(type(3.14))      # 输出：<class ‘float’>
print(type(“Python”))  # 输出：<class ‘str’>
print(type(True))      # 输出：<class ‘bool’>

my_var = 100
print(type(my_var))    # 输出：<class ‘int’>
```

## 总结

本节课中我们一起学习了Python编程的起点：**变量**与**基础数据类型**。

*   **变量**是存储数据的命名容器，通过赋值操作符 `=` 来创建和赋值。
*   Python的**基础数据类型**主要包括：
    *   **整数 (int)**： 表示整数值。
    *   **浮点数 (float)**： 表示带小数的数值。
    *   **字符串 (str)**： 表示文本信息。
    *   **布尔值 (bool)**： 表示逻辑真 (`True`) 或假 (`False`)。
*   我们可以使用 **`type()`** 函数来检查任何对象的数据类型。

掌握这些概念是理解后续更复杂主题（如运算、数据结构、控制流）的关键。请务必动手尝试创建不同类型的变量，并使用 `print()` 和 `type()` 函数来观察它们。
