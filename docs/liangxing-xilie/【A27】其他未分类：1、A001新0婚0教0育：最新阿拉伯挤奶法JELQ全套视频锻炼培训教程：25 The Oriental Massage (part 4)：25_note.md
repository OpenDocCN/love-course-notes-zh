## 动作详解

该动作的核心是进行直线拉伸并与身体接触。

以下是动作执行的步骤列表：
1.  将目标部位直线向外拉伸。
2.  使其头部接触同侧大腿。
3.  然后恢复原位。
4.  将此套动作在身体两侧各重复执行九次。

**动作公式**可概括为：`拉伸 -> 触碰大腿 -> 回位`，每侧重复 **9** 次。
