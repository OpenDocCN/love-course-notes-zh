## 关于健身计划制定

制定个人健身计划是成功的第一步，但初学者常感到困惑。以下是几个关键原则。

*   **循序渐进原则**：训练强度与容量应逐步增加，公式可表示为 **`本次训练量 ≤ 上次训练量 + 增量`**，避免突然激增导致受伤。
*   **专项性原则**：训练效果针对所练习的动作与能量系统。例如，想提升卧推力量，就应多进行卧推及相关辅助训练。
*   **恢复原则**：肌肉在休息时生长。确保充足的睡眠与营养，为身体恢复留出时间。

## 关于训练动作与技巧

正确的动作模式是有效且安全训练的基石。以下是两个常见问题的澄清。

*   **动作幅度**：大多数力量训练应完成全程动作，以充分刺激肌肉。例如，深蹲时髋关节应低于膝关节水平。
*   **呼吸方法**：发力时呼气，还原时吸气。在进行大重量训练时，可采用瓦式呼吸：深吸气绷紧核心，完成动作后调整呼吸。

---

## 关于饮食与营养

饮食是支撑训练成果的另一半。以下是针对增肌与减脂期的基础饮食概念。

*   **热量平衡**：体重变化核心在于热量收支。公式为：
    **`体重变化 = 热量摄入 - 热量消耗`**
    增肌需热量盈余，减脂需热量缺口。
*   **蛋白质摄入**：它是肌肉修复与合成的关键材料。推荐每日摄入量为每公斤体重 **`1.6 - 2.2 克`**。

## 关于平台期与心态

遇到进步停滞时，调整策略比盲目坚持更重要。

*   **调整训练变量**：尝试改变动作顺序、组数、次数、休息时间或增加新动作，给肌肉新的刺激。
*   **记录与回顾**：保持训练日志，定期回顾，客观评估进度与计划的有效性。
