# 艳舞私房课：06：慢动作分解 - 摇摆转身胸部扭动式 🕺

在本节课中，我们将学习一个名为“摇摆转身胸部扭动式”的初级舞蹈动作。我们将通过慢动作分解，详细讲解每一个步骤，确保初学者能够轻松跟上并掌握核心要领。

## 动作分解

以下是该舞蹈组合的完整步骤分解。我们将从脚步移动开始，逐步加入臀部和上半身的动作。

**步骤1：侧移与转身**
*   向侧方迈步，同时旋转你的髋部。
*   做一个交叉步并转身，然后向后撤步。

**步骤2：重心转移与画圈**
*   将身体重心保持在前腿，向前迈步并再次转向观众。
*   用你的髋部画一个圈。

**步骤3：交叉步与移动**
*   再次做一个交叉步并转身，接着完成撤步和向前迈步的动作。
*   将你的双手沿着身体向上移动。

**步骤4：转身与下蹲**
*   完成一个转身。
*   双脚分开站立，双手上举。
*   然后单膝跪地蹲下。

**步骤5：肩部与胸部动作**
*   左右移动你的肩膀。
*   用你的胸部画圈。
*   交换支撑的膝盖。

**步骤6：腿部动作与起身**
*   抬起非支撑的那条腿。
*   身体向下，然后向后推起自己。
*   先向右转，再向左转。

本节课我们一起学习了“摇摆转身胸部扭动式”的完整分解动作。我们从基础的侧移和转身开始，逐步加入了髋部画圈、肩部扭动和胸部绕环等元素，最后以腿部动作和方向转换结束。请记住核心要领：保持节奏，分解练习，并享受舞蹈的过程。
