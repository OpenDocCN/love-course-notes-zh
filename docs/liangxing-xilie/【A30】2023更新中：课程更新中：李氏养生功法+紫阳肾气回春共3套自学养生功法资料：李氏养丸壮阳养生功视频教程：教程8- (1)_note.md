# 李氏养生功法：第八节：控精术详解

在本节课中，我们将学习控制射精的核心方法——控精术。这是基础课程的最后一节，旨在解决早泄与射精过快的问题，帮助大家掌握自如控制射精欲望的技术。

控精术分为**外控经**与**内控经**两部分。外控经侧重于利用身体外在动作来控制射精，是基础方法；内控经则专注于腹内动作与肌肉控制，是更高级的技巧。两者结合使用，效果更佳。

## 外控经：身体动作控制法

上一节我们介绍了控精术的整体概念，本节中我们首先来看看**外控经**。其核心思想是：通过调动全身的动作与力量来抑制射精冲动，属于初级控制手段。

**前提**：此方法需在通过性交或自慰达到即将射精的临界点时进行练习。当你感觉高潮即将来临、射精欲望强烈且一触即发时，方可开始以下训练。

以下是外控经的具体操作步骤，所有动作需几乎同时完成：

1.  **深吸气并闭气**：立即长长地深吸一口气，闭住，并将气感引至小腹。
2.  **按压会阴穴**：用左手紧紧按住会阴穴。
3.  **捏紧阴茎根部**：右手用拇指、食指、中指平均用力捏紧阴茎根部，使阴茎保持勃起状态。
4.  **扣紧阴茎根并提肛**：左手手指用力扣住阴茎根部，同时用力收缩PC肌（提肛）。
5.  **调动全身肌肉**：瞪大眼睛，使全身、手、腿、腹部肌肉紧张僵硬，以此分散和转移性兴奋。
6.  **强化收缩与上挺**：咬牙强忍，用力收缩肛门、阴部及腹内肌肉，随吸气向上提。吸气时，将阴茎向上挺；呼气时，让其回落。

**关键**：必须依靠毅力忍住，绝不射精。当第一次射精感被抑制后，应继续刺激至临界点，并重复上述6个动作再次抑制。如此反复，直至阴茎疲软，但始终不射精。

古代养生典籍对此有精要总结，其口诀揭示了外控经的精髓：
> 缘经欲走，把荆愁。守足下纳钳与钩。二阴加境强忍地。登目缩经一元猴。

其含义是：当想射精时，立即抽出阴茎，手脚紧张如钳钩，肛门阴部加紧强忍，瞪眼缩颈，状如猿猴。

## 内控经：腹内肌肉控制法

掌握了利用外在动作控制的基础后，本节我们来看看更高级的**内控经**。其含义是：通过小腹内的收缩动作来控制射精，这是最根本且高效的高级手段。

内控经的核心是锻炼**PC肌**（耻骨尾骨肌）。我们的祖先早已认识到PC肌的力量对控制射精具有决定性的作用。若PC肌无力，则绝对无法控制射精。此练习不仅锻炼肌肉，更能提升神经系统对射精阈值（即引发射精所需的刺激强度）的控制能力。

内控经可随时练习，无需达到性兴奋临界点，站、坐、卧姿均可。它分为**基本练习**与**强力练习**。

### 内控经基本练习

以下是内控经的三种基本呼吸与收缩配合练习法，请跟随指引进行：

*   **半拍子法**：一呼一吸为一个循环。吸气时，用力收紧（提肛、收缩腹内）；呼气时，深深放松。
    ```
    吸气 -> 收紧
    呼气 -> 放松
    ```

*   **一拍子法**：一个呼吸专注于一种状态。第一个呼吸（吸+呼）全程保持收紧；第二个呼吸（吸+呼）全程保持放松。难度有所增加。
    ```
    呼吸1（吸+呼）-> 保持收紧
    呼吸2（吸+呼）-> 保持放松
    ```

*   **多拍子法（以三拍为例）**：收缩与放松的过程都延长至多个呼吸。例如，用三个呼吸的时间缓缓收紧至顶点，再用三个呼吸的时间缓缓放松到底。要求力量平均分配。
    ```
    呼吸1 -> 开始收紧
    呼吸2 -> 继续收紧
    呼吸3 -> 收紧至顶点
    呼吸4 -> 开始放松
    呼吸5 -> 继续放松
    呼吸6 -> 放松到底
    ```

### 内控经强力练习

在基本练习熟练后，可以进行更强化的训练。强力练习主要分为两种方法：

*   **慢收快放**：用较长时间（如5个呼吸）缓慢收缩至最强力度，然后在一个呼吸内突然完全放松。重复3次。收缩时，脚趾可抓紧地面。
    ```
    // 示例：5吸收紧，1呼放松
    吸气1 -> 渐紧
    吸气2 -> 渐紧
    ...
    吸气5 -> 收紧至顶点
    呼气6 -> 突然完全放松
    ```

*   **快收慢放**：在第一个呼吸时快速收缩至顶点并再加一点力，然后用较长时间（如后续5个呼吸）缓慢地、逐步地放松。重复3次。
    ```
    // 示例：1吸快速收紧，后续缓慢放松
    吸气1 -> 快速收紧至顶点并加力
    呼气2 -> 开始缓慢放松
    呼气3 -> 继续放松
    ...
    呼气6 -> 放松到底，感受腹内热感
    ```

内控经的训练有一定难度，需要大家耐心修炼，认真体会，并持之以恒。

## 总结

本节课中，我们一起学习了控制射精的完整技术体系。我们从基础的**外控经**开始，学习了如何在临界点通过一系列身体动作来抑制射精冲动；进而深入到高级的**内控经**，掌握了通过锻炼PC肌和调节呼吸来从根本上提升控制力的方法。请记住，外控与内控需辩证结合，相辅相成，并在日常坚持练习，方能在关键时刻应用自如。
