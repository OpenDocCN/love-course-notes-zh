# 性情大师训练馆：08：如何理解男性出轨性心理？

在本节课中，我们将从性心理学角度，探讨男性出轨的心理动因、社会现象背后的本质，并提供一些维护关系的思路。课程内容基于学术研究，旨在提供新的思考视角。

## 概述

本节课我们将分析出轨现象，特别是男性的出轨心理。我们将探讨出轨是否普遍、其背后的根本原因，以及如何应对伴侣出轨的情况。课程将从数据和理论出发，帮助你更理性地看待这一问题。

## 第一节：出轨现象的普遍性

上一节我们介绍了课程主题，本节中我们来看看出轨现象是否普遍。

第一个问题是：男人都喜欢出轨吗？这个问题没有绝对答案，因人而异。但我们可以通过数据观察趋势。

中国人民大学潘绥铭教授的团队曾进行了一项长达15年的调查，对象是18至61岁的非单身人士。调查问题是：“在你跟伴侣保持性关系的同时，你有跟别人发生过性关系吗？”

以下是调查的核心发现：
*   2000年，丈夫出轨比例约为11%，妻子约为4%。
*   到2015年，丈夫出轨比例上升至约34.8%，妻子上升至约13.3%。
*   这意味着，截至2015年，大约每三个丈夫中有一个曾出轨，每7.5个妻子中有一个曾出轨。
*   中国女性的出轨率在此数据中位列世界第一。

数据表明，并非所有男人都会出轨，但出轨比例不低。同时，女性的出轨率也在显著上升。因此，出轨已成为一种社会常态。

## 第二节：出轨的根本原因

了解了出轨的普遍性后，本节我们探讨其背后的根本原因。

许多人将出轨归咎于道德滑坡。出轨确实会造成伤害，因此是不道德的。大多数出轨者内心也明白这一点并备受煎熬，但他们仍然选择了出轨。

如果只是个别现象，可以归咎于个人道德问题。但当出轨成为大面积的社会现象时，仅从道德角度批判就难以解释，也无助于解决问题。数据显示，无论道德批判多么激烈，出轨率仍在持续攀升。

那么，应该从什么角度思考呢？答案是：人性，更具体地说是性心理。

以下是基于性心理学的核心观点：
*   **多偶倾向**：无论男女，人类本身都具有多偶倾向。这是由生物进化决定的。
*   **男性天性**：雄性天生具有竞争意识，倾向于广泛传播自己的基因。这是刻在基因里的特性，虽经文明弱化，但并未消失。
*   **女性天性**：女性同样有多偶倾向。但由于怀孕生子的高风险性，历史上女性对伴侣的选择更为挑剔。随着避孕技术发展和舆论压力变化，女性的多偶倾向也逐渐显现。

两个具有多偶倾向的人结合，期望一生只爱一人、只与一人发生性关系、共处几十年，这本身非常困难。人的一生可能会对很多人动心。

出轨反映了人的侥幸心理。当制度或环境缺乏监督，使行为有可能不被发现时，侥幸心理就容易滋生。婚姻制度压抑了多偶本能，但其本身非常粗糙，缺乏有效的监督和奖惩机制，这导致了大量侥幸心理和出轨现象。

因此，**出轨是本能，忠诚是选择**。许多出轨者并非不爱伴侣，而是在新鲜感、激情向往和侥幸心理的驱动下，陷入了“既焦虑又侥幸”的循环。

此外，补偿心理也是一部分原因。当人在婚姻中某些需求（如价值感、亲密感、性满足）得不到满足时，可能会试图从婚外关系中寻求补偿。

## 第三节：如何降低伴侣出轨的可能性

在理解了出轨的人性基础后，本节我们探讨如何在现有婚姻制度下，降低伴侣出轨的可能性。

虽然婚姻制度存在缺陷，但我们仍可在其中努力经营。以下是几个建议：

首先，承认出轨是人的天性。这不是鼓励出轨，而是让你看清本质，从而更有危机感地去用心经营关系，提前弥补可能引发对方侥幸心理或补偿心理的漏洞。

其次，增强监督机制。婚姻制度本身监督薄弱，我们可以设法弥补。

以下是具体方法：
*   **增加相处时间**：尽可能创造更多共同相处的时间和空间，减少对方单独与其他异性深入接触的机会。
*   **签订协议**：在律师协助下，签订具有法律效力的婚前或婚内协议。约定若一方出轨，另一方可在财产分割、子女抚养权等方面获得更多权益。这能形成强大的威慑力。

最后，减少对方的补偿心态。关注伴侣在婚姻中可能缺失的需求，并尝试满足。

以下是几个关键方面：
*   **价值感与成就感**：特别是中年男性，若在事业和家庭中缺乏价值感和征服感，可能转向婚外关系寻求认可。伴侣可以多给予赞赏和肯定。
*   **性收益**：性关系中的“收益”（包括生理和心理满足感）至关重要。公式可以简单理解为：**性行为选择 = 感知收益 - 感知成本**。如果对方在与你关系中总体性收益为负，他可能减少互动或向外寻求。提高对方的性收益（方法在上节课已详述），有助于将他的性需求留在婚姻内。

## 第四节：面对伴侣出轨，该如何应对

最后，我们讨论最现实的问题：如果伴侣出轨，该如何应对？

这个问题没有标准答案，最终取决于个人价值观和具体情况。但处理思路可以大致分为两种。

第一种选择是原谅并继续婚姻。数据显示，伴侣出轨后，超过70%的人选择留在婚姻中。如果选择原谅，最大的挑战是信任重建。

重建信任需要一个过程和新的起点。建议尝试与伴侣共同建立一种新的关系或合作项目，例如一起学习新技能、共同运动（如跑马拉松）、规划旅行等。在新的共同经历中，逐步重新建立连接和信任。

第二种选择是果断离开。如果你内心完全无法接受，认为信任已无法重建，那么体面离开是更好的选择。

如果决定离开，请注意以下两点：
*   **收集证据**：在摊牌前，冷静地收集有效证据（如聊天记录、转账凭证等），并咨询律师。这有助于在离婚时争取更多合法权益。
*   **调整心态**：认识到这只是遇到了一个未能克制住人性弱点的人。人的一生会喜欢很多人，对方并非唯一。

最后需要明确的是：
*   不是所有人都适合婚姻，婚姻制度本身存在粗糙之处。
*   单亲家庭同样可以给予孩子充足的爱，维持貌合神离的家庭对孩子的伤害可能更大。
*   无论选择留下还是离开，最终目标都应是活出最好的自己。

## 总结

本节课我们一起学习了以下内容：
1.  出轨是一种普遍的社会现象，并非男性特有。
2.  出轨的根本原因源于人性的多偶倾向，以及在缺乏有效监督的婚姻制度下产生的侥幸心理和补偿心理。
3.  通过承认人性、增强监督、关注并满足伴侣需求，可以在一定程度上降低出轨风险。
4.  面对伴侣出轨，没有固定解决方案，可根据自身情况选择原谅重建或果断离开，核心是做出让自己未来更好的决定。

希望本课程能为你提供一个看待出轨问题的新视角，将其从纯粹的道德批判中解放出来，更理性地面对这一复杂的社会现象。
