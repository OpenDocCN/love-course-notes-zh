# 道家内功入门：A27：5.1：调息炼气基础 🧘

在本节课中，我们将学习道家内功入门中关于“调息炼气”的核心方法与原理。我们将深入理解“凌晨调息调息凌神，凌神入气血”这十三字真传，掌握如何通过调整呼吸来凝聚心神、滋养气血。

## 核心概念解析

上一节我们介绍了打坐的基本姿势（调形），本节中我们来看看如何调整呼吸，即“调息”。

“凌晨调息调息凌神，凌神入气血”是道家内功入门的十三字真传。理解这十三字，能解决打坐时调息、调心、调神的问题。

### 什么是“凌神调息”？

“凌神”指集中我们的注意力与心神，专注于一件事。专注做什么呢？调息。

“调息”指调整我们的呼吸。呼吸如何调整？首先将注意力集中在鼻孔位置，感受空气进出的感觉与快慢。调息的目标是将呼吸调整得**均匀、细长**。集中注意力在鼻孔处，调整呼吸至均匀细长。

### 什么是“调息凝神”？

“调息凝神”指通过调整呼吸（调息），达到凝聚心神（凝神）的目的。反过来，通过调息也能加强精神集中、心神专注的效果。这就是调息凝神。

### 什么是“凌神入气血”？

“气血”指我们的下丹田。下丹田是腹部肚脐下方的一个区域，并非一个精确的点。

如何“凌神入气血”？即如何将心神引入下丹田？方法是“调息凌神”。通过在下丹田处进行调息，来达到将心神引入气血（下丹田）的效果。这就是凌神入气血。

## 下丹田调息方法

上一部分我们理解了核心概念，现在来学习具体的操作方法。

下丹田如何调息？最初我们在鼻孔处调息。此外，下丹田所在的腹部也可以调息。原因在于我们采用**顺腹式呼吸**：吸气时腹部自然鼓起，呼气时腹部自然内凹。我们可以有意识地调整腹部鼓起与内凹的节奏与幅度，通过这种调整来实现“调息”。当注意力集中在此处进行调整时，心神便自然到达下丹田，实现“凌神入气血”。

“凌神入气血”即是很多人所知的“意守丹田”概念。但许多人只知概念，不知具体做法。这十三字真传详细指导了我们如何做到意守丹田。

## 原理探讨：为何意守丹田能“炼精化气”？

意守丹田的一个主要功效是“炼精化气”。为何意守丹田可以炼精化气？在众多解释中，陈式太极拳实用拳法海外掌门人陈中华老师的解释最为形象。

陈老师解释道：我们的腹部下丹田区域内部是什么？是**大肠**，其中存有粪便等消化后的废物。这些东西可以“化腐朽为神奇”。生活中类似的事物是**沼气池**。沼气池是一个密闭空间，填入动物或人的粪便后，经发酵产生沼气，成为一种可利用的能量（如烧火煮饭）。

我们的下丹田就类似一个小的“沼气池”。但使用过沼气池的人知道，沼气产量与温度密切相关。夏天温度高，产量大；冬天温度低，产量小。我们的下丹田这个“小沼气池”同样需要**热量**。

“炼精化气”的“炼”字带“火”字旁，意味着需要热量。人体的热量从何而来？来自于我们的“神”，即心意与注意力。注意力集中在哪里，人体的热能就汇聚在哪里。因此，当我们意守丹田时，该区域就会发热。热量能将体内的物质再次炼化，转化为能量（气），使气更充足，并滋润全身。

这就是“凌晨调息，调息凌晨，凌晨入气血”原理的一种形象阐释。

## 实践步骤指南

理解了原理后，以下是具体的练习步骤。请注意，调息必须在调形（调整姿势）的基础上进行。形不正，则息难调。若未学习调形，建议先掌握正确的打坐姿势。

以下是打坐调息的具体步骤：

1.  **调整姿势**：首先，将打坐的姿势调整端正。
2.  **凌神调息（鼻孔处）**：将注意力集中在鼻孔位置，感受呼吸。初步将呼吸调匀、调细、调长、调深。
3.  **转移注意力至下丹田**：随后，将注意力从鼻孔向下转移至腹部。
4.  **下丹田调息**：感受腹部随呼吸的起伏。吸气时，让腹部缓慢、均匀地向外鼓起；呼气时，让腹部缓慢、均匀地向内凹陷。保持呼吸的均匀、细长。此时，注意力完全集中在腹部。
5.  **达成状态**：通过以上步骤，即可做到“凝神入气血，调息于丹田”。
