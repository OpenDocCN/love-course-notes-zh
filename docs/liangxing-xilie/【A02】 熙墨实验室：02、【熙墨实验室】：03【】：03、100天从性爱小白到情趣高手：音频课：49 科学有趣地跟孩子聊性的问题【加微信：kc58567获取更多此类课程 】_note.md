# 性教育父母指南：02：科学有趣地与孩子谈论性

在本节课中，我们将学习如何以科学、坦然的态度与孩子谈论性话题。我们将探讨父母应有的认知、核心原则以及实用的教育资源，旨在帮助孩子建立健康的性观念。

---

上一节我们提到了性教育的重要性，本节中我们来看看与孩子谈性的核心要点。以下是三个关键要素。

**第一个要素是以平常心谈论性，不避讳提问，不打击好奇心。**

许多父母询问开始性教育的最佳时机。答案永远是越早越好。研究表明，婴幼儿在0至2岁就开始对身体和性产生好奇。他们会触摸自己的性器官，观察身体的不同。因此，当孩子开始学习语言和身体部位名称时，就可以自然地引入性教育。

对于0-2岁的幼儿，可以使用叠词、可爱的形容词或小故事来解释。例如：
*   当孩子指着碗问“这是什么？”，你会回答“这是碗，用来装饭饭的”。
*   同样，当男孩指着阴茎问时，你可以回答“这是**阴茎**，它有个小名叫小鸡鸡，用来尿尿，长大后还负责生宝宝”。

如果孩子问“宝宝是怎么生出来的？”，你可以说：“妈妈两腿之间有一个开口叫**阴道**，宝宝就是从那里出来的。”关键在于，将性相关词汇与其他身体部位名称同等对待，不特殊化，才能消除孩子不必要的困惑和过度好奇。

如果孩子没有主动提问，可以在日常认知教育中一并介绍。例如，在教孩子认识眼睛、鼻子时，也自然地介绍乳房、阴茎、肛门等。坦诚的解答能给孩子带来安全感，并减少未来青春期出现问题的可能性。

**第二个要素是警惕孩子欠缺的隐私意识，而非自慰行为本身。**

婴幼儿没有隐私意识，可能会当众触摸自己的性器官。从生理健康角度看，**自慰**本身无害，只需注意手和器官的清洁。真正有害的是父母的批评和羞辱带来的心理伤害。

父母若遇到孩子自慰，应告知：
1.  这是无害的私密行为。
2.  只能在私人房间进行，不能当着他人。
3.  要注意清洁卫生。

同时，必须向孩子明确**隐私部位**（如阴茎、阴道、乳房、肛门）的概念，并制定规则：除了孩子自己和父母（在必要情况下，如洗澡、就医），任何人都不可以触碰。有些父母担心孩子会在外谈论这些词让自己难堪。但请记住，孩子的安全教育和自我保护远比成人的面子重要。必要时，你可能需要向周围的其他成年人解释进行性教育的必要性。

**第三个要素是妥善处理孩子撞见父母亲密行为的场景。**

如果孩子撞见父母性爱，这并非坏事，可以转化为教育契机。你可以告诉孩子：“这是爸爸妈妈相爱和表达爱的方式，你就是通过这样的方式来到我们身边的。”借此机会，可以结合性教育绘本讲解生命的诞生。

之后，需要向孩子说明界限：
1.  **隐私界限**：这是父母的私密行为，如果房门锁上，不可以随意闯入。
2.  **年龄界限**：性很美好，但需要等身体成熟后才能体验。
3.  **意愿界限**：性必须基于双方同意，不能强求他人，也要懂得拒绝。
4.  **关系与安全界限**（对青春期前的孩子）：未来发生性行为需在平等、负责任的关系中，并做好避孕和疾病预防。

向孩子传递“爱与性是美好的”这一观念，同时明确其界限，对孩子未来的情感生活有积极意义。

上一节我们探讨了核心原则，本节中我们来看看有哪些实用的资源可以帮助进行性教育。以下是推荐的书籍和动画片。

**首先推荐的是适合与孩子共读的绘本：**
1.  **《小威向前冲》**：用故事讲述精子“小威”的竞赛，巧妙地解释了生命的诞生，同时融入了坚持、爱与接纳不完美的主题。
2.  **《乳房的故事》**（土屋麻由美/相野谷由起）：以温情的方式满足孩子对乳房的好奇，解答“妈妈的胸部为什么比较大？”“里面有牛奶吗？”等问题。
3.  **《小鸡鸡的故事》**（山本直英）：讲解男女身体结构的不同、生命的诞生、清洁保护以及防止性侵害，传达“每个生命都是宝贵的”情感。
4.  **《成长与性》（上/下）**（胡萍）：图文并茂，内容涵盖身体常识、预防性侵犯、青春期变化、预防艾滋病等，适合年龄稍大的儿童。
5.  **《东方儿童性教育绘本》（全3册）**：包括《我的弟弟出生了》、《我是女孩我弟弟是男孩》、《我宝贵的身体》。
6.  **《和妈妈一起看的性教育绘本》（全3册）**：包括《我的故事（男孩篇）》、《我的故事（女孩篇）》和《两个人的故事》，涵盖生命起源、身体差异、婚姻与爱。

**接下来是推荐给父母阅读的指导书籍：**
*   **《善解童真：0-6岁孩子的性发展与性关怀》（胡萍）**：这是一本基于大量案例的中国父母性教育宝典，帮助家长理解孩子各阶段的性发展，并学会正确引导。书中强调，真正的“性纯洁”是拥有健康、科学的性观念，能保护自己不受伤害。

**最后是适合亲子共赏的动画资源：**
*   **《“丁丁豆豆”成长记》**：由广东省绿芽乡村妇女发展基金会出品，共6集动画片，内容涵盖出生教育、身体认知、自我保护、青春期教育等，从多角度解析性问题。
