# 机器学习入门：13：核心概念答疑

在本节课中，我们将解答一些机器学习初学者常见的核心概念问题。通过澄清这些基础概念，有助于你建立更清晰的知识框架。

上一节我们介绍了模型评估的方法，本节中我们来看看一些容易混淆的基本概念。

以下是关于“特征”与“标签”的常见疑问。

*   **特征**：是输入模型的变量或属性，用于进行预测。在代码中通常表示为变量 `X`。
*   **标签**：是我们希望模型预测的输出结果。在代码中通常表示为变量 `y`。

理解了数据的基本构成后，另一个关键问题是模型如何从数据中学习。

以下是关于“训练”与“预测”过程的解释。

*   **训练**：模型通过分析带有标签的**训练数据集**，学习特征与标签之间关系的过程。核心目标是找到最优的模型参数。
*   **预测**：将新的、无标签的特征数据输入已训练好的模型，由模型输出预测标签的过程。

模型学习的目标是尽可能做出准确预测，这就引出了如何衡量模型表现的问题。

以下是“损失函数”的核心作用。

*   **损失函数**：用于量化模型预测值与真实标签之间的差异。模型训练的本质就是通过优化算法（如梯度下降）**最小化损失函数**。均方误差（MSE）是一个常见例子，其公式为：
    **MSE = (1/n) * Σ(y_i - ŷ_i)^2**
    其中 `y_i` 是真实值，`ŷ_i` 是预测值，`n` 是样本数量。

区分了训练和评估的差异后，我们需要了解模型面对新数据时的表现。

以下是“过拟合”与“欠拟合”的概念对比。

*   **过拟合**：模型在训练数据上表现极好，但在未知数据（测试集）上表现很差。这好比死记硬背了习题答案，却不会解新题。
*   **欠拟合**：模型在训练数据和测试数据上的表现都不佳，说明模型未能捕捉到数据中的基本规律。

本节课中我们一起学习了机器学习中几个最核心的概念：特征与标签的定义、训练与预测的区别、损失函数的作用以及过拟合与欠拟合的现象。清晰理解这些概念是后续学习更复杂模型和算法的重要基础。
