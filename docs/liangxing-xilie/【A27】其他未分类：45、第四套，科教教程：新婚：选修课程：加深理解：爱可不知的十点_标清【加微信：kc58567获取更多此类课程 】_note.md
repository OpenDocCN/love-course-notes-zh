# 人体生理学趣谈：A27：关于性高潮你可能不知道的十件事

在本节课中，我们将通过一系列有趣的科学研究和历史轶事，探讨关于人类性高潮一些鲜为人知的生理现象和理论。内容涵盖从胎儿行为到动物研究等多个领域。

## 1. 子宫内的行为观察

上一节我们介绍了课程概述，本节中我们来看看一项关于胎儿行为的独特研究。

《超声医学杂志》曾发表过一篇非常引人入胜的论文，其标题是《对子宫内手淫现象的观察》。

*   论文描述：在超声图像中，可以观察到胎儿的手（图像左侧的大箭头所示）以及阴茎（图像右侧）。放射科医生伊斯雷尔·迈斯纳描述道，图像显示手以类似于手淫动作的方式握住了阴茎。由于这是超声影像，所以记录的是动态画面。

## 2. 高潮的反射本质与非常规触发

了解了胎儿期的可能行为后，我们进一步探讨高潮的生理机制。高潮是自主神经系统的一种反射。这部分神经系统负责控制我们无意识的行为，如消化、心率和性唤起。

高潮反射可以由异常广泛的刺激触发。

以下是几个非常规触发案例：

*   常规刺激：生殖器刺激。
*   特殊案例：金赛曾采访过一位女性，她可以通过被抚摸眉毛而达到高潮。
*   脊髓损伤患者：截瘫或四肢瘫痪者常在损伤部位上方发展出一块异常敏感的区域。
*   文献记载：存在“膝盖高潮”的案例。
*   最奇特的案例：一份病例报告记录了一位女性每次刷牙时都会经历高潮。复杂的刷牙感觉运动动作触发了她的高潮。神经学家检查后排除了牙膏或牙龈刺激的原因，确定是整个刷牙动作所致。然而，该女性认为自己被恶魔附身，转而使用漱口水进行口腔护理。

## 3. 自主引发高潮与神经科学

从外部触发转向内部控制，我们来看看大脑的能力。我曾采访过一位能够通过意念使自己达到高潮的女性。她是罗格斯大学一项研究的参与者。在餐厅外，她用了大约一分钟进行了演示。她表示自己上次这样做是在迪士尼乐园的缆车上。

高潮的脊髓神经中枢位于骶神经根。用电极精确刺激该点可以引发高潮。事实上，在某些死者（即脑死亡但依靠呼吸机维持器官供氧的捐献者）身上也能触发脊髓反射，例如“拉撒路反射”（一种肢体抽动）。专家斯蒂芬妮·曼认为，如果骶神经仍有氧供，理论上也可能在死者身上引发高潮反射，但这对其本人已无意义。我曾向一位研究者建议进行此类实验，但她表示难以获得伦理审查委员会的批准。

## 4. 历史中的气味理论与打嗝疗法

现在，让我们回顾一些历史上的奇特观点和偶然发现。根据1930年代的婚姻指南作者西奥多·范·德·维尔德的说法，在性交后约一小时内，可以从女性的呼吸中检测到轻微的精液气味。他自称能区分年轻男子（气味清新提神）和成熟男子（气味类似西班牙栗子花，时而清新，时而浓烈）的精液气味。

1999年，以色列一名男子持续打嗝，尝试各种方法无效。在一次与妻子性交后，打嗝停止了。他的医生将此作为病例报告发表在《加拿大医学杂志》上，标题为《性交是治疗顽固性打嗝的潜在方法》。文章甚至建议单身打嗝者可以尝试手淫。

## 5. “上吸”理论与受孕

历史上，高潮曾被认为与受孕密切相关。20世纪初，许多妇科医生相信女性高潮时的收缩会吸吮精液通过宫颈，快速将其送达卵子，从而提高受孕几率。这被称为“上吸理论”。追溯到希波克拉底时代，医生认为女性高潮不仅有助于而且对受孕是必要的。当时的医生常告诫男性取悦妻子的重要性。婚姻指南作者范·德·维尔德甚至引用了一个典故：据说哈布斯堡王朝的特蕾莎女皇怀孕困难时，宫廷医师建议“在性交前一段时间刺激陛下神圣的外阴”。

然而，到了20世纪50年代，马斯特斯和约翰逊对此理论表示怀疑。他们设计了实验：让女性佩戴装有“人工精液”（含放射造影剂）的宫颈帽，并在X光机前自慰，观察精液是否被“上吸”。结果未发现证据。你可能会问如何制作人工精液？文献中有多种配方，例如面粉和水或玉米淀粉和水。其中一份配方在列出成分后，标注的产量是“一次射精量”。

## 6. 男性角度与动物界的证据

高潮提升生育力可能还有另一种涉及男性的方式。在体内留存一周以上的精子会开始出现异常，降低其受精能力。英国性学家罗伊·列文推测，这或许解释了为何男性在进化中形成了频繁手淫的倾向——以持续产生新鲜精子。这提供了一个有趣的进化论借口。

同时，在动物界存在相当多的“上吸”证据。例如在丹麦，国家养猪委员会发现，在对母猪进行人工授精时加以性刺激，其产仔率会提高6%。因此他们制定了一套针对母猪的五点刺激方案，并制作了教学DVD。方案模仿公猪用鼻子进行的求偶动作，包括按压母猪背部、抓挠腹侧、骑跨施加重量、按摩阴部区域等。害羞但有抱负的养猪户甚至可以购买一种安装在输精管上的“母猪振动器”，因为母猪的阴蒂在阴道内部。研究人员指出，不能从母猪面部判断其是否愉悦，因为猪像狗一样，更多用耳朵等上半部脸表达情绪。

## 7. 灵长类的表情与实验设备

与猪不同，灵长类动物更多用嘴部表达。这是截尾猕猴的射精表情。有趣的是，在雌性猕猴身上也观察到类似表情，但仅出现在其骑跨另一只雌性时。

马斯特斯和约翰逊在20世纪50年代决心研究整个人类性反应周期。对于女性，许多过程发生在体内。但这并未阻止他们，他们开发了一台“人工性交机器”。本质上，这是一个带摄像机的电机驱动假阳具——一个透明的丙烯酸假体，装有摄像头和光源，连接在一个往复运动的电机上。女性与之进行性交以供观察。遗憾的是，该设备已被拆除。

## 8. 精液射程实验

最后，我们来看一个关于测量本身的奇特实验。阿尔弗雷德·金赛曾决定计算精液射出的平均距离。这并非出于无聊的好奇心。当时（20世纪40年代）有一种理论认为，精液撞击宫颈的力度是影响生育力的一个因素。金赛认为这是无稽之谈，于是开始工作。他在实验室召集了300名男子，使用卷尺和电影摄像机进行测量。结果发现，四分之三的男性精液只是缓缓流出，并非强力喷射。然而，纪录保持者的射程接近8英尺（约2.4米）。金赛在实验报告中写道：“铺了两张床单以保护东方地毯。”这在他所有著作中是我第二喜欢的句子，最喜欢的是：“在一对交配的老鼠面前撒上奶酪屑，会分散雌性的注意力，但不会分散雄性的注意力吗？”

本节课中我们一起学习了关于性高潮的多种生理现象、历史理论及趣味研究，从胎儿行为、非常规触发机制、神经控制原理，到历史上的“上吸理论”、精液气味说，乃至针对动物和人类的特殊实验。这些内容揭示了人类性反应复杂多面而又时常出人意料的特点。
