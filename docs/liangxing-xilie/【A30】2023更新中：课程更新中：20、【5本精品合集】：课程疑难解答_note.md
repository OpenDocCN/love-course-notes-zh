# Python编程入门：1：Python基础语法

在本节课中，我们将要学习Python编程语言的基础语法。内容包括变量、数据类型、运算符以及基本的输入输出操作。这些是编写任何Python程序的基础。

## 变量与数据类型

上一节我们介绍了课程概述，本节中我们来看看Python中如何存储和处理数据。

变量是用于存储数据的容器。在Python中，你可以直接为变量赋值，无需预先声明类型。

Python支持多种数据类型，主要包括：
*   **整数**：如 `10`, `-5`
*   **浮点数**：如 `3.14`, `2.5`
*   **字符串**：用单引号或双引号包围的文本，如 `‘你好’`, `“Python”`
*   **布尔值**：表示真或假，即 `True` 或 `False`

以下是给变量赋值和查看数据类型的示例代码：
```python
name = “小明”  # 字符串变量
age = 20       # 整数变量
height = 1.75  # 浮点数变量
is_student = True  # 布尔值变量

print(type(name))  # 输出：<class ‘str’>
print(type(age))   # 输出：<class ‘int’>
```

## 基本运算符

了解了如何存储数据后，我们来看看如何操作这些数据。Python提供了丰富的运算符。

运算符用于对变量和值执行操作。以下是几种常用的运算符类型：

*   **算术运算符**：用于数学计算。
    *   `+` 加法
    *   `-` 减法
    *   `*` 乘法
    *   `/` 除法
    *   `%` 取余
*   **比较运算符**：用于比较两个值，返回布尔值。
    *   `==` 等于
    *   `!=` 不等于
    *   `>` 大于
    *   `<` 小于
*   **赋值运算符**：用于为变量赋值。
    *   `=` 基本赋值
    *   `+=` 加后赋值，例如 `a += 1` 等价于 `a = a + 1`

以下是使用运算符的公式和代码示例：
```python
# 算术运算
result = 10 + 3 * 2  # 结果为16，遵循数学运算优先级
print(10 % 3)  # 输出余数1

# 比较运算
print(5 > 3)  # 输出：True
print(5 == 3) # 输出：False
```

## 输入与输出

程序需要与用户交互。本节介绍如何从用户那里获取输入，以及如何将结果输出到屏幕。

输入输出是程序与外界沟通的桥梁。Python使用 `input()` 函数获取用户输入，使用 `print()` 函数输出信息。

以下是实现基本输入输出的步骤：
1.  使用 `input()` 函数获取用户输入的字符串。可以提供一个提示信息。
2.  使用 `print()` 函数输出一个或多个变量的值。

以下是输入输出功能的代码示例：
```python
# 获取用户输入
user_name = input(“请输入你的名字：”)
user_age = input(“请输入你的年龄：”)

# 输出信息
print(“你好，”, user_name)
print(“你的年龄是：”, user_age)
```
**注意**：`input()` 函数获取的所有内容都是字符串类型。如果需要作为数字计算，需使用 `int()` 或 `float()` 函数进行转换。

本节课中我们一起学习了Python的基础语法。我们掌握了如何使用变量存储不同类型的数据，如何运用运算符进行运算和比较，以及如何通过 `input()` 和 `print()` 函数实现基本的程序交互。这些是构建更复杂程序的基石。
