# 李氏养生功法：7-2：户外功法十二式

在本节课中，我们将学习李氏养生功法中适合在户外练习的“户外功法十二式”中的两个核心动作：“鸣天鼓”与“摩丹田”。这些动作简单易行，旨在帮助练习者在不同环境下都能有效锻炼身体。

## 鸣天鼓 🥁

上一节我们介绍了基础的热身动作，本节中我们来看看第一个招式“鸣天鼓”。这个动作主要通过敲击手指来刺激穴位，达到醒脑安神的效果。

以下是“鸣天鼓”的具体步骤：

1.  用左手中指敲打右手中指，共敲打 **20** 次。
2.  用右手中指敲打左手中指，同样敲打 **20** 次。
3.  在整个敲打过程中，需用左右手掌心捂住耳朵。

## 摩丹田 🌊

完成了头部的“鸣天鼓”练习后，接下来我们将注意力转移到腹部，学习“摩丹田”的练习方法。这个动作有助于温养丹田，但根据练习环境的不同，操作方式有所区别。

**核心概念**：按摩下腹丹田区域。

*   **室内练习**：一只手直接在上腹部进行按摩。
*   **户外练习**：在户外不适合直接按摩，需采用以下方式：
    1.  双手快速搓热后，迅速捂住丹田区域。
    2.  此动作重复进行 **50** 次。

关于户外练习“摩丹田”的补充说明：

*   在天气暖和时，可以将衣服撩起，让手直接贴住皮肤进行搓捂。
*   在冬天寒冷时，应将外套撩起，用手贴住内衣进行搓捂，以避免寒气侵入身体造成伤害。

本节课中我们一起学习了“户外功法十二式”中的“鸣天鼓”与“摩丹田”。这两个动作简便有效，尤其适合喜欢在户外锻炼的朋友，便于大家随时随地进行李氏户外养生功的修炼。
