# 武当山济肾堂功法：A27-2：清肺内功练习准备 🧘

在本节课中，我们将要学习练习清肺内功前必须完成的准备工作。充分的准备是功法练习安全、有效的基础。

上一节我们介绍了清肺内功的概况，本节中我们来看看开始具体练习前需要做哪些准备。

## 环境准备 🌳

练习清肺内功的第一个准备是选择一个空气质量良好的环境。因为清肺内功的呼吸方式比较独特，与平常的呼吸方式完全不同。公式表示为：**练习环境 = 空气质量优**。

如果所处环境空气质量很差，必须避开该地点，寻找一个空气污染较小的环境才能练习此功法。

## 器具准备 🎗️

第二个准备是最好准备一条腰带。这条腰带应是布质的，不要使用皮带。因为在使用过程中发力时，皮带容易崩断。

以下是腰带的规格要求：
*   **长度**：约2.5至3米。
*   **宽度**：约10公分。
*   **用法**：可以围绕腰部系两圈。

系腰带时，需将腹部尽量往回收，并稍微勒紧，然后系好即可。

## 辅助物品准备 🧻

第三个准备是准备好纸巾。准备纸巾的原因是，在后续练习中会用到鼻腔喷气的技巧。

喷气时，非常容易将鼻涕喷出。因此，需要准备好纸巾进行擦拭。

---

本节课中我们一起学习了练习清肺内功的三项核心准备工作：**空气质量良好的环境**、**符合规格的布质腰带**以及**用于清洁的纸巾**。做好这些准备，将为后续的实际练习打下坚实基础。
