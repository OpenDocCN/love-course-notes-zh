# 捆绑教程：A24：双手捆绑1

在本节课中，我们将学习一种极其简单的捆绑方法，用于限制手腕和双手的活动。这种捆绑被称为“爪缚”，其功能在于限制被缚者的小臂活动，同时允许其肩部自由活动。这种捆绑方式会给被缚者带来极大的挫败感，也可用于限制其推离地面或阻止他人靠近的能力。

## 🎼 核心概念与准备

上一节我们介绍了“爪缚”的基本用途，本节中我们来看看实施捆绑前需要了解的核心概念和准备工作。

“爪缚”的核心在于利用绳索在手腕处形成约束环，其基本原理可类比为一个简单的绳结循环。关键是要确保约束牢固但不过紧，以免影响血液循环。

以下是实施捆绑前需要准备的物品清单：
*   一根长度适中的绳索（建议约3-5米）。
*   一把安全剪刀，置于随手可及之处，以备紧急情况使用。
*   一个平坦、安全的环境。

## 🤲 捆绑步骤详解

了解了核心概念和准备工作后，现在我们来一步步学习“爪缚”的具体操作流程。

1.  **对折绳索**：首先，将绳索对折，找到其中点。
2.  **放置手腕**：请被缚者将双手手腕并拢。将绳索的中点置于其双腕下方。
3.  **交叉缠绕**：将绳索的两端分别向上拉起，在手腕上方交叉，然后绕回手腕下方。此过程重复2-3次，形成数道横向束缚。
4.  **纵向固定**：完成横向缠绕后，将剩余绳端在横向缠绕的绳索上下穿行，进行纵向固定，以加固整体结构。
5.  **收尾打结**：最后，打一个牢固但易于解开的结（如平结）来完成捆绑。务必检查松紧度，确保能插入一根手指。

## ⚠️ 安全注意事项与检查

完成了捆绑步骤，安全始终是首要考虑。本节将重点强调捆绑后必须进行的安全检查。

每次捆绑后，都必须进行以下检查，确保被缚者的安全与舒适：
*   **血液循环**：检查被缚者手指颜色是否红润，询问其是否有麻木或刺痛感。
*   **神经安全**：询问被缚者手指是否能自如活动，感知是否正常。
*   **绳结安全**：确认绳结牢固，不会意外松脱或越勒越紧。
*   **持续沟通**：在整个过程中与被缚者保持沟通，尊重其感受和界限。
