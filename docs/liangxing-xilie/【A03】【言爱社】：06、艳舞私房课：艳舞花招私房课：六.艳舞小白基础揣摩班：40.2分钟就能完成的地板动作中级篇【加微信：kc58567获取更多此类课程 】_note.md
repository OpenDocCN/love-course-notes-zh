# 艳舞私房课：06：2分钟完成的地板动作中级篇 🩰

在本节课中，我们将学习一个更具趣味性的地板动作。这个动作在光滑地面上更容易完成，在地毯上则会稍有挑战。我们将跟随Julia的示范，分解动作要领。

## 动作要领概述

这个动作的核心在于利用手臂支撑身体重量，同时用双脚在地面画圆。以下是完成该动作的具体步骤。

以下是完成该动作需要遵循的步骤列表：

1.  将身体重量转移到手臂上。
2.  保持双腿伸直。
3.  用双脚在地面上画出一个圆形轨迹。
