## 枯燥性爱的危害

增加性爱新鲜感不仅是为了乐趣，更是为了维护情感与婚姻关系的稳定。枯燥的性爱可能对关系造成深远伤害。

为了说明这一点，我将分享一个真实案例。

### 一个案例的启示

我曾有两位朋友，感情原本很好，结婚三年多。但在即将进入第四年时，女方出轨了。男方作为我的朋友，向我诉苦，不解为何会走到离婚地步。他告诉我，起初也不信传言，但开始观察妻子后，发现她开始注重打扮，周末总以约了同事为由拒绝他的邀约。最让他确信的是，妻子连每月不多的几次性爱也拒绝配合，总以累、无聊或身体不适推脱。最终，他看到了妻子与他人的私密聊天记录，其中提到彼此已无感情、生活无聊、性爱缺乏激情，只是为了孩子在一起。后来他们离婚了。

很久之后，我遇见那位女性朋友，她已恢复单身。我问她当初为何出轨。她直言，与丈夫的性爱总是重复相同的动作和姿势，感觉永远在敷衍，让生活索然无味。她表示并非想破坏家庭，只是追求生活品质、满足性需求，但被发现后不得已离婚。

虽然不清楚他们之间是否还有其他问题，但“性生活不和谐”成为一个重要理由。听众可能觉得这位女性不守妇道，但请换位思考，这个理由是否耳熟？它也是某些男性出轨的常见借口，例如觉得与妻子做爱没意思，所以外出寻求刺激。

这个问题本身不分男女。有时，并非情感先出问题导致性爱变质，而是性爱本身变得枯燥乏味，进而蔓延至生活，导致感情变淡。当身体最亲密的接触都变得无趣，日常生活的亲密感又从何谈起？

### 反思与行动

大家可以回想，恋爱或结婚多年后，你们性爱的频率和质量是否有变化？例如次数减少、激情消退。世上没有一劳永逸就能白头偕老的爱情，也没有源源不断天赐的欲望。热恋期的新鲜感只是荷尔蒙的见面礼，用完即止。过了那个阶段，夫妻更有义务主动创造激情。

那么，我们该如何主动创造呢？首先，可以在性爱时变换更多体位。这引出了我们今天课程的第二部分。
