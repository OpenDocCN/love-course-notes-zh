# 性治疗案例解析：A18：早泄的心理干预与行为治疗

在本节课中，我们将通过一个具体的临床案例，深入探讨早泄问题的心理根源与行为治疗方法。我们将跟随性治疗师童嵩珍的诊疗过程，理解如何结合心理疏导与行为训练来帮助患者。

## 案例背景介绍

上一节我们介绍了早泄问题的普遍性，本节中我们来看看一个来自科技行业的男性患者的真实经历。

这名患者从事科技行业，生活中缺乏稳定的异性交往关系。深入沟通后发现，他与女性仅通电话就会引发焦虑并导致早泄。因此，他从不穿浅色裤子，以避免尴尬。

## 寻求治疗的契机

童嵩珍医师将性医学与心理学结合，撰写了三四十篇相关文章进行网络推广。正是通过这些文章，这位科技行业的患者找到了她并前来求助。

## 患者的创伤经历

以下是患者分享的一段对其造成严重心理伤害的经历。

他与朋友一同光顾一家按摩店。他的朋友服务尚未结束，他却已提前“缴械”。当他出来后，服务他的小姐紧随其后，并向排队等候的其他人拍手宣称：“他已经结束了，叫他练快一点。”这令他感到极度羞辱。

患者表示，这是他最不愿告知他人的隐痛，但他的朋友却因此知晓。此事之后，他提前离开，并且再也不敢与那位朋友联系，自觉颜面尽失。

## 治疗过程与方法

基于上述经历，治疗师为他制定了系统的治疗方案。

我们逐步对他进行治疗，主要采用**减敏感训练**并结合**心理疏导**。童嵩珍医师提到，起初这位患者对她并不完全信任，是在多次沟通后才开始接受治疗。

## 治疗方案的核心：减敏感训练

减敏感训练是行为治疗的核心，旨在降低患者对性刺激的过度敏感反应。其基本逻辑可简化为一个训练公式：

**刺激暴露 + 放松练习 → 敏感度阈值提高**

主要训练步骤如下：
1.  在放松状态下，逐步接触不同等级的性刺激。
2.  当感到兴奋度上升至临近失控时，立即暂停刺激。
3.  待兴奋感平复后，再次开始刺激。
4.  循环此过程，逐渐延长刺激时间。

## 治疗结果与启示

经过三个多月的系统治疗，患者的情况显著好转。如今，他已经结婚并育有子女。

童嵩珍医师总结认为，多数的性功能障碍患者都伴随有相应的心理问题。单纯的行为训练必须与心理层面的疏导和支持相结合，才能取得稳固的疗效。

本节课中我们一起学习了早泄治疗中“身心结合”的重要性。通过这个案例，我们看到，早泄不仅是生理反应过快的问题，更常与焦虑、创伤经历等心理因素紧密相连。成功的治疗需要像剥洋葱一样，层层解开心理症结，同时通过科学的行为训练重建健康的性反应模式。
