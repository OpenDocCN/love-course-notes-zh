# 玉蛋功：A15：分段跟随练习一丨热身抖动、脊椎呼吸 🥚

在本节课中，我们将学习玉蛋功的热身部分，包含两个核心动作：热身抖动与脊椎呼吸。这些练习旨在激活身体能量，为后续训练做好准备。

## 热身抖动 🎵

上一节我们介绍了课程目标，本节中我们来看看第一个热身动作：抖动。这个动作的目的是通过有节奏的抖动，激活全身能量。

以下是动作分解步骤：

*   双腿与肩同宽站立，双膝自然弯曲。
*   跟随音乐节奏抖动全身。
*   抖动到腹部。
*   抖动双手。
*   抖动双脚。
*   向左侧扭身抖动。
*   向右侧扭身抖动。
*   双手上举抖动。

在抖动过程中，感受能量从心脏扩散到身体末梢。手指、脚趾乃至发丝都充满能量，身体变得柔软且有活力。

动作完成后，恢复直立站姿。

## 脊椎呼吸 🫁

完成了全身能量的激活后，我们进入脊椎呼吸练习。这个动作通过呼吸与躯干的伸展收缩，来灵活脊柱并深化呼吸。

以下是动作分解步骤：

*   双腿略宽于肩站立，双膝自然弯曲。
*   用鼻子吸气，同时张开双臂从两侧上举。
*   双肩尽量打开，胸部前挺，双眼看向上方。
*   张开嘴，尽力从腹部呼气。
*   呼气时发出“哈”的声音，同时握拳收回双肘，合在胸前。
*   背部尽量向上拱起，低头将脸埋在胸前，使整个上身缩成一团。

整个过程的节奏是：**吸气时伸展，呼气时收回**。此动作需重复进行9次。

动作完成后，双手自然分开下垂。

---

本节课中我们一起学习了玉蛋功的热身部分。我们首先通过**热身抖动**激活了全身能量，使身体变得柔软有活力；随后通过**脊椎呼吸**练习，掌握了吸气伸展、呼气收缩的节奏，灵活了脊柱。这两个动作为后续练习奠定了良好的身体与呼吸基础。
