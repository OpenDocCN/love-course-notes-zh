# 武当山济肾堂强肾清肺功法：2：清肺内功第三式详解 🫁

在本节课中，我们将要学习清肺内功的第三式。这一式主要通过特定的呼吸与动作配合，来强化肺部功能，促进气息运行。我们将分解动作要领、呼吸方法及练习中的注意事项。

## 动作与呼吸分解

上一节我们介绍了基础站姿，本节中我们来看看第三式的完整流程。核心在于呼吸与发力的三步递进。

**动作起始**：双脚保持基础站姿。双手交叉，男性左手在内，女性右手在内。交叉后，在胸前提前打开。打开时，身体下蹲，双手随之下落至膝盖上方。需注意身体不可前倾，保持立掌姿势，此为吸气阶段。

**运气与发力**：吸气后，双掌闭气绷紧，开始运气。想象双手托举重物，将其向上拖起。双手向中间合拢，至胸前时，手腕翻转，如同转动一个连环。随后，将想象的重物向前推出。

以下是动作的三个关键细节：
1.  **推出时**：身体随动作前倾，但腿部并非完全伸直。在最终推出的瞬间，腿部猛然伸直。
2.  **手臂姿态**：手臂向前推出时，不可完全伸直，应保持微微弯曲，这与出拳原理相同。推出后，掌心朝外。
3.  **回收动作**：推出后，合掌，将双手收回并向下压回。

## 呼吸进阶三步法 🌀

本式的呼吸分为三步，强度逐步递进。

以下是具体的呼吸节奏：
1.  **第一次**：吸入约三分之二的气，然后一次性全力呼出。
2.  **第二次**：吸入全部的气，分三次缓慢呼出。
3.  **第三次**：一次性吸满气，一次性全力呼出。

## 核心要领与发力技巧 💪

练习此式时，需掌握全身协调发力的方法。

**发力公式**：初学时使用 `70%` 的力。熟练后，可逐渐增加至 `100%` 甚至 `120%` 的力。

**身体协调代码**：
```plaintext
吸气绷紧 -> 脚趾抓地 + 膝盖内扣 + 提肛收肾 -> 全力推出
```

以下是练习时必须同步进行的身体要点：
1.  **根基稳固**：脚趾用力抓地。
2.  **下肢绷紧**：膝盖有意识地向内扣紧。
3.  **核心收紧**：提肛、收缩肾区，腹部绷紧，气沉丹田。
4.  **全身协同**：在最终推出的瞬间，全身用力，喊出声音以助发力，同时腿部蹬直。

## 常见问题与纠正 ⚠️

练习此式容易导致头晕，若发生，需立即用意念引导气息下行，即可缓解。

以下是教练在指导中强调的常见错误与纠正方法：
1.  **手臂过直**：推出时手臂完全伸直。**纠正**：手臂应保持微曲。
2.  **下盘松散**：膝盖未内扣，站成外八字。**纠正**：始终保持膝盖内扣意识。
3.  **发力松懈**：在运气和推出过程中身体未绷紧。**纠正**：从吸气开始到动作完成，全身需持续绷紧。
4.  **回收随意**：合掌收回时，手臂弯曲或路线错误。**纠正**：收回时手臂应伸直，从身体侧面直线下落。
