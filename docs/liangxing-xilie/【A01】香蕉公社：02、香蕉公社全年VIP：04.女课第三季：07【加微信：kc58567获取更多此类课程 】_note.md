## 什么是“奇怪”的性要求？

很多女性将“奇怪”的性要求等同于“变态”的性要求。这个逻辑是：因为这件事我没见过、没听过、没感受过，所以一般人不会这么做；你要我这么做，那你就是变态。这个逻辑实际上是有待商榷的。

所谓“奇怪”，就是超出你认知或想象范围的东西。但超出你认知范围的东西，不一定不合理，也不一定就是变态，可能只是你认知和想象力不足。

举一个生活中的例子：一到冬天，北方人买白菜一买就是一吨或好几吨，而南方人无论何时都只买一顿。北方人会觉得南方人很奇怪，南方人也会觉得北方人很奇怪。但如果让一个在南方和北方都生活过的人来看，这两种行为都很正常。因为在北方，冬天寒冷，白菜会迅速减产，如果不一次性多囤积，可能整个冬天都买不到白菜，而且寒冷天气利于储存。在南方，天气不算太冷，白菜供应如常且不易保存，所以每天买一顿的量即可。无论是南方人还是北方人，都在自己的处境下做出了最有利的决策，这一点都不奇怪。

同样的道理，性也是这样的。当你觉得某件事很奇怪时，首先要做的不是排斥和拒绝，而是用心去考虑：你了解过这个性行为吗？它会带来哪些好处或坏处？这才是一个理性的人对于“奇怪”应有的态度。

从我的咨询经历来看，绝大部分中国女性认为“奇怪”或“变态”的行为，认真考究下来其实是正常的，且常常有利于双方的性收益。中国女性之所以认为很多性行为奇怪，主要是因为见得少、了解少，想象力和认知范围狭窄，思想中有很多禁锢。直到今天，还有很多人认为口交是变态的、自慰不正常、用手指插入阴道不可以、性交就应该在床上。所以，在上一季女课分享技巧前，我都会先做思想上的铺垫，因为必须先从认知上接受，才能实践。

**所以，我们不要用“习不习惯”来评价一个性行为，而要用“合不合理”。**

## 如何判断性要求是否“合理”？

你们可能会问：我怎么知道一个性要求合不合理？有没有标准？下面我提供一些可供参考的标准：**互惠、安全、无害**。

首先，这个行为应该对双方都有利，双方都能在这项性行为中获得性收益，无论是生理上还是心理上。

同时，这个行为必须是安全的，危险系数完全可以掌控。举个例子，性行为中有一种玩法叫“性窒息”，即让对方掐住自己脖子，或用密封塑料袋包住头来进行性爱。部分人在大脑缺氧时反而更容易高潮，获得更强烈的生理快感。但这种性行为非常危险，操作不当可能危及生命安全。几乎每年都有人因性窒息而死亡。所以，这种行为不太推荐，即使要做也必须非常谨慎。

此外，这个性行为还不能产生伤害。伤害类型包括生理伤害和心理伤害，伤害对象包括性行为的当事人和其他人。举个例子，如果在做爱过程中想拍小短片，很多人喜欢这样做。你们拍完自娱自乐看一看，其实是正常且有情趣的事情。但如果一方将视频或图片发到网上，就可能对当事人乃至社会风气产生严重的负面影响，这时伤害就产生了。所以，评价是否伤害需要多角度考虑，把控风险。

### 4. 做爱时说粗口
