# 私密健康训练：014：勃起状态下的PC肌训练法 💪

在本节课中，我们将学习一种进阶的PC肌训练方法——在勃起状态下进行锻炼。这种方法旨在提升性爱过程中的耐力与控制力。如果你已经掌握了基础PC肌的定位与“10乘3”训练法，那么本节课将帮助你进入下一个训练级别。

上一节我们介绍了如何找到PC肌并进行基础收缩训练。本节中，我们来看看如何在更具挑战性的勃起状态下，有效地锻炼这块肌肉。

## 训练准备 🛠️

开始训练前，你需要准备一条拧干的小毛巾作为辅助工具。

## 训练步骤详解

以下是勃起状态下PC肌训练的具体操作方法。

### 步骤一：进入勃起状态
首先，你需要通过各种方式使阴茎自主勃起。可以一边听讲解一边进行。

### 步骤二：放置毛巾
勃起后，将准备好的小毛巾放在阴茎上。放置的位置大约在阴茎中段以上、靠近龟头的区域。

### 步骤三：执行收缩与举起动作
通过收缩PC肌（即之前课程教授的收缩法），尝试将毛巾微微举起。举起后保持2到3秒，然后放松肌肉使毛巾落下。此为一个完整动作。

### 步骤四：重复循环
重复“收缩-举起-保持-放松”这一过程，建议完成10次左右。

## 要点与调整建议 🔧

如果在尝试过程中遇到困难，可以根据以下建议进行调整。

*   **感觉毛巾过重或难以举起**：这可能是因为勃起硬度尚未完全达标。你可以尝试将毛巾拧得更干一些，或者换用一条更小、更干燥的毛巾。
*   **调整毛巾位置**：不必让毛巾太靠近龟头，那样可能需要更大的力量。可以将其稍微下移，放置在阴茎中段位置再进行尝试。
*   **选择训练姿势**：提供了两种主要姿势以适应不同情况。

以下是两种推荐的训练姿势。

1.  **站立姿势**：适合勃起时阴茎自然上翘的男性。双腿张开，略宽于肩，膝盖微曲。臀部微微向后坐，仿佛要靠向一把不存在的椅子。然后臀部向前倾，直到腰部与臀部大致平行。可以将一只手反撑在腰间作为辅助。在此姿势下进行上述的毛巾举起训练。
2.  **坐姿**：如果勃起时不易上翘，可采用坐姿。坐在椅子上，保持背部挺直。将毛巾置于阴茎上，然后进行PC肌的收缩与举起练习。坐姿通常需要PC肌发出更大的力量。

## 训练要求与进阶 🎯

请注意，仅尝试一次训练是不够的。你需要私下坚持练习，直到能够轻松地重复完成10到30次举起动作为止。达到这个标准后，你才能更顺利地进行下一阶段的训练。

本节课中，我们一起学习了在勃起状态下利用毛巾进行PC肌抗阻训练的方法，包括准备物品、详细步骤、姿势选择以及坚持练习的重要性。持之以恒是取得效果的关键。
