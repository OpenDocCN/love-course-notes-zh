# 艳舞私房课：06：百分百睡服他的臀部旋转诀窍

在本节课中，我们将要学习艳舞或膝上舞的核心动作——臀部旋转。这是所有研磨动作的基础，掌握它将为你的舞蹈增添无限魅力。

## 🎼 尝试臀部旋转

臀部旋转是艳舞或膝上舞的核心动作，也是所有研磨类动作的基础。其变化形式是无限的。

上一节我们介绍了臀部旋转的重要性，本节中我们来看看如何进行基础练习。以下是几种可以练习的姿势：

*   站立姿势。
*   蹲姿。
*   坐在舞伴腿上的姿势。
*   平躺在地板上并抬起臀部的姿势。
*   双手双膝着地的姿势。

练习得越多，动作就会越纯熟。

## 😊 让双手也忙碌起来

在练习臀部旋转的同时，别忘了让你的双手也参与进来。这能大大增加舞蹈的表现力和诱惑力。

以下是几种手部动作的建议：

*   让双手在身体上下滑动。
*   将双手高高举起。
*   将双手向身体两侧伸展。
*   与你的头发或衣物进行互动。
