# 马氏徒手壮阳：1：壮阳22式11 🧘‍♂️

在本节课中，我们将要学习一套徒手锻炼方法，旨在疏通身体经络，促进气血循环，从而提升身体活力与精神状态。我们将详细讲解动作的完整路径与循环。

## 动作路径解析

上一节我们介绍了动作的基础，本节中我们来看看完整的动作循环路径。该动作主要作用于头部与面部，并贯通身体前后的主要脉络。

动作从身体后方的长强穴开始。我们之前已经从身体前侧向下运行完毕。路径一直向下延伸到会阴穴。

## 任督二脉循环

以下是动作在身体后侧的上升路径，构成一个完整的任督二脉循环：

*   **起点**：从后方的长强穴开始。
*   **上行**：向上经过长强穴，抵达上髎穴。
*   **继续上行**：经过腰阳关穴、筋缩穴、神道穴。
*   **至顶端**：向上运行至大椎穴，再到哑门穴、风府穴。
*   **完成循环**：最后向上到头顶，再向前返回起点。这就完成了一个循环。

你需要将整个循环路径打通。刚才演示的只是其中一半，现在我们来完成另一半。做完另一半后，身体同样会感到非常轻松。
