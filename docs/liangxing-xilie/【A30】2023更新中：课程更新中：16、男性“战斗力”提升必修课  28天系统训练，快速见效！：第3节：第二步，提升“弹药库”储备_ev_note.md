# 男性战斗力提升必修课：第3节：第二步，提升“弹药库”储备 💪

在本节课中，我们将要学习影响男性“弹药库”（即射精量）的关键肌肉。这些肌肉的紧张或失衡会直接影响盆腔空间和盆底肌功能，从而减少储备。我们将逐一认识这些肌肉，学习如何评估它们的紧张度，并掌握简单的处理方法。

## 认识影响“弹药库”的肌肉 🏋️

上一节我们介绍了整体概念，本节中我们来看看具体是哪些肌肉在起作用。以下是六块直接影响盆腔和膀胱位置的肌肉，它们的紧张会导致骨盆位置不正，从而挤压空间、影响功能。

1.  **内收肌**
    *   **位置**：从耻骨联合连接到大腿内侧。
    *   **功能**：主要功能是内收大腿（将腿向内侧并拢）。
    *   **紧张的影响**：会向内牵拉骨盆，导致盆腔空间变小，膀胱容量受限。

2.  **腹直肌**
    *   **位置**：从第5-7肋骨连接至耻骨联合（即常说的“腹肌”）。
    *   **功能**：使躯干屈曲。
    *   **紧张的影响**：会将骨盆向后下方牵拉，导致骨盆后倾，改变膀胱承受的压力。

3.  **髂腰肌**
    *   **位置**：从腰椎和髂骨内侧连接至股骨小转子。
    *   **功能**：屈髋（抬起大腿）。
    *   **紧张的影响**：会导致骨盆前倾，对盆底肌产生压力，并使膀胱偏离正常位置。

4.  **腘绳肌**
    *   **位置**：从坐骨结节连接到小腿胫骨和腓骨。
    *   **功能**：屈膝和伸髋。
    *   **紧张的影响**：会将坐骨向下拉，导致骨盆后倾，同样影响膀胱位置。

5.  **臀大肌**
    *   **位置**：从髋骨后侧连接至股骨和大腿筋膜。
    *   **功能**：伸髋和外旋大腿。
    *   **紧张的影响**：若单侧紧张，会拉动骨盆旋转，导致盆腔内器官位置不正。

6.  **髋外旋肌群**
    *   **位置**：位于臀部深层，从骨盆侧面连接至股骨大转子。
    *   **功能**：主要使大腿外旋。
    *   **紧张的影响**：单侧紧张同样会导致骨盆旋转，影响脏器功能。

## 肌肉紧张度评估与处理方法 🩺

了解了这些肌肉后，接下来我们学习如何评估它们是否紧张，以及出现问题时如何进行处理。请按照以下步骤逐一进行测试和拉伸。

### 1. 内收肌评估与拉伸

**评估方法**：
侧卧，身体躺平。下方腿屈膝支撑，上方腿伸直，脚尖始终朝前（不可朝上）。将上方腿向上打开，过程中腰部不能下塌。打开角度达到**60度**为及格。

**处理方法**：
若角度不足，需进行拉伸。坐姿，双脚脚底相对并拢，膝盖向外打开。身体缓慢向前俯身，直至大腿内侧有拉伸感。保持30秒，重复3-5组。

### 2. 腘绳肌评估与拉伸

**评估方法**：
仰卧，身体躺平。一条腿伸直贴地，另一条腿伸直向上抬起，膝盖打直，脚尖绷直。正常男性抬起角度应达到**80度**（女性为90度）。

**处理方法**：
若角度不足，可仰卧，用弹力带或毛巾套住脚掌，缓慢将腿拉向身体，保持膝盖伸直。感受大腿后侧的拉伸。保持30秒，重复3-5组。

### 3. 腹直肌评估与拉伸

**评估方法**：
站立，双脚并拢，膝盖打直。身体缓慢向后仰。正常向后仰的角度应能达到**45度**。若无法达到，可能腹直肌或髂腰肌紧张。

**处理方法**：
使用“花生球”进行放松。俯卧，将花生球置于耻骨联合（下腹部正中骨性凸起）上方。双手缓慢撑起身体，使腹部有拉伸感。保持30秒，重复3-5组。

### 4. 髂腰肌评估与拉伸

**评估方法**：
俯卧，骨盆平贴地面。单腿伸直向上抬起，身体不可旋转或侧倾。抬起角度达到**30度**为及格。

**处理方法**：
采用弓步拉伸。若右侧紧张，则左腿在前呈弓步，右腿向后伸直。身体挺直，右手向上延伸。感受右侧腹股沟前方的拉伸感。保持30秒，重复3-5组。

### 5. 臀大肌评估与拉伸激活

**评估方法**：
臀大肌因久坐常被拉长无力，通常无需角度评估，直接进行拉伸和激活。

**处理方法**：
*   **拉伸**：仰卧，将右脚踝置于左膝上方，双手抱住左大腿拉向胸口。感受右侧臀部的拉伸。保持30秒。
*   **激活**：保持上述姿势，被拉伸的右腿膝盖用力下压，与双手的阻力对抗5-10秒，然后放松。此为一次，重复8-10次为一组，做3组。

### 6. 髋外旋肌群评估与拉伸

**评估方法**：
俯卧，双腿并拢，屈膝使小腿与地面垂直。然后让双膝自然向两侧分开。正常单侧小腿与身体中线夹角约为**45度**。

**处理方法**：
拉伸姿势与臀大肌类似但角度不同。仰卧，将右脚踝置于左膝上方，但右脚位置更靠近左膝。然后身体向左旋转，用左侧肩膀去靠近右膝。感受右臀深处的拉伸感。保持30秒，重复3-5组。
