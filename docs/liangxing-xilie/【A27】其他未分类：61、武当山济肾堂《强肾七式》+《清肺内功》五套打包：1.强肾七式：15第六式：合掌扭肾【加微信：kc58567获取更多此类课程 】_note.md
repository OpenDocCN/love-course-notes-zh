# 武当山济肾堂强肾七式：第六式：合掌扭肾 🧘‍♂️

在本节课中，我们将学习《强肾七式》的第六式——合掌扭肾。这一式的核心在于通过特定的手臂划圈轨迹，配合腰部的反向扭转，来温和地活动肾脏区域与脊柱。掌握其动作路线与发力要点是关键。

## 图解与原理 📐

上一节我们介绍了预备式，本节中我们来看看第六式的动作蓝图。理解下方的原理图是正确练习的第一步。

该式动作轨迹基于三个假设的圆形平面：
*   **上圆**：从眉毛到肩膀的平面。
*   **中圆**：从肩膀到腰部的平面。
*   **下圆**：从腰部到膝盖的平面。

以下是完整的动作路径描述：
1.  **起点**：从左肩开始。
2.  **上行弧线**：从左肩画弧线上行至眉毛。
3.  **下行弧线**：从眉毛画弧线下行至右肩。
4.  **继续下行**：从右肩画弧线下行至左腰。
5.  **降至低点**：从左腰画弧线下行至右膝。
6.  **开始上行**：从右膝画弧线上行至左腰。
7.  **返回高点**：从左腰画弧线上行至右肩。
8.  **循环**：动作在右肩稍作停顿后，可推回左肩（起点）开始下一次循环。

在整个路径中，有四个关键的身体定位点需要牢记：**眉**、**肩**、**腰**、**膝**。动作就是在这些点之间连接出平滑的弧线。

## 动作要点详解 🔑

理解了路线图后，我们来看看执行时的具体规则。图中标注的“转”和“正”是核心要点。

**关于“转”**
路径上标注的“转”字，表示此处手掌需要翻转。具体翻转时机如下：
*   起点（左肩）
*   从眉到肩
*   从肩到腰
*   从腰到膝
*   从腰到肩
*   **注意**：从膝上回到腰的这段弧线，**不需要**翻转。

**关于“正”**
图中标注的“正”字，表示动作运行到此时，身体（特别是腰胯）需要回正到中立位置，手掌也随之摆正。这是为了在扭动中建立节律和平衡。

## 分步动作示范 🌀

下面我们将配合图解，分解演示整个动作流程。

**第一步：起势**
1.  双脚并拢，身心放松安定。
2.  双手从体侧缓缓抬起，双掌合十。
3.  指尖朝前，掌心虚空。
4.  将合十的双掌上提至胸口，然后平推至左肩位置（起点）。

**第二步：第一个“转”**
在左肩起点处，进行第一次翻转：手掌由外向内翻转，直至掌根朝上。**全程保持指尖方向始终朝前**。

**第三步：运行轨迹**
现在，我们开始沿着图示轨迹运动：
1.  从起点（左肩）画弧线运行至**眉** → 此处“正”，身体回正。
2.  从**眉**画弧线运行至右**肩** → 此处“转”，手掌翻转；此处“正”，身体回正。
3.  从右**肩**画弧线运行至左**腰** → 此处“转”；此处“正”。（**注意**：从此处开始，配合屈膝下蹲）
4.  从左**腰**画弧线运行至右**膝** → 此处“转”；此处“正”。
5.  从右**膝**画弧线运行至左**腰** → 此处“正”。（**注意**：此段无“转”，同时开始直膝起身）
6.  从左**腰**画弧线运行至右**肩** → 此处“转”；此处“正”。（回到高点，膝盖应已伸直）

## 核心技巧与常见错误 ⚠️

掌握了基本动作后，以下几点能帮助你提升练习效果，避免错误。

**核心技巧**
1.  **手与腰的对抗**：本式名为“扭肾”，关键在于用手的运动引导腰部的反向自然扭动。例如，手向左画弧时，腰应向右微微扭转，形成一种温和的对抗力。公式可简化为：`手运动方向 ≠ 腰扭转方向`。
2.  **保持平面**：手臂划弧时，应始终想象在一个固定的玻璃平面上运动，不要向前或向后突出。
3.  **中正下蹲**：屈膝下蹲时，保持腰背挺直，不可弓背或翘臀。想象脊柱垂直于地面，直上直下。

**常见错误**
*   手指尖方向在翻转过程中改变。
*   手臂运动脱离假想的平面。
*   下蹲时弯腰驼背。
*   只用手臂摆动，而没有用腰胯发力带动。

## 收势动作 🙏

完成主要的划圈扭腰动作后，需按以下步骤收势：
1.  双手在终点（右肩）伸展分开，掌心相对，目光注视两掌之间。
2.  以腰为轴，带动手臂如钟摆般左右轻柔摆动数次。**要领是腰动带手，而非手动**。
3.  摆动停止后，双手回到中间，再次合十，拇指、中指、食指轻轻相触。
4.  将左脚向侧边打开，脚尖点地，脚跟抬起。
5.  合十的双手向上举起，伸直，同时仰头。
6.  缓缓呼气，放松。
7.  放松颈部、脊柱、肩部，双手自然下垂。
8.  屈膝下蹲，收回左脚。
9.  全身放松片刻后，可开步站立，调整呼吸。

## 总结 📝

本节课我们一起学习了《强肾七式》中的第六式“合掌扭肾”。我们首先解读了动作的轨迹原理图，明确了“眉、肩、腰、膝”四个关键点以及“转”和“正”的运用时机。随后，通过分步示范梳理了从起势到收势的完整流程。最后，重点强调了**以腰胯带动、手腰反向对抗**的核心发力方法，以及保持身体中正、动作在同一平面的要点。请记住，练习时宜慢宜松，重在感受身体内部的牵拉与扭动，而非追求外形幅度。
