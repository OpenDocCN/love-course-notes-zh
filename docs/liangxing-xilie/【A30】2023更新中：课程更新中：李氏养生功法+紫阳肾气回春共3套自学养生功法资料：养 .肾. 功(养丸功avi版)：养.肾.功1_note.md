# 李氏养生功法：第一课：养肾功基础与呼吸训练 🧘‍♂️

在本节课中，我们将要学习养肾壮阳养生功的基本原理、三种核心呼吸训练方法以及前三个基础招式。理解这些是后续所有练习的根基。

## 概述：功法核心原理

上一节我们介绍了课程主题，本节中我们来看看这套功法的核心理念。

为什么这套功法被称为“养肾壮阳养生功”并围绕“养丸”展开？理解这一点至关重要。明白了其中的道理和原理，大家在练功时就有了方向、动力和指路明灯。

生活中我们常说“肾虚”。中医所说的“肾虚”之“肾”，并非现代解剖学上的肾脏，而是指**外肾**，即**睾丸**。

**核心概念**：**睾丸是男人的命根子**。它控制着男性的性功能，负责生产**精子**和**雄性激素**。有了精液，身体才有了“子弹”和发射的欲望；有了**雄性激素**，才能成为真正的男人，对异性产生吸引力，并使阴茎勃起。睾丸与脑中枢形成反馈，分泌性激素。因此，养肾壮阳功的核心就是养护睾丸，通过各种功法完成这一重任。

## 基础训练：三种呼吸法

理解了原理后，我们进入最关键的实践部分。任何养生功的基础都是呼吸训练，它将贯穿我们整个功法修炼过程，并最终成为无需刻意控制的自觉性动作。

呼吸训练能让我们吐故纳新、益精回春、预防疾病、增强性能力，是一种古老的回春术。

以下是三种必须掌握的呼吸方法：

1.  **腹式呼吸**
    *   **要点**：吸气时，让小腹部**凸起**；呼气时，让小腹部**凹下去**。同时，舌尖始终顶住上齿龈后方。

2.  **胸式呼吸**
    *   **要点**：这是日常无意识的呼吸。吸气时，胸腔**胀大**；呼气时，胸腔**收缩**。此呼吸法较易掌握。

3.  **综合呼吸**
    *   **要点**：此法是腹式呼吸与胸式呼吸的综合，但并非简单结合。
    *   **流程**：
        *   **吸气**：腹部凸起到最大时，用力**提肛**、**缩阴**。
        *   **憋气**：此时憋住气，小腹微微回收，停留几秒，双脚掌紧紧抓地。
        *   **呼气**：不是缓缓呼出，而是用力将气从鼻腔**喷出**。同时，小腹部向外用力一突，整个阴部、肛门**放松**。

## 招式演练：前三式

掌握了呼吸法，我们就可以开始学习具体的招式了。以下是养肾壮阳养生功的前三个招式。

**第一式：手托天**

*   **动作要领**：
    *   双脚与肩同宽，双膝微曲。
    *   双手劳宫穴护住丹田穴，收腹挺胸，采用**腹式呼吸**。
    *   吸气时，双手掌心向上慢慢端起，感觉端着一团有分量的阳气，沿任脉上举至头顶，同时掌心翻向上，呈“双手托天”状，并抬起脚后跟，整个身体向上伸展。
    *   呼气时，两臂向两侧展开，掌心向下画一个大弧，同时身体下蹲。
    *   双手从地面搂起精华，同时吸气，缓缓端起身。
    *   手上端至胸前时，可缓一口气（熟练后可省略），然后继续吸气上举至头顶托天，再呼气开始下一个循环。
    *   **练习次数**：做9个完整的呼吸循环。

**第二式：通元气**

*   **动作要领**：
    *   接上式，两手从丹田处抬起，掌心向下，如按压皮球。
    *   双脚紧紧抓地，两眼瞪大，采取**综合呼吸法**。
    *   吸气时，用力**提肛缩阴**，向上提睾丸，维持几秒。
    *   然后随着喷气，整体向下撑。
    *   **核心锻炼**：提肛动作旨在锻炼性爱肌（PC肌，即耻尾联合肌）。
    *   **练习次数**：做13次呼吸（喷13次气）。

**第三式：提耳朵**

*   **动作要领**：
    *   传统医学认为“肾开窍于耳”，耳部布满诸多与肾脏及脏腑相关的穴位。刺激双耳可调理机能，强肾健脾。
    *   采用**腹式呼吸**。
    *   右手经头顶，用拇指和食指捏住左耳上耳廓，向上提拉滑动20次。
    *   然后换左手，同样方法提拉右耳20次。提拉到最后时要脱手。
    *   **后续课程**：在后续课程中，还会教授“拨耳”、“灸耳”、“弹耳功”。
