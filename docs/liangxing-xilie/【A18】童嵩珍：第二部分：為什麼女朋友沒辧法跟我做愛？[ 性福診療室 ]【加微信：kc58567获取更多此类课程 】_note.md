# 性福诊疗室：A18：第二部分：为什么女朋友没办法跟我做爱？🔍

在本节课中，我们将探讨女性性交障碍（原房障碍）的成因、医学定义及其对伴侣双方的影响。我们将了解其核心并非简单的生理疼痛，而是一种涉及心理与生理的复杂恐惧。

## 传统心理分析的视角

上一节我们介绍了课程主题，本节中我们来看看传统的解释。女性性交障碍在心理分析理论中，常归因于过往的负面性经验。

*   这些经验可能包括曾遭受性骚扰、性侵害或其他不良的性体验。
*   这些经历被认为会导致女性对性行为产生恐惧，从而无法顺利进行。

## 临床观察的新发现

然而，仅用心理创伤解释并不全面。在临床实践中，我们发现这类女性通常具备一些共同特质。

以下是两类常见的个性与生理特征：

1.  **个性控制感强**：她们从小个性上控制感很强，凡事都要求按照自己预设的方式处理。
2.  **身体痛阈较低**：在身体上，她们也比一般人更怕痛。

当这两种特质的指数越高时，出现性交障碍的情况就越多。

## 医学定义的演变与澄清

那么，这种障碍在医学上究竟如何定义？其命名经历了一个逐步精确化的过程。

最初，它被称为**阴道痉挛**。但许多男性会质疑：“我又没有插入，哪来的痉挛？”（痉挛即抽筋）。

后来，改称为**性交疼痛**。丈夫们可能又会说：“我还没有插入，哪来的痛？”因此，这个定义也只对了一半。

最终的准确答案应是**性交恐惧**。这意味着在插入行为发生之前，女性就已经感到害怕。针对这种恐惧，治疗方式需要从心理（性）与生理（生）两方面共同入手。

## 问题的核心与对双方的影响

性交障碍的核心问题在于，伴侣双方无法通过性来达成和谐的关系。这最终导致婚姻中缺乏性的协调，就像两片没有粘合剂的吐司，很容易分离。

如果朋友们遇到性交障碍、性交疼痛导致性生活匮乏的问题，可以尝试自己进行脱敏疗法处理。如果不行，应尽早求医。

因为性交障碍不只影响女性，也会严重影响男性。一旦性生活减少，男性可能出现两种后果：

*   **阳痿**：因为长期缺乏亲密互动，导致靠近伴侣时无法勃起。
*   **早泄**：由于太久没有性生活，造成阴茎敏感度过高，进而导致早泄。
