# 持久训练课程：A27：丁丁按摩训练1-推肾俞

在本节课中，我们将要学习一种结合了腰部按摩与盆底肌锻炼的训练方法，旨在通过特定的手法和节奏来促进相关区域的血液循环与肌肉控制。

## 🎼 训练步骤详解

上一节我们介绍了课程的基本目标，本节中我们来看看具体的操作步骤。以下是该训练的核心执行流程。

1.  **准备动作**：在手掌上涂抹几滴精油，将双手掌心互相搓热。
2.  **放置手掌**：手掌并排，将双手向下按在腰后部。
3.  **推按动作**：从腰部开始，向尾椎部方向，上下来回推按。
4.  **速度与节奏**：推按速度不宜过快。同时，需配合推按的频率，有节奏地快速收紧肛门，然后放松。
5.  **训练时长**：该完整步骤需持续进行2分钟。

## 总结

本节课中我们一起学习了“推肾俞”训练法。我们掌握了从涂抹精油、搓热手掌，到定位腰后部并进行有节奏的上下推按，同时协调进行肛门收紧与放松的完整过程。关键在于保持动作的节奏性与持续性，整个训练需严格计时2分钟。
