# 强性健肾保健操：1：现代性健康概论与保健操介绍

## 概述
在本节课中，我们将要学习现代性健康的基本理念、当代男性面临的普遍性问题，并介绍一种有效的自我锻炼方法——强性健肾保健操。我们将了解其目标、来源与核心作用。

一个民族由每一个人组成。一个民族的整体健康，由每一个人的健康组成。我们每一个中华民族的儿女都应努力加强保健，增进自身健康，这也是对民族的贡献。

传统或原始的健康概念仅仅是身体健康。后来加入了心理健康。但时代发展至今，健康和保健理念加入了新的内容。现代健身四理念是：健身、健心、健脑、健性。这四种健康分别是身体健康和心血管健康、心理健康、脑血管健康和智力健康，最后加上性健康。四个方面缺一不可。

当今社会，现代男性在这四个层面的健康都在不断下降，而性健康形势更加严峻。

## 当代男性性健康的八大普遍问题
以下是当代男性在性健康方面普遍存在的八个问题。

第一种，精液质量大幅度降低。具体表现为：精子密度减少一倍，每次射精量减少一倍，精子活性降低很多。

第二种，性生殖疾病多发化、年轻化，过早出现男性更年期综合症。

第三种，性衰老年龄大大提前。现代人工作紧张，生存压力大，许多男性刚过40岁已经出现性欲衰退，感到力不从心。

第四种，性生活时间偏短。男性在性生活中的普遍情况是时间过短，具体体现在女方还未达到高潮，自己也未尽兴，就已经结束。作为男子汉，没有让女性身心酣畅，不能充分体现自身能力，这是相当沮丧的。

第五种，性障碍增多。早泄、阳痿等性障碍的人开始增多，不过其中一半人是属于功能性的，完全能够纠正。但是他们并不知道有效的治疗方法。

第六种，出现了新问题：性欲过度。现代生活中性刺激随处存在，有些人在不知不觉之中，性生活已经超量。而性欲过度、频繁射精导致体力透支，对身体健康造成很大损害。

第七种，中老年人过早停止性爱。男人性生活本可持续到70多岁，可是很多男性早早“停工”，结束了性生活。其实，持续而有规律的性生活，对健康长寿都是有利的。

第八种，性疾病传播增大，艾滋病威胁加剧。

在这八大问题里面，对广大男性影响最大、对生活幸福影响最多的则是射精过早，坚持时间短。这可以说是男性最普遍的问题，是男性雄风受挫的第一位问题。
