# 艳舞私房课：06：头会动，才能掌握艳舞诀窍 🕺

在本节课中，我们将要学习艳舞中一个至关重要的基础动作——头部波浪。掌握头部的灵活运动，是提升舞蹈表现力与流畅度的关键。

## 概述

头部波浪动作在艳舞中非常重要。在开始练习前，你可能需要先进行一些颈部的热身和拉伸活动。

## 动作详解

上一节我们介绍了头部动作的重要性，本节中我们来看看具体的练习方法。

请尝试用你的头顶画出尽可能大的圆圈。以下是练习的两个核心步骤：

1.  **起始与回拉**：伴随音乐节奏，将头部向后仰，再缓慢向前回拉。
2.  **保持与律动**：在动作过程中保持身体的律动感，让动作与音乐融为一体。

## 总结

本节课中我们一起学习了艳舞的头部波浪基础动作。记住，从用头顶画大圈开始练习，并确保在动作前充分活动颈部。持之以恒的练习将使你的舞蹈更具表现力。
