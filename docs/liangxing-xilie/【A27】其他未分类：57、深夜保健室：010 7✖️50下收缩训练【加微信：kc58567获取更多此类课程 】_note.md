# Python入门：01：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是如何工作的，是编写任何程序的第一步。

## 概述

变量就像一个贴有标签的盒子，我们可以用它来存放数据。数据类型则定义了盒子里可以存放什么种类的东西，比如数字、文字或真假值。本节将详细介绍如何创建和使用变量，以及Python中几种基本的数据类型。

---

## 变量：数据的容器 📦

变量是程序中存储数据的基本单元。你可以把变量想象成一个带有名字的储物箱，你可以往里面放东西（赋值），也可以随时查看或更换里面的东西。

### 如何创建变量

在Python中，创建一个变量非常简单，只需要使用等号 `=` 进行赋值即可。等号左边是变量名，右边是你想存储的值。

```python
my_variable = 10
```

这行代码创建了一个名为 `my_variable` 的变量，并将整数值 `10` 存储在里面。

### 变量命名规则

给变量起名字需要遵循一些规则，以下是主要的命名规范：

*   **可以包含**：字母（a-z, A-Z）、数字（0-9）和下划线（_）。
*   **不能以数字开头**：例如 `1variable` 是无效的。
*   **区分大小写**：`myVar` 和 `myvar` 是两个不同的变量。
*   **不能使用Python关键字**：例如 `if`、`for`、`while` 等是Python语言本身使用的，不能作为变量名。
*   **建议使用描述性的名称**：使用像 `user_age`、`total_price` 这样的名字，能让代码更容易理解。

---

## 基本数据类型 🔢

上一节我们介绍了如何使用变量作为容器。现在，我们来看看这个容器里可以存放哪些基本类型的数据。Python有几种内置的基本数据类型，最常用的是以下三种。

### 1. 整数 (int)

整数是不带小数点的数字，可以是正数、负数或零。

```python
age = 25
temperature = -5
count = 0
```

### 2. 浮点数 (float)

浮点数是带有小数点的数字，用于表示实数。

```python
price = 19.99
pi = 3.14159
average = 85.5
```

### 3. 字符串 (str)

字符串是由一系列字符组成的文本。在Python中，字符串需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。

```python
name = “Alice”
greeting = ‘Hello, World!’
message = “Python is fun.”
```

### 4. 布尔值 (bool)

布尔值只有两种可能：`True`（真）或 `False`（假）。它们通常用于逻辑判断。

```python
is_raining = True
has_finished = False
```

---

## 查看与理解数据类型 🔍

在编程时，有时你需要知道一个变量里存放的是什么类型的数据。Python提供了一个非常方便的函数 `type()` 来查看任何值或变量的数据类型。

### 使用 `type()` 函数

你可以将变量或值直接放入 `type()` 函数中，它会返回对应的类型。

```python
print(type(10))        # 输出：<class ‘int’>
print(type(3.14))      # 输出：<class ‘float’>
print(type(“Hello”))   # 输出：<class ‘str’>
print(type(True))      # 输出：<class ‘bool’>

my_number = 100
print(type(my_number)) # 输出：<class ‘int’>
```

了解数据类型非常重要，因为它决定了你可以对这个数据执行哪些操作。例如，你可以将两个整数相加，但不能将一个整数和一个字符串直接相加。

---

## 总结

本节课中我们一起学习了Python编程的基石：**变量**和**基本数据类型**。

*   **变量**是存储数据的命名容器，通过赋值操作符 `=` 来创建和修改。
*   **基本数据类型**包括：
    *   **整数 (int)**：表示整数值。
    *   **浮点数 (float)**：表示带小数的数值。
    *   **字符串 (str)**：表示文本信息，用引号包裹。
    *   **布尔值 (bool)**：表示逻辑真 (`True`) 或假 (`False`)。
*   可以使用 **`type()`** 函数来检查任何变量或值的数据类型。

理解这些概念后，你已经可以开始用Python存储和处理简单的信息了。在接下来的课程中，我们将学习如何利用这些数据做更多的事情，比如进行数学运算和逻辑判断。
