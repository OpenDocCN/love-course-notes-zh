# 玉蛋功：A15：热身抖动与脊椎呼吸 🥚

在本节课中，我们将学习玉蛋功的两个基础热身练习：全身抖动与脊椎呼吸。这些练习旨在激活身体能量，为后续的功法练习做好准备。

## 热身抖动 🎵

上一节我们介绍了课程概述，本节中我们来看看第一个热身动作：抖动。这个动作的目的是通过有节奏的抖动，激活全身的能量循环。

双腿与肩同宽站立，双膝自然弯曲。

以下是跟随音乐节奏进行抖动的步骤：

*   抖动全身。
*   抖动腹部。
*   抖动双手。
*   抖动左侧身体。
*   抖动右侧身体。
*   将双手上举并抖动。

在抖动过程中，感受能量从心脏扩散到身体的末梢。你的手指、脚趾乃至发丝都应充满能量，身体会逐渐变得柔软而富有活力。

完成抖动后，恢复直立的站姿。

## 脊椎呼吸 🌬️

完成了能量激活的抖动练习后，接下来我们学习脊椎呼吸法。这个练习通过呼吸与动作的配合，进一步伸展和放松脊柱。

双腿略宽于肩站立，双膝自然弯曲。

以下是脊椎呼吸的完整步骤：

1.  用鼻子吸气。一边吸气，一边张开十指，从身体两侧将双臂向上举起。
2.  双肩尽量打开，胸部微微前挺，双眼看向上方。
3.  张开嘴，尽力从腹部呼气。呼气时发出“哈”的声音。
4.  在呼气的同时，握拳并收回双肘，合在胸前。背部尽量向上拱起，低头将脸埋在胸前。

整个过程中，吸气时身体充分伸展，呼气时上身缩成一团。我们将这个吸与呼配合动作的过程，一起重复9次。

---

本节课中我们一起学习了玉蛋功的两个核心热身练习。通过**全身抖动**激活了身体能量流，接着通过**脊椎呼吸**（吸气伸展，呼气收缩）实现了脊柱的伸展与放松，为深入练习打下了良好的身体基础。
