# 玉蛋功：A15：分段跟随练习六丨阴道举重 💎

在本节课中，我们将要学习玉蛋功的第六个分段练习——阴道举重。这个练习旨在通过负重训练，增强盆底肌（PC肌）的力量与控制能力。我们将学习如何选择合适的重量，并跟随指导完成三个核心动作：蝴蝶飞舞式、芝麻开花式以及它们的变体。坚持练习，可以提升盆底肌的弹性与力量，为身心健康带来益处。

## 重量选择标准 ⚖️

开始训练前，选择合适的负重至关重要。以下是判断重量是否合适的标准：

*   **合适的感觉**：当你双腿与肩同宽站立时，盆底肌（PC肌）只需稍微用力收缩，就能保持吊坠不掉落，并能较为轻松地完成后续训练。
*   **增加原则**：可以尝试以**20克**为单位逐步增加重量。
*   **重要警告**：切勿为了追求效果而让盆底肌负担过重，过重的负荷反而会导致肌肉损伤。

3.  练习时速度要慢：**呼气时收缩、挤压、握紧玉蛋；吸气时放松**。
