# 艳舞私房课：06：膝部动作私密互动舞教程

在本节课中，我们将学习一个需要稍加练习的膝部互动舞蹈动作。这个动作适合中级舞者，能增强舞蹈的私密互动感。

## 🪑 准备工作

上一节我们介绍了舞蹈的基础热身，本节中我们来看看这个特定动作所需的准备。以下是开始练习前必须完成的两个关键步骤。

1.  **确保座椅稳固**：选择一把足够坚固和稳定的椅子，这是安全完成动作的前提。
2.  **充分热身**：在尝试此动作前，务必完成你的热身练习，以避免受伤。

## 💃 动作练习方法

完成准备工作后，就可以开始练习动作本身了。你可以根据舒适度选择以下任一方式进行练习。

*   **独自练习**：你可以单独在椅子上练习这个动作。
*     **结伴练习**：如果感觉舒适，也可以与一位朋友一起练习。
