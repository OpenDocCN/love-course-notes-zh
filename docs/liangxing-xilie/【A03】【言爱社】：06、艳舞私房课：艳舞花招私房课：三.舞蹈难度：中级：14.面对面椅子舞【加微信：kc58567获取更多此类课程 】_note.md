# 艳舞私房课：06：面对面椅子舞 🪑

在本节课中，我们将学习一个中级难度的舞蹈编排——面对面椅子舞。这个舞蹈仅需两把椅子、一双高跟鞋和一些练习即可完成。

## 舞蹈准备 🪑

上一节我们介绍了舞蹈的基本概念，本节中我们来看看这个特定舞蹈所需的准备工作。以下是开始练习前需要准备的物品清单：

*   两把椅子
*   一双高跟鞋

## 舞蹈练习要点 💃

准备好物品后，我们就可以进入练习阶段了。面对面椅子舞的核心在于利用椅子作为支撑和道具，与舞伴进行互动。以下是练习时需要注意的几个关键点：

*   确保椅子稳固，练习时注意安全。
*   穿着高跟鞋练习需逐步适应，保持身体平衡。
*   与舞伴保持眼神交流和节奏同步。

## 总结 📝

本节课中我们一起学习了面对面椅子舞。这是一个中级难度的编排，核心要求是准备好两把椅子和一双高跟鞋，并通过持续练习来掌握动作与配合。
