# 性福诊疗室：A18：无性恋的认知与应对策略

在本节课中，我们将探讨一个特殊的性取向——无性恋。我们将了解其核心特征，分析相关个案，并明确在伴侣关系中寻求治疗的核心目标与方向。

## 个案现象：对两性均无兴趣

最近我们遇到一个个案。他与妻子结婚34年，但从未有过性生活。他强烈怀疑自己是同性恋，因此加入了一个同性恋社群。然而，在加入后，他发现自己对同性恋群体中的成员也无法产生爱恋之情。

他因此感到困惑：既不爱男性，也不爱女性，自己究竟属于哪个群体？

## 核心概念：认识无性恋

现在存在一个大众了解较少的群体，称为**无性恋**群体。上述个案的情况，很可能就属于不爱男性也不爱女性的无性恋状态。

然而，在性方面，这类人群在与自己独处时，是可以进行自慰的。他们拥有正常的**射精**功能，也能产生性唤起（勃起）。但他们确实对异性缺乏兴趣。

## 求治动机：源于关系压力

大部分无性恋者不会主动寻求治疗。他们前来求治，最主要的原因是与其伴侣的相处出现了问题。

具体而言，是无法与伴侣完成**性行为**这件事。因此，他们求治的目标通常设定为：能够与他人完成这件事，目的可能是为了传宗接代，或者是为了满足伴侣的期待。

## 治疗目标：完成特定任务与达成共识

如果“完成性行为”这个部分可以实现，那么不论当事人本身是否有性欲或享受其中，治疗的首要目标便算达成。

当双方对此结果都感到满意时，便没有人有权利去限制他人喜欢何种性别，或是去质疑一个人“根本不喜欢任何性别”这件事。

## 总结

本节课中，我们一起学习了无性恋的基本概念。我们通过个案了解到，无性恋者可能对两性均无恋爱情感，但可能保有自慰能力。他们寻求帮助的主要动力往往来自伴侣关系中的压力，核心治疗目标应聚焦于解决具体的、共识性的关系任务，而非改变其性取向本身。
