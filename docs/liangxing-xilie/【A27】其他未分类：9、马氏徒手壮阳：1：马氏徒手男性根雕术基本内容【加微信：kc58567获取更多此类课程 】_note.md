# Python入门：01：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是如何工作的，是编写任何程序的第一步。

## 什么是变量？ 📦

变量可以看作是一个贴有标签的“盒子”，我们可以在里面存放数据。在Python中，创建一个变量并给它赋值非常简单。

上一节我们介绍了课程的目标，本节中我们来看看如何创建和使用变量。

### 变量的创建与赋值

在Python中，使用等号 `=` 来给变量赋值。等号左边是变量名，右边是想要存储的值。

```python
# 创建一个名为 message 的变量，并存入文本“你好，世界！”
message = "你好，世界！"
```

执行这行代码后，Python解释器会做两件事：
1.  在计算机内存中开辟一个空间来存放字符串 `"你好，世界！"`。
2.  将这个内存空间的“地址标签”（即变量名 `message`）指向这个数据。

之后，当我们使用变量名 `message` 时，Python就知道去对应的内存位置找到这个数据。

以下是变量命名的基本规则：
*   变量名只能包含字母、数字和下划线。
*   变量名不能以数字开头。
*   变量名是区分大小写的（例如，`my_var` 和 `My_Var` 是两个不同的变量）。
*   应避免使用Python的保留关键字（如 `if`, `for`, `while` 等）作为变量名。

## 数据类型是什么？ 🔢

数据类型定义了变量中存储的是哪种数据，以及可以对这种数据执行哪些操作。Python有一些内置的基本数据类型。

上一节我们学会了如何把数据放进“盒子”（变量），本节中我们来看看“盒子”里可以放哪些不同类型的“东西”。

### 常见的数据类型

Python中几种最常用的数据类型如下：

1.  **整数**
    *   表示没有小数部分的数字，可以是正数或负数。
    *   **代码示例**：`age = 25` 或 `temperature = -10`

2.  **浮点数**
    *   表示带有小数点的数字。
    *   **代码示例**：`price = 19.99` 或 `pi = 3.14159`

3.  **字符串**
    *   表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
    *   **代码示例**：`name = “小明”` 或 `greeting = ‘Hello’`

4.  **布尔值**
    *   只有两个值：`True`（真）和 `False`（假）。通常用于逻辑判断。
    *   **代码示例**：`is_student = True` 或 `is_raining = False`

### 使用 `type()` 函数查看数据类型

我们如何知道一个变量里存的是什么类型的数据呢？可以使用Python内置的 `type()` 函数。

```python
# 定义几个不同类型的变量
my_integer = 100
my_float = 3.14
my_string = “Python”
my_bool = True

# 使用 type() 函数查看它们的类型
print(type(my_integer))  # 输出：<class ‘int’>
print(type(my_float))    # 输出：<class ‘float’>
print(type(my_string))   # 输出：<class ‘str’>
print(type(my_bool))     # 输出：<class ‘bool’>
```

## 变量的重新赋值与动态类型 🔄

Python的一个特点是**动态类型**。这意味着：
*   你不需要提前声明变量的类型。
*   同一个变量名可以在程序运行过程中被重新赋予不同类型的值。

```python
# 变量 x 最初是一个整数
x = 10
print(x)          # 输出：10
print(type(x))    # 输出：<class ‘int’>

# 之后，我们可以将 x 重新赋值为一个字符串
x = “现在我是文本了”
print(x)          # 输出：现在我是文本了
print(type(x))    # 输出：<class ‘str’>
```

这个特性非常灵活，但在复杂程序中需要注意，避免因为意外改变变量类型而导致错误。

## 总结 📝

本节课中我们一起学习了Python编程的基石：
1.  **变量**是存储数据的命名容器，通过 `=` 赋值。
2.  **数据类型**定义了数据的种类和可执行的操作，主要学习了**整数**、**浮点数**、**字符串**和**布尔值**。
3.  使用 **`type()`** 函数可以查看任何变量或值的数据类型。
4.  Python是**动态类型**语言，变量可以随时被重新赋值为不同类型的值。

理解并熟练运用变量和数据类型，是您迈向Python编程高手的第一步。在接下来的课程中，我们将学习如何使用这些数据来进行运算和做出决策。
