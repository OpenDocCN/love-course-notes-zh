# 深夜保健室：A27：男人真心话大调查

在本节课中，我们将通过一份问卷调查，探讨男性在亲密关系中一些通常不愿让伴侣知晓的真实想法。我们将从几位男性的分享中，了解他们对经验、主动性、身材等话题的看法，并通过快问快答的形式揭示一些普遍观点。

## 疯狂性事分享

上一节我们介绍了课程主题，本节中我们来看看几位男性分享的、令他们印象深刻的个人经历。

以下是三位男性的分享：

*   **挑战者**：我曾为了在朋友面前证明自己，试图在一晚内完成七次。但到第六次时，我在过程中睡着了，未能完成挑战。对我而言，第二次之后的行为就失去了意义，纯粹是为了挑战自我。第五次时，我是靠回想朋友们质疑的表情才完成的。
*   **初次经历者**：我最印象深刻的是第一次经历，因为当时是在朋友家或旅馆，有其他朋友在旁边睡觉。这让我非常紧张，不确定他们是否听到了声音。
*   **公共场所冒险者**：我曾因一时冲动，在营业场所的洗手间里独自解决。相比车震，我认为洗手间的空间反而更宽敞舒适，因为车里会有排挡杆等障碍物。

听完三位的分享，主持人认为这属于“正常男性”范围，并未过于夸张。

## 快问快答：男性真实想法

了解了他们的个人故事后，我们通过六道快问快答题目，进一步探索男性群体的普遍观点。

### 第一题：偏好经验丰富的女性吗？

对于“是否更喜欢经验丰富的女性”这个问题，现场男性给出了不同理由。

以下是他们的观点：

*   **支持方**：与处女相比，我**不会不喜欢处女**，但经验丰富的女性更能勾起我的欲望和兴致。与什么都不懂的女性相处感觉奇怪，像在带未成年人，甚至有种“污染”他人的感觉。
*   **契合度考量**：经验匹配很重要。如果对方需要一步步教导，互动会不协调。就像跳舞，双方水平相当才能玩得开心，有互动感。

调查数据显示，喜欢经验多和经验少的女性比例相近（各约8%），而明确喜欢处女的只占20%。

### 第二题：女性松紧度会随年龄或生育改变吗？

本题探讨男性对女性身体变化的认知。现场三位男性均认为“不会”，这与更大范围的调查结果（三分之二男性认为“会”）相反。

以下是现场观点：

*   **与生育无关**：认为这与身材比例有关，和生育没有直接关系。经验丰富的女性可能更懂得控制肌肉。
*   **与经验有关**：认为与年龄有关，因为随着年龄增长，性经验可能更丰富。一位男性提出了一个比喻：**女生就像一个漱口袋，当经验丰富之后就变松了**。但他随即澄清这并非贬义。
*   **个人不适**：有男性提到，因包皮过长，太紧反而会导致疼痛不适。

讨论中也提到了“阴道紧缩手术”等医学手段，有人认为这会影响男性体验。

### 第三题：希望女性主动吗？

所有现场男性都举牌表示“希望女性偶尔主动”，这与调查中三分之二男性的选择一致。

以下是他们希望女性主动的原因：

*   **参与感**：希望女性有参与感，不一定是要会很多姿势，即使是躺着配合律动也可以。这是一种分工合作。
*   **主导权交替**：喜欢有来有往的主导权交替，而非始终由自己掌控。
*   **怕累**：有男性直言，自己健身已经很累，结束后不想再全程主导运动。

### 第四题：女性身材走样会影响性欲吗？

现场男性一致认为“会”，调查数据也显示超过三分之二的男性认同此观点。

主要看法是，如果伴侣身材走样，性欲会下降。有男性分享，曾因看到心仪女性露出松懈的腹部而瞬间失去兴趣。他们通常也对自我身材有要求。

### 第五题：喜欢女性在床上更放得开吗？

所有现场男性迅速举牌表示“喜欢”。调查显示高达五分之四的男性喜欢伴侣更主动、放得开。

他们认为，有时女性会因怕显得经验丰富而故意装乖，但男性其实更喜欢对方主动营造气氛。主动的举例包括：伴侣在半夜睡着时突然开始挑逗。

### 第六题：女性哪些自以为性感的动作其实很解嗨？

约三分之一的男性认为存在这类动作。现场讨论了一些可能“解嗨”的行为：

*   **模仿A片**：例如身材不符却硬要模仿某些姿势，或突然讲日文等，会让人出戏。
*   **含手指**：有男性表示不喜欢被含手指，感觉奇怪。但也有男性喜欢这种触电般的接触感。

## 给女性的个别告诫

在分享了普遍观点后，每位男性给出了一条具体的个人建议。

以下是他们的告诫：

*   **告诫一（关于亲吻身体）**：我个人非常不喜欢被亲吻身体，感觉很奇怪。我喜欢前戏后直接进入正题。我认为持续亲吻超过三分钟就足够了。
*   **告诫二（关于沟通）**：当感觉不舒服或不够爽时，我会直接换个动作，但不会说出来打击对方。我认为很多女性也不确定对方的反应是真好还是假好。
