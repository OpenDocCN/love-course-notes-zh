# 玉蛋功课程：A15：放置玉蛋与取出方法 🥚

在本节课中，我们将学习玉蛋功练习中两个关键步骤：如何正确放置玉蛋，以及练习结束后如何安全地取出并清洁玉蛋。掌握这些基础操作是安全有效练习的前提。

## 放置玉蛋 🎼

上一节我们介绍了练习前的准备，本节中我们来看看如何放置玉蛋。放置时，身体姿势和放松心态至关重要。

一般来说，建议采取半蹲姿势进行放置。但对于初次尝试者，如果担心阴道肌肉无法握紧玉蛋，可以选择躺下放置。

以下是放置玉蛋的具体步骤：
*   保持微笑，进行自然呼吸。
*   用手握住玉蛋较小的一端，将较大的一端朝向身体内侧。
*   将玉蛋顶端抵在阴唇上，用中指顶住玉蛋。
*   找到一个合适的角度，轻轻向里推送玉蛋。保持微笑，避免紧张，持续轻轻推动，直到玉蛋不再向内移动。
*   吸气，同时放松身体。

呼气时，收缩PC肌（耻尾肌）。此时可以借助腹部肌肉的力量，想象将玉蛋深深吸入身体盆腔处。

正确放置好的玉蛋，在身体自然放松时应无异物感。完全放入后，可尝试轻轻收缩挤压玉蛋，并将注意力转移到阴道内部。

去感受体内的玉蛋：放松时是什么感觉，挤压时又是什么感觉。部分人可能会敏感地感受到玉蛋的存在甚至其细微震动，但大多数人可能需要经过练习才能逐渐感知到玉蛋。

## 姿势调整与初步感知

如果你刚才躺着放置了玉蛋，现在可以尝试站起来，双腿与肩同宽。慢慢放松PC肌，感受玉蛋自身重力对盆腔的刺激。

如果你的PC肌力量较为薄弱，可能会感到玉蛋有明显的下坠感，甚至可能脱落。若出现这种情况，接下来的训练应以躺姿为主。或者在练习时，将双腿并拢一些，以减轻PC肌的负担。

找到合适姿势后，可以用力收缩一下PC肌，感受玉蛋在肌肉挤压下的运动。当我们挤压PC肌时，玉蛋会向上运动。多尝试几次，熟悉用PC肌操控玉蛋的感觉。

现在，你的阴道和玉蛋已经完成了“相互认识”，便可以进入正式的练习阶段了。

## 取出与清洁玉蛋 🧼

练习结束后，你可以选择继续将玉蛋留在体内进行日常活动，或选择其中某些动作重复练习。但需注意，单次专注练习的时间不应超过**20分钟**。此20分钟不包括热身、冥想和放松的时间。此外，每天放置玉蛋的总时长也不应超过**2小时**。

需要取出玉蛋时，请按以下步骤操作：
1.  用手捏住玉蛋上的牙线。
2.  慢慢呼气，同时将玉蛋轻轻取出。

取出玉蛋后，请务必进行清洁：
1.  用热水冲洗干净。
2.  或在滴有CJ茶树精油的清水中消毒浸泡几分钟。
3.  自然风干后，妥善收藏即可。
