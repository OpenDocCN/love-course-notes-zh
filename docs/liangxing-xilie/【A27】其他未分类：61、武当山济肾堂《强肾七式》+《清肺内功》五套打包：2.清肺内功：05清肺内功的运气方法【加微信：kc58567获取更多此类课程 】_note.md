# 武当山济肾堂强肾清肺功法：2.2：清肺内功运气方法详解 🫁

在本节中，我们将学习清肺内功的核心——运气方法。我们将了解运气的基本原理、具体步骤与身体各部位的要求，掌握如何通过正确的运气来增强身体。

## 概述

清肺内功通过特定的运气方式疏通经络。其原理在于运气时能促使气血加速循环，从而疏通堵塞的经络，保持经络畅通。

## 运气原理与作用

上一节我们介绍了呼吸的环节，本节中我们来看看如何运用气息。运气时，我们的整个腹部会形成一个如同充满气的轮胎的状态。

**核心原理公式**：`运气 -> 气血加速循环 -> 疏通经络`

当轮胎充满气后，用力击打它，力量会被内部的空气阻隔，形成一层保护膜，因此只能伤及表皮。练习清肺内功后能够抗击打，正是基于这个原理。

运气是在闭气时进行的。呼吸方式包含吸气、闭气、喷气三个环节，运气则是在闭气（不呼不吸）的状态下完成的。

## 运气具体要求

以下是进行运气时，对身体各部位的具体要求。请务必按照步骤逐一调整身体状态。

**1. 脚部要求**
双脚脚趾需如大树深根，牢牢抓地。即十个脚趾头要用力向下扣紧地面。

**2. 膝盖要求**
两个膝盖需使用“暗劲”向内扣合。力量方向是朝向身体中线。
*   **明劲**：外形发生明显变化的用力。
*   **暗劲**：力量内蓄，膝盖已向内扣，但外形无明显动作。

**3. 裆部要求**
提纲收肾。
*   **提纲**：收紧并上提肛门。
*   **收肾**：男性需用力上提睾丸；女性需收紧外阴。目的是将前后二阴整体向上收紧。

**4. 气息与腹部要求**
气沉丹田，并绷紧腹部。
*   **气沉丹田**：吸气饱满后，将气息下沉至丹田（小腹），此时腹部会自然向外鼓起。
*   **绷紧腹部**：腹部鼓起后，需主动绷紧腹壁肌肉，使腹部触感坚硬，而非柔软。最终状态应如充满气的轮胎般坚实有弹性。

**5. 头部与整体要求**
*   手部动作需配合具体的功法招式用力。
*   头部应尽量放松，不可用力。
*   简言之，运气时，除了头部放松，身体自头部以下均需处于用力状态。

## 运气流程总结

本节课中我们一起学习了清肺内功的运气方法。现在我们来总结完整的运气流程：

1.  **吸气**：深吸气，使气息饱满。
2.  **闭气**：进入不呼不吸的闭气状态。
3.  **运气**：在闭气状态下，按顺序做到：十趾抓地 -> 膝盖暗劲内扣 -> 提纲收肾 -> 气沉丹田并绷紧腹部 -> 头部放松。

通过持续练习，腹部的绷紧程度和抗击打能力会逐渐增强。具体的功法动作与运气配合，我们将在后续章节详细讲解。掌握这些要点，能帮助你在后续练习中更准确地运气。
