# 性福诊疗室：A18：外遇一定跟性有关吗

在本节课中，我们将探讨一个常见问题：伴侣关系中的外遇行为是否必然与性技巧或性功能障碍直接相关。我们将通过一个具体案例，分析其背后的心理因素与解决路径。

## 案例引入与分析

最近有来访者询问，她的丈夫对夫妻性生活不满意，并将原因归咎于她的技巧不佳。她因此考虑是否可以向职业工作者学习技巧。这实际上是两个不同的问题。职业工作者或许掌握特定技巧，但性爱中最核心的要素是**爱**，而非单纯的技巧。

因此，面对此类情况，我的建议是从**爱**出发。双方应首先致力于培养沟通，明确彼此的需求：男性究竟需要何种互动来提升性生活质量，女性又能提供怎样的情感支持来激发男性的欲望。这是一个值得深入探讨的议题。

## 核心问题：心理障碍而非生理疾病

上述案例并非孤例。另一位来访者的情况是，其丈夫可以在婚外完成性行为，唯独对妻子不行。我们的首要步骤是厘清问题：丈夫是否存在性功能障碍。

通常，丈夫对妻子无法完成性行为（“行房”）。妻子的第一反应往往是怀疑丈夫有阳痿问题。然而，经过专业检测，我们发现这位丈夫实际上并无器质性的**阳痿**或**早泄**问题。

问题的根源在于心理层面。他在既往的性经历中遭遇了勃起方面的挫折，导致自尊心受损，形成心理阴影，因而不敢在妻子面前再次尝试勃起，并会寻找各种理由来掩饰无法勃起的真实原因。

## 解决方案：伴侣协作与心理疏导

在治疗过程中，我们手把手地引导妻子学习如何进行爱抚，以及如何面对丈夫在性行为过程中可能出现的勃起中断（“突然软掉”）的情况，并学习如何帮助他。

后来，他们成功地恢复了和谐的性生活。这个案例表明，当你发现伴侣在床上的表现不佳时，急于责怪对方并非解决之道。相反，我们应该探究其背后原因：究竟是存在真实的生理功能障碍，还是在面对亲密伴侣时，存在**心理上的焦虑**。

这些心理焦虑，正是造成勃起功能障碍最主要的因素之一。

## 本节总结

本节课我们一起学习了，外遇或性表现问题不一定源于技巧缺失或生理疾病。很多时候，核心在于亲密关系中的心理互动与自尊感受。解决问题的关键是从爱与沟通出发，识别并协同克服心理障碍，而非单纯追求技巧或相互指责。
