# XO初级训练自学课：02：XO挤与臀桥 🏋️‍♀️

在本节课中，我们将要学习如何将XO挤技巧与臀桥动作相结合，以强化核心肌群与盆底肌的协调性。这套训练有一定难度，但效果显著，请跟随指导反复练习。

## 概述
本节训练是本周第五天的内容，重点在于XO挤与核心肌的强化。训练需要平躺在瑜伽垫上进行，建议开启音频循环播放模式，跟随练习3到5遍以达到最佳效果。

## 准备姿势
首先，我们需要摆好正确的起始姿势。

平躺在瑜伽垫上，双腿膝盖弯曲，脚掌踩实地面。确保膝盖与脚掌方向一致，避免内扣或外扩。双脚分开，距离与个人胯宽相同。每个人的胯宽不同，请找到适合自己的宽度。

## 热身激活
在开始正式训练前，我们需要先激活目标肌群。

以下是静止状态下的XO挤热身动作，用于唤醒盆底肌：
*   **X型挤**：盆底肌发力，想象向外推。
*   **O型夹**：盆底肌发力，想象向内收紧。

准备好，我们开始热身：**X型挤，345…678…O型夹，2345678…S型挤，345…678…O型夹，234567…停**。

好，热身完毕，盆底肌已被激活。

## 臀桥与XO挤结合练习
上一节我们激活了盆底肌，本节中我们来看看如何将其与臀桥动作结合。

现在开始做臀桥。首先用骨盆发力，使臀部慢慢离开地面。然后用骨盆带动脊椎，再带动腰部逐渐抬高。

以下是动作的关键配合要点：
*   在臀部抬高的过程中，同步进行**X型挤**（盆底肌向外推）。
*   只需抬起到一半高度即可，无需达到最高点。

抬起到位后，我们慢慢收紧臀部肌肉，同时将身体缓缓放下。
*   在臀部回落接触地面的过程中，同步进行**O型夹**（盆底肌向内收紧）。
*   臀部贴地后，继续用力收紧。

这个动作的核心目的是锻炼核心肌群与盆底肌之间的协调性。

## 动作协调性详解
熟悉了基本动作后，我们需要理解其中的协调要领。

在臀桥的抬起和放下过程中，**X型挤**和**O型夹**的发力应是**过渡式**和**渐变式**的。它们需要与臀部动作同步进行，而不是割裂地先完成挤夹再做臀桥。

准备好，我们再来一组以体会这种同步性：抬起做**X型挤**，臀部离地至一半高度…保持…现在开始慢慢放下，臀部回落地面时做**O型夹**…用力收紧…收紧…放松。

## 交替循环训练
掌握了协调要领后，我们正式进入交替循环练习。

现在调整呼吸，做好准备：深呼吸…吐气。

以下是完整的交替训练循环，请跟随节奏练习：
*   **抬起阶段**：做**X型挤**，骨盆带动脊椎抬起，盆底肌向外推。节奏：**1, 2, 3, 4, 5, 6, 7…**
*   **放下阶段**：做**O型夹**，臀部逐渐收紧下落，回到地面后盆底肌再收紧。节奏：**…走， 5, 6, 7…**

我们继续循环：
抬起，**X型挤**，盆底肌向外推…**1, 2, 3, 4, 5, 6, 7…** 换。
放下，**O型夹**，臀部收紧下落…收紧，再收紧…**5, 6, 7…**

加快速度练习：
抬起，**X型挤**…**4, 5, 6, 7, 8…**
放下，**O型夹**…**3, 4, 5, 6, 7…**
再来，抬起，向外推**X型挤**…**5, 6, 7, 8…**
放下，向内收**O型夹**…**5, 6, 7…停**。

## 放松与总结
训练结束，请平躺放松全身肌肉。深吸一口气…再深呼气。

本节课中我们一起学习了将XO挤技巧与臀桥结合的训练方法。这组训练能帮助你更好地在实战中运用XO挤，从而与伴侣获得共同的愉悦。请务必多加练习，尽早熟练掌握今天的动作。
