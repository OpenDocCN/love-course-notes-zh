# 性爱科学：05：如何应对勃起功能障碍（二）

在本节课中，我们将学习作为伴侣，如何以积极、合作的态度应对男友的勃起问题。我们将探讨心态调整、沟通技巧以及具体的鼓励方法。

上一节我们介绍了勃起在性爱中的作用及非药物预防措施。本节中我们来看看，当问题发生时，伴侣应如何共同面对。

## 核心理念：问题是共同的

在亲密关系中，任何问题都应被视为双方共同面对的挑战，而非单方的责任。例如，伴侣一方出现性冷淡，这是两人共同遇到了“性冷淡”这个问题。同理，当男友出现勃起障碍时，应理解为“我们遇到了勃起障碍的问题”，而不是“他不行了”。这种思维转变是有效应对的第一步。

夫妻是人生的合作伙伴。既然决定携手共度一生，遇到问题时就不应将责任完全归咎于一方。

## 行动指南：两大原则

基于共同面对的理念，我们可以从以下两个方面努力。

### 1. 不强求，懂克制

即使你当时性欲强烈，若伴侣确实无力满足，也应适当克制冲动。切勿施加压力强制要求对方。这会给对方带来巨大心理负担。若需求无法通过性交满足，可以自行处理，或邀请伴侣以其他方式（如爱抚、观看）参与你的自我满足过程。

### 2. 及时给予鼓励

1.  **给予信任**

2.  **表示接纳**

3.  **表示感激**
    *   **原则**：对他为你做的任何事情（包括在性爱中的付出）及时表示感谢，不要视为理所当然。

4.  **表示赞美**
    *   **重点**：赞美他的身体和性感。
    *   **话术示例**：“只要跟你这样性感的身体接触，我就已经非常满足了。”

5.  **表示鼓励**
    *   **目的**：给予正向期待，减轻本次失败的压力。
    *   **话术示例**：“放心吧宝贝，今天只是累了状态不好。早点休息，明天我给你炖点汤补补，肯定马上就好了。”

## 问题解决逻辑与医学治疗

了解如何鼓励后，我们来看完整的解决逻辑。

**勃起障碍的解决应遵循以下步骤：**

1.  **男方自查与调整**：检查并改善生活习惯，包括**保证睡眠、均衡饮食、进行有氧运动**，以及加强**XO脊和下肢肌肉训练**（例如进行**深蹲**训练）。
2.  **伴侣表达支持**：女方明确表达共同面对、共同努力的决心与信心。
3.  **寻求医学诊断**：若经上述调整后问题依然存在，则应共同前往正规医院检查，确诊是否为勃起功能障碍。
4.  **遵医嘱治疗**：确诊后，在医生指导下进行治疗。一般医院可能推荐以下方法：
    *   **药物治疗**：如**伟哥（西地那非）、他达拉非（希爱力）** 等。
    *   **阴茎注射治疗**：主要用于器质性或严重的功能性勃起障碍。
    *   **激素治疗**：用于内分泌性勃起障碍。
    *   **注意**：具体治疗方案必须咨询正规医院的专业医生。
