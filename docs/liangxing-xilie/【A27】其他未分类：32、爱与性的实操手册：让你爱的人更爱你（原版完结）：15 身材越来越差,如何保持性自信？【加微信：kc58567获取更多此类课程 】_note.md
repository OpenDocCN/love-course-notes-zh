## 身材变化是“丧失”，需要经历心理过程

面对丧失，我们常常要经历**否认、愤怒、妥协、抑郁和接受**这五个阶段。

以下是这五个阶段的具体表现：
*   **否认**：内心认为“我不相信我瘦不下去”。
*   **愤怒**：产生“凭什么让我长这么胖？”的想法。
*   **妥协**：想法转变为“我不求回到原来，让我瘦几斤就可以了”。
*   **抑郁**：对一直瘦不下去感到难过，对很多事情失去兴趣。
*   **接受**：最终意识到“我接受我的情况就是这样，我还可以好好过日子”。

有的人可以顺利度过到最后的接纳，有的人则可能会卡在前面的某个阶段里出不来。如果女性对自己的身体变化不接纳，卡在了某一个阶段上，就可能会引发对性的抗拒。因为只有对自己的身体接纳以后，才能相信丈夫对自己身体的喜爱，享受和伴侣的性生活。

## 促进自我接纳的练习方法
前面我们讲了不接纳的负面影响，那么我们可以做些什么来促进自我接纳呢？以下是两个实用的练习方法。

### 方法一：身体感知冥想
这是一个增加对身体觉察的冥想练习。

```

1. 以舒服的姿势坐下、躺下或站立。
2. 将注意力置于全身上下，穿过每一个部位（头部、脸部、前胸、后背、双手、腹部、臀部、生殖器官、双脚）。
3. 感受身体的存在方式、放松或紧绷的部位、任何疼痛或刺痛感。
4. 集中注意力在紧绷或疼痛的部位，试着画出这些部位的精确边界。
5. 感受皮肤的感官，感受身体作为一个整体的连接。
6. 在内心对身体说：“谢谢。”
```

可以反复体验身体的每一个部分和你的感知。也可以在网络上搜索其他相关的冥想音乐辅助练习。

### 方法二：照镜子练习
舞蹈演员通过镜子与身体保持连接，我们也可以使用这个工具。

以下是练习步骤：
1.  在温度适宜的房间，脱掉所有衣服，站到全身镜前。
2.  观察自己的身体，感受这是属于你的独特身体。
3.  感知每一个部位带给你的感受和记忆。
4.  如果对某个部位有负面想法，尝试转化它。例如：
    *   抚摸剖腹产刀疤，想象孩子出生时的幸福与自豪。
    *   抚摸乳房，感受它的健康，告诉自己这是属于自己的独特乳房。
    *   抚摸腹部，与它沟通：“这么多的你来到这里是我曾经选择的结果，现在让我们合作让它缩小一些，变得更健康，可以吗？”
5.  练习时请带着接纳和合作的态度。如果有些感受难以消化，可以寻求专业帮助。
6.  感到安全时，可以邀请丈夫一起练习，彼此帮助对方获得更多对身体体验和回忆。这种深度感知对性的亲密很有帮助。
