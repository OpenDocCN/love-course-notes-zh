# 性学硕士教你科学延时法：A27：3.延时的骗术有哪些？🚫

在本节课中，我们将要学习如何识别网络上关于男性延时的常见骗术与误导信息。了解这些陷阱，能帮助你避免经济损失和身体伤害，并学会通过掌握正确的知识来科学地解决问题。

## 警惕不实医疗信息

上一节我们介绍了延时的重要性，本节中我们来看看需要警惕的第一个陷阱：不实的医疗广告信息。

网络上充斥着大量爆炸性的信息。例如，在搜索引擎上查询“包皮过长割不割”这类问题时，搜索结果前列往往会出现当地一些医院的广告信息。

这些医院大多是私立医院，以商业盈利为主要目的。它们经常利用信息不对称，对咨询者造成经济损失，甚至可能带来身体伤害。

因此，一再提醒大家，应前往公立性质的大型医院就诊，例如二级甲等或三级甲等医院。

此外，还需注意另一种坑：有些医院虽是公立性质，但将其部分门诊转租给市面上的私立医院经营。这让人防不胜防。

大家务必注意，最好选择名气较大、规模较大的医院。

以前也教过大家一个方法：在搜索引擎查询医院时，一定要查看“医院类型”栏目。该栏目会写明医院属于公立性质还是私立性质，以及是公立几级医院等信息。这些信息写得比较清楚。

大家一定要查询对自己有效的信息，并多加分辨。

## 识别虚假保健品

除了医疗信息，另一个需要警惕的领域是所谓的“壮阳补肾”保健品。

市面上确实存在很多此类保健品。但众所周知，中国的保健品市场很混乱，很多产品并非真正的保健品。

需要有效地辨别市面上哪些东西属于真正的保健品。为此撰写过一篇文章，大家一定要去看一看。在微信公众号搜索并关注“吴墨希”，查看文章《没有病，请勿乱吃药》。

识别方法是：查看所谓的保健品包装上是否有“蓝色帽子”标志。然后，在网络监管平台查询该蓝帽标志对应的批准文号是否属于该产品。一个批号对应一个产品。

这也需要大家自己去查询。你需要具备这种能力，去查询这些产品对自己而言是否正规。

## 掌握知识而非仅靠工具

我发现大家在很多问题上都只有“技法”，没有“心法”。例如，经常说飞机杯可以用于延时锻炼。

大家听进去了，然后就什么也不管不顾，直接去买了一个飞机杯。买了之后，就自己在那里反复使用，却不去学习正确的使用方法。

他们只是很快速地想要解决问题，但不去学习基本的知识。然而，最关键的东西往往不在产品本身，而在于是否理解了相关知识。

如果你不具备这些知识，就不具备能力去辨别网络上爆炸的信息。如果你不具备一些知识，就无法有效指导自己的行为。

你有可能还会上当受骗，有可能还是不知道应该怎么做。正如一句话所说：“我懂得很多道理，但是我依然过不好我这一生。”如果你连道理都不懂，那就会越走越偏。

## 重新认识“早泄”

大家一定要注意一个问题：“早泄是一种病”这个说法，可能是全世界最大的谎言。

在我看来，早泄很多时候不是一种病，而是一种“关系病”。这个问题在之前就已经给大家讲过了。

所以，最大的坑其实是性知识匮乏所导致的。今天要跟大家强调：**知识才是最好的延时类喷剂**。

只有当你具备这些知识时，才能掌握很多有效信息，才能真正自己治疗好早泄问题，并学习到关于延时的一种技能。

最后，花一点时间快速梳理一下关于早泄的知识点。大家一定要认真听：到底早泄是什么？

我们发现，即使去医院，医生在做诊断时，标准也不统一。在同一个医院，不同的诊室、不同的医生都会有不同的标准来定义早泄。

*   有些医生判断早泄是进入阴道不足一分钟。
*   有些人认为是进入阴道不足30秒。
*   有些医生计算的是阴道内抽动不足15次，就会认为是早泄。
*   还有的医生判断标准是：如果时间没有让伴侣满意，那也可能叫早泄。

所以，从医学视角来看，对早泄问题的研究本身也并非高度一致。

而且，早泄问题在以前某个年代根本就没有成为这样的困扰，也没有成为一个临床问题。直到1984年，才逐渐开始被临床研究。

为什么早泄（时间短）以前不是问题，到了1984年（约30年前）我们才开始关注呢？这与我们性活动的目的息息相关。

早期的性活动目的是以生育为目的，性活动就是为了生孩子。后来我们有了情感需求，有了婚姻家庭的联结，就开始追求愉悦感。

追求愉悦感时，初期所有的性行为是服务于男性的，只考虑男人快乐就好。这是在男权思维意识下。

但现在，随着平权运动的兴起、女性高潮被证实，以及女性对高潮的期许和期待，我们现在更追求的是双方的愉悦感。我们对性生活有了更高的要求。

以前也跟大家讲过，通过性反应周期理论，我们能够看到：男生的性反应普遍快于女性。男生获得性高潮可能需要2到6分钟，但是女生获得性高潮可能要8分钟以上。

这就产生了一个时间差。正是因为此，男生才会对时间产生焦虑。也正是因为此，我们才会想要去延长自己的时间。

大多数男性想要延长自己的时间，其实是想要满足自己的伴侣，想让伴侣有更好的体验，而不是单纯地完成插入行为，更是想获得情感的升华。

所以，早泄问题应该是一种“关系病”，而不是真正器质上的病变问题。

举一个简单的例子：
*   如果你一分钟就能让伴侣达到高潮，伴侣一接触你就兴奋，你一进入伴侣瞬间高潮，你还会为自己一分钟感到苦恼吗？不会。
*   如果你30分钟都没让伴侣高潮，甚至每次累得筋疲力尽，都害怕性生活了，伴侣还是没有高潮，那你还算早泄吗？这也是值得大家思考的问题。
