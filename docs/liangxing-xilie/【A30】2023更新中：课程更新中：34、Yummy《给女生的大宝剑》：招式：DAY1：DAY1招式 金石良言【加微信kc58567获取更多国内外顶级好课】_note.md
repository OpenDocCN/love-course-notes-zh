# 恋爱技巧课程：34：给女生的大宝剑：DAY1招式 金石良言

在本节课中，我们将学习“金石良言”这一招式。该招式旨在通过一系列体贴的互动，在两人独处时营造放松与亲密的氛围，核心在于自然、关心的引导。

## 第一步：表达关心与放松引导

当你们两人在一起时，可以先与对方进行闲聊，关心对方的当前状态。

以下是具体的操作步骤：

*   询问对方是否感到疲惫。例如：“今天会累吗？”
*   引导对方躺在你的腿上，为其提供依靠。
*   在对方躺下后，再次轻声询问其感受：“怎么样？今天会累吗？”
*   开始为对方进行简单的头部放松，例如按摩太阳穴。

上一节我们介绍了如何开启话题并引导至放松状态，本节中我们来看看如何自然过渡到下一阶段。

## 第二步：提出邀请与转换场景

在完成头部按摩后，可以请对方坐起来。

接着，以认真且略带深意的眼神看着对方，提出邀请：“走吧，我们去洗澡吧。”

这个邀请需要语气自然，但眼神传递出明确的意图。

## 第三步：浴室中的互动

当对方接受邀请，一同进入浴室后，互动进入下一阶段。

以下是进入浴室后的步骤：

*   首先，主动为对方脱下衣服。
*   然后，在对方面前，自然地脱下自己的衣服。

本节课中我们一起学习了“金石良言”招式的完整流程：从通过关心和按摩建立初步的亲密感，到以认真的态度提出共同沐浴的邀请，最后在浴室中通过相互脱衣进一步拉近距离。整个过程强调自然、体贴的引导。
