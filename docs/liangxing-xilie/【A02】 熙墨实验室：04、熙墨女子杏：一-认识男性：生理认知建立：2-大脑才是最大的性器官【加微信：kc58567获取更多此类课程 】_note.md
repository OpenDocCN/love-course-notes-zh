## 身体的角色：神经系统的工作流程

理解了大脑的核心作用后，我们来看看身体在其中扮演什么角色。这部分知识可以回溯到高中生物关于神经系统的内容。

神经系统最基本的结构包括：**感受器 -> 传入神经 -> 神经中枢 -> 传出神经 -> 效应器**。这个过程描述起来可能有些抽象，但通过一个例子你就能明白。

3.  **大脑处理信号**：大脑接收到信号并进行分析，产生“舒服”或“讨厌”等感受。
4.  **信号传出并反应**：大脑将处理后的指令，通过传出神经传到身体相应部位。
5.  **效应器做出反应**：如果抚摸令人舒服，全身肌肉会放松，呼吸变得和缓；如果令人不适，则会肌肉僵硬并闪躲。
