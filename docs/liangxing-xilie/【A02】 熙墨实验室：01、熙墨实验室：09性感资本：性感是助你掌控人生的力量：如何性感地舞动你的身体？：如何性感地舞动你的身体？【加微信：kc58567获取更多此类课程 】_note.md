## 核心理念：为何人人皆可舞动？

上一节我们介绍了课程主题，本节中我们来看看其背后的核心理念。我知道你现在可能在质疑两件事。第一，既然是教跳舞的课程，怎么能只用语音和图片来讲。这不够直观。第二，我的专业是两性关系，并非舞蹈老师，学跳舞为何不直接找舞蹈老师。

这你们就没想到。其实我们今天的主题叫做“人人都能跳的贴身蜜舞”。主题的第一个关键词是“人人”。你不需要有任何舞蹈基础，不需要拥有蜜桃臀或马甲线，就可以很性感。我知道很多女生羡慕能跳诱惑舞蹈的舞者。我说那你就学几个简单的小动作，在你爱的人面前跳起来。他们马上说，我不会跳舞，动作难看怎么办？我对身材不自信，担心撩汉未遂反而尴尬。甚至有些有舞蹈基础的女生，一说到跳给恋人看就很害羞放不开，总想练到跟舞蹈老师差不多时才敢跳。

但遗憾的是，你永远不可能练得跟舞蹈老师差不多。人家是专业的，我们无法与专业人士相比。如果你非要跳到专业水平才敢展示，这舞你可能永远都没法跳。

好，我们来回顾一下熙墨实验室的性爱欲望理论。我们跳贴身蜜舞，就是为了激起对方的欲望。欲望是什么？还记得那个公式吗？

**欲望 = 新鲜感 + 信任感**

大家都看过美剧《绯闻女孩》。里面有一个情节，花花公子Chuck长期活跃于超模圈和脱衣舞酒吧，阅女无数。有一次在脱衣舞酒吧，Blair突然兴起上台与脱衣舞女郎共舞。Blair的舞技比专业舞者差很多，但Chuck那次却被她深深吸引并爱上了她。我年轻时看剧很不理解，为何她跳得差，Chuck仍会欣赏。后来随着对男性心理的研究，我意识到欲望与舞跳得好坏根本无关。他们需要的是“反差萌”，即制造强烈反差所带来的新鲜感。

然而，制造新鲜感的本质就是要迎难而上，去挑战那些对方认为不可思议的事。他也清楚你不可能比专业舞者跳得更好，但是你的舞蹈只为他而跳。只有他能够看到这个不一样的、多变的你。这种“独占”的属性，也就是信任感。

第二个关键词是“贴身蜜舞”。我们跳舞的目的不是为了表演，而是要加深跟对方的连接，增强前戏的趣味性。这是两个身体之间的沟通，也是两个人表达爱意的方式。跳舞是次要的，重要的是你们两人如何通过这个特别的前戏更好地进入状态、投入进去。

所以，虽然我今晚也会推荐一些简单的舞蹈动作，但比动作更重要的是“跳起来”和“跳入戏”。

## 日常练习与注意事项

我们将分两部分讲解贴身蜜舞。第一部分是日常练习与注意事项。第二部分将介绍三种不同类型的男士分别喜欢哪种风格的舞蹈。今天是第一部分。

以下是日常练习需要注意的八个要点。

### 1. 利用走路机会锻炼
我曾经在视频直播里说过，平时走路时要尽可能锻炼腰部和臀部的灵活性。站直身体，肩膀放松，头平视前方。走路时，双脚尽量保持在一条直线上。既然保持了一条直线，那就会自然而然地带动腰部和臀部左右扭动。这种扭动不光有锻炼效果，还可以消耗更多热量。比如同样是走一公里，这种走法可能比普通走法多消耗10%左右的热量。

### 2. 穿3到5厘米的低跟鞋
虽然穿高跟鞋更容易走出摇曳身姿的体态，但长期穿着对脚踝损伤较大。穿3到5厘米的低跟鞋，比完全的平底鞋对足部健康更有利。平底鞋脚后跟完全没有支撑，走久了反而更容易累。3到5厘米的鞋跟比较适合正常行走和练习动作。

### 3. 锻炼乐感
跟着音乐在跑步机上或公园里一边走动，一边感受音乐的节奏。大脑可以开始幻想：如果让我现在就着音乐编排一段舞，我会把它编成什么样子？

### 4. 多看视频获取灵感
课后我会推荐几个性感的舞蹈视频。并非让大家模仿里面的动作，因为你我都无法跳得和舞蹈演员一样。但我们可以从中获得灵感，挑出几个你认为最好看的动作，并把它简化成你自己能够做到的动作。简单点，再简单点，越简单就越能够练习得更加精湛。你并不是在表演，不需要制造舞台效果。你是在撩汉，简单的动作就足够性感。

学舞蹈需要练习基本功，但简单的贴身舞对基本功要求不高。平时有空可以练一下呼啦圈、扭腰、臀桥和卷腹，这些基本就够用了。

### 7. 善用道具
多用沙发、椅子、墙壁、杆子做道具。在道具上练习那种自己先从远处走来，慢慢坐在对方身上，用你的身体贴紧对方的身体，两个身体之间缓慢摩擦的感觉。
