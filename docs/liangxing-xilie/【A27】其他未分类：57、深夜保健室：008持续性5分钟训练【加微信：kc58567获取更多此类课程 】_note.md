# Python入门：01：变量与数据类型 📚

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是如何工作的，是编写任何程序的第一步。

## 什么是变量？ 🏷️

变量可以理解为一个**带标签的盒子**，用于在计算机内存中存储信息。你给这个盒子起一个名字（变量名），然后把数据（值）放进去。之后，你就可以通过这个名字来使用盒子里的数据。

上一节我们介绍了变量的比喻，本节中我们来看看如何创建和使用一个变量。

在Python中，创建一个变量并赋值非常简单，只需要使用等号 `=`。等号左边是变量名，右边是你想存储的值。

其基本语法是：
```python
变量名 = 值
```

以下是几个创建变量的例子：
*   `name = "小明"` - 创建了一个名为 `name` 的变量，里面存储了文本“小明”。
*   `age = 20` - 创建了一个名为 `age` 的变量，里面存储了数字20。
*   `is_student = True` - 创建了一个名为 `is_student` 的变量，里面存储了逻辑值“真”。

## 数据类型 📊

数据类型描述了变量中存储的是什么样的数据。就像盒子可以装水果、书本或工具一样，变量也可以存储不同类型的数据。Python有一些内置的基本数据类型。

上一节我们学会了如何把数据放进“变量盒子”，本节中我们来看看这些“数据”都有哪些种类。

了解数据类型很重要，因为它决定了你可以对这个数据做什么操作。例如，数字可以进行加减乘除，而文本通常不能。

以下是Python中最常见的几种基本数据类型：

1.  **整数**
    *   **描述**： 没有小数部分的数字，可以是正数、负数或零。
    *   **代码示例**： `my_score = 100`， `temperature = -5`

2.  **浮点数**
    *   **描述**： 带有小数点的数字，用于表示更精确的数值。
    *   **代码示例**： `pi = 3.14159`， `price = 19.99`

3.  **字符串**
    *   **描述**： 由一系列字符组成的文本数据。在Python中，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
    *   **代码示例**： `greeting = “你好，世界！”`， `author = ‘Alice’`

4.  **布尔值**
    *   **描述**： 逻辑值，只有两种可能：`True`（真）或 `False`（假）。通常用于条件判断。
    *   **代码示例**： `is_raining = True`， `has_key = False`

## 如何查看数据类型？ 🔍

有时你可能不确定一个变量的数据类型是什么。Python提供了一个非常方便的函数 `type()` 来查看。

其使用方法是：
```python
type(变量名或值)
```

以下是使用 `type()` 函数的例子：
*   `print(type(10))` 会输出 `<class ‘int’>`，表示10是整数类型。
*   `print(type(3.14))` 会输出 `<class ‘float’>`。
*   `name = “Python”`
    `print(type(name))` 会输出 `<class ‘str’>`（str是string的缩写）。

## 总结 🎯

本节课中我们一起学习了Python编程的基石：**变量**和**数据类型**。
*   我们知道了**变量**就像一个命名的存储盒，用 `=` 来赋值。
*   我们认识了四种基本**数据类型**：用于整数的`int`，用于小数的`float`，用于文本的`str`，以及用于逻辑判断的`bool`。
*   我们还学会了使用 `type()` 函数来检查任何值或变量的数据类型。

理解这些概念后，你就可以开始用变量存储信息，并根据数据的类型进行相应的操作了。下一节课，我们将学习如何让这些数据之间进行“交流”和“运算”。
