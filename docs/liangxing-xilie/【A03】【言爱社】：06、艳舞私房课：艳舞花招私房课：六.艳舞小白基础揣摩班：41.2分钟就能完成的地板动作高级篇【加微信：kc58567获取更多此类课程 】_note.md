# 艳舞私房课：06：2分钟高级地板动作教程

在本节课中，我们将学习一个高级地板动作。这个动作需要一定的身体控制力与灵活性，请确保在充分热身且身体健康的情况下谨慎练习。

上一节我们介绍了基础的地板动作控制，本节中我们来看看一个更具挑战性的肩部翻滚动作。

## 🎼 动作概述与安全须知

这是一个高级地板动作。**仅建议无健康问题、无背部疼痛且已完成热身活动的学员尝试**。

请注意观察示范者Julia在此动作中**没有弯曲她的颈部**。动作的诀窍在于**用肩膀翻滚，而非用颈部**。她的髋部略微向侧面转动，以协助完成肩部翻滚。

**切勿将双腿直接向后甩**，这可能导致颈部受伤。我们建议对此动作格外小心，仅在对自己柔韧性有信心的情况下进行练习。

## 🔑 核心动作分解

以下是完成该动作的关键步骤与要点。

1.  **起始姿势**：从坐姿或跪姿开始，确保背部挺直，核心收紧。
2.  **肩部引导**：向一侧倾斜身体，将重心移至肩膀，而非头部或颈部。**关键公式：发力点 = 肩部**。
3.  **髋部辅助**：转动髋部，使其朝向侧面，为身体翻滚提供动力和方向。**代码逻辑：hip.rotation = sideways**。
4.  **腿部控制**：有控制地带动双腿，避免利用惯性猛然甩动。
5.  **完成翻滚**：依靠肩部与核心力量，流畅地完成整个翻滚动作，回到初始或下一个姿势。

## 💡 练习建议与过渡

在掌握了上述分解要点后，你可以尝试将动作连贯起来。从慢速分解练习开始，逐步加快速度，并确保每个阶段都对身体有良好的控制。

本节课中我们一起学习了一个强调**肩部发力**与**髋部协调**的高级地板翻滚技巧。记住，安全永远是第一位的，务必在能力范围内循序渐进地练习。熟练掌握此动作将极大地增强你在地板舞段中的表现力与流畅度。
