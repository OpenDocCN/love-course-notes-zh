# XO初级训练自学课：02：XO挤与空中单车 🚴♀️💪

在本节课中，我们将学习两个核心训练动作：“空中单车”与“XO挤”。这两个动作旨在协同锻炼你的腰臀、大腿及盆底肌群，提升肌肉力量、耐力和控制力。课程包含动作分解与耐力训练，请跟随指引练习。

## 第一部分：空中单车动作详解 🚴♀️

上一节我们介绍了课程目标，本节中我们来看看“空中单车”的具体做法。这是一个模拟踩单车的动作，重点在于用腰臀发力，而不仅仅是腿部。

**动作要领**：
1.  仰卧，抬起腰臀。
2.  左腿向前迈出，在空中划一个大圈。
3.  左腿落地时，配合“O型夹”（盆底肌向内收紧）。
4.  换右腿向前迈出划圈，用后臀和腰部肌肉发力。
5.  右腿落地时，配合“X型挤”（盆底肌向外推）。
6.  交替进行，逐渐加快速度。

**练习提示**：
*   动作初期应放慢速度，专注于划出最大的弧度。
*   感受腰臀和盆底肌的协同发力。
*   保持身体平衡，避免只用腿部力量。

完成一组后，如果感到腰臀酸胀是正常的。以下是放松步骤：
1.  停止动作，休息片刻。
2.  站起来活动身体，轻轻甩动腰臀。
3.  用双手拍打腰臀和大腿肌肉，帮助放松。
4.  进行深呼吸，休息一分钟。
*建议今天至少循环练习三遍。*

## 第二部分：XO挤耐力训练 💪

完成了动态的“空中单车”，我们进入静态的“XO挤”耐力训练。本节重点在于长时间保持肌肉收缩状态，以激活并强化盆底肌。

**X型挤（向外推）**：
1.  从阴道口开始，想象将其慢慢张开。
2.  将发力点扩展到G点附近的肌肉，继续向外推。
3.  再将力量延伸到A点、B点（宫颈口附近）。
4.  最后延伸到耻骨区域。
5.  整体感觉像有一股向外推开发力的力量。
*保持这个收缩状态，即使肌肉发抖或难以坚持也属正常。*

**O型夹（向内收）**：
1.  从阴道口开始，向内收紧，仿佛要关闭通道。
2.  将收紧感传递到G点。
3.  再延伸到A点、B点。
4.  最后到达耻骨。
5.  用盆底肌的力量向内挤压，仿佛挤压阴蒂。
*练习时，可以稍微并拢双腿，脚尖内扣成内八字，用大腿内侧力量辅助盆底肌发力。*

**训练循环与休息**：
以下是训练步骤：
1.  执行“X型挤”，尽可能长时间保持。
2.  切换到“O型夹”，同样长时间保持。
3.  完全放松所有肌肉，进行深呼吸调整。
4.  重复上述循环。
*本次是耐力训练，不以快速切换为目标，每次收缩需坚持较长时间。能坚持完成非常棒。*

## 课程总结 📝

本节课中我们一起学习了“空中单车”和“XO挤”两个核心训练。
*   “空中单车”侧重于腰臀的动态协调与力量。
*   “XO挤”侧重于盆底肌的静态耐力与控制。
请确保按照指引完成练习组数与放松，持之以恒才能收获效果。加油！
