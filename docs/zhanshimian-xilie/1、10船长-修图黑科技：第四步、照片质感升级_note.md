# 10船长-修图黑科技：1.4：照片质感升级

在本节课中，我们将学习如何使用VSCO软件对照片进行光线和质感的精细调整。我们将重点介绍曝光、对比度、饱和度、色调、白平衡、锐化、清晰度和颗粒感等核心参数的调节方法，旨在让照片看起来更干净、清晰且富有质感。

上一节我们介绍了初步的光线调节，本节中我们来看看如何利用VSCO进行更深层次的照片质感升级。

## 软件与准备

我们使用的软件是VSCO。请打开软件，并导入上一节课中已经过初步光线调整的照片作为操作对象。

VSCO提供了丰富的滤镜和调光功能。其中，HSL（色相、饱和度、明度）功能需要VIP才能使用，但本教程不涉及此功能。我们主要利用VSCO进行光线升级，因此不建议使用其滤镜。当然，在完成光线调整后，你可以自行尝试使用滤镜。

## 核心调整参数与步骤

以下是本教程将用到的VSCO核心调整功能列表：

1.  曝光
2.  对比度
3.  锐化
4.  清晰度
5.  饱和度
6.  色调（高光/阴影）
7.  白平衡（色温/色调）
8.  肤色
9.  颗粒

所有参数的调整都需遵循一个基本原则：**调整幅度宜小不宜大，多数参数调整范围建议控制在±1以内**，以避免照片失真。

### 1. 曝光与对比度

观察照片，可能会发现整体有些暗沉，对比度略低。这是上一步调整后的常见情况。

首先调整曝光，让照片稍微变亮。操作时只需微调。

**代码示例（调整逻辑）：**
`曝光 += 0.2  // 具体数值根据照片情况决定，通常不超过1`

增加曝光后，照片可能显得有点“发灰”。这时需要增加一点对比度，让画面更清晰、真实。

**代码示例（调整逻辑）：**
`对比度 += 0.4  // 具体数值根据照片情况决定，通常不超过1`

### 2. 饱和度

饱和度控制色彩的鲜艳程度。如果觉得照片色彩过于浓艳或肤色偏黄，可以适当降低饱和度。

**代码示例（调整逻辑）：**
`饱和度 -= 0.5  // 降低饱和度，使色彩更柔和`

### 3. 色调（高光与阴影）

“色调”下的“高光”功能用于压暗照片中过亮的区域（如灯光、反光）。向右滑动是减淡高光。

**代码示例（调整逻辑）：**
`高光减弱 += 0.8  // 压暗高光区域，数值通常不超过1.5`

“阴影”参数在本教程中建议保持不动，调整它容易影响整体照片质感。

### 4. 白平衡与肤色

白平衡包含“色温”和“色调”。
*   **色温**：向左（冷调）让照片偏蓝，向右（暖调）让照片偏黄。可根据环境光调整，使照片色调更自然。
    `色温 -= 0.2  // 减少暖调，让照片更清爽`
*   **色调**：向左让照片偏绿，向右让照片偏红。微调增加红色可以让人物肤色看起来更红润。
    `色调 += 0.2  // 轻微增加红色调`

“肤色”功能可以单独调节皮肤颜色走向（偏红或偏黄）。同样进行微调，让肤色更健康自然。
`肤色 -= 0.5  // 向偏红方向微调`

### 5. 锐化与清晰度

现在回到之前跳过的“锐化”和“清晰度”。
*   **锐化**：增强物体边缘的清晰度，使照片看起来更锐利。但过度锐化会产生不自然的白边。
    `锐化 += 2  // 通常不超过3`
*   **清晰度**：通过增加局部对比度来让照片细节更突出、更立体。但过度调整会使照片显得脏乱。
    `清晰度 += 1  // 通常不超过1.5`

### 6. 颗粒感

最后，添加“颗粒”可以为照片增加一种胶片质感，提升整体格调。颗粒感不宜过重。

**代码示例（调整逻辑）：**
`颗粒 += 1  // 通常不超过2，轻微添加即可`

## 效果对比与总结

调整完成后，请务必对比调整前后的照片。你会发现，经过VSCO精细调整的照片在光线、清晰度和整体质感上都有显著提升：光线更均匀，画面更干净，人物更清晰，并且拥有了细腻的质感。

本节课中我们一起学习了使用VSCO进行照片质感升级的全流程。核心在于理解每个参数的作用，并遵循“微量多次、反复对比”的原则进行调节。记住，参数值没有固定标准，一切以你眼睛看到的最终效果为准。通过练习，你将能更熟练地运用这些工具，让照片焕然一新。
