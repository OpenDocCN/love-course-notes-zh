**摄影入门：01：展示面图片基础**

在本节课中，我们将学习展示面图片的基础概念，包括其定义、核心要素以及如何选择和处理图片。

展示面图片是指在社交媒体等平台上，用于展示个人形象、生活状态或兴趣爱好的照片。一张好的展示面图片能有效传递信息，吸引他人注意。

上一节我们介绍了展示面图片的定义，本节中我们来看看构成一张合格展示面图片的核心要素。

以下是展示面图片应具备的几个关键点：

*   **高清晰度**：图片应清晰，避免模糊或像素化。这通常意味着使用高分辨率的原始图像。
*   **良好构图**：遵循基本的构图法则，如三分法，将主体置于视觉焦点。
*   **适当光线**：光线应充足且自然，避免过暗或过曝。公式可表示为：`理想曝光 = 充足光源 + 正确测光`。
*   **主题明确**：图片应有明确的主题或故事性，避免杂乱无章。
*   **真实自然**：图片应反映真实状态，过度修饰可能适得其反。

理解了核心要素后，接下来我们探讨如何为你的展示面选择和处理图片。

选择图片时，应优先选择那些能体现你积极生活状态、兴趣爱好或专业领域的照片。处理图片时，可以进行适当的裁剪、调色和锐化，但需保持自然。例如，使用代码式描述调整流程：`原始图片 -> 裁剪构图 -> 调整亮度/对比度 -> 微调色彩 -> 导出成品`。

本节课中我们一起学习了展示面图片的基础知识。我们明确了其定义，分析了高清晰度、良好构图等核心要素，并简要了解了选择与处理图片的基本原则。掌握这些是打造优质个人展示面的第一步。
