# 老吴《六节课从素人到达人》：1.4：快捷修图法 🚀

在本节课中，我们将学习老吴独创的快捷修图法。这套方法旨在通过一系列高效、标准化的步骤，帮助初学者快速掌握修图核心技巧，提升照片质感。

上一节我们了解了修图的基本思路，本节中我们来看看如何将这些思路转化为具体、可操作的快捷步骤。

## 核心原理

快捷修图法的核心在于建立标准化流程。通过固定的步骤处理常见问题，可以大幅提升修图效率与效果的一致性。其核心流程可概括为以下公式：

![](img/1b78107df71f449dbb7b8ed69c66db44_0.png)

**快捷修图流程 = 基础校正 + 风格化调整 + 细节优化**

---

## 操作步骤详解

以下是快捷修图法的具体操作步骤，我们将按照流程逐一讲解。

### 第一步：基础校正

基础校正是修图的基石，目标是解决照片的曝光、色彩和构图等基本问题。

![](img/1b78107df71f449dbb7b8ed69c66db44_1.png)

1.  **校正曝光与对比度**：使用色阶或曲线工具，确保照片高光不过曝、阴影有细节。
2.  **调整白平衡**：纠正照片色偏，使白色物体呈现为纯白色，色彩还原准确。
3.  **裁剪与构图**：根据三分法或引导线等构图原则，裁剪掉多余部分，突出主体。

### 第二步：风格化调整

在基础校正之后，进行风格化调整可以为照片注入个性与情绪。

1.  **色彩分级**：分别对高光、中间调和阴影区域添加不同的色彩倾向，例如高光加暖、阴影加冷。
2.  **应用预设/滤镜**：可以套用成熟的滤镜预设，然后根据当前照片微调参数，快速确定风格基调。
3.  **局部调整**：使用渐变滤镜、径向滤镜或调整画笔，对特定区域（如天空、人脸）进行强化处理。

### 第三步：细节优化

![](img/1b78107df71f449dbb7b8ed69c66db44_2.png)

最后一步是进行精细打磨，提升照片的质感与完成度。

1.  **锐化与降噪**：适度锐化以增强细节清晰度，同时进行降噪以保持画面干净。
2.  **修复瑕疵**：使用修复画笔或克隆图章工具，去除传感器灰尘、人物皮肤上的明显瑕疵等。
3.  **最终输出检查**：将视图比例缩放到100%，仔细检查全图细节，确认无误后导出为最终文件。

## 技巧总结

本节课中我们一起学习了老吴的快捷修图法。我们首先理解了**建立标准化流程**的核心原理，然后逐步演练了从**基础校正**、**风格化调整**到**细节优化**的完整三步法。掌握这套方法，能帮助你有条不紊地处理大多数照片，高效地实现从“素人”到“达人”的修图质量飞跃。

> 记住：熟练源于重复。将这套流程应用于不同照片进行练习，是掌握它的最佳途径。
