# 老吴《六节课从素人到达人》：05：如何轻松carry朋友圈 且看八大类修图实操案例

在本节课中，我们将学习如何通过八大类修图实操案例，轻松提升朋友圈照片的质感与吸引力。我们将从基础概念入手，逐步深入到具体的修图技巧。

上一节我们介绍了修图的基本思路，本节中我们来看看具体的实操案例分类。

以下是八大类修图实操案例的详细列表：

1.  **风景类**：调整天空的蓝色与饱和度，突出云层细节。核心操作是使用**HSL工具**单独调整蓝色与青色。
2.  **人像类**：进行皮肤磨皮与美白，消除瑕疵。常用**修复画笔工具**和**曲线工具**提亮肤色。
3.  **美食类**：增强食物的色泽与对比度，使其看起来更诱人。重点是调整**色温**偏暖，并提升**清晰度**。
4.  **静物类**（如产品）：打造干净、有质感的背景，突出主体。通常使用**径向滤镜**局部提亮主体。
5.  **夜景类**：降低画面噪点，提升暗部细节，强化灯光效果。关键步骤是使用**降噪**功能并拉亮**阴影**。
6.  **宠物类**：提亮眼睛，增强毛发质感，让宠物更灵动。可使用**局部调整画笔**涂抹眼睛区域。
7.  **建筑类**：校正镜头畸变，让线条横平竖直，体现结构感。主要依赖**变换工具**中的垂直与水平校正。
8.  **黑白风格类**：通过调整不同颜色的明度，创造有层次感的高对比黑白照片。核心是使用**黑白混合**面板。

每个类别都有其侧重的调整方向，掌握这些核心思路能让你事半功倍。

在具体操作时，你可以遵循一个通用流程：先进行基础曝光和色彩校正，再根据照片所属类别进行针对性调整，最后进行锐化或特效输出。

本节课中我们一起学习了朋友圈修图的八大核心案例类别及其对应的关键修图技巧。掌握这些分类和工具，你便能更有针对性地处理各类照片，从而轻松提升朋友圈的视觉效果。
