# 手机摄影教程：第05课：用手机做后期：课时2 · Snapseed局部处理

![](img/9d36ff267302be46af507f877e24ca4e_1.png)

在本节课中，我们将学习如何使用Snapseed进行局部处理，包括画笔工具、修复工具、局部调整、变形和晕影功能。这些工具能帮助你精确地调整照片的特定区域，弥补拍摄时的不足，提升照片的整体效果。

## 画笔工具：局部曝光补偿 🖌️

上一节我们介绍了Snapseed的基础调整功能，本节中我们来看看如何对照片的局部进行精细处理。首先，我们打开一张逆光拍摄的海鸥照片。这张照片整体曝光不足，显得有些暗，这是为了在拍摄时保证主体清晰度和光影效果而有意为之。

现在，我们学习如何调整它。进入Snapseed后，选择“工具”菜单。今天我们将使用“调整图片”、“裁剪”、“变形”以及“画笔”功能。“变形”工具用于校正照片的水平或垂直角度偏差。

“画笔”是一个功能强大的局部调整工具。它的作用相当于后期处理中的曝光补偿（EV），可以对照片的特定区域进行增加或减少曝光的操作。

以下是使用画笔工具调整局部曝光的步骤：

1.  点击“画笔”工具。
2.  选择“加光减光”模式。屏幕上会出现“+”、“-”符号，代表增加或减少曝光。
3.  将曝光值滑块调整到+0.7。
4.  用手指在照片中较暗的区域（例如海鸥的黑色羽毛部位）轻轻涂抹。
5.  被涂抹的区域亮度会立即提升，暗部细节变得更加明显。

如果效果过强，可以重复操作并降低曝光值，例如设置为+0.3，进行更精细的调整。通过这种方式，我们可以有针对性地提亮翅膀等阴影部分的细节，弥补拍摄时曝光不足的问题。

## 修复工具：移除画面干扰物 ✨

处理完曝光问题后，我们可能会发现画面中存在一些不必要的干扰物。这时可以使用“修复”工具。这个功能类似于Photoshop中的污点修复工具，可以轻松移除画面中的小物体或瑕疵。

例如，照片水面上有几只小海鸥干扰了主体。我们可以使用修复工具将它们移除。

以下是使用修复工具移除物体的步骤：

1.  点击“修复”工具。
2.  将图片放大到需要处理的区域。
3.  用手指轻轻涂抹想要移除的小海鸥。
4.  软件会自动计算并填充背景，几乎不留痕迹。

对比处理前后的照片，可以看到干扰物被干净地移除，画面变得更加简洁。这是一个非常实用的污点修复功能。

## 局部调整：精确控制点参数 🎯

“局部”调整功能比画笔工具更为精确。它允许你对照片中某个具体的点进行调整，控制其亮度、对比度和饱和度。

我们来看看如何使用它。在工具菜单中选择“局部”功能。点击左下角的“+”号，然后在照片的暗部区域点一下，屏幕上会出现一个控制点。

以下是使用局部调整功能的步骤：

1.  按住控制点并上下滑动屏幕，可以在“亮度”、“对比度”、“饱和度”参数间切换。
2.  选择“饱和度”后，左右滑动手指即可增加或减少该点的饱和度。
3.  一个强大的功能是：长按控制点，选择“拷贝”设置。
4.  在另一个需要相同调整的位置点击“+”号，选择“粘贴”，即可快速应用相同的调整参数。
5.  你还可以使用双指缩放手势，像放大镜一样查看局部调整的细节效果。

与画笔的涂抹方式不同，“局部”调整通过控制点实现更精准的微调。例如，我们可以单独提高海鸥翅膀受光面的暖黄色饱和度，使其在逆光环境下更加突出。

## 其他实用工具：变形与晕影 🔧

除了上述核心功能，Snapseed还有其他实用的局部处理工具。

**变形工具**：用于校正照片的透视和水平线。当你拍摄的建筑或风景照片出现倾斜时，可以使用此工具进行水平或垂直方向的校正，让画面更端正。

**晕影工具**：用于创建或调整暗角效果。你可以控制暗角区域的大小、内部亮度和外部亮度。喜欢暗角效果的用户可以用它来突出照片中心的主体，营造氛围。

工具菜单下的“镜头模糊”、“魅力光晕”等功能也值得尝试，它们能快速营造特殊的视觉效果。而“滤镜”菜单则提供了一系列具有固定风格的滤镜，大家可以自行点选预览效果。
