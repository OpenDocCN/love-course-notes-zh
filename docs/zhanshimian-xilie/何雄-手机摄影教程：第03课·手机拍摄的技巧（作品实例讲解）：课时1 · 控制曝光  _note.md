在本节课中，我们将通过具体的作品实例，学习手机摄影中控制曝光的核心技巧。曝光决定了照片的明暗，是影响画面效果的关键因素。

首先，我们分享一张逆光拍摄的照片。在逆光环境下，如果测光点选择在最亮的天空部分，会导致暗部区域（如海鸥和水面）变得非常黑，失去细节。

为了应对这种情况，我采用了增加曝光补偿的手法。具体操作是，将手机屏幕上的测光点（通常是一个小方框或圆圈）对准画面中较暗的海鸥或水面区域。这会导致相机认为环境较暗，从而自动提高整体亮度，使暗部细节得以清晰呈现。虽然天空会因此过曝（失去细节），但当时云层本身并不出彩，所以这个取舍是值得的。这个技巧可以总结为：**对暗部测光，提亮整体**。

以下是这种“加曝光”手法的要点：
*   适用场景：逆光拍摄，希望保留前景或主体暗部细节时。
*   操作方法：在手机拍摄界面，点击屏幕中较暗的区域进行对焦和测光，并向上滑动屏幕（或使用曝光补偿滑块）增加曝光值。
*   效果：暗部变亮、细节可见，亮部可能过曝。

上一节我们介绍了通过增加曝光来提亮暗部的方法，本节中我们来看看相反的情况。

第二张照片展示的是“减曝光”的拍摄手法。与上一张相比，这张照片拍摄时光线强烈，天空的云层细节非常丰富。

为了压暗天空，突出云层的质感和层次，我进行了减曝光操作。同时，因为光线充足，海鸥白色的翅膀呈现出通透、明亮的质感。这个技巧可以总结为：**对亮部测光，压暗整体**。

以下是这种“减曝光”手法的要点：
*   适用场景：顺光或光线强烈时，希望保留天空、高光区域细节，或创造深沉影调时。
*   操作方法：在手机拍摄界面，点击屏幕中较亮的区域（如天空）进行对焦和测光，并向下滑动屏幕（或使用曝光补偿滑块）减少曝光值。
*   效果：亮部细节得到保留，画面整体影调更沉稳，色彩可能更浓郁。

接下来，我们看一张在下午顺光条件下拍摄的照片。顺光拍摄时，光线从正面照射，能很好地展现物体的色彩和形态。

这张照片的曝光控制介于“减曝”和“加曝”之间，属于正常的曝光手法。测光点选择在远处房屋被夕阳照亮的光影部分，从而获得了一个整体均衡的曝光效果。这提醒我们，**根据画面中最重要的部分进行测光**，是获得理想曝光的基础。

最后，我们分析一张利用闪光灯补光的作品。不要小看手机的内置闪光灯，在特定情况下它能发挥关键作用。

这张人物特写拍摄时，由于头饰遮挡，在面部形成了难看的阴影。为了消除阴影，让面部光线均匀、细节清晰，我开启了手机的闪光灯进行近距离拍摄。这不仅能消除阴影，还能为眼睛带来明亮的眼神光，并增强皮肤的质感，获得更戏剧化的效果。

以下是使用闪光灯补光的要点：
*   适用场景：近距离拍摄，主体面部或细节处于阴影中时。
*   操作方法：在手机拍摄设置中开启闪光灯（通常为“强制开启”模式）。
*   效果：消除阴影，提亮前景，增强细节和质感。

本节课中，我们一起学习了四种控制曝光的核心技巧：
1.  **加曝光**：对暗部测光，用于逆光时提亮前景。
2.  **减曝光**：对亮部测光，用于保留高光细节或创造氛围。
3.  **正常曝光**：对画面关键中间调区域测光，获得均衡效果。
4.  **闪光灯补光**：强制开启闪光，用于近距离消除阴影、补充光线。

![](img/34f56b734cd79d17cea4262d251ccc10_6.png)

记住，曝光没有绝对的正确，关键在于它是否服务于你想表达的主题和情感。多尝试不同的测光点与曝光补偿组合，是掌握手机摄影曝光控制的最佳途径。
