# 木西-用普通手机拍出专业级照片（完结）：07：拍摄人像与静物

在本节课中，我们将学习生活中最广泛的手机摄影题材：人像与静物（美食）摄影。我们将以实战形式，学习如何选择地点、运用光线、与模特沟通、正确摆盘以及后期处理，最终拍出专业级的照片。

## 课程回顾与概述

上一节我们以城市风光为例，学习了如何在户外拍摄并后期处理风光作品。本节中，我们将聚焦于人像和静物这两个核心题材。

人像摄影涉及拍摄人物，例如朋友或伴侣。静物摄影则以美食为代表。本课程将通过实战演示，涵盖从前期拍摄到后期处理的完整流程。

## 人像摄影实战：室内餐厅场景

在拍摄人像时，餐厅、咖啡厅等室内空间是常见场景。这些空间通常较小，客人较多，因此我们主要拍摄半身或大半身人像。在这种限制下，核心是**利用好环境光线**来展现人物最美的一面。

我们需要带着模特在环境中寻找最适合的光线位置。餐厅光线复杂，可能包含无方向的白炽灯和有方向的射灯，找到合适的位置至关重要。

### 拍摄前的沟通

与模特的沟通是第一步。即使是专业摄影师或相处多年的伴侣，也可能不完全了解对方最喜欢的拍摄角度和状态。

以下是沟通的关键点：
*   了解模特认为自己最美的角度和状态。
*   询问其喜欢的衣着搭配和希望展现的气质（如高冷或热情）。
*   确认其偏好，例如是否喜欢看镜头或侧脸。

一个小技巧是让模特用你的手机自拍，观察她喜欢的角度和效果，然后你可以参照进行拍摄，事半功倍。

### 光线的运用与观察

光线有两个关键特性：**强度/方向**和**色温**。

*   **色温影响**：窗外光线偏冷（冷色光），能使皮肤更白，气质显得冷酷；餐厅内部光线偏暖（暖色光），使人显得热情。
*   **强度影响**：朝向室内强光侧，脸部明暗对比强；朝向窗外弱光侧，光线柔和散射，皮肤显得更白皙。

你需要根据场景光线和与模特的沟通，决定最终使用的光线效果。

### 特殊用光技巧：蝴蝶光

在拍摄半身人像时，可以利用一种特殊光线——**蝴蝶光**。当灯光从人物正上方向下照射时，会在脸颊两侧产生阴影，鼻梁下方形成蝴蝶形状的高光。

**公式：顶光 → 脸颊阴影 + 鼻下高光 = 脸部视觉变瘦**

这种光线能增强脸部立体感，让人物看起来更瘦。

### 利用背景与手机功能

许多手机具备人像模式或背景虚化功能，能模拟大光圈效果，让人像更接近单反相机质感。

这时，我们需要主动寻找适合虚化的背景，例如餐厅中灯光密集、能形成漂亮光斑的区域。关键在于多尝试，并有意识地去寻找这类背景。

## 人像摄影实战：户外场景

来到户外，人物可以与更大的建筑背景互动，展现更多体态。有以下几个要点需要注意。

### 构图与线条

确保人物的头部或身体主要部位不要与背景中的线条（如地平线、建筑护栏）交叉或贯穿，这会影响画面美感。

### 衣着与背景的搭配

正确选择人物衣着与背景颜色的关系。例如，如果模特穿着黑色衣服，应尽量避免以黑色或深色为背景，否则人物轮廓无法突出。选择蓝天或浅色建筑作为背景，能更好地衬托出衣着和体态。

**核心原则：衣着颜色与背景形成对比，以突出主体。**

### 弱光环境下的补光

当天色渐晚，手机在暗光下拍摄能力有限。此时，可以请同伴用另一部手机打开手电筒功能进行补光。

**对比：有补光 vs 无补光，画面效果差异巨大。**

### 拍摄角度与透视

*   **拍摄全身人像时**：摄影师放低机位，从下往上拍，可以使模特的腿显得更长。
*   **拍摄半身人像时**：摄影师提高机位，从上往下拍，可以使脸的下半部分（如下巴）显得更小，人物更显年轻。

### 与环境互动

注意观察环境中的特色元素并加以利用。例如，利用建筑中漂亮的曲线作为构图元素，与人物结合，可以拍出既干净又富有艺术感的人像照片。

### 利用复杂环境光

城市环境中常有各种颜色、色温、方向的光源。例如，街头随机的射灯可以作为造型光，强烈照亮人物脸部，通过明暗对比突出脸部线条，让脸部更立体或更瘦。

## 静物（美食）摄影实战

现在我们来到咖啡厅，这是拍摄静物和美食的常见场景。拍摄美食与室内拍人像有相通之处，都需要精心把握光线和构图。

### 核心：观察与利用光线

摄影是用光的艺术。在咖啡厅，我们需要观察光线的颜色、聚散，并找到最适合的拍摄角度。

通常，环境中有一盏最亮的灯作为**主光源**。主光源的方向可以通过影子的方向来判断（影子方向与光源相反）。

如果不顾光源方向直接拍摄，手机或人的影子会投射在食物上，影响画面。

**解决方法**：绕开主光源，从侧面进行拍摄，这样可以避免影子出现在食物上。

### 食物的摆放与构图

与人像不同，食物体积小且可以随意摆弄。因此，我们可以对食物进行更精致的摆放和调整。

以下是基本步骤：
1.  **清理画面**：移走无关的杂物，如凌乱的纸巾、不必要的餐具。
2.  **重新摆盘**：将餐具、食物摆放到美观的位置。陪体（如一块巧克力）可以放在合适角度点缀画面。
3.  **结合光线**：在避开主光源影子的角度进行拍摄。

**两个基本技巧**：
1.  找到并避开主光源，防止影子破坏画面。
2.  主动对食物进行摆盘和构图，让画面更精致。

### 创造性补光

有时无论如何调整角度，都难以完全避免阴影。此时，可以创造性地使用另一部手机的**手电筒功能**进行补光。

**操作要点**：
*   用补光手机的手电筒照亮阴影区域。
*   补光距离要合适：太近光线太硬，容易过曝或产生新阴影；太远效果不足。需要不断调整距离和角度，找到光线柔和自然的位置。
*   **代码/流程模拟**：
    ```python
    if 画面存在顽固阴影:
        打开另一部手机的补光灯
        while 补光效果不理想:
            调整补光手机的距离和角度
            测试拍摄效果
    ```

## 静物摄影后期处理

完成拍摄后，后期处理必不可少。静物后期相对于人像更简单，可塑性更强。我们以 Snapseed 为例进行演示。

后期核心是：**首先符合人眼观察习惯，其次表达个性化风格。**

### 后期处理步骤

1.  **调整图片（基础调整）**
    *   **亮度**：根据直方图（大部分像素偏左表示偏暗），适当提高亮度，让曝光正常。注意避免最右侧过曝。
        *   **公式**：`调整后亮度 = 原亮度 + Δ（适量增量）`
    *   **对比度**：增加对比度，让画面更通透。
    *   **饱和度**：谨慎增加，避免色彩失真。如需突出特定食物（如抹茶蛋糕），可考虑后续使用蒙版单独调整。
    *   **氛围**：可加可减。本例中略微降低氛围，能使白色盘子和蛋糕更突出。
    *   **高光**：控制盘子等高光区域的亮度，保留细微反光以体现材质（如陶瓷质感）。
    *   **阴影**：降低阴影可以增强画面对比度和物体立体感（如叉子的金属质感）。

2.  **突出细节**
    *   **结构**：降低结构可以使画面局部细节柔和，让食物看起来更干净、亲切。但降低结构可能导致画面变模糊。
    *   **锐度**：在降低结构后，增加锐度可以恢复画面的整体清晰度。
        *   **组合技巧**：`降低结构 + 增加锐度 = 画面干净且清晰`

3.  **旋转与裁剪**
    *   调整角度，使画面中的参考线（如灯的倒影）垂直，让画面更平稳。

4.  **视角调整**
    *   使用视角工具轻微调整透视。例如，让画面下半部分增大，会产生叉子变大、盘子远去的纵深感；让上半部分增大，则感觉食物离得更近。根据喜好调整，让透视更符合人眼观察。

5.  **修复**
    *   使用修复工具点除画面中的污点、杂点，使桌面和食物看起来更干净。

6.  **色调对比度**
    *   分别调整高调、中调、低调的对比度，可以增强不同亮度区域的立体感。本例中，适当降低高调和中调对比度，让盘子和蛋糕的细节更柔和。

7.  **魅力光晕**
    *   添加光晕可以使明暗过渡更柔和，画面氛围更温馨。调整光晕强度、色温（暖色调/冷色调）和饱和度，达到舒适自然的效果。
        *   **效果**：硬朗过渡 → 柔和过渡，食物质感更显柔软可口。

**后期流程总结**：基础调整 → 突出细节（降结构、加锐度）→ 旋转校正 → 视角微调 → 修复瑕疵 → 色调对比度增强质感 → 魅力光晕营造氛围 → 保存。
