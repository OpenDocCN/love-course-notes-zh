# 装逼的艺术：1：课程导论与核心概念 🚀

在本节课中，我们将学习“装逼”这一概念的核心定义，并了解本课程的整体框架与学习目标。

如今社会，人们通过互联网等渠道结识他人。互联网本身成就了个人影响力，其中最高表现形式就是“网红”。

从早期的凤姐到现在的各类网红，他们都具备非常出色的技能：拍得了照片，修得了图片，写得了段子，装得起高冷。他们都是把魅撩汉的大神，逼格满满。正因如此，我们诞生了这门装逼课程。

说到装逼，首先要提到的一个词就是“逼格”。更专业的解读是：**逼格 = 装逼的档次**。但“档次”这个词本身就很装逼。人类这种生物很喜欢划分等级。

因为有了等级划分，就产生了抬高自己和贬低别人的可能性。装逼者首先要做的事情，就是将自己与大众划分为不同的等级，并且把自己的等级抬高，以凸显优越感，满足虚荣心。

逼格指的就是不同的等级。逼格越高，说明你处在的装逼食物链层次就越高。能够凭借与低等级的人对比，从而获得一种优越感。

总结成一句话：**追求高逼格是装逼的核心**。

具体到社交（例如把妹）场景，所有用到的逼格可以分为两大类。

以下是外在的逼格体现：

*   **个人形象**：男生最主要的是发型、胡子、肢体语言。
*   **穿衣风格**：包括服饰与饰品。
*   **所处环境**：身处的背景与环境。

以下是内在的逼格体现：

*   **谈吐**：说话的方式与内容。
*   **气质**：由内而外散发的个人特质。
*   **思维方式**：思考与解决问题的角度。

我们的课程将会细分以上所有要点。

上一节我们定义了装逼与逼格，本节中我们来看看课程的具体内容安排。课程将从个人形象、穿衣搭配、肢体语言开始。

接着，我们会深入拍照技巧，包括拍摄角度、光线选择、取景构图，并介绍一些低成本高逼格的拍照地点与拍摄方法。

然后，课程将涵盖后期修图技术、朋友圈文案撰写。

最后，我们会探讨内在的装逼，包括谈吐、气质以及思维方式的塑造。以上所有内容在我们的课程中都会全面涵盖。

请做好准备，让我们开始装逼之旅。
