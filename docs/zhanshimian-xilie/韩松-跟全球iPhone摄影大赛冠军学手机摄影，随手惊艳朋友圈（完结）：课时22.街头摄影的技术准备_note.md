# 手机摄影：22：街头摄影的技术准备

![](img/cdc3b9ced1d1ffe3d8434239e3ab19fe_0.png)

在本节课中，我们将学习街头摄影的核心概念与技术准备。我们将探讨如何捕捉街头瞬间、利用光影与视角，以及做好拍摄前的器材与操作准备。

街头与人物摄影，即人文摄影或街头摄影，本质上是运用观察力记录社会的真实状态。街头摄影具有明显的不可控性，这也正是其有趣之处。本节课将围绕街头摄影的核心——抓拍展开，包含多个议题：如何在混乱场景中拍到好看的内容、哪些元素值得发现和拍摄、街头有哪些道具可用、与陌生人拍摄时是否需要交流，以及街头场景的有趣表现方式。本课将逐一解答这些问题。

## 什么是街头摄影

上一节我们介绍了课程概述，本节中我们来看看街头摄影的定义。街头摄影是一种关于城市公共生活的摄影类型，是摄影师对特定城市生活观察和个人体验的表现。之前提到了“人物摄影”，这与“人像摄影”截然不同。人像摄影主要表现人物本身，而街头人物摄影主要表现人物的活动与空间的融合及关系，往往带来更强的故事感和叙事性。

## 街头摄影的关键要素

理解了基本概念后，我们来看看拍好街头摄影的几个关键点。以下是几个核心要素：

*   **形式感**：在传统街头摄影中，有一种审美取向是在真实杂乱的场景中提炼出有意思的形象。这类照片通常有一个视觉中心，画面简洁且有说服力。形式感可以理解为“看上去好看且有可读性”。当代摄影的取向则另当别论，将在最后一课简单介绍。

接下来分享几张极具形式美感的街头摄影照片。

第一张在纽约现代艺术博物馆拍摄。图书馆管理员穿着黑色西装，与背景墙壁形成鲜明黑白对比，制造了画面中唯一的视觉中心。这与第二堂课讲到的“单一纯粹”美学规律一致。

第二张照片中，夜空中的一盏灯非常显眼，带来了幽绿色彩。用色彩表现街头，常能制造极大亮点。

第三张照片运用了对比法则。左边站岗士兵的严肃与右边游客的放松，形成了情绪对比。

第四张照片首先能感受到色彩对比：左边的红墙与右边的黄墙。同时，左边人物穿黑衣，右边人物穿白衣，又构成黑白对比。

第五张照片写实地出现在街头，具有临场感。

第六张照片同样应用了对比。墙内的人与墙外的车形成大小对比，渲染出画面情绪。

## 捕捉画面情绪与情节

除了形式感，捕捉画面的“情绪爆点”也是街头摄影的利器。情绪爆点在于给画面制造故事感，抓拍人物行为不稳定的瞬间，让人感觉画面会继续发展。

第一个场景在纽约街头，抓拍到一个穿红衣女子高兴兴奋、两手张开的瞬间。这个动作与下方鸽子飞舞的动作存在联系。鸽子的动感与女孩极具张力的表情组合，表现出动态之美。

第二张照片在二楼拍摄。女孩的头发处于散开的不稳定瞬间，似乎下一秒就会回到肩上。

第三张照片中，两个人大步流星向前跑去，这是一个不平衡的瞬间，下一秒钟姿势就会变化。抓拍这种瞬间容易造成画面的情绪高潮。

我们还可以在街头摄影中抓拍有情节的内容，例如冲突、巧合和故事感。

第一张照片中，前景的狗狗欢快跑过，背景有一排英文写着“穿着夹趾拖鞋，你的生活会更美好”。通过文字传达出画面俏皮又温馨的感觉，体现了表意功能。

第二张照片体现了符号的传达。远处打牌的人和近处穿拖鞋经过的人产生了一种年代感。这种场景常出现在传统老区或生活样式保存较好的区域。

第三张照片用冲突表现画面。两边是大片花朵，中间玩游戏的小男孩像是强行插入，形成了与花朵的冲突感。

第四张照片体现了故事感。一个倒着的牌子写着“轻轨3号线”，但它指向的方向却是一个小广告。这种文字表现出的故事性，构成了趣味的街头拍摄。

## 第一部分要点总结

上一节我们看了许多具体案例，现在来总结一下第一部分的要点。

1.  **手机是理想的街头摄影工具**：它非常轻便，焦距适中。实际上，很多街头摄影大师都在使用与手机相似的焦段，即**30毫米左右**的焦段进行拍摄。
2.  **形式感、情绪化和情节感是关键**：只要我们能抓住这些元素，街头摄影往往能取得较好效果。
3.  **街头摄影不限于街头**：其核心在于用独到的眼光观察和记录人与人之间的关系，以及人与环境之间的互动。这是街头摄影非常重要的表现议题。

## 拍摄前的技术准备

了解了核心理念后，本节我们来看看要拍好一张街头照片，需要进行怎样的技术准备。

### 器材准备

以下是进行街头摄影时推荐的器材清单：

*   **手机**：当然是必要的拍摄工具。
*   **耳机线**：闲暇时可听音乐。之前课程提到，街头拍摄时可用耳机线当快门，更加隐蔽。
*   **硅胶套**：保护手机，防止滑落。
*   **充电宝**：手机拍摄非常耗电，准备一块**10000毫安以上**的充电宝非常有必要，可保证一天拍摄。
*   **充电线**：除了充电，还可充当稳定手机的作用。
*   **背包和水**：也是非常重要的装备。

### 操作准备

说完了器材，我们再来说说街头拍摄的操作准备。最重要的就是快速进入拍摄状态。因为街头元素都是突然出现的，所以能在最短时间内进入拍照操作非常重要。

这里以苹果手机为例，国产手机大多类似。

1.  **快速启动相机**：在锁屏界面下，点亮屏幕，拇指从屏幕边缘迅速向左滑动，即可直接进入相机界面，无需输入密码，这是最快的方法。
2.  **单手操作**：街头摄影中常需单手操作。首先按上述方法快速进入拍摄界面，然后用食指按下音量键（作为快门）。如果时间允许，拇指可进行对焦，以更高效率完成拍摄。

### 街头拍摄的可用材料

![](img/cdc3b9ced1d1ffe3d8434239e3ab19fe_2.png)

接下来分享几个街头摄影中非常棒的抓拍材料。

*   **抓拍光线**：顺光、逆光、侧光都可以运用。例如右侧照片采用了逆光拍摄。
*   **利用影子**：例如右侧照片中，夕阳下人物的影子被拉得非常细长，用影子参与构图。另一张照片是人物剪影，由于背景光线较亮，人物呈现剪影状态。这张在室内拍摄的照片，因曝光时间较长，人物产生了拖影，用拖影表现动感也是非常棒的方法。
*   **反射题材**：街头可以看到很多反射题材。例如这张照片拍到了路人眼镜中反射出的美国国旗。
*   **特殊视角**：我们平时最常见的视角是大约1.5米到3米范围内的平视。如果突然改成仰视或俯视，会带来更新鲜、新颖的拍摄效果。例如这张从上往下拍摄的一对情侣牵手的剪影效果就非常有意思。
*   **窥视视角**：例如右侧照片，其中的圆圈实际上是卷帘门上的圆孔。离圆孔很近时，圆孔显得很大。通过圆孔观察对面街道，形成窥视视角，这种方式在电影中也常运用。
*   **极近距离拍摄**：将被摄体拉得极近拍摄，往往会造成比较夸张的效果。例如上面这个人，近得可以看到他衣服的材质和包上的花纹。通过观察这些细节，可以抓到更有意思的材料。
*   **极远距离拍摄**：除了更近，还有更远的拍摄，以抓取人物与城市完全融合的感觉。

## 第二部分要点与总结

最后，我们来总结一下第二部分的要点，即街头摄影的身心准备。

1.  **保持习惯的拍摄方式**：尽量保证自己用最习惯的方式拍照，这一点非常重要。只有用最习惯的方式，才能提升拍摄效率，最大可能地抓拍到满意的街头照片。
2.  **善用光影与反射**：光影和反射都是渲染照片情绪的绝佳材料，可以多多观察和使用。
3.  **创造独特形式感**：极高或极低的视角、极远或极近的拍摄距离，都是不错的拍摄手法，可以给画面带来独特的陌生感。
4.  **具备独到眼光与快速反应**：独到的眼光、快速的反应和拍摄速度是提高出片率的保障。这几点请大家注意。
