# 何雄手机摄影教程：第05课·用手机做后期：课时7 · Stackables 📱

![](img/cbedddd73d09066456a63136cb3e3901_1.png)

在本节课中，我们将要学习一款名为Stackables的冷门但功能强大的手机后期处理软件。这款软件的核心特色在于其图层叠加功能，类似于桌面端的Photoshop，允许用户通过添加和调整多个特效图层来创造出独特的视觉效果。

## 软件简介与界面概览 🖼️

上一节我们介绍了多种后期工具，本节中我们来看看Stackables这款软件。它虽然相对冷门，但其图层叠加功能非常强大。

打开软件后，首先需要导入一张照片。软件界面顶部显示了照片的格式信息，如JPG或PNG。右侧的勾选按钮用于确认操作。

## 强大的裁剪与画幅功能 ✂️

导入照片后，软件提供了一系列智能裁剪选项。

以下是可用的画幅比例：
*   原片比例
*   1:1
*   4:3
*   3:2
*   16:9

软件能智能识别照片的横竖构图，并自动提供相应的裁剪建议，当然用户也可以进行自由裁剪。通常选择“原片”比例并打勾确认即可进入编辑界面。

## 核心：滤镜、特效与图层叠加 🎨

进入编辑界面后，我们可以看到上方有丰富的调整选项。

这些选项主要包括：
*   各类滤镜和视觉特效。
*   用于调整锐度、颗粒感等参数的图层。
*   一个位于右上角、图标类似瓶子的“预设”功能。这里汇集了许多经典的预设效果，例如模仿胶片或特定艺术风格的样式，选择非常丰富。

我们可以尝试应用一个预设效果。每个效果图层的强度都可以通过左下角的滑块进行左右滑动调节，实现强度的增加或减少。

## 高级图层管理 🔄

Stackables最强大的功能在于其图层管理系统。

在界面右下角，有一个带有加号和小脸谱的按钮。点击加号，可以执行以下操作：
*   **添加图层**：在同一张照片上叠加新的特效、质感或纹理图层。
*   **复制图层**：复制当前已应用的图层效果。
*   **删除图层**：移除不需要的图层。
*   **合并所有图层**：将所有调整合并为一张最终图像，类似于Photoshop中的合并图层操作。

而那个“小脸谱”图标，主要用于调整阴影、光晕等光影效果的参数变换。
