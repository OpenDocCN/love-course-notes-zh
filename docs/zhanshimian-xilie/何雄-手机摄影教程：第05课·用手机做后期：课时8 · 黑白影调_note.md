# 手机摄影教程：第05课：用手机做后期：课时8 · 黑白影调 🖤

在本节课中，我们将学习如何使用手机APP将一张彩色照片处理成一张高对比度的黑白作品。我们将使用Snapseed和KUNI Cam两款软件，通过调整滤镜、亮度、对比度、颗粒感等参数，来获得纯粹且有冲击力的黑白影像。

---

上一节我们介绍了黑白后期处理的整体思路，本节中我们来看看具体的操作步骤。

首先，我们打开Snapseed APP，并导入需要处理的照片。这张照片拍摄于中午，画面中有海鸥与人的对比元素。

**步骤一：应用黑白滤镜**

我的思路是将这张照片处理成高对比度的黑白效果，以突出其纯粹和简单的味道。

以下是具体操作：

1.  点击右下角的工具图标，进入工具菜单。
2.  选择“黑白电影”滤镜。这个滤镜能提供高对比度和颗粒感，符合我的预期。
3.  在滤镜选项中，我选择了“H01”或“H02”预设。它们能带来强烈的影调变化。
4.  我个人偏好纯粹的效果，因此将“颗粒”强度降为0，以消除我不喜欢的软件颗粒感，使画面更平滑。
5.  略微降低“亮度”。
6.  “柔焦”功能可以柔和明暗过渡，但本照片需要高对比度，因此保持不动。
7.  确认调整，点击勾号保存。

此时，照片已经完成了第一步的黑白转换，但高光部分可能仍然过于跳跃，不够厚重。

---

上一节我们应用了基础的黑白滤镜，本节中我们来进行更精细的影调调整。

**步骤二：精细调整影调**

为了达到理想的高对比度效果，我们需要进入“工具”>“调整图片”进行手动调节。

以下是核心参数的调整：

1.  **氛围**：增加氛围值，可以提亮暗部细节（如人物的领口部分），同时不过度影响高光。
2.  **高光**：降低高光值，让过亮的部分（如海鸥的腹部）变得柔和，减少刺眼感。
3.  **阴影**：为了强化对比，将阴影值向左滑动降低（例如减到-30），使暗部更暗，从而让对比更强烈。

经过这些调整，一张高对比度的黑白照片初步完成。点击“导出”>“保存副本”。但这并非最终步骤。

---

上一节我们在Snapseed中完成了主体调整，本节中我们将使用KUNI Cam软件进行最终优化和输出。

**步骤三：使用KUNI Cam进行最终优化**

退出Snapseed，打开KUNI Cam软件。导入上一步保存的照片。

1.  点击右上角“+”号，导入照片。
2.  习惯性查看照片参数（如尺寸、拍摄信息）后，点击“编辑”。
3.  首先，将**饱和度**降至最低，确保照片是纯粹的黑白。
4.  关键步骤是**锐化**。高对比度作品需要清晰的质感。将锐化值增加到10%左右，可以观察画面细节的变化。
5.  更关键的一步是添加**胶片颗粒**。KUNI Cam的颗粒质感非常出色。选择“胶片颗粒”效果，强度设置为17左右。这会使画面颗粒紧密，增加胶片感和输出质感。
6.  由于经过不同软件处理，影调可能稍有变化。此时可以再次微调**高光**和**阴影**，确保高光不过曝，暗部细节和厚重感符合要求。

**步骤四：添加边框与导出**

我个人习惯为作品添加边框。

1.  KUNI Cam的加框功能很强大，它是在原图外围扩展画布添加边框，而非裁剪原图，因此不损失画面像素。
2.  选择喜欢的边框样式（例如简单的黑框或模拟装裱效果的边框）。
3.  完成后，在导出前，务必检查设置。点击右下角三条横线图标，进入设置。
4.  为了无损保存，将画质设置为**专业（TIFF格式）**。这是我认为值得输出的作品格式。
5.  导出最终照片。

之所以选择在KUNI Cam中完成最终锐化、加颗粒和保存TIFF格式，是因为Snapseed无法保存TIFF，且KUNI Cam的颗粒算法和输出控制更能满足我对作品纯粹性和完整性的要求。

---

本节课中我们一起学习了制作高对比度黑白照片的完整流程。我们首先在Snapseed中利用“黑白电影”滤镜奠定基础影调，并通过调整图片工具精细控制高光、阴影和对比度。随后，在KUNI Cam中完成最终优化，包括去除杂色、添加锐化与胶片颗粒，并最终以无损的TIFF格式输出成品。这个过程结合了不同软件的优势，旨在获得质感强烈、影调纯粹的黑白摄影作品。
