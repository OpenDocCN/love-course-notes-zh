# 男哥展示面：1.1：展示面图片基础

在本节课中，我们将要学习展示面图片的核心概念与基础要求。展示面是个人形象在网络社交中的视觉呈现，其质量直接影响他人对你的第一印象。

## 概述

展示面图片是构建个人社交形象的第一步。一张合格的展示面图片需要传递积极、真实且有吸引力的信息。本节将分解展示面图片的构成要素与基本标准。

上一节我们介绍了课程的整体框架，本节中我们来看看展示面图片的具体要求。

## 核心要素

一张好的展示面图片通常包含以下几个核心要素：

以下是构成展示面图片的四个关键点：

1.  **清晰度高**：图片必须清晰，避免模糊或像素化。这代表了你的认真态度。
2.  **光线适宜**：光线应自然、明亮，能清晰展现面部轮廓与细节，避免背光或过度曝光。
3.  **背景简洁**：背景需干净、整洁，避免杂乱无章的元素分散注意力。
4.  **表情自然**：表情应放松、自信，展现积极的精神状态。

## 技术参数

为了达到上述要素，在拍摄或选择图片时，需关注一些基本技术参数。

以下是需要注意的几项技术性要求：

*   **分辨率**：图片分辨率不应过低，确保在各类设备上查看都清晰。
    *   代码示例：通常建议图片宽度**不小于1080像素**。
*   **构图**：人物应处于图片的视觉中心位置，常用构图法如三分法。
    *   公式描述：人物焦点 ≈ 画面黄金分割点（≈1:0.618区域）。
*   **格式**：推荐使用通用性强的格式，如JPG或PNG。

## 内容主题

除了技术层面，图片所表达的内容主题更为重要。

以下是几种常见且有效的展示面内容方向：

- **生活场景**：在咖啡馆阅读、参与户外运动等，展示生活方式。
- **专业技能**：在工作场景中专注的样子，体现职业感。
- **兴趣爱好**：如演奏乐器、绘画，展现个人才华与情趣。
