# 修图黑科技：1.3：照片光线调节 📸

在本节课中，我们将学习如何使用Snapseed软件对照片的光线进行精细调节。光线不均匀是影响照片质量的关键因素，我们将通过三个核心功能，让照片的光线分布变得均匀、自然。

上一节我们介绍了面部修图的基础，本节中我们来看看如何调整照片的整体光线。我们使用的工具是被称为“手机版Photoshop”的Snapseed，它的调光功能非常强大。

![](img/d1ed0f5253614616e94fbe70143c4993_1.png)

## 第一步：使用“调整图片”功能进行整体调节

首先，我们打开需要调整的照片。这张照片存在明显的光线问题：地面有亮斑、吊灯过亮，而门后和墙面区域则过暗，导致整张照片明暗不均。

![](img/d1ed0f5253614616e94fbe70143c4993_3.png)

“调整图片”功能可以对照片的整体曝光、色彩等进行初步校正。以下是具体的操作步骤：

1.  **亮度**：由于原图整体偏暗，我们首先增加亮度来补光。**注意**：亮度调整值建议不超过`15%`，以避免照片失真。本例中增加了`10%`。
2.  **对比度**：增加亮度后，照片可能略显平淡。适当增加对比度（本例为`+4`）可以让画面更真实。**注意**：对比度调整值同样建议控制在`±15%`以内。
3.  **饱和度**：如果觉得肤色或其他颜色过于鲜艳或暗淡，可以微调饱和度。本例略微降低了`5%`的饱和度。
4.  **氛围**：这是让光线变均匀的关键参数。它能智能提亮暗部并压暗过亮区域。**注意**：调整值不宜超过`20%`，否则画面会变模糊。本例增加了`20%`。
5.  **高光**：此参数专门控制画面中最亮的部分（如灯光）。适当降低高光（本例微调）可以让过曝区域恢复细节，画面更清澈。
6.  **阴影与暖色调**：**阴影**参数极易导致画面发灰、变“糊”，通常不建议调整。**暖色调**会影响整体颜色，建议在后续调色步骤中处理，此处保持不动。

经过“调整图片”的初步调整，照片的整体明暗已经协调了许多，但局部仍有问题。

## 第二步：使用“局部”功能进行精确修正

“调整图片”处理的是全局，而“局部”功能允许我们对照片的特定区域进行独立调整，这是解决局部过亮或过暗问题的利器。

以下是使用“局部”功能的要点：

1.  **定位与选择**：点击画面中需要调整的区域（如过亮的地板），会出现一个红点。双指缩放可以控制调整的影响范围（红色覆盖区域）。
2.  **调整亮度**：选中区域后，上下滑动即可单独调整该区域的亮度。**核心原则**：无论是提亮还是压暗，单次调整幅度最好控制在`±15%`以内。
3.  **处理多个区域**：可以添加多个控制点，分别处理地面亮斑、过暗的墙面、过亮的台灯等。如果控制点数量用完，可以保存后再进入“局部”功能添加新的调整。
4.  **进阶技巧 - 增强轮廓**：“局部”功能还可以用于塑造立体感。例如，在脖子侧的阴影处添加控制点并适当压暗，可以让人物轮廓更清晰。**公式**：`适当降低阴影区亮度 = 增强轮廓感`。

通过“局部”功能的精细涂抹，我们成功消除了抢眼的亮斑，提亮了死黑的暗部，使背景光线与人像主体更为协调。

## 第三步：使用“曲线”功能进行最终协调

在完成整体和局部调整后，我们使用“曲线”功能对全图的光线分布做最后的微调与平衡。

“曲线”功能通过一个坐标图来映射输入亮度与输出亮度的关系。操作步骤如下：

1.  选择 **“中性”** 滤镜，这是最常用的基础曲线。
2.  点击曲线图左下角的圆圈图标，选择 **RGB通道** 进行调整。
3.  **锁定亮部**：在曲线右上部分（代表高光区域）点击添加一个锚点并轻微拖动，目的是固定最亮的部分，防止其过曝。
4.  **提亮暗部**：在曲线左下部分（代表阴影区域）点击添加锚点，并**向上**轻微拖动。这可以整体提亮画面中最暗的部分，让细节更清晰。**注意**：拖动幅度要小，避免画面发灰。
5.  **微调中间调**：观察曲线中部，可根据需要轻微调整，使光线过渡更自然。如果觉得脸部光线因提亮暗部而有些平淡，可以尝试将曲线中部**稍稍向下**压一点，以恢复些许对比。

曲线调整的精髓在于“微调”。每一次拖动都要观察画面的整体变化，确保光线更加均匀、自然，而不是制造新的不协调。

## 总结

本节课中我们一起学习了使用Snapseed调节照片光线的完整流程：
1.  **整体校正**：使用“调整图片”功能，初步解决曝光和基本对比度问题。
2.  **局部修复**：使用“局部”功能，精准打击画面中过亮或过暗的特定区域。
3.  **全局平衡**：使用“曲线”功能，在RGB通道下微调亮度分布，使光线过渡达到最终和谐。

记住调光的核心目标是 **“均匀”与“自然”** 。每一步调整都应遵循“少量多次”的原则，避免参数过大导致失真。多对比调整前后的效果，你就能逐步掌握让照片焕发光彩的秘诀。
