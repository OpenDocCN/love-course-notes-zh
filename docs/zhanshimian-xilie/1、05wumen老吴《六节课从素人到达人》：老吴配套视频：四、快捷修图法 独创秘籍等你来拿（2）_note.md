# 老吴《六节课从素人到达人》：1.5：快捷修图法（2）📸

在本节课中，我们将继续学习老吴独创的快捷修图法。上一节我们介绍了基础调整，本节中我们来看看如何运用核心技巧进行高效美化，让照片质感迅速提升。

## 核心原理：分离色调与局部调整

快捷修图法的核心在于**分离色调**与**局部调整**。其原理公式可概括为：

**最终效果 = 基础调色 + Σ(局部调整图层 × 蒙版)**

这意味着，我们不依赖单一滤镜，而是通过多个针对性调整的叠加，精准控制画面不同区域。

## 核心操作步骤

以下是实现快捷修图的具体操作流程，每一步都至关重要。

1.  **建立观察层**：首先，创建一个黑白调整图层，将照片转为灰度。这有助于排除色彩干扰，专注观察画面的明暗关系与瑕疵。

2.  **瑕疵修复**：使用**修复画笔工具（快捷键：J）** 或**污点修复画笔工具**，去除人物皮肤上的明显痘印、斑点或环境中的杂物。

3.  **光影塑造**：新建一个曲线调整图层，以“S”型曲线轻微增加对比度。然后，使用黑色画笔在曲线图层的蒙版上涂抹，擦除对高光区域的提亮，仅保留对中间调和阴影的增强，从而立体化主体。

4.  **色彩润饰**：添加**色彩平衡**或**可选颜色**调整图层。例如，在高光中加一点青和蓝，在阴影中加一点红和黄，能为照片快速定下冷暖基调，增强氛围感。

5.  **锐化与输出**：最后，合并所有可见图层（Ctrl+Shift+Alt+E），执行“滤镜 > 锐化 > USM锐化”。数值设置参考：数量80%，半径1.2像素，阈值2色阶。此操作能增强细节，让照片更清晰。

## 关键技巧与注意事项

掌握步骤后，了解以下技巧能让你的修图事半功倍。

*   **蒙版是精髓**：几乎所有的调整都应配合图层蒙版使用。用黑色画笔隐藏效果，用白色画笔显示效果，这是实现局部调整的关键。
*   **数值宁少勿多**：每次调整的幅度要小，通过多个图层的微弱效果叠加，比单一图层进行剧烈调整更自然。
*   **建立个人预设**：将验证有效的调整图层组合保存为“动作”或“预设”，之后遇到类似风格的照片便可一键应用，极大提升效率。

本节课中我们一起学习了快捷修图法的核心技巧与完整流程。从建立观察层到最终锐化输出，关键在于理解**分离调整**的思想并熟练运用**图层与蒙版**。请记住，高效修图不在于步骤复杂，而在于每一步都精准有效。多加练习，你就能快速掌握这套方法，提升修图速度与质量。
