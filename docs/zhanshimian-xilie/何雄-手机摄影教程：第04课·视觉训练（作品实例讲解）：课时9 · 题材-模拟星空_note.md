# 何雄-手机摄影教程：第04课·视觉训练（作品实例讲解）：课时9 · 题材-模拟星空 🌌

![](img/9c01a55e489534c2760bd806f152e6f2_0.png)

在本节课中，我们将学习如何利用手机APP的创意滤镜来模拟星空效果，并探讨如何从日常生活中发现和创造独特的拍摄题材。

![](img/9c01a55e489534c2760bd806f152e6f2_0.png)

这张照片看起来非常梦幻。我曾经在朋友圈分享过很多类似的照片，朋友们常会惊讶地询问这是哪里拍摄的。能达到这种效果，让我感到非常开心。

这其中的亮点，在于一个功能强大的手机APP所提供的滤镜。接下来我们浏览一下。这张照片拍摄于下午，而另一张则拍摄于早晨。

![](img/9c01a55e489534c2760bd806f152e6f2_2.png)

这种效果就是通过软件模拟星空实现的。我将其称为“模拟星空”效果，这是该APP众多创意滤镜中的一个功能。这里以西山的照片为例进行说明。



![](img/9c01a55e489534c2760bd806f152e6f2_2.png)

西山位于昆明滇池边。这个例子告诉我们，可以从生活中非常常见的场景中发现拍摄题材。无论是街上的行人、行驶的车辆、建筑的轮廓、人物的瞬间、动物、植物，甚至是光影，都可以成为创作和拍摄的对象。

这就是手机摄影中，创意视角训练最基本的内容。通过这种方式拍摄，可以保持我们对任何事物的创作激情，以及通过摄影进行表达的欲望。生活中没有什么是不可以拍摄的。当然，我也会拍摄各种题材，包括自拍。
