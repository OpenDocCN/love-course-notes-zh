# 老吴《六节课从素人到达人》：1.5：快捷修图法

在本节课中，我们将学习一种高效的快捷修图方法，掌握如何快速提升照片的视觉表现力。

上一节我们探讨了构图的基础，本节中我们来看看如何通过后期调整快速优化照片。

## 核心原理：亮度与对比度

快捷修图的核心在于快速调整画面的**亮度**与**对比度**。亮度决定画面的明暗基调，对比度则影响明暗部分的差异程度。一个简单的调整公式可以表示为：

![](img/f151cb686fc7c37debfb62dab3d0feab_0.png)

**调整后图像 = 原图像 × 对比度系数 + 亮度增量**

通过平衡这两者，可以立刻让照片显得更通透、更有层次。

## 操作步骤详解

以下是使用常见修图软件进行快捷调整的具体步骤。

1.  **导入图片**
    首先，在修图软件中打开你需要处理的照片。

![](img/f151cb686fc7c37debfb62dab3d0feab_1.png)

2.  **调整曝光/亮度**
    观察照片整体明暗。如果偏暗，则适当增加曝光值或亮度；如果过曝，则适当降低。目标是让画面细节清晰可见。

3.  **增强对比度**
    适当增加对比度，可以让亮部更亮，暗部更暗，从而突出主体，增强画面的立体感。

4.  **微调高光与阴影**
    单独调整高光（最亮部分）和阴影（最暗部分）。降低高光可以恢复过亮区域的细节；提亮阴影可以显示暗部隐藏的信息。

5.  **确认并导出**
    对比调整前后的效果，确认满意后，导出最终成片。
