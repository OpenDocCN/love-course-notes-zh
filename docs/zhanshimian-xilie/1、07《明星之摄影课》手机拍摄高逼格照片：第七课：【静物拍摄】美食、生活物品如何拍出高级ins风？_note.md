# 明星之摄影课：07：静物拍摄——美食与生活物品如何拍出高级ins风？

在本节课中，我们将学习如何使用手机拍摄静物，特别是美食和生活物品，以获得具有高级感和ins风格的照片。我们将从背景、光线、道具、构图、拍摄角度和后期处理等多个方面进行详细讲解。

上一节我们介绍了人像摄影，本节中我们来看看如何拍摄静物。静物拍摄是记录生活美好瞬间的重要方式，掌握技巧能让你的照片质感大幅提升。

## 背景选择

静物照片的美观度，背景起到了关键作用。选择背景的原则是**越简洁越好**。简洁的背景是照片高级感的来源，近年流行的ins风、极简风、北欧冷淡风都遵循这一原则。

![](img/6cdfb475524f4692c0edb9ba6a05fd2d_2.png)

以下是实现背景简洁的方法：

*   **选择单一材质**：使用纯色墙面、干净桌面或带有格纹的桌布作为背景基调。
*   **个性化选择**：在单一材质基础上，可根据喜好选择颜色或纹理，例如粉色墙、大理石桌面或原木桌面。
*   **实用推荐**：黑白格纹桌布是质感很高的拍摄背景。

![](img/6cdfb475524f4692c0edb9ba6a05fd2d_2.png)

## 光线条件

![](img/6cdfb475524f4692c0edb9ba6a05fd2d_4.png)

选好背景后，我们需要考虑光线。室内拍摄静物常受限于采光，而光线是较难人为控制的客观条件。

因此，需特别注意：

*   **选择光线充足均匀的位置**：例如窗边或阳台边。
*   **处理强烈日光**：正午时，可使用透明白色窗帘过滤部分光线。
*   **优先使用自然光**：自然光比室内灯光更柔和，不易形成强烈的阴影和光影对比。

![](img/6cdfb475524f4692c0edb9ba6a05fd2d_4.png)

## 道具准备

合适的道具能为静物照片增添内容和个性标签，直接影响作品风格。例如，第一节课演示的小花瓶就是ins风静物摆拍的常见帮手。

以下是三类推荐的小道具：

*   **花草类**：如满天星、绿叶、鲜花、白色棉花球。
*   **时尚单品类**：如墨镜、口红、包包。
*   **文艺气息类**：如外文杂志、外文报纸、木棍香薰、茶色玻璃瓶、明信卡片。

这些道具多为日常搭配品，不常见的也可在电商平台以适中价格购得。

## 构图技巧

构图是静物拍摄的核心。在第三节课的基本构图法则基础上，我们需要引入**拍摄画面的主体数量**这一概念，它决定了使用何种构图方式。

主体数量并非指物品的绝对数量，而是根据画面呈现效果来判断。例如，拍摄一盘食物时，主体是“一盘食物”。但若拉近镜头俯拍盘中的四个草莓，主体则变成了“四个草莓”。

根据画面主体数量，常用构图方法可分类如下：

*   **单一主体**：常用**中心构图法**、**景框式构图**和**三分法构图**。公式表示为：`主体位置 ∈ {中心点，三分线交点}`。
*   **两个相似主体**：常用**对称式构图**，例如ins上流行的“对称早餐”。
*   **两个差异主体**：可营造主次分明的效果，例如餐盘（主体）与杯子（次体）的搭配。
*   **多个主体**：常用**垂直俯拍（Flat Lay）** 手法。拍摄时需将镜头置于主体正上方垂直向下，并注意避开阴影。iPhone用户可让取景框中心的两个十字架重叠以辅助对齐。

除了上述构图，静物拍摄也注重**留白**手法的运用：

*   **三分法自然留白**：主体置于三分线位置，另一侧自然形成留白。
*   **三角构图留白**：让主体在画面中形成三角形的三个顶点，并注意保留部分留白以突出意境。
*   **不规则留白**：多用于自然风光拍摄，如以天空为背景拍摄树枝，恰当的留白能提升意境。

## 拍摄角度

拍摄角度需根据物品特点选择。以美食拍摄为例：

食物呈现方式主要分两类，拍摄角度也相应不同：

*   **敞口容器（如盘子）**：拍摄角度多样，可侧面拍也可俯拍。关键在于找到食物最好看的一面。
    *   **立体感强的食物（如汉堡）**：建议侧拍，以展现层次和丰富内容。
    *   **注重摆盘的食物**：建议俯拍，以完整展现精美摆盘。
*   **非敞口容器（如锅、不透明杯子）**：更多使用俯拍，才能更好地展现食物。

此原则同样适用于其他静物或风景拍摄，即找到物品最具特色和最美的一面进行记录。

## 后期处理

本节课教授的静物拍摄主要围绕ins风格。Instagram应用内置了便捷的拍摄和编辑功能。

后期处理基本流程如下：

1.  打开Instagram，点击加号开始拍摄。
2.  构图并调整曝光。
3.  按下快门。
4.  在众多滤镜中选择喜欢的风格（例如温暖的色调）。
5.  使用编辑功能调整参数（如暗部细节、清晰度、亮度等），使照片更完善。
