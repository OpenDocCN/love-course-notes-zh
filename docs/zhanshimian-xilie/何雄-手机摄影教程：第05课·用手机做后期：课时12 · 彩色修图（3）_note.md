# 手机摄影教程：第05课：用手机做后期：课时12 · 彩色修图（3）

![](img/3f8a560dff343fe2122db88eefb451de_0.png)

在本节课中，我们将学习如何使用Snapseed对一张彩色人像照片进行后期处理。我们将通过一系列步骤，包括应用滤镜、增强细节、精细调整和局部处理，来提升照片的色彩表现力和视觉冲击力，最终导出高质量的照片。

---

上一节我们介绍了黑白照片的冷暖度处理。本节中，我们来看看如何对一张彩色人像照片进行后期修图。

我们将在Snapseed中导入一张使用手机闪光灯拍摄的肖像照片。这张照片的特点是人物面部有丰富的彩绘，因此进行彩色后期处理可以更好地凸显色彩的惊艳度和吸引力。

## 第一步：应用“怀旧”滤镜

首先，我们使用“怀旧”滤镜为照片奠定一个柔和的基调。这个滤镜通常能带来低饱和度和朦胧的复古感。

以下是操作步骤：

1.  在Snapseed中打开照片，点击“工具”。
2.  选择“怀旧”滤镜。
3.  系统提供了多种预设样式，可以逐一尝试，观察照片的变化。我们选择了一个使皮肤看起来更柔和、温暖的样式。
4.  点击样式后，屏幕上会出现可调节的参数：**亮度**、**饱和度**、**对比度**和**样式强度**。
5.  我们不需要怀旧效果中的“划痕”和“漏光”元素，因此将“划痕”强度降至0。
6.  由于怀旧滤镜默认降低饱和度，而我们希望色彩突出，因此将**饱和度**参数提高到100%。
7.  **对比度**保持不变。
8.  调整**样式强度**来控制滤镜效果的强弱，我们将其设置为70-80%。
9.  调整完成后，点击右下角的对勾确认。

经过这一步，照片整体呈现出一种柔和的朦胧感，为后续的色彩强化做好了铺垫。

## 第二步：使用“戏剧效果”增强立体感

为了使照片的色彩更加突出、立体感更强，我们接下来使用“戏剧效果”工具。这个工具类似于HDR，能增强画面的层次和细节。

以下是操作步骤：

1.  回到主工具菜单，选择“戏剧效果”。
2.  工具提供了多种特效（如戏剧1、戏剧2、明亮等），可以逐个尝试。我们选择了效果较为中性的“戏剧1”。
3.  点击特效后，可以调整两个参数：**滤镜强度**和**饱和度**。
4.  为了获得强烈的立体感，我们将**滤镜强度**设为100。
5.  应用该效果后，照片饱和度可能会降低，因此我们需要将**饱和度**参数加回来，提高到100。
6.  点击对勾确认。

对比处理前后的效果，可以看到经过“戏剧效果”处理后，照片的灰暗感消失，色彩和立体感得到了显著增强，面部皮肤也保留了自然的柔和质感。

## 第三步：进行“调整图片”精细修复

“调整图片”工具可以对照片的光影和色彩进行非常精细的全局调整，是后期处理中的关键步骤。

以下是操作步骤：

1.  再次点击“工具”，选择“调整图片”。
2.  首先增加**氛围**参数（可理解为智能HDR），这能让画面细节更丰富、更惊艳。我们将其增加到70%。
3.  接着，提高**饱和度**，让色彩更加鲜艳夺目。
4.  降低**高光**参数，防止面部过亮区域失去细节。
5.  最后，微调**暖色调**参数。为了让色彩更惊艳且不与背景过于融合，我们适当降低了暖色调，使其偏向冷色一点，设置为-19。
6.  点击对勾保存调整。

这一步的调整进一步分离了人物与环境，使面部的色彩和表情更具张力和视觉焦点。

## 第四步：利用“局部”工具进行微调

“局部”工具是Snapseed中非常强大的功能，它允许我们对照片的特定区域进行独立的亮度、对比度和饱和度调整，实现分区控制。

虽然这张照片不一定需要，但我们演示一下其用法：

1.  在“工具”中选择“局部”。
2.  点击“+”号，在画面中需要调整的位置（例如眼睛）点击一下，会出现一个控制点。
3.  用手指在屏幕上上下滑动，可以选择调整**亮度**、**对比度**或**饱和度**。
4.  左右滑动手指，可以增加或减少所选参数的值。
5.  双指在屏幕上捏合或张开，可以控制该调整点的影响范围。
6.  通过添加多个控制点，可以分别压暗背景、提亮主体，从而更突出主题。

这个工具能实现非常精细和个性化的调整。

---

本节课中我们一起学习了彩色人像照片的完整后期流程。我们通过四个核心步骤来优化照片：

1.  **应用“怀旧”滤镜**：奠定柔和基调，去除不必要的划痕，并提高饱和度。
2.  **使用“戏剧效果”**：大幅增强画面的立体感和细节表现力。
3.  **进行“调整图片”**：精细控制氛围、饱和度、高光和色温，使色彩惊艳且协调。
4.  **探索“局部”工具**：了解如何对特定区域进行微调，以突出主体。

如果你非常喜欢这个效果，可以记住这个步骤顺序并多加练习。

## 最终导出与格式转换

照片在Snapseed中处理完成后，为了获得最佳质量，我们通常将其导出为TIFF格式，并可能进行最终锐化。

![](img/3f8a560dff343fe2122db88eefb451de_2.png)

1.  在Snapseed中点击“导出”，选择“保存副本”（或“导出”为TIFF格式）。
2.  将导出的照片再次导入Snapseed或其它软件（如Photoshop Express）。
3.  由于跨软件可能导致画质损失，可以适当添加一点**锐化**，使细节更清晰。
4.  也可以添加少许**胶片颗粒**，增加照片的质感。
5.  根据喜好，可以为照片添加一个白边。
6.  完成所有调整后，保存最终成品。

经过这一系列处理，最终的照片文件大小会增加，画质细节也得到了更好的保留。这就是一张彩色人像照片从原始状态到惊艳成片的完整后期流程。
