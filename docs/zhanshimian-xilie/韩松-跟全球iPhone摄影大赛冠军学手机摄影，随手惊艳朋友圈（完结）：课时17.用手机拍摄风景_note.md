# 韩松手机摄影课：课时17：用手机拍摄风景

在本节课中，我们将学习如何用手机拍摄风景照片。核心在于为风景画面引入视觉焦点，并探索通过不同拍摄手法与后期处理，创造出风格独特的作品。

上一节我们介绍了城市建筑摄影，本节中我们来看看如何将类似的构图思维应用于更广阔的自然风景之中。

## 引入画面的“主心骨”

拍摄风景时，最重要的技巧是在场景中引入一个“主心骨”。这个主心骨可以是人物、建筑，或是风景自身的结构线。它能有效提升画面的精神气，同时不会妨碍主体元素的表现。

以下是几个具体的例子：

*   **建筑作为焦点**：在一张背景为峭拔山峰的照片中，一座位于画面正中央的小房子构成了视觉中心，成为画面的主心骨。
*   **人物增添趣味**：在拍摄冰岛草帽山时，如果只表现山峰会略显单调。在前景中加入手持气球的人物，能为画面注入俏皮感。观看者的视线会先被人物吸引，再扩散至背后的山体。
*   **引导线汇聚视线**：在一张阴天公路的照片中，背后山雨欲来的云层极具视觉冲击力。前景的公路从近处向远方收拢，形成视觉上的集合效果，增强了画面的重点。
*   **突出形态与质感**：雪山本身的形态和质感就极具冲击力。在这种情况下，集中表现其形态与质感即可。
*   **色彩与元素呼应**：一张照片中，奶蓝色的湖水与天空的颜色形成呼应。画面中冒烟的电场则增添了人为与自然的对比感。

## 创造风格化的风景

![](img/0530135a2d5b1a28b8b8407ac59ef7eb_1.png)

制造不一样的风景效果，需要借助一些特定的拍摄或后期手段。例如，可以使用慢速快门（Slow Shutter）进行长曝光，或利用iPhone的实况模式。此外，通过后期裁剪与构图，也能制造出极简风格的效果。

以下是几种风格化风景的创作方法：

*   **极简构图**：可以拍摄雪后简洁的山坡，依靠前景的线条质感形成极简效果。有时，直接仰拍山的走势，隐藏其他杂乱元素，也能创造出极简的美感。
*   **长曝光效果**：使用长曝光拍摄冰岛黑沙滩的海浪，能够凝固住海浪拍岸的瞬间，形成如颜料画般的朦胧质感。拍摄长曝光可以使用Slow Shutter等App配合三脚架，或者简单地使用iPhone的实况模式拍摄后，在相册中选择“长曝光”效果。

## 核心要点总结

本节课中我们一起学习了手机风景摄影的几个关键点：

1.  **寻找视觉焦点**：在风景拍摄时，找到或安排一个“主心骨”至关重要。
2.  **把握拍摄时机**：早晚阳光倾斜时是拍摄风景的黄金时间。但阴天、雨天或逆光等条件也并非不能拍摄，反而可以利用这些条件制造独特的风格。
3.  **利用前景与关联**：拍摄风景时，除了远景，应多利用画面的近景，让人物或道具与环境产生联系。
4.  **避免单一视角**：风景摄影容易陷入“大而全”的单一模式。通过观察局部、寻找画面元素间的对应关系，并从多种角度拍摄，可以增强画面的张力。
