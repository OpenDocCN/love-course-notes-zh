# 老吴《六节课从素人到达人》：1.5：八大类修图实操案例（二）🎨

在本节课中，我们将继续学习八大类修图实操案例的第二部分。上一节我们介绍了基础的人像与风景修图，本节中我们来看看如何对食物、静物以及创意合成图片进行处理，让你的朋友圈内容更加丰富和吸引人。

## 概述
本节课将聚焦于三类常见的修图场景：**美食摄影**、**静物摆拍**以及**创意合成**。我们将通过具体的操作步骤和核心工具，帮助你掌握让图片“焕然一新”的技巧。

## 美食摄影修图
美食图片的关键在于激发食欲，修图重点是增强色彩饱和度和突出细节。

以下是处理美食图片的三个核心步骤：

1.  **调整色温与色调**：使用**色温（White Balance）**工具，让食物呈现暖色调（如增加黄色），通常能更好地营造美味可口的氛围。公式表示为：`最终色温 = 原始色温 + 黄色增量`。
2.  **增强色彩与对比度**：适度提高**饱和度（Saturation）**和**对比度（Contrast）**，让食物颜色更鲜艳，明暗关系更清晰。
3.  **突出细节与锐化**：使用**清晰度（Clarity）**或**锐化（Sharpen）**工具，强化食物表面的纹理，例如蛋糕的糖霜或牛排的肌理。

## 静物摆拍修图
静物图片注重质感与格调，修图目的在于营造氛围和突出物品的精致感。

以下是提升静物照片质感的四个要点：

1.  **统一背景与去除杂物**：利用**修复画笔（Healing Brush）**或**克隆图章（Clone Stamp）**工具，清理画面中分散注意力的元素，保持背景干净简洁。
2.  **强化光影与立体感**：通过**曲线（Curves）**或**局部调整画笔（Adjustment Brush）**，人为加深阴影、提亮高光，增强物体的三维感。
3.  **应用滤镜营造氛围**：尝试添加微妙的**胶片颗粒（Grain）**或**暗角（Vignette）**效果，为图片增添复古或聚焦的视觉感受。
4.  **精确色彩校正**：如果物品有特定颜色（如品牌Logo色），可使用**HSL工具**单独调整该颜色的色相、饱和度和明度。

## 创意合成修图
创意合成是将不同元素组合成新画面的艺术，关键在于实现无缝拼接与和谐统一。

以下是完成一次基本创意合成的操作流程：

1.  **构思与选材**：明确合成主题，并挑选背景、主体元素等素材，确保它们的光照方向和分辨率大致匹配。
2.  **抠图与融合**：使用**钢笔工具（Pen Tool）**或**选择并遮住（Select and Mask）**功能将主体从原图中精确抠出，放置到新背景上。
3.  **调整颜色与光影**：匹配所有合成元素的色调和光影。为核心概念，可使用**颜色匹配（Color Match）**命令，并利用**曲线工具**统一高光和阴影的颜色倾向。代码逻辑可简化为：`背景层色调 -> 映射到 -> 主体层色调`。
4.  **添加统一效果**：最后对整个合成画面进行统一调色，并添加全局光影（如阳光、雾气）以使合成效果更加真实自然。

## 总结
本节课我们一起学习了针对**美食**、**静物**和**创意合成**这三类图片的修图方法。关键在于理解不同场景的修图目标：美食要“诱人”，静物要“有质感”，合成要“无违和”。掌握这些核心思路与工具组合，你便能轻松应对大部分朋友圈图片的修饰需求，持续产出高质量内容。
