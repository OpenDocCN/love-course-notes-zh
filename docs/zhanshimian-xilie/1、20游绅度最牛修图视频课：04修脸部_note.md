# 20游绅度最牛修图视频课：04：修脸部 🎨

在本节课中，我们将学习如何通过后期修图技术，对面部轮廓和五官进行精细调整，从而改善照片中的人脸外观。我们将从基础概念讲起，逐步演示从调光到使用专业工具进行“微整”的全过程。

## 概述：理解“微整”概念

在开始修图之前，我们需要理解一个核心概念：**“微整”**。这并非指现实中的整形手术，而是指通过后期软件，对人脸的面部构造进行数字化调整。其意义在于，当我们在重要场合拍摄的照片因角度或技术问题不理想时，可以通过修图技术将其修复到更完美的状态。

上一节我们介绍了修图的意义，本节中我们来看看具体的操作步骤。

## 第一步：基础光影调整 💡

修图的第一步永远是调整照片的光影，确保脸部曝光正确、细节清晰。以下是操作要点：

*   **亮度与氛围协调**：单独提高亮度可能导致背景过曝。正确做法是适度调整亮度（例如50），同时增加“氛围”值，使脸部自然清晰。
*   **高光与阴影**：简单降低高光，提亮阴影，可以平衡画面光比。
*   **局部调整**：使用局部调光功能，单独提亮脸部区域。
*   **锐化**：最后，适当增加锐化，让轮廓更清晰。

完成以上步骤，我们就为后续的面部修饰打下了良好的基础。

## 第二步：使用“美图秀秀”调整面部轮廓 ✏️

接下来，我们进入改变面部物理构造的核心环节。这里我们使用“美图秀秀”的瘦脸功能，一般建议将强度调到**4档**，以便于精细控制。

**请注意**：修图的目的是进行优化和微调，而非变得面目全非。掌握“大修”技术是为了更好地进行日常“小修”。

以下是使用瘦脸功能的正确手法和顺序：

1.  **操作方向**：**从脸部外侧向内侧轻轻推动**，而不是从里向外。这样可以更自然地收缩轮廓。
2.  **调整咬肌**：大多数人咬肌都略显突出，可适当向内推，使下颌线更流畅。
3.  **修饰下巴**：根据原图下巴情况，可轻微拉长或收窄，塑造清晰的下颌轮廓。
4.  **处理苹果肌与额头**：如果苹果肌过于饱满或额头形状不佳，可进行适度平滑与修饰。
5.  **调整鼻子**：可缩小鼻翼，并轻微推动鼻梁两侧，让鼻形更挺拔。
6.  **修饰嘴角与眼睛**：将嘴角微微向上推，制造微笑感。外眼角可以适当拉长，使眼睛看起来更大。
7.  **整体协调**：完成局部调整后，缩小视图观察整体比例，必要时再次微调，使脸部轮廓和谐。

通过以上步骤，脸部的整体轮廓和五官位置就得到了初步优化。

## 第三步：使用“Facetune”进行精细化平滑处理 🧽

在塑造好基本脸型后，我们需要处理皮肤上的褶皱、不平整等细节。这里我们使用 **Facetune** 的“平滑”功能。

这个功能相当于数字磨皮，可以抹平细纹和瑕疵，让皮肤看起来更光滑。操作时，在需要处理的区域（如法令纹、眼袋、皮肤褶皱处）轻轻涂抹即可。为了达到最佳效果，可以重复涂抹2-3次。

## 第四步：使用“细节”功能强化轮廓 🌟

“细节”功能相当于化妆时使用的**侧影（修容）**。它的原理是强化局部对比，让涂抹的区域更突出。

以下是使用细节功能的技巧：

*   **勾勒五官**：用细节功能轻轻涂抹眉毛、眼眶、嘴唇边缘，能让五官更立体清晰。
*   **强化轮廓**：沿着发际线、脸颊边缘、下巴轮廓涂抹，可以起到“修容”效果，让脸部轮廓在视觉上更加紧致和分明。

操作时需沿着结构线轻轻涂抹，避免过度使用导致画面失真。

## 第五步：添加滤镜与最终调整 🎭

最后一步是为照片添加风格化效果，提升整体质感。

![](img/bc5790ad34e9405245633a54711d9195_5.png)

![](img/bc5790ad34e9405245633a54711d9195_7.png)

1.  **添加滤镜**：选择一个合适的滤镜（如红色或黄色调），为照片定下基调。
2.  **增加清晰度**：再次轻微锐化，确保画质。
3.  **营造氛围**：使用“褪色”或类似功能，添加一丝朦胧感，让照片更具氛围。

完成所有步骤后，务必对比修图前后的效果，检查整体是否自然和谐。

## 总结与核心建议 📝

本节课中我们一起学习了从光影调整、轮廓重塑、皮肤平滑到细节强化、滤镜添加的一整套人脸修图流程。

**核心公式可以概括为：理想人像 = 正确光影 + 协调轮廓 + 平滑肌肤 + 立体细节 + 风格化滤镜**

最后，给所有初学者一个最重要的建议：**修图应追求“润物细无声”**。本节课演示的是一种相对夸张的手法，旨在展示技术的可能性。在实际应用中，尤其是用于社交展示的照片，应以“微整”和自然优化为主，避免修饰过度。当你掌握了“大修”的技术力度后，就能更精准地控制“小修”的程度，做到真实与美观的平衡。

记住，修图是让照片更接近你理想状态的工具，其目的是增强自信，而非制造虚假。
