# 展示面构建教程：1：展示面核心概念与图片选择

在本节课中，我们将要学习展示面的核心概念，并了解如何选择合适的高质量图片来构建你的个人展示面。展示面是你在社交平台上的视觉名片，直接影响他人对你的第一印象。

## 什么是展示面？

展示面特指在社交软件（如微信朋友圈、微博、探探等）中，用于展示个人生活、品味、价值和状态的图片集合。它并非简单的照片堆砌，而是一个有策略的视觉呈现系统。

上一节我们介绍了展示面的定义，本节中我们来看看构建优质展示面的核心原则。

## 展示面核心原则

以下是构建有效展示面必须遵循的几个核心原则：

1.  **真实性**：展示面应基于真实生活。过度摆拍或使用虚假素材会损害信任。
2.  **多样性**：内容需涵盖多个生活维度，避免单调。公式可表示为：**优质展示面 = 生活 + 兴趣 + 价值 + 社交**。
3.  **高质量**：图片的构图、光线、清晰度必须过关。低质、模糊的图片会传递负面信息。
4.  **积极导向**：整体基调应积极、向上，展现热爱生活的一面。

理解了基本原则后，我们将进入实操环节，学习如何具体选择和处理展示面图片。

## 图片处理与发布要点

选择好图片后，适当的处理与发布方式同样重要。

1.  **基础处理**：使用裁剪、滤镜、调色等工具优化图片，但切忌过度修饰。代码式的精确要求是：`修饰度 < 30%`。
2.  **配文技巧**：配文应简洁、有点睛之效，或与图片内容互补。避免冗长描述。
3.  **发布节奏**：保持稳定、不刷屏的发布频率。建议每周2-4条高质量内容为宜。
