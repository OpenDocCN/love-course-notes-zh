# 老吴《装逼课》：8：萌宠互动装逼指南 🐶📸

![](img/c48de2d569295319027be35c34433427_0.png)

在本节课中，我们将学习如何与宠物（特别是狗狗）进行互动并拍摄出自然、有爱的照片。课程将涵盖拍摄前的准备、互动技巧、拍摄姿势以及后期选片思路。

## 拍摄前的准备

上一节我们介绍了课程主题，本节中我们来看看拍摄前需要做哪些准备。

与宠物拍照，首先需要解决“宠物从哪里来”的问题。以下是几种获取“拍摄道具”的途径：

*   **借用朋友的宠物**：联系身边养狗的朋友。
*   **加入宠物社群**：进入宠物爱好者的圈子，寻找机会。
*   **饲养自己的宠物**：这是最理想的情况。

## 拍摄场所的选择

解决了宠物来源问题后，接下来需要选择合适的拍摄场地。一般来说，有两个场所比较理想：

*   **室外公园**：选择有树木、水景等自然环境的公园。
*   **宠物游乐场**：专门的宠物活动场所。

建议选择下午的时间段在公园进行拍摄。

## 核心互动与拍摄技巧

选好场地后，真正的拍摄环节就开始了。与宠物拍照的核心在于**展现互动与爱心**，而不是僵硬地摆拍。

如果宠物不是自己饲养的，在拍摄前需要花时间与它培养感情，具体方式可与主人沟通。

### 拍摄姿势与互动

许多人会选择直立站着与宠物拍照，这样拍出的照片比较普通。更好的方式是沉浸在互动中。以下是几种推荐的拍摄姿势和互动方法：

*   **半蹲互动**：采用自然的半蹲姿势，与狗狗玩耍，例如抚摸它的头。拍摄时无需刻意看镜头，将注意力完全集中在宠物身上。
*   **地面玩耍**：可以坐下或与狗狗一起躺在地上玩耍，进行互动。
*   **动态行走**：请助手帮忙，拍摄你牵着狗狗向镜头走来的画面。可以来回走几次，捕捉动态的、有生活感的瞬间。

**核心要点**：在整个拍摄过程中，你需要多与宠物互动。随着感情加深，拍摄会越来越顺利。关键在于投入互动，把宠物当作朋友，而非道具。

### 眼神与焦点

拍摄时，眼神的处理很重要。如果宠物没有看镜头，你却盯着镜头，画面会显得奇怪。最自然的方式是**你在拍摄时看着宠物**。

## 后期选片与思路

拍摄完成后，我们进入选片阶段。通常先拍摄几十张甚至上百张照片，然后从中筛选。

选片时，可以关注以下几点：

*   **互动感强的照片**：挑选那些能体现你与宠物自然互动的瞬间。
*   **“对话感”照片**：寻找一些仿佛你正在和宠物交流的照片，这类照片充满温情。
*   **动态照片**：行走、玩耍中的抓拍，富有动感和生活气息。

选中满意的照片后，可以将其标记（如标星），以便后续进行修图和配文。好的配文能让朋友圈分享的效果更出色。

如果第一次拍摄没有选出理想照片，可以继续尝试拍摄。

## 课程总结

本节课中，我们一起学习了与狗狗拍照的完整流程：

1.  **前期准备**：解决宠物来源，选择公园或宠物乐园作为场地。
2.  **互动拍摄**：通过半蹲、玩耍、行走等姿势进行自然互动，拍摄时看着宠物，投入感情。
3.  **后期选片**：从大量照片中筛选出互动感强、有爱、自然的照片，用于后续修图和分享。

核心在于**真诚互动**，将宠物视为伙伴，才能拍出打动人心的照片。
