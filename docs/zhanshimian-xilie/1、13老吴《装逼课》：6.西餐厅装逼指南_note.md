# 老吴《装逼课》：6：西餐厅装逼指南 🍽️

![](img/728108002429895b63c801e28f3fec02_1.png)

在本节课中，我们将学习如何在西餐厅用餐时，通过得体的着装、优雅的餐桌礼仪以及专业的拍照技巧，自然地展现出高格调。课程内容涵盖从点餐、摆盘到用餐、拍照的全过程，旨在帮助初学者掌握“无形装逼”的核心要领。

## 着装与点餐

上一节我们介绍了课程概述，本节中我们来看看进入西餐厅前的准备工作。着装和点餐是塑造第一印象的关键。

去西餐厅用餐，着装不宜过于花哨。简洁的着装（例如白色上衣）能让画面更干净，避免衣服在照片中过于抢镜。

关于点餐，西餐通常包含浓汤、牛排、披萨等菜品。以下是常见的点餐组合：
*   与朋友或约会对象用餐，可以点三个菜。
*   或者，每人点一份牛排，搭配汤和红酒（本节课暂不涉及红酒）。

## 菜品摆盘与构图

点好餐后，面对多个菜品，如何摆盘是拍出好照片的第一步。将菜品摆成一条直线很难拍摄，我们需要进行穿插摆放。

以下是摆盘的核心步骤：
1.  将主菜（如牛排）的盘子放在自己面前。
2.  将其他菜品（如披萨、配菜）围绕主菜摆放，注意间隔开，不要紧挨在一起。
3.  可以将汤碗特意摆放在构图中间。
4.  利用桌上的鲜花等小物件作为点缀，与菜品形成**三角形联动构图**。

正式的西餐厅，餐具摆放有讲究：**刀和汤勺在右边，叉子在左边**。餐盘下通常会有桌布或餐垫。

## 拍摄技巧与角度

摆好盘后，就可以开始拍照了。由于菜品丰盛，建议使用长方形的画幅进行拍摄。

开始拍照前，有一个关键细节：务必检查并擦拭手机摄像头，确保没有指纹或污渍，以保证照片清晰干净。

西餐风格较为正式，适合拍摄稳重、自然的风格。拍摄时，焦点应对准食物本身。构图是重点，可以根据餐厅光线，使用手机拍摄界面的**亮度调节功能（太阳图标）** 进行微调。

除了平拍，可以尝试多种角度：
*   **30度角拍摄**：较为常见的角度。
*   **45度角拍摄**：能带来不一样的视觉感受。
*   **90度俯拍**：站起来垂直拍摄。此时需保持手机水平，并注意避免灯光造成的阴影。如果发现有阴影，可以向后移动再拍摄。

如果只有一两道菜，可以采用**对角线构图**。摆盘的美观关键在于颜色搭配，要么统一，要么互补，同时菜品本身（如牛排）的品相要好。

## 牛排的选择与礼仪

牛排是西餐装逼的重点。选择牛排很有讲究，这直接关系到用餐体验和“装逼”的成败。

好的牛肉（非冷冻冷藏肉）讲究吃其鲜味，因此**推荐五成熟或三成熟**。品质一般的牛肉才需要做到七成熟以上来掩盖缺点。如果餐厅服务员推荐七成熟，通常意味着其牛排品质一般。

关于价位，一份分量适中、价格在百元左右的牛排即可，无需追求过于昂贵的。

## 与同伴用餐的拍照策略

与同伴（尤其是约会对象）用餐时，如何自然地拍照是需要技巧的。

如果对方打扮得体，可以主动提出为其拍照。对方满意后，再请对方帮你拍照，整个过程会显得非常自然顺理成章。如果对方不愿帮忙，可以考虑下次是否再约。

当需要拍摄人物与餐食的互动时（例如对方坐在对面），建议采用**正方形构图**以提升逼格。拍摄人物时，姿势很重要：背部挺直，可微微前倾，但不要趴在桌上；气息要稳，姿态舒展。

## 用餐姿势与细节

优雅的用餐姿势是“无形装逼”的核心。以下以喝汤和吃牛排为例。

喝汤时，背部保持挺直微弯。整个过程可以分解为三个自然的姿势进行抓拍：
1.  将汤勺放入汤中。
2.  舀起一勺汤。
3.  将汤送入口中。

吃牛排时，礼仪更多。首先，好的牛排通常只搭配**黑胡椒和粗海盐**，而非厚重的酱汁。摆盘时，可将牛排斜放，并与配菜（如绿色蔬菜、小番茄）间隔摆放，显得舒展。

使用调味料时，不要直接撒在牛排上。正确做法是将胡椒或盐**撒在盘子一侧**，然后用切好的牛肉去蘸取。

使用刀叉时，**左手拿叉，右手拿刀**。刀尖不要指向同伴。双手姿势应优雅，手肘微弯，**手腕悬空，不要压在桌子上**。

## 切割与进食

切割牛排时，动作要优雅，使用阴力，**尽量避免刀叉与盘子摩擦发出刺耳声音**。切下的肉块要**大小适中**，方便入口。

进食时，用叉子叉起肉块，优雅地送入口中，然后**闭口慢慢咀嚼**，不要发出声音。如果想拍照，可以在切肉时或切好肉准备送入嘴时抓拍。准备送入嘴时，可以右手稍放低，左手抬高，看向镜头。

用餐中途若暂停，可将刀叉呈“八”字形放在盘中。用餐结束后，将刀叉并排放在盘子右侧。

## 照片后期与分享

前期拍摄到位，后期修图就很简单。核心在于前期拍摄的自然与优雅，后期无法弥补姿势的低俗。

后期主要是调整滤镜和基本参数，让照片色调舒适。如果拍摄了人物，可以进行适当的面部修饰。

分享到朋友圈时，可以精选两张照片组合：例如一张丰盛的美食全景，搭配一张优雅用餐的人物特写。

本节课中我们一起学习了西餐厅装逼的全流程：从简洁得体的着装、合理的点餐，到专业的摆盘构图、多角度的拍摄技巧；从牛排的选择、优雅的用餐礼仪，到与同伴互动的拍照策略，以及最后的照片处理与分享。记住，核心在于自然与细节，做到“无形的装逼”才是最高境界。
