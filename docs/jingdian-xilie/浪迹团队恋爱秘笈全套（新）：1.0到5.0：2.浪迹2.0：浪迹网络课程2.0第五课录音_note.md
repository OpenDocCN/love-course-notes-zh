## 心态建设与高阶技巧

**关于目的性与“无心插柳”**
通过学员“莎莎哥”和一个女生的实际案例，我想说明一个关键点：**目的性太强往往是失败的主要原因**。

**关于投资与带领**

一个重要的亲身体验是：**千万不要疯狂地、主动地进行DHV（展示高价值）**。这种行为非常令人讨厌。DHV应该自然地融入你讲述的有趣故事中，一点点透露即可。
