# 聊天训练营：第一课：开场白模型与实战

在本节课中，我们将学习聊天互动的核心模型，并重点掌握三种有效的开场白方式。通过理解模型框架和完成实战作业，你将能够顺利开启与女生的对话。

## 聊天核心模型概述

上一节我们介绍了课程的整体安排，本节中我们来看看聊天的基本流程模型。这个模型概括了从认识到邀约的完整步骤。

以下是聊天互动的五个核心阶段：

1.  **开场**：对话的起点。
2.  **破除陌生感**：适用于街头搭讪或社交软件，旨在建立初步联系。
3.  **安全感与舒适感**：进行基本的信息交换，让对方了解你，建立信任。
4.  **筛选**：测试对方的兴趣度和服从度。
5.  **邀约**：提出线下见面的请求。

> 注：在特定情况下（如追求快速关系），可能会涉及额外的“聊骚”环节，但常规约会遵循以上五个步骤。

我们将用七天时间详细讲解每一环节。现在，让我们进入第一课的重点：开场白。

## 三种开场白方式详解

开场是建立联系的第一步，一个好的开场能显著提高获得回应的概率。以下是三种经过验证的开场方式。

### 1. 表情开场

这种方式非常简单直接，即发送一个表情符号作为对话的开始。

**代码示例：**
```
😊
😏
😢
😳
```

**应用场景：** 主要用于社交软件（如探探、陌陌）。在街头搭讪后使用时，通常需要在表情后附加一句话来主动开启话题。

### 2. 价值开场

价值开场通过分享状态或赞美对方，展示自身价值或表达关注，营造一种熟悉、自然的对话氛围。

以下是几个实用惯例：

*   **赞美头像**：`你的头像好萌。`
*   **分享状态**：`刚刚给学生讲课，中途脑袋空白了一下，好丢脸。`
*   **关联搭讪**：`刚和朋友吃完火锅，好辣。不过想起了今天，感觉还蛮回味的。`

**应用场景：** 适用于街头搭讪、探探、陌陌等多种场合。“头像好萌”惯例在社交软件上尤其常用。

### 3. 好奇开场

好奇开场通过制造悬念或提出有趣观察，激发对方的好奇心，从而引导她回应。

以下是几个实用惯例：

*   **假装相识**：`是你？`（营造似曾相识的感觉）
*   **提出观察**：`我发现一个特点。`（对方通常会问“什么特点？”，可接`你面相很喜庆`或`你好像面相旺夫`等）
*   **具体赞美**：`我发现你的眼睛好像梅花鹿。`

理解其原理后，你可以自行创造更多类似的开场白。
