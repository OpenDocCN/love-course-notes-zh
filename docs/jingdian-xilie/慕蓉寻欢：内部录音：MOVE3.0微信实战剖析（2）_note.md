## 控制需求感的重要性

这里需要说明：如果你唱歌五音不全，反而可能更好，因为这带有娱乐心态。但关键在于，唱歌时不能显得非常兴奋。如果你的情绪过于亢奋，无论是说话还是行为，都会暴露出你对她的需求感。

回顾我上一段和这一段的聊天，我的需求感都非常低。为什么要用低需求感去聊天？因为当你的需求感越低时，你对对方没有任何欲求，不会让她感觉你想从她身上索取什么。这时，你在整个关系中就拥有主导权和主动权。

例如，牙叔之前喜欢一个目标，我反复告诉他不要对她百依百顺、不要那么顺从。但后来牙叔没过多久就表白了。一旦表白，你的需求感就暴露了，在之后的相处中就会失去主动权。

## 核心模型：诱惑加速论

以上两个案例就分析到这里。你们从中感受到或学到了一些东西吗？

这里我分享一个内部模型，叫做“诱惑加速论”。这是一个针对社交软件聊天的系统模型，分为七个步骤。但请注意，不是每个目标都需要完成所有步骤，它揭示的是网聊的一般规律。

以下是该模型的七个核心步骤：

1.  **展示**：指你的展示面，包括微信朋友圈、头像、个人资料，乃至你的衣着搭配、拥有的资产（如车）等。这是第一印象。
2.  **锁定**：确保目标愿意继续与你聊天。如果前期无法“锁定”对方，后面的步骤都无从谈起。例如，在线下，通过道具互动（如互换佩戴物）可以建立连接；在线上，则需要通过提供有趣、有价值的话题来吸引对方持续互动。
3.  **性张力**：在互动中适当加入与性相关或能引发性联想的内容。例如，让目标“帮忙洗内裤”或提及“一起洗澡”，这能在对方脑海中创造画面感。性张力不宜过早或过多使用。
4.  **演绎**：通过语言、语音创造画面感和情绪波动。这是后续课程的核心内容之一。
5.  **观念重组/重塑**：输出你的价值观，影响或改变对方的某些观念。这对高质量或框架强的女性尤其重要。
6.  **安全感**：在整个互动中，让对方感到被尊重和理解，觉得与你见面是安全的。这通常贯穿在整个互动过程中。
7.  **TD（推倒）**：最终目的。本节课不详细展开。

这个模型的价值在于它将零散的网聊技巧系统化了。

## 低需求感的本质与练习方法

最后，新宇老师补充一点关于低需求感的体会。

从我旁观寻欢聊第一个案例（那个声音很撩人的女生）的过程来看：任何一个正常男人听到那种声音都会有生理和心理的冲动，需求感会飙升。寻欢也不例外。

但关键在于，当他每次要回复语音时，他会先屏住呼吸，调整身体状态，以最放松、最平稳、最稳重的状态去回复。也就是说，他先控制住了自己的生理和心理反应，然后再进行互动。

这就是线上练习的意义：当你在线上聊天感到兴奋时，不要马上回复。隔一两分钟，调整好情绪，以最平静、最有框架的一面去回应。通过不断的线上练习，强大自己的心态，然后再应用到线下互动中。

低需求感是我们需要具备的基本素质。只有保持低需求感，你才能更好地掌握关系的主动权，建立自己的框架。这就是“无欲则刚”的道理。

例如，在第一个案例的观点重组中，我表达的含义是：“一段好的感情是两个人彼此付出。真正懂得爱你的人，是不会让你迷失自己的。”我只需要表达我的观点，她是否听懂，我并不关心。这就是框架。
