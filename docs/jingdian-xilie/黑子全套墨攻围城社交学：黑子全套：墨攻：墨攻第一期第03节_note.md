# 黑子全套墨攻围城社交学：第一章：03：课程意义与能力概述

在本节课中，我们将要学习本课程的核心意义，并初步探讨个人能力，特别是沟通能力的重要性。

## 课程的核心意义

大家不要以为花了一些钱来听课就怎么样。其实更重要的是我们相互认识的过程。因为课程有时间限制，会一期一期地结束。

但是我们的沟通是不断的。我今天下午特别开心，因为我的一位河北学员，比我小一两岁，他今天告诉我他马上要和女朋友结婚了。

我听了之后非常开心。下午我跟他打了一个小时的电话，讨论了他和女朋友之间发生的一些摩擦，以及未来怎么走。我觉得这是一件好事。这是我的学生，有事会想到我，告诉我，听听我的意见。

我通过他的事情给了他一些意见。然后他再回到现实中去操作和锻炼。这样他切切实实地进步和成熟了。所以说，我觉得这是我们课程最大的意义。

在课程中，我只能把一些基础知识给大家普及，简单地沟通你们目前存在的问题。更重要的是，我需要教大家一种解决问题的方式，一种思考的方式。昨天我首先给大家讲心态，就是因为心态是基础，我们需要有一个良好的心态。

我的每一节课之间都存在相互的联系，不是想到哪里就讲到哪里。

以上是我在课前和大家沟通的事情。也希望昨天和子玲沟通的学员，心理上不需要不爽，不需要有太多不愉快的感觉。因为你是我的学生，他不是了，就这么简单。

## 正式课程：能力概述

好，那么我们现在开始今天正式的课程。第二章，我主要给大家讲的是一个人的能力。

大家去思考这样一个问题：什么是能力？你们凭空想象能想象出多少种一个人可以具备的能力呢？

大家可以跟我互动一下。可以扣字，你们去想能力可以包括几种，什么能力。

以下是大家可能想到的一些能力示例：
*   语言方面的能力
*   沟通能力
*   技巧方面的能力
*   协作能力
*   协调能力
*   社交能力

很多东西都可以在一个名词后面，贯上“能力”两个字。有无数种能力。

## 本节重点：沟通能力

那么，我今天想给大家讲的第一点是**沟通能力**。这是我今天要讲的。然后呢，还会给大家讲**观察能力**。接下来呢，还要体现一个**表现能力**。最后给大家讲**听取分析判断的能力**。

当然，这些内容今天一天不一定讲得完，所以争取讲到哪算哪。因为下一堂课依然还会强化能力这一块。

我为什么要上来跟大家讲沟通能力？因为我觉得这个能力，从与女人交往的角度来说，是尤为重中之重的一种能力。我们去认识女性、向女性表述自己、谈情说爱等等，沟通能力显得尤为重要。

在沟通能力中，我分为了5个小点为大家陈述。

## 沟通能力第一点：形象

首先第一点是**形象**。我把形象放在沟通能力里来讲。为什么要放在这里？一个人和你见面的时候，首先听到的不是你的声音，而是看到你的整体形象。不同的人会给我留下不同的印象，这就是你的形象。

但是在这里，我需要强调一点，是希望大家可以做到：你的形象要保持住一种风格，要保持住两个字——**干净**。

一个人你的形象，不管穿多少钱品牌的衣服，浑身上下的衣服、裤子、鞋到底搭配不搭配，你的发型是烫了还是染了，还是光头，干净是基础。
