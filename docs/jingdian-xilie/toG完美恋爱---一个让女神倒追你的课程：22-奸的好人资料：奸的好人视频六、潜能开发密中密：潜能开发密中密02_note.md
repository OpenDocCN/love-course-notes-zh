# 完美恋爱课程：22：潜能开发密中密02

在本节课中，我们将探讨关于学习与改变的核心信念。我们将分析流行的“归零学习”观念，并通过科学进步与自然界的例子，揭示成功改变所必需的真实态度与条件。最后，我们将介绍构成成功的五个关键因素。

## 关于学习的态度

上一节我们介绍了课程主题，本节中我们来看看关于学习应有的态度。

市场上流行“终身学习”的观念。学习需要保持特定的信念。许多导师会告诉你，学习必须“归零”。所谓“归零学习”，意思是像一个装满水的杯子，必须先倒空才能装入新水。他们认为，只有成为“空杯子”，才能接受别人的知识。

但这里存在一个问题：你如何判断别人倒入的是“清水”还是“脏水”？“归零学习”的态度是否正确，抑或只是流行说法？

## 科学进步与“归零学习”

值得注意的是，整个科学进步史中，并不存在“归零学习”。以著名科学家牛顿为例，他曾说自己的成就是“站在巨人的肩膀上”。同样，所有科学进步都基于**归纳法**，即总结前人的经验，形成体系供后人遵循。

科学还有另一种精神：**打破定理**。但它的首要前提是必须站在前人（巨人）的肩膀上，才能推翻旧理论并建立新理论。因此，人类文明与科技的演变，从未依靠“归零学习”，而在于如何继承与发展。

## 改变需要付出的代价

很多人会谈论改变，但人类究竟如何改变？我们可以从演化过程中找到例子。

老鹰是鸟类中最长寿的动物之一。当老鹰活到50多岁时，身体会发生巨大变化：嘴角变长抵至胸部，捕食困难；羽毛变得又长又厚，难以高飞；利爪变钝，无法捕捉猎物。

此时，老鹰若想获得突破与转变，必须改变自己，但这需要付出沉重代价。它只有两条路：等待衰老死亡，或是选择重生。

重生的过程包含以下三个步骤：

1.  老鹰需飞上高山，用嘴猛啄岩壁，直到旧喙完全脱落，等待长出新的尖嘴。
2.  利用新长出的尖嘴，将老化的指甲和利爪一根根拔除，等待新的利爪长出。
3.  将身上又长又厚的羽毛一根根拔掉，忍受寒冷，等待新的羽毛长出。

经历这三个痛苦阶段后，老鹰才能获得新生，重新翱翔天际。

同理，一个人若想从A性格转变为B性格，例如从软弱变为坚强，也必须付出相应代价，才能实现生命的重大改变。

## 成功者的五个条件

就像老鹰的重生需要三个步骤，成功者也由特定条件构成。我曾通过一个心理测试，发现了构成成功的五个关键因素。

当时，我与数位本地及外国企业家共进晚餐。席间，大家谈起如何赚到人生第一桶金，气氛热烈。我有做笔记的习惯，便将他们的成功经历记录下来。在记录过程中，我突然发现，这些人之所以能成就大业、成为市场赢家，离不开五个共同因素。

我随即在餐巾上写下这五个条件，并将其推到餐桌中央。当所有企业家看向餐巾时，我向他们提问：“在你们看来，这五个因素中，哪一个最重要？”

我要求他们只选一个最重要的因素。结果，现场发生了奇怪的现象：企业家们开始激烈争论。这种争论并非导致不欢而散，而是越争论越兴奋，话题也越来越多。

他们的争论甚至吸引了邻桌的企业家加入，最后连餐厅老板也忍不住参与进来。这让我首次见识到这五个问题的巨大威力。

第二次，我在与一位商业大亨会面时再次使用了这五个问题。对方起初态度强势，只给我两小时会谈时间。但在谈话中，我巧妙引入这五个问题，情况发生了巨大转变。会谈从原定的两小时延长至六小时，他甚至忘记了吃饭，并邀请我继续回办公室深谈。他向我详细讲述了自己从建筑工人到保险业，再到成为马来西亚首个举办公开展览的企业家的发家史，以及目前在中国的投资情况。

这五个关键因素，即我们今天要告诉你的：**鉴定手段、性格、资源、技能与机遇**。

## 总结

本节课中，我们一起学习了关于学习与改变的核心观点。我们分析了“归零学习”的局限性，认识到真正的进步建立在继承与发展之上。我们通过老鹰重生的例子，理解了任何深刻的改变都需要付出巨大代价。最后，我们探讨了构成成功的五个关键因素：鉴定手段、性格、资源、技能与机遇。理解这些，将为你个人的成长与发展提供坚实的方向。
