# 魔鬼咨询师VIP课程4.0：公开课 SameDirection

在本节课中，我们将跟随一位学员的亲身经历，了解他从社交新手到掌握有效方法的转变过程。课程将涵盖搭讪、短信沟通、约会技巧以及心态调整等核心环节，并通过具体案例揭示常见错误与正确做法。

## 1：初识搭讪：从怀疑到实践

上一节我们介绍了课程背景，本节中我们来看看这位学员是如何开始接触搭讪的。

我于2008年研究生毕业后来到北京。一次偶然的机会，我在网上看到了魔鬼咨询师关于教授搭讪的帖子。当时的我感到非常震惊，因为直接在大街上与陌生女孩交谈并索要联系方式，完全超出了我以往的认知范围。

巧合的是，当时只需支付200元即可参加在五道口举办的搭讪课程。我没有犹豫，立即乘坐城铁前往。到达后，我拨打了咨询师的电话，接听的是一位声音好听且感觉年轻的人，他指引我到了指定地点。

## 2：首次实践与观念冲击

在前往课程地点前，我曾潜意识地认为，学习搭讪的学员应该都是外表出众、打扮时髦的帅哥。然而，到场后我发现大多数学员都和我相似，估计多数也从事IT行业。

最让我受刺激的是，这些外表普通的学员，竟然能与在我看来非常漂亮、生活中罕见的女孩谈笑风生地互动。这个场景让我彻底放下了顾虑。当咨询师为我指定一个目标后，我毫不犹豫地冲了上去，完全没有新手的搭讪焦虑。

我上前对那位女孩说：“嗨，你好，我想认识你。”她回应说可以，但需要急着去买眼霜，店铺快关门了。我立刻表示没问题，并快步跟上与她聊了起来。这位被我称为“五道口美眉”的女孩，至今仍是我的好友。

她买完东西后，我低头兴奋地记下了她的电话号码。一抬头，却发现她身高一米八几的模特男友就在旁边等着。后来得知，她男友是位家境优渥的富二代。我看到后便立刻离开了。

首次搭讪的经历非常顺利。那天我一共尝试了6次，成功获得了4个号码。按照今天的标准，这也是一个很高的成功率。

## 3：瓶颈出现：低约出率的困境

虽然能成功获得号码，但我很快遇到了严重的问题。在北京的一年里，我大概尝试了100到120次搭讪，获得了五六十个号码，数量可观。

然而，真正能成功约出来见面的，回忆起来只有5个人，约出率仅为10%左右。约出率低下的原因显而易见：我的短信沟通能力很差。

当时课程主要教授搭讪，对短信技巧涉及很少。我曾将自己与女孩的短信对话拿给咨询师看，他的评价是：“你这个短信怎么发的这么二啊！”

当时流行的一种方法是，将获得的10个号码在某个时间点群发“在干嘛呢？”，然后只与回复的那几个人深入聊天。这个方法对于有一定短信沟通基础的人可能适用，但对我这样发短信很“挫”的人来说，效果很差。

我的工作非常忙碌，平时根本没有时间与女孩好好聊天。每到周末有空时，我才想起要约人，于是便群发“今天有空吗？一起出来吃个饭？”。女孩们收到这样的短信，自然不予理睬。因此，当时的约会情况非常惨淡。关键原因就是不懂如何有效地进行短信沟通。

## 4：约会中的经典错误案例

即使偶尔能成功约出女孩，我在约会过程中也犯了许多经典错误。

**案例一：机场地勤女孩**
我曾在北京机场搭讪一位地勤女孩，过程非常顺利。我们一同玩耍，我甚至搂了她的腰防止她跌倒。坐在长椅上聊天时，我说了一句搞笑的话，她笑得特别厉害，还顺手打了我一下。这让我感觉非常好。

回去的公交车上，我收到了她的短信：“你做我男朋友好不好啊？”当时，我以典型的男性思维回复道：“其实我们还挺有缘分的，聊得挺开心。但我们还是要互相了解一下，考虑成熟了，你也喜欢我，我也喜欢你，那时才可以成为男女朋友。”

经过咨询师分析，这是一个特别错误的回复。原因在于完全不了解女性思维，不知道女孩当时是怎么想的，只是从自己的逻辑出发去回应。

**案例二：中国传媒大学的女孩**
我搭讪了一位身材很好的女孩，回头发现相貌也非常出众，按标准至少8分以上。我们聊得不错，也约会了一两次。后来她请我去修电脑。

我觉得女孩找我帮忙，一定要保证完成任务，达到最好效果。于是我坐城铁过去，没多聊就开始重装系统。她的索尼笔记本有专用恢复盘，结果电脑没修好，我们也没有充分交流，女孩还生气了。在那种气氛下，我感到很挫败和拘谨，不知如何活跃气氛，导致局面越来越僵。

临走时，我想再聊聊天。当时天气很冷，我选择了她寝室楼下人来人往、吵闹的会客区，而不是附近的咖啡厅或音乐吧。结果气氛越聊越冷，最后我不得不主动离开，她也如释重负。这又是一个不懂约会的典型案例。

**案例三：身材高挑的女孩**
我幸运地约出了一位身材、走路姿态和相貌都非常出众的女孩，打分在8.5到9分。她非常会聊天，短信中常开玩笑。我问她学什么，她说“练铁人三项的”，我竟然信了并开始分析，其实她是学服装设计的。

我们约在麦当劳见面，因为之前开玩笑说请吃麦当劳。约会时聊得还不错。但离开时我又开始犯错。站起来后，身高至少1.70米的她看着1.72米的我，说：“你怎么这么矮啊？好像不高。”我一下子语塞，不知如何回应，只是低着头走路。

咨询师后来告诉我，当时应该幽默地回应，比如“我知道我错了”。回去时我们坐的地铁，这也不太好。既然花了时间和金钱约会，如果没有车，至少应该提出打车送女孩回去。如果女孩懂事并坚持坐地铁，再坐地铁不迟。之后我再给她发短信，她一条都没回过。

这些惨痛经历的主要原因是：会搭讪、能要号，但不懂得如何发短信交流，也不知道约会的基本要领。

## 5：惨淡的相亲经历与反思

结束北京的生活回到杭州后，我几乎中断了所有搭讪和相关学习。这段经历我回忆不起来，可能太痛苦了。原因可能有两个：一是在北京时大家一群人玩很有气场，但我没有培养出独立的搭讪人格，独自一人时气场不足，受挫后容易放弃；二是工作非常忙碌。

于是，父母开始为我安排相亲，经历也相当惨痛。

以下是几个印象深刻的相亲案例：

*   **案例一：** 第一个相亲对象，见面后发现外貌与期待差距极大，我只能硬着头皮应付，带她去便宜的地方随便吃了顿饭。
*   **案例二：** 介绍人声称“妹妹很漂亮，姐姐肯定不会差”。见面发现对方身材非常肥胖，腿粗到无法正常交替迈步，还穿着细高跟鞋。我一上车就感觉车震了一下。这次我学聪明了，只带她喝了20分钟饮料就结束。
*   **案例三：** 相亲对象外形有些畸形，体毛很长，沟通存在严重障碍。我说搞计算机（IT），她完全听不懂，解释电脑是“上网聊QQ、看网页、有键盘鼠标的”也不明白。她介绍自己的工作，我也没听明白。

经历了极其惨淡的相亲后，我感到非常气愤和沮丧。我开始怀疑，难道在介绍人和亲友眼中，我只配和这样的人在同一层次吗？这甚至引发了家庭矛盾。

## 6：重拾希望与系统学习

在人生非常暗淡的时期，我想起了魔鬼咨询师和他的课程。我给他打了电话，他立刻叫出了我的名字（SameDirection），让我感到一阵温暖。与他聊了我的现状后，我豁然开朗，感觉人生又出现了光明。

我当即报名了他的网络课程。作为老学员，我享受了一点优惠。开始学习后，我发现新课程已经非常系统化和理论化，便于理解和学习。

我自认情商（EQ）不高，社交直觉不强，因此需要咨询师总结出简明扼要的框架和法则，例如“遇到这种情况，你就这么做”。有了这些框架和成功案例作为基础，我再去感性体会和认识，就能形成自己的理解，更灵活地运用理论。

## 7：关键提升：心态、选目标与法则

然而，起步阶段仍有困难，尤其是在选择搭讪目标上。我当时容易被打扮妖艳、穿着高跟鞋丝袜的女孩吸引，而忽略了那些穿着普通（如T恤牛仔裤球鞋）但可能身材样貌有7、8分的女孩，导致感觉找不到合适目标。

后来，咨询师来到杭州，与他接触让我感触很深。他让我感受到，他是一个能够欣赏和体会生活中美好的人。无论是他教学的短信内容，还是他个人的气息，都让人觉得生活可以很有趣、很美好。这改变了我，让我明白最重要的不是明天，而是过好每一个今天，让当下开心幸福。

在选目标方面，经过咨询师指导和课程培训，我也有了进步。现在，我对自己的约会生活已经很满意了。有段时间状态很好，21天里有14个约会，成功约出了多位姑娘。

## 8：核心法则：三三法则的应用

有一点我想特别强调：有些东西在我们自身的水平和层次上很难摸索出来，需要依靠咨询师和众多学员共同总结的法则。

让我印象最深、很受震撼的一个法则就是“三三法则”。说起来简单，但难就难在应用。大家发短信时可能都遇到过这种情况：你发一条，她回“呵呵”；再发一条，回“哈哈”；再发一条，她说“我要去洗澡了”，或者回“哦”。遇到这种情况，我以前会感到无奈、低落甚至气愤，觉得对方把我当屌丝对待，有时就放弃了，或者情绪激动地指责对方。

在咨询师指导并了解三三法则之前，我遇到这种情况要么放弃，要么有一次特别气愤地发短信说“像你这样的两个字短信，你就不要发了”。结果那个女孩马上打电话过来，解释说感冒咳嗽身体不舒服，还咳了两声。但下次我发短信，她干脆不理我了。

三三法则是一个理性且高效的交友沟通方式。只要你遵守这个方式，从长期来看，你的约出率能够接近50%。你无需纠结女孩回“呵呵”还是“哦”，也不用去想“从她回呵呵到成为男女朋友距离有多远”。你只需要按照法则做好该做的事，以最高效率去处理，就一定能成功，达到像我这样的约出率。

实际上，我认为自己的短信和约会水平只是一般，可能刚及格。如果大家系统学习，相信凭借更好的基础，能取得更优异的成绩。

## 总结

本节课中，我们一起学习了这位学员从对搭讪感到不可思议，到亲身实践并遇到瓶颈，再到通过系统学习克服困难、提升自我的完整历程。核心问题集中在**短信沟通能力不足**、**不了解女性思维**以及**缺乏基本约会技巧**。解决之道在于**系统化的理论指导**（如三三法则）、**正确的心态调整**（关注当下美好）以及**持续的实践与反思**。对于社交直觉不强的人来说，明确的框架和法则至关重要，它们能提供有效路径，避免陷入个人思维惯性的误区，从而显著提升情感社交的成功率。
