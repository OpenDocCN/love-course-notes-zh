# 手机影像入门：课时04：寻找最佳自拍角度

在本节课中，我们将学习如何根据不同的脸型，找到最适合自己的自拍角度，从而拍出更上镜、更有格调的照片。

上一节我们介绍了自拍的基础准备，本节中我们来看看如何通过调整角度来扬长避短。

## 自拍常见错误

在开始学习具体角度前，我们需要先了解几个自拍时应避免的常见错误。以下是几个关键注意事项：

*   **光线要适中**：自拍环境不能太暗，也不能太亮。
*   **构图忌太满**：即使外貌出众，也不要让整张脸填满整个屏幕。
*   **放弃老土姿势**：例如“剪刀手”等过于常见的姿势。
*   **注意手臂入镜**：避免让白花花的手臂在画面中显得突兀。
*   **善用自拍杆**：不要觉得自拍杆不够时尚，在需要时它是实用工具。
*   **留意背景**：选择一个简洁或不杂乱的背景。

## 圆脸的自拍角度

圆脸通常看起来可爱，适合卖萌，但自拍时两颊的肉感可能不太上镜。

那么，圆脸在自拍时，选择怎样的角度才能让脸型更显清瘦、轮廓更立体呢？我们可以尝试将手机举高，采用**45度俯拍**的角度。

这个角度之所以流行，是因为它能**放大眼睛、缩小下巴、显露腮骨线条、拉长脸部轮廓**，从而达到显瘦的效果。自拍时，可以尝试睁大眼睛，并利用手势。

以下是适合圆脸的手势建议：

*   **猫爪式或握拳**：将手成拳放在颧骨两边。
*   **遮阳式**：将手放在额头附近，仿佛在遮挡阳光。

在平拍时，圆脸美眉可以尝试将脸侧转**45至60度**。通常选择自己更上镜的一侧脸对准镜头（多数人选择左脸）。这个角度同样能让脸在镜头前看起来更清瘦。

## 梨形脸的自拍角度

梨形脸的特征是额头较窄，下巴两侧较突出，下半部分脸型较宽平。

自拍时，我们的目标是遮挡或弱化下巴较宽平的部分。可以尝试**45度角**的自拍方式。梨形脸通常更具文艺气质，因此手势上建议采用**托下巴**的姿势，用手势自然遮挡住下半脸较宽的区域。

平拍时，可以通过神态和手势转移视线：

*   **眼神放空或迷离**。
*   **将手放在嘴唇上**，仿佛在诉说秘密。
*   **使用可爱的小道具**，遮挡住部分脸颊。

## 长脸的自拍角度

对于长脸型，如果有刘海（斜刘海或长刘海）会更容易修饰，可以遮挡部分额头，缩短脸型的视觉长度。

如果脸颊清瘦，容易显得没精神，这时表情应尽量生动。建议采用**从下往上**的仰拍角度，将手机放在大约与胸齐平或稍低的位置。可以**微微低头，眼睛向上看**，并搭配笑容。**侧脸角度**也非常适合长脸型。

## 瓜子脸的自拍角度

瓜子脸（包括锥子脸）是所有脸型中最上镜的，对各种角度的适应性很强。

无论是俯拍、仰拍、左侧拍、右侧拍还是平拍，瓜子脸的美眉在自拍时没有太多限制。核心要点是**保持自然，避免做作**，捕捉最真实的瞬间。

## 总结与练习

本节课我们一起学习了如何根据圆脸、梨形脸、长脸和瓜子脸的特点，选择对应的最佳自拍角度和手势。

为什么有时角度和姿势都对了，拍出来还是不好看？请别担心，除了技巧，设备和练习同样重要。不要过于与他人比较，**多练习**是找到最适合自己风格的关键。
