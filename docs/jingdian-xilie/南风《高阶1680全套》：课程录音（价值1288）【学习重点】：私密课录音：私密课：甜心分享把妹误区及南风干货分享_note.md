### 第三部分：四种常见的错误心态及颠覆

**聊天非常重要，它决定了你和这个人能不能继续走下去。但我们不能把所有阶段都停留在聊天，甚至精心编排每一句话。聊天只是一块敲门砖。** 如果你觉得这几天聊得不错，非常建议把女生约出来。通过“网上聊天 -> 约会 -> 再聊天 -> 再约会”这种渐进的方式，才是比较正确的。而不是在网上聊了365天却一次面都没见过。

**总结：** 聊天是感情的敲门砖，但**核心在于面对面的约会和渐进的关系推进**。

**总结：** **避免直接明确的表白**，采用暗示性话语进行关系试探，根据对方的回应逐步推进。
