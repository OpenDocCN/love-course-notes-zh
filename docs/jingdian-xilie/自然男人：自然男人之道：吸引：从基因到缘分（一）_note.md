# 自然男人之道：1：吸引：从基因到缘分（一）🧬➡️💞

## 概述
在本节课中，我们将要学习两性关系中“吸引”的本质。我们将从生物学和进化心理学的角度出发，探讨男女相互吸引的底层动机，并分析这些理论在解释人类复杂情感和行为时的局限性。课程旨在找回被现代媒体遮蔽的历史智慧，理解吸引不仅是生理本能，更是深层次的心理与文化现象。

## 男女相互需要的基本动机
关于男女关系的学问，古已有之。但现代人常被媒体信息束缚，难以看清历史全貌。本系列课程的目标，正是找回这些丢失的智慧。

从现实角度看，男人需要女人，女人需要男人，这是基本事实。但男女结合的需求究竟是什么？许多男人在得到女人后，反而感到空虚，这涉及一个基本动机问题。该动机随着时代变迁而变化。

## 生理与心理层面的动机
本节我们将探讨生理和心理两个层面的动机。

从心理层面讲，一个常见的词是“安全感”。女人寻找男人，常是在寻找一种安全。但安全感具体指什么？男人如何知道自己能否提供安全感？这是今天要讨论的问题之一。

另一个问题是“成就感”。在当今社会，男人对女人，常被视为一种成就，得到一个女人象征着成功。人们寻求的往往是这种感觉。因此，男女是相互需要的。

男女结合并非简单的物质概念，它包含深层次的需求。接下来，我们先从生理方面探讨这种需求。

## 基因与进化论的视角
首先要谈的是基因。达尔文提出的进化论指出，生物物种是不断变化的，人类由猩猩、猴子进化而来。进化生物学作为重要分支，研究物种如何进化、自然选择如何进行。例如，为什么一只孔雀会被雌孔雀选中，而另一只不会？

人类也是一种生物，因此可以从某些角度比较观察人类的变化。相关学科包括灵长类动物学，以及专门研究人类的进化心理学。进化心理学不只研究人，还通过人类进化历程来理解行为。从200万年前人类社会形成开始，那时的人与现代人看到异性时的反应截然不同。虽然当时人类在体型、毛发、直立程度上已接近现代人，但心理仍在不断变化。进化心理学正是研究这种变化及其原因。

例如，中国人与西方人存在差异。从文明角度追溯，可归因于两千年前老子、孔子奠定的宗教文化，与西方耶稣基督教文化的不同，文字也不同。但这只是一方面。为何会创造出不同的文字与文化？这可以进一步追溯到进化心理学的范畴。

若要深入研究基因、进化及人类生理差异的根源，涉及上述多个领域。本节课将借用进化心理学领域的框架。

## 基因支配下的繁殖动机
基因的存在支配着男女（或雌雄）繁殖后代的行为。例如，理查德·道金斯在《自私的基因》中指出，生物有一种先天冲动，受基因掌控。雄性看到雌性就想交配，公狗看到母狗就想交配，这是基因控制的行为，目的是繁殖后代。

人类也需要繁殖后代，这必须依靠男人和女人。在基因理论中，这种繁衍被认为是人类两性关系最根本的动机。在此动机下，不同生物发展出不同的求偶现象。

求偶通常是雄性追求雌性。方式是通过展示吸引力或优势，例如击败其他雄性，或展现更优质的体魄和特点，从而让雌性选择自己。这在生物界很普遍。

雌性则根据雄性的表现选择最佳配偶。最佳雄性意味着其基因也是最佳的。当然，在更复杂的生物群体（如处于食物链较高位置的黑猩猩）中，还存在其他情况。例如，雄性被选中可能因为其地位和资源更高。猴王能与最多的母猴交配，而最弱小、资源最少、地位最低的猴子则没有交配机会。这在动物界是较明显的现象，在大部分动物群体中都可观测到。

该理论的基础认为人类也是如此。从某种角度讲，人类确实继承了这方面的特性。在基因框架和进化心理学中，行为模式、选择模式、社会模式都受此影响。如果雄性的基因使其个体最适应环境，那么它就是最优秀的雄性，从而繁衍更多后代。这可以用八个字总结：**物竞天择，适者生存**。

在人类社会中，如果你生存有问题，机会就很小。如果你生存得很好，地位更高，资源更多，你在两性关系中的吸引力优势就更大。某种程度上，我们确实能看到这一点。

## 基因理论的启示与基本问题
虽然基因和进化心理学理论对我们有很大启示，但基因框架存在一些基本问题。以下列出四点进行说明。

### 1. 费希尔发现的“失控选择”现象
首先以动物为例，鸟类中的孔雀存在一种“失控选择”现象。越有吸引力的雄孔雀，意味着其尾巴越漂亮、越大，上面的眼状斑纹越明显、越亮。雌孔雀选择时，雄孔雀会排队或在特定区域展示翅膀（它们不打斗），雌孔雀走过并观察，最漂亮的那只会优先获得交配机会。它在每次选择中都能被选中，其后代成活率或数量最多。

但存在一个问题：雄孔雀的大尾巴越大，它就越难逃脱天敌的捕捉。也就是说，吸引力越高，意味着风险越高，生存率越低。费希尔发现了一对相悖的现象：**最吸引雌性的特质，反而最不适合生存**。当吸引力特质被无限放大时，会产生反向作用。

应用到人类身上是什么意思？观察城市与农村人口，或高素质与低素质人群的区别，你会发现：越是素质高、聪明的人群，其后代越少；越是贫穷、落后的地方，反而后代越多。这说明并非越有吸引力就越能促进生存，吸引力指向的是别的东西，而非简单的优质基因问题。

### 2. 黑猩猩社会中“头领”的动态性
第二个现象关于黑猩猩部落中的“头领”。在黑猩猩部落中，雌猩猩会带着年轻雌猩猩一起长大玩耍，雄猩猩则带着年轻雄猩猩，男女分开，这与人类原始部落很相似。雄猩猩群落常有较多争斗，大多是为了争夺首领地位。

黑猩猩中的“头领”是雄猩猩争夺的目标，就像人类争夺冠军或首相职位一样。在基因理论中，对于两性关系的说法是：男人中地位最高的，会得到最多雌性的吸引。也就是说，如果你有这样的地位，就能得到更多女人的注意和吸引。这在一定程度上确实发生。

但有一个问题：在黑猩猩社会中，头领并非固定不变，而是频繁交替，一天可换好几次。如何更换？例如，一只猩猩发现一根树棍，折下来后因其较长较硬，便挥舞旋转，吓跑其他猩猩，此时它就成了头领。或者另一只猩猩发现一个铁罐，踢来踢去发出声响，吓跑其他猩猩，它便成为头领。等到其他猩猩发现是铁罐发声，过来抢夺时，第二只猩猩就变成头领了。

因此，“头领”地位是会变化的，并非固定不变。它随环境和情况的变化而变化。许多动物群体都是如此。有些动物群体（如狮群）有相对固定的首领，例如猴王能在位一段时间，但并非所有猴子都认为它是王，而是在第二只猴子出来争夺之前，它是王。这是一个动态概念。

换到人类社会也是如此。你可能在一个圈子里是老大，换一圈人可能就不是了。所谓“头领”的吸引力只是相对的、动态的概念，没有什么是最终决定性的，也没有固定不变的因素。基因因素并非如此决定这一点。

### 3. 动物价值与人类价值的区别
第三个问题是动物价值与人类价值的区别。可以粗浅地看：动物活着没有其他追求，没有美学、文学、探索宇宙的思维，这一切都没有。对动物而言，最大价值是生存。在这套基因进化论理论中，这确实合理。

但作为人，这就产生了一个问题：人不是动物。人类活着的价值是否只是活着、能生存就行？人是有追求的，人想要实现某种东西，人活着不能简单地只为活着。人是有意识的，需要在生存与意识之间做协调。人有一些特殊的价值追求，没有人会说只要活着就好。

如果用进化论和基因理论套用在人身上，存在一个基本问题：对动物合理的，对人未必如此。人不能简单地被等同于动物。同时，人还有另一个层面的需求，另一个层面的世界，可称之为“人文的精神世界”。

从基因论和进化心理学中，我们看到的是一个科学事实的世界，一个用数据和现象说话的世界。但人类的美感、故事对心智的作用、我们的困惑，这些怎么办？两性之间不是一个简单的交配问题。我们择偶、组建家庭，无论东方西方，都涉及人的需求，而非动物需求。

因此，虽然进化论和生物学理论给了我们一些启示，帮助我们理解某些东西，但我们要反过来看：人类是否仅仅如此？我们要尊重事实。

### 4. 人类吸引力的心理本质
例如，狗发情时交配，公狗见到母狗没有别的反应，第一反应就是上前交配。但对于人，男孩子们，你们看到一个美女，第一反应是心理的还是生理的？有些人会说“生理的”，但实际上不是。仔细想一想，第一眼看到一个女孩，第一反应是心理的，先是心动。

因此，人的第一需求、配偶选择包括吸引力这个概念，我们看到一个人为什么会被吸引，很大程度上不是一个生理问题。明白这一点，就知道进化心理学研究虽有贡献，但非常局限。要了解吸引力的本质、工作模式，我们需要探索很多，其中最重要的就是心理问题。

例如，男孩子想要一个女孩子当女朋友或结婚，是不是什么人都行？不是。女孩子也一样，甚至更苛刻。我们如何选择？是什么告诉我们这个人行不行？很多时候并非清醒的意识。你看到一个女孩，喜欢与否，有时说不清，但你知道。喜欢一个人不是靠意识决定的。吸引力不是一种意识，而是一种下意识、一种感受。

有些人会说，女孩子会找个子高、长得帅、家里富有的男孩子。不排除这种可能，什么样的男人都会有女孩子喜欢。但是不是长得高、有钱、帅就一定有女孩子喜欢？不一定，而且不是所有女孩都喜欢。你见过所有女人都喜欢的男人吗？没有。

还有其他层面的问题，不光是看起来如何，还有一种感受。与人接触后，他的教养、基本素质、有无上进心等细节，是否也能决定一个男人的吸引力？所谓的吸引，远远大于基本的生理问题。它不是定量的，而包含文化因素：一个人的文化层次、心理感受好坏。这是动物界的基因论研究无法达到的高度，是无法用科学事实显示的现象，但我们都知它存在。

此外，即使你让一个女孩子觉得很喜欢、很有吸引力，她也可能不选择你。例如，有女孩子会觉得这个男孩子缺乏安全感，虽然被他吸引，但不会选择他。男人也一样，会被很多女人吸引，但会不会选择她们？不见得。

## 安全感与成熟
说到男人，女孩子常提到一个关键问题：安全感。安全感到底是什么？安全感是某种成熟，成熟的男人能提供某种安全感。那什么是成熟？这里要讲的就多了，我们需要看几个关键问题。

刚才我们讲了半天生理方面的问题。除了人类，动物有成熟与不成熟之分吗？没有这一说。因此，成熟与否是心理上的，是人类社会特有的东西。而且它在两性关系中，例如女人对你的看法和选择，是决定性的。

学习人类社会中这种变化的最佳途径，是理解原始社会中人的成熟是什么。最典型的例子是，在几乎所有原始社会中，都有一个特定仪式：**男性成人仪式**。当小男孩长到性成熟，开始对女孩感兴趣时，部落酋长会把这个孩子抓起来，进行某种启动仪式。仪式可能包括抽打、浸泡在河里几乎不能呼吸、让虫子爬满身体，或村里其他人化妆成魔鬼将他从妈妈身边夺走，进行残酷的肉体折磨。

通过身体折磨，他在此过程中产生心理变化：从不成熟心态转变为成熟心态。这个变化是什么？人类学对男性成人仪式的研究指出，关键变化在于：之前男孩子的不成熟心态是“依赖的心理机制”。某种程度上，他依赖大人，有困难或危险时会投向大人寻求保护。转型后发生的变化是“责任机制”：当危险、困难或任何消极事情出现时，他会先站出来担当。这是在原始部落中的一个概念。

这种从依赖机制到责任机制的转型非常关键。首先，男孩子是达不到的。例如，澳大利亚某些部落的仪式类似于此：用袋鼠和狗打架等方式进行，男孩子蹲在那里看。如果孩子没通过，例如吓跑或在拷打时昏过去，只要显示他已失败，部落的人会把他杀掉并煮了吃。为什么？因为他不能达到成熟的要求。

因此，所有通过仪式的男孩都会成为男人，没通过的都被杀掉吃了。部落里凡是过了那个年纪的，都是成熟男人。这对部落的生存至关重要。因为部落生活在自然环境中，需要打猎、面对凶猛野兽和其他部落侵袭。如果一个男人在学习打仗、制作工具、拼杀时，心理机制却是回去找妈妈、依赖别人，那么这样的个体不能要，是害群之马。男人的成熟与否决定了部落的存亡。

今天，我们的社会已经变了，不再有这种启动仪式，也不可能再做这样的仪式。但男人在社会中仍有成熟和转变的过程。这个过程特别复杂，下节课我们再慢慢涉及这方面的知识：今天的成熟意味着什么。但要记住最基本的一点：这不是一个生理问题，成熟是心理结构的变化。它对女孩子是否选择你，甚至两性的吸引力（所谓的那种触电感受），很大程度上是由这个心理结构决定的。

## 总结
本节课我们一起学习了吸引力的多维度本质。我们从基因和进化心理学的角度出发，理解了繁殖作为两性关系的底层动机，以及“物竞天择”在动物界的体现。同时，我们深入探讨了基因理论的局限性，包括“失控选择”现象、领导地位的动态性、动物与人类价值的根本区别。最后，我们指出人类吸引力核心在于心理层面，并引入了“安全感”与“成熟”的概念，通过原始社会的男性成人仪式，说明了心理结构从依赖到责任的转变对吸引力的决定性作用。下节课，我们将继续探讨现代社会中的成熟意味着什么。
