# 撩妹高手自我修养：二：V花间 (1)

在本节课中，我们将探讨在追求心仪对象时，许多人普遍缺乏的几个关键要素。我们将逐一分析这些要素，并提供清晰的思路，帮助你理解并掌握更有效的互动方式。

## 概述：你缺乏什么？

许多人在与异性相处时感到困惑或受挫，往往是因为缺乏一些核心的认知和技巧。以下是四个最常见的问题，也是你需要掌握的关键点。

1.  **耐心**
2.  **稀缺性**
3.  **铺垫**
4.  **平衡与真诚**

接下来，我们将逐一展开，详细解析每一个要点。

但是，很多人愿意花同样的价钱去买一个性价比略低的高档品牌车。如果不了解车，可以换一个例子：人们常说“穷玩车，富玩表”。你开一辆80万的卡宴，别人可能还说你是暴发户；但你戴一块四五万的欧米茄手表，别人就会说你有品位、是贵族或有钱人。这不奇怪吗？80万的车难道还比不上四五万的手表吗？

这是因为**稀缺性**。手表戴在身上看时间，你可以用手机，为什么一定要用手表？手表花四五万块钱戴在手上看时间，其实没有那么大的实用价值。正是因为它“没有用”，才会让人觉得：“你这么没用还这么贵，真有品位。”把妹也是一样，就是这种**稀缺性**。

就像有钱人有钱以后，他需要通过消费来向别人标榜自己的不同，所以他会想买一块四五万的手表戴在手上，让人们看出他是多么不一样。大家懂这个意思了吧？所以你去把妹时也是一样的。

这就是我说的**稀缺性**。你要做的是跟别人不一样的东西。你要让别人在挑选和筛选你时，感受到你和别人不一样，你有特别的地方。这就是所谓的稀缺性。

所以，我曾经回答过一个问题：女生聊天时说感冒不舒服，这时怎么办？现在大家应该明白，当别人都在说“喝热水”、“休息一下”、“洗个澡”时，你要直接**用行动**来表明你跟别人不一样。

也就是说，当你再去面对女生、想去撩动女生的心时，你要懂得用不一样的方式。不一样有两种：一种是特别，一种是更有行动力。100个人里，可能有99个都会说，用嘴皮子功夫表达得很漂亮，但可能只有二三十个是敢于去行动的。这个“行动”就是第一步的让你跟别人不一样。

所以，我很想让大家理解：在与女生相处时，嘴巴上说的东西可以少一点，更多的是你要用行动去做。这是一种稀缺性的表现，是跟别人不一样的一种方式。当然还有很多其他的不一样。

所以说，稀缺性我刚才说得可能还不够准确。它不仅是让你自己跟别人不一样（这是本质，让你在女生眼里看起来不一样），你还要为这个稀缺性赋予很多东西，例如陪伴、行动、带领等。这些都是你可以赋予进去的特质，或者说你让人感觉不一样的地方。这是一个最开始的思路。

## 第三点：铺垫与耐心 ⏳

讲完稀缺性的思路，我们自然会遇到一个新的问题。例如，之前有过这样的案例：女生说她感冒了，你拎着汤和药去她家时，她可能会不舒服，觉得“你怎么跑到我家来了？这样会不会不太好？”她可能觉得你太冒昧了。或者你用特别的方式去关心她、连接她时，她采取了一种回避、反感或不接纳的态度。

## 第四点：真诚与平衡 ⚖️

讲完了前期的铺垫与交战，我们来看最后一个部分：**真诚与平衡**。也就是从那个紧张的、暧昧的、刺激有趣的环节，进入到让大家终于可以舒服地在一起、彼此信任、可以真诚真实地恋爱的环节。

真正的**真诚**是什么呢？是一种**平衡**，即“你喜欢我的时候，我也喜欢你”，这才是真诚。如果你不喜欢我的时候，我还喜欢你，这就不是真诚，这有问题。这个要展开讲有点深，有点哲学，我们不太展开讲。
