# 但丁魅力约会与自由社交完整版录音：第6课：社交自由与领袖能力

在本节课中，我们将要学习如何摆脱“战胜他人”的竞争思维，建立真正的社交自由与领袖能力。核心在于理解“无我”状态，并通过整合资源、建立独特连接来创造不可替代的价值，而非在他人制定的规则下竞争。

## 理解“无我”与竞争思维

上一节我们介绍了社交中的基本心态，本节中我们来看看阻碍我们社交自由的核心思维模式。

许多男性在社交，尤其是与女性互动时，陷入了一种“必须战胜其他男性”的竞争思维。这种思维的表现是：

*   纠结于能否“快速”与女性发生关系。
*   试图通过战胜其他男性来凸显自己的优越感。
*   表现为“我可以接受她的过去，但认识我之后不行”，即使对方并非你的女朋友。

所有这些思维的根源，都是**你想要战胜别人**。而当你想要战胜别人时，你本身就是**自卑**的。因为一个真正强大的人，根本不需要与弱者比较。

**公式：想要战胜他人 = 内心自卑**

当你和别人比较时，你就把自己拉低到了和对方同一层次。就像一个普通人去骂国家领导人，对方根本不会回应，因为你们不在一个档次。他若回应，反而降低了自己。

## 竞争思维的局限性与痛苦

这种“战胜他人”的思维在现实中会带来巨大痛苦。例如，当你和一个女性有发展可能时，你会担心她过年回家、换城市，担心其他男性出现。

问题的核心在于：**你究竟是要她这个人，还是要她的心？**

如果你追求的是两个人在一起，外界有太多不可控的阻碍。但如果你要的是她的心，只要心在你这，总有机会。而当你一心想要战胜其他男人时，其实是在背离这个目标。

这引出了一个根本问题：**女性是否有权利选择自己的亲密关系？** 答案是肯定的。正如你再爱一个女人，也无法绝对阻止她与其他男人发生关系。你越试图阻止、要求、控制，对方只会越反感。

这背后隐藏的关键是：**你把自己看得太重了**。你觉得自己是“天选之子”。而在“无我”的境界中，“我”是不存在的。女生给我的任何反馈，都只是一种体验。

你为什么特别在乎她？原因有二：
1.  **意味着需求感**：你对她有强烈的需求。
2.  **意味着对结果的执着**：你特别想要一个确定的结果。

当你特别想获得一个结果时，你必须明白，完成一件事除了自身努力，还有很多不可控的外部因素。例如，你约一个女生，她临时爽约，可能只是因为她有急事，心情不好，而并非你的问题。时间拖得越久，这种“非你因素”的影响就越大。

**核心观点：千万别太把自己当回事。**

弗洛伊德研究认为，人一生所追求的可以概括为两个字：**装逼**。即永远在追求证明自己比别人优越。但事实上，没有人是全面的赢家。我约会厉害，不代表我在其他方面都强。人永远如此。

## 如何与高价值人群建立有效连接

接下来我们不讲理论，讲具体操作。承接上次的社交内容，假设你在生活中遇到了很多高价值的人（如老总、局长、媒体人），交换了名片，但之后不知如何跟进，无法建立有效连接。

与这些人交往，主题不应该是“赚钱”或“玩女人”，因为他们可能不缺这些。他们真正关心的主题是：**社交**。他们希望通过社交认识更多有价值的人。

以下是建立和维护这种高端社交圈的具体方法：

**1. 提供价值与分享感悟**
*   与他们聊天时，分享你独特、细节化的经历。例如，聊你在其他城市或国外举办社交活动的经验。
*   可以分享你组织活动的感悟，比如：“一开始请太多女生，大家都不聊正事了；后来女生太少，气氛又不好。最终发现10个人，男女比例3:7时效果最好。”
*   你可以把他们加入你的联系人列表，但不要群发广告。多分享你的思考和感悟，刷存在感。

**2. 策划事件，提升价值感知**
*   搜索并关注上海等地的高端活动。如果你知道某个活动，可以提前一天晚上给那位老总发信息：“我和朋友办了个XX活动，感觉很高端，您有兴趣吗？” 即使他大概率不去，但他会认为你在主办高端活动，你的价值在他心中会提升。
*   找一个借口，把所有这些高价值的人一次性约出来。最好的借口之一是：**你的生日派对**（不一定真是你生日）。
*   当这些牛人齐聚一堂，看到彼此都是高价值人士，他们会自然而然地认为你也很牛。
*   你还可以邀请一些高价值的女性（白富美）参加你的生日派对。当这些牛人看到你身边有如此优质的女性资源时，你在他们心中的地位会再次提升。

**3. 成为资源枢纽，建立深度利益绑定**
真正的连接不仅仅是维护关系，更需要建立切实的项目合作，并将关键人物变为**利益共同体**。

**案例：整合电影项目与虎扑资源**
1.  **契机**：在一次峰会上，一个投资人和一个电影导演想拍一部关于恋爱教学的电影。
2.  **脱颖而出**：我之所以被他们选中，是因为我的表达**通俗易懂**，而其他讲师用了太多“购买度”、“购买欲”之类的术语，外人根本听不懂。说话的关键是让听众听懂，而不是自我陶醉。
3.  **资源整合**：我做了以下事情：
    *   对投资方：我提出可以利用我在“虎扑”论坛的影响力进行宣传。虎扑有3000万用户，广告位很贵，但可以由电影投资方出钱购买。广告链接直接导向电影预告片，并写上“但丁监制”，同时附上我的微信二维码。
    *   对虎扑管理层：我通过朋友引荐认识了虎扑的超版。为了打消他“我是骗子”的疑虑，我特意约他吃饭，并叫上两位上海的普通论坛会员。在饭局上，我现场为那两位会员解答情感问题，证明了我的能力和诚意。此后，我与虎扑建立了深度合作。
4.  **核心策略**：我成为了投资方和虎扑之间的**枢纽**。我让双方资源互换，并为自己争取了核心股份。**钱永远不要一个人赚，要分给关键节点的人。**

**4. 用心维护，创造独特印象**
*   拿到名片后，在背面写上见面日期和对方说过的一句让你印象深刻的话。下次见面时提起：“我特别记得你，因为那天你说了一句……对我影响很大。” 人都需要被重视和认可的感觉。
*   针对不同层次的人，要用不同层面的东西打动他。对于高层次的人，他们很难被钱打动，但往往会被**境界和理念**打动。例如，我对一位成功的女企业家说：“我知道您不缺钱，您真正想要的是看到下一代成长起来，这是一种希望和成就感。” 这深深打动了她。

## 制定自己的规则：建立不可替代的独特连接

社会或女性常常给我们设定规则：你要有钱、要帅、要有魅力才能获得爱情。但问题在于，如果你遵守别人的规则，就永远会有人比你更有钱、更帅、更有魅力。规则制定者随时可以淘汰你。

真正的解决方案是：**建立独特的、不可替代的连接**，让对方进入**你制定的规则**。

**核心思路**：你不需要在社会的通用规则下去战胜别人，因为那里总有更强者。你要做的是，创造一个对方独有的需求，并由你来满足，从而让他同意遵守你的规则。

**如何建立独特连接？**
*   **案例**：很多20岁出头的漂亮女性，对未来的事业感到迷茫。如果你能在这上面帮她，甚至一起合作一个项目（比如开一家店），你们之间就建立了除了感情之外的**事业连接**。这种连接比单纯的情感吸引更牢固。
*   **实例**：我与一位女性朋友就是如此。我们不仅是情人，我更一直在她的事业上出谋划策，甚至计划共同投资一个密室逃脱项目。即使她有男朋友，我们依然保持合作，因为我们的连接是独特且多维的。

**具体方法示例：面试法则**
*   为什么很多PUA要做“约会面试”？因为当女性同意来参加面试时，她就**同意了你制定的规则**。
*   规则由她制定时，是她挑剔你。规则由你制定时，就变成了你筛选她。你的心态会从“渴望得到她”转变为“审视她是否合格”。

本节课中我们一起学习了：
1.  摒弃“战胜他人”的自卑竞争思维，追求“无我”状态。
2.  与高价值人群社交时，要提供价值、策划事件、成为资源整合的枢纽，并建立利益共同体。
3.  不要陷入他人制定的社会规则，而要主动创造独特价值，建立不可替代的多维连接，从而制定属于自己的规则。

通过以上方法，你将从社交的追逐者，转变为真正的引领者和规则制定者，获得真正的社交自由。
