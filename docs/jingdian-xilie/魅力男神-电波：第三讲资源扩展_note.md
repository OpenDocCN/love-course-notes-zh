2.  友善地说：“嗨，你好。”

**间接开场**
间接开场是隐藏自己的真实意图，寻找搭讪的链接。在目标分数较高，或自身状态不太好的情况下，推荐使用这种方式。这种方法容易打开目标进行互动，不容易被打枪，收号成功率也高。

### 错误的展示面

以下是需要避免的错误展示面：

1.  **转载一堆文章**：朋友圈全是转载文章，没有关于你个人的内容，妹子无法了解你。
2.  **背景很垃圾的自拍照**：在厕所、杂乱房间等低层次环境拍照，会让妹子觉得你层次低。
3.  **像素低、光线差、无美化的自拍照**：模糊、昏暗的照片让人没有欲望和你聊天。
4.  **角度错误的自拍照**：避免从下往上拍等错误角度。最好请别人帮你拍，或自己从正前方、斜上方拍。
5.  **豪车晒方向盘照片**：单纯晒方向盘或车内饰，妹子会觉得无聊或你在盗图。即使车是你的，自己拍也容易显得“很作”、装逼。最好请别人帮你拍。
6.  **全是风景照，没有自己**：妹子会怀疑你是否真的去过这些地方，是不是盗图。
7.  **用别人的照片**：用明星或他人照片，是极度不自信的表现，且无法见面。
