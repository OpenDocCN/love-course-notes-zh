说回我们今天的主题，我们讲的是“如何分手”。分手当然有上、中、下策。在讲方法之前，我想说的是，分手是一个过程，大家需要注意。我们这里不会理会你们因为什么原因分手，我们不会评判对错。

举个例子，就像我们刚才说的，你可以因为想换伴侣而分手，可以因为家人反对而分手，可以有各种各样的理由。但这些理由其实都不是根本理由。根本原因是你这段关系本身出了问题，你才想分手。如果关系本身没问题，就不会走到分手这一步，或者正在走向分手。

在讲分手方法之前，我们不如先讲讲分手的结果。因为分手的方法有很多，但结果大致只有三种。下策通常会对应下等的结果。这里要记住，你用什么策略去分手，通常就会得到相应的结果。我们之后会举一个例子，曾经有人用了很好的方法分手，但分手后因为一些后续处理不好，结果从上策变成了下策。

如果讲分手的结果，下策无非是反目成仇、做不成朋友、对方说你的坏话。通常这种情况下，对方一定会说你的坏话，甚至无中生有，明明你没做过的事也赖在你头上。如果一个女人不要面子，她比男人更在乎面子。在传统观念里，分手时很多人会认为女方是受伤害的弱者。如果她说了些对你不利的话，人们可能会倾向于相信女方的话。举个极端的例子，如果她笑你在床上不够持久，这种事无法证实。只有当事人两个人知道的情况下，女方一说出来，男方就很麻烦了。这就是“唱衰”的极致。所以大家听这集的时候要记住，小心这几种经典情况。

如果是中策，其实之后有很多可能性。因为中策的结果通常是还能做朋友。这种方法很多人都能做到。只要分手时你用的理由和方法不是低级到爆，其实真的能做到。这不难，关键是心态要放得下。如果你放不下，那就什么都不用说了，还分什么手呢，直接归入下策。所以中策的结果通常还能做朋友，从零开始。从零开始之后发生什么事就没人知道了。中策我们好像有位明星也是这样，就是前两集讲过的那位，和飞飞那样。因为他们两个分手是没人知道具体原因的。他们的官方宣布是经纪公司的策略，不一定是他们两个人实际说的话。宣传策略而已。这个世界已经发展到这个地步了。

如果讲到上策，那当然最好。最好的策略，最好的结果就是多了一个“炮友”，或者说红颜知己。通常先做回红颜知己，然后不知不觉，反正大家都有需要的时候就做了炮友。达到“恋人以上，朋友未满”的那种状态。
