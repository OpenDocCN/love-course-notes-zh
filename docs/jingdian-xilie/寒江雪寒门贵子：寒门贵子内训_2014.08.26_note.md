# 寒门贵子内训：1：情绪的觉察与接纳

在本节课中，我们将要学习心理学中一个核心且基础的概念：情绪。我们将探讨情绪的本质、它如何驱动我们的行为，以及如何通过觉察练习来更好地理解和管理情绪。

## 什么是情绪？

上一节我们介绍了课程的主题，本节中我们来看看情绪到底是什么。情绪是人感受到的一种内在体验。例如，当我们说“我很生气”时，描述的就是一种内在的感受。这种感受驱动着我们的外在行为、表情和言语。

心理学，尤其是从弗洛伊德开始的精神分析学派，重点研究的就是驱动人行为的各种情绪从何而来。弗洛伊德将这种驱动力量称为“情绪动能”。

## 情绪与潜意识

情绪常常与“潜意识”或“潜在动机”联系在一起。这些动机通常是当事人自己难以察觉的，却深刻地影响着我们的选择和行为，例如职业选择。

以下是几个例子，说明潜在的情绪动机如何影响职业：

*   **通信行业**：从事制造手机、电话等通信设备的人，其潜在动机可能是渴望与他人沟通，而现实中他们可能恰恰不擅长直接的人际沟通。
*   **电池行业**：生产电池的人，其潜在动机可能是内心感觉动力不足，希望储存和释放能量。
*   **演艺行业**：演员或明星的潜在动机往往是希望获得大量的关注和被看见。

这些内在的驱动力就像“情节”——一种情绪的固结，尤其是童年时期形成的情绪固结。它成为一个人特别在意、一旦被触及就会产生强烈反应的领域。

## 情绪的觉察练习

理论需要结合实践。为了让大家切身感受情绪，我们现在来进行一个简单的练习。这个练习需要两个人一组，模拟一个日常场景。

以下是练习步骤：

1.  两人一组，分别扮演“麦当劳顾客”和“服务生”。
2.  场景设定为顾客因故对服务生不满，双方发生争执。
3.  自由发挥对话，尝试表现出“吵架”的情绪状态。

（练习过程略）

练习结束后，引导者和参与者可以围绕以下问题进行反思和讨论：

*   当你扮演角色时，你的感受是什么？
*   当对方说某句话时，你的情绪产生了什么变化？
*   你注意到自己在对话中有哪些身体反应或习惯（如结巴、抚摸某物）？

这个练习的核心是引导注意力回到自身，持续询问“你的感觉是什么？”。这种练习在心理学训练中被称为“督导”，即在指导下进行自我觉察的练习。

## 练习的核心：从外转向内

在练习和日常觉察中，一个常见的误区是将注意力转向外界，评价他人。真正的关键在于将注意力拉回自身。

例如，当有人说“他那样做是在调戏别人”，督导者会问：“他那样做，让你感觉怎么了？” 始终围绕自身感受进行探索。

同样，对于同一句话，不同的人会有不同的感受和理解，这被称为“投射”。**投射是我们感知外界的普遍方式，也是唯一方式**。我们总是通过自己的心理滤镜去理解他人。

## 觉察的益处与日常作业

持续的自我觉察练习，就像锻炼情绪的“肌肉”，能带来诸多好处：

*   **提升情绪敏感度**：能更敏锐地察觉自己和他人的情绪变化。
*   **增强情绪管理能力**：在情绪升起时，能有所觉察，从而不被其完全带走，为理性应对留出空间。
*   **加深自我了解**：看清自己的行为模式、潜在动机和情绪盲点。
*   **促进校准行为**：如同闭眼走路会偏航，睁开觉察的“眼睛”，就能及时调整自己在生活、社交、事业中的方向。

本节课的作业是：每位同学找到一位“同桌”，在课后进行一对一练习。互相询问对方在日常生活事件中的感受和情绪，练习将注意力从对外界的评判转向对内在体验的观察。

## 总结

本节课中我们一起学习了情绪的基本概念，了解了潜意识与情绪动能如何影响我们。更重要的是，我们体验并学习了自我觉察的核心方法：不断询问“我的感觉是什么？”，并将此作为理解自己、管理情绪的起点。通过持续的练习，我们可以培养更敏锐的觉察力，从而在复杂的生活情境中做出更恰当、更清醒的回应。
