# 倪外卖方法：1：一对多约会策略

在本节课中，我们将要学习如何处理一种常见的约会场景：一对多约会。这不是指多人游戏，而是指当你与目标对象约会时，对方可能会携带朋友（即“障碍”）一同前来。我们将探讨如何应对这种情况，并巧妙地将“障碍”转化为你的助力，而非真正的阻碍。

上一节我们介绍了课程的整体方向将更加细化。本节中我们来看看“一对多约会”这个具体课题。

## 课程内容细化说明

我们团队后续推出的内容会逐步更加细致。例如，以往课程标题可能过于宽泛，如“如何约会”、“如何诱惑”或“街头搭讪”等。这些话题范围非常大。

若用一两个小时来讲解如此宽泛的主题，内容难免会显得稀释，干货比例可能很低。我们的意思是，今后的内容将进行更精细的划分。

## 本节课核心主题：一对多约会

今天我们讨论的主题是：面对一对多约会。具体而言，当你的约会对象携带同伴前来时，应如何应对。

例如，你的目标对象可能会带一位或两位朋友，甚至男性朋友。在这种情况下，我们应该如何面对？

我们的目标是：**让“障碍”成为你的助力，而非真正的阻碍**。“障碍”可以转化为聊天契机。这是本节课要探讨的核心课题。
