# 职业素养课程：第一章：职业态度与职业化层次

## 概述
在本节课中，我们将学习职业发展的核心态度，并了解职业化从合格到卓越的三个层次。掌握这些原则，有助于职场新人建立正确的职业观，实现稳步成长。

## 职业态度：从内在成长开始
上一节我们提到了职场中的基本认知，本节中我们来看看如何建立正确的职业态度。许多职场人常陷入一个误区：认为个人能力强但薪资低，而资方则认为支付了高工资却未获得匹配的能力。这构成了劳资矛盾。然而，从经验看，职场人往往高估了自己的实际能力。

**核心矛盾公式**：
`员工感知：自身能力 > 所得报酬`
`雇主感知：支付报酬 > 员工能力`

如果抱持“给多少钱，干多少活”的观望心态，只完成分内工作，拒绝额外任务，并急切追求升职加薪，却忽略了内在能力的提升，那么职业发展将停滞不前。关键在于每月、每季度、每年是否取得进步。若能力数年如一日，却期望更高的职位与薪水，这并不合理。

与同龄人攀比职位高低时，更应衡量自身内在成长了多少。如果三年前是主管，如今仍是主管且能力未增，那么晋升经理的依据何在？因此，正确的职业态度是：**做好你喜欢的工作，更要做好你不喜欢的工作**。

以下是关于此态度的具体阐述：

将工作视为一个整体，由喜欢与不喜欢的任务共同构成。只有将所有任务都完成好，才能获得脱离不喜欢工作的机会。例如，一名服务员只有把服务工作的方方面面（包括喜欢的和不喜欢的部分）都做好，才有可能晋升或转岗，不再从事不喜欢的任务。反之，若只挑喜欢的工作做，会给人能力不全面的印象，难以获得提拔。

**成长策略代码逻辑**：
```python
if perform(all_tasks) == "well":
    opportunity_to_leave_disliked_tasks = True
else:
    stuck_with_disliked_tasks = True
```

从成长规律看，通常只需忍耐约一年时间，全力做好所有工作（包括不喜欢的部分），之后便可能不再接触那些任务。但若敷衍应对，则可能长期陷于不喜欢的任务中，直至忍无可忍辞职。

## 把握当下：职业生涯管理的最佳策略
在确立了基本态度后，我们需要有效的执行策略。职业生涯管理的最佳策略是：**把握当下，着眼未来**。

这意味着要做好每个阶段的事情，具体可通过制定`月计划`、`周计划`、`日计划`来实现。职场新人切忌因忙碌或娱乐而放松要求，忽略当下。没有当下的坚实积累，就无法抵达理想的未来。反之，若总是拖延，用明天的时间补今天的任务，甚至数周无法完成工作，这种效率和态度将让人难以信任，从而错失承担更重要工作的机会。

**策略对比**：
*   **上策**：做好当下，为未来稍作准备。
*   **下策**：空想未来，忽略当下。

## 职业化的三个层次
明确了态度与策略后，我们来系统了解职业化的发展阶段。职业化大致可分为三个递进的层次：合格、优秀、卓越。

以下是每个层次的核心特征：

1.  **合格**
    *   **定义**：明确并服从公司的要求、制度和规范。
    *   **表现**：能做好本职工作，熟悉规则，不犯重大错误，并开始纠正个人不良习惯。处于“在框架下完成任务，基本不犯错”的状态。

2.  **优秀**
    *   **定义**：在合格的基础上，主动提升自我。
    *   **表现**：不仅能遵守规范完成工作，还能成为部门榜样，其职业化水平被视为标杆。体现了更高的自觉性和示范性。

3.  **卓越**
    *   **定义**：在优秀的层面上，能够推动积极的改变。
    *   **表现**：不仅是榜样，还能在现有框架下进行适当的调整、优化和创新，例如改进工作流程。这是基于规则并超越规则的创造。

对于职场新人，首要目标是达到“合格”，即遵守规则，完成指令。其后才是追求“优秀”，成为榜样。最终，在积累足够经验和信任后，才有可能达到“卓越”，推动创新。

## 总结
本节课中我们一起学习了职业发展的核心态度与职业化层次。我们认识到，与其抱怨外部环境，不如聚焦内在成长，通过“做好不喜欢的事”来赢得发展机会。职业生涯管理应遵循“把握当下，着眼未来”的策略。同时，职业化是一个循序渐进的过程，需经历从合格（遵守规则）、优秀（成为榜样）到卓越（推动创新）的三个阶段。职业化贯穿整个职业生涯，要求我们在思维、言行等各方面都以专业的标准要求自己。
