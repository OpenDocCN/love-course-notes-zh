有句话是这样说的：不要去追一匹马，用追马的时间来耕种草地，洒上水，施予阳光。等到夏天草长莺飞的时候，自然会有很多马来到你的身边。

后悔药是没得吃的。现在，我们有一个最好的身体。说一件很感慨的事情。我有一个朋友，他最初喜欢一个女孩时不懂方法，没有把握住她。等过了很多年，他学会了方法，再次找到那个女生，用尽方法，但三天之后他告诉我他很空虚，他发现找不回18岁那年的感觉了。

所以，大家明白了吗？为什么我们这个恋爱补习班存在？一开始，我们只是试着帮身边的人答疑解惑。但后来我发现，身边很多朋友像当年的我一样，把最精彩的自己交付在了一段段毫无意义的感情和一个很无知的自己身上。看到这些，我就想起当年的自己，特别痛苦，特别不想再看到这些事情发生。

所以我们成立了这个组织，叫做恋爱补习班。我最爱教大一大二的孩子，不是对年纪有偏见，而是因为他们觉醒得越早，优势越大。我再也不想看到有人把大好的青春耗费在无意义的自己身上。

所以，我跟大家讲第一堂课时，往往不会先教方法。怎么聊天、搭讪、约会，这些都很简单，网上有很多案例。这些真的不是关键。方法就像武功招式，需要内功来衬托。如果你有内功，简单的招式也能发挥巨大威力。如果只有招式没有内功，那就是花拳绣腿。为什么同一段话，导师说出来就很有用，你说出来可能就没用？就是因为内功没有修炼到位。

在过去的7天里，我们讲了很多招式，也讲了内功心法。但内功心法不能一蹴而就，需要你慢慢去内化。什么是内功？就是所谓的**框架和心态**。简单来说：**你是谁？你想要什么样的感情？你想要什么样的自己？你喜欢什么样的女人？不喜欢什么样的女人？**（除了漂亮，因为大家都喜欢漂亮的，这无可厚非）。你心里想成为的自己是怎么样的？你想要的感情是怎么样的？这是关键。

所以在这里，我也想问问大家，请你们自己也思考一下：你们想成为什么样的人？你们想要的感情是什么样子的？能跟我分享一下吗？我非常想听听大家，尤其是我们这7天训练营的同学，在经过了集训、学了这么多方法之后，心态有没有一定的改变？你们现在想追求、想去相处的女生，是什么样子的？这也是我觉得，我想为大家做的最大的改变。

有时候，我们面对女生往往是惶恐、不安、需求感爆棚的。甚至在分辨对方、认清自己是什么样的人之前就贸然下决定，这不单是对对方不负责，也是对自己、对感情不负责。框架和心态，是志宏课程讲义里最重要的一关。如果你能找到，后面的东西就驾轻就熟；如果找不到，不光“泡”妹子，你的爱情观都会出问题。

## 我的背景与长期关系心得

在这些恋爱补习班成立之后，我想跟大家分享一下我的看法。大家了解我吗？志宏HIT。志宏是我的名字，HIT是我的学校哈尔滨工业大学。我高中在山东上学，那是一个教学极为严格的地方。我们大学的校训是“规格严格，功夫到家”，是一个很讲究科研精神、话说回来也是很“闷”的学校。

所以说，我的整个成长过程都是比较规规矩矩的。现在我在百度工作，在百度贴吧。这也是一个工科生聚集的地方，很多人都是学技术出身。所以我的生活圈子里充满了男性思维，或者说“直男癌”比较严重的同学。他们很多人很不了解女生是怎么想的。

而我们恋爱补习班上来就会跟大家说，你要想“泡”女生，要先做到两点：
1.  **了解**：了解她们是怎么想的。
2.  **尊重**：你要做到这两点，就可以慢慢进入她们的生活。

我现在其实已经结婚一年了。虽然大家看我平时在群里有点“禽兽”，发一些耸人听闻的照片，但其实我是一个已婚的人，而且我很爱我老婆。你可能会觉得我有点浮夸，一边做着短期关系的事，一边还说很爱老婆。但事实就是这样子。短期关系并不影响我们遇见爱情。

她是我学妹。我大二时她大一，我帮一个老乡搬行李那天遇见了她。那时候没有一下子就喜欢上，但我觉得她是一个很值得我去付出的人。经过大概两年的感情纠葛（包括暧昧、那时她还有男朋友等），我在大三的时候跟她在一起了。那时我20岁，到现在我26岁快27岁。我们恋爱6年，结婚1年。

我觉得我很幸运，能遇到她、把握住她。但你们知道这个过程需要多大的坚韧吗？我当时做PUA，是因为来到北京后遇到了我大学的舍友，他就是Teddy。我们大四时在校外租房子，我跟我老婆住一间，他跟他女朋友住另一间。我们四个人一起住了一年，但两段感情走向了不同的结局。所以我再见到他时，会唏嘘很多。

往往在长期关系中，家庭因素是我们无法规避的，总会在某个时间段出现并影响发展。交流也是一个很重要的原因。

我跟大家说一下，长期关系需要经营四个阶段。如果你能度过这四个阶段，你就会拥有最完美的爱情。今天讲的内容是我自己总结的特别精华的部分。

**第一阶段：吸引与愉悦的相处**
这就是我们现在教给大家的。我们教给大家的短期关系、把妹、约炮、吸引女生、挽回，第一步都是这样：**吸引而导致的愉悦的相处**。想必大家之前听过很多：女生不是追来的，而是吸引来的。所以吸引是第一课。

恋爱其实很简单，尤其是在校园里，还没涉及到谈婚论嫁。当然，很多人没有恋爱就直接结婚了，那叫做“很正确的爱情”或“婚姻”——双方家庭条件差不多，至少不相互讨厌，有着相似的家境和世界观，就凑成一对一起前行。但我想，这应该不是大家想要的。我们肯定想要一个能让我们动心的姑娘，无论是因为容貌、性格还是其他。在那一刻，我们脑子里没有多余想法，就是很想把这个人搂在怀里，很想跟他一起走下去。

这样的爱情，第一步都是做到吸引，做到愉悦的相处。这一步其实很简单，说白了就是我跟一个女生可以很开心地一起玩，而不觉得尴尬、乏味。只要你们能到这种状态，其实就可以建立恋爱关系了。我们相互吸引，一起出去玩会觉得有趣、充实。

所以大家，我其实挺看不起那些“一见钟情、非他不娶”的人，你都不了解人家，凭什么这么说？对自己、对爱情负责吗？第一步，我们要求目标不要太高，不要说“真的很爱对方，一定要结婚”。你只要能够做好吸引对方，能跟对方愉悦地相处，这就是一段美好爱情或短期关系的开始了。

（学员“子豪”说：喜欢约炮。）喜欢约炮没有问题。约炮需要做到两点：第一，你可以约炮，但不要骗炮。哄着女生说谈恋爱，骗了一两次，我觉得挺不道德的。第二，你约炮的时候，哪怕只有一晚上，你都可以让对方感觉到快乐、愉悦。虽然时间长短不同（比如Teddy只有3分钟，我一个半小时），但在这个过程中，我们都能让对方感觉到快乐，这就足够了。

所以，时间不重要。我们能够做到一个点，就可以理直气壮地说约炮是可以接受的：**在对方遇见你之后，他比遇见你之前更快乐、更充实了。他会觉得遇见你是一件人生中值得纪念、想起来就快乐的事情。** 我举个特别浮夸的例子，比如《色戒》里汤唯和梁朝伟。任何一个平凡女子，如果觉得跟梁朝伟有了一夜之情后会特别荣幸、开心，纪念那个夜晚。这是一个很绝对的状态。但我希望大家也能找到这样的状态，哪怕只有一天，你要能够让对方感觉到跟你在一起很快乐，遇见你很值得。

所以第一点，要做到吸引、愉悦的相处。我没有跟大家讲方法，而是讲心态、阶段、过程。你们要找到自己。

其实很多美丽的爱情，说句实话，有时候也是从“约”开始的。还真见过一开始只是约个炮，结果就走在一起了。但不要打着恋爱的幌子去约炮。约完之后往后怎么走，其实每个人都预料不到。

**第二阶段：包容与依赖**
我们做到吸引、愉悦相处后，如果能迈入第二阶段，就可以成为真正亲密无间的情侣。这个第二阶段叫做**包容与依赖**。

什么叫做包容与依赖？我们在跟女生相处一段时间之后，可能会觉得乏味。人都是这样，对新鲜刺激的事往往很有兴趣。第一次做爱会觉得超级happy，第二次有点乏味，第三次可能就有点厌恶了。很多男生有这种情况。

但我们要跟一个人长期走下去，除了享受对方的优点，肯定要去接受、接纳彼此的缺点。为什么现在这么多人喜欢保持暧昧状态？有人能回答我吗？

（学员回答：刺激感；展示自己；新鲜感；安全感不足；习惯了。）

“Star”说得非常对。暧昧状态就是保持在一个适当的距离，这个距离足够近，近到我们可以享用彼此的优点；但又足够远，远到我们不会接触到彼此的缺点，或者不会被彼此的缺点所伤害。

但我们真正要跟另一个人走下去怎么办？我们肯定要去包容彼此的缺点。我跟大家讲个段子。我之前跟我女朋友合租的时候，一个屋檐下，各种摩擦。两个人住在一起时，所有细节的性格劣势都会被无限放大，稍微一根小的导火索都会把你们两个点燃。

举个最简单的例子：有一天我们一起起床，她说饿了，让我煮饺子。我就去煮饺子，为了不粘锅，我撒了点盐。我老婆就问我为什么煮饺子要放盐。我说这是我们家的秘方，煮饺子放盐容易熟，而且饺子不粘在一起。她说不对，她妈煮饺子从来不放盐，这样饺子会很咸。我说没错，我妈每次都放盐。她说：“你妈做错了吧？”我说：“你敢说我妈做错了？你还饿不饿了？”她说：“你爱不爱我呀？”这就吵起来了。现在回想起来，是一件很傻的事情。

但我想告诉大家，生活中各种各样的小事都会点燃你们两个，很无厘头。因为你离他太近了，你要去接纳他的缺点。有时候我们忍不了他很多缺点，比如我很爱干净，你回家袜子随便一脱；我很有时间规划，你每次都迟到；我能记得你的生日、你爱吃什么，你连我爱用什么牌子都记不住。很多观念的冲突会导致恋人分开。

但是，如果你爱他，就会愿意谅解他、包容他，不会计较吃亏与否。你会知道，这件事我很吃亏，这件事我无法接受，触犯了我的原则。但是我爱他，我想跟他更好地生活，所以我选择退步、忍耐。这就是爱情，这就是爱情的进阶状态：包容与依赖。

有一首歌叫《喜欢两个人》，歌词说：“心会累，爱会冷，这是感情必经的过程。只是有人就放弃，而有人愿意等。”所有的感情都会冷、会厌倦，女生也会，男生也会，是人都会。但在这个阶段，你愿不愿意付出、愿不愿意去包容，决定了你的爱情能不能继续走下去。

**第三阶段：信念的守护**
有时候我们跟对方相处很愉悦，也愿意容忍对方的一些缺点。但爱情往往很脆弱，会被一些很现实的东西打垮。比如刚才同学说的，父母不接受；比如毕业的恋人，突然相隔两地；比如必须抉择是留在城市工作还是去他的城市相信爱情。

这个时候我们会发现，人生中不只有爱情，还有家庭、父母、事业、梦想。有时候我们会想，作为一个男人，爱情可能只是生命中的一部分，为了保全其他东西，我们不得不放弃爱情。我觉得这个选择无可厚非。

但我们往往也会羡慕其他爱情，比如《何以笙箫默》（有人看过打个“1”），一个人等一个女生等了大概十年。当然我不是说这是正确的，只是说这种感情特别可遇不可求。

第三阶段叫做**信念的守护**。爱情会受到很多外界因素的冲击：家庭反对、社会压力（如买不起房）、异地、新认识的人等等。但只有你愿意去守护，愿意用能量去把它握在手里、主动把握，只有你足够强——你是一棵树，不是一棵草，不会让风一阵阵吹走——你才能够做到这一点。

我跟大家讲三个发生在我身边的故事。

**第一个故事**：我大学的一个同学，典型的工科男生，特别羞涩。他大学四年一直跟我们班一个女生关系特别好，但始终没在一起。大四时，男生终于鼓起勇气表白，两人在一起了。他们决定一起考研，但最后男生考上了哈工大的研究生，女生的第一志愿落榜了，要去湖南的一个学校读研。一个在哈尔滨，一个在湖南，距离特别远。男生做了一个特别勇敢的选择：他放弃了哈工大的研究生，选择去女生所在的城市工作。你可能觉得这很傻，万一以后分手怎么办？但他做决定时根本没考虑过他们会分开。现在他们结婚了，很幸福。这个故事告诉我们：**爱情是可以被选择的**。

**第二个故事**：我大学时在主持人俱乐部社团，里面有一个女孩特别秀外慧中，在男生居多的学校里风靡万千少男。大四时，她跟一个很优秀的学长在一起了，是大家眼中的王子公主。但两个同样优秀、强势的人，感情不容易一帆风顺。毕业后，男生要去美国加州读硕士，女生要去英国读金融。异地恋已不靠谱，何况异国恋？大家都不敢问他们怎么样了。但在去年6月，我看到男生发了一个朋友圈，地点在女生所在的城市，他抱着那个女孩幸福合影。这么多年，我们不敢问，却看到这样一个温馨的结果。那种对爱情的相信感真的很温暖。2014年10月，他们在加州登记结婚了。女生家庭条件不错，但绝对是奋斗的典型。这个故事告诉我们：**爱情是可以被坚持的**。

**第三个故事**：关于我自己的。我在感情中也遇到过很多纠结的问题：异地恋、第三者等。大学刚毕业时，我去了深圳，女朋友在北京。我觉得特别没有安全感，所以离开深圳来到了北京。她毕业前一直陪我留在北京，但毕业后，她妈妈特别反对她留在我的城市。她家里是一个不错的二线城市。她妈妈不让她过来，我就一个人在北京呆了一年半。那一年半里，我从来没有约过炮，因为怕自己迷失，我要守着很好的状态等她回来。但她一年半都没回来，我越来越绝望。

我家里是山东人，比较传统。有一天他们让我去考公务员，说比较稳定。我复习了两三个月，考得还不错，考上了山东省某个市的交通局。在报道的前一天，我想如果报道了，就回家乡了，应该不会再回北京。我给她打了个电话，告诉她我考上了公务员，周末要回去报道。她支支吾吾，听出来不开心。我问她：“如果我在这儿继续等你，你爸妈是不是不会让你来我的家乡工作？”她说：“是。”我问第二个问题：“你是不是不想让我回去？”她特别果断地说：“不，我不想让你回去。”我感受到这句话，瞬间做了一个决定。那个周末我没有回去报道，也没接父母电话，把手机关机睡了两天。周一报道期错过，我失去了资格。父母臭骂了我一顿，但我继续留下来了。

直到2014年3月，她跟她妈妈吵了二三十次架之后，一个人倔强地拿着行李来到北京。作为一个985高材生，她选择了一份月薪只有3500块的财务工作。2014年9月9日，我们恋爱五周年时，我们回到学校、回到我的家乡领证，并做了一次短途旅行。

回想这三个故事：
1.  第一个故事告诉我们：**爱情是可以被选择的**。
2.  第二个故事告诉我们：**爱情是可以被坚持的**。
3.  第三个故事告诉我们：**爱情没有不可能**。

在这个年代，我们为什么不容易找到爱情？是因为这个时代节奏太快，我们要经历的太多，懂得太少，所以错过了美妙的感情以及值得陪伴一生的人。
