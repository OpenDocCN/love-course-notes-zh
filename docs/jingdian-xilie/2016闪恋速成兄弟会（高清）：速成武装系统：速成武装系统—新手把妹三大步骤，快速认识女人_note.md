# 恋爱速成：第一章：速成武装系统

在本节课中，我们将要学习“速成武装系统”的核心思想与具体操作方法。这是你快速认识女性、建立平等心态的第一步。我们将从核心定律讲起，逐步介绍如何利用社交软件高效收集资源，并布置本周的实践任务。

## 核心定律：斜天秤定律

上一节我们介绍了课程的整体目标，本节中我们来看看指导所有行动的核心定律。

我们兄弟会总结出的核心定律是：**斜天秤定律**。

这个定律的含义是：**你想要搞定价值为X分的女性，你就必须展示出（X+1）分的自身价值**。例如，你的目标是6分女性，那么你至少要展现出7分男性的价值。

这个定律之所以是“斜”的，而非完全平衡，是因为在现实中，男性必须比女性更主动一点。如果你和女性完全平衡（即天秤平衡），由于社会竞争，其他主动的男性会抢先一步。但如果你过度主动（多出不止一分），则会显得你在“跪舔”；反之，如果你不够主动（少一分），则会显得你毫不在乎。这两种情况都会导致你无法成功。因此，最有效率的策略是：**永远只比女性多展示一分价值，多主动一分**。

## 心态建设：投资自己，而非女性

理解了核心定律后，我们需要建立正确的心态。这是武装系统的思想基础。

你必须记住第一点：**把你所有计划投注在女性身上的金钱和精力，转而投资到你自己身上**。许多男性无法吸引女性的原因，并非自身条件差，而是竞争对手变强了。在互联网时代，即使条件普通的女性，也能通过修图、化妆等方式在社交软件上展示出更高价值，从而拥有大量追求者。如果你还保持着过去的思维和展示方式，就会在竞争中处于劣势。

因此，我们的“武装系统”旨在教你通过技巧快速提升展示价值，而不是花费数年时间去真正奋斗成为“高富帅”。女性天生懂得包装自己，我们也要学会相应的技巧。

## 资源收集：社交软件高效用法

建立了正确心态，接下来我们看看如何快速、大量地认识女性。在网络时代，街搭效率低下，社交软件是更高效的途径。

以下是闪电会总结出的高效社交软件列表：
*   **探探**
*   **陌陌**
*   **世纪佳缘**
*   **微信**

对于新手（8级以下），我们建议主攻 **世纪佳缘**。因为探探和陌陌非常依赖高质量的“展示面”（即个人照片），在新手阶段效果不佳。世纪佳缘则是一个更严肃、高效的平台。

具体操作步骤如下：

1.  **设置资料**：最简单的方式是模仿。课程结束后，助理会将我的社交账号资料分享给大家，你们可以直接模仿我的资料设置，这是最快速的学习方法。
2.  **选择目标**：像“选妃”一样，在软件上筛选你感兴趣的女性。
3.  **发送招呼**：打开对话框后，**只发送一个友好的表情**。字数越多，越显得你不自信、需求感强。
4.  **引导至微信**：如果对方回复，不要在原软件上多聊，统一回复：“**微信聊吧，方便点。**”目的是将联系方式转移到交流效率最高的微信上。
5.  **处理拒绝**：如果对方拒绝加微信，可以补充一句：“**那发你的我加好了，我看你又不算坏人。**”用幽默化解尴尬，并再次尝试。

请记住：如果严格按照步骤操作后对方仍不回应，那不是你的问题，只是你不符合对方的偏好。不要纠结，继续寻找下一个目标即可。
