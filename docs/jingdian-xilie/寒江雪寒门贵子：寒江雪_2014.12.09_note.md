## 课程参与情况分析

已经晚上8点，今天是12月9日。12月9日是一个学生运动的纪念日。由于毕业已久，具体细节已不太记得。

教室里和频道里的学员都比较少。这与今天没有群发微信通知有关。以往会群发通知，今天仅在群里做了简单告知。这说明两个问题。

以下是具体分析：

1.  **信息接收习惯**：许多学员不常查看群消息。群发私信时，参与人数多；仅在群内通知时，参与人数少。这表明部分学员可能屏蔽了群聊，只关注私信。
2.  **课程记忆依赖**：部分学员并未记住每周二有固定课程。他们依赖于收到通知才来上课，没有通知则容易忘记。

今天未发通知看似是件坏事，但也能帮助我们看清以上两个现实问题。那么，我们就为在场的学员上课。

## 上期作业回顾与思考

上期布置的作业是继续进行信贷市场调研并完成报告。有学员完成了吗？无论是文案调研还是实际调查，都可以分享。如果没有完成，也可以谈谈原因。

有学员反馈因为忘记而没有完成。也有新学员表示因不知情而未做。这引出了一个更深层的思考：如何促使学员主动学习和完成作业？

我思考了很久，结论其实很简单：**学习本质上不是老师能强行推动的事情**。回顾我哥哥的例子：他从小学习不好，父亲经常督促甚至责打，但他最终仍未能考上大学。而我自己学习较好，却很少主动寻求老师帮助，多是自学。

这说明了核心问题：**学习的内驱力（想学）比外部的督促（抽着学）更重要**。如果学员自己不想学，老师或家长用任何方法强迫，往往只会让双方都很疲惫，且效果甚微。

想通这一点后，我决定不再为此焦虑。学习是学员自己的事，想学的自然会学好，不想学的，外力难以改变。我们应该接受这个事实，这样双方都能更轻松。

## 本周新作业：股票交易实践 📈

上一节我们探讨了学习的内驱力问题，本节我们将布置一项新的实践作业，将焦点转向外在的技能锻炼。

本周的作业是：**进行股票交易**。

具体要求如下：
*   在12月9日至15日这一周内（约5个交易日），进行实际的股票买卖操作。
*   可以多次交易。
*   在下周二上课时，每位学员需要分享自己的交易经历。

分享内容可以包括：
*   买了哪些股票（代码或名称）。
*   买入和卖出（或当前持有）的价格。
*   交易的理由。
*   投入的资金、盈亏情况。

**公式示例：**
*   收益率 = (卖出价格 - 买入价格) / 买入价格 * 100%
*   盈亏金额 = (卖出价格 - 买入价格) * 持股数量

**代码示例（伪代码）：**
```
if (current_price > buy_price):
    print("当前盈利")
else:
    print("当前亏损")
```

市场目前比较活跃，大家可以体验一下。无论是否有经验，都可以参与。赚钱或亏钱都是自己的经历和责任。如果实在不想操作，也可以仅进行模拟或观察了解。作业的核心目的是获得第一手的市场体验。

## 课程体系说明

最后，我们来明确一下本课程体系的定位。

周二的本课程主要解决“外在”的实际操作问题，如市场调研、增加人脉、投资交易等，侧重于经济学、成功学等解决具体问题的学科。

而周一的“聆听”课程则由另一位老师负责，主要解决“内在”的心理与认知问题，涉及心理学、哲学等领域。

两者虽有联系，但分工明确：一个主内，一个主外。本课程就是大家锻炼解决实际问题能力的平台。
