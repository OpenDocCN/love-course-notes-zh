以下是关系的三种基本定位：

该系统基于一个六边形模型进行评估。模型上方评估**主观繁殖价值（R）**，决定短期吸引力，即你对对方外在的喜欢程度。模型下方评估**主观内心特质（X）**，这是长期吸引力的关键。

**主观繁殖价值（R）** 包含三个子项：
*   **RA（肉体核心需求）**：例如身材、腿型等。
*   **RB（面容核心需求）**：例如脸型、眼睛、嘴唇等。
*   **RC（身体附加需求）**：例如发型、皮肤、体态等。

**主观内心特质（X）** 也包含三个子项：
*   **XA（本质特性）**：例如善良、单纯、孝顺、聪明、贤惠等内在品质。
*   **XB（两性互动姿态）**：例如在关系中是否个人主义强、脾气大小、沟通方式等。
*   **XC（生存姿态）**：例如性格是安静、阳光、活泼、文静还是多愁善感。

## 如何进行审判打分

现在，我们来看看如何使用这个系统进行具体操作。

首先，将你理想中伴侣的每一项核心需求列出5条。例如，RA（肉体核心需求）可以列出“喜欢长腿”、“偏好某种体型”等5条标准。

然后，为你遇到的女性进行打分。每满足一条标准得2分，满分10分。分别计算R值（RA、RB、RC的平均分）和X值（XA、XB、XC的平均分）。

**核心公式**：
`最终R值 = (RA + RB + RC) / 3`
`最终X值 = (XA + XB + XC) / 3`

评估原则是：**X值决定长期关系适合度**。如果X值不及格，说明不适合发展长期关系，只能考虑短期关系。R值主要决定短期吸引力。

学习这些技术的目的是筛选掉R值和X值低的女性，从而不错过心中接近满分的伴侣。你需要明白，你爱上的不是某一个女人，而是某一类具备你欣赏特质的女性。

## 长期关系的维护方法

当你确定对方适合发展长期关系后，以下是维护关系的几个关键方法。

1.  **保持新鲜感与空间**
    长期关系中需要保持一定的新鲜感和神秘感。双方不应时刻黏在一起，需要有自己的私密空间。过度亲密容易导致厌倦。

3.  **建立时间框架**
    明确告知对方你何时有空闲。例如，每周哪几天、每天什么时间段可以陪伴或聊天。这能避免恋爱扰乱你正常的生活节奏。

4.  **保持一致性**
    不要追求时百依百顺，确立关系后就态度大变。应保持前后一致的原则和态度。你的框架和原则不应因为关系确立而放弃。

5.  **引导对方投资**
    在长期关系中，引导对方投资（时间、精力、情感、物质）至关重要。对方对你投资越多，感情就越深，你在她心中的地位也越高。双方投资平衡是健康关系的基础。
