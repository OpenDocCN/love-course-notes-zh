# 自然男人英语习得法：三：建立英语思维 🧠

在本节课中，我们将要学习“建立英语思维”这一核心概念。这是自然英语习得法理论的最后一部分，也是决定你学习速度和最终水平的关键。我们将从理论到实践，逐步解析如何摆脱对母语的依赖，构建独立的英语意义系统。

上一节我们介绍了从依赖文字系统过渡到声音系统的重要性。本节中，我们来看看如何迎接英语学习的第二个里程碑——建立英语思维。

## 第一部分：理论

首先你需要理解的是，英语思维和汉语思维有巨大的不同。对英语的学习，你所学到的并不是已知汉语世界的一个翻译版，而是一套无法翻译的、全新的思维习惯与逻辑。用语言学来讲，这是另外一套**语义**和**语用**系统。

如果你还记得我们上节课里说的：语言是一种声音信号，经过声音解码器到达我们大脑中的认知部分，即**意义系统**（`system of meaning`），然后才被我们理解。那么，这个“意义系统”就是我们今天要深入理解的部分。

在我们的汉语文化中，我们习得了一套包含**语法**、**语义**和**语用**的意义系统。例如，我们打招呼时会说“吃了吗？”，这背后是一整套与语言密不可分的文化习惯。

同样的，在英语世界里也有一套独特的语法、语义和语用，构成了其**英语的意义系统**（`the English system of meaning`）。这个系统往往更庞大，因为英语是由两个主要语系构成的：
*   **源自古德语的盎格鲁-撒克逊语系**：词汇短而精，有力道。
*   **源自古拉丁的法语系**：词汇较长，含义较广，更文雅。

几乎每个概念都有两个不同语系的词汇对应，例如：
*   `pork`（猪肉 - 法语系词）
*   `ham`（火腿/猪肉 - 撒克逊语系词）

这导致英语拥有丰富的表达层次。请看例句：
*   **拉丁用法**：`The disgraceful woman conducted fabrication from actuality.`
*   **撒克逊用法**：`The bitch lied.`

这两句话意思相同，但给人的感受截然不同。语言归根结底，是我们用一套特定的声音来表达内心无穷无尽的思想（`idea`）和感受（`feeling`）。

语言的进化与文化密不可分。例如，汉语在味觉和嗅觉方面的表达特别丰富，而法语据说有上千种嗅觉表达。英语中也有很多复杂表达（尤其是科学用语）在汉语中并不直接存在，例如`cognition`（认知）、`native`（原生）、`perception`（感知）。

因此，学习语言的最终目的，不仅仅是词汇积累，更是你**意义系统的扩展**。它能增加你思想的复杂度和感知的细腻度，从而带来更好的判断力、决策力和更丰富的人生。

## 第二部分：实践

理解了语言意义系统与思维感受的关系后，我们来看看如何在实践中运用理论来建立英语思维。

以下是三个核心的实践原则：

**1. 脱离母语干扰**
成人学习第二语言时，常依赖母语翻译来“快速理解”，这是一种错误的态度。真正的习得需要单独建立**第二语言的意义系统**（`a second system of meaning`），而不是以第一语言系统为基础。因为两个系统重叠有限，用汉语思维去理解英语会造成大面积曲解。你需要像幼年学汉语一样，依靠大脑的认知功能去猜测和建立声音与各种因果关系之间的联系。

**2. 你不需要理解所有的意思**
成人学习的习惯是拼命为每个词找到汉语中的对应，以求“安心”。但这恰恰阻碍了新系统的建立。正确的做法是接纳那种“感觉上能够理解，但说不上来”的状态。允许某种不确切的感觉带着你前进，这正是在逐步建立独立的英语意义系统。建议使用**英英词典**来辅助理解，并记住不要过度依赖文字。

**3. 放下意识，用身体感受语言**
越是绞尽脑汁、刻意去学，你的意识反而会阻碍习得。相反，放下意识，让身体去感受语言，不拿已知的东西去套用新的东西，则会让你得到真知。语言学习也是一种“悟”。你天生拥有这种感受力，甚至在学会第一语言之前，你就具备了下意识思考的能力。带着这种感受去学习英语，就是自然习得之道。

---

本节课中我们一起学习了建立英语思维的理论基础与实践方法。核心在于认识到英语是一套独立的**意义系统**，并通过**脱离母语干扰**、**接纳不确切的理解**以及**用身体感受语言**这三项原则，在实践中逐步构建它。记住，这不是一个后期目标，而是应从学习初期就开始培养的习惯。
