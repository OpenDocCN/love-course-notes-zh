# 自然男人之道：3：社会机制下的吸引

在本节课中，我们将要学习吸引力的深层社会机制。我们将探讨性别概念、婚姻制度等社会建构如何从根本上塑造我们对“吸引力”的感知和反应，理解这些机制将帮助我们更清晰地认识自身感受的来源。

上一节我们从心理机制探讨了吸引，本节中我们来看看埋藏在心理机制之下、更深层次的社会性影响因素。

## 性别：一个社会建构的概念

我们首先需要理解“男女”这一基本概念。所谓性别差异，首先是一个客观事实，其次是一个被社会赋予意义的概念。男女之间的相吸，可以用荣格的“阿尼玛”与“阿尼姆斯”理论来理解。

*   **阿尼玛**：指男性心灵中的女性意象。公式表示为：**男性个体 = 阳性面 + 阿尼玛（内在女性意象）**。
*   **阿尼姆斯**：指女性心灵中的男性意象。公式表示为：**女性个体 = 阴性面 + 阿尼姆斯（内在男性意象）**。

荣格认为，从灵魂层面看，男性内在是女性化的，女性内在是男性化的。这种内在意象并非天生固定，而是在后天成长中被逐渐塑造的。

*   对于男性，最初接触的女性（通常是母亲）的性格特质，会奠定其“阿尼玛”的基础。随后生命中遇到的其他女性形象、社会文化中的女性模板会不断叠加其上，最终形成一个综合的“阿尼玛”。它决定了男性看到某个女性时会产生何种反应。
*   对于女性，其“阿尼姆斯”的塑造始于对父亲（或第一个重要男性形象）的认识，随后同样受到生活中其他男性形象和社会文化的影响。

因此，一个人会被何种特质吸引，很大程度上由其生命经历决定，而非由被吸引方单方面决定。吸引力在深层次上是高度主观的。

## 性别的文化多样性

我们通常认为的“男女”概念并非全球通用，而是特定社会文化的产物。以下是不同文化对性别的理解：

*   **婆罗洲岛（加央族）**：性别按社会分工划分。擅长采集种植者被视为“女人”，擅长伐木建房者被视为“男人”。因此，生理上的男性若承担“女性”工作，也会哺乳养育孩子。
*   **新几内亚（桑比亚部落）**：性别区分极端严格，认为女性会“污染”男性。男孩需通过严酷的成人仪式（包括吞食精液）来清除女性影响，转变为“真正的男人”。
*   **巴西（某些土著部落）**：性别分类基于性行为角色，分为“男性”、“女性”和“被肛交者”（Travesty）。他们的观念与现代社会的同性恋概念完全不同。

这些例子表明，“男性气质”与“女性气质”的内涵因文化而异。例如，在中国文化中常见的女性“撒娇”行为，在西方文化中可能被视为幼稚，不被欣赏。这种差异源于深厚的历史文化渊源。

## 婚姻：塑造吸引的社会制度

婚姻作为一种古老的社会制度，比“爱情”概念早出现数万年，它深刻影响着我们对长期伴侣吸引力的判断。婚姻的基本类型主要有以下几种：

*   **一夫一妻制**
*   **一夫多妻制**
*   **一妻多夫制**

根据对1231个人类社会的研究，其中588个社会普遍实行一夫多妻制，而一夫一妻制社会有186个。婚姻制度还伴随着各种规则：

*   **族内婚**：必须与族内人通婚。
*   **族外婚**：禁止与族内人通婚。
*   **宗教规定**：如犹太教、伊斯兰教、天主教等对信徒通婚有严格限制。
*   **民俗规矩**：如中国传统的“父母之命，媒妁之言”、“门当户对”等。

以下是世界各地一些独特的婚姻实践：

*   **西非尼日尔（沃达比部落）**：实行一夫一妻制，但每年举行男性选美大会，女性有权选择心仪的男性共度一晚，妻子会全力支持丈夫参与。
*   **猎头族**：婚姻与生死契约相连。新郎必须在婚前猎取一个外族人的头颅，未来孩子的名字将取自这位牺牲者。
*   **中国摩梭族**：实行“走婚制”。男女不组建固定家庭，男子夜间到女子住处相聚，拂晓离开。孩子由女方家族抚养，舅舅扮演父亲角色。
*   **宗教婚礼**：现代常见的白色婚纱、教堂仪式等，源于基督教文化传统，现已全球传播。

婚姻制度是人类为了协调两性关系、稳定社会而发明的机制。它从早期的杂乱状态中发展出规则，使文明得以有序发展。

## 总结与过渡

本节课中，我们一起学习了吸引力背后的社会机制。我们探讨了“性别”概念的文化建构性，以及“婚姻”制度如何作为社会规范塑造我们对伴侣的期待和选择。理解这些，有助于我们认识到，个人对“吸引力”的感受并非纯粹天生或偶然，而是被漫长的文化历史和社会规则所深刻影响的。

当前社会的快速转型，使得传统机制与现代个人诉求之间产生诸多冲突与困惑。下一节课，我们将探讨对这种社会约束机制的反抗——即“爱情”与“浪漫”观念的兴起，看看它们如何成为现代吸引力的核心叙事。
