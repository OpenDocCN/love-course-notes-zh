# 搭讪技巧教程：1：便利商店与快餐店场景实战

在本节课中，我们将要学习如何在便利商店、快餐店等日常场景中，自然地与店员（如收银员、点餐员）展开对话并获取联系方式。这些场景的特点是互动时间短，需要快速建立连接。

很多人想学习搭讪，以为听完课程就能立刻融会贯通，与任何人流畅交谈。这种想法不现实。

你应该从小处着手，让自己成为一个热情大方的人。例如，早上买早餐时，可以对店员说：“看你每天早上都这么早起床，应该很辛苦吧。”对方可能会回答“是”或“还好”。你只需简单聊两句，然后说“谢谢，再见”即可离开。通过经常与店员、店员或周围的人进行简短交流，你能逐渐训练自己与任何人聊上几句的能力。这种训练能帮助你在真正需要搭讪时，快速观察对方的行为、言语或正在听的内容，并以此开启对话。

当你看到一个想认识的店员，例如走进一家便利商店时，可以这样开始对话。如果你住在附近，可以这样说：“第一次看到你，你也住在这附近然后在这里打工吗？”对方的回答是或否并不重要。之后你可以说：“我先去拿个东西，等下再聊。”在拿好东西后，再回来继续对话。

以下是一个导演朋友的实战案例。

他在肯德基点餐时，看到一位漂亮的女服务员，直接对她说：“你是我见过肯德基最漂亮的服务员。我可以帮你拍张照然后发给你吗？”接着他补充道：“但我的相机蓝牙有点问题，不太好传。不如你直接把电话给我，我用蓝牙传给你。”就这样，他成功要到了联系方式。

在便利商店或肯德基这类场合，由于后面可能有人排队，你必须行动迅速，讲话直接突出重点。

你的态度越自然大方，对方的态度才可能越自然大方。如果你讲话紧张、小声，对方很快就能察觉，气氛会变得尴尬。因此，请记住，你表现得越自然，对方才越可能以自然大方的态度回应。

就这样直接向对方索要联系方式。

如果你觉得在排队人多时交谈不便，可以尝试以下方法。

想办法观察这位女生的上下班时间。每次点餐时与她简单聊两句。大约一周后，你可以在她上班前或下班后，在公司门口制造“巧遇”。在这种场合下再与她交谈并索要联系方式，你就不必担心旁边有人注视，也不必担心索要电话会影响她的工作。

要么在搭讪场景中当机立断，要么在工作场景外制造巧遇。这两种方法都能使你的搭讪过程更顺畅，对方也更容易把联系方式给你。

本节课中我们一起学习了在便利商店与快餐店进行搭讪的核心方法：通过日常小对话训练自然感，在目标场景中快速直接地表达意图，或者通过观察制造工作外的“巧遇”机会。关键在于态度自然大方，行动果断。
