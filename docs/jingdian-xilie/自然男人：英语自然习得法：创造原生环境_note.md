# 自然男人英语习得法：2：创造原生环境 🏡

在本节课中，我们将学习如何人为地创造一个英语原生环境，这是实现自然习得的关键。我们将探讨三个核心心态要素，并理解它们如何共同作用，帮助你自然而然地吸收英语。

上节课我们介绍了英语习得不依赖于书本或培训机构，而依赖于原生环境。我们还了解到，原生环境是一种需要人为创造的“软件”条件。本节中，我们来看看如何具体创造这种环境。

## 核心要素一：对英语文化的好奇与开放 🌍

创造英语原生环境所需的最基本心态，是对英语文化保持**好奇**和**开放**。

这个心态可以说是最重要的决定性因素。一个从小生活在中文环境中的成年人，若没有对英语文化的好奇与开放，是不可能把英语讲好的。我们学习任何一门外语，都必须对其文化感到好奇和兴趣，并想去了解。

长期努力学英语却学不好的人有一个共同特点。他们首先认为英语只是一个工具，一个不得不学的东西，用以应付工作、考试或论文。但他们心底并未真正认可英语文化。他们认为英语文化只是语言不同，与汉语世界没有本质区别。他们还会认为，英语世界就是他们眼前世界的另一种语言翻译版本。他们认为汉语已经足够丰富，中国文化足够博大精深，自己也不打算在国外生活，因此领悟外国文化没有必要。

你可能读过我写的一段关于男人和女人世界大小的话。原文是：**你的世界大小决定了你身边女人的优秀程度**。因为优秀的女人往往是见多识广的，她不会选择一个世界比她小的男人。在当今世界，真正优秀的人世界观非常开阔，这种开阔需要建立在对世界文化的理解之上。

你需要知道，英语文化代表的并不仅仅是英国或美国文化。英语文化中涵盖的是世界文化。世界上最经典的文学、艺术、音乐，最尖端的科学技术，最先进的商业管理与投资理念，最古老、最有深度的智慧，几乎都有英语版本。英语世界中文化的丰富程度，远远超过包括汉语在内的任何其他语言。

一个人从汉语文化走向英语文化，开启的是一扇窗户，通往一个在质和量上都远超其原有认知的世界。我们都知道，现在最好的教育多在英语国家的大学，或欧洲顶尖的英语教学大学中。送孩子出国留学已成为许多家庭的常态。越来越多条件优越、受过良好教育的女性都有留学经历或深受世界文化熏陶。这些女性眼界开阔，意味着她们需要选择一个眼界同样开阔的男性作为伴侣。一个经过英语文化洗礼、拥有真正世界观的女人，不会选择一个只把中国当作整个世界的男人做男朋友。

这就好比一个在城市长大的女人，不会考虑一个在农村长大的男人做男朋友。对于一个农村人，来到城市意味着告别村庄、田园和朴实的乡亲，进入人际关系复杂、竞争激烈的环境。他必须放弃原先舒适的生活，迎接新的挑战。即便城市生活残酷、人心险恶、充满诱惑，一旦决定走向城市，就必须开放地迎接这一切。

## 核心要素二：牺牲与放弃已知的语言实践 🚶‍♂️

相同地，一个走向英语文化的人，必须学会**放弃**与**牺牲**，放弃汉语世界曾带来的舒适感。这就是创造原生环境的第二要素。

你需要放弃很多东西。例如：
*   放弃中文电视剧、电影、杂志。
*   放弃一些人际关系。
*   放弃你喜欢听的歌、熟悉的歌星与影星。

你只有放弃很多之后，才会有精力和时间去迎接世界文化。你不可能既期望享受农村的山水空气、熟悉的邻里，又期望完全融入城市生活。你必须放弃前者，将身心置于新的环境中，否则你顶多只是一个生活在城市里的农村人。

因此，你需要开始在一定程度上远离汉语文化。你已经长大，熟知本土文化的主流，懂得人际交往规则，已有足够了解。你可以过几年再回来享受中国文化的精髓。但如果你不走出去看看，不站在更高层面进行比较和参照，恐怕无法真正理解中国文化的精髓是什么。

这相当于农村人适应城市生活后，再回农村时会看得更清楚、了解更透彻，处理事情也更游刃有余。他仍然可以选择生活在农村，但此时的他已与从前不同。同样，一个走向世界后再回看中国文化的人，也会感到更清楚、更游刃有余。

语言习得过程本质上是英语文化渗透你的过程。在这个过程正式开始前，你必须学会放弃令你感到适应和舒适的汉语文化，勇敢地走向未来与未知世界。只有这样，创造英语原生环境才有可能。否则，你将没有任何动力持续推进自己了解英语文化，会觉得英语没什么用处。当兴趣和好奇失去时，任何方法都不会奏效。

## 核心要素三：让语言成为辅助过程 🔧

当你学会放弃，准备好迎接世界文化时，英语就真正变成了一门**语言**，而不是一门**科目**。它变成了一扇通往世界的窗户。也就是说，语言从一线的“科目学习”退居到了二线，变成一个辅助工具。只有当语言退居二线，成为通往你渴望方向的桥梁或通道时，它反而会发挥出最大威力。这就引出了创造语言环境的第三要素。

相信你还记得我前面讲的两个重要概念：**外显记忆**和**内隐记忆**。
*   **外显记忆**属于意识下的记忆，例如人名、历史事件，容易忘记。
*   **内隐记忆**则是过程性的，例如骑自行车、游泳等技能，相对永恒。

语言的自然习得大部分属于**内隐记忆**，是一种过程性、无意识的学习。真正的语言习得是在你不知不觉中听懂和会说的。观察孩子在自然环境中习得语言的过程，你会发现他们并不需要认真坐下来背单词、学语法或翻词典。他们是在使用语言的过程中，自然而然地获得了这种能力。我们每个人都具备这种自然能力。

这种自然能力的发掘，不在于你花多少时间学教材或上培训课，而在于你是否通过语言来达到你的**第一目的**，而语言本身只是为这个目的服务的。

我们打个比方。当你在国外购物时：
1.  你需要读包装盒或问商家，这是不是你要买的东西？
2.  讨价还价时，你需要会说“多少钱”（`how much`）或“太贵了”（`too expensive`）。
3.  决定购买时，你要核对现金数额并听懂对方的话。
4.  对方找钱时，你需要确认金额是否正确。

假设你第一次学这个国家的语言，买两次东西之后，就知道这些话怎么说了，也能听懂他们说什么。例如，我去柬埔寨旅游时学会了几句柬语：“`adad`”（太贵）和“`a供`”（不要）。我只跟导游学了一次，到现在还没忘，而且连这些字怎么写、怎么拼都不知道。

为什么能记住这些话？就像你出国后会清晰记得用过的语言一样，原因是语言成为了为**购物**而服务的第二过程，一个下意识、非刻意的过程。这种习得就会变得非常自然。

听的过程也一样。如果你试图理解对方在说什么，并借助对语言的理解听懂了，那么以后他再说同样的话，你就立即知道意思。前提是，你不是在学那句话的英文单词、分析语法或背单词。那样做效果甚微。反之，如果你的目的是想知道他在说什么，那么语言学习过程就变成自然、下意识且迅速的，甚至可以达到过耳不忘。这就是小孩子学语言迅速的原因——他们不是冲着学习语言去的，他们都有一个第一兴趣或目的，语言只是辅助工具。回忆你的童年，不难发现你正是这样学会汉语的。

## 总结与作业 📝

本节课中我们一起学习了创造英语原生环境的三个核心心态要素：

1.  **对英语文化的好奇与开放**。
2.  **放弃牺牲汉语文化，走出舒适区**，迎接英语与世界文化。
3.  **让语言成为辅助过程**，为你的第一目的服务。

这些是你学好英语必备的心态。你可以把这三条写下来，贴在电脑屏幕或日记本上，时刻提醒自己。

以下是为你布置的家庭作业，分为两部分：
