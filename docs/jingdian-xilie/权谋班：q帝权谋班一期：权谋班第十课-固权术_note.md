# 权谋班：第十课：固权术

## 概述

在本节课中，我们将要学习权谋中至关重要的一环——**固权术**。我们将从权利的来源开始分析，探讨如何巩固和运用手中的权力，并学习在管理和人际交往中如何运用权谋思维来保护和发展自己。课程内容将结合经典案例，力求简单直白，让初学者能够理解并应用。

---

## 一、权利的来源：知其然，方能固其本

上一节我们探讨了权谋的宏观概念，本节中我们来看看权力的根基——**权利的来源**。想要巩固权力，首先必须明白权力从何而来。

根据对《西游记》取经团队这一经典成功样本的分析，我们可以将权利的来源分解为以下五个方面：

1.  **身份**：指个人的出身、背景或社会地位。例如，唐僧的身份是金蝉子转世、状元之子，并得到当朝皇帝的认可。这种身份赋予了他天然的权威。
2.  **道德**：指占据道德制高点，拥有崇高的目标或行为准则。唐僧“十世修行的好人”和“普度众生”的宏愿，使他获得了道德上的合法性。
3.  **授权**：指权力获得的合法性与历史渊源。它意味着权力并非凭空而来，而是有“师出有名”的依据。唐僧的权力来源于如来内定、观音点化和通关文牒（官方授权）。
4.  **外部力量**：指来自外部（如同行、上级、其他组织）的认可与支持。这相当于一种社交认证。唐僧的外部力量主要来自如来和观音。
5.  **强力**：指强制力或暴力，是维护权力的最后手段。对唐僧而言，强力就是控制孙悟空的“紧箍咒”。

**核心公式**：
`权力 = 身份 + 道德 + 授权 + 外部力量 + 强力`

理解这五个来源，你就知道巩固权力需要从哪些方面着手，而不是盲目地使用蛮力或权术。

## 二、权谋的三大核心理念：行动的指南针

在深入具体技巧之前，我们必须掌握权谋思维的三大核心理念。这些理念将贯穿你运用权谋的始终。

### 1. 利益至上

一切行动都应基于**利益计算**。你需要权衡长远利益与眼前利益、整体利益与局部利益。例如，在婚姻选择上，不能只看感情，还要考虑双方家庭背景、性格契合度等长远利益。

### 2. 整体博弈

放弃“单打独斗”的精英思维。现代社会竞争是**团队与团队**、**资源与资源**的较量。要善于团结一切可以团结的力量，动用所有可用资源去达成目标。就像街头斗殴，能群殴绝不单挑。

### 3. 不择手段

这里的“不择手段”并非鼓励违法犯罪，而是指在合法范围内，为了达成目标，要**快、准、狠**，不拘泥于常规或面子。执行力要强，出手要果断，打击要精准。

**核心总结**：这三大理念构成了权谋行动的逻辑链条：**通过利益计算明确战略方向 -> 通过整体博弈组织所需资源 -> 通过不择手段（快准狠）高效执行**。

## 三、管理中的权术运用：激励与控制

当你拥有权力并开始管理团队时，如何运用权术进行有效的激励和控制？

### 激励的艺术：画饼与分饼

对不同的人，要采用不同的激励方式：

*   **对新员工（画饼）**：他们经验不足，最需要的是梦想和前景。要用公司未来的宏伟蓝图（“画饼”）来留住他们，进行“事业留人”。
*   **对老员工/技术骨干（分饼）**：他们经验丰富，需要养家糊口，且可替代性低。必须给予实实在在的利益（“分饼”），如高薪、奖金等实际好处。
*   **对中坚力量/亲信（一起吃饼）**：他们是你的核心团队，需要与他们“同甘共苦”，建立深厚的认同感和归属感。

### 管理的原则：分而治之与制衡

在控制团队时，要遵循以下原则，防止权力被架空：

*   **以点带面**：不需要控制所有人，只需牢牢抓住几个关键人物（如团队中的“孙悟空”），通过他们来管理整个团队。
*   **分而治之**：避免下属形成铁板一块的小圈子。要有意制造平衡，让下属之间存在适度的竞争或制衡，这样他们就会更依赖你。
*   **有名无实 / 有实无名**：不要让同一个人同时拥有巨大的名声（虚权）和实际的利益（实权）。例如，给了管理权（名），就不要让他再掌管财权（实），反之亦然。这能有效防止尾大不掉。

### 摧毁小圈子

如果下面形成了可能威胁你的小团体，你需要：

1.  **经常调动**：定期轮换岗位或负责人，不让人在一个位置上待太久从而培植亲信。
2.  **业务分解**：不讓單一團隊或個人掌握從頭到尾的完整業務鏈，使其無法形成獨立的閉環。
3.  **经常洗脑**：不断强化组织文化和共同目标，让员工认同大集体而非小团体。

### 控制个人的技巧

对于难以管理的个别人员，尤其是“有背景”的人，可以采用：

*   **诿过术**：安排他去做容易出错的事，一旦出错，就抓住这个“把柄”。有了把柄，就有了控制和敲打他的资本。
*   **架空术**：如果抓不到把柄，又无法剔除，就把他放到一个闲职上，剥夺其实际权力，让他无法妨碍你。

## 四、总结与展望

本节课我们一起学习了**固权术**的核心内容。

我们首先剖析了**权力的五大来源**（身份、道德、授权、外部力量、强力），这是巩固权力的基础。接着，我们掌握了权谋行动的**三大核心理念**：利益至上、整体博弈、不择手段，这是思考问题的框架。最后，我们探讨了在具体管理中如何运用权术进行**激励、控制、摧毁小圈子以及对付个别人**。

权谋的学习绝非一朝一夕，本节课只是一个开始。真正的精通来自于：
1.  **持续观察**：在生活中、历史上、新闻中观察和分析各种权力运作的案例。
2.  **深度交流**：与有共同思维基础的人（如本课程的同学）讨论、碰撞，能极大提升认知。
3.  **谨慎实践**：在理解原理的基础上，于实际工作和生活中小范围、低风险地尝试应用。

记住，学习权谋是为了更好地**自我保护**和**实现目标**，而不是为了害人。希望大家能将所学用于正道，不断提升自己的生存和发展能力。
