# 约会游戏：04：8.3 疯狂史诗故事

在本节课中，我们将分享一些在过去八年外出与女性互动过程中，发生的更为疯狂和史诗般的故事。这些故事是我回顾这段旅程时真正脱颖而出的时刻，是那些让我难以置信的经历，也是我真正珍视的记忆。希望这些故事能激励你，让你也想为自己创造类似的巅峰体验。

上一节我们讨论了故事的重要性，本节中我们来看看如何将经历转化为值得讲述的“疯狂史诗”。

## 核心心态：将游戏变得史诗化

这些故事之所以精彩，关键在于一种心态：**将每一次互动都视为创造史诗故事的机会**。这迫使你将焦点从“生存”（例如，避免被拒绝）完全转向“蓬勃发展”（例如，如何让体验变得非凡）。即使面对困难的客观条件，这种心态也能将障碍转化为精彩故事的组成部分。

以下是创造史诗体验的几个核心原则：

*   **拥抱挑战**：将不利的客观条件（如糟糕的 logistics）视为让成功显得更酷的挑战。
*   **寻求乐趣**：这是一个有严肃规则的愚蠢游戏。你必须全力以赴，采取正确行动，但在此过程中，你必须享受乐趣。如果过程毫无乐趣且过于严肃，你将无法持久。
*   **主动创造**：不要被动接受现状。不断问自己：“**如何能让此刻变得更具史诗感？**”

## 早期故事：瑞士的疯狂开端

我的第一个史诗故事发生在瑞士，那是我外出约会的早期阶段。当时我已取得一些成果，但体验仍比较平淡，我更多是在“生存”而非“蓬勃发展”。

一晚，我和朋友决定喝醉（注：此为早年不成熟行为，不推荐模仿）。在俱乐部，我径直走向一个女孩，几乎想直接亲吻她。她拒绝了，但我再次尝试，然后要到了她的号码。第二天我发短信邀约，她竟然对我前晚在朋友面前的大胆行为感到兴奋。我们当晚相约见面。

为了延续这种疯狂，我决定再次喝醉赴约。在极度兴奋的状态下，我让她开车来接我。上车后，我让她随意开，最终我们到了一个可以俯瞰城市和湖泊的公园。在见面后的5到10分钟内，我们就发生了关系。后来得知她当时甚至已订婚。整个经历发生得极快且疯狂，当时我觉得：“这太他妈酷了。”

这次经历让我“上瘾”了。我开始假设结果必然会发生，而我的全部焦点变成了：**如何让每次经历都像这次一样辉煌？**

## 将不利条件转化为史诗素材

拥有“史诗化”心态后，你可以将任何看似不利的客观条件（Logistics）转化为精彩故事的背景板。

**瑞士的“荒野求生”式 Logistics**
在瑞士，从俱乐部带女孩回家是一段“旅程”：需要步行10-15分钟到火车站，沿途只有毒贩在叫卖“可卡因”；然后在车站可能等待长达一小时；坐上火车10-15分钟到下一站；最后还要在漆黑、寒冷的田野中步行30分钟回我的住处。

你可以视之为噩梦，也可以想：**“如果我能在这种情况下成功，那该是多棒的故事！”** 我选择了后者，并成功带许多女孩穿过了那些田野。如今回想，那非常酷。

**洛杉矶的“屋顶格拉瓦”时期**
初到洛杉矶时，我很穷，住在治安很差的街区。为了约会，我通常会把女孩带到朋友的屋顶。那不是一个浪漫的屋顶，而是铺满碎石子、没有景色、没有椅子的简陋地方，有时还得担心管理员上来。

同样，这可以视为不便。但我的想法是：**“这将会是一个史诗般的故事。”** 我告诉女孩们“去参加 after-party”，然后把她们带到这个屋顶。只要你的意图清晰，女孩们即使发现没有派对，也不会多问。我在那个屋顶上与很多女孩共度了时光。

## 更多疯狂故事实例

以下是几个具体的故事，展示了如何在各种情境下应用“史诗化”思维。

**故事一：玻璃窗惊魂**
有一次在朋友房间，我和一个女孩亲密时，她的腿意外踢穿了薄玻璃窗，碎片扎进脚里，血流得到处都是。我快速处理完，她的朋友非常冷静地帮忙清理、包扎，甚至给了我们修窗户的钱。更疯狂的是，事后我和朋友交换了女伴。整个经历既惊险又荒谬。

**故事二：迈阿密惊魂夜**
在迈阿密，一个非常火辣的女孩在凌晨4点发来地址让我过去，车程30分钟。我的心态是：**“即使什么都没发生，这也将是一个绝妙的故事。”** 我打车过去，发现是她和男友的公寓，男友夜班，清晨才回。意识到危险后，我偷偷用手机记下地址和她的名字以防万一。然后，在“男友可能随时开门进来”的巨大恐惧中，我与她发生了关系。那可能是我经历过最紧张的性爱。

**故事三：拉斯维加斯电梯困境**
在拉斯维加斯峰会期间，我和朋友在酒吧成功搭讪两位澳大利亚美女，准备一起回酒店房间。一位从2010年就关注我的粉丝也加入进来，但他对我的朋友的女伴行为过界，导致场面尴尬。走向电梯时，我另一位朋友巧妙地让保安拦下了他。在电梯门即将关闭时，那位粉丝不断喊我的名字求救，所有人都看着我。我面临选择：承认认识他会毁掉这次约会；不承认则显得冷酷。最终，我移开了目光，电梯门关上。第二天我们澄清了误会，但那一刻如同电影场景。

**故事四：房间里的“敬业”DJ**
成功进入女孩们的酒店房间后，我们发现里面还有一个男性朋友，他正在用电脑工作。我们以为他很快会离开，但他却从行李箱拿出一个DJ混音台，开始认真地打碟。他持续了30-45分钟，完全没有停下的意思。我和朋友只好在床上与各自的女伴开始亲密，而他……居然配合地改变了音乐节奏。更戏剧性的是，我的女伴发现我从事约会指导，并说出我一位企业家好友的名字，原来她正是那位好友正在约会的人，手机里全是他们的亲密合照。整个经历充满了荒诞的巧合。

**故事五：浴室与“沉睡”的室友**
一次，我把女孩带回她的房间，发现她的女性朋友已经睡在床上。我本可以在浴室解决问题，但我想：**“怎样会更史诗？”** 于是我把女孩带到床上，就在她“沉睡”的朋友旁边开始亲密，甚至试图弄醒那位朋友。未果后，在结束时，我转向那位“沉睡”的朋友，将体液留在了她的背上。这无疑将那个平凡的夜晚变成了一个令人难忘的（尽管有些离谱）故事。

## 注意事项与平衡之道

追求“史诗化”需要智慧，避免惹上麻烦。

我曾与一位洛杉矶俱乐部推广人发生矛盾。我与他喜欢的女孩约会，并幼稚地制造了戏剧冲突，导致他在俱乐部当场难堪。为了“让事情更史诗”，我随后故意与他最好的女性朋友以及其他他心仪的女孩约会，并在他面前炫耀。这是一种非常幼稚的 ego 行为，差点让我被所有俱乐部封杀。请务必小心，**不要让对“史诗”的追求损害你的社交关系或人身安全**。

## 美好而史诗的夜晚

史诗故事不总是疯狂或荒谬的，也可以是美好而辉煌的。

我印象最深的一个美好夜晚在纽约。在一个周日晚，我与朋友们以及一位刚恢复单身的意大利美女一起。在一家酒吧，我搭讪了一位金发美女，制造了意大利美女的嫉妒情节，并拿到了金发美女的号码。在下一家酒吧，我又搭讪了一位极其出色、聪明的女孩，她竟然与我所在的创业峰会圈有交集，我们非常投缘。最终，我成功带这位聪明女孩回到了她位于中央公园旁的顶层豪宅公寓。望着窗外的中央公园，那一刻感觉无比辉煌。第二天我飞往迈阿密，心情依然在高点。

## 总结与行动指南

本节课中我们一起学习了如何通过“史诗化”心态，将约会游戏转化为创造难忘故事的旅程。

请记住以下要点：
1.  **被激励**：看到从普通人到经历这些精彩故事的可能性。
2.  **被娱乐**：在过程中找到乐趣，娱乐自己。
3.  **主动创造**：无论顺境逆境，都问自己：“**我如何能让此刻变得更具史诗感？**”
4.  **正确看待困难**：困难是故事的调味品。如果一切顺利平淡，故事也将索然无味，你甚至会很快忘记。正是这些充满挑战、疯狂或美好的史诗时刻，构成了你真正为之生活的记忆。

那些你以为你想要的、“一切顺利”的普通约会，往往是无聊且容易被遗忘的。你不会忘记的，是这些史诗故事。

现在，去创造属于你自己的史诗吧。
