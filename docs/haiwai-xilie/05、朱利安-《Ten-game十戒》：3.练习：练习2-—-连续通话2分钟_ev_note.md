# 社交突破训练：05：练习2 — 连续通话2分钟 🗣️

在本节课中，我们将学习一个名为“两分钟连续通话”的练习。这个练习旨在帮助你克服内心的阻碍，学会流畅、不间断地表达自己，从而在社交场合中更加自信和自如。

## 练习概述

这个练习需要两人一组进行。它的核心规则非常简单：在指定的两分钟内，你必须背对同伴并持续不断地说话，不能有任何一秒的停顿。

## 练习规则详解

以下是进行此练习的具体步骤和规则。

1.  **分组**：参与者需要两两配对。如果是四人小组，就分成两对。
2.  **姿势**：其中一人转身，背对自己的同伴。
3.  **计时**：练习将严格计时两分钟。
4.  **核心要求**：背对同伴的练习者必须开始不间断地说话。在这两分钟内，**不允许有任何一秒的停顿**。
5.  **内容自由**：你可以讲述故事、提出问题，或者谈论任何想到的内容。内容本身不重要，关键在于“持续不断”。
6.  **同伴职责**：聆听的同伴需要确保说话者没有停顿。如果出现停顿，将会有相应的“后果”（旨在增加练习的挑战性和趣味性）。

## 练习的核心目的

上一节我们介绍了练习的规则，本节中我们来看看这个练习背后的深层目的。

这个练习听起来只有两分钟，但实际做起来会感觉非常漫长。要顺利完成它，**唯一的办法就是关闭你内心的评判机制**。

*   如果你在过程中有任何自我形象顾虑，或者评判自己说的话是否合适、是否有趣，你将无法快速流畅地表达。
*   你可以慢慢地说话，但若想达到要求的快节奏，你就不能去思考或分析你正在说的每一个字。
*   话语应该自然而然地流淌出来。一旦你开始逻辑分析，比如想“这句话通顺吗？”或“这值得说吗？”，你的语速就必然会慢下来。

这个练习能有效打破你脑海中的阻塞，帮助你摆脱无法充分表达自己的困境。

## 如何进行日常练习

这个练习非常适合日常训练。我建议你每天都可以做。

*   **基础练习**：每天早上起床后，可以对着镜子中的自己进行这个练习。站在镜子前，开始不间断地说上两分钟。
*   **进阶挑战**：当你能够轻松完成两分钟后，可以逐渐增加时长：尝试五分钟、十分钟，甚至十五分钟。
*   **效果**：如果你能做到连续十五分钟不停顿地快速表达，你会发现自己在真实社交场合中的表现将得到巨大提升。

## 练习示范与执行

现在，让我们切换角色，开始第二轮练习。记住，用你所能达到的最快速度去说，就像唱一首快节奏的饶舌歌。

准备好：3， 2， 1， 开始！快，加快速度！
