更进一步说，它不仅仅是“良好情绪”，而是**总体上积极的强烈情绪**。这一点非常重要。托德曾谈到，人们是在情绪发生变化时感受到情绪的。这意味着，当你从感觉糟糕转变为感觉良好时，那种情绪的**转变**会让你感觉最好。转变越大，冲击力就越强，产生的积极感受也就越强烈。

你可以在典型的电影情节中看到这一点：开头不错，然后有所铺垫，接着出现低谷（总有些糟糕的事情发生），最后是美丽的结局。你在结局感受到高潮，正是因为你经历了从“天哪”到“他赢回了她”的转变。这种转变产生了强烈的冲击。

在与女孩互动时，我之所以能真正脱颖而出、深深吸引她们，就是因为给了她们这些强烈的情绪波动。很多男人害怕带来负面情绪，他们来自“讨好者”的心态。但我能做到把她们的情绪带向低谷，然后再把她们拉起来。当我之后把她们的情绪提升时，其他任何男人都无法为她做到这一点。她永远不会在其他男人那里体验到如此强烈的积极转变。于是她对此上瘾了。
