# 大卫X规则课程：17：一致性原则 🧩

在本节课中，我们将深入探讨“一致性”这一核心规则。我们将回顾前两条规则的重要性，并通过具体的故事和作业示例，帮助你理解如何在实际互动中应用这些原则，克服恐惧，并建立有效的沟通模式。

上一节我们介绍了规则的基本框架，本节中我们来看看如何将“不害怕拒绝”这一核心理念付诸实践。

规则一和规则二至关重要。尽管你现在可能尚未完全理解，但随着你不断回顾、消化和领悟，你会意识到它们极其重要。基于此，你几乎可以应对任何情况。

你不能害怕拒绝。许多男性存在这种恐惧。当拒绝发生时，你往往会觉得“哦，也没那么糟”。我们恐惧的是未知。拒绝，你可以被拒绝一千次，但只需被接受一次。这一次就足以弥补那一千次。

许多男性问我关于那些我接近过的漂亮女性。我接近过成百上千位。但我成功的次数，多到我自己都应付不过来，甚至多过我实际想要的。有时女性会抱怨：“你从不给我打电话，你根本没时间陪我。”她们说得对。我的贪婪，我无法控制。我看到另一个，就想尝试。我必须拥有。我想要一个意大利女友，我想尝试希腊女孩，我尝试过法国女孩、德国女孩、波兰女孩。我喜欢波兰女孩，她们更温柔。我在开玩笑。

我必须完成我的清单。我列了一个清单，写下所有我想尝试的类型。我很幸运。蒙特利尔就是这样一座城市，移民众多。我们这儿的法裔加拿大女性对性的看法不同。

以下是一个故事，可以概括法裔加拿大女性的特点，因为你们可能对此有不同印象。大多数和我一起长大的英语区男性不去法语区，因为他们对说法语感到不自在。我从未有这种舒适度问题。我和一个人成了朋友。有一次，他带我去他家，见他的新婚妻子（她比我年长，我们是同事）。他带我去他家，在他母亲家的一次圣诞派对上，我见到了他的新婚妻子。她说：“你为什么不和我妹妹约会呢？”我看了看她妹妹，她很漂亮。我说：“哦，当然好。”妹妹会说英语，但她的父母几乎不会说英语。

于是我和妹妹约会。我告诉我的朋友：“哦，我爱法国女性，她们太有趣了。”她过去常在她家里，父亲在隔壁房间看电视时，带我去卧室给我口交。我当时觉得：“不，你父亲不应该发现。”有一天，我去接她，她还没准备好。典型的法裔加拿大人，在冰球之夜，他们会坐下来，搬一箱啤酒，对着电视，整晚喝酒看电视。法裔加拿大人爱他们的啤酒和电视。我进去后，她父亲递给我一瓶啤酒。法语里他们说“o tablet”，意思是室温啤酒，不冰。我喜欢喝冰啤酒，但我是客人。我怎么能拒绝？我喝着啤酒。那个男人看着我说：“你喜欢我女儿吗？”我说：“哦，是的。”他说：“她床上功夫不错，嗯？”我当时……啊啊，哦。我可不是个经常语塞的人。好吧。我坐在那里喝着啤酒，你无法想象我脑子里在想什么。然后他接着说：“她妈妈把假牙取下来后，口活很好。”我坐在那里，不知道他是不是在试探我。这时，女朋友进来了。我当时脸色一定红一阵白一阵，反正肯定不正常。我喝不下啤酒，惊呆了。她说：“我们走吧，是我父亲打扰你了吗？”我说：“不，不，不，不，他很好。”他说：“好的。”她告诉他。他对她说：“在你取下假牙之前给他口交。”我坐在那里，心想：“我们现在能走了吗？我想离开这儿，因为你父亲太不可思议了。”你知道他问我什么吗？她说：“是啊，我床上功夫好吗？”我说：“那是你父亲。我该怎么回答？我该怎么……”我选择沉默。我的家人就是这样。

几周后，在另一次派对上，在她祖母家，我不得不去参加。我们在那里，他们都在谈论当他问我她床上功夫好不好时，我脸有多红。我的朋友和他妻子（她的姐姐）也在那里，他们刚生了个小宝宝。然后我看到他和妻子说话，妻子又和她父母、祖母交谈，他们都开始大笑。祖母的丈夫去世了，但她请了当地的牧师来家里。突然，祖母对牧师说了些什么。牧师对我说（他英语说得很好）：“去卧室，请把她梳妆台上的耶稣像拿来。”我走过去拿起耶稣像，就这样走进房间，他们都歇斯底里地大笑。我心想：“有什么好笑的？”我女朋友说：“把你的手转过来。”我照做了。结果另一面是一个阴茎，一个这么大的该死的阴茎。她说：“我祖母和牧师玩那个。”我当时想……无论何时我在那里，我总是更受惊吓。这让你对法裔加拿大人看待性的方式有了一点了解，他们对此非常开放。他们像巴西人一样吗？我不知道。

当我告诉我的朋友时，没有一个人相信我。他们说：“得了吧，父亲问你是不是觉得他女儿床上功夫好？”我说：“是的。”他们说：“那你怎么办了？”我说，那是我人生中第一次如此尴尬、害羞，不知道该怎么回答。我告诉我父亲（我和父亲关系很好），我去她家，她父亲问我她床上功夫好不好。他看着我，只说了一句：“法国人。”这就是全部。

关键在于：你永远不会不感到惊讶。你认为可能的事，是可能的。你梦想的事，会成真，因为你会让它成真。

上一部分我们通过一个生动的故事了解了文化差异和开放性，现在我们来探讨具体的实践作业。

我今晚给你们布置的作业，也是我给每个人的第一次作业，是：他们必须坐下来，写下从研讨会或第一次课程中学到了什么。然后我希望他们发邮件给我，我会批改。我告诉他们什么是重要的，什么是他们没看到的。因为我需要进一步推进。

第二次作业是去获取一本禾林小说。去书店，看看那个专区有多大。进去，随便挑一本你想要的。别买太贵的，我不知道这里卖多少钱，但就几美元。带回家。这是我最喜欢的部分：阅读它。好吧，我永远不会读禾林小说，因为我懂，所以不需要读。当然，你会看到那些内容。进去后，要么把你读过的那本给她（因为平装书读过后会有痕迹），告诉她你读过；要么给她一本全新的。就像我说的，显然这根本不重要，送她一份礼物。这是很好的圣诞礼物。很快。你甚至不用看封面。送给她，你会惊讶于它的效果有多好。我曾经送出过144本。

当我过去给女孩买礼物时，要么是一朵玫瑰（那时一美元），要么是这些小小的钥匙扣泰迪熊。我会告诉她们：“我有个礼物给你，见面时给你。”这只需几美分。但我会说：“当你孤独想我时，有个小泰迪熊陪你。”哦，你真浪漫。我从不买昂贵的礼物。最便宜的小礼物往往意义最重大。

要保持连续性。如果你和她约会，不能每次想和她上床都送禾林小说。对同一个女孩，只能送一次。你可以送给另一个女孩另一本，这没关系，但对那个女孩，不要做两次。有时事情会适得其反，那就尝试别的。不送泰迪熊，送点别的，比如哨子。在我发现泰迪熊之前，我常送哨子。“看到我时就吹哨子。”“当你送我时，你知道的，一些浪漫又傻气的东西。”因为浪漫和傻气是我们认为的，但她们认为那是浪漫，觉得你很贴心。

你可以尝试这些，也可以想其他东西送给她们。

在学习了如何通过小礼物建立联系后，我们来看看如何通过提问来深入了解对方。

另一项作业是，你可以列一个问题清单，但在列清单之前，你必须在一张纸上写下每次见到一个女人时，她吸引你的地方是什么？她长什么样？你想对她做什么？她是金发，三围是36，22，36，我想舔她的阴部，把我的阴茎放进她的屁股，或者任何你想到的，写下来。给她编个号，比如“这是我做作业以来的第三个”。第一个，为自己列一大堆东西。不是她认为你想说的，而是你想对她做的事。如果你不知道想对她做什么，发张照片给我，我告诉你我想对她做什么。好吧，我通过你们来间接体验生活。就是这样。

现在，作业的最后一项：列一个问题清单。每一个问题。比如：“你相信60%的女性从未有过性高潮吗？”“你认为关系中最重要的东西是什么？”“你有几个兄弟姐妹？”“你会在工作服下面穿性感内衣吗？”“你最喜欢的颜色是什么？”我喜欢把非常普通的问题混在里面。还有性问题，但不要1，2，3都是性，否则她们会认为你的模式就是性。这就是为什么我说，要像这样安排。比如：“你最喜欢的水果是什么？”“你最喜欢的蔬菜是什么？”“你第二喜欢的水果是什么？”“第二喜欢的蔬菜是什么？”

通过这样做，当你提问时，你会学到和我一样的模式。你会明白它们的含义。“你喜欢在烛光下泡澡吗？”“你喜欢香槟和橙汁吗？”“你喜欢伏特加吗？”“你喜欢我吗？”或者任何你想问她们的问题。你想了解她们，你想让她们认为你想了解她们。你这样做是为了获取信息，了解她们，了解你想要什么，以及如何引导她们。

所有的问题。“你是个随性的人吗？”“你在关系中有没有一套规则，第一步、第二步、第三步、第四步做什么？”“到什么程度你会和男人上床？”“你和多少男人有过关系？”“上一次恋爱是什么时候？”“谁提出的分手，是你还是他？”永远不要问发生了什么，这不重要。就问谁提出的分手，是你还是他？这些问题，你必须思考。当我和女人交谈时，这些问题会自然流露出来。实际上我打算录制更多内容放到我们的网站上，你们可以听所有这些我有的问题。我告诉过你们，我有一个非常有趣的。我总是，嗯，在和女性说话时，我总是想看看我在和多么聪明的人交谈。所以有一天我在想，世界上最愚蠢的工作是什么？一些正常人绝不会相信的，最愚蠢的，真正底层的工作，如果你对一个男人说，他会像看傻子一样看你。你会惊讶有多少女性相信。你知道吗，好问题。这就是你需要想明白的。这不是审问。我想了解你，我想沟通，她们也想沟通。所以你觉得是审问，但如果你觉得是审问，我告诉你：如果不问问题，别人怎么能了解你呢？在任何事情上，当一个人犯罪时，他们会坐下来问他问题，进行访谈。因为通过问题，真相会浮现。你在审问她们，好吧，如果她们不喜欢，不愿意回答你的问题，那就找下一个。你在想“如果……”，你又违反了第一条规则。你看，你在想什么。我从来没有遇到过不想回答问题的女性，抱歉，我得收回这句话。我遇到过不想回答问题的女性，她们糟糕得可怜。“我不回答问题。”“谢谢。”当我开始问问题，并且本打算请她吃晚餐时，我发现这样更好。我停一下。
