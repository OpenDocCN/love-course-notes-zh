# 言语游戏：09：1-8周：第一周（接近焦虑）：3.开场白：3.间接打开 💬

在本节课中，我们将要学习间接开场白的核心概念。我们将探讨它的定义、适用场景、优缺点，以及它如何融入你的整体互动策略。理解这些基础后，后续学习的具体技巧将更容易掌握。

上一节我们介绍了直接开场白，本节中我们来看看它的对立面——间接开场白。

间接开场白在本质上与直接开场白相反。直接开场白的优势在于**快速切入主题并推动互动向前发展**。其缺点是，它要求对方（女孩）立刻做出决定。而间接开场白的优势在于，**它不要求对方立刻做出决定**。这为你争取了更多时间，让你能在要求对方做出决定前，充分展示你的个性与积极特质，给她留下良好印象。因此，如果你是一个懂得正确操作的“酷家伙”，这将是一个优势。

然而，间接开场白也有其缺点。你推动互动发展的速度远不如直接开场白快。这意味着在开场之后，你需要做更多后续工作。在某些特定环境下，你甚至可能因为互动发展过慢而“时间耗尽”——女孩可能被朋友拉走、到达目的地，或者在你切入正题前就上了网约车。这同样可能造成不利影响。

总而言之，间接开场白是一种**进展较慢的构建方式**，即使成功，其即时达成的效果也不如直接开场白。但它的优势在于，**能以良好方式开启互动的成功率（百分比）更高**。
