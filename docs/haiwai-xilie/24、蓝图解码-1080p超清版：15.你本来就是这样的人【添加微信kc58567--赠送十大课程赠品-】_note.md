## 现实强度的深层定义

上一节我们介绍了现实强度的表面定义，即你对自己信念的坚定程度。本节中，我们来看看这个定义的扩展。

我们将现实强度定义为：你对自己信念的坚定程度，以及这种如磐石般稳固的确定性将他人吸引进你世界的能力。这是现实强度的表层定义。

现在，我想扩展这个定义。我将把这个定义提升到一个新的层次。

如果你从未这样思考过，你可能会开始审视你认识的人，并可能感到好笑。

人们可以通过两种不同的方式拥有强大的现实感。一种是像这样。

**知识结构：**
```
信念1
信念2
信念3
...
```

如果这是你思维的构建方式，那么当你推倒其中一根支柱时，整个结构会崩溃吗？

许多拥有非常坚定现实感的人，包括许多固执的领导者，比如某些以自我为中心的CEO或政治领袖，他们的现实感看起来更像是这样，甚至像这样。

**网状信念结构：**
```
信念A <-> 信念B <-> 信念C
  ^          ^          ^
  |          |          |
  v          v          v
信念D <-> 信念E <-> 信念F
```

这里发生了什么？他们将信念相互连接成一个网络。人类这样做是绝对迷人的。在我们讨论之后，我会请你们去观察他人，当然也包括自己，看看是否能观察到这种现象。

## 强化现实感的策略

有些人为了增强自己的现实感，为了给自己更强的确定性，会持续地将一个信念与另一个信念联系起来。

他们会说：“当然，那会成功。我做过这个。如果那是真的，那么这个也一定是真的。”你们明白吗？你们是否认识这样的人？

举个例子，比如一个简单的信念：“我永远不会窒息，因为我需要呼吸。”你希望相信氧气是充足的，因为你感觉你需要能够呼吸。

这类似于我之前谈到的“由评判、比较、解释和标签构成的不透明之墙”。如果你总是试图给事物贴标签以寻求安全感，比如“这个标签关联那个，那个标签关联这个”。

## 重新定义强大的现实

那么，这把我们引向什么方向？首先，它重新定义了我们关于强大现实的概念，不是吗？因为拥有那种现实的人，第一，他们拥有强大的现实；但第二，它实际上是脆弱的。你们听说过“结构越僵化，适应性越差”这个概念吗？

假设生活就像一条路。起初，路很宽，这是你的信念结构。但假设后来路变窄了。这是你庞大的信念结构，它来到这里，会发生什么？它崩溃了。

从进化的角度来看，结构越僵化，越容易崩溃。当我们谈论自我时，当你对自己有一个宏大的概念，突然你得到的反馈显示你不是那个人，你的自我感就会崩溃。

这实际上回到了古老的东方哲学表达，有人能说出它吗？是“以柔克刚”吗？不，我喜欢那个。也许是“上善若水”。好吧，“像水一样”。基本上意思是：你没有那种僵化的结构。

这在武术中如何应用？基本上，在武术中，你可以在李小龙的《截拳道之道》这本书中读到。他基本上是说，假设你对打斗有一个概念，比如，大多数人那样出拳。当他们那样出拳时，我这样回应；当他们那样出拳时，我那样回应。然后你在现实世界中进入一场真正的打斗，如果他们攻击你的方式不是你预期的，你的整个自我感就会崩溃，因为你将自我投入到了你的武术方法中，所以发生的是你崩溃了。

这是李小龙在《截拳道之道》一书中的观点，一个真正迷人的想法，至少据我理解，它革新了武术世界。

## 两种现实概念

所以，关于弱现实和强现实的两种概念。一种是，是的，你有一个强大的信念系统。但另一种，是一个灵活的信念系统，能够通过瓶颈。

那么，我们接下来的概念是：**重新排序感知以保留地图**。

这个概念非常重要，我要请你们迁就我，跟着我重复。首先我说，然后你们重复：“重新排序感知以保留地图。”

人们一直在这样做。让我告诉你们。快速筛选想法并接受或拒绝它们的能力，实际上是一种力量和技能。事实上，甚至不止于此。你越能适应，允许相互冲突或矛盾的现实存在于你的脑海中而不被它们关闭，你就能越容易地在不同地图之间切换。在别人会跌倒的地方，你将屹立不倒。

让我们看一些矛盾现实的例子。你们记得我说过人们如何经常合理化吗？他们会合理化并做对自己最有利的事，那是一种现实概念。但你们还记得我也谈过当你看到人们最好的一面，甚至在所有人身上看到自己时，人们会非常积极，你愿意相信个人的最好一面吗？这在表面上有矛盾吗？是的，表面上当然有矛盾。

另一个概念，比如“在他人身上看到自己，成为整体的一部分”，对比像理查德·道金斯或马特·里德利类型的现实，我们说“自私的基因”。这在表面上有矛盾吗？这是一个难题。你们是否遇到过真正沉迷于整个进化论，并真的相信他们的整个生活就是生存和复制，他们必须出人头地，就像马基雅维利式的概念？这很有趣，因为你看，不同的人有不同的个性侧面。有时人们可以展示一面，有时可以展示另一面。

但每当我看到人们沉溺于那种想法时，在他们沉溺于其中的时候，他们看起来如此空虚。我记得看到理查德·道金斯本人在科尔伯特报告上。他在谈论，你知道，人们有他们的自私意图，这就是他们如何获得性等等。我看着道金斯本人，我想，这家伙是个了不起的科学家，在各个层面上都是了不起的人，一个我很荣幸能见到的人，但我不认为犹太人能泡到妞。你们明白我的意思吗？那不是我的印象。他看起来非常僵化。我其实很想有机会带他去夜店，真的，你们懂我的意思吗？并从他那里学到无数东西。我敢肯定他可能没兴趣，但我是说，他是个科学人，他走了那条路，我只有尊重。但关键是，有些人通过那种“哦，每个人都是自私的，都想得到自己的”来解释世界，这本身就有些不对劲。

所以我们必须能够处理那些矛盾的现实。一方面，我们可以看待像“自私的基因”这样的东西。另一方面，我们可以在他人身上看到自己，并且不被那干扰，就像不需要解决它，如果那说得通的话，不需要让它必须合理。

为什么？因为地图不是疆域。你们听说过吗？这是一个NLP术语。地图不是疆域。我们拥有的每一个现实概念都只是一个概念，是对现实的标签，不是现实本身。所以，尽管地图可能很棒，它只有有限的能力让你了解你所看到的东西。
