# 言语游戏：09：接近焦虑导言 🎯

在本节课中，我们将要学习“接近焦虑”这一核心概念。接近焦虑是许多人在尝试与陌生人，特别是心仪对象交谈时，会体验到的一种紧张或恐惧感。我们将探讨它的本质、根源以及如何克服它。

## 接近焦虑的本质：真实与虚幻

上一节我们介绍了课程主题，本节中我们来看看接近焦虑的矛盾本质。

我教授恋爱技巧已有15年。接近焦虑是一个有趣的困境。一方面，它非常真实。另一方面，它完全是胡扯。

在我学习恋爱技巧的成长过程中，我有很长一段时间深受接近焦虑的困扰。直到18岁，我想和很多女孩交谈，但从未付诸行动。我总是给自己找借口，比如“她很忙”、“我不知道该说什么”或者“她不够有魅力”，即使她明明很有魅力。我会找理由不去行动。这可能是阻碍我提升与女性交往能力的最大障碍。是我自己限制了自己。从这个意义上说，它非常真实。

从另一个角度看，接近焦虑又非常不真实，或者说完全是胡扯。我之所以这么说，是因为在我的课程中，每一个有接近焦虑的学员，只要给予足够的欲望、动力和社会压力，他们都能够克服它。这从来不是一个无法逾越的障碍。所以，它显然是你头脑中的产物，并非真实存在于现实世界。但它确实是一个非常真实的困境。

## 进化根源与当代现实

公平地说，接近焦虑有其真实的来源。它源于我们进化于小型部落的事实。我们一生都围绕着相同的人群成长。做出社交上愚蠢或尴尬的行为，可能对我们产生非常负面的影响。

例如，想象你生活在一个小部落里。部落里最漂亮的女孩是酋长的妻子。如果你去接近她，尤其是方式不当，但即使是任何接近行为，都可能导致酋长杀死你或将你驱逐出部落，让你在沙漠或森林中自生自灭。因此，这种强烈的恐惧感确实有其进化根源。

然而，在21世纪这个非常安全的世界里，这种恐惧在某种程度上没有意义，可以说是胡扯。

## 理解与接纳你的焦虑

所以，我希望你理解，接近焦虑确实有其道理。你有这种焦虑并不意味着你是个坏人，也不代表你不够男子气概。它有进化的原因。但是，也要明白它是非常不理性的，在当下并不真正合理。

这就像任何恐惧一样。任何恐惧都源于自然，但在某些情况下可能没有根据。我记得小时候第一次走上高台跳板（实际上是跳下去，因为跳水更可怕），我走到跳板尽头往下看，吓得走回去爬下楼梯，整个泳池的人都在看着我。但当我最终说服自己跳下去后，发现乐趣无穷，想一次又一次地尝试。接近焦虑对你来说也应该是同样的道理。它是恐惧，但这种恐惧也是肾上腺素带来的兴奋感，同样可以转化为积极的事物。

## 课程路线图：理论与行动

接下来，在接近焦虑课程的其余部分，我将为你提供五种看待和处理接近焦虑的不同方式。

以下是五种全局性的解决方案框架：
*   **视角多元**：包括生理、哲学和生理心理层面的方法。
*   **剖析根源**：深入分析焦虑产生的原因。
*   **拆解策略**：提供具体的拆解和克服方法。
*   **方案实操**：每种视角都对应可操作的建议。
*   **综合应用**：帮助你从多角度攻克焦虑。

然后，我们有两套不同的行动课程。

第一套行动课程适用于尚未进行过很多搭讪的学员。你们将进行我称之为“渐进式脱敏”的训练。以下是其步骤：
*   **从易开始**：通过相当简单的体验逐步建立信心。
*   **循序渐进**：过渡到稍微困难一点的场景。
*   **持续挑战**：逐步增加难度，直到你能完成一次完整的搭讪。

第二套课程适用于那些已经进行过搭讪，但在某些特定情况下仍会感到接近焦虑的学员。例如，普通搭讪可能没问题，但接近一群女孩、接近有男性同伴的女孩，或在众目睽睽之下接近，可能仍会让你感到害怕。这是我们行动课程的第二阶段。
