# 价值大师班：06：你的价值来自哪里：生活在现实中 🧠

在本节课中，我们将探讨“生活在现实中”这一核心概念。我们将学习如何停止与现实的对抗，并通过具体的练习来培养活在当下的能力，从而提升个人效能与魅力。

## 与现实的冲突 🚫

如果你与现实发生冲突，你将在与人相处、获取成功等方面遇到巨大问题，并会制造大量借口。

因此，生活在现实中的第一步，我称之为“不要对自己开枪”。顺便一提，这个短语托尼·罗宾斯也使用，但含义不同。我喜欢这个说法，所以借用了它。在这里，我指的是停止使用“应该”这个词。

“应该”这个词可能是应对现实最糟糕的方式，因为它本质上是在否定现实。每当你认为“这件事应该发生”，你实际在说“现实是这样，但我反对它”。每当你认为“这件事不应该发生”，你也是在说“现实是这样，但我反对它”。与现实对立是行不通的，这不是一种有效的生存方式。

## 摒弃“应该”的练习 🔄

每当你发现自己使用“应该”这个词时，你实际上是在无谓地制造压力。你在制造压力，因为你声称现实如此，而你却与之对立。

以下是我想让你们做的练习。

每次你使用“应该”这个词时，我希望你将其反转。例如，如果你在想“我应该有一份更好的工作”，那么一旦你意识到自己说了“应该”，我希望你立刻转换思路。

你应该这样想：在现实中，我不应该有一份更好的工作，因为我没有完成学业，或者因为我每天上班迟到，又或者因为我还没有整理简历去申请更好的工作，等等。

因此，与其抱着世界欠你什么或事情“应该”如何的念头，不如认清现实——在现实中，事情“本就不应该”如你所愿。然后，开始专注于改变那个现实。

我希望大家意识到，每次使用“应该”这个词时，你很可能错了。因为现实，至少在平均意义上，就是它“应该”的样子。当然，有些人非常不幸。但即便如此，例如“我不应该遭遇车祸”——实际上，从统计上看，我应该，因为在美国每年都有一定数量的人会遭遇车祸，这只是我的统计概率事件。另一方面，我“本不应该”出生，因为那是一个万亿分之一的机会。所以，我能出生非常幸运，而遭遇车祸则是不幸。世界基于统计运行。我们必须接受现实，与之和解，然后继续前进并寻找解决方案。专注于“应该”是个坏主意。

## 活在当下的力量 ⏳

关于“生活在现实中”这一点，我主要想推荐一本书，即《当下的力量》。这本书非常出色。就书籍推荐而言，我强烈推荐这本书。书中的核心理念是：只有“现在”是存在的，过去不存在，未来也不存在。你越专注于实际存在的事物——即“现在”，你的状态就会越好。这是一个改变人生的概念，但大多数人并未践行。

我将聚焦于此，并提供一个练习。你可以去读那本书，但我给你的练习是：将其作为对自己的一个挑战，努力保持临在。这非常困难，让你的思绪不飘向过去或未来是很大的挑战。

尝试在冥想时保持临在一分钟、五分钟等。而我认为更有用的真正挑战，是尝试在日常生活中保持临在。关键在于，如果你发现自己走神了，比如在做白日梦，只需觉察到“我刚才走神了”。不要责备自己，这与我们的目标背道而驰。你应积极对待自己，只需提醒自己，并开始培养尽可能活在当下的习惯。这对你完成任务、提高效率将具有巨大的力量，同时也会让你更具魅力。

许多人在社交和谈话中缺乏魅力的原因，正是因为他们忙于担心五分钟前说过的话，或思考五分钟后要说什么，而没有关注当下正在发生的事。具有讽刺意味的是，你能说出的最精彩的话，往往是对谈话对象在那一刻所说内容的自然回应。因此，与其思考要说什么或说过什么，不如专注倾听，然后让你要说的话从当下的临在中自然流露。

## 总结 📝

本节课中，我们一起学习了“生活在现实中”的重要性。我们认识到，使用“应该”一词是与现实对抗的表现，并通过“反转‘应该’”的练习来学习接纳现实。接着，我们探讨了活在当下的价值，通过《当下的力量》一书引入了“现在”是唯一存在的理念，并提供了保持临在的练习方法。摒弃对过去的懊悔与对未来的焦虑，专注于当下，不仅能提升效率，还能增强个人魅力。记住，改变始于接纳现实，并在此基础上有意识地行动。
