## 理解“长定拉力”的关键点

首先，需要理解几个关键概念。基于之前提到的法则，以下是长定拉力的核心要点。

### 夜晚的时间划分与目标

以下是关于夜晚不同阶段互动策略的要点：

1.  **夜晚前半段（通常到凌晨12:30）**：这个阶段，女孩的目标通常不是寻找性伴侣。她们精心打扮出门，是为了享受一个有趣的夜晚，性通常是夜晚高潮的结尾。因此，在这个阶段试图带走女孩几乎不可能。
2.  **互动时长的自然规律**：没有女孩愿意整晚只和同一个男人聊天。即使在前半段互动良好，到了一定时间点，互动也会自然冷却。这不是你做错了什么，而是因为她有其他的社交目标。如果你试图强行留住她，反而会变成她达成目标的障碍，从而失去吸引力。
3.  **夜晚后半段（凌晨12:30之后）**：此时，女孩们已经和不少人交流过，开始更倾向于寻找可以深入发展的对象。这是你需要“定下来”，与一个女孩进行长时间互动（45分钟到1.5小时）并引导向性的阶段。

### 时长是基本要求

在游戏技巧方面，时长是一个基本要求。如果你的互动平均时长只有5到10分钟，那么无论你的技巧多么出色（即使达到**10分/10分**），也几乎不会有持续的结果。因为时长未达标。

你的首要焦点应该是达到**45分钟**或**1.5小时**的互动时长。只有达到这个基准线后，如果还没有进展，再去考虑提升互动技巧。

### 承诺与耐力

在凌晨12:30左右，你需要选定一个女孩并坚持下去。你可能会遇到“选择困难”，觉得其他女孩更好，但关键在于“承诺”。选对一个或选错一个，都可能没有结果，但如果不做承诺，则肯定没有结果。

这里的技能在于培养“耐力”的肌肉。很多成功带走女孩的案例都依赖于耐力——将互动维持45分钟到1.5小时。这很困难，因为我们习惯了短互动。

反直觉的是，**忍受良好反应比忍受糟糕反应更难**。当互动顺利时，我们可能会恐慌、不知所措，甚至无理由地主动退出。这是一种“成功障碍”，你害怕事情真的变好。因此，需要锻炼的肌肉是学会与良好的反应共处，并“忍受”互动的顺利进行。

接下来，我们将分解一个长互动实例。这是一个实际持续了**1小时10分钟**的互动，被剪辑到**24分40秒**。如果完整播放可能会显得冗长，但本课的重点在于，你需要开始以此为目标。如果你觉得这个视频很长，那正是一个需要修正的重要信号——说明你过于关注短互动了。

在这个互动中，我的策略是“更多相同的内容”。我的游戏方式是一系列小循环的重复：一点乐趣、一点推拉、一点肢体接触（靠近、远离）、建立舒适感。我就像锻炼肌肉一样，持续这个循环**45分钟**到**1.5小时**，然后引导离开。

这是最理想的带走情况。大多数成功结果来源于此类互动，而不是在夜晚刚开始或即将结束的混乱阶段。这是可预测、可控的方法。

## 核心要点与进阶技巧

上一节我们分解了长互动的过程，本节中我们来看看其中的核心心法和可能遇到的挑战。

### 氛围与自我感受

另一个关键是，你必须达到一个“性成为可能”的地点，比如你的公寓或她的公寓。如果你不知道如何引导她去那里或和她一起去那里，那么什么也不会发生。

这应该让你意识到：**我需要做更长时间的互动**。如果你没有达到45分钟的基准线，这应该是你的第一要务。下次外出时，你的挑战就是：今晚，完成一个长互动。看看会发生什么。

当你有了基础，感觉良好时，一个常见问题是如何激励自己在一个互动中停留那么久。你会想：“她喜欢我，我知道……好吧，我喜欢那样。”首先，要认识到这背后的荒谬性。

这源于你对自己的看法过低。潜意识里，你会想：“她喜欢我？她有什么问题？为什么她会喜欢我？”你想要她喜欢你，但一旦她喜欢了，你又觉得奇怪。所以，你总是陷入一种两难境地。

你必须经历一个过程，戒掉对认可的依赖，重新让自己习惯于那种“给予”的氛围。一个挑战是：和一个伙伴外出，给他1000美元，并告诉他今晚不要号码，不要带走女孩。你消除了所有可以“获取”的东西。你仍然必须外出，但现在没有东西需要完成。无论你做什么，都不会“得到”女孩。你的动机必须来自不同的地方。

这就是每个人必须发现的：当我外出时，不是为了得到女孩，而是做我真正想做的事。我们太习惯于看着女孩，想着“女孩喜欢什么？我能做什么来得到女孩？”，以至于从未花时间问自己：“我喜欢什么？我能做什么让自己感觉良好？”

你可能会遇到空白，但那就是游戏自己。你需要弄清楚。如果你不知道，就坚持多这样做几个晚上，直到最终领悟。

### 关于“感觉良好”的界定

一个细微的问题是：如果你非常沉浸在“此刻玩乐”中，她也被吸引了，但你是否有感觉到回馈的吸引力？当我说“乐趣”时，想得更宽泛一些：**感觉良好**。乐趣是其中一部分，但你的目标是任何让你感觉良好的事情——乐趣、无忧无虑、激情、被吸引。

这甚至应该是你评判夜晚成功与否的唯一标准。你的唯一标准就是：**你感觉有多好**？如果你持续感觉越来越好，只要这涉及到女性，你就在正确的轨道上。

区别在于：哦，如果她在那里，很酷。但如果她离开，不再有那种低落感。你已经感觉很棒了。什么会让你感觉更好？与一个女孩分享。什么会让你们俩都感觉更好？一起过夜。

### 如何培养“不在乎结果”的心态

如何培养那种不担心结果、非常踏实的心态？我会给出理想方法和实践方法。

*   **理想情况**：你意识到你本就足够，然后你到达那里，根本不在乎。
*   **实践方法**：找到一种方法，在你认为她能给你的结果上，实现“没有她”也能达到。这就是做法。

例如，你想要她感觉良好。这就是为什么第三条法则是“让自己感觉良好”。**我如何能在没有她的情况下让自己感觉良好？** 因为如果你想要她感觉良好，并且你知道如何让自己感觉良好，你就不需要她来感觉良好。现在，你处于富足状态。

这是一个思维上的真相。你想要女孩感觉良好，所以你依附于此。假设是：我感觉不好，我需要她来让我感觉好。如果得不到她，我就感觉不好。所以你来自稀缺和索取。

那么，实际问题就变成了：我如何能来自富足，从而不需要她来让我感觉好？**在没有她的情况下让自己感觉好**。一旦你达到感觉良好的状态，就不再需要女孩来感觉良好，因为你已经在那里了。然后你就可以以富足的心态与她交谈。

这就是为什么第三条法则是：我如何获得乐趣？我如何让自己更无忧无虑？我如何更兴奋、更有激情？我如何对自己做这些，而不是如何让女孩把这些给我。

大多数男人走向女孩，然后她的反应回来——好或坏。而这是走向自己，对自己好，然后传递给女孩。

### 夜晚第三阶段：深夜游戏

最后，我们来谈谈夜晚的第三阶段，因为这仍然是其中的一部分。

*   **第一部分**：获取电话号码，了解情况。
*   **第二部分（凌晨12:30开始）**：是时候做出承诺了。
*   **第三部分**：如果你承诺的对象失败了怎么办？现在你进入了第三部分。有时男人在第三部分放弃。这也是看待夜晚的一种不同方式。

大多数男人认为夜晚的开始比第二部分更重要。**第二部分才是游戏开始的地方**。直到凌晨12:30的第一部分，那是练习。游戏甚至还没有开始。我宁愿你们在前半部分做得差，也不要在后半部分做得差，因为无论你在前半部分做什么，什么都不会发生。

在第二部分进行游戏。如果你承诺的互动失败了，现在你进入第三部分，这时俱乐部快要关门了，人们都在街上。购买欲望很高，这对你有利。但人们也要回家了。你现在没有足够的时间花45分钟和一个女孩聊天，因为没有女孩会站在街上和你聊一个小时。

所以你必须考虑：开场，然后要么带走，要么跟她走。但你必须和女孩去某个地方，你最好知道要去哪里。可能是你的地方，可能是她们回家时去她们的地方，也可能是去吃点东西。要有一个去处。

还要意识到，在夜晚的第三部分，你的很多结果将来自“跟她走”，而不是“带走她”。例如，陪她走回家，陪她回家。
