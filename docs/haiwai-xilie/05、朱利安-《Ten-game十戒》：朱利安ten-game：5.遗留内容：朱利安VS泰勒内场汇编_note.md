# 朱利安《Ten Game十戒》：05：遗留内容 - 朱利安VS泰勒内场汇编 🎬

在本节课中，我们将学习一种名为“偷女孩”的互动游戏。这个游戏的核心并非字面意义上的恶意争夺，而是一种用于打破社交焦虑、训练心态和提升互动技巧的练习。我们将通过分析朱利安与搭档欧文（Owen）的实际视频片段，来理解其运作原理和训练价值。

## 游戏概述与核心理念 🎯

这是一种我们过去常玩的游戏。其核心要点是我们会互相“偷走”对方正在交谈的女孩。

观看这些视频时，你可能会喜欢它，觉得有趣，并受到启发去尝试。我的建议是：如果你受到启发想尝试，必须确保你的搭档知情并同意。否则，你会疏远每一个搭档。过去有些家伙跑来抱怨：“兄弟，我做了那个偷女孩的事，现在没人愿意和我一起出去了。”这毫不奇怪。

需要记住的是，当我们互相“偷女孩”时，我们双方都清楚规则，都是为了乐趣而进行，不涉及自尊心。我们主要在夜晚的前半段进行这个游戏。如果你已经在一个女孩身上花了30分钟，那就是你的目标，你不能去偷搭档的女孩。但在夜晚刚开始时，大家都没有明确目标，做什么都无所谓，反正你也不会和那个女孩发生什么。

## 游戏的核心价值 💡

通过这个游戏，你不仅能开始享受过程本身，而不是专注于女孩或寻求认可，还能摧毁你的自我。**真正能吸引女孩的人，是自我卷入最少的人**。你会发现，每当你的自我卷入时，你就会失去女孩。

这也是一个很好的测试方式：如果你因为搭档“偷”了你的女孩而生气，那说明他能“偷”走，意味着你从一开始就没有真正“拥有”她。

此外，这个游戏迫使你学会保持镇定。**反应最少的人总是能得到女孩**。

有时当你的搭档介入时，你可能会僵住，陷入自己的头脑中，要么被推入“旁观者模式”。或者，当女孩的男性朋友加入组合时，你也会僵住。我常看到这种情况：一个家伙正在和女孩聊天，突然她的男性朋友出现，他就只会说“哦，嘿”，然后完全僵住。这个游戏能训练你应对这种情况。

总的来说，这是一种有趣的方式，能让你开启夜晚的社交，并摆脱自我设限。

## 实战片段解析 📽️

以下是游戏中的一些关键片段，展示了心态和框架的博弈。

### 片段一：突然亲吻 😘

这是欧文。他正在和一个女孩聊天。以下是一种让你摆脱头脑思考的方法。

我走过去说：“你想和我在帝国大厦楼顶见面吗？”然后，我直接亲吻了她。

这几乎可以称为“即时亲吻”。当这种情况发生时，请真正想象一下那个场景。从他的视角来看，这能让你理解如何避免被推入自己的头脑中：你正在和一个女孩聊天，可能已经聊了10分钟，完成了全套吸引流程。你正在说：“是啊，你想在屋顶见面吗，等等等等。”突然，一个家伙走过来，抓住她，当着你的面亲吻了她。

一方面，你会想：“靠，这说明我本来可能可以亲到她的。”另一方面，你会想：“搞什么鬼？”这会震惊到你，因为它太出乎意料了。再者，你会想：“妈的，我刚刚浪费了10分钟，因为我才不要吃别人的剩饭。”

你被推入了自己的头脑中，你能看到这一点。你可以看到反应的程度。你会在他的反应中看到。在接下来的其他视频中，你也会在我身上看到。注意他是如何反应的：他看向摄像机，笑了起来。那是真心的笑吗？不，那是一种“好吧，好吧，你看到那个反应了”的笑。我还会捶他的胸口，说“抓到你了”之类的。这让他有点…我的口红都沾上去了。

### 片段二：设定互动节奏 🎵

下一个片段，我正在和一个女孩聊天，欧文走了过来。我看到他过来，这是我的手，试图推开他的头。他躲开了，跳了进来。

你可能听我谈论过“设定互动的节奏”。你们听说过吗？就是说，谁主动，谁就在设定节奏；谁被动，谁就在反应。他跳进来，设定了节奏，把我推入了我的头脑。观察一下。

“哦，你好，我叫欧文，很高兴认识你。我是个法裔加拿大人，我是个混蛋，我是个讨厌鬼，我选了你，妈的，我还是个蠢货，你还想知道什么？不错的围巾，我讨厌它，现在我又爱它了。”这就是在设定节奏。就像在问“你还有什么想说的？”当你和女孩聊天时，有时你会说“是啊，怎么了？”，然后你就僵住了，只会说“是啊，就像…你要怎么做？”你知道，你会做出一些面部表情。

“你让我想起…让我想起我在货车上看到的一个女孩，她穿得很紧身。太棒了。你是谁？告诉我。”现在连我都有点环顾四周，心想“靠，你知道吗，你长得像我来自斯特普的前女友，她有点…你也是加拿大人吗？你叫什么名字？”

这其实不需要有逻辑意义。当你介入时，**含糊地说一些话，比说逻辑通顺的话更好**。你可以说：“不错的衬衫，这个，手表，妈的，裤子，生命，死亡，嘿”，类似这样。这样更好，因为你仍然处于“因”的位置。你也可以不说话就做到这一点，做一个“沉默开场白”，即你不做反应，让她对你做出反应。比如你只是看着她，然后她会说“什么？”，现在她就在对你做出反应。这就像：谁是“因”，谁是“果”？谁在主导，谁在反应？

“别跟他说话。”现在即使在这里，我也在说“别跟他说话”，这本身就是在对他做出反应，把价值放在了他身上。“不。不。好好。惠特尼，惠特尼。爱，别理那该死的齐。看。”

### 片段三：背后突袭 👻

这个片段我很喜欢。这实际上是在芝加哥，他甚至不知道我在这个城市，更别说在同一个俱乐部了。他正在和这个女孩聊天，然后我从背后突然介入。

我在这里，从一秒到下一秒，他仍然没看到我，女孩已经转向我了。你会看到，他花了一秒钟才意识到发生了什么，然后反应过来，骂了句“混蛋”，并立即做出了反应。“混蛋。”所以即使从那里开始，“混蛋”，现在无论他做什么，都本身会是一种反应吗？

“去吧，小姐。别这样。当。我在后面。报告。是的。他是个虫子，他是个讨厌鬼。我恨他，他总是，我恨他。但是看，你有非常可爱的面部表情。你喝这个穿得好汤。是的，就跳会儿舞。你知道我今天看到什么了吗？我看到一个小孩，对吧？”

所以就像这个家伙在示意，当你做出反应时，那就相当于你被推入了自己的头脑中，或者叫“旁观者模式”，随便你怎么称呼，你就是僵住了。所以欧文在这里处于那种状态，此刻你能想到的只有像2002或2003年的一个老惯例，关于一个戴着黑色头巾的小孩之类的，所以他只能求助于那个。

“他戴着像…他戴着黑色头巾。混蛋。你知道，我们有输有赢，兄弟，我们互相整得很惨。我们不认识彼此。不，不，别这样。哇。你好。”他走进来，设定了节奏。我的反应是什么？没有那些狗屎。我不接受。“不，不，别因为他更高更帅就无视我，我有个…”现在我被推入了自己的头脑，我能想到的只有她会说西班牙语，这是洛杉矶一家叫…的快餐店吗？所以我说的就是，在某个时刻，我会说：“哦，是的，我很尴尬，它会看起来…”

### 片段四：无声干扰 🤫

这个我很喜欢。我已经在这个场地待了一会儿了，我正处在状态中。欧文刚到，就像刚开始的几个组合之一，你知道，就是让互动自然发生。

我在他后面，我只是戳了他的肚子，这就是我做的全部，我他妈一个字都没说。发生了什么？“不，朱利安。我们竞争很激烈，要么在那里。不知道，别缠着他，别缠着他。医疗的。”

如果你去俱乐部，和女孩聊天，你必须开始说话，否则你会卡在自己的头脑里。你会变得很奇怪，所以你碰巧开始说话。“你对我太好了，你对我太好了。不，混蛋，不，不，他讨厌朱利安比我先进入状态。就带走她们，就带走她们。我你。”

### 片段五：说唱对决 🎤

这个片段简直像他妈的说唱对决。如果你们看过埃米纳姆的《8英里》。就像我们站在她两边，互相较量，而那个比对方高出那么1%的人得到了女孩。

“哦，我的天哪，那是最尴尬的事。真好啊。は。我。Z 是个男的 Z 是个男的，看，我来了。跟我来。跟我来。我们爱你。是的，去他妈的家伙。是的，太多热狗了，太多他妈的热狗在这屎里了。让那家伙滚出去，让他他妈滚出去，我的天…”

“你们，好吧，所以承诺并不真的想亲女孩的卧室，给你，给你。那太糟糕了。你看到我发生什么了吗？不，发生了什么？他很糟糕。我情感上受到了创伤和被骚扰。哦，我们做得不好。我是个极具攻击性的人。”

所以，你可以再次看到那里的动态。这也是很重要的一点：**每当可能有其他男人出现或类似情况时，真正关键的是谁反应最少**。所以，即使你…你知道，比如有个“阿尔法男”或者你想称之为的其他家伙，她不认识的人试图得到她，不要试图过度表现，因为那样你反应更多，她反而会走向他。即使你正在和她的女孩聊天，突然某个家伙出现，大多数男人会“哦，天哪”，立即反应，那你就完了。

所以理想情况是，如果一个男人进入组合，我通常会先保持冷静，甚至10秒钟，让他完全接管，然后主动地，我会说：“总之…”，进行带领，试图把他置于对我的反应框架中。**这是一场关于谁反应更多的战斗**。这很微妙，但这就是它的运作方式。

## 总结 📝

本节课中，我们一起学习了“偷女孩”游戏作为一种心态训练工具。其核心在于通过刻意制造社交压力，来**摧毁自我、训练镇定、学习设定互动节奏**。关键要点是：**反应最少的人赢得互动**，过度反应或试图逻辑辩解会让你陷入被动。这个游戏强调过程乐趣而非结果，适合在社交初期与知情搭档共同练习，以打破僵局、摆脱“旁观者模式”。记住，真正的吸引力来自于低自我卷入和主动引领框架的能力。
