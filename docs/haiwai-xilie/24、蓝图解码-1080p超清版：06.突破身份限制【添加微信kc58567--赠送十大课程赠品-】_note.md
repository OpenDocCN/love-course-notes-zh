# 蓝图解码 1080p超清版：06.突破身份限制

在本节课中，我们将要学习一个核心概念：**身份**。我们将探讨身份如何塑造你的社交行为，以及如何突破身份的限制，成为你想成为的人。理解这一点，是提升社交技能和吸引力的关键。

## 身份是什么？

上一节我们讨论了社交反馈如何影响你的现实。本节中，我们来看看**身份**这个概念。

首先，需要明确的是，不同的人对“身份”有不同的定义。我在这里使用的定义，是我用来与你们沟通的工作定义。任何概念都只是引导你进入某种状态的“路标”，重要的是这些原则最终将你带往何处。

那么，我认为身份是什么？身份是一个将你与你的社交环境联系起来，同时也将你从中分离出来的概念。它是你对自己是谁的概念，它定义了：
*   你与他人的不同之处。
*   你在人群中的位置。
*   什么使你成为你。
*   相对于他人，你拥有何种地位。

你可以想象自己在一个图腾柱上，可能处于顶端、中间或底部。这取决于你的自我概念。我们总是通过脑海中这个叫做“身份”的种子来感知世界。我们感知到什么、如何感知、如何看待他人、情绪如何反应，都受其影响。

## 身份的双重作用

身份既能帮助你，也能限制你。

**身份如何帮助你？**
当你清楚“我是谁”时，它能给你力量和清晰的个人边界。例如，身处逆境的人可能会说：“我知道我是谁，这就够了。”

**身份如何限制你？**
同时，身份也会成为你最大的枷锁。从社交角度看，你的潜力几乎是无限的。真正阻止你的，是你对自己身份的信念。我们每个人心中都有一个“酷家伙”、“受欢迎的人”或“在女性面前成功的人”的形象，但我们常常不给自己成为那个人的**许可**。

一个常见的限制性想法是：“那不是真正的我。” 无论是尝试新的个性特质，还是学习新的社交技能，这个想法都会跳出来阻止你。在引导学员时，这是最常需要处理的问题之一。

## 身份的任意性

有趣的是，你的身份和个性在很大程度上是**完全任意**的。它本可以朝另一个方向发展。

你的很多个性特质是在**反应**中形成的，源于你成长过程中收到的**社交反馈**。例如：
*   如果你尝试自信并得到了积极反馈，你可能会发展出自信的特质。
*   如果你尝试自信却遭到打压（比如被欺负），你可能会形成负面的联想，走向相反的方向。

你可能会想：“这就是我，我无法改变，我无法成为我想成为的人。” 但事实是，构成你核心本质的部分从未改变，而许多围绕它的个性特质是可以演变的。

**关键在于：你可以成为你想成为的人。** 你的核心保持不变，但你的风格、个性、观点都可以改变。不要被“这不是我”的想法束缚。

## 你应得什么？

在你的身份概念中，包含着你认为自己**应得**什么的想法，尤其是在女性方面。

想一想：你认为自己在图腾柱的什么位置？你真正觉得自己应得什么样的女性？

当你觉得自己和对方“在同一水平”时，一切都会变得自然。你行为从容，容易产生吸引力。相反，当你觉得自己“低人一等”时，你的行为会在细微处发生变化，例如：
*   口吃或紧张。
*   寻求认可。
*   声音音调升高。
*   在脑海中拼命思考下一句该说什么，试图推进对话或给她留下印象。

这些细微的**讨好行为**，即便很隐蔽，也是在放弃你的力量。区分“取悦他人的表演者”和真正具有性吸引力的“成功者”的核心因素，往往就在于他是否清楚“自己是谁”。

## 社交反馈与雷达系统

这让我们回到昨天讨论的概念：**社交反馈**。你的大脑有一个雷达系统，总是在“探测”，以弄清楚：
*   你是谁。
*   在当前环境中，哪些行为是被允许的。
*   什么是正常的。

你并非独自弄明白世界如何运作，你依赖二手信息。这个雷达系统会观察他人，判断：“这个人的现实感比我的更准确吗？” 如果某人显得非常确定，你的大脑可能会倾向于相信他。

因此，当有人（比如学校里更受欢迎的孩子）非常确定地贬低你时，会对你施加一种心理压力，让你认为自己就是他所说的那种人。许多愚蠢的人仅仅因为非常自信，就能施加这种影响。但你不必屈服于这种压力。

## 印记：你心中的模板

接下来是一个重要的概念：**印记**。

每个人心中都有一个概念（一个模板或粗略草图），关于：
*   一个有价值的、很酷的人是什么样子。
*   一个没有价值、不酷的人是什么样子。

学习过社交动力学的人，可以清晰地描述这个模板（眼神接触、放松的肢体语言等）。即使没学过的人，也有一个无意识的版本，并会对它做出反应。

你的大脑总是在根据当前情境，参照这些印记来决定你应该如何行动。你的情境不断变化，大脑则不断查看社交反馈，以判断你拥有多少价值，并决定此刻哪种个性最合适。

这解释了为什么你有时在一个场合可以很酷、很有魅力，在另一个重要场合却无法做到。你的雷达系统被打乱了，它只允许你展现出它认为在当前价值水平下“合适”的个性。

## 为什么大脑要限制我们？

这就引出一个问题：为什么大脑只允许我们展现出它认为别人会喜欢的部分？这似乎适得其反。

在文明社会，暴力威胁已大大降低。你通常可以尝试新行为而不会受到身体伤害。主要的“威胁”其实是**出丑**，即显得愚蠢或尴尬。

任何新手在尝试新行为时，都会看起来像个傻瓜，因为尚未校准。如果你害怕这个，你的个性范围就永远不会扩大。

**掌握与女性相处技巧的一个非常重要的部分，就是能够反复承受羞辱。** 有趣的是，许多男人为了维护“男子气概”的身份，愿意打架甚至受伤，但却极度害怕让自己出丑，因为“看起来像个傻瓜”不是他们身份的一部分。

## 突破限制

那么，如何突破身份的限制？

1.  **认识到身份的任意性**：明白你的很多个性是后天形成且可以改变的。
2.  **挑战“那不是真正的我”**：主动尝试你想成为的那种人的行为、着装和思维方式。
3.  **承受不适**：愿意在学习和校准新技能时，经历看起来笨拙或尴尬的阶段。
4.  **选择能挑战你的朋友**：结交与你观点不同、能拓展你现实感的朋友，而不是只认同你现有观念的人。
5.  **像对待他人一样对待自己**：就像在培训中，教练会以100%的确定性将你视为一个充满魅力的人，从而对你施加积极的心理压力一样，你也可以对自己这样做。

记住，你的核心本质是恒定的，但围绕它的一切——你的表达、风格、能力——都可以无限拓展。你有权成为你想成为的任何人。
