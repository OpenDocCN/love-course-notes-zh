# 言语游戏：09：游戏时代：3.自然流时代

在本节课中，我们将探讨游戏发展历程中的第三个主要时代——自然流时代。我们将了解它为何出现、其核心特点、相较于前代的改进之处，以及它自身存在的显著问题与局限性。

## 概述：自然流时代的兴起

自然流时代是继罐头游戏时代之后出现的一个主要阶段。

为了理解这个时代，你首先需要明白它为何会出现。

它的出现并非因为人们最初认为它实际上更好。

它之所以出现并变得突出，很大程度上与市场营销有关，与创造一种让大众易于接受、并且可能不被视为冒犯或政治不正确的东西有关。

## 自然流时代的背景与动因

上一节我们介绍了罐头游戏时代，本节中我们来看看它的演变。自然流游戏在《The Game》这本书出版后开始盛行。

这本书在某种程度上让人们注意到了许多罐头游戏策略。

从政治正确的视角来看，这些策略可能被视为具有操纵性，或者可以被置于负面解读之下。

考虑到这一点，一种自然风格的游戏理念开始流行，其核心是“做最好的自己”。

只需展现最自信的自己。这种态度似乎更易于被接受，更易于推销，也更适合公开宣扬。

同时，当时许多实践者在使用罐头游戏时感到不适。

他们觉得自己没有在做真实的自己。他们感觉像是在欺骗女孩。

因此，对于研究此道的受众来说，自然流也成了一种更易于接受的方案。

但最主要的原因，再次强调，是出于市场营销目的，因为它是一种更容易被包装和接受的概念，能让人们相信这是我们希望成真的事情。

你当然希望“做自己就能成功”是真的。而这，再次几乎让人回想起无游戏时代。

## 核心特点：与无游戏时代的对比

自然流时代实际上与无游戏时代非常相似。

但它对无游戏时代做了一些改进。

以下是自然流时代相较于无游戏时代的主要改进：

*   **继承了罐头游戏时代的发现**：它恰好出现在罐头游戏时代之后，因此罐头游戏时代发现的某些概念得以延续到自然流时代。例如，“成为奖品”或视自己为奖品的框架和理念，在罐头游戏时代被积极传播，这个理念在自然流时代得以保留。所以，你现在拥有的是“无游戏时代”加上“我是奖品”。同样，在罐头游戏时代发展出的某些升级技巧也被带入了自然流时代。因此，我们得到的是“无游戏时代”加上“我现在知道如何升级关系”或“我现在有了一些理论依据”。

你可以将其视为一个略微改进的无游戏时代。

## 自然流游戏的主要问题

但是，自然风格游戏存在巨大的问题，其中许多与无游戏风格的问题相同。

以下是自然流游戏的两个主要问题：

*   **再次面临无话可说的困境**：在无游戏时代，无话可说是一个大问题。在罐头游戏时代，至少在开场初期，这不是问题，因为你有惯例和话术。但在自然流游戏中，突然间，“我的开场白是什么？我该说什么？如何让对话超过最初的30秒？”这又成了问题，回到了起点。
*   **缺乏推进的序列**：如果你只是去“做自己”或展现自信版的自己，那么偶尔在时机恰好时，可能会得到女孩的良好反应。但如果你只是重复做同样的事情，你并没有推动互动向前发展。例如，如果你只是戏弄女孩、给她制造难题，这可能会让她对你产生吸引力，但这不会让你与她建立情感连接，也不会让你与她发生肢体接触或达成关系。因此，重复做那些最初奏效的事情，在后续阶段并不正确。

让我给你一个比喻。有人曾告诉我，创造一个糟糕喜剧演员的最好方法，就是让一个人上台，没有剧本，没有准备好的素材。

因为会发生的是，这个人会很紧张，或者他会追求第一个快速、廉价的笑点。所以他可能会制造一个滑稽的场面，比如在观众中模仿放屁的声音，然后这奏效了，他就会做类似的事情，最终形成一个全是放屁声的喜剧表演，这可能是你见过最糟糕的表演。或者他可能会在舞台上滑倒，或者做一些奇怪的事情来获得廉价的笑声，并且他永远无法超越这一点，因为他认为这就是喜剧的全部。

自然流游戏开始变成这样。它开始变成人们做一些出格、荒谬、愚蠢的事情，因为这能得到反应，并且认为这就是游戏的全部。

于是你看到很多这样的人：“哦，我能引起女孩的反应，但我从未真正让她们喜欢我。”或者“我已经到了可以随心所欲说话、并且充满自信的地步”，但他们不谈自己的实际成果，因为成果并不存在。

因此，这种**缺乏结构**、**缺乏推进**的特性，在自然流时代确实造成了很大伤害。

## 自然流时代的积极面与“状态”概念

自然流时代好的一面是，实践者确实感觉更真实一致，也感觉更自由。

另一个在某种意义上好坏参半的发现，是“状态”这个概念。

“状态”基本上是指，当你处于心流中，当你重复做某事足够多次或完全投入某事时，你自身的一些自然魅力就会显现出来。

事实上，很多实践自然流游戏的人都在追逐状态，他们想：“我只需要进入状态，然后我就能进行游戏。”这是一种可悲的游戏方式，因为假设你走出前门，看到一个美女，而你不在状态。你该怎么办？难道要先和六个邻居聊天来进入状态，然后再去和她说话吗？这在实践层面行不通，在很多实际场景中都有问题。

此外，我们发现，如果你试图追逐状态，试图让自己进入好心情、进入心流，总在思考自己是否处于心流中，这恰恰是**最糟糕**的进入心流的方式。这是让你陷入自我意识、脱离心流的最佳方法。所以，虽然“状态”这个概念确实存在，虽然确实存在心流状态和最佳表现水平，但追逐它或依赖它实际上是达到它的最差途径。

这造就了一批“状态依赖者”。想象一下两个情况：第一，你不知道对女孩说什么；第二，你觉得必须处于状态才能和她说话。你又会开始产生强烈的接近焦虑，开始出现所有在无游戏时代发生的问题，甚至可能更糟，因为现在你为自己设定了更高的游戏标准。

## 自然流时代的升级与结果

自然流时代真正好的一点，是它确实让升级变得更顺畅，因为实践者更真实一致。如果你能公开告诉女孩你喜欢她，并且你已经学会了诸如升级序列之类的东西。

例如，“进两步退一步”，或者先进行小幅度的升级试探，如果她反应良好，再进行更大程度的升级。

这至少意味着，当一个女孩喜欢他时，这样的男人能够推动互动向前发展并取得结果。

所以，一个无游戏的男人常常会有女孩喜欢他，但他不知道如何在肢体上推进，不知道如何升级关系。

而一个实践自然流游戏的男人，如果有一个女孩真的喜欢他，他至少知道如何在肢体上推进，或者至少对如何升级有一些概念。虽然他未必能做得非常流畅或精准，但至少他知道前进的方向。这意味着他至少能取得一些结果。

## 自然流时代的成果与局限

当一群人在游戏中表现得随意且随机，他们知道如何与喜欢他们的女孩取得结果，但不知道如何很好地开启对话，也不知道如何让不喜欢他们的女孩喜欢上他们时，会发生什么？

最终结果是，许多质量平平的女孩被推进了肢体接触阶段，而这正是男人们最终发生关系和建立关系的女孩类型。

因此，从罐头游戏转向自然流游戏，**性行为数量**确实可能有所增加。

但所涉及女孩的**吸引力**却明显下降。

同时，感觉男人非常具有压迫性、过于直接，并因此感到被冒犯和厌烦的女孩数量也大幅增加。

## 总结：自然流时代的利弊

本节课中，我们一起学习了自然流时代的兴起、特点与影响。

以下是自然流游戏的利弊总结：

**积极方面**：
1.  它让我们理解了“状态”概念，尽管其在自然流游戏中的实用性非常有限。
2.  它让升级变得更容易。
3.  它让实践者感觉更真实一致，可能让某种类型的男人觉得游戏更有趣、感觉更真实。
4.  它创造了一种男人更容易接受的东西，并且让游戏“洗白”，因为它更具市场性，从而吸引了更多人加入社群分享想法。

**消极方面**：
1.  再次面临无话可说和缺乏互动结构的问题。
2.  过度依赖难以捉摸的“状态”，反而加剧焦虑。
3.  在追求最顶尖美女方面，效果极不稳定，最多只能说是时好时坏。
4.  可能导致互动质量下降，吸引到的女性平均吸引力降低，并可能引起部分女性的反感。

自然流时代是游戏发展中的一个重要阶段，它强调真实性与一致性，但因其缺乏系统结构和可重复的方法论，在效果上存在显著天花板。这为后续更结构化、更系统化的游戏方法的发展埋下了伏笔。
