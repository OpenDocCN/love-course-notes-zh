# 老板 BOSS：11：高地位肢体语言

在本节课中，我们将要学习什么是肢体语言，它如何在社交互动中帮助你，以及高地位肢体语言的具体表现和内在感受。我们还将探讨其优势，并学习如何将其融入你的日常行为中。

## 什么是肢体语言

肢体语言是女孩通过观察你的身体来了解你的能力。她看着你，看到你的身体姿态，并从中获取关于你的信息。这是一个她在瞬间做出的判断。她并不认识你，但会基于此做出决定。

我们的大量交流是非言语的。据说80%的沟通是非言语的。很多搭讪技巧也是情感层面的，而非言语层面。当你看到我完全投入、不加评判地进行搭讪时，你可以看出我的身体非常放松。这种状态会让女孩产生兴趣，因为她通过观察我控制身体的方式，能够感知我内心的感受和想法。

对她而言，我的具体姿势并不重要。重要的是，我的肢体语言像一扇窗户，让她得以窥见我的内心和情感。她想知道我是谁。正如我常说的，我采用的是基于性格的搭讪方式。

**基于性格的搭讪 = 你的性格 + 你的想法 + 你的感受 + 你的关注点**

正是这些因素，而非你说的话，决定了你搭讪和社交互动的质量。

## 肢体语言为何有效

肢体语言之所以如此有效，是因为它很难伪装。要做到零评判、没有伪装的人格，你必须真正地放下心防。为了吸引女孩，你必须真实地做到这一点。

在训练营中，我们有时会进行“无声开场”练习。你需要在20分钟内进行搭讪，但不说任何话。即使没有言语，对方也能从你的肢体语言中了解到你的很多信息。他们能看出你是否风趣、是否认真对待自己、身体是否放松、是否有幽默感。

我曾展示过一个片段：我走向一个女孩，做了个挥舞手臂的滑稽动作，她说：“我甚至不认识你，但你真的很好。我们信任你，喜欢你。”这听起来毫无逻辑，但在瞬间，她通过观察你的肢体语言来理解你是谁。眼睛的交流也同样重要，她也通过你的眼睛来观察你。

## 谁应该学习肢体语言

我认为每个人都应该学习肢体语言。它不仅在你进行搭讪时有用，在自我发展、工作汇报、销售等任何你可能遇到的情况下都非常有用。它适用于你周围的陌生人，也适用于你认识的人，甚至对你自己也很有益。

请理解这个概念：肢体语言不仅是为了让别人觉得你很酷。它也是为了改变你对自己的感受。

每一天的每一刻，你都在向你的大脑发送信号。无论你是否意识到，你的脑海中总有一个声音在说话。这个声音有时会说一些负面的话，比如“我不够酷”、“我的发型不好”、“我不够有趣”。

但你可以让这个声音说积极的话。当你通过肢体语言告诉大脑积极的信息时，这会变得更容易。例如，如果我这样站着，我的背部会承受更多压力，呼吸不顺畅，看着地板，我会感觉不那么积极。这向我的大脑发送了关于我如何看待自己的信号。

相反，如果我站直，身体挺拔，脊柱笔直，重心落在脚后跟，肩膀向后，呼吸更深沉、更自然（用腹部而非胸部呼吸），我对自己的感觉就会不同。我说话的方式会放慢，按照自己的节奏，享受话语说出的过程。这向我的大脑发送了积极的信号。

因此，肢体语言不仅对女孩或工作中的同事重要，让他们认为你聪明、有趣、自信，对你自己也同样重要。你通过肢体语言在每时每刻“编程”你的大脑，告诉自己应该成为什么样的人。你听从自己身体建议的次数最多。

此外，你的身体总是在与你对话。它是一个神奇的工具，总是告诉你什么是对的，什么是错的。例如，如果你注意到自己在紧张或过度思考时会频繁挠胡子，或者有抽搐的小动作，这就是你的身体在与你对话，这是身心连接的表现。

如果你发现每次搭讪时都把手插在口袋里，你的身体在告诉你：“嘿，我有点紧张，我想用这种方式处理这种紧张感。”前几天有学员说：“我很紧张，不知道说什么。”我告诉他，那就走过去说：“我很紧张，不知道说什么。”至少这样，你能与自己的身体及真实感受保持和谐。

所以，不要与你的身体对抗。倾听它在告诉你什么。在俱乐部里，如果你感到紧张、身体紧绷，去感受是肩膀、胸部还是脖子感到紧张，这会告诉你很多关于自己的信息。当我压力很大时（不是指紧张，而是指工作非常努力），我的身体会感到紧绷。这时我会去按摩，放松身体，从而也放松了心灵。身心是和谐统一的。

## 传统肢体语言与高地位肢体语言

接下来，我们谈谈“如何做”。我要讲的不仅仅是常规的肢体语言，因为你们随处可以找到关于肢体语言的书。有一本非常著名的书叫《肢体语言》，我推荐大家阅读。

传统的肢体语言书籍会告诉你一些基础知识。例如，当某人看向左上角或右上角时，可能意味着他们在说谎或说真话。但我不完全相信这个，因为有时我看向那里只是在创造性思考。

传统理论还会说，如果有人搓拇指，可能意味着他们紧张、在思考或不确定。没有哪种肢体语言是100%准确的。书上可能说，当某人微笑但下巴紧绷时，意味着他们并非真正快乐。或者，当某人交叉双臂时，可能意味着他们情感上不开放、不想听你说话、不确定、不相信，或者也可能意味着他们坚强、快乐、自豪。

当某人双手叉腰时，可能意味着骄傲或评判。当某人手插口袋时，可能意味着自信或紧张。这些都是非常细微的差别。传统理论认为，手这样插口袋是紧张，而这样插口袋是放松。但拇指的差异真的那么大吗？不，并没有。

传统理论还说，像这样站着意味着紧张，因为不想占用空间；双脚分开站立意味着更放松、重心分散，对所说或所做之事更有信心；单脚站立意味着非常放松，只享受当下而非关注外表。这些都是非常传统的肢体语言知识，你们可以在网上找到。

但我不想过多关注这些。我想聚焦的是**高地位肢体语言**。

## 肢体语言并非全部

需要明确的是，肢体语言并非万能，它也可能“说谎”。例如，我可以假装紧张地接近某人，表现出糟糕的肢体语言：手插口袋、耸肩、弯腰、低声说话、低头。从传统角度看，这是糟糕的肢体语言。

但在RSD（真实社交动力学）中，我们有一个原则：**真我总会显现**。这意味着，即使你的肢体语言很差，即使你做了一个糟糕的搭讪，只要你的内心是平和的、快乐的，你选择拥有积极的情绪和想法，那么这次搭讪通常也会成功。

这就是为什么在RSD，我们说不要专注于技巧，要专注于概念。可能有人想做“反向击掌”，但做得不好，感觉像是在索取。而当我只是简单地分享一个击掌时，感觉就不同了。关键不在于你做什么，而在于你行动时的心态和意图。

如果你的意图是让对方感觉良好，那就会是好的互动。如果你的意图是享受对话，你的肢体语言自然会好。如果你的意图是不要显得愚蠢，你的肢体语言反而会变差。

因此，除了我将要教你的技巧，我希望你专注于在每时每刻的感受和想法。你可能会说：“麦迪森，你说得对，但我通常感觉糟透了，所以这没帮助。”这时，你需要**选择关注什么**。你必须果断，切断脑海中其他所有想法，专注于你最想拥有的、对你有帮助的想法。你关注什么，什么就会增长。

这并不是说你不会有坏想法或消极想法。你会有。不要评判它们，承认它们的存在，然后专注于对你有帮助的想法。这就像为了学校考试而临时抱佛脚，或者为了健康而抵制冰淇淋的诱惑一样，你需要将所有的意图和精力集中在那件事上。

在搭讪时，我也是这样做的。我非常清楚周围有其他男人在看，她有朋友在身边，她很漂亮，她可能不会和我这样的男人约会。但我选择专注于：“她很漂亮，我想认识她。”直接聚焦于此。

当我外出时，通常有一两个学员在看我，还有一些从网上认出我的人跟着我。在多伦多时，简直像有一群人跟着我。此外，通常还有斯蒂夫或另一个人拿着相机和麦克风对着我，这是价值数千美元的设备压力。而且，网上会有成千上万的人观看并评论。所有这些目光都集中在我身上，但我必须屏蔽掉这些，专注于眼前与我连接的女孩。

我 literally 在 mind fuck 自己。因为我本可以关注所有那些事，但我必须只专注于眼前的女孩。这就是为什么我让你们做眼神接触练习，因为它帮助你只专注于眼前的人，降低防御，变得脆弱，与面前的人建立连接。

当你专注于此时，你会做得更好。

对我来说，高地位肢体语言是阿尔法男性、酷小孩的肢体语言。我最初并没有，我必须学习它。我是怎么学的？很简单：我去俱乐部观察男人。我观察那些受女孩欢迎的男人，看他们如何与其他男人和女孩交流，然后模仿并将其融入我的人格中。

我将从头开始，向下讲解。

### 1. 眼睛与面部

首先，我们来谈谈我的脸。高地位肢体语言或高地位面容，通常那些受女孩欢迎的人，他们的面部很有表现力。我们先说眼睛。

他们的眼睛是清醒、有神的。我称之为“眼中的星光”。人们总是通过看你的眼睛来窥探你的内心世界。

如果我在评判自己，担心别人怎么看我，担心这次搭讪是否顺利，我的眼睛会变成这样。你们看到了吗？我试着在台上做一下：台下的人付了钱来这里，哦，这有点让人紧张……你们看到我如何退缩到自己的脑海里了吗？就像刚才我还与你们同在，但瞬间就隔了一堵墙。

相比之下，如果我想：“嘿，看看这些很酷的家伙，我面前这些很棒的人，他们玩得很开心，看起来很放松，还在做笔记，太棒了。”你们看到我眼中的不同了吗？这几乎就是我所说的“眼中的星光”。你眼中的星光完全来自于你的关注点。无论你关注什么，都会创造或带走这种星光。

所有酷小孩都有这个。如果你看一张婴儿的照片，婴儿是什么样子？如果你走进电梯，里面有个婴儿，婴儿就这样盯着你看五六层楼的时间。你会觉得有点怪，但婴儿只是长时间盯着你。不知为何，你开始感觉良好，因为婴儿没有自我包袱，没有故事，没有内心独白告诉自己“我四个月大的人生一直被拒绝”或“高中时没人喜欢我”。婴儿没有这些包袱，所以眼睛明亮、非常有表现力。

当你自然地走向一个女孩时，她首先看的就是你的眼睛。这就是为什么我让你们做眼神接触练习。她首先看你的眼睛，观察它们多有表现力。如果你的眼睛有表现力，她就能与你连接。你本质上是通过在那一刻选择关注什么，来“欺骗”自己成为一个酷小孩。

选择在那一刻关注什么，会产生巨大的不同。这可能是最重要的事情之一，甚至就是最重要的事。因为就像我之前说的，即使我肢体语言很差，但如果我的眼睛在说“这很有趣”，那么肢体语言就不算差。一切都体现在眼睛里。

所以，要拥有“眼中的星光”，你需要关注女孩有吸引力的事实，关注你只是来分享，关注你想从每次搭讪中学习，不要关注这个女孩是那个能让你人生圆满的人，关注你想在夜晚结束时处于良好状态，关注你正在学习，关注你只是出来行动。

昨晚在俱乐部有个时刻，我心想：“这他妈的就是俱乐部。”然后我意识到：“等等，我就在俱乐部里，就像RSD视频里那样。等等，我就是那个RSD的家伙。”这很疯狂。我开始为此感到兴奋。即使现在在台上，你们也能看到我的眼睛开始亮起来。

所有酷小孩的共同点是他们的心理。让我解释一下酷小孩的心理。有一天，我和前女友去公园，我注意到她和她朋友的朋友们在一起。那些家伙都很酷：六块腹肌，脱了衬衫扔飞盘。一个家伙大声说：“嘿，我有个三人行，太疯狂了。”他在公园里大声告诉女孩们这些事。另一个女孩说：“我爸爸想送我去那里，我不知道。我们之后应该去那儿。是的，我有我爸的信用卡。走吧。”我当时想：“这是个什么样的世界？我没有爸爸，没有爸爸的信用卡，没有六块腹肌，不帅，没有频繁的三人行。”我感到自卑。

然后我意识到，这些孩子之所以酷，是因为生活总是给他们好东西，对吧？“我很强壮，我有杂志上的六块腹肌，因此我可以感觉很酷，因此我的眼睛可以发光，因为世界告诉我我足够好。”那个女孩走在街上会有五个男人搭讪，打开Instagram，她穿着比基尼的屁股照片有500个点赞。整个世界都告诉她她很漂亮、很棒。她走进俱乐部得到认可，在网上得到认可，有爸爸的信用卡可以无限制消费。所以她感觉很好，因为世界告诉她她很棒。她没有那些消极想法。

但作为男人，作为这个房间里的男人或观看视频的男人，你可能没有世界告诉你你很棒的这种奢侈。所以，你必须根据自己的标准告诉自己你很棒。“我很棒，因为我刚刚他妈的进行了搭讪。”“我很棒，因为我给了自己一个机会。”“我很棒，因为我虽然害怕做这个，但我正在冒险踏入未知，冒险变得更好。”“我正在采取的每一步，都让我很棒。”“有很多人甚至不会在这个房间里。你们投资了一个课程，来到这里学习，这让你他妈的很棒。”“你正在变得更好的道路上，你在学习，这让你很棒。”

所以，当你走向女孩时，眼中要有星光。并不是因为女孩会喜欢你，你眼中才有星光。你眼中之所以有星光，是因为你知道自己在做正确的事，你在采取行动，采取正确的行动，采取下一步行动。你眼中之所以有星光，是因为你感觉自己在那条道路上向前迈进，你在取得进步。即使你当晚表现很差（你总会有表现差的夜晚），至少你从那些夜晚中学到了东西。这样你就能放松，拥有那种状态。

接下来，谈谈面部表情。有两种面部表情：常规表情和微表情。

常规表情就像这样，只是一个愚蠢的表情。微表情则是这样。你捕捉到了吗？这是在面部表情之间的小表情。有趣的是，在面部表情中，微表情比实际表情更能传达信息。

朱利安很久以前教过我一个叫“洛杉矶式表情”的东西。常规表情就像你对某人印象深刻，这样：向上看，向下看，微笑。洛杉矶式表情则是这样。我再做一遍。这是在表情之间的小表情，说明了一切。洛杉矶式表情就像：“我看到你了，我并不印象深刻，但我假装我是。”“我看到你了，我不喜欢我所看到的，我感到厌恶，但我假装喜欢你。”因为在洛杉矶，每个人都这样微笑，但他们心里恨死你了。这是洛杉矶。所以他们非常不真诚。我认为这与苏黎世的文化非常不同，这里的女孩不太这样做。

面部表情在你的交流中非常重要，可以成就也可以破坏交流。眼神接触和面部是最重要的。面部表情能传达很多信息。你需要有一张**动态的脸**。

动态的脸意味着你的脸能很好地配合你说的话。我可以这样做，也可以这样做。每次我说的时候，我的脸都意味着不同的东西。女孩会再次阅读我的脸。她总是关注潜沟通。所以，她不是看你说了什么，而是看你话语背后的含义；不是看你做了什么，而是看你行为背后的意义。这才是关键。

动态的脸。我之前展示过无声开场，像这样。脸看起来很有表现力，几乎像金·凯瑞，但没那么夸张。很多男人，取决于你来自的文化，你的脸可能是静态的。如果你来自俄罗斯文化或中东文化，传统是男人要坚强，男人不表露情感。在训练营里，我有这样的学员，这被称为“学习微笑的三天”。我说“微笑”，他们就这样。我们实际上会走上前，像这样帮他们弄脸，他们的脸会疼。他们的微笑在头几天看起来真的很不舒服，看起来像他们有胀气或要拉肚子。微笑对他们来说很难，因为他们还不习惯。这需要练习来适应，因为本质上，他们的心态在控制他们的脸。社会文化告诉他们必须坚强，所以大脑控制了脸。

他的问题是：“我什么时候做这些？”我自始至终都在做。当你带她回家，和她单独在一起时，这也非常重要。通常，我会坐在床角或沙发上，她在这边，我和她聊一会儿，给她一些空间。她离我三四英尺远，我只是和她聊一会儿。因为我想让她知道，我足够舒适，不会在她还没准备好时对她采取行动。我足够舒适，可以不去追求她，即使她说“不”。我在展示我有平衡，我有某种克制。她已经知道我可以吻她。她已经知道我是一个会采取行动的男人，因为在搭讪过程中，我可能已经抱起过她，和她手指交叉，吻过她的脖子。然后我靠回去，给她一些空间。

关于肢体语言，关键在于这是一种“进与出”的游戏。你不是总是在“进”。有时你“进”，有时你“出”。有时你“进”，有时你“出”。你总是在玩弄你和女孩之间的空间。就像你有一个空间是你的，中间有一个你们共享的空间，然后她有一个空间是她的。有时你走进去，有时你退出来，有时你走进去，有时你退出来，你也留出机会让她走进来。明白吗？很好的问题。

另一个问题是：“女人欣赏你给予她全部、不分心的关注。”这绝对正确。她们确实欣赏你能给予全部、不分心的关注。但事实是，她们并不想一直拥有它。她们想要片刻，但她们也想知道她们可能会失去你的关注。她们也想知道她们可能会搞砸。因为如果我给你我100%的关注一小时，我就这样盯着你。我能永远这样盯着你。最终，如果你是一个女孩，你会对自己说：“如果我愿意，我可以和这个男人上床。这对我来说没有挑战，没有乐趣。我知道这会怎么发生。我带他回家，如果我说‘好’，他会说‘是的’。”所以这不再有游戏性了。游戏中的情绪峰值不仅仅是积极情绪，而是积极和消极的完整情绪范围。所以是“进”和“出”。“进”和“出”创造了完整的情绪范围。她需要两者才能感到吸引。她基于你让她感觉如何而被你吸引。这让她感觉良好，5或10分钟。但之后变得无聊，新鲜感消失。就像你拿到新iPhone 6，什么是新iPhone，iPhone 7？有人有iPhone 7吗？你有，大约两三周很酷吧？现在，就像“哦，是的，什么，你昨天就有了？你昨天就拿到了？是的，来了。”所以据我所知，你拥有iPhone 7大约一两周，兴奋感就消失了。他就像：“哦，好吧，我有了。”你刚得到那些鞋时很兴奋，你现在还像第一天那样爱它们吗？第一天，新鲜感消失了。所以这对女孩来说是兴奋的，然后变得无聊，新鲜感消失。你必须保持一点不可预测性，这样她不知道接下来会发生什么。

他的问题是：“在约会中，你做什么肢体语言？”永远，永远，永远，永远，永远，永远，永远，永远不要坐在桌子旁约会。桌子是一大块木头，是一个障碍，它阻碍了你们的交流。我总是坐在吧凳上。我像这样坐在吧台旁，她像这样坐在吧凳上，我们的膝盖会碰到。这样，如果我想碰她的膝盖，我在这里；我可以搂她一会儿；我们在这里；我们可以有那种身体联系，那种身体交流。桌子扼杀了中间的能量。所以我从不坐在桌子旁。我从不在美国式的两人桌约会。即使我去吃晚餐，我也会和女孩坐在桌子的同一边。是的，我这样做。因为我意识到，每次我最终只是坐在她对面，付了餐费，然后她说谢谢，是时候走了。她说：“实际上，我要回家了。”我就想：“什么？”所以我就不再那样做了。我只是避免那样做，因为它没有任何帮助。

本节课中，我们一起学习了高地位肢体语言的核心概念、重要性及具体应用。从眼睛和面部的表达，到头部、肩膀、手部、坐姿、站姿的细节，我们探讨了如何通过肢体语言不仅影响他人对你的看法，更重要的是改变你对自己的感受和心态。记住，关键在于真我显现、选择关注点以及与自身身体的和谐。将这些原则融入实践，你将能更自信、更有效地进行社交互动。
