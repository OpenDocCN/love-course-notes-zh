# 自然流约会学：2.08：收尾的心理学 🧠

在本节课中，我们将要学习强效收尾背后的核心心理学原理。理解这些心态，能帮助你克服在关系推进关键时刻的犹豫与恐惧，从而更自信地采取行动。

很多男性在收尾阶段遇到问题，因为他们不想显得过于强势，或者认为强硬地收尾是混蛋行为。这种想法源于多种原因，可能来自与强势销售人员的交往经历，也可能来自成长过程中被良好社会化、成为一个有礼貌的人的经历。

但事实是，当你收尾过于软弱时，你并没有为对方带来任何好处。原因如下。

你足够关心对方的感受，以至于希望温和地收尾、希望保持礼貌，这一事实本身就意味着你很可能是一个会善待她的好人。然而，如果你作为这个会善待她的好人却一点也不强势，结果就是你将无法完成收尾，而最终与她达成关系的，将会是那个强势的混蛋，一个她本不该与之有关系的人。所以，你以为自己是在表现友善。你以为自己是在对她保持礼貌，避免让她有哪怕一丝负面的体验。但你实际在做的是，你为她决定了：她必须和一个混蛋约会。请理解这一点。这非常、非常重要。当你放弃或停止收尾时，你并没有在帮她。

话虽如此，确实存在“过于强势”这回事。也存在“不接受拒绝”这回事。我并不鼓励那样做。如果对方明确对你说“不”，请将其视为真实的回应，适时后退，不要给自己惹麻烦，不要在那之后继续强求。但除非你得到一个明确的“不”，否则不要自行断定她不感兴趣，让她来告诉你她不感兴趣。不要自行断定她永远不会喜欢你这样的男人，让她自己做出选择。

如果你相信自己是一个优质产品（你最好坚信自己是一个优质产品），那么你对自己、对她都有责任去帮助她获得这个产品。这听起来可能是个有点夸张的例子，但如果你在沙漠中看到一个神志不清、快要渴死却甚至没有意识去要水的人，你试图给他水他却推开，你会想“哦，看来他不渴，看来他不需要水”吗？你会说：“兄弟，你需要水，我会确保你喝到。”因为你关心他，你会确保他得到此刻需要的东西。这个例子有点极端，但道理是相通的。很多时候，女性处于一种奇怪的困境中：她们想和优秀的男人发生关系，想和优秀的男人建立关系，但她们没有太多自由去主动追求，没有太多自由去主动推进关系。同时，她们也多少被迫要表现得有点难以得手。

首先，她们试图避免显得放荡，因为如果显得放荡，很多男人会评判她们。所以她们陷入了这种两难境地。因此，如果你太快放弃，或者太快认为“哦，我没什么价值可提供”，或者认为“哦，我在强加于人”，你对自己或她都没有任何好处。所以，我希望你这样想，我在讲开场白时也提到过这种心态：想象你口袋里有一张巨额支票，你的工作就是把它递给她。在这种情境下，如果你知道自己有价值可以给予，你还会对收尾犹豫不决吗？你绝对不会。所以我希望你知道这一点。要知道你有价值。要知道你可以给予价值。因为当你不知道时，她们不会尊重你，也不会信任你。

有太多、太多、太多的互动因为未能收尾而失败。我个人的例子是，在我学习约会学的头九个月里，我约会不断，因为我擅长搭讪，擅长认识女孩、拿到电话号码、安排约会。但我不知道如何进行肢体接触，不知道如何亲吻女孩。在一次次美好的约会结束后，我一次又一次地看到那种眼神，我知道我需要像个男人一样去吻她，但我就是没做，因为我太害羞，或者觉得时机不对，我不想让场面尴尬。于是，为了避免尴尬，我干脆不去尝试。然后你会看到女孩脸上那种非常难过、失望的表情，你知道她希望我吻她。但最有趣的是，尽管我知道、她也知道我本该吻她，很多时候她会在下一次约会时放我鸽子。因为我没有吻她，这向她证明了我要么不是我所展示的那种男人，不是能满足她需求的男人；要么就是个不懂女人、在女人面前没用的男人；要么就是个不相信自己的男人。如果以上任何一点成立，那么她就有非常充分的理由不再和我约会，不想吻我。所以，这些年来，我因为不去尝试而失去的互动、失去的关系，远比因为尝试而失去的多得多。我不希望这发生在你身上。

原因一，那样你会失去更多。
原因二，那样更难承受。如果你尝试了但失败了，你至少可以坦然接受，知道自己尽力了。如果你没尝试就失败了，你他妈余生都会在想“如果当初……会怎样”。

所以，不要让自己陷入那种境地。你对自己、对那个女孩都有责任去相信你是有价值的，去展现自己，去尝试收尾。😡

如果你尝试收尾但失败了，没有伤害，没有犯规，你已尽力。如果你不尝试就失败了，那是不可原谅的。😡

**本节课总结**

本节课中，我们一起学习了强效收尾背后的关键心理学。核心在于转变心态：**将收尾视为向对方提供价值，而非索取**。犹豫和退缩往往源于过度礼貌或害怕被拒，但这实际上可能将机会让给并不善待对方的人，并且会因缺乏自信和决断力而失去对方的尊重与兴趣。关键在于区分“强势推进”与“尊重拒绝”——在未得到明确“不”的信号前，应自信地推进；一旦被明确拒绝，则必须尊重并后退。请记住，自信地尝试收尾，无论成败，都远胜于因恐惧而放弃，留下遗憾。
