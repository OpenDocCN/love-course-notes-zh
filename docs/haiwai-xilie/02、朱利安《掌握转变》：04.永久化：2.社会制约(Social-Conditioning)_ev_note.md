# Python入门：01：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。它们是构建所有程序的基石，就像盖房子需要砖块和水泥一样。

## 什么是变量？📦

变量是计算机内存中用于存储数据的一个“盒子”或“标签”。你可以把数据放进去，也可以随时取出来或更换它。

在Python中，创建一个变量非常简单，只需要给它起一个名字，然后用等号 `=` 给它赋值即可。

```python
# 创建一个名为 message 的变量，并存入字符串 "Hello, World!"
message = "Hello, World!"
```

上一节我们介绍了变量的基本概念，本节中我们来看看如何给变量命名。

### 变量命名规则

以下是关于变量命名的一些重要规则和最佳实践：

*   **只能包含字母、数字和下划线**：例如 `my_variable`、`score1` 是有效的，而 `my-variable`、`1score` 是无效的。
*   **不能以数字开头**：`variable1` 是有效的，`1variable` 是无效的。
*   **不能使用Python的关键字**：例如 `if`、`for`、`while`、`print` 等是Python语言本身使用的单词，不能用作变量名。
*   **区分大小写**：`age`、`Age` 和 `AGE` 是三个不同的变量。
*   **建议使用描述性的名字**：使用 `user_name` 比 `a` 或 `x` 更能清晰地表达变量的用途。

## 数据类型 🧱

数据类型定义了变量中可以存储什么类型的数据，以及可以对它进行哪些操作。Python有一些内置的基本数据类型。

上一节我们介绍了如何创建变量，本节中我们来看看Python中几种最常见的数据类型。

### 1. 字符串 (String)

字符串用于表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。

```python
name = "Alice"
greeting = 'Hello'
```

### 2. 整数 (Integer)

整数就是没有小数部分的数字，可以是正数、负数或零。

```python
age = 25
temperature = -10
count = 0
```

### 3. 浮点数 (Float)

浮点数就是带有小数点的数字。

```python
price = 19.99
pi = 3.14159
```

### 4. 布尔值 (Boolean)

布尔值只有两种可能：`True`（真）或 `False`（假）。它们通常用于条件判断。

```python
is_student = True
is_raining = False
```

## 查看和转换数据类型 🔍

有时我们需要知道一个变量是什么类型，或者将一种类型转换为另一种类型。

### 使用 `type()` 函数

`type()` 函数可以返回任何变量的数据类型。

```python
print(type(“Hello”))  # 输出：<class ‘str’>
print(type(42))       # 输出：<class ‘int’>
print(type(3.14))     # 输出：<class ‘float’>
print(type(True))     # 输出：<class ‘bool’>
```

### 类型转换

你可以使用特定的函数来转换数据类型：

*   `int()`：将数据转换为整数。
*   `float()`：将数据转换为浮点数。
*   `str()`：将数据转换为字符串。

```python
# 将字符串转换为整数
num_str = “100”
num_int = int(num_str)  # num_int 现在是整数 100

# 将整数转换为字符串
age = 30
age_str = str(age)      # age_str 现在是字符串 “30”

# 将整数转换为浮点数
my_int = 5
my_float = float(my_int) # my_float 现在是浮点数 5.0
```

**注意**：不是所有转换都能成功。例如，你不能将 `“hello”` 这样的文本直接转换成整数。

## 总结 📝

本节课中我们一起学习了Python编程的起点：**变量**和**数据类型**。

*   **变量**是存储数据的命名容器，创建和命名它需要遵循一定的规则。
*   **数据类型**定义了数据的种类，我们重点学习了四种基本类型：
    *   **字符串 (str)**：用于文本。
    *   **整数 (int)**：用于整数。
    *   **浮点数 (float)**：用于小数。
    *   **布尔值 (bool)**：用于真/假值。
*   我们学会了使用 `type()` 函数来检查数据类型，以及使用 `int()`、`float()`、`str()` 等函数在不同类型之间进行转换。

理解这些概念是编写任何Python程序的第一步。在接下来的课程中，我们将学习如何使用这些数据来进行运算和做出决策。
