# 赢家游戏：3.0：实战训练计划：3.2：新手任务：3.2.7：任务8—收号 📞

在本节课中，我们将要学习实战训练计划中的关键一步：如何有效地结束互动并获取联系方式。这是从“练习”转向“赢取”的重要环节。

上一节我们专注于开启对话和给予赞美，本节中我们来看看如何为对话画上一个有成果的句号。

## 核心概念：要求成交

在销售领域有一句名言：**“讲述不等于销售”**。这意味着，如果你想卖出东西，就必须主动要求对方下单。搭讪也是同样的道理。你不能只是聊天，必须在离开前主动“要求成交”，即获取对方的联系方式。

以下是本次任务的具体步骤：

1.  **开启对话**：像之前一样，走上前说“嗨”并给予赞美。
2.  **进行简短交流**：进行一段自然的对话。
3.  **执行收号**：在离开前，必须说出类似以下的话来结束互动：
    *   “和你聊天很愉快，你的号码是多少？”
    *   “我们应该保持联系，你的邮箱是？”
    *   “我们改天再聊，可以给我你的联系方式吗？”

我的建议是直接尝试要电话号码，因为这是最有效的收号方式。但如果你对此感到紧张，可以先从要求名片或社交账号开始，让自己逐步适应。关键在于，你必须为自己创造未来能再次联系到对方的机会。

## 心态调整：关注过程而非结果

因为这是你在所有任务中第一次真正向对方“要求”某样东西，我必须特别提醒你，不要过于纠结于成功或失败。

**“高度赞美 + 最终索取电话号码”** 这个流程，本质上就是完整的“冷搭讪”过程。这是你将在未来数周、数月乃至数年中不断练习、改进和完善的核心技能。所以，不要认为你现在就必须做到完美。只需理解这是一个循序渐进的步骤。

最重要的事情是，你遵循了正确的流程，并且终于给了自己一个可能成功的机会。

*   如果你通过这次任务要到了号码，请为此感到高兴，享受这份成就感，并为自己骄傲。
*   如果你没有要到很多号码，那也没关系。你正在做正确的事情，你正在向上迈进。
