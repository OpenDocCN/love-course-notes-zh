# BOSS课程：11：德里克访谈 - 建立你的行为准则与世界观

在本节课中，我们将与RSD资深导师德里克进行深度对话。我们将探讨如何通过审视自身经历、定义个人原则，并将这些内在准则转化为与外界（包括人际关系和约会）互动的方式。核心在于，真正的成功和吸引力源于内在的坚定与和谐，而非外在的技巧。

---

## 从环境到自觉：自我认知的起源

上一节我们介绍了课程的核心目标，本节中我们来看看德里克如何形成他强大的世界观。他认为每个人都是环境和遗传的产物。

德里克分享了他的成长经历：童年曾享有优越生活，后因父亲生意伙伴的问题家道中落。这种巨大的反差让他意识到，若想重获优质生活，必须依靠自己主动创造，而非等待馈赠。

他早期缺乏杰出的导师或榜样，这促使他一生都在寻找能指引他方向的人。他不断思考：
*   我想过什么样的生活？
*   这种生活是什么样子？
*   什么样的人应该成为这种生活的一部分？
*   什么样的人不应该？

德里克认为，驱动人们行动的是其自身的利益。因此，要达成合作，关键在于找到双方利益的交汇点，创造双赢局面。自私并非坏事，它是人类行为的基本动力。

## 挑战权威与发现潜能：成长的阵痛

德里克和麦迪森都经历了与权威体系的冲突，这种经历反而塑造了他们独立思考和行动的能力。

德里克在学校时，常常因为觉得规则“不必要”或“愚蠢”而陷入麻烦。例如，他不理解为何不能把脚搭在课桌下的横杆上。这种不愿盲从、对无意义规则的反抗精神，恰恰成为他日后创业的关键特质：独立思考、承受社会压力、以非常规方式解决问题以及行动偏好。

麦迪森的经历更为极端，他曾因无法适应学校环境而被五所高中开除，甚至身处危险之中。然而，这段经历让他对日常压力（如在俱乐部被女孩拒绝）产生了强大的免疫力。

**核心转变**在于，他们都**选择**将这些挑战性的经历转化为力量和动力，而不是负担。他们意识到自己拥有更大的潜力，并决定全力冲刺。

## 从工作到事业：独立意识的觉醒

德里克分享了他职业生涯的转折点。他的最后一份工作是在沃尔玛，从装卸工迅速晋升为部门经理。当他查看业绩数据时，发现自己的表现远超前任。

然而，当区域经理告诉他下一次晋升可能需要等待数年时，他有了一个突破性的想法：**如果我将同样的努力和职业道德投入到自己的项目中，而不是作为别人机器上的一个齿轮，我难道不能赚到比在这里可能赚到的更多的钱吗？**

这个想法促使他在19岁后再也没有为别人工作过，成为了“成功的失业者”。他意识到，将时间投资于自己，回报远大于为体系工作。

## 内在准则与外在互动：一致性是关键

将内在原则应用到外部世界，尤其是在约会中，是成为“老板”的核心。

德里克早期接触约会社群时，流行的是基于话术和技巧的“套路式”撩妹。他虽然因此获得了更多机会，但感觉非常虚伪和不满足，因为这并非真实的自我表达。

他转而专注于建立个人原则和行为准则。在约会中，他不再试图讨好所有人，而是进行筛选：
*   **我不喜欢大多数女性**：在俱乐部交谈的女性中，大约85%会让他感到厌烦。
*   **寻找“同类”**：目标是找到那些拥有相似价值观、目标和抱负的人。
*   **价值交换**：思考自己能为对方提供什么价值，以帮助对方实现其目标。

**核心公式**：**吸引力 = 内在一致性**。当你与女性互动的方式与你和其他人（同事、陌生人）互动的方式没有区别时，你就达到了内在的和谐。女性能够感受到这种真实与坚定。

## 价值主张与筛选机制：像经营企业一样经营社交

如何将你的原则有效传达并吸引对的人？这需要清晰的价值主张和严格的筛选。

德里克将互动视为提供“机会”。他的责任是清晰传达这个机会可能为对方带来的价值和生活改变。如果对方不感兴趣，他并不感到被拒绝，就像给流浪汉20美元却被骂一样，只会觉得可笑或是不合适。

他提出了一个尖锐的观点来强调自我价值和时间的重要性：**一个阴道值多少钱？** 在全球范围内，其货币价值可能很低（例如，在某些地方仅需几美元）。因此，容忍一个带来负面情绪或浪费你时间的女人，其“成本”可能远超其“收益”。

**核心行动指南**：
1.  **定义你的价值**：明确你想过什么样的生活，你代表什么。
2.  **有效传达**：学会向他人清晰展示你的价值主张。
3.  **严格筛选**：只在你认为值得的人身上投入时间。你的时间是你最宝贵的资源。
4.  **无惧“拒绝”**：如果对方因为真实的你而拒绝，那只是说明不合适，并非失败。

## 处理非议与建立声誉：专注创造，而非讨好

当你开始坚持自我时，难免会遇到反对者。德里克认为，如果你没有任何批评者，很可能是因为你做得还不够突出。

他并不追求外部认可，而是专注于成为自己想成为的人，并帮助那些真正渴望改变的人。他不在演讲中推销，只是真诚分享自己的思考和经验。这种真实和不强求的态度，反而吸引了最积极、最投入的学员。

他的声誉建立在成果之上：那些最积极、最投入的学员在他的指导下取得了显著突破。这是一种自我选择的过程——真正想“搞定事情”的人会被他吸引，并最终取得成果。

---

本节课中我们一起学习了德里克如何从个人经历中提炼原则，并将这些内在准则转化为强大、一致的外部互动方式。关键在于：**定义你是谁，清晰传达它，严格筛选你的圈子，并为你提供的价值感到自豪。** 真正的力量来自于内在的坚定与和谐，当你做到这一点时，你自然能吸引正确的人和机会，在生活的各个领域成为“老板”。
