# 言语游戏：09：1-8周：第三周（吸引）：吸引技巧：傲慢的框架 🎭

在本节课中，我们将要学习一种非常强大的吸引力策略：建立一种傲慢或自负的框架。这种框架的核心在于，你向外界传递出一种信息，即“如果这是真的，就会导向亲密关系”或“如果这是真的，就意味着她会被你吸引”。我们将探讨如何设定这种框架，并提供具体的开场白示例。

上一节我们介绍了吸引的基本概念，本节中我们来看看如何通过特定的思维框架来塑造互动。

传统的假设是，当你接近一位女性时，她是高价值的，你需要去打动她。傲慢的框架则要求你彻底翻转这个假设。你采取的姿态是：**你才是高价值的一方**，她需要来打动你，或者她需要表现得自然才能吸引你。

这种框架的美妙之处在于，**维持一个既定的框架（维持现状）比改变一个既有的框架要容易得多**。因此，如果在互动开始前，还没有任何框架被设定，你就抢先设定“你是有吸引力的”、“她在追求你”的框架。那么，她将需要付出努力来打破这个框架，这远比一开始就设定框架要困难。

很多时候，你需要在互动初期、任何框架尚未形成时就采取这一策略。

以下是几个具体的开场白示例，它们都植入了傲慢的框架。

**示例一：“你很害羞吗？”开场白**

你走过去说：“打扰一下，你很害羞吗？”她可能会说：“不啊，为什么这么说？”你可以回应：“哦，因为我在这儿站了大概五分钟了，你都没过来跟我打招呼，也没来捏我的屁股。我猜你可能需要点帮助，所以我就过来跟你说话了。”

在这个例子中，你设定的框架是：你和她说话，是因为她一定是不敢主动跟你说话（显然她想跟你说话），你这是在帮她一个忙。这与你直接说“嘿，我想认识你”截然不同。这种做法应该带点轻松戏谑的语气，因为显然是你主动走过去的。但由于你带着一点幽默和调侃，你依然成功设定了框架。如果她抗拒这个框架，你显然可以轻松化解，比如笑着说“开个玩笑啦，别太认真”，然后继续进行正常的互动。

**示例二：“让自己开心”开场白**

另一个更微妙、可能整体上更好的开场白是：“打扰一下，你看起来在这儿挺无聊的。我想我该过来让自己开心一下。”

在这里，你不仅设定了傲慢的框架——你是来“让自己开心”的，并且你并不把和这位美女搭讪当作一件大事——这本身就是一种自负的姿态。你还运用了一点“推拉”技巧：听起来像是“我想过来让你开心”，但最后一刻却改口成了“让我自己开心”。这显得有点自负、以自我为中心，甚至有点傲慢或自恋。你再次设定了框架：**你是这次互动中最重要的人**，你只是来找乐子的，并没有太把她当回事。

**示例三：“我们好看的人”框架**

下一个技巧，我经常以多种方式使用，核心是**将我自己和她都框定为“好看的人”**。

例如，如果她在谈话中提到生活中被讨厌的男人搭讪之类的事情，我可能会说：“我懂，我们这些好看的人得团结一致。长得像我们这么有吸引力，生活可真不容易啊。”

我是在调侃她“生活不易”的处境，但实际在做的是：将我们两人都框定为有吸引力的人。这样，“我很有吸引力”这个框架就被巧妙地置入了对话。在这种情况下，她很可能不愿意反驳这个框架，因为它与她“有吸引力”这一点绑定，而这是她希望成真的事情。

这是一种传递带有**些许性暗示和自负性质框架**的思路。如果这个框架被接受，会让你更接近亲密关系。如果未被接受，你总可以笑着说“开个玩笑”。很多时候，如果框架未被接受，你会立刻得到一个“废物测试”，这在互动初期是一个绝佳的机会，能比冷启动时更好地展示你的价值。

本节课中我们一起学习了“傲慢的框架”这一强大的吸引力技巧。我们了解到，关键在于抢先设定“你才是高价值方”的互动基调，并通过具体、略带调侃的开场白来植入这一框架。掌握这一技巧，能让你在社交互动中占据更主动、更有利的位置。
