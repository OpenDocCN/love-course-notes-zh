# 朱利安《Ten Game十戒》：05：任务三

## 概述
在本节课中，我们将学习任务三的具体内容。这个任务将延续每日外出的习惯，并引入新的心态挑战与社交技巧练习。我们将重点探讨如何建立个人权威、获取电话号码、开启混合组合以及如何在互动中加入幽默元素。

## 任务核心要求
上一节我们专注于物流和习惯的建立，本节中我们来看看如何在此基础上增加新的挑战。

以下是任务三的核心行动要求：
*   **持续外出**：在接下来的7天里，每天至少外出30分钟。这将使你连续外出的总天数达到21天。
*   **禁止提问**：本周禁止在任何论坛或群组中提出任何问题。你必须学会自己寻找答案或信任自己的判断。
*   **获取号码**：本周内尝试获取5个电话号码。
*   **开启混合组合**：每天尝试开启一个包含男性的社交组合（混合组合），并进行简短的闲聊。
*   **自我娱乐**：在互动中，尝试尽可能多地、不露痕迹地插入“like a dog”这个短语。

## 心态调整：建立个人权威
本周的心态训练重点是**建立个人权威**和培养**自我信任**。许多人习惯于将决策权交给外部，不断寻求他人确认，这削弱了内在的权威感。

我的挑战是：**本周不允许提出任何问题**。当你产生疑问时，请遵循以下步骤：
1.  问自己：“**如果永远没人给我答案，我现在会怎么做？**”
2.  然后据此行动。你可能会自己研究、查阅资料、进行测试，或者最终信任自己内心的答案。

你需要做的不是提问，而是在完成自我解答后，写下你原本的问题，并附上你是如何找到答案的。这个过程旨在重建你对自己判断力的信任。

## 技巧练习：获取电话号码的五个步骤
在获取电话号码时，遵循清晰的步骤可以提高成功率。以下是五个关键步骤：

1.  **明确意图**：在交谈中，自然地提及再次见面的可能性。例如：“你这周哪天有空？我们可以一起喝杯咖啡。”让她明白你要号码是为了邀约。
2.  **确保记住**：确保对方记住了你的名字。很多女孩会忘记。
3.  **自然拿出手机**：在询问号码之前或交谈过程中，就自然地拿出你的手机。不要先问“能给我你的号码吗？”，等对方同意后再慌乱地找手机。要假设对方会同意，直接问：“你的号码是多少？”
4.  **保持低调**：如果她身边有朋友，尽量选择不那么引人注目的时机交换号码，避免让她感到尴尬或有压力。
5.  **立即备注**：拿到号码后，立刻发一条包含你名字的短信给她。例如：“嗨，我是[你的名字]，很高兴认识你:)”这能确保她存下你的联系方式。

## 社交拓展：开启混合组合
为了训练自己在任何社交场合都能感到自在，你需要挑战更“困难”的情景。

**混合组合**指的是包含男性和女性的社交团体。我们的目标不是去搭讪其中的女性，而仅仅是进行**30秒左右的闲聊**。例如，走过去说：“这地方不错，对吧？”聊几句后，自然结束：“好了，祝你们玩得开心，再见。”

这样做的目的是积累“**参考经验**”，证明即便场中有男性，也不会发生什么可怕的事情。大多数男性都和你我一样，不会因为有人来闲聊就反应过度。

## 自我娱乐：加入幽默元素
最后一项练习是为了给互动增添趣味，并让你学会自娱自乐。

挑战在于，在一次互动中，你能多巧妙地、不引人注意地插入“**like a dog**”这个短语。例如：
*   对方：“我叫杰西卡。” 你：“哦，这名字不错，like a dog。”
*   对方：“我是做设计的。” 你：“哦，很酷，like a dog。”

**重点不在于这个短语本身能否吸引女孩，而在于它能让你自己感到有趣，成为你与自己之间的一个内部笑话**，从而放松心态。

## 总结
本节课我们一起学习了任务三的完整内容。我们明确了需要**连续7天、每天外出30分钟**的核心行动。在心态上，我们专注于**通过禁止提问来建立个人权威和自我信任**。在技巧上，我们学习了**获取电话号码的五个步骤**，并挑战自己每天去开启一个**混合组合**以积累社交信心。最后，我们通过尝试在对话中插入“**like a dog**”来练习**自我娱乐和保持轻松心态**。请坚持执行，并在社群中发布你的每日报告。祝你好运。
