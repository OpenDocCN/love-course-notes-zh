# 掌握转变：02-06：早晨指导发布 🧘

在本节课中，我们将学习一个早晨的指导练习，旨在帮助你释放杂念、连接身体，并以更轻松、专注的状态迎接新的一天。整个过程分为三个核心步骤：觉察、体验与释放。

---

## 环境准备

开始练习前，请确保你处于一个不受外界干扰、无人会打断你的环境中。

确保你的手机和电脑通知都已关闭。

然后，挺直腰背坐在椅子上，以保持警觉和临在状态。

将双手放在膝盖上，掌心向下，然后闭上眼睛，我们开始。

---

## 第一步：觉察你想要释放的事物 🎯

上一节我们做好了环境准备，本节中我们来看看如何开始觉察。首先，我们将引导你从头脑中走出来，进入你的身体。

请可视化你两耳之间的那个“话匣子”——那个总是在喋喋不休的小声音。它可能在思虑即将到来的一天，为之兴奋、担忧或 dread。

现在，请可视化这个小声音改变位置，缓慢地一路向下移动，直到你的脊柱底部。这个位置也被称为你的根脉轮或基础脉轮。这是与你的身体建立深层连接的地方。

请缓慢地将你的意识带到那里。那个小声音，逐渐下移。一切变得越来越安静，最终安顿在脊柱底部。

你现在从那里思考，从那里呼吸，从那里运作。你与身体连接在一起。

让我们通过进行三组**腹式呼吸**来锁定这个位置。具体做法如下：

```
吸气（通过鼻子）：数6下
屏息：数3下
呼气（发出“嘘”声）：数6下
屏息：数3下
```

我们将从脊柱底部进行这个呼吸。

以下是呼吸循环的引导：

1.  现在开始吸气，数6下：1, 2, 3, 4, 5, 6。屏息，数3下：1, 2, 3。呼气，数6下：嘘, 2, 3, 4, 5, 6。屏息，数3下：1, 2, 3。正常呼吸。
2.  进行第二组循环。从脊柱底部吸气，数6下：1, 2, 3, 4, 5, 6。屏息，数3下：1, 2, 3。呼气，数6下：嘘, 2, 3, 4, 5, 6。屏息，数3下：1, 2, 3。正常呼吸。
3.  进行最后一组循环。现在吸气，数6下：1, 2, 3, 4, 5, 6。屏息，数3下：1, 2, 3。呼气，数6下：嘘, 2, 3, 4, 5, 6。屏息，数3下：1, 2, 3。正常呼吸。

现在，一切都更安静了一些。你已走出头脑，进入了身体。

---

## 第二步：全然体验这些感受 🌊

在第一步中，我们学会了将意识带入身体并安顿下来。现在，请开始觉察身体内出现的各种感觉。

以下是可能出现的感受类型，请留意它们：

*   任何被激起的情绪、紧张、紧绷或不适。
*   你可能发现自己还带着前一天晚上残留的情绪，可能来自睡梦或噩梦。
*   当你想到即将到来的一天时，伴随出现的身体感觉。

请思考即将到来的一天：你要见的人、要去的地方、要做的事、要完成的工作。让任何浮现的感觉自然出现。

现在，我们进入第二步：允许自己全然体验这些感觉。

你需要做的只是与这些感觉待在一起，沉浸其中，让它们接管你。

以下是本步骤的关键原则：

*   不要试图修复它们。
*   不要试图改变它们。
*   不要试图理解或分析它们。

这里没有需要修复、改变或理解的东西。你所要做的就是向它们 surrender（臣服）。给予这些感觉一个信息：你无条件地全然与它们同在，你愿意接纳它们。

让我们花一些时间，只是与这些和未来一天相关的感受待在一起，无论它们是什么。继续从脊柱底部向它们呼吸，让它们接管。更深入地沉浸其中，全然与它们同在。

---

## 第三步：释放所有感受 🕊️

在充分体验了身体的感受之后，我们进入最后一步：让所有这些感受升起，然后将它们释放出去。

你要邀请它们升起。记住，它们想要离开，想要完成它们的自然进程然后消失。我们只需要停止通过抵抗来紧抓它们。

无论那些感觉是什么，让我们停止抵抗。欢迎它们，允许它们完成进程并消失。

我们将再进行三组腹式呼吸，向这些感觉呼吸。这一次，伴随着每一次吸气，请可视化这些感觉升起、接管；伴随着每一次呼气（发出“嘘”声），请可视化它们穿过你并被释放出去。

以下是释放的呼吸引导：

1.  现在与那些感觉连接。吸气将它们带起，数6下：1, 2, 3, 4, 5, 6。屏息，数3下：1, 2, 3。呼气将它们释放，数6下：嘘, 2, 3, 4, 5, 6。屏息，数3下：1, 2, 3。正常呼吸。
2.  可能还有些感觉在那里，让我们连接它，将它带起并释放。吸气数6下：1, 2, 3, 4, 5, 6。屏息数3下：1, 2, 3。呼气数6下：嘘, 2, 3, 4, 5, 6。屏息数3下：1, 2, 3。正常呼吸。
3.  最后进行一次循环。无论下面还有什么，连接它，吸气将它带起，数6下：1, 2, 3, 4, 5, 6。屏息数3下：1, 2, 3。然后全部呼出，数6下：嘘, 2, 3, 4, 5, 6。屏息数3下：1, 2, 3。正常呼吸。

---

## 完成与回归 🌅

现在，注意事物是否感觉更轻松了一些？一种如释重负的感觉，仿佛重担被卸下。

不再是对未来一天的担忧、恐惧、兴奋或执着于某事是否发生，而是一种更轻盈的感觉。你处于一种欢迎未来一天到来的状态。

让我们再花一些时间，让自己体验这种解脱，体验这种不抵抗的欢迎状态。

现在，从脊柱底部进行最后一组腹式呼吸。现在开始吸气，数6下：1, 2, 3, 4, 5, 6。屏息，数3下：1, 2, 3。呼气，数6下：嘘, 2, 3, 4, 5, 6。屏息，数3下：1, 2, 3。正常呼吸。

我们准备慢慢回来了，准备好度过美好的一天。慢慢开始轻微活动身体，将一些意识带回到你的周围环境，准备睁开眼睛。

---

本节课中我们一起学习了早晨指导发布的完整流程：从环境准备开始，通过**觉察**想要释放的思绪并将其带入身体，然后**全然体验**伴随的感受而不加评判，最后通过呼吸**释放**这些感受以达到轻松、临在的状态。这个练习可以帮助你以更清晰、平和的心态开始新的一天。
