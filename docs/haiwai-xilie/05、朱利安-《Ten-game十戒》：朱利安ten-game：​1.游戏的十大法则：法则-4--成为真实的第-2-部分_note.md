# 搭讪艺术：05：游戏十诫之法则四 - 成为真实的自己（第二部分）

在本节课中，我们将深入探讨“游戏十诫”的第四条法则——成为真实的自己。我们将解析为什么真实具有巨大价值，并提供具体的练习方法，帮助你摆脱社交中的伪装与操纵，学会真诚地表达自己。

## 理解真实的价值

上一节我们介绍了真实性的基本概念，本节中我们来看看为什么真实如此重要。你必须尝试去想象一个女孩所处的现实环境，或者观察大多数人的行为方式。你会发现并可以讨论人们有多么虚伪。

人们上前搭讪并表现得友善，并非因为他们真心想友善，而是因为他们试图得到某些东西。有些男人会故意对女孩刻薄，因为他们从网上读到“混蛋更受女孩欢迎”的说法。他们这样做也是为了得到某些东西。几乎没有人是真正真实的。这一切都只是一种操纵形式。大多数沟通都是操纵。这非常疯狂。

## 真实是稀缺品

想象一个百万富翁在俱乐部里，周围都是穷人。每个人都会上前搭讪，说些有趣的话。他们很友善，有些人则很刻薄，但背后总有一个小小的议程——他们想要钱。对于一个女孩来说，情况类似。每个和她说话的人都想获得她的认可、她的肯定，或者想和她上床来满足自己的感觉。没有什么是真正真诚的。

当某个男人走上前去，能够真正做自己，摆脱那种操纵时，就像一股新鲜空气。这本身就蕴含着巨大的价值。这种价值会随着你摆脱对认可的许多需求而自然产生。

## 如何练习真实

以下是几种帮助你练习真实性的具体方法：

*   **假设已达成目标法**：例如，假设你和这个女孩已经有了“今晚会发生关系”的默契。你会如何行动？另一个方法是，当你走上前时，可以尝试“假设吸引力”练习。想象你已经从女孩那里得到了你想要的良好反应。去感受那种情绪，体验那种“良好反应”的感觉，然后基于“我已经得到了”的假设去行动。问问自己：“现在，我得到了所有想要的。接下来呢？”这样会让你更接近真实。
*   **接纳当下感受**：我们常常感到羞耻，会想：“这种感觉好吗？这感觉坏吗？”这又是贴标签的行为。拥抱你的真实想法。例如，当你没话可说时，就真实地承认：“我没话可说了。”这就是真实。现在你就有话题可以聊了——聊聊“没话可说”这件事本身。如果你感觉糟糕，就告诉对方你感觉糟糕；如果你感到悲伤，就承认你感到悲伤。不要评判情绪，只需承认它们。真实的东西就有价值。
*   **展示真实状态**：我曾解释过一个例子：一个学员在咖啡店看到一个女孩，他当时感觉并不好，有点累，有点暴躁，咖啡也还没喝到。他本可以只是走上前，告诉女孩自己很暴躁。以暴躁的状态去提供价值，并不意味着你要试图让她来安抚你。你可以说：“嘿，你知道吗，我现在真的很暴躁。我的咖啡还没好。”你提供的是真实。谁说这一定是坏事？我们被社会条件反射地认为必须总是感觉棒极了。感觉棒极了固然有价值，但如果你没有那种感觉，展示真实本身也有价值。要始终真实。

## 保持内外一致

这与“内外一致”的概念相同。例如，我晚上刚出门时，感觉通常和夜半时分不同。在开始时，如果我感觉还没到位，强迫自己热情地说“嘿，玩得怎么样？”就不够真实。我不会强迫或伪装。开始时，我可能只是简单地说：“玩得怎么样？很高兴认识你。你是洛杉矶人吗？哦，酷。这些是我的朋友。”仅此而已。但这内外一致，这就是真实，价值就在于此。不要强迫，让它自然流露。始终与你当下的感受保持一致。

理想情况下，这可以回溯到“自我扩展”的概念。你就像靠在悬崖边上，随着夜晚的深入，越来越多的真实自我会展现出来。

## 明确你的意图

这也意味着要明确你的意图。你可能听说过：不要以朋友对朋友的方式相处，要以男人对女人的方式。这也是真实的一部分。让她知道你的想法，让她知道你被她吸引。我们常常隐藏这一点，因为感到羞耻或内疚。现在，要假设你的欲望，把它表达出来。学会真实。

以下是一种帮助你习惯于此的“训练轮”方法：

*   **练习表达意图的陈述**：你可以说：“嘿，我喜欢你。”、“嘿，你真漂亮。”或者“爸爸回家了。”这是一个开启对话的好方法。走上前说“爸爸回家了”，你就无法隐藏你的意图了。长期来看，你不需要一直这样做，但短期练习能让你体会到：“当我真实时，没什么坏事发生。”、“让她知道我的想法是可以的。”你让她知道得越早越好。大多数男人会隐藏意图，这又是一种操纵形式。他们像朋友一样和她聊天，然后在最后才“亮出底牌”——“顺便说一句，我有那方面的想法，我想和你上床。”她之前并不知道。要让她知道你为什么在那里。这应该是清晰的能量、清晰的情绪和清晰的沟通。一切都应该清晰明了。

## 超越语言的真实

这也更多地体现在眼神接触中。你可能听说过“打破融洽感”，即你的声音略微降低。这其实是寻求认可和操纵的终极形式。有时进行一些肢体接触也可以，但你并不一定需要做所有这些。很大程度上在于氛围。明确你的意图不是你“做”出来的事，而是她能“感觉”到的事。

即使你不说“爸爸回家了”这样的话，甚至不需要肢体接触，只是谈论我走到俱乐部的过程，也可以是以男人对女人的方式。比如：“我迈出了第一步，然后又迈出了下一步。”她知道我喜欢她。这更多是允许自己将那种情绪和意图释放和投射出来。就像解除阻塞，消除你的恐惧。

## 克服对真实的恐惧

那么，你害怕什么呢？如果你真实，你害怕什么？这可以回溯到法则一，可能是害怕拒绝，可能是害怕对方不喜欢，也可能是再次提醒自己认识到自身的价值。是什么阻碍了你变得真实？然后就是花时间练习，比如使用一些表达意图的陈述，向自己证明没什么坏事会发生，习惯那种情绪，然后抓住那种感觉，摆脱那些固定的句子，并培养它。这非常重要，且价值巨大。

## 其他辅助练习方法

以下是一些其他的“训练轮”方法：

*   **主导对话**：例如，主导对话的走向。如果女孩在谈论一些你并不真正喜欢的话题，你可以引导话题转向。你听着并假装喜欢，这并不真实。切断某些话题线索，切断某些主题，稍微表示一下不同意。这非常重要。
*   **练习推拉技巧**：你可能听说过“推拉”技巧。它基本上是练习真实性的“训练轮”。如果你真实，推拉自然会发
生。如果你真实，总会有一些女孩产生共鸣的东西，也有一些她不共鸣的东西。自然会有一些“推”和“拉”。当我外出时，我不会担心“我现在是在推吗？我是在挑战还是不同意？”它只是自然发生，因为我是真实的。但如果你不习惯，你可能需要加上这个“训练轮”——时不时地表示一下不同意。

## 允许自己脆弱

最后，关于真实，我想说的是，这实际上是在某种程度上允许自己脆弱。有句话说：越个人化，越好。你让事情越个人化，效果就越好。这也与发挥创造力有关。你觉得什么有趣？你想说什么？即使你说的是别人说过的话，你是否真的想这么说？要让这真正关于你自己。不要试图复制别人。如果你复制别人，你只会成为那个人的拙劣模仿版。只有真正属于你个人的东西才具有力量。所以，把你个人的东西带到互动中来。

## 课程总结

本节课中，我们一起学习了“成为真实的自己”这一法则的深层应用。我们明白了真实之所以具有巨大价值，是因为在充满操纵和伪装的社交环境中，真诚本身就是一股稀缺的“新鲜空气”。我们探讨了通过“假设已达成目标”、“接纳当下感受”和“明确表达意图”等方法来练习真实。关键在于保持内外一致，不强迫自己，并克服对拒绝和评判的恐惧。记住，越个人化、越真实的表达，越有力量。真实性不是一种技巧，而是一种需要培养和释放的内在状态。
