# 掌握转变：02：月度计划：2月：1.内疚与自我憎恨 😔

在本节课中，我们将要学习内疚与自我憎恨的区别，理解它们如何导致自我破坏，并掌握如何从中解脱出来的具体方法。

欢迎来到“转变”系列的第二个月，主题是内疚与自我破坏。首先需要明确的是，当我们谈论内疚时，我指的是**状态**，而非**情绪**。感到内疚这种情绪是必要且有益的。它向你传递信息，让你知道自己搞砸了。它告诉你，你的行为与你的价值观不一致，或者违背了你的原则，是你不想重复的事情。内疚的情绪是暂时的。

然而，内疚的状态是持续性的。当你接收到内疚情绪传递的信息后，你没有去承认它、从中学习、采取可能的补救行动并继续前进，而是紧抓着它不放。你成为了内疚的囚徒，不断纠结于自己做过或没做过的事，自我攻击和自我虐待随之而来，并主导了你的生活。你持续地沉浸在悔恨和24/7的内疚感中。这种状态在外在世界中的表现就是**自我破坏**。因为你觉得自己有罪，所以不配获得成功。

上一节我们区分了内疚情绪与内疚状态，本节中我们来看看为什么我们会沉溺于这种有害的状态。

首先，你需要认识到，持续感到内疚、处于这种状态中，没有任何用处。它没有任何目的。它只会导致你自我破坏，让你停滞不前。我们之所以容易沉溺其中，是因为它让我们感觉良好。这源于我们的成长环境：我们被教导要一直做个“好人”。通过感到内疚，我们是在惩罚自己，以维持“好人”的形象。我们在为自己应得的“正义”买单。如果我们不感到内疚，我们就会成为“坏人”。这就是我们紧抓不放的原因。

这里总伴随着某种依恋和抗拒。依恋是：“我是个好人。如果我感到内疚，我就应该感到内疚。我应该为我所做的事付出代价。” 在内心深处，你喜欢这种感觉。你必须对自己诚实。你可能认为你想摆脱内疚，但你的潜意识有一部分**热爱并需要**内疚。自我破坏也是如此。那个不断拖你后腿的部分这样做是有原因的，它不是为了故意搞砸你，而是为了让你成为一个“好人”，让你看到“正义”并为自己的行为付出代价。而恐惧则是：“如果我放下内疚，我就是个坏人。”

现在，我们来看看如何摆脱这种内疚状态。

第一步是**切断它的势头**。如果你现在正处于内疚状态，听到我说“你必须放下它”时，很容易因为感到内疚而更加内疚，陷入恶性循环。因此，首先要接纳你此刻感到内疚的状态。承认它。然后，坚定地承诺并决定愿意原谅自己、放下它。你必须对此做出承诺，因为它非常容易上瘾。

需要明确的是，原谅自己、放下内疚，并不意味着你做过或没做的事是可以接受、正确的，也不意味着你应该继续这样做。内疚情绪告诉你搞砸了，你就不应该再犯。自我接纳与认可某事是有区别的。原谅自己是为了让你能够继续前进。

接下来，你需要**精确识别**是什么在滋养你的内疚。你后悔什么？你为什么感到内疚？如果自我破坏出现在你的生活中，很可能就存在内疚。你必须坐下来，诚实地识别它，停止视而不见或否认。

以下是帮助你识别内疚根源的一些方法：

*   **冥想练习**：想象自己死去，从那个不再需要达成什么或逃避什么的视角回顾一生，看看会浮现出哪些遗憾。
*   **心理回溯**：在脑海中回到过去，重演当时的情景。以你现在的认知和视角，你会采取哪些不同的做法？这能极大地强化你从中学到的教训。

你必须现实地看待过去，认识到**在当时，你已经尽了最大努力**。错误是你在反思时希望自己当时能做得不同的事情，而“反思”是关键。我们总是在事后才知道什么是正确的做法。在犯错时，我们并不知道那是错误。没有人会故意选择去犯一个将来会后悔的错误。我们当时都在尽力做到最好。如果你带着同样的视角和信息回到过去，你**会做出完全一样的事情**。你必须接受这一点，并认识到错误的价值。正是通过犯错，你才得以学习和成长。

此外，内疚不会改变已经发生的错误，感到内疚也于事无补。错误永远不会降低你的自我价值。我们被教导要过于在意自己表现得有多好。你需要**放下完美主义**，给自己一些喘息的空间。我们为自己制定了太多内部规则，很多内疚感源于我们未能达到所有这些规则。很多内疚与现实无关，只是因为你给自己设定了“我必须永远如何”的规则。

在社交媒体时代，我们被灌输了“除了自己，别人都很完美”的观念。不要相信这一点。你在网上看到的只是人们最好的一面。拿自己最糟糕的一面去和别人最好的一面比较，你永远不会赢。要认识到**没有人是完美的**。每个人都有自己的问题，这很正常，只是他们不展示出来而已。

另一个关键点是**原谅他人**。如果你很难原谅自己，很可能你也很难原谅他人。原谅他人有助于你原谅自己。试着先去理解，再寻求被理解。设身处地为他人着想，你会发现，别人对你所做或未做的事，也是他们在当时尽了最大努力的结果。每个人都在以自己的方式应对生活。理解这一点并不意味着你认可他们的行为，但能帮助你原谅他们。心怀怨恨毫无益处，和平更好。

最后，我们通过一段与客户的对话来具体分析内疚与自我破坏的关联。这位客户总是在人际关系和机会面前自我破坏。

对话中，我首先让他**觉察**自己感到内疚的是什么。我问他：“如果你是一个女孩，你会愿意和你自己约会吗？为什么不会？”他列出了自己的缺点：懒惰、不坚持己见、不跟进事情、感觉自己的意见不重要。

接着，我让他**列出自己的优点**。这是对抗内疚、减少自我破坏的有效工具。内疚让我们只盯着自己不喜欢的地方。列出你为他人带来的价值，并时常重温这个列表，能极大地改变你的心态。这就像销售中，列出产品能给客户带来的所有好处，能让你在推广时充满底气，而不是感到内疚。

此外，一个短期工具是**现实地观察周围**。当你觉得自己配不上某人时，看看周围，那些“更好的选择”真的存在吗？还是只是你的想象？这有助于打破完美主义的幻觉。

然而，这些工具只能暂时缓解。要触及根本原因，你需要**深入挖掘**：你为什么觉得自己没有价值？这种感觉从何而来？对于这位客户，我们深入探讨了他的童年经历：父亲缺席、家庭冲突、目睹祖母去世时无法哭泣、觉得自己让父亲失望……这些经历让他背负了沉重的内疚和自我憎恨。

关键在于，你需要意识到这些“都是你的错”的想法，并**挑战它们**。实际上，这些都不是你的错，但你的潜意识相信是。你必须深入这些记忆，**原谅当时的自己**。以当时的认知和能力，你已经做到了最好。学习**自我认可**，停止自我憎恨和自我抛弃。当你开始处理这些压抑的情绪，真正地原谅和接纳自己时，根本性的改变才会发生。

**总结**：本节课中我们一起学习了内疚情绪与内疚状态的根本区别。内疚状态是持续性的自我惩罚，会直接导致自我破坏。我们沉溺其中是因为它维持了我们“好人”的自我形象。要摆脱它，需要切断势头、识别根源、原谅自己、放下完美主义、现实看待他人与自己，并通过深入挖掘童年或过往创伤来触及问题的核心，最终学会自我认可与接纳。记住，持续的自我憎恨无法带来任何益处，唯有放下内疚，你才能从自我破坏的循环中解脱，真正向前迈进。
