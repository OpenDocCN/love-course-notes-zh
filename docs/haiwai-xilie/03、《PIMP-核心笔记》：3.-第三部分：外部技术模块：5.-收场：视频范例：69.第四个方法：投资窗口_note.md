# 《PIMP 核心笔记》：3. 第三部分：外部技术模块：5. 收场：视频范例：69.第四个方法：投资窗口 💰

在本节中，我们将学习一个名为“投资窗口”的互动方法。这个方法的核心是引导对方主动进行情感或社交上的投入，从而建立更紧密的联系。

上一节我们介绍了其他收场技巧，本节中我们来看看如何通过“投资窗口”来深化互动。

---

### 方法原理

“投资窗口”是指通过言语或情境设置，为对方创造一个需要主动表态或做出选择的机会。当对方选择向你靠近或为你辩护时，就相当于对你进行了“投资”，这有助于提升你在关系中的主导地位。

### 具体范例与应用

以下是一个在约会场景中的应用范例。

假设约会时，女孩对你说：“今天本来有一个闺蜜约我的，我推掉了。”

此时，你可以进行一个轻微的曲解。你可以说：“你闺蜜约你？如果你不想和我见面，不用勉强。我不想因为我的原因，让你没法去见闺蜜。”

这样做的目的是创造一个机会，让女孩对你说：“不，没关系，我愿意和你聊天，我愿意留下来。”

当女孩这样回应时，她就承担了一定的社交风险（例如，可能显得重色轻友），这标志着**她主动跳进了这个“投资窗口”**。

在实际操作中，你会发现女孩很少会直接说“我愿意留下来”。相反，她们通常会寻找借口。

以下是她们可能使用的常见借口：

*   她们会说：“没关系，她们几个人在逛街呢。”
*   或者说：“估计她们现在在吃饭呢，我不饿。”
*   也可能说：“没事，反正我也是打算来这边喝咖啡的。”

**当女孩帮你找借口来圆场或转移话题时，这是一个对你有利的积极信号。**

在得到这类回应后，你可以再次抛出一个“投资窗口”来强化效果。你可以说：“如果你有事的话，现在可以回去，没关系的。如果你想见闺蜜，现在走还来得及。”

总之，核心是持续**给女孩创造机会来追逐你和证明她的意愿**。
