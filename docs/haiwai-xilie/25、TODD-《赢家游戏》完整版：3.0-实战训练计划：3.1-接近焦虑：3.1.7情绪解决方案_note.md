# TODD-《赢家游戏》完整版：3.0-实战训练计划：3.1-接近焦虑：3.1.7 情绪解决方案 💡

在本节中，我们将探讨克服接近焦虑的最终技术方案。这个方案的核心在于重新定义你内心的“痛苦”与“胜利”，将行动本身转化为一种不可违背的个人准则。

上一节我们讨论了认知重构，本节中我们来看看如何从情绪层面彻底解决接近焦虑。我最终克服接近焦虑的方法，正是基于以下这个核心转变。

最终，我将“不去接近”所带来的痛苦，关联得比“接近可能产生的任何结果”所带来的痛苦都要大。

原因如下。大约从8、9岁起，我就意识到自己喜欢女孩。但从那时起直到18岁，我从未进行过搭讪。即使看到想交谈的女孩，我也毫无行动。即使是同班同学，我也什么都没做。我只是不采取行动。

每一次，我没有挺身而出；每一次，我没有去和女孩交谈；每一次，我没有约女孩出去；每一次，我没有展现出男人对女人的姿态，我都感觉自己不像个男人。我对自己感觉更糟，这很痛苦。

这种痛苦变成了失望。事实是我知道我在拒绝自己。我甚至没有给女孩拒绝我的机会，是我自己在拒绝自己。

这种痛苦最终累积到了我无法承受的地步。我到达了一个临界点，心想：去他妈的，不能再这样了。我绝不再拒绝自己。这太他妈可悲了。

这成了我肩上一个沉重的包袱，我绝不允许这种情况再次发生。我别无选择，只能去接近，因为我知道如果不去接近，对我而言将是多么痛苦。

我不知道和女孩的互动是否会顺利，但我知道如果我不去接近，我他妈的会懊恼一整天。我已经拒绝自己太多次，绝不能再让这种事发生。这就是我做出的决定。

就像宗教信徒遵守他们的戒律一样，我也有了自己必须遵守的戒律。我绝不会违反它们。这成了我的一条戒律。

**戒律：汝不可拒绝自己。** 😡

**戒律：汝绝不可再拒绝自己。**

她拒绝你，没关系，甚至是胜利，是积极的，是好事。但如果你拒绝自己，那他妈算什么？😡 你不被允许拒绝自己。这成了我的信条。

因为，我经历了太多次那种痛苦。再次承受那种痛苦、想到我可能又拒绝了自己，这比女孩可能对我做的任何事情都更让我受伤。

公平地说，你从她那里得到的任何回应，都不是对你个人的反映。但你没有挺身而出，这却是对你个人的反映。所以，把你的自尊心附着在这件事上。

不要把你的自尊心附着在“我是否受女孩欢迎”或“我是否得到了好的回应”上。把你的自尊心附着在：“我是否让自己失望了，还是我挺身而出了？”“我是否做了我知道正确的事，还是我选择了当下更舒适但更懦弱的答案？”

你需要改变你对“对与错”的赋值。改变你对“胜利与失败”的定义，并为自己立下一条戒律：绝不再退缩，绝不再拒绝自己。这就是我所做的。从根本上说，正是这一点让我克服了接近焦虑。

那么，我克服接近焦虑是指我不再感到害怕吗？不是。我克服接近焦虑是指，无论我多么害怕，我都知道正确的事该怎么做。无论你多害怕，都要做正确的事。正如人们所说，勇敢并非没有恐惧，而是**心怀恐惧，却依然行动**。😰 只是不再给自己找借口。

这就是我用来确保克服接近焦虑的根本方法。你真正需要做的根本之事，就是拥有足够强烈的渴望。

你需要有足够强烈的渴望去追求你想要的东西。坦率地说，你需要有足够强烈的渴望去爱自己、忠于自己，从而不再拒绝自己，不再让自己失望。

因为如果女孩拒绝你，没关系，无论如何我可以接受，我可以带着被女孩拒绝的经历继续生活。但你怎么能带着“知道自己正在拒绝自己”的认知去生活？你怎能那样活着？

这对你来说必须是痛苦的。它必须痛苦到让你说“绝不再犯”，这就是我的戒律：**汝绝不可再拒绝自己**。

本节课中，我们一起学习了克服接近焦虑的终极情绪解决方案：通过将“不行动”与巨大的个人痛苦关联起来，并立下“绝不拒绝自己”的绝对戒律，从而将恐惧转化为必须行动的坚定决心。关键在于重新定义胜利（挺身而出）与失败（自我拒绝），让勇敢行动成为你对自己不可违背的承诺。
