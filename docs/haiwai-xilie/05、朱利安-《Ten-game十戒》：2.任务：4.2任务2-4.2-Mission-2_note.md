# 约会游戏十戒：05：任务2

## 概述

在本节课中，我们将学习并执行第二个核心任务。此任务旨在巩固你的外出习惯，并引入新的技巧，包括混合日间与夜间游戏、培养积极心态、掌握互动中的后勤信息，以及使用一个高能量的开场白。

## 任务回顾与引入

上一节我们完成了连续七天外出的第一个任务。现在，我们将在此基础上继续前进。

本节的任务将持续七天，每天至少外出30分钟。这意味着你的连续挑战天数将从七天延长至十四天。本周，我们将在任务中加入一些新的调整。

## 核心任务要求

以下是本周你必须完成的四项核心要求。

**第一，混合日间与夜间游戏。**
在接下来的七天里，你必须混合进行日间游戏和夜间游戏。我们总是倾向于回到自己最习惯的模式。为了让你成为无论在白天、夜晚或任何环境中都能轻松自如地与女性互动的人，本周你必须走出舒适区。如果你只习惯夜间游戏，本周必须尝试日间游戏；如果你只习惯日间游戏，本周必须尝试夜间游戏。

**第二，禁止自我否定。**
在接下来的七天里，完全禁止否定自己。这是保持正确心态的关键。无论发生什么——被拒绝、没拿到电话号码，或者行动未达预期——你都不能否定自己。这意味着你必须寻找积极面，发现自己做得好、成功的地方，并且只能认可自己。在你的每日报告里，不允许出现任何负面、抱怨或受害者心态的内容，只允许积极和鼓励。

**第三，获取后勤信息。**
你需要开始关注互动的逻辑层面，即后勤信息。在接下来的七天里，你每天至少需要从一次互动中获得以下五个后勤问题的答案：
1.  你和谁一起来的？
2.  你住在哪个区域？
3.  你怎么回家？
4.  你有室友吗？
5.  你今晚/今天晚些时候/明天有什么安排？

**第四，使用特定开场白。**
本周，你需要使用“爸爸回家了”这个开场白来开启三次对话。这个开场白要求你完全表明意图，不能含糊其辞。你必须以自信的姿态说出，同时也要能自嘲，否则会显得怪异。记住，这只是开场白，说完后要继续对话。

## 任务总结

本节课中，我们一起学习了第二个任务的具体内容。你需要做的是：连续七天、每天至少30分钟外出；混合进行日间与夜间游戏；在心态上只关注成功、禁止自我否定；在技巧上每天获取一组完整的后勤信息；并使用三次“爸爸回家了”作为开场白。
