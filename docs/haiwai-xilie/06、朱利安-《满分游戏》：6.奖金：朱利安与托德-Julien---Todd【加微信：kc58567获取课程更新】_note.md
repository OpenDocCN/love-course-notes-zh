# 满分游戏：06：奖金：朱利安与托德 🎯

在本节课中，我们将学习如何批判性地思考约会技巧，超越非黑即白的教条，理解其背后的核心原理与适用情境。我们将探讨如何建立自己的认知模型，并通过科学实验的方法，将知识内化为有效的行动。

我所看到的一切，都会与我已有的世界模型联系起来。我的世界模型基于我的个人经验、阅读、观察以及学生的反馈，包括过去有效和无效的方法。因此，当我获得新信息时，我不会毫无背景地盲目看待它。我会将其置于我的模型中，并问自己：根据我现有的模型，这为什么有效？同时，我也会问自己：这违反了我现有的模型吗？因为大多数时候，新信息都能在我已有的良好模型中解释。但偶尔，你会看到一些违反你模型的东西，这时你应该感到兴奋，因为这是你成长的机会。我总是将信息置于背景中思考。

例如，在过去“罐头式”游戏是常态时，“打压”是一个重要概念。人们普遍认为它的作用是降低对方的价值。但我将其置于背景中分析，发现人们使用“打压”的方式非常情境化。它确实涉及自尊层面，但另一个作用是让互动与女孩相关，让她有机会进入自己的思绪。我意识到，有很多其他方法也能达到让她进入自己思绪的效果，而不必使用“打压”。于是我开始在别人使用“打压”的情境中，试验使用这些替代方法。我理解了“打压”的真正目的，获得了更深层次的理解。如果我当时只是盲目模仿朱利安、泰勒或谜男过去的做法，我就会毫无章法地尝试，希望它能奏效。如果你不科学地做事，你永远只能猜测。但当你理解了“为什么”，你的行动就会像投掷飞镖一样精准，而不是像扔水球。

关键在于，每当我看到别人在做某件事或阐述某个原则时，我不会只是重复它。我会问自己为什么，问自己这代表了什么核心原理，以及他们在什么背景下说这些话。我想补充的一点是，还要问自己：这如何适用于现在的我？比如，你看到一些现场录像。不同导师的游戏风格可能非常不同，因为我们都在打造自己的游戏。但我们也有很多做法是相同的。不要过分关注差异，这是很多人常犯的错误。要关注相似之处，关注我们都遵循的核心原则或动态，这才是游戏的本质。对你而言，要透过表面看本质，不要试图成为克隆人，那行不通。要理解它为什么有效，看到我们共同在做的事情，然后问自己：我的版本是什么？这就像和女孩打招呼，你的“最近怎么样”是你的版本。在更大的层面上，游戏也是如此，你的游戏版本是什么？通过这些视频，我试图让你思考：如何让游戏成为你自己的？观察我的现场、托德的现场、每个导师的现场，然后思考：什么是我独有的？这些原则如何适用于我？

这与我们即将讨论的“开场白”话题有关。不要非黑即白地思考，要学会看到那些细微差别。接下来，让我们介绍开场白并进行分析。

开场白是我们几分钟前刚讨论过的话题。我最近发布了一些关于开场白的视频，是技术性视频，详细解释了如何规划前30秒，而不是杂乱无章。很多人觉得这很棒，但我也收到一些评论说：“但是朱利安说过开场白不重要。”我想，他可能在某个背景下确实说过。但我敢肯定，如果我问朱利安：“开场白不重要吗？”他不会说“不，当然不重要”。从逻辑层面看，如果你的开场白糟糕到让女孩反感，你甚至没有机会挽回。所以，至少在最低限度上，拥有一个不糟糕的开场白是重要的。除此之外，我相信你也注意到，当你外出时，如果在互动早期得到良好回应，后续一切都会更顺利；如果早期回应不佳，后续就会更困难。这个回应是对什么的回应？是对你的开场白的回应。当然，你可以从一个糟糕的开场白中恢复。当然，仅仅因为得到良好回应，也不意味着你在后续互动中无需努力。但说它“不重要”是愚蠢的。人们总是对这类事情发表绝对言论，比如“外表不重要”、“开场白不重要”、“金钱不重要”。这些说法都很绝对。它是唯一重要的吗？不是。它是整个互动中最重要的吗？也不是。但说它“不重要”是愚蠢的。这是一种非黑即白的思维。

我认为，在游戏和生活中，任何包含“总是”或“从不”的句子都可能是错误的，因为几乎所有事情都处于灰色地带。对于游戏的每个精确答案，几乎都以“看情况”开头。这取决于背景、你、女孩、情境以及无数其他因素。因此，当你从别人那里获得建议时，理解其“为什么”和“来源”至关重要，同时也要理解建议者的个性特点，因为不同的人擅长或不擅长的方面不同。有些人天生在游戏中具有高价值，但很难让女孩感到舒适。这些人在学习过程中会意识到“舒适感”的重要性，并认为“一切都关乎舒适”，因为他们天生就会做高价值的事情。另一些人则天生善于让女孩感到舒适，但不擅长展示价值。他们会认为“游戏全关乎价值”。如果你是一个擅长价值但不擅长舒适的人，却向一个擅长舒适但不擅长价值的人学习，他的课程可能与你需要的完全相反。但如果你能分析他的行为，理解其背景并问自己“为什么”，你仍然可以非常有效地从他那里学习。

这些细微差别是学习游戏的最大障碍之一。很多人会归咎于“是我的长相”、“是我的金钱”。在大多数情况下，阻碍你的更多是你自己的思维僵化。很多问题都源于非黑即白的思考方式。如果你能超越这种思维，看到矛盾背后的统一性，看到细微差别，你就能解决很多问题。

以下是一些例子：

首先是开场白。对于紧张到不知该说什么的人，听到“开场白不重要”可能是一种解脱，让你有勇气行动。但另一方面，细微差别在于：如果你走过去说“嘿”，但女孩没听清，或者你的语气很猥琐，比如“最近怎么样？”，那效果就不好。我曾见过有人这样开场，女孩只是站在那里，毫无反应。他不知道如何传达自己的气场和个性。另一方面，也有人开场时过于亢奋，让女孩感到压力。如果你不能与女孩建立连接，不能向她展示“我是个很酷的人”，她就不会和你交谈。所以，从这两个角度看，两种说法都可以是正确的。但这其中涉及更多背景。

另一个例子是，你听我说“要练习和你不感兴趣的女生搭讪，这样才能搞定辣妹”。但你也可能陷入另一个陷阱：因为害怕而只敢和不吸引你的女生搭讪。所以，另一条建议是：“你必须去和那些辣妹说话！走出去，直接搭讪现场最辣的女生。”这取决于具体情况。

再比如，你需要“鞭策自己”。如果你外出时退缩了，你需要适当鞭策自己，以便第二天更有动力行动。你也需要“自我接纳”。但如果你做得太过，把自己打击到无法行动，那么就需要“不要过分苛责自己”，这反而能激励你采取更多行动。很多人会困惑：到底该不该鞭策自己？很多时候，这取决于常识。通常，你可以鞭策自己到足以采取行动的程度，如果过度了，就引入自我接纳。如果你自我接纳过度，开始懈怠、什么都不做，那又是一种合理化，这时就需要回到鞭策自己。

学会看到这些细微差别，甚至在气场中也是如此。你听说过“提供价值”和“索取价值”。有人会问：你问问题，比如“你叫什么名字”，这算是索取价值还是提供价值？他们看不到其中的细微差别：你问“你叫什么名字”时，你的气场决定了这是索取还是提供。

另一个例子是“不要表现出需求感”。有人会把它和开场联系起来：我听说不能有需求感，但你是主动走过去搭讪女孩的人，这难道不是有需求感吗？他们没有看到，这关乎情感层面的出发点。无法看到这些细微差别，是阻碍很多人的原因。他们执着于或教条地相信某件事，认为“我听说就是这样，永远是这样”，不愿意放下那些僵化的信念，而这正是阻碍你的东西。

同样的情况也适用于我们讨论的技术层面。你听说需要掌握这些技术性内容，需要开始逻辑思考。我们讨论过关于后勤、铺垫等话题。但与此同时，矛盾的是，你也要“享受当下，不要考虑未来”。很多人会问：到底是哪一个？你必须学会超越非黑即白的思维，驾驭这些矛盾。

一个很好的例子是“给女孩买饮料”。在搭讪社群中，曾有一条规则：永远不要给女孩买饮料。这个想法源于“不要讨好”。不要做那种试图通过买饮料来换取女孩好感的需求感强的人，而是要相信你本身就足够有吸引力。这是因为很多刚进入社群的人过去太“好”了，他们受电影影响，认为搭讪就是给女孩买杯酒。这完全将你置于“供养者”框架，建立了一种你在追求她、她更有价值的情境。所以人们想摆脱这种框架，于是有了“永远不要给女孩买饮料”的想法。然而，我告诉你，当我想带走一个女孩时，最有效的事情之一就是和她一起去买杯饮料。原因如下：第一，这是一个将她从朋友中隔离出来的绝佳借口。第二，它建立了一种“你们在一起进行某项活动”的背景和氛围，这非常有益。第三，虽然你不是想灌醉她，但她摄入了一些酒精后，她脑海中的叙事会是“我们在玩乐、在派对，这只是自然而然发生的，我不需要负责”，这让她可以参与某些行为，而不必感到自尊受损或受到评判。所以，有三个很好的理由给女孩买饮料。另一个很好的理由是，在酒吧相遇时，人们通常就在做这件事，融入这种氛围有助于你更协调、更被接受。这并不是说如果你从不喝酒，就应该迫于压力去买饮料。但我想说的是，在很多情境下，这样做有合理的理由。所以说“永远不好”是非常愚蠢的。当然，说“总是很好”也会让你很快变穷。所以这不是“总是”或“从不”，而是一个灰色地带。

我想补充一点，当我听到有人因为拒绝和女孩喝一杯而搞砸了带走机会的故事时，我感到震惊。女孩说：“好啊，和我喝一杯吧。”而他们却说：“不，我拒绝。”这太固执了，死守着规则，没有意识到具体情况，结果搞砸了。这也是同样的概念：要看到潜在的原则。即使是“打压”或“假性时间限制”也是如此。“打压”在过去源于“我需要降低女孩的价值”。如果你从不同角度思考，另一个背景是“戏谑地挑战女孩”。如果出发点不同，完全可以这样做。“假性时间限制”也是如此。过去人们会这样开场：“嘿，我只有两分钟，因为我朋友马上来了。”其背后的假设是“我是个烂人，你只需要忍受我两分钟”。但如果你更实际地使用它，比如当女孩担心“我朋友要来了”时，你可以用它来消除她的顾虑，这样她就能放松下来听你说话，你也能展现个性。如果朋友真的来了，你可以对朋友说：“哦，就两分钟，我们马上回来。”这非常有效。所以关键在于微妙之处、细微差别和出发点。任何事情都没有绝对的“总是这样”或“总是那样”。

你可能会问：我该如何处理这些矛盾？我理解这些矛盾存在，但我该怎么做？我还是很困惑。以下是我要说的几个原则：

第一，愿意实验。我把游戏看作获取性爱的手段，但也将其视为科学，是对真理的探索。在科学中，有“科学方法”。你提出一个关于世界如何运作的假设，测试它，观察结果，然后验证或修正这个假设。如果你在特定情境下不知道该怎么做，你应该根据你的经验和知识储备，尽力而为，并相信“尽力而为”本身就是最好的。接受任何结果，但要分析结果。如果有效，就反复做，并问自己为什么。如果无效，你可能暂时失去机会，但如果你有经验或天赋，或许可以挽回。之后，不要在场上想太多，事后问自己：为什么会这样？这挑战了我对游戏的看法，这是你学习和深化模型的机会。

很多人问我关于技术性游戏的问题是：如何在场上思考所有这些事情？事实上，我在场上并不思考。但我在场外思考很多。因为我头脑中有一个非常丰富的模型，我深信并内化了这个模型，正确的行为就会自然流露。这不是成长模型的问题，而是我不需要思考它。就像一个运动员学习接棒球，你不会思考所有物理原理，你的潜意识会根据以往类似轨迹的棒球经验做出反应，你的脚会跑到正确位置，手会以正确方式抬起，你信任你的肌肉记忆。在游戏中也是如此：你通过思考来增强肌肉记忆和理解，内化更多概念，但当你真正外出时，你信任你的肌肉记忆，尽力而为。大多数时候，你的肌肉记忆会很好地为你服务。偶尔失效时，你就回到起点，分析原因。如果只是偶尔，可能是运气问题。但如果某种模式反复无效，比如连续5、6次，那就是你的问题了。让我试试这个，试试那个，找到解决方案。因为你通过系统的方法找到了解决方案，你就永远解决了这个瓶颈。这就是以技术性、科学的方式做事的好处。但话说回来，你不需要在场上一直思考，不需要在女孩面前做微积分。

分析场外技术细节时，不仅仅是看互动的表面。要审视每一个变量：我说这话时感觉如何？她当时感觉如何？出发点是什么？是晚上什么时间？她的情况如何？和朋友一起吗？在酒吧还是俱乐部？她是哪种类型的女孩？是更回应强势行为还是温和行为？是更极端还是更冷静？所有这些小变量都需要你关注。随着时间的推移，分析每一个参考经验，最大化其价值，这就会成为你的一部分，在场上自然处理。这不仅仅是“我说了这个，没用；我说了那个，有用”。你需要深入审视每一个细节。很少仅仅是话语本身，而是整个一揽子因素。

你必须拥有这种心态。如果你不在场外进行分析，你就不会进步。另一个与这些矛盾相关的是“意图”与“对结果无执念”。我听到有人说“对结果无执念”，然后就变得无所谓，甚至不打算得到女孩。对那个人来说，他更需要的是“执着于结果”，去追求女孩。但同时，这不能源于“我需要她来完整我自己”的心态。这是“计划去赢”（意图得到女孩）和“我需要她来完整自己”之间的区别。或者“自我娱乐”：有些人会走到女孩面前做疯狂的事，女孩跑掉了，他们还说“是啊，自我娱乐”。不，互动中的自我娱乐必须包含女孩，并且导向性。所以，任何你现在认为是矛盾或对立的事情，挑战在于，我通过这个视频告诉你：尝试看到背景，尝试理解两者如何在特定背景下都正确，尝试超越非黑即白的思维。如果你能做到，你的游戏将突飞猛进，也会影响你的分析方式，这非常重要。你不能只是非黑即白，要愿意挑战你现有的每一个信念。如果你现在有坚定的信念，问问自己：是否有相反的证据？另一个重点是：不要试图“固化”游戏。有些客户总想寻找“游戏的宏大统一理论”。没有这样的理论，它是所有不同模型的集合，它们共存，并在不同情境下发挥作用。

关于技术性游戏，我常被问到的另一个问题是：当人们变得技术性时，会有点完美主义。我想告诉你，没有完美的游戏。我从未运行过一个完美的组合，你也永远不会。但好消息是：你不需要。你不需要运行任何接近完美的组合。你可以到处都是瑕疵和错误，但仍然能得到结果，只要你做对了足够多的事情，足以压倒那些瑕疵和错误。所以这不是关于完美。对于游戏中的任何特质，我总是将其视为一个范围。例如，“友善”和“混蛋”之间有一个范围。如果你太友善，你会被践踏，得不到尊重。如果你太混蛋，女孩无法与你产生共鸣，不觉得自己特别，她会为了自身安全和自尊而拒绝你。但在“太友善”和“太混蛋”之间有一个很宽的范围，你在这个范围内的任何位置都可以。这些特质的范围相当宽。实际上，这就是为什么你会看到不同风格的原因。你可能看到一个人处于范围中更混蛋的部分，另一个人处于更甜美、友善的部分。我们可能处于范围的不同位置，但关键是我们都轻松地处于有效范围内。如果我们任何一个人变得太友善或太混蛋，都不会奏效。同样的情况也适用于我们对女孩的关注和投入程度。如果我们太关注女孩，就会显得需求感强。如果我们太冷漠，女孩会认为她没有机会，从而失去兴趣或放弃。我们可能处于这个光谱的不同位置，但我们都处于有效范围内。你会发现每个人的游戏风格都在那个有效范围内。在你自己的游戏中，你会发现当你偏离有效范围时，就会失败。好处是，当你做出调整时，你不需要调整到完美，只需要调整到大致在范围内的某个位置即可。

这也给了你灵活性和自由。如果追求完美，你会压力巨大。只需瞄准那个范围，找到它，测试它。很多这类的判断其实是感性的。你怎么知道临界点在哪里？通过试错。你需要培养对每一个细微部分的感觉。就像学习平衡：我要向右倾斜多少才会摔倒？试试看。不要让任何事情阻碍你，挑战每一条建议。再举个例子，我谈到夜晚的节奏：在夜晚的前半段，你很少能带走女孩，通常是收集号码、安排后勤、积累线索。然后有人遇到了一个购买温度很高、已经准备好的女孩，却因为“时间还早”而放弃。不，游戏的每一条规则，都要愿意打破它。这些规则、想法，正如你所说，只是指导方针，给你方向。不要让它们成为你的枷锁，让这些指导方针为你服务。同样，关于夜晚的时间：要理解“为什么”。为什么夜晚的前半段不容易带走？这是关于购买温度的视角。但如果你在晚上10点遇到一个从下午2点就开始喝酒派对的女孩，在她的世界里，已经是“凌晨4点”的模式了，即使俱乐部还是“晚上10点”的模式。所以要问自己为什么。或者，有些女孩就是会立刻答应，虽然不是大多数，但你会遇到一些。所以，理解潜在原则并应用，你就能在当下调用所需的工具。有时你需要这个工具，有时那个工具可能是最愚蠢的选择。有所有这些变量，去实践，去尝试。

最后我想总结的是，在游戏中，真正重要的是那些精彩的经历和故事，当你回顾旅程时，你会觉得“太棒了，这就是我投身游戏的原因”。如果你能分享一个近期或总体上让你觉得“这就是为什么我在游戏中”的史诗般经历。

对我来说，最好的经历是当技能遇到机会的时候。我和很多辣妹在一起过，那些总是很棒。拥有美好的关系，总是很棒。但我真正喜欢的是，当你学到新东西并立即应用它，或者遇到以前没遇到过的挑战并能够克服它。我最喜欢的经历之一是在拉斯维加斯。简单来说，我甚至还没打算开始我的夜晚，结果就搭讪了一个女孩，让她带我进俱乐部。她开始为此找茬。这个组合的好处在于，它在某种程度上是“意外开场”。所以从互动的第一刻起，我就处于可能最高的价值位置。我说：“算了，我找别人吧。”然后她就开始追我。我们最终带她进了俱乐部，基本上领着她走下楼梯，就这样。我说：“酷，我要去享受俱乐部了。”她说：“什么？你要走了？”我说：“嗯，我们可以喝一杯什么的。”我们喝了酒，然后我的朋友要去另一个俱乐部，我们把她带到另一个俱乐部，碰巧有熟人，所以我们直接插队进了俱乐部。看到她对真正超级高价值的反应，真的很棒。最终我带她回家了，实际上很快，整个组合可能就一个小时，包括换场地，然后带回家完成带走。当我带她回去后，我故意放慢节奏，因为我喜欢带女孩回去后慢慢来，喜欢让女孩来引诱我。这个女孩恳求着上床，这很棒，因为这是一个纯粹基于价值的带走。这对我很重要，因为当我刚开始学习游戏时，我总是太友善，价值不够。所以这是一个非常两极化的例子：当价值足够而舒适感不足时，如果做得恰当，就会这样。同时，她也是我遇到过的最辣的女孩之一。所以这是一个很好的结合：我做了这件事，这是我心目中游戏应有的样子，效果很好，女孩很辣，而且她最终也是一个很棒的女孩。所以这很史诗，因为它把所有东西都结合起来了。不仅仅是她是个辣妹，不仅仅是我做了什么，而是整个经历。这就是你在游戏中的原因。对我来说，性爱很棒，但如果它有意义就更棒了。那次经历对我有意义，因为它象征着我到那个时刻的成长。

所以，要超越矛盾，看到细微差别，聪明地思考，让游戏为你服务。

最后还有什么要说的吗？

是的，就是思考。为自己思考。不要在场上思考，在场上就采取行动。但不要只看表面价值。如果我说了什么与你的经验相悖，或者朱利安说了什么与你的经验相悖，首先相信你自己的经验，胜过相信我们说的。然而，可以将我们视为权威，尝试一些东西，看看是否能获得新的经验。但归根结底，你是自己生活的主角，你是必须首先为自己思考的人。所以，永远不要把这个责任交给任何人，永远不要完全非黑即白地思考。
