# 《PIMP 核心笔记》：03：外部技术模块：5. 收场：视频范例：72. 破坏和谐感声调示范+寻求和谐感声调示范 🎤

在本节中，我们将通过具体的视频范例，学习两种关键的声调模式：**破坏和谐感**与**寻求和谐感**。理解并掌握这两种声调的区别，对于建立有效沟通至关重要。

上一节我们介绍了沟通中的核心感概念，本节中我们来看看如何通过具体的声调来实践它。

## 破坏和谐感声调示范

破坏和谐感的讲话方式，其核心在于通过声调传递出主导与自信，而非寻求对方的认可。这种声调模式的关键特征是句尾音调**向下降**，并带有一种直接的、略带命令感的浅沟通语气。

以下是破坏和谐感声调的具体示范与要点：

*   **核心特征**：句尾音调下沉，不拖长音。
*   **语气**：直接、平稳、略带命令感，而非询问或讨好。
*   **错误示范对比**：避免使用上扬、轻快的“寻求认可”式语调。

例如，当你想认识某人时，正确的破坏和谐感说法是：
> “嘿，等一下。你叫什么名字？”
> “嘿，等一下，我想认识你。”

这里的“嘿”发音短促，句尾的“名字”、“你”音调自然下降，陈述事实而非提问。

## 寻求和谐感声调示范

与破坏和谐感相反，寻求和谐感的讲话方式旨在通过声调迎合对方，以获取好感或认可。这种模式通常表现为句尾音调**上扬**，语气显得犹豫、友好或过于热情。

以下是寻求和谐感声调的典型表现，请注意其与上一种模式的对比：

*   **核心特征**：句尾音调上扬，类似提问。
*   **语气**：友好、试探、充满期待，隐含寻求对方回应的意味。
*   **效果**：这会削弱自身的主导地位，将互动的控制权交给对方。

例如，同样是想认识某人，寻求和谐感的错误说法是：
> “嘿嘿，你好！你叫什么名字呀？”

这里的“嘿嘿”语气词轻快，“呀”字音调上扬，整个句子听起来像是在小心翼翼地请求对方互动，从而**建立了和谐感**，而非破坏它。

因此，在需要展现主导性的沟通场景中，你说话的目标应是**破坏和谐感**，而非寻求它。

本节课中我们一起学习了两种核心的沟通声调：破坏和谐感与寻求和谐感。关键在于记住：**破坏和谐感 = 句尾降调 + 直接陈述**；**寻求和谐感 = 句尾升调 + 试探询问**。通过有意识地控制句尾音调，你可以有效地管理沟通中的气场与主导权。
