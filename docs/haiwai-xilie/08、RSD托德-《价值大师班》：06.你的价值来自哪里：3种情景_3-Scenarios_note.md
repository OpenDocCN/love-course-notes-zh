# 价值大师班：06：你的价值来自哪里：3种情景

在本节课中，我们将探讨如何将“高价值”的核心概念应用于三个具体的生活场景。我们将学习如何通过理解他人需求、创造共赢局面来提升自身价值，而非进行零和博弈。

上一节我们探讨了高价值的内在逻辑，本节中我们来看看如何将这些原则应用到实际生活中。我们为你准备了一些情景挑战，现在分析其中几个，后续课程中还会涉及其他有趣的情景。核心问题是：如何将社交中的高价值理念应用到不同场景中，或者决定不应用（因为某些技巧，例如调情或推拉，并不适合用在老板身上）？关键在于如何将这些技能因地制宜地运用。

以下是三个具体情景的分析，我将简要阐述如何以高价值的方式处理。有些方法可能直观明显，有些则未必。

## 情景一：谈判加薪

谈判加薪是一个非常有趣的场景。它与街头搭讪的一个主要区别在于，你已与对方建立了既定关系。既然你与老板或公司已有关系基础，许多准备工作应在正式谈判前就已完成。你无法在工作表现糟糕、老板几乎想解雇你时，仅凭高超的谈判技巧就成功加薪。与他人关系越久（无论是商务还是私人关系），对方对你的评价就越基于你的实际表现，而非社交触发点。

回想昨天的比喻：在任何情境中，你只注意到某些元素。认识某人越久，你注意到的情境元素就越全面，印象也就越整体。因此，谈判加薪的成功很大程度上取决于事前铺垫，你至少需要在谈判时获得大致积极的评价。如果老板已经对你生气或不满，此时谈判加薪将难以成功。所以，第一点是提前做好基础工作。

如果我要去谈判加薪，方法有好坏之分。直接说“我觉得我需要加薪”是一种方式。如果你工作表现极其出色，或因角色大幅扩展而薪酬明显过低，这种方式可能有效。但即使有效，也可能无法最大化成果。在确定能加薪的情况下，通常有更有效的方法来争取更好的条件。

这里引出一个对许多场景都关键的概念：**首先了解对方的蓝图或价值观**。例如，如果你知道自己工作出色，可以先询问老板对你工作的评价，探讨你还能在哪些方面提供帮助。然后，你可以将这些“额外帮助”与加薪联系起来。但需谨慎，这可能让你承担更多责任却未获加薪。

更好的方法是：你可以说：“我想聊聊，看看如何能在这里提供更多帮助。我想了解，作为公司管理者，您认为最重要的优先事项是什么？如果实现哪些目标会对公司产生最大影响？我希望把精力用在最关键的地方。” 这样，你引导老板谈论他的愿景和目标，在某种意义上是在向他“推销”这个愿景。然后你可以表示愿意协助，并讨论所需的额外资源，例如：“显然我需要投入更多时间，因此需要相应的补偿”或者“可能需要他人接手我部分现有职责”。关键思路是：**首先找出对方的优先事项，了解他们看重什么**，因为他们的价值观可能与你不同。

我们稍后会讨论“共赢”而非“零和”，以及“非零和博弈”的概念。如果你直接对老板说“我要加薪，你要付我更多钱”，你就设定了一个零和场景：你每多得一块钱，他就损失一块钱。这是达成目标的糟糕方式。相反，你应该设定一个让他觉得“给你更多就是为自己得到更多”的场景，即共赢局面。这样你才有可能获得更大的加薪空间。

核心要点是：**首先找出对方想要什么**。价值是主观的，了解什么对他们有价值是关键。

## 情景二：在朋友派对上

这是一个社交圈场景，你可能想成为高地位者，或在派对上认识女孩。同样，我想将其与街头搭讪区分开。街头搭讪有一些限制和前提：其一，如果你没给对方留下印象，你可能再也见不到他们，这在朋友派对上不成立；其二，你可以不计后果地行动，因为搞砸了也不会再见面，这在朋友派对上也不成立。

因此，在这些更社交化的情境中，你需要更间接地表达需求。在直接追求目标前，应先提供普遍价值。例如，如果你在朋友派对上看到一个美女，直接只跟她说话而忽略其他所有人（包括邀请你的朋友），这将是非常糟糕的策略。原因如下：一、你会冒犯某些人；二、这对你的朋友是种冒犯；三、这会显得目的性太强、非常明显，别人能一眼看穿。他们甚至可能告诉那个女孩，或当众让你难堪，从而破坏你的机会，尤其如果这个女孩很受欢迎。

所以，超级直接可能不是好主意。同样，对不同的人使用相同的套路台词也可能是坏主意。在所有人面前制造大场面，可能也是坏主意。在派对场景中，假设派对将持续数小时，你的目标应该是：**在有利条件下被介绍给你想认识的人**（无论是女孩、社交关系还是商业伙伴）。这可能发生，也可能不发生，但你从一开始就应以此为目标。

你应该先与主办人、邀请你的人交谈，认识他们正在交谈的人，通过介绍建立你的社交阵列，建立社交联系，并享受派对乐趣。如果你能在你想认识的人附近这样做，那就更好了。你希望以某种方式为派对提供价值，带来乐趣，产生积极影响，并逐渐接近你最终想进入的圈子。

如果实在无法建立联系，那情况可能变得更像街头搭讪。但即便如此，由于在派对上，你们仍有更多共同点，有更多空间进行随意交谈，你不必那么快速行动，也不必像在其他场景中那样展示强烈的“男女前提”。如果你们有共同朋友并成为朋友，你很有机会安排再次见面，而不必让一次互动承受太大压力。

最后，如果在朋友派对上与某人（无论是女孩还是你想建立更紧密联系的人）进展顺利，你应设法稍微远离派对，进行一对一交谈或相处，例如去后院散步或一起拿杯饮料。尤其是如果你会反复参加同一群人的派对，你肯定不希望公然在派对上升级关系，以免给自己带来负面声誉。

朋友派对的要点是：**放慢节奏，享受与已有熟人的对话和氛围**。顺便说一句，在朋友派对上，你不必一定要认识新朋友。你可以有目标，也可以单纯享受派对。这完全可以。我之所以讨论在派对上有目标或想认识某人，是因为那是困难的情景。如果只想享受派对，直接去享受就好。许多学习社交或搭讪的人认为永远不能那样做，但绝对可以。事实上，这也能为未来积累价值。总之，不要在公开场合过度升级关系，不要成为派对上“那个人”。

## 情景三：社交网络

这是一个类似的概念。我真正想问的是：你是在专门的社交活动上，还是在一般场合进行社交？如果你在专门的社交活动上，人们会有更多相互交流的预期，他们更愿意进行简短互动，除非有非常好的联系，否则不希望独占他们的时间。而如果你在普通活动中社交，情况会有点像朋友派对，你只是试图建立社交联系并展开对话。

可以将社交网络想象成**没有“男女前提”的搭讪**（实际上前提不同，但类似）。要非常清楚和意识到场景。最重要的是采取行动，尤其是在专为社交设计的活动中。在这种情况下，你能做的最重要的事就是去做：与人交谈，不要害羞，开始对话。因为大多数人都在四处走动，希望自己能与人开始对话，并感到害羞、乏味和尴尬。所以，即使只是说声“嗨”，你可能也在帮助他们。

## 情景四：讨论开放式关系

关于此话题有一个很棒的免费资源，我先推荐它，然后再补充几点。这个免费资源是YouTube视频“关系宣言”，其中有一整节关于讨论开放式关系的内容。

但就本次演讲与价值相关的要点，我想说明：当讨论关系性质（无论是开放式关系还是任何关系）时，你**可以选择时机**。你可以选择从一开始就明确：“我们是什么关系？是男女朋友吗？” 你也可以选择只是做一个高价值的人，给对方良好的体验，并让关系保持大体未定义的状态。通常，你越是能保持高价值并让关系未定义，对方最终越可能因为感觉与你更亲密、想要更多而主动找你。这意味着你将以最佳状态进入那次对话，对方对你的好感度达到最高。因此，比起在不利条件下试图成为主动定义关系的一方，你的处境会好得多。

关于开放式关系及此类话题的完整演讲，我不想在这里展开，因为它需要较长时间，并且可以在YouTube上免费获取。请查看“关系宣言”演讲。

但我想在此提及更普遍适用的一点：**在任何定义关系、条款或进行谈判的情境中，事前铺垫非常重要**。谁找谁谈、对话的框架如何设定，这类事情都很重要。请关注这一点，并确保谈判是在你自己的条件下开始的。

本节课中，我们一起学习了如何将高价值原则应用于谈判加薪、朋友派对、社交网络及关系讨论等具体场景。核心在于：**首先理解对方的需求与价值观，致力于创造共赢局面，并根据不同社交情境的规则灵活调整策略**。记住，价值是主观的，真正的力量来自于让对方觉得满足你的需求也符合他们的利益。
