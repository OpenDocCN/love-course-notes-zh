# 大卫X规则：17：一致性

## 概述
在本节课中，我们将学习大卫X提出的两个核心规则。这两个规则旨在帮助你建立自信，并在与女性互动时保持一致性。我们将深入探讨每个规则的含义，并通过简单的比喻和例子帮助你理解如何应用它们。

## 规则一：别在意她的想法

上一节我们介绍了课程概述，本节中我们来看看第一个核心规则。

规则一的表述很简单：**别在意她的想法**。这并不意味着对女性不尊重或抱有贬低态度。它的核心含义是：**不要试图替她思考**。

你无法知道她内心的真实想法。如果你总是试图猜测她的喜好或反应，你很可能犯错。你已经有足够多的事情需要为自己思考了。

以下是忘记这条规则时常见的错误做法：
*   你会想：“我要这么说，因为她会喜欢。”
*   你会想：“我不能这么做，因为她会不喜欢。”
*   你的行为会基于你猜测的她的想法，而不是你自己的真实意愿。

据统计，90%的互动问题都源于你忘记了这条规则。你不再关注其他一切，而是陷入了猜测她想法并试图预判的循环中。

## 规则二：你才是最重要的

理解了不要替女性思考后，我们来看看第二个规则，它甚至更简单。我们会经常回溯到规则一，以确保不会忘记它。

规则二是：**你，你，你，你才是关系中最重要的那个人**。

因为你是那个需要主导关系方向的人。你的意愿很重要。在开始任何指导之前，我都会告诉学生：你必须先决定自己想要什么。

你今天想要的东西明天可能会变，这没有错。我自己的目标就变过很多次。但你必须从今天的目标开始，并且保持专注。如果你只是想出去约会，那很好，但请保持专注，不要轻易改变。

没有人能影响你真正想要什么。这一点至关重要。我敢保证，如果我问你们任何一个人是做什么工作的，你们很可能非常擅长自己的工作，并且在工作中自然而然地践行着我所说的原则。但一涉及到女性，你就关闭了这种思维，觉得事情变难了。

## 克服无谓的恐惧

我们讨论了以自我意愿为中心的重要性，但很多人却因为恐惧而无法行动。这种恐惧从何而来？

想象一下，一个女性可能花了两三周挑选衣服，花了几个小时化妆，就是为了出门遇见像你这样的人。而你却在害怕她。这说不通。

你无法害怕一个实际上并不可怕的事物。如果她真是武术专家，你可以去报个班；如果她是火箭科学家，你可以寻找其他女性。但你究竟在害怕什么？这种恐惧源于你自身。

## 回归简单：像买咖啡一样思考

为了理清思路，让我们做一个简单的类比。

当你去买一杯咖啡时，你会对店员说：“请给我一杯咖啡。”你不会想：“他会不会对我说‘滚出去’？”不，你根本不会考虑他会说什么。你只是想要一杯咖啡。

同样的事情你可以做上千次，但一涉及到女性，一切都变了。我不是要你成为我，你无法成为我，你必须做你自己。我给你的不是一套需要死记硬背的“大卫X语录”。这是一件会让你感到轻松的事：停止思考她希望你做什么或想什么。

你是一个自由的个体。你知道自己人生的方向，也知道你希望它走向何方。不能仅仅因为遇到了一个女性就改变这一切。你必须保持专注，清楚自己想要什么：是想要一个女朋友，还是想认识一位不错的女性，是想带她回家，还是享受彼此的陪伴。

## 平等的关系观

保持自我框架的同时，也需要建立健康的关系认知。

她并没有给你任何别人无法给予的东西。这不是一份“礼物”，并非如此珍贵。仅此而已。世界上有成千上万的女性。

## 轮胎的故事：过度思考的陷阱

为了说明“不要猜测他人想法”这一点，我想讲一个关于轮胎的故事。

两个男人在乡间开车，天气很热。突然，“砰”一声，轮胎爆了。他把车停到路边，打开后备箱，拿出备胎——发现备胎也是瘪的。原来他几周前爆胎后就没修备胎，只是把它扔回了后备箱。

于是他开始沿着路把备胎滚向加油站。他一边滚，一边胡思乱想：“也许加油站今天没开门？”“也许他不补胎？”“我已经看不到我的车了，也许我该用另一个轮胎？”“也许这个轮胎修不好？”他想了各种可能，天气又热，他越想越焦虑。

终于到了修车厂，店员走出来说：“先生，需要帮忙吗？”这个男人说：“**拿着这该死的轮胎，塞进你的屁股里吧！**”

你明白了吗？这是一回事。在到达之前，你根本不知道对方会说什么。你无法知道他们在想什么。你只能思考你自己的事情。

生活没有其他方式。当你去申请工作时，你尽可能打扮得体。面试官开始和你交谈，你逐渐放松。你会想他希望你说什么吗？你会告诉他关于你自己的真实情况。然后由他决定是否录用你。与女性交流也是同样的道理。

沟通，尤其是真诚的自我表达，至关重要。

## 总结
本节课中我们一起学习了大卫X的两个核心规则：
1.  **别在意她的想法**：不要试图替女性思考，你无法知道她的心思，专注于自己的想法和行动。
2.  **你才是最重要的**：在互动和关系中，明确并坚持自己的意愿和目标，你是主导者。

关键在于停止无谓的猜测和恐惧，像处理生活中其他简单事务（如买咖啡）一样去行动和沟通。保持真诚，专注于自身，才能建立自然、平等的关系。
