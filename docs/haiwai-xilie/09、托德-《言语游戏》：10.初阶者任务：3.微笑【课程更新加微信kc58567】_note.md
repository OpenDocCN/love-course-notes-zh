# 言语游戏：10.初阶者任务：3.微笑 😊

在本节课中，我们将学习第二个任务：在保持眼神接触的基础上，增加微笑。这个练习旨在帮助你以积极的方式与他人建立初步的、非言语的互动。

上一节我们介绍了如何与他人保持眼神接触。本节中，我们将在眼神接触的基础上增加一个关键元素：微笑。

## 任务核心：传递积极能量

这个任务的核心是**向世界传递积极能量**。关键在于你的付出，而非对方的回应。无论对方是否回以微笑，只要你完成了眼神接触并微笑，你就已经为世界增添了积极因素，任务即告成功。

如果对方也对你微笑，甚至开始与你交谈，那当然非常美妙。但这并非本任务的目标，而是一个额外的奖励。本任务的目标是让你自己感到舒适：**😊，进行眼神接触，微笑，并轻微地“侵入”他人的个人空间**。

## 具体执行步骤

以下是你在日常生活中执行此任务的方法。

*   **寻找机会**：在你处理日常事务时，抓住任何与人接触的机会。
*   **眼神接触**：不要盯着地面，要看向人们的眼睛。
*   **给予微笑**：当对方看向你时，给予一个微笑。即使这个微笑看起来有点刻意、不自然或奇怪，目前也无需担心。
*   **放下自我意识**：不要为此感到难为情。你只是在传递积极能量。
*   **调整心态**：这样想：如果有人因为一个陌生人的微笑而感到被冒犯，那是他们的问题，不是你的。你的任务就是进行眼神接触，并传递出那份积极。

通过这个练习，你将获得与他人进行常规人际互动的体验。这有助于你走出自我封闭的内向世界，在日常中与他人建立简单的、积极的连接。

本节课中，我们一起学习了如何通过“眼神接触加微笑”的组合，以积极、非侵入性的方式开启与他人的互动。记住，重点在于你主动传递积极能量的行动本身，而非对方的反应。请在日常中多加练习，逐渐适应这种与人连接的方式。
