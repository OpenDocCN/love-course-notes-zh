# 《PIMP 核心笔记》：3.3：上钩模块 - 上帝之手技巧

在本节课中，我们将要学习一种名为“上帝之手”的实用技巧，用于在社交互动中邀请坐着的女孩站起来交谈。这种方法的核心在于通过一个简单、自信的肢体动作发出邀请。

## 技巧概述与命名

上一节我们介绍了上钩模块的基本概念，本节中我们来看看一个具体的肢体引导技巧——“上帝之手”。这个技巧的核心是，当目标女孩坐着时，你主动走过去并向她伸出手。这个伸手的动作被赋予了“上帝之手”的名称，它传达出一种不容置疑的、带领性的姿态，仿佛在无声地说：“过来。”

## 执行步骤详解

以下是“上帝之手”技巧的两个关键执行步骤。

**步骤一：伸出邀请之手**
*   你走向坐着的女孩。
*   你直接向她伸出手，同时配合语言邀请，例如说：“嗨。”

**步骤二：引导站立与转换**
*   如果女孩服从，将手放在你的手上，你便轻轻抬起手臂，将她拉起来。
*   在引导过程中，可以使用语言进行安抚和鼓励，例如说：“来，我们站起来一下，就10秒钟而已。” 或者说：“站起来就10秒钟，就一下下。”
*   一旦女孩成功站起来，你们便可以自然地转为站立姿势进行交流。

## 技巧原理与总结

之所以在女孩站起来后可以顺畅地继续交谈，是因为此时双方的“舒适度”达到了同一水平。站立交谈消除了坐立的高度差，创造了更平等、更便于互动的物理空间。

本节课中我们一起学习了“上帝之手”这一肢体引导技巧。我们了解了其自信直接的邀请方式、具体的执行步骤，以及通过统一站立姿态来平衡双方舒适度的基本原理。掌握这个简单的动作，可以帮助你更有效地发起互动并建立连接。
