# 老板 BOSS：01：简介 - 一个平凡的世界 🚪

在本节课中，我们将要学习如何以正确的心态开启这段旅程。课程的核心在于引导你从“知道”走向“做到”，并准备好迎接彻底的自我升级。

---

你终于来到了这里，迈出了第一步，抵达了门前。现在的问题是，在你踏入这扇门之前，你准备好了吗？你准备好踏入未知了吗？我觉得你和我有很多共同之处。即使我不认识你，我也觉得我们有很多共同之处。因为我为你制作了这一切，这一切都是为了你。同时，这也是为我自己制作的。

我思考过，当我还是一个年轻的男性开发者时，我想要什么。我思考过，当我还是一个年轻的男性开发者时，我缺少什么。我思考过，那些从来没有人告诉过我的该死的事情。你知道，生活中有一些事情你知道你需要做。你知道你应该去上学，你知道你应该偶尔给奶奶打个电话，你知道你应该对陌生人友善，你知道你应该刷牙，你知道你应该吃蔬菜，你知道所有你应该做的事情。

但生活中还有一大堆你甚至不知道你应该做的事情。你甚至不知道你不知道什么，这很疯狂，对吧？有时我们在街上，或者在直播时，人们会来找我们，他们非常感激我们，眼睛闪闪发亮，有时甚至热泪盈眶，声音里充满热情。他们说，伙计，你极大地改变了我的生活。我问为什么，他们说，因为你告诉了我需要听到的话，你告诉了我别人不会告诉我的事情。这就是我想在这里为你做的，老板。我想确切地告诉你别人不会告诉你的事情，我想给你真相。

当我制作这个课程时，我对自己说，这一切都是为了你，这个我唯一关心的人。即使你是唯一一个选择这个课程的人，这也是值得的，我会很高兴。如果你是这个课程的唯一使用者，而其他人只是随便浏览一下，但你却带着真诚和感激去做，感激我为了获取这些信息所付出的挖掘泥土、跪地爬行的努力，并且你如此感激地应用它，你真的应用了你听到和看到的一切，那么这一切就是值得的。

如果你观看视频，看到评论里可能有一些笔记，但你要做你自己的笔记。做你自己的笔记。那么对我来说，这就是值得的。我认为人们开始我们的课程时总是充满热情，他们进入一个课程时会想，就是它了，我很高兴，现在课程会完成所有工作。但本质上，我认为这并不会真的发生。我认为，**由你决定这个课程能为你做什么**。这由你决定，没有其他人能为你决定。

所以，如何从这个课程中获得最大收益？你需要绝对认真地对待它。你进入课程时，不要像在教室后排那样，只是放松地看着。你要带着奉献精神来，带着专注的练习来。你不仅仅是练习技巧，这对你来说是一种情感上的练习。我希望这门课程对你来说是一次情感上的冥想。所以，当你想观看时，关掉手机，关掉电脑上的Wi-Fi（实际上你需要Wi-Fi来观看），但关掉手机上的Wi-Fi，对吧？走进一个房间，关上门。这是你的时间，只属于你，没有别人。你独自坐在那里，与自己安静相处，也许这是一天中第一次你与自己独处。

你观看视频，我与你交谈，你与我交谈，我们一起对话，我们一起建立关系，我们一起分享，一起成长。你观看这些视频，更多地了解自己。我希望一天中的这个时间不是你害怕去做的时刻，比如“哦，我又得做那个该死的课程了”。我希望你对此感到兴奋，每天带着激情来参加与我的这个环节。我希望你在精神上做好准备，以一种好的方式感到渴望和兴奋，想着：就是现在，这是我成长的时刻，这是一天中只属于我的时刻。

在这一天中，你放下所有的焦虑，那些让你压力山大的人，你必须完成的那篇论文，你老板给你的那个超出你职责范围的工作项目（但为了升职你还是会去做）。为什么不把这些都暂时放在一边，和我一起待在这里呢？与我融为一体，与我同在，与我分享你自己，因为我会把一切都分享给你，伙计。我会给你我拥有的一切，我会打开每一扇门，不留任何未解决的缝隙，我不会保留任何信息，我会给你一切。

所以，如果我给了你一切，你就再也没有借口了。就是这样。你绝对没有任何借口了，这就是终点线，你打算怎么做？你他妈到底打算怎么做？你是要继续维持现状？还是在这里为自己划清界限？

当我开始一个训练营项目时，我通常只告诉学生两件事：**责任**和**担当**。责任是知道你应该做什么。担当是**无论如何都要去做**，就是去做。所以，这是你的时刻，这是你现在的时刻，这是你历史上的时刻。这是你生命中可以达到关键点的时刻，问自己：我是想继续做知道该做什么的“负责任”的事，还是想走上“担当”的道路，真正去做我知道应该做的事？

这个课程不仅仅是关于更多信息，它不是关于更多信息。你已经有了海量的信息，你已经拥有了所有你他妈需要的信息。现在我们要调整它，现在我们要提炼它，现在我们要让你成为最好的版本。我们将让你成为**2.0版本**。不是麦迪森的2.0，是**你的2.0**。

为此，你不能再是原来的你了。你必须升级你的思维、你的哲学、你的生活方式、你的观点、你的心态、你的范式，一切都必须改变。如果你为此准备好了，我们可以一起做到。你不会孤单，我会在这里陪着你，支持你，想着你，即使我们从未谋面，我也会想着你，想着这对你意味着什么，想着我能为你多做些什么。好了，我们可以做到，你在这里，你已经抵达。让我们开始吧。

---

本节课中，我们一起学习了开启这段旅程所需的心态准备。核心在于理解**责任**（知道该做什么）与**担当**（付诸行动）的区别，并承诺以专注、投入和开放的心态参与课程，目标是实现个人从内到外的全面升级，成为**2.0版本的自己**。记住，课程的价值取决于你的应用。
