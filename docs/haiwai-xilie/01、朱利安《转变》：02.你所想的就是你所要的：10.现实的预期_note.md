# 朱利安《转变》：02：你所想的就是你所要的：10：现实的预期 🎯

在本节课中，我们将要学习关于动机的一个重要心态：保持现实。我们将探讨如何诚实地评估自己的现状，设定切合实际的目标，并理解改变所需的时间。保持现实是避免挫败感、维持长期动力的关键。

## 诚实地评估你的现状

上一节我们讨论了动机的重要性，本节中我们来看看如何设定现实的预期。一切始于诚实地评估你当前的技能水平。

你需要不带偏见地审视自己。放下你的自尊心，停止将自己视为受害者或高人一等。只需实事求是地问自己：在吸引女性方面，我处于什么水平？是新手、有一定经验的中级者，还是已经比较熟练？

以下是进行自我评估的步骤：

*   **放下偏见滤镜**：停止自我欺骗和合理化。
*   **评估你的强项与弱点**：诚实地看待自己的意志力、过往的成功经历和不足之处。
*   **接受事实**：即使真相可能伤害你的自尊，也要接受它。例如，如果你在这方面表现不佳，就承认“我目前在这方面很糟糕”。只有接受事实，你才有改变的材料。

**公式：自我评估 = 放下偏见 + 诚实审视（强项 & 弱点）**

承认自己“很糟糕”并不可耻，这反而是强大的动力来源。从最底层开始意味着你没有什么可失去的，任何进步都是向上的。记住这句话：**没有人第一天走进健身房就是肌肉发达的**。每个人都是从新手开始的。比较一个两岁孩子和一个五岁孩子谁“更好”是荒谬的，他们只是处于成长旅程的不同阶段。同样，技能水平的不同不代表人格的优劣。

## 设定现实的期望与目标

在诚实地了解自己的起点后，下一步是设定与之匹配的现实期望和目标。不切实际的期望是导致沮丧和放弃的主要原因。

许多初学者在接触相关知识后，期望自己能够立刻像高手一样“大杀四方”，轻松吸引最漂亮的女性。这就像从未打过篮球的人，看了几个教学视频后就期望自己能像乔丹一样扣篮。当现实与这种幻想不符时，他们就会变得消极、沮丧并失去动力。

避免这种情况的关键是：

*   **不要进行不公平的比较**：不要将自己与那些已经在此领域深耕多年的人（比如教学者本人）比较。他们的“高光时刻”视频是经过剪辑的，并非互动的常态，就像电影里的洛杉矶比现实更光鲜一样。
*   **不要被“高光时刻”吓倒**：这些片段旨在激励，而非展示全过程。真正的进步源于枯燥、重复的练习。
*   **专注于过程本身**：成功来源于日复一日的基础练习，而不是对最终结果的幻想。

一个生动的例子是，一位28岁从未接触过女性的学员，在第一天的实践中表现远超预期，与多位女性进行了愉快的互动。然而第二天他却崩溃大哭，因为他认为自己“失败”了，理由是他还没有“用手指或与女性发生关系”。他的期望完全脱离了自己的起点，进行了不公平的比较。

**核心思想：现实的期望导向现实的目标。**

## 如何设定并达成目标

理解了设定现实期望的重要性后，我们来具体看看如何制定可执行的目标。关键在于将大目标分解为可持续完成的小步骤。

你需要将挑战与自身能力相匹配。就像健身，你不可能一开始就举起最重的重量，那样只会拉伤肌肉。你必须“小块分解”你的目标。

以下是设定目标的方法：

1.  **确立“首要目标”**：这些是简单、线性、在你当前技能水平内可达成的小目标。例如：
    *   向一位女性说“嗨”。
    *   进行一次30秒的对话。
    *   进行一次更长的互动。
    *   要一个电话号码。
    *   尝试约一次会。
2.  **确立“最终目标”**：这是一个能深刻改变你的远大目标。但要记住，“首要目标”是通往“最终目标”的垫脚石，不要满足于停留在小目标上。
3.  **保持目标简单、清晰、精确**：复杂性是执行力的敌人。把目标写下来有助于使其具体化。
4.  **逐步推进**：一次只专注于下一个“首要目标”，像婴儿学步一样积累动力和力量。**不要追求全垒打，而要追求一连串的安打**。

**行动建议**：现在就去写下你的“首要目标”和“最终目标”，并公之于众（如学习小组），用此来督促自己 accountability。

## 对时间保持现实：这是一场马拉松

最后，我们必须对改变所需的时间保持现实。一个常见的问题是：“我需要多久才能变得擅长？”

答案是：**比你应得的更快，但比你希望的更久。** 有太多变量，无法给出确切时间。更重要的是，**这是一个持续一生的旅程，没有终点线**。你永远不会“足够好”，因为总可以变得更好。无论是单身还是恋爱中，你都可以继续成长，更接地气，发展人格。

不要将这个过程视为达到终点（如“得到某个女孩”）后才能快乐的苦差事。**快乐在于行动本身，在于旅程，而非目的地**。外部奖励（如成功）只是额外收获。你应该为了“做”这件事本身而去做，就像冥想是为了冥想，而不是为了开悟。

从现实角度看，这需要时间，需要**数年**。但不必恐慌，因为你本就打算终身投入。这是一场**马拉松，而非冲刺**。理解这一点也能帮助你在遇到平台期或连续失败时保持动力。

**记住，你无法跳过过程**。要得到你想要的，你必须配得上它。要吸引美丽的女性，你必须成为最佳选择。这就像学习吉他，即使你知道所有步骤，也必须经过漫长练习才能像范·海伦一样弹奏。你必须“配得上”那个水平。

我们的社会充斥着寻找“神奇药丸”和捷径的文化，总想以最小付出获得最大回报。但真正的成长要求你**拥抱艰苦的工作**。你投入得越多，回报就越大。请与自己约定：永不偷工减料，永不寻找借口。建立扎实的基础需要**练习和耐心**。来得快的东西，去得也快。

同时，也无需匆忙。女性不会消失，你也不会。以可持续的、稳定的步伐前进即可。而且，你并非要苦熬数年毫无结果。**拥有未来价值的潜力，与已经拥有即时价值，同样具有吸引力**。只要女性看到你是一个有动力、有潜力、正在向上走的人，她就会被你吸引。
