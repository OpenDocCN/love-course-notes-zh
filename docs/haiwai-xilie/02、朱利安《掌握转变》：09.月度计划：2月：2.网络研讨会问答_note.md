# 掌握转变：09：月度计划：2月：网络研讨会问答 📝

在本节课中，我们将基于内容视频进行补充，探讨如何正确处理内心的愧疚感。我们将学习区分何时需要采取外部行动来弥补过错，以及何时只能进行内部的情感释放。

上一节我们介绍了处理愧疚感的基本心态，本节中我们来看看一个关键的补充原则。

如果条件允许，你仍应尽你所能去纠正错误。我曾在内容视频中提到过这一点。这适用于你因做过某事、未做某事或本应做某事而感到内疚，并且这种情绪正在内心折磨你的场景。你可能在观看内容视频后意识到，自责、后悔和沉溺于过去毫无帮助，并决定放手。这很棒，你必须放手。正如我们所讨论的，**愧疚状态对你毫无益处**。

然而，在某些情况下，你可以选择放手，但如果外部仍有可以采取的纠正措施，你应当采取这些行动，否则愧疚感将持续折磨你。

当他人仍在承受伤害时，你很难真正放下内疚。例如，如果你欺骗了某人、对某人撒谎、从某人那里偷窃或伤害了某人，只要存在可以采取的补救行动，你就应该去行动。

这不仅仅是独自静坐、试图在内心“放下”那么简单。如果你不去采取可用的补救行动，你会发现自己永远在试图“放下”，因为内心有一部分始终知道，你做了后悔的事且本应去弥补。只要可用的补救行动存在而你未采取，这种想法就会不断浮现。

因此，如果存在任何可以采取的行动，就去采取。然后，同时从内心放下它。

当然，有时你可能无能为力，有时为时已晚。在这种情况下，你所能做的就是在内心与自己和解，释放这种情绪。但如果存在外部行动，即使只是道歉，你也应该去做。

例如，仅仅是找到某人并说一句“嘿，对不起”也可能是一种补救。

以下是本月可以实践的一个具体任务：

*   如果你想到任何可能曾被你伤害过的人，去向他们道歉。

本节课中我们一起学习了处理愧疚感的进阶方法：**当存在外部补救可能时，采取行动是内心释放的前提**。我们明确了“采取纠正行动”与“内部情感释放”需要同步进行，并给出了一个简单的实践任务。记住，真正的释怀往往始于负责任的行动。
