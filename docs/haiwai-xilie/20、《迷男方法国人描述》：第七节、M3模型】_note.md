# 迷男方法国人描述：第七节：M3模型

在本节课中，我们将要学习《迷男方法》第四章的核心内容，即正确的游戏规则与架构。我们将重点介绍迷男发明的M3模型，并详细讲解一个完整的恋爱流程，帮助你理解如何在实际互动中应用这些理论。

## 标准化与内化

上一节我们介绍了游戏的基本概念，本节中我们来看看如何将理论转化为实际能力。迷男忠告我们，要想练成技能，必须在现场进行游戏。你不能总躲在幕后，或仅通过微信聊天。这样做成长会很慢。要想成长得快，就要追求人在现场。

人在现场才能积累很强的社交直觉。我们一开始谈恋爱会失败，或在男女互动中犯错误，都是因为没有按正确的游戏进行。我们学习的理论，必须到现场去实践。这样才能够微调自己，最终做到在正确的时间做正确的事。

你能做到在正确的时机做正确的事，就达到了“标准化”。标准化就是追求游戏的标准化。迷男说，你到现场去练习。现场就是妹子多的地方。包括你经常出门，参加一些活动聚会，去一些人流密集的社交场所。夜店也算。

人在现场，大概经过几个星期（快的）或几个月（慢的）的时间，你就会对自己做一个彻底的改造。你对现场的任何社交反应都会很清新，也就是你活得比较清醒。你更能够解读现场的一些行为，而不是一个社交白痴。社交白痴就是你去到一个现场，各种搞不懂状况，对于别人怎么互动，他们的关系你是看不懂的。

去现场有利于我们摆脱宅男、矬男的心智模式，提升自己的社交智慧，达到一种社交上的标准化。一开始，这个标准化是经过我们思维去提示自己的。也就是说你会想：我下一步该怎么办？我现在处在什么位置，现在是什么情况？你是有意识地调整自己。

你之所以有意识调整，是因为你学了正确的方法和理论，也知道了以前错误的方法和行为，并有意识地避免错误，让自己用正确的方式互动。这个有意识的过程，会随着你慢慢的练习，变成习惯，变成无意识。就是有一天你发现，你自然而然地就会做出一些正确的事，不用再自己提醒自己了，就没有这个声音再提醒你了。

一旦你发现这样的时候，就意味着你对于整个正确的游戏、标准化的游戏规则和架构，实现了一种“内化”。内化就是习惯成自然，进入你的潜意识。大概需要多长时间呢？要养成一个好习惯，需要最短21天的时间。这就是内化。这也是迷男让我们到现场去训练的价值和意义。我们练习这个游戏，是为了做一个准备好了的人，在时机到来的时候，大幅改善我们的人生。这是一个准备的过程。

## 心态建设：成长与练习

在去现场实践时，我们的心态建设至关重要。我们的注意力往往会很分散。比如你参加一个聚会，或去到一个夜店，诱惑你的东西太多。你可能不知道该把注意力放在哪，可能就是想我该去“收货”了，我该去泡妞了，我今天晚上该TD。

迷男的忠告强调，我们到现场，不要把心思放在今晚你是否能够成功泡到妹子上。你应该把自己的注意力放在提升自己的技能和技巧上。就是说，我们要建立一个“练习的心态”。这就是每一个PUA的成长心态。你要想成长，就得建立成长心态。成长心态的关键就是要建立练习的心态。

迷男说，过程重于结果，不要在乎一时一地的得失。也就是说今天晚上，你如果真正练习了你的技术，哪怕是你没泡到这个妹子，它对于你来说也是一种成长，也是一种收获。我们要建立起享受过程的心态。

以下是练习心态的两个核心要点：

1.  **享受过程，活在当下**。
2.  **游戏心态**。什么叫游戏心态？就是你要有一种无所谓、不在乎的意识提醒你，就像打游戏一样。你打游戏打输了，对你有什么损失吗？其实没有，无非就是你再重来一盘。练习把妹也是这样。你今天晚上或者说两三天的时间，搞一个妹子最后没搞成，对你有什么损失？有兄弟说你有损失，我请她约会还多花了钱。这种损失可以忽略不计，这是你练习成长必须要付出的代价。你如果连这点东西都不愿付出，那就别成长了。所以说它的损失基本是零，没什么太大损失，而你得到的东西就会很多。

我们要建立这个成长心态，就是我们时刻要告诉自己，在一段时期之内，我们要以练习自己的能力为主。什么能力？就是社交力学。简单说，就是练习自己在正确的时间做正确的事，做出正确的反应，而去除那些以前错误的、很宅的、很low的反应。就是练习享受过程，然后活在当下，游戏心态就是不在乎成败。

关于心态上的建设很重要，是因为兄弟们学了这个理论之后，或许还比较茫然，就是我们缺乏一个成长的路线图。听完这个，不代表你就成PUA了，或成恋爱达人了。错了。听完这个之后，只是说你在思想上、意识上知道自己该怎么做了，它并不代表着你恋爱能力实质上有多么高的提升。

你从一个情感小白，从一个AFC成长成一个恋爱达人，成长成一个有能力的、有魅力的男人，怎么成长？大多数兄弟他不知道，他就困在这儿了。而迷男在一开始第四章，就提出一个成长的路线。就是说，建立成长心态，然后人在现场，你不断地去练习，不断地精进自己的表达和技巧。以后书中讲到的很多东西，都是这种技巧的具体应用。

## M3模型：正确的恋爱流程

好，这就是关于前期的成长路线和心态建设。那么下面咱们就进入真正的主题，就是介绍一下迷男的M3模型。M3模型是迷男的一个最伟大的创造。这个东西对于我们男人来说，尤其有意义和价值。因为它是模型框架，便于理解，便于操作。它也是一个很正确的恋爱流程。

有很多人，他没看过迷男，他道听途说，就总以为迷男方法是什么速推方法，就是约炮方法。这是大错特错的。还有的人呢，他也没看过迷男方法，他一听这个名字，就认为是一个夜店方法。因为迷男主张去夜店，他就认为迷男方法不适用于中国，不适用于长期关系，不适用于正常的恋爱关系。这都是通通错误的观念。这些错误都是源于你不了解迷男方法造成的误会。

那么正规的迷男方法，它是一个完整的恋爱流程。为什么这样说呢？因为迷男说，任何一段正常的恋爱，都是从吸引开始，然后再建立舒适感，记住它是有顺序的，最后再进行诱惑。也就是说他分了三个阶段。这三个阶段就叫M3模型。

**M3模型 = 吸引 (A1-A3) + 舒适感 (C1-C4) + 诱惑 (S1-S3)**

它适用于你听过的、见过的、读过的、体验过的所有的关于恋爱的情感经历和故事。它适用于所有。它能够解释一切。所以说它是一个科学，就是社交力学。这就是正确的游戏的规则与架构。

## 四个错误的顺序

那么，如果你不按正确的顺序进行，就会出现四个错误，也就是颠倒顺序的四个错误。

以下是四个常见的错误顺序及其后果：

1.  **直接从诱惑开始**。一接近这个妹子，一发现目标，就直接开展诱惑，忽略了吸引和舒适感。这是第一个错误。这样会陷入一种什么后果呢？在迷男看来叫“愚人的笨招”。它可能会成功，但记住是“可能”，这是一种小概率。什么叫小概率？就是你正好碰见这个妹子，她需求感很大，或者比你还大，或者是你的价值很高，她的价值很低。记住这都是假如的情况下，小概率就是万分之一。它会成功，你也会TD。但我们要教一个正规的恋爱流程，不能这样教。你这样教的话，大概率的情况下都不是这样的。兄弟们学了等于白学。所以说我们要避免第一种情况，就是叫“愚人的笨招”。就直接诱惑，就是总想当一个推土机，推的越多越好。推土机就会陷入这种错误。

2.  **没有吸引，直接建立舒适感并试图诱惑**。第二个顺序错误是，一开始接触这个妹子，发现目标，就试图建立舒适感，并试图诱惑。这是第二个错误。那么这样就会陷入什么？迷男说叫“好好先生”。好好先生就是好人卡。这样就会激起妹子的“防护罩”。因为你没有吸引，在妹子看来你就没有价值。因为价值产生吸引。她没有发现你的价值，只发现你是个好人。你和所有接触她的这些“苍蝇”围着的“好男人”、“好好先生”都一样。你就是试图接触她，你当好好先生，你对她讨好、建立舒适感，在她面前各种乖、听话，或者是献花、请客吃饭，做乖乖猫、乖乖狗。你这个行为在妹子看来，只有一个目的：你想跟她上床，没别的。你和这些男人都一样。这就会陷入第二个错误，这就是你会被发好人卡。妹子会说：你是个好人，但我对你没感觉；你是好人，咱俩不合适。这就是没有吸引，直接追求舒适感的后果。

3.  **有吸引，没有舒适感，直接诱惑**。第三个错误的顺序是这样的：有吸引，然后没有舒适感，略过了舒适感，直接就开始诱惑。那么这样就会陷入“事后反悔”。这是第三个错误。

4.  **有吸引，有舒适感，但困在舒适感，无法进行诱惑**。第四个错误的顺序就是：有吸引。他接触这个妹子，一开始接触这个目标，也是按正确的方式先建立吸引，然后再追求舒适感。但是，他现在困在舒适感里，出不去了。这叫什么？这叫“友谊区”。就是无法进行诱惑。这个就叫友谊区。现在有很多词汇形容这个友谊区，比如“男闺蜜”。就是跟妹子聊天聊得太好了，聊成男闺蜜了。还有就是“备胎”。备胎就是陷入妹子的备胎了。也就是我跟你保持这种朋友关系，但是就是再也没有上床的机会了，你就在那备着。什么叫备胎？备胎就是平时不用，在那挂着。然后妹子一旦她和别的男人分手了，可能会借由你来满足一下她这两天的虚荣心，你会安慰安慰她，请她吃顿饭，看看电影。备胎上位是很难的。那么有方法吗？也有，这个回头咱们再讲。

## 三个玩家陷阱

接下来再讲一下三个叫“玩家陷阱”。陷阱就是很危险的雷区，就是很容易造成失败的三个点。

以下是三个玩家陷阱：

1.  **无法把吸引正当化**。就是有了吸引了，然后下面不知道怎么做了，无法正当化。因为迷男说，只有吸引还是不行。那妹子对你感兴趣，不代表着最后你们能走在一起。就是你要和他建立不起舒适感，最终无法诱惑的话，你们还是等于零。迷男说吸引有的时候就像服务员一样，不持久。那么怎么才能避免无法把吸引正当化呢？这就要提到一个叫“可得性”。什么叫可得性呢？就是说，你的这个可得性太高的情况下，妹子很快就会对你丧失兴趣。也就是说她对你一表现兴趣，你马上就让她感觉到她马上就把你拿下了，这样的话她对你兴趣很快就会丧失。那么为了避免这样，迷男方法才创造了一系列的方法，就是建立自己的高标准，建立自己的筛选态度，在和妹子互动的时候来跳过这个第一个陷阱。

2.  **没有建立起很好的舒适感，就试图诱惑**。第二个陷阱就是：有吸引，没有建立起很好的舒适感，然后就试图诱惑。这也是一个陷阱。那么这样会造成什么后果呢？这样会造成这个妹子感觉到你就想玩她。他会和你在一起感觉很不舒适，很别扭。就是你不想了解我，你也不想和我有什么长期关系，你就想和我，说白了，你就想和我打打“眼炮”。当然了，迷男说，如果说你赶上这个妹子她也很需要，那么也可以，但是大部分情况下是不可以的。所以说这是一个陷阱。

3.  **事后反悔**。第三个陷阱就是这个“事后反悔”。什么叫事后反悔呢？这一个事后反悔，它指的很多时候不是说，你把这个妹子你们带入私密空间，做完了事儿之后，妹子反悔了。她还不是完全这样理解的。当然也有情况下是这样理解的，就是你们都做完了，然后妹子反悔了，她感觉到和你不合适。那么还有一种情况更悲剧。是什么？就是他跟你，你无论想什么办法，你们两个人走到了一个亲密阶段，走到了一个私密空间，也进行了前戏，最后没弄成，把妹子放跑了。妹子这时候出现了事后反悔。这也叫事后反悔，就是说他没给你做成这件事儿，也叫事后反悔。她就不理你了，对你越来越冷淡。

## 不归点原理
