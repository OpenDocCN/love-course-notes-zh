# 赢家游戏：3.0：实战训练计划：3.2：新手任务：3.2.1：任务1—眼神交流 👁️

在本节中，我们将开始“接近焦虑脱敏系列”的第一个任务。这个任务的核心并非直接与人搭讪，而是专注于打破社交壁垒，为后续的互动打下基础。

上一节我们介绍了实战训练的整体框架，本节中我们来看看这个系列中最基础、也最关键的第一步：眼神交流。

## 任务概述

你的第一个任务不涉及主动接近他人。它的核心在于**打破社交屏障**。这涉及到轻微地“侵入”他人的个人空间。有一句话说：如果你注视某物（或某人）的眼睛超过3秒，通常意味着你想与之发生关系或想杀死它。我们当然不会注视那么久。

我们要做的是开始**增加眼神接触**。我们将开始明确地尝试与陌生人进行眼神交流。

## 具体操作方法

以下是本任务的具体执行步骤。

假设你走在街上，有人正向你走来。我希望你真正地看着他们，并尝试进行眼神接触。当你在办公室或学校里穿行，有人迎面走来时，同样尝试与他们进行眼神接触。

关键点在于：我希望你**将眼神接触保持得比让你感到舒适的时间再长那么一点点**。不要立即移开视线。保持住那额外的一刹那，然后如果你想的话，就可以移开目光。

这样做的目的是克服社交障碍，开始成为一个更善于社交的人。任务内容就是：四处走动，创造眼神接触。

## 注意事项与深入理解

现在，当你与他人交谈时，你并不需要一直盯着对方保持眼神接触。你可以时不时地看向他们的眼睛。

我希望你通过这个任务，建立更人性化的连接。人们常说，眼睛是心灵的窗户。当你直视某人眼睛时，你确实在与他们建立连接；反之，则连接较弱，尤其是对于陌生人而言。你需要习惯并适应这种感觉。

你可能会发现以下情况：
*   有些人会立即移开目光。这没关系。
*   有些人会回看，并且目光可能非常强烈。这也没关系。在这种情况下你可以移开目光。

你无需变得具有对抗性，或与对方产生任何冲突。你的目标仅仅是**适应并习惯进行眼神接触**。

## 任务总结

本节课中我们一起学习了“接近焦虑脱敏”的第一个新手任务。这个任务非常简单，是开启社交互动、在这个世界上轻微地展现自我的**绝对最低限度要求**。

总结来说，你的任务就是：当有人看回来时，你要将眼神接触保持得比你认为舒适的时间再长那么一小会儿。这就是你现阶段需要做的全部。
