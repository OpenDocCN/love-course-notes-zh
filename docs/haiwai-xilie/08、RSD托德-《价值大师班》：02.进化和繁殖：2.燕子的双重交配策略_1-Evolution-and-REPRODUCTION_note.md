# 价值大师班：02：进化和繁殖：燕子的双重交配策略 🐦

在本节课中，我们将要学习吸引力与择偶行为背后的进化根源。我们将从动物的例子入手，理解生存与繁殖如何塑造了包括人类在内的所有物种的行为模式。

## 一切源于进化

从根本上说，我们是动物，尤其在情感层面更是如此。我们的认知大脑、深层逻辑和理性思维的发展速度，远不及我们的情感。因此，作为动物，我们首先受情感支配。我们与猫、狗、鸡、鱼的相似之处，可能比你想象的要多得多。我们所做的很多事情，在生物学层面上早已被预先设定。

上一节我们介绍了进化的基础，本节中我们来看看繁殖的核心驱动力。

## 生存与繁殖

这一切都归结于生存与繁殖。需要指出的是，是**生存与繁殖**，而不仅仅是生存。例如“适者生存”的观点，其本质并非为了生存而生存，而是为了**繁殖**而生存。

以下是几个关键点，用以说明繁殖的核心地位：

*   **进化筛选**：像在70多岁时易患阿尔茨海默症的倾向，对生存与繁殖的影响可能微乎其微。在远古时期，当人们的平均寿命较短时，这种特质不太可能被自然选择淘汰，因为到了70岁，个体通常已经过了繁殖期。
*   **关键时期的影响**：然而，一个男性在16、17、18、19、20岁时的地位，决定了他如何在社会中生存、在部落中处于何种位置，这可能会对他的生存与繁殖产生巨大影响。因此，这类特质会受到更强烈的自然选择压力。

所以，需要强调的是，进化的核心是**繁殖适应性**，而不仅仅是生存适应性。任何与繁殖窗口期之后年份相关的特质，其重要性都会降低；与直接传递基因无关的特质，其受重视的程度也会低于直接相关的特质。

本节课中我们一起学习了吸引力与择偶行为的进化基础，理解了“繁殖适应性”是驱动一切的核心法则。我们从动物的普遍策略出发，认识到人类行为深植于为传递基因而竞争的生物本能之中。
