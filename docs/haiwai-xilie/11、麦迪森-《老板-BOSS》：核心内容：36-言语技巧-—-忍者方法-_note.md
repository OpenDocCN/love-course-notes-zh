# 搭讪艺术：第11课：避免“忍者式”接近法 🥷

在本节课中，我们将要学习一种关键的社交技巧：如何避免使用“忍者式”接近法。我们将探讨这种方法的弊端，并理解为什么展示真实的自我是建立核心自信的唯一途径。

## 什么是“忍者式”接近法？

上一节我们介绍了课程主题，本节中我们来看看什么是“忍者式”接近法。它指的是在接近他人时，试图隐藏自己的真实意图和欲望，就像忍者一样悄无声息。

以下是其常见的表现形式：

*   **言语上的“忍者”**：例如，在公交车上问：“你知道这趟车终点站在哪吗？”或者在星巴克问：“请问最近的星巴克在哪？”然后突然转向真实目的：“你有男朋友吗？”
*   **肢体上的“忍者”**：以非常轻微、像蚊子一样的方式触碰对方，例如轻轻地、快速地拍一下。正确的触碰方式应该是**放松、坚定但不具侵略性**，就像开玩笑一样，目的是让对方明确感知到你的存在。

## 为什么应该避免“忍者式”接近法？

那么，为什么男性会使用这种“忍者式”接近法呢？核心原因在于**安全**。这种方法可以避免所有可能的拒绝，因为它本质上不算一次真正的“接近”。你不承担风险，不把自己置于可能被拒绝的境地。

但是，建立**核心自信**的唯一方法，恰恰是把自己真实的一面展现出来，并接受由此带来的接纳或拒绝。**没有捷径**。

> **核心自信公式**：**核心自信 = 展现真实自我 + 承受接纳或拒绝的结果**

如果你坚持使用“忍者式”接近法，你实际上是在**自我设限**，无法获得真正的成功。

## “忍者式”接近法的两大弊端

以下是“忍者式”接近法带来的两个主要问题：

1.  **对方没有机会真正了解你**：你只提供了一个糟糕的“样品”。即使你本身非常优秀（例如是医生或战争英雄），对方也完全无从知晓，因为你表现得像只“小蚊子”。
2.  **你试图避免的恰恰会发生**：你因为害怕被拒绝而采用“忍者”方式，结果对方完全看不到你的个性，互动无法真正开始，这本身就导致了“拒绝”的结果。你**恰恰招致了你试图避免的事情**。

## 正确的理念：提供你的“个性鸡尾酒”

我认为，搭讪的艺术就像是走向一个女孩，并递给她一杯你的“个性鸡尾酒”。她需要看到你的全部。

以我最喜欢的鸡尾酒“古典鸡尾酒”为例。它的原料并不复杂（方糖、苦精、波本威士忌、橙皮等），但不同地方调出的味道千差万别，关键在于**各种原料的混合比例与方式**。

人际互动也是如此。你的目标是让对方看到你个性的完整混合：
*   一点快乐
*   一点悲伤
*   一点愤怒
*   一点性感
*   一点风趣
*   一点严肃

我的互动目标就是让她看到我个性的所有这些方面。正如我通过视频展示全部自我，让观众感觉认识我一样。当你使用“忍者式”接近法时，对方**完全看不到**你的任何个性。

## 总结

本节课中我们一起学习了“忍者式”接近法。我们了解到，这种方法源于对安全的追求和对拒绝的恐惧，但它会阻止对方了解真实的你，并且恰恰会导致你害怕的结果——无法建立真正的连接。记住，建立核心自信的唯一方式是勇敢地**展现真实的自我**，并坦然接受任何结果。摒弃那种“忍者”式的做法，大方地展示你的“个性鸡尾酒”。
