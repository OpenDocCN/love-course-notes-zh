# 价值大师班：05.价值的内在博弈：2 “状态”与“心流”相关概念 🧠💫

在本节课中，我们将探讨“状态”与“心流”这两个核心概念。我们将了解它们是什么，如何利用它们提升表现，以及为何不应过度依赖它们作为借口。

## 什么是“状态”？

“状态”指的是当你处于更好的情绪或特定心态时，你往往能把事情做得更好。这个观点是正确的，大量心理学研究已证实，处于不同情绪中的人通常表现更佳。例如，研究表明，当人们在听特定类型的音乐时学习，他们在听同样音乐时能更好地回忆起所学内容。这类现象已被反复验证，因此“状态”是真实存在的。

然而，“状态”的问题在于，如果你持续依赖它，例如总想着“我必须进入状态”或“我必须心情好”，就可能产生负面影响。以学习时听音乐为例，如果你在考试时也能听同样的音乐，这或许有益。但你不应过度依赖它，以至于当老师不允许考试时听音乐，你就完全乱了阵脚，无法正常发挥。你也不应在考试时一直想着“我讨厌这个老师不让我听音乐”，而不是专注于眼前的试题。

因此，“状态”是美好的，在你能利用它、聪明地适应并运用它以获得最大益处时，你应当这么做。但你绝不能让自己陷入依赖“状态”的境地。“状态”不能成为借口。你不能去上班时对老板说：“我今天心情不好，没法工作。”现实并非如此。

理解“状态”是好事，围绕它制定策略、进行规划也是好事。但它绝不能成为一个糟糕的借口。我不希望你用“状态”作为借口，我更倾向于使用“心流”这个概念。

## 什么是“心流”？

“心流”是指你完全沉浸于正在做的事情中。你如此投入，以至于周围的世界仿佛都消失了，你能真正地专注并发挥出最佳水平。

一个颇具讽刺意味的最佳“心流”例子是电子游戏。游戏的设计目的就是让你进入心流状态。它们旨在提供恰到好处的挑战、成功与失败，以保持你的参与度。这就是为什么你可以开始玩一个游戏，然后玩一整夜，忘记时间，早上醒来发现自己除了玩游戏什么都没做，因为你完全迷失在那个世界中，进入了心流。

但是，如果你给自己设定的挑战过大（例如尝试玩对你来说过难的关卡并反复失败），你就会脱离心流，变得沮丧，表现也会变差。同样，如果你开始玩一个已经通关16遍的游戏，你会感到无聊和自满，也会脱离心流，表现不佳。

因此，在考虑“状态”时，我关注的重点是如何进入“心流”。如何为自己创造恰到好处的挑战？如何创造恰当的参与度？如何创造能引导我走向结果的情境？但你绝不应依赖它。不在心流状态不能成为借口。你应该做的是执行那些能让你进入心流的正确流程。

## 如何间接控制状态与心流？

这里存在一个讽刺之处：你无法为了进入“状态”或“心流”而刻意去做某些事。你必须因为那是正确的事而去做，并让它自然发生。这就像一种非自主功能。例如，打嗝：我能命令你现在就打嗝吗？很可能不能。你能仅凭意念就让心跳显著加速吗？也许站起来跑可以，但仅靠思考心跳，这是非常困难的事，因为它是一种非自主功能。

因此，直接控制状态和心流非常困难，但你可以间接控制它们。你可以通过执行正确的流程、采取正确的行动、专注于正确的事情来控制状态和心流，但你不能刻意为之。如果你不断检查“我进入心流了吗？我进入状态了吗？”，你实际上是在让自己陷入过度思考和分析，这与处于心流或状态是相反的。检查和试图进入心流或状态，恰恰是真正进入它们的反面。

## 核心要点总结

这些是伟大的概念，你绝对应该围绕它们来构建你的实践。但你绝不能允许自己依赖它们。你需要能够在心流内外、状态内外都能工作，并且绝不应将它们用作借口。你不应依赖这些现象。

本节课中，我们一起学习了“状态”与“心流”的概念。我们明白了它们能提升表现，但过度依赖则会成为障碍。关键在于通过执行正确的行动来间接引导自己，而非刻意追求或将其视为表现不佳的理由。
