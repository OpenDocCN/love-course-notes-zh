# 掌握转变：02：觉知：01：找出原因 🔍

在本节课中，我们将要学习一个核心概念：**范式**。理解范式是进行深度工作并触及问题根源的第一步，而不是仅仅在表象上做文章。

## 什么是范式？🧠

上一节我们介绍了课程的整体框架，本节中我们来看看什么是“范式”。

范式是我们头脑中用于理解世界的不同“地图”或框架。你可以把它们想象成我们观察世界的不同“镜片”。它们影响着我们感知、理解、解释和过滤生活的方式。

有趣的是，我们通常意识不到自己拥有这些范式。这是我们首先要认识到的一点：我们以为自己客观地看待事物，但事实并非如此。

**核心公式**：`你看到的并非世界本身，而是你被设定好看到的样子。`

从你出生那一刻起，来自家庭、朋友、学校、媒体、网络等各方面的信息就在不断塑造你看待世界的方式，塑造你过滤生活的“范式镜片”。这就像有人给你戴上了一副眼镜，而你因为一直戴着它，甚至意识不到它的存在。

## 范式的类型与假设 🥽

既然我们通过范式观察世界，那么下一个关键问题是：我们从未质疑过这些范式是否准确。

为了简化，我们可以将范式分为两种基本类型：
*   **丰盛范式**：其根本假设是你的**默认状态是丰盛的**，即 `基线 = 10/10`。
*   **匮乏范式**：其根本假设是你的**默认状态是匮乏的**，例如 `基线 = 3/10`。

你也可以将其理解为幸福与不幸福、完整与不完整、足够与不足够。正是这些根本假设，衍生出了我们所有的态度、行为和行动。

以下是判断你当前处于哪种范式的方法：
1.  进行第一节视频中的练习：觉察身体感受，并以1到10分进行评分。
2.  问自己一个问题：**如果我什么都不做会怎样？** 如果停止努力、停止寻求认可，我是会坠入匮乏，还是安住于丰盛？

不幸的是，由于我们被设定的方式，绝大多数人都默认假设自己是匮乏的。这副“匮乏镜片”被一代代传递下去。

## 错误范式的巨大代价 ⚠️

假设匮乏，是造成我们所有痛苦的根源。它是自我憎恨、逃避现实、自我放弃、自我破坏、慢性压力及健康问题的温床。

这也解释了为何我们在个人成长上投入大量努力，结果却总不持久或并非真正所需。因为我们试图在**匮乏的范式**中追求**丰盛**。这就像拿着底特律的地图在芝加哥找路，无论你多么努力调整态度、改变行为，都无济于事。

**核心洞见**：`在某个范式内所做的任何事情，都会强化该范式的根本假设。`

例如：
*   **努力提升自己**：假设是“我不够好”，那么所有提升行为都在强化“我不够好”这个假设。
*   **激励自己奋斗**：假设是“我天生懒惰、缺乏动力”，那么所有激励行为都在强化“我缺乏动力”这个假设。

你为摆脱某物所做的任何努力，都是该物的延续。不仅如此，你投入越多，就越难放弃这个错误的范式，甚至会无意识地寻找证据来强化它，忽略相反的证据，形成自我实现的预言。

## 解决方案：转向内在，触及根源 🧘

那么，关键何在？关键在于**觉察**，并**退后一步审视**我们所处的范式及其根本假设。最终，你必须放下错误的范式。

这就像《黑客帝国》中的尼奥，你需要彻底转换范式。这意味着，这是一场**内在的旅程**。触及根源，就是深入你内在的那个默认状态、那个根本假设。

**核心方法**：`这是一条由内而外的路径。`

我们的内在模式创造了外部现实。一个总是愤怒的人，在任何环境中都能找到令他愤怒的理由。生活的质量取决于你内在的体验品质。如果你内心丰盛，你就能处处看到并创造丰盛。

我们习惯于 rearranging the external（改变外部），希望借此影响内部，但这只是在处理表象。如果答案真的在外界，以我们这代人的尝试程度，早就该找到了。我们是历史上最舒适的一代，却也可能是有史以来最不快乐、基线状态最差的一代。

所以，是时候停止拐弯抹角了。让我们触及问题的根源。

## 总结 📝

本节课中我们一起学习了：
1.  **范式**是我们意识中过滤世界的“镜片”，我们通常对其毫无察觉。
2.  两种基本范式：基于**丰盛**假设与基于**匮乏**假设。
3.  在错误范式（通常是匮乏范式）内努力，只会不断强化根本问题，无法获得真正持久的改变。
4.  真正的转变始于**内在**：通过觉察，审视并放下错误的根本假设，从内部根源入手，实现由内而外的改变。

放下对外部效果的执着，开始这场深入内在、直达根源的旅程吧。
