# 约会加速器：02：社交焦虑

在本节课中，我们将深入探讨阻碍男性在约会和社交中取得成功的核心问题——社交焦虑。我们将剖析其根源，并学习如何转变心态，将恐惧转化为前进的动力。

## 概述：焦虑的根源

在接下来的内容中，我们将深入探讨这个问题，分析这些焦虑的潜在原因，并寻找解决方案。

## 恐惧的本质：我们究竟在怕什么？

首先，我们需要审视：你究竟在害怕什么？

让我问你一个问题：如果你生活在一个相当繁忙的城镇或城市，你每天真正应该担心的是什么？最可能危及你生命的东西是什么？除非你生活在战区或街头暴力频发的地方，否则对你最危险的东西是**交通**。汽车非常危险，它们以惊人的速度飞驰而过，而我们常常站在离它们仅几英寸的地方。每一辆车都可能终结我们的生命。然而，我们大多数人对驶过的汽车并无恐惧，因为我们已对此脱敏，不将其视为威胁。

一些理论家试图解释男性为何害怕接近女性。理论大致是：在原始社会，我们生活在小型狩猎采集群体中。如果我接近群体中的一位女性并遭到拒绝，其他女性看到后都会认为我是失败者，不再愿意与我交往，从而导致我的基因从基因库中被淘汰。因此，我对女性的拒绝有一种根深蒂固的生物性威胁反应。

这个理论可能包含一些真实元素。作为社会性动物，我们关心自己的声誉和地位，会在社会中比较自己。我们确实在深层非常在意他人对我们的看法。然而，我们也有多层次的**条件作用**覆盖其上。我们的大脑具有极强的适应能力，能够适应各种环境。就像我们适应了汽车驶过的恐惧一样，我们也可以适应接近女性时的感受。

最初，接近女性可能感觉非常可怕。我曾指导过一些学生，他们的身体会僵住，无法说出话来。我自己刚开始外出与女性交谈时，最大的问题也不是说什么，因为一旦对话开始，即使话题普通，我也能感觉尚可。真正的挣扎在于**首先克服障碍走过去与她交谈**，这花了我数年时间才达到现在的状态。

如今，当我想到接近一位女性时，我仍然会感受到你们和其他男性会感受到的某些东西。但我对这些感受的**解读**与你们不同，也与十年前的我不同。我将在后面展示如何开始转变你对自己感受和想法的感知方式。

## 思维与情绪的陷阱

我们的思维过程和情感过程可以是强大的盟友，也可以是囚禁我们的狱卒。不幸的是，后者常常发生。

如果你没有临床焦虑症，那么将其称为“焦虑”并不是一个好主意。迄今为止，我一直将这种恐惧称为“接近焦虑”，这是普遍的认知。但我认为这是一个**糟糕的标签**。它本质上是在给自己贴上一个“病症”的标签。当你说“我有接近焦虑”时，你是在说“我得了这种病”。这会导致你认为自己无法做任何事，因为你感到害怕。

你如何知道自己害怕？因为你的胸部有心悸，胃部紧绷，手心出汗，声音哽咽，双腿发抖……所有这些身体表现，伴随着一种情绪反应，以及附加的想法：“我做不到”、“她太美了”、“我是个失败者”、“我不知道说什么”、“她可能有男朋友”、“这反正行不通”、“等我有了车和好工作再说”……所有这些内心对话，这些糟糕的“人生指导”，与那种感觉糟糕、渺小、不足的情绪结合在一起，身体同时做出反应，这一切变成一团乱麻。然后你做了什么？**你退缩了**。

或者她可能天生有才华，或者她真的努力提升自己，通过冥想、瑜伽、戏剧或其他什么真正探索和成长。我记得有几次我遇到一个女孩，心想：“哇，她太棒了。”并不是我不够棒，而是我还没到能吸引这个女孩的阶段。我可以看到这一点。我必须做出决定：我可以非常个人化地看待它，认为“好吧，我是个失败者，我永远得不到那样的女孩”；或者，正如我所做的，我把它看作一个基准：“我想我需要更敞开心扉，我要在这方面努力”，或者“我需要更性自信，我要在这方面努力”，或者“我需要更善于表达和真实”，或者“我需要更内敛和策略”。有很多不同的领域，我多年来成长、改变和学习，这意味着，一级一级地，我开始得到以前得不到的女孩。并不是过去的我不好，只是稍微不那么成熟的我。那时我也有和我水平相当的酷女孩。

所以你需要明白，揭露自己并不意味着会被所有女性拒绝。有时意味着会被一群女性拒绝。这并不超级个人化。不是你是个可怕的人，或者你不如她。只是如果她处于一个社交上很协调、人脉很广的位置，她有很多选择。而且她非常美丽。处于这些位置的美丽女性拥有的选择，比你或我能想象的要多得多。现在有了社交媒体和互联网，她们真的有成千上万的男性在Instagram上关注她们，给她们送礼物、点赞，以各种方式奉承她们。她们有很多选择，有时现实是，你在她的优先顺序阶梯上只是低于其他某个男性。这不是个人化的。不意味着那个男性是比你更好的人。意味着他社交上更 connected，或者更社交 aware，或者他知道如何与她周旋，知道如何以她认为更令人兴奋和有吸引力的方式与她交流。

所以，这种对被揭露、被评判的恐惧，是男性在感到这种去接近女性的恐惧时，所面临的大多数问题的潜在原因。

有很多事情你可以开始做来转变这一点。这不是本课程的重点。我想为你打开思路，开始思考和研究，但这里不会深入超级深层的内心游戏转变。你可以在我的其他产品《马歇尔冥想方法》中看到，当然，我们也在现场课程和各种其他课程中做这个。但重要的是审视：好吧，我真正害怕的是什么？现在，我想整天待在那里思考这个，躺在心理学家的沙发上谈论它五年吗？我认为这不必要。我的意思是，对某些人来说，可能是。但总的来说，让我们知道：好吧，我意识到，是的，我担心如果我向她展示自己，她会以某种方式评判我。她会认为我不配，或者只是平庸，或者更糟。她会认为我是个令人毛骨悚然、猥琐下流的怪胎，我 somehow 是某种变态。所以，保护自己、完全不被人看见更好。当然，不被人看见并不更好。被忽视并不更好，因为当你被忽视时，你不会发生关系。

所以，我们必须能够超越这些恐惧。现在，这并不意味着我们需要解决所有问题。你不需要在能够去得到你真正想要的女孩之前，变成一个完全实现、毫无问题、自信的男性。记住，女性并不完美，她们也不期待完美。我们所有人都有问题，都有包袱，有时都感到不安全。即使是最自信的人。我通常是个自信的人，生活成功，通常能得到我想要的。但在我的一生中，我也经历过重大危机，有过崩溃的时候，抑郁过，充满自我怀疑和自我厌恶。然后我从那里面走出来，进入另一个阶段，进入一个自信的阶段。这在你的整个生命中会持续发生，程度不同。多年来我发展的是：总的来说，我学会了如何更快地从低谷中反弹，我的整体环境——比如休息时的自信水平——比过去高得多。这就是你应该通过个人发展、冥想学习、社交学习所追求的：总的来说，你正在变得更像一个男人，更像你自己，你学得更多，扩展了你的意识、心灵、情感、经历、技能。日复一日，这增加了，你成为更好的自己。当然，当你成为那个更好的自己时，你开始吸引更好的女性。

但我们需要看看：那么，即使我们可能无法解决问题，我们如何能够穿越这个恐惧，去接近女性？

为了穿越这些恐惧，我建议的第一件事是：**你完全停止将其视为恐惧**。
