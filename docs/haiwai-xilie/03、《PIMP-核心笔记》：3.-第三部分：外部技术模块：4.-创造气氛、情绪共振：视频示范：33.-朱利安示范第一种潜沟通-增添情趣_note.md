# 《PIMP 核心笔记》：3. 第三部分：外部技术模块：4. 创造气氛、情绪共振：视频示范：33. 朱利安示范第一种潜沟通-增添情趣 😊

在本节课中，我们将学习如何通过潜沟通为互动增添趣味。核心在于以积极主导的方式传递信息，让交流过程变得轻松愉快。

上一节我们介绍了潜沟通的基本概念，本节中我们来看看如何具体运用“增添情趣”这一技巧。

关键在于以一种为互动增加乐趣的方式表达。你可以走向一个房间，这样说：“嘿，你叫什么名字？” 然后对方回答：“哦，我叫XX。” 你可以接着说：“酷。你叫什么名字？” 此时互动依然带有目的性。你仍然在打破和谐感，比如反复问“你叫什么名字”。但这种方式背后带有一些趣味性。就像带着一点微笑，这是一种积极的主导。例如：“嘿，你叫什么名字？” “哦，真的吗？那太棒了。酷。你叫什么名字？” 这就是在增添乐趣。

以下是运用此技巧的两个核心要点：

1.  **保持积极主导的姿态**：在交流时，通过微笑和轻松的话气传递自信与友好。
2.  **在目的性中加入趣味**：即使是在进行有目的的对话（如询问名字），也要通过重复或略带调侃的方式，让过程变得有趣。

总而言之，本节课我们一起学习了如何通过“增添情趣”的潜沟通技巧，以积极主导和富有趣味性的方式开启或推进互动，从而创造更轻松、有吸引力的交流氛围。
