# 迷男方法国人描述：第14节：进挪升高

在本节课中，我们将要学习《迷男方法》中一个至关重要的概念：**进挪升高**。我们将详细探讨什么是进挪、为什么它如此重要、以及如何在实际互动中正确运用进挪升高的原则，来引导女性进行投资并锁定关系。

## 什么是进挪？

上一节我们介绍了如何引导女性进行投资，那么具体怎么做呢？这就引出了一个核心概念：**进挪**。

进挪，可以理解为所有你与目标女性之间发生的肢体接触。这是对进挪最简单直接的理解。

为什么我们如此强调进挪呢？因为人与人之间要建立起亲密的情感连接，仅靠语言上的沟通、调情、赞美或否定是不够的，**肢体上的触碰**是必不可少的一环。

## 进挪的核心原则

关于进挪，有几个关键点需要掌握。迷男对此有一个很好的解释。

首先，**不要过度重视**进挪。过度重视意味着过度紧张，总是在寻找“关键点”，比如“我该什么时候牵手、拥抱或亲吻”。为了避免这一点，我们需要建立一个观念：**这没什么大不了**。你要把肢体接触想象成男女交往中自然而然会发生的行为。这就是我们的框架——我们认为这是合理且应该发生的。

*   **框架示例**：我就该牵你的手，牵完手之后拥抱、亲吻，最后亲密。如果对方抗拒，你可以用框架打压她：“你会不会谈恋爱？你是出来谈恋爱的还是瞎闹的？”

记住，进挪不是吃豆腐，不是占便宜。如果你觉得“我没追到这个女生，但亲了她一口也不错”，这种想法就太没出息了。进挪有更深远的意义。

## 进挪的意义：逆向合理化

为了完全讲清楚进挪是什么，我们需要深入细节。迷男说，如果初吻怪异又尴尬，那很可能就是最后一吻了。女性对这类事情很苛求，她们有邂逅白马王子的完美幻想，追求一切**自然而然**地发生。

如何让对方感觉自然？前提是**你自己首先要感觉放松和自然**。如果方向正确，永远不会有一个“鼓起勇气、创造奇迹”的重大时刻。进挪是一个**由浅入深**的自然流程。

以下是几种具有象征意义的进挪：
1.  **牵手**
2.  **拥抱**
3.  **亲吻**（包括舌吻）
4.  **亲密**

这些象征性进挪的作用是：**锁定你们的关系**，实现**逆向合理化**。

例如，第一次约会后，如果什么都没发生，女生会合理化你们为“普通朋友”。如果你们牵了手，她就会想“我们已经牵过手了”，从而将关系合理化为“超越普通朋友的暧昧情侣关系”。通过肢体接触，而不是言语表白，来逆向合理化你们的关系。拥抱、亲吻乃至亲密，都是如此。你不需要说“做我女朋友吧”，通过一系列进挪做成这件事，她自然就成了你的女朋友。

## 进挪升高的流程

进挪应该是从**打开组合**（即使是一个人）开始，直到发生亲密关系之间的一连串自然流程。一切应该天衣无缝，由一连串细微、自然的时刻组成，很少有让你印象深刻的波折。

这是一个逐步**升高**的过程：
*   从轻轻碰手，到正式牵手。
*   从搭肩搂腰，到拥抱。
*   从亲吻额头、脸颊，到拥吻。

**不要一上来就试图拥抱或亲吻**。进挪必须**由浅入深，逐步升高**。这就是“进挪升高”的原则。

迷男做了一个形象的比喻：想象两只水母彼此靠近漂浮。起初只有几根触须几乎触碰到对方，接触、滑开、再触碰。最后越来越多的触须加入“舞蹈”，接触增多，其中几根开始用力拉扯，随着交缠而积累能量。你和你的目标就应该像这样。没有某个特定的“出招时刻”，而是几十个特定招式累积出隐晦的、有意无意的加速作用。

## 进挪的操作要点

基于以上原则，在操作进挪时需要注意以下几点：

**1. 敢于出手，不必询问**
事先不必询问或试探，直接出手。敢于去牵手、搂腰、拥抱、亲吻。不要问“我可以牵你的手吗？”，这会让氛围变得尴尬。

**2. 有升高，也要有释放**
调情的关键在于**升高与释放**。如果你一直牵着对方的手，时间长了可能会别扭。牵一下，然后释放，这是在调节对方的情绪。进挪上的升高和释放本身就是一种很好的调情方式。

**3. 抓住“机会之窗”**
如果你的游戏正确（展示IOD、高价值、筛选），女性对你的“机会之窗”会**周期性开启**。这时你可以进行进挪。如果窗口开启时你没有行动，窗口就会关闭。错过太多机会，女性会认为你胆小、无能或对她没兴趣，这都是低价值的表现。男人不敢进挪，往往是担心留下坏印象，但这反而会展示你缺乏社交智慧、胆小的低价值面。

所以，**永远要假设你们彼此来电**。如果她还在和你互动，你就继续进挪。

**4. 练习与内化**
对于练习者来说，敢于进挪是改善你“社交标准化”的最佳方法。通过不断勇敢地尝试，积累社交直觉和经验，你就能知道何时该做何事，并将这种能力内化。这不仅是技术，更是胆量问题。

**5. 另类触摸**
触碰女性不能像青少年宣誓主权那样正式（如很刻意地伸手）。要自然轻松，可以采用“另类触摸”，比如玩游戏（如拇指大战）、指滑等。这些轻微、快速的触碰更容易在前期获得服从。

**6. 永远主导，循序渐进**
不要幻想女性会主动来进挪你。男人要**永远主导**，承担起主动进挪的责任，并且必须遵循**循序渐进**的原则。

## 应对抗拒的策略

进挪升高时，常常会遇到女性的ASD（反荡妇防卫机制）抗拒，比如缩手、推开、说“太快了”。这时该怎么办？

**策略一：进一步，退两步**
例如，你们牵手很舒适了，你试图搂腰（进一步），但遇到抗拒。这时你不仅退回搂腰，甚至**连手都不牵了**（退两步）。这会制造一种因她抗拒而产生的尴尬，让她感觉是自己“不懂事”、“不会谈恋爱”。过一会儿你再尝试进挪，往往能获得更好的服从。

**策略二：无所谓的释放与坚持**
更高级的社交直觉是：在她抗拒**之前**，你先主动释放。比如，牵手走路时，感觉她快不舒适了，马上主动放手。这种释放带来安全感，让她相信你不会强迫她。遇到抗拒后，你停止，互动一会儿再升高；再抗拒，再停止，再互动，再升高。关键是保持**无所谓的态度**，不被她的抗拒影响情绪（不沮丧、不恼羞成怒）。这样她会感到安全，并积累**服从动能**，越来越服从你。

**策略三：剥夺**
如果抗拒非常激烈，可以使用“剥夺”。即给予IOD、否定或打压，让她明显感觉到自己做错了。在S阶段，这类似“冷冻”。例如，约会中她表现不好，你可以暂时不理她，去干别的事。如果一个妹子出来约会只吃喝却不让你碰，很可能对你没意思，这时应该“剥夺”你的关注，及时闪人。

## 进挪回响与服从性测试

**进挪回响**是指两人间吸引力增强的现象。例如，你推她肩膀，她也推开你；你牵她手，她回握你；拥抱时她也用力抱你。这是积极的回应，是很好的兴趣指标和服从信号，标志着关系正在升温。

理解了进挪升高，我们就可以引出另一个核心概念：**服从性测试**。

服从性测试是迷男方法的核心概念之一，贯穿整个互动阶段。CT就是你为了让目标女性在一段关系中投资，而让她做的事。
*   **CT示例**：`让她帮你拿饮料`、`伸手牵她`、`让她帮你收快递`、`提出去开房`。

如果她服从了，你就达成了几件事：
1.  这是一种**兴趣指标**。对于社交直觉不灵敏的人，CT是判断对方是否喜欢你的最靠谱方式。其他可能是虚的，但服从是真的。
2.  你**升高**成功了，她更习惯你的触碰。
3.  她的框架进一步被你的框架吸收。

运用CT要循序渐进，从小的测试（如拿饮料）开始积累。她每服从，你就奖励（IOI）；不服从，则给予IOD或轻微惩罚。以此积累**服从动能**。

## 互动模型与服从门槛

迷男方法有一套基础的**互动模型**，反映了人们社交互动的方式：

1.  你向目标展示 **IOD**（无兴趣）和 **DHV**（高价值）。
2.  目标有两种反应：**IOD**（无兴趣）或 **IOI**（兴趣指标）。
3.  如果收到IOD，则循环回第一步，继续展示IOD和DHV，直到收到至少三个靠谱的IOI。
4.  收到足够IOI后，发出 **CT**（服从性测试，如进挪）。
5.  目标对CT的反应：**IY**（服从）或 **IOD**（抗拒）。
6.  如果抗拒（负面服从），则对她发出IOD，展示DHV，重新吸引，再循环。
7.  如果服从，则**奖励她IOI**（同时可间歇性给予IOD调节）。

这个模型包含了社交互动的五种核心概念：**IOI, IOD, DHV, DLV, CT**。

**服从门槛**是指，当一个人服从到一定程度，感觉“过头了”时，就会开始抗拒（如女性说“我们太快了”）。我们使用互动模型和进挪升高的目的，就是不断去**突破她的服从门槛**（即底线），让她在你面前逐步放弃矜持和原则。无论是短期还是长期关系，让女性在你面前放下矜持，由你接管她的“自尊”，是关系变得亲密的关键临界点。

## 奖励的方式

当女性通过服从性测试（如进挪）后，需要给予奖励。奖励的方式包括：

1.  **赋格**：表达她符合了你的高标准。`“你还挺符合我对女朋友的想象的。”`
