# 《PIMP 核心笔记》：03：第三部分：外部技术模块：5. 收场：视频范例：15. 常见的“朋友借口”以及破解方法 🎯

在本节课中，我们将学习在夜店、酒吧或派对等社交场合中，当你向女孩发出邀请（如参加轰趴或冒险）时，她最常见的拒绝借口——“朋友借口”。我们将详细分析这一借口，并学习几种简单有效的破解方法。

## 常见的“朋友借口” 🚫

当你提出邀请后，女孩可能会使用与朋友相关的理由来拒绝。以下是最常见的几种说法：

*   不行，我还有朋友在这儿呢。
*   我不能丢下我朋友一起的。
*   我朋友和我一起来的。
*   对不起，我必须陪我朋友。
*   我必须和她在一起的，我们是连体婴儿。

你会反复遇到这类借口。接下来，我们来看看如何应对。

## “朋友借口”的破解方法 💡

破解此借口的关键在于，为女孩提供一个能让她暂时、合理离开朋友并与你相处的理由。以下是几种有效的解法。

### 解法一：短暂离开的借口

当女孩已经和你单独相处，但你计划带她去别处（如你家或另一个派对）时，可以使用此方法。其核心是让她告诉朋友一个需要暂时离开的正当理由。

*   告诉你朋友，你去趟厕所，然后到我这儿来。
*   告诉你朋友，你来和我说两句话，然后就回去。
*   告诉你朋友，你喝多了，然后给他发条短信。

这个解法非常有效，因为它制造了一个短暂且合理的分离。

### 解法二：提前退场的借口

如果你打算带女孩离开当前场所，这个解法能帮助她向朋友解释为何需要提前走。

*   告诉你朋友，你有点累了，你要先回家了。
*   告诉你朋友，你真的喝多了，你真的撑不住了，得先走了，得先回家了。

这个方法同样简单直接，为离场提供了合理的解释。

### 解法三：快速往返的提议

此方法适用于你想带女孩进行一个短途体验（如去你家看一眼），它降低了行动的“心理门槛”，强调离开时间很短。

**公式：** `提议去附近地点 + 承诺短时间返回`

你可以说：“我住的地方就在附近，我们去看一眼/去溜达一下。5分钟就回来，10分钟就回来，20分钟就回来。你朋友甚至都察觉不到你出去过。”

你要运用以上这些解法，来破解她提出的“朋友借口”。
