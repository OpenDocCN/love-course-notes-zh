# 《PIMP 核心笔记》：3.3：上钩模块 - 视频示范：12. 用热情与兴奋感吸引对方 😊

在本节中，我们将学习如何通过展现自身的热情与兴奋感，将互动对象自然地吸引并融入你所设定的氛围框架中。

上一节我们探讨了框架设定的重要性，本节中我们来看看如何通过情绪感染来强化这一框架。核心在于，你主动展现出的强烈情绪（如好奇、兴奋）能引发对方的好奇心，从而将她“吸入”你主导的互动氛围中。

以下是一个具体的应用实例：

我曾经在一次约会中，对女孩使用了这个方法。我指着地上的一个井盖，用非常热情和兴奋的语气说：“哇，你快看这个井盖！天哪，这简直太特别了！”

此时，女孩的典型反应会是充满疑惑地问：“啊？什么？怎么了？”

通过这个简单的互动，女孩的注意力被瞬间捕获，她被我表现出的热情和兴奋感所吸引，从而进入了由我发起和主导的对话框架中。

**核心操作公式**：**你的强烈情绪展示 + 指向一个具体事物 = 吸引对方进入你的情绪框架**

---

本节课中我们一起学习了“情绪吸引”的技巧。关键在于主动、真诚地展现你的热情或兴奋，以此作为“钩子”，引导对方产生好奇并进入你营造的互动氛围。这比单纯的语言内容更能有效地建立吸引和连接。
