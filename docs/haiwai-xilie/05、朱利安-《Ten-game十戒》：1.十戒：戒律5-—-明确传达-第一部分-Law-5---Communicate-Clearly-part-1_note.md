# 朱利安《Ten game十戒》：05：戒律5 — 明确传达 第一部分 🗣️

在本节课中，我们将要学习“明确传达”这一戒律的第一部分。核心在于理解，即使你内心状态良好，也必须通过清晰、有效的方式将这种状态传递给对方。这涉及到一些基本的沟通规则。

上一节我们介绍了建立内在状态的重要性，本节中我们来看看如何将这种状态有效地向外传达。

以下是确保有效沟通的几个基本要求。如果你未能满足这些要求，那么你实际上并未真正参与到互动中。

*   **确保对方能听清你**：如果对方听不清你说话，你就没有参与到互动中。她或许会勉强回应几句，但听不清会带来不便，互动将无法持续。解决方法是：**提高音量** 或 **靠近她的耳边**。如果你声音太小，这不会导向亲密关系。
*   **必须外出社交**：无论你当下是否想出门，如果你不走出家门，你就无法进行任何社交游戏。这是参与互动的前提。
*   **保持微笑**：微笑是传递积极状态最直接的信号之一。当互动进展不顺时，检查自己是否在微笑是一个有效的调整方法。
*   **保持眼神接触**：稳定的眼神接触能传递自信和专注，是连接双方的重要纽带。
*   **面向对方并保持适当距离**：你的身体朝向应该面对她，并且需要进入足够近的距离，以便她能充分感受到你的存在和能量。

这些是五个基本要求，它们的作用是帮助你有效地传递我们之前讨论的那种内在状态和氛围。

本节课中我们一起学习了“明确传达”的五个基本实践要点：确保可听度、主动社交、保持微笑、眼神接触以及适当的身体朝向与距离。它们是成功互动不可或缺的基础。
