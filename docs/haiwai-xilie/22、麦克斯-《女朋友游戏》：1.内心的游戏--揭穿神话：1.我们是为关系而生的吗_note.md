# 女朋友游戏：1.1：内心的游戏——揭穿神话：我们是为关系而生的吗？💭

在本节课中，我们将探讨一个核心问题：人类天生是为亲密关系而生的吗？我们将审视关于关系本质的不同观点，并分析几种常见的情感关系模式。

---

现在，让我们直接开始讨论这个问题。

你可能来自“把妹达人”的圈子，认为“我只想找女孩，恋爱很蠢”；或者你可能深受主流好莱坞式浪漫故事的影响，认为“恋爱当然是人生唯一重要的事”。这里有一个更接近事实的观点。需要说明的是，这里的某些内容可能会让你感到不快。这是一个全新的视角。

作为一名专业的约会教练，我基于在RSD见证过的真实情况，带来了这个观点。我们对于恋爱关系的看法非常清醒，并非外界那些被美化、充满糖衣的叙述。我们看到了真相。所以，让我们来谈谈它。

事实的真相是，是的，我们天生是为关系而生的。这确保了后代的养育，从而保障了我们物种的延续。如果你深入思考这一点，有一本很棒的书叫《性在黎明》，作者是克里斯托弗·莱恩和西塞莉达·杰斯，我强烈推荐这本书。

但你必须意识到：我们是为关系而生的，然而，我们可能并不完全是为**一夫一妻制**的关系而生的。我先把这一点提出来，因为它目前并不改变任何事实。关键在于，我们天生需要与异性建立亲密关系。关于“男朋友”这个角色，我们会在后面的章节详细讨论。

我们同样天生会经历不同的阶段，这些阶段可能持续数年。例如，四处游历、与许多女孩发生关系的阶段。这也很正常。同样，这些阶段可以持续数年。很可能你现在就处于这样一个阶段。

如果你了解RSD，或者正处于这样一个阶段，那么最重要的一点是：你正在学习**阿尔法男性行为**。在那些四处游历、与女孩发生关系的阶段学习阿尔法男性行为，这与和女性（或广义上的异性）建立亲密关系一样正常。

然而，完全不自然的是“新手”的角色。前面我们谈了“男朋友”（亲密关系），也谈了“玩家”（四处游历的阶段），现在我们来谈“新手”。这是完全不自然的：四处游荡却不采取行动。沮丧、困惑，这就是新手。这他妈完全不自然。

这是当今社会的产物：我们能接触到无限量的色情内容，我们被主流叙事所“阉割”。这不该存在。这完全不自然。大自然母亲没有预见到这一点。在这里，我们联系到了一个几年前发布的名为“自然”的课程，它探讨了什么是自然，男女之间自然的吸引力是什么，以及我们如何遵循它。

你看，新手感到沮丧和困惑，四处游荡却不行动、没有经验，这完全不自然。对于新手来说，生活中正在发生一件非常、非常糟糕的事情：他的**生物钟**。每分每秒都在滴答作响，把他逼疯。因为那个时钟知道：兄弟，你快要孤独地死去了。没有繁衍，没有获得任何经验。这就是为什么处于“新手”阶段非常糟糕。我曾身处其中，我自己也经历过。

那么，让我们谈谈下一个话题：什么是爱？什么是关系？老实说，这是一个深层的文化问题。我保证，你对于爱和关系的看法是100%被扭曲的。我的也是100%被扭曲的，因为这本身就是一个深层的文化概念。

我们的文化几乎不接受一夫多妻制或多边恋。无论你生活在印度、中国、北美还是欧洲，我保证你的文化接受一夫多妻或多边恋是非常罕见的。简单问问你的父母或朋友，告诉他们你除了女朋友之外还和其他女孩发生关系，每个人都会说：“哦，我的天，这太荒谬了，你怎么能这样？”

但再看看其他文化，例如印度的包办婚姻，或者在某些文化中，强奸被认为是女性的过错。如果你去迪拜，有多起案例是女性被强奸反被认为是她们的错，她们还被关进监狱。这多疯狂？你可以看到文化对我们的关系和爱情观念的影响有多大。

再看看像《性在黎明》这样的书。这些书采取了完全不同的、非常清醒的视角，类似于我们在RSD所做的：嘿，让我们看看关系本身。根本没有证据表明我们天生就适合排他性的一夫一妻关系。与我们关系密切的灵长类动物都没有这种关系，这很疯狂。如果你想深入了解，我再次推荐看看这本书。这里的很多观点都源自那本书。

我只是结合我自身、我的客户、同事和朋友们经验来阐述。那么，我为什么说这些？为什么谈论文化？听着，我不是要求你重写社会规范。我只是要求你对改变你自己关于关系的观念保持**开放心态**，并挑战你迄今为止认为的唯一方式。

猜猜怎么着？如果这听起来很熟悉——挑战你认为的唯一方式——这可能和你刚开始学习搭讪时的情况一样。你曾认为“正常”的做法就是那些被稀释的主流废话建议，比如“只要友善”、“做你自己”，这些对任何人都没帮助。然后你发现了我们这类内容，比如“做个男人”、“保持框架”等等。所以，你需要在这里也保持一定程度的开放心态。这正是我要求你在这个课程中做到的。

接下来的三个视频，我特意放在开头，因为你现在需要**自我评估**你处于这个三阶段结构中的哪个位置。你需要评估你现在在哪里，你的误区是什么，你当前的信念是什么。然后，我们将努力把你提升到下一个阶段，并为你创造一个开放的心态，准备好采取行动并向上迈进。

接下来我们将讨论：
1.  新手角色（新手阶段）
2.  玩家角色（玩家阶段）
3.  男朋友角色（男朋友阶段）

这三个阶段各自都有特定的错觉。新手全是关于等待，认为事情会以某种方式自动发生，充满了幻想。玩家已经看到了曙光，更像是“哦，我准备好了，去他的所有恐惧”，但他难以放松、与女孩深入交流、感受甚至稍微放下防备。而男朋友也充满了错觉，因为他需要保持敏锐、保持吸引力，才能留住他的女人。

这三个阶段都是完全自然的，你不需要对其中任何一个感到糟糕。自然的过程会引导你度过它们。那么，让我们开始吧。
