# 老板任务：11：国际象棋任务 🏆

在本节课中，我们将学习本周的任务：国际象棋。我们将探讨选择国际象棋作为任务的原因，分析它能带来的核心益处，并明确你的具体实践步骤。

上一节我们介绍了任务背景，本节中我们来看看选择国际象棋的深层原因。我并非凭空选择这个任务，而是基于大量研究。研究发现，每一位成功的企业家或大型公司的CEO，都曾投入时间学习国际象棋。这是一项极具价值的技能，能带来多重益处。

以下是国际象棋带来的主要益处：

*   **延迟满足**：国际象棋教你延迟满足。例如，在社交场合，你或许知道应该等待更合适的时机采取行动，但冲动却让你立刻行动。这就是缺乏延迟满足。国际象棋能训练这种能力。
*   **战略思维**：它教授战略技能。这类似于《孙子兵法》的理念，要求你理解全局和对手，而非仅仅攻击眼前目标。你需要思考长远策略。
*   **识人与自我管理**：游戏过程中，你需要运用观察力，分析对手的性格与思维模式（例如，是否易怒、反应快慢）。同时，它也是管理自我情绪的绝佳机会。当理性与感性冲突时，是情绪化地走出错棋，还是冷静下来相信过程并做出正确决策？
*   **应对逆境**：国际象棋教你如何在逆境中思考。当你失去重要棋子时，是转入防御模式苦苦支撑，还是设法寻求和棋甚至绝地反击？这本质上是人生的模拟。
*   **建立正确人生视角**：人生不是跳棋，并非“一步一杀”。人生是国际象棋，是一场讲究长期策略的游戏。你会有损失，也会有胜利，关键是在大局中保持前进的节奏。

理解了国际象棋的价值后，我们来看看你的具体任务。

以下是本周的任务要求：

1.  **进行1到2次国际象棋对局**。面对面进行为佳，因为这能提供真实的人际互动和观察机会。如果找不到对手，在线进行也可。
2.  **如果你是新手**：建议通过Meetup.com等平台加入象棋小组，或直接邀请朋友一起喝咖啡、下棋。
3.  **关键心态**：对局时，如果输了，不要沮丧。要学会欣赏自己在失败中学习的过程，对此抱有感激之心。
4.  **任务后反馈**：完成任务后，请在BOSS任务的应用或Facebook群组中发布你的反馈。描述你的经历、学到的关于自己的东西，并回答以下问题：你学到了什么？经历了什么？看到、感受到、听到了什么？我将阅读每一条反馈并至少给予回应。

本节课中我们一起学习了国际象棋作为人生训练工具的多重价值，包括培养延迟满足、战略思维、识人能力、情绪管理与逆境应对。你的任务是完成1到2次对局，并带着学习的心态去体验。我期待看到你们的进展。
