我发现生活中，大多数人倾向于对成就采取一种我称之为“防御性”的视角。这有点像你在学校里学到的东西：你参加一场考试，重点在于不出错，在于不犯错误。然而，生活并非如此运作。

在生活中，那些取得惊人成功的人会犯下海量的错误。他们只是把足够多的事情做得极其出色，以至于那些错误变得无关紧要。因此，他们是以一种**积极主动**的方式在“游戏”，而不是试图通过只丢两分来获得98分。他们的目标是得到100分，同时可能丢500分。他们以一种进取的方式，努力超越、突破任何人的既有认知。

为了更清晰地理解，我们可以用一个**公式**来描述这两种心态：
*   **防御心态**： 结果 = 避免犯错 → 追求有限范围内的完美。
*   **积极心态**： 结果 = 大量正确行动 - 可承受的错误 → 追求突破性的卓越。

上一节我们探讨了心态对结果的影响，本节中我们来看看如何将这种积极心态应用到具体场景中。

### 从网球到生活：进攻的必要性

假设你开始打网球。起初，如果你的水平很差，想要成功，你只需要不断把球回过去，很快你的对手就会失误。但在高水平的网球比赛中，这种策略行不通，它实际上会导致失败。因为迟早你会遇到能真正打出制胜分的对手，或者能主动得分的对手，那时你就无计可施了。

因此，如果你渴望达到高水平，你必须主动进攻。你不能只打防守，不能只是等待对手对你做什么。你必须开始**主导节奏、掌控局面，并让事情发生**。这在任何情境下都是真理。

以下是积极心态在不同领域的应用示例：
*   **销售情境**：你不能只是走进去，等待客户自我推销。你必须主动去销售。
*   **竞争领域**：在任何竞技场中，情况都是如此。
*   **与女性互动**：这一点尤为明显。

### 积极心态如何加速学习与成长

如果你只是走进去，等待事情发生在你身上，你的大脑将很难分析你得到的反馈。你进去，发生了一件随机的事——“好吧，随便”。下一次互动又发生了另一件不同的事——“随便”。这无法分析。

但如果你走进去，试图让某事发生——比如你试图进行肢体接触——你会得到各种回应，这些回应会直接针对你刚才的行为。你的大脑可以根据**因果关系**来解读这些反馈。

用**代码**来类比这个过程：
```python
# 被动反应模式（难以学习）
feedback = random_event() # 反馈是随机事件
learn_from(feedback) # 难以从随机事件中学习

# 主动创造模式（易于学习）
action = take_proactive_step() # 采取主动行动
feedback = get_response_to(action) # 获得针对该行动的回应
learn_from(feedback) # 易于从因果关联中学习
```

因此，学习成长变得非常、非常容易。如果你渴望成长，渴望卓越，渴望在任何事情上达到伟大的水平，采取防御姿态，只是走进去希望不犯错，希望侥幸过关，蹒跚地得到一个结果，这是行不通的。

你必须走进去，努力打出全垒打。你必须走进去，努力表现得令人惊叹。这不仅会在当下为你带来成功，也会在学习和成长方面助你一臂之力，让你在每一次后续尝试中变得更好。
