# 《PIMP 核心笔记》：3. 第三部分：外部技术模块：4. 创造气氛、情绪共振：视频示范：30. 查户口话题植入强大潜沟通 💬

在本节课中，我们将要学习如何通过看似简单的“查户口”式提问，在对话中植入强大的潜沟通信息，从而建立更深层次的情绪连接。

许多人在与女孩聊天时，会持续使用“查户口”式的问题进行对话。选择这种方式，是因为这类问题本身非常简单，且每句话都很短促。例如，你可以问：“你是哪里人？”、“你多大了？”、“你做什么工作？”或者“你一会儿要去哪？”。

在这些基础的查户口话题中，你可以植入更深层次的潜沟通潜台词。这种沟通方式能够植入强大的前沟通信息。这种方法远比谈论那些内容上看似有趣、好玩，但实际上冗长、复杂且停留在肤浅层面的话题要有效得多。

因此，在我的聊天实践中，我会在查户口话题中，着重植入我讲话时的感觉和态度。

现在，请回顾你纸上记录的那五句话。这五句话构成了你整个聊天大厦的基本结构。查户口话题是搭建这座聊天大厦的基石。

在练习这个聊天结构时，你需要对每一个在现场搭讪的女孩，都说出这五句查户口话题。让这种做法成为一种习惯，并与我们之前课程中讲解过的其他内容结合起来运用。我本人在搭讪每一个姑娘时，都会贯彻这一方法。

本节课中，我们一起学习了如何将查户口话题作为对话的框架，并通过在其中植入潜沟通来提升互动深度。关键在于将简单的提问与你的个人态度结合，使其成为一种有效的沟通习惯。
