通常那些背负着所有内疚的人，就像再次，在训练营里，就像……你就是不能，就像，你不让那些过去，在那之后，无论你做得多好，你就是无法接受。是的，而且，嗯，你从不接受，就像你不能允许自己感到完全快乐。你不能允许自己承认自己的成功。一旦事情变得有点太好，你就不断地搞砸它，因为你觉得自己不配得到它，因为，你知道，就像，你知道，要么你会背叛另一个人。而且因为有那种自我憎恨，就像你甚至不想，你就像，我不被允许成功，因为我搞砸了那个。我不被允许，因为我不是一个好人。我不被允许，因为我不喜欢自己。

**是的，那是真的。是的，那个……我想这也阻碍了我……完全地，像完全地……哀悼，就像我……我已经大概五年没哭过了。而且发生了很多事，像很多……不同的事情发生了，按理说……你应该哭。**

**为什么不？**

所以有时会发生这种情况，你开始释放一些非常真实的东西，比如，你知道，这种自我憎恨，这种内疚，你有点意识到这没关系，你知道，开始对它有一些觉知是可以的，然后更多的事情会浮现，比如，你知道吗，我也很久没哭了。你知道，然后，当然，你可以跟随恐惧到那里。而且这非常普遍，比如情感淡漠，人们无法哭泣，无法大笑，无法感受。现在，这里也有一个小提示，阻力会以这种方式显现：你只是不断提出更多更多的事情，以避免深入第一个问题。所以你必须感受一下，并告诉自己，嘿，我在这里提出这些东西，是诚实的，因为，你知道，这没关系，我只是想继续释放一些真实性，还是我在这样做以避免深入其中？所以有时我会打断别人，我就像，不，不，不，停留在第一个问题上，或者，好吧，这里发生了什么。我们也深入探讨那个。

**你哭的话就是个小娘炮。是的，那就像无处不在。就像作为男人……就像，如果你哭就是娘炮。就像，像个男人一样。你是个娘炮吗？像个男人一样。不允许哭，但我……我知道……完全地……就像我读了很多……像灵性书籍之类的，所以我……对感受它没问题，但我仍然有自我意识，不想让自己感受它。为什么？恐惧在哪里？如果你做了，会发生什么？**

关于情感淡漠，也有一个小提示，一个极其常见的陷阱是人们把它视为好事。他们把它视为目标。他们视自己为开悟了。猜猜怎么着？我从不哭。我再也不难过了。我一定是开悟了。不，因为那仍然是对哭泣的阻力。开悟意味着完全的自我接纳，当你处于全然不抗拒的状态，你欢迎一切。情感淡漠是……阻力，情感淡漠是麻木。我经历过一段这样的时期，当……我在媒体丑闻期间，在我跌入谷底之前。有一段时间，我只是像，你知道吗，我现在很好，但我只是处于完全的麻木状态，那并不好。你知道，那是你在欺骗自己。因为事情只会变得越来越糟，越来越糟，直到我跌入谷底。那是，你知道吗，这里很好。然后它们只是越来越糟，失望接踵而至。之后我就像，你知道吗，去他妈的。然后你就有点放弃了。你以为那是，哦，我放弃了。但不，那只是你变得麻木了。而那又是你，抛弃了自己很大一部分。而且你这样做得越多，你越会意识到你开始抛弃其他情绪，甚至包括快乐，直到你甚至他妈的什么都感觉不到，只是完全的情感淡漠，不只是哭泣，而是一切，你只是……一个人的空壳。你知道，那不是开悟，那不是你追求的。你需要深入对感受的阻力。如果你哭了，有什么不好？如果你笑了，有什么不好？如果你感受了，有什么不好？你知道，你来这里是为了感受。没有感受的生活不是生活，你来这里是为了拥有那种对比和那种体验的范围。

培养对自己的同理心，释放那种阻力。调谐到当我说“感受”，当我说“体验负面情绪”时，浮现的东西。释放那些。

**总结**

在本节课中，我们一起学习了如何识别和处理转变过程中的内在阻力。我们了解到，阻力会以各种形式出现，例如：停留在表面为自己找理由（“因为感觉不好”）、用“我不知道”来阻断探索、强迫性行为（如取悦他人）、固着于识别出的模式而不深入原因、用“不值得”等借口掩盖真实感受（如“不配得”）、以及压抑真实浮现的情绪（如自我憎恨、内疚）。关键在于始终问“为什么”，并不断深入，直到触及核心的恐惧和信念（如害怕孤独、不被爱、不值得）。处理这些深层情绪，如自我憎恨和内疚，并通过接纳和原谅过去的自己来释放它们，才是实现根本转变的途径。记住，目标是完整的自我接纳和感受的能力，而非情感上的麻木或逃避。
