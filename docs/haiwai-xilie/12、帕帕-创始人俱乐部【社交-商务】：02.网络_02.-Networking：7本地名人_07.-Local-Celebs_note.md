# 帕帕创始人俱乐部【社交 商务】：02：网络：7 本地名人 🎯

在本节课中，我们将学习如何识别并有效地与“本地名人”建立联系。这些人在当地社区拥有影响力和资源，掌握与他们交往的技巧能极大地扩展你的社交网络。

## 概述

每个城市，无论大都市还是小镇，都存在本地名人。他们拥有影响力，能接触到不同的事物。本节将探讨如何通过共同兴趣自然地与这些人建立联系，并利用这种联系了解他们吸引社区其他人的特质。

## 识别与接触本地名人

上一节我们介绍了网络的重要性，本节中我们来看看如何定位并接触你所在社区的“中心人物”。

无论是在哈佛想结识被称为“哈佛市长”的社交名流，还是在拉斯维加斯想接触紧密的创业者社区核心人物，抑或是在洛杉矶认识那位知晓所有热门派对和人物的“社交蝴蝶”，关键在于**以正确的方式与他们建立联系**。这种能力是许多人都希望掌握的。

## 建立联系的核心心态

与本地名人交往时，心态至关重要。以下是你需要秉持的核心原则：

*   **纯粹交往**：与他们相处时，应注重乐趣本身，而非试图索取价值，如金钱或特殊渠道。即使你暂时无法提供对等的价值，愉快的相处本身就能为长远关系打下基础。
*   **成为连接器**：如果你已经结识了一些名人，将不同城市的他们相互引荐会非常有力，因为这样你就成为了**连接器**。名人通常深谙人际网络的价值，他们的地位往往就来自于高超的社交能力。
*   **尊重与平等**：像对待朋友和家人一样对待他们。秉持“每个人都是你关心的人”这一理念，尊重其作为个人的价值。更重要的是，**尊重他们的时间**。本地名人的时间通常非常宝贵。

## 寻找共同点与提供价值

要跨越时间障碍，关键在于发现他们关心的事物并提供相应价值。

*   对一些人可能是**金钱**。
*   对另一些人，如唐纳德·特朗普，可能更希望你**捐赠给他关心的慈善机构**。
*   还有人则欣赏那些**支持他们共同愿景和目标**的人，例如支持某项社区艺术活动。

**共享共同兴趣非常有价值，但更有效的是共享他们社交圈内的共同朋友。** 有些人能如此深入地融入你的社交圈，以至于你不可避免地会与他们成为朋友。

## 出现在正确的地点

本地名人的朋友不一定是名人，大多是普通人。名人并非只与其他名人交往。

然而，如果你经常出现在成功人士和受欢迎人士聚集的**同类场所**，你就有更多机会与他们产生小范围的互动。你频繁出现在这些场所的次数越多，结识他们的可能性就越大。这一点同样适用于结识极具吸引力的女性。

以下是具体方法：

*   **邀请他们参加社交活动**，尤其是你知道他们常去的固定场所。
*   **利用现有的兄弟会、组织或社交圈网络**。许多强大的社交网络建立在已有的成熟结构之上。

例如，在洛杉矶，借助像迈克尔·萨顿（一位著名娱乐推广人之子）所建立的、连接好莱坞名流、模特与商人的庞大社交基础设施，比从零开始构建网络要高效得多。通过付费加入这类组织，可以获得接触结构的权限，并最终发展出真挚的友谊。

## 主动研究与融入组织

寻找并融入合适的组织需要主动研究。

*   这不像电影里的秘密组织。许多组织并不主动营销，只是存在的社交圈。
*   它们可能有一个网站，也可能需要通过**口碑**和询问圈内人来了解。
*   你可以花时间研究如何加入这些组织，无论是为了商业还是社交目的。

例如，**RSD Nation** 就是一个我们为学员创建的“智囊团”基础设施结构。我们的本地名人就是像泰勒这样在全球教授课程的导师。

通过融入多个城市的此类组织，你可以建立一个由各领域“名人”组成的大型网络——因为几乎每个人都因某事而知名，无论是商业、特定技能，还是其他资源。

## 总结

本节课中我们一起学习了与本地名人建立有效网络的方法。关键在于以正确的心态（纯粹、尊重）接触他们，通过寻找共同点和提供价值来建立联系，并主动出现在他们所在的场景或融入现有的社交组织中。结合本课程教授的所有网络技巧，理解并找到那些适合你生活方式、目标和需求的本地兄弟组织，你将能构建一个强大而多元的社交网络。
