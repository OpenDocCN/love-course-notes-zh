# 约会学：17：B计划：跟随她 🚶‍♂️➡️🏠

在本节课中，我们将学习当无法按原计划（A计划）将女孩带回你的地点时，如何灵活切换到B计划——跟随她去她的地点，并最终达成目标。我们将探讨如何应对复杂的社交情境，逐步推进，直至创造私密空间。

上一节我们介绍了A计划的核心执行逻辑。本节中我们来看看，当A计划因客观条件限制无法实施时，我们该如何应对。

## 接受A计划的局限性

A计划并非总能成功。你面对的是人，而人际互动总存在随机性，客观条件（如社交情境、同行朋友的状态）也可能不允许你执行A计划。例如，女孩可能说：“我和我喝醉的室友在一起，我是指定司机，我得送她回家，不能离开她。” 在这种情况下，强行执行A计划可能会让女孩感到压力过大，显得你过于固执，反而可能失去她。

因此，有时需要接受A计划行不通。但这并不意味着放弃。对于你已决定要带回家的女孩，如果A计划受阻，就启动B计划：**跟随她**。改变行动路线，最终去她要去的地方。你仍然可以进入“豁出去了，没什么可失去”的心态。

## B计划的核心逻辑：创造最终独处机会

B计划不像A计划那样完全由你掌控（“我带她回家，一切就绪，我为每一步都找好了借口”）。现在你需要面对更多未知。但目标依然可以实现：记住，女孩和她的朋友（比如室友）**最终总会分开**。你只需要在夜晚结束时，确保自己与女孩处于**一对一、隔离**的状态，这样你就有机会让事情发生。

## 实战案例分解

假设你在夜店后半场认识了一个女孩。你开场、建立吸引、筛选、并判断了客观条件。你决定：“就是她了，我今晚要花时间在她身上，最终要发生关系。” 此时她的“下载进度条”（兴趣指标）很高，已准备好进行亲密行为。

但她的客观条件是：她和喝醉的室友在一起，她是指定司机，她们形影不离，无法去你的派对。A计划行不通。

以下是应对步骤：

### 第一步：改变路线，提出同行

你的目标是和她一起行动。首先，不要想得太远，先解决**如何上车**。因为她要回家，你需要全程和她待在一起。

以下是你可以说的话术示例：
*   “真的吗？你住哪个区？… 巧了，我也住那边/我朋友也住那边。”
*   “我正要去那边的一个派对/朋友家。你能捎我一程吗？”

在她的思维里，她会想：“好吧，反正他找了个小理由/借口，我可以顺着这个理由让他跟着，没问题。” 你的目标是说任何能让你**更近一步**的话。

### 第二步：在途中埋下“种子”

上车后，你需要为下一步进入她的公寓做铺垫。如果到达目的地后你突然说“我能上去吗？”，会显得很突兀，让她措手不及。

因此，在车上或同行途中，你需要**埋下“种子”**。例如：
*   “你家里有酒吗？有伏特加吗？我们待会应该喝一杯。”
*   “我现在真的需要用下洗手间。”

通过这样做，你为“喝酒”或“用洗手间”创造了理由。这样，到达她公寓楼下时，你就可以引用之前埋下的“种子”：“对了，不是说喝一杯吗？” 或 “我实在得用下洗手间了。” 这不会显得太突兀或诡异。

### 第三步：到达后，逐步进入并留下

到达目的地后，你的目标是说任何话进入公寓，然后想尽办法**留在里面**，直到室友最终去睡觉。

以下是进入公寓的借口示例：
*   “我在给我朋友发信息，但他没回。我能在楼上和你们一起等吗？我不想在外面冷风里站着。”
*   “我朋友拿着我的钥匙，我进不去门了。我能跟你们上去等吗？他很快就到。”
*   “就喝一杯告别酒嘛。”
*   “我能用下洗手间吗？我从来不用公共洗手间。”
*   “我能送你到公寓门口吗？我是绅士。… 我能看看你公寓的view吗？我正在这附近找房子。”
*   “我就上去看一眼，马上下来，我保证。我累了，什么也不会发生。”

**核心是：说任何能让你更进一步的话。** 一旦进入公寓，就运用同样的原则（为她卸下责任、逐步推进），想办法留下来，直到你与女孩独处。

## B计划的通用原则与心态

1.  **核心公式**：**跟随 -> 创造独处机会 -> 推进**。
2.  **保持轻松氛围**：当你跟随她（比如步行送她回家）时，不要让她感到尴尬或自觉。不要问“我们这是一起走回家吗？”。保持轻松的闲聊，就像你们在自己的小气泡里。话题可以是：“你什么时候搬来洛杉矶的？”、“你有兄弟姐妹吗？”。你不是在继续“游戏”或“推拉”，你只是**坚持待在她身边**。
3.  **细节保持模糊**：在对话中，关于你自己的去向等信息，开始时尽量保持模糊，必要时再补充细节。例如，如果她问你去哪，你可以说“我也往这个方向”或“我朋友在这附近有个派对”，而不要一开始就给出精确地址。
4.  **地点不限于公寓**：B计划的目标地不一定是她的公寓。如果她去停车场，可以尝试在车里；如果无法离开场所，可以尝试在洗手间。关键是**她去找，你就跟到哪里，直到找到可以隔离的机会**。
5.  **拥抱不适感**：执行B计划时，你可能会感到尴尬或“像个变态”。例如，默默跟着女孩走回家。你必须拥抱这种感觉。你想成为夜晚结束时最后站在她身边的人。当你们最终独处时，她的心态会发生**转变**：从“这家伙想干嘛？”变成“反正没人会知道，我也喜欢性，何乐而不为？”。这种隐私和便利性会极大降低她的心理防线。
6.  **接受有时仍会失败**：有时，即使你送她到家门口，她仍会说“抱歉，你不能上来”。不要为此沮丧。把它看作一次宝贵的处理复杂社交情境的练习。你增加了“得分”的机会。
7.  **养成习惯**：将B计划变为习惯。成为那个**坚持待在互动中、不轻易离开**的人。就像冰球比赛，想增加进球概率，就要**不断把球打到球门前**。执行B计划，就是不断将自己置于可能发生亲密关系的各种情境中，把“球”打到“球门”前。

本节课中我们一起学习了当A计划受阻时，如何切换到B计划——跟随女孩去她的地点。我们拆解了从提出同行、途中铺垫、到进入并留在目标地点的具体步骤和话术。关键在于保持灵活，坚持留在互动中，利用一切机会创造最终的一对一独处环境，并始终记得为她卸下心理责任。通过拥抱过程中的不适感并养成执行B计划的习惯，你能显著增加成功的可能性。
