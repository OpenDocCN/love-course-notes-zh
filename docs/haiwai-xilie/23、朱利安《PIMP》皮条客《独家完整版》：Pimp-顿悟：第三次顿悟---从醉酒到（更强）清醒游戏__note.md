## 我的个人经历

我个人很幸运，最初是以清醒状态进入这个领域的。17岁时，在接触“游戏”概念之前，我在瑞士有过一次小事故。瑞士允许饮用啤酒和葡萄酒的年龄是16岁。在那次学校派对上，我喝得烂醉，那是我人生中最糟糕的经历之一，几乎给我留下了心理创伤。自那以后，我就决定不再喝酒。所以当我后来发现“游戏”时，我正处于完全不饮酒的时期。因此，我从未想过要借助酒精。

我走的是艰难的道路。我在这个系列的第一个视频中谈过，我以清醒状态面对接近焦虑，在街头做日间搭讪，内心感觉快要死掉。第一次去夜店也是人生中最糟糕的经历之一，我害怕极了，不知道如何与人交谈，最后只是坐在巨大的音响旁边，身体像要垮掉一样，等待清晨的第一班火车回家。所以，我是在清醒状态下处理了所有必须面对的心理创伤和麻烦。

有趣的是，即使经历了所有这些，我也从未想过要喝酒。这可能也因为那是七年前，饮酒问题不像现在这么突出。那时的许多游戏哲学也更注重“惯例”，如果你喝得太多，就记不住惯例了。所以，我清醒地游戏了一段时间。

后来在大学时期，就在我辍学搬去洛杉矶之前，我对生活极度不满，变得非常虚无主义。我想“去他的”，于是开始喝酒和外出饮酒。

## 饮酒的“好处”

为了全面分析，我们先从饮酒的一些所谓“好处”或优势说起。很多人这样做，很多人喜欢外出喝酒并与女性互动。我曾经也喜欢这样做。

以下是主要的几点优势：

1.  **赋予你展现自我的许可**：当我开始喝酒时，我注意到它允许我，或者说给予我自己许可，去更多地展现原本的个性。你感觉更有资格，更不在乎别人的看法。当你喝醉时，你更倾向于做真实的自己，你会想“这就是真实的我，去他的，人们会喜欢的”。这让你能在那个层面上扩展自我，或认识到内心深处你是谁、你喜欢什么。它允许你走得更远。我记得当我喝酒时，我开始尝试在互动中“试探边界”，看看我能推进到什么程度。这让我了解到，当我毫无保留、完全不在状态时，内心深处自然流露出来的是什么。而且，你总是可以把责任推给酒精，不必承担太多责任。你可以进行实验，如果人们反应激烈，你可以说“听着，我喝醉了，对不起，我平时绝不会这样”。这就像生活在一个小小的现实里，在那里一切都被允许。

2.  **让你更少顾虑，更专注当下**：因为你更不在乎了，你更专注于自己。你不会想得太远，也不会沉湎于过去，你只是处于当下那种醉醺醺的朦胧感中。这让你能够不去思考，只在当下采取行动。很多新手从饮酒中获得了这一点。如果他们无法接近女性，喝几杯后就可以做到了。当然，这是暂时的，我们会在弊端部分讨论。但它让你开始采取行动。很多男性只是僵在原地，但喝酒后，你不会考虑后果，不会想过去的经验或可能出错的事情，你只会想“去做吧”。这让你以一种低于意识层面而非高于意识层面的方式活在当下。

3.  **给你一些事情做**：很多男性在外出时面临的困难就是干站着，尤其当他们不去接近别人时，什么都不做会让人焦虑。如果你手里有杯酒，你可以小口啜饮，或者至少拿着它。这让你看起来像是在做点什么，而不是独自傻站着。有时，你也会感觉更融入环境，因为其他人都在喝酒。如果你不喝，虽然没人会真的指指点点，但你可以用喝酒这个动作来隐藏自己。

4.  **给你更多能量**：我记得在我最初清醒游戏的日子里，我遇到一个朋友，他总是非常积极、快乐、精力充沛。我问他怎么做到的，他说：“伙计，是酒精。”酒精能给你能量的冲击。当然，这种能量通常在夜晚结束时消退。但在开始时，如果你喝点像伏特加红牛这样的东西，你会感觉精力充沛，假设你不是那种喝酒后变消极的人，你可以像有无穷能量一样推进。你不需要有意识地调动储备，你就像拥有无限的能量。

在我看来，以上是饮酒的四个主要“优势”。

## 饮酒的弊端

现在，让我们看看弊端。

1.  **耐受性递增与临时解决方案**：起初，你可能因为无法接近女性而想“也许喝一杯我就能做到了”，或者喝一两杯就能进入状态。但随着时间的推移，就像任何事情一样，你会逐渐对那个酒精量脱敏，你总是需要越来越多。最终你会达到一个点：你需要喝很多才能进入那种放松的状态，但另一方面，当你达到那个状态时，你已经完全醉了。所以，一开始可能不错，作为新手，一两杯酒让你放松但仍保持一定意识。但随着脱敏，你需要更多，然后你就醉得一塌糊涂，以至于游戏表现很差。**这只是一个临时解决方案**，长期来看并不好。

2.  **过早达到峰值**：我不断强调的一点是，夜晚最重要的部分是后半段。那时你才能开始进行更长的互动，女性的“购买温度”更高，她们最终为性做好准备。而前半段真的不重要。如果你喝酒，在前半段你会疯狂地和每个人交谈，你可能玩得很嗨，但你也可能因为过于活跃、不合时宜而“烧毁”整个场地。然后到了关键时刻，你已经累了，或者因为想保持感觉而喝了更多酒，结果醉得太厉害。当需要取得结果时，你已经不在状态了。这就像你在练习赛中打得很猛，但当真比赛开始时，你已经精疲力竭，或者因为太醉而无法准确判断时机，甚至看不到带女性离开的机会窗口。

3.  **无法记住参考经验，影响不深**：我长期以来的一个观点是：**游戏是什么？最好的进步方式就是在尽可能短的时间内获得尽可能多的参考经验**。如果你外出喝酒，你可能采取很多行动，做疯狂的事，和很多女孩交谈甚至带她们回家。我也有过一些喝醉后的精彩经历，但我并不总是记得所有细节。当你记不住时，你就错过了所有这些最终能让你个人成长或游戏进步的参考经验。如果你今晚和20个女孩交谈，并记住了所有细节，你会锚定很多教训，内心会发生很多变化。如果你一个都不记得，你可能会进步一点，但不会像你应该的那么快。快速进步的理想方式是外出时尽量记住发生的一切，回家后坐下来复盘，写下发生的一切，并将每件事与概念或视频联系起来。每当你这样做，你就在内化知识。如果你喝醉了，你什么都记不住，所有这些参考经验都浪费了。其次，正如之前所说，它们对你的影响也不会那么深刻。真正的进步是当某些事情在深层情感上影响你时，无论是像接近焦虑这样的负面情感创伤，还是一次惊人的成功带来的强烈正面冲击。这些深刻的冲击才能改变你的核心。如果你喝醉了，你就剥夺了这些参考经验本应具有的强烈冲击力。

4.  **无法发展核心自信，只能发展情境自信**：这是最大的一点弊端。当我戒酒之后，我深刻理解了一句话：**你的个性是在你最常处的状态下发展起来的**。这意味着什么？如果你经常外出喝酒，并与女性互动，你仍然会取得一些进步。但是，你所取得的进步将被锚定在“醉酒”这种状态的人格矩阵中。要获取所有这些进步或你发展出的魅力，你总是需要处于醉酒状态。当你清醒时，你就像变了一个人。如何判断自己是否如此？问问自己：如果你白天看到一个女孩，那些没有发展出核心自信的男性会想“哦，今晚还会有其他像她一样的热辣女孩，我等晚上喝了酒再说”。如果你有约会，你可能会想“我可能应该在约会前喝几杯，才能展现出那种个性”。所以，你并没有真正改变“你”，你改变的是“醉酒的你”。你所取得的那点进步，改变的是醉酒的你。这是我决定戒酒的主要原因之一。我想在深层改变自己，我想要核心自信。我想达到一种境界：无论白天在夜店，还是其他任何时候，我都能完全清醒地召唤出这种状态。我不需要依赖酒精。

这些就是饮酒的弊端。**弊端远远大于好处**。如果是这样，那就彻底戒掉。我现在看待生活中任何事情的方式都是：**它带来的好处是否多于它索取的代价？如果答案是否定的，那就彻底摒弃**。你可以用这个标准看待任何事情，比如约会一个情绪消耗极大的女孩，那短暂的愉悦是否值得其余的所有麻烦？如果不值得，就彻底摒弃。

所以，基本上，如果你想变好，如果你想在深层真正改变自己，你必须保持清醒。

## 如何从醉酒游戏过渡到清醒游戏

如果你正在看这个视频，并且已经在喝酒玩游戏，这个转变会非常困难。

当你第一次在清醒状态下外出时，你会注意到以下症状：

*   **感觉不完整**：你感觉没有完全在场，你意识到自己真实的个性可能很糟糕，因为你只发展了醉酒时的人格。
*   **感觉像个局外人**：你环顾四周，看到每个人都在玩乐、跳舞、处于状态、喝酒。你觉得和他们不在同一个状态，感觉自己像个巨大的局外人。
*   **经历很多内心抗拒**：你在那里，但你不开心。你知道如果喝一杯，你就能开心起来。你会想“我不开心，但我希望我开心。如果我喝了酒，我就会开心。该死的，现实不是我想要的”。这种内心抗拒会产生很多负面情绪。
*   **感到非常疲惫**：你不会拥有同样的“杀手本能”去完成收尾。你会想“我累了，去他的，不知道值不值得”。你可能会专注于所有可能出错的坏事，而不是让它变得精彩。
*   **不知道如何自娱自乐**：你需要学习实际的热身过程。喝酒时，你可以绕过这个过程，酒精让你自然跳出思维。但清醒时，你困在脑子里，你需要知道如何在不借助酒精的情况下，实际地让自己进入状态。

现在，为了帮助你度过这个阶段，我能给你的最重要的心态是：一方面，接受你为了更大的利益必须在进步上暂时退一步。如果你从醉酒状态转到清醒状态，而你还没有发展出清醒时的人格，你会直线下降。但当你以正确的方式重建时，你将能达到比醉酒时高得多的水平。这就像先下降，然后再上升。你必须接受这一点，并真正让自己处于“滴酒不沾”的立场。完全不要喝，即使你想“就一杯，就两杯”。一方面，你仍然是在围绕那一两杯酒发展个性；另一方面，这更容易，因为会有很多抗拒，比如“我只想喝一杯”。制定一个严格的规则更容易：无论如何，不喝酒。你可以做一个30天挑战：**30天零酒精**，无论我感觉如何，无论我多么消极。这能消除那种抗拒。

另一种消除抗拒的方法是直接告诉女孩，并自嘲。比如：“嘿，我真希望我喝了酒。如果我喝了酒，会好玩得多。我还没发展出清醒时的个性，我总是需要喝酒。天啊，我真是个酒鬼。”看，这甚至成了你的开场白。

此外，正如我提到的，你需要学习如何热身。这非常重要。遵循四条法则，采取大量行动。当你喝酒时，你可以坐等酒精自然发挥作用。但当你清醒时，如果你只是坐着，你就完了。你会陷入思维，状态不会自然出现。所以，找到让自己进入状态的方法。遵循四条法则。另一个好方法是，如果你是第一次清醒外出，并且有同伴，可以玩“五秒游戏”。告诉你的同伴，在接下来的五分钟里，你不允许离开组合超过五秒，如果超过了，他就用尽全力打你的肩膀同一个位置。于是你倒数5、4、3……然后你会想“糟糕”，接着就开始不停地跑动、接近。**当你的行动快于你的思维时，你就能以这种方式让自己进入状态**。

同样要接受的是，重建需要很多时间。你对自己游戏水平的印象还停留在“还不错”，但你必须接受回到近乎新手的地位。你可以把这当作一种强化：你之前没有以正确的方式发展个性。你会想“我应该能清醒地做到”。但之前我用了作弊码，去他的。这一点很重要，因为如果你经常醉酒外出，你的朋友们已经习惯了你那种水平。我记得当我戒酒时，正是我开始协助泰勒工作的时候。在训练营里，我清醒地辅助，会想“学生们会怎么看我？如果我喝了酒，我的游戏水平会高得多”。你必须放下那种自我形象，真正投入到长期目标中：学会如何进入状态，超越意识，享受清醒的乐趣。

你会意识到，当你第一次开始在清醒状态下自娱自乐时，你会觉得“嗯，这很有趣，但不如喝醉时有趣”。你必须坚持做下去。然后你会发现，即使你偶尔作弊一晚去喝酒，你也会意识到，你记忆中旧式的醉酒乐趣被过度美化了，它并没有你想象的那么有趣。你会觉得“哦，清醒的乐趣好像更好一点”。这就像如果你一直吃披萨之类的垃圾食品，然后转向健康饮食，你会觉得“哦，还行吧，但披萨更好”。但当你投入到健康饮食中，三四个月后再吃披萨，你会觉得“它没我记得的那么好”。清醒游戏也是如此，你必须投入到清醒游戏中。然后，将来每次你回到醉酒游戏，都会强化这一点：“是的，它确实没那么好，完全不如我记得的那么棒。”
