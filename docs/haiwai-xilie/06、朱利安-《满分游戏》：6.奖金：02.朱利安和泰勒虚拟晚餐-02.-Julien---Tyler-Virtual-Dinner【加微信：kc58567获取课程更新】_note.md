# 满分游戏：06：奖金：02. 朱利安和泰勒虚拟晚餐

在本节课中，我们将跟随朱利安和泰勒进行一次非正式的虚拟晚餐直播。这不是一次传统的教学，而是一次“幕后”式的交流，旨在展示更轻松、随意的互动氛围。我们将探讨学员提出的各种问题，涵盖从游戏技巧到个人发展的多个方面。

直播开始，朱利安和泰勒与在线观众打招呼，并鼓励大家抓住这个私密交流的机会提问。

他们强调，这次虚拟晚餐的主要目标是放松和闲聊，展示幕后状态，而非正式传递信息。这是一种“闲逛”的氛围。

> 不要因为我们让矮个子实习生在我们身后酿造康普茶而大惊小怪，这是幕后花絮，不是我们常规宣传的东西……只要我们玩得开心就行，别评判我们。

## 核心技巧问答环节

在闲聊之后，他们开始回答观众提出的具体问题。以下是部分问答的整理：

**关于自我证明**
学员问：如何在互动初期感到更自在，避免觉得需要立刻证明自己？
*   **核心概念**：这涉及到“自我证明”。一旦你开始自我证明，你就展示了低价值。
*   **解决方案**：转向**自我娱乐**和不在乎的态度，甚至可以进行**自我 disqualify**。
*   **示例**：你可以开玩笑说“我害羞得像根小湿胡萝卜”或“我来自小镇，是个 Applebee‘s 的服务员，很天真”。这能以幽默的方式吸引对方注意你的优点，而不是生硬地自我证明。关键在于能拿自己开玩笑，这展现了真正的自信和自我接纳。但注意不要过度，否则会显得虚伪。

**关于互动推进**
学员问：在洗手间接吻后，对方不愿进一步发展，称不想太快发生关系，该如何处理？
*   **分析**：对方愿意在洗手间接吻，已经说明了很多。她提到“不再那么快发生关系”，这可能是她个人的新边界。
*   **建议**：表示尊重，享受当下的乐趣，保持联系，后续再尝试邀约。

**关于状态**
学员问：不在状态（out of state）时是什么感觉？
*   **幽默比喻**：就像有一种诡异的黑色能量从你身上散发出来，大家都跑开了……然后你回家写一份负面报告，说游戏对你永远没用。
*   **核心**：指出这个问题本身可能意味着你很少不在状态。保持乐观 outlook 很重要。

**关于试探的限度**
学员问：如果玩笑开过头，对方走开了怎么办？
*   **核心概念**：注意 **压力与释放的节奏**。
*   **解决方案**：拒绝通常不是非黑即白的。对方会给出许多信号（如身体紧绷、笑容减少、向后靠、看向别处）。当你注意到这些信号时，就应该**慢下来**。任何事做太多都会变成不安全感。

**关于线下活动与线上课程**
学员问：Transformational Mastery 线下活动与线上产品相比，值得购买吗？
*   **线上产品优势**：可永久拥有、反复观看、做笔记，包含每日引导式释放练习和案例分析。
*   **线下活动优势**：亲身体验、导师台上指导、突破阻力层、内容根据现场定制、利用团队压力和互助。
*   **结论**：两者各有优缺点，都非常好。组织方会确保即使两者都参加，也能获得不同价值。

**关于开场方式**
学员问：最喜欢的无声开场白是什么？
*   **示例**：凝视、进出互动、拉着对方的手放在自己胸前然后说“看着不错”、做鬼脸、挥手。
*   **核心**：**间接开场**（sidestep opener）是革命性的，能极大减轻你的压力。

**关于友好游戏与直接高价值游戏**
学员问：友好游戏与更直接的高价值游戏，各有何利弊？
*   **分析**：友好游戏有效，直到它无效为止。**默认应采用友好方式**，因为这是从A点到B点的最短路径。
*   **策略**：如果感觉不对或凭直觉知道无法吸引对方，再转向戏弄或更具挑战性的方式。

**关于建立自信**
学员问：如何建立自信？
*   **途径**：
    1.  达到社会标准：外表好、发型好、衣着得体。
    2.  拥有资源：很多钱、资产或人们想要的东西。
    3.  社交认证：身边有很多朋友。
    4.  内在价值：源于内心的自我价值感。
    5.  当下互动：通过与人互动获得动力，意识到这没什么大不了。

**关于舞池游戏**
学员问：舞池游戏技巧？
*   **核心策略**：走上前后先别跳舞，可以说“嗨，怎么样？”，利用现场音乐作为靠近她耳边的借口。
*   **关键目标**：**把她带离舞池**，带到可以交谈和工作的地方。不要在舞池浪费太多时间，除非你热爱跳舞。

**关于内在成长与游戏的关系**
学员问：Transformational Mastery 对游戏有帮助吗？
*   **回答**：当然有巨大帮助。它虽然不是专门的游戏产品，但能改变生活，让一切（包括游戏）变得更轻松、不费力。

**关于学习对象**
学员问：应该只跟随一位导师学习，还是向所有人学习？
*   **建议**：主要跟随一位导师，但也可以博采众长。可以参考朱利安频道最新关于“创建智囊团”的视频。

**关于白天约会与夜间约会**
学员问：如何看待白天约会？
*   **商业类比**：业务需要潜在客户开发、转化和留存。
*   **分析**：夜店就像一个为你引荐**合格潜在客户**（即愿意社交的女孩）的高端场所。虽然她们也会被其他男人搭讪，但这仍然是高质量的线索。
*   **结论**：你应该学会白天约会，但也不应忽略夜间约会这片充满合格潜在客户的“沃土”。这就像拒绝利用托尼·罗宾斯的邮件列表进行推广一样不明智。

**关于被称作“玩家”**
学员问：总是被叫作“玩家”或发生关系后被说坏话，难以进入恋爱关系，怎么办？
*   **分析**：吸引之后还需要**信任**。女孩需要感觉到你真正在关注她、了解她。你需要学会恰当地“证明她值得”。
*   **层级差异**：中级阶段容易被叫“玩家”。当达到更高层级时（如贾里德、莱昂纳多），对方可能甚至不敢这样称呼你，怕惹你生气。
*   **根本原因**：有时“玩家”感源于显得过于努力或套路化，对方看不到真实的你。**尝试增加互动中的真实性，放慢节奏，改变方式。**

**关于“游戏”一词**
学员问：你会永远从事“游戏”吗？
*   **辨析**：“游戏”只是一个来自《游戏》一书、易于理解的词汇。它本质上就是社交互动。
*   **核心**：你会一直希望与人交往并学习社交动态。“游戏”这个词带有操纵的负面涵义，但我们可以使用它，同时明白其本质是做自己并与人互动。

**关于拒绝**
学员问：接近时对方立刻停下并拒绝（例如说“我有男朋友”），怎么办？
*   **建议**：尝试只是**保持礼貌**。你不应该总是遭受如此直接的打击。

**关于旧我与新我**
学员问：能变回以前的朱利安吗？
*   **哲学观点**：人像俄罗斯套娃，一层包裹一层。你从未真正改变，只是进化出新的层面和侧面。旧的你依然存在，只是你进化到了新的范式。那个部分你永远可以调用。

**关于咖啡因**
学员问：与咖啡和咖啡因的关系？
*   **个人做法**：偶尔喝，但不每天喝。
*   **建议**：如果你每天喝，建议戒掉。长期依赖后，没有咖啡因时可能会情绪不佳。注意观察自身感受，一旦出现依赖迹象就戒掉。

**关于推荐书籍**
学员问：改变你生活的书籍？
*   **大众推荐**：《超人》（大卫·R. 霍金斯）、《当下的力量》（埃克哈特·托利）、《唤醒内在的巨人》（托尼·罗宾斯）、《高效能人士的七个习惯》（史蒂芬·柯维）。
*   **进阶推荐**：《能量层级》（弗雷德里克·道森）、《致幻剂》（特伦斯·麦克肯纳）、《汉尼拔与我》（类似罗伯特·格林风格）、《艺术之战》和《专业主义》（史蒂芬·普雷斯菲尔德）。
*   **注意事项**：阅读进阶书籍需要**辨别力**，有些观点可能不完全认同，需谨慎吸收。

**关于最大顿悟**
学员问：目前最大的顿悟是什么？
*   **核心顿悟**：与其关注“做什么”，不如关注“**不做什么**”。保持冷静，别像傻瓜一样行事（如自我证明、显得饥渴、过于努力）。
*   **游戏顿悟**：大量接近，大量开场，然后对其中反应好的人**大量推进关系**。大多数人开场不足，而能做到的人又往往在推进关系时技巧不足或轻易放弃。

**关于快乐与技能**
学员问：因为你们的新产品（Transformational Mastery），如何能在没有技能的情况下感到快乐？例如，没有技能怎么能成为外科医生？
*   **辨析**：这是一个错误的类比。快乐或富足不需要依赖任何东西，它不应该依赖任何条件。外科医生是一种技能，而快乐不是技能。你不需要满足任何条件才能快乐。

**关于情绪管理**
学员问：你是否有时情绪极差，感到丑陋、无价值、恶心？而大多数时候你都能接受自己？
*   **核心观点**：Transformational Mastery 的关键不是永远快乐或积极，因为试图永远快乐是在逃避不快乐。
*   **目标**：**稳固你的内在基础**，这样你就不会被情绪起伏带走。你仍有情绪，但它们不再那么影响你。
*   **方法**：当你情绪不好时，不要自责，不要因为情绪不好而感觉更糟。**接纳它**，让它自然流逝。埃克哈特·托利称之为“在情绪周围创造空间”。

**关于自嘲**
学员问：如何自嘲？
*   **重要性**：如果你不能自嘲，别人就会替你嘲笑你。就像美国白宫前发言人肖恩·斯派塞，因为太把自己当回事，成了《周六夜现场》的绝佳嘲讽对象。
*   **核心**：你周围有一种能量。如果你不能放松并自嘲，就等于向别人敞开了嘲笑你的大门。你必须自己先做这件事。

**关于展示男性能量**
学员问：如何展示更多男性能量，感觉更阳刚？
*   **核心概念**：源于将意识安住在内在的“空无”之中，而非外部环境。女性是接收者、包容者（如同性爱中），而男性是给予者。如果你总是关注外部环境，会让你显得更女性化。

**关于游戏的目的**
学员问：游戏只是为了到处喝酒和发生关系吗？
*   **反讽回答**：是的，我们教授的一切的唯一目的，就是持续不断的自我满足。就像把脸埋进披萨和蛋糕里一样，没有更深层的东西。这就是我们25年综合经验所得的全部，因为世界充满了傻瓜，我们也是傻瓜。

**关于公开演讲**
学员问：计划做一个帮助人们学习的项目，但对公开演讲感到害怕，如何改进？
*   **建议**：进行大量公开演讲练习。对着摄像机谈论任何话题（比如拉链），讲10分钟。
*   **核心**：**重复，重复，再重复**。同时记住，虽然表达方式有帮助，但真正重要的是你的信息和内容。如果你的信息有益且能与人产生共鸣，人们会原谅你表达上的不足。专注于分享和帮助，你的能量就是开放的；如果只关注自己在镜头前的样子，能量就是收缩的。

**关于白天约会训练**
学员问：你们会举办白天约会训练营吗？
*   **解释**：在迈阿密等项目中进行大量白天商场教学，因为那里人多。关键是**数量**，而不是白天或夜晚。需要确保有足够的数量让学员保持练习动力。

**关于处理负面人际关系**
学员问：如何应对生活中的负面人物，尤其是试图在你创业或提升时拖你下水的家人或亲近之人？
*   **埃克哈特·托利观点**：“如果你想测试自己是否开悟，就和家人共度一个周末。”
*   **分析**：家人总会触发你内心的某些东西，这是一个深入探索并释放的好机会。
*   **同理心**：意识到他们试图拖你下水，通常与你无关，而与他们自己有关。你的成功像聚光灯一样照出了他们的不足，这可能是他们感到不安的原因。
*   **策略**：对他们抱有更多同理心，不要生气。然后**限制与他们相处的时间**，或只在特定情境下见面。

**关于掌握“10分游戏”**
学员问：如何知道自己掌握了“10分游戏”？
*   **终极答案**：当你不再问这样的问题时，你就掌握了。

## 直播总结

本节课中我们一起学习了朱利安和泰勒在虚拟晚餐中的即兴分享。这次交流涵盖了从具体的游戏技巧（如自我证明、舞池策略、应对拒绝）到更广泛的个人发展话题（如自信建立、情绪管理、人际关系处理）。核心在于理解互动中的自然流动、避免过度努力，并通过自我接纳和内在成长来提升整体状态。记住，游戏不仅仅是技巧，更是关于真实地与人连接和持续的自我进化。
