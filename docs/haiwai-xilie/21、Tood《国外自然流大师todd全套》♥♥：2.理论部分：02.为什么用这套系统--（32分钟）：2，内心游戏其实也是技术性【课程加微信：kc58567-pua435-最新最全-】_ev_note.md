# 国外自然流大师Todd全套：2：理论部分：为什么用这套系统

在本节课中，我们将探讨一个长期存在的核心辩论：内心游戏与外在游戏的关系。我们将分析为何不应将两者对立，并揭示有效的内心游戏本质上也是一种可操作、可训练的外在技术。

---

过去十年左右，另一个主要的辩论焦点是内心游戏与外在游戏之间的悖论。你究竟想要哪一个？是专注于做正确的事，还是专注于感受正确的方式，或是让事情源自正确的地方。这似乎是一个悖论，看起来你只能选择专注于其中一方。

首先需要明确的是，没有理由不能同时专注于两者。即使从表面上看，也没有理由不能同时拥有良好的内心游戏和良好的外在游戏。因此，它们并非相互排斥。

但我想更进一步。我想说，实际上，**内心游戏本身就是一种外在游戏**。

其逻辑在于：如果你想改变你的内心游戏，这需要具体、明确的步骤和过程。而具体、明确的步骤和过程，几乎从定义上讲，就是外在游戏。例如，人们会说“我想改善我的内心游戏”，然后他们会给自己设定一个任务，去做某件事来让自己对恐惧脱敏。这个任务就是一个外在游戏任务。你正在开发一套技能，你正在建立一个过程。

因此，内心游戏，至少是有效的内心游戏，是**基于行动的内心游戏**。它基于这样的认知：“我感觉这样，但如果我有不同的感觉会更好”，或者“我有这种态度或心态，如果我训练出不同的心态会更好”。你可以尝试仅仅通过思考来进入那种心态，但这行不通。那只会让你更沉浸在自己的头脑里。

实际上，你能真正获得更好心态、更好人生观的方法是通过**经验**。因为我们学习和相信的东西，是通过经验获得的。如果我手里拿着东西并松开，你知道它会掉到地上。你知道这一点，不是因为良好的自我信念或积极的态度，而是因为你从小婴儿时期就一遍又一遍地看到东西掉到地上。你反复看到它，并且它极其可预测、可靠，以至于你知道这不是你对世界的看法，而是根据你的经验所认识到的世界真相。

所以，最好的内心游戏实际上来自经验，而最好的内心游戏训练实际上来自采取行动并获得反馈。即使是为了克服恐惧等，最好的内心游戏训练也来自真正采取行动，经历克服恐惧的过程，经历表现得无所畏惧的过程。

但关键是：**内心游戏是有流程的**。你的内心游戏越是模糊不清，比如“我必须感觉某种特定方式”，你就越是内心游戏的奴隶。

游戏中有个概念叫“状态”，这是一个非常有趣的概念。它基本上指你的思维运作方式：你是否处于心流状态？你是否活在当下？你是否处于最佳状态？有趣的是，在我学习游戏的漫长时期里，很长一段时间“状态”这个词从未被提及。当情况如此时，没有人会“不在状态”，因为如果状态这个概念不存在，你就不可能“不在状态”。所以，如果你心情不好，你他妈的就直接上去做了，你不会找借口。然后发生的是，你尽力而为，在心情不好的情况下也取得了一些成果。而这些成果以及你采取了积极行动的事实，会让你对那次行动感觉良好，这往往会让你进入好心情，然后你实际上就进入了“状态”——只不过你当时还不知道那是什么，因为这个词还没被发明出来。

但本质上就是那样。现在，突然到了“状态”这个概念存在、被贴上标签和命名的时代。给它贴标签是有用的，因为它在某种程度上是一种真实现象。但问题是，一旦你给它贴了标签，它现在就变成了一个借口。几年前，在我的课程中，我会派一个学生去搭讪，他会说：“嘿，我想先进入状态。” “我想先进入状态。” 首先，这是什么蹩脚的借口？你需要进入状态才能去和女孩说话？这绝对荒谬。她不需要进入状态来和你说话。没有比这更低价值、更把她放在神坛上的思维框架了：“我现在的样子不够好，我必须做这个复杂的事情才能配得上她。” 与此同时，她不需要做任何事就足以配得上我。所以这在很多方面都是根本错误的。

其次，“我需要去进入状态。” 你具体计划怎么进入状态？无论你的计划是什么，它都需要行动。你的计划可能是通过做一堆搭讪来进入状态——那是行动，是外在游戏。你的计划可能是通过跳舞、提升生理状态来进入状态——猜猜怎么着？那在某种意义上也是行动，实际上是外在游戏的一种形式，是执行计划并在现实世界中采取真实行动。假设你计划在下一次互动中只是开玩笑、胡闹、不在乎结果，并把这当作一个练习。猜猜怎么着？那也是外在游戏。

所以，所有有效的内心游戏技巧从根本上都归结为**采取行动**，这意味着它们从根本上仍然是外在游戏的解决方案。事实上，思考你的内心游戏、思考你的心理、思考你如何思考，这本身就是“沉浸在自己头脑里”的定义。让自己脱离头脑的最好方法，实际上是如此专注于现实世界、如此专注于女孩、如此专注于你正在做的事情，以至于你大脑里根本没有剩余的内存去沉浸在自己的头脑里。然后，你实际上就不可能沉浸在自己的头脑里——你正在强迫自己脱离头脑。但同样，你是通过行动来做到这一点的。

因此，我想在这里提出的观点是双重的：
1.  **内心游戏与外在游戏之间没有悖论**，没有理由你不能两者都擅长。
2.  **内心游戏有其自身的外在游戏组成部分**。无论你的内心游戏状况如何，都有行动计划、行动步骤，有你可以做的具体有形的事情、具体的练习，甚至是思维过程——这些仍然是外在游戏，因为它是“我正在采取具体行动以获得特定结果”。任何你无法通过采取具体行动来获得特定结果的领域，你就是在要求事情变得不可预测，如果你依赖好心情去做某事，你就是在让自己失控。你必须等待直到心情好，或者只是希望当可爱的女孩走过时你碰巧心情好——这是一种非常糟糕的生活方式，是一种你让自己受制于生活、成为因果关系的被动接受者的生活方式，你实际上把自己视为环境的受害者，这可能是你在游戏中最糟糕的哲学之一。

所以，你想要掌控自己的生活。你想要采取行动，并且你想要将**一切**都视为你可以在任何特定时刻采取积极、主动的步骤来改进的东西。当你拥有这种态度，并且当你学会了这些步骤（我将在整个课程中帮助你），你将能控制你的结果，你将获得可预测的结果，并且你还将获得一个非常顺畅、流畅的学习过程，在这个过程的每一步你都能掌控。

---

本节课中，我们一起学习了内心游戏与外在游戏并非对立，而是相辅相成。有效的内心提升必须通过具体的、可执行的行动（即外在游戏）来实现。掌握这种将内在状态转化为外在步骤的能力，是获得稳定、可预测进步的关键。
