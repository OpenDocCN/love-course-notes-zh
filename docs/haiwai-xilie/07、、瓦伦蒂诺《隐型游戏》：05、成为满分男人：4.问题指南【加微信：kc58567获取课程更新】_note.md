**《隐形游戏》课程：07：成为满分男人：4. 问题指南**

在本节中，我们将探讨一个塑造你所有行为和人生方向的根本力量：你持续向自己提出的那个核心问题。我们将学习如何识别它，评估它，并有意识地将其替换为更具赋能性的问题，从而从根本上提升你的状态和吸引力。

在上一节，我们讨论了自我对话。我们了解到，我们其实一直在向自己提问。

现在，我们将聚焦于那个你潜意识或无意识中一直在问自己的问题。我喜欢称之为“指南针问题”。它就像是你始终在问自己的一个基础问题，是你所有行动的基石，真正指引着你人生的“真北”方向，引导着你整个道路、使命和生活的走向。

每一刻，你都在问自己一个问题。通常，大多数人的提问方式倾向于逃避痛苦或寻找事物背后的缺陷。例如，很多人会持续问自己：**为什么我不够好？** 为什么这行不通？为什么我又不能做到完美？或者，我如何才能完美并确保每个人都爱我？

要找到你的那个问题，你需要问自己：什么事情如果发生或不发生，会带来**极端**的后果？

我过去的指南针问题是：**我如何才能以最快的方式成为最好的？** 这听起来像是个好问题，似乎能引领人走向成功。但同时，它会让你抓狂，因为你总想着：我必须成为最好的，必须尽快做到。这会给你带来压力，有时甚至会妨碍你与他人的深度连接，因为你总是以成功为导向。

关键在于，你的价值观也与此相关。审视你的核心价值观，你会发现这个问题通常试图去满足那些核心价值观。但即使你不了解价值观理论，只要能找出那个你持续提问的核心问题即可。

通常，如果你现在就问自己：我在问自己什么？这一切的源头是什么？你就能找到它。一旦你找到了那个指南针问题——它通常被触发，或其满足与否会带来极端、甚至致命般的后果——例如，“我如何才能以最快的方式成为最好的？”如果做不到，感觉就像天塌了一样。

这个问题被满足或不被满足的后果，通常感觉如同生死攸关。这会给你线索去发现自己的问题。虽然当面练习能更精确地定位，但凭借刚才的知识，仔细思考，你就能找到它。或者，“我如何才能学会这个？”也可能是另一个例子。提问的方式多种多样，但本质是通用且适用于几乎所有情境的。

现在，你可以做的是：找到那个问题，然后问自己：问这个问题带来的后果是什么？它是在**赋能**我还是在**削弱**我？

如果它是一个削弱你的问题，你可能需要改变它。以下是一些例子供你参考。

我最喜欢的例子之一来自《精要主义》这本书。作者的问题是：**现在什么最重要？** 这个问题将所有焦点都带到当下，因为当下是我们唯一生活的时刻。我们总在思考过去或预想未来，却错过了当下发生的一切美好。

另一个我喜欢的问题来自托尼·罗宾斯。他的问题是：**我如何才能更欣赏此刻的完美？** 这个问题预设了你已经在欣赏，预设了此刻就是完美的，它能唤起你的感恩之心，将你带入一个更好的状态，从而更有效地行动。

还有一个我喜欢的问题是：**做什么事，能让20年后的我为之骄傲？**

这些只是你可以开始问自己的问题的几个思路。要将新问题内化，方法就是选择一个，比如“现在什么最重要？”，然后去森林里散步，反复默念：现在什么最重要？现在什么最重要？……就这样不断将其刻入你的大脑。下次当你感到有点压力时，你就会想：现在什么最重要？也许为这些事抓狂并不重要，更重要的是与我生命中真正所爱和重要的人建立连接。

另一件我常做的事是进行至少5分钟的晨间仪式。如果你早上连5分钟都抽不出来给自己，那你的生活确实需要调整。醒来后，坐下来，可以冥想，闭上眼睛（睁着也行，只要不分心），然后问自己以下几个问题。

第一个问题是：**我爱谁？谁爱我？** 这让你想到生命中最重要的人，让你感受到你已经拥有的爱。你可能会觉得没人爱你，但仔细想想，通常至少有那么几个人，会为你的存在感到无比高兴，并且无论如何，在他们内心深处都爱着你。这能将你带到对你最重要的事情上，让你在一天中能据此行动，因为早晨为接下来的一整天奠定了基调。

第二个问题是：**此刻，有哪三件事值得我感恩？** 我首先总会想到一些我能感受到、正在经历的事情。比如我正在呼吸的新鲜空气，我能触摸到的氛围，我有眼睛能看，有耳朵能听，我能感知事物，我存在，我能意识到我的存在。这些非常简单的事情，能再次将你带入一个状态，让你在接下来的一整天里更有效地行动。这让你变得极具吸引力，女孩们会忍不住想待在你身边，因为你在最深的层次上掌控着自己。有谁会做这样的事？有谁甚至意识到自己在问自己什么问题？有谁意识到这会引领他们去往何方？几乎没人。如果你这么做，会以积极的方式震撼到女孩，她会想：哇，我得留住这家伙，他有点东西。

最后一个你可以在早晨问自己的问题是：**今天有什么即将发生的事让我感到兴奋？可能是什么？我可能对什么感到兴奋？** 这比起“为什么我要这么早起床？”“为什么我必须做这个？”“为什么不能睡懒觉？”这样的问题，更能为你的一天设定成功或满足的基调。

关键是，你反正已经在不停地问自己问题了。所以，不如有意识地去问，从而真正从你拥有的时间里获得最大的收益。

**总结**
本节课中，我们一起学习了“指南针问题”的概念。我们了解到，识别并审视那个持续影响我们的核心自我提问至关重要。如果它是在削弱我们，我们可以有意识地将其替换为像“现在什么最重要？”或“我如何才能更欣赏此刻的完美？”这样更具赋能性的问题。通过晨间仪式等练习反复强化新问题，我们能从根本上改善自己的状态、专注力和内在吸引力，从而更有效地主导生活。
