# 做个转变：04：行动_3. 获取你的推荐信 📝

在本节课中，我们将要学习如何通过获取大量“推荐信”（即实践经验）来快速提升你的社交技能。核心在于，你需要在最短时间内积累尽可能多的互动经验。

上一节我们介绍了行动的重要性，本节中我们来看看如何高效地获取这些关键的实践经验。

## 为何选择夜店与酒吧 🍸

获取推荐信最快的方式，是前往人流量大的场所，例如酒吧和夜店。当你听到这个建议时，很多人的第一反应是抗拒。

以下是常见的抗拒理由：

*   我不喜欢夜店，太吵、太拥挤。
*   我不喜欢那里的音乐，感觉很可怕。
*   那里的女孩太年轻、喝醉了、很虚伪，不是我喜欢的类型。

如果你因为这些理由而避开夜店，那将严重阻碍你的进步。

## 夜店的优势分析 🚀

夜店是获取实践经验的最佳场所，原因如下：

**1. 高流量与高效率**
在夜店，你不必在互动之间浪费时间寻找下一个目标。那里有源源不断的女性可供练习，你可以快速连续地进行互动，高效积累经验。公式可以概括为：**经验积累速度 ∝ 目标数量 / 单位时间**。在夜店，这个值最大。

**2. 容错率高**
可选目标越多，你承受犯错压力的能力就越强。如果只有一个目标，你会感到巨大压力，害怕搞砸。但在夜店，即使搞砸了几个，你还可以立刻转向下一个。这种环境降低了你的心理负担。

**3. 相同的社交对象**
不要认为白天遇到的“好女孩”晚上不会去夜店。事实上，她们也会去社交。如果你只喜欢白天遇到的类型，那么在夜店你同样会遇到她们。

**4. 有利的社交条件**
在夜店或酒吧，人们通常会停留较长时间，这为你提供了充足的练习时间。相比之下，白天人们往往行色匆匆，没有时间进行长时间互动。

**5. 匿名性与低风险**
在夜店，即使被拒绝，你也可以迅速消失在人群中，不会显得突兀。那里的氛围轻松，人们不会太较真，你也能更快地被遗忘，从而减少尴尬。

## 从陌生社交圈开始练习 🚫

不要犯一个常见错误：在你的日常社交圈（如同事、同学）中开始练习。

你必然会搞砸，不要把麻烦带到日常生活中。应该从完全陌生的“冷接触”开始，在对你日常生活没有影响的场合练习。当你掌握技能后，再自然地将其应用到日常生活中。

## 克服对夜店的恐惧 😨

夜店确实令人害怕。音乐嘈杂、竞争激烈、环境混乱。但这正是它的价值所在。

**越艰难的练习场，越能锻造强大的你。** 这就像学游泳：如果你一开始就在狂风暴雨的海洋中学习，那么以后面对儿童泳池就会觉得轻而易举。如果你能在夜店这样混乱的环境中游刃有余，那么在任何其他场合你都会感到轻松自在。

随着经验积累，你的心态会从“夜店好可怕”转变为“夜店是我的游乐场”。你会感到像在家一样舒适自在。

## 如何有效获取推荐信 🎯

一旦你进入夜店，请遵循以下原则：

**1. 与所有人开启对话**
你的目标是成为一个社交达人，与遇到的每个人交谈。每一次与女性的互动经验都是宝贵的，无论她是谁、是否聪明或美丽。

**2. 对所有人认真对待**
一个常见的借口是：“我只想和美女聊天”。如果你这样做，进步会非常缓慢。一晚可能只有3-5个你感兴趣的目标，这意味着你只能获得寥寥几次经验。相反，你应该和每个女孩交谈，获取经验。代码逻辑如下：
```python
if girl_in_sight:
    approach() # 执行接近，无论吸引力如何
    gain_reference_experience() # 获取经验值
```

**3. 行动重于结果**
不要将你的行动与结果挂钩。有时，你的目标只是“拿到她的号码”或“带她回家”以获得相关经验，而不是真的要发展什么。你可以之后删除号码。重要的是你完成了“获取号码”或“带回家”这个动作，积累了参考经验。不要等到“真命天女”出现时才第一次尝试，那时你毫无经验，注定失败。

**4. 这是关于你的修炼**
这个过程是为了提升你自己，培养有吸引力的人格，向世界展示你是谁。不要挑剔互动对象。让每个人都感受到你的魅力，然后你再选择你喜欢的。如果你只对你认为有吸引力的女性展示魅力，你会把她们捧上神坛，变得患得患失，这反而会吓跑她们。

## 心态调整：平等与乐趣 😄

想要吸引那些非常美丽的女性，你必须愿意和每个人交谈，并且能从与每个人的互动中获得真正的乐趣。

如果你发现自己总是在环境中扫描美女，这说明你认为“价值在别处”，而不是在你身上。无论你和谁在一起，无论你在哪里，派对的核心应该是你。

当你能够平等地与所有人享受乐趣时，你就不会把任何人捧上神坛。即使在与一位美女交谈时，你也会想：“如果搞砸了也没关系，我也可以和那边那个女孩玩得一样开心。”这种不需求感、充满选择权的心态，反而会让女性更想追逐你。

## 创造无评判的现实 🌈

如果你根据外貌来评判女性（例如“她不够美，价值较低”），你就在强化一个“人人相互评判”的现实。你会因此担心别人也这样评判你，从而变得束手束脚，害怕尝试。

相反，你应该强化一个“无评判”的现实：在这里一切都好，没人在乎。做你想做的事。事实上，如果一个美女看到你和一位不那么吸引人的女孩愉快交谈，她反而会觉得你自信、放松、有趣，从而更被你吸引。

记住，女孩会成为你预期中的样子。如果你认为“她是美女，是件大事”，她就会进入这个框架，变得难以接近。如果你保持轻松、好玩、不在乎的心态，她也会放松下来，这样你才能得到她。
