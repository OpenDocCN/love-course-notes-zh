# 掌握转变：03.证明：5.释放指导 🧘

在本节课中，我们将要学习一个名为“释放指导”的具体练习。这个练习旨在帮助你安全、有效地释放身体和情绪中积压的感受与张力。我们将分步进行，从准备环境到完成释放循环。

## 准备环境

首先，你需要创造一个不受干扰的环境。

确保你身处一个没有外部干扰、无人能打断你的空间。

确保你的手机和电脑通知都已关闭。

真正创造一个你能100%专注于自身的环境。

我个人喜欢坐在椅子上进行这个练习。

我建议你也采取坐姿，背部挺直。不要向后靠。

保持背部挺直，以维持警觉状态。

保持临在，将双手放在大腿上，掌心向下。闭上眼睛。

让我们开始。

## 第一步：觉察

上一节我们介绍了如何准备环境，本节中我们来看看练习的第一步：觉察你想要释放的东西。

第一步是觉察你想要释放的东西。

这将是你在当下正在体验的感受。

所以，让我们先帮助你从头脑中走出来。

为此，我希望你想象在你双耳之间的那个“话匣子”。

就是那个总在喋喋不休、贴标签、评判、比较的小声音。

“这里发生了什么？”、“我做对了吗？”、“这对我有用吗？”、“这和我平时的冥想相比如何？”、“这要花多长时间？”、“我之后有计划吗？”，就是那个小声音。我希望你想象它。

改变位置，一路向下移动到你的脊柱底部。

这也被称为你的海底轮或基础轮。

这是一个与你的身体建立深层连接的地方。

所以，将你的觉知带到那里，想象那个声音，慢慢移动到你的脊柱底部。

然后，停驻在底部。

你不是从那里思考。而是从那里呼吸。

事情会变得安静一些。感觉会变得轻盈一些。

现在，让我们通过三次循环呼吸来锁定这个位置。

以下是具体做法：

你将通过鼻子吸气，数到6。屏息3秒。

然后呼气6秒，发出“嘘”声，再屏息3秒。

所以，从脊柱底部开始，吸气数到6。现在开始：1，2，3，4，5，6。

屏息3秒：1，2，3。呼气6秒：嘘，1，2，3，4，5，6。再屏息3秒：1，2，3。正常呼吸。

让一切安定下来。

我们将进行另一个循环。开始。吸气6秒：现在开始，1，2，3，4，5，6。

屏息3秒：1，2，3。呼气6秒：1，2，3，4，5，6。再屏息3秒：1，2，3。再次正常呼吸。

还有一次，让你的身体保持临在。吸气6秒：1，2，3，4，5，6，屏息3秒：1，2，3，呼气6秒：1，2，3，4，5，6，再屏息3秒：1，2，3。正常呼吸。

让一切安定下来。

你现在是从脊柱底部运作。你从那里思考，从那里呼吸。你脱离了头脑。

你进入了身体。

## 第二步：感受与接纳

在将意识带入身体后，现在我们需要去感受并接纳出现的任何感觉。

我希望你调频到你的胃部和胸部区域。

开始觉察正在发生的不同感觉。

让我们看看是否有任何东西被搅动起来，是否有任何紧张、紧绷、不适或不安。

开始注意所有的感觉。

不要评判它们。不要给它们贴标签。

你现在在你的身体里。你只是在感受。

我希望你现在做的是，允许自己去充分体验它们。

这是第二步。

我希望你与那些感觉待在一起，真正沉浸其中。让它们接管。

不要试图修复、改变它们，或试图理解它们。

这里没有什么需要修复、改变或理解的。

你要做的就是，臣服于正在发生的一切。臣服于那些感觉。

给它们传递一个信息：你无条件地完全与它们同在，你百分之百开放地去体验它们。

让我们花一些时间真正与它们在一起。

继续从脊柱底部呼吸。让它们接管。臣服。甚至更深入地沉浸其中。

向它们呼吸。与它们完全同在。

好，再来一次呼吸。

现在让我们进入第三步也是最后一步。

## 第三步：释放

在充分感受和接纳之后，最后的步骤是邀请这些感觉离开。

这就是：让一切都升起来。然后释放出去。

这就是我们现在要做的。

我希望你邀请所有这些感觉升起来。

记住，它们想要离开。它们想要完成自己的过程，达到自然的终点，然后消失。

我们只需要邀请它们，并停止通过抗拒来紧抓它们。

所以，你要做的是再进行三次循环呼吸，真正地向这些不同的感觉呼吸。

但这一次，随着每一次吸气，我希望你想象它们升起来。

随着每一次呼气，发出“嘘”声，我希望你想象它们穿过你，让它们出去。

所以，调频到它们，注意体内被搅动的任何东西，现在把它们都带上来。

吸气数到6：1，2，3，4，5，6。屏息3秒：1，2，3。呼气6秒，让它出去：1，2，3，4，5，6，屏息3秒：1，2，3。再次正常呼吸。

现在，可能下面还有更多。让我们调频到它，现在把一切都带上来。

吸气：1，2，3，4，5，6，屏息3秒：1，2，3，然后让一切都出去：1，2，3，4，5，6，屏息3秒：1，2，3。正常呼吸。

现在，让我们再次调频。进行最后一次循环。让我们现在把一切都带出来。

任何剩下的东西，带上来吸气数到6：1，2，3，4，5，6，屏息3秒：1，2，3，然后让一切都出去。送走它。放手：1，2，3，4，5，6。屏息3秒：1，2，3。呼吸。

让一切安定下来。

将你的觉知带回你的身体。只需注意事物感觉更加轻盈。

有一种如释重负的感觉，仿佛重担已被卸下。

让我们就在那里待一会儿。让自己体验这种解脱。这种轻盈。这种对自己的新体验。

现在，一切都好。一切都完美。

你终于能感到快乐、平静。

我们将在脊柱底部再做一次循环呼吸。

吸气数到6：现在开始，1，2，3，4，5，6。屏息3秒：1，2，3。呼气6秒：嘘，1，2，3，4，5，6，屏息3秒：1，2，3。正常呼吸。

现在我们要准备回来了。

开始慢慢地移动一下身体。开始动动指尖。开始动动脚。

将你的觉知带回到周围环境。

准备好慢慢睁开眼睛。

## 总结

本节课中我们一起学习了“释放指导”的完整流程。

这就是你放手的方法。它并不神奇。也不玄乎。它极其直接，是我们所有人都拥有、都能做到的自然能力。

我们只是从未这样做，也从未被教导如何去做。

我想在此强调，这是你必须亲自去做的事情，因为它看似简单直接。我可以永远谈论它，但正如你在此体验到的，你不能只是谈论。你必须经历它。你必须亲身经验。

这是我建议每天进行的练习。我在下方包含了一个带有引导释放录音的部分，有一个清晨引导释放和一个晚间引导释放。在接下来的章节中，我们还将更深入地探讨如何释放埋藏在你潜意识深处的东西。
