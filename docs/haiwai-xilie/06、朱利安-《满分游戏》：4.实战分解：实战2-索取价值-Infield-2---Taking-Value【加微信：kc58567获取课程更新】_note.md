# 满分游戏：06：实战分解：实战2-索取价值

在本节课中，我们将通过一个实战案例，深入剖析“索取价值”的互动模式。你将看到，当心态处于“索取”而非“给予”状态时，即使行为表面相似，互动的氛围和结果也会截然不同。理解这种核心差异，是提升社交吸引力的关键。

上一节我们探讨了“提供价值”的互动，本节中我们来看看与之相反的“索取价值”模式。

## 🎯 互动案例：索取价值模式

这是一个非常典型的互动场景，可能代表了95%男性的社交尝试。这种互动不会立即被拒绝，但也不会真正成功。

我将播放这段互动，请注意感受氛围上的差异。和之前一样，我会提问：你注意到了什么？感受到了什么？不要过于关注具体“做了什么”，有趣的是，你会发现“行为”本身与我处于最佳状态或上一节互动中的行为相当类似。

**区别在于**：这些行为源自不同的内在状态。

（以下是互动对话内容整理）
*   “你好。我今天看了电影《珍珠猎人》，很受启发。我在寻找‘巨魔’。我是个‘巨魔猎人’。”
*   “你叫什么名字？……我是朱利安。”
*   “你真的来自奥斯陆吗？再次确认，你来自卑尔根还是奥斯陆？”
*   “你喜欢还是讨厌它？（指奥斯陆）”
*   “所以你是瑞典人？……你的科学知识来自远方？我懂一点点。”
*   “下周末很棒。是好是坏？是更好还是更差？”
*   “你喜欢亚历山大·斯卡斯加德这位演员吗？”
*   “你多大了？”
*   “如果你必须在瑞典和挪威之间选择，你会选哪个？”
*   “你去过美国吗？你去过斯堪的纳维亚以外的地方吗？非洲呢？”

即使只是观看，你也能感觉到不同：它不那么有吸引力，你无法像之前那样投入其中。

## 🤔 核心分析与关键区别

以下是关于这段“索取价值”互动的关键观察点：

**1. 互动的本质与结果**
这个女孩在15分钟内没有拒绝我，这很奇怪。为什么？因为我的表现并不糟糕。她的替代选择是什么？她和正在与某个男人交谈的朋友在一起。她要么成为朋友和那个男人之间的“电灯泡”，要么听我说话。这不糟糕，但不足以让她想和我发生关系。

这就是那种你与女孩交谈、可能都有过的经历：聊了一会儿，当你索要号码时，她说“不”、“我得去洗手间”、“我有男朋友了”或“我没手机”。互动毫无进展。这就像地狱。这就是我说的，**95%男性的互动现状**：投入大量努力，维持着对话，但什么也没发生。

**2. 内在心态的试金石**
所有这些问题都在试图“索取”。我在问：“你会说瑞典语吗？这个怎么样？那个怎么样？”我试图获得那些好的反应，来验证或证明我并非处于低价值状态。因为在这个互动中，我感觉自己处于低价值状态。我处于稀缺心态，我在试图索取。

而在另一个（提供价值的）互动中，如果那个金发女孩（拍照的）离开了，我会感觉有差别吗？不会。在这个（索取的）情境中，如果女孩离开，我感觉如何？**糟透了**。这就是判断你真正处于“提供价值”还是“索取价值”心态的方法：如果你感到失落，说明你期望得到某些东西却没得到；如果你已经拥有了一切，那就无关紧要。

**3. 行为背后的意图**
在这个互动中，我说的很多话其实在我状态好时也会说。但在这里，其背后有不同的意图。它有点**强迫和不自然**。我试图把状态好时会说的话当作一种**技巧**来使用，以获得那些反应。它不再是纯粹的表达，而是带有**操纵性**。

**4. 外在表现与氛围**
身体语言更僵硬，不那么轻松自在。有点更紧张，更易受外界影响，一点也不洒脱。有点失去平衡。而另一个互动则显得轻松、有趣、真实、真诚。这里则显得失衡，试图索取，感觉处于低价值，僵硬且有点勉强。

**5. 女性关注的焦点**
这些细微差别和精妙之处，才是女孩们在与你交谈时真正关注的东西。她们关心这些远多于你的言辞或行动。在视频中你可以清楚地看到：言行本身，和我状态好时说的东西一样，但有些别的东西不同了，那就是**氛围**。

## 💡 重要启示与心态调整

**关于“高手”的误解**：阻碍我们的一个观念是，我们认为擅长此道的人永远不会“陷入头脑”（过度思考）。所以一旦你陷入头脑，就感觉糟透了，心想“和那些从不陷入头脑的高手比，我太逊了”。事实并非如此。

**接纳不完美**：我仍然会时不时陷入头脑。我仍然经常被拒绝。你应该被拒绝，因为如果你真实地展现自己，不是每个女孩都会与你产生共鸣。这是事物的本质。如果你能和每个女孩都合得来，那才奇怪。唯一能做到的方法就是戴上面具、妥协真实性和自我感受。所以，你必须现实一点：我会搞砸，我会陷入头脑。

**目标不是完美**：这里的目标不是达到一种任何时候都完美的幻想现实。目标是走出去互动。事实上，你会搞砸一次互动，我也会，比如说了些无聊的话，让气氛有点冷场。这反而让整个过程变得有趣。女孩们总是看**整体画面**。

**核心障碍：对结果的执着**：在那个（糟糕的）互动中，阻碍我的主要是**对结果的执着**。通常，对结果的执着会阻碍你，让你显得非常“需要”某些东西。在第一个（好的）互动中，我在和多个女孩交谈，所以即使一个离开，我还有其他人。我创造了一个外在的丰裕情境。但更重要的是，我并不在乎，我不需要那个女孩留在那里。

**情境的影响**：那晚是训练营的最后一晚。结束方式是在俱乐部关门前结束，以便给大家创造“带女孩回家”的机会。当时有两个学员成功了，还有一个在场。我带着相机，突然之间，我心想：“我想做什么？我想待在外面吗？我想带一个回家吗？如果我能拍到一个成功的实战视频，那就太棒了。**如果拍到了，我会对今晚感觉更好**。”就在那时，我完了。“我需要这个实战成功来让今晚变得成功。”这种想法一旦出现，心态就转变了。

## 📝 本节总结

本节课中，我们一起深入分析了“索取价值”的互动模式。我们看到了当心态从丰裕的“给予”转变为稀缺的“索取”时，即使外在行为相似，整个互动的能量、氛围和潜在结果都会发生根本性变化。关键在于识别并调整自己的内在状态：关注过程而非结果，保持真实表达而非技巧操纵，并接纳互动中自然存在的不完美。真正的吸引力源于内在的丰盈与自在，而非外在的索取与证明。
