上一节我们介绍了控制力的基础，本节中我们来看看如何正确表达意图。

## 核心概念解析

意图清晰意味着直接、坦诚地表达你的兴趣或目的。在社交互动中，这可以类比为操作电视机：目的明确，但姿态平等。

**核心公式**：
`意图清晰 = 直接表达兴趣 + 保持自我框架`

关键在于，表达兴趣的同时，不能伴随低价值的恳求或讨好行为。以下是一个错误示范的代码描述：

```python
# 错误示范：低姿态的“卖家”模式
def approach_with_wrong_intent():
    intention = “我喜欢你”  # 意图看似清晰
    posture = “乞求与讨好”  # 但姿态是低价值的
    return low_value_interaction

# 这种模式会导致吸引力下降。
```

正确的做法是，像筛选商品一样保持选择权。你的话语是筛选器，用于判断对方是否符合你的标准，而非推销自己。

## 正确做法示例

以下是实现“意图清晰”且不沦为“卖家”的具体做法。

我的做法是主动筛选。我会用一个简单直接的问题开启对话，例如：“嘿，你多大了？” 这句话的作用是：

*   **表达兴趣**：主动开启对话本身就是一种兴趣指标。
*   **进行筛选**：问题本身包含了对年龄信息的筛选。
*   **保持框架**：提问者姿态是获取信息、做出判断的一方，而非请求认可的一方。

这种方法确保了互动从平等甚至略占优势的位置开始。

## 总结

本节课中我们一起学习了“意图清晰”的正确应用。我们明确了意图清晰是直接表达，而非低声下气地讨好。关键在于保持自我框架，通过主动筛选来维持互动中的平衡与吸引力。记住，你是筛选者，而非卖家。
