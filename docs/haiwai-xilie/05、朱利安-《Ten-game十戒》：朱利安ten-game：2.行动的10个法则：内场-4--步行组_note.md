# 朱利安Ten Game：05：行动的10个法则 - 内场4：步行组 🚶‍♀️

在本节课中，我们将学习如何应对移动中的目标（即“步行组”）。核心在于理解她们的目标是**从A点移动到B点**，你的策略不应成为其障碍，而应顺势而为，创造双赢局面。

## 概述

上一节我们介绍了内场互动的基本法则，本节中我们来看看当目标处于移动状态时，我们该如何行动。关键在于**不要强行阻止**，而是学会**伴随移动**或**巧妙吸引**，以保持互动的延续。

## 核心法则：顺势而为，不要阻挡

女孩步行的目标是从A点移动到B点。多数男性试图直接拦住她们。虽然你与许多女孩可能是完美匹配，但若你成为她目标的障碍，她就会拒绝你。

**法则公式：**
```
你的目标 ≠ 阻止她到达B点
你的目标 = 与她同行 或 吸引她停留
```

因此，你无需在开场的地方就和女孩完成整个互动。应该与她同行，实现双赢。

此外，请记住：**你应尽可能远离开场的地点**。因为你不会在那里与她发生什么。在你开场的地方，什么都不会发生。但我们却总试图把女孩留在原地。例如：“嘿，来这边。”“不，就待在这儿。”“嘿，我去看看朋友。”“哦，留在这里。”“我们去喝点东西吧？”“不，留在这里。” 你为何试图把她留在这里？这不是双赢的策略。

## 互动策略示例

以下是处理步行组的一些具体方法，核心是**保持互动进行**，无论是否让她停下。

### 方法一：直接吸引

不要只是盯着看。说些能激发她兴趣、把她吸引过来的话。可以加入一点挑战和激情来互动。

**示例代码（互动流程）：**
```plaintext
你：（伸出手臂）你是迈阿密人吗？你从哪来？
女孩：挪威/芝加哥。
你：来迈阿密做什么？
女孩：度假。
你：我也是刚来。我不住芝加哥，我住LA。（对朋友说）他来自挪威奥斯陆。
（通过分享和提问，将她带入你的对话圈）
```

在这个例子中，你是在吸引她加入。这个女孩是独自一人，且走得有些慢、漫无目的。你可以事先判断哪些女孩可以拦下。对于那些看起来像有任务在身、快步行走的女孩，最好立即开始与她们同行，并且要表现得随意。

### 方法二：轻触开场与同行

如果女孩看起来走得很快，你可能会说“嘿，停一下”，她或许会停下。但需要明白，即使她停下，可能也只停留2、3分钟。她终究还是想去B点，这时你就需要跟她一起走。

很多我的开场是“轻触”式的。比如她正在走，轻拍一下，说“打扰一下”，或者直接开始并肩走。**你的唯一目标是让互动持续下去**。无论用什么方法，只要能保持互动就行。

**示例：**
```plaintext
你：你是迈阿密人吗？你是迈阿密人吗？你从哪来，密歇根？俄罗斯？
（如果她们继续走）
你：等等，一秒，一秒，砰！（用声音或手势吸引）
（然后你走上前）
```

此时你不需要逻辑完美，可以快速说一些话。词语本身不重要，你只是想通过氛围尽可能快速有效地传达你是个什么样的人。这就是达到“挂钩点”的关键。词语不必担心，即使你含糊地说些废话，让她感受到你的存在，也比什么都不说要好。

开场可以大胆一些。比如：“嘿，停一下，迈阿密…什么鱼…嘿，之类的。”这比因为想不出词而僵住要好。要像机关枪一样持续表达，活在当下。

### 方法三：明确意图，消除戒备

当你与步行组互动时，需要给她时间意识到自己被搭讪了，并被吸引进来。她还不清楚你是否是个有趣的人。

对于步行组，很重要的一点是：**让她知道你为何拦住她**。可以通过氛围传达，如果你想强化这一点，可以握手。逻辑上，她会想：“这家伙是谁？他想干嘛？哦，他想认识我，可以。”让她知道你在那里的原因，而不是只是喃喃自语。

所以你可以说：“握个手吧。你从哪来，法国？”…“幸会。你们在聊什么？”…“不，你们从哪来，就两分钟，停两分钟。我昨天刚到这里，你们住这儿吗？”

每次我说“两分钟停留”后，我会保持谈话。我意识到，你搭讪她时，大多数女孩会拒绝那些拦下步行组的男人，因为她们觉得对方是无家可归者或试图向她推销什么。

就像如果你走在街上有人拦住你，我们的默认反应是：继续走。除非你会和每个拦住你的人交谈。因为大多数人拦住你都是：“你想签名吗？”“你有钱吗？”而我是：“我在这里，我搭讪你是为了认识你，不是向你推销东西，不是向你要钱，是为了认识你。”握手也有助于巩固这一点。“我不想要什么，我不是那种推销员。”这就是我甚至叫住她的原因。她会想：“你想要什么？”我说：“我什么都不想要，姑娘，不。我来旅游，昨天刚到，所以我想问问你现在住这里吗？”

## 关键技巧：坚持与突破

即使在这个开场中，也会有一点尴尬阶段，我们停下了，然后…（沉默）。她们可能什么也不回应。有时她们也会故意封闭自己，观察你的反应。比如：“如果我封闭起来，什么也不给，他会坐立不安吗？他会慌乱吗？他是否需要从我这里得到什么？他是否想从我这里拿走什么？”如果你真的不是，你就不会介意。如果你有所图，你就会坐立不安、慌乱。

所以她们会这样做。你只需要继续**让自己感觉良好**（法则3）。坚持度过那个尴尬阶段，因为这不在于你做更多不同的事，而基本上在于**耐力**。持续表达你自己，保持“看看这个”的状态。在某个时刻，就会有足够的信息让她被吸引进来。你的目标是**激发她被吸引**，而不是把她拉进来。

## 总结

本节课中，我们一起学习了应对步行组的核心策略：
1.  **不要成为女孩目标的障碍**，学会与她同行或吸引她。
2.  **互动目标是保持对话持续**，而非一定要在原地拦住她。
3.  **可通过直接吸引、轻触开场同行、明确意图等方法来互动**。
4.  理解女孩的戒备心理，通过握手、表明来意等方式化解。
5.  最重要的是**坚持表达自己**，依靠耐力度过初始尴尬期，积累足够互动“数据”来自然吸引对方。

记住，你的角色是激发兴趣，而非生拉硬拽。通过练习这些方法，你将能更自如地应对移动中的目标。
