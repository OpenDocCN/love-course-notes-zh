# 约会学：15：收尾与达成目标 🎯

在本节课中，我们将学习如何为一次互动收尾，并明确你的最终目标。我们将探讨“达成目标”的含义、应有的心态，以及面对不同情况时的具体行动计划。

上一节我们介绍了如何与女孩建立互动和共鸣。本节中，我们来看看如何为这次互动画上句号，并实现你的最终目的。

无论你寻求的是一夜情、性伴侣还是一段关系，这一切都始于性。性是最重大的投资形式，因此你应始终以此为目标。尽可能快地尝试与她发生性关系。

## 学习过程：重复完整流程

即使你是新手，也应始终以性为目标。学习约会技巧的方法是重复整个流程：开场、吸引、建立共鸣，最终达成目标。你不应分阶段“精通”，例如先精通开场，再精通吸引。你应该在每次互动中都练习并推进整个流程。

你的最终目标始终是性。你通过始终瞄准“终点区”来学习整个流程。因此，要始终以达成目标、与女孩发生性关系为方向。

## 三种可能的结果：计划A、B、C

关于达成目标，一次互动有三种可能的结果：计划A、计划B和计划C。

以下是三种计划的具体内容：

*   **计划A：带走她**
    这是你的首要目标。带走她意味着带她回你的住处、你的车里，或任何你们将要发生性关系的地点。这是最简单的方式，你掌控着局势，无需处理她的朋友或未知的复杂情况。如果可能，就带走她。

*   **计划B：跟她走**
    如果你无法带走她（例如因为她的朋友不允许或 logistics 不允许），那么选择计划B：跟她走。这意味着去她所在的地方。希望她的室友不会和她睡同一张床，或者最终她的朋友会离开，情况会变得对你有利。

    只有当计划A和计划B都完全不可行时，你才选择计划C：获取她的电话号码。但请记住，获取号码不是计划B，而是计划C。拿到号码后，你应尝试计划第二次见面，以便在 logistics 对你有利时，执行计划A或计划B。

既然你现在以性为目标，你需要做好准备并更加主动。尤其是新手，你的思维可能会忽略某些方面。

*   随身携带安全套。
*   提前准备好你的房间。
*   开始期待性行为的发生。

对于新手来说，性行为常常在最意想不到的时候发生。如果你没有准备，就会错失良机。所以，做好准备，保持主动。
