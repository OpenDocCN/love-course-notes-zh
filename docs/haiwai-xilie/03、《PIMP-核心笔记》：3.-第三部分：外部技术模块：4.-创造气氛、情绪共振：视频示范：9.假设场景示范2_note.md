# 《PIMP 核心笔记》：3. 第三部分：外部技术模块：4. 创造气氛、情绪共振：视频示范：9. 假设场景示范2 🎬

在本节中，我们将学习一种名为“假设场景”的对话技巧。这种技巧的核心是使用两个特定的提问句型，来轻松、有趣地开启话题，特别是那些通常难以直接讨论的话题。

上一节我们介绍了创造气氛的基本概念，本节中我们来看看一个非常实用的具体方法：假设场景。

假设场景方法的核心在于使用两个固定的句型框架进行提问。以下是这两个关键句型：

1.  **“要是……（某种情况），你会怎么样？”**
    *   这个句型用于设定一个虚构的场景，并询问对方的反应或感受。

2.  **“你会不会……（某种行为或反应）？”**
    *   这个句型在设定场景后，用于更具体地引导对方想象自己的可能行为。

这两个句型之所以有效，是因为你可以将场景设定得**非常荒谬**或夸张。例如：

*   “要是你遇到一个**那个部位**特别小的男人，你会怎么办？”
*   “要是你和一个白天是直男、晚上是同性恋的男人约会，你会怎么办？”

通过这种荒诞的假设，你可以非常自然地引入一些敏感或私密的话题，例如性话题。同时，由于讨论的是完全虚构的、甚至不可能发生的情景，你无需为自己提出的问题承担任何现实中的责任或压力。这就是假设场景方法的主要优势。

本节课中我们一起学习了“假设场景”这一对话技巧。我们掌握了两个核心提问句型，并了解了如何通过构建荒诞的假设来安全、有趣地引导对话方向，从而有效地创造气氛和促进情绪共振。
