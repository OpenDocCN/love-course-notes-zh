# 老板BOSS：11：野心 🚀

在本节课中，我们将要学习“野心”这一核心概念。野心是驱动你前进的内在燃料，但同时也是一把双刃剑。我们将探讨它的定义、作用以及如何明智地运用它。

## 什么是野心？

上一节我们介绍了课程的整体框架，本节中我们来看看“野心”究竟是什么。野心是对成就更多、成为更好的渴望。它是将动机转化为行动的动力。如果你正在学习这个BOSS课程，你很可能已经怀有野心。

我认为，野心是 **动机在行动中的体现**。如果你有动机，那是一种思考和感觉。而野心，则是你将这种动机付诸实践的过程。

## 野心的两面性

理解了野心的定义后，我们需要认识到，野心既可以是积极的，也可以是消极的。

野心可能带来害处。追求在某个领域取得巨大成功，通常需要在一段时间内做出很多牺牲，导致生活失衡。你可能没时间去健身房，没精力保持健康饮食，因为需要把资金投入事业。有时你甚至需要暂时放下人际关系。野心也可能让你冲动地做出错误决定，比如过快地签署合同或进行不明智的借贷，从而让自己陷入困境。

然而，野心在很多方面也很有用。它能推动你进步，帮助你设定并追求目标，让你变得更好。BOSS课程本身就是一个充满抱负的产品，学习它意味着你想唤醒自己内在的“老板”特质。

因此，野心可以是积极的，也可以是消极的。

## 野心：汽车的燃料 🏎️

认识了野心的双重性后，我们用一个比喻来理解它的核心作用。我认为，**野心是汽车的燃料**。

你已经手握方向盘，脚踩刹车，知道自己想去哪里。你设定了目标，清楚自己能够并想要达成的成就。但如果没有燃料，汽车就无法启动。野心就是驱动你驶向目标的燃料。

## 英雄之旅中的野心

将野心比作燃料后，我们来看看它在你的个人成长旅程中扮演的角色。在你的“英雄之旅”中，你将需要野心来“屠龙”——即克服重重障碍。

BOSS课程的理念建立在“英雄之旅”的基础上。当你踏上这段征途，你会感觉自己已经改变，然后回归现实世界去对抗你的“恶龙”与“心魔”。你第一次很可能会失败，而这正是预期之中的。据说，90%的创业公司在第一年就倒闭了。因此，一个成功的商人，在成功之前很可能已经历过四五次失败的创业，并从中学习。整个“把妹”游戏的核心理念也是：你走出去，失败很多次，但在失败中学习和感受，从而变得更好。

所以，你需要有 **敢于失败的野心**。你需要有经历多次失败、甚至经历重大失败的野心。如果你还年轻，比如二三十岁，你可以快速失败并迅速恢复，因为你可能还没有家庭、房贷等重大责任压身。随着年龄增长，拥有这种“失败野心”会变得更难。如果你正处在人生中可以冒险的阶段，那就全力以赴。但无论如何，野心的概念都将极大地影响你前进的道路，因为在首次“屠龙”失败后，你需要大量的野心来支撑你再次站起。

---

本节课中我们一起学习了野心的本质。我们了解到野心是驱动行动的动机，它像汽车的燃料一样推动我们前进。同时，我们也认识到野心具有两面性，既能助人进步，也可能带来风险。关键在于，在你的“英雄之旅”中，需要运用野心去勇敢尝试，甚至包括勇敢地面对失败，并从中汲取力量继续前行。
