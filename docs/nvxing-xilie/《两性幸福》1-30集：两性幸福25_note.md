# 《两性幸福》课程：1-30集：两性幸福25

## 概述
在本节课中，我们将学习人生成长的三重境界，并掌握一套由四句话组成的疗愈方法。这些内容旨在帮助我们建立更清晰的自我认知，处理人际关系，并最终获得内心的平静与幸福。

## 人生道路上的三重境界 🎼

上一节我们探讨了幸福的基础，本节中我们来看看个人成长的三个关键阶段。

### 第一重：建立自我
建立自我是认识自己的起点。如果不知道你是谁，你就需要建立自我。这个过程意味着在心中为自己树立一个形象，并拥有一个明确的定位。建立自我会让你产生一个坚定的念头或信念，明确自己要成为什么样的人，并为之设立目标、努力奋斗。**建立自我是一切行动的动力源泉**。

### 第二重：达到忘我
忘我是建立自我之后的更高境界。什么是忘我？就是忘记自我。这是一种高度专注、执着、不达目的誓不罢休的状态。处于忘我状态的人会全心投入，感受不到疲惫、时间流逝，也不在意他人的看法。这种状态普通人难以理解和达到，却令人欣赏和向往。因此，许多人一生都在追求从建立自我到进入忘我的境界。

### 第三重：实现无我
无我是建立在自我与忘我之上的终极升华。无我就是融入自然，达到天人合一的境界。在这世上，能达到无我境界的人非常稀少。**人生的道路从建立自我开始，人生的优秀与卓越从忘我开始，而人生的伟大则从无我开始。**

## 四句疗愈真言 😔

了解了成长的阶段后，我们可能会遇到各种伤痛与困境。接下来，我们学习一种简单而强大的疗愈方法。

无论遇到何种困难或陷入何种困境，都可以尝试用以下四句话来疗愈自己。以下是这四句话的具体含义与应用方法：

*   **对不起**：说出“对不起”，意味着你终于认识到自己在这段挣扎中也扮演了角色。你原本可能认为错误都在别人，但现在明白自己也需要负起一份责任。即使不清楚具体错在思维、情绪还是行动，持续念诵“对不起”也能打开内心空间，让灵感指引你发现需要改进之处。有时我们会抗拒：“我明明没错，都是他的错，凭什么我说对不起？”这时需要明白，如果我们内心没有“我”与“他”的分裂对立，又何来争执？此时的“对不起”是在说：我忽略了彼此一体的神性，只看到了物质表象并执着于此。

*   **请原谅**：当你明白自己的过错与责任后，自然会请求对方原谅。这是谦卑的开始，也是放下“我执”的第一步。主动请求原谅是伸出友善之手，也是勇者的风范。你可以请求上主（或更高力量）的原谅，但请记住，上主也存在于对方心中。不要以上主为借口来躲避面对他人或隐藏自己的骄傲。如果暂时无法直接面对对方，可以先请求上主的帮助，让你获得面对他人的勇气。

*   **我爱你**：爱能超越因果，带你进入恩典与慈悲的世界。爱能消弭距离，让人进入同一体的境界。爱是认识彼此的真我。你与我之间，除了爱，还能是什么？哪一个思维清晰的人会拒绝爱及其带来的一切？

*   **谢谢你**：说“谢谢你”，是因为你理解了这次相聚的深层含义，看到了这段关系的可贵价值。在时空里，一切皆是关系，所有学习也离不开关系。有意义与价值的关系才是真实的。当一个人还看不到某个关系的意义时，他在这个关系课题上的学习就尚未结束。时空会一次次将他带回这个关系里，直到有一天，他能看到其意义与价值，并能由衷地说出“谢谢”。

**原来，学会说这四句话的人，才是智者。原来，爱的表现形式，就是：对不起，请原谅，谢谢你，我爱你。**
