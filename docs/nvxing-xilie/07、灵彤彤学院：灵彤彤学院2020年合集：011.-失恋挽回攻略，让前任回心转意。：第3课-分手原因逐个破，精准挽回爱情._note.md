# 灵彤彤学院：011：第3课-分手原因逐个破，精准挽回爱情

## 概述
在本节课中，我们将学习如何精准分析分手原因，并针对不同原因制定有效的挽回策略。理解分手的根本症结是成功挽回的第一步。

## 课程正文

很多人在被分手以后，甚至没有好好分析爱情失败的原因就闷头去挽回了。实际上，这种鲁莽的做法不可能将对方追回来。你连症结在哪里都不知道，如何对症下药呢？

就挽回前任这件事来说，关键在于找出你们分手的**真正原因**。对方提出分手时，出于各种心思，未必会说出真实原因。想要找到真正导致分手的原因、做好挽回的第一步，需要你自己根据实际情况去分析。

无论因为哪种原因分手，都会让人伤心欲绝。所以很多人分手后迫切想复合。如果你想要挽回对方，请在做出决定前，先好好思考如下几个问题。

以下是需要你冷静思考的几个核心问题：
*   **第一个，你为什么要挽回？** 你是真心爱她，还是不甘心被甩的感觉？你是真的觉得做错了希望改过自新，还是怀疑自己再也找不到这么好的女人了？你是否能承受挽回路上漫长而痛苦的折磨？
*   **其次，性格是否能够磨合？** 试想如果你们重新在一起，你是否能包容她离开你及之后所做的一切？你是否有信心今后的生活能够幸福，而不是像以前一样天天吵架冷战？
*   **接着，审视自己的内心。** 如果你能想清楚以上问题，真实认识到自己的错误，能够包容她且不推卸责任，请问问自己到底有多爱她。如果她是你生命中的真爱，请你义无反顾地去做。
*   **最后，端正挽回动机。** 如果你是个多情的种子，对每个喜欢的人都投入很多感情，无法释怀对方的离开，想从技术层面操作，请谨记切勿走火入魔。对于一段没有结果的感情，一开始就应该戛然而止。

当你想清楚这些问题后，对这段感情就会有更清晰的认识。决定挽回后，第一件事不是去纠缠对方，而是找出分手的根本原因。只有解决了这一点，你的挽回才能有好的基础。

如果你实在不知道从何分析，下面的几种常见分手原因可以给你启发。

### 原因一：自身缺点过多或太明显
这是导致分手的最常见情况。当你的缺点太多或太明显时，这些缺点就成了陋习。如果你不去改正，对方在难以接受的情况下必然会选择分手。

你可能会说对方既然选择和你在一起，就应该包容你的不足。爱情需要包容没错，但你要知道，人的耐心是有限的。如果你毫无改正的想法甚至变本加厉，对方怎么可能接受得了。

**挽回方法：提升自我**
核心策略是**用新的闪光点去对冲过往的不足**，同时最好也能修复缺点。
1.  把对方喜欢的运动、娱乐、食物等兴趣喜好列举出来，挑重点一样一样去提升自己。
2.  确保再次见面时，给对方一种焕然一新的感觉，用全新的闪光点吸引他。这暗示着：分手后，我努力改变了自己，为了有一天你能回到我身边。
3.  让自己变得更加优秀，指的是全方位的提升，包括外在美、内在美、生活格调和自身品味。例如，以前你喜欢奶茶，现在可以品味红茶拿铁，这就是一种提升。
4.  更加优秀的目的不是为了让他高攀不起，而是为了重新吸引他的目光，让他忽略感情中出现的漏洞，进而惊讶“原来你有这么好”。

爱情很简单，无非是相互吸引，吸得住就放不开。就挽回而言，你越是骄傲，对方就越不可能回头。向自己喜欢的人服软并不丢人，与未来的幸福相比，稍微放低姿态并无要紧。

### 原因二：犯了原则性错误
人与人相处都有底线，爱情也不例外。既然选择了和对方在一起，就不要去尝试触犯对方的底线，这是相互尊重的表现。原则性问题绝不能触碰，例如：**出轨、家暴、强迫对方做不愿意的事**。

从你们决定在一起的那一刻开始，就应该为爱情构建框架，然后严格执行。有奖有罚，才是维系爱情的基本方法。

因为这种原因分手，一般是真心分手，看上去没有挽回余地，其实不然。但这时最忌讳的就是你放不开手。在这个时间点，对方肯定对这份感情死了心，会向往自由。你不能做锁住他的锁链，而是要大方地松开手：“你走吧，我们还是朋友。”

接着你要做的就是等待。时间会改变一切。他会慢慢厌倦一个人的生活，开始怀念与你在一起的时光。不过，我们要按兵不动，因为这时他是犹豫徘徊的，你任何暴露需求感的动作，都会把他推远。保持联系，慢慢观察。

当他走出犹豫期，开始给你一些暗示（例如：“最近一个人经常有点烦闷”），就主动出击吧。因为她的深层意思是“我想你了”。如果此刻你能感受到两人依旧彼此相爱，只是心里有些失望导致暂时分开，你要做的便是积极主动。当然，不是跪舔，而是落落大方地把对方追回来。

**落落大方**就是有所坚持地承认错误：“是的，我有错，我承认，但我们的感情没有错。”不用做太多，把自己的内心表达出来就足够了。

### 原因三：暴露太多需求感
这是最容易被忽略的一点。有些人在爱情里会有极高的需求感，当这种需求感暴露出来后，伴侣就会感到巨大压力，一旦承受不住，就会选择分手。

例如在热恋时，要求对方时刻以自己为中心，事事汇报，甚至跟踪、怀疑对方，这种强烈的控制欲会慢慢磨灭对方的恋爱热情，他只好选择逃离这段关系，让自己可以自由呼吸。对方并不是不爱了，而是“累却不爱了”。

这种情况下，如果对方提出分手，你还要歇斯底里地责备或表达爱意，只会让情况更糟。

**挽回方法：冷冻**
很多人会走进雷区：分手后想尽办法联系对方、约见面辩解，甚至采用暴力行为强行挽回。这里的“冷冻”指的是你必须接受分手的事实，不能再以伴侣的身份自居。在这段时间内好好整理过去，找出分手的真正原因，解决、完善并提升自我。

挽回一定不能着急，你越着急就越可能犯错。要做好心理准备，挽回是一场持久战。要给对方和自己留有足够的时间冷静。对方不可能立刻对你的行为做出反应，你不能给对方太多压力。分手后，就把对方当做普通朋友来相处。平时可以多一些交流增进感情，但绝不能逼迫对方给出回复。

虽说挽回不简单，但把握好时机真的很重要。抓住一个好时机，能让挽回成功率倍增。所以你在分手后的断联期里，要好好把握时机。机会并非总能抓住，你也应该学会创造时机。

**创造机会的关键在于你的存在感和对方对你的好感。** 你要不断刷存在感（当然不是直接出现在对方面前，而是间接出现），向对方传达你还喜欢他的信号，然后创造一个机会将对方一举拿下。

因此，冷冻的时间不能太长。一旦错过最佳挽回时机，对方可能已经完全放下，甚至身边出现了新的追求者。不过在此之前，你要学会调整自己的情绪，不可以患得患失或胡思乱想。用一周时间冷静一下，然后跟对方好好沟通，改变以后的相处方式，这样的挽回机会还是挺大的。

### 原因四：家庭反对
经常会有这样的情况：“我家里人好像不太喜欢你，我夹在中间好累，我们还是分手吧。”这种情况的分手原因不在于你们之间，算是外部因素。要成功挽回，必须对症下药，解决家庭问题。

**挽回攻略：**
1.  **了解反对原因并积极改正：** 先要知道对方家里人反对的原因。有些问题如果可以改正，尽量去改。你可以向他父母表达你们的真心，用心去感动他们，让他们对你改观。懂得尊重长辈，接受建议并承认自己的问题，才有可能扭转局面。
2.  **在家长面前做好自己，提升价值：** 家长反对多是觉得对方不适合，或怕你日后受苦。他同意分手，也可能是因为认同了家长对你的看法。所以你要做好自己，提升自己，改变过往的坏习惯。例如：改正晚睡恶习，多看书提升内涵，培养乐器、绘画等健康优雅的爱好。你的价值提升了，生活健康了，他自然会对你的印象改观。
3.  **降低和家人的矛盾：** 专家说过，不到万不得已，一定不能增加与对象家庭成员的矛盾，否则会给未来的婚姻生活留下隐患。没有父母是不爱自己子女的，他们反对肯定是为儿女好。你要站在他们的角度想，明白他们的想法。每个家长都希望儿女幸福，并且认为自己的儿女值得最好的。所以，面对反对，首先要做的是尊重他们的意见，通过了解原因来改变自己，重新赢得同意。

## 三种不该挽回的分手情况
接下来要告诫大家，有三种分手情况不该去挽回。

以下是需要警惕的三种情况：
1.  **感情基础不牢靠：** 例如异地恋、网恋、单恋；一方恣意付出而另一方不懂珍惜的卑微恋爱；还有因父母反对、门不当户不对、学历外形差异大、婚恋观不同、彼此心智不成熟等导致情感基础不牢固的情况。倘若具备以上几点，这样的恋爱非常危险。
2.  **游戏人生的态度，遇人不淑：** 对方从一开始就是抱着玩一场暧昧游戏的目的。当新鲜劲儿过了，他便草草结束。而你已热血沸腾，却只能无奈看着他撤退。这时你越追，他跑得越快。扩大范围说，品性及个性有缺陷的人，发现势头不对，应趁早了断。
3.  **见异思迁的渣人：** 见一个喜欢一个。无论他以什么理由道别，不过都只是一个借口：要么已经有新人在等，要么正准备追求新人。当一个男人急于投向另一个女人的怀抱时，无论如何你都阻挡不了。一哭二闹三上吊，除了毁掉自己的形象，别无他用。

以上这些原因分手的情侣，虽然有些能够复合，但结局基本都一样：同样的问题导致再一次分手。所以希望你能仔细认清这段感情，也能拿得起，放得下。
