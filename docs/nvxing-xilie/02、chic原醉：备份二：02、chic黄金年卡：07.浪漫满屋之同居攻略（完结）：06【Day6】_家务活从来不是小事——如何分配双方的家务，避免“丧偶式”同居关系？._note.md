# 同居生活指南：02：家务活从来不是小事——如何分配家务，避免“丧偶式”同居关系？ 🏠

在本节课中，我们将学习如何与伴侣共同承担家务，建立公平合理的分工模式，从而提升同居生活的幸福感和协调性，避免因生活琐事积累矛盾。

## 概述

有些人认为因家务吵架分手很夸张。说这话的多半是很少分担家务的男生。他们不知道日积月累的家务活扛在一个人肩上有多重。从起床洗衣、出门晾晒，到下班买菜、做饭刷碗，家务永无止境。华盛顿大学的社会心理学家约翰·高特曼发现，对于最初相爱的伴侣，最容易导致分手的并非巨大困难，而是**生活的琐碎**。建立处理生活琐碎的配合模式，既能减轻体力劳动，又能排解不满情绪。本节课将探讨如何与男友成为共同经营生活的好搭档。

## 共同承担家务的重要性

首先，女生不必包揽所有家务。承担家务即承担责任。懂得分工合作的伴侣幸福感更强。

以下是家居环境对关系影响的一些数据：
*   89.5%的人认为，改善家居环境可以提高家庭幸福指数。
*   84.6%的人看到家里脏乱心情会变差。
*   53.8%的人会因家里脏乱与配偶吵架。
*   40.5%的人认为家里脏乱可能成为离婚原因之一。

这些数据说明，同居环境对双方关系有重要影响。家务活若全由一人负责，抱怨就会接踵而至，例如“你的衣服不要乱丢”、“拖鞋不要乱放”。每次说这些话都令人无奈，怕对方觉得是刁难。共同承担家务，能提升对伴侣的评价和同居满意度。

另一方面，在家务分配上达成共识的情侣，未来发展会更顺畅。因为各自清楚扮演的角色和任务，遇到麻烦时就不会纠结由谁处理或推诿责任。

## 从案例看家务的影响

给大家推荐一部短片《甜蜜与分离》，它真实叙述了一对情侣从甜蜜到分离的过程，而分离的部分原因正来自琐碎的家务。

片中，男生同居前匆忙收拾房间等待女生到来。女生入住后，房间变了模样：化妆品占据空间，墙上贴满合照。起初房间井然有序，但随时间推移逐渐凌乱。后来女生忙于工作，注重整洁的男生对糟糕环境越来越不满，双方发生争吵。女生苦恼地说：“你就不能帮帮我吗？我只需要你帮我一次。”这些话并未换来帮助，分歧扩大后，女生搬离了小屋。

很多人会遇到类似状况：当我们疲惫或忙碌时，对方不会主动替代完成工作。一起做家务能规避此问题，因为那意味着双方都需拿出态度维护共同空间，制造舒适感。我无暇顾及时你可以帮我，你有麻烦时我可以接下交接棒。共同承担家务，就是相互协调地面对未来生活。

## 应对责任分散效应，共同制定分工

接下来我们看看如何应对男友的责任分散效应，共同制定家务配比。

令女生头疼的是，很多男生会推卸干家务的责任，甚至理直气壮地说“家务活就是由女生干的”。除部分人有根深蒂固的封建思想外，还有些人是为了偷懒而故意将责任全安在女生身上。

对于前者，男友可能因成长背景认为家务应由女生完成。我们可以向他表明，同居需要共同承担家务。你可以这样说：
```plaintext
亲爱的，一个人的精力有限，我们需要一起维护我们的家。
```
然后布置一些他不参与就有损自身利益的事，比如晾衣服，并告诉他：“这个任务安排给你了，如果忘了晾，你下次就穿不上干净衣服了。”从简单的事着手，可逐渐改变其固有思维。

对于第二种找借口推卸责任的情况，可以用**责任分散效应**来解释。该效应指，当一件事由群体参与时，个人责任感会变弱；若由一人负责，其责任感则较强。例如，大学个人作业自己会负责完成，但小组作业时大家责任感变弱，觉得“总会有人去做”，最后基本由责任感最强的人承包。

男生在家务上也会弱化自身责任。因此，可以商量出明确分工，各自负责具体事项。分工方式可根据自身情况而定：

以下是几种常见的分工思路：
*   **按领域分工**：一人负责整理打扫，一人负责外出购物缴费。
*   **按劳动类型分工**：一人负责洗碗、拖地等轻活，一人负责修理、扛东西等重活。
*   **按喜好分工**：谁喜欢做饭、洗碗、扔垃圾就各自认领。

## 分配家务时的三个注意事项

分配家务时需要注意以下三点。

**第一，分配家务不是上街买菜，无需斤斤计较。** 若过分计较谁做得多、谁做得少，商量配比可能变成商务谈判。两人的付出不会完全对等，这一次他接送你，下一次你给他织围巾；这一次他帮你买药，下一次你给他洗袜子。这些事无法准确衡量价值，正是这些不计得失的小小付出，才显露出人情味。

**第二，给伴侣一个学习适应的过程。** 谁也不是天生的劳动高手。男朋友第一次操作时，我们可能缺乏耐心，急切希望他变为家务能手。其实我们自己打理家务的能力也是一步步学来的。例如综艺《出发吧爱情》中，吴京和谢楠因做饭争吵。谢楠在整个煮面过程中事无巨细地指导：“你这火不行，鸭血煮不熟”、“血水不要倒到锅里”。这些指导对吴京而言都是否定，他觉得“煮个面而已，怎么什么都不行”。或许谢楠出发点好，希望做出美味晚餐，但不断指示变成了监督和评价。夸张地说，若有人告诉我迈步会踩脏地板、摸到东西会留汗渍，我在这个空间就会如履薄冰，丢失共同生活的舒适感。这种舒适感也是我们要给做家务的伴侣营造的，让他觉得家务不是烦人事。

具体而言，你可以问他需要什么帮助。若暂时不需要，就放手让他实践。像吴京，就让他用自己的方法煮一锅面。鸭血熟不熟、口感好不好，他自己能吃出来，下次也知道改进。若有些问题他意识不到，比如碗边没洗干净，可以等下次用碗时，用非命令语气说：
```plaintext
亲爱的，下次碗边可以再刷一刷，你看这边还有一点污渍。
```
不要等过段时间再提，因为死无对证，他通常不会承认，还会想你是否在找他麻烦。

**第三，偶尔一起做一件家务，增进情感交流。** 同居生活中的事都可用来加深感情，包括一起做家务。关键在于无所谓家务成果好坏，而要注重两人在同一件事上的亲密互动和交流。就像电影《人鬼情未了》的经典片段，男女主在一起亲密接触、呢喃细语地做陶艺。你们伴随轻松音乐，将情感投注在一件家务事里，就是一件浪漫的事。

## 如何改变已形成的不平等分工

有人可能会说：“我已经包揽了几乎全部家务，要怎么解除这种不平等条约？”有此困惑，通常是因为男朋友已习惯被照顾，不愿接受直接提出的分配建议。那么，要让男生心甘情愿妥协，最好的办法是展现你的脆弱与无助，让他知道没有他帮助，你也会疲惫无措。当你在做某件家务时，用请求而非要求的话说：
```plaintext
亲爱的，可以帮我一下吗？我应付不了。如果你能分担一部分，我会非常高兴。
```

最后我们会发现，有序的家务模式进行到另一阶段时可能会卡住，比如男朋友以工作太累为由打破规则。对方这样耍赖皮时，你可以先主动承担部分工作，启动**链状效应**。链状效应指我们会受周围人或环境影响，做出正面或负面反应，同时我们也可以成为影响别人的人。当对方不想承担家务时，先接手一部分，让对方度过心理抵触期。前期不要表现出不耐烦，你可以让对方看到你帮他时也并非得心应手，希望他能与你共同承担工作之余的压力。

## 总结

本节课我们一起学习了如何通过合理分工共同承担家务。首先我们知道了，共同承担家务更有益于同居情侣获得强烈幸福感，且两人对未来生活的协调性会更高。具体方法是双方共同制定都能接受的家务配比，分配时不必太严苛，避免变成讨价还价。正式实行后应给伴侣适应过程，因为实践是检验真理的唯一标准，让他在实践中试错好过你扮演批评者。此外，可以时不时一起做同一件家务，以充分利用此机会增进同居关系。当男朋友偷懒时，可先承接部分工作，但不要显得你要彻底接受，还要让他看到长期忙家务并不轻松，你仍需他的帮助。
