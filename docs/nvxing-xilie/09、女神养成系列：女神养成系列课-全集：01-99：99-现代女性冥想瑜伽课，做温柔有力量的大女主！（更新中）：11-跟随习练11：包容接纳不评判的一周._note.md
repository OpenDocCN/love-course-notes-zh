# 女神养成系列：11：跟随习练11：包容接纳不评判的一周

## 概述
在本节课中，我们将学习一种名为“十八点制感法”的全新冥想技巧。该方法通过引导注意力依次关注身体的十八个敏感部位，帮助我们将感官从外界收回，向内专注，从而进入深度冥想状态。课程包含呼吸觉知练习、十八点制感法练习以及“OM”总持练习。

---

## 呼吸觉知练习
上一节我们介绍了课程的整体目标，本节中我们来看看练习的起点：呼吸觉知。这是为后续制感法做准备的基础练习。

首先，请来到一个舒适的坐姿。
保持眼睛闭合。
在呼吸时，无论呼吸是快、是慢，还是紊乱，都投入其中去感受和体验。
如果呼吸变快，就去感受它，不去改变。
如果呼吸是乱的，也体验这种乱的呼吸，不去改变。
如果呼吸是慢的，也只是感受。
如果呼吸被搅乱，没关系，只是感受，不需要主动改变它。
以这样的方式观察自己的呼吸，你的呼吸会充满觉知。
这个充满觉知的呼吸，可以运用到即将进行的制感练习中。
即便呼吸改变了很多次，也没关系，我们都只是感受，不去人为地改变和干预它。
如果你走神了，没关系，一次再一次地感受现在的呼吸，只是持续地感受它。
如果你感觉到现在的呼吸是缓慢且充满觉知的，那么就可以仰卧下来，开始练习十八点制感法。

## 十八点制感法练习
在制感的练习中，当你在体会敏感部位时，应该运用到刚才的、缓慢且充满觉知的呼吸。

在我们的身体中有十八个十分敏感的部位。我们将依次引导大家关注这些部位。以下是具体的引导步骤：

首先，从鼻尖开始。在吸气和呼气的时候，感受鼻尖。一吸一呼为一轮，在感受鼻尖时，做四轮呼吸。

然后，来到口腔内上颚的后侧。同样的，进行四轮呼吸。在吸气时，感受上颚的后侧有什么感觉；呼气时，有什么感觉。

现在，来到双眼。在吸气和呼气时，感受双眼发生了什么。

现在，来到眉心与眉心轮。同样的，一吸一呼为一轮。我们来感受在吸气和呼气的时候，眉心与眉心轮发生了什么。

然后，感受整个前额，即整个额头的部分。当你吸气和呼气时，整个前额有什么样的感受。

然后，来到喉轮，即喉咙的部分。在吸和呼的时候感受喉咙。

现在，将意识带到心轮。在四轮的呼吸中，来感受心轮发生了什么。

然后，来到肚脐。你的呼吸依然是正常、自然的。感受肚脐发生了什么。

现在，来到生殖器和肛门。用同样的方式去感受，在这两个部位分别发生了什么。

现在，来到你的大腿。在吸气和呼气时，感受整个大腿发生了什么。

现在，感受两个膝盖，重点感受膝盖后侧膝窝的位置。在吸气时，这里有什么感觉；在呼气时，这里又有什么感觉。

现在，我们来到小腿肌肉的上半部分，也就是靠近膝盖的部分。同样的，保持四轮呼吸，在呼吸中感受小腿肌肉上半部分。

然后，来到小腿肌肉的下半部分。在吸气和呼气的时候，只需要感受小腿肌肉下半部分发生了什么，充分地感觉。

现在，来到脚踝。在吸气时，脚踝有什么感受；在呼气时，脚踝又有什么感受。

然后，来到脚掌和整个脚底。

脚掌之后是两个大脚趾。让我们在吸气和呼气时，充分地感受两个大脚趾。

## 反向练习
然后，我们反向再来一遍，再次关注这十八个敏感部位。

从大脚趾开始，吸气和呼气时感受它。
从大脚趾，来到脚底。
从脚底，来到脚踝。
现在，让我们来观照小腿的下半段肌肉。
现在，来到小腿的上半段肌肉。要有耐心，每一个部位至少要保持四轮的吸气和呼气。如果忘记了某些部位，没关系，只需要慢慢地回到鼻尖，重新开始就可以了。
接着，来到双膝的区域，主要感受膝窝的位置。
从膝窝来到大腿的整个区域。在吸气和呼气时，感受这里发生了什么。
现在，来到生殖器。
从生殖器来到肛门。依然在感受每个部位的时候，保持四轮的呼吸。
现在，来到肚脐。
从肚脐，来到心轮。
现在，来到喉轮。从喉咙往上，身体上部的这些位置可以驻留的时间稍微长一些。
从喉轮向上，来到顶轮，即头顶的位置。
从顶轮来到额头。在吸气和呼气时，分别感受额头发生了什么。
现在，来到眉头和眉心轮。
然后，来到双眼。在吸气的时候，双眼发生了什么；在呼气的时候，双眼发生了什么。让我们至少保持四轮呼吸，甚至更长的时间。
现在我们来到口腔内部上颚的后侧。
最后，回到鼻尖。在吸气和呼气时，我们感受鼻尖有什么样的感觉。

以上是十八点制感法的练习。在这期间，依然保持仰卧，不要左侧卧，也不要右侧卧。

## OM总持练习
现在，我们来进行OM的总持练习。

这个练习是这样的：在仰卧的时候，当你吸气时吸3秒，感受气息好像去向了根轮的位置。在这3秒之内，内心默念OM中“O”的音。3秒后，当气息到达根轮时，在心里默念“M”，让气息从根轮达到顶轮。在默念“M”的音时，从根轮向上默念6个种子音，这6个种子音分别对应着我们各个不同的脉轮。

这6个种子音从下往上分别是：**Lam**、**Vam**、**Ram**、**Yam**、**Ham**、**OM**。
它们从根轮开始，经过腹轮、脐轮、心轮、喉轮、眉心轮，到达顶轮。

所以，这个OM的总持训练，是我们在内心默念的：
*   吸气3秒，内心默念“O”的音，感受气息来到根轮。
*   呼气，内心默念“M”的音，分别默念 **Lam**、**Vam**、**Ram**、**Yam**、**Ham**、**OM**，从根轮到顶轮。

现在大家可以跟随引导来默念，也可以发出声音。

以下是十轮练习的引导：

1.  吸气，默念O。呼气，默念：Lam, Vam, Ram, Yam, Ham, OM。
2.  吸气，默念O。呼气，默念：Lam, Vam, Ram, Yam, Ham, OM。
3.  吸气，默念O。呼气，默念：Lam, Vam, Ram, Yam, Ham, OM。
4.  吸气，默念O。呼气，默念：Lam, Vam, Ram, Yam, Ham, OM。
5.  吸气，默念O，保持3秒，把气息从鼻尖带到根轮。呼气，默念种子音，把气息从根轮依次经过腹轮、脐轮、心轮、喉轮，一直到达顶轮。
6.  吸气，默念O，气息从鼻尖带到根轮。呼气，默念种子音：Lam, Vam, Ram, Yam, Ham, OM。
7.  吸气，默念O。呼气，默念：Lam, Vam, Ram, Yam, Ham, OM。
8.  吸气，默念O，保持3秒，自己做。呼气，默念种子音，同时把气息从根轮带到腹轮、脐轮、心轮、喉轮、眉心轮直至顶轮。
9.  最后一遍。吸气保持3秒，在心里默念O。呼气，默念：Lam, Vam, Ram, Yam, Ham, OM，依次把气息从根轮向上带到顶轮。

## 感受与静默
现在，感受通过刚才的练习，自己的身体发生了什么，起到了什么效果。这种效果与其他方法带来的效果有什么不同？我们不需要任何语言的描述。

回想一下自己十八点制感练习的功效。只需要静静地等待一会儿，就会知道这个功效。

现在，再一次回到顶轮，感受整个身体之内发生了什么。我们不能期待没有发生的事情，我们只是简单地等待。在单纯的等待中，冥想即将就会发生。

记得在这时，我们能觉知到整个身体，觉知到双腿、双臂、头部、胸部、后背。在整个身体内有个说不出的感觉一直在发生。继续去感受已经发生的事情。

然后有一种舒适感袭来。在内心观察这种舒适感，越来越深，越来越深。静默出现了。继续停留在静默之中。那么，就进入冥想了。

如果你没有进入冥想，继续留在静默中，慢慢地感受两个脉轮：感受腹部的中心和头部的中心。不要去考虑下一步，只是停留在当下，感受当下腹部的中心和头部的中心。

然后，来到顶轮。如果你感觉到超越头顶之外、十二指的位置发生了什么，那么就去感受十二指之外的明点轮。如果你感觉到这种感受继续上行，超越了十二指的位置，那么就去感受这个上行。如果没有，就依然停留在顶轮或者明点轮。

现在，简简单单地忘掉一切，忘掉脉轮，就去感受这种静默，只有静默。

如果你感觉可以了，随时都可以坐起来，但是不要着急，要慢慢地。如果还没有准备好，那么可以继续仰卧着。

坐起来以后，继续保持2分钟的静默。
