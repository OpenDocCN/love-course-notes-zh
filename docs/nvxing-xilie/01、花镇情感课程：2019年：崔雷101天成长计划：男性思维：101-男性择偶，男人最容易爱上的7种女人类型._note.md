# 花镇情感课程：01：男性思维：101-男性择偶，男人最容易爱上的7种女人类型

在本节课中，我们将要学习一个核心概念：**投其所好**。我们将探讨其真正的目的，并分析在亲密关系中，如何通过调整思维模式来掌握主动权，而不是被动地迎合对方。

## 投其所好的真正目的

上一节我们介绍了课程主题，本节中我们来看看“投其所好”的本质含义。

想象一个场景：中间有一个火圈，火圈的一端站着一只老虎。你在火圈的另一端放了一块鸡肉。放鸡肉的目的是什么？目的是引诱老虎跳过火圈。

那么，放榴莲行不行？放西红柿、苹果、香蕉或橘子行不行？或者放一瓶雪碧呢？你认为榴莲是水果之王，营养好；苹果能让人远离疾病；香蕉能让人心情愉悦。但这些你认为好的东西，是老虎想要的吗？不是。老虎只对肉感兴趣。因此，放置诱饵的目的，是为了**引导对方产生我们期望的行为**。

所谓的“投其所好”，指的就是：**用对方在意、喜欢或需要的东西，去引导或交换，使其产生你期望的行为**。关键在于“对方需要”，而非“你认为好”。

记住一个核心点：**投其所好一定不是付出你认为好的东西**。你给老虎榴莲，它不感兴趣；但肉可能你不喜欢，老虎却喜欢。关键在于对方的需求。

## 思考：男人到底要什么？

理解了投其所好的定义后，我们自然会思考一个问题：在亲密关系中，男人到底要什么？

举个例子：小红和小明恋爱。小红发现小明喜欢吃糖醋里脊，于是她研究这道菜，希望小明能因此早点回家。后来她又发现小明喜欢番茄炒蛋，就在菜里多放鸡蛋，希望小明能多回来一分钟。她甚至一次炒菜放30个鸡蛋，希望小明提前回来30分钟。之后，小明又喜欢上锅包肉、牛排、果汁……小红就不停地学习制作新食物。

小红每天都在琢磨小明想要什么，并且每次琢磨得都很准。但有一天，小明突然对食物失去了兴趣。小红变得很慌张，因为她发现自己不明白小明要什么了。当小明“变”了，小红就觉得自己失去了价值。

那么，一个女人在什么情况下，会持续思考男人要什么呢？通常是在她试图通过满足对方需求来维系关系时。

但这里有一个关键观点：**一个女人永远无法通过掌握男人的需求来留住男人**。因为需求是会变化的，男人自己可能都不知道为何会变。当你不断琢磨对方喜欢什么时，谁才是这段关系中的赢家？是那个被琢磨的人——男人。他无需付出，就能持续获得满足。

## 转换思维：让对方来“投你所好”

因此，比“投其所好”更重要的是：**想办法让对方愿意来给我们“投其所好”**。

我们应该让男人去思考：“我的女朋友/妻子想要什么？”“我该怎么做，她才会满意？”当男人开始思考这些问题时，双方的格局就发生了转变。获益的不再只是单方面，关系也会更加平衡和稳定。

我们学习知识时，常陷入一个误区：认为一定要不断迎合对方。但更高级的做法是引导对方来关注和满足我们的需求，这样我们在关系中才能占据更主动、更稳固的位置。

## 核心要点总结

以下是本节课的核心要点总结：

1.  **投其所好的本质**：用对方需要的东西，引导其产生你期望的行为，而非付出你认为好的东西。
2.  **需求的局限性**：人的需求会变，试图通过掌握并满足对方所有需求来维系关系，是不稳定且被动的。
3.  **转换关系格局**：比“投其所好”更有效的是，**引导对方来“投你所好”**。让伴侣思考你的需求，是关系平衡的关键。

本节课中我们一起学习了“投其所好”的真正含义及其在亲密关系中的应用。关键在于从“我要如何满足他”的思维，转向“如何让他愿意来满足我”，从而建立更健康、平等的情感互动模式。
