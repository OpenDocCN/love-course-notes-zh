# 幸福感提升的秘诀：02：掌握幸福的主动权

## 概述
在本节课中，我们将要学习长期亲密关系中的核心挑战，并探讨如何主动掌握幸福，而非将其归因于运气。我们将分析长期关系中的本质矛盾、改善关系的实际困难，并从经济学和心理学理论中寻找提升幸福感的方法。

---

## 长期关系中的两大难题
上一节我们介绍了幸福是一种需要主动经营的能力。本节中，我们来看看当情侣进入长期稳定关系时，普遍会遇到的两项关键难题。这是掌握幸福主动权的第一步。

### 第一项难题：长期关系的本质矛盾
俄国作家马尔林斯基说过：“毫无经验的初恋是迷人的，但经得起考验的爱情是无价的。” 这表明，在不同恋爱阶段，我们的感受和面临的矛盾截然不同。

社会心理学将爱情分为四个阶段：共存期、反依赖期、独立期和共生期。
*   **独立期**：双方萌生逃离对方或希望拥有独立空间的想法。
*   **共生期**：亲密关系经过反复磨合后进入相对稳定的状态，通常表现为长期的恋爱或婚姻关系。

进入共生期（即长期关系）后，大多数时间双方能彼此珍惜。但也会因为大脑中**多巴胺**、**复合胺**及**睾丸激素**的大量减少，导致激情褪去，产生审美疲劳。最常见的表现是亲密互动减少、各自忙碌、对彼此的耐心降低。

一旦发现对方越来越偏离自己理想中的形象，或暴露的缺点越来越多，就会开始怀疑感情，甚至产生放弃的念头。这就是长期关系的本质矛盾，是时间带来的巨大考验。

### 第二项难题：制造幸福感的实操困难
当个人感到孤独苦闷时，我们可以通过适应环境、设立小目标、更换饮食等方式自我解压，提升幸福感。但在亲密关系中提升幸福感，常会让人感到无力。

以下是造成这种无力感的两方面原因：
1.  **关系涉及双方**：亲密关系由两人组成，仅靠一方的努力很难扭转局面，还需要花费精力引导对方共同付出。
2.  **需要极强耐心**：这要求实施方有极强的耐心，利用较长时间不厌其烦地尝试，直到看到预期结果。

例如，在综艺节目《爱情保卫战》中，一对交往三年的情侣，女生一直单方面制造惊喜，却得不到男生的积极反馈。她最终忍不住开始无理取闹，设定苛刻条件，导致男友产生抵触情绪，感情面临分手危机。这正说明了单方面付出维持关系的困难性。

## 为何必须维持长期关系：沉默成本的角度
上一节我们了解了长期关系中的难题，本节我们从经济学的“沉默成本”概念出发，分析为何解决长期关系矛盾刻不容缓。

在感情的共生期分手最为可惜。这涉及到 **“沉默成本”** 的概念。
沉默成本是指以往发生的、与当前决策无关的支出。人们在决定是否做一件事时，不仅看未来收益，也会考虑过去已投入的成本。这些已发生且不可回收的支出，如时间、金钱、精力，就是沉默成本。

因此，相恋三年分手比相恋三个月分手要艰难得多，主要原因就是“舍不得”——舍不得已在对方身上投入的感情和时间。这是在决策时考虑了沉默成本的权重。

针对沉默成本，可能存在以下两个误区：

**第一个误区**：在一段亲密关系中，沉默成本并非单方面付出，而是两人感情的共同见证。陷入感情瓶颈时，负面情绪会放大自身单方面的成本，忽略对方的付出，并否定已达成的感情成果。未经改善措施就放弃，十分遗憾。犹豫时间越长，造成的感情停滞或损伤越大，沉默成本越多，后续修复也越难。

**第二个误区**：选择分手后，与他人重建亲密关系的风险同样很高，沉默成本也开始了从零计费的过程。一组微博投票数据显示，与前任复合并最终走到一起的概率有52.5%。这证明大多数人会从沉默成本中找到坚持下去的理由（无论是爱还是责任感），而非选择开展新恋情或独身。

## 制造幸福感的起点：爱情三角理论
了解了维持长期关系的必要性后，我们来看看制造幸福感的具体方向应从何处入手。

有位姑娘留言说，每次和男友吵完架都感到全身无力，觉得多年的交往似乎被无休止的冲突占据，开心时刻很少，但分开又舍不得，于是归因于自己“运气不好”。

实际上，在爱情中能否幸福，不是运气问题。幸福需要自己争取，它是一种经营能力。就像酿制红酒，需要手艺调配、时间发酵，才能收获美酒。

美国心理学家罗伯特·斯滕伯格提出了 **“爱情三角理论”** 。该理论认为爱情由三个基本元素组成：**亲密**、**激情**和**忠诚**。
*   **亲密**：指在爱情关系中能引起温暖体验的情感联结。
*   **激情**：指爱情中的性欲成分，是情绪上的着迷。
*   **忠诚**：指维持关系的决定、期许或担保。

这三个元素像三原色，通过不同比重组成不同形式的爱情。例如：
*   **迷恋式爱情**：只有激情，缺乏亲密与忠诚（如短暂的初恋）。
*   **喜欢式爱情**：亲密感很强，但缺乏激情和忠诚（如互有好感但不愿在一起的异性朋友）。
*   **完美式爱情**：同时具备三要素，是爱情的理想状态，但较难达到。

因此，想要制造长期关系中的幸福感，可以从平衡爱情三元素的比例入手，让感情状态趋于完美。

同时，美国心理学家伊莱恩·哈特菲尔德根据社会交换理论，将爱情分为**激情之爱**与**伴侣之爱**。这可以看作是斯滕伯格理论中两种要素组合形式：
*   **伴侣之爱**：在长期相处中形成了亲密与忠诚，激情比重很少。
*   有的夫妻甚至会出现亲密和激情同时减少的情况，导致关系难以为继，可能做出违背承诺的事。

由此可见，在一段亲密关系中，如果**亲密**、**激情**、**忠诚**这三个要素的量值都下降，会严重影响双方的幸福感。

## 总结
本节课中，我们一起学习了如何掌握长期关系中的幸福主动权。

首先，掌握幸福主动权需要了解长期关系中的两项难题：**时间带来的本质矛盾**以及**改善关系的实操困难**。

其次，从 **“沉默成本”** 的经济学角度出发，解决长期关系矛盾是刻不容缓的，草率分手或更换伴侣并非最佳解决方案。

最后，我们从斯滕伯格的 **“爱情三角理论”** 中找到了提升幸福感的灵感。长期关系中幸福感的来源，在于掌握并平衡**亲密**、**激情**和**忠诚**三要素，通过塑造趋向完美的爱情，在平淡中收获稳固的幸福。

在接下来的课程中，我们将通过深入分析这三种爱情元素，帮助大家找到提升幸福感的具体秘诀。
