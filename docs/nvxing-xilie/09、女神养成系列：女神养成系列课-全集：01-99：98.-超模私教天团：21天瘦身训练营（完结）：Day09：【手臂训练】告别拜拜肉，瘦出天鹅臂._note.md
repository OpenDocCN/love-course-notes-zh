## 热身环节

开始正式训练前，必须进行充分热身以激活身体、预防受伤。以下是热身动作序列。

**肩关节热身**
双手置于肩关节处，进行顺时针绕环。
1, 2, 3, 4, 5, 6, 7。
接着进行逆时针绕环。
1, 2, 3, 4, 5, 6, 7。

**肘关节热身**
保持上臂不动，仅活动肘关节。先向前活动。
1, 2, 3, 4, 5, 6, 7。
然后换反方向活动。
1, 2, 3, 4, 5, 6, 7。

**髋关节热身**
双手反手叉腰，双脚打开，脚尖向外45度。顺时针活动髋关节。
1, 2, 3, 4, 5, 6, 7。
接着逆时针活动。
1, 2, 3, 4, 5, 6, 7。

**膝关节热身**
双手放在膝关节位置，进行膝关节屈伸。
1, 2, 3, 4, 5, 6, 7。

**弓箭步动态拉伸**
双脚脚尖朝前，迈出最大步幅成弓箭步。双手叉腰，上半身挺直。吸气准备，吐气时身体向下压。注意髋关节朝向正前方，上半身挺直以拉伸髂腰肌。
左侧：1, 2, 3, 4, 5, 6。
换右侧：1, 2, 3, 4, 5, 6。

**大腿后侧与内侧拉伸**
双脚横向打开，脚尖朝前，膝盖伸直。双手交叉置于身前，挺胸立腰。膝盖伸直，身体向下压以拉伸大腿后侧。
1, 2, 3, 4, 5, 6。
接着，屈左腿，右腿保持伸直。用左手膝盖顶住左腿膝关节，右手按住右腿膝关节保持伸直，进行侧弓步压腿以拉伸大腿内侧（内收肌）。
左侧：1, 2, 3, 4, 5, 6, 7。
换右侧：1, 2, 3, 4, 5, 6, 7。

**手腕脚腕活动**
活动手腕脚腕。
左脚：1, 2, 3, 4, 5, 6, 7。
换右脚：1, 2, 3, 4, 5, 6, 7。

热身结束，下面开始正式训练。

## 第一组训练

以下是第一组训练动作，每个动作需完成15次（特别说明的除外）。注意保持动作规范与呼吸节奏。

**1. 跪式俯卧撑**
*   **动作要点**：身体下压和推起时，腰背挺直，收紧腹部，避免臀部后撅。
*   **次数**：15次。

**2. 俯身哑铃划船**
*   **动作要点**：身体俯身时保持腰背挺直，不可弓腰。向上拉哑铃时，感受肩胛骨收紧。
*   **次数**：15次。

**3. 站姿哑铃推举**
*   **动作要点**：向上推举哑铃时，始终保持腰腹收紧，避免塌腰或挺肚。手腕保持直立，不可折叠。
*   **次数**：15次。

**4. 转体哑铃上勾拳**
*   **动作要点**：转体时保持腰腹收紧。手持哑铃的手腕保持直立，不可折叠。然后做出上勾拳动作。
*   **次数**：左右各完成一次算1次，共15次。

**5. 哑铃站姿颈后臂屈伸**
*   **动作要点**：双手托住哑铃置于颈后，保持挺胸收腹状态，避免塌腰、弓背或挺肚。手臂伸直时，注意肘关节不要超伸锁死。
*   **次数**：15次。

**6. 仰卧卷腹**
*   **动作要点**：上半身起身，直至肩胛骨离开地面即可，然后长长吐气，感觉将腹腔气体排空。
*   **次数**：15次。

**7. 俯撑交替肘膝碰触**
*   **动作要点**：保持腰腹收紧，避免塌腰或弓背。
*   **次数**：左右各8次。

## 组间休息

第一组训练完成，现在进入组间休息，时长约3分钟。

休息期间，可以小口、少量地饮水。建议四处走动，不要坐下或躺下。可以进行轻微的动态拉伸，但避免长时间的静态拉伸，以免肌肉过度松弛影响后续训练状态。利用这段时间深呼吸，调整均匀呼吸，放松身体，让心率平复。

休息结束，我们开始第二组训练。

## 拉伸环节

训练结束后，必须进行拉伸以帮助肌肉恢复，提高柔韧性。以下是上半身拉伸动作，每个静态拉伸保持10秒钟。

**1. 背部拉伸**
双脚打开。一手触碰对侧肩胛骨，另一手扳住该手肘。侧屈身体并吐气，拉伸背部。
保持10秒，换边重复。

**2. 肩部拉伸（三角肌）**
*   **左侧**：将左臂拉近身体，吐气，左肩下沉并向后拉，尽量将手臂贴紧身体。保持10秒。
*   **右侧**：换右臂重复上述过程。保持10秒。

**3. 肩前束拉伸**
双手在身后交叉，上半身向下俯身，同时手臂向上抬至极限。保持10秒。

**4. 胸部拉伸（需垫子）**
*   **胸大肌拉伸**：趴于垫上，右手外展贴地，左手撑于左胸外侧地面。推起身体向右扭转，抬头向上看。保持10秒。换边重复。
*   **胸小肌拉伸**：姿势同上，但将贴地的手臂弯曲至90度，再次推起身体扭转。保持10秒。换边重复。

**5. 腹部拉伸**
双手撑于胸部两侧，将上半身撑起，头向上看，拉伸腹部。保持10秒。

**6. 侧腹拉伸**
回头看左脚，吐气，尽量将左侧髋关节向地面下压，感受侧腹拉伸。保持10秒。换边（看右脚）重复。

**7. 背部与脊柱拉伸**
臀部坐于脚后跟上，身体向前趴下，双手尽量向前延伸，拉伸整个背部和脊柱。保持10秒。

拉伸部分结束。
