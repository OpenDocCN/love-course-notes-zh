# 舒化课程：18：课程回顾三：7.21 追逐奖品的跑道（奖品性）🎯

在本节课中，我们将要学习一个核心概念——**奖品性**。我们将通过一个故事引入，详细解释奖品性的含义、重要性，以及如何通过具体的方法提升自己的奖品性，从而在感情中占据高位，引导对方进行有价值的追逐。

## 概述：什么是奖品性？

我们先讲一个招工的故事。朋友在人事部，老板要求招100个人，但一周只招到10个人。他很发愁，担心完不成任务会被扣工资。我帮他修改了招工广告。新广告写道：“由于本公司业务不断扩大，前一段时间已招满18至24岁女工200名。在此基础上，面向社会招聘100名生产工人。”朋友将新广告贴出，一周后，公司果然招够了100名年富力强的青壮年。

这个故事说明，女孩子天生具有**奖品性**，而男生天生喜欢**追逐**。群里有时会看到女生说：“他送我礼物，我不敢收。让他发红包，他会不会觉得我物质？收了他的东西，我感觉欠他的。”这时，老学员常会问：“你的奖品性呢？”奖品性与收红包、收礼物有什么关系？

奖品性，就是让自己具备**奖品的特性**。奖品大家都不陌生。幼儿园老师奖励的小红花是奖品。运动会上，第一名获得的奖杯和奖状也是奖品。但奖品是对一个人能力的证明，并非人人可得。只有跑得最快的第一名才能拿到。奖品是被追逐的，且追逐有规则，只有按规则表现最好的人才能获得。

## 为何要将自己打造成奖品？🎁

将奖品性放到女孩子身上，第一个想到的是**校花**。校花通常非常美丽、优秀，被许多男生追逐。谁追到校花，大家都会觉得这个男生厉害。校花就像奖品一样被追逐。

我们要把自己打造成奖品，拥有高奖品性。目的是通过对方追逐的过程，以及我们自己制定的规则，来筛选出最符合标准的男生。有的女孩觉得：“我喜欢他，他符合我的条件。他追我，我就答应，用真心换真心。”😊 这种情况下，女孩最后往往伤心。因为男人和女人在感情中不同。

在感情里，男人做的是**减法**，对你的感情可能越来越少。而女孩不同，如果一个人持续关心、照顾、为你花钱、提供情绪价值，女孩可能会感动、习惯、慢慢喜欢并依赖这个人。但男人不会，不喜欢就是不喜欢。即便你主动追他、对他好，他可能依然不喜欢你。

你必须让他**追逐**你。在追逐过程中，让他付出努力、时间、精力和金钱，像闯关一样。你则要不断对他进行**推拉**，让他的情绪起伏，让他觉得你深不可测。男人的爱情必须有心理波动，一会儿觉得你喜欢他，一会儿觉得你不喜欢他。同时，他必须有足够的付出，你才能让他推进到下一步关系，就像一关一关地闯。

如果你想要发展长期关系，就要让自己**最难得到**。人性告诉我们，最难得到和永远失去的才是好的。你让他最难得到，经历更多关卡，他才会觉得你好、珍惜你、觉得你珍贵。电视剧里，男主喜欢女主，最初被外貌吸引，但一定会经历磨难和挫折才在一起。这些磨难和挫折，就是他的心理波动。

所以，一定要让自己像一个奖品，让男人去追逐。

## 如何判断与提升奖品性？📈

但是，被男人追逐就代表有奖品性了吗？不是的，这只是开始。你先吸引他，前提是他来追逐你。关键环节在于，想得到奖品是有规则的，必须按规则表现最好才能获得。就像跑800米，发令枪响后才能跑，想得第一必须跑得最快。

作为奖品，你的规则是什么？你的标准明确吗？框架硬吗？如果你很漂亮，身材好，工作能力强，但别人一追你就答应，没有筛选测试机制，那么你的奖品性也不高。外表可以吸引他，但没有追逐过程。

奖品性高低很容易理解。举个例子，你花50元买一个包，不会特别珍惜，想用就用。如果你攒了几个月工资，花2万元买一个包，你肯定会很爱惜，使用时也非常注意。这就是成本不同。

如果男生追你，你要让他闯关，这个成本就是你的价值。你要让自己很“贵”，他才会高度重视你。如果你让自己很“便宜”，就像50元的包，随时可能被换掉。

奖品性高的女孩都有自己的规则，通常是高标准，并且能严格用高标准筛选追求者。之前课程讲过标准和框架，每个人都有作业：写下自己的标准。

以下是标准举例（每个人标准不同，此为例）：
*   **有教养，懂得尊重别人。**
*   **有公德心。**
*   **会做饭。**
*   **幽默。**
*   **有正能量。**
*   **大方，能够为你付出。**
*   **专一。**

有了这些标准，要按照标准去筛选，但不要直接告诉对方“我要找这样的人”。正确做法是先通过观察，看他是否满足最基本要求（如是否有教养、公德心）。只有基本要求达标，才继续了解。如果连最基本要求都达不到，可以直接删除联系方式。不让你删是怕你后悔，但这种人没必要留在微信里当备胎。

对于其他标准，可以侧面传达。例如，要求他会做饭，可以说：“我感觉会做饭的男孩子都特别体贴，有魅力。”这就是在传递你的标准和框架。喜欢幽默的男孩，可以说：“我特别喜欢听笑话，不开心时如果有人讲笑话哄我开心，我就觉得这个人很厉害。”你也可以做服从性测试：“我今天不高兴，你要讲个笑话哄我开心。”如果男生照做了，你要夸他：“你怎么这么幽默？从哪找到这么好笑的段子？我以前没发现你还挺有魅力的。”这时要用**赋格**，给他正面情绪价值，引导他变成你喜欢的类型。

我们喜欢大方、愿意付出的男生。如何知道他是否如此？必须通过筛选和测试。收红包只是筛选测试的途径和手段，目的不是要那5.21或52.1元。当然，不是刚认识就让对方发红包。而是在他追求你时，观察节日是否有表示，惹你生气时让他发红包他是否愿意。如果这些时候都不买礼物、不发红包，那他很可能不是大方、愿意为你付出的人。

以上标准是举例，每个人标准不同。你要知道自己要什么：是情绪价值，还是大方付出？要有侧重点。在他追你时，要让他知道你想要什么类型，引导他向这个类型靠拢。

千万不要让他追你的时间太短，时间短他也容易失去兴趣。也不要怕推得太远他不追了，只要他被你吸引且没有得到你，他就想征服你，还会追你。你要做的就是保持美好，在他追你时立好规矩。这是你需要做的正确事情。

不要遇到喜欢的男生就扑上去。不要追男人，要**吸引**男人来追你。而吸引他来追你，你的奖品性至关重要。

## 如何提升奖品性？✨

怎么把自己打造成奖品，或者说怎么提高奖品性呢？以下是具体方法。

### 1. 植入“我是奖品”的思维🧠

首先，你要植入一个思维：**你是奖品**。你需要别人来追逐你、适应你。一定要记住，是他来适应你。你喜欢大方为你付出的，如果他小气，直接筛选掉。如果他想追你，就必须为你付出。这时你不能不好意思。男人追逐你，要么想让你做女朋友，要么想发生关系。他们有这种想法，你有什么不好意思提要求的？

你是一个奖品，他们在追逐你，你在选择他们。你的选择权非常值钱、有价值。是他们需要你，是他们来追逐你。所以你必须告诉他们你的标准是什么。男人天生喜欢为自己想追逐的女人付出，天生就该为自己喜欢的女人花钱。其实所有男人都知道，追一个姑娘是需要付出的。

你一定要让自己变得很“贵”，让别人知道你的规则。一定要让男生付出足够多，你才可以跟他进行下一步。以前我也不懂，别人发红包我也不收，不知道以什么立场收。后来才知道，你不收，他反而会没面子，这是他的一种表达方式。你可以接受他的好意，夸奖他、感谢他，并告诉他你收到了他的付出，很高兴很开心。😊

要不断给自己洗脑：我是奖品，别人要追逐我，就必须为我付出。

### 2. 成为一个美好的人🌟

第二，你是一个奖品，想奖品性高，你必须是一个**美好的事物**。之前课程讲过，美要漂亮、身材好。这很容易做到，你不能懒，要多运动、做面膜、做保养。还要多学习、多看书。这是外在美。

内在美呢？就是多学习、多看书，你会变成一个非常有趣的人。与人聊天时，他会觉得你特别舒服，你也懂他。你们能沟通、聊得来，这样他就会非常想跟你在一起。你的奖品性，他想追逐你的欲望就高了，这就是奖品性提高了。

除了外在，还要**自信**。有的女孩不是不漂亮，而是不自信。可能觉得年龄大了，工作不顺心，体型不够好，或经历过一段婚姻。对于所有事情，都要积极面对。任何事情都有两面性，积极应对生活中的事，你就是一个积极的人。当你变得积极，吸引来的也会是积极的人。如果你消极、不自信，觉得自己年纪大、不年轻了，你身上的光会暗下去。

一旦有了自信，不管40岁还是50岁，你身上自信的光都非常吸引人。如果身材不够好，就找到适合自己的穿衣风格去改善，依然可以很自信。如果经历过一段婚姻，要有积极态度：“我之前失败过，知道错在哪儿，正在改进，下一段一定会是更好、更适合我的人。”要有这种积极心态。

要学会接受自己所有的点，并用积极心态看待、面对。如果你能积极面对自己的缺点和不足，就会越来越自信。

给自己定一些小目标，例如：
*   **素食一个星期。**
*   **扩大社交圈。**
*   **考出一个证书。**
*   **每天学习一个新知识。**
*   **保持好奇心。**

每天要注意体态，抬头挺胸。晚上可以拿出一小时冥想、反思或放空自己，这些会让你更了解自己、更自信。有的女孩不自信是因为放不开。其实，伸一个大大的懒腰也是舒缓自己的方式。

每天对自己微笑，对别人微笑。去看一次演唱会，尽情欢呼雀跃。提前几十分钟写下计划，整理好自己和每天的生活。或者简单尝试一种没试过的指甲油颜色。这些都会让你开心。每天开心，喜欢自己，你就会越来越自信。😊

有时不要想太多，把身边小事做好，心情会越来越好，自己也会慢慢变得更美好。

### 3. 控制情绪并引入竞争机制⚖️

第三点是**情绪控制**，这非常重要。奖品性高的女孩一定能控制住情绪，或努力学着控制情绪。有时我们因为一个人生气焦躁，等不到回信息就心烦。这时你必须要有**备胎**，转移注意力。不跟他聊，可以跟别人聊，或找自己的事情做。备胎非常有效，因为你希望有人提供情绪价值，而备胎完全可以做到。

其实，**竞争机制**对男人非常有效。刚才提到的校花，并不一定多么漂亮，但有很多人追。越是追的人多，优秀的男孩就越想去挑战。别人都追，如果我追到，是对我能力的证明。有时他并不喜欢这个女孩，但看到很多人追，他也去追。这就是第三点：情绪控制，并引入竞争机制。

### 4. 制定并坚守规则📏

第四点是**制定规则**。规则就是我们的标准。我们一定要有高标准，因为男人都喜欢挑战，你要让他觉得你有挑战性。

我画了一个图。之前讲过高位和低位，女神在高位，支撑她的就是她的规则和框架。几个标准就像台阶，他要一关一关地过，每个标准都符合，才能到你的位置。但注意，标准不用太苛刻。例如，5项标准有4项合格，就可以给他机会。因为有的女孩标准写得非常多，不止10条，可能20条。你要挑出最重要的5到6条，让他符合，就可以给机会。

这个跑道就是我们的框架，只有在框架内的人，我们才可以给他机会。框架内就是按我们的规则来追逐我们的人。例如，最右边的蓝色追求者，他在跑道外面。他可能达到了4个标准，也会做饭、幽默、有正能量、肯为你付出，但**他不追逐你**。他不追逐，就不在框架内，你就不要去想。

如果他没有做到你的标准，或不符合你的标准，必须果断筛选掉，不要犹豫。不要让男人轻易得到你。有的人会说花言巧语，有的女孩耳根子软，听到甜言蜜语就动摇，这是非常错误的。所以你的框架必须要硬，因为你的框架决定了别人能欺负你的程度。

追逐你的人，有的想发展长期关系，有的只想发生关系。他们都有需求。这时你就要引导追逐：“你想追我，想跟我发展关系，那么你就要怎样做。”你要把框架和要求告诉他，具体告诉他怎么对你好。男人的思维大部分很简单，但有的人会装傻。这时你必须去测试他。有的人不懂，你就要具体跟他说。这样，装傻的人没法装，不懂的人也会懂，就知道怎么对你好、怎么追求你。

其实男人都知道，谈恋爱、追女神是需要付出的。具体怎么引导付出、追逐、投资？这需要你来练手。

## 总结与核心：练手的重要性🎯

本节课我们一起学习了**奖品性**的核心概念。我们明白了为何要成为被追逐的奖品，以及如何通过植入思维、提升自我、控制情绪和制定规则来提高奖品性。

最后，必须强调**练手**的重要性。练手首先说明你是有人追逐的。这时你要展示高价值，释放一点点可得性，然后引导追逐。在引导追逐过程中，让他的情绪对你上瘾。

撩汉或谈恋爱，很重要的两个方面：一是**心态**（你的态度），二是**技术**（话术、节奏掌握、推拉、竞争嫉妒、赋格后撤等）。这两样都要有。当你遇到喜欢的人时，心态经常不稳。那怎么撩、怎么谈？练手就是磨练技术的时候。哪怕心态崩了，但遇到喜欢的人或事，你直接就会懂该怎么处理、应对。所以练手非常重要，能帮你避免可能犯的错误。

我们必须有高的奖品性。只有奖品性高，我们才会在高位，男生才会来追逐我们。
