# 韩式肌肤的秘密：02：先导课 - 老师及课程介绍

在本节课中，我们将了解讲师背景，并预览如何仅通过家庭护理来打造健康肌肤的完整课程体系。

大家好，我是来自韩国首尔的皮肤管理师Emily，很高兴见到大家。

我在韩国担任模特，并负责皮肤科海外业务，同时也是一家韩国皮肤管理院的院长。今天能在这里向大家传授仅通过家庭护理就能打造健康肌肤的方法，我感到非常高兴。那么，我们所说的健康肌肤是指什么样的皮肤呢？没错，肤色透亮、紧致有弹性、不干燥、且不易敏感，这样的皮肤就是大家公认的健康肌肤。想必大家都渴望拥有这样的健康肌肤吧？那么，从今天起，就让我们一起努力打造健康肌肤。

首先，很多中国朋友都问过我一个问题：为什么韩国女性的皮肤那么好？她们到底做了什么护理？结论是，韩国人并没有接受什么特殊的护理。在中国能做的水光针、超声刀等项目，韩国人同样也在做。但在我往返于韩中两地的过程中，感受到的最大差异是：韩国人非常重视家庭护理，而中国朋友对家庭护理的看法则过于简单，不够重视。家庭护理其实至关重要，我个人也非常看重这一点。所以从今天开始，我将向大家传授我个人的家庭护理方法。首先，为了给大家带来希望。

我将展示一张2016年的皮肤照片。看起来非常可怕，对吧？这是前年我在中国开设皮肤管理院时，由于过度劳累和压力，突然爆发的炎症性痘痘。但作为一名皮肤管理院的院长，如果皮肤如此糟糕，谁会愿意向我学习呢？我自己也会缺乏教学信心。由于当时需要四处出差，无法在管理院接受护理，我只能在工作结束后回到酒店，独自进行家庭护理。那么，大家觉得我现在的皮肤看起来如何？今天我仅使用了基础护肤品，没有进行任何化妆。是的，现在很多人正为各种皮肤问题所困扰，去皮肤科固然重要。

但如果缺乏家庭护理的支持，效果将大打折扣。大家现在明白家庭护理有多重要了吧？接下来，我将简要介绍我后续课程的内容。首先，第一课是皮肤类型管理方法。要进行正确的家庭护理，必须先准确了解自己的皮肤，才能采取精准的管理方法。第二课，我们将学习正确的洁面方法。我在2016年非常疲惫，睡眠不足。

睡前卸妆时有些偷懒。结果，那些压力和化妆品残留物一起转化成了炎症性痘痘。由此可见正确洁面法的重要性。第三课，我们将学习去角质的必要性以及针对不同皮肤类型的去角质方法。健康的皮肤会以大约4周为周期，重复老旧角质脱落、新角质生成的过程。然而，当受到外部刺激、不良营养状态或压力等因素影响，这一正常过程被打破时，未能及时脱落的角质就会引发皮肤问题。它会堵塞毛孔，降低皮肤免疫力，使皮肤变得粗糙。这时就需要进行去角质护理，因此适度去角质非常重要。第四课，我们将学习。

如何为自己选择化妆品。市面上化妆品琳琅满目，在这众多产品中，究竟什么才是适合自己的化妆品？其选择标准又应该是什么？我们将准确学习这些内容。从第五课到第八课，我们将学习很多人最为困扰的四大问题管理法：敏感/干燥皮肤管理法、美白管理法、毛孔管理法以及痘痘管理法。第九课，我们将学习如何打造许多人羡慕的、韩国明星般的“水光肌”。第十课，针对。

希望看起来更年轻的朋友们（我也希望看起来更年轻），我将传授让我看起来年轻十岁的皮肤打造方法。第十一课，日常生活中打造小脸的方法。大家都想要小脸吧？其实小脸是每个人都可以拥有的。在第十一课中，我们将学习如何在日常生活中打造小脸。很多人实际的脸比看起来要小，这是因为生活中的不良习惯导致脸部浮肿，从而显得更大。所以我们将学习如何消除浮肿，打造小脸。第十二课，学习了这么多之后，有些人的皮肤可能会变化，比如今天还是干性，到了夏天就变成了中性皮肤，对吧？没错，我们的皮肤会随着季节。

和环境的变化而产生改变。因此，我们将学习针对不同季节的皮肤管理方法。怎么样？这就是我们将要一起学习的12堂课内容，大家期待吗？让我们一起下定决心学完这12课，直到大家都成为皮肤美女的那天，加油！再见。
