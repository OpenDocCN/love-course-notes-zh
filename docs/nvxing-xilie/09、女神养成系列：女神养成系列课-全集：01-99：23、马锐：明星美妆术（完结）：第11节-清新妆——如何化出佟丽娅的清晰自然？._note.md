# 五行美学：01：清新型妆容教程

在本节课中，我们将学习如何打造“清新型”妆容，其核心是追求自然、通透、无妆感的视觉效果，并以明星佟丽娅的清爽造型为范例进行解析。

## 概述：什么是五行美学？

上一段我们提到了课程主题，本节中我们来了解一下其理论基础。五行美学是一套将女性风格分为五种类型的体系，分别是：**清新型**、**优雅型**、**可爱型**、**性感型**和**中性型**。这套体系有两个主要作用：第一，帮助发现一个人身上不同的风格潜力；第二，为想要成为某种风格类型的人提供定制化的美妆方案。

## 清新型妆容的核心特点

了解了五行美学的基本概念后，我们聚焦到“清新型”。这种妆容适用于任何场合，其目标是打造出干净、年轻、有朝气且极度自然的整体形象。

以下是清新型妆容的六大具体特点：
1.  **底妆**：以白皙、透亮、嫩滑为主。
2.  **眉形**：采用标准眉形。
3.  **眼部**：追求明亮清澈，拒绝假睫毛和粗眼线。
4.  **腮红**：颜色需接近自然肤色。
5.  **唇部**：以裸色系为主，强调浅淡自然。
6.  **发型**：倾向于长发、直发，发色以自然黑色为佳。

## 分步打造清新型妆容

### 第一步：打造光泽肌底

清新型妆容的成败关键在于底妆的质感。底妆的目标是营造出皮肤自然的光泽感和水润度，仿佛没有上妆。

1.  **妆前准备**：在完成基础护肤（水、精华、妆前乳）后，取少量精华乳在掌心，轻轻按压在面部。这个动作能**增加皮肤的光泽度与光泽感**，同时让后续粉底更滋润。
2.  **局部提亮**：为了强化皮肤的自然光感，可以在苹果肌、眼下、额头、下巴等部位点涂少量美容油。公式可以理解为：`自然光泽 = 皮肤基底 + 精华乳 + 局部油分`。
3.  **上底妆**：
    *   工具：选择轻薄的粉底液和空气感足的软毛刷。
    *   方法：用刷子将少量粉底液均匀点涂在脸上。然后，**清新型妆容必须用手上妆**。将手上多余的粉底擦掉，用指腹快速、轻柔地拍打面部，将粉底“按压”进皮肤。这种方式能最大程度保留皮肤质感，且毫无粉痕。

### 第二步：局部定妆

为了保持清新型妆容特有的光泽感，我们不定全脸妆，只针对易出油部位进行局部定妆。

以下是需要定妆的关键区域：
*   法令纹
*   额头
*   鼻翼外侧
*   眼下
*   唇角外侧

使用小型定妆刷，将散粉轻轻按压在这些区域即可。

### 第三步：塑造自然眼妆

底妆干净后，眼妆的清晰度就尤为重要。清新型眼妆的原则是“增亮不增色”。

1.  **眼影**：选择与肤色接近的裸色调眼影。采用**平涂**手法，在眼皮上淡淡扫一层即可，目标是看不到明显颜色，但能增加眼部亮度。下眼睑也可用同色系轻微带过。
2.  **眼线**：只画内眼线。在双眼皮褶皱以内，用眼线笔轻轻填满睫毛根部，**目的是增加眼睛亮度，而不是看到眼线的形状**。眼尾处也保持自然，不要拉长。
3.  **睫毛**：拒绝假睫毛。只需用睫毛夹将睫毛**微微夹翘**，然后刷上纤长型（非浓密型）睫毛膏，简单梳理，让睫毛看起来干净、有精神即可。

### 第四步：描绘原生感眉妆

眉妆要避免明显的画眉痕迹，追求根根分明的自然感。

1.  先用眉刷梳理眉毛，并擦掉眉毛上多余的粉底。
2.  选择灰黑色的眉笔，采用**填补式画法**。即一根一根地描画，只填补眉毛间的空隙，不要勾勒出明显的眉形边框。

### 第五步：点缀好气色腮红与唇妆

腮红和唇色是提升气色的关键，但必须自然得像从皮肤里透出来一样。

1.  **腮红**：为减少粉感，建议使用**腮红膏**。颜色选择粉橘色系。使用方法：先在手上晕染开，然后用气垫粉扑或手指，从颧骨外侧向面中轻轻拍打上妆。这样正面看只有淡淡的好气色，侧面看则有自然的轮廓感。
2.  **唇妆**：清新型妆容不建议使用口红。应选择与自身唇色接近的**润色唇膏**或**唇彩**。避免使用过于哑光（显憔悴）或高珠光（显肤浅）的产品，只需滋润、提亮唇部即可。

## 总结

本节课中我们一起学习了五行美学中的“清新型”妆容。我们来回顾一下核心要点：
*   **底妆**是灵魂，需通过拍打手法和局部提亮，打造出**光泽、水润、通透**的伪素颜肌底。
*   **眉妆**采用**填补式画法**，追求无痕自然。
*   **眼妆**的核心是 **“增亮不增色”** ，通过裸色眼影、内眼线和自然睫毛来实现。
*   **腮红与唇妆**要选择**裸肤色调**（如粉橘色），并使用膏状产品，以拍打方式上妆，营造出仿佛天生的好气色。

掌握这些要点，你就能化出如同佟丽娅一般清新、自然、通透的日常妆容。
