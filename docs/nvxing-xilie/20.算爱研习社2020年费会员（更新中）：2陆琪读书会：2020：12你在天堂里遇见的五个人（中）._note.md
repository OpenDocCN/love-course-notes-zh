# 算爱研习社2020年费会员（更新中）：2：陆琪读书会：2020：12：你在天堂里遇见的五个人（中）

在本节课中，我们将继续解读《你在天堂里遇见的五个人》的中篇内容。我们将跟随主角艾迪，在天堂中遇见他的第二个人和第三个人，并深入探讨战争创伤、自我牺牲、原生家庭影响以及如何与自己和解等核心主题。

## 艾迪的17岁生日与战场回忆

上一节我们介绍了艾迪在天堂遇见的第一位蓝皮人。本节中，我们首先会看到一段关于艾迪17岁生日的插叙，然后他将遇见第二位天堂向导。

艾迪17岁生日时，遇到了他生命中最重要的姑娘玛格丽特。生日当天，家人以此调侃他，而他的母亲则欣喜地拉着他跳舞，为他感到高兴。

接着，艾迪在天堂的旅程继续。他脚下的地面变得坚实，周围景象变换，他发现自己回到了当年参战的战场。虽然书中未明确战争名称，但根据背景推断应为二战。

在参军前，艾迪曾在红宝石码头练习打枪。他父亲的朋友米基告诫他：
```
“战争不是游戏。如果你需要开枪，你就开。别自责，别犹豫。你开枪，再开枪。别去想在朝谁开枪，或者杀谁，或者为什么。你想回家，你就得开枪。”
```
这些回忆浮现时，艾迪已在战场之上。

## 遇见第二个人：上尉

艾迪在天堂遇见的第二个人是他服役时的上尉。这位上尉教会了艾迪许多战争中的生存技能。

艾迪问上尉自己为何在此。上尉没有直接回答，反而先问艾迪是否还与当年的战友保持联系。艾迪记得那些名字，但战后大家并未联系。战争既能将人们紧密联结，也能因创伤而将人们排斥开。

关于战争创伤，书中提到了美国的越战纪念碑。其设计曾引发巨大争议，但最终，这个被称为“大地伤口”的黑色三角形碑体，刻满阵亡士兵名字，成为了最具影响力的纪念地，它诠释了战争的本质。

上尉问艾迪是否回到了他们曾约定要去的游乐场。艾迪苦笑，他因伤腿被困在了那里。

## 战俘营的残酷记忆

他们的对话迅速将艾迪拉回一段残酷的回忆：他们曾被俘，关押在矿场，遭受看守的残酷折磨。

一位名叫雷伯佐的战友生病了，仍被强迫劳动。艾迪站出来维护他，却遭毒打。最终，一名看守将枪塞进雷伯佐的耳朵并开了枪。艾迪感到“自己的身体被撕成两半”。

这段经历后，艾迪不再祈祷，只与上尉讨论逃跑计划。他们发现敌人也濒临绝境。

## 逃亡、纵火与腿伤

艾迪利用在游乐场学会的杂耍技巧吸引看守注意，然后突然用石头攻击，带领战友们发起暴动，成功逃脱。

获得自由后，士兵们被愤怒驱使，决定用找到的喷火器烧毁营地。艾迪也参与其中。

然而，在焚烧一个窝棚时，艾迪恍惚看见一个小身影跑进了火海。他坚信有个孩子在里面，不顾战友阻拦，执意要冲进去营救。

混乱中，艾迪的腿突然感到一阵撕裂般的剧痛，中弹倒地。他最终被拖离了火场。

## 真相与“自我牺牲”的真谛

回到天堂的对话。上尉告诉艾迪，他的腿伤并非敌人所致。
```
“是我开枪打了你的腿。”
```
艾迪震惊且愤怒。上尉解释道，当时艾迪执意要冲进火海，时间紧迫，为了救他的命，只能开枪打伤他的腿，以便将他拖走。

艾迪追问上尉是如何牺牲的。上尉回答，在带领大家撤离时，他踩中地雷，被炸成了碎片。

接着，上尉阐述了本段的核心主题：**自我牺牲**。
```
“自我牺牲其实是生命的一部分。就应该是这样。它是不需要惋惜的，它是值得追求的东西。”
```
上尉指出，艾迪一直对自己的牺牲（健康的腿、原本的人生计划）感到愤怒和耿耿于怀。但牺牲并非失去，而是传递。
```
“有时候当你牺牲了一件宝贵的东西，你其实并没有真正的失去它，你只不过将它传递给了另一个人。”
```
上尉开枪打伤艾迪，履行了“不让任何人掉队”的诺言；艾迪失去了一条健康的腿，但保住了生命，也得到了其他东西（虽然他当时并未察觉）。

艾迪最终与上尉握手，达成了和解。上尉的天堂景象随之变化，展现出一片未被战争玷污的、原始而美丽的世界。

在离开前，艾迪再次追问自己是否在码头事故中救出了小女孩。上尉没有给出答案。

**本节总结**：艾迪与上尉的相遇揭示了“自我牺牲”的深层含义。我们所有的失去与获得，在这个世界上是等价的流转。我们的“牺牲”可能成全了他人，而我们的“获得”也常源于他人的付出。理解这一点，有助于我们接纳生命中的不公与失去。

## 遇见第三个人：神秘的老妇人

上一节我们探讨了牺牲与获取的平衡。本节中，艾迪将遇见第三位天堂向导，并直面原生家庭的影响。

艾迪发现自己身处一个新的环境，脚下是雪，身体恢复了衰老病痛的状态。他看到一个餐车式饭店，透过窗户，他震惊地发现了自己的父亲。他疯狂敲打窗户呼喊“爸爸”，但父亲毫无反应。

这时，一个声音响起：“不要生气，他听不见你说话。”说话者是一位衣着典雅的老妇人。艾迪并不认识她。

老妇人带艾迪来到山脚下，艾迪意识到她才是自己要见的第三个人，而非父亲。这让他感到孤独和困惑，他接连发问：能看到地球吗？能和上帝说话吗？能回到过去改变一切吗？

老妇人被艾迪想“回去”的念头逗乐了。艾迪激动地说，他只记得事故中抓住了小女孩的手，然后就在这里，这一切让他觉得毫无意义。

老妇人说了一句关键的话：
```
“你会得到安宁的，当你不再跟自己过不去的时候。”
```
这句话点明了许多人痛苦的根源——过于在意他人的评价，活在别人的评判体系里，无法接纳自己。真正的安宁源于拥有“被讨厌的勇气”，即允许自己不被所有人喜欢，专注于自我的内在标准。

## 老妇人的故事：红宝石码头的起源

老妇人开始讲述自己的故事。她叫鲁比，年轻时是“海象烧烤”餐车的女侍者。一位名叫埃米尔的富有冒险家爱上了她，并为纪念他们的爱情，建造了一座游乐园——这就是**红宝石码头**的前身。

然而，一场在独立纪念日燃放的烟花引发大火，烧毁了乐园。埃米尔为救火身受重伤，破产后卖掉了土地。买家保留了“红宝石码头”的名字重建公园，但已物是人非。鲁比此后从未踏足那里。

艾迪恍然大悟，他在码头旧入口处见过鲁比的画像。但他不解，这场发生在他出生前的大火，与他有何关系？

鲁比解释道：
```
“在你出生之前发生的事情仍然会影响到你。我们到过的许多地方，如果不是因为那些先于我们而来的人，可能压根不存在。”
```
我们所有人都站在前人的基础之上。我们今天的成就与生活，并非凭空而来，而是层层累积的结果。

## 父亲之死与原生家庭的烙印

鲁比告诉艾迪，她此行的目的是要告诉他**父亲真正的死因**。

回忆展开：艾迪的父亲在码头附近病倒，高烧肺炎。艾迪不得不接替父亲在码头维修的工作，以维持家庭的收入。这与艾迪离开码头、追求自己人生的愿望相悖。

一天，艾迪去医院看望已多年未交谈的父亲。他什么话也说不出，最后只是举起自己沾满油污的双手，向父亲表明他在干他的活，随后便离开了。

书中对此有一段深刻的论述：
```
“父母们很少会对自己的孩子放手，所以孩子就对他们的父母放手。他们向前走……直到很久以后，当他们的皮肤变得松垂了，心脏变得衰弱了，他们才会明白：他们的故事和他们所有的成就，都是基于父母的经历建立起来的，就像生命之河里的石头，层层叠叠。”
```
**核心解读**：父母是我们最初的“依恋物”。健康的人生需要逐步完成与父母情感上的分离，形成独立的自我。但这并不意味着否定父母的基础。我们今日的一切，都建立在父母提供的基石之上。分离与感恩，两者并不矛盾。

艾迪的父亲最终死于肺炎。而在天堂的餐车外，艾迪发现自己内心深处，依然渴望得到那个始终沉默的父亲的认可与爱。

鲁比的故事讲完了。艾迪天堂之旅的下一站，又将遇见谁？

**本节总结**：通过鲁比的故事和艾迪的回忆，我们探讨了原生家庭的深远影响。我们既要学会从情感上独立，摆脱过度依恋，也要认识到并感恩父母为我们人生打下的基础。同时，获得内心安宁的关键，在于停止过度关注外界评价，学会与自己和解。

本节课中，我们一起学习了艾迪在天堂与上尉、鲁比的相遇。我们探讨了“自我牺牲”的本质是价值的传递，审视了战争带来的创伤，并深入分析了原生家庭的影响以及如何通过“被讨厌的勇气”来获得内心的平静。下一节课，我们将继续跟随艾迪，遇见他在天堂的第四个人。
