## 第一步：建立舒适感

建立舒适感是对方不排斥与你继续发展关系的前提。那么，如何营造这种相处氛围呢？

以下是两个营造舒适感的小技巧。

**技巧一：建立平行沟通，先成为有“话聊”的朋友**

美国作家艾拉·惠勒·威尔科克斯说过：没有友情做基础的爱情，就像是沙子上盖起来的房子。想要俘获直男的心，你得先和他做朋友，在柔软舒适的友谊温床上孵化出你们的爱情。

想要和他们成为朋友，首先应该建立“平行沟通”的相处模式。平行沟通，是指跟同事、同辈或朋友之间的沟通方式。相对于“上行沟通”（给予长辈、上司）以及“下行沟通”（给予晚辈、下属），这是最舒服的安全型相处模型，无需小心翼翼讨好，也无需专制于人。

平行沟通需要注意聊天的技巧。在聊天的艺术这门课中，有关于怎样说话的详细课程。这里概括讲讲：如何把优越感让给对方，让其发现你的独特面。面对面或线上交流时，你都可以抓住对方的优势予以赞美，让他自信心爆棚。如果对方愿意向你吐露自己的委屈和脆弱，适时的关怀和鼓励也要毫不吝啬地给他。也就是要成为他的“M型”对手。

日本人将情侣在恋爱时的状态分为两种：M型和S型。M型是指总为对方考虑的一方。S型是善于主导的一方。M型的恋人给人满足感和安定感，而S型恋人给人激情，也让人觉得失控。如果我们在与直男暧昧时成为他的M型对手，就可以轻松制造舒适感。

在这里，成为M型对手的必备手册是：**提升对方的自我价值**。赞美、关怀和鼓励就是给男生贴心的好方法。此外，聊天时最重要的技能是能站在直男的视角，发现他做事和说话的深层原因，而且这个原因要对他有利。

让他发现你的观察入微、善解人意，是个特别的人。这就是运用最常用的赞美、关怀和鼓励的话术，给直男送去最窝心的温暖。

这里有一点要强调：在暧昧期不要为对方制造烦恼。因为直男的耐心通常有限，不要为了跟他套近乎，就不断说你的私事，甚至不断把你的抱怨和悲伤这样的负能量传播给他。尽量让他发现你好的一面、积极和阳光的地方，因为直男不愿意“费事儿”。

## 第二步：塑造吸引力

建立了基础的舒适感后，我们需要更进一步。直男魅惑术的第二步是塑造你的吸引力，让对方开始为你魂不守舍。

大家会发现在和直男暧昧的时候，会遇到这样的情况：你明明对他很好了，但他就是不跟你表白。如果是这样，就说明对方有很大的几率是把你作为其中一个选择，而不是必选。而培养自身魅力，让直男主动被你迷惑却不同。这是一种不管你向对方做不做出行为，甚至你认不认识对方，都会让他心动的力量。

就像我们在校园里会对一个只见过几次的男生产生心动的感觉。这不是因为他对我们好，而是因为他长得好看，或是笑起来十分阳光，他对周围的人热情大方，都可能让你小鹿乱撞。所以我们可以发现，内在吸引力是一种情绪反应。想要提升魅力，就要学会调动直男的情绪，让他们为我们欢喜，也为我们忧愁。

**让他们欢喜**，相对简单。只要你注重外在的魅力培养，就可以让对方赏心悦目。美国加利福尼亚大学的詹姆斯·罗尼教授研究过：来自于异性的吸引力，会让对方的睾丸素和皮质醇激素增加，导致他们的心情愉悦。如果跟一个大家都觉得很有魅力的异性面对面座谈，我们通常会不知不觉比平时话多，精神也会饱满些。所以这里建议大家能够关注自己的形象管理，连头发和牙齿的细节也要注意到。具体的我们在接下来的课程中也会讲到。

**让他们忧愁**，则需要一点技巧。大家有没有发现：不想恋爱的时候，就开始有对象追求你；当你很渴望爱情的时候，往往都没有好的选择。原因很简单，当男人觉得你的重心不在他身上，你就会进入一种对他欲擒故纵的状态。

同样的，如果当你们要开始升级关系的时候，也不能表现得太主动。
*   **公式/策略**：坚持初次约会不和男生接吻 → 他会更加想要得到你。
*   **公式/策略**：约会邀请了好几次，你只答应了一两次 → 他会更加期待和你见面。
*   **公式/策略**：你们一起吃过饭，但回家后没接他电话 → 他也会多少产生失落的感觉。

这些都是你的魅力所在，可以轻易猎获他心中的小鹿。你也千万不要担心不让他在第一时间得手，他就不会再追求你。如果对方因为求稳失败而不再追你，那就代表他不是一个值得托付的好男人。你也因为自己的坚持间接预防了一段糟糕的恋情。

## 第三步：激发占有欲

塑造了足够的吸引力，我们来到最后一步。直男魅惑术的最后一步是激发他的占有欲，让她开口向你表白。

*   **代码/话术示例**：“你怎么这么幽默呀，我以后也要找一个你这样的男生当对象。”
*   **代码/话术示例**：“你这么会做料理，谁嫁给你都会很幸福啊。”

这些都是在给对方吃定心丸，就是让他知道你是欣赏他的。他就会有信心表白，不会担心被你拒绝。

你还可以在约他的时候说已经有约了，而且还是单独和一位异性朋友出去。这个时候男生就会产生危机感，他的占有欲也会爆发，可能就会迫不及待地要跟你表白了。

**公式**：在恰当的时间内，越晚接受告白 → 遇到好男人的概率越大。
