# 女神养成系列：01-99：61 腿部激活：005.第四课：子宫养护瑜伽

## 概述
在本节课中，我们将学习一套专门针对子宫养护的瑜伽练习。这套练习旨在通过特定的体式，加强骨盆区域的血液循环，稳定骨盆结构，从而滋养子宫并预防相关问题。课程包含坐姿、仰卧及站立体式，适合产后妈妈及希望进行子宫保养的女性练习。

---

## 第一节：吉祥坐姿与髋部放松
上一节我们介绍了课程目标，本节中我们来看看第一个基础坐姿练习。这个体式能有效伸展髋部，加速骨盆区域的血液循环。

**体式名称：吉祥坐**
*   坐于垫面，双腿交叉盘坐，可将一侧脚踝置于另一侧小腿中部。
*   若感觉背部无法挺直或腰部后倾，可将折叠的毛毯垫于臀部下方。
*   保持脊柱向上延展，双手食指交叉，掌心向前推出，同时肩胛骨向后收，形成对抗力。
*   微屈手肘，感受手臂肌肉收紧，保持均匀呼吸。

**练习要点：**
*   保持5至8次均匀呼吸。
*   若感觉平衡不佳，可靠墙练习。
*   此体式有助于放松身心，适合产后42天以上的妈妈练习。

## 第二节：仰卧束角式与深度放松
完成了坐姿的热身，接下来我们进入仰卧的放松体式。这个动作能极大地放松骨盆区域和横膈膜。

**体式名称：仰卧束角式**
1.  坐立，双脚脚心相对，双膝向两侧打开。
2.  在双膝下方各垫一块瑜伽砖作为支撑。
3.  将高度约8-11厘米的靠枕（可用两个枕头叠放）竖放于身后，靠近骶骨位置。
4.  缓慢向后仰卧，让头部、背部完全由靠枕支撑。
5.  双臂自然放于身体两侧，掌心向上。

**练习要点：**
*   确保腰部贴实靠枕，不要悬空。
*   彻底放松髋部与腹部，想象子宫得到充分休息。
*   保持5至8次均匀呼吸，可有效缓解泌尿系统不适。
*   结束时可双膝并拢，转向一侧缓慢起身。

## 第三节：树式与骨盆稳定训练
在充分放松后，我们需要加强力量训练以稳定骨盆。首先从树式开始，寻找会阴上提、核心收紧的感觉。

**体式名称：树式**
*   山式站立，双脚与髋同宽，可手扶椅背保持平衡。
*   重心移向左脚，随吸气屈右膝，将右脚掌放于左大腿内侧、膝盖或小腿内侧。
*   支撑腿（左腿）微内旋，屈膝腿（右腿）主动外展。
*   感受从脚底到头顶的向上延展感，重点收紧会阴并向上提。
*   目视前方固定点，保持身体直立，避免塌腰。

**练习要点：**
*   保持3至5次均匀呼吸后换边。
*   能力允许时，可尝试双手离开椅子，挑战平衡。
*   此体式能有效强化核心，改善因重心前移导致的骨盆前倾。

## 第四节：战士二式及其变体
掌握了单腿平衡后，我们进一步加大强度，练习战士二式及其变体。这些体式能极好地增强下肢及骨盆稳定性。

以下是战士二式及其侧弯变体的练习步骤：

**体式一：战士二式**
1.  双脚大大分开，距离约等于或大于腿长。
2.  右脚尖外转90度，身体和肚脐仍朝前方。
3.  收紧臀部，呼气时屈右膝向下，使右大腿尽量与地面平行，膝盖对准第二脚趾。
4.  双臂向两侧平举，或手扶椅背。后腿伸直，髌骨上提。
5.  保持会阴上提，胸腔展开。

**体式二：侧角伸展式（变体）**
1.  从战士二式开始，将右手放于右脚内侧地面。
2.  左臂向上伸展，也可扶住前方的椅背以加深伸展。
3.  保持骨盆稳定，脊柱延展。

**共同要点与细节：**
*   每个体式保持3至5次均匀呼吸。
*   **关键细节**：后脚足弓内外缘均匀压地，避免膝盖内扣或过度外翻，始终保持髌骨上提。
*   练习结束后，缓慢起身，双脚内外八字收回。
*   完成一侧后，换另一侧重复以上两个体式。

## 第五节：摊尸式放松与产后休息
在完成一系列站立体式后，身体需要充分的休息与恢复。本节我们学习摊尸式的放松方法，这对产后恢复尤为重要。

**体式名称：摊尸式（辅具版本）**
1.  将准备好的厚靠枕（8-11厘米）竖放于垫子后端。
2.  缓慢侧身，然后仰卧，让整个背部和头部舒适地靠在靠枕上。
3.  双腿自然分开，脚尖微微外八即可，无需过度打开。
4.  双臂放于身体两侧，掌心向上，双肩放松。
5.  下巴微收，保持颈椎自然曲度，可额外垫高颈部。

**练习意义与时长：**
*   此体式能让人彻底放松，摄入充足氧气，有助于精力恢复和提高母乳质量。
*   **特别适合剖腹产妈妈**：在产后至恶露排净期间（约42天内），可每天练习2次，每次保持15-20分钟。
*   结束练习时，先屈膝，身体转向一侧，再用手推地慢慢坐起。
