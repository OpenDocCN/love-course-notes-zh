# 简七学理财【大众版28节】：17：家庭经济支柱，怎么买保险

在本节课中，我们将学习如何为家庭经济支柱配置保险。我们将遵循三个核心规划原则，并详细探讨寿险、重疾险、意外险和医疗险这四大险种，重点从保额和保障期限两个角度进行规划。

上一节我们介绍了如何从需求出发买对保险。总的来说，重疾险、寿险、意外险是保障的主力军，而医疗险是性价比很高的补充。不过，许多朋友会有疑问：每个家庭成员都需要这四款保险吗？购买这些产品会不会很贵？保障到什么时候最合适？接下来的两节课，我们将详细聊聊如何做好具体的保障规划。本节课信息量较大，但干货满满。如果你觉得有些地方不太明白，不妨反复学习。坚持学完，相信你会有很大收获。

在此之前，我们首先需要了解为家庭做好保障规划的**三个重要原则**。

以下是三个重要原则：

1.  **原则一：家庭成员保险配置顺序不同**。在同样的风险下，家庭中哪位成员遭遇风险对家庭经济的打击最大，谁就最需要保障。显然，收入最多的家庭经济支柱是最需要保障的重点。因此，从家庭责任角度看，应优先为家庭主要劳动力配置保险，其次再配置孩子和老人的保险。
2.  **原则二：不同家庭成员保障重点不同**。例如，未成年的孩子一般不太需要寿险保障。而对于有房贷、需抚养孩子的家庭支柱来说，寿险则是必备产品。
3.  **原则三：不同家庭适用的保额与保费各不相同**。预算宽裕可选更高保额、更长保障期限；预算有限则需抓住重点，保障责任最重的时期。这里需回归保险本质：**用能够承受的钱（保费）去转移无法承担的重大经济风险（保障）**。在保费上，建议预留家庭年收入**5%到8%**，最多不超过**10%** 的钱作为每年的保障投入。这个比例既能选到相对充足的保障，又不会对家庭经济造成过多压力。

理解了以上三个原则，本节课我们将重点探讨家庭经济支柱的保障配置。家庭经济支柱往往需要最全面的保障，我们将分别针对四大险种——寿险、重疾险、意外险和医疗险进行介绍，并重点从保额和保障期限两个角度做好规划。

## 寿险规划

首先我们来聊聊第一个产品：寿险。我们将学习如何确定寿险的保额和保障期限。

上节课提到，寿险的本质是为了转移不幸身故后未尽到的家庭责任。因此，确定寿险保额与你的家庭责任息息相关。

家庭责任具体指哪些呢？一个简单的计算方法是梳理大额负债和未来确定的大额刚性支出，进行估算。**寿险保额 ≈ 未还完的房贷 + 其他大额负债 + 子女教育等刚性大额目标**。

举个例子，朋友Cindy希望通过寿险转移的风险有：未来房贷200万、未来十年女儿的教育费用70万、车贷和朋友欠款30万。把这几项加起来，Cindy一家比较适合的寿险保额是 **200 + 70 + 30 = 300万**。她和丈夫都有不错收入，寿险保额可按收入比例分配。例如，她月薪1万元，丈夫月薪2万元，那么按比例，Cindy自己需要100万保额的寿险，丈夫则需要200万保额的寿险。

除了保额，寿险的保障期限也是重要选择。我们再次回归家庭的经济责任。通常，普通人一生的经济责任是一条类似山峰的曲线。二三十岁组建家庭后，家庭责任逐步上升。随着房贷还完、子女长大、财富积累，所肩负的家庭经济责任会越来越少。因此，对于普通人来说，寿险并不需要买终身那么长。一般保障期限覆盖房贷等大额负债的偿还期限，或者到子女成年就足够了。

仍以Cindy举例，假设她的房贷需25年偿还，女儿研究生毕业还需10年，车贷5年后还清。那么她与丈夫就比较适合选择保障期限为**25年**的寿险。

## 重疾险规划

说完了寿险，我们再来看看第二种非常重要的保险：重疾险。这是大家最关心、问得最多的保险种类。我们将从保额、保障期限和保障疾病种类三个方面详细探讨。

首先是重疾险的保额。之前介绍过，重疾险保障我们在不幸罹患重大疾病时所面临的巨额经济损失。我们需要通过分析这个巨额经济损失的构成来确定保额。一般来说，至少需考虑三个方面：

以下是确定重疾险保额需考虑的三个方面：

1.  **治疗费用**：保额需能覆盖常见重疾的平均治疗成本。根据近几年的专业数据，重疾治疗的平均费用通常在**20万到30万**甚至更高，其中医保可覆盖一部分。
2.  **康复费用**：对于大多数重疾，治疗只是第一步，术后休养往往还需要**3到5年**。除治疗费用外，还需准备休养期的康复费用、后续医药费等。
3.  **收入补偿**：身患重疾后，一段时间内继续工作的可能性很低，工作收入大幅减少。因此，需估算休养期间的收入损失，纳入重疾险保额。

所以，重疾险保额大概可以等于：**治疗费用 + 康复费用 + 收入补偿**。一般来说，**50万**是重疾险的基础保额。如果低于这个保额，保障往往不够充分。如果你还有能力，建议可适当增加保额，比如保到**80万或100万**都是合理的。适当从宽规划主要有两个原因：一是考虑到医疗费用会不断上升；二是随着年龄增长，同样保额的重疾险保费会上升，甚至可能因身体状况变化而无法投保，以后再补充存在一定风险。当然，前提还是要量力而为。

那么，购买重疾险，我们选择保障到多少岁更合适呢？大家都知道，年老后患病概率会越来越高。我们当然希望重疾险保障久一点，甚至保障到终身。但问题还是要回归保险的初衷：用合理的成本充分转移重大风险。因此，选择保障期限需结合自己的预算。

如果你的收入不错，预算充足，希望得到更持久的保障，终身型的重疾险更适合你。如果你的保险预算有限，就只能抓住重点矛盾，保障到**60或70岁**会更适合。通过适当缩短保障期限，可以在经济负担最大的时期获得充足的保额。

我们来举一些例子。比如年收入6万元的D先生，年保费预算上限设置在6000元比较合适。面对同一款重疾险产品，他有以下三种选择：

以下是D先生的三种重疾险选择方案：

*   **选项一**：重疾保额50万，保障至终身，年保费5850元。这个选择的问题在于预算超支。如果再加上寿险、意外险、医疗险等，总保费会大幅超过6000元，压力过大。
*   **选项二**：重疾保额30万，保障至终身，年保费3510元。这个选择的问题在于保障不够充分。作为家庭经济支柱，30万保额往往无法覆盖重疾带来的经济损失。
*   **选项三**：重疾保额50万，保障至70岁，年保费3700元。50万保额比较充足，保障期限覆盖了整个工作期（即重大经济责任时期），保费也合理。所以这个方案更适合他。

另外，细心的朋友可能发现，不同重疾险保障的疾病范围不同，有的80种，有的100种，这两款会有多大区别呢？这里要特别强调：**保障的疾病种类并非越多越好**。根据保险公司理赔数据显示，中国保险行业协会统一定义的**25种重疾**，已经覆盖了大约**98%** 的重大疾病发生率。所以，市场上常见的保50种、70种、100种疾病的重疾险，其实都包含了规定的25种重疾。如果保费差距不多，可以适当选择保障范围更广的；但如果因为增加重疾数量导致保费大幅度上升，那就不划算了。

## 意外险规划

聊完了重疾险，我们来看看第三类保险：意外险。我们同样关注保额和保障期限两个方面。

意外险用于保障意外事故带来的伤残和死亡风险。那么保额的确定重点就要落在意外身故和意外伤残这两方面的损失评估上。对于身故风险，我们有寿险可以作为保障，搭配意外险可以多一重保障。但这并不意味着意外险不重要。

一方面，面对意外伤残的风险，寿险、重疾险都无法发挥作用，而且伤残的赔付比例是按照伤残程度划分的，赔付比例在**10%到100%** 不等。因此我们更需要配足意外险的保额，否则按比例赔付下来，可能会杯水车薪。另一方面，意外险保费很便宜，是保障杠杆最大的保险，一般几百块就可以买到上百万的保障，是性价比很高的一类保险。

所以我建议大家，意外险保额最少在**50万**，理想保额在**100万以上**。由于大部分互联网渠道销售的意外险保额上限是50万，如果你预算充足，可以叠加多份意外险，将保额做得更充分。

聊完了保额，再来看看意外险的保险期限。一般建议你选购**一年期**的综合意外险，每年续保。原因主要是这样购买从保费方面看最划算。有朋友可能会问，这样的话，意外险续保会不会很难？意外险和寿险、重疾险不同，保费一般不随年龄增长而增长，而且对身体健康程度的要求很低，所以续保通常都很容易。另外，如果你出差出行比较多，也可以再考虑补充一些交通相关的意外险。

## 医疗险规划

除了上面几款保险之外，预算充足的朋友不妨再补充一份商业医疗险。在确定医疗险的保额前，我先科普一个小知识点：和重疾险直接给付保额不同，医疗险通常属于**报销型保险**。简单理解就是花多少报多少，没有额外补偿。所以，医疗险的保额其实是一个报销额度的上限。

举个例子，如果小A购买了一款100万额度的商业医疗险，看病花了20万元，那么只能在20万以内进行报销，剩余保额这次是用不到的。但如果这个疾病符合重疾险的赔付条件，小A之前买了50万元的重疾险，那么小A可以获得的重疾险赔付就是**50万（保额）**。这也是为什么说医疗险更适合做重疾险的补充，而不是直接替代重疾险的原因。

这里多说一句，重疾险的赔付和医疗险的报销是**不冲突**的。还是拿小A举例，如果同时购买了上面的重疾险和医疗险，按照之前提到的情况，他一共可以获得**70万元**的保险赔付，即看病花的20万元医疗险报销以及50万元的重疾险赔付。

目前市面上常见的百万保额的医疗险，几百块钱的花费就能买到百万的报销额度，对于大多数朋友来说是很不错的选择。它既可以补充重大疾病的保障，又可以弥补一些非重疾引起的住院医疗和手术费用。其实这些费用一旦产生，往往也不是小数目。

最后我们来聊一聊医疗险的保障期限。值得注意的是，医疗险大多是缴费一年、保障一年的消费型产品，而且投保时对健康情况有比较严格的要求。所以，医疗险的续保条款，你尤其需要关注以下四个问题：

以下是选择医疗险时需要关注的四个续保问题：

1.  续保时是否需要重新健康告知？
2.  理赔过是否还能续保？
3.  续保时会针对个人情况单独调整保费吗？
4.  产品如果停售了，还能续保吗？
