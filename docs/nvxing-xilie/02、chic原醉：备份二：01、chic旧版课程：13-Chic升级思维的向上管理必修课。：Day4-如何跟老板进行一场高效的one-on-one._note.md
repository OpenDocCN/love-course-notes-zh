# Chic升级思维的向上管理必修课：02：如何与老板进行高效的一对一沟通

在本节课中，我们将学习如何与老板进行一场高效的一对一沟通。这种沟通是保持信息通畅、确保认知一致、并让老板更好地理解我们处境的关键方式。

## 一对一沟通的特点与价值

上一节我们介绍了沟通的重要性，本节中我们来看看一对一沟通的具体特点。

通过沟通保持与老板之间的信息交换通畅，是所有职场人士的必修课。它是让我们与老板的认知保持一致的最好方式，同时也能让他对我们遇到的问题更好地感同身受。

我们常说见面三分情。职场中也是类似，经常与老板沟通的员工在公司里也更容易得到老板的支持和保护。

与老板沟通有书面和面对面两种方式。书面交流一般主张写一张精炼简洁的**One Page**来汇报工作的结果和问题。本节课主要讨论后者，即如何进行一场面对面的交流。

这种在外企被称为**One-on-One**的沟通方式有几个显著特点：

以下是其三个核心特点：
1.  **时间的完整性**：一对一沟通通常是提前与老板确定双方都方便的时间，相比碎片化的沟通更加完整。
2.  **空间的相对独立性**：一般会选择在老板办公室、会议室等地点，避免了被打断和谈话内容泄露的风险。
3.  **交流的深入性**：由于时间和空间的完整与独立，可以就某个问题与老板进行系统、深入的沟通。

## 一对一沟通的应用场景与预约

由于需要相对完整的时间和空间，一对一沟通不能随时随地展开。相对于日常沟通，其主要应用场景有两种。

以下是两种主要的应用场景：
*   **事务型汇报**：通常与某个具体项目或工作事项有关，汇报事件进程，时间比较随机。
*   **总结型汇报**：用来回顾总结阶段性工作情况，通常有固定的时点，如月总结、季度汇报。

无论是哪种类型的一对一沟通，第一步都是确定时间。

一般来说，只要不是临时紧急的汇报，都需要提前一周左右，通过邮件或信息等能够准确留痕的方式，与老板确认会议时间。

例如，可以这样沟通：
> 领导您好，关于XX项目的进程情况，我想在某月某日下午占用您一小时的时间跟您做一个简单的汇报，不知您届时是否方便。

接下来可以根据老板的时间观念和汇报内容的重要性来选择提醒频率。如果老板时间观念较差，或汇报内容非常要紧，期间可以多提醒几次。

在约定会议时间时，也要留出余量，尽量提前。例如，事情希望在下周五解决，那么最好能约到下周一的时间。

还有一个加分操作是，在发预约邮件时附上一份**Pre-reading**，即你需要汇报的大致内容或大纲。这能帮助老板提前了解你的思路，也帮助你自己理清逻辑，降低双方的沟通成本。

## 如何进行事务型汇报

以上是汇报前需要做的准备事项。下面的内容我们来具体说说每种类型的一对一沟通到底该怎么做。先来说事务型的汇报。

当我们要跟老板汇报一件事情的进展和结果时，首先需要明确汇报的性质和目的：顺利与否、是好消息还是坏消息、是否需要征求老板的建议或帮助。

一个大的原则叫做**结论先行**。即无论汇报的是好消息还是坏消息，都需要首先点明结果。

如果项目进展一切顺利，没有遇到大的阻碍。在汇报完最新进展后，可以接着说下一步的计划以及预期的完成时间。并在汇报结束时寻求反馈，例如问：“您对我们下一步计划有什么看法？”

如果项目结果不好，或遇到了阻碍，需要寻求老板的帮助。首先，也要在最开始点明事情的结果和汇报的意图。接着介绍事情的进展、原因和结果。

在求助过程中要注意，除了客观理性地汇报问题，也要分条缕析地说清楚我们已经尝试过的努力。这能节省老板的思考成本，让他知道我们已经努力设想过很多方案。

征求老板的意见也有诀窍。老板喜欢做选择题，而不是简答题。我们不能直接丢给老板一个开放式问题，而是要率先提取几个备选方案（A/B/C），并分析各自的优劣及所需资源，最后给出自己的倾向性建议。

这样实际上提高了讨论的效率。就像用橡皮泥捏小人，如果你给他的是接近完工的半成品，他只需要画龙点睛，就能得出完美的成品。

## 如何进行总结型汇报

关于事务性汇报就先说到这里，下面我们来聊一聊总结型的汇报该怎么做。

总结型汇报的主题通常是围绕阶段性工作总结和规划来做的，周期也相对固定。一般来说，随着入职时间增长，总结性汇报可以逐渐减少。刚入职的新人建议每1到2周做一次，一年后可减少到每月1到2次。

下面我们来看看总结型一对一沟通需要汇报哪些内容。

以下是总结型汇报的四个核心步骤：
1.  **上一阶段的总结和复盘**：总结可分为正反两方面。正向汇报收获、成绩、成长；负向说明意识到的问题、不足和反思。建议日常准备一个手账，随时记录，作为汇报素材。
2.  **咨询老板的预期和建议**：说完自己的总结，下一步是请老板说说他对你的预期和建议。升职加薪的第一步是持续超越老板的预期，而定期沟通是明确老板流动预期的绝佳方式。可以问：“您对我们目前的工作状态还满意吗？对我有什么好的建议吗？”
3.  **根据老板的预期和建议，陈述下一个阶段的规划**：在汇报前，我们一般会有大致的规划，但需根据老板的新期待进行调整。例如，原本计划完成70万销售额，但老板希望负责新产品，那么整个规划都要围绕新产品展开。
4.  **寻求反馈**：汇报结束后，要主动问老板：“您觉得我这样的计划可以吗？还需要做哪些调整？”一方面确认是否有遗漏，另一方面再次与他确认预期。

## 课程总结

本节课中我们一起学习了如何与老板进行高效的一对一沟通。

我们先介绍了一对一沟通的三个特点：时间的完整性、空间的相对独立性以及交流的深入性。这意味着我们需要提前预约，并针对具体问题与老板深入沟通。

接下来我们分别讲解了事务型汇报和总结型汇报的具体做法：
*   **事务型汇报**需要秉承**结论先行**的原则，首先点明结果，分析原因，最后寻求反馈。在征求老板意见时，要提供经过思考的选项。
*   **总结型汇报**一般有周期性，主要分为四个步骤：总结复盘、咨询预期、调整规划、寻求反馈。
