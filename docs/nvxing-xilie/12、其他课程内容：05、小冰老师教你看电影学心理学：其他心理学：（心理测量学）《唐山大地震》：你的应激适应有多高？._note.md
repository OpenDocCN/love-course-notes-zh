# 心理学入门：05：从《唐山大地震》看应激与心理测量

在本节课中，我们将学习心理学中的两个重要概念：**应激**与**心理测量学**。我们将通过电影《唐山大地震》的情节来理解应激反应，并介绍一个科学的评估工具——社会再适应评定量表。

## 🎬 电影《唐山大地震》与应激概念

上一节我们介绍了课程的基本框架，本节中我们来看看电影《唐山大地震》如何诠释心理学概念。

这部电影由冯小刚执导，徐帆主演。故事背景设定在1976年的唐山。卡车司机方大强和妻子李元妮有一对双胞胎子女。在大地震发生的夜晚，夫妻二人在外无事，但两个孩子被同一块楼板压住，救援其中一个就必须放弃另一个。父亲为救孩子而丧生，母亲则被迫做出人生中最艰难的选择：保儿子还是保女儿。她听到儿子的呼救声，判断儿子生存概率更大，最终选择了儿子。

当时女儿意识清醒，随后被其他人救走，并由军人家庭领养，从此母女、姐弟天各一方。直到32年后，他们在汶川大地震的志愿者工作中重逢。

这部电影引出了心理学中的一个核心概念：**应激**。

## ⚡ 什么是应激？

那么，什么是应激呢？应激是指个体在面临极端压力时产生的情绪与生理反应。“应”是应对的应，“激”是激动的激。极度的恐惧、愤怒、悲哀甚至快乐都可能引发应激反应。

灾难和战争是最容易导致应激的事件。因此，在重大灾难或战后，通常会有心理咨询师对相关人员进行心理干预和疏导。例如，2012年庐山地震时，成都震感强烈，许多人经历了强烈的恐惧，这种体验就是应激反应的一种。

## 🔄 日常生活中的应激

虽然战争和大地震在生活中不常发生，但我们仍会遭遇各种应激源。以下是生活中常见的应激事件：

*   人际关系危机
*   更换工作或失业
*   迁居到新城市
*   失去亲人

应激反应的持续时间因人因事而异，可能持续几小时、几天，甚至数年。个体不得不面对它。如果我们能妥善应对应激，自然是好事。若不能，应激不仅会干扰情绪，还可能威胁健康。

心理学家通过大量研究证明，当生活中需要为重大事件进行心理调整时，患病的几率会相应上升。这种由外部变化引发的压力，就是生活应激。

## 📊 评估你的应激水平：社会再适应评定量表

每个人对应激的反应不同，但可以肯定的是，每个人都会遭遇应激。现在你可以回想一下，过去一年你经历了多少应激事件，承受了多大压力。

心理学家通过研究，总结了43个常见生活事件。他们根据大数据统计对这些事件进行了评估，为每个事件赋予了一个分值。这张表格被称为 **社会再适应评定量表**。

以下是该量表的部分示例：

*   **丧偶**：100分
*   **离婚**：73分
*   **结婚**：50分
*   **失业**：47分

你可以回顾过去一年发生的生活事件，将对应分值累加，得到一个总分。这个总分对应不同的风险区间：

*   低于**150分**：患病几率较低。
*   **150-199分**：患病概率增加约37%。
*   **200-299分**：患病概率增加约50%。
*   **300分以上**：患病概率增加约79%。

这个量表有两个主要用途：第一，帮助我们调整自身情绪；第二，辅助我们做出决策。如果你发现近期应激指数已经很高，就不宜再从事可能引发高应激的活动。

## 🧪 关于心理测量学的补充说明

在介绍了应激和评估量表后，我们需要正确认识心理测量本身。

许多人将心理学等同于网络上流行的趣味心理测试。然而，那些测试大多缺乏科学依据。心理学中有一门严谨的学科叫做**心理测量学**。所有科学的量表和测试都是基于统计学原理严格设计出来的。

量表是心理咨询师的重要工具，其性质类似于医生使用的X光机或彩超仪。设计一份**信度**（可靠性）和**效度**（有效性）都高的合格量表，其难度不亚于制造一台医疗仪器。

目前，中国心理学界使用的量表多是引进国外量表后，在本土人群中进行标准化修订而成的。因此，请大家以科学的态度看待心理测试，不要轻信那些如同星座分析一样的伪科学测试。

## 📝 总结

本节课中，我们一起学习了两个核心心理学概念。

首先，我们结合电影《唐山大地震》的情节，探讨了**应激**的概念，即个体面对极端压力时的身心反应，并了解了它在日常生活中的表现。

其次，我们介绍了一个科学的评估工具——**社会再适应评定量表**。这是一个基于统计学的心理测量工具，用于评估个体过去一年所承受的应激水平，并预测相关的健康风险。
