**冥想瑜伽入门：07：核心呼吸法与基础体式**

在本节课中，我们将学习冥想瑜伽的两个核心组成部分：**腹式呼吸法**与**山式站立**。掌握正确的呼吸是连接身体与意识的桥梁，而稳固的体式则是所有练习的根基。

上一节我们介绍了冥想的基本概念，本节中我们来看看如何通过具体的呼吸和体式练习来奠定基础。

**腹式呼吸法**

腹式呼吸，又称 diaphragmatic breathing，是瑜伽练习的根基。它有助于激活副交感神经系统，促进深度放松。

以下是腹式呼吸的练习步骤：

1.  采取舒适坐姿或仰卧，将一只手轻放于腹部，另一只手放于胸口。
2.  用鼻子缓慢吸气，感受腹部像气球一样自然隆起，胸腔保持相对静止。
3.  用鼻子缓慢呼气，感受腹部自然向内收缩，排空空气。
4.  保持吸气和呼气的时长均匀，例如吸气计数4秒，呼气计数4秒。

其核心可概括为公式：**吸气时，腹部隆起；呼气时，腹部内收。**

掌握了平静的呼吸方式后，我们需要一个稳定的身体姿态来配合。接下来，我们将学习最基础的站立体式。

**山式站立**

山式是所有站立体式的起点，它教会我们如何对称、稳定地站立，建立身体的空间感和警觉性。

以下是山式站立的要点解析：

1.  双脚并拢或微分开，脚掌均匀压实地面，感受重心位于足弓。
2.  膝盖微微放松，不要锁死。大腿肌肉轻微上提。
3.  骨盆保持中立，尾骨微微向下，腹部轻柔内收。
4.  双肩向后向下沉，远离耳朵。手臂自然垂于体侧，掌心朝向身体。
5.  头顶向上延伸，下巴微收，视线平视前方。

想象你的身体像山一样稳固而挺拔，代码可以抽象地表示为：`姿态 = 对齐(脚，膝，髋，肩，耳)`。

本节课中我们一起学习了冥想瑜伽的入门关键：通过**腹式呼吸**来平静心神，以及通过**山式站立**来建立身体的稳定与对称。请在日常中多加练习这两个基础环节，它们将为后续更复杂的练习打下坚实的基础。
