社交焦虑治愈指南：02：克服社交焦虑的前提：找准你心里的“焦虑因子” 🔍

在本节课中，我们将要学习社交焦虑背后的核心心理障碍。我们将了解导致焦虑的三种情绪阶段，并深入分析三种常见的“焦虑因子”：过度的自我关注、过度的完美主义和过度的自我否定。掌握这些是克服社交焦虑的根本。

日常人际交往中，许多宝贵机遇可能因不必要的焦虑而错失。例如在聚会中，他人尽情结交朋友，而你却躲在角落；在工作中头脑风暴时，你不敢说出想法，从而失去高光时刻；在恋爱中，看到心仪的男神，主动的女生已成功搭讪，而你只会脸红和手抖。

如果你在社交中始终保持焦虑畏缩的状态，你将永远没有机会发现自己其实是受欢迎和受认可的，也没有机会挑战自身潜能。更重要的是，你很难为自己搭建可靠的社交网络，以获得外界的帮助。

为了改变这种状态，有人会直接寻找各种速成技巧，例如教如何说话、如何显得合群等。但发现关键时刻仍用不上，该害怕的还是害怕。因为技巧如同速效救心丸，只有一次性效果。要真正克服社交焦虑，需从根源上破除心理障碍，这才是治本的方法。本节课将带你找到自己的心理障碍是什么，以及如何治疗这些焦虑因子。

在分析心理障碍之前，我们先了解一个概念：社恐背后的情绪三阶段。

美国心理学家菲利浦·金巴多曾提出，每个行为背后都隐藏着复杂的情绪变化。社恐背后主要由三种情绪支配：畏惧、悲伤和愤怒。

皮克斯电影《头脑特工队》生动展现了人脑内的情绪世界。其中，每个人的大脑里都住着五位情绪指挥官：乐乐、忧忧、怕怕、厌厌和怒怒。这五个小人分工合作，掌控你的喜怒哀乐，支配你的一举一动。电影主人公莱利原本是个阳光小女孩，但搬家后面对新同学和新环境，突然变得不愿与人交流。这个过程恰恰反映了情绪三阶段。

莱利来到新学校的第一天，代表恐惧的小人“怕怕”给他列出一整串可能出现的坏情况，包括新同学可能不喜欢自己、融入不了集体、做自我介绍时忘词等。于是莱利十分犹豫，在学校门口左右徘徊。最后，五位情绪小人发生争执而失控，让莱利在新同学面前真的出丑了。

这就是形成社交恐惧的第一阶段：畏惧的情绪最先占据主导地位。

在这个阶段，主要表现是内心戏特别多，纠结应该积极面对还是仓皇逃避，并对交流结果抱有消极预期。

因为第一天上学就出丑，莱利看到别的同学对自己议论纷纷。这时代表悲伤的小人“忧忧”控制了情绪按钮，让莱利忍不住眼睛变红、流下眼泪，把自己藏起来。

这就是形成社交恐惧的第二阶段：从畏惧转换到悲伤。

在这个阶段，主要表现是自责，对自己感到失望，因别人的某些反应产生羞耻感，从而不再尝试接近他人。

最后，非常失落的莱利回到家里，莫名其妙地向爸爸妈妈发脾气。自此之后，莱利开始封闭自己，甚至无法与人进行正常沟通。

这就是形成社交恐惧的第三阶段：悲伤过后，怨恨的情绪从心中升起。

在这个阶段，主要表现是觉得别人都没把自己当回事，开始责怪世界让自己变得孤独、无法合群。

可以发现，形成社交焦虑背后有着复杂的情绪变化，而每一种情绪都由不同的心理障碍引起。但如果克服了这些心理障碍，你就能控制情绪系统，重新恢复社交自信。接下来我们进一步分析具体有哪些心理障碍。

第一种心理障碍是过度的自我关注。

这类人往往把更多注意力放在自己身上。例如在人群中，总感觉有人在注意自己细微的一举一动，仿佛有一个放大镜在照着。于是表现出手抖、脸红、心跳、说话结巴等生理反应。就像电影《国王的演讲》中，乔治六世每次面对公众演讲时，就会不知所措、紧张口吃。

我们日常生活中也有很多类似的公众场合，例如商业酒会、年会、聚餐、联谊活动等。

有同学问，如果实在太紧张而出丑了怎么办？

其实很多人觉得被人关注会尴尬，是因为没有掌握出现问题后的应急办法。自嘲就是一个万事万灵的好方法，既能缓解紧张气氛，又能给自己找到台阶下。

就像《国王的演讲》中，乔治六世最终克服了演讲恐惧的心魔。即使不小心口吃也不再自卑，而是自黑地说：“口吃可是我的标志，这才能让百姓们知道是我呀。”大家都不禁为他的风趣而鼓掌。

克服这种过度敏感的心理障碍，还有一个方法可以推荐：先从小圈子突破，再逐步融入更大的圈子。

因为越快融入群体，越能减少个人化的特征和感受。通常在一个活动内，整个环境是一个大群体，其中以组织者为中心是核心圈子，其他零星分散的是边缘圈子。你可以先从自身出发，找到有共同点的小圈子融入进去，这样能大大降低你的社交压力。

说完了第一种心理障碍，我们继续来看第二种心理障碍。

有些人在社交中害怕出糗，是因为抱着不切实际的高标准来要求自己，这就是过度的完美主义。

我有一个师妹小A，曾因过度和自己较劲，在职场中失去了年底升职加薪的机会。她原本是个极度追求完美的人，不到100分宁可放弃也不将就。就像平时工作，领导问有没有擅长商务英语的人可以去接待宾客。她因为自己雅思口语不是满分就没举手，结果让一个刚过六级的同事争取了机会并受到赏识。

最受打击的是，在一次头脑风暴会上，她原本打算发表自己独特的见解。可每当她组织好语言想要发言时，又会习惯性地否定自己的观点：“不行不行，这个观点也太普通了。”于是，讨论时间快结束了，她还是没说出看法，感到非常沮丧。最后，一个刚毕业的小男生竟然把她的观点说出来了。他讲得很浅，根本没有思考清楚，可是立刻有领导表示赞同并开始补充，而这个思路竟成了最终方案。

会议结束后，领导甚至还留下她谈心，评价她性格太“收”了，多和同事交流才能进步。她感到非常懊恼。

逻辑学上有一个概念叫做“完美的谬误”。

意思是完美主义营造了一种二选一的假象，但在现实中，更多是多项选择题里择优。更重要的是，在人际关系中，最吸引人的其实不是完美的形象，而是那些偶尔犯错的人。

我们总以为自己表现得精彩绝伦别人会记得，可事实上，过于谨慎和表现完美，反而让人觉得高冷而有距离感。人们津津乐道的是那些逗趣的时刻，这就是美国心理学家阿伦森提出的“犯错效应”。

要改变对完美主义的执着，我们要习惯将大目标拆分为小目标，降低自己对完美的预期。

例如，小A其实可以把自己的想法拆分成多个环节。在第一次发言时，先讲30%；在后续大家的补充和碰撞中，再逐步把剩下70%的观点缓缓道来，反而会带来持续的惊喜感。另外，在日常生活中，我们可以做一个“暴露缺点”的练习：不妨主动给自己找一个无伤大雅的缺点，并用夸张的方式表达，以此来锻炼自己对于缺点的接受度。

说完过度的自我关注和完美主义，接下来是最后一种心理障碍：过度的自我否定。

简单来说，就是当人际互动中出现一个不好的信号时，他们会认为那是自己的问题，并把这种负面结果带入所有类似的情境中。举个例子，当他们鼓起勇气和对方说话时，没有得到对方热情的回复，他们就会倾向于将之解释为“一定是我太无趣了”，而不是“对方可能正在忙”，并且以此来对所有类似事件做出整体的预判：“我主动搭话都是这样的结果，别人都会觉得我厚脸皮。”

要克服自我否定的心理障碍，就要学会正确的归因。

最近，主持人吴昕痛哭的话题上了热搜。原因是在访谈节目里，吴昕说到有一次跨年晚会，10个主持人的节目里只有她的被拿掉了。她不停地质问自己：“为什么别人的都不拿，只拿你的呢？”最后归因道：因为自己就是最差的，不够漂亮也不够专业。

这件事对她来说是致命打击，甚至改变了自己对待外部世界的看法。此后，在主持节目和嘉宾互动时都会小心翼翼，甚至在上恋爱节目时，无论潘玮柏怎么暗示对她的喜欢，她都下意识地躲避，被网友嘲笑“注孤生”。她痛哭，为什么自己变成这样，在社交中丧失了自信。

实际上，事实真相是当年节目里不只有她的节目被拿掉了，并且当晚只是紧急调整而已。但吴昕没有想明白，一味责怪自己，长期下来让自己遇到什么事情都习惯自我否定。

要掌握正确归因，首先，你需学会在事情发生时开启“上帝视角”，切记不要让自责和愤怒的情绪盖过理性。只有保持全方位的视角去看待，你才能让自己少背没有意义的锅。

美国社会心理学家凯利曾提出一个归因理论：每一件事情的背后基本都由三个因素组成：
1.  归因于自己
2.  归因于你的对手
3.  归因于外部环境

造成一件事的后果必然有各方面的原因。如果习惯归因给自己，其实是一种走认知捷径的懒惰思维。

为了避免信息不对称导致的归因错误，这里推荐一个小方法：你可以尝试列思维导图的方式。
1.  首先，将中心事件分为三个分支：自己、对手、外部环境。
2.  然后，把所有影响的可能性分类到这三个因素下。
3.  最后，衡量各因素占整个中心事件的百分比。

梳理清楚后，你将对整件事有一个清晰的判断，而不是一味怪责自己。

说到这里，今天的内容也就快结束了。现在让我们来回顾一下本节课的重点。

社恐的形成其实由三种情绪支配：恐惧、悲伤和愤怒。而每一个情绪阶段都对应着不同的心理障碍。

第一种心理障碍是过度的自我关注。我们可以先从小圈子突破，再逐步融入更大的圈子，来减少自己的社交压力。即使出丑，也可以用幽默的方式化解。

第二种心理障碍是过度的完美主义。我们可以拆分大目标为小目标，并且日常做暴露缺点的练习。

第三个心理障碍是过度的自我否定。我们要学会正确的归因，而不是一味让自己背锅。

通过今天这节课，希望大家能真正找到自己的心理障碍，克服这些焦虑因子。我们下节课再见。
