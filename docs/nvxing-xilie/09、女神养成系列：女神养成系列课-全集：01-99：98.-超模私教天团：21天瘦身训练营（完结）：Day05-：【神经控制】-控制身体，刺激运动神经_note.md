# 女神养成系列：01-99：98. 超模私教天团：21天瘦身训练营（完结）：Day05：【神经控制】控制身体，刺激运动神经 🏃‍♀️💪

在本节课中，我们将要学习激活期的最后一节训练课程。课程内容旨在通过一系列动作，帮助你控制身体、刺激运动神经，从而提升训练效果。我们将从热身开始，逐步进行正式训练，最后以拉伸运动结束，确保训练的科学性和完整性。

## 热身运动

在开始正式训练前，进行充分的热身至关重要。热身能激活肌肉，提高关节灵活性，预防运动损伤。以下是热身动作列表。

以下是热身动作的具体步骤：

1.  **肩部绕环**：双脚分开站立，向前和向后绕环肩部各8次。
2.  **髋关节绕环**：双脚分开站立，顺时针和逆时针绕环髋关节各9次。
3.  **膝关节活动**：双脚并拢，微屈膝盖，进行左右绕环活动。
4.  **手腕脚腕活动**：分别活动手腕和脚腕，进行顺时针和逆时针绕环各10次。

热身运动已经结束。接下来，我们将进入正式训练环节。

## 正式训练

正式训练由多个动作组成，每个动作重复15次，旨在全面刺激身体肌肉群。我们将依次完成以下动作。

以下是第一个训练动作的要点：

*   **箭步蹲**：身体保持直立向下蹲，膝关节不超过脚尖，均匀呼吸，臀部收紧。

完成箭步蹲后，我们进行下一个高强度的全身性动作。

以下是第二个训练动作的要点：

*   **波比跳**：跳起时均匀呼吸，向上击掌，避免塌腰，收紧臀部和腹部。

波比跳对心肺功能要求较高。接下来，我们进行一个针对腹部的训练动作。

以下是第三个训练动作的要点：

*   **仰卧自行车**：蹬车时扭转身体，使手肘与对侧膝盖触碰，扭转幅度要大。

腹部训练后，我们进行一个结合平衡与柔韧性的动作。

以下是第四个训练动作的要点：

*   **哑铃单腿出够**：手持哑铃或水瓶于身前，单腿站立，另一条腿抬起，身体前倾用手触摸物品，站直时臀部收紧。

接下来，我们进行一个针对背部的训练动作。

以下是第五个训练动作的要点：

*   **单臂俯身哑铃划船**：身体保持直立，腰腹收紧，肘关节贴近身体向上划船，背部收紧，避免耸肩。

背部训练后，我们进行一个针对肩部的动作。

以下是第六个训练动作的要点：

*   **哑铃侧平举**：身体稍前倾，膝盖微屈，手臂向两侧平举至与肩平行，保持节奏和呼吸。

肩部训练后，我们进行一个核心稳定性训练。

以下是第七个训练动作的要点：

*   **俯卧支撑交替触肩**：身体保持一条直线，交替用手触摸对侧肩膀，臀部摆动幅度尽量小。

核心训练后，我们进行最后一个针对侧腹肌的动作。

以下是第八个训练动作的要点：

*   **侧卧抬腿**：侧卧，身体呈一条直线，腿部、臀部、腰部收紧，向上抬腿，感受侧腹收缩。

完成第一轮所有动作后，我们将有3分钟的组间歇休息时间。休息后，请重复以上全部训练动作，共完成3轮。

## 拉伸运动

训练结束后，进行拉伸运动可以帮助缓解肌肉酸痛，促进恢复。以下是拉伸动作列表。

以下是拉伸动作的具体步骤：

1.  **肩部拉伸**：拉伸肩部肌肉，保持呼吸。
2.  **肱三头肌拉伸**：拉伸手臂后侧的肱三头肌。
3.  **小腿拉伸**：拉伸小腿后侧肌肉。
