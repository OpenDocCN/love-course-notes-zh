# 零售的哲学：02：Seven-Eleven的创立与经营之道

在本节课中，我们将学习日本7-Eleven便利店是如何从无到有创立，并凭借其独特的经营策略与心法，成为全球零售业典范的。我们将从它的创立背景、核心经营策略以及背后的成功心法三个层面进行剖析。

## 第一节：Seven-Eleven的创立之路

上一节我们概述了课程内容，本节中我们来看看7-Eleven是如何在日本诞生的。它的创立并非一帆风顺，而是源于创始人铃木敏文对行业现状的深刻洞察与打破常规的勇气。

创始人铃木敏文先生最初并非零售业出身。上世纪50年代，他进入出版行业，并在工作中掌握了统计学与心理学知识，这为其日后经营打下了基础。后来他转入伊藤洋华堂，尽管对流通领域是外行，但他决心认真对待新工作。

当时正值日本经济快速增长，大型超市兴起，严重冲击了中小型零售店。铃木敏文认为，经营的核心在于产品品质与服务，与店铺规模无关。他萌生了一个想法：创立一种现代化的中小型零售店模式，与大型超市实现共存共荣。

在美国考察时，铃木敏文偶然发现了7-Eleven便利店。这种小店商品种类齐全，给他留下了深刻印象，并决心将其引入日本。然而，这个想法在公司内部遭到了强烈反对，但他凭借全新的视角，分析了小型商店衰败的原因在于生产效率低下和未能适应市场变化，最终说服了公司。

1973年，经过精心准备，铃木团队开始与美国南方公司（7-Eleven品牌拥有者）谈判。过程异常艰难，美方提出了诸多严苛条件。铃木敏文采取的策略是**站在对方立场思考，谋求长远合作**，最终双方在特许经营区域、开店数量和权利金比例上达成妥协。

签约后，铃木团队赴美研修，却发现课程内容空洞，无法提供实际运营指导。他意识到，日本的7-Eleven必须从零开始自己摸索。基于“让小型商店与大型店共存共荣”的初心，他决定第一家店采用**加盟模式**，而非更利于积累经验的直营模式。

团队组建和首家店铺的寻找同样困难。铃木通过报纸招聘，聚集了一批没有经验但充满热情的年轻人。第一家加盟店是一位经营酒坊的店主，店铺面积小、位置一般，但店主热情高涨。铃木承诺，若三年后经营失败，将把店铺恢复原样归还。就这样，日本的7-Eleven在摸索中踏上了历史舞台。

**本节总结**：7-Eleven在日本创立于零售业停滞期，它从美国引入，却完全依靠本土团队从零开始摸索。其创立初衷是创新中小零售店模式，实现与大型超市的共存共荣。

## 第二节：Seven-Eleven的核心经营策略

上一节我们了解了7-Eleven艰难的创立过程，本节中我们来看看支撑其成功的三大核心经营策略：运营创新、产品导向与高效沟通。

7-Eleven的一切经营都围绕“为顾客提供便利”展开。其策略的过人之处可以从以下三个方面总结：

**首先，是运营方式的开拓性创新。** 当时日本零售业通行大批量采购与配送，但这不适合小型便利店，容易导致库存积压。铃木敏文力排众议，推动了**小额配送**体系。同时，为实现全年无休营业，他坚持要求供应商在正月（日本新年）也照常送货，这彻底改变了行业惯例。

在门店扩张策略上，7-Eleven没有选择快速全国铺开，而是坚持**区域密集开店**。这不仅能提高物流配送效率，更能形成强大的品牌曝光和广告效应，加深区域消费者的认知与消费习惯。这一策略后来成为便利店行业的黄金法则。

随着发展，铃木发现顾客将便利店视为“另一个冰箱”。他认为，仅售卖商品不足以体现“便利”，需增加生活服务。于是，他决定在店内引入ATM机，提供取现和代收公共服务费等功能。在银行合作受阻后，他做出了一个惊人决定：**自己开设银行**。

7-Eleven的ATM机功能专注结算、营业时间与门店同步，并且将成本控制到了行业平均水平的**四分之一**，三年即实现盈利。顾客在排队时浏览商品，取钱后顺便购物，无形中提升了销售额和顾客忠诚度。

**其次，是极致顾客导向的产品开发与选品。** 7-Eleven始终根据顾客需求变化进行创新。例如，发现老年人和职业女性需求上升后，便增加长保质期副食和半加工蔬菜。

在2005年日本经济低谷期，行业普遍滞销，铃木却认为这是消费者在追求新的产品价值。他果断推出高品质的**自有品牌（PB）产品**，大获成功，引领了行业从追求廉价到追求品质的转变。

7-Eleven首创了米饭、便当、关东煮等鲜食产品。其对产品品质要求极为严苛，奉行一条原则：**任何新产品上市前，若有一位高管认为口味未达卓越，就必须返工调整配方**。铃木坚信，只有令人感动的品质，才能赢得市场。

**最后，是注重自上而下的直接沟通。** 铃木敏文认为，经营理念必须准确无误地传达到每一位加盟店长和店员。为此，他不惜成本定期举办大型会议。

以下是7-Eleven内部重要的沟通会议：
*   **经理会议**：面向所有管理层。
*   **区域顾问会议**：针对指导加盟店的经营顾问，铃木会亲自出席并宣讲理念。

有人建议用IT技术节省会议开支，但铃木认为，信息在传递中会被无意识过滤，**自上而下的经营管理必须依靠直接沟通**。因此，坚持每两周召开一次区域顾问会议，这已成为7-Eleven的传统。

**本节总结**：7-Eleven的经营策略围绕便利性展开，通过运营创新（如小额配送、密集开店、引入ATM）、极致的产品开发与严苛的品质控制，以及坚持直接沟通的管理方式，构建了其强大的竞争壁垒。

## 第三节：Seven-Eleven的成功心法

上一节我们剖析了7-Eleven具体的经营策略，本节中我们将深入其背后，探讨那些可以借鉴的成功心法与思维模式。这些心法可总结为三点。

**第一，勇于主动创新的思维模式。** 7-Eleven的历史就是一部创新史：引入鲜食、全年无休、搭载POS系统、提供ATM、首创送货上门和网络购物、设置店内Wi-Fi和休息区等。每一次创新都引领行业之先。

铃木敏文不认同“经济不好，顾客不愿花钱”的论调。他认为，**产品滞销的根本原因，是公司的工作方法已无法满足时代和消费者需求的变化**。这种思维让他始终作为生意的主人去主动探索，而非随波逐流。

例如，通过调研和媒体报道，他预见到老龄化与少子化趋势，力排众议推出了**送餐服务**。即便喜欢自取的顾客，知道有送餐选择也会感到便利性大增。在送货方式上，他选择由**店员而非第三方配送**，虽然增加了人力成本，但店员送货带来的亲切感和安全感，极大地提升了顾客体验。

这种主动创新思维也用于改造美国7-Eleven。当时美国门店由总部统一配货，导致商品滞销、体验差。铃木改革的核心是：**要求每家店必须自主订货**，哪怕实习生也要参与。这极大地激发了员工的主动性和创新热情。

主动创新常意味着“逆流而上”。在通货紧缩时期，7-Eleven推出售价**100日元**的现磨咖啡，远高于市场均价。这一定位“家中难以实现的味道”的创新，成功吸引了女性消费者，改变了即饮咖啡的市场格局。

**第二，创造“信息钩子”。** 铃木的许多决策看似独断，实则源于深刻的洞察，这得益于他独特的“信息处理”心法。

他习惯于让自己**随时置身于信息流中**（如看电视、听广播、浏览杂志），目的并非获取特定结论，而是保持与世界的连接，感知流行趋势和社会关切。

更重要的是，他会在脑中设置“**钩子**”。这些“钩子”就是他日常经营中时刻思考的**核心问题**。当外界信息流过时，相关的内容会被这些“钩子”自动捕捉，从而激发新的思考与决策。铃木认为，**比起收集信息，如何运用信息才是关键**。

**第三，刻意保持经营的“紧张感”。** 为确保最高效率和精益求精，铃木有意在合作中避免形成“好朋友”式的松懈关系。

例如，在与供应商联合开发产品时，他特别要求双方**不要因共同奋战而变得过于亲密**。因为一旦产生“同盟意识”，面对未达标的产品就容易妥协，认为“大家都不容易”。这种松懈会直接导致产品在市场上被淘汰。

在开发红豆糯米饭团时，因工厂设备限制，糯米是用煮的而非蒸的，导致口感不佳。研发人员希望体谅，但铃木坚持**立即下架该产品**，直到找到合适工艺和设备后才重新上市。他的逻辑是：顾客不会为“设备不行”的理由买单，他们只关心产品是否达到期望的价值感。

**本节总结**：7-Eleven的成功，根植于其主动创新的文化、创始人善于从信息中提炼洞察的能力，以及在经营中刻意保持紧张感、杜绝妥协的严苛准则。这些心法共同确保了企业持续的竞争力与生命力。
