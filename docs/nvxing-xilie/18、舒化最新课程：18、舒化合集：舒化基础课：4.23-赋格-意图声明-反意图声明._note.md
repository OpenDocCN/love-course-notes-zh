# 舒化基础课：4.23：赋格、意图声明与反意图声明 🧠

在本节课中，我们将要学习三种核心的聊天技巧：**赋格**、**意图声明**与**反意图声明**。这些技巧能帮助你更好地理解对方的意图，并在互动中建立有利的框架，从而提升你的沟通能力和吸引力。

---

## 一、什么是赋格？ 🎭

上一节我们介绍了课程概述，本节中我们来看看第一个核心技巧——赋格。

赋格是一种沟通技巧，通常由男性对女性使用。从字面上理解，**赋格**就是“赋予对方一种人格”。具体来说，它指的是在对方尚未表现出某种特质时，你提前将这种特质“赋予”给对方，就像给对方戴上一顶“高帽子”。

**公式**：`赋格 = 提前赋予对方某种积极的人格特质`

例如，你并不完全了解对方，但你可以说：“我发现你真是个善良又靠谱的人。” 这就是在对方展示这些特质之前，你先进行了“赋格”。

以下是赋格的两个关键特点：
1.  它是一种预先设定的积极评价。
2.  它能为后续互动设定一个积极的框架。

## 二、赋格的实战案例 💡

理解了赋格的基本概念后，我们通过一个具体案例来看看它是如何应用的。

在一个案例中，男生对女生说“以后你的口红我全包了”。女生随后提出：“口头承诺不可信，除非你现在付个小定金。” 当男生发来一个金额较小的红包（如5.20元）后，女生回复：“你的诚意我感受到了，你果然是个说到做到的好男人。”

**代码描述**：
```python
# 赋格应用示例
if 对方做出承诺：
    提出服从性测试（如要小额红包）
    if 对方通过测试：
        给予赋格：“你果然是个[对方承诺相关的优点]的人。”
```

这句话就是一个典型的赋格。它提前将“说到做到的好男人”这个人设赋予了对方，无论红包大小，都强化了“他兑现了承诺”的框架。

## 三、意图声明与反意图声明 🔄

学会了赋格，我们再来看看一组组合技巧：意图声明与反意图声明。它们像“推拉”一样，需要配合使用。

**意图声明**：明确告诉对方，你喜欢他/她的某个**具体行为或特质**，而不是泛泛的“我喜欢你”。
*   **公式**：`我喜欢你[某个具体行为/样子]。`
*   **示例**：“我喜欢你给我付钱时大方的样子。” 或 “我喜欢你带我打游戏时专注的样子。”

**反意图声明**：在对方没有做到你曾声明的“喜欢点”，或做了让你不开心的事时，收回之前的喜欢声明。
*   **公式**：`你居然[做了我不喜欢的事]，我不喜欢你了。`
*   **示例**：（接上例）如果对方后来拒绝付钱或对你发脾气，可以说：“你居然凶我，我不喜欢你了。”

以下是两者的使用逻辑：
1.  **先播种**：通过意图声明，在对方心里种下“我怎么做会让她喜欢”的种子。
2.  **后收割**：通过反意图声明，在对方行为不符时进行“惩罚”，引导对方调整行为。

两者结合，可以有效地“调教”对方，让他明白如何做能获得你的好感，如何做会让你疏远。

## 四、核心心态：将男人视为狗来调教 🐕

掌握了具体技巧，我们还需要建立正确的心态。一个形象的比喻是：**将男人当成狗来调教**。

这个比喻的核心是“奖惩机制”：
*   **奖励**：当对方做出你喜欢的、符合你“意图声明”的行为时，给予正面反馈（夸赞、赋格）。
*   **惩罚**：当对方做出你不喜欢的行为时，要像训练小狗一样，明确给出负面信号（冷淡、反意图声明），并且要**忍住**不去主动和好，给对方时间“摇着尾巴过来哄你”。

**关键点**：很多女生失败在于，对方的“尾巴”还没摇起来，自己就先忍不住去讨好对方了。必须保持耐心，让奖惩机制生效。

## 五、实用话术与框架回收 🛡️

最后，我们学习一个应对特定场景的实用话术，以及如何回收框架。

**场景：对方临时邀约**
即使你非常想见对方，也要使用推拉技巧，树立自己的框架。
*   **推**：“你下次必须提前预约我哦！”（树立标准，制造轻微挫败感）
*   **拉**：“这样我才能好好准备，恭候大驾呀！”（给予尊重和期待）
这句话能在对方情绪上制造起伏，既显得你价值高，又让对方感觉被重视。

**场景：对方发小红包并质疑你**
当对方发了小额红包（如5.2元），并说“刚认识就发红包，这样的男人你敢要吗？”来甩框架时，可以这样回收框架：
1.  **加倍返还红包**：例如，回一个66.66元的红包。
2.  **附言**：“我也是这么想的。”
这个操作实现了多重打击：展示了你的大气和不贪图小利；用“我也是这么想的”赞同他的观点，反而让他自相矛盾；利用人性的好奇与贪婪，让他陷入“领与不领”的纠结，从而夺回对话主导权。
