# 女神养成系列：09：日杂赤茶妆教程 🍂

在本节课中，我们将学习如何打造一款经典的日系妆容——日杂赤茶妆。这个妆容的核心在于追求色调的统一与整体的干净感，营造出邻家、温柔且精致的氛围。我们将从认识妆容特点开始，逐步进行实操演示，帮助你掌握色彩统一的技巧。

## 第一部分：赤茶妆的认识

上一节我们学习了韩剧女主妆，本节中我们来看看日系妆容的特点。日杂赤茶妆源自日本杂志风格，其精髓在于**妆容色彩的极致统一**。整体色调以红棕色（赤茶色）为主，眼影、眉毛和口红的颜色相互呼应。

以下是日系妆容的几个关键特点：

*   **干净通透**：底妆追求白皙、无瑕的肌肤质感，修图也偏向提高亮度。
*   **重点突出**：妆容重点通常放在眼影和口红上，腮红相对较弱。
*   **自然亲和**：发型和妆面设计都偏向可爱、邻家风格，没有强烈的距离感。
*   **弱化线条**：眼线细且自然，睫毛追求根根分明而非夸张卷翘，眉毛边界柔和。

让我们欣赏几种不同风格的赤茶妆表现：

1.  **气质赤茶色**：采用全包式下眼影，搭配棕色美瞳，极致强调红棕色调的统一。
2.  **干净杂志风**：妆感非常淡，重点修饰眉毛和口红，突出模特本身的自然美感。
3.  **传统与创新**：传统画法注重整体晕染；创新画法则可能运用“脏脏唇”画法、单簇假睫毛等，在统一色调中增添时尚细节。

**核心概念**：日杂赤茶妆的公式可以概括为 **统一色调（红棕色） + 干净底妆 + 自然线条**。

## 第二部分：妆容实操演示

认识完妆容特点后，本节我们将通过实操来一步步还原这个妆容。本次模特眼部条件为内双、略有肿眼泡，更贴近多数初学者的实际情况，方便大家参考。

### 第一步：底妆与遮瑕

首先进行基础打底。模特已涂抹隔离，我们直接使用遮瑕产品处理痘痘、痘印等瑕疵。注意，遮瑕可以在粉底前或粉底后进行，依个人习惯即可。

```
操作要点：将遮瑕产品点涂在瑕疵部位，并用指腹或工具轻轻拍开。
```

接着使用粉底液，以点涂方式均匀铺满全脸，打造白皙干净的底妆。

### 第二步：眼影刻画

眼影是本次妆容的重点。我们使用红棕色系眼影盘来塑造层次。

以下是眼影上色步骤：

1.  **打底**：使用大号眼影刷，选取接近肤色的浅色眼影，在整个眼窝进行大面积打底。
2.  **铺色**：换用中号刷子，选取偏橘的红棕色，从睫毛根部开始向上晕染，作为主色调。同时，将余粉轻轻扫在下眼睑的后2/3处。
3.  **加深**：换用小号细节刷，选取更深的红棕色，重点加深睫毛根部以及眼尾三角区，并向前自然晕染。
4.  **晕染**：每一步上色后，都需用干净的刷子将色块边缘晕染开，确保过渡自然。最后可选用极细的深色珠光眼影，少量点缀在眼皮中央，提升质感（初学者可省略此步）。

```
技巧提示：单眼皮或肿眼泡，眼影范围不宜过大，重点强调睫毛根部和下眼影。
```

### 第三步：眉毛描绘

为了保持色调统一，眉笔应选择**棕色系**，避免使用黑色或灰色。

以下是画眉步骤：

1.  **梳理**：用眉刷将眉毛梳理整齐。
2.  **定型**：先用眉笔清晰地勾勒出眉毛的**下边缘线**，这决定了眉毛的基本形状和走向。
3.  **填充**：顺着眉毛生长方向，填充眉毛中间空缺的部分。
4.  **晕染**：使用眉刷，将勾勒的线条和填充的颜色向上、向四周晕染开，特别是眉头部分要晕染自然。
5.  **调整**：用遮瑕笔修饰眉毛边缘，使眉形更加干净精致。

```
核心概念：眉毛的精致感来自于“晕染”。通常只有下边缘线是画实的，上缘和眉峰都是靠晕染出来的。
```

### 第四步：鼻影与修容

选择灰棕色调的修容产品，打造自然立体的轮廓。

1.  **鼻影**：从眼窝凹陷处开始，向下轻扫鼻梁两侧。注意不要一笔到底，鼻梁中部可稍淡，鼻头两侧稍作修饰，最后轻扫人中与下唇下方。
2.  **面部修容**：轻扫发际线、颧骨下方以及下颌线，让脸型更显小巧。

```
操作要点：修容务必少量多次，追求自然晕染的阴影效果，而非明显的色块。
```

### 第五步：眼线与腮红

1.  **眼线**：日系妆眼线追求极致自然。仅需用眼线笔填充内眼线，并在眼尾轻轻拉出一条细而短的外眼线即可。可轻微加深下眼睑后1/3处，与眼影衔接。
2.  **腮红**：选择偏橘调的浅色腮红，用点拍的方式轻扫在苹果肌上，稍微向修容区域过渡，营造好气色。

### 第六步：定妆与唇妆

1.  **定妆**：用散粉刷蘸取少量散粉，轻扫全脸进行定妆，使妆面更持久、干净。
2.  **唇妆**：选择红棕色系（如砖红色）口红。首先将颜色涂在嘴唇内侧，然后抿嘴，让颜色自然扩散。再叠加1-2次，增加饱和度。最后用唇刷或棉签精细勾勒唇形，使唇妆饱满且精致。

```
技巧提示：口红是妆容的灵魂。通过多次叠加和精细修饰，可以极大提升妆容的完整度和高级感。
```
