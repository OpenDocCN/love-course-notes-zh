# 尊巴燃脂舞：01：初级全身挑战：冲击心率，燃烧全身能量 💃

在本节课中，我们将学习一套初级的尊巴燃脂舞。这套动作旨在通过全身性的舞蹈组合，有效提升心率，促进能量消耗。课程将分解基础舞步，并指导你如何协调手臂与腿部动作，以达到最佳的燃脂效果。

## 课程概述

本节课程是一套连贯的尊巴舞蹈序列，包含基础步伐、手臂摆动和身体转向。跟随音乐节奏，我们将完成一系列旨在提高心率、活动全身主要肌群的动作组合。

## 动作分解与指导

以下是本课程的核心动作序列。每个动作都旨在协调你的全身，并逐步提升运动强度。

**准备姿势**：双脚自然分开，与肩同宽，膝盖微屈，保持核心收紧。

1.  **热身与启动**
    *   首先，跟随音乐节奏原地踏步，让身体热起来。
    *   将手臂向身体两侧平举，然后向前伸展，重复数次。

2.  **基础步伐与手臂组合**
    *   向右侧迈两步，同时将双手举过头顶。
    *   接着向左侧迈两步，双手慢慢放下。
    *   重复此左右移动的序列，动作由慢到快。

3.  **加入转身与跳跃**
    *   在原地进行小跳步，同时双手在胸前交替摆动。
    *   加入90度转身跳，跳转时保持手臂动作有力。
    *   公式：`小跳步 + 转身 = 提升心率`

4.  **侧向移动与核心收紧**
    *   向侧面滑步，同时将对侧手臂向上伸展。
    *   感受侧腹肌群的收缩。
    *   代码式描述：`while 滑步: 保持核心稳定 && 对侧手臂上举`

5.  **高潮部分：全身冲击**
    *   组合前述所有动作：踏步、侧移、转身、跳跃。
    *   加快动作频率和幅度，让心率达到高峰。
    *   关键：保持笑容，调整呼吸。

6.  **缓和与整理**
    *   动作速度逐渐放慢，恢复基础踏步。
    *   进行深呼吸，双臂缓慢画圈，帮助身体放松。

## 练习要点总结

本节课我们一起学习了初级尊巴燃脂舞的全套动作。核心在于通过 `基础舞步 + 手臂动作 + 方向变化` 的组合，持续活动身体，冲击心率区间，从而高效燃烧能量。请记住，全程保持核心收紧、呼吸顺畅，并享受音乐与运动的快乐。
