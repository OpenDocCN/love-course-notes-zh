# Chic原醉：02：倾听的艺术

在本节课中，我们将要学习如何成为一个优秀的倾听者。很多人知道如何说话，但大多数人却不知道如何正确地倾听。这常常导致沟通障碍。正确的倾听技巧本身也是一种效果显著的心理暗示术。我们将通过三个核心部分来掌握这门艺术。

## 倾听的意识

上一节我们介绍了倾听的重要性，本节中我们来看看倾听的第一个要点：倾听的意识。

首先你要知道，倾听对方讲话其实是一种报酬。很多同学可能时常觉得自己不会聊天，但有时，即使不用很会说，你依然可以获得对方的认可与好感。这个方法就是学会倾听对方的倾诉。

就拿我来说，当一个患者找到我的时候，如何才能让对方快速地对我产生信任感，并且对我的引导进行配合呢？最好的方式就是静静地听他诉说。因为你越是认真地倾听一个人的谈话，对方就越有和你交流的欲望。

很多沟通出现问题、建立不下去的主要原因，就是听者太急于倾诉自己的想法。这打乱了对方的发言逻辑，也让对方有种得不到尊重的感觉。沟通的顺序应该是先倾听再去表达。也就是当对方开始和你说话时，你要听他把他想说的内容全部倾诉完成之后，再从他刚刚陈述的内容中选择重点进行表达。这样的沟通才会有一个很好的持续互动过程。

所以说我们要形成倾听的意识，也就是说要克制自己迫切表达的情绪，了解了对方想表达的内容再进行回馈。

男女沟通的一个重大误区，就是男生间的沟通趋向理性化，会比较注重倾听理解。而女生聊天的时候，基本是各持己见，大家都在表达自己的想法。所以有时男生会觉得女生聊天的时候很吵闹、很不能理解，主要就是因为这个。

在倾听的时候，有些细节是一定要注意的。以下是需要注意的几点：

*   在对方和你说话的时候，一定要时常点点头。
*   尽量盯着对方的眼睛，不要转移视线。
*   如果是第一次认识的话，你可以每隔半分钟调整一下视线，比方说缓缓闭眼，或者向对方的手臂扫一眼，然后迅速把目光转移回对方的眼睛。

这是因为第一次见面的时候，如果一直盯着对方会给他无形的压力，所以要做一定的调整，给对方一个舒适的介入过程。

## 学会附和

了解了倾听的意识后，我们来看看如何通过积极的反馈来深化倾听效果，这就是倾听的第二要点：学会附和。

虽说倾听很重要，但是也不能只是目光呆滞、傻傻地听着。因为这样会让对方觉得你心不在焉，也会降低对方沟通下去的欲望。所以在倾听的过程中，我们要学会配合附和对方，让他们知道我们在听，并且在理解。只有这样的沟通状态才能让诉说者有很大的被认可度与继续交流的欲望。

这就像是开车一样，你的附和就是汽油，只有不断地附和，才能有让对方前进沟通下去的动力。

那怎么正确引导对方说更多的话呢？我们要从倾听者转向善于询问。很多人附和诉说者的方式，大多都是“嗯”、“啊”、“是”，这种附和的效果实在是太过单薄，你根本没有办法用这种附和方式激起对方心中的波澜。

附和分为两种，一种被称之为**随口性附和**。其表达方式也就是我刚才说的那种附和类型，你会给对方一个你对于这个话题兴致不高的暗示。你可以仔细回想一下，当有一个人和你巴拉巴拉说着你不感兴趣的事情，你又不想伤了对方面子的时候，都会选择这种心不在焉的附和方式，那聪明的诉说者当然能够感受得到，于是就会尽快地结束话题。

但是如果你想要引导对方说更多的话，更喜欢与你交流，就一定不能用随口性附和，而要用**肯定性附和**。肯定性的附和会有更多的语言表达方式，它的内容都是向对方展现出“我对你的话题很感兴趣”、“我很想知道你后面要说什么，请你继续说下去”的暗示性语言。

比方说在和对方说话的时候，你可以尝试用这些语言去附和对方：“真的吗？”、“然后呢？”、“厉害！”、“的确如此”、“原来还可以这样”。你可以对比一下两种附和的方式，后者明显要比前者积极得多，会让交流进入一个更加积极热烈的状态。

这个方式，如果你要运用好的话，你甚至可以让那些或许对你聊天欲望不是很高的人都爱上和你聊天。

在这里教大家一个我经常运用的讲话方式，用来激起那些不爱说话或者对我兴致没有那么高的人和我沟通的欲望。以下是具体步骤：

1.  **提示话题**：在开始的时候，我会进行第一步提示话题。那这个话题可以是他的工作，也可以是他生活环境、兴趣爱好等等。比方说我前几天遇到的一个台湾女孩，在我和他开始聊天的时候，我先问他：“说起来台湾那边生活环境很舒服吧。”
2.  **引导对方的话**：当我说这句话的时候，其实就进入了第二步，引导对方的话。他会和我讲：“对啊，台湾生活还是蛮舒服的，就是收入呢不是很高。”
3.  **肯定性附和**：当你引导出对方的话的时候，就可以进入肯定性附和阶段了。于是我接着说：“真的吗？”他果然兴致高涨地对我讲：“对啊，台湾普遍收入都不高的，所以我才来大陆这边工作。”
4.  **催促继续讲述**：那下面呢我就会用肯定性附和，催促对方继续讲述。我会问他：“为什么？”然后他开始跟我进行解答。

就这样呢，对方无意识地开始跟我进入了一个愉悦的聊天状态。然而，我却又不会暴露很多的需求感，因为我只是在进行引导，但是更多的呢我是扮演着一个倾听者的角色。这就是肯定性附和的重要性。如果你能善于使用，我相信你无论和任何人沟通，都会处于一个被极度认可和信赖的状态。

## 倾听的回馈

学会了倾听和附和，沟通的闭环还需要最后一步。上一节我们探讨了如何积极反馈，本节中我们来看看倾听的第三部分：回馈。

我之前说过，讲话是你来我往的事情，只是倾听与附和是不够的，你要适当地给对方回馈。那回馈的重要性在于两点：第一点是你让对方知道你理解了他说的内容；第二点是在你理解的内容中总结自己的观点，这样呢才可以形成交流。

关于回馈，我可以给大家一个法则：**LOVE法则**。它主要分为四个部分：

1.  **L (Listen)**：倾听对方的讲述。也就是我们第一点所说的倾听。
2.  **O (Observe)**：对对方进行观察，观察他对自己行为产生的结果，是不是变得更加积极。
3.  **V (Value)**：认可对方的价值，也就是肯定性附和。
4.  **E (Evaluate)**：根据对方的内容进行整理，并且评价，这就是所谓的回馈。

就比方说和那个台湾的女孩讲述完成后，我会进行总结：“所以说台湾真的是一个很适合生活的地方，只不过如果要去那边生活的话，对生存能力还是一个很大考验的。”这一句话就是最后的回馈。对方根据这个回馈，还可以进行继续的回复。

到这里基本就形成了整个倾听的流程。

## 总结

本节课中我们一起学习了如何成为一个完美的倾听者。其实这节课主要的目的是为了让大家学会去做一个倾听者。最近发现很多女生让我帮他们分析聊天的时候，聊天中大多都扮演着一个诉说者的身份，而且是那种自说自话的状态。

记住：**你说的越多，暴露的越多**。然而你学会倾听，不但可以让你不会暴露太多需求感，还会让你更加了解对方，增加对方对你的信任程度。
