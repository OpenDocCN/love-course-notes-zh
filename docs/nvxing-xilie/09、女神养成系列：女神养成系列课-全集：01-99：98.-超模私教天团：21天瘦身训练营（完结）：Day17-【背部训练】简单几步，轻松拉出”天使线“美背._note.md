# 女神养成系列：01-99：98. 超模私教天团：21天瘦身训练营（完结）：Day17 【背部训练】简单几步，轻松拉出“天使线”美背 💪

在本节课中，我们将要学习一套完整的背部及全身塑形训练流程。课程内容包括热身、三个大组的核心训练动作以及训练后的拉伸放松。目标是帮助你塑造漂亮的肩颈线条、平坦的小腹，并缓解之前训练产生的肌肉酸痛。

## 课程概述与原理

本节课的训练任务非常明确，旨在练出美背、漂亮的肩颈和平坦的小腹。同时，训练也会帮助身体排掉一些上次臀腿训练产生的乳酸，减轻身体负担。

浑身酸痛是因为身体产生了乳酸。乳酸产生后，需要通过继续训练来帮助身体排酸。越不动，疼痛感可能会越强。因此，我们直接开始训练。

## 第一步：热身激活 🏃‍♀️

训练开始前，必须进行充分的热身，以激活目标肌群并保持最佳身体状态，预防受伤。

以下是热身动作，每个动作完成15次。

**1. 掌心向上直臂后震**
*   掌心向上，双臂伸直向后震动。
*   过程中，肩胛骨要始终收紧。

**2. 屈肘后震**
*   保持掌心向上，屈肘，进行向后震动。

**3. 微型上举后震**
*   完成微型上举动作，配合后震。

**4. 掌心向前肘下压**
*   掌心向前，肘关节向下压。

热身完成，准备开始正式训练。

## 第二步：第一大组训练（循环3次）🔄

上一节我们完成了热身，本节中我们来看看第一个训练大组。本大组包含三个动作，旨在强化背部与核心。

以下是第一个大组的三个训练动作，需按顺序完成，三个动作为一小循环。完成一个小循环后休息90秒，总共重复三个小循环。

**1. 俯身哑铃划船**
*   **动作描述**：俯身，哑铃起始位置在髋关节两侧（而非肋骨两侧）。背部收紧，肩胛骨收紧。沿大腿正面向髋关节方向滑行提起哑铃。
*   **练习要求**：完成15次。
*   **要点**：呼吸均匀，深吸深吐。背部起伏要小，集中感受背部发力，避免借力。

**2. 徒手深蹲**
*   **动作描述**：先屈髋，保持背部平行并收紧肩胛骨，再屈膝下蹲。蹲至髋关节低于膝关节即可。
*   **练习要求**：完成15次。
*   **要点**：身体向下时同步屈膝。股四头肌应有发力感。

**3. 平板支撑**
*   **动作描述**：身体呈平板支撑姿势。
*   **练习要求**：保持1分钟。
*   **要点**：不要塌腰或弓背，重心不要过度前移。核心收紧，稳定腰椎，保持均匀呼吸。

第一个大组的全部训练动作都已完成。现在休息2分钟，准备开始第二大组的训练。

## 第三步：第二大组训练（循环3次）🔥

在完成了以背部和下肢为主的训练后，本节我们将重点转移到肩部和全身性的燃脂动作上。

以下是第二个大组的训练动作，同样以三个动作为一小循环。完成一个小循环后休息90秒，总共重复三个小循环。

**1. 哑铃提拉**
*   **动作描述**：站姿，双手握哑铃。发力点应集中在肩部三角肌，而非肘关节。避免含胸、驼背或身体后仰。肘部放松，注意力放在肘关节的移动轨迹上。
*   **练习要求**：完成15次。
*   **呼吸与发力**：**呼气时三角肌发力上拉，吸气时下放**。初始可能感觉动作别扭，但坚持标准动作会有进步。

**2. 站姿哑铃推举**
*   **动作描述**：站姿，双手握哑铃于肩侧。肘关节不要内扣，避免含胸和仰身。上推时呈A字形轨迹向上，哑铃不要相碰；下放时按原轨迹还原，肘关节不要外撇。
*   **练习要求**：完成15次。
*   **呼吸与要点**：**呼气上推，吸气下放**。在最顶端时肘关节伸直，但哑铃不要相碰。错误姿势会增加受伤风险。

**3. 波比跳**
*   **动作描述**：完成一次标准的波比跳。
*   **练习要求**：完成15次。
*   **要点**：蹬腿时不要塌腰，不要耸肩或低头。背部收紧挺直。保持均匀呼吸和稳定节奏，双脚离地跳跃。

第二大组的全部训练动作都已完成。现在休息2分钟，开始最后一个大组的训练。

## 第四步：第三大组训练（循环3次）✨

经过前两个大组的训练，我们的背部、肩部和心肺都得到了锻炼。最后这个大组将针对臀部和腹部进行塑形，让线条更加完美。

以下是第三个大组的训练动作，继续以三个动作为一小循环。完成一个小循环后休息90秒，总共重复三个小循环。

**1. 哑铃单腿出够（罗马尼亚硬拉变式）**
*   **动作描述**：单腿站立，手持哑铃。俯身时不要弓背，目的不是触碰哑铃，而是让臀大肌产生收缩感。
*   **练习要求**：单腿完成15次后换腿。
*   **要点**：集中感受臀大肌发力，这个动作只练习臀部。**吸气准备，吐气发力**。

**2. 俯身臂屈伸（俯撑）**
*   **动作描述**：俯身，肘关节收紧，与身体保持约75度夹角，腰背挺直。
*   **练习要求**：完成15次。
*   **要点**：手臂不要抬得过高，集中感受肱三头肌发力。保持均匀呼吸。

**3. 卷腹**
*   **动作描述**：仰卧，进行卷腹动作。
*   **练习要求**：完成15次。
*   **要点**：肩颈放松，不要低头或靠颈部发力。眼睛向上看，手臂自然放置。**呼气时腹部发力卷起**，上腹部应有明显感觉。

今天所有的训练动作都已经完成。接下来我们进行拉伸放松。

## 第五步：拉伸与放松 🧘‍♀️

训练后的拉伸至关重要，它能帮助肌肉恢复，缓解紧张，提升柔韧性。

以下是全套拉伸动作，请按顺序完成。

**1. 站姿体前屈（分腿）**
*   双腿分开与肩同宽，俯身向下呼气，静止保持15秒，还原。

**2. 站姿体前屈（并腿）**
*   双腿并拢，俯身向下呼气，保持15秒，还原。

**3. 跪姿股四头肌拉伸**
*   跪在垫子上，双腿保持一定距离，手向后撑地。撑臂抬头，将腿向前伸出，均匀呼吸，拉伸股四头肌。然后向后坐，拉伸臀大肌。

**4. 婴儿式放松与阔背肌拉伸**
*   坐在脚踝上，俯身放松15秒。然后原地做旋转，拉伸背阔肌，保持15秒后还原，换另一侧。

**5. 屈肘后拉伸**
*   做一个热身时的屈肘、掌心向上动作，但幅度要比热身时更大，保持拉伸。

拉伸完成。
