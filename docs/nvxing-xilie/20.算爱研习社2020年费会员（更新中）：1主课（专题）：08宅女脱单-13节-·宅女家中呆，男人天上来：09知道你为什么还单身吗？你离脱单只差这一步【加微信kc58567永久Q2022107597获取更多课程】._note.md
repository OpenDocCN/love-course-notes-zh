## 概述
在本节课中，我们将要学习如何界定“普通人”在择偶时的合理要求范围。我们将通过定义“可预期社交圈”和“要求等级”，帮助你认清自身定位，从而制定出切实可行的脱单策略。

上一节我们定义了“才女组”，即拥有突出才华的人群。本节中，我们来看看“普通人”该如何定义。

## 普通人的定义
普通人很好定义：没有突出的颜值，没有特殊的才艺，并且没有特别高的要求。

普通人的特点不在于你没有什么，而在于你愿意为你所没有的东西而放弃什么。因为你什么都没有，所以你选择放弃那些不切实际的条件，这才叫普通人。如果你什么都没有，却还有诸多要求，那你不是普通人，而是“追梦人”。

追梦人想要脱单就不那么容易了。那么，普通人为什么容易脱单呢？因为普通人懂得放弃。作为一个什么都没有的人，你脱单的重点在于明确你的要求是什么。

## 关注你的要求
为什么说谈恋爱是一种社会化的行为？因为这不是单靠你一个人就能完成的事情。在社会化行为的逻辑里，“我要什么”并不重要，重要的是“我可以要什么”。

很多姑娘说“我要谈恋爱，我要这个，我要那个”。这就像是一个大学毕业生跑去公司说：“我要5万月薪，我要当主管。”这很可笑，对吗？关键在于“你可以要什么”。当然，你可以通过努力提升颜值、增加才艺，从而“可以要”更多。但如果你什么都没有，就必须对“现在的你能要到什么”有一个明确的评估。

所以，普通人并不好当。普通人要学会放弃。学不会放弃，就不是普通人，而是前面提到的“追梦人”或“做梦的人”。

## 界定“要求不高”
这就带来一个问题：很多姑娘说“我没有才华，也没有颜值，所以我要求一点都不高，一点都不特别”。这只是个人的认知，而认知很可能存在偏差。

我们需要对“要求高不高”、“要求特别不特别”有一个明确的界定。最广为人知的说法是“男的就行”。但当我们深入追问时，比如“瞎一只眼的行吗？”、“身高1.4米行吗？”、“欠债2000万行吗？”，很多人就会不断添加新的条件。

其实每个人的要求都很多，只是从未仔细梳理。最后只归结为“活的、男的就行”，但这并不能解决问题。所以，我们需要科学地界定普通人的“要求不高、不特别”。

## 如何科学界定要求？
我们不以身高、体重、年龄、收入等具体标准来界定，因为这些标准因人而异。不同圈子的人对“有钱”、“优秀”的定义天差地别。

科学的界定方法是：以你的 **可预期社交圈** 来定。

**可预期社交圈** 包括以下三类人的集合：
1.  你已经认识并聊过天、见过面的人。
2.  你们互相有微信联系方式，但平时不怎么聊天的人。
3.  你通过一定努力（例如通过共同朋友介绍）就能够加到微信的人。

简单理解，就是“你已经认识”和“你很快就能认识”的人。

## 要求的四个等级
在你的可预期社交圈内，我们可以将你对男朋友的要求划分为四个等级，用游戏化的方式表述为：**正常、高级、稀有、传奇**。

以下是每个等级的具体定义：

*   **正常要求**：在你的可预期社交圈内，有超过5个**单身**男性完全符合你的要求。
    *   *示例*：要求“211/985毕业，月入1万+，身高1.7米以上，在北京/上海”。翻看朋友圈，发现符合条件且单身的男性有七八个。

*   **高级要求**：在你的可预期社交圈内，只有1到3个单身男性符合你的要求。
    *   *示例*：要求“金融公司副总裁”。翻遍朋友圈，发现只有一家公司的副总裁最近刚离婚，符合条件。

*   **稀有要求**：在你的可预期社交圈内，符合你要求的单身男性一个都没有，或者只有一个。
    *   *示例*：要求“身高185cm，长相如小鲜肉，才艺出众，未来必须疼我入骨”。这种对未来行为的期望，很难在现有社交圈中找到对应人选。

*   **传奇要求**：符合你条件的男性，全世界可能都不超过100个。
    *   *示例*：要求“长得像蔡徐坤/彭于晏/吴秀波”。这属于传奇级别。

## 给普通人的建议
**普通人应该将择偶要求限定在“正常”范畴内。**

如果你是颜值组或才艺组，可以奢望一下“高级”要求。至于“稀有”要求，请先审视自己是否也属于同等稀有的类型。

穿越社会阶层是一件非常困难的事情。颜值、金钱、才华、或者为他人做出巨大奉献的品德，都可以成为穿越阶层的资本。但如果你什么都没有，却什么都想要，那就是“拧巴”，是认不清自己的位置。

**核心逻辑**：
```
如果 (自认无颜值、无才艺，且不打算改变) {
    那么 请将要求限定在“正常”范畴；
} 否则 {
    你可以通过升级自我（如提升颜值、培养才艺）来获得选择更高要求对象的资格；
}
```

所以，请把你的要求限定在正常范畴：即在你的可预期社交圈里，有超过5个单身男性完全符合你做男朋友的标准。

## 总结
本节课中，我们一起学习了如何科学地界定自己的择偶要求。我们引入了“可预期社交圈”的概念，并将要求划分为“正常、高级、稀有、传奇”四个等级。作为普通人，认清自身定位，将要求设定在“正常”范畴，是迈向脱单最切实的一步。如果你想提高择偶标准，唯一的方法是先投资自己，升级自我。
