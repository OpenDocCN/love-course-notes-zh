# 职场产品思维课：02：用户思维：你为谁代言？（上）

## 概述

在本节课中，我们将要学习产品思维的第二个核心要素：**用户思维**。我们将探讨什么是用户思维，它为何重要，以及构成用户思维的三个关键元素。理解并运用用户思维，能帮助我们在职场中更有效地解决问题、提升工作成果。

---

上一节我们介绍了产品思维的第一个要素“去情绪化”。本节中，我们来看看产品思维的第二个支柱：**用户思维**。这部分内容较多，我们将分为上下两节课来分享。

所谓的用户思维，就是站在用户的立场上去打磨和完善产品的思维方式。

虽然每种产品的设计都是为了满足某种需求，但用户思维的渗透程度，会极大地影响产品对需求的满足程度。

举个例子，三星级酒店和五星级酒店带给人的感受完全不同。前者提供衣食住行的基本设施，而后者则会从房间布置、浴巾摆放、空调温度、服务质量等细节，让人全方位感受到被悉心照料。

这种让人感受到被优待的精致体验，是产品设计者从用户视角出发的最直观体现。它能直接影响用户对产品或服务的价值认定和品质判断。

就像我们不能指望50元的电风扇让房间冬暖夏凉，却可以对5000元的空调提出恒温、静音等个性化要求。

而人的有趣之处在于，一旦体验过更顺手的产品，就对其他产品提不起兴趣。所谓的使用习惯就是这么来的。

所以我们发现，在互联网领域，即使是同样的需求催生的同质化竞争，不管先来后到，总是那些更贴合用户需求和痛点的产品，能以最快速度引起市场注意，培养用户习惯，从而跑赢对手，占领更大市场份额。

甚至在一个非常传统的赛道，只要切准市场，把用户体验做到极致，也有机会后来居上。

就像传统白酒行业中，近年来出现的品牌**江小白**。它从众多老牌品牌中杀出一条血路，是一个非常典型的互联网品牌。它很好地贴合了90后年轻人群的需求和特点：独立自主。小包装设计迎合了这一点。此外，瓶身设计有趣，每个包装上的文案都能代表年轻人某个时刻的心声，让人有拍照分享的冲动。这客观上为产品和品牌赢得了更大的传播空间。

所以我们看到，具备良好用户思维的产品，相对于其他产品，总是更容易打开局面，直击痛点。

实际上，在我们的职场中也是这样。面对自己的用户，比如老板、客户或消费者时，如果能充分站在对方的立场思考和解决问题，无论是执行效率还是对方的主观体验，都会大大提升。

想要练就用户思维，首先需要知道由哪几种元素构成这种思维。

以下是用户思维的三个核心要素：

**第一点是共情能力。**
所谓共情，指我们要跳脱自己主观的感受和体验，去设身处地体察别人感受和体验的一种能力。让对方觉得“你懂我”。
对于产品设计者来说，共情能力是打造良好用户体验的重要前提。
举个例子，几年前很火的外卖APP“回家吃饭”。它最初因地铁广告走进人们视线：一个老奶奶手捧热汤说：“加班辛苦了吧，刚给我闺女熬好的鸡汤，也给你盛一碗。”这一点很快击中了许多以外卖为生的都市白领，让人印象深刻。类似广告文案的传播，使产品能在短时间内迅速引起用户兴趣。这就是强大的共情。

**第二点是需求的场景化能力。**
每种产品都对应一种需求，或者说产品本身就是对某种需求的场景化。
打个比方，**Keep**是一个很受欢迎的健身软件，它对应的需求是年轻人普遍的减肥瘦身和运动需求。
而实体健身房要么价格昂贵，要么过度营销。由此催生的场景是：在家里或任何地方都能进行的专业规范的健身训练。
产品的巧妙之处在于，它可以准确捕捉用户需求，并把它落实到具体场景上。

**第三点是精准表达。**
也就是说，尽量节约对方的理解成本，直击要害，会大大提高用户体验。
不管是工作还是生活中，我们都会发现一个现象：许多看似巧妙、暗藏玄机的设计，创作者花了100分心思，到了用户那里，可能只接收到50%甚至更少。这就是表达不够精准惹的祸。
记得多年前的全球移动互联网大会上，小米手机的巨幅海报让很多人印象深刻。不是因为它设计巧妙，相反，它是全场最简陋的海报：橘红底色上只写了四个大字“小米手机”。对比两旁精心设计但字小看不清的海报，它格外醒目。实际上，并非小米设计师不用心，而是他们深知在这种环境下，只有足够大的字，远处的行人才能看清。否则，设计再精巧，用户看不清也无济于事。

以上是用户思维的三个主要关键点。

接下去的内容，我们主要讲讲如何把这种互联网产品中的用户思维，运用到更宽泛的职场中，帮助职业生涯走得更加顺畅。

**第一步是锁定你的用户。**
互联网产品都有自己对应的受众，用户群越精准，越容易对症下药。
职场中也是类似：我们所做的每一件事都是一个“产品”，也都有它对应的“用户”。可能是你的老板、合作伙伴、关联部门同事或真实消费者。
不同职位面对的用户各有不同，但所有人都有一个共同用户：**老板**。他们是我们在职场中最需要重点维护的用户。老板对我们的体验和评价，很大程度上决定我们在职场中的上升和去留。
除此之外，外向型职位（如公关、销售、投资）也面临外部客户和消费者。好好服务他们是这些人的职责所在。
而一些基础性和研究型的内向型职位，则更多要面对公司内部各个关联部门的用户。
明确自己主要的用户是哪些人，能帮助我们在工作过程中精简思路，找到解决问题的主要切入点。在此基础上，不断优化升级用户体验，是我们在职场中能够不断成长和升级迭代最主要的内在动因之一。

**第二步是寻找共情点。**
前面说到，共情能很好地拉近我们与用户之间的距离。准确捕捉这种共情因素，来自于我们对用户特点、处境和心理诉求的捕捉和体察，也就是所谓的**用户画像**。
比方说，“上班族”这个群体，他们的特点是经常乘坐公交地铁、每天吃外卖、怀念家的感觉。“回家吃饭”这个APP从这些方面综合入手，就能很好地引起这些人的兴趣和共鸣。
对于职场人士来说，共情很多体现在我们是否能够与老板和客户同频思考问题，而不仅仅是从自己的利益诉求出发。
举个例子，在对外的商务谈判时，常规做法是“在商言商，等价交换”。但这种心态会让我们损失很多与客户成为朋友、拉近距离的机会。
而共情的做法是，站在服务提供者的角度，思考对方真正想要的是什么。通过这种方式，引起双方情感上的共鸣，拉近彼此距离，从而促成合作。

## 总结

本节课中，我们一起学习了产品思维的核心要素之一：**用户思维**。我们明确了用户思维就是站在用户立场思考的思维方式，并深入分析了构成它的三个关键元素：**共情能力**、**需求的场景化能力**和**精准表达**。最后，我们探讨了将用户思维应用于职场的第一步：**锁定你的用户**和**寻找共情点**。理解这些概念，是我们在工作中提升效率、优化成果、赢得认可的重要基础。下节课我们将继续学习如何培养和运用用户思维。
