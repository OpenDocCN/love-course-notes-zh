# 女神养成系列：02：常见花材识别与养护（下）

在本节课中，我们将继续学习几种常见花材的识别方法与养护技巧，包括菊花、龙胆花、绣球和睡莲。掌握这些知识，能帮助你更好地欣赏和照料它们，延长花期。

## 认识菊花

上一节我们介绍了多种花材，本节中我们来看看菊花。菊花在中国传统文化中，一直被视为高风亮节的花。从小我们就会背诵“采菊东篱下，悠然见南山”的诗句。还有“待到秋来九月八，我花开后百花杀”的诗句，讲的也是菊花，因为它的花期很长。

但由于一些习俗关系，今天我们并未准备很多单头大菊花。你更常见到的是小雏菊。小雏菊是一种非常常见的配花，也是花点时间最钟爱的配花之一。它的花期很长，既可以独立成束，也可以与其他花材搭配。当玫瑰、百合、康乃馨凋谢后，小菊往往依然存活。

以下是今天会看到的几种小菊：

*   **粉丝小菊**：一种常见的小雏菊。
*   **粉丹特小菊**：一个特别的品种，花型稍大，花瓣形状更锐利。
*   **绿色小菊**：花型也有不同，有些较小。

除了常见的小雏菊，还有另一种花材。它的学名叫**洋甘菊**，有黄色的花心和清浅的白色小花瓣，看起来像小野花，能带来很多野趣。这种花在国内市场上比较少见，通常来自进口。

我们还会经常看到**乒乓菊**。乒乓菊是圆形的，看起来很萌，一副呆头呆脑的样子。但它花期很长，可以开放很久。

## 认识龙胆花

说完了菊花，我们再来看看其他常见花材。比如龙胆花，它也叫做**洋桔梗**，经常出现在花束中。

有些用户分不清龙胆花和玫瑰。如果把两者放在一起对比就很清楚：玫瑰的花头是圆润的；而龙胆花在一枝上会有很多分叉和花头。尤其是香槟色的龙胆花，开饱满时也是圆圆的。但你收到时，它可能被压扁。这时可以对着花心轻轻吹气，让它恢复原状。

龙胆花在插花时，最需要做的一件事是“断舍离”。因为同一枝花上的分叉高低不同，为了与其他花材在视觉上协调，有时需要剪掉位置过低的分枝。虽然不舍，但为了整体花束的美观，需要舍弃不够协调的部分。剪下的花枝可以插成迷你花束。

龙胆花有各种颜色，其中一种非常特别的颜色是翠绿色的洋桔梗，绿中带一点黄，看起来非常清新，是一个较少见的品种。

还有一种花叫做**日本龙胆**，价格较贵。在鲜花中，紫色和蓝色是比较少见的颜色。这种花用来搭配香槟色等颜色非常美，花期也很长。

## 认识绣球花

接下来我们认识两种非常美丽但养护稍有难度的花。第一种是绣球花。绣球花最常见于婚礼上，因为它花型雍容，颜色饱和度很高，适合与其他花搭配。

绣球花有粉色、蓝紫色等颜色，这与种植土壤的酸碱度有关。还有一种特别的颜色叫做**秋色绣球**，通常只在9、10月份出现，很适合做干花，只需自然风干即可。

绣球花虽然美，但很容易脱水。因为它的花头很大，需要很多水分。深水养护绣球时，通常需要剪**十字剪**。

**十字剪步骤如下：**

1.  将绣球花茎底部横向剪一刀。
2.  再纵向剪一刀，形成一个十字形切口。
3.  可以看到茎内有白色的芯，可以用剪刀尖将其掏出来，以利于吸水。
4.  绣球的花茎留得越短，越有利于吸水。最好将叶子掰掉，避免水分被叶子消耗。
5.  将处理好的花茎插入深水中。

如果绣球脱水了，有两种补救方法：

1.  **敷面膜法**：将一张纸巾盖在蔫软的花头上，然后向纸巾上喷水，保持湿润。
2.  **浸泡法**：将蔫软的绣球花头朝下，整个浸泡在深水中，像泡“绣球汤”一样。大约三四个小时后，花瓣吸足水就会重新饱满起来。

## 认识睡莲

最后我们来认识睡莲。睡莲通常在夏天比较多见，是一种时令花材，到了冬天就没有了，请珍惜它。

你收到的睡莲可能含苞未放。如果摆了很久都不开，可能是因为没有做这件事：可以尝试将包裹在外层的绿色花瓣轻轻掰开。你可以掰掉它，如果不想掰掉，就掰到大约90度的位置即可。掰开后，通常能看到它微微绽放。

睡莲还有一个神奇的特性：在白天对着阳光时会绽开，到了晚上则会闭合。

养护睡莲有一个重要的点：**一定要深水养**。因为睡莲（荷花）原本就生长在水里。深水养是指水位几乎要没到它的花茎顶端。想象一下，从花头下方开始，整根花茎都浸在水里，它会开得很好。睡莲不需要频繁剪根，因为它可以从底部一直向上吸水。

**示例插法**：将睡莲插入很高的花器中，让水一直淹到它的“肩膀”（即花头下方）位置。
