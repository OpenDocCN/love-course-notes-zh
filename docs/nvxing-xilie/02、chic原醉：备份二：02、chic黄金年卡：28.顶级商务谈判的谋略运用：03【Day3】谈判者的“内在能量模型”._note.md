# 谈判哲学：02：顶级商务谈判的谋略运用：03【Day3】谈判者的“内在能量模型” 💡

在本节课中，我们将要学习一个关键的谈判工具——**内在能量模型**。这个模型帮助我们理解并调动自身不同的人格属性，从而在谈判中更好地掌控节奏、表达意图并达成目标。

上一节我们介绍了谈判中的信息战，掌握了信息收集与处理的基础。然而，仅仅拥有信息还不够。本节中我们来看看，当面对实际谈判压力时，如何运用内在能量来有效表达并重掌主动权。

很多时候，即便准备充分，真正谈判时却难以有效表达。要么说出来的话与初衷不符，要么轻易被对方带偏节奏。这时，我们需要一个工具来调整自身状态。

这个工具就是**内在能量模型**。它指的是每个人在人际沟通时，内在能量源自几种不同的人格属性。

哈佛大学法学院的福克斯教授研究发现，人在沟通时呈现的人格并非单一，而是由几种属性综合而成。以下是构成内在能量模型的四种人格属性：

*   **思想者**：人格中最理性、注重微观细节且富有逻辑的部分。它帮助我们判断细节、数据和逻辑框架，在谈判中起到稳定步调的作用。例如，当对手提出不切实际的承诺或语言中设下陷阱时，思想者能帮助我们拨开迷雾，看清本质。
*   **梦想家**：人格中宏观、理想化且富有战略眼光的部分。它善于制定战略愿景，对目标和方向有清晰认知。在谈判中，梦想家的作用有二：一是从气场上让对方产生敬畏，表明我们是有目的、有战略的；二是把握整场谈判的大方向。那些立场坚定、不易被影响的人，通常具备明显的梦想家特质。
*   **勇士**：负责人格中勇气与执行力的部分。它在谈判中能展现出不达目的誓不罢休的决心。例如，在企业扩张初期，与知名公司谈合作常会碰壁。勇士的力量驱使一些人选择顽强坚持，而非退缩。
*   **恋人**：人格中最感性、温柔细腻的部分。其特点是懂得聆听、善于共情。在谈判中，恋人帮助我们与对方维护良好关系，体察对方情绪，起到居间调停和润滑的作用，避免谈判陷入僵硬尴尬的气氛。

以上四要素共同构成了谈判者的**内在能量模型**。

听到这里，你可能觉得其中某一两项更像自己平时的状态。但实际上，这四种属性存在于每个人的人格中，只是在面对不同情形时，主导的人格属性不同。

在四者之中，梦想家和勇士是紧密结合的整体。前者负责“我想要什么”和“我不要什么”这类战略问题；后者负责“我要去做什么”和“我不要去做什么”这类执行层面的战术问题。

而思想者和恋人则是一对相互牵制的矛盾体。思想者负责探寻情感因素以外的真相，但过于理性易使谈判陷入僵局，需要恋人的感性来中和。

在任何一场谈判中，只有充分调动这四者，才能很好地控制节奏。如果缺失任何一种，谈判都可能陷入困境；反之，如果将任何一种发挥到极致，后果也不堪设想。

所以，接下来的内容主要讲解如何正确运用内在能量模型，调动不同板块的人格属性。

前面讲到，谈判中势均力敌的情况不常见，双方往往存在一定的势能差。具体到现实中，大致可分为以下三类：

*   **劣势谈判**：对方在公司实力、影响力等各方面都超过我方，我方处于相对劣势。换言之，对方可能并不在意谈判结果，因为他们有众多合作伙伴可选；但对方对我方而言却是非常重要的合作对象。
*   **优势谈判**：与我方相比，对方处于弱势，我方更有话语权。此时，我们主要关注三个问题：一是我方对合作对象的最大诉求是什么，对方能否满足；二是如何在谈判中争取更大现实利益；三是对方还有什么其他优势或潜能，能在远期为我所用。这类谈判的难度较第一种轻得多。
*   **互有所求式谈判**：双方相对平等、和谐，彼此都有合作需求。这种情况比较容易达成合作。但我们之前讲过“复合需求”概念，即双方可能在某个方面相互需要，但在其他方面仍可能产生分歧。例如，投资人和创业者都期待长期股权增值，但可能对短期公司估值有异议。

以上三种常见的谈判格局，都可以运用本节课讲到的内在能量模型来应对，只是在不同的应用场景和案例中，侧重点可能不同。

下面我们以最难的**劣势谈判**为例，做一个拆解模拟。

对于这类谈判，我们首先需要对局势和双方情况有清晰认知，思考通过什么方式赢得合作机会。在这个过程中，我们首先需要请出**思想者**和**恋人**这一对矛盾体。

由**思想者**分析清楚当前的局势和双方各自的真实实力、优势与劣势；用**恋人**体察对方的立场和诉求，目的是找到我们接近对方、让对方让步的突破口。

在这一步完成之后，我们需要请出**梦想家**和**勇士**这一对“虎将”。一个负责确定目标和战略，另一个负责制定计划并实施战略。在真实谈判中，二者的作用相辅相成。试想，如果谈判对手只会定方向、唱高调，而没有任何规划和执行力，我们也会觉得他不靠谱。

因此，在制定战略的同时，一定要对战略进行战术化拆解，明确每一步具体该怎么做。

例如，假设我们是一家成立不久、市场影响力不大的投资顾问服务机构，想通过与大型投资机构合作来提升行业知名度。

这时，我们需要先动用**思想者**去思考：我们与对方的真实实力差距如何？周围有哪些竞争者？我们自己有什么优势可作为置换筹码？

然后，用**恋人**去询问和感知：对方对这场合作可能有怎样的期待？目前最需要我们这类机构帮他们做什么？最大的痛点和难点是什么？

接着，以此为依据，结合我方意图，动用**梦想家**去制定谈判的战略目标。例如：我们要满足对方哪些需求？要帮对方对接到怎样的项目？我们自己想从中获得什么？

最后，用**勇士**去制定具体计划，思考如何满足对方的种种需求。

这样就基本完成了我们通过**内在能量模型**进行谈判准备与实践的过程。

本节课中我们一起学习了**谈判者的内在能量模型**。这个重要概念主要包含四个部分：**思想者**、**梦想家**、**勇士**和**恋人**。它们的相互配合，能帮助我们在谈判中发挥出理想水平。

除此之外，我们还讲到了谈判的三种常见类型：**劣势谈判**、**优势谈判**和**互有所求式谈判**。
