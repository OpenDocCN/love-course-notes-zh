# 超模私教天团：21天瘦身训练营：Day10：【有氧运动日】无课

在本节课中，我们将要学习有氧运动日的核心任务与科学原理。我们将明确今日的训练目标，并理解有氧运动在整体瘦身计划中的关键作用。

上一节我们安排了力量训练，本节中我们来看看如何通过有氧运动高效燃脂。

今日的训练核心任务是进行**室外慢跑或快走5公里**。执行此任务时，需要将配速控制在**6至9分钟每公里**的范围内。这个强度的运动能够达到让身体微微出汗的效果。

然而，仅仅微微出汗对于有效燃脂是不够的。为了达到理想的瘦身效果，有氧运动的持续时间至关重要。

以下是关于运动时长的关键要求：
*   有氧运动必须持续**30分钟以上**。
*   总运动时长不应超过**1小时**。

因此，在实际的5公里训练中，请确保你的运动时间落在上述区间内。在当前季节进行室外跑步，完成5公里后通常能达到全身湿透的效果，这表明运动强度是足够的。

关于如何安全有效地进行跑步，有一些重要的技术细节需要注意。

以下是跑步时的关键注意事项：
*   正确的跑步姿势可以保护膝盖和脚踝免受伤害。
*   详细的姿势教学已在试听课中讲解，学员可返回专栏复习相关内容。

最后，我们需要理解有氧运动与力量训练之间的关系。力量训练能增加肌肉，提升基础代谢；有氧运动则直接消耗热量，提升心肺功能。

两者结合才是最科学高效的瘦身方法，公式表示为：**最佳瘦身效果 = 力量训练 + 有氧运动**。切勿忽视有氧运动的重要性。

有氧运动提升心肺功能，力量运动增强肌肉强度，两者结合才能使身体状态达到巅峰。坚持21天，你将完成蜕变。
