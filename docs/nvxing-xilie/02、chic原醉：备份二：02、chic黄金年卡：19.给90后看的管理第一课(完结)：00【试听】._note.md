# 团队管理术：02：708090后的职场欲求画像

在本节课中，我们将要学习如何识别和理解70后、80后、90后不同年龄层员工的职场核心诉求。掌握这些差异是进行有效团队管理的第一步。

当你决定学习本课程时，我猜你已经是一位新进管理者，或者是一位预备管理者了。只是在面对团队管理时或多或少的碰了壁，或是正面临着与团队的博弈。

心里没底儿，比如如何管理比自己牛的人，如何让年龄大的下属听自己指挥。以前老话说，新官上任三把火，现在是你刚上任，下属给你挖撒坑，该怎么躲避？再或者你带的团队流动性还很大，老板开始质疑你的领导力。暗中招募接替你的人怎么办？

类似的棘手问题还有很多，不过没关系，通过本套课程的学习，你一定会获得超强的解题思维与能力。

好，要想解决如何带好团队的问题，你得先搞清团队成员们的职场性格与诉求。如今，公司里大多是三代同行的现状，作为新进小领导的你，要面对的下属大多是90后、80后，甚至还有70后的老叔叔老阿姨们，如何带动团队里不同年龄层的人一起工作，让团队良性高效运转起来，着实考验年轻的你。

如果你偷懒，想用同一种方式管理不同年龄段的人，结果会很难看。去年网上出现过一个很火的段子，不要大声责骂年轻人，他们会立刻辞职。但你可以往死里骂那些中年人，尤其是有车有房有娃那些。

实际上这说的就是差异化管理的道理。

所以这第一节课我想跟你分享的是708090后的职场欲求画像，更是一个人随着年龄增长，职场欲求不断发生的变化。你只有摸清了下属在职场中的欲求命脉，抓住了他们的痛点，你的管理才能对症下药，变得有效高效。

## 职场欲求的五个维度

那随着年龄增长，人们的职场欲求到底会发生哪些变化呢？为此，我咨询了一位资深的人力管理者，他是这么说的。

职场人的职场欲求大致能概括出5个维度，分别是**技能获取**、**价值认同**、**薪酬**、**晋升与权利**以及**自由度**。只不过不同年龄层的人，在各个维度上需求强弱度不同。

这很好理解，比如说工龄越长，你对工资水平的期望和要求也就越高。

## 90后的核心职场欲求

那708090这三个年龄层的人，他们各自最为突出的职场欲求是什么呢？我先和你说说90后的职场欲求。

90后是当下最鲜活的一股生命力，最能代表他们的词是**成长**，尤其是技能成长，同时伴随的还有心智成熟。

不难理解，从校园、书本到职场实践，90后最先要突破的是岗位技能大关，像是与现实市场接轨的策划，实打实的项目开发和运营。

所以90后最突出的职场欲求之一是**技能获取**。
