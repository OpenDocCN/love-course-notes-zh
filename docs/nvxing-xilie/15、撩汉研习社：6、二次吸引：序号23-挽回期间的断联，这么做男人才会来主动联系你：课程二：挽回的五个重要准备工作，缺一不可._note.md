# 撩汉研习社：6：二次吸引：挽回期间的断联，这么做男人才会来主动联系你

## 课程二：挽回的五个重要准备工作，缺一不可 🎼

在本节课中，我们将要学习挽回前任过程中至关重要的五个准备工作。这些步骤是成功挽回的基础，缺一不可。我们将逐一详细解析每个步骤的具体做法和背后的原理。

### 概述

我理解被分手者的悲伤。但时间有限，我们没有太多时间沉溺于难过或祭奠逝去的感情。我们需要做的是立即行动，力挽狂澜，重新俘获他的心。在采取行动之前，必须做好充分的准备工作。今天的课程将讲解挽回前任的五个重要准备工作，你必须牢记于心。

以下是五个关键的准备工作：
1.  对方表达分手时，控制情绪，保持冷静。
2.  冷静反思这段关系。
3.  避免偏激行为，增加挽回成本。
4.  查缺补漏，改造自己。
5.  请专业人士帮忙。

下面我们来逐一分析这几点。

### 一、保持冷静

当对方表达出分手意愿时，我知道很多女性会情绪失控，做出冲动行为，例如大吵大闹、跪求复合，甚至以死相逼。要知道，这些行为不仅无效，还会起反作用，只会增加挽回的难度。

那么，此时应该做什么呢？

心理学有一个概念叫做**峰终体验**。通俗来讲，就是人们对一段经历的记忆，主要由两个因素决定：**高峰时（最痛苦或最开心）** 和**结束时**的感受。

根据这个概念，引出了一个常说的理论：分手时才是看人品的时刻。因此，如果他提出分手后，你又哭又闹、以死相逼，他对你的“峰终体验”就停留在最坏的一面。如果他提出分手，你平静接受，他对你的“峰终体验”就会停留在好的一面，甚至可能产生愧疚。

那么，该如何面对这尴尬而痛苦的分手场景呢？我们假设几个场景：

*   **场景一：短信告知分手。**
    *   **正确做法：** 回复“尊重你的选择，谢谢你给我的爱。” 不要长篇大论。

*   **场景二：当面告知分手。**
    *   **正确做法：** 可以含泪且柔情地说：“我知道我有很多做得不对的地方，对不起。我尊重你的选择。”

*   **场景三：托人告知分手。**
    *   **正确做法：** 平静且难过地说：“谢谢你的转告，他是个很好的人。我祝福他。”

*   **场景四：完全不联络表示分手。**
    *   **正确做法：** 此时不能发跪求复合或承认错误的信息。应该努力回忆，仔细分析他分手的原因，并着手进行自我提升，为二次吸引做准备。

### 二、冷静反思关系

上一节我们介绍了如何控制情绪，本节中我们来看看如何理性分析关系。虽然生活中我们无法像狄仁杰办案那样抽丝剥茧，但起码要有黑猫警长的机警。好好思考一下，为什么两个人会走到这一步？自己曾经做过什么？他做过什么？

同时，要把在他身上的情绪进行抽离。要知道，他现在对于你来说，连普通朋友都算不上。不接电话、不回短信甚至拉黑你，都很正常。不要为此难过伤神，这会影响你对这段感情的判断，增加挽回难度。

此时可能有姑娘会说：“难道这是要让我放弃的节奏吗？” 错。我知道你们要挽回，挽回对你们而言是人生大事，怎么会叫你放弃呢？我是要让你**放下**。暂时放下因被分手而变得乱七八糟的心情，放下那些海阔天空却悲伤满满的想法，放下他在你心中的分量。只有这样，才能轻装上阵，早传捷报。

### 三、避免偏激行为

分手后，切忌做过激行为来增加挽回的成本。

以下是必须避免的行为列表：
*   **切忌夺命连环电话。**
*   **切忌找上门去**，包括他的家、公司、朋友家、前女友家等。
*   **切忌胡乱搬救兵**，包括闺蜜、哥们、双方亲朋等。
*   **切忌以死相逼。**

这些都是挽回的大忌，也是大部分人在挽回过程中会犯的通病。用这些死缠烂打的方式去祈求对方回心转意，是低级且无用的，最终容易引起对方的厌恶心理。

### 四、查缺补漏，改造自己

趋利避害是人的本性。对方要离开你，一定是你身上有他无法接受的东西，或者他有了比你更好的选择。找到自己的不足，然后进行提升，例如外形、气质、内涵、脾气等。

千万不要认为真正爱你的男人就会接受你的一切。亲爱的姑娘，这是骗人的，谁信谁就被坑了。

### 五、寻求专业帮助

如果没有头绪，就要请专业人士进行问题分析和解决。闺蜜或亲戚往往不能客观理智地为你解决问题，甚至可能起到反作用。

### 总结

本节课中，我们一起学习了挽回前任必须完成的五个重要准备工作：

1.  **控制情绪，保持冷静：** 对方表达分手时，跪舔和冲动的行为只会增加挽回难度。
2.  **冷静反思关系：** 认真思考走到分手这一步的真正原因，并抽离情绪。
3.  **避免偏激行为：** 夺命电话、不停纠缠、以死相逼等过激行为是挽回大忌，只会让关系彻底恶化。
4.  **查缺补漏，改造自己：** 找到对方无法接受你的地方和自身不足，是提升的关键。
5.  **请专业人士帮忙：** 寻求客观理智的分析和解决方案。

之前的课程中，我们一直在强调挽回时需要断联，需要给自己时间去蜕变和升华，也需要给对方时间去平静和重新接受。那么，关于为什么要断联？断联的真正目的和作用是什么？这些你都知道吗？下节课我们就来讲讲分手之后的断联环节，以及断联后你的正确做法应该是什么。
