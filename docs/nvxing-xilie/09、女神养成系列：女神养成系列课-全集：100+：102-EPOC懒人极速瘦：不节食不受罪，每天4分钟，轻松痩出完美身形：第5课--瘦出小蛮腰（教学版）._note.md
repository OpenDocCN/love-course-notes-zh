# 女神养成系列：第5课：瘦出小蛮腰 🎯

在本节课中，我们将学习如何通过强化腰腹及核心肌肉群，来有效减小腰围、塑造纤细腰部。课程包含四个针对性动作，无需节食，每天仅需4分钟。

## 概述

许多人常为腰围发愁，尝试多种方法但效果不佳。腰部赘肉不仅源于脂肪堆积，也常因腹部周围肌肉松弛所致。因此，要真正拥有小蛮腰，除了减少脂肪，还必须强化腰腹肌肉群。本节课程将针对腰腹及核心肌群进行训练。

上一节我们介绍了整体训练理念，本节中我们来看看具体的腰腹塑形动作。

## 动作教学

以下是本节课的四个核心训练动作。

### 1. 侧平板支撑

这个动作主要训练腹部两侧肌肉，改善侧腰松弛和脂肪堆积。

**动作要领：**
1.  侧卧于垫上，双腿伸直。
2.  用下方手臂的小臂支撑身体，肘部位于肩部正下方。
3.  确保头、背、臀、脚在同一直线上。若感觉困难，可先背靠墙壁练习，找到身体平直的感觉。
4.  发力将髋部向上顶起，感受侧腹收缩，然后缓慢下放至接近地面。

**呼吸配合：**
向上顶起时用嘴巴吐气，向下放时用鼻子吸气。

**练习建议：**
每侧练习30秒，完成后换另一侧。

### 2. V字固定

此动作能充分训练腹部深层及表层肌肉群，对侧腹、下背和腹直肌都有良好效果。

**动作要领：**
1.  坐于垫上，身体向后倾斜约45度，保持背部呈“C”形，收紧腹部。
2.  肩膀下沉，远离耳朵。
3.  缓慢抬起双腿，保持身体稳定。

**呼吸配合：**
保持均匀呼吸。

**难度调整：**
若感觉困难，可先将双脚放在垫上，仅保持身体后倾姿势。适应后再尝试抬起双腿。

**练习建议：**
保持该静态姿势30秒。

### 3. 跪姿平板支撑

这是一个经典的静态支撑动作，能增强腹部深层肌肉、肩膀、髋部和膝盖的稳定性，像“天然腰带”一样保护腰椎。

**动作要领：**
1.  双膝跪地，以前臂和脚尖支撑，肘部位于肩部正下方。
2.  头部向前延伸，肩膀下沉。
3.  腹部收紧，保持头、背、腰、臀在一条直线上，避免腰部塌陷或过度拱起。
4.  臀部收紧，想象夹住一张纸。
5.  用整个前臂力量支撑，而非仅用肘关节。

**练习建议：**
保持该姿势30秒或更长时间，尽力维持身体稳定。

### 4. 杰克跳（开合跳变式）

这个动作能有效提升心肺功能，促进全身脂肪燃烧，作为训练的收尾。

**动作要领：**
1.  站立，保持头、背、臀、脚在同一直线。
2.  跳跃时双脚开合，同时双臂上下摆动。
3.  跳跃全程保持身体稳定，避免晃动。

**重要技巧：**
落地时，务必用脚尖先着地，然后过渡到脚跟，同时膝盖微屈以缓冲冲击力，像“避震器”一样保护膝盖。避免全脚掌直接砸向地面。

**呼吸配合：**
跳跃打开时吸气，合拢时吐气。

**练习建议：**
可进行5次一组，或持续跳跃30秒。

## 总结

本节课我们一起学习了四个针对腰腹塑形的关键动作：**侧平板支撑**、**V字固定**、**跪姿平板支撑**和**杰克跳**。通过规律练习这些动作，可以强化腹部两侧及深层肌肉，收紧腰腹线条，同时提升核心稳定性和心肺功能。请记住，持之以恒是获得完美身形的关键。
