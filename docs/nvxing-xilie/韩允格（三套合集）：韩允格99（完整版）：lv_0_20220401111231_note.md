# 职场情商课：01：边界感的建立与运用

在本节课中，我们将要学习如何在职场中建立并维护健康的个人边界感。边界感是保护个人时间、精力和情绪的重要工具，能有效防止过度消耗，并帮助我们更清晰地聚焦于职业发展目标。

上一节我们介绍了边界感的重要性，本节中我们来看看如何具体地树立边界感。

## 🛡️ 如何树立个人边界

首先，你需要学会用语言和姿态向他人表明你是有底线的人。想要获得他人的尊重，你必须先学会尊重自己，这意味着要正视并表达自己的需求。例如，明确告知他人：“我的需求同样重要，我的私人时间同样需要被尊重。”

以下是树立边界感的核心步骤：

1.  **明确表达**：清晰、直接地沟通你的界限，例如“我无法在下班后处理这项工作”。
2.  **学会拒绝**：对于超出职责范围或能力的不合理要求，要敢于说“不”。
3.  **保持一致性**：一旦设定了边界，就要坚持，避免因他人施压而妥协。

## 😨 建立边界可能面临的挑战

然而，在树立边界感的过程中，你可能会遇到一些挑战。拒绝他人时，你可能会产生内疚或恐惧，害怕因此不被喜欢、被孤立或在背后被议论。

同时，被你拒绝的人也可能产生负面反应。以下是他们可能出现的心理及行为：

1.  **产生意见**：对方最初可能会对你有不满情绪。
2.  **施加压力**：对方可能尝试道德绑架你，或在背后说坏话、制造障碍。

一个成熟的成年人需要有能力承担自己选择带来的后果，而不是将其强加于你。如果对方无法接受你的合理边界，说明他本身缺乏界限感且较为贪心。对于这种人，应尽量保持距离，减少接触，否则你只会被持续消耗，且对方成为“白眼狼”的概率很高。

## ⚖️ 边界感与职场情商的平衡

但是，在职场中必须明确一点：拥有边界感不等于没有情商。你需要区分不同的社交场景。

例如，在小团队中与同事和睦相处，扮演“老好人”，或是为了老板的任务积极投入甚至牺牲部分个人时间，你是哪一种？有些人错误地认为“得人心者得天下”，试图与所有人搞好关系以期在升职时获得支持。

实际上，如果你需要选择站队，建议优先考虑与**老板、主管及上级领导**建立良好关系。因为决定你升职加薪的是你的上司，而非你身边的同事。你必须拎清这其中的轻重关系。

建立边界感时，切忌方式过于生硬。在职场中，老板很难喜欢一个浑身是刺、难以协作的员工。这不是鸡汤，现实中确实有人终日忙碌却得不到提拔。因此，在保护自己边界的同时，也要懂得如何向上管理，谋求发展。这便涉及到情商的运用。

## 🧠 职场情商的本质与应用

那么，情商究竟是什么？情商并非“跪舔”。情商的本质是**树立一个明确的目标，然后以投其所好、不令人反感的方式去达成它**。这才是高情商的表现。

因此，做事不可只听一半。我教你树立边界感，并非让你一旦被触犯就“暴跳如雷”。你需要辨别对象：对于那些只会消耗你的人，当然要坚决拒绝；但对于那些能为你赋能、让你学到东西、锻炼你能力的机会或人，则应该以开放的态度对待。

本节课中我们一起学习了职场边界感的建立方法、可能遇到的挑战，以及如何平衡边界感与情商，从而在保护自我的同时，更有效地实现职业目标。
