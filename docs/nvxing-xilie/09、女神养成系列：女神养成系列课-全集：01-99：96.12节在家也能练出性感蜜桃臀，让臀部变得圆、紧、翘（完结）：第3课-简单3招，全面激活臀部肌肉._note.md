# 女神养成系列：03：全面激活臀部肌肉的3个简单动作

## 概述

在本节课中，我们将学习三个专门用于激活臀部深层肌肉的动作。这些动作旨在全方位锻炼臀部肌群，包括臀大肌、臀中肌和臀小肌，帮助塑造圆润、紧致、上翘的臀部曲线，并促进下半身血液循环，辅助燃脂。

上一节我们介绍了臀部训练的基础概念，本节中我们来看看具体的激活动作。

## 臀部肌肉与训练原理

我们的臀部主要由三块肌肉构成：**臀大肌**、**臀中肌**和**臀小肌**。其中，臀中肌和臀小肌属于深层肌肉，位于臀部上侧。锻炼这两块肌肉对于塑造漂亮的侧面臀部曲线至关重要。

长期久坐且缺乏运动，会导致臀部及下肢血液循环不畅。当血液循环难以充分到达这些区域时，脂肪就容易堆积。本节的训练能够像启动一个小马达一样，促进下半身血液循环，从而帮助燃烧脂肪。

## 动作详解

以下是三个全面激活臀部肌肉的动作，请依次练习。

### 动作一：站姿腿后摆

这个动作主要针对臀大肌，通过静态收缩来增强肌肉的紧实感。

1.  自然站立，挺胸收腹，腰背挺直，双手叉腰。
2.  将左腿伸直，向后抬起至个人能达到的极限。抬腿时，两腿之间的夹角大约在15到30度之间。
3.  在腿后摆发力的顶峰位置，固定姿势5到10秒钟，让臀大肌彻底收紧，然后缓慢还原。
4.  换右腿重复以上步骤。在发力顶峰，即臀部感觉最酸胀时，务必保持5到10秒，这能让臀部持续发紧，达到紧实效果。

### 动作二：站姿侧抬腿

这个动作旨在激活位于臀部上侧的臀中肌和臀小肌。

1.  自然站直，双手掐腰，双脚微微分开。
2.  将重心落于右脚，向侧面抬起左腿至最高点，然后缓慢放回。
3.  控制动作速度，左脚全程不着地，上半身保持稳定。
4.  抬腿外展时呼气，落腿时吸气。若想增加难度，可以勾脚尖发力。

完成此动作后，如果感到臀部侧面有酸胀感，说明发力正确。

**动作要点：**
*   与第一个动作一样，站立时需挺胸收腹。
*   抬腿时保持身体平衡和稳定，尽量让支撑腿垂直于地面，避免身体东倒西歪。

### 动作三：保加利亚深蹲

这是一个能同时锻炼臀部和腿部的复合动作，对塑造整体臀腿线条很有帮助。

1.  挺胸收腹，单腿站立。将另一条腿的脚面搭在身后的椅子或凳子上。椅子高度可根据个人身高调整，一般不建议超过膝盖高度。
2.  缓慢下蹲，直至前腿膝盖和后腿膝盖均弯曲成大约90度。与徒手深蹲要求一致，前脚膝盖不要超过脚尖，后脚不能触碰地面。
3.  驱动前腿，向上伸髋伸膝站起。整个过程中，腹部要始终收紧，脊柱保持挺直。

**发力要点：**
做动作时，注意力可集中在后腿。在身体上升过程中，后侧大腿要主动向上绷直，这样才能有效刺激臀部。同时，颈椎保持挺直，腰、腹、臀部都要收紧。
