# 女神养成系列：15：请客与送礼的礼仪要点

在本节课中，我们将学习商务与社交场合中至关重要的请客与送礼礼仪。掌握这些要点，能帮助你在职场中建立良好的人际关系，展现专业与得体的形象。

上一节我们介绍了如何通过自我介绍给人留下良好印象。本节中，我们来看看如何通过成功的宴请和恰当的赠礼来深化关系。

一次失败的宴请可能源于对基本礼仪的疏忽。例如，误让客人坐在主位并买单，会显得主人失礼且准备不足。商务宴请不仅是吃饭，更是一个展示个人与公司形象、促进交流的重要平台。在轻松的氛围中，人们更容易放下戒备，增进了解。

一次成功的宴请离不开周密的准备工作。以下是通用的注意事项。

**第一点是预约。** 预约包含对餐厅和对客人的预约。
*   **餐厅预约**：需提前1-3天沟通就餐人数、时间、区域（如吸烟区）、特殊需求（如生日蛋糕）、饮食禁忌及大致消费标准。
*   **客人预约**：重要宴请最好提前一周甚至一个月发出邀请，以示尊重，并在宴请前再次确认。准时赴约是最基本的礼貌。

**第二点是点菜。** 点菜是展现诚意的开始，需考虑多方面因素。
*   **宗教饮食禁忌**：必须尊重客人的宗教信仰相关的饮食习惯。
*   **特殊人群禁忌**：需提前了解客人是否有疾病（如心血管、肝肾问题）或食物过敏史。
*   **饮食偏好禁忌**：注意地域口味差异（如喜辣或清淡）及文化差异（如有些国家不吃宠物、动物内脏）。
*   **职业禁忌**：例如，公务员公务餐有标准限制，司机不能饮酒。
*   **点菜技巧**：菜单应先递给客人；点菜数量可比人数多1-2道；注意荤素、口味搭配，尤其照顾女性客人需求；点完菜应询问酒水偏好。

前面提到了西餐礼仪，这里我们重点说明中餐礼仪。

首先是**座位安排**。主人通常坐于主位，确定主位有五个原则：
1.  **以中为上**：离主人近的位置为尊。
2.  **以右为上**：按国际惯例，主人右侧为上位（国内政务礼仪有时以左为尊，需视情况而定）。
3.  **面门为上**：主位一般正对门口。
4.  **以远为上**：主桌或上位应远离通道。
5.  **特殊原则**：观赏角度最佳的位置可为上座。

入座后，应与邻座适当交流。勿玩弄餐具或手托下巴，双手可自然放于膝上。

其次是**餐具使用**，需格外留意。
*   **筷子**：使用不当会显得缺乏修养。需避免“八忌”：
    1.  **舔筷**（将筷子放入口中吸吮）。
    2.  **迷筷**（持筷犹豫，在桌上巡找）。
    3.  **扒筷**（用筷子在菜盘中翻扒挑拣）。
    4.  **扭筷**（扭转筷子）。
    5.  **插筷**（将筷子插在饭中）。
    6.  **舞筷**（说话时挥舞筷子）。
    7.  **跨筷**（将筷子横跨在碗碟上）。
    8.  **剔筷**（用筷子剔牙或夹取食物以外的东西）。
*   **勺子**：主要用于取食，注意：取食勿过满；不用时放下，勿同时持筷和勺；勿用嘴吹凉食物；勿反复舔食。
*   **食碟**：暂放取自公盘的菜肴，注意：取菜勿过多；骨、刺等残渣应轻放于碟前端，由服务员及时更换。
*   **辅助餐具**：
    *   **水杯**：专用于喝水，勿倒扣。
    *   **湿巾**：仅用于擦手。
    *   **牙签**：尽量避免当众剔牙。必要时，应以手掩口，勿叼着牙签。

宴请不同对象，侧重点也不同。
*   **宴请领导**，关键在于“**尊**”，体现在客人级别、宴请规格及环境的私密性。
*   **宴请客户**，关键在于“**敬**”，需匹配客户身份，讲究舒心、贴心。
*   **宴请同事**，关键在于“**和**”，菜品应可口，环境宜温馨，便于交流。

此外，还需考虑餐厅位置与交通便利性，并根据宴请目的（如拉近关系或商谈工作）选择相应氛围的场所。

宴请之外，送礼也是增进感情的重要方式。一件恰当的礼物能有效促进关系发展。

关于送礼，有以下三个方面需要注意。

**第一是礼品的定位。**
*   私人交往，礼品宜定位为**纪念品**。
*   企业交往，礼品宜定位为**宣传品**（如带有企业标识的物品）。

**第二是礼物的选择与赠送。** 关键在于用心，需入乡随俗、换位思考、量力而行。思路可遵循：人无我有，人有我优，人优我“心”（更显心意）。具体可参考 **5W原则**：
1.  **Who**：送给谁？
2.  **What**：送什么？考虑时尚性、独特性、便携性。
3.  **Where**：在何处送？公务礼品在办公地点送，私人礼品在私人场合送。
4.  **When**：何时送？做客时宜见面即送；主人送礼宜在客人告辞时。
5.  **Which**：如何送？尽可能亲自赠送；国际交往中需注重礼品包装。

**第三是接受礼品。**
1.  **大方接受**：可收的礼物应坦然接受，但现金、有价证券或过于贵重的礼物不宜收。
2.  **表示感谢**：当面致谢。需注意，国内习惯事后拆礼，而国际惯例常当面拆开并赞赏。
3.  **保持低调**：不宜张扬；无特殊原因，收到的礼物勿转送他人。

针对不同对象的送礼要点：
*   **送领导**：价值适中，以示尊重，注意时机与场合。
*   **送客户**：重在心意，需全面考虑客户背景，让其感受到被尊重与关怀。
*   **送同事**：象征友好，宜实用，包装不必过于隆重。

本节课中我们一起学习了商务交往中的宴请与送礼礼仪。宴请旨在增进感情、促进发展，需从预约、点菜到座次安排、餐具使用等环节严格遵守礼仪，特别是座次的五个原则和筷子的“八忌”。送礼同样为增进感情，需遵循因人而异、换位思考的原则，参考5W原则选择与赠送礼品，接受礼物时也需大方得体、懂得感谢与回礼。
