# 见家长攻略：02：善始善终做好收尾工作——给拜访画上完美句号

在本节课中，我们将要学习如何为见家长活动画上一个圆满的句号。我们将从见面尾声的礼仪，到分别后如何持续维护关系，系统地讲解收尾工作的核心要点，帮助你巩固好印象，为长期关系打下坚实基础。

不知不觉，本套课程已接近尾声，见家长活动也到了要分别的时刻。上一节我们探讨了拜访中的互动技巧，本节中我们来看看如何做好至关重要的收尾工作。

当你见完家长之后，可能会感到如释重负。特别是当男友告诉你他的父母对你印象很好时，你可能会开始憧憬未来。但其实还有一个非常重要的步骤需要妥善完成，才能为你和男友的长期关系建立更稳固的基础。这个步骤就是见家长的收尾工作。

下面我们会从见面尾声讲到分别后彼此回归各自生活时，与对方父母的正确相处方式，来告诉大家究竟怎样收尾，才能和他的父母维持好感情。

## 见面尾声：得体接受见面礼

首先，在见面活动接近尾声时，你和男友的家长已经相互有了一定的了解。如果你在前期的表现不错，对方父母可能已经对你表示肯定。这个时候，他们常会通过递上红包或送上其他见面礼来表达心意。

还有可能准备一些当地特产，让你转交给你的父母，以表达他们的友好和想要与你们家进一步发展关系的意愿。

这个时候，女孩们的第一反应可能是礼貌地推辞一下。但这并不是接受见面礼最好的方法，有时甚至会起到相反作用。

在传统意义上，对别人赠送的礼物表示推辞是一种礼节。但实际上，这在对方父母眼里可能是一种距离感，甚至会引起不悦。站在对方角度来看，给你见面礼就是表达对你的认可，想要和你做一家人。如果你拒绝了，可能会让他们没面子，还会觉得你太客套，有意和他们拉开距离。

所以，当对方家长要给你见面礼时，最好的应对方式就是不要客套，直接用双手接过来，并且表示自己的感恩和欢喜。这对他们来说就是十分恰当的反馈。

就像在真人秀《如果爱》中，颖儿第一次去见付辛博的家长，会面结束后，对方爸妈也给颖儿的父母准备了一些礼物。颖儿的做法是开心地接过来，鞠躬连说了三个谢谢，表现得得体大方，非常有亲和力。

不过，有时我们也需要先推辞一下再接。前提是对方父母送的礼物实在太贵重。为了不让自己有很大的心理负担，我们可以先推辞一下。

当对方家长要将价值不菲的礼物送给你时，你要先拒绝他们的好意。推辞的时候，你可以这样说：“阿姨，礼物实在太贵重了，这样不太好。”如果对方执意要给你，你再接过来并表示感谢就没问题了。

而如果对方将祖传的宝贝直接给你，你两眼一亮，立马就抱在怀里，可能会让对方感到内心失落，甚至会对你产生误解和疑虑。

## 告别时刻：巧用“峰终定律”留下好印象

当你和对方父母正在告别的时候，学会巧用“峰终定律”能给对方留下最后的好印象。

心理学上有一个重要的定律叫“峰终定律”（Peak-End Rule）。它是指人们对一项事物的体验之后，所能记住的只是在体验过程中**峰值**（最高峰）和**终点**（结束时）的感觉。

*   **峰**：指体验中情绪或感受的最高潮。
*   **终**：指体验结束时的感觉。

例如，你去一家咖啡店，咖啡味道极好是让你最开心的时刻（**峰**）。而结账离开时，店员真诚的微笑和注视会给你留下很好的印象（**终**）。正因为这两种感受很好，即使你觉得这家店有其他不足，下次仍可能光顾。

在见家长活动结束时，我们就要利用“峰终定律”，把最后的气氛调动起来，加深他们对你的好印象。

举个例子，我有一个朋友第一次见家长时，因为见面安排仓促，礼物是临时买的，对方父母对她的第一感觉不是很好。后来见完面离开时，她顺手将门口的垃圾带了出去。就是这个小小的举手之劳，却让男友的父母记在了心底，以后逢人就说她是个不错的姑娘。

这就是在会面结束时，通过一个刺激对方家长重新认识她的行为，让他们猛然发现她的细心，从而开始喜欢她，是“峰终定律”的正确运用。

见家长时，想要顺其自然地将“峰终定律”发挥最大作用，有两个方法。

以下是具体操作建议：

*   **制造引人回味的小细节**：通过行为上的小细节感染他们，从而改变他们对自己的看法。你可以从以下细节入手：
    *   整理好自己住过的房间。
    *   离开时带走家里的垃圾。
    *   换鞋时将鞋柜里的鞋子码整齐。
    *   细心提醒男友检查行李。
*   **通过亲热接触制造情绪峰值**：在告别时，尝试说一些温暖人心的话，例如感谢照顾、提醒他们注意身体、表示会常来看看。此外，你还可以给对方一个亲热的拥抱，或握住对方妈妈的双手亲切慰问几句，以此带动大家分离时不舍的情绪。

## 分别之后：持续维护线上关系

最后，分别以后，你要尝试用持续的问候给对方父母送去温暖，让对方时刻惦记着你的好。

当见家长步入尾声时，不少姑娘会感到松懈。还有的姑娘巴不得快点和男友恢复私人空间，将刚刚与男友家长好不容易建立的感情抛诸脑后。

我之前认识一位姑娘，见完家长后，对方妈妈发消息问她是否安全到家，她却故意装作没看见。还有一位姑娘，对方爸爸加她微信后常发一些养生帖，她不仅不回，还向男友抱怨。如果对方父母已经主动给你发消息示好，你的态度不咸不淡，只会让他们觉得你不重视或不喜欢他们。这对于感情升级以及建立稳固的长期关系是百害而无一利的。

那么，见家长之后，我们该怎么继续搭建准婆媳关系的小船呢？

首先，我们要了解后现代心理学中的一个概念叫做“同在理论”。它是指交往时，各主体之间的情绪会相互影响。参照这个理论，在见家长时，我们的态度往往会带动对方父母的情绪。所以，要想在彼此分开的情况下持续维护好感情，我们在向对方反馈自己的情绪时，一定要采取积极的方式，来引导他们感知到我们的正面情绪。

接下来，我们将通过以下四个步骤讲解如何操作：

1.  **主动添加联系方式**：强调“主动”二字，即自觉表达想要持续建立连接的信号，为以后的友好建交奠定基础。
2.  **第一时间表示感谢**：分别后的第一时间，通过手机向他们表示感谢，包括本次拜访中受到的照顾、吃到的美食、收到的礼物等，让他们体会到你的大方懂事。
3.  **及时回应对方消息**：如果对方父母主动给你发了消息，应该及时回应。切记出现怠慢或冷淡的态度，否则“当面一套背后一套”只会让人对你失去信任。
4.  **逢年过节常联系**：在逢年过节时，也要与对方父母常联系，勤发祝福短信，慰问身体健康，给他们持续的关注，让他们打心眼里更加喜欢你。

## 总结

本节课中，我们一起学习了如何为见家长活动做好收尾工作。想要做好收尾，就要掌握接受见面礼的正确方式，了解推辞的尺度；还要在分别时利用“峰终定律”给对方父母留下难以忘怀的好印象；并且在往后的生活中，能通过线上交流，给家长们送去持续的温暖。做好这些，才能为你们的长期关系垫上坚实的基石。

祝姑娘们的见家长活动，都有一个圆满的结束。
