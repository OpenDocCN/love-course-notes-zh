**5W法则模板如下：**
`Who（谁） + When（什么时间） + Where（什么地点） + What（做什么事） + Why（为什么要做）`

这样指令明确，他就不会只收一部分。

**举例二：接下班**
“亲爱的（Who），晚上8点（When），你来公司正门外的停车场（Where）接我下班（What）好吗？然后我们一起去吃那家很好吃的甜点，我特别想你陪我去（Why）。”
