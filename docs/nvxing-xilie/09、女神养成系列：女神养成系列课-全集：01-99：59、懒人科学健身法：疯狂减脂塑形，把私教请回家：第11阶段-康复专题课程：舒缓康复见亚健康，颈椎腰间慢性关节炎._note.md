# 健身康复专题课程：第11阶段：舒缓康复与亚健康管理

在本节课中，我们将学习针对现代人常见的亚健康问题与慢性劳损的科学康复方法。课程将涵盖颈椎、腰椎、膝关节的常见问题解析，以及高血压、痛风、糖尿病等代谢性疾病的运动管理策略。我们将从基础解剖知识入手，理解问题根源，并学习具体、安全的康复训练动作。

## 脊柱基础知识与常见问题

上一节我们概述了课程内容，本节中我们来看看康复训练的基石——脊柱。理解脊柱的结构是解决许多慢性疼痛的第一步。

脊柱是由不规则骨组成的支柱。形成脊柱的骨头分别由颈椎、胸椎、腰椎和骶椎构成。
我们所俗称的颈椎就是我们的脖子，颈椎总共有7节。
胸椎是由十二节组成。
腰椎是由五节组成。
骶椎是由四节已经合成一块骶骨的。
骶椎的四节骨与骨之间是完全连接连死的，是不能活动的。

我们的身体最先容易出现的问题，几乎都出现在脊椎上，尤其是我们的颈椎和我们的腰椎。
脊椎骨首先它是有个基本的生理曲线，就像一个S型的生理曲线。
你的颈椎从侧面看是做了一个向前的反弓型。
你的胸椎是这样的一个反弓型，是向后开始弓的一个反弓型。
腰椎是向前的一个反弓型。
而骶椎是向后的一个反弓型。
所以我们正常的生理曲线就是一个S型，颈椎向前弓，胸椎向后弓，腰椎向前弓，骶椎向后弓。

而我们最常出现的脊椎上面的问题就是我们的颈椎腰椎发生一些变形。

## 颈椎问题与康复训练

了解了脊柱的基础结构后，本节我们聚焦于最常出问题的部位之一：颈椎。我们将探讨其常见问题及针对性的康复方案。

我们刚刚上面已经讲过了，颈椎由七节组成，它的正常形体是一个向前的反弓。
颈椎怎么样会出现问题呢？
我们平时的生活习惯，比如经常低头玩手机，经常低头写作业，经常低头使我们的颈椎违反了我们的生理曲线。
长期下来，首先引发就是颈椎不良感应。
你的颈椎不良感应是指你在头晕眼花，然后出现恶心等症状。
由于你长期的肌肉习惯，使你的肌肉保持了一直的向后的这种弓，所以你的肌肉的牵引力就形成了后面的张力非常强，而前面的收缩力也非常强，将你的颈椎完全拉成了一个像虾米一样的弓形。
那么在这个时候，除非你通过做运动，做训练，将你的肌肉力量重新把牵引力拉回来，把你从这样的颈椎的形象，直接给拉回一个反弓形，拉回到正常该有的形象。
否则的话，无论你用药物还是牵引，你的颈椎病始终处于一种常年复发的情况。

而颈椎病这里面呢，到了后期你感觉到严重的时候的时候会形成你的手会经常的不自觉的开始发抖。
如果是你的右手发抖，就说明你的右臂从神经被压迫。
如果是你的左手发抖，就说明你的左臂从神经在压迫。
而这些普遍在你的颈四椎到颈七椎之间。
这就是你颈椎最容易犯病的问题。

所以在关于颈椎病的康复里面，在这里给大家讲做颈椎的病的康复，每天要做大概20分钟的训练。
首先就是一个你的正绕感，不过用弹力带，反复的正绕，然后再做些提拉，包括一些拉伸。
而这个训练，每天你都要坚持20分钟，每天不断，一直坚持。
只有你一直坚持下去，你的颈椎问题才能得到些根本的康复。
但是当如果你说你的颈椎病已经非常严重，趁在手麻脚麻的时候，都已经严重到无法做训练的时候，那这时候你只能先通过药物和牵引，让你恢复到基本形态，再做康复训练。
所以在这里面给大家讲，希望大家千万不要等到你的颈椎病完全犯到很严重的时候，才想起来去做锻炼。

以下是教练演示的颈椎康复训练动作示例：

**动作一：靠墙站立**
*   动作要领：整个手臂贴于墙面，保证你的头、肩、臀部贴近墙面，腰部离开墙面。
*   执行：保持姿势，进行深呼吸。

**动作二：肩胛提肌拉伸**
*   动作要领：首先我们把头向左边，然后向右前方45度转，然后向后仰头拉伸。
*   执行：左边和右边各持续3个8拍。

**动作三：斜方肌拉伸**
*   动作要领：将头向左边侧屈，感受拉伸。
*   执行：每边重复3个8拍。

**动作四：颈后肌群拉伸**
*   动作要领：我们将手食指交叉置于脑后，然后轻轻向下施加压力。
*   执行：持续3个8拍。

## 腰椎问题与康复训练

上一节我们学习了颈椎的康复，本节我们来看看另一个疼痛高发区——腰椎。腰椎问题往往与我们的日常坐姿习惯密切相关。

我们最容易出现的问题就是指腰椎病，而这个腰椎病不只仅仅指的是你的5节腰椎，还包括你的骶椎。
腰椎问题常见就常现在你的腰4、腰5和骶1，在我们的医学上面称之为L4到L5之间。
腰椎骨是椎骨当中椎骨形态最大的骨头，也是最容易出现的问题。
腰椎骨的正常情况下，它是形成了一个向前的反弓。
而我们依旧是有很多的生活的不良习惯。
比如说我们坐凳子喜欢翘二郎腿，我们在长期翘二郎腿的过程中，我们的骨盆是倾斜的。
我们坐在凳子上的时候，我们的身体是向前趴的，整个腰椎形成了一个向后的反弓。
这样子的情况下非常容易让我们犯腰椎病。

而这里面最出名的一种病就是叫做腰椎间盘突出。
腰椎间盘突出，它分为四个等级。
第一个等级就是腰椎间盘膨出，它膨出就是你的腰椎只进行了一点点的膨出。
第二个就是腰椎间盘突出，整个的椎间盘已经开始向外突出了。
第三个就是所谓的腰椎间盘破裂，椎间盘外面的这层软骨组织已经完全破裂了。
而第四层就是腰椎间盘脱出，整个髓核突出。
我们所讲的腰椎间盘突出能够在于康复的环节只限于突出和膨出。
如果说在突出膨出之间，你不进行一个坚持的运动康复训练，那么等一旦到了腰椎间盘破裂或者髓核脱出的时候，那么你这辈子都将受到腰椎间盘病的困扰。

而做腰椎间盘病训练的话，每天也是一样，要做20到30分钟，坚持就用这几个最常见的牵引动作，使你的肌肉力量强行将你的腰椎拉回至你的正常的前弓位。
这里重点的就是来康复你的臀大肌和你的腹直肌。
但是有一点在做腰椎间盘突出训练的时候，大家不要去想我自己腰椎间盘是向前突出还是向后突出。
因为只要能够把你的腰椎间盘拉到正常位，你的腰椎间盘突出就是能够康复的。
而且在这里还有一点要注意的就是刚开始做腰椎间盘突出训练的时候，你会非常痛苦。
因为你经常性的翘二郎腿或者保持不良的生活习惯，那肌肉有可能形成了几年甚至十几年的向后牵引力。
而这个时候你要向它进行一个向前的牵引力，会给你造成非常巨大的神经痛苦。
在出现这种疼痛的时候，大家一定要忍住，忍过这段时期，恢复1到2个月以后的情况会越来越好。

以下是腰椎康复训练动作示例：

**动作一：仰卧提膝**
*   动作要领：仰卧，抓住一个固定物，腹部核心收紧，脚尖勾起，向上提膝。
*   执行：一组12个，每组做5组。

**动作二：鸟狗式**
*   动作要领：采取四足支撑位，双手与地面垂直，身体形成一个平行四边形。核心收紧，对侧手和膝同时离地。
*   执行：每组动作做20次，共做3组。

**动作三：猫驼式**
*   动作要领：四足支撑位，从腹部发力身体向上做一个拱起，顶到极限后呼气放松。
*   执行：每个动作做15次，共做3组。

**动作四：臀桥**
*   动作要领：仰卧，足跟和坐骨对齐，从坐骨处发力吐气向上将臀部抬起，脊柱逐节向上抬起，然后放松向下。
*   执行：每个动作做15个，共做3组。

**动作五：仰卧转体**
*   动作要领：仰卧，双手伸直，两腿并拢。髋关节以下贴于地面，头部和上半身向一侧扭转，深呼吸。
*   执行：保持20秒，做3组。

## 膝关节问题与康复

解决了躯干的核心问题，我们来看看支撑我们行动的膝关节。膝关节结构复杂，一旦损伤往往恢复缓慢，因此预防和科学康复至关重要。

在讲膝关节问题之前，首先我们要了解膝关节的构成。
你的膝关节构成就是421。
4是指你的股骨（大腿骨）、胫骨、腓骨和髌骨这4块骨头。
2指的是两个半月板（内侧和外侧）。
1指的就是整个膝关节关节囊。
所以你的膝关节就是由这些东西组成的。
其他的像韧带属于结缔组织。

所以我们在这里面说，如果你的膝关节只要不是出现了我刚说的421的这种骨头上的基础问题，那么膝关节是不会出现太大的病变的。

好，在这里面我今天给大家介绍康复的就是介绍一个关于半月板的损伤处理。
半月板损伤意味着很难自我修复，因为你的膝关节周围血管很少，血液无法进入到膝关节内部对半月板进行营养供应。
半月板常见的损伤分为半月板横裂、半月板纵裂、半月板边角裂，还有一种叫做半月板提篮状撕裂。
当你的半月板出现问题以后，你是没有办法进行大的爆发训练，大的负重训练。
我们所谓的这种康复，也只是帮你恢复到一个好的运动行走，不让你的生活受损的状态。
在这个时候你就要非常注意加强你的肌肉力量，当你一旦出现了半月板的损伤时，忌做过多加强你股四头肌（大腿正面）的训练，因为你的股四头肌肉每增强一点都会让你的半月板承受的压力更大一点。
而这个时候要重点加强你的腿后的腘绳肌群和臀大肌群的训练，这样子可以让你的力量向后，将你的股骨和胫骨之间拉的距离拉开，对保护你的半月板有很好的作用。

以下是膝关节康复训练动作示例：

**动作一：坐姿抬腿**
*   动作要领：坐在地面上，两腿伸直，上半身挺直，勾脚，将腿向上抬起。
*   执行：一侧做20次，共做三组。

**动作二：俯卧后抬腿**
*   动作要领：俯卧，双手向前伸直，两腿并拢。双脚勾起，向上向后提起臀部。
*   执行：一边做20次，共做三组。

**动作三：靠墙静蹲**
*   动作要领：后背靠墙半蹲，腰部保证自然曲度，采取腹式呼吸，使腹部始终处于收紧状态。
*   执行：每组保持20秒，共3组。

**动作四：股四头肌拉伸**
*   动作要领：坐于地面，将一侧腿向后折叠，身体微微向后倾斜，感受大腿前侧拉伸。
*   执行：保持20秒钟，做三组。

**动作五：相扑深蹲**
*   动作要领：两腿打开比肩略宽，脚尖打开45度，手持壶铃，向下蹲时膝盖随脚尖方向。
*   执行：每组10-12次。

**动作六：跪姿侧抬腿**
*   动作要领：四足支撑位，核心收紧，使一侧腿向侧上方抬起。
*   执行：每边做8次，共做5组。

## 代谢性疾病的运动管理

前三个部分我们处理了具体的骨骼肌肉问题，接下来我们看看影响更广泛的代谢性疾病。这些疾病无法通过运动根治，但科学的运动可以极大改善生活质量和控制病情。

代谢性疾病是不可能治疗好的。
代谢性疾病主要指三个，第一就是高血压，第二指的是痛风，第三指的是糖尿病。
这三种代谢性疾病也被称为不死癌症。

**高血压的运动管理：**
高血压是指你的收缩压（高压）和舒张压（低压）不在正常的范围之内。
正常的血压值是80到120mmHg。
一旦你患有高血压，当你要去锻炼之前，切记一点，不要吃完降压药再做训练，这种情况下你无法体会到自己的血压增高，容易造成心脑血管疾病。
关于原发性高血压的康复来说，原发性高血压又可以分为假性高血压和真性高血压。
假性高血压很多时候你并非得的是高血压，你只是这段时间出现血压增高，然后你被误诊了。
若果只是血压暂时升高，比如由肥胖引起，你只要减减肥，你的血压值就会下来。
在做高血压的康复训练的时候，大家要有几个注意点：
第一，不要做过大重量训练（建议不超过自身体重的80%）。
第二，在做训练时候不要憋气。
第三，不要做头低于心脏的运动（如某些瑜伽倒置动作）。
第四，不要做爆发性的训练。
得了高血压以后做的最好的训练就是耐力性有氧的训练，推荐游泳和慢跑。
患有高血压的时候，你要注意降低自己的体重，这点至关重要。

**痛风的运动管理：**
痛风也是代谢性疾病的一种，主要是你的尿酸大量堆积在了关节处。
痛风一旦得了，这辈子治不好的。
但是如果你经常注意，包括你的吃东西注意训练注意的话，你的痛风的复发几率会非常低。
得了痛风以后，首先是要从饮食上面要注意，尽量避免喝酒、吃羊肉和海鲜。
痛风做训练的注意点，千万记住，不要去做过多的跳跃类的训练。
得了痛风以后，尽量去控制体重，因为你的体重每增高一斤对你的膝盖脚踝压力就加深。
做训练还是以最基础的慢跑、有氧游泳和辅助力量性训练为主。
同时记住不要做大的力量性训练，这个力量值的极限就是超过你体重的百分之百。

**糖尿病的运动管理：**
糖尿病分为一型糖尿病和二型糖尿病。
一型糖尿病是天生胰岛素分泌有问题，身体非常消瘦。
二型糖尿病就是传说中的富贵病，可以百分之百的确定来说，二型糖尿病就是祸从口入，95%以上的患者是胖子。
糖尿病是无法根治的，一旦得了以后是真的无法根治的。
一型糖尿病患者在做训练的时候一定要做增肌性的力量训练为主，增强你的肌肉体积。
你的身体的净体重每增加一斤对你的健康程度就会提高一分。
而二型糖尿病患者的训练环节来说，那真的就是减肥，除了减肥别无他用。
二型糖尿病患者，其实不一定你一定要去健身房才能锻炼，主要训练目标就是减肥减脂，推荐的还是以慢跑跑步为主的训练，然后配合其他的力量训练为辅助。
二型糖尿病患者在饮食上一定要注意控制，饮食控制起到70%的作用，训练缓解只能起到30%的作用。

针对糖尿病患者的训练建议：
*   **一型糖尿病患者**：采取增肌的训练模式，主要以胸、背、腿的大重量训练为主（低次数高组数）。
*   **二型糖尿病患者**：采取减脂训练模式，采取一些心肺训练以及力量训练相结合的多次数低组数的训练模式。

例如，坐姿划船动作：坐于器械上，上半身挺直，腹部核心收紧，肩带下沉，发力吐气向后拉，保证小臂垂直于地面。每组做12次，共做三组。
