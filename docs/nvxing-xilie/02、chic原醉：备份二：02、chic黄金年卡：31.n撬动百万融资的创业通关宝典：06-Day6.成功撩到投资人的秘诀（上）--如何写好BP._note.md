# 创业资本论：02：成功撩到投资人的秘诀（上）- 如何写好BP

## 概述

在本节课中，我们将学习如何撰写一份出色的商业计划书。商业计划书是创业者与投资人建立联系的第一步，一份优秀的BP能清晰传达你的商业逻辑与价值，为成功融资奠定基础。

上一节我们介绍了知名投资机构的对接技巧。在融资过程中，让投资人认识你的项目始于一份BP，接着是路演与谈判。成功吸引投资人更多依赖于前期在BP和路演阶段的努力。

我们发现许多创业者存在一个共性问题：不知道如何撰写BP，内容冗长且没有重点。有些创业者甚至仅凭一份Word文档就去寻找投资人，最终往往石沉大海。

即使成功通过BP的初步筛选，在路演阶段又会面临新的挑战——讲故事的能力。许多创业者在首次路演中，由于缺乏公开演讲经验且未提前构思好故事框架，导致路演超时或让投资人感到乏味。

从本节课开始，我们将从BP撰写和故事讲述两方面，分析成功吸引投资人的秘诀。由于内容较多，我们将分三节课详细讲解。

## 商业计划书的重要性

一个好的故事能让你的融资之路事半功倍。例如，当年马云在厕所遇到软银董事长孙正义，双方交谈六分钟后，孙正义决定向阿里巴巴投资2000万美金，如今这笔投资回报已高达700亿美金。

讲故事非常考验创始人的功力。一个好故事能将创始人与投资人的立场从对立转化为统一。我们今天分享的重点并非口头讲述技巧，而是讲故事的前提：拥有一份扎实可靠的商业计划书。

经历过或即将进行融资的创业者，都对商业计划书有所了解。起初可能有人认为这不过是制作一份PPT，但其中蕴含很大的学问。

融资计划书是留给投资人的第一印象。在短短几页PPT中，必须解释清楚商业逻辑，精准表达公司的商业模式，并向投资人展现公司的真正价值。

真正擅长撰写BP的，往往是那些总结过多次失败经验的连续创业者。BP主要分为两种：一种是发给投资人审阅的，另一种是用于路演讲解的。

这两者有何区别？前者需要让投资人在短时间内了解项目概况，因此需简要写出产品、模型及相关经营数据。就像企业HR需要浏览大量简历一样，投资人的邮箱也常堆满来自各路创业者和投资顾问的项目BP。他们在查阅时，很少会从头到尾仔细研究。

尤其对行业有一定了解的投资人，通常会直奔他们最关注的几个要点，包括我们之前反复提到的数据、产品和模型等。

后者则要求更加简洁。除了融资目的，它通常还带有一定的宣传和科普作用，旨在让与会者直接了解你的产品与行业前景，并对你所创业的领域产生更深刻的认知和兴趣。

## BP的核心组成部分

我们所提到的商业计划书主要包含以下几部分内容：
*   产品或业务模型
*   商业模式
*   竞品分析
*   团队介绍与行业概况分析
*   运营和财务数据现状
*   融资规划

在详细拆解BP的撰写步骤之前，我们先讲解几点注意事项。

以下是撰写BP时的五项核心注意事项：
1.  **篇幅控制**：BP整体长度不宜过长，最好控制在10页之内。
2.  **阶段侧重**：不同融资阶段的BP内容要求不同。对于早期项目，团队是最大卖点。核心成员的行业经历和头衔是BP中的亮点。若核心人员有知名公司从业经验或是行业专家，能为公司提供无形的信任背书。
3.  **表达精炼**：PPT中尽量避免堆砌大量文字，应通过提炼关键词进行描述。
4.  **善用数据**：投资人每天需阅读大量BP，数字和图表能更直观、高效地展现项目的亮点与竞争力。
5.  **实事求是**：不要夸大其词，应真实反映公司现状。

下面，我们来拆解BP中的六个重要撰写步骤。

## 步骤一：构建产品模型

在BP的开篇，需要解释清楚产品的定位及其解决的痛点。这是投资人最关心的部分之一，关系到你所切入的市场和人群，以及该市场是否具备刚需属性和变现价值。

例如，如果我们计划开发一款社交类产品，就需在BP中清晰描述目标市场。假设我们的目标是单身男女青年。

接下来，我们需要针对这一群体的痛点和需求做出解释。单身男女青年的普遍状态是感到孤独，并希望寻找新的社交途径。

最后，在产品定位模块，我们需要综合考量目标市场与竞争对手的产品定位，找出自身的独特亮点。以微信和QQ为代表的熟人社交软件，与以陌陌、探探为代表的陌生人交友软件，其产品目标市场需求存在差异。

如果我们作为创业者进入该领域，从社交产品功能本身挖掘差异化可能较难。那么我们的差异化在哪里？我们可以参考近期的一些社交产品，例如今日头条的多闪，其与其他社交APP的差异化在于主打视频社交。从用户来源看，其用户大多从抖音平台导流而来。

## 步骤二：规划商业模式

有了基础的产品构想后，接下来至关重要的一步是规划公司的商业模式。投资人投资项目的根本目的是获取财富增值，因此，公司的盈利模式是需要重点考量的指标。

前面提到，早期项目的商业模式虽不一定非常具体，但必须具备足够大的延展空间。此时，投资人通常最关注两方面：一方面是初始数据规模，主要是需求端产生的需求是否足够刚性、高频且覆盖范围广；另一方面是潜在的变现属性，例如产品本身是否具有交易价值，用户群体的付费能力、复购率和转化率如何，能否产生持续现金流等。

我们需要向投资人说明项目的核心业务流程是怎样的，是可以短期实现盈利，还是需要长期积累后才能变现。

如果我们开发一款社交类APP，这属于依靠流量变现的行业。因此，通常需要经过长期的数据积累与沉淀后，才能转化为实际收入。

此外，之前提到的核心技术或独特优势也应在此部分呈现，让投资人对我们的差异化竞争力一目了然。

## 总结

本节课我们一起学习了商业计划书的重要性、核心组成部分、撰写注意事项，并详细拆解了前两个关键步骤：**产品模型的构建**与**商业模式的规划**。一份优秀的BP需要精炼、直观、有重点，并能清晰展现项目的价值与潜力。
