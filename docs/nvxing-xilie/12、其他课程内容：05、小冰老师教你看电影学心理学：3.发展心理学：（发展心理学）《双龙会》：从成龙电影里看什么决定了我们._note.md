# 小冰老师教你看电影学心理学：05：发展心理学：《双龙会》：从成龙电影看什么决定了我们

## 概述

在本节课中，我们将通过电影《双龙会》来探讨发展心理学中的一个核心议题：究竟是什么因素塑造了我们？是先天遗传，还是后天环境？我们将结合电影情节与心理学研究，为你解析“遗传决定论”这一重要观点。

## 电影情节简介

上一节我们介绍了本系列课程的目标，本节中我们来看看第一部电影《双龙会》。

这部电影由成龙主演，他一人分饰两个角色。这两个角色是一对双胞胎兄弟，在刚出生时因匪徒劫持而分离。

弟弟名叫“玩命”，被一名舞女捡到并抚养长大。他长期生活在社会底层。为了帮助朋友追求女孩，他结识了由张曼玉扮演的舞女芭芭拉，并因此得罪了黑社会。

哥哥名叫“马友”，被一对富裕的夫妇收养并带到美国。他接受了精英教育，长大后成为一名著名的音乐家。他回到香港举办音乐会时，结识了由利智扮演的世伯之女唐心。

在阴差阳错中，兄弟二人互换了身份，进入了对方的生活。大家开始分不清谁是马友，谁是玩命，因此闹出许多笑话。

芭芭拉和唐心这两位女主角也在这一过程中心思发生了变化：哥哥的女朋友爱上了弟弟，弟弟的女朋友则爱上了哥哥。

电影的高潮发生在音乐会上：玩命通过胡乱指挥大获成功。而马友则在黑帮的胁迫下，开车去拯救玩命的好友泰山。最终，兄弟二人联手击败了强敌。

## 电影与心理学的联系：双胞胎研究

为什么要推荐这部电影呢？因为它关联到生活中一个神奇的现象：双胞胎。

现实中，许多双胞胎确实像电影里演的一样，一出生就被分开了。许多心理学家对此现象感到好奇。他们想知道，如果双胞胎由不同的养父母抚养，在不同的文化和教育环境下长大，那么他们在成年后会有多少相似之处。

心理学家托马斯·布沙尔进行了一项长达20年的双胞胎研究。其中最著名的案例是“双胞胎吉姆”的故事。

这对双胞胎都叫吉姆，有趣的是，这个名字是他们的养父母分别取的。他们在出生几周后就被分开抚养。当他们近40岁在佛罗里达海滩偶然重逢时，发现彼此身上有许多行为完全一致。

以下是他们在测试和生活中展现出的惊人相似性：

*   **心理测试**：他们在人格、智力、态度和兴趣测试中的得分几乎一致。
*   **健康状况**：两人都有轻度高血压，都做过输精管切除手术，并且都有偏头痛。
*   **生活习惯**：他们喜欢同一品牌的香烟和啤酒，购买的汽车也是同一品牌。
*   **婚姻家庭**：两人都结过两次婚。第一任妻子都叫琳达，第二任妻子都叫贝蒂。他们都有很多孩子，其中一个孩子的名字都叫詹姆斯·艾伦。他们养的宠物狗名字也相同。
*   **职业**：两人都是社区警官。

这个案例在心理学上非常著名。心理学家托马斯·布沙尔据此坚定地认为，双胞胎80%的行为是由先天决定的。也就是说，我们大多数行为、性格、喜好，甚至对伴侣的选择，在成为受精卵的那一刻就已被决定。

这听起来有些宿命论的感觉，仿佛“龙生龙，凤生凤”。仔细想想，生活中似乎也能看到类似现象，例如许多优秀人物往往出自同一家族。

## 核心理论：遗传决定论

开创“遗传决定论”这一学派观点的，是著名心理学家弗朗西斯·高尔顿。他的表兄正是大名鼎鼎的查尔斯·达尔文。高尔顿深受达尔文进化论思想影响，并将其引入对人类的研究。

他坚定地认为：**一两的遗传胜过一吨的教育**。

这一派观点认为，人的能力主要源自遗传。高尔顿提出了具体的遗传比例公式：

**人的特质 = 1/2（来自父母） + 1/4（来自祖父母） + 1/16（来自曾祖父母） + …**

这个观点在19世纪被提出，高尔顿也因此成为近代优生学的开创者。发展到现代，许多基因检测技术也为这一理论提供了不少证据。

这个理论带有一些宿命论的色彩，听起来可能让人感到悲观，仿佛“总统的儿子注定当总统，农民的女儿注定是村姑”。虽然我个人并不完全认同，但它作为心理学史上一个非常重要的观点，有必要分享给大家了解。

## 总结

本节课中，我们一起学习了发展心理学中关于“什么塑造了我们”的探讨。

我们通过电影《双龙会》的情节，引出了现实中双胞胎研究的经典案例，并详细介绍了由此衍生的**遗传决定论**核心观点。该理论强调先天遗传在决定个人行为、性格和命运中的主导作用。

在心理学上，关于这个问题的答案目前尚无定论，就像一场辩论，存在好几派不同的观点。今天我们介绍的遗传决定论，是最早发展出来的理论之一。
