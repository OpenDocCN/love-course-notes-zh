# 女神养成系列：16：上肢燃脂计划（教学版）💪

在本节课中，我们将学习一套专门针对上肢和核心肌群的燃脂训练计划。这套计划包含四个高效动作，旨在燃烧脂肪的同时，增强腰腹核心力量，帮助改善久坐带来的腰部不适。

## 动作一：全程平板支撑

上一节我们介绍了课程目标，本节中我们来看看第一个核心动作——全程平板支撑。这个动作能有效训练背部及腹部周边的核心力量，帮助对抗久坐导致的腰肌劳损，并增强腰部周围的肌肉耐力。

以下是动作要领与步骤：

1.  **起始姿势**：俯卧在垫子上，双脚并拢，脚尖并在一起。双手打开，与地面保持平行。
2.  **手部支撑**：双手手指指节朝向正前方支撑地面，以保护肩关节。避免手肘弯曲角度过大，以免挤压肱骨位置，损伤肩关节。
3.  **身体抬起**：吸气准备，吐气时将身体支撑起来。
4.  **身体姿态**：头、背、腰、臀、脚保持在一条直线上。腰部不能下沉，臀部也不能撅起过高。身体保持平板式。
5.  **细节调整**：头部尽量向前延长，肩膀下沉远离耳朵。大臂与地面垂直，小臂指向前方。用整个小臂发力支撑，而非肘关节。双手朝向前方，不要相对重叠放在胸前。
6.  **发力要点**：双腿用力向中间夹紧。臀部同样用力夹紧。脚尖并拢。保持均匀呼吸，身体尽量稳定。
7.  **保持时间**：保持该姿势30秒，然后休息。

**核心公式**：`身体姿态 = 头 + 背 + 腰 + 臀 + 脚` 成一条直线。

这个动作比之前学过的跪姿平板支撑难度更高。在练习时，你会感觉到腹部、臀部、腰部及核心肌群都在发力，以对抗地心引力。对于久坐人群，多加练习可预防腰肌劳损，严重者还可能预防腰椎间盘突出。

**重要提示**：
*   做静态收缩动作时不能憋气，需保持匀速呼吸。憋气可能导致血管压力增大、大脑缺氧，引起头晕。
*   核心收紧的同时保持呼吸。
*   血压偏高的伙伴可采用跪姿的初级训练方式，并时刻关注身体感受，量力而行，感到不适立即停止。

## 动作二：俯身划船俯卧撑

掌握了静态的核心支撑后，本节我们来学习一个动态的复合动作——俯身划船俯卧撑。这个动作名称听起来很酷炫，它能同时锻炼到胸部、肩部和背部等多个肌群。

以下是动作分解：

1.  **起始姿势**：四点支撑，保证头、背、腰在一条直线上，双脚支撑起身体。
2.  **基础支撑**：大臂与地面保持垂直，肩关节主动发力对抗引力，不要被动支撑。头部向前延长。
3.  **划船动作**：先将外侧手臂抬起，做一个划船动作。注意肘关节贴紧身体向上滑动。
4.  **俯卧撑动作**：手臂支撑回地面后，完成一次标准俯卧撑。
5.  **交替进行**：换另一侧手臂进行划船，然后再次完成俯卧撑，这算一次完整动作。
6.  **降阶选择**：如果感觉困难，可选择跪姿完成。在跪姿基础上，抬起一侧手臂做划船，然后完成跪姿俯卧撑，再换边。

**代码描述动作流程**：
```plaintext
起始（四点支撑） -> 右臂划船 -> 俯卧撑 -> 左臂划船 -> 俯卧撑 -> 循环
```

这个动作对女性颇具挑战性，建议初学者从跪姿开始。随着肌肉力量和耐力的提升，再尝试完成全程动作。

## 动作三：俯身两头起

接下来，我们学习一个针对身体后链的练习——俯身两头起。这个动作能有效刺激下背部（竖脊肌）和上臀部肌肉，这两处是支撑腰椎、承担上半身重量的关键肌群。

以下是执行步骤：

1.  **起始姿势**：俯卧在垫子上，双手向前展开。
2.  **发力抬起**：吸气准备，吐气时，通过腰椎和臀部的发力，让胸部和膝盖同时离开垫子。
3.  **控制还原**：在顶峰稍作停留后，有控制地慢慢还原到起始位置。

**关键要点**：做此动作时，腹部千万不可放松。必须让腹部和背部同时收紧，协同发力来完成动作，这样对腰椎最安全。

**重要安全提示**：如果你有严重的腰椎间盘突出，建议跳过此动作。为确保安全，当腰椎疼痛严重时，不要完成这个动作，可以直接进行下一个练习。

完成这个动作后，可以做一个婴儿式来放松和拉伸下背部，会感觉非常舒适。

## 动作四：腹部弓步跑

最后，我们来学习今天最后一个复合动作——腹部弓步跑。这个动作能高效启动腹部和下背的核心肌群，同时还能锻炼到腹内外斜肌及侧腰。

以下是动作详解：

1.  **起始姿势**：以平板支撑姿势开始，大臂与地面垂直，肩膀下沉，头部向前延长，核心收紧，身体保持平板式，双腿支撑地面。
2.  **侧向提膝**：利用腰腹侧面的肌肉发力，带动一条腿从身体侧方向前上方延长，尝试用膝盖去触碰同侧的肘关节。
3.  **交替进行**：完成一侧后，换另一侧腿进行同样的动作。

**动作感受**：在做这个动作时，肌肉可能会发抖。这是肌肉接近力竭时向大脑发出的信号，表明肌肉正在得到有效训练。如果感觉非常累，可以提前休息。

今天的四个动作组合在一起，能全面锻炼到腹部、下背部、侧腹以及整个核心肌群（包括腹横肌）。激活腰部周围的这一圈肌肉，使其更有力量，能更好地承担上半身对腰椎的压力，从而对腰椎起到保护作用，预防久坐带来的腰肌劳损或腰椎间盘突出。
