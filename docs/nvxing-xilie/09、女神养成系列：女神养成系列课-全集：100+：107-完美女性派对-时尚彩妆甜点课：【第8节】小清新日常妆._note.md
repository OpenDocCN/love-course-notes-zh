# 女神养成系列：09：小清新日常妆

## 概述
在本节课中，我们将学习如何完成一个完整的“小清新日常妆”。这个妆容包含底妆、眉毛、眼妆、修容、腮红和口红等所有步骤，适合学生、上班族等日常场合，旨在提升气色和整体形象，是所有妆容的基础。

---

## 第一部分：日常妆的认识

上一节我们介绍了课程的整体安排，本节中我们来看看什么是“日常妆”。

日常妆是适合任何人的基本妆容。它比“伪素颜妆”略浓一些，通过粉底、眉毛、眼线、口红、腮红和修容等步骤，改善肤色和脸部状态，达到“锦上添花”的效果。化妆的终极目标是通过光影和色彩，让人产生视觉上的改变。

日常妆是所有妆容（如欧美妆、韩妆等）的基础。掌握好日常妆的技法，是学习其他复杂妆容的第一步。

日常妆适合上学、上班、面试、游玩等多种场合。它给人一种干净、精致的感觉，容易被大众接受，并能有效提升个人气质。

## 第二部分：日常妆的步骤

认识了日常妆的概念后，接下来我们详细拆解画一个完整日常妆的具体步骤。

以下是画一个日常妆的标准流程：

1.  **观察**
    *   **观察肤质**：分为油性、干性、敏感性和混合性肌肤。不同肤质需选用不同的底妆产品。
        *   **油性肌肤**：适合使用粉底霜。
        *   **干性肌肤**：妆前需做好保湿，适合使用粉底液。
        *   **敏感性肌肤**：应选择抗敏感配方的产品。
        *   **混合性肌肤**：T区易出油，脸颊偏干，需分区护理。
    *   **观察脸型**：分为标准脸、梨形脸、圆脸、方脸、长脸、菱形脸等。根据脸型设计眉形、修容和腮红的位置。

2.  **打底**
    *   目标是底妆轻薄、色号自然。
    *   **先遮瑕**：针对不同问题使用不同颜色的遮瑕产品。
        *   **黑眼圈**：使用偏橘色的遮瑕膏。`公式：黑眼圈 -> 橘色遮瑕`
        *   **痘痘、痘印**：使用接近肤色的米色遮瑕膏。
        *   **红血丝**：使用绿色遮瑕膏中和红色。`公式：红血丝 -> 绿色遮瑕`
        *   **局部暗黄（如嘴角）**：使用紫色遮瑕膏提亮。
    *   **后上粉底**：将粉底均匀拍打在面部，并轻微带过脖子，使妆效更自然服帖。

3.  **眉毛**
    *   **颜色**：眉笔颜色需与发色接近。黑发选深棕或灰色，浅发选浅棕或深棕色。`原则：眉色 ≈ 发色`
    *   **画法**：眉毛应“淡而有型”。先勾勒眉毛底线并画平，再填补眉毛空隙，最后用眉刷晕染自然。画完后可用遮瑕膏修饰眉毛边缘，使眉形更精致。

4.  **眼妆**
    *   **眼影**：选择大地色系。用最浅色铺满整个眼窝打底，再用深一阶的颜色从睫毛根部向上晕染，范围不超过双眼皮褶皱。下眼影画在眼下凹陷处。
    *   **眼线**：要求自然、流畅。先画内眼线（睫毛根部），再沿睫毛根部画一条极细的外眼线，眼尾轻微拉长即可。
    *   **睫毛**：夹翘睫毛后，刷上睫毛膏，避免出现“苍蝇腿”。下睫毛也需轻微带过。

5.  **唇颊（口红与腮红）**
    *   **腮红**：颜色不宜过重。用刷子斜向扫在颧骨位置，少量多次。刷完后可蘸取微量粉底轻拍在腮红上，制造“由内而外”透出的好气色。
    *   **口红**：日常妆适合“裸色”系口红，即接近自身唇色、能提升气色的颜色。先涂在嘴唇中央，再用手指或棉签向外晕染开。

## 第三部分：妆容实操演示

了解了理论步骤后，本节我们将通过真人模特演示，将上述步骤付诸实践。

（演示过程概要：老师为一位18岁的模特现场上妆，逐步展示了从遮瑕、打底、画眉、眼妆、修容到唇颊的完整过程，并穿插讲解了工具选择（如一次性粉扑、NYX六色遮瑕盘、深蓝眉笔、kiss me眼线笔等）和实用技巧（如遮盖眼镜压痕、晕染手法、定妆重点等）。）

**实操核心要点回顾：**
*   **底妆**：遮瑕要精准，粉底要轻薄，并用散粉重点定妆眼周等易出油部位。
*   **眉毛**：遵循“先底线，后填空，再晕染”的顺序，并用遮瑕膏精修边缘。
*   **眼妆**：使用多把刷子分层晕染眼影，眼线要细且自然。
*   **修容**：鼻影从眼窝开始晕染，不要画成两条直线。面部修容是一个“面”，而不是一条“线”，需充分晕染开。
*   **整体**：所有步骤都遵循“少量多次”的原则，最终妆效清新自然，提升了模特的整体气色和精致度。
