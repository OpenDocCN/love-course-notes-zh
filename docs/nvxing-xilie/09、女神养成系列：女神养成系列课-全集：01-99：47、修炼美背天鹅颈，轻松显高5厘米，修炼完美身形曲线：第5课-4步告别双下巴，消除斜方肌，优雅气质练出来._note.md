# 女神养成系列：第5课：4步告别双下巴，消除斜方肌，优雅气质练出来 💃

在本节课中，我们将要学习如何通过简单的训练动作，有效改善双下巴和斜方肌问题，从而塑造更紧致、挺拔的颈部与肩部线条，提升整体气质。

## 概述：双下巴与斜方肌的影响

双下巴和斜方肌问题会直接影响上半身的整体气质，也是决定穿衣是否好看的关键因素。本节我们将了解这两者形成的原因，并学习改善方法，让颈部看起来更加紧致、挺拔、细长。

## 第一部分：认识与改善双下巴

上一节我们介绍了双下巴和斜方肌对气质的影响，本节中我们来看看如何具体改善双下巴问题。首先需要了解其成因。

双下巴，医学上称为下颌脂肪袋，是由于下颌脂肪组织堆积过多，加上皮肤老化松弛及重力作用导致下垂形成的。日常不良体态，如经常低头玩手机、驼背，会加速其形成。

为了消除双下巴，必须先了解其形成原因，才能找到有效的解决办法。

以下是双下巴的三个主要形成原因：

1.  **脂肪堆积因素**：身体脂肪过多时，颈部下颌部位同样会堆积脂肪，形成视觉上的双下巴。公式可理解为：`身体总脂肪量高 → 局部脂肪堆积概率增加 → 双下巴形成`。
2.  **皮肤松弛因素**：皮肤中的胶原蛋白和弹力纤维蛋白随年龄增长而流失，导致皮肤松弛，支撑力下降，下巴脂肪随之下垂。
3.  **咀嚼习惯差**：咀嚼次数过少，面部和下颌肌肉得不到有效锻炼，会变得松弛缺乏弹性，长期只吃软食容易引起双下巴。

了解了原因后，我们可以通过针对性的训练来改善。以下是三个消除双下巴的练习动作。

### 动作一：下颚颈练习

此动作旨在锻炼下颌及颈部肌肉，增强紧致度。

1.  肩膀放平，脖子伸长，头部平行，目视前方。
2.  将一只手放在下颌位置。
3.  抬头向上看天花板，同时用手轻轻抵住下颌，感受颈部拉伸。
4.  头部回正，目视前方，换另一只手进行。
5.  左右交替进行，每次做4个八拍。

**练习建议**：每天练习4个八拍。可在办公室、地板上或椅子上完成。

### 动作二：长颈鹿亲吻练习

此动作能有效拉伸颈部前侧肌肉，强化下颌线条。

1.  侧身站立，双手叉腰，肩膀下沉。
2.  将脖子向斜前方最大限度地伸出，同时嘴唇用力向前撅起。
3.  在最远处保持8秒，感受下颌的酸胀感。
4.  缓慢收回，重复进行。
5.  注意保持脖子向上挺拔延伸，而非向下缩。

**练习建议**：每天可练习20次。

### 动作三：眼镜蛇舌练习

此动作通过舌部运动深度激活下颌肌肉群。

1.  将一根手指轻轻放在下颌处。
2.  用力将舌头向最远的方向伸出。
3.  伸出时，应感觉到下颌肌肉完全收紧。
4.  收回舌头，重复进行。

**练习建议**：每天练习20次。

运动是紧致肌肤、消除赘肉的关键法则，坚持练习能使皮肤更加紧致Q弹。

## 第二部分：认识与消除斜方肌

上一部分我们学习了消除双下巴的方法，接下来我们看看如何改善斜方肌问题。优美的肩部线条能让人看起来更显瘦、优雅。

斜方肌是位于上背及中背的表层肌肉，连接肩颈部。过于发达的斜方肌会让人显得脖子短粗、脸大、气质下沉。

想要消除粗壮的斜方肌，首先要了解其成因。大部分人的斜方肌问题并非源于高强度健身，而是不良生活习惯所致。

以下是导致斜方肌粗大的常见原因：

*   **不良体态**：如走路时弯腰、驼背、含胸，会导致斜方肌过度紧张。
*   **重力负担**：斜方肌需要持续对抗头部、肩部和上肢的重力，在久坐、伏案工作、长时间使用电子设备时负担尤重。

因此，除了针对性锻炼，在工作间隙进行肌肉舒缓也非常重要。

### 动作四：芭蕾手位练习（消除斜方肌）

此练习借鉴芭蕾基础，旨在提升肩部姿态，放松并下沉斜方肌。

1.  手指张开，大拇指轻触中指第三指节，小指并拢。
2.  双手轻轻置于腿部前方，保持肩膀下沉。
3.  缓慢将手抬至芭蕾二位手位置（手臂呈弧形于身体前方），全程保持斜方肌下沉，脖子像长颈鹿一样向上伸长。
4.  打开至七位手（手臂向身体两侧打开），然后呼吸收回。
5.  跟随节奏重复练习，注意视线随手动，肩膀始终向下沉。

**练习要点**：练习过程中，注意力集中在让斜方肌向下沉、放松，避免耸肩。
