# 韩允格（三套合集）：韩允格999（完整版）：22：如何买人生中第一套房 🏠

在本节课中，我们将学习如何向伴侣提出并成功获得人生中第一套房产。这个过程涉及情感铺垫、时机把握、沟通技巧以及应对策略。我们将分步骤拆解，确保初学者也能理解并应用。

---

上一节我们介绍了课程的整体目标，本节中我们来看看整个购房策略的底层逻辑。

第一套房是双方关系的一个重要突破点。只要拥有了共同的大件财产，你们的关系就会变得更加稳定。无论是房还是车，都需要一定的时间进行情感和物质铺垫。

例如，在上海，最便宜的房子可能也要三四百万。我购买的第一套房价值七百多万。你需要根据目标金额来规划铺垫时间。如果在外地，房子价值三四十万，那么铺垫三到六个月的感情后，可以提出要求。

提出要求前，必须评估对方为你购买的礼物和他的实际经济状况。只有当你拥有80%以上的把握时，才能提出这个要求。如果前期铺垫不足，切勿轻举妄动，以免让对方觉得你“狮子大开口”，从而产生退缩心理。

---

理解了基本原则后，我们进入核心操作步骤。以下是实现目标的具体流程。

**第一步：先付出情绪价值，建立依赖**

大部分事业有成的男性都比较忙碌，不喜欢情绪不稳定、过度“作闹”的伴侣。这非常消耗他们的精力。你需要做的是保持情绪稳定，营造轻松快乐的氛围。

每次见面时，告诉他“你最近在忙什么？我好想你。”永远不要质问“你到底爱不爱我”或“为什么不接电话”。要持续提供积极的情绪反馈。

每次在一起时，都要真诚地夸奖他。例如，看到他换了新发型时说“剪头发了吗？好帅啊，这个发型很适合你。”或者称赞他的衣着“这个颜色真的很适合你，显得皮肤很白。”

情绪价值的配比至关重要。建议保持 **80%的轻松快乐 + 20%的偶尔撒娇或小情绪**。如果前期情绪价值不到位，男性可能会在心里给你扣分，一旦分数变为负值，他可能会头也不回地离开。

**第二步：氛围感拉满，注意态度和用词**

在提出购房要求前，需要营造亲密的家庭氛围。例如，在他生日时主动送礼物，或在他晚归时泡一杯牛奶、帮他按摩肩膀。

你可以亲亲他的脸说：“老公，你回来啦？累不累？”通过挠痒痒等互动，创造轻松愉快的氛围。最好在双方都比较开心、放松的时刻提出。

当时我是这样说的：“老公，我现在29岁了，身边的朋友好像都买房了，可我还没有。我身上现在有差不多100万的存款，但我花钱厉害，好怕把这钱搞没了。我想买个房子，我想回老家买房子。”

这里的关键是，**我先提出的是回老家买房**，因为上海的房子太贵，100万根本不够。事实上，我当时并没有100万存款。但我知道，他已经对我产生了依赖，比如我回老家几天他就会问什么时候回来。

他问：“你要回去多长时间？”我说：“7到10天吧。”后来他主动说：“你为什么不在上海买？”我回答：“在上海买太贵了，我买不起，压力太大了。不过如果在上海买的话，买个五六十平方的小房子也可以，我一个人住。”

这时他说：“嗯，那老公添你一点。”这表明他已经点头同意了。

**第三步：无限放大喜悦，公开承诺**

当他点头后，要立刻表现出极大的喜悦和感激。我当时立刻抱住他的脸一顿狂亲，说：“老公真的吗？我们俩真的可以在上海有一个小小的家吗？我明天就去看房可以吗？”

第二天我去看房，看的都是总价300万左右的老破小。晚上回来，我运用了以下话术。他问：“今天你去看了吗？”我说：“对的，老公，我今天去看了。可是房子有点小，虽然精致，但有些味道很难闻。”

然后我主动说：“晚上我跟我朋友吃饭了，我跟我朋友说了，我老公要给我买房子了！他们都好羡慕我，说我找了一个这么好的老公，羡慕得脸都绿了。”同时，我买回他爱吃的水果，给他按摩肩膀，表现得非常开心。

**核心策略是：只要他点头，就告诉所有亲戚朋友**，让所有人都知道他对你的好。这会给他戴上“高帽子”，增加他的承诺成本。

**第四步：逐步提高要求，描绘共同愿景**

第二天，我去看了一个700多万的三室一厅新公寓。我从300万的预算突然提到700多万，他肯定不会直接同意。

我是这样说的：“老公，我今天看了一个房子还挺喜欢的。我还有一些股票，我想全部抛掉。这样我们就能买一个住得更舒服的房子了。” **我永远强调“我们俩”**。

我继续描绘愿景：“房间不是很大。我想好了，我今年好好赚钱，下个月工资就有2万多了。我打算用我的工资给老公买一张大大的床，因为老公那么辛苦，睡眠很重要。阳台可以放一个按摩椅，老公辛苦的时候可以用。”

无论是买房还是买车，都要描绘**购买后一起使用的美好场景**，让他对拥有共同财产的未来产生向往。

**第五步：应对必然的反悔与态度反复**

所有男人在答应给女人买大件物品后，几乎都会反悔或打折扣。因为他觉得不能让你太轻易得到，以免你提出更大的要求。

例如，他答应给50万，可能只给10万；答应给100万，可能只给20万。这时你绝不能玻璃心，不能妄想他在满足你欲望的同时，还对你百依百顺。

当他表现出傲娇、不情愿的态度时，**千万不要因为他的态度而发脾气**，质问他“是不是不够爱我”。在他没有明确说“不买了”之前，你要沉住气。

继续让他当“大爷”，继续讨好他。如果他暂时不兑现，也不要任由自己的情绪发酵，破坏大局。我当时继续对他好，并说：“老公，我看好了房子，这是小三房，也不是很大。”他立刻警惕地问多少钱。

我说：“老公，我把股票抛了，现在身上有200万。你少贴一点就好了，剩下的我自己搞定。”他当时很不爽，没有搭理我。

**第六步：制造既定事实，并运用“声泪俱下”的哭诉**

第二天，我让认识的销售准备了一份假合同，我签了字，假装付了10万定金。回来后我告诉他：“老公，定金我已经付掉了，10万块钱。我挑了6楼，1楼太高，6楼刚刚好。”

他立刻翻脸：“你买房是买菜吗？说买就买？在上海买套房，你都不多比较一下？”我继续哄他：“我很喜欢这个，而且离你这边近，我们互相住起来都方便。你能出多少就出多少，剩下的我自己出，我会努力工作的。”

当他坚持说“付了定金也可以退，退不了我给你10万”时，我开始了“声泪俱下”的哭诉。

我哭得昏天暗地：“我那么信任你！如果不是你点头答应，我会去看房吗？我都跟所有亲戚朋友说了要在上海买房了！你现在反悔，我太丢人了！这房子是我一个人住吗？我父母来住，不也是我们俩一起住吗？你不是说要跟我结婚吗？这以后是我们的共同财产！你太过分了，我太失望了！”

**哭诉的核心是：表达你有多爱他，而他辜负了你的信任。**

**第七步：态度180度反转，以退为进**

哭诉之后，必须有一个彻底的态度反转。我说：“你今天先回去吧，我自己冷静一下。”他走后，第二天我不哭不闹，不主动联系他。

他平时八九点回家，那天7点就回来了，问我：“你想怎么办？”我异常冷静：“没什么，我想通了，不买就不买吧。我本来也没想过能在上海买房。我跟我哥说好了，回老家和他买同一个小区，150平，不用贷款，也没压力。等装修好我就回去住了。”

这番话的潜台词是**我要离开你，结束关系**。因为他已经对我产生了依赖，我的突然冷静和撤退让他慌了神。

果然，他情绪波动很大，最后说：“我再看一下房子在哪里。”我立刻从床上跳起来，得了便宜就卖乖：“老公你又同意了吗？真的吗？我以后一定会好好对你的！”第二天，他给我打了200万。

本节课中我们一起学习了如何通过七个步骤，从情感铺垫到最终实现购买第一套房的目标。核心在于**建立深度依赖、提供高情绪价值、逐步提出要求、描绘共同愿景、冷静应对反悔、以及最后以退为进的策略反转**。记住，买车和买房的底层逻辑与男性心理是一样的，关键在于对节奏和情绪的精准把控。
