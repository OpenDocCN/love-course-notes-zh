# 妈妈塑形训练营：008：平坦小腹“马甲线”第二课

## 概述

在本节课中，我们将继续腹部塑形训练。上一节我们重点进行了腹部深层核心的激活，帮助大家找到正确的发力模式。本节中，我们将加入更多动态练习，目标是打造平坦小腹和塑造马甲线。所有动作均基于普拉提原理，只需一张瑜伽垫即可在家练习。关键在于掌握动作要点，并通过循序渐进的练习，让身体恢复运动记忆。

## 动作一：行军踏步

行军踏步是普拉提中的经典动作，能有效稳定骨盆和核心区域，帮助找到内核心与外核心协同发力的感觉，从而更好地收紧腹部。

以下是动作步骤：

1.  **准备姿势**：屈膝仰卧在瑜伽垫上，确保后脑勺完全贴地。双手掌心向下，置于身体两侧。检查身体位置：髋、膝、脚踝应在一条直线上，双膝间距约一个拳头。
2.  **调整姿态**：想象头顶有绳子向上牵引，延展脊柱。双肩放松下沉，贴紧地面。微收下巴，避免颈部受压。将一只手平放于腰骶下方，调整骨盆至中立位（手掌刚好能放入，不紧不松）。
3.  **启动核心**：运用横向呼吸法。吸气时，气息充满肋骨两侧。呼气时，先将肋骨降下，用嘴缓缓吐气。接着，找到提肛和轻微憋尿的感觉（收紧骨盆底肌），然后收腰收腹。注意收腹时，骨盆保持中立，不要后倾。目标是找到**肚脐拉向脊柱**的感觉。
4.  **执行动作**：
    *   吸气，缓慢抬起左脚脚跟，脚尖点地。
    *   呼气，缓慢将左腿抬至“水平桌面位”（大腿与地面垂直，小腿与地面平行），脚尖轻绷。
    *   吸气，抬起右脚脚跟。
    *   呼气，缓慢将右腿抬至“水平桌面位”。抬腿时，始终保持核心收紧，避免腰部拱起。
5.  **动态练习**：双手保持掌心向下。呼气时，想象脚尖轻点水面，将左脚缓慢向下点。吸气时，收回至桌面位。换右侧重复。注意动作源自髋关节（髋伸），而非膝关节，保持大小腿角度不变。若难以控制，可在双膝间夹一个小球辅助。

**退阶版本**：若核心力量较弱，可单腿交替进行。步骤为：吸气抬脚跟，呼气抬腿至桌面位，吸气落腿点地，呼气脚跟落地。换另一侧。

## 动作二：死虫子式

接下来，我们在行军踏步的基础上加入手臂动作，形成“死虫子式”。这个动作能有效激活腹内外斜肌，锻炼身体的交叉链稳定性。

以下是动作步骤：

1.  **准备姿势**：仰卧，将双腿抬至水平桌面位，保持骨盆稳定。双手伸向天花板，指尖垂直向上。
2.  **执行动作**：呼气时，将**左脚向下点地**，同时**右手向头后方向延伸**，左手则顺着左腿方向自然下落。吸气时，手脚同时收回至起始位置。呼气，换另一侧（右脚点地，左手后伸）。
3.  **要点**：动作过程中，时刻关注腹部是否凸起，需始终保持核心内收，维持身体稳定。建议完成8组练习。

## 动作三：百次拍击

这个动作能强力激活内核心（腹横肌）和腹直肌，同时增强骨盆底肌力量。

以下是动作步骤：

1.  **准备姿势**：仰卧，双手放于身体两侧。吸气，将双腿抬至水平桌面位，保持核心收紧与骨盆稳定。
2.  **抬起上身**：呼气，将上半身抬离地面，直至肩胛骨下角接触垫面即可。若颈部不适，可用一只手轻轻托住后脑勺，手与头产生轻微对抗。双手向前平举，与髋部同高，掌心向下。
3.  **呼吸与律动**：采用快速的“泵式呼吸”。鼻子快速短促地吸气两次（像吸鼻涕），嘴巴快速短促地呼气两次（发出“嘶嘶”声）。呼吸的同时，手臂配合上下小幅拍动（约离地15厘米）。例如：**吸-吸，呼-呼**，同时手臂上下拍动两次。
4.  **进阶**：可从两吸两呼开始，逐渐增加至三吸三呼、四吸四呼、五吸五呼。每组呼吸模式连续做三组。若觉吃力，可保持上身贴地，仅做手臂拍击与呼吸配合。

## 动作四：单腿朝天

此动作专门针对女性容易堆积脂肪的下腹部进行强化练习。

以下是动作步骤：

1.  **准备姿势**：仰卧，调整身体至中立位（头颈延展，双肩放松，骨盆中立）。伸直双腿，脚尖回勾。若伸直腿时腰部拱起（骨盆前倾），则需收紧核心将骨盆调回中立位，或微屈膝完成。
2.  **抬起上身**：预先收紧核心（收骨盆底肌、收腹）。吸气准备，呼气时将上半身抬离地面，视线看向双脚之间。
3.  **执行动作**：双腿呈剪刀状（一上一下）。双手可环抱上侧腿的大腿后侧轻微借力。伴随呼吸，将上侧腿向胸部方向拉近两次。例如：**吸-呼（拉近）-吸-呼（拉近）**。下侧腿始终保持悬空。完成一侧后换腿。建议每侧完成8组。

## 动作五：单腿伸展与双腿伸展

这是针对腹横肌及腹内外斜肌的练习，单腿伸展是基础，双腿伸展是进阶。

**单腿伸展步骤**：

1.  **准备姿势**：仰卧放松后，将双腿抬至水平桌面位，收紧核心。吸气，抬起上半身。
2.  **执行动作**：双手放在双膝两侧。呼气时，伸直右腿，左腿保持屈膝。右手放在右腿外侧，左手放在左膝内侧。吸气时，交换腿和手的位置（伸直左腿，屈右膝，左手放左腿外侧，右手放右膝内侧）。配合呼吸交替进行，完成5-8组。若抬起上身困难，可尽量贴地完成；颈部不适可用手托住头部。

**双腿伸展（进阶）步骤**：

1.  **准备姿势**：从仰卧开始，预先收紧核心。吸气，将双腿抬至桌面位。呼气，抬起上半身，双手放在双膝两侧。
2.  **执行动作**：吸气时，将双腿和双手同时向远处伸展（腿伸直，手向后打开）。呼气时，收回至屈膝抱腿姿势。配合呼吸重复进行。

## 课程总结与拉伸

本节课我们一起学习了五个针对腹部塑形的动态练习。从稳定核心的“行军踏步”，到激活交叉链的“死虫子式”，再到强化内核心的“百次拍击”，以及针对下腹的“单腿朝天”和整合腹斜肌的“伸展”系列。所有动作都强调预先收紧核心（腹横肌与骨盆底肌）。

训练结束后必须进行拉伸，以减少肌肉乳酸堆积，缓解后续酸痛，并塑造更漂亮的肌肉线条。

**拉伸动作（上犬式变体）**：

1.  俯卧，双手置于胸部两侧，脚背压地。
2.  吸气，双手推地，将上半身抬起。如果可以，让髋部离地；如果困难，保持肘部微屈即可。
3.  感受腹部的拉伸感，臀部放松，双肩下沉。保持3-5组深呼吸后缓慢落下。

希望各位妈妈坚持练习，逐步建立身体的本体感受，最终收获平坦的小腹和理想的马甲线。
