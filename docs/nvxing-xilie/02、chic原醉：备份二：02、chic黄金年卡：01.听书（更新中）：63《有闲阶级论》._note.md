# 经济学与社会批判：02：有闲阶级论解读

在本节课中，我们将一起学习美国经济学家凡勃伦的经典著作《有闲阶级论》。我们将探讨“有闲阶级”的定义、其价值观对社会的影响，以及这个阶级跨越时代的共性。通过本次学习，你将理解财富、休闲与社会地位之间复杂的历史联系。

## 什么是有闲阶级？这个阶级是如何产生的？

上一节我们介绍了课程概述，本节中我们来看看“有闲阶级”的具体定义及其历史起源。

所谓有闲阶级，指的是那些**远离手工生产劳动，同时占有更多财富**的社会群体。这个阶级在数千年前便已诞生。

在早期人类部落中，狩猎和采集是主要的食物获取方式。生产力低下的时代，工作差异不大，明确的阶级尚未形成。随着社会发展，工作性质差别越来越大，阶级雏形开始出现。生产力达到新阶段后，人们对工作价值的判断向两个方向转化。

用书中的语言概括，可以分别命名为**勋绩**和**生产**。中国古话“劳心者治人，劳力者治于人”与此有相似之处，但不完全相同。

*   **生产者**：从大自然中采集资源，并将其加工成可供族群使用的生活用品。
*   **勋绩者**：以智慧和力量改变整个族群的命运，担负起族群的兴亡。在远古时期，这同样需要付出体力，例如围猎危险动物或保卫部落安全。

在野蛮时代，战斗力更强的男性更容易成为勋绩者。他们因此渐渐远离一线手工生产，这是步入有闲阶层的第一个标志。此时，职业鄙视链产生：有闲阶级的工作被视为高贵，而普通手工生产被视为低贱。这种观点延续至今。

随着时代进步，私有制被确立。有闲阶级中的强者开始争夺战利品，从财产到俘虏。许多俘虏沦为奴隶，人类进入奴隶时代，这又被称为“掠夺时代”。**财富的占有程度**成为区分阶级的明显标志，并长期保留下来。直到如今，财富仍是衡量有闲阶级的重要标志。

在我们这个时代，财富常与荣耀画上等号。人人追求财富，向往有闲阶级的身份，希望跃迁到更高阶层，攀比财富的风气也流行开来。例如，文艺复兴后，贵族竞相建造奢华豪宅以胜过旁人。

那么，如何积攒更多财富？通常认知是“开源节流”，依靠勤劳和节俭。但在有闲阶级看来，**劳动是身份低贱的标志**，高贵人不应参与劳动，保持休闲状态才能彰显身份。

在奴隶时代，奴隶主靠剥削奴隶维持财富地位，是典型的有闲阶级。封建时代的王公贵族亦是如此。即使家道中落，昔日的“有闲阶级”仍坚持骄傲，不愿参与生产，强撑体面。

以日本封建时代为例，武士、将军、领主属于有闲阶级，不被允许从事劳动生产，因此部分武士生活穷困。而经济可能更稳定的农夫和商人却不属于此阶级，见到武士必须毕恭毕敬。

## 有闲阶级的价值观为世界带来了哪些影响？

上一节我们探讨了有闲阶级的定义与起源，本节中我们来看看这个阶级的价值观如何塑造了社会。

有闲阶级为社会带来深远影响，无论是礼貌、审美风尚，其背后都受有闲阶级左右。根本原因在于，有闲阶级掌握绝大多数财富。为谋求生存，其他人不得不向其靠拢，最终将其生活习惯扩散，改变大众行为。

**礼貌**与有闲阶级息息相关。中国古语“仓廪实而知礼节”正是如此。有闲阶级无需为生计担忧，于是在日常生活中增加繁琐仪式和习惯，希望与无产阶级划清界限。久而久之，礼仪在大众中流行，缺乏礼貌者被厌恶。例如，18世纪法国宫廷贵族对礼仪的狂热追求，曾长期影响整个欧洲的礼仪风尚。

在资本主义时代，拥有财富的有产阶级驱使奴仆服务。这些仆人唯一工作是侍奉主人，不参与生产劳动。按照书中定义，他们被称为 **“越位休闲者”** 。一个优秀仆人要熟知家规，对主人卑躬屈膝。为彰显身份，他们会穿着特别衣服，消费有闲阶级才能享用的物品。

在早期时代，只有战功赫赫的男性才能享用精美食物和稀有材料制成的奢侈品，普通劳动者则无此资格。随着时代发展，麻醉性饮料（如酒）和药品成为追捧的消费品。文学作品中常出现醉醺醺的管家和仆人，这便是“越位休闲者”的消费方式之一。

我们可以将这种**维持基础生活之外的消费命名为“炫耀性消费”**。它是区分有闲阶级的重要标志，而昂贵的奢侈品则是顶层有闲阶级的专属。

如何评价奢侈品的价值？在珠宝鉴定技术不成熟的时代，人们多从感性角度，根据外观判断价值。与普通的钢铁和石头相比，璀璨的黄金和多彩的宝石更受追捧，成为奢侈品原料。由此可见，**原材料的稀有程度**是关键因素。

例如，有闲阶级若想建造绿草茵茵的牧场彰显地位，不会选择田间劳作的牛作为点缀，因其不够名贵。他们更倾向于选择来自远方的名贵食草动物，如非洲的瞪羚、小鹿等。当然，上层有闲阶级的审美也会变化，有时会追求返璞归真的美感。

如今的炫耀性消费，主要体现在同类消费品的品牌选择上。例如，部分人追随苹果手机的创新潮流，另一部分人则选择奢侈品牌Vertu以高昂价格彰显身份，也有人选择低调的华为或小米以避免注意。

炫耀性消费的大趋势不变。通常，与农业生产息息相关的猪、牛、山羊等，不会成为炫耀性消费目标。而最初作为消遣伙伴被驯养的猫、狗、马等，则可归入此范畴。人们用富丽堂皇的词语形容它们，实际上，狗是打猎帮手，马是赛马赌博伙伴，这沿袭了掠夺时代的身份，符合现代人挥霍炫耀的心理。

有闲阶级对**女性审美**的影响则更加耐人寻味。在炫耀性消费时代，人们往往喜欢纤细精致的女性。为迎合此审美，许多女性主动减肥、整容，这一切做法都源于有闲阶级的审美标准。

为了让有闲阶级彰显审美，人们设计出除美感外无实用价值的装饰品和衣服。廉价的机械制造衣服往往不受欢迎，而昂贵、别致的衣服却受追捧。例如，身穿整齐西装、锃亮皮鞋的男性，不易让人联想到手工工人；身穿裙子、高跟鞋的女性亦然。这些衣服的设计目的，就是显示穿着者不可能从事一线生产劳动。但绝不能忽视，这些炫耀意味十足的衣服往往穿戴不便、不舒服，唯一作用就是彰显身份。

当然，时代变化使衣服蕴含的意味越来越复杂。本书诞生于100年前，当时西装革履的男性易让人联想到有闲阶级。但在今天，写字楼普通职员也被要求穿正装，其薪水可能非常微薄。西装的高贵感在这个时代已可忽略不计。因此，要以发展的眼光看待作者观点。

即便如此，仔细观察仍能发现，有闲阶级的西装搭配风格（在颜色、装饰上翻新）与房产销售者的流水线式西装衬衫并不相同，同样营造出卓尔不凡的效果。由此可见，服装仍是区分有闲阶级的特点，只是潮流引领者与追随者之间存在时间差。

## 纵观古今，有闲阶级存在哪些共性？

上一节我们分析了有闲阶级价值观的影响，本节我们将总结这个阶级跨越历史的共同特征。

有闲阶级往往通过消费彰显地位，而消费需要庞大财力支撑。为攫取更多财富，有闲阶级通常具有**武力和欺诈**这两种特性，这也是其赖以生存的不二法门。

古代酋长为彰显身份，头上插五颜六色的羽毛，但现代富豪未必如此。因此，在人群中很难一眼辨认富豪与平民。许多高阶人士甚至会主动淡化与普通人的距离，以消弭底层仇富心理。但**每个人都热衷于炫耀性消费**是不争的事实。在我们这个时代，若能辨认一个人的消费方式，借此明确其身份，也非难事。

在远古奴隶时代，奴隶收入仅能勉强维生，劳动积极性不高。进入资本主义时代后，关系改变：资本家成为新有闲阶级，工人成为新无产阶级。若工人付出较多劳动，资本家会相应付出更多酬劳，因此工人劳动积极性远胜古代奴隶。

对普通无产阶级来说，维持基础消费后，收入往往还能剩余一部分。为表现物质条件充裕，他们也将有闲阶级的炫耀性消费沿袭到生活中。

在城市中，人口集中、流动性大，消费习惯更容易普及。这种消费风尚的主要带动者，是一些流动性强、人数多的职业，最典型的是流水线工人。他们趁年轻加班换取更多收入，但未必长期在一家工厂工作。赚到钱后，他们会辗转下一个城市，并购买苹果手机、笔记本电脑或高级化妆品，以拉近与有闲阶级的距离。这种消费彰显着人们对有闲阶级的向往。久而久之，消费习惯转化为生活习惯的一部分。让人主动降低消费标准非常困难，毕竟由俭入奢易，由奢入俭难。

为维持消费水平，城市中产阶级生活往往捉襟见肘，因此不愿生育子女，以免生活水平下降，无法维持昔日的炫耀性消费。这种生育观是消费时代的共性。

在消费至上理念下，人们的价值观也产生变化。如果身居高位者获取不义之财，却用于奢侈消费，反而会被一部分群众谅解。这说明，**财富至上的理念已被大多数人所认可**。

根据之前观点，光靠劳动无法累积大量财富。有闲阶级热衷的职业却能让他们财源广进。从掠夺时代进化至今，他们的职业也悄然改变。

*   **律师**：是典型的掠夺性职业，与手工生产几乎无关，却能凭借机敏和狡诈，获得普遍尊重、畏惧及巨大金钱收益。
*   **金融从业者**：需强大预判能力和卓著心理素质，只有如此才能在竞争中脱颖而出，获得更大财力。

若想保住自己的阶层，必须深谋远虑、谨慎行事，以狡诈的智慧谋得更好明天。

有意思的是，古代有闲阶级推崇的**尚武精神**也在现代得到继承。古代人争夺资源或配偶，往往通过你死我活的斗争完成。人类进入文明社会后，相对文明的决斗取代死斗。在西方国家，决斗制度曾流行很长时间，上层有闲阶级将这种好斗习惯沿袭下来，军队军官则是其发扬者。

决斗被明令禁止后，这种习俗并未消亡，而是被**体育竞技**所代替。如今的体育巨星身价不菲，他们虽靠体力赢得收入，但仍属于“炫耀性休闲者”，因其职业与真正的体力劳动相差甚远。他们是上古尚武精神的代表，而从蛮荒世界遗留下来的残忍与狡猾，则在体育运动中体现得淋漓尽致。

如果我们希望实现阶级跃迁，绝不能只满足于为人打工，而是要累积实力、拓展人脉，尽量增大社会影响力。当自己从一个普通生产者，晋级为拥有市场渠道及稳定订单的资本家时，收入将会增长。累积到一定程度后，也将晋升为有闲阶级的一员。

## 总结

本节课中我们一起学习了凡勃伦《有闲阶级论》的核心内容。

首先，我们讲述了有闲阶级的定义及其诞生过程。在远古时期，随着人类部落发展，分工日益明显。一部分人从事手工生产以维持族群生存，另一部分人成为统治者并远离手工生产。他们凭借自身优势占有大量资源，借此彰显不凡身份，成为最早的**有闲阶级**。即便历经千年巨变，对有闲阶级的定义——财产充裕和远离生产劳动——仍未发生太大变化。至今，人们仍追求庞大财富，并蔑视基层手工劳动者。

然后，我们讲述了有闲阶级的价值观为世界带来的影响。在财富和地位支撑下，有闲阶级制定了关于礼貌的判定标准。这些仪式感十足的做法不仅在上层社会蔚然成风，更渐渐扩散至整个社会。在有闲阶级带动下，他们的仆人也远离普通生产，成为受雇佣者的雏形。此外，有闲阶级对审美方向的界定深刻影响着社会对美的看法。人们热衷于购买使用外形华贵的奢侈品；女性通过节食、整容等方式迎合有闲阶级审美；无论男女，都热衷于穿戴不适合手工生产的衣着以彰显阶级身份。

最后，我们讲述了有闲阶级存在的共性。在我们的时代，富豪外表与普通人差别不大，但他们仍热衷于在基础生活之外进行额外的**炫耀性消费**。这种消费风潮也向底层普通民众蔓延，迫使城市中产阶层不再热衷生育。观察新时代的有闲阶级，如金融从业者、律师及体育竞技者，我们会再次发现，这些新生代有闲阶级大都拥有**武力和欺诈**这两种特性，这无疑是有闲阶级的共性。

虽然作者对有闲阶级持有批判态度，但作为读者，我们不妨同样以批判性眼光看待书中观点。了解有闲阶级的来源、对社会的影响和共性，在自己不断积累财富的过程中，朝着有闲的生活方式努力，同时也避免其弊端。
