# 恋爱课堂：03：追求的艺术【完结】

## 01. 为什么要学习谈恋爱

在本节课中，我们将探讨学习恋爱知识的必要性，并理解为什么追求爱情并非天生技能，而是一门值得系统学习的艺术。

---

我相信很多人在第一次看到课程名称时，内心的第一个想法是：谈恋爱还需要学吗？谈恋爱有什么好学的？

抛开那些情场失意、情路坎坷、从未追到过喜欢的人，或渴望改变的人不谈，我相信大多数人会认为，谈恋爱和追求是天生就会的事。大部分男生可能会这样想，并感到疑惑。

他们会想，谈恋爱有什么好学的？而且产生这种想法时，可能还带有一丝嘲讽和轻视。

我经常对学员讲，在很多不了解恋爱课堂的人看来，你们交钱上课学习谈恋爱的行为非常傻。这种想法很正常，就像没人认为走路、跑步这种天生就会的事，还需要交钱专门上课学习。所以有人这样想非常正常。

可是大家有没有想过，如果谈恋爱真的是人天生就会的技能，那为什么有些人就是追不到自己喜欢的人？而且还有很多人，可能条件、长相、人品等各方面都不如你，却偏偏能找到令人羡慕的对象。

更可悲的是，很多人的心上人，最后往往被一些大家公认很渣、很不好的人抢走。所以他们心里苦，却无处诉说。

在我多年的教学生涯中，我见证了无数男孩女孩凄苦的感情史。我见过很多人，他们在追求喜欢的人时特别用心，人也很好，但追求的方式往往错误百出。总结起来，他们的方式几乎都一样。

很多没有学习过如何追求的人，在追求别人时一开始都特别迷茫。他们不知道该怎么追求，不知道聊什么天、怎么说话，甚至不知道要约别人出来。

所以他们追求别人，只会用最传统的方式：对对方很好、关心体贴、言语间试探暗示、悄悄送礼物、制造惊喜、请吃饭、陪聊天、在对方难过时安慰。他们希望有一天对方能发现其用心良苦，并因此感动。

然后一直等，发现对方没什么表现，最后终于忍不住，突然来一个郑重其事的表白，最后收获一张金灿灿的“好人卡”或“好妹妹卡”。

这种情况其实还好，至少他还愿意尝试沟通、了解和行动。我还认识另外一种人，他们喜欢一个人，很想和对方聊天，但根本没有话题，也不知道如何开口，几乎每次聊天都聊死了。

每次他们打开对话框，都不知道要和对方说什么。思考半天，想了很久，最后往往只发出：“在吗？”“在干嘛？”“吃了吗？”。对方回复也往往非常不积极，都是：“在呀。”“没干嘛。”“吃了。”或者就是：“嗯。”“呵呵。”“干嘛？”“去洗澡。”——即所谓的“女神三法宝”。然后就冷场了。

很多时候对方完全不回复，或者聊着聊着就没声音了。他们很难过，一会儿看手机怎么还不回消息，一会儿翻看是不是没收到。或者聊着聊着发现对方突然不回了，然后翻聊天记录，思考是不是哪里说错了话。

这是很多人在追求喜欢的人时的常态。讲了这么多，你会发现，谈恋爱这件事非常有必要去学，而且它不是天生就会的。如果是天生的事，为什么这部分人会这样？还有一部分人却天天被很多人喜欢？

我想告诉大家，谈恋爱这件事不是天生的，它很有必要去学习。我们想要学习谈恋爱，首先需要对这件事有一个正确的认知。

其实，学习谈恋爱准确讲，应该叫学习两性关系的知识。两性关系不是指两性生活，希望大家不要误解。两性关系是指男性和女性之间的关系，这是一门非常严谨的科学。

但市面上很多相关书籍和名作的角度都太宏观，内容大多讲的是恋爱后、结婚后发生的亲密关系。所以在国内或国外，几乎没有任何名家作者专门研究“如何追求”这个过程当中的两性关系。

再加上中国传统文化对于找对象、追求心上人这些事情比较含蓄或羞涩，说出来大家会觉得没面子或难为情。所以恋爱课程，特别是追求这方面，一直没有一个明确的市场，也没有在大众价值观中充分曝光，导致很多人对其认知有偏差。

可是，大家都知道这个社会的竞争非常残酷，人和人的竞争无处不在，感情中更是如此。我记得很小的时候听过一首歌叫《明明很爱你》，里面有一句歌词是这样唱的：“明明很爱你，明明想靠近。但是你的身边有人捧花总是拥挤。我凭什么一一打败情敌，敢大声说要做你的唯一。”

对呀，我凭什么一一打败情敌呢？这首歌是我十几岁失恋时天天听的。我就在感叹，我凭什么打败别人，凭什么拥有他？这就是人和人之间总是有竞争。

我经常对学生讲，人与人之间的比较和竞争，并不是比谁跑得快，也不是比谁跑得远。因为人的一生总会遇到很多坎坷和挫折，而人与人之间所谓能力的差异，就是看谁在遇到困难挫折、遇到一个坎的时候，在这个最关键的时刻跳得高、跳得远，跳过那个坎。这才是人与人之间最大的区别。

就如同你在追求你喜欢的人，还有另一个人也在追求。当你们同时喜欢的这个人出现某些问题、某些考验时，谁有能力跳过，谁就可能赢得对方的芳心，赢得对方的喜欢。就是这个道理。

所以说，学习谈恋爱，我打一个比方：如同在学习“作弊”。很多人会对“作弊”嗤之以鼻，觉得可耻。其实作弊是一种解决问题的能力，是在关键时刻能够解决问题的能力。

我不知道有多少听课的人小时候考试作过弊，反正我作过。最重要的是，每次我作弊成功，某一科成绩合格，我都非常开心。我达到了自己的目的，并且从不认为我可耻，反而特别有成就感。

同样我也相信，但凡曾经作过弊的人，哪怕失败了，也不会觉得自己很可耻、很无耻。他反而还是会觉得自己很光荣，只是运气不好。所以说，很多人对于作弊的看法和对于学习谈恋爱的看法其实都一样。

就如同谁不想作弊？或者准确讲，谁又不想稳稳通过，拿到一份满意的成绩呢？就如同谁不想成为一个情场高手，追到自己喜欢的人，过上幸福美满的生活呢？但这样想又有什么用？

现实生活中，很多人的感情生活非常困顿、迷茫。他们都很想知道：为什么我喜欢的人不喜欢我？我又该如何让他们喜欢我？怎么样？他们很多时候在迷茫、在渴望，但很多人并没有真正努力探索。

他们虽然情场失意，但并没有把感情问题真正当作需要解决的问题。他们经常表现出非常痛苦、难过、难受，但这种表现全都是无病呻吟。甚至有很多人认为，我如此悲伤，或许会有人喜欢我、被我打动。

他们其实就是闲得无聊，并不会去努力思考和学习。就算面前真的出现一堂教你谈恋爱、教你如何追求的课程，他也会在心里“呵呵”，觉得谈恋爱有什么好学的。然后明天依然说：“为什么他不喜欢我？我该怎么办？”

所以我特别佩服能够点开这堂课，并且能听到这里的人。我觉得你们都特别棒。不管你是出于好奇，还是真的想要学习，我都佩服你们。至少你们没有对知识抱有歧视，至少没有在一开始就将它完全抹杀。

如同我在前面讲，作弊就是一个在关键时刻跳得高、跳得远的能力，它是知识解决问题的方式。为什么要对它有成见呢？比如说，如果你作弊成功了，能够顺利毕业、上名牌大学、通过面试……人生中有很多关键时刻。

我就认识一个人，他要去一家公司面试，然后做了一个假文凭，进入了公司。最后凭借自己非常出色的业务能力，为公司创下巨大利益，最后升职加薪。几年后老总发现他的文凭是假的，那又怎样？老总不会开除他，老总还说：“看不出来呀，你小子还骗我。”开开玩笑就过去了。

当然，我和大家讲这些，不是让大家将来都去办假文凭，还是得好好读书。对于学生，我想告诉大家，学习谈恋爱，其实最重要的就是我们要通过学习，拥有一种能力。一种在我们追求爱情、追求喜欢的人时，用来解决问题的能力。

我们通过学习以后，就能够拥有这种在最关键时刻找到最优解法去处理问题的能力，最后达到我们想要的目的。比如你考试想要过关，比如你能追到你喜欢的人。不管你是男人还是女人，难道你能说这种能力不重要？

如果你认为这种能力不重要，那将来你遇到你喜欢的人，刚好也有另一个人喜欢他。当你出现竞争者的时候，你又凭什么认为胜利的一定是你？再或者，你将来喜欢上一个人，但他喜欢一个人渣，你又如何去救他于水深火热之中呢？

诸如此类的问题太多了。所以我们是一定需要通过学习，拥有这种能力，然后去解决问题。

那么，说了这么多，我们学习谈恋爱，到最后到底是在学习什么？其实学的东西听起来也挺简单的。我们需要搞清楚几个非常重要的问题。

以下是几个核心的学习方向：

1.  **感情与人性的基础本质**：我们需要去搞清楚感情和人性之间最基础的本质。人性是什么？感情又是什么？这些最基础不变的本质。就如同你不知道什么是爱情，你不知道“爱”和“情”是两个东西。
2.  **人际交往的基础道理**：我们需要去学习人与人之间交往最基础、最重要的道理。比如说，人喜欢什么，不喜欢什么，讨厌什么；什么东西招人喜欢，什么东西叫有魅力，什么东西没有魅力等等。这些最基础的为人、待人接物的道理。
3.  **解决问题的方法论**：我们要去学习解决问题方法背后的原理，掌握方法论。

这些东西一点都不夸张。我认识很多人，他们以为自己非常懂，其实他们一点都不懂。什么叫不懂？我可以跟大家举一个非常简单的例子。

比方说，我们可以看到，从小到大遇到很多人，同学、朋友也好，你看到身边的人谈恋爱或追到自己喜欢的人的过程就是这样的：不停地聊天、发短信（以前打电话），然后不停地关心问候，不停地约、聊。约个四五次、五六次，然后很突然两个人有一天就变成了情侣。就是这样一件事。

很多人看到的都是表象，然后他最后得出一个错误的结果。他们以为，我只要不停和喜欢的人发生聊天、约会这样的事情，对方就能喜欢上我，然后两个人就会在一起。其实这些都是错的，这里面有很多他们不知道的东西。

比如说，他们不知道一个人喜欢上另一个人，是因为这个人本身很有魅力，而不是因为两个人在一起约会的次数很多、聊天的时间很长。约会和聊天本身不具备吸引力。约会和聊天只是用来展示你本身魅力的载体和途径。

而那些约了很多次会最后喜欢上对方的人，都是因为在约会聊天的过程中，被这个人展现出来的魅力所吸引。不管是物质的魅力，还是精神人格的魅力。

也就是说，对于一个你本身没有吸引力的人而言，你和异性约1000次、1万次，也不会有异性喜欢上你。但是很多人可能并不知道什么叫吸引力，他只知道盲目追求，不知道原来我还需要有吸引力。

太多太多的人不懂得这些，他们只能看到表象。于是乎，他们去追求一个人、喜欢上一个人、要去追求的时候，就会变成这个样子：编着理由去和对方聊天，而对方不回复，还会一次一次地问；或者就是单独去约对方，一次又一次地约，最后终于把对方约出来了。然后陪他吃了个饭、看了个电影，接着就回家了。然后可能就再也没有别的了。

回家以后发个消息：“到家了吗？”“到了。”然后他就不知道说什么了。“今天开心吗？”“你对我感觉怎么样？”对，就是这样。然后或者对方就完全不回消息，他就黯然神伤，默默伤心，想着我是不是要放弃了，想着对方是不是不喜欢我。

再有的就是受不了相思的煎熬，就直接果断选择表白，最后换一张金灿灿的好人卡。而女生可能还要更悲剧一点，因为又有很多男生，他们会贪图女生的身体，但是又不想负责任。所以他就不干脆地把女生拒绝掉，然后导致这些女生一直认为自己还有希望。其实有很多男生也是这样。

反正他们最后就会变成了一个社会新兴的产物：叫“备胎”。

所以，说了这么多，大家也就发现了，其实谈恋爱是很有必要去学习的。因为我们需要这些知识，需要搞清感情和人之间本质的道理，人和人之间交往最本质、最基础的道理。

当然，我也遇到过有一些人，他们也知道需要学习，但是呢，他们只会在情感受挫的时候来找我，渴望我给他们一个办法去解决他们当前的问题。然后我给他们办法以后，可没几天他又来了，他又遇到了新的问题。

而且还有一些人特别喜欢问我那种很宏观的问题。比如说，有一天突然有一个人知道了：“你是一个网络著名的情感专家，对不对？那你快告诉我，我怎么追姑娘啊？我怎么追到喜欢的男生啊？我怎么去撩汉子？”

他们就直接问我这种问题，请教我如何追喜欢的人。这么宏观的问题，我怎么可能几句话回答呢？因为两性关系的知识是系统的呀，这是《追求的艺术》，后面还会有《相处的艺术》、《处理危机的艺术》，对不对？还有《备胎反击的艺术》。很多很多的东西，它都是系统的，绝不能断章取义。

很多人也不知道这个，他们渴望快速解决问题，甚至会有很多人跟我说：“哎，你不要跟我讲大道理，你就直接告诉我，我要用什么办法才能追到他，我要用什么办法才能搞定他。你给我一个方法，你给我一个具体的方案，或者你给我几个话术。”

很多人不分男女，我甚至还遇到有些人拿着聊天记录跟我说：“大神，我要如何回复他，他才会喜欢我？”其实我当时特别想对这个人讲：“孩子你要想想，我跟他回一个消息，他就喜欢上我了，你觉得我还会在这里吗？我估计早就被国家带走了。”

我跟大家讲这么多话，都是想要告诉大家，两性关系是非常系统的，而且是从头到尾有联系的。就像我在后面课程会提到的“相识即吸引”一样，对吧？就是认识的时候，就是吸引对方最好的时候。这个东西，它都是从头到尾有联系的。你在一开始做不好，后面就根本没有办法做。

当然，后面会讲，不要着急。所以我们在学习的时候，并不是在学习某个案例里面的几句话、几个套路。我们需要的是：他为什么要去说这个话？他为什么该这样做？我们要学习的是这个方法背后的理论，也就是去创造方法的方法，就是**方法论**。

而只有学会了方法论，我们才会真正掌握知识。如果你只是去学什么话术、学什么套路、技巧，根本没有用。而且有一天如果你遇到了一个人，他跟你说，只要办法用得好、技术用得好，你就能撩到、追到你喜欢的人。我跟你说，这种事情就完全没常识，他绝对是骗你的。他要这么牛，他怎么不去泡个大明星看看？你不是技术用得好什么都能泡得到吗？

因为什么呀？谈恋爱、追求人不是做机械性的产品。机械性产品的目的不会变，就像烤蛋糕，只要你明确步骤、时间、用量，你可以机械性地生产出很多蛋糕，而且味道都差不多。你换个人去烤也一样。

但是谈恋爱和追求这个东西就不一样，人是会变化的，而且人还都不一样。你指不定同样的一句话，我说出去成功了，你说出去，那你可能就被打了。所以我们需要学习的是**方法论**，需要学习的是创造方法的方法，要去学习最本质的那些道理和原理。因为方法论是本质，不会变的。

一旦我们掌握了方法论以后，我们就可以根据不同的人、不同的环境、不同的时间，创造出当下最适合解决问题的方法，也能够创造出专门属于你的方法。就比如说你自己有什么样的特征，那这个方法可能只适用于你，不适用于我。

所以，在我接下来讲的所有课程里面，所有关于实际操作方面的一些道理，基本都是方法论。比如说：主观喜欢客观否定、斩断可能性、障碍性的表达兴趣等等，全部都是方法论。当然你不用担心你会听不懂，因为我会讲得非常详细，而且还会结合大量的案例来和你讲。

所以在两性关系里面，不管是男人追女人，还是女人追男人，其实本质的道理都是相通的，没有什么区别，只是在细节层面上有差别。本质道理都相同，都一样。

然后，如果我一上来就告诉你，第一堂课就告诉你，你该如何追，那我告诉你绝对是骗子，绝对是忽悠。为什么呢？因为在追求之前，我们需要先对两性关系的知识有一定的了解。你们大家都需要很多那种颠覆你三观的知识。你相信我，我跟你讲完这些内容以后，你绝对会瞠目结舌。

就像下一堂课《揭开两性吸引的秘密》，这堂课告诉你吸引力到底有多么重要，什么是吸引力，如何让自己具备吸引力，什么东西才能吸引到异性，你吸引不到异性是为什么，为什么很多你认为能够有用的、能够吸引到对方、能够让对方喜欢你的东西，到最后对方都没有喜欢到你。

就说你很难过，所以下一堂课的内容就是解决你的很多思维上的迷惑，让你能够思维通畅，然后你才能够知道第一步干嘛，第二步干嘛。初次见到异性该如何，后面如何聊天沟通。这一整套下来才是完整的。
