# 瑜伽理疗入门：01：告别含胸圆肩的肩颈练习 🧘‍♀️

在本节课中，我们将学习一套针对肩颈问题的理疗瑜伽序列。这套练习旨在通过循序渐进的温和动作，帮助改善肩周炎、圆肩驼背和颈椎前引等问题，逐步建立身体的稳定性并放松紧张的关节与肌肉。

## 课程概述

肩颈理疗练习分为三个阶段，旨在逐步改善肩周炎与颈椎问题。常见的肩颈问题包括肩周炎、圆肩驼背、颈椎前引以及椎间盘突出。通过系统练习，可以建立身体稳定性，打开关节，缓解不适。

## 练习准备

选择一种舒适、放松的坐姿，确保身体垂直于地板，双膝自然下沉。保持骨盆摆正，肋骨内收，肩膀放松。

## 肩部环绕激活

首先，通过简单的活动来放松肩膀。许多人长期处于耸肩圆肩的状态。

以下是肩部环绕的步骤：
1.  双手轻搭于双肩。
2.  吸气时，手肘向上抬起，尝试让手背在颈后相触。
3.  呼气时，手肘向下落，尽量靠近肋骨两侧。
4.  按照自己的呼吸节奏，缓慢重复此动作10到15次。

**注意**：肩周炎患者若手臂无法抬高或双手无法相触，只需抬到力所能及的高度即可，避免手肘向前扣，应尽量向外展开。始终保持骨盆中立，避免前倾或塌腰。

完成15次后，将双手手肘在胸前相碰。

## 手肘开合练习

上一节我们激活了肩部，本节通过手肘的开合进一步打开胸腔。

以下是练习步骤：
1.  保持手肘在胸前相触。
2.  吸气时，手肘向上并向两侧展开。
3.  呼气时，手肘下落相触，同时收腹、低头拱背。
4.  跟随呼吸，重复此开合动作15次。

**关键点**：每次吸气时尽力打开胸腔，每次呼气时充分拱背。动作需缓慢进行，确保效果。

完成15次后，在最后一次呼气手肘相触时，开始进行环绕练习。

## 肩部环绕练习

接下来，我们将进行完整的肩部环绕，进一步增加肩关节活动度。

以下是练习步骤：
1.  从手肘相触开始，吸气时，手肘向后、向上划圈，尽量让手背在颈后相触。
2.  呼气时，手肘向前、向下划圈，回到相触位置，同时低头拱背。
3.  缓慢、有控制地重复此环绕动作15次。

**核心要领**：动作越慢，对肩膀的放松和激活效果越明显。避免快速、无控制的晃动。

完成15次后，将手臂向两侧打开落下，放松肩膀。

## 坐姿侧弯与扭转

现在，我们通过侧弯和扭转来伸展侧腰与背部，同时活动肩颈。

以下是练习步骤：
1.  吸气，双臂侧平举。
2.  呼气，将右手带向身体右侧，左手轻放于右膝，身体向右侧弯。
3.  吸气，身体回正向上；呼气，收腹，身体向左后方扭转，右手放于体后地面。
4.  保持双肩平行于地板，扭转时避免耸肩。感受背部的伸展。
5.  吸气，后方手臂向上抬起；呼气，身体向左侧弯，保持身体呈一个平面。
6.  保持深长呼吸，下方肩膀向前推，上方肩膀向后打开，下方臀部坐实垫子。
7.  吸气，身体回正，手臂打开。换反方向重复以上步骤。

## 手臂上举与后支撑

本节我们通过手臂上举和后支撑来伸展肩部与灵活胸椎。

首先进行手臂上举：
1.  双手十指交扣，翻转掌心向上。
2.  吸气，手臂向上伸展，身体向上延展。
3.  呼气，双肩主动下沉，远离耳朵。
4.  肩周炎患者若手臂无法伸直，可向前延展，抬到力所能及的高度，但务必保持肩膀下沉，避免耸肩。
5.  保持几个呼吸，感受肩部的伸展。

呼气时，手臂落回体侧。

接着进行后支撑：
1.  双手置于身体后侧，指尖朝前，掌根压地。
2.  吸气，胸椎向上提，抬头，伸展身体前侧，下巴微收。
3.  保持肩膀放松，膝盖向下压。
4.  此动作有助于灵活胸椎，改善驼背，缓解颈椎后侧压力。
**注意**：抬头时眼睛看向斜上方即可，避免过度后仰给颈椎造成压力。
5.  吸气，身体还原。

## 猫式伸展

现在，我们来到四角板凳式，通过猫式灵活脊柱，特别针对胸椎。

以下是动作要点：
1.  手腕在肩膀正下方，膝盖在髋部正下方，四肢垂直地面。
2.  脚背平行或向下压地，避免重心过度前移或后移导致手腕或膝盖不适。
3.  吸气时，抬头，胸椎向前向上推，尾骨向后向上，肩胛骨向后拉，充分伸展脊柱前侧。
4.  呼气时，低头，拱起上背部，眼睛看肚脐，腹部收紧。
5.  重点在于胸椎段的充分活动。吸气时尽力向前延展胸椎（而非过度下压腰椎），呼气时尽力向上拱起上背部。
6.  保持手肘相对，避免肘关节超伸。按照自己的呼吸节奏，重复15次。

完成15次后，将重心后移，臀部坐回脚跟上，身体直立。

## 颈部针对性拉伸

最后，我们针对紧张的颈部肌肉进行拉伸。颈椎前引通常伴随颈前侧肌肉紧张、后侧肌肉缩短。

首先拉伸颈侧斜方肌：
1.  身体坐直，下巴微收。
2.  将右手放于左耳外侧，肩膀下沉。
3.  吸气延展，呼气时，手臂轻轻带动头部向右肩方向倾斜。
4.  保持下巴内收，感受左侧颈部的拉伸，保持30秒。换边重复。

接着拉伸胸锁乳突肌：
1.  头转向左侧，用右手手指按住左侧颈部前方的两条肌腱（胸锁乳突肌）。
2.  吸气，呼气时，头部向右侧斜后方缓慢倒去，感受左侧颈前肌肉的拉伸，保持30秒。换边重复。

最后拉伸枕下肌群：
1.  将双手大拇指按在后脑勺下方头骨边缘的凹陷处（枕下肌群）。
2.  吸气，呼气时，微收下巴，头顶向前方延展，仿佛要将后脑勺拉向远方。
3.  感受大拇指按压处肌肉的伸展，保持30秒。
**注意**：是头顶向前延展，而不是用力低头。

## 课程总结

本节课中，我们一起学习了针对肩颈问题的理疗瑜伽序列。我们从肩部激活环绕开始，逐步进行了手肘开合、坐姿侧弯扭转、手臂伸展、猫式灵活脊柱，最后完成了颈部的针对性拉伸。

虽然动作简单，但关键在于练习的质量：动作要缓慢到位，呼吸要深长配合，注意力要集中在身体的感受上。请定期练习，循序渐进，你的肩颈状况将得到有效改善。
