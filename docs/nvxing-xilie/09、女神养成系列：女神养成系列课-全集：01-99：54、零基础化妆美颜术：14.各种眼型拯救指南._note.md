# 零基础化妆美颜术：14：各种眼型拯救指南

在本节课中，我们将学习如何针对不同的眼型进行妆容调整。我们将重点探讨单眼皮的调整策略、双眼皮贴的使用技巧，以及如何通过睫毛夹、睫毛膏和假睫毛来改变眼型，让眼睛看起来更有神采。

上一节我们介绍了眼妆的基础画法，本节中我们来看看如何针对具体的眼型问题进行改善。

## 单眼皮的调整思路

决定眼睛是否需要调整的根本因素，不是单眼皮或双眼皮，而是眼睛是否看起来明亮有神。需要调整的单眼皮，通常是因为眼皮过多地遮盖了眼珠，导致眼睛显得无神。调整的目标是扩大眼珠的露出范围，提升眼皮。

对于单眼皮的调整，重心不应放在画多宽的眼线或眼影上。因为单眼皮或内双非常“藏”妆，闭眼时画得很宽，睁眼后可能完全看不见。真正有效的方法是改变眼睛的构造。

以下是两种核心的调整方法：
1.  使用双眼皮贴，创造出双眼皮褶皱。
2.  使用假睫毛，或通过夹翘、刷翘睫毛来支撑和改变眼型。

## 双眼皮贴的使用方法

市面上有多种双眼皮贴。最常见的是单面塑料贴，它依靠自身的支撑力撑开眼皮，形成褶皱，但缺点是表面可能反光，不够隐形。

专业化妆中常使用需要自行修剪的蕾丝或纤维网状双眼皮贴。它颜色接近肤色、表面可上妆、不反光，因此非常隐形。但使用时需要配合剪刀修剪，技术难度较高。

日常生活中，推荐使用双面胶式的双眼皮贴（如日本DUP）。它的原理是通过两面粘性，将眼皮粘合出一条褶皱。由于胶条被粘进褶皱里，所以比较隐形。

以下是使用双面双眼皮贴的步骤：
1.  用附赠的小叉子两端分别按压住双眼皮贴的两边，将其粘起，避免手直接接触胶面。
2.  将双眼皮贴放置在预设的双眼皮褶皱位置上方。
3.  先不要睁眼，用叉子底部将眼皮向上推，使胶面与上方眼皮粘合。
4.  睁开眼睛，进行微调。

使用要点：
*   取用工具要保持干净。
*   放置位置不宜过高，刚好压在原有褶皱线上方即可。
*   此方法适用于本身有一定褶皱空间的眼睛（如有内双或褶皱不明显的情况）。如果眼皮特别紧致，可能无法形成持久褶皱。

## 睫毛的重要性与处理技巧

如果双眼皮贴不适用，睫毛是改善眼型的重要利器。

睫毛的重要性体现在：
1.  **妆感自然**：睫毛是眼睛原生部分，处理得当最能打造裸妆效果。
2.  **弱化眼影**：卷翘的睫毛像帘子，能让背后的眼影产生若隐若现的妙感。
3.  **调整眼型**：通过改变睫毛的朝向，可以纵向放大眼睛或横向拉长眼型。
4.  **快速出门**：匆忙时，刷翘睫毛加上一点眼尾眼线，就能让眼睛有神。

### 如何使用睫毛夹

正确使用睫毛夹是关键。推荐使用适合亚洲人眼弧的睫毛夹（如资生堂、植村秀）。

以下是正确步骤：
1.  **清洁夹口**：确保与睫毛接触的橡胶垫干净，无残留彩妆。
2.  **找准角度**：不要将睫毛夹垂直紧贴眼皮。应将夹子与面部呈约45度角，斜向放入。
3.  **分段夹翘**：首先夹住睫毛根部，停留几秒；然后轻轻将夹子向外移动一点，夹住睫毛中段；最后再向外移动，夹一下睫毛梢部。整个过程是“根部 -> 中段 -> 梢部”的渐进式夹法。
4.  **处理边角**：对于难夹的眼头眼尾睫毛，可以使用局部睫毛夹。或者在使用普通睫毛夹时，用手轻轻提起眼尾皮肤，将夹子偏向眼尾方向进行夹取。

检验标准：夹好后睁眼平视，应能看到自己上翘的睫毛。睫毛膏有重量，会稍稍压低睫毛，因此夹翘的程度要略高于最终理想效果。

### 如何刷睫毛膏

初学者建议选择刷头干净、纤维少的睫毛膏，更容易刷出根根分明的效果。细刷头睫毛膏（如Innisfree）不易结块，也适合刷下睫毛。

基本刷法：
1.  将刷头横放，从睫毛根部开始。
2.  以小幅度的“Z”字形手法，向睫毛梢部移动。
3.  根据想达到的效果调整：想眼睛更圆，可着重刷中部睫毛并向上提拉；想眼睛更长，可着重刷眼尾睫毛并向外拉长。
4.  想要更浓密纤长的效果，可以等第一层睫毛膏干后，重复夹翘和刷睫毛的步骤。

刷下睫毛的技巧：
*   可以用纸巾或化妆棉垫在睫毛下方，防止膏体沾到皮肤。
*   竖拿刷头，轻轻梳理下睫毛。

### 特殊睫毛问题的解决

*   **睫毛短**：可考虑嫁接睫毛或粘贴假睫毛。
*   **睫毛硬、易塌**：夹翘后，先使用睫毛定型液，再刷睫毛膏，有助于保持卷翘。

## 假睫毛的挑选与佩戴

假睫毛能显著改变眼型。挑选时需关注：

1.  **长度**：短款更自然，适合新手；长款妆效更明显。
2.  **形状与密度**：稀疏自然的款式接近原生睫毛；簇状、浓密的款式放大效果更好。
3.  **长度分布**：
    *   中部加长型：能让眼睛变得更圆。
    *   眼尾加长型：能让眼睛变得更长。
4.  **梗的质地**：
    *   透明梗：有弹性，较隐形，但眼头眼尾可能翘起。
    *   棉线梗：无弹性，贴合度好，但需要搭配眼线遮盖。

日常可从216、217等接近自然睫毛形态的款式入手。

### 假睫毛佩戴步骤

1.  **修剪长度**：将假睫毛放在眼睛上比对，剪掉过长部分，使其与眼长吻合。
2.  **涂抹胶水**：将胶水挤在背板上，用假睫毛梗部去蘸取，确保胶水涂抹均匀。
3.  **等待片刻**：等待胶水半干（变透明），此时粘性最强。
4.  **佩戴**：眼睛向下看，先将假睫毛中部对准眼中部贴在真睫毛根部，再调整粘好眼头和眼尾。
5.  **调整与修饰**：用镊子调整翘度。胶水干后，可用眼线液笔填补梗部与眼皮间的空隙，使妆效更完美。
6.  **卸除**：用沾满卸妆液的化妆棉湿敷眼睑，待胶水溶解后轻轻取下，切勿直接撕扯。

## 不同眼型的调整策略汇总

掌握了以上核心技巧后，我们可以针对不同眼型进行分析和组合运用。

以下是常见眼型的调整思路：
*   **长眼/丹凤眼**：希望变圆时，强调眼中部。可加粗中段眼线、刷浓中部睫毛，眼影采用中间向两边晕染的方法。
*   **圆眼/杏仁眼**：希望变长时，拉长眼尾。眼线、眼影、睫毛都着重向眼尾水平方向延伸。
*   **吊眼（眼尾高于眼头）**：
    *   若眉眼距宽，可通过加高眼头（如加宽眼头眼影、双眼皮）来平衡。
    *   若眉眼距窄，则需压低眼尾，眼线平拉或稍向下，避免上扬。
*   **垂眼（眼尾下垂）**：重点提拉外眼角。使用“小飞翼”眼线画法，画出微微上扬的眼线。
*   **凹陷眼**：避免使用深色加重凹陷感。应在眼窝处使用浅色、珠光或暖色调眼影，打造饱满感，睫毛可浓密些。
*   **肿泡眼**：避免使用珠光、暖色眼影。可使用冷灰色调眼影制造“后退”感，并强调睫毛的支撑力，将眼皮“撑”起来。
