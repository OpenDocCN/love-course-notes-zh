# 理财入门：1.10：简单易上手，从指数基金开始投资 📈

在本节课中，我们将要学习一种被称为“懒人投资利器”的基金——指数基金。我们将了解它的本质，并掌握如何一步步筛选出适合投资的指数基金。

## 什么是指数基金？

上一节我们介绍了一些基金的基础知识，你可能已经发现基金的筛选过程比较复杂。本节中，我们来看看一种相对更容易筛选和判断的基金类型。

想要了解指数基金是什么，首先要理解什么是指数。我们以股票指数为例进行解释。当然，也存在债券指数等其他金融指数，其原理是类似的。

官方的股票价格指数定义是：每一种股票价格指数都会选取一组有代表性的股票，将它们的价格进行加权平均，通过特定的计算得出。各种指数在股票选取和计算方法上会有所不同。

每个标准的指数就像一份严谨的菜谱。它会明确标注纳入该指数的“原材料”（即股票或其他指定投资品），并严格规定每种“原材料”的投放比例，也就是每个成分股的具体权重。

这些严谨的“菜谱”往往质量很高，很有价值。通过观察这些指数，我们能快速掌握对应投资市场的整体行情。

**指数基金**就是严格对照这份标准“菜谱”进行“做菜”的基金，它力求对指数进行一模一样的复制。

就像普通人对照标准菜谱也能做出对应口味的菜一样，即使我们不会挑选具体的股票，通过购买对应的指数基金，也能完成对相应市场的投资。

你可能会问，在哪里可以查看这些指数的列表？

国内比较官方的查询地址是**中证指数有限公司**的官网。你可以在网站主页的“指数系列”板块，查看到按不同类别划分的各类指数。你也可以在最上方的搜索栏输入关键词，直接搜索感兴趣的指数。

## 如何选择适合投资的指数？

了解到什么是指数之后，你可能会发现，光是中证系列指数就有几百种。面对如此多的指数，我们该如何选出适合投资的那一个呢？我们可以通过定性和定量两个角度来解决这个问题。

### 定性筛选：选对指数

首先，我们可以用定性的方法筛选指数。对指数的定性筛选，我们可以考虑以下三个方面。

以下是三个主要的定性筛选角度：

1.  **优先选择综合程度较大的指数**。一般来说，覆盖股票数量越多、筛选范围越广泛的指数往往越综合。例如，同样代表大盘企业股票，覆盖沪深两市300只股票的**沪深300指数**，就比只含沪市50只股票的**上证50指数**更综合。长期投资指数基金的核心原则，是确保指数背后的经济方向长期看涨。因此，大型综合指数是普通人长期投资的优选。这里先介绍三支有代表性的综合指数：
    *   **沪深300指数**：选取沪深两市中，符合一定条件且市值规模最大的300家公司。它基本是中国企业的标杆，其对应的指数基金也是国内影响力最大、最重要的。
    *   **中证500指数**：选取规模排在沪深300成分股之后的500家交易活跃的公司，综合反映中小市值公司的整体状况。
    *   **标普500指数**：由标准普尔公司编制，基于美国主要交易所中规模前500的企业，选股范围广，代表性强，通常被视为美国大盘股的股指。

2.  **选择行业的代表指数**。例如医药指数、房地产指数、金融指数等。很多时候我们看好某个行业的发展，但作为门外汉很难选择具体股票。此时，行业指数基金就提供了投资该行业的便捷途径。反之，应避开那些长期处于下行趋势的指数，例如钢铁指数自2007年以来整体下行，尽管可能存在短期机会，但长期下跌的概率较大。

3.  **留意指数的主观程度**。现在有越来越多的创新指数，它们往往与特定的投资策略挂钩，主观性更强。例如，**中证价值回报量化策略指数**就是按照主观的财务参数和价值回报策略筛选编制而成。这类指数更像是主动型基金和被动型基金的过渡体。一般建议在进行被动指数投资时，尽量避开它们。

> 补充说明：指数背后的成分公司并非固定不变，它会保持非常好的新陈代谢，将不符合条件、经营不善的公司剔除，并纳入更优质的股票。

### 定量判断：选择买入时机

讲完了对指数的定性筛选，接下来我们再聊聊如何从定量角度判断当前指数的价格是否适合买入。

生活中有一个朴实的道理：想要赚钱，最好能够低买高卖。投资指数基金也是同样的道理。但具体如何判断一只指数当前是否处于划算的价格区间呢？

有一个简单的指标可以作为衡量指数是否高估或低估的标准，这个指标叫做**市盈率**。

**市盈率（PE）** 是最常用来评估股价水平是否合理的指标之一，其计算公式为：
`市盈率 (PE) = 股价 / 年度每股盈余 (EPS)`

换个角度看，市盈率反映出按照现在的盈利水平，需要多少年才能收回投资成本。

我们假设一个更简单的情况：假如你开了一家馒头店，投入10万元，过去一年赚了1万元，那么这家店的市盈率就是10（10万 / 1万）。这意味着，如果保持每年赚1万元的水平，你需要10年才能收回初始的10万元成本。

现在已经有很多工具帮你计算各大常见指数的市盈率，例如“且慢”平台的每日估值功能、“理性人”网站等。以“且慢”为例，看看如何找到低估的指数：

1.  进入且慢官网，在顶部找到“每日估值”功能。
2.  且慢会对比指数的历史PE数据，列出当前估值所处的水平（如历史高位、中位、低位）。
3.  如果PE值处于历史高位，该指数往往被高估；反之则被低估。且慢会以颜色直观显示：绿色背景代表低估，黄色代表估值适中，红色代表估值较高。你可以选择被低估的指数进行投资。

## 如何找到并选择具体的指数基金？

选好了指数，你可能会问，该如何找到跟踪这些指数的指数基金呢？

一般来说，指数基金都会在基金名称中显示出它所追踪的指数。因此，最简单的方法就是在基金代销平台（如天天基金网）上直接搜索指数关键词。

例如，想找到投资中证500的指数基金，就在天天基金网搜索框输入“中证500”，即可看到在售的相关基金。

此外，还有一个更权威、信息更完整的查找方法：直接在中证指数公司官网搜索感兴趣的指数。

例如，搜索“中证500”，在结果页点击指数名称进入专属页面。在指数主页左下方有一个“相关产品”区域，点击“更多”即可看到所有跟踪该指数的基金产品列表及相关基本信息。

通过搜索，你可能会发现，针对同一个指数，会有多家基金公司发行对应的基金产品。那么，具体的指数基金我们该怎么选呢？

这就好比不同的厨师按照同样的菜谱做菜。我们判断优劣的方法很简单：看这只基金是否精准地复制了“菜谱”（即目标指数）。具体来说，就是查询基金的**跟踪误差**。跟踪误差越小，意味着基金复制指数的能力越强。

判断跟踪误差有一个简单的方法：下载该基金的季报或年报，在其中的“业绩表现”板块寻找相关信息。

以下是在天天基金网查找基金报告的方法：

1.  打开天天基金网，在首页“基金数据”模块打开“基金公告”。
2.  在智能查询中输入你想了解的指数基金代码，点击查询。
3.  在出现的公告列表中，点击“定期报告”筛选出该基金的季报和年报。
4.  在报告中搜索关键词“业绩表现”，找到对应内容。该部分一般会有针对“基金跟踪偏离度”的说明，这个数值就是对基金跟踪指数误差的统计。

**报告期内，跟踪偏离度绝对值越小的基金，就是在该期间复制指数能力越强的基金，可以优先考虑选择。**

## 投资指数基金的三个锦囊

到这里，从了解指数基金到选出适合投资的具体产品，我们就讲完了。但在你正式开始投资旅程之前，还有三个锦囊相送，相信会帮助你更好地完成这场投资。

以下是三个重要的投资锦囊：

1.  **何时卖出？遵循巴菲特的三原则**。股神巴菲特不仅推崇普通人购买指数基金，也对卖出时机提出过三个原则：
    *   **指数过于高估时**：这是最直接的卖出信号。
    *   **基本面恶化时**：就像超市降价处理发霉的馒头，再便宜也不能吃。我们必须选择那些长期趋势向上的指数进行投资。
    *   **有更好的选择时**：就像习惯在某超市买馒头，但隔壁超市搞促销（价格相同但买二送一），就可以换到更优惠的地方。投资也一样，如果出现了代表新科技、新动力的新兴指数，并且你看好它，不妨关注和比较。

2.  **严格遵守投资纪律**。追涨杀跌、懊悔买贵卖便宜是常见心态。最好的应对办法是，在一开始就建立完整的投资方法，明确每种情况下的应对标准。我们前面提到的“看估值”以及锦囊中的卖出标准，就是这样的完整策略。记住一句话：**我们卖出一项投资的唯一原因，是我们买入时设置的理由消失了。**

3.  **和时间做朋友**。所有长期投资策略，最难的不是策略本身，而是坚持。中国股市历史上牛短熊长，有时需要面对漫长的熊市。保持耐心，并确保用于投资指数基金的是长期闲置的资金，才能真正赚到想要的收益。

## 课程总结

本节课中，我们一起学习了指数基金投资的基础知识。

首先，我们认识了什么是指数。**指数**就像一份有明确目的的标准化的“菜谱”，规定了需要的“原材料”（成分股）及其比例。而严格复制指数的基金就是**指数基金**。

其次，我们从定性和定量两个角度学习了如何选择指数。最好选择综合性较强或看好的行业代表指数进行投资，并注意避开指数价格被高估的时点。

最后，当多只基金追踪同一指数时，我们可以优先选择对指数复制效果最好（即跟踪误差最小）的基金进行投资。
