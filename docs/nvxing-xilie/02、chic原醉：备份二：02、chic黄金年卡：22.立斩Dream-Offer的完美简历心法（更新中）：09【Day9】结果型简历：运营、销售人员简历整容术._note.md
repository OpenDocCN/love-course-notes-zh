# 高薪简历整容术：02：结果型简历撰写心法

在本节课中，我们将学习如何为运营、销售等结果导向型岗位撰写一份极具说服力的简历。这类岗位的招聘方尤其关注求职者的工作业绩与成果，因此简历的呈现方式至关重要。

上一节我们介绍了其他类型的简历修复技巧，本节中我们来看看如何针对结果型岗位进行简历“整容”。

## 理解结果型简历

所谓结果型简历，是指企业在评估求职者岗位胜任力时，会特别关注其个人工作业绩的简历。这类岗位相比其他岗位更加结果导向。

根据以上说明，你可以思考身边有哪些岗位属于这种类型。销售是最典型的结果导向型岗位，其销售额直接等同于销售实力。当然，也有很多岗位名称不叫销售，但工作本质是销售，例如医药代表、教育咨询顾问、会销讲师等。

再比如，新媒体领域的用户增长岗位，需要用拉新量证明用户运营能力；商务岗位最终会用可用资源关系量来衡量个人岗位胜任力。

这些岗位的求职简历通常面临一个共性挑战：如何通过一纸简历凸显结果导向的岗位胜任力。为了确保结果型简历设计更具说服力，我们根据对众多HR的采访，整理了结果型岗位的底层胜任力模型。

这种底层能力，如同冰山处于海平面以下的部分，HR不易直接见到，但却是他们最需要、最想挖掘的核心。

接下来，我们将揭秘这个底层胜任力模型，并教你一些能让HR看出你具备这些底层胜任力的撰写技巧。

## 结果型岗位的底层胜任力模型

结果型岗位的底层胜任力模型共包含四种能力素养，从最深层到最浅层分别是：**成就动机**、**探索欲**、**洞察力**和**自控力**。

### 1. 成就动机：模型的根基

成就动机是指你希望从事对自己有意义、存在一定困难和挑战的事情，并且你希望取得完满的结果和成绩，甚至是超越他人。它是整个胜任力模型的根基。

很显然，对于结果型岗位，成就动机直接指向工作结果。若想让招聘人员看到你的成就动机，可以参考以下方法：

以下是三种在简历中体现成就动机的技巧：

*   **直接呈现工作结果**：清晰陈述以往工作经验中的工作成果。
    *   **例如**：`超额完成年度销售任务150%`，`数次获得公司年度销售冠军`，或`实习期6个月，累计拉新60万用户`。
*   **展示数据增长趋势**：利用经历中不断攀升的数据结果，凸显个人敢于面对挑战、追求突破的能力。
    *   **例如**：`经过6个月的连续运营，获客量呈阶梯式增长，月环比增长超过45%`，或`在职期间带领销售团队连续三年创公司销售业绩新高`。
*   **描述推动过程的心力**：成就动机也体现在为加速目标达成所付出的努力。对于运营、商务、销售等岗位，内部协同至关重要。
    *   **例如**：你可以说明为推动目标达成所做的具体事情，如`主动协调财务、法务部门，清晰说明客户时间节点要求，将平均合同评审周期缩短了30%`。

好，分享了结果型岗位底层胜任力模型的根基，我们再来看它之上的第二种底层能力素养。

### 2. 探索欲：成就动机的外化驱动力

探索欲在胜任力模型中的作用，相当于把成就动机外化，转化为向外拓宽业务范围的驱动力。例如探索新的获客渠道、挖掘新的用户增长点、开发新的客户资源以及发现新的方法等。

事实上，只有当我们对达成目标有深厚的兴趣、热爱和好奇心时，才会有主动探索的行为发生。你的探索欲越强，越可能成为该领域的专家。

因此，在简历中彰显探索欲，你需要突出兴趣、好奇心、为此付出的实践以及转化结果。

以下是体现探索欲的表述示例：

*   **示例**：`对用户增长（Growth Hacking）有浓厚兴趣与强烈好奇心，积极探索短视频等新兴获客渠道，并实践出一套高效增长方法论；曾通过优化一条短视频的标题与封面，在5分钟内实现粉丝增长量提升10倍。`

探索欲的最佳表现形式，就是用兴趣和好奇心为自己包装出这个岗位的专家级人设。

结果型岗位的第二种底层能力素养就分享到这里，我们继续分享第三种。

### 3. 洞察力：探索实践中的认知能力

如何理解结果型岗位特别需要洞察力？在生活中，你一定遇到过还不了解你的需求，就直接推销产品的经历。这样的销售通常是新手。

真正的销售高手往往不会一开口就谈产品，他们会先与你谈生活、爱好、价值观等，从中洞察你的真实需求，再将产品优势与之结合，用你认同的价值观说服你，最终促成交易。

所以说，在销售过程中，较好地调用洞察力，可以极大提升成单率。

再举一个例子，通常能把电商运营做好的人洞察力都比较敏锐，他们善于洞察人性根本需求。例如母婴电商运营，他们善于洞察宝妈人群的核心关注点：养育孩子、自我提升、维护夫妻关系，以及强烈的社交诉求。

为此，他们会建立不同属性的社群（如宝宝喂养群、潮妈群、吐槽老公群），通过共同话题培养信赖感，最终在这些社群中完成销售，并利用成员间的信赖进行口碑互推。

以上例子说明，敏锐的洞察力对于结果型岗位非常重要，它相当于将探索欲付诸实践后，对实践过程的深刻认知。

因此，你需要在简历中体现个人的敏锐洞察力，例如对客户需求的洞察、对数据变化的洞察、对行业动态的洞察。具体体现哪方面，需根据岗位内容而定。

以下是体现洞察力的表述示例：

*   **示例1**：`熟悉行业用户增长模式，对用户行为数据变化高度敏感，能敏锐透过数据波动发现潜在问题，并提出有效应对策略。`
*   **示例2**：`善于洞察并把握核心用户需求，据此设计行之有效的用户激励与成长体系，使核心用户群月活跃度提升40%。`

你可以根据以上话术，设计适用于自己岗位的洞察力表述。

结果型岗位的第三种底层能力就分享到这里，下面继续分享最后一种能力。

### 4. 自控力：基本而重要的职业素养

结果型岗位的自控力主要体现在挫败情境下的情绪自控力，以及对职业操守的自控力。

这不难理解。结果型岗位的绩效评判通常比较直接，只有超额完成、完成和未完成三个维度。在面对订单迟迟拿不下、绩效考核未达标时，难免产生负面情绪。但更重要的是如何快速调整，而非自我放弃、转移情绪给客户，或进行违规操作。

情绪调整方法有很多，如认知法、呼吸法等。但对于结果型岗位，非常建议使用“转移注意力法”：立即将关注点转移到问题和解决方法上，而非当下的情绪感受。这种方法的核心思维与结果型岗位的目标导向思维一致，能帮你快速化解情绪并确保效率。

因此，你可以在简历中这样表达情绪自控力：`具备良好的情绪管理能力，遇到困境时通常的化解方法是直面问题本身，积极探索最佳解决策略。`

关于职业操守，我亲身经历过一些案例。例如，一位新媒体运营新手因阅读量和增粉压力，找到刷单渠道完成KPI，最终导致账号充斥“死粉”，打乱公司运营计划，并损害了个人职场口碑。

另外，像销售岗位向客户“吃拿卡要”或私拿回扣都属于违规之举，会严重损害个人和企业形象。因此，职业操守是结果型岗位非常在意的素养，有必要写进简历，它就像你的个人承诺。

话术可以简练，例如：`具备良好的职业操守与责任心，结果导向，能扛起数据增长重担并为最终结果负责。`

虽然自控力在模型中处于较浅层，但并非其重要程度低，而是指这种能力更为基本和普遍。

好了，结果型岗位的最后一种底层胜任力就分享到这里。今天的课程也临近尾声。

## 课程总结

本节课中，我们一起学习了为运营、销售等结果型岗位设计简历的核心心法。

最后，我们一起回顾一下这节课的核心知识点：

1.  **底层能力模型**：招聘方非常看重结果型岗位应聘者是否具备四种底层能力素养，由深到浅分别是：**成就动机**、**探索欲**、**洞察力**和**自控力**。成就动机是根基，探索欲是其外化，洞察力是对具体事物的认知，自控力则是基本的职业素养。
2.  **成就动机体现**：在简历中体现成就动机，你需要清晰写出工作结果、不断攀升的数据趋势，以及为加速目标达成所采取的具体行动。
3.  **探索欲体现**：在简历中体现探索欲，需要用兴趣、好奇心、付出的实践及转化结果，为自己包装出该岗位的专家级人设。
4.  **洞察力体现**：在简历中体现敏锐的洞察力，需要根据岗位具体内容，从对行业、数据、用户需求等层面分别描述你的洞察。
5.  **自控力体现**：自控力包含情绪自控力和职业操守。较好的展示方法是直接说明面对困境时直面问题、探索策略，并承诺为结果负责。
