# 舒化课程：第9节：赋格、意图声明与反意图声明 🧠

在本节课中，我们将学习三种核心的沟通技巧：**赋格**、**意图声明**与**反意图声明**。这些技巧能帮助你更好地理解社交互动中的潜在框架，并提升你的沟通掌控力。我们将通过简单的定义、生动的例子和实际案例，让你轻松掌握这些概念。

---

## 什么是赋格？🎭

上一节我们介绍了推拉等基础技巧，本节中我们来看看**赋格**。赋格是一种沟通技巧，通常指在对方尚未表现出某种特质时，你主动将这种特质“赋予”对方。简单来说，就是提前给对方“戴高帽”。

从字面上理解，“赋”即赋予，“格”即人格。你提前为对方设定了一个你希望他扮演的角色或拥有的品质。

**赋格的核心公式**可以概括为：
`你是一个{正面特质}的人，所以你一定会{做某件符合该特质的事}。`

以下是赋格的几个关键点：
*   它是一种预先设定的框架。
*   常用于引导对方的行为或强化其某种形象。
*   对方通常不愿意否定你赋予的正面人格。

### 赋格实例解析

为了让你更好地理解，我们来看几个例子。

**例子一：教育孩子**
假设你有一个不爱刷牙的7岁儿子。你可以对他说：“无忌，你是一个爱干净的小王子，是打败变形金刚的男神。所以，男神今天晚上一定会乖乖刷牙的，对不对？”这里，你提前赋予了孩子“爱干净的男神”人格，引导他去刷牙。

**例子二：社交对话**
朋友约会迟到，你抱怨说“我要打死你”。对方回应：“你这么温柔，怎么舍得打我呢？你是一个温柔的女神，不会打我的。”这就是对方对你使用了赋格，将“温柔”的人格赋予你，从而软化你的态度。

**例子三：两性互动（案例复盘）**
在我们的教学案例中，学员菲儿遇到一个男生。男生承诺“以后你的口红我包了”。我们指导菲儿说：“口头承诺就是骗人的，除非你现在给我打一个小小的定金。”男生随后发了一个5.2元的红包。
菲儿按照指导，回赠了一个66.66元的红包，并说：“**你的诚意我感受到了，你果然是一个说到做到的好男人。**”
这句话就是典型的**赋格**。在男生行为（发5.2元红包）与承诺（包所有口红）不完全匹配时，我们主动赋予他“说到做到的好男人”这一人格，既抬高了对方，也设立了我们的期望框架。

## 意图声明与反意图声明 🔄

理解了如何为对方“赋格”后，我们来看看如何清晰表达自己的喜好与边界。这就是**意图声明**与**反意图声明**，它们像推拉一样，需要结合使用。

### 什么是意图声明？

意图声明不是简单地说“我喜欢你”，而是具体、准确地告诉对方：**我喜欢你身上的某个特定行为或特质**。

**意图声明的公式**为：
`我喜欢你{做某件事}的样子。` 或 `我喜欢你{拥有某种特质}。`

以下是意图声明的特点：
*   具体而非笼统。
*   为后续的互动埋下种子。
*   类似于“心锚”，能让对方记住并关联到特定行为。

**例子：**
*   “元儿，我喜欢你做笔记时认真的样子。”
*   “菲儿，我喜欢你帮我P图时专注的样子。”
*   对爱付钱的男友说：“我喜欢你付钱时大方的样子。”
*   对游戏玩伴说：“无忌，我喜欢你带我吃鸡时厉害的样子。”

### 什么是反意图声明？

反意图声明是意图声明的反向应用。当对方没有做到你之前声明的“喜欢点”，或做了让你不开心的事时，你可以收回之前的“喜欢”。

**反意图声明的公式**为：
`你居然{没有做某事/做了某事}，我不喜欢你了。`

以下是使用反意图声明的场景：
*   当对方忽略了你声明的喜好时。
*   当对方对你态度不好或拒绝你时。
*   用于表达轻微不满，调整对方行为。

**例子（接上文）：**
*   你曾声明：“我喜欢你带我吃鸡的样子。”之后他玩游戏却没叫你，你可以说：“哼，你居然自己偷偷吃鸡不带我，我不喜欢你了。”
*   更通用的用法：当对方凶你或冷落你时，直接说：“你居然敢凶我/不理我，我不喜欢你了。”

### 两者如何结合使用？

1.  **先种下意图声明**：不断告诉对方你喜欢他的哪些具体行为（如付钱、专注、带你玩）。
2.  **适时使用反意图声明**：当他行为不符时，用“我不喜欢你了”来否定。因为他之前接收了那么多“喜欢”，这一句否定会让他更在意。
3.  **目的**：通过一正一反的声明，潜移默化地“调教”对方，让他明白如何做能获得你的喜欢，如何做会让你不满。

## 核心心态：将男人视为狗来调教 🐕

学会了具体技巧，我们还需要一个核心心态来指导所有行为。请记住这个比喻：**男人是狗，女人是猫**。你需要像调教一只小狗一样去调教男人。

这个心态的核心是：**奖励与惩罚必须分明**。

以下是调教“小狗”的步骤，完全适用于与男人的互动：
1.  **设立规则**：让“小狗”知道什么是对的（如去厕所尿尿），什么是错的（随地尿尿）。
2.  **及时奖励**：当“小狗”做对时，立刻给予奖励（零食、夸奖）。对应到男人，就是当他做了你“意图声明”中喜欢的事时，给予正面反馈（夸赞、热情回应）。
3.  **立即惩罚**：当“小狗”做错时，要立刻给出负面反馈（轻拍、严肃说不）。对应到男人，就是当他让你不满时，使用“反意图声明”或冷淡态度，让他知道“主人生气了”。
4.  **保持耐心**：调教需要时间。不要在他“尾巴还没摇起来”（还没主动认错或讨好）时，你就心软去“跪舔”。请至少冷静几天，让他感受到你的情绪。

记住，能调教好小狗的人，通常也能很好地处理两性关系。关键在于始终掌握“奖励”与“惩罚”的主导权。

## 实战话术：当对方约你时 🗓️

最后，我们学习一个立即可以使用的推拉话术，适用于当男人约你时，即使你内心非常想赴约。

**话术结构**：
1.  **推**：强调规则，抬高姿态。“**你必须要提前预约我哦！**” 或 “你看，就是因为你没提前预约，我明天都没时间了。”
2.  **拉**：给予甜头，让对方感到被重视。“**因为我得准备好，恭候大驾呀。**”

**对话示例**：
男：“说话，明天有空吗？想请你吃饭。”
你：（内心高兴，但表面克制）“哎呀，明天我的行程都满了呢。**你以后要约我必须提前预约哦！**”（推）
你：（稍作停顿后）“不过既然是你约…我挤挤时间吧。**毕竟我得好好准备，恭候大驾嘛。**”（拉）

这句话术能在短时间内让对方情绪产生起伏，从觉得你“难搞”转变为感觉被“特别对待”，从而大大提升你的吸引力和框架感。

## 本节课总结 📚

本节课中我们一起学习了：
1.  **赋格**：提前将某种正面人格或特质赋予对方，引导其行为的技巧。公式：`你是一个{特质}的人，所以你会{行为}。`
2.  **意图声明**：具体表达你喜欢对方某个行为。公式：`我喜欢你{做某事}的样子。`
3.  **反意图声明**：当对方行为不符时，收回你的喜欢。公式：`你居然{行为}，我不喜欢你了。` 需与意图声明结合使用。
4.  **核心心态**：像**调教小狗**一样，对男人做到**奖励分明**，保持耐心，掌握主导权。
5.  **实战话术**：面对邀约时，使用“**必须提前预约我**”（推）+“**我得恭候大驾**”（拉）的组合，制造情绪波动。

请从今天开始，有意识地将这些技巧融入你的日常交流中，特别是赋格和意图声明。通过不断练习，你将能更从容地识别他人的框架，并稳固地建立自己的沟通优势。
