# 瑜伽理疗：第15课：生理期理疗，保持年轻状态

在本节课中，我们将学习一套专为女性生理期设计的瑜伽理疗序列。这套练习旨在帮助缓解经期不适、安抚子宫、促进骨盆区域血液循环，并教导如何在经期后通过温和的倒置帮助身体恢复。课程包含暖身、主题练习及放松环节，确保练习安全有效。

## 暖身与呼吸准备

暖身练习至关重要，它能唤醒身体，为后续体式做好准备，有效预防运动损伤。

### 坐姿暖身与调息

首先，我们以舒适的坐姿开始，专注于脊柱的延展和深长的呼吸。

1.  **简易坐姿伸展**：
    *   坐在垫子上，双腿自然交叉，脊柱挺直，双肩打开。
    *   双手合十于胸前。吸气时，双臂向上高举，感觉脊柱向上延伸。
    *   呼气时，双手缓慢落回胸前。重复此过程，注意手臂上举时肩膀不要紧张。

2.  **猫牛式流动（坐姿变体）**：
    *   保持坐姿，吸气时，胸腔上提，脊柱逐节向上延展。
    *   呼气时，腹部微收，身体轻柔地向前卷动。再次吸气，从尾骨开始逐节卷起脊柱。
    *   呼气时，打开胸腔，双肩下沉。重复此流动，专注感知脊柱的一节节运动。

3.  **颈部与肩部放松**：
    *   双手向后撑地，指尖朝后。双肩外旋，下巴微抬，在此保持深呼吸。
    *   将左肩向上提起，下巴抬高，感受左侧颈部的伸展。换右侧重复。
    *   这个练习有助于释放肩颈压力，提升免疫力。

4.  **侧腰伸展**：
    *   右手向上吸气，感受右侧臀部和手臂的侧向拉长。
    *   呼气时，手臂带动身体向左侧弯，感受右侧腰的伸展。保持几次呼吸后回正，换左侧重复。

5.  **坐姿扭转**：
    *   右手抵住左膝外侧，左手向后撑地，吸气延展脊柱，呼气时身体向左后方扭转。
    *   保持呼吸，感受脊椎的旋转。回正后换另一侧。

### 肩颈与手腕激活

上一节我们活动了脊柱和侧腰，本节中我们重点激活肩关节和手腕，为支撑类体式做准备。

1.  **肩部绕环**：双手指尖搭肩，以肩关节为轴，向前向后缓慢画圈，活动肩颈。
2.  **手腕热身**：
    *   来到四足跪姿，先活动一下手腕。
    *   将一只手立起，用另一只手轻轻将每个手指向后拉伸。
    *   握拳，然后用力张开手指，重复数次，确保手腕温暖灵活。

### 猫牛式与脊柱活化

现在，我们进入经典的四足跪姿，通过猫牛式进一步灵活整条脊柱。

1.  双手分开与肩同宽，十指张开压实地面，手肘微屈不锁死。双膝分开与髋同宽，脚背贴地。
2.  吸气时，抬头提尾骨，腰部自然下沉，脊柱逐节延展。
3.  呼气时，收腹拱背，低头看向肚脐，像生气的猫一样。重复5-8次，感受脊柱的波浪形运动。
4.  接着，加入颈部侧倾：保持四足跪姿，将头缓慢转向左侧，回正，再转向右侧。重复几次。

完成脊柱活化后，让身体休息一下：臀部向后坐向脚跟，手臂向前延伸，额头触地，进入婴儿式放松。

## 生理期主题理疗练习

暖身结束后，我们进入本节课的核心——针对生理期不同阶段的理疗练习。

### 经期不适舒缓练习

如果你在生理期感到疼痛或不适，以下练习可以帮助安抚子宫、放松背部。

1.  **抱枕婴儿式（缓解剧痛）**：
    *   在身前放一个抱枕。从四足跪姿开始，双膝分开略比髋宽，大脚趾相触。
    *   呼气时，臀部缓慢向后坐向脚跟，将胸部和下腹部完全放松在抱枕上，头转向一侧休息。
    *   在此保持自然呼吸，彻底放松腹部和整个身体。几分钟后，可将头转向另一侧继续休息。
    *   **核心概念**：这个体式通过温和的前屈和支撑，为腹部创造温暖、安全的空间，有效缓解子宫痉挛。

2.  **深蹲式与扭转（促进循环）**：
    *   如果你没有疼痛，只是希望促进盆腔血液循环，可以练习深蹲。
    *   从站立慢慢下蹲，双脚分开与垫子同宽。手肘抵住膝盖内侧，帮助打开髋部，保持背部挺直。
    *   吸气延展脊柱，呼气时，可以双手抱头，身体微微前倾，感受背部伸展。
    *   在此姿势中，可以将一只手放于地面，另一只手推对侧膝盖，进行温和的扭转，进一步放松腰背。

**重要原则**：生理期内，应避免任何使骨盆高于心脏的倒置体式（如头倒立、肩倒立）。

### 经期后恢复练习

生理期结束后的一周内，适合进行温和的骨盆倒置练习，有助于清除残留代谢物。

1.  **辅助桥式肩倒立**：
    *   仰卧，在骶骨下方垫一个厚抱枕或折叠的毯子，使骨盆略高于心脏。
    *   双腿伸直或弯曲，双手可抱膝，自然呼吸。这是一个非常温和的倒置。
    *   **代码描述**：`体式目标 = 骨盆轻微倒置 && 心脏高度 > 骨盆高度`

2.  **束角式前屈**：
    *   坐立，脚心相对，膝盖向两侧打开。双手抓住脚趾，吸气挺直脊柱。
    *   呼气时，从髋部向前折叠，停在感觉舒适的位置。保持呼吸，感受腹股沟和骨盆底的放松。

练习结束后，记得补充温暖的食物，如红糖姜茶，并使用暖水袋热敷腹部，让子宫保持暖和。

## 放松与修复序列

完成主题练习后，需要通过一系列舒缓的体式让身体彻底放松，整合练习效果。

以下是全身放松的序列：

1.  **坐姿侧伸展**：坐立，右手向左上方延伸，左手推地，感受左侧腰伸展。换边。
2.  **仰卧穿针式**：仰卧，左脚踝放在右膝上，双手抱住右大腿后侧拉向胸腔，伸展左臀。换边。
3.  **仰卧手抓脚趾伸展**：仰卧，抬起右腿，用手或毛巾勾住脚掌，保持腿伸直或微屈，伸展大腿后侧。换边。这是缓解下背痛的关键。
4.  **快乐婴儿式**：仰卧，双膝打开靠近腋窝，双手抓住脚掌或小腿，轻柔地左右摇晃，放松下背和髋部。
5.  **仰卧束角式**：仰卧，脚心相对，膝盖下沉，双手放松在身体两侧。这是极佳的修复体式。

### 摊尸式深度放松

最后，进入最重要的**摊尸式**，进行至少5-10分钟的深度放松。

1.  仰卧，双腿伸直分开与髋同宽，脚尖自然外八。
2.  双臂放于身体两侧，与躯干成45度角，掌心朝上。
3.  有意识地放松身体每一个部位：从脚趾、脚踝、小腿……一直到头皮。
4.  关闭视觉，将注意力完全放在自然深长的呼吸上，让身心完全沉向地面。
    *   **公式描述**：`放松状态 = 零肌肉紧张 + 自然呼吸节奏 + 意识内观`

结束时，先动动手指脚趾，将意识慢慢收回。身体转向右侧卧，用左手推地，缓慢坐起。
