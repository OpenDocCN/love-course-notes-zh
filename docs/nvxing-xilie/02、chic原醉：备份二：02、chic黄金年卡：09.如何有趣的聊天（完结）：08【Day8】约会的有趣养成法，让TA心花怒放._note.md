# 幽默感养成：02：约会中的幽默感，让TA心花怒放

## 概述
在本节课中，我们将学习如何在约会中运用幽默感，为双方创造快乐的回忆。我们将重点掌握三个核心技巧：抓取幽默细节、学会打趣以及制造假投降。

## 抓取幽默细节
上一节我们概述了幽默感的重要性，本节中我们来看看如何从对话中抓取幽默细节。一项来自堪萨斯大学的研究发现，约会中双方的欢乐越多，彼此成为潜在亲密对象的可能性就越大。因此，即使在紧张的情况下，一个笑容也能为约会带来良好开端，并化解误解。

为了让对方笑起来，你需要学会从对方的话语中发掘幽默细节。有些人天生幽默，例如学生时代总有人能在老师讲话的间隙插科打诨。这种技巧的核心是“咬话尾”。

### 什么是“咬话尾”？
“咬话尾”指的是利用对方最后一句话的点来制造幽默，通常是对原话进行曲解或夸张的发挥。这与常见的“抬杠”不同。抬杠通常是各执己见的争论，而“咬话尾”的目的并非争胜，而是为了逗乐大家。请注意，切勿让“咬话尾”演变成斗嘴。

**公式**：`幽默回应 = 曲解/夸张(对方最后一句话)`

以下是“咬话尾”的一个例子：
*   **场景**：男孩搭讪女孩。
*   **女孩说**：“看来你挺喜欢搭讪嘛。”
*   **男孩回应**：“看来你也挺享受被搭讪吧。”

这个回应抓住了女孩话尾的“搭讪”一词，进行了巧妙的发挥。学会“咬话尾”，就能在对话中持续制造幽默。因此，在约会中寻找有趣线索，应从仔细聆听对方的话语开始。

除了“咬话尾”，你还可以尝试“反转语序”来制造笑点。

### 什么是“反转语序”？
“反转语序”是指对方陈述一个现象，你则提出一个能扭转该现象的说法。

以下是“反转语序”的例子：
*   **对方说**：“我的脸型戴什么眼镜都好看。”
*   **你可以说**：“我的眼镜什么脸型戴上都好看。”
*   **对方说**：“怎么每次看到你，你都在打游戏，你有没有追求啊？”
*   **你可以回答**：“每次打游戏都会看到你，你真会挑时间啊。”

如果你不擅长“咬话尾”或“反转语序”，可以从对方感兴趣的事物入手，反复测试以发现其笑点。

例如，当聊到喜欢的明星时：
*   **对方说**：“我喜欢周杰伦。”
*   **你可以说**：“我也很喜欢他，不信你看我模仿他唱歌给你看。”（随即模仿周杰伦的嗓音表演）
    *   **关键**：无需模仿得惟妙惟肖，抓住特色即可有趣。
*   之后，当你发现对方在某方面很优秀时，可以用周杰伦的标志性语言夸赞：“哎呦，不错哦。”

这就是从对方兴趣点入手，共同互动创造的乐趣。如果对方喜欢相声小品，你们甚至可以邀请彼此进行即兴表演，相互捧逗。

综上所述，寻找幽默细节并不难，核心在于认真倾听，发现对方的兴趣所在。

## 学会打趣对方
在掌握了如何发现幽默线索后，我们来看看如何主动制造幽默，即“打趣对方”。在两性交往中，打趣意味着以玩笑的方式与对方互动。

以下是打趣的几个方法：

### 1. 踩边线挑逗
这种方法通过制造误解、引导对方深入，最后揭露真相并（假装）谴责对方来产生幽默。

**步骤**：`制造误解 -> 引导深入 -> 抖包袱揭露 -> (假装)谴责`

*   **例子**：你问女孩：“什么东西是世界上最硬、女孩最喜欢，特别是结婚的女人爱死了？”
    *   在女孩浮想联翩时，你告诉她：“天哪，你想哪儿去了？这个东西就是钻石啊！”

**注意**：踩边线技巧利用了典型的幽默结构——制造悬念、积累、放大，最后释放以产生笑点。但请注意，涉及“荤段子”的踩边线在关系不够亲近时存在风险，需谨慎使用。

### 2. 吐槽对方特点
在互动中，可以幽默地吐槽对方的特点，包括说话方式、标志性动作等。例如，吐槽对方平翘舌不分、经常撩头发或走路不爱摆胳膊。

**关键**：约会中的打趣目的是拉近关系，而非制造隔阂。因此，在“黑”完对方后，务必用温柔的情话化解，让对方明白你并无恶意。

**化解话术示例**：“你太特别了，我真的没有见过比你还可爱的人。”

### 3. 从过往故事着手
谈论对方过去的经历时进行打趣，可以在愉快的分享中增进了解。
*   谈论对方早恋时，可以打趣他“成熟的早”，并在对方可能紧张时说：“我也想这么早的时候就认识你啊。”
*   吐槽对方小时候的糗事后，可以说：“你小时候真的好可爱啊。”

**核心原则**：两性打趣的关键在于 **先制造紧张，再用情话化解**。这能有效促成亲密感。

然而，在成为亲密伴侣前，打趣需要适度：
1.  **切忌用力过猛**：玩笑如同菜肴里的葱花，适量可提鲜，过量则令人厌烦。在约会中，如果从头到尾都在取笑对方的槽点，会显得过分。最好只选取对方鲜明且无伤大雅的特点。
2.  **注意对方反应**：如果对方无法理解你的笑点，请及时停止。并非人人都适合被频繁打趣。在了解不深时，应由浅入深，循序渐进，才能最终“笑到一起”。

## 制造假投降
最后，我们来看看如何在约会出现争论时，运用“假投降”来幽默化解矛盾。“假投降”主要用于争论场景，其核心是“先抑后扬”。

### 什么是“假投降”？
“假投降”指先对对方的要求或谴责表示让步妥协，然后再进行反击或解释。有时也表现为“假承认”，常用“是啊，不过……”的语序。

**公式**：`回应 = 假意承认/妥协 + 幽默解释/反击`

以下是两种“假投降”的例子：

#### 类型一：谴责反讽（更具攻击性）
*   **例子**：于丹教授讲座后，有观众质疑其穿着过于西化，应穿中式服装讲《论语》。于丹幽默反问：“是啊，你看我现在裹小脚还来得及吗？”
*   **分析**：这是基于对方不当预设的“假承认”加反问，带有反讽意味，攻击性较强。

#### 类型二：向谴责求和（适用于约会）
*   **例子**：因堵车约会迟到半小时，对方很不高兴地说：“第一次见面你就迟到了。”
    *   **错误回应**：试图讲道理说明原因。
    *   **正确回应（假投降）**：“是呀，昨天晚上做了一宿关于你的梦，没有睡好就迟到了。”
*   **分析**：这里的“假投降”是向对方求和，传递的信息是：“是的，抱歉我迟到了，但绝不是因为我不在意你。”这成功地将矛盾转化为甜蜜的互动。

**对比总结**：约会中的“假投降”应侧重于 **“向谴责求和”** ，即表达“我错了，不过你误会了我的初衷，我们和好吧”，而非带有攻击性的“谴责反讽”。当出现争论时，不要急于讲道理，先假装投降再用幽默化解，才是维持约会氛围的妙计。
