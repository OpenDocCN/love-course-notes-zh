# 快速消费品行业揭秘：02：快消行业的周边机构：唇齿相依的第三方专业机构

在本节课中，我们将要学习快消行业中那些至关重要的第三方专业服务机构。它们与快消公司紧密合作，共同构成了完整的商业生态。

上节课我们介绍了快消品行业的五种制胜模式。本节中，我们来看看那些与快消行业唇齿相依的第三方专业机构，以及它们带来的机遇。本节课内容将分为三个板块：调研公司、广告与代理机构、数据公司。它们都属于第三方服务公司，专注于产品周期的前期研发、中期推广运营和后期数据分析，是消费品公司业务的补充和智慧外挂。

无论是调研公司、广告与代理机构还是数据公司，都是快消品业务强有力的补充。这三个方向的从业者也是消费品公司青睐的人才库。如果你有这三个方向相关的积累，再进入快消品公司，尤其是市场部，会非常有竞争力。

## 调研公司：洞察消费者的眼睛

无论什么消费品公司，最重要的任务就是要满足消费者的需求。因此，找到需求，即获得消费者洞察，是工作的重中之重。调研公司就是在产品周期的前期开发中，帮助消费品公司来获取消费者洞察。

调研公司具体会通过技术手法和专业知识来帮助消费品公司开展消费者调研。以下是其典型工作内容：

*   **产品概念测试**：评估新产品的市场接受度。
*   **品牌传播主题测试**：检验广告语或品牌信息的有效性。
*   **包装测试**：分析不同包装设计对消费者的吸引力。

调研公司会依据需求去召开访谈和测评。访谈一般是获取定性信息，测评则多半是定量的分析。通过访谈和测评，调研公司会一层层地挖掘消费者表达之下内心真正的需求。

调研领域的入门门槛并不是很高。很多调研公司的从业人员在积累了足够的专业知识和客户人脉之后，就自主创业开办小型调研公司。由于进入门槛不高，调研公司也是一个红海市场，各家之间没有非常本质性的竞争差异，常常流于人脉限制和价格竞争，起步虽容易，但做大做强很难。

想要进入这个行业创业的同学，可以先进入一家比较大的调研公司，如 **Ipsos（益普索）**、**Nielsen（尼尔森）** 等，全面接触和掌握调研的技术和工作流程。这个过程快则两三年，慢则五到八年，基本上可以在专业层面过关。在企业内工作的中后期，需要比较留心甲方资源，维护好客户关系，就很容易自立门户。当然，由于调研公司提供的服务比较同质化，这也注定是一个“小而美”的生意。但在这一行中积累起来的许多能力，像是对消费者的洞察能力、对整个市场的分析和把控，都是未来去到正规的快消公司或创业做别的生意所必备的能力和素养。

## 广告与代理公司：品牌与市场的桥梁

接下来我们来讲一讲产品周期中期的智慧外挂——广告与代理公司。相信大家对广告都不陌生。对于快消品公司来说，公司最关心怎么做出产品、怎么卖出去和获得利润。品牌和宣传只是这长长链条中的一个环节。当我们把目光聚焦到市场部，他们需要获得市场中的洞察，需要在内部调动各种资源把产品生产出来，那么在宣传的部分就需要借助更多外部的力量了。所以广告和代理公司也是高度商业化社会中催生出来的专业服务机构。

广告公司一头对接甲方（快消品公司），一头对接各个媒体和活动。他们的日常工作就是在这中间匹配资源和创造价值。以下是其典型工作流程：

1.  **策略讨论**：与甲方一起讨论用什么方式抵达产品的目标人群。
2.  **方案提案**：例如提案邀请某位符合品牌调性的明星代言。
3.  **内容制作**：拍摄广告片等宣传物料。
4.  **媒体投放**：将广告投放到电视、网络、户外、手机等各个媒体。
5.  **活动执行**：在重要时节，于核心地段举办线下活动，营造声势。

广告公司创造的价值就在于，在快消品的策略指导下，为甲方寻找到最合适的媒体资源，用他们的专业知识提出最高效的媒体方案。

这里需要解释一下广告公司和代理公司的区别。广告公司承担的职责更多，比如广告片的拍摄、投放和策略等。而代理公司则更专精于对接媒体资源。例如，一个大品牌想要在爱奇艺、优酷、腾讯、央视、微信朋友圈等平台投放广告，品牌方很难有精力去研究每一个媒体生态的适配方式和价格，代理公司就专门帮品牌方处理媒体端的事情。由于代理公司汇集了很多甲方需求，通过他们去接触媒体也经常能获得更好的价格。

因此我们能发现，代理公司需要在大小媒体都有资源，进入门槛更高。而广告公司只需要在一些领域有专业能力，就可以服务于品牌方。在市场上，代理公司已是几家大机构覆盖，比如 **Starcom（星传媒体）**、**Mediacom（竞立媒体）** 等。但是广告公司则是林林总总，数不胜数，有的在社交媒体比较强，有的专注于创意，有的专门拍广告片。

广告公司的竞争也比较激烈，像调研一样，起步容易但做大很难。进入门槛和调研也非常类似：先进入一家大型国际4A公司，从专业层面积累创意、策划、文案、媒介等知识，然后在客户积累到一定程度，就有可能自行创业。当然，由于从事这一行的同学普遍对于品牌解读和塑造更加擅长，未来在做新的生涯规划时，一切和品牌相关的工作，如品牌策划、品牌推广等，都是可以参考的方向。

## 数据公司：洞察市场的雷达

最后我们来讲讲数据公司。这要从消费品的生意模式说起。消费品是我们每个人日常生活都会购买使用的东西，所以消费品的消耗非常广泛和分散。对于消费品公司来说，他们知道每个大客户（如沃尔玛、大润发、全家便利店）一个月下了多少单。但是一旦产品摆上了货架，到底有多少产品被消费者买走了？这些厂商们可就不知道了。

你可能会问，为什么消费品公司想要了解有多少产品被消费者买走了呢？难道不是客户下了单，拿回货款就可以了吗？的确，客户下单回款是对于消费品公司来说生意的第一步。但是产品要被消费者买走、真正被用掉，才会长久地支持生意增长。否则客户下单的产品最后卖不掉，还是可能找厂商退货的。产品只有完成从厂商到消费者的闭环，才是生意的闭环。

所以，数据公司就是帮助消费品公司来监测产品从摆上货架之后，最后卖掉了多少。这也就是产品后期数据分析的外部服务。

你可能会问，厂商既然把东西卖给了客户，那客户很清楚什么被买走了呀，为什么厂商不直接问客户呢？有两个原因。第一，对于厂商来说，客户是甲方，客户下单买了东西之后的产品周转事情是厂商想了解的，但对于客户来说并不是首要任务，所以客户并不会额外花时间去帮助厂商整理那些终端数据。第二，厂商不仅想知道自己的产品周转，也想了解竞争品牌的终端销售情况，而这些数据，客户就更不可能披露了。所以数据公司作为专业服务公司就应运而生。

数据公司获取数据的方式主要有三种：

1.  **战略合作**：与大型连锁超市战略合作，直接进行终端数据的交换。这样的数据最准确。
2.  **技术抓取**：针对很多电商平台，通过技术手段从网页上抓取数据。
3.  **人脉网络**：公司内专业人士通过自己的人脉网络获取数据。

最后他们服务的呈现形式就是各种各样的数据库和分析报告。快消品的数据服务公司有四大巨头：**尼尔森（Nielsen）**、**凯度（Kantar）**、**益普索（Ipsos）** 和 **欧睿（Euromonitor）**。在很多公开报道里，也经常会引用来自这四家公司的数据。

从数据公司的介绍中，我们会感知到，数据公司是一个进入门槛比较高的细分行业，非常规系统驱动。现在快消品的数据服务也相对比较饱和，不是一个好的创业方向。但数据公司从业者非常重要的核心竞争力在于信息。他们对全行业的信息有一种天然的敏感性，能够准确感知市场的风向。对于很多公司来说，他们需要具备这种能力的人来当做信息终端。因此，他们的就业天花板也远远不止于数据公司本身，而是能够被迁移到许多领域，比如快消行业的市场部门、销售部门等。

## 总结

本节课中，我们一起学习了快消行业的三大周边服务机构。

*   第一个是帮助快消品公司获得消费者洞察的**调研公司**。
*   第二个是作为快消品外脑和手的**广告与代理公司**。
*   第三个是帮助了解零售终端竞争格局的**数据公司**。
