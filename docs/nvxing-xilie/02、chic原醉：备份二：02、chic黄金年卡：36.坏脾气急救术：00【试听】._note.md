# 坏脾气急救术：02：为什么你总掉入发脾气的陷阱？

在本节课中，我们将要学习坏脾气的成因。很多人认为坏脾气是天生的，但事实可能并非如此。我们将从科学和心理学角度，探讨坏脾气形成的后天机制，帮助你理解自己情绪反应的根源。

## 坏脾气的普遍影响

坏脾气往往是导致悲剧的主要原因。

影视作品中不乏其例。《都挺好》中，脾气暴躁的苏明成打妹妹、打老婆，最终将自己送进了看守所。《武林外传》中，郭芙蓉天天暴打吕秀才，差点导致秀才移情祝无双。

这类事情不只体现在影视剧中，在生活中同样如此。一个坏脾气的人，总容易使事情往坏的方向发展。他们还像一颗定时炸弹，一不小心就会炸伤身边的朋友或恋人。

## 改变坏脾气的常见困境

许多暴脾气的人自己也很着急，想试着变成一个知书达理的人，但总是感觉很难。坏脾气仿佛是天生的属性，甩也甩不掉。

本节课我们首先来解决第一个问题：为什么你总掉入发脾气的陷阱？难不成真是天生暴脾气？

## 先天论：星座与八字的解释

有些人相信八字或星座的解释。八字可能说“我坐下七杀，性格刚烈，脾气不太好”。星座则可能告诉你“我太阳是白羊，上升是狮子，月亮是射手，基本上集齐了火象星座的暴脾气特征”。

因此，结论常是“我要拯救我的坏脾气，基本是天命不可为”。我们暂且不考究星座八字的科学性，而是从更科学的角度来看待这个问题。

## 科学视角：先天与后天的影响

科学研究发现，只有少数文献提到脾气会随基因遗传。例如，《认知神经科学杂志》曾发现，与坏脾气相关的自我控制基因是有可能遗传的。

但是，更多的研究倾向于相信，**后天环境**会给坏脾气带来更大的影响。这也是为什么有的父母脾气差，而孩子的脾气却正好相反。

既然坏脾气主要是后天形成的，那它具体是怎么形成的呢？

## 核心概念：情绪模式的形成

接下来，我们引入一个核心概念：**情绪模式**。

它是指某个人因受到**同一因素**持续刺激时，逐渐形成的一系列固定反应模式。

为了让你更好地理解，以下是几个具体的例子：

*   **案例一：创业者的反应**
    我的一位朋友曾决定去北上广创业，家人却强烈反对，他因此和家人起了很大冲突，关系一度紧张。如今他已是公司老板，但每当他的计划或决定被反对或终止时，他就会变得激动、气愤，并且固执己见。
    *   **长期刺激因素**：被不断否定和反对。
    *   **形成的情绪模式**：一被反对就生气。生气逐渐变成了他的固定反应。

*   **案例二：同事的请求**
    某位同事平时不会找你，但只要找你，就会拜托你处理一些棘手的工作。因此，他每次来找你，你就会立刻警惕、怀疑，心生戒备。
    *   **长期刺激因素**：对方出现即意味着麻烦。
    *   **形成的情绪模式**：一见此人便自动进入戒备状态。

## 另一种后天成因：代际传递

除了个人经历，还有另一种后天形成的状况，叫做**代际传递**。

你从小会耳濡目染父母处理事情的方式。如果父母遇到问题时最容易发脾气，那么你在成长过程中，也可能因为**找不到更合适的解决办法**而模仿性地选择发脾气。

这里所说的代际传递，主要指对父母**行为和心理的模仿**，而不只是先天基因上的遗传。
