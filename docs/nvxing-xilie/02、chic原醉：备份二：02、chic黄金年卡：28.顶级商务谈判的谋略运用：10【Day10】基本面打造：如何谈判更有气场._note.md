# 谈判哲学：02：顶级商务谈判的谋略运用：10【Day10】基本面打造：如何谈判更有气场 💼

在本节课中，我们将要学习如何在商务谈判中塑造强大气场。我们将探讨气场的内涵、构成要素，并提供具体、可操作的修炼方法，帮助你在谈判桌上展现自信从容的姿态。

上一节我们介绍了谈判的内功心法，本节中我们来看看如何将这些内在能力通过外在表现发挥出来，即打造你的谈判气场。

我们常说的“气场”是一个复合概念。它既包含外在的穿着打扮、妆容造型带来的整体感受，也包含内在的自信从容状态。多种因素综合在一起，形成了每个人独特的气场。这种微妙的直观感受虽不直接决定谈判成败，但其产生的蝴蝶效应不容忽视。例如，在商场购物时，导购员的亲和力可能让你对其推荐的产品产生连带信任感。

我们发现，在谈判或日常生活中，能塑造良好印象的气场主要包含以下几个元素。

以下是气场构成的三个核心要素：

1.  **精致得体的妆容和造型**
    有一句话叫“You are what you wear”（衣如其人）。妆容和造型传递着丰富信息：你的角色、对对方的态度、自身审美、公司实力与行业属性等，都能通过整体造型传递出来。例如，金融从业者与媒体人即使同为90后，散发出的气场也截然不同：前者更显精致干练，后者则相对随性自然。

2.  **柔和坚定的表情和眼神**
    眼神和表情在谈判中起到无声传递信息的作用。你的态度是激进强势还是胆怯懦弱，通过表情一看便知。谈判是合作中夹杂对抗的特殊场合，因此表情管理尤为重要：既不能过于凌厉强势，也不能唯唯诺诺。谈判开始时，最佳状态是自信而柔和的微笑。面对对方时，眼角可稍稍弯曲，以吸纳代替进攻，这能有效减少双方的紧张僵持感。谈判过程中，可根据具体情境，通过表情表达感兴趣、疑惑、坚定等多种态度。

3.  **逻辑清晰、简练大方的语言表达**
    语言是传递信息的最直接途径。表达的逻辑性和连贯性，既能反映你解决问题的思路是否清晰，也直接体现你对场面的把控能力。一个语言节奏和逻辑俱佳的人，总能轻松引领整场谈判的气氛和走向。反之，吞吞吐吐则容易显得立场不坚定。即使表达相同意思，自信与否也会带来完全不同的感觉。例如：
    *   **第一句**：`这个报价稍微有点太高了，我们可能有点难以接受。`
    *   **第二句**：`当前的报价超出了我方的预算，我们可能需要重新梳理一下。`
    两句话都表示价格太高无法接受，但第二句明显更自信。原因在于其语义更聚焦，在表达立场的同时明确了自身诉求，并且避免了“稍微有点”等模棱两可的词汇，整体感觉更加坚定和诚恳。

综合以上三点，气场从某种程度上是气质和立场的代名词。只要我们抓住重点并持续练习，就能有效提升。

为了帮助大家系统性地塑造强大气场，以下是三个具体的修炼步骤：

*   **第一步：寻找风格偶像，进行角色扮演**
    对于初学者，有一句箴言屡试不爽：**Fake it till you make it**（假装成功，直到真正成功）。你需要在心里树立一个想要成为的形象（风格偶像），并想象自己就是那种形象。例如，你可以联想董明珠、电影《穿Prada的女王》中的角色，或干练的影视形象如苏明玉、唐晶等。锁定形象后，尝试在妆容、服装、动作姿态上模仿。例如，建议年轻娇小、想修炼知性气场的学员模仿法国女性苏菲·玛索，通过深色高级感装束衬托气质，并训练坚定优雅的体态与表情。模仿是一个突破自身瓶颈、调试并最终形成个人风格的过程。

*   **第二步：语言方面，提前梳理逻辑，优化表达**
    对于语言沟通有障碍的同学，唯一的方法是提前设计表达内容。建议在谈判前一天，写下谈判的逻辑顺序，包括：
    1.  **谈判主题**
    2.  **谈判目的**
    3.  **对方身份**
    4.  **主要矛盾点与解决方案**
    这能帮助你理清思路，减少无用表达。针对每个主要矛盾点（如涨价原因、增值服务等），预先准备好应对话术。例如：
    > `选择20%的价格增幅，主要是出于市场环境方面的考虑。`
    > `我们的增值服务前期已得到市场正向验证，用户评价和反馈都在这里。`
    同时，要注意明确表达自身需求，并戒掉“然后”、“其实”、“就是”等语气助词或口头禅，这类词语频繁出现会显得准备不足、底气不足。

*   **第三步：表情神态方面，管理体态与微表情**
    体态和气质虽需长期积累，但可通过训练改善。例如，每天抽半小时进行背部和手臂的力量训练，以具备良好的身体形态，在人际交往中自然做到抬头挺胸。在此基础上，一定要**微笑**。微笑能有效化解生硬尴尬的气氛。伴随微笑缓慢点头，表示你在倾听并对内容感到好奇，能从心理上拉近与对方的距离。最后，注意**减少眨眼频率**。在交谈中频繁眨眼也是一种不自信的表现，控制眨眼频率能让对方感受到你坚定的态度。

本节课中我们一起学习了谈判气场的塑造方法。我们明确了气场的三个关键点：精致得体的妆容造型、柔和坚定的表情眼神、逻辑清晰简练大方的语言表达。为了达成这三点，我们提出了三个核心建议：寻找风格偶像进行角色扮演、提前梳理语言逻辑与表达、以及通过体态训练和微表情管理来优化神态。至此，我们全套谈判哲学课程就全部结束了。希望大家未来能在真正的谈判中，熟练运用内在能量模型与外在气场塑造方法，从容应对各种挑战。
