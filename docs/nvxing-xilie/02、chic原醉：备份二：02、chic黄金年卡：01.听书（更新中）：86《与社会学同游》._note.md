# 社会学入门：02：与社会学同游：人文主义视角解读

在本节课中，我们将一起学习彼得·伯格的经典著作《与社会学同游》。我们将探讨社会学的基本概念、人与社会的关系，以及个人如何在社会的结构性约束中寻求自由。通过四个核心部分，我们将理解社会学如何作为一种思维方式，帮助我们更清晰地审视我们所处的世界。

---

## 什么是社会学？社会学的议题是什么？

上一节我们概述了本书的主题，本节中我们来看看社会学的定义及其核心议题。

社会学是西方历史特定阶段产生的科学，它并非像哲学一样源远流长，而是近代社会变革的产物，起源于19世纪末期。社会学使用各种研究方法进行实证调查和批判分析，以发展和完善一套有关人类社会及活动的知识体系。

**社会学** 的公式化描述可以理解为：
```
社会学 = 实证研究 + 批判分析 -> 社会知识体系 -> 社会福利改善
```

作者彼得·伯格继承了马克斯·韦伯的社会思想并有所创新。他认为，社会学既是方法论意义上的科学，也是人文意义上的人文学科。简单来说，社会学是一种关注一切社会现象和人类行为的思维方式。

以下是社会学关注的四个核心议题：

**揭露真相**
现实生活中亲眼所见不一定是真相。学习社会学能让我们透过现象看到本质，更加理性地对待遇到的事情。例如，社会学对婚姻的解释超越了“爱情的产物”这一浪漫观点，会分析其背后的阶级、收入、教育、种族和宗教等结构性因素。

**不恭敬的态度**
社会可以一分为二：上层阶级与下层阶级。社会学，尤其是历史上的芝加哥学派，常以“不恭敬的态度”揭露社会的阴暗面（如贫民窟生活、犯罪世界），这实际上是一种保持清醒、进行多层次社会认识的意识。

**中性化趋势**
在社会学语境中，“中性化”指的是不标榜某一种社会或文化为唯一标准，而是以变化、宽容、尊敬的态度去对待其他的社会、文化和世界。这体现了现代社会日益开放的思维特征。

**世界眼光**
这指的是拥有开放的意识，对世界开放，对自己的思维和行动开放。优秀的社会学家内心向无限丰富的人类潜力开放，热情追求新视野和人类意义的新世界。

## 为什么把社会比喻成监狱？

上一节我们介绍了社会学的基本视角，本节中我们来看看社会如何对人形成约束。

人在享用各种社会资源的同时，也必然受到诸多限制。作者伯格持有社会决定论观点，认为社会如同监狱，人就像囚徒。法国社会学家涂尔干也指出，社会从内到外给人定位，界定人的角色和身份，从而决定人的命运。

人是通过一系列社会控制机制被管理的：

以下是主要的社会控制机制：

1.  **暴力**：最古老的终极控制手段，是一切政治秩序的终极基础。
2.  **社群控制**：通过道德、风俗和礼节进行约束，违反者可能面临社会性惩罚。
3.  **初级群体控制**：在小型社群（如邻里）中，通过规劝、嘲讽、议论和羞辱进行控制。
4.  **职业系统控制**：一个人的工作决定其大半生的行动范围、社交圈、居住地和地位。
5.  **个人生活圈子控制**：来自亲朋好友的评价和态度，其影响可能比外部控制更为深刻。

此外，**社会分层**（特别是阶级体系）深刻影响个人命运。一个人的经济条件决定其阶级，而阶级地位又影响着生活水平、医疗条件、子女教育等方方面面。虽然存在阶级流动的可能性，但良好的家庭出身往往能为后代提供更高起点。

**社会情境**和**制度**也是理解社会结构的关键。社会情境指社会环境如何定义并影响人们的行为和人生策略。制度（如法律、婚姻、宗教）则是一套特色鲜明的社会行为规范，引导并塑造着人们的行为和期望。

## 为什么说人是在木偶剧场里表演？

上一节我们探讨了社会作为外部监狱的约束力，本节中我们来看看社会控制如何内化于人自身。

尽管社会枷锁使人压抑，但人们能够承受，因为人们渴望得到社会赋予的身份和角色来体现自我价值。为了解释这一现象，作者提到了三种理论。

以下是三种解释社会内化于个人的理论：

**角色理论**
角色是对典型期待的典型回应。社会提供剧本（模式），个人则根据剧本在具体情境中演出。例如，一名新晋军官会逐渐适应并维护其军官的角色和风度。身份是社会赋予的，并通过社会互动得到支撑和转换。

**知识社会学**
这是研究思想社会定位的学科，探讨“谁在说什么”。**意识形态** 就是一种为社会既定利益服务的思想，它塑造人们的观念以符合社会期许。例如，美国南北战争前，南方编造的“种族神话”旨在让黑人奴隶甘于被奴役。

**参照群体理论**
该理论在角色理论与知识社会学之间架起桥梁。参照群体为我们提供行为范式，我们通过与之对比来调整自己的行为。例如，大学生通过比较和选择，加入能带来归属感的社团或小组。

通过这些理论可以看出，人就像在一系列不间断的剧场中表演的木偶，被肉眼看不见的社会“木偶线”操纵着。作者认为，我们受制于社会的枷锁，但这个枷锁并非完全由外力强加，而是我们与社会“合谋”打造的，我们常常心甘情愿地跳入社会的陷阱。

## 人如何逃离社会的枷锁，寻得自由？

上一节我们分析了人如何被社会规则所牵引，本节中我们来看看个人是否以及如何能获得一定程度的自由。

生存需要社会的承认以获得自我形象和身份，反之，社会的存在也需要我们的承认。这种相互依存的关系，也为我们提供了抵制社会控制、寻求自由的可能性。

作者伯格提到了三种抵制社会控制、寻求自由的手段：

**变革**
历史上的每一次社会变革都是对社会控制的反抗。通常，在制度被暴力推翻前，统治者在人们心中的合法性与意识形态支持已然瓦解。

**巧妙利用**
个人可以巧妙利用制度规则进行非正式反抗。例如，利用工作便利办私事。但这属于违规行为，一旦发现会受到惩罚。

**超然**
这是作者重点提及的方式。**超然** 指内心采取退让姿态，与社会角色保持距离，甚至拒绝扮演，进入一种“自我流放”的精神境界。例如，中国古代的老子或希腊的斯多葛学派。在现代社会，某些“另类”群体或特立独行的个人也体现了这种超然态度，他们不受既定社会等级的束缚。

自由必须以一定程度的意识解放为前提。安于现状则自由无望。游离于主流社会之外，独自面对不确定性，正是走出舒适“洞穴”、追求自由的行动。

## 总结

本节课中我们一起学习了彼得·伯格在《与社会学同游》中阐述的核心观点。

首先，我们了解了**社会学**是一门研究社会的科学，也是一种思维方式，其核心议题包括揭露真相、不恭敬的态度、中性化趋势和世界眼光。

其次，我们探讨了**社会如同监狱**的比喻，分析了社会通过暴力、社群、职业系统等多重控制机制以及社会分层、情境和制度，对人形成的层层拘束。

接着，我们通过角色理论、知识社会学和参照群体理论，理解了**人如何像木偶一样在社会剧场中表演**，揭示了社会控制的内化过程以及我们与社会的“合谋”关系。

最后，我们看到了寻求自由的三种可能路径：**变革、巧妙利用和超然**。其中，超然——即内心与社会角色保持距离——被作者视为一种重要的精神自由途径。

社会学为我们提供了一种审视世界与自我的独特视角。它帮助我们理解社会的结构性力量，同时也启示我们，尽管身处社会“牢笼”，但通过意识的觉醒与选择，我们仍有可能获得一定程度的自主与自由。
