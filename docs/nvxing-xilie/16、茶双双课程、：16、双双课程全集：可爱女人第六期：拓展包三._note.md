# 茶双双课程：16：可爱女人第六期：拓展包三

在本节课中，我们将要学习如何识别和应对感情中的焦虑情绪。焦虑是许多人在亲密关系中面临的主要挑战，它可能源于外貌、体重或对关系的过度担忧。本节将通过具体案例，分析焦虑型人格的特征及其对关系的影响，并提供调整心态的建议。

## 认识焦虑型人格

上一节我们介绍了关系中的常见挑战，本节中我们来看看一种特定的情绪模式——焦虑型人格。经过大量的咨询经历和案例观察，我发现感情中出现问题的女性，多数具有焦虑型性格。这种焦虑从外在的外貌焦虑、体重焦虑，延伸到内在的负面猜想，使得伴侣间的小事都可能被视作危机，最终让人际关系变得越来越糟糕。

焦虑型性格的人通常表现出以下主要特征：

以下是焦虑型人格的主要特征列表：
*   **吹毛求疵**：会放大自己身上的缺点。
*   **错误归因**：将生活上的不如意归结于自身的某个缺点。
*   **渴望亲密**：喜欢和恋人在一起，渴望深层的亲密关系。
*   **持续担忧**：常常处在不确定中，担心恋人与自己不够亲近。
*   **过度敏感**：特别容易察觉感情生活中的细微波动，并将其无限放大。
*   **负面臆想**：对恋人的一举一动非常敏感，容易否定恋人对自己的感情。

## 焦虑的恶性循环

这种焦虑会消耗个体大部分的心理能量，让人感到疲倦，并陷入“草木皆兵”的状态。一个典型的恶性循环思维模式是：
`男朋友回微信慢 -> 猜想他不在意自己 -> 猜想他在和别的女生玩 -> 猜想他有外遇`

科学研究表明，人的情绪如同月亮一样有阴晴圆缺。女性有生理周期，男性其实也有。伴侣的情绪波动和喜怒哀乐，并不完全是由你引起的。然而，焦虑型人格会因为自己的负面假设而感到异常压抑。当压抑积累到一定程度，他们可能通过作闹、黑脸、吵架甚至跟踪窥视等方式向伴侣释放压力，这些行为都会伤害彼此的感情。

相反，如果伴侣给予一点正面确认，他们的情绪又会立刻变得愉悦。他们的情绪经常因为一些小事而大起大落。

## 案例分析：体重焦虑

接下来，我将以一位女孩的体重焦虑为例，具体说明现代社会女性的外貌和体重焦虑。这是一位咨询者的真实情况，我将以第一人称进行叙述：

> 我身高1.62米，目前体重50公斤。最胖时57公斤，最瘦时45公斤。从高中就开始减肥，曾认为“连体重都控制不了的人，怎么能控制人生？”。我特别在意别人的看法，有人说我瘦了我会特别开心，但体重一旦反弹，整个人就会崩溃，并强制开始下一次减肥。戒糖常常导致我情绪不稳定。
>
> 减肥时我常出现这种状况：反复打开冰箱又关上。男朋友问我在干嘛，我说饿。他说那就吃，我又说在减肥。有一天晚上，当我第五次打开冰箱时，男朋友说：“你非得这么样减肥吗？”我回答：“那不然怎么减。”他说国外女孩去健身房，身材线条好。我说不想锻炼，好累。他说不喜欢过瘦的，没意思。我说别人瘦又不是给你看的。他说反正也觉得不好看。
>
> 这种焦虑让我注意力不集中，整个人处在戒备状态。第二天，男朋友买了一个汉堡，我很生气。他说刚才问过我说不吃。我说我确实说了不吃。然后他说“真不吃？来尝一口。”我们因此产生了小冲突。因为我觉得如果他买了汉堡，我忍不住偷吃，会特别沮丧并持续自我贬低，产生许多负面想法。例如：“我连体重都控制不好，难怪男朋友对我不好。”我会有一种错觉，只要体重达标，一切都会顺利起来。

之后，她的男朋友又打压她说胸变小了，这让她感到悲痛，双方因此产生分歧，让彼此氛围变差。

这位咨询者还提到，减肥这几年她还算幸运，没有绝经或得厌食症，但经常胃痛和低血糖。现在，50公斤的体重是她不用特别努力就可以维持的。所谓“不特别努力”，意味着她可以偶尔吃点零食、雪糕，也会稍微锻炼一下。在她不强求体重不过百之后，反而快乐放松了许多。因为不用长期断糖挨饿，在平稳幸福的状态下减少了自我贬低，和男朋友的相处反而更好了。

她对此的总结是：**可能没有人喜欢和一个讨厌自己的人在一起吧。**

## 如何应对焦虑：爱自己与调整视角

我制作这个拓展包的目的，并非让大家松懈下来不去做自我提升，而是希望大家不要因为一点不对劲就陷入恐慌与焦虑。应对焦虑，可以从以下几点入手：

以下是调整心态、应对焦虑的建议列表：
*   **对自己宽容一点**：多爱自己一点。没有人会因为你的一句话、一个行动、一件事没做好，就否定你的全部。
*   **稳定心理状态**：不要让心理状态一直处于危险期，不要让安全感短暂而稀少。
*   **避免焦虑主导**：不要让焦虑成为感情的主基调，不要让生活悬在钢丝之上。
*   **谨慎反应**：不要过早下结论，也别过快做出反应。多等片刻，能更准确地理解周围世界。
*   **善用敏锐**：让你的敏锐真正成为优势，不要一有风吹草动就不加思索地回应，那样容易犯错并伤害彼此。

此外，还需要调整看待自己与他人的视角：
*   **不要把自己看得太重**：你的一言一行没有那么受到别人的重视，也不会轻易伤害到别人。
*   **不要把别人看得太重**：别人的一言一行，也没有那么容易能伤害到你，对你来说也并没有那么重要。

## 总结

本节课中我们一起学习了感情中焦虑型人格的特征与影响。我们通过一个体重焦虑的真实案例，看到了焦虑如何引发恶性循环并损害亲密关系。关键在于，要打破这种模式，需要学会爱自己、对自己宽容，并调整看待自己与他人的视角。不要被焦虑主导你的生活和感情，更不要因为过度自省和自我贬低而推开爱你的人。希望大家能够更加无惧无忧地活在这个世界上。
