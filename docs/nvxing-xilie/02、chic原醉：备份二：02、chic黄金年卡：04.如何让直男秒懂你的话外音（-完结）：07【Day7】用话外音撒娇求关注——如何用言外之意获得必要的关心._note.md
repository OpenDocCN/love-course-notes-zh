# 话外音沟通术：02：用话外音撒娇求关注——如何用言外之意获得必要的关心

## 概述

在本节课中，我们将学习如何通过“话外音”有效表达需求，从而获得伴侣的关心与关注。许多女性常常感到伴侣无法理解自己话语背后的真实意图，导致期待落空。我们将从自身表达和引导伴侣两个角度，提供具体、可操作的技巧，帮助你用言外之意，巧妙地获得必要的关心。

## 为何男生总听不懂你的需求？

女生常常希望得到男生更多的关注。生病时，期待对方陪伴和笨手笨脚的照顾。失落时，想要对方给予拥抱和甜言蜜语。在海滩时，希望他眼中只有自己，充满激情。有情绪时，想让他放下一切，逗自己开心。

但是，男生总是不得要领，让女生的期待不断落空。这不禁让人疑惑：到底要怎么做，才能让他明白我的心思？

上一节我们探讨了沟通不畅的现象，本节中我们来看看具体的解决技巧。

## 技巧一：用SCQA框架，清晰表达困境

让男生理解并关注你是有技巧的。我们可以根据以下三个技巧逐一掌握。

首先，从我们自身的表达方式入手。核心在于**详细准确地表述你的困境**，让男友切实体会到你需要关心。

我们来分析一个常见现象：为什么男生总回应女生“多喝水”？仿佛热水是包治百病的万金油。明明你想要的是关心和拥抱。

如果想让他回应你的核心诉求，可以改变陈述方式。有一种故事写作方法能很好地帮助你与男友展开高效沟通，即 **SCQA故事框架**。

`SCQA`是英文首字母组合的缩写，分别代表：
*   **S** (Situation): 情境
*   **C** (Complication): 冲突
*   **Q** (Question): 疑问
*   **A** (Answer): 回答

用在沟通上，就是当你想让某人做一件事时，可以循序渐进：先陈述场景，再说冲突，然后提出疑问，最后给出回答。这样他回应你的概率会非常高。

用这个框架引导男生关心自己，我们需要着重第二个步骤——**对于冲突的表达**。即放大你的不舒服，尽可能详细地描述身体的疼痛和心里的难过，以此让他对你产生更多关注和重视。

例如，你生病了，希望男生哄哄你。可以根据SCQA框架表达：

1.  **陈述场景(S)**：“我发烧了。”
2.  **放大冲突(C)**：“直接烧到了39度，现在嘴唇干裂，脑子发懵。要是再烧下去，我就真的没有知觉了。”
3.  **提出疑问(Q)**：“（如果对方还没行动）我现在情绪很低落，怎么办？”
4.  **引导回答(A)**： 这时，他就会根据你提问的指向来安慰你的情绪，而不是简单地让你“多喝水”。

形象具体的冲突描述，容易使他代入不舒服的感受，并对你表示出关心和同情。

**需要着重说明的是**：前面提到的“放大冲突”，只是把难过、焦虑等身体感受尽可能全面地说出来，而不是夸张地描述不合实际的事。

## 技巧二：假装生气时，记得留个台阶

在说明如何留台阶之前，需要明确：**假装生气是一种撒娇的方式，而不是真的对男生埋怨或嫌弃**。

然而，很多时候我们表现生气时会刹不住车，不仅翻白眼，还咬牙切齿。关于这种“弄假成真”的情况，心理学家威廉·詹姆斯解释说：我们之所以快乐是因为我们笑了，悲伤是因为我们哭了。**身体反应会导致情绪反应**。

“身体反应”在这里指我们皱眉、嘴角下拉等生气的行为。“情绪反应”就是逐渐强烈的生气情绪。当我们不停表现生气时，大脑就会接收指令，并且慢慢相信我们是真的生气了。而一旦相信，行为动作又会进一步被放大。

所以，在男生眼中，他看到的是一个怒目圆睁的你。他从男性思维出发，很难分辨出你是在“演戏”。

把假装生气的重点放在“假装”上，问题就比较好解决了。关键在于：**你在组织生气的语言时，要包含别的信息，让男友的注意力集中在这个信息上**。

例如，你可以说：“你看你总是惹我生气，要不是你长得这么帅，我才不稀罕你呢！”

将这样的话说出口，男友的第一反应是你在夸他帅、你舍不得他，而不会觉得你是真的生气。

## 技巧三：用“装傻”法，暗示心仪礼物

很多女生经济独立，买得起钟爱的物品，去得了梦想的地方。但当男生送我们礼物时，我们依然很开心，因为我们认为这是男生爱自己的方式——那些没有直接说出口的爱，被他们装点成了礼物。

如果我们生硬地直接提出要礼物，意思就不太一样了。比如你跟他说：“你不是刚发了年终奖吗？给我买一双鞋吧。”他十有八九不会很高兴，因为他会有一种被紧盯荷包的不舒适感。

我的一个朋友有一个让男朋友乐意买礼物的方法：**期待礼物时，一定要用撒娇的语气表达出来**。而她平时表现的是一个约会主动付账、分担房租压力的自强形象。当她向男朋友撒娇要礼物时，男朋友会觉得她反常的撒娇有点可爱。

除此之外，她还很注意撒娇要礼物的**时机**。这个时机通常选在男朋友向她提出某些小请求时，比如让她缝扣子、捏捏背。双方进入一个良好的沟通氛围后，撒娇要礼物就显示出情调来了。

如果你还想把要礼物表现得更含蓄，那么需要在日常生活中表露你喜欢什么样的物品、是哪种品味。喜欢程度越高，重复得当，他的印象就会越深刻。

关于礼物贵重的问题，其实不必太在意。只要他是真心准备一份礼物，目的是让你开心，都值得好好感谢。你甚至可以用非常明显的话外音说：“你真的买了呀？讨厌！”同时两眼放光地看着礼物，让他看到你毫无保留的喜悦。当他意识到送你礼物能让你开心，他才会送你下一次、再下一次。

## 如何培养男友的“高情商”？

除了以上三个从自身出发的技巧，要让一个男生明白女生说话是在求关注，他需要具备一定的“高情商”，即**站在他人角度看事情的能力**。这里我们教大家一些提高男友情商的办法。

### 方法一：引导他觉察并管理自我情绪

首先，要让男友的脑回路学会“绕弯”，需要给他进行日常的情商训练。著名心理学家彼得·萨洛维和约翰·梅耶提出了情商的概念，并将其拓展到5个领域，其中三个都与情绪有关：
1.  了解自我情绪
2.  管理情绪
3.  识别他人情绪

可见，情绪的认知和管理对情商的培养很重要。男生无法深刻感受到女生的情绪，更深层的解读是：**他们对自我情绪的感知也有所欠缺**。

当他的工作陷入瓶颈、生病或被误解时，他的情绪可能悲伤、委屈甚至愤怒。然而大多数情况下，为了表示自己坚强有气度，他们不会把这些埋在心底的情绪表露出来。

**让男生懂得觉察和体会自己的情绪，他才能更好地体会别人的情绪**。那么，你可以去引导他发现自己的情绪：告诉他你察觉到了他的不舒服，和他一起分析为什么会这样，再帮助他发泄出来。如果他表示想先静一静，就说：“好，你想说的时候记得告诉我。”这不仅是一种鼓励，还是一种示范。

当你下次面对沮丧时，他可能就学习到了这种方法，停下手中的事问你：“你怎么了？先休息一下吧。”

### 方法二：用文艺作品营造浪漫氛围

第二种提高男友情商的方法，是通过培养电影、音乐、书籍等爱好，让他处在浪漫的氛围里。

每个男生表面上可能很“直”，但偶尔还是会为了让女朋友高兴去制造一些浪漫。细想之下，你知道你的男朋友是出于什么原因制造浪漫吗？很可能是他受到了某些环境的“点醒”。

公园漫步的情侣，让他想给你送一束花；球场上男生给女生绑鞋带的画面，让他想给你做一顿早餐。这些两个人的幸福场景都会让他意识到，也有一个人与他朝夕相处，共同铸造温馨生活。

我们有很多机会去“点醒”他，**电影、音乐和书籍就是最好的方式之一**。我的一个男同事就是因为看了一部韩国电影《我爱你》，跑来问我送女朋友什么礼物好。电影中主人公晚年相遇却享受独特浪漫的恋爱方式感染了他，他的浪漫情绪仿佛一下被唤醒，不仅准备了礼物，还和女友共同规划起了生活上的闪光细节。

如果你的男朋友很久没有说贴心的话、有亲密的举动，或是在你主动示好时毫无表示，那么通过电影、音乐和书籍，他会领悟很多。

### 方法三：给予正向反馈与“甜头”

最后，假使他初次体验到了主动理解你的“反差”效果，记得巧妙地给他点“甜头”。美国管理专家拉伯福说，他悟出的世界上最伟大的管理原则是：**人们很愿意去做收到奖励的事，并且会做得很漂亮**。

当他真正理解到你话里的意思时，不要错过机会夸赞他。你可以说：
*   “你太体贴了！”
*   “你关心我的时候，让我又有了怦然心动的感觉。”
*   “我认识了你不同的一面。”

或者深切的亲吻他。如果他免不了认为听你的话外音是一种“猜测”，那显然参加“有奖竞答”，比答错“电击”要有诱惑力多了。

## 总结

本节课中，我们一起学习了如何用话外音求得男友的关注。可以从两个角度寻找办法：

**首先，从我们自己的角度出发**，就是要从三个方面改变自己说话的方式：
1.  **详尽表达冲突**：使用SCQA框架，特别是放大冲突细节，让他不得不回应你。
2.  **生气时转移焦点**：通过语言中的其他重点（如夸赞）转移他的注意力，让他知道你是假装生气。
3.  **撒娇装傻提需求**：在良好氛围中，用撒娇或日常暗示的方式提出礼物需求，制造生活中的小情调。

**其次，从男友角度出发**，就是我们要通过日常引导来培养他的高情商：
1.  **引导情绪管理**：帮助他觉察和管理自我情绪，他才能更好地理解你的情绪。
2.  **营造浪漫氛围**：利用电影、音乐、书籍等，点醒他的浪漫感知。
3.  **给予即时甜头**：当他做出正确回应时，及时给予夸赞或亲密奖励，强化他的积极行为。

掌握这些方法，你将能更有效地通过言外之意，获得你渴望的关心与爱护。
