# 爵士舞零基础教程：16：成品舞分解教学《如你》（一）

在本节课中，我们将学习韩国女团歌曲《如你》的爵士舞片段。课程将从基础站姿和节奏练习开始，逐步分解舞蹈动作，包括手部动作、脚步移动和身体律动。我们将确保每个步骤都清晰易懂，适合零基础学员跟随。

## 🎵 准备与起始姿势

首先，请准备一件外套。舞蹈中包含拉开外套的动作。如果没有外套，可以直接将手打开完成相应动作。

音乐开始时，我们背对镜头。听准节奏，从“1234”开始。

## 🧍‍♀️ 第一组动作分解（8拍）

以下是第一个八拍的动作分解。

1.  **第1-4拍**：左腿作为支撑，做左胯动作，左手掐腰。头转向左边。节奏为“1234”。
2.  **第5-8拍**：右胯向上抬，然后下坐。节奏为“5678”。

我们连贯练习一遍：`1234，5678`。

## 👣 第二组动作分解（8拍）

上一节我们介绍了起始的胯部动作，本节中我们来看看脚步与手部的配合。

以下是第二个八拍的动作分解。

1.  **第2拍**：右脚脚跟踩地。
2.  **第2-3拍**：左脚跟上，右手从嘴唇边向外拉开。
3.  **第4拍**：重心移到右边。
4.  **第5拍**：重心移到左边，右手在上。
5.  **第6拍**：重心移回右边。
6.  **第7拍**：重心移到左边，左手在嘴前拉开，右手放在头后。
7.  **第8拍**：抬脚。

我们将这八拍连起来：`2， 23， 4， 5， 6， 7， 8`。

## 💃 第三与第四组动作衔接

接下来，我们将前两个八拍连接，并加入转身动作。

1.  接上一个八拍，做“三抬脚”动作。
2.  向侧面迈步，右手掐腰，左手前指，左顶胯。
3.  向后迈步，转身，做拉开外套的动作。
4.  双手在胸前顺时针划圈。

这四个八拍的完整串联是：`1234567大8， 223大45678， 32345678， 42345678`。

## 🚶‍♀️ 行进间步伐组合

现在，我们学习一段向前行进的动作组合。

以下是具体步骤：

1.  **第1-3拍**：右脚前迈，左手撩头发；左脚前迈；双手打开。
2.  **第4-8拍**：双手下按，重心右移，垫步向前，最后左胯上提。

我们从“567”开始准备：`567走123， 567到8`。

## ✋ 手部与方向变化

这一部分注重手部细节和身体方向的转换。

以下是动作要点：

1.  **第2拍**：左脚旁迈，左手摊开，右手外放。
2.  **第4拍**：右手点于左手。
3.  **第6拍**：左手放胯，右手至嘴边。
4.  **第7-8拍**：右脚前弓箭步，然后向前迈步。

我们从“567”开始连接：`567， 123， 567大8， 22345678`。

## 🔄 综合段落练习

我们将目前所学的两个行进段落整合起来练习。

完整序列如下：
`678走1234567大8， 2234大5678大1， 2345678`

## 👐 波浪手与蹲起组合

现在学习一个带有波浪手和身体起伏的新组合。

动作分解如下：

1.  双手掌心朝外，左脚开始向前迈三步。
2.  身体转向左边，双手合十向下，再打开。
3.  身体转向右边。
4.  向下蹲。

我们从“5678”开始：`5678， 123， 567`。

## 💫 绕胯与点地动作

这个组合包含绕胯和精致的点地姿势。

以下是详细步骤：

1.  **第2-4拍**：右手前伸，左手搭上，向后迈步。
2.  **第5-6拍**：向右顶胯，左手手掌，右手食指。
3.  **第7-8拍**：双手绕圈，左脚吸腿。

连接练习：`5678， 223， 5678`。

## 🌪️ 大绕胯与移动

这是一个幅度较大的绕胯和重心移动组合。

动作流程如下：

1.  左脚旁迈，重心左移，胯部向后绕至前点。
2.  左手扶头，右手在胯旁绕圈下砸三次。
3.  重心移到中间，再移到右边，下坐。

我们从“5678”开始：`5678， 323， 4567到8`。

## 🏁 结束段落与连接

最后，我们学习本片段的结束动作，并将其与前面内容连接。

结束动作分解：

1.  **第4-8拍**：右脚旁迈，双手左上交叉；重心右移，左脚点地，右手扶头；身体向左绕回。
2.  **下个第1拍**：重心移右脚，左腿抬起。

四个八拍的完整串联是：
`5678， 123， 5678， 223， 5678， 32345678， 423456781`

## 📝 本节课总结

本节课中我们一起学习了《如你》爵士舞第一段的大部分内容。我们从基础的节奏和胯部动作开始，逐步增加了手部姿势、方向转换、行进步伐以及综合组合。重点在于将动作与音乐节拍对齐，并保持身体的协调与律动。请反复练习每个分解动作，直至能够流畅连接。
