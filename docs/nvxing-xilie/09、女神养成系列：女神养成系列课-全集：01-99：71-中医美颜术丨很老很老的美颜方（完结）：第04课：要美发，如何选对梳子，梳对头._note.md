# 女神养成系列：04：要美发，如何选对梳子，梳对头 💆‍♀️

在本节课中，我们将要学习如何通过中医的视角观察头发健康，并了解如何选择适合的梳子以及掌握正确的梳头方法，以达到美发养生的目的。

## 头发的“发诊”：望闻问切

上一节我们介绍了身体外在表现与内在健康的关系。本节中我们来看看如何通过头发进行诊断，即“发诊”。发诊同样遵循中医“望闻问切”的原则，旨在通过头发状态了解身体内在的寒热虚实。

### 望诊：观察头发的状态

望诊主要通过观察头发的颜色、光泽、皮屑、油腻度、干枯分叉等情况来判断身体状况。

以下是望诊的具体观察点：

*   **颜色**：观察头发颜色。白发通常反映身体的寒热虚实状态。
*   **光泽**：观察头发是否有自然光泽。光泽不足可能表示身体“精”气不充足。
*   **皮屑**：观察是否有头皮屑。大片头屑可能与过敏有关；小片头屑可能与熬夜等内在状态有关。
*   **油腻**：观察头发是否油腻（需排除长时间未洗头的自然状态）。油腻可能代表身体有痰湿。
*   **干枯分叉**：观察头发是否干枯分叉。这可能表示有阴虚或血虚的状态。
*   **粘结成束**：观察头发是否像枯草一样粘在一起。若非先天因素，则可能是肾阴不足的表现。

通过望诊，我们可以对身体状态有一个初步的认识。

### 闻诊、问诊与切诊

除了观察，诊断头发还需结合其他方法。

*   **闻诊**：闻头发的气味。正常的头发气味自然。若头皮有疮或疹，可能闻到油腻味，用以判断痰湿或个人卫生问题。
*   **问诊**：询问生活习惯。例如是否熬夜、饮食是否偏好油腻辛辣等，以了解引起头发问题的内在因素。
*   **切诊**：用手指感受头发。方法是用左手拇指和食指掐住发根，右手同样掐住头发，匀速向发梢捋动。感受是否顺滑，中间是否有“涩滞”感，以此判断人体气血和津液的状态。

综上所述，发诊的目的是通过头发了解身体状态，进而通过调理身体来改善头发问题。

## 梳子的选择：材质决定功效

我们每天最常做的一件事就是梳头。接下来，我们来看看如何选择一把合适的梳子。古人多用木梳，因为不同材质的木梳对人体有不同的促进作用。

以下是几种常见材质梳子的功效：

*   **桃木梳**：桃木在民间被视为吉祥物，有祛邪扶正之意。桃木木质细腻坚硬，有淡淡清香。
    *   **功效公式**：`桃木梳 → 疏通经络 + 活血化瘀 → 改善头皮供血`
    *   适用于常感头部胀痛、发紧的人。
*   **枣木梳**：枣木颜色偏红，有花纹。
    *   **功效描述**：通经脉，令发易长。能帮助乌发、止痒、醒神健脑。
    *   适用于头发长得慢、有白发、头皮瘙痒的人。
*   **牛角梳**（以水牛角常见）：手感温润厚实，梳头无静电。
    *   **功效描述**：清凉止血，清热解毒。可缓解热毒引起的头皮疮、热性头痛、高血压头昏。
    *   质地坚硬，不伤皮肤，护发护头皮，能按摩头皮、促进血液循环、缓解疲劳。
*   **檀木梳**：质地紧密坚硬，香气芬芳持久。
    *   **功效描述**：辟邪治病，强肾养精，提神降火去痛。其芳香可缓解因外感邪气引起的头痛，并能提神醒脑。
*   **黄杨木梳**：主要功效是清热利湿解毒。
    *   **功效描述**：抑制真菌生长，因此止痒和去头皮屑的效果较好。

选择一把适合自己发质和身体状况的梳子，是护发的第一步。

## 梳头的方法：循经梳络，事半功倍

选好了梳子，我们来看看如何正确使用它。中医认为头部分布着许多经络，按照经络走向梳头，效果更佳。梳头能疏通经络、活血化瘀，改善头皮营养。

为了达到理想效果，我们需要了解头部的经络分布。

### 头部主要经络

以下是头部几条主要经络及其作用：

1.  **督脉**：位于头部正中线（从大椎穴到前发际）。掌管阳气，调节中风、热病、头部疾病、神志病等。
2.  **足太阳膀胱经**：位于督脉两侧约一寸（个人拇指指关节宽度）处。主要缓解后头痛、颈椎不适、腰背疼痛。
3.  **手少阳三焦经**：环绕耳后至眉尾区域。主要影响侧头部、耳部、眼部疾病及咽喉痛。
4.  **足少阳胆经**：位于头部两侧。主要影响侧头部、耳部、胁肋部，并与眼病、神志病相关。

### 正确的梳头顺序与手法

梳理时，可根据自身状况侧重相应经络。如果全身舒畅，则可系统梳理全部经络。

以下是推荐的梳头顺序与手法要点：

*   **顺序**：
    1.  先梳正中的**督脉**，从前发际梳到后发际（包括后脖颈）。
    2.  再梳一侧的**膀胱经**，梳通舒畅后，换另一侧。
    3.  接着用手指梳理耳后的**三焦经**区域。
    4.  最后用木梳梳理头部两侧的**胆经**区域。睡眠不好、焦虑急躁者可多梳理此部位。
*   **核心要点**：`梳头 = 梳头皮`，而非只梳头发丝。
*   **手法**：将梳子伸进头发，接触头皮，然后**徐徐缓缓**地用力均匀地向后梳。切忌快速猛刮或用力过猛。
*   **作用原理**：`柔和梳头 → 促进头皮平和稳定的血液循环 → 血液输送营养至毛囊 → 头发获得充足供养`
*   **替代方法**：也可直接用**手指指腹**稍加用力，插入头发接触头皮，缓缓向后梳理，力度更易掌控。
*   **次数建议**：每日可梳多次，每次至少梳100下。持之以恒，对改善营养不良、白发等问题有良好效果。

## 总结

本节课中我们一起学习了中医的“发诊”方法，了解了如何通过望闻问切观察头发健康。同时，我们认识了不同材质梳子（如桃木、枣木、牛角、檀木、黄杨木）的特有功效，并掌握了沿着头部督脉、膀胱经、三焦经、胆经等经络进行柔和、缓慢梳头的正确方法。坚持正确的梳头习惯，不仅能美发，更能通过刺激头部经络，达到养生健脑、缓解疲劳的功效。
