# 《两性幸福》课程：第5集：如何实现思想同频与共同成长 🧠❤️

在本节课中，我们将探讨两性关系中一个至关重要的课题：当伴侣的成长速度不一致时，如何通过智慧的方法实现思想同频，从而维护和深化感情。我们将学习具体的沟通策略与行动步骤。

## 核心问题：成长不同步

讲实话，你们觉得杨老师有没有智慧？有没有一点悟性？但是有时候我觉得我相当愚蠢。因为当老师出外学习，学到很高的境界的时候，我突然觉得我跟不上了。我一直在想我的智商、我的情商、我的财商都不可能是跟不上他的成长的。甚至我的悟性在1万个人里面都是顶级的。为什么我突然跟不上呢？

突然发现一点：他接受了另外一个价值观的时候，我没有接受。所以我们思想不同频了。当思想不同频的时候，精神上就达不到同一个层次了。当精神达不到一个层次的时候，夫妻间即便说我们感情没有问题，问题也已经出现了，你认可还是不认可？

## 应对策略：避免指责，主动学习

所以在座的女性们，这个时候你们一定不要暴躁，不要质问“我老公怎么能进步那么快？”或“我老公怎么能不带上我提升？”。就像我老公经常说的一句话：“你这个女人那么强，我就想超越你，我都还没有想保护你的时候，我就想超越你。”

所以我就一直在想一件事情。那如果是这样子的话，很简单，怎么去改良呢？我就发现我老公学的东西所接触的境界跟我不一样。

## 具体行动步骤

以下是实现思想同频的具体操作步骤，关键在于主动、耐心和策略。

**第一步：表达支持与好奇**
我就开始喊着我老公：“老公，老婆以后支持你好不好？”怎么天天这么听话不正常。他怀疑你第一天不正常，你千万记得要有耐心。接着说：“老公，我突然发现你最近学习的太棒了。你在哪里学的？”

**第二步：提出共同学习的请求**
然后他就告诉你在哪里学的，你接着说：“你帮我也报一个这样的课程，好不好？”然后老公就会说了：“没关系的，等我学会了，我来教你。”

**第三步：识破“我来教你”的陷阱**
你千万不要相信男人说这样的话是为了你好。要讲实话，他要能教会你，早就教会了，也轮不到今天他还来教了。老婆也不是被老公教出来的，一定是被别人教出来的，你认可还是不认可？所以我深刻的知道这个道理，他如果想教我，我还没接受，我的价值观就走向偏离。他一定是有目的想让我听他话，所以才这样说。

**第四步：运用智慧说服**
但是如果一个老师这样教你说要对男人怎么样，你会不会觉得这个老师是跟你老公串通好的，所以故意来这样说的，会不会？不会。所以我就开始哄我老公：“老公，我知道。但是你有没有想过时间比钱还重要？老婆其他都不想你老出差，老婆就想跟你待一块。所以能不能我们不为了这个钱考虑，那你也帮老婆报一个，让老婆跟你一起进修。进修的目的就是以后你说啥就是啥。”说完以后，老公立马给你刷卡。

## 策略背后的原理

上一节我们介绍了具体的行动步骤，本节中我们来看看这些策略背后的深层逻辑。

然后女人就说了，骂的老子有钱，为什么要男人刷？我想我的财富也不会比你少，我肯定也有钱。但是我不小心看了一篇文章，我是这么看的：**凡是让男人刷钱了，他就不会在别人身上刷了**。因为他在你身上付出那么大成本以后，他的胆子就变小了。不然算一下，投资回报率不值。

所以我一想，还得让老公刷。刷完以后怎么办呢？果然思想同频了。思想同频，感情有进步，越来越恩爱了。等到下一次的东西能够增长你智慧的时候，都不用你说“老公帮我报一个呗，我以后听你的呗”，老公是想方设法把你送到那个通往更高境界的地方去。因为他以为把你送到更高对他的好处最大。

## 根本受益者与战略思维

其实我想告诉在座的各位，那个世界上一定有个最大的受益者，不是你老公，而是你本人，你认可还是不认可？所以很多东西我一直在想：**战略上先把战略定下来，再调整战术**。这条战术不对，就换个战术没有关系。还有三十六计走为上计，管他们这条不管用，换下一条。

## 男女思维差异
