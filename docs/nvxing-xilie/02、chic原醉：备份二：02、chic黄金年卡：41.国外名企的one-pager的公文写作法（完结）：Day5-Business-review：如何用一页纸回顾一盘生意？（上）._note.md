# 公文写作法：02：商业回顾型一页纸（上）

在本节课中，我们将学习第二种一页纸公文——商业回顾（Business Review）的写作方法。我们将重点探讨如何避免流水账式汇报，并通过运用结果性数据和结构化原因分析，清晰、有力地向上级展示工作成果。

上一节我们介绍了事件型一页纸（Issue Sheet）的写法，本节中我们来看看如何用一页纸对一段时期的业务进行回顾与总结。

## 为何要避免流水账？

我们每个人都做过商业总结，需要向老板汇报过去一段时间的工作进展和成果。这是一种常规的周期性汇报。但在回顾工作时，许多人容易陷入流水账的思维模式。

相信大家对以下汇报内容感到熟悉：“上周共进行5项工作：1. 某项目前期沟通；2. 与某客户业务洽谈；3. 参与某某会议；4. 跟进某某决议；5. 完成某某合同起草。”

这种汇报会产生一种视觉差。汇报者觉得很有成就感，但倾听者（尤其是老板）可能感觉未接收到有用信息。其核心问题是**结果性信息的缺失**。

作为执行者，我们容易关注执行过程。但向老板汇报时，必须切换到结果思维，让老板清楚我们**做成了什么事**。这并非否认努力，而是通过技巧让努力被更好地看见和利用。

## 核心一：运用数字的逻辑

我们的课程多次强调汇报中数字的运用。要达到最佳效果，关键在于注重**结果性数字**的运用。

结果性数字指能直接代表工作成果的数据，例如销售额、同比增长率等。失败的汇报常充斥过程性信息（如“参加会议”），这些信息难以直接用于决策。而结果性信息（如具体数额、百分比）能清晰展示进展，并辅助下一步决策。

结果性数据主要有三种：

以下是三种关键的结果性数据类型：

1.  **绝对数字**
    *   **公式/概念**：`具体金额` 或 `数量`
    *   **作用**：直接揭示结果本身，开门见山。例如：销售额、销售量、产量、用户量。
    *   **应用建议**：无论结果好坏，都应如实呈现，通常置于汇报开头。例如：“上周A产品完成50万元销售额。”

2.  **纵向相对数字**
    *   **公式/概念**：`(本期数据 - 上期数据) / 上期数据 * 100%` （同比/环比增长率）
    *   **作用**：对比自身过去与现在，显示发展动态（增长或收缩）。例如：同比增长25%。

3.  **横向相对数字**
    *   **公式/概念**：`自身数据 / 市场总数据 * 100%` （市场份额）或 `行业排名`
    *   **作用**：跳出自身，在全行业中进行比较，明确相对位置与竞争态势。例如：“占据30%市场份额，位居行业第二。”

在商业回顾的开头，通过列举这三组数据，就能让老板快速把握业务现状。

**示例**：“A产品上月完成20万销量，实现100万销售额，环比增长25%。目前，该产品在X品类中占据30%的市场份额，位居行业第二。”

了解这些结果后，老板自然会问：“取得这些成果的原因是什么？” 这就过渡到下一部分。

## 核心二：进行原因分析

商业回顾中的原因分析，侧重于**阶段性的变化**，而非事件型一页纸中更聚焦的时点性分析。

这里介绍两种分析工具：

以下是两种实用的原因分析方法：

1.  **SWOT分析法**
    *   **概念**：从**优势(S)**、**劣势(W)**、**机遇(O)**、**威胁(T)**四个宏观维度分析项目。
    *   **示例**：增长源于技术优势(**S**)，但品牌知名度不足(**W**)；市场处于扩张期(**O**)，但面临巨头竞争(**T**)。
    *   **适用场景**：中长期回顾（如季度、年度）。优点在于提供全景视角，缺点是不够具体。

2.  **原因分解三要素法则**
    为弥补SWOT的不足，可采用此方法，从三个维度由表及里地拆解原因：
    *   **直接原因**：肉眼可见、不加感情色彩的现象。
        *   *示例*：“销量下滑的直接原因是出现大规模客户退单。”
    *   **主要原因**：对结果产生最大影响的因素，关联前期决策或市场变化。
        *   *示例*：“退单的主要原因是竞争对手发起了价格战。” 此原因可直接指导下一步行动（如制定应对策略）。
    *   **根本原因**：更宏观、更核心的深层原因，涉及市场环境、消费心理或公司基因。
        *   *示例*：“用户注册数下降的根本原因在于产品功能场景存在局限。”

通过对原因进行这三个维度的拆解，就能透彻解释当前的业务进展。接下来，自然需要基于此提出未来的行动方案与目标。

本节课中，我们一起学习了商业回顾型一页纸的前半部分核心：**运用结果性数字清晰呈现成果**，以及**使用结构化方法深入分析原因**。下一节，我们将继续讲解如何制定具体的后续行动计划与目标。
