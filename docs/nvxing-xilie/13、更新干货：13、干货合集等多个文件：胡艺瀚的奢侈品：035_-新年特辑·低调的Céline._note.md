# 奢侈品入门：035：低调的Céline品牌全解析

在本节课中，我们将深入了解奢侈品牌Céline的前世今生、其独特的品牌哲学，以及那些风靡一时的经典手袋。我们将从品牌历史讲起，探讨其如何在不依赖明星营销的情况下保持优雅与热度，并重点解析由前创意总监菲比·菲洛设计的标志性“IT Bag”。

## Céline的前世今生与LVMH收购

上一节我们概述了本节课的内容，本节中我们来看看Céline品牌是如何创立并最终成为LVMH集团一员的。

Céline的发展史始于一个悲伤的故事。1945年，设计师Céline Vipiana和她的丈夫在巴黎开设了一家以她名字命名的店铺，名为“Céline”，最初是一家专门为儿童定制鞋履的制靴作坊。

童鞋销售状况良好，摩纳哥卡罗琳公主和亲王阿尔伯特二世都曾是顾客。Céline的知名度逐渐在上流社会流传开来，Céline女士也开始设计男鞋和女鞋。

到了1959年，Céline才开始致力于女装和手袋的设计。当时手袋上的双轮马车、花朵和半月标记，都是那个时代的经典标志。

1967年，Céline Vipiana发布了她的第一个运动休闲装系列，名为“Couture Sportswear”。产品包括裙裤、运动衫和皮马甲等。这些成衣系列将巴黎女性高贵优雅的气质与日常生活中轻松惬意的感觉融为一体，引领了新的时尚潮流。

20年后，LVMH集团总裁伯纳德·阿诺特找上门来，询问Céline女士是否愿意出售品牌。当时Céline是一家生产精品女装和皮具的家族小企业。Céline Vipiana负责设计，丈夫理查德管理财政。他们每年的2000万美元营业额中，利润只有500万美元，因为店铺均由经销商特许经营，他们只抽取部分比例的销售额。

夫妇俩已年过花甲，考虑退休，但儿子却不想继承家业。阿诺特提议买下Céline。他最初提出的条件是：阿诺特负责公司运营，占有三分之二的股份，夫妇俩保留三分之一。

然而，事实并非如此。合作签署几个月后，夫妇俩就被迫离开公司，并被告知无需插手品牌的任何事宜。这让夫妇俩备受打击，无奈之下只能将剩余的三分之一股份也全部卖给了阿诺特。最终，Céline在1996年正式加入了LVMH集团。

我们现在试想一下，如果Céline当初没有被阿诺特买走，到如今会有怎样的发展呢？

## 低调的哲学与菲比·菲洛的革新

上一节我们回顾了Céline被收购的历史，本节中我们来看看品牌如何凭借独特的低调哲学获得成功，以及创意总监菲比·菲洛的关键作用。

大家可能已经发现，当LV、Gucci等国际大牌纷纷邀请明星或流量小生代言时，Céline却一直保持着自己的步调，不请明星代言。其国内门店很少，也从不打广告大肆宣传。公众所知的消息通常是歌手王菲每年会自行飞往巴黎观看Céline的时装秀。

然而，Céline这几年越是低调，就越是火爆。其每一次推出的新款包袋都能成为“IT Bag”。这一切主要得益于刚刚离职的创意总监菲比·菲洛。

大家知道吗？Michael Kors曾担任Céline的创意总监。他在任期间，一度为品牌华丽的风格注入了美式旅行的风潮。但到了2004年，Michael Kors离开Céline，开始致力于自己的同名轻奢品牌。从那以后，Céline的设计风格一度陷入混乱，甚至出现衰落的情况。

菲比·菲洛的加入让Céline起死回生。在此之前，她接手好友Stella McCartney的职务，成为品牌Chloé的创意总监。在接手的五年内，她把Chloé打造成巴黎时装周炙手可热的品牌，并取得了巨大的商业成功。但后来因为结婚生子，菲比淡出了时尚圈。

当她重出江湖时，Céline还不知名，也没有知名设计师愿意接手。LVMH集团为了聘请她担任Céline的创意总监，不惜将整个工作室从巴黎搬到伦敦，并答应菲比进行了以下几项重大改动：
1.  团队大换血。
2.  关掉一些不必要的门店。
3.  销毁所有的旧货库存。

所幸的是，LVMH的选择和让步都是值得的。在菲比任职的近十年间，Céline的年营业额从之前的2亿欧元逐渐增长到了7亿欧元。她设计的第一款包袋Classic Box就成为了至今仍很抢手的“IT Bag”。

品牌火爆有其原因。要知道，2008年那会儿，大众审美还停留在用大牌来炫富的年代。例如，维多利亚·贝克汉姆就是一个典型例子。当时她以为自己穿着紧身衣，手拎爱马仕包包，脚踩高跟鞋就已经走在时髦前列了。在这样的风气下，菲比·菲洛重新塑造了Céline，也重新定义了奢侈品。

菲比的设计一向奉行极简主义。她的设计哲学是“看不到的设计”以及“廓形的剪裁”。当大家穿紧身裤时，她设计了阔腿裤。去年十分流行的毛毛拖鞋和“奶奶鞋”，菲比其实在几年前就设计过了。许多大牌都借鉴过她的设计，有的甚至如出一辙。因此，菲比的设计让Céline一直走在时尚的前端。

此外，自她接管以后，她统一了店面的装修，整体采用大理石墙面作为店铺形象。整个展示空间也非常干净通透，与她家的衣服和包包一样，没有多余的设计，看起来舒适又不做作。很多人形容Céline为“性冷淡风”。无论是手袋还是时装，Céline给人的感觉永远是简单、高级，没有过多复杂的装饰。

菲比本人一直行事低调，没有社交网络账号，仍保持着看报纸的习惯。每次时装周一结束，她就带着女儿离开，并不接受媒体采访。也许是受菲比的影响，品牌也一直都很低调，没有大力宣传。在这个网络时代，当其他品牌纷纷开启社交网络，Gucci甚至开通网购渠道时，Céline在2018年2月才开通了Instagram账号。不过，它很快就拥有了近70万粉丝。

前几年，Céline在中国还鲜为人知，只有王菲在背后一直默默地为它“打call”。直到菲比·菲洛设计的“笑脸包”（Luggage）横空出世，品牌的知名度才立刻在中国传播开来。

## 菲比时代的经典“IT Bag”盘点

上一节我们了解了菲比·菲洛为Céline带来的变革，本节中我们来细数一下Céline这些年推出的那些经典“IT Bag”。

以下是Céline在菲比·菲洛时期推出的几款标志性手袋：

*   **The Luggage Tote（笑脸包/囧包）**：大家最熟悉的应该是The Luggage Tote，即俗称的“笑脸包”。因为包袋的手柄和拉链组合起来很像一个“囧”字，故也得此昵称。这款包包很实用，容量大，也很有辨识度。当年它非常火爆，从街拍中明星的携带频率就可见一斑。欧美女明星和时尚博主几乎人手一个。后来Céline又推出了小尺寸款式，可拎可背，更受欢迎。
*   **Trapeze（秋千包）**：在2011年冬季，Céline乘胜追击推出了Trapeze“秋千包”。Trapeze指体操或马戏团用的高空秋千。这款包结合了Luggage Bag的“耳朵”和Classic Box的锁扣，具有独特美感且功能性十足。其间隔设计实用，容量也非常大，曾一度是女性上班包的首选。包身采用不同颜色和材料的拼接，首季推出的宝蓝色便卖到全球断货。
*   **Trio系列**：2013年秋冬季推出的Trio系列也很值得一提。它是由三只小包用金属锁扣组合起来的包包。整个包身采用小羊皮，触感柔软。Trio包包没有花哨的图案，仅在顶部印有“Céline”字样，简洁大方，不易过时。若出门不想拿大包，它是很好的小包选择。拆下肩带，可直接当做手拿包使用。
*   **Belt Bag（鲶鱼包）**：在2014年早秋系列中，Céline推出了一款名叫Belt Bag的包包。因为包袋正面有两条长长的纽结皮带，与鲶鱼的胡须很像，故得名“鲶鱼包”。
*   **Clasp Bag**：在2017年春夏系列推出的Clasp Bag让复古趋势回潮。整个包袋的外形是梯形的，开口处有金属边框和开关。合上包包时，开关会发出清脆的“咔哒”声，像是上世纪50年代的包袋。各种配色也透露出浓浓的复古气息。它共有大、中、小三个尺寸，后来还有Wallet on Chain款式。美中不足的是包身宽度不够，从侧面看是三角形，容量并非很大。

## 永恒的经典：Classic Box手袋深度解析

说了这么多“IT Bag”，Céline最出名的还是Classic Box。Classic Box被称为“IT Bag”纯属巧合，因为这款包包是菲比根据Céline在1970年设计的一款包包改进而成的。她保留了原有的包型，只是把马车锁扣换成了黄铜方形锁扣。当时她给这款包包取的名字是“Anti-IT Bag”，没想到它会如此受欢迎。

就连电视剧《我的前半生》里，唐晶经常背的Céline Box手袋也让人印象深刻。

Classic Box最初的经典款采用抛光牛皮。这种皮质是用于掌将头层牛皮卷滚，让其表面微微裂变成半反光的状态。随着时间的流逝，表面的光泽会越来越好看。抛光牛皮表面容易出现划痕，但无需担心，这些划痕和使用印记恰好能呈现出复古的感觉。

后来，Box Bag出现了更多的皮质种类，包括荔枝纹牛皮、羊皮、水波纹牛皮等，有的更有光泽，有的更防划伤，适合不同需求。

内里皮质是区别Classic Box和其他奢侈品包袋最明显的一点。大部分奢侈品包袋内里采用布料或单层皮革。而Box采用的则是实打实的Napa小羊皮，并在表皮和内衬之间加入了一层轻薄的毛毡料，从而形成边缘巧妙的弧度和包体圆滑的曲线。

Box的金属扣是未经任何处理的黄铜扣环。除了故意做旧以呈现复古美感之外，这种做法还能防止仿冒品的滋生。它共有三种大小，最大的在2013年就已停产，现在市面上售卖的是中号和小号。

## Céline手袋的购买策略与市场现象

刚才说到Box的金属扣是未经处理的黄铜扣环。这里分享一个朋友购买Box Bag的亲身经历。

前年圣诞节时，我的朋友在伦敦路过Céline专卖店，想看看是否有中号焦糖色的Box。询问售货员后，没想到真的有货。这款颜色非常难买到。当初安妮·海瑟薇在电影《The Intern》里背过之后，立刻成为爆款，导致全球断货。我朋友一听说有货，便要求看一看。售货员表示，这个颜色太火，必须确定要买才能看。我朋友心仪已久，便说确定要买。结果拿出来一看，发现黄铜金属扣看起来旧旧的，便询问售货员，结果被售货员“鄙视”了。售货员嘴上说着“Box bag is like this.”（Box包本来就是这样的设计），我的朋友赶紧付完钱拿着包走人了。后来他告诉我，背着这个包才体会到设计师的用心，因为每一个细节都在诠释经典。

除了焦糖色、红色、墨绿色等火爆配色，每年Céline还会推出新颜色，但每次都会出现一包难求的情况。

由于销量实在太好，这款包从去年开始就实行了“配货制”。什么是配货制呢？就像爱马仕的Birkin和Kelly一样，不是你想买这个包去了专柜就能买到，你必须先购买同等价位的其他商品，买够了才有资格购买这只包包。好在它不像爱马仕那样是1:1的配比。如果你想买Classic Box，通常需要配一款Céline的首饰。这种销售方式目的是带动配饰的销量。

而且，Céline还为Box采取了限购政策：凭护照购买，每个人在三个月内最多只能买1到2只Box手袋。这种做法是为了杜绝代购的肆意生长。

因为购买Classic Box有各种限制和政策，包袋价格一年比一年高。中号Box从最初的2万元人民币涨到了现在的3万多元，小号也要2万出头，比一般奢侈品牌价格要高，稀有皮质款式就更贵了。但这丝毫不影响大家的购买热情。

## Céline手袋选购指南：推荐与避坑

最后让我们来聊一聊Céline的包袋有哪些“雷区”，以及最值得购买的有哪些。

以下是Céline手袋的选购建议：

*   **最值得购买**：毫无疑问是Céline的Classic Box，特别是那些经典配色。不过，配色和大小最好还是根据自己的喜好来选择，这样才能提高包袋的使用率。
*   **普遍缺点**：Céline家的包袋普遍有一个缺点，那就是皮质较厚，自重较重。因此，并不建议大家购买大号的笑脸包和秋千包。很多身边的朋友买了之后都表示有些后悔。
*   **避坑建议**：如果你的身材并不是特别高大，建议避开任何Céline的托特包。Céline最新推出的Cabas托特包可能非常大，背上身很容易压个子。
*   **其他推荐**：
    *   **Trio包包**：小巧又实用，是不错的选择。
    *   **Clasp Bag**：喜欢复古风的女生可以考虑购买，这款包袋无论背多少年都不会过时，非常值得入手。
    *   **Frame**：最新推出的“IT Bag”Frame也很受欢迎，设计简洁大气，粗肩带的设计也大大提高了舒适度。喜欢拼色包包的人可以考虑。

## 总结

在本节课中，我们一起学习了Céline品牌的发展历史，了解了其被LVMH集团收购的曲折过程。我们深入探讨了前创意总监菲比·菲洛如何以其极简主义哲学和低调的行事风格，让品牌在不依赖传统营销的情况下重获新生并定义新的奢侈内涵。随后，我们详细盘点并解析了由她主导设计的多款经典“IT Bag”，特别是永恒的Classic Box。最后，我们还提供了实用的选购指南，帮助大家了解哪些款式值得投资，哪些可能需要谨慎选择。希望通过对Céline的解析，你能对这个低调而高级的品牌有更全面的认识。
