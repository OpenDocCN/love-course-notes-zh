# 女神养成系列：05：龙脊功前立圆训练 🐉

在本节课中，我们将要学习“龙脊功”中的一个核心训练动作——前立圆训练。这个动作旨在灵活我们的脊椎，即身体中的“大龙”。功法简单易行，站立、坐姿或卧姿均可练习。为了演示清晰，我们将以站姿进行正面和侧面的讲解。

## 预备姿势

首先，我们来学习正确的预备姿势。以下是具体步骤：

1.  两脚开立，与肩同宽。
2.  立身中正，头部摆正，目视前方。
3.  全身保持挺拔且放松的状态。

## 核心动作详解

上一节我们介绍了预备姿势，本节中我们来看看如何完成前立圆的画圈动作。这个动作的关键在于用身体的运动带动手指轨迹。

以下是动作分解步骤：

*   **参照物设置**：将左手悬于空中固定，作为画圆的参照物。想象左手中心有一个垂直的圆形轮廓。
*   **手部放置**：将右手手掌贴于腹部，中指尖轻轻点在肚脐位置。
*   **轨迹描绘**：以身体（特别是脊椎）的运动，带动紧贴身体的右手。用右手中指的意念，沿着假想中左手所在的圆形轨迹，进行画圈。
*   **画圈方向**：轨迹方向为 **向前 → 向下 → 向后 → 向上**，形成一个向前的垂直立圆。

**核心要点**：右手始终轻贴身体，动作的动力来源于躯干和脊椎的旋转与屈伸，而非单纯的手臂运动。

## 侧面演示与要点

从侧面可以更清楚地观察动作的垂直面轨迹。画圈时，请始终牢记“向前、下、后、上”的顺序，保持动作的圆润与连贯。

这个功法主要激活和锻炼我们后背的椎骨节节。如果您在练习时感觉难以控制臀部的晃动，可以采用坐姿（例如坐在凳子上）进行练习，以更好地隔离和专注于脊椎的活动。

## 总结

本节课中，我们一起学习了龙脊功的前立圆训练。我们掌握了从预备姿势到以身体带动画圆的核心动作，并了解了通过坐姿辅助练习的方法。坚持练习此动作，有助于增强脊椎灵活性，为整体健康打下良好基础。
