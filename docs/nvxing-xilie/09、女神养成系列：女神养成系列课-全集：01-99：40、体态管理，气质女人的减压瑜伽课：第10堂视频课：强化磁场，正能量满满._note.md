**练习方法**：
1.  **手印**：食指交叉，中指（智慧指）伸直并拢，其余手指交叉握住。女性左手大拇指在右手之上，男性则相反。
2.  **起始动作**：吸气，手臂向上伸展，上臂位于耳朵两侧，舒展脊柱，肩膀放松下沉，核心收紧，下盘稳定。
3.  **动作与唱诵**：唱诵“Sat Nam”。唱“Sat”时，丹田自然内收上提；唱“Nam”时自然放松。形成“收-放”的律动。

    *   **错误示范**：用喉咙唱诵会导致声音嘶哑；发力错误会导致手臂和肩膀随之上下移动。

5.  **放松**：练习后需立即平躺放松。练习3分钟则放松2分钟，练习5分钟则放松3分钟，再慢慢起身。

### 体式二：强化磁场三式
这套动作自成体系，共需9分钟，每个动作精准完成均有相当难度。

**第一式：单腿下犬式动态练习**
1.  从下犬式开始。
2.  吸气，右腿向上抬起，避免髋部外翻，腿无需伸得很高，感受腿部延展。
3.  呼气，屈膝，头部靠近地面。
4.  吸气，手臂伸直；呼气，再次靠近。动态重复。
5.  **要点**：核心和臀肌收紧。若吃力，抬起的腿可放低一些，支撑腿保持放松。
6.  结束：深深吸气打开，呼气右腿落下，双腿交替踩踏放松骨盆，回到婴儿式休息。

**第二式：盘坐手臂练习**
1.  盘坐。左手伸直，掌心朝右；右手翻转，拇指在下，掌心也朝右。双手手背外缘相扣。
2.  保持手臂与地面水平。吸气，耸肩将手臂上抬至60度角；呼气，回到水平位置。
3.  有力呼吸带动动作，练习3分钟。
4.  结束：深深吸气，保持在60度角，屏息，收根锁，关注眉心。然后深深呼气，慢慢落下放松。放松时避免大幅度活动肩部，以保持能量状态的稳定。

**第三式：水平开合臂**
1.  双臂向前伸直，与地面平行，掌心相对，相距约15公分。
2.  吸气，双臂向两侧后方尽可能打开；呼气，回到起始位置。
3.  **要点**：脊柱和根基保持稳定，避免甩动。
4.  练习3分钟。后期手臂可能下沉，但请尽量坚持，动作可以放慢但不要停止。接近结束时，你可能会捕捉到心理电磁场被净化的精细感觉。
