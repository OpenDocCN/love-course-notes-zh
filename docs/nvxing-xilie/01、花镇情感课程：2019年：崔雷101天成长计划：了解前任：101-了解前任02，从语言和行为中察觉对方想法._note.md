# 情感关系课程：01：从语言和行为中察觉对方想法

在本节课中，我们将学习一个核心思想：个人在情感问题中的责任归属，并探讨如何通过提供有价值的建议来建立更深层次的关系连接。我们将通过具体的例子和策略，帮助你理解如何有效地支持他人，同时维护健康的互动边界。

## 思想核心：责任归属

上一节我们介绍了情感问题中常见的无助心态，本节中我们来看看这种心态背后的根本逻辑。

人在遇到情感问题后，常会陷入“我没有办法”的自我暗示中。这相当于将自己置于一个弱者的位置，如同四肢无力一般。当人陷入这种状态时，外界的任何建议通常都无效。这就像身处雨中，却声称自己无法让雨停下，将自己放在一个无能的状态里。

此时，人极度渴望被他人拉起，希望别人替自己承担责任。但这种状态并无益处。人必须学会自己站起来，对自己负责。需要明白一个关键点：无论做什么事，即使别人为你出谋划策、给出建议，最终的执行者和责任人依然是你自己。

我举一个最简单的例子。假设一个人对你说：“我没有办法赚钱了。”我告诉你：“你去偷东西吧，偷了就有钱了。”你听了我的话去偷窃。请问，你会真的听从这种建议吗？大部分人不会。

但假设你听了，并且因此被警察逮捕。那么，坐牢的人是你还是我？你或许会辩解：“不是我要偷的，是那个姓崔的人让我偷的。”你认为警察会接受这个理由吗？警察一定会说：“是你自己实施了偷窃行为。”理解了吗？

到那时你才会后悔，甚至怨恨出主意的人。这个逻辑就是：你让别人替你做出决策，但你去执行，责任人依然是你自己。

再比如买彩票。你问我：“崔老师，该买哪一注彩票？”我告诉你：“老夫今天掐指一算，这个号码可能中奖。”结果你中了500万。那么，这500万你会分给我吗？领奖时，你会把奖金给我吗？显然不会。

同理，我指导一个男生去追求一个女生。如果追成功了，这个男生会说“这个女孩是崔老师教的，因此我要把她介绍给你”吗？不会。为什么？因为好处他会自己留着。

这意味着：别人指导你做一件事，**好处是你自己的，坏处也是你自己的**。

所以，今天第一个核心思想是：**任何人给你任何决策建议，最终的执行人和责任人都是你。不要指望别人替你谈恋爱或承担后果，这是不存在的。** 当然，我也不会指挥你去偷东西。即便你真的去偷了，责任人是你，而不是我。

## 建立连接的关键：提供价值

理解了责任归属后，我们来看看如何在关系中通过行动创造积极影响。当对方提出具体困扰时，你的回应方式至关重要。

有人问：“追求男人最重要的两个点是什么？”这个问题非常好。第一个点是**好看**。第二个点，说到底是**好看**。但除了外在，内在的支持同样关键。

例如，你的伴侣抱怨：“我的销售团队很难遇到好的销售人才，不是爱抱怨，就是话术一般。”你该如何回应？如果你帮不上忙，就无法体现你的价值。

以下是两个具体的策略：

**策略一：主动吸引人才**
你可以建议：“亲爱的，你可以将个人信息登记在保险公司的网站上。当你提交信息后，每天都会有不同公司的销售人员给你打电话。那些坚持不懈、试图说服你、有韧性、有激情、懂得客户心理、一遍遍联系你并试图加你微信的销售，正是你需要的人才。你可以通过接听这些电话，来观察和识别优秀的销售者。”

**策略二：直接挖掘人才**
你可以进一步建议：“你可以尝试与其中优秀的销售约见面谈。你可以说：‘我对你的产品很感兴趣，但觉得当面聊更好。’可以约到家里或咖啡馆。见面后，你可以直接与他沟通：‘小伙子，我觉得你销售能力很强，有没有兴趣来我的公司？’这样，你就能直接为公司挖掘到人才。”

这意味着，男性在日常工作和生活中，常会遇到寻求社会认可（如领导、同事、客户认可）的痛点。当他确实有困难时，虽然我之前说不要轻易给建议，但有时**给出切实可行的建议，能在他心中大大加分**。如果给不出，至少不会扣分。

因此，你脑子里需要储备一些“套路”和方法。当他遇到销售团队的问题，你给出可立即操作的策略，这就是在提供价值。他若采纳并取得成效，你就帮到了他。

## 回应赞赏：谦逊与赋能

当你的建议生效，对方夸赞你“亲爱的，你真棒”时，你该如何回应？有人说：“那你要怎么奖励我？”这句话的言外之意是“我比你棒”或“我就是很棒的”。也有人说：“那是因为你比我更棒。”

更好的回应方式是：“这个建议只是我临时想到的，我没想到你能执行得这么出色。亲爱的，我很佩服你。”

这营造了一种类似“皇上与大臣”的感觉：皇上（对方）看似没有主意，建议由大臣（你）提出。皇上采纳后，大臣要说“皇上英明”。你给了他建议，帮到了他，然后你把功劳归给他，维护了他的面子。

男人不傻，他知道哪个“大臣”是真正能帮到自己的，也知道谁在浑水摸鱼。你这么做，等于让他欠你一个人情。他日后会以某种方式回报你。

所以，当男人真诚地向你求助时，不要回避或说“我相信你就是对的”。你需要灵活应对，给出切实的建议和策略。当他因你的建议而变好时，你要将功劳归于他的执行。

---

本节课中我们一起学习了情感关系中的责任归属原则，明白了个人永远是自身行动的最终责任人。同时，我们探讨了通过提供具体、有价值的建议来在关系中建立深度连接的方法，并学习了在建议被采纳并取得成效后，如何以谦逊的方式回应赞赏，从而巩固关系。核心在于：**赋能他人，而非替代他人**。
