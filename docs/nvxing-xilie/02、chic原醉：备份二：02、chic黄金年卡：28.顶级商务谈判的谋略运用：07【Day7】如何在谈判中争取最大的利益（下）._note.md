# 谈判哲学：02：顶级商务谈判的谋略运用：07【Day7】如何在谈判中争取最大的利益（下）

在本节课中，我们将继续学习在优势谈判格局下，如何运用内在能量模型中的“梦想家”与“勇士”人格属性，来最终达成谈判目标，并争取最大利益。

上一节我们介绍了如何利用“思想者”和“恋人”来思考双方的复合需求与底线。本节中，我们来看看如何运用“梦想家”和“勇士”来推动谈判走向成功。

## 🎯 梦想家的目标：化对立为统一

“梦想家”主要负责目标的制定。在优势格局中，我们拥有更强的话语权和选择权，因此谈判的核心目的是尽可能多地争取自身复合需求的满足。

与劣势格局下只能满足1-2个核心诉求不同，优势格局为我们提供了争取更多利益的筹码。然而，任何一方都不希望自己过于被动。若对方感觉利益被持续挤压，合作关系也难以长久维持。

因此，“梦想家”需要完成两项主要任务：
1.  准确识别谈判目的：即尽可能多地争取复合需求的满足。
2.  缔造双方共识：将针锋相对的对立局面转化为统一目标，让对方感到其诉求同样能得到满足，而非被压榨。

这听起来可能有些不可思议，但通过一个著名案例可以清晰理解。

### 案例分析：阿里巴巴收购高德地图

2014年，阿里巴巴斥资14亿美元全资收购高德地图。当时，高德虽是美股上市公司，但在用户数量和资金体量上远不及阿里。作为占据31%市场份额的手机地图领头羊，高德正面临用户破亿却难以商业化的瓶颈。

阿里收购高德，旨在补足自身在电子地图领域的短板，并借助其数据能力开拓线上线下联动市场。对高德而言，阿里的资金和用户资源极具吸引力。

这个案例展示了优势谈判中“梦想家”将双方目标统一的三步法：

以下是“梦想家”统一目标的三步法：

1.  **找出对方核心痛点，以己方优势填补**：阿里找到了高德商业变现的痛点，并以自身雄厚的资金实力作为解决方案。
2.  **避免暴露己方痛点和渴望，保持姿态**：即便占据绝对优势，也应避免流露对达成合作的强烈渴望，以防主动权旁落。可采用“佛系”姿态，暗示己方拥有其他选择。
3.  **包装己方诉求，转化为共同价值**：将己方的部分需求，包装成能为对方开辟新发展路径的机遇。例如，阿里将高德定位为大数据来源，为其业务发展提供了新的可能性，从而将谈判焦点从价格对峙转向商业版图的统一。

在实际谈判中，例如品牌方与供应商因价格僵持时，可以转而强调合作能为对方带来的品牌背书价值，从而创造共赢点。

至此，“梦想家”的使命已基本完成。接下来，轮到“勇士”登场，负责落实具体的执行细节。

## ⚔️ 勇士的执行：以退为进的策略

在相对强势的位置上，我们虽在价格和条件上拥有更大话语权，但为了尽可能满足复合需求，同时避免显得巧取豪夺，需要一种巧妙的执行策略。

当前期通过“思想者”和“恋人”探知对方底线，并通过“梦想家”统一目标后，“勇士”可以采用 **“以退为进”** 的方法。此策略旨在让对方心理产生足够的“消费者剩余”，从而强化合作意愿，促成最终合作。

**消费者剩余** 是一个经济学概念，指消费者愿意为某商品支付的最高价格与实际支付价格之间的差额。这个差额会让消费者感到“占了便宜”。

具体操作步骤如下：

以下是“以退为进”策略的执行步骤：

1.  **开出高价**：在探知的对方底线附近，首先提出一个会让对方感到非常为难的较高要价（或较严苛的条件）。
2.  **主动退让**：随后，做出让步的姿态，退回到一个相对更合理、但仍有利于我方的价格（或条件）。
3.  **创造剩余感**：我方主动退回的这部分利益，会在对方心理上产生显著的“消费者剩余”感，使其觉得获得了实惠，从而更愿意达成合作。

例如，在商场中，一件标价1000元的衣服可能让人犹豫，但若旁边出现“5折”标识，消费者会立刻感到“捡了便宜”，购买意愿大增。在谈判中，处于劣势的合作方对于强势方的任何让步都会格外敏感，因此“以退为进”能有效促成合作的临门一脚。

## 📚 本节总结

本节课中，我们一起学习了在优势谈判格局下如何运用“梦想家”与“勇士”。

*   **梦想家**的核心任务是**降低对立、统一目标**。通过识别对方痛点、隐藏己方渴望、包装共同价值这三步，将谈判从零和博弈转化为共赢局面。
*   **勇士**的核心任务是**落实执行、促成合作**。采用“以退为进”的策略，通过先扬后抑的方式创造对方的“消费者剩余”心理，从而强化其合作意愿，最终达成协议。
