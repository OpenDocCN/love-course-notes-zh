# 算爱研习社2020年费会员（更新中）：1主课（专题）：04恋爱·这样谈恋爱才不会分手：20为什么你一谈恋爱就好像换了个人

在本节课中，我们将要学习四大情感模式的本质、特点及其在恋爱关系中的具体表现。了解自己的情感模式，有助于我们理解自身在亲密关系中的行为模式，从而更好地经营感情。

上一节课我们介绍了什么是情感模式，以及情感模式带给我们的指导意义。每个人都通过测试题测出了自己的情感模式，可能是单独优势型，也可能是两两混合型。

这里需要特别说明，我们所测的情感模式是与生俱来或已经养成的，而非被矫正的模式。被矫正的情感模式是指，例如你原本是自虐型或掌控型，但为了伴侣而将自己调整为取悦型。短期内你们可能相处融洽，但时间一长，双方都会回归自己原本的情感模式，争吵便会增多。改变或调整只能持续一小段时间，无法长久。许多恋情开始时感觉特别合适、舒服，但时间久了就开始各种争吵，原因并非人变了，而是一开始双方伪装了自己的情感模式。时间长了，伪装无法维持。恋爱越谈越难、争吵越来越多，根本原因在于双方的情感模式本就不匹配，却硬生生地调整成看似匹配的样子。当真相显现时，结果会说明一切。

所以，从这节课开始，我们将深入讲解四大情感模式的本质、其真正的为人处事方式，以及在恋爱过程中的具体行为。我会详细展开讲解每一种模式。大家不要只听自己的模式，也要了解其他模式，因为你未来的伴侣可能是其他模式。了解哪种模式更适合你，对你有帮助。每一种情感模式没有好坏之分，它只是决定了你在各种情境下会做出何种抉择、如何相处。模式本身并无对错，就像星座、血型一样，你只需了解自己属于哪种模式即可。

在展开讲解四大情感模式时，我刻意规避了每个模式中的缺陷或缺点。因为人类不爱听自己的缺点。如果我指出享受型模式有一堆缺点，你可能会抗拒，甚至不想成为这种模式。因此，我尽可能讲述中性或偏好的方面，先帮助大家接纳自己的模式。

## 第一种模式：享受型模式 😌

享受型模式的核心特点是心安理得地接受伴侣带给自己的一切好处。

以下是享受型模式的主要特点：

*   **心安理得地接受**：认为从伴侣那里获取好处是天经地义的。例如，伴侣提供帮助、赠送礼物、支付账单时，会欣然接受，不会过多思考背后的含义或觉得亏欠。
*   **高忍耐度源于需求满足**：如果伴侣能满足其大部分需求（如物质、陪伴、精神需求），享受型的人会有很高的忍耐度，甚至会强迫自己忽略对方的不完美之处。他们会认为此人是“天生一对”。
*   **情绪直接，雨过天晴**：其生气往往源于需求得不到满足。一旦需求获得满足，情绪会很快好转。如果长期无法满足核心需求（如物质、陪伴、精神共鸣），则可能爆发并导致分手。
*   **在关系中容易“放弃思考”**：这不是缺点，而是一种特点。在恋爱中容易变得“傻”一些，愿意依赖对方，可能在一定程度上丧失独立性和自主能力，喜欢安逸和稳定。
*   **心大，忽略细节**：过得好的时候不会去多想不好的事情或潜在危险，因此通常日子过得比较舒心，显得“没心没肺”。
*   **反应直接**：对事情的反应直接了当，不会过度思虑。例如，伴侣抱怨被冷落，会直接回应“我多陪陪你”，而不会陷入自责或过度分析。
*   **吵架后易和好**：只要伴侣愿意继续满足其需求，甚至愿意主动降低一些需求以便和好。除非伴侣完全无法满足其绝大部分需求，否则关系不易破裂。
*   **被分手概率较大**：因为不关注细节、心大，可能对伴侣的变心（如出轨、移情别恋）后知后觉，常常在莫名其妙中被分手。
*   **发现并选择“供养者”**：擅长发现并能满足自己需求的人，并选择这样的人作为伴侣。但这也带来一个相关性：这样的伴侣可能同样能满足他人需求，且因感知到你的高忍耐度而缺乏恐惧感，因此出轨概率相对较大。

**核心公式/代码描述**：
`满足度 = 伴侣提供的需求； 忍耐度 = f(满足度)； 当满足度高时，忍耐度极高，忽略缺点；当满足度持续低下时，关系破裂风险激增。`

## 第二种模式：自虐型模式 🥀

自虐型模式的核心本质是因内心脆弱而表现出的坚强与警惕。

以下是自虐型模式的主要特点：

*   **警惕心强**：对于伴侣突然的好意或赠与会心存疑虑，思考原因，不会像享受型那样坦然接受。
*   **不主动提要求**：倾向于自己的事情自己做，能找朋友就不找伴侣。在爱情中更希望自己的需求能被对方主动发现并满足，认为“要来的就不值钱了”。
*   **隐藏情绪**：如果需求被主动满足，会暗自高兴（暗爽）；如果没被满足，则会生闷气，但都不会充分表现出来。越是亲密的关系，越倾向于隐藏真实情绪。
*   **渴望“懂得”**：认为“如果你懂我，就不需要我说”。对方若能不言自明地理解其喜好，才会觉得对方有资格在一起，但不会认为“非他莫属”。
*   **因脆弱而坚强**：内心脆弱，害怕受伤和失去，因此像刺猬一样用“刺”（冷漠、警惕）保护自己。对伴侣一丝一毫的冷漠都非常敏感，并随时准备离开。
*   **怀疑人性**：认为人都会变，随时做好对方会变的准备。
*   **不愿主动追求**：即使知道自己想要什么，也不愿主动追求，更希望对方能懂并主动给予。
*   **不展现软弱**：在恋爱中不愿展现痛苦和软弱的一面，最多以冷漠脸示意。
*   **情感决绝**：信条是“不喜欢的，热情了也没用；喜欢的，冷漠了就不可能”。一旦感知到对方的冷漠或分手意图，往往会果断选择分手，且大多不吃“回头草”。

**核心公式/代码描述**：
`安全感 = 感知到的对方主动理解与付出； 关系稳定性 = g(安全感)； 当安全感低时，防御机制启动，倾向于疏离或决绝分手。`

## 第三种模式：掌控型模式 🧭

掌控型模式的核心特点是在关系中追求主导权和改变对方，以实现自己设定的目标或标准。

以下是掌控型模式的主要特点：

*   **希望对方听从与改变**：伴侣听话、对自己好，会觉得满意；反之则会感到“膈应”（不舒服）。会主动要求伴侣满足自己某些需求。
*   **乐于付出，但方式强势**：经常思考伴侣需要什么，并直接去办，但可能不同询对方意见。总觉得对方不够成熟，像需要自己照顾的“孩子”。
*   **总想改变对方**：希望改变伴侣的性格、工作、生活习惯等，但总觉得对方“很难改”，容易忽略对方已做出的改变，而紧盯未改的部分。
*   **感到“累”**：因为总觉得在单方面督促、照顾对方，而对方改变缓慢，所以相处中常感到疲惫。
*   **对情感走向有把握**：对自己的情感决策比较理性、有逻辑，分手后痛苦但不会持续很久，能自主走出来。纠结少，但会有目标未实现的失落感（如物质目标）。
*   **对物质偏佛系**：对物质满足态度较为淡然，有也可，没有也可，更看重掌控感。
*   **希望了解对方动向**：更希望掌握对方的行踪、密码等信息，了解对方在做什么。
*   **事实上的主动者**：在亲密关系中通常处于事实上的主动地位（但有些人因心理防御不愿承认）。姐弟恋（女方掌控型）或男方年龄偏大（男方掌控型）的组合较常见。
*   **“大男子主义/直男癌”倾向**：男性掌控型模式可能表现出此类特征。若双方都是掌控型，容易产生权力争夺。

**核心公式/代码描述**：
`控制感 = 对方符合期望的程度 + 对关系的了解程度； 满意度 = h(控制感)； 当控制感强时，关系稳定；当控制感弱（对方不听话、难以改变）时，产生疲惫感和冲突。`

## 第四种模式：取悦型模式 🤝

取悦型模式的核心特点是希望通过满足他人需求来获得价值感与和谐关系，但长期单向付出可能导致情绪问题。

以下是取悦型模式的主要特点：

*   **不由自主希望对方开心**：总是通过满足对方需求的方式来让对方舒服，不仅在亲密关系，在生活中也常如此。
*   **在“被需要”中获得满足**：伴侣需要自己时，会产生强烈满足感；不需要时，则容易焦虑、胡思乱想。
*   **察言观色，同理心强**：特别在乎他人感受，不想让人难受或不满。擅长察言观色，能敏锐察觉他人需求，甚至不需要对方明说。
*   **害怕尴尬**：有强烈的“尴尬症”，害怕气氛冷场，会主动活跃气氛。
*   **付出多，获得少**：在情感中通常付出较多，但自我需求感较低，因为觉得“接受意味着要加倍回报”，很累。
*   **出轨率与主动分手率可能偏高**：原因包括：(1) 对人好，机会多；(2) 对单一伴侣情感依赖度可能相对不高；(3) 取悦行为并非专属一人，对建立情感联系的人都会取悦；(4) 长期单向付出导致情绪积压，可能通过出轨或分手寻求宣泄。
*   **主动热情，追求容易**：在感情中偏热情，会主动发起攻势，且因天生的取悦特质，追求成功率较高。
*   **忍耐度好，容忍底线低**：可以容忍出轨、暴躁、争吵等多种在常人看来超出底线的事情。喜欢照顾人，甚至延伸至伴侣的亲友。
*   **自我认知偏低**：即使外人觉得很好，自己却常觉得不够好，发生问题时容易从自己身上找原因，自责。
*   **情绪爆发与“情绪吞没”**：长期单向情绪输出会导致情绪积压，可能在某个节点突然爆发（情绪宣泄），或陷入“情绪吞没”（被情绪占领，失去理性）。但这种爆发通常是宣泄，宣泄后可能自我调整恢复。然而，没有永远的取悦，长期不平衡的付出终将导致关系崩坏。

**核心公式/代码描述**：
`自我价值感 ≈ 感知到的被需要程度 + 他人愉悦程度； 情绪负载 = 持续单向付出 - 获得的情感回馈； 当情绪负载超过阈值时，发生情绪爆发或关系破裂。`
