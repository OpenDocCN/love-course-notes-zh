## 二、当代婚姻的挑战与机遇

近年来，离婚率快速上升。有调查显示，在2018年的离异人群中，由女性主动提出并坚持离婚的比例超过50%。这反映了女性社会地位和经济实力的提升，她们对婚姻和生活品质的要求也越来越高。

然而，这也带来了一个潜在风险：人们越来越习惯于“以新换旧”，越来越容易放弃。爱情似乎变成了一种消费品，一言不合就分手。当我们越来越不习惯坚持时，放弃的东西也会越来越多，从一件礼物到爱情、婚姻乃至个人梦想。

## 三、一个关于“放弃”的故事

我的闺蜜“想想”最近果断向男友提出分手，原因是男友从欧洲回来后联系变少，她认为对方不再在乎自己。男友看到分手信后，连夜打电话解释，原来他因倒时差和重病而无力频繁联系。

如果不是男友的坚持，这段感情就会终结于一场误会。想想可能会继续寻找下一段感情，但类似的场景很可能重演。我们身边有多少人像想想一样，因为轻易放弃而不断与真爱擦肩而过？

你可能会说，你遇到的不是这种小误会，而是真正的痛苦，比如伴侣出轨。即便如此，痛苦是真实的，但这不代表关系无法挽回。当你开始爱自己时，你便有了解决麻烦的力量，快乐会越来越多，伴侣也很有可能带着爱意回归。

这是作者的亲身经历，也是许多学员的真实经历。你同样可以做到。

## 五、本书的三个核心关键点

以下是本书精华内容的三个关键点，我们将逐一进行探讨。

### 关键点一：了解你的结婚动机

首先，我们需要问自己：我为什么要和这个人结婚？了解结婚动机，能让我们接近未经修饰的爱的真相。

这个真相背后，可能隐藏着其他需求：你觉得他魅力非凡、才华横溢、功成名就、生活富裕、温柔体贴。很多时候，人们选择婚姻并非因为爱，而是爱背后的这些需求。

**案例分析：小南的故事**

我的朋友小南曾是品牌总监，理性干练。她爱上了一位新加坡富二代“清”，但清的家族情况复杂，婚姻不能自主。在确认关系的不确定性后，小南感到疲惫，此时大学同学“丹”出现并求婚。小南清楚自己并不爱丹，但丹能给她一个安稳的家。她接受了求婚，动机是满足自己“休息”的需要，而非爱情。

小南清楚自己的动机，因此对丹没有过多的爱的要求，婚姻得以平静维系。了解自己的结婚动机，有助于在婚姻中做出明智的取舍。

**反面案例：丽丽的故事**

学员丽丽为了留在北京而结婚，婚后却开始渴望浪漫爱情。她的丈夫为养家拼命工作，无暇顾及她的情感需求。丽丽的不满和丈夫的委屈导致关系恶化，最终以离婚收场。如果丽丽能意识到最初的结婚动机（留在北京），并对丈夫的付出心存感激，结果或许会不同。

结婚动机多种多样：可能是累了想休息，可能是改善物质生活，也可能是寻找灵魂伴侣。重要的是，在婚前清晰了解自己的动机，并明白选择伴随的后果。

**公式：结婚动机的后果**
```
选择对方的魅力 → 可能需承担其向他人献殷勤的后果
选择对方的财富 → 可能需承担孤独与寂寞
选择对方的体贴 → 可能需分担其软弱与纠结
```

### 关键点二：认清婚姻的真相

上一节我们探讨了结婚动机，本节中我们来看看婚姻的真相。婚姻并非实现所有幻想的天堂，它充满了没完没了的冲突。

童话故事总在“王子和公主幸福地生活在一起”处戛然而止，因为婚姻里多是鸡毛蒜皮的磕绊。我们会不自觉地将许多未完成的期待投射到伴侣身上。

从心理学家庭治疗的观点看，在所有的人际关系中，爱人是唯一可以替代“重要他人”（如父母）的角色。我们在婴幼儿时期的创伤，可以在健康的亲密关系中获得疗愈。因此，我们常将自身缺乏的关爱、内心的脆弱都寄托于婚姻中的另一半，一旦期望落空，便会产生无尽的争吵。

**婚姻中最棘手的问题：外遇**

在诸多婚姻麻烦中，外遇最让人伤心。本书作者认为，**外遇不是第三者的错**。从家庭治疗视角看，当一个人无法在亲密关系中得到回应时，就会把注意力转向第三方（如工作、孩子或婚外情）。因此，**被背叛的人往往是先从关系中抽离的人**。

**案例分析：辛娜的故事**

学员辛娜的丈夫出轨并提离婚，她感到无比委屈。辛娜包揽了家里一切大事小情，买房、装修、照顾双方老人。她与丈夫几乎没有交流，并且因为不信任丈夫的能力而拒绝生孩子。在这段婚姻里，丈夫成了一个“不存在”的代号。

当丈夫在婚姻中找不到自己的位置和互动对象时，便将注意力转向了工作和第三者。辛娜通过学习意识到是自己先冷落了丈夫。她主动道歉并邀请丈夫回家，同时尊重他的选择。最终，丈夫回归，两人关系重修旧好，辛娜也怀孕了。

亲密关系中的忽视不一定是冷言冷语。当反复无视对方的意见，按照自己的想法行事，关系便会断裂。虽然同住一个屋檐下，但真实的关系已不复存在，外遇便可能发生。

本书的一大警示是让我们看到婚姻的真相：矛盾、冲突、性、外遇、金钱问题等都是婚姻生活的一部分。幸福并非从不吵架，而是在争吵后仍愿意握着彼此的手说：“我爱你，谢谢你给了我这个婚姻。”

### 关键点三：如何通过爱自己获得幸福婚姻

在我们了解了结婚动机和婚姻真相之后，我们来看看如何通过爱自己获得美满的婚姻。婚姻中的所有问题都源于我们自身的过往，那么，怎样才是爱自己？

作者给出的解决方案可以简化为一个公式：

**核心公式：爱 = 自律 + 坚持 + 宽容 + 收获**

以下是这个公式的具体解读：

1.  **自律**：对自己有较高要求，提醒自己放下执念，打开心扉，不断学习。
2.  **坚持**：在面对障碍时，愿意给彼此多一次机会，不因一句话或一次冲突就轻易放弃。
3.  **宽容**：首先是对自己宽容，然后才是对他人。原谅自己的过错，允许自己不完美，便不会再苛责他人。宽容不等于将他人的过失揽到自己身上，而是识别出自己内心的伤害标记，决心抛弃往日积怨。
4.  **收获**：做到以上三点，自然会收获惊喜、快乐、爱情和稳定持久的幸福婚姻。

你可以通过训练来培养宽容，以伴侣、父母、朋友等为练习对象。每天提醒自己对他们保持宽容。这虽然不易，但坚持下去，你会发现自己内心变得柔软，关系也越来越融洽。

## 六、课程总结

本节课中，我们一起学习了《爱自己，和谁结婚都一样》的核心内容。

首先，我们需要知道自己结婚的动机是什么，并真实地面对它。
其次，我们要认清婚姻的真相，拥有解决矛盾和冲突的勇气与方法。
最后，我们要通过爱自己——主要是**自律、坚持和宽容**——去收获稳定、持久的婚姻。

爱自己，意味着你能从伴侣身上发现自己的内心弱点，并共同成长。
爱自己，意味着你能把婚姻变成疗愈心灵的港湾，与爱人建立有效沟通。
爱自己，意味着你能为孩子创造健康的家庭环境，让他们学会如何建立和睦的家庭。

当你真正开始爱自己，整个世界都会来拥抱你。你便能实现“执子之手，与子偕老”的温暖与幸福。
