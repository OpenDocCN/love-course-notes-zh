# 理财投资基础：第28课：如何找到一家好公司并成为它的“创始人” 👑

## 概述
在本节课中，我们将学习股票基本面分析的核心方法。我们将从“创始人”的视角出发，学习如何通过定性分析和定量分析，在好行业中筛选出一家好公司，为后续在好价格时买入打下基础。

股票投资方法众多，主要思路分为两大流派：基本面分析和技术分析。本节课从基本面分析说起。

基本面分析包括宏观经济运行态势和上市公司的基本情况。通俗来说，基本面派的目标是在好行业中选出一家好公司，然后在其价格便宜时买入。即：好行业、好公司、好价格。基本面派，其实是在挑选股票市场上的“三好学生”。

这三者构成了一个比较完整的股票投资体系。本节课，我们先探讨前两“好”：好行业和好公司。

## 定性分析与定量分析
你可能会觉得分析公司很遥远，不妨换个角度思考：如果你是你现在所在公司的老板，你会给自己的公司打几分？无论分数高低，你都能给出理由。大到行业前景、公司盈利能力，小到内部文化、员工待遇。判断一家上市公司的方法也很类似。

我们可以把所有信息分为两大维度：定性和定量。

顾名思义，定性分析需要我们更多关注那些与数字无关，但与公司所处行业、业务增长、内部结构等相关的要素。对于定性分析，建议学习管理学知识并关注行业报告。

定量分析则是从数据出发，我们可以重点关注公司的财务数据。

## 如何进行定性分析？
我们先来探讨定性分析。可以借用管理学中一个著名的框架分析方法：SWOT原则，来看如何定性分析一家好公司。

这个模型通过两个维度（外部/内部、好的/坏的），将一个企业分为优势、劣势、机会和威胁。你可以把目标企业的重要信息罗列进去进行分析。

这样做的好处是：投资一只股票，本质上是成为这家企业的股东。像创始人一样分析企业的优劣势以及潜藏的机会和威胁，可以帮助你做出更正确的投资决策。

所以，我们不妨站在创始人的角度，一起来分析公司的经营情况。

以下是四个常见的定性分析角度：

### 1. 行业前景
这个时代人人都怕入错行，不同行业的股票差别也非常大。我们列出三个代表性行业的历史数据：最近4年，互联网、食品饮料和钢铁行业的涨幅分别是27%、80%和40%，差别巨大。尽量选择高速发展的朝阳行业，远离那些容易“踩坑”的夕阳行业，是我们的选择。

当然，筛选行业的同时，你还需要做更多有针对性的研究。阅读行业研究报告是比较好的方法。

### 2. 竞争对手分析
街边卖炒饭的摊贩都需要考虑周边有哪些饭店。在确定行业之后，我们也需要看看行业里的竞争对手。

如何查找行业竞争对手？可以通过专业财经网站的行业板块，先找到对应行业，再查看其中有多少家上市公司。也可以在PC端的股票行情软件和手机端的股票APP中方便地查找到。

### 3. 竞争优势
如何与竞争对手竞争？只有具备竞争优势，扬长避短才行。

我们以大家熟悉的白酒企业贵州茅台为例。在白酒行业中，贵州茅台业绩长期排名第一，属于龙头企业，拥有强大的竞争力。它有两个非常独特的优势：一是品牌，在消费者心中认知度非常高；二是地理位置，茅台镇独特的气候很适合微生物生长，酿出的酒质口感有特色，无法复制。

显然，因为拥有这些优势，白酒行业中其他公司很难超越茅台。比如同样作为白酒企业的洋河股份，没有得天独厚的自然环境优势，也没有“国酒”名号，所以更需要通过广告宣传让更多人了解。两者竞争力上的差距在股价上也可以看到。

具体如何了解一家企业的竞争优势？首先推荐阅读上市公司的财报，其中“公司业务概要”和“经营情况讨论与分析”这两节就有对公司情况的具体介绍。此外，也可以从券商机构的分析研报中获得有价值的信息，例如在慧博资讯等平台可以检索到相关研报。

### 4. 公司战略
最后一个角度，如何把竞争优势和外部机会结合在一起，很大程度上决定了公司的发展策略。

还是用贵州茅台举例，因为茅台有非常独特的生产优势，结合白酒在中国高端饭局中的独特地位这一行业机会，茅台一直走高端战略，能把优势发挥到最大。

再对比洋河股份，没有茅台的独特生产优势和品牌优势，其目标消费群体定位是中高端人群。为了满足更多消费者的不同需求，它还推出了一些中低端品牌的酒，以争取更多的市场份额。

所以，虽然都是白酒企业，但因为两家竞争力方面的差距，导致公司选择了不同的发展策略。这些都是在挑选公司股票时需要事先了解的信息。

至于公司战略，除了同样可以从财报的“公司业务概要”和“经营情况讨论与分析”中找到文字介绍外，还可以搜索公司高层的新闻报道和公开讲话，也能找到不少有用信息。

当然，上面四个角度只是一个简单的定性分析框架。管理学中还有很多其他方法，比如波特五力模型、波士顿矩阵等。

## 如何进行定量分析？
说完定性分析，再来看看定量分析。想要更深入了解一家企业的经营情况，学会阅读财报是必不可少的技能。

上市公司的财报就像一份细致、全方位的官方“体检报告”。它不仅是一手资料，而且有统一的标准。你可以从公开渠道获取其中的财务数据做详细分析。

### 如何查找财报？
在阅读财报之前，得先学会怎么查找财报。我们列举了四个主要渠道：
*   上海证券交易所官网
*   深圳证券交易所官网
*   巨潮资讯网
*   上市公司官网的“投资者关系”栏目

不妨课后就动手试试，找找你心仪公司最近几年的财报。

下载财报后，你会看到披露期间的财报一般分成4种：一季报、半年报、三季报、年报。时间跨度越大，内容越详细，其中年报含金量最高。

### 财报重点看什么？
一份财报通常有100多页，内容多且专业。我们应该看哪部分？除了上面提到的“公司业务概要”和“经营情况讨论与分析”这两节之外，还有一个章节叫做“财务报告”。整份财报最重要的财务数据就在这里。

我们重点来聊聊会计报表。会计报表最核心的就是这三张表：**资产负债表**、**利润表**和**现金流量表**。要读懂财报，最核心的就是读懂这三张表。

#### 利润表：看公司赚钱能力
利润表反映一个时期内公司的赚钱能力强不强。可以看其中两个典型指标。

**首先，看成长速度。**
可以从营业收入和净利润看一个公司成长速度如何。一个简单的方法是，用今年的营业收入和净利润跟去年对比，看看增长速度。以贵州茅台和洋河股份为例，从数据能看到茅台的成长能力很强，高过后者。

**其次，看赚钱效率。**
通俗来说就是这个企业能不能赚钱。这个能力我们用**毛利率**来衡量。

什么是毛利率？以餐厅为例：
*   如果2016年营业收入是10万，营业成本是5万，那么毛利润就是5万块。
*   毛利率就是用毛利润除以营业收入：`毛利率 = 毛利润 / 营业收入`。
*   所以毛利率是 `5万 / 10万 = 50%`。

这5万块毛利润还要再减去人工、水电等费用，到手的才是赚到的钱。所以假如毛利率太低，餐厅就有亏本的可能。一般来说，好公司的毛利率往往不低于30%，否则在扣除税费、管理费等其他支出后，很可能就赚不了多少钱。

同样，我们来看看贵州茅台和洋河股份的毛利率。茅台毛利率超过了90%，非常惊人，也远远高于洋河股份，印证了它强大的竞争力。

为了帮助大家理解，我们模拟了一张简单的利润表。其实上市公司的利润表大体都是类似格式。看懂这张表和背后的计算逻辑，再看复杂一点的利润表时，也不会觉得没方向。

#### 资产负债表：看公司财务实力
资产负债表顾名思义有2块内容：资产和负债。
*   **资产**：指企业拥有的具有商业或交换价值的东西，预期未来会给企业带来经济利益或资源，比如现金、厂房、机器设备等。
*   **负债**：指企业所承担的能以货币计量，将来需以资产或劳务偿付的债务。

这张表主要反映一个企业在某个时点上的财务实力。也给大家介绍两个其中比较重要的信息。

**首先，关注负债是否合理。**
一般来说，合理的企业负债是为了更好地经营生产，提高自身竞争力。所以合理借款对企业是好选择。但要是欠钱太多还不起，就会导致企业破产。所以高负债的公司很危险。

怎么判断一个公司负债是否合理？最简单的办法是看**资产负债率**：`资产负债率 = 总负债 / 总资产`。

不同行业之间资产负债率差别比较大。但总体来说，不要超过50%，尽量不超过60%。

除此以外，还要重点关注企业整体的借款利率，也就是它的负债成本。通过关注财务报表中的“利息支出”等相关栏目，就能找到信息。企业越是能够用低成本借到钱，那这个负债就越有价值。

**其次，关注经营效率。**
这里介绍一个重要的参考指标：**资产周转率**。这个指标可以对应不同的具体资产周转率，比如应收账款周转率、存货周转率。这些周转率的主要价值是反映企业的经营效率高不高。

我们以**应收账款周转率**为例。
*   资产负债表有一个项目叫“应收账款”，是指卖出东西却没收到的那部分钱。
*   应收账款周转率的公式是：`应收账款周转率 = 营业收入 / 平均应收账款`。（营业收入来自利润表）

怎么理解这个指标？假设有一个老顾客每月来你店里吃饭，每次记100块钱账，月底再还。那么这个人一年总共消费1200元。而这个赊账让他保持着100元的应收账款？计算一下，得出应收账款周转率的结果就是 `1200元 / 100元 = 12次`。也就是说，应收账款在一年里有12次变成了现金。假如这个人一年结一次账，那么应收账款周转率就变成了1。

所以，**应收账款周转率越低，说明账期越长，应收账款收回越慢；周转率越高，则意味着账期越短，收回越快**。也就是说，这个数值越大越好。

当然要强调，这些财务比率的比较一定要在同行业内进行才有价值。另外，存货周转率也有类似作用。不过需要注意，不同行业情况不一样，选取的具体资产周转率指标就会不同。比如白酒行业比较特殊，会有大量存货，所以存货周转率这个指标就不太适合用来分析白酒行业，而更适合用前面提到的应收账款周转率。

#### 现金流量表：看公司现金健康度
现金流量表的原理非常简单，理论上它是对企业现金流入和流出的汇总。大致可以理解成是“流水账”。与利润表相比，现金流量表的水分更少，更难造假，所以更具有真实性。

为什么这么说？因为现金收付会有真金白银或银行电子记录。还是拿餐厅作为例子，假如老顾客吃完都记账不付钱，理论上这些钱也会在利润表中被记作一笔营业收入。但是，在现金流量表中就不会有任何现金流入。长此以往，如果这些钱没有全部收回来，营业收入其实就有水分。更重要的是，如果一直让别人白吃白喝收不到现金，这家店就算生意再好，也难以经营下去。

现金流量表中主要分为：**经营现金流**、**投资现金流**和**融资现金流**。我们分析个股时，最需要重点关注的就是**经营活动现金流**这个指标。它是指平时的经营活动所产生的现金流，所以最能反映企业的实际经营情况。

**首先，关注正负值。**
一家企业经营活动现金流如果大于零，说明流入现金大于流出现金，企业可支配的现金比较多。如果小于零，说明企业不但没有可支配的现金，还要靠借钱过日子，这就有点危险。

**第二，与净利润对比。**
假如经营活动现金流大于净利润，说明公司的净利润大部分变成了现金，赚到的是真金白银。反之，则像我们刚才举的例子一样，餐厅里大部分人吃饭都是赊账的，那我们就要小心了。

除了经营现金流通常正向比较好之外，投资和融资现金流的正负则很难说哪一种更好。我们需要结合企业的发展阶段、行业等具体情况具体分析。

最后提醒大家，会计报表附注中会有一些重要数据，可以有选择地翻阅。

## 总结
本节课我们一起学习了股票基本面分析中如何分析一家公司。

之前我们提到，股票代表了公司一部分的所有权。而基本面分析就是希望在好行业中选出一家好公司，在好价格时买入。也就是说，在其股价被低估时买入，以提高我们的投资胜率。

具体分析包括了定量分析和定性分析两种。我们需要建立一个管理者的视角来审视公司。一个简单的模型就是用SWOT分析法，来综合分析多方因素做一个定性分析。从财务报表中的数据，我们则可以对一个公司进行定量分析。大家要重点关注利润表、资产负债表和现金流量表这三张最重要的表格。
