# 女神养成系列：47：聪明进补，如何冬补不上火？ ❄️

在本节课中，我们将要学习如何根据个人体质，在冬季进行聪明、有效的进补，避免因不当进补而导致上火等问题。

立冬意味着冬季正式开始。春生、夏长、秋收、冬藏，四季各有不同。冬季处于封藏时期，此时服用补品补药，可以使营养物质更易于吸收，发挥更好的效果。民间有句谚语：“今年冬令进补，明年三春打虎”，意思是冬季补好了，下一年身体会很棒。但是，进补不是乱补，需要根据自己的体质来。常有患者在冬季进补后出现流鼻血等症状。这节课我们就讲一讲不同体质的人在冬季如何进补。

针对不同的体质，我会分别给出不同的食疗方案。中医将我们的体质分为9种，分别为平和型、气虚型、阳虚型、阴虚型、痰湿型、湿热型、血瘀型、气郁型、特禀型。最后一种特禀体质一般指易过敏的体质。这8种体质都是不健康的。我们往往不是单一的体质，很多时候多种体质并存。比如有的患者既是血瘀又阴虚，同时还伴有痰湿，这种情况很常见。

在这9种体质中，我们选三种最常见的体质给大家讲一讲在冬季如何进补。这三种体质分别是气虚型、阳虚型和湿热型。

在讲解这三种体质之前，我们先说一下平和型体质。这类体质是我们调理体质的终极目标。我们调理体质的目的是为了把自己调理成平和型的体质。平和型体质的人总是精力很充沛，心态也很好，很少生病。这类人在冬季进补时，多吃温补性的食物就可以，选择的范围比较大。比如像人参、党参、大枣、燕窝、牛肉、羊肉等等。这些食材偏温，对于平和体质的人来说，冬季多吃一点没有关系。这个体质因为没有特别需要改善的地方，我们就不多说了。如果你属于平和型的体质，继续保持自己的生活习惯就好了，要珍惜拥有的好体质。

接下来，我们来说第一种需要重点关注的体质：气虚体质。气虚体质的人在冬季进补很有必要。这类型的人典型的特点就是老觉得累。下班回到家，第一件事就是往床上一躺，或者往沙发上一躺，一躺就是半个小时，要不然缓不过劲儿来。还有患者说，为什么睡了一天还是感觉到累。气虚体质的人容易感觉到疲劳、气短、易出汗。除此之外，还爱感冒，更严重的会导致内脏下垂，像胃下垂、脱肛，女性子宫下垂等等。气虚的人一般脾也不强。我们在前面课程里讲到过，脾是气血生化之源。脾不断地消化吸收食物的精华，然后转化成气。所以气虚的人应该多吃一些益气健脾的食物，像人参、山药、桂圆、粳米、鳝鱼、牛肉。秋冬季节可以多吃萝卜，但是对气虚的人来说，要少吃这些耗伤气血的食物，特别是生萝卜。

人参和黄芪是公认补气最好的中药。这里给大家推荐两款食疗方。

以下是两款适合气虚体质的食疗方：
*   **人参莲子汤**：用10克的人参，10枚去芯的莲子，再加少量的冰糖放在碗中，加水没过食材，隔水蒸一个小时，然后喝汤吃莲子。
*   **黄芪桂圆鸡**：准备一只家养的小公鸡，10克的黄芪，50克的桂圆。另外，可以根据自己的口味爱好，加点香菇、冬笋等配料。小公鸡焯水去血沫，起锅烧油，放入姜末。然后将切好的鸡块、黄芪、桂圆下锅翻炒一会儿，加水，改小火慢炖。吃肉喝汤。

这两个食疗方对益气补虚很有帮助。气虚体质的人，冬季可以时不时做上一顿。

上一节我们介绍了气虚体质的进补方法，本节中我们来看看阳虚体质。上面讲到的气虚体质和接下来要讲的阳虚体质，是最适合冬季进补的两种体质。阳虚体质最大的特点是怕冷，常年手足冰凉，不敢吹空调。用中医的话说，就是身体内产生的热量不够。这类患者中，年轻女孩和老太太偏多。夏天穿着长袖和长裤，冬天裹好几层，遇到冷就会拉肚子，经常感到肚子凉、膝盖凉。这类人的舌头颜色比较淡，舌头看起来有点胖，边上也有齿痕。

现在阳虚体质的人很多，为什么呢？因为吹空调、喝冷饮、熬夜这些现代生活方式所造成的。民间有句谚语：“家备小姜，小病不慌”。姜对阳虚体质的人来说可谓佳品，可以做饭炒菜的时候多放些姜丝。在冬季可以多吃一些羊肉，像板栗、糯米等温补的食物。一定要少吃寒凉的食物。

在这里我推荐阳虚体质的人多喝当归生姜羊肉汤，多吃涮羊肉。当归生姜羊肉汤这个方子在我国中医四大经典张仲景的《金匮要略》中有记载，它有温中散寒的作用，可以补充身体的阳气，驱寒的同时还能补血，特别适合冬天使用。

以下是当归生姜羊肉汤的做法：
*   可以用20克的当归，30克的生姜，一斤羊肉。
*   将当归和生姜用清水泡软，切成一片片备用。
*   将羊肉过开水，去血水后捞出，切成一片片的。
*   然后把当归、羊肉、生姜一起放在砂锅中，大火炖烂就可以。简单方便。

阳虚体质的人可以抓住冬季这个时机，好好进补。

最后我们来讲一下湿热体质。湿热体质的人在冬季进补时稍微一不注意就会上火。比如上面说的当归生姜羊肉汤，湿热体质的人吃完会雪上加霜，会便秘、长痘。这是因为湿热体质的人本来就容易长痘，甚至是三四十岁还长青春痘。这种体质的标志之一就是满脸长痘，口舌生疮。三四十岁了还长青春痘，是真的年轻吗？当然不是，这是因为不健康，是体内有湿热，表现在皮肤上就会长痤疮。女孩子爱长在额头两边，男孩子爱长在胡须的周围。当吃了羊肉串、水煮鱼等上火的东西后，第二天满脸都是痘，还会有长脓包。甚至有的人除了脸上长，脖子上也会长，甚至连头发里边也可以摸到。

此外，湿热体质的人还长湿疹和口腔溃疡。所以这部分人群在冬季进补的时候要注意。酒是形成湿热体质的推手。湿热体质的人，首先要限制饮酒的次数。饮食应以清淡为主，适合吃冬瓜、薏仁、苦瓜、空心菜等等。而对于辛辣燥热、大热大补的食物，像生姜、牛肉、羊肉、蒜，应尽量避免食用。

在这里我们推荐一道鲤鱼冬瓜汤。
*   用一斤冬瓜，半斤的鲤鱼肉，放在锅里一起炖煮。
*   可以先将冬瓜煮10分钟，然后加鲤鱼炖5分钟，出锅撒上葱花和香油就可以食用了。

鲤鱼和冬瓜清利湿热的效果非常好。有人说我是湿热的体质，但是我控制不住吃火锅涮羊肉怎么办呢？这个时候可以喝点金银花之类的凉茶挽救一下。但大家听了这么久我的课也知道，我的宗旨就是自律，先把自己的体质调理好再说。

以上就是今天的内容。

本节课中我们一起学习了冬季进补的智慧。总结一下，冬季是气虚体质和阳虚体质的人进补最好的时机。这类人需要抓住这个进补的机会，为来年打下一个好的身体的底子。而湿热体质的人，尽量不要盲目地进补。好，今天就到这里，我们下节课见。
