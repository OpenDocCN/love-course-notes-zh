# 女神养成系列：10：心肺滋养：养出红润好气色 💖

在本节课中，我们将学习一套滋养心肺、促进血液循环的瑜伽练习，并了解一款有助于清热养颜的饮品。通过疏通手臂经络和调整呼吸，帮助身体获得充足氧气，从而改善气色，让面容更加红润年轻。

## 心肺滋养的重要性

上一节我们介绍了课程的整体目标，本节中我们来看看心肺滋养为何如此关键。心肺功能直接影响全身血液循环与氧气供应。心脏有力、气血通畅，人就会容光焕发，显得更年轻。现代人常因气血淤塞、三焦不畅导致血管问题。因此，疏通手臂的阴阳两面经络至关重要，它能提升肺活量，确保身体获得充足氧气。

## 手臂与手腕活化练习

以下是激活手臂、疏通经络的初步练习。这些动作能伸展心经、心包经和肺经，影响全身脉络。

1.  **手指伸展与握拳**：坐于椅上，双脚轻轻打开。双手向前平举，指尖尽力张开。呼气时，握紧实拳，向内卷到极限，然后向前伸直。感受手臂完全伸展。保持深呼吸。
2.  **手指逐次开合**：保持手臂伸直。将手掌心朝上，然后一根手指接一根手指地向外打开。接着，手指向下撑到最远。深呼吸，掌根向外推。最后，再一根手指接一根手指地向内收回。
3.  **手腕活动**：双手十指交扣，活动手腕。这个动作能缓解因使用鼠标、抱孩子等造成的手腕过度用力。
4.  **手指拉伸**：一只手手指伸直，用另一只手将其大拇指一同包住，轻轻向上然后向后拉伸。深呼吸。换手重复。
5.  **拳头拉伸**：握紧拳头，向前伸，然后轻轻向下拉，再带回。这个动作能伸展手臂内侧多条经络。

## 肩背与手臂综合伸展

完成基础活化后，我们进一步伸展肩背和整个手臂筋膜。

1.  **交叉手前伸**：双手相对，十指交叉翻转相握。从内侧像掏出礼物一样，将手臂向前伸出（伸直与否均可），在极限处保持深呼吸，然后慢慢收回。
2.  **鹰式手臂（简易版）**：
    *   右手在上，左手在下，双臂缠绕。如果手掌无法相对，则手背相对即可。
    *   双手稍微向上抬，感受肩胛骨和背部的伸展。
    *   吸气，眼看上方。呼气时，身体向前倾，低头，进一步打开背部。
    *   吸气回正，手放下解开。换边重复（左手在上，右手在下）。
3.  **手臂画圆**：站立，右手放于体侧，左手上举。呼气时，左臂向前画一个大圆落下。吸气时，右臂上举，重复画圆。左右交替进行数次。
4.  **手臂筋膜伸展**：站立，双臂侧平举。想象用小指引领手臂向上旋转，再用大拇指引领向下旋转。配合呼吸重复多次，感受手背筋膜和肌肉的伸展。

## 结合球椅的强化练习（可用小球或徒手）

上一节我们活动了上肢，本节中我们加入球椅（或徒手/持小球）来强化训练，并加入下肢动作。

1.  **持球深蹲**：双手将球压紧于胸前。脚尖微朝外。吸气，屈膝下蹲（膝盖朝向脚尖，尾骨内收，臀部不后翘）。呼气，臀部夹紧站起。重复。
2.  **持球侧向移动**：在深蹲最低位，将重心移到左脚，球也移向左侧。吸气，双手持球上举；呼气，将球水平推向右侧。配合呼吸左右移动。
3.  **持球体前屈**：将球向前推置于地面，双脚伸直，感受背部和大腿后侧的伸展。深呼吸。双手扶球，向左右两侧移动，加深侧腰伸展。
4.  **坐姿侧伸展**：坐于球上，将球带到身体右侧，臀部向左移动，感受左臀到右手指尖的拉长。右手将球推远，眼看上方，深呼吸。呼气收回。换边重复。
5.  **骨盆画圆与扭转**：坐于球上，顺时针轻轻转动骨盆，灵活脊柱。然后，将骨盆移向右侧，左手扶球，右臂向上延伸拉伸侧腰。回到中间，换边重复。

## 深度伸展与放松

经过一系列练习，现在进行更深度的伸展来放松身体。

1.  **坐姿扭转**（以右侧为例）：坐姿，左脚伸直。右手从外侧扶住左膝外侧，身体慢慢向左扭转向下。如果可以，用左手抓住右脚踝，右臂向上伸展，眼看上方。保持呼吸后慢慢收回。换边重复。
2.  **抱肩式**：坐直，双手环抱自己，左手在上抓右肩，右手在下抓左肩。吸气打开肩胛骨，呼气低头拱背。吸气回正。换右手在上重复。
3.  **收束**：双手打开，吸气上举，在头顶合十。呼气时，双手带回胸前。调整呼吸，将脚尖脚跟收回。

## 养颜膳食：雪梨蜂蜜苦瓜汁 🍐

练习结束后，内在滋养同样重要。许多人因心火燥热导致长痘或脾气暴躁。以下是一款清热养颜的饮品制作方法。

**材料**：雪梨、蜂蜜、苦瓜。
**功效简述**：
*   **雪梨**：滋润、化痰、辅助降血压。
*   **蜂蜜**：润肠通便。
*   **苦瓜**：滋肝明目，清热降火。

**注意事项**：此饮品性质偏寒凉，建议在中午或下午饮用，避免晚上喝。

**制作步骤**：
以下是制作雪梨蜂蜜苦瓜汁的简单步骤。
1.  准备适量雪梨、苦瓜和蜂蜜。
2.  将雪梨和苦瓜洗净切块。
3.  将切好的雪梨和苦瓜放入榨汁机中榨汁。
4.  根据个人口味，在汁液中加入适量蜂蜜搅拌均匀即可。
