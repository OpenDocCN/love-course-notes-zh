# 女神养成系列：012：第十一课：舒缓情绪术

## 概述
在本节课中，我们将学习一套专门为产后妈妈设计的瑜伽练习。这套练习旨在帮助缓解紧张、焦虑的情绪，提升精力，并应对因角色转变带来的身心挑战。我们将使用一些常见的家居物品作为辅助工具，通过六个舒缓的动作来达到安抚情绪、恢复精力的效果。

## 课程准备
在开始练习之前，我们需要准备一些辅助工具。以下是所需的物品列表：

*   **带靠背的椅子**：一把普通的带靠背椅子即可。
*   **抱枕或枕头**：家里常用的枕头即可。
*   **瑜伽砖**：标准瑜伽砖即可。若无，可用厚字典或硬纸盒代替。
*   **毛毯**：一条普通的毛毯。
*   **圆形靠枕（可选）**：一个较厚的圆形靠枕。如果没有，可以用额外的枕头或折叠的毛毯代替。

准备好这些工具后，我们就可以开始今天的练习了。

## 动作详解
上一节我们准备好了辅助工具，本节中我们来看看具体的练习动作。每个动作都旨在通过不同的方式舒缓身心。

### 动作一：辅具辅助下犬式
下犬式可以按摩横膈膜，有助于稳定大脑、消除敏感脆弱的情绪。使用椅子辅助能让初学者更舒适地找到伸展感。

首先，调整站姿。双脚分开约一个拳头的距离，与髋部同宽。脚趾用力张开再下压，感受大脚趾球、小脚趾球和脚跟三点稳稳压向地面。保持骨盆中立，避免过度翘臀或塌腰，肩膀自然展开，肋骨内收。

面向椅子站立，双手扶住椅面。吸气时延展脊柱，呼气时臀部向后向上推，进入下犬式。确保双脚在臀部正下方，脚跟尽量压地。双肩远离耳朵，不要耸肩。在此保持3到5次均匀呼吸。

**核心要点**：重心稳定，脊柱延展。
**公式/代码描述**：`姿势 = 双脚与髋同宽 + 三点压地 + 脊柱延展 + 肩下沉`

完成此动作后，缓慢屈膝，双手扶髋，慢慢起身回到站立。

### 动作二：瑜伽砖辅助加强下犬式
这个动作是下犬式的变体，能进一步对抗心悸、缓解头痛。我们使用瑜伽砖和靠枕来辅助完成。

将两块瑜伽砖以与椅子腿相同的间距放好，后方放置准备好的厚支撑物（靠枕或毛毯）。双手虎口用力压实在瑜伽砖上，如果困难，可将食指指向正前方。脚尖回勾，臀部向上推，脊柱延展。先踮起脚尖，然后尝试让脚跟落地，同时将额头轻轻靠在支撑物上。保持肩部向后推，远离耳朵，肋骨内收。在此保持3到5次均匀呼吸。

完成练习后，吸气抬头，屈膝跪地，然后坐下。

### 动作三：辅具辅助肩倒立式
肩倒立式有助于增强记忆力，缓解因记忆力减退带来的焦虑。我们将使用椅子和靠枕进行安全辅助。

面对椅子坐下，将靠枕放在椅子前方。然后，臀部坐在靠枕上，双腿依次抬起放到椅面上。头部微微后仰，让下巴微扬，为颈椎创造空间。双肩下沉，双手可以抓住椅子腿以保持稳定。注意靠枕是垫在臀部下方，帮助骨盆微微后倾，而非垫在腰部。在此放松，保持5到8次均匀呼吸。

练习结束后，缓慢而有控制地放下身体。

### 动作四：仰卧束角式
这个动作能放松全身，滋养腹部脏器，改善骨盆区域血液循环。

坐立，双脚脚心相对。将瑜伽砖垫在双膝下方以获得支撑。身体缓慢向后躺下，双手放在身体两侧，掌心向上。确保双肩沉向地面，腰部不要过度拱起。如果膝盖无法碰到瑜伽砖，可以调整砖的高度。在此完全放松，保持5到8次均匀呼吸。

### 动作五：支撑婴儿式
这个舒缓的练习可以按摩心脏，放松下背部，缓解不适。

跪立，双膝向两侧打开，双脚并拢。如果臀部无法坐到脚跟上，可以在脚后跟处垫上折叠的毛毯。首先，微微收紧腹部（感受腹横肌发力），然后骨盆前倾（微微翘臀），身体缓慢向前俯卧。可以将上半身趴在椅子上放松。

如果想加深练习，可以在前方放一个高支撑物（如枕头），让额头枕在上面，进一步伸展脊柱。保持双肩远离耳朵，臀部尽量向后坐向脚跟。在此保持5到8次均匀呼吸，感受情绪的安抚。

完成后，缓慢用手撑起身体，回到跪姿。

### 动作六：坐立前屈（辅具辅助）
坐立前屈能拉伸腿部后侧，但需要注意避免腰部代偿。使用椅子辅助可以帮助我们找到正确的发力模式。

坐立，用手将臀部肌肉向后拨开，让坐骨前端扎实坐地。如果感觉困难，可以面向椅子，双脚蹬住椅子下方的横杠，双手扶住椅子腿。吸气时延展脊柱，呼气时从骨盆处开始前倾（非腰部弯曲），让胸腔向前向上提。在能保持脊柱延展的位置停留，感受腿后侧的拉伸。保持3到5次均匀呼吸。

练习结束后，缓慢起身，收回双腿，选择舒适的盘坐姿势。

## 总结与建议
本节课中，我们一起学习了六个舒缓的瑜伽动作，包括下犬式、肩倒立、束角式、婴儿式和坐立前屈。这些练习结合了前屈、后仰和仰卧，能全面帮助放松身心、提升精力。

建议各位妈妈可以在每晚睡前、身体疲惫时或午休时练习这些动作。每个动作都可以根据自身情况保持更长时间（如5到8分钟）。坚持练习，不仅能改善身体状态，更能从内而外地调整情绪，以更好的精力应对妈妈的新角色。
