# 舒化课程：11：话术复习与核心技巧回顾 🗣️

在本节课中，我们将系统回顾舒化基础课第一晚学习的关键话术与沟通技巧。这些技巧旨在帮助你在社交互动中建立框架、引导对话并避免暴露需求感。

## 课程内容回顾

上一节我们介绍了课程的整体框架，本节中我们来详细回顾第一晚学习的七个核心话术技巧。

以下是第一晚学习的核心话术要点列表：

1.  **使用“稍等”**
    *   当不知如何回复或需要对方等待时，使用“稍等”而非其他词语。这是一个基础的缓冲话术。

2.  **进行服从性测试**
    *   当对方询问“在干什么”而你不知如何开启话题时，可以使用测试对方服从性的话术来破冰。
    *   **示例话术**：“你亲我一下，我就告诉你。”

3.  **补救说错的话**
    *   如果发言失误，可以使用“321”方法进行补救，将被动转化为主动，让对方重新进入你的对话框架。
    *   **核心要点**：此方法仅能使用一到两次，频繁使用会显得笨拙。

4.  **进行示弱性打压**
    *   当对方过度展示优越感时，可采用示弱的方式进行温和打压，以平衡双方姿态。
    *   **示例话术**：“我害怕你成为我的男神”、“我害怕你不要爱上我了哟”。
    *   目的是打压对方过度的自尊，表明自身框架不易被突破。

5.  **铺垫高价值形象**
    *   通过朋友圈或日常聊天，自然地展示自身价值，为后续发展做铺垫。
    *   **操作方法**：在聊天中偶然提及生活细节，例如“我在做面膜”、“我在涂口红”。长期坚持，让对方逐渐了解你的生活品质。
    *   **核心原则**：高价值形象需要长期铺垫，无法一蹴而就。

6.  **避免套路重复**
    *   初期交流时，避免连续使用单一或多种套路。
    *   **错误示例**：对方每个问题都用“你亲我一下/给我发个红包，我就告诉你”来回应。
    *   这样会导致对方厌烦，效果适得其反。

7.  **敢于试探并避免低级提问**
    *   **敢于试探**：不要害怕改变聊天风格（如尝试撒娇）。通过试探逐步了解对方性格。
    *   **避免低级提问**：永远不要直接询问“你想我了吗？”或“你对我第一印象怎么样？”。这类问题会彻底暴露你的需求感。
    *   **错误话术分析**：“想知道我对你的第一印象吗？”这句话的潜台词是“我迫切希望你知道我对你有好感”，需求感暴露程度为百分之百。

## 核心技巧深度解析

上一部分我们回顾了基本要点，现在我们来深入分析两个关键技巧：需求感补救与舒适聊天标准。

### 需求感暴露的紧急补救

当不慎发出暴露需求感的信息（如询问第一印象），而对方尚未回复时，可立即采取补救措施。

**补救方法**：迅速追加发送“321，我不告诉你了”。

**作用原理**：将原本严肃的提问转化为一个玩笑，扭转被动局面，让对方认为你是在调侃而非认真询问，从而拉回自己的姿态平衡。

### 何为“舒适的聊天”

判断一次聊天是否舒适，有一个简单的标准。

**舒适度标准**：当你回顾整个聊天记录时，内心感到淡淡的甜蜜，并且是微笑着看完的。这种体验是聊天舒服度的直观体现。

## 课程总结

本节课中，我们一起复习了舒化基础课第一晚的核心内容。我们学习了包括使用“稍等”、进行服从性测试、用“321”法补救失言、进行示弱性打压、长期铺垫高价值形象、避免套路重复、敢于试探以及绝不能询问“第一印象”等暴露需求感的问题在内的七大技巧。掌握这些技巧，有助于你在社交互动中更好地把握节奏，建立吸引。
