# 恋爱课堂：02：洞察人性

在本节课中，我们将探讨人性的核心驱动力，特别是“自我认知”与“欲望”如何深刻影响我们的行为、情绪和人际关系。我们将从“虚假自我”与“真实自我”的投射开始，逐步深入到对“存在感”和“性欲”这两种基本人性追求的剖析。

## 虚假自我与真实自我

上一节我们介绍了人性的一些基本概念，本节中我们来看看“自我”的构成。一个人专注于某项事物，这属于自我的一种投射。这种投射可能是虚假自我的投射，也可能是真实自我的投射。我们需要明确知道这一点。

在现实生活当中，你是依靠什么而存在的？你依靠什么来存在？虚假自我存在于在意他人的负面评价中，而且这个负面评价和你虚假自我当中的自我认知是截然相反的。

一旦你的自我认知（虚假的自我认知）和他人给你的评价相反，就相当于在你的精神上一刀又一刀地凌迟，所以你会特别痛苦。这种痛苦会让你想要去自我保护，然后你会去反驳、争执、吵闹。这其实是你对自己的一种捍卫。

所以你就知道，当别人说你不好的时候，你为什么跟别人吵、跟别人闹。而当现实情况是，一个人来说你不好时，最正确的做法是：先想想你是不是那样的人。然后确认你到底是一个什么样的人。最后再想想，向他证明你是这样的人有没有意义。

例如，你的人生有一件很大的事情要做，你有一个很明确的目标。那你想想做这件事情有没有意义。

以下是关于真实自我的定义：
*   **真实自我**是自己给自己的定义。
*   **真实自我**其实就是一个人的人生观。

人生观是什么？就是你到底要做一个什么样的人。你要做一个依靠内在而存在的人，例如勇敢、聪明、智慧、踏实、靠谱、上进、友爱等品质。它不是指你要挣多少钱，或者要多么飞黄腾达、成为名人。它关乎你在关键时刻能够坚持的品质。

## 人生观与价值观

那么，价值观是什么？价值观是能够为他人提供价值、能够成就他人。人生观和价值观，哪个在前？

**价值观决定人生观。** 价值观明确后，做那样的事情需要一个什么样的人？你就要成为那样的人，才能够去做那样的事。**做事大于做人**。因为你想要去做一件事，你就势必要成为做成那件事所需要的人。

例如，一个自私自利、只顾利益的人，他能开好饭店吗？开不好。对于一个自身没有强大资源可以依附的人，想要通过自己创业或奋斗成功，任何大企业永远坚信的一个事物都是：**为他人提供价值至上**。哪怕他们实际上做的并非如此，但他们的口号都是这样喊的。

例如，你想开一家面馆并做好它。唯一的方法就是把面做好吃。你的面做得越好吃，生意就会越好。只要你的面是真的好吃，客人自然就会多。如果你说你的面很好吃但没有客人，那可能你的面根本不好吃，或者那个地方的人不喜欢吃面。

所以，任何餐饮大品牌，一旦开连锁加盟店，其品质就容易下滑。因为加盟商是为了赚钱，最好的手段就是缩减成本，从而把品牌搞砸。除了肯德基和麦当劳，很少有餐饮品牌加盟后越做越火爆，大多数都做着做着就倒闭了。因为他们大多数人没有坚持去提供价值。

## 自我实现与他者意志

所以，做人就是真实的自己，就是你的人生观。你自己到底是一个什么样的人，心里要有数。而你要做什么事，那是你自己的事。

我是一个有高强度意识形态的人，我把我的意识形态和想法传递给你们，你们就受我影响。但你们最终应该成为的是你们自己，而不是成为下一个我。如果你们成为了我，那么你们所实现的目标都是我的目的，你们相当于在帮我做事，而不是帮自己。

这就解释了为什么每个大企业都有企业文化。企业文化其实都是企业家、管理层、股东层的意愿。他们需要不停地向大家灌输，把自己的意愿传递给别人。例如，我把我的理想，变成了你们的理想，那你们所有人最终实现的理想是谁的理想？是我的理想。

这很有意思，统治者不都干这种事情吗？实现一个“中国梦”。那其实就是他自己的梦。当然，这也没什么不好。生活在中国挺好的。如果你家庭条件一般，想要发家致富，千万不要听信那些谎言去国外。你老老实实待在中国发家致富吧。因为在中国发财特别简单。

你看那些抖音上天天发鸡汤的人，抄来抄去，一样可以赚钱。就这么简单。你买一本五几年、六几年的心灵鸡汤书，把里面的话抄下来，做成音频节目或配上图片发出去，你也可以火。因为现在的人就是这样。

## 对存在感的追求

以上是关于虚假自我和真实自我的探讨。接下来我们看看人的第一个追求：**追求开心，追求快乐**。这靠什么满足？靠**存在感**来满足自己的开心和快乐。

这是对于情绪的追求，是人大多数行为动力的核心源泉。例如，你们经常在晚上看到很多人喝了酒坐在大马路上吃烧烤，不停地讲自己多厉害。他们就是在找存在感。

那么，追求存在感，是属于主观世界还是客观世界的追求？存在感是属于**客观世界**的追求。客观世界的规则是根本无法改变的。获取他人的关注其实就是一种**虚荣心**。

满足一个人的虚荣心为什么杀伤力极强？因为满足虚荣心让他获得了大量的**关注度**，关注度让他感觉极其有存在感。

例如，很少有女人能扛得住和一个男生去饭局，一进去就被叫“嫂子”，一堆小弟倒茶递烟、端板凳夹菜。更不用说如果她的男朋友在演唱会上把她请上台当面表白，那种关注度太强了，几乎没人扛得住。

为什么有些人喜欢进豪车？因为跨进豪车的那一刻，他就知道很多人在看他，关注度极其强大。别人都在关注他、尊重他、欣赏他、仰慕他。

你以为你喜欢钱吗？钱是你的人生追求吗？错了，钱不是你的人生追求。**被大家关注、尊重、欣赏、喜欢，让你有存在感，这才是你的人生追求**。

这就是对于存在感的追求，俗称“刷存在感”。但你要知道，**找到你可以刷存在感的方向，比你去盲目刷存在感重要太多**。

## 对性欲的认知与压抑

除了获取他人关注能使人开心快乐，还有什么能让你感觉很爽？除了钱，除了别人关注你，还有什么？是**性**。

性欲是人所有欲望当中最为**肤浅**的，也是最本能的。食色性也，吃饭和性是人的本性。它不罪恶，也不可耻。

但为什么在中国的环境（乃至许多社会）中，性变成了难以启齿、被视为羞耻的事情？这和**当权者**、**统治阶层**有很大关系。

性欲是客观世界的规则，是外在的、人无法改变的。如果人能控制自己的性欲，那就太厉害了。

性欲会使人体产生荷尔蒙和神经递质，例如**多巴胺**。多巴胺会使一个人对另一个人产生极强的控制欲和占有欲。那么，一个当权者对他喜欢的女人产生了极强的控制欲和占有欲后，他允许这个女人跟别的男人睡吗？不能。

于是，当权者就用各种规则（如爱情、婚姻）来制约他人，甚至**否认性欲的存在，妖魔化性欲**。婚姻的本质就是男人用来制约女人的工具，其作用包括保证血统纯正性、保证女人不逃跑、保证自己继承权。

**否认性欲，就是否认人类最初始、最本质的欲望。** 一个正常人一定会有性欲。如果你认为自己没有性欲，要么有病需要看医生，要么就是认知出现了偏差。

## 性欲不满足的表现与影响

一个18岁的、保守的姑娘，可能特别喜欢男生陪她聊天。为什么？因为她对自己的性欲认知不清晰，认为自己没有性欲。但实际上她有，性欲得不到满足，她就会不舒服、焦虑、心慌意乱，需要慰藉。她会找异性聊天、陪伴，这本身就是一种精神上的性欲满足。

为什么不愿意承认自己有性欲？因为在你的认知体系中，性欲是一件羞耻和恶心的事情。如果有一天迈过这个坎，你会发现世界如此简单。

## “能”与“不能”的补偿机制

关于性欲，我们需要认知两个观念：**“能”** 与 **“不能”**。

如果一个人的性欲能得到满足，他会显得特别宽容、随和、谦和。如果一个人的性欲不能得到满足，他就会在其他方面表现自己特别“能”。

例如，异地恋的双方，性欲不能满足，就会不停地在情感方面试图控制对方，要求对方按自己的规范做事。这就是“不能”在“能”上的补偿。

如果你的性欲、经济条件、存在感都能得到极大的满足，你还需要苦苦追求一段感情来获取慰藉吗？可能就不那么需要了。

一个人如果某方面的欲望得不到满足，他就一定需要在另外一个方面证明自己无所不能。反之，如果某些欲望能得到满足，他在其他方面就会显得特别随和。这就像大坝决堤，一个口子漏水，你堵住它，水就会从其他地方冒出来。

所以，如果你在情感方面否认自己的性欲，认为自己没有性欲，导致性欲得不到满足，你谈恋爱一定会很“作”。你会牢牢抓住存在感，对方今天不关注你，你就生气、冷嘲热讽、拉黑删除，看到对方求你就感觉很爽。

## 总结

本节课中，我们一起学习了人性的几个核心层面：

1.  **自我认知**：区分了**虚假自我**（依赖他人评价）和**真实自我**（内在的人生观），并指出痛苦源于两者冲突。捍卫真实自我需要内省而非外争。
2.  **价值观驱动**：明确了**价值观**（为他人提供价值）决定**人生观**（成为什么样的人）。成功的事业往往源于价值提供，而非单纯逐利。
3.  **存在感追求**：揭示了快乐的核心驱动力之一是**存在感**，表现为对**关注度**和**虚荣心**的满足。真正的方向比盲目“刷存在感”更重要。
4.  **性欲的本质与压抑**：承认**性欲**是人类本能、客观的欲望。其在社会中被妖魔化和压抑，主要源于统治阶层为控制资源而制定的规则。
5.  **欲望的补偿机制**：提出了 **“能”与“不能”** 的模型。当基本欲望（如性欲）**不能**满足时，人会在其他领域表现**能**（如控制欲）。全面满足欲望有助于人格的平和。

理解这些概念，有助于我们更清晰地认知自己行为背后的动机，在人际关系中减少不必要的内耗和冲突，从而更坦诚地面对自己的需求，建立更健康、更真实的生活与情感模式。
