# 爵士舞零基础教程：02：头部基础练习

在本节课中，我们将学习爵士舞中头部的基础动作。头部动作是舞蹈表现力的重要组成部分，掌握正确的发力方式和动作轨迹至关重要。我们将从四个基本方向开始，逐步学习平移、画圆、甩头等技巧。

## 🧭 头部四个基本方向

上一节我们介绍了课程概述，本节中我们来看看头部最基础的四个方向动作：前、后、左、右。

核心动作路径是：**中 → 左 → 中 → 后 → 中 → 右 → 中**。

以下是练习步骤：
1.  向前低头。
2.  回到正中位置。
3.  向左倒头。
4.  回到正中位置。
5.  向后仰头。
6.  回到正中位置。
7.  向右倒头。
8.  回到正中位置。

**关键点**：回到正中位置时要求有爆发力，而向前、后、左、右方向运动时则不需要爆发力。

反方向练习路径为：**中 → 右 → 中 → 后 → 中 → 左 → 中**。

将以上动作连贯起来，即可完成头部的绕圈动作。

## 💨 如何甩动头发

在跳舞转圈时，若想漂亮地甩起头发，需要掌握正确的发力方法。

核心要领是：**头部转动要快，并利用头顶的力量在最后一下爆发性地甩回来**。

练习方法如下：
1.  头部快速转向一侧。
2.  在动作终点，用头顶的力量爆发性地将头带回原位。
3.  注意头部要有向下的蓄力过程，再向上爆发。

## ↔️ 头部左右动作

接下来，我们学习头部的左右动作。身体需保持直立并收腹。

动作分解如下：
1.  头部倒向你的左肩方向，目光看向左方。
2.  头部回到正中。
3.  头部倒向你的右肩方向，目光看向右方。
4.  爆发用力，有控制地回到正中位置。

## 📏 头部平移动作

头部平移是一个相对较难的动作。我们先从准备姿势开始：双腿打开，双臂在身体两侧架起。

核心概念是：**用你的左耳和右耳分别去找同侧手的方向，头部在水平面上向两侧平移**。

**关键点**：
*   脖子向两侧平移，但头部不要倾倒。
*   头顶方向始终保持向上。
*   肩部保持稳定不动。

初学者练习方法：
1.  背靠墙角，用墙角卡住双肩，强制肩部固定。
2.  在此姿势下，练习头部向左右的纯粹平移。
3.  掌握后，离开墙角，双手架起，朝着手的方向做平移练习。

这个练习主要**提升脖子两侧的灵活性**。

## ↕️ 头部前后动作

现在，我们来看头部的前后动作。身体同样保持直立不动。

**向前动作**：
1.  下巴引领，头部尽量向前伸。
2.  身体不要跟随前倾。

**向后动作**：
1.  头部尽量向后靠，下巴微收。
2.  身体不要跟随后仰。

将前后动作连贯起来：前 → 后 → 前 → 后，身体保持稳定。

## 🔄 头部画圆与画8字

将前、左、后、右四个点连接起来，即可完成头部画圆动作。顺序为：前 → 左 → 后 → 右 → 前。

反方向顺序为：前 → 右 → 后 → 左 → 前。

**头部画8字**是一个进阶动作，用你的下巴尖在空中描绘一个平面的数字“8”。
1.  下巴向左下方移动。
2.  向右上方抬起。
3.  向右下方移动。
4.  向左上方抬起。
头部跟随下巴的运动轨迹移动。

## 💃 安全且漂亮的甩头技巧

许多人在甩头时，头发容易糊在脸上，这是因为发力部位不对。

正确方法：
*   **核心发力点**：利用**胸腰**和**旁腰**的力量将头带起，而不是只用脖子的蛮力。
*   **头顶力量**：在甩起来的最后瞬间，头顶要用力上顶，这样头发才能向后飞扬。

错误示范：如果仅用脖子发力且头顶不用力，头发就会垂在面前。

**侧面甩头**：
1.  上半身向旁侧下腰。
2.  利用旁腰和头顶的力量将头甩起。

**前方甩头**：
1.  胸腰向前下弯。
2.  起来时，用头顶用力和胸腰带动，将头抬起。

**重要原则**：务必使用胸腰和旁腰发力，避免单独使用脖子发力，以防受伤。

## 💎 总结

本节课中我们一起学习了爵士舞头部的基础练习。主要内容包括：
1.  头部的四个基本方向（前、后、左、右）及其爆发力要点。
2.  头部的平移动作及借助墙角的练习方法。
3.  头部画圆和画8字的轨迹。
4.  **安全甩头的核心技巧：用胸腰和旁腰带动，配合头顶爆发力，避免只用脖子蛮力。**

头部平移动作较难，请按照教程所述，在墙角固定双肩进行练习，熟练后再进行徒手练习。所有头部动作都需要大家课后**多加练习**，才能熟练掌握。
