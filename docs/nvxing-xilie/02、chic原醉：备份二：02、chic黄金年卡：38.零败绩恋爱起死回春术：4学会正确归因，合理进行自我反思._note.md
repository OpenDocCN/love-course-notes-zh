# 恋爱挽回课程：02：学会正确归因与自我反思

## 概述
在本节课中，我们将学习如何为分手进行正确的归因，并在此基础上进行合理的自我反思。这是挽回过程中重塑对方认知、重建吸引力的关键一步。

## 课程内容

上节课我们介绍了冷冻期的操作技巧。接下来，在为期半个月的冷冻期内，我们并非无所事事，而是需要开始为建立新的联系做准备。

在这段时间里，我们需要持续向对方传递正向信息，以重塑他对我们的认知。如果在操作中遇到任何问题，可以添加老师的微信号进行反馈或提问。

在上次课程结束后，许多同学向老师提问：如何才能最大概率地挽回？大家都希望在与课程同步操作时，能有一个明确的挽回方向。在开始本节课之前，我们先来解答这个问题。

一直关注课程的同学应该听过，**吸引是一个从零到一的过程**。从未知到了解，只需要我们做足吸引，为对方建立一个对你的正向认知，就很容易让对方产生爱慕之情。

而**恋爱的过程是从一到多的过程**。你们会越来越了解彼此，认知也随之提升，同时也会产生矛盾。但在磨合之后，感情会得到进一步升华。

然而，**分手却是一个从多到负的过程**。也就是说，我了解你越多，分手后我对你的负面认知也就越强。如果我对你有5分的了解，分手后我可能也会对你有5分的负面印象。由于分手的关系，你好的一面会被对方忽视，而你坏的一面则会扩大化地存在于对方的脑海中。

这时，男生便会产生反抗情绪。即使他之前对你很温柔，分手后也可能异常冷漠，拒绝任何沟通。同时，为了避免再次进入一段错误的感情，他也会拒绝你的挽回。

因此，我们挽回要做的事情，首先是改变对方对你的负面认知，让他意识到你与他所想的不同。这样才能引发男生的自我反思，并对你重新建立认知。如果新建立的认知是正向的，就很容易重拾对你的好感，达到挽回效果。

**所以，挽回的过程应该是：先推翻对方对你的负面印象（归零），然后再重新吸引（建立好感）。** 我们上节课所讲的技巧，正是基于这个“推翻认知”的原则。当认知被推翻后，我们要做的就是重新建立印象。

在我们接下来的操作和相处中，要做的同样是“推翻”与“重塑”。只有印象重塑成功，再加上一点推波助澜，才有可能真正达到挽回效果。

因此，我们必须知道分手的正确原因是什么，才能更好地了解对方对你的负面认知是什么，从而实现真正的印象重塑。老师无法为每个人单独归因，所以现在教大家正确的归因方法，帮助大家找到挽回的方向与思路。

许多女生认为，分手原因就是分手时对方给出的理由。事实并非如此。分手时的理由往往只是感情破裂的爆发点。而矛盾激化的真正原因，大多是**长期情绪积压**的结果，最终导致了分手。

所有矛盾的源头，其实可以归结为两点：**无知**与**恐惧**。

以下是关于这两个核心概念的详细解释：

*   **无知**：指在生活中并不真正了解对方，非常自我，坚持主观意识。不考虑对方的感受，坚持以自己认为正确的方式与对方沟通。最后问题不仅得不到解决，反而会激化。男生为了平复你的情绪选择忍让，看似问题解决了，但实际上在他心里留下了深刻的负面印象。
*   **恐惧**：指拒绝面对现实，缺乏安全感。因为害怕失去、害怕产生矛盾，所以在问题出现时一味选择逃避，不去解决。问题就会不断积压，最终导致对方情感崩溃。

以上两点是引发冲突的主要原因。你可以根据你们的日常生活，将自己归入相应的类别。

当这些问题累积到一定的爆发点时，彼此可能会迎来**争吵**、**冷战**，还有一种情况是**内疚**。

在这里，内疚这一点需要着重说明。争吵基本是因为“无知”而形成的矛盾升级；冷战基本是男生开始重新审视这段关系，或刻意回避矛盾时的表现，这两点都比较好理解。

而**内疚**，是因为你一味的逃避、不去解决问题、不面对现实，而让对方产生的一种自我负罪感。这就像是你很喜欢一个人，但在相处过程中，对方一直处于忍让、默不作声的角色。这段关系在你眼中也会变得怪异，你时常会有一种负罪感：是不是因为自己而让对方变得不快乐？是不是因为自己而让对方陷入了一个被动的环境当中？这时，我们可能就会克制自己的感情，甚至因为这种负罪感而疏远对方。

感情上也是同样如此。这就不难理解，为什么有些男生连分手理由都说不出来，明明很喜欢你，印象里也都是对你的正面认知，却又偏偏坚持分手。

**所以，分手的基本逻辑是**：当你们在一起后，面临的第一个问题就是开始暴露问题。当问题暴露到一定程度时，开始觉得无法容忍，继而产生冲突。当你们产生冲突后，会看到彼此最不可理喻的一面，这时幻想被打破、理想破灭。你们会开始越来越多地关注彼此的不足之处，最后形成感情破裂。

那我们最行之有效的挽回方式，就是**寻找根源问题**，也就是造成“理想破灭”的点在哪里，再加以改进。等到和他再次联系、出现在对方生活中时，展现出与之相反的正面形象，就可以很好地把他带回到曾经的回忆状态，开始回忆你曾带给他的理想状态，形成感情回溯，最终重新拾起对这段感情的渴望与兴趣。

以下是针对不同归因结果的行动建议：

*   如果你的归因是自己太任性幼稚，让对方难以忍受，那就要在接下来的相处中展现出一定的稳重感，并且让自己的思维更有条理。
*   如果是因为你的自我，让对方感到压抑，那你就要表现出你会站在其他立场思考问题，不是以自我为中心的形象。
*   然而，如果你是因为逃避而造成的分手，那你要做的就是把对方约出来，开诚布公地谈一谈。

这些改变后的表现，你可以展现在平时传递给他的内容里（如社交动态），也可以展现给你身边的人，通过他们传达给他。我们要基本做到向外传递的形象与实际吻合。这样，男生在和你接下来的相处中，才不会因为感觉太过唐突而选择退缩或拒绝交流。
