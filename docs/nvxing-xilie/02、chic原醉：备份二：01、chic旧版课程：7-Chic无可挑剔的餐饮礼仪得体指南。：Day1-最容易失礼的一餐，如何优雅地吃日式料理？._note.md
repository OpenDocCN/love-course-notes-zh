# 餐饮礼仪得体指南：02：优雅享用日式料理

在本节课中，我们将学习如何优雅地享用日式料理。课程将涵盖常见的礼仪错误、日料店的通用规则以及寿司的特定用餐礼仪，帮助你避免失礼，展现得体与尊重。

## 理解礼仪的本质

上一段我们提到了课程主题，本节中我们来看看礼仪的核心。日常饮食中的礼仪本质，绝非虚假客套的代名词。礼仪是能让餐厅里所有人都感到舒适的言谈举止。厨师会因此觉得你值得被更好对待，你也能真正品尝出食物的美味。

有人认为礼仪无非是保持安静、动作轻慢。这些确实是基本原则，但仅能让人看着不粗鲁。要给人留下有家教、得体优雅的印象，其间还有很大距离。

就像日式料理，很多人都吃过，却少有人了解其具体的饮食礼仪。你一定有过类似经历。

## 大多数人常犯的礼仪错误

以下是人们在日料店常犯且容易忽视的礼仪错误。

1.  **错误混合蘸料**：吃寿司时，将酱油和芥末在同一个碟子里混合搅拌，然后用底部的寿司米蘸取混合物食用。这里犯了两个错误：混合蘸料会破坏寿司原本的味道层次；用寿司米蘸取会使饭团失去应有温度甚至散开。**正确做法**：将酱油单独倒入碟中，用寿司的上层食材部分蘸取。芥末加或不加皆可，因为寿司师傅通常已在制作时放入适量芥末。如果你是重口味爱好者，可以再抹一点于食材之上。高级寿司一般无需额外蘸取芥末和酱油，因为师傅已做好调味。

2.  **不当就餐姿势**：日料店的吧台或桌子通常较低，很多人习惯弓着背，几乎把脸趴在桌面上吃东西。这被称为“猫背犬食”，意指只有猫狗才会这样不得体地进食。**正确做法**：吃饭时应直起后背，将碗碟端起来慢慢享用。但吃拉面时不必端碗，应用筷子夹起拉面放入勺子，再将勺子送到嘴边食用。

3.  **用筷子扎取食物**：遇到难以夹起的圆滑食物时，许多人下意识会用筷子扎着吃。这会显得很粗鲁。**正确做法**：务必使用筷子夹取，如果实在不行，用勺子也是可以接受的。

4.  **喷洒香水用餐**：喷香水去日料店用餐是绝对禁忌。它会阻碍你品尝食物最原始的味道，同时也会干扰料理师傅和其他客人。因此，身上没有气味才是最好的。

以上用餐礼仪在国内的日料店应留心遵守，在日本则必须特别注意。因为日本人通常不把喜怒挂在脸上，即使你行为失当，他们也不会开口指正，只会在心里直接给出差评。这种印象可能会一直伴随你，直到你的礼仪改善为止。

## 日式料理店的通用礼仪

既然了解了常见错误，那日式料理店有哪些通用的礼仪呢？要学习日料的用餐礼仪，先从它的种类说起。日料主要分为寿司、烤串、火锅、怀石料理、本膳料理、会席料理等。由于每种日料在用餐方式和细节上有所不同，不能一概而论，所以今天我们重点聊聊日料店的通用礼仪。

为了方便理解记忆，下面我们按照从进店到离席的顺序来具体讲解。

**进店时**
初次走进一家日料店，你想到的第一件事可能是四处拍照。实际上，在日料店这么做特别招人厌，既影响其他客人用餐，也会干扰专心做料理的师傅。**正确做法**：想拍照的话，最好只拍自己桌前的食物。若拍合照时有店内其他人入镜，应事先询问知会，这样才比较礼貌。

**用餐中**
正式的用餐环节，礼仪的重点是“静”。咀嚼食物或使用餐具都应尽量不发出声音。最稳妥的办法就是放慢自己的动作。即使是和同行的人聊天，也应该把音量调低。

这里还要提到一个细节：吃饭时切记不要让袖子粘到菜品。要越过餐盘拿取碗碟，胳膊不要盖在餐盘上。放在左边的碗碟要用左手去取，放在右边的，要先将手中的筷子放下，然后用右手去拿。

倘若你参加正式聚会或宴席，不要试图去够离你较远的菜品。在日料的观念中，那样会显得你很贪心，只吃自己面前的几道才是得体的。这些都是你吃饭稳重的体现。

不过，以太过慢动作的速度吃饭自然也是不行的。日本餐厅的用餐时间大多限制在2小时左右，所以大家还是要自己控制好时间。

**离席时**
酒足饭饱后，你可能会因为点菜过多而需要打包。这与国内不同，大部分日本餐厅不允许外卖和打包，只有酒可以带走。原因是食物一旦离店，就有保存不当的风险。吃坏了肚子，不仅顾客体验不好，餐厅声誉也会受到影响。但目前来看，国内的日料店大多可以打包，这也是饮食文化差异的体现。

## 寿司的特定用餐礼仪

说完了通用礼仪，那作为日式料理中最常吃到的寿司，有哪些重要的用餐礼仪呢？事实上，寿司的用餐礼仪有很多。在日常生活中，如果店家没有在招牌上特别声明是哪一种寿司，通常都是主流的“握寿司”，即下面是饭团，上面盖着食材的那种。

握寿司顾名思义，即不用任何模具，全靠寿司师傅手工握制而成。正是因为这样的制作方法，决定了好的寿司是有温度的。所以当寿司师傅将寿司摆在你面前时，你应该直接在**3到5秒内**食用，这样才能获得最好的味觉体验，同时也表达出对寿司师傅的尊重。如果你迟迟不动口，甚至不太在意近在眼前的寿司，师傅可能会理解为你不喜欢或不想吃，稍后便帮你将它收走，这也是一种不礼貌。

此外，吃寿司还存在许多规矩，接下来我们就来系统地告诉大家。

以下是吃寿司时需要特别注意的几点：

1.  **不要掀开食材**：对待面前摆好的寿司，万万不要掀开上层的食材。这个受好奇心驱使的行为，对很多寿司师傅来说是极大的不尊重。
2.  **取用方式**：一般来说，坐在吧台位时用手拿起寿司食用；而在4人或6人的餐桌，建议用筷子比较妥当。
3.  **一口吃完**：寿司的正确吃法是一口解决。寿司师傅在制作过程中也会将它的大小控制在一口吃完的标准，并且竭力让食材和米的比例达到近乎完美的状态。所以不要咬一口寿司再将其放回去，这会影响口感，是一种不懂吃又不讲究的做法。
4.  **姜片的用途**：寿司店给顾客提供的姜并不是小配菜，而是用来重置味觉的。当你吃下一个寿司后，可以吃一片姜，再吃下一个，这样两种寿司才不会串味。所以如果你把姜当菜吃，是会被人笑话的。
5.  **回转寿司礼仪**：在回转寿司店用餐时，应该吃一盘拿一盘。这样不会让寿司失去原本该有的温度，也不会使你给人留下浪费食物的印象。
6.  **军舰卷的蘸取方法**：军舰卷和握寿司蘸酱油的方法不一样。所谓军舰卷，也就是将米饭用海苔裹成椭圆，上面放着鲑鱼籽或其他食材的那种寿司。它要用海苔的一侧蘸取酱油。这里再教你一个小技巧：如果店里有提供姜片，你可以用姜片蘸上酱油，接着涂在军舰卷上就可以了。这样别人也会觉得你很懂行。

好了，寿司部分就到这里，大家可要记牢啦。

## 其他日料礼仪要点

除了寿司，还有一些有关其他日料的礼仪问题要分享给大家。

*   **烤串**：大阪的烤串店特别出名。吃烤串时，你手边一般会放着盛有酱汁的器皿。蘸一下，吃一口，再蘸一下，这是吃烤串的大忌。老板或烤串师傅看到的话，必定会阻止你。因为器具里的酱汁是供不同客人重复使用的，那样的吃法会影响其他客人的食用。所以**一次性蘸取足量的酱汁**才是正确做法。
*   **怀石料理**：面对一出场就是大阵仗的怀石料理，细节要处理得当。比如吃完桌上的多道料理后，碗绝对不可以叠起来，那样很不礼貌。还有，不是所有的食器都可以用手端起来吃。这其实很好分辨：看起来比较轻的碗可以拿起来吃，而看着个头很大很重的餐具，用筷子夹着吃就好了。
*   **饮酒**：如果你爱喝酒，在日料店吃饭自然少不了喝点特色啤酒。这里要提醒大家，啤酒沫溢出来时，千万不要用嘴去吸，这是很失礼、很不得体的行为。用纸巾把杯子擦干净就好了。
*   **拉面店**：当你吃完拉面之后，如果需要将碗交还给师傅，切记要**两只手端碗**。一是对师傅表示尊重，另外，一只手很难拿得稳，不小心把剩余的汤面撒出来就很不雅了。

## 课程总结

本节课中，我们一起学习了如何优雅地享用日式料理。

最后想要告诉大家，用餐礼仪的确有些繁杂，但它真的可以让你的行为举止变得更得体，同时也可以给予食物足够的尊重。

在结束之前，让我们再来复习一下本节知识的重点：

1.  **留心常见错误**：千万不要盲目相信自己的饮食操作，留心避免人们常犯的礼仪错误。毕竟这些用餐细节完全可以体现出一个人的家教。
2.  **遵守通用规则**：在日料店从进店到离席，切记这两个关键词：**安静**、**缓慢**。但是要注意用餐时间。另外，很多日料店是不允许打包和外卖的。
3.  **尊重食物本身**：不管是寿司的制作还是食用，处处都表现出对食物本身的尊重。每一个人都应该用心记住这些吃寿司的规矩。
