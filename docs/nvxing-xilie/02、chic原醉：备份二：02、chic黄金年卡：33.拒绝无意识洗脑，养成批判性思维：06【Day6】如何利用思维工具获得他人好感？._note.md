# 批判性思维：02：如何利用思维工具获得他人好感？

在本节课中，我们将学习如何运用批判性思维中的具体工具，来更有效地与人沟通并建立良好关系。课程将聚焦于两个核心思维工具：摆脱双重标准与运用引导式沟通。

在之前的课程中，我们探讨了个人思维中常见的问题，迈出了提升批判性思维的第一步。从本节课开始，我们将进入思维提升的实践阶段。

批判性思维有助于培养我们理性说理的习惯，建立清晰的公共话语逻辑。它能指导我们如何讲道理，如何识别他人（有时也包括我们自己）的不合理之处，并进行有效反驳。掌握这些思维工具，实质上也是在提升我们的情商。

## 第一部分：摆脱双重标准，关注问题本质

当我们评判事物或与人交往时，首先需要抓住问题的核心。生活中，人们常会不自觉地“贴标签”或带着偏见看问题，这会导致判断流于表面，难以令人信服。

因此，思考问题时，应尽量剥离无关因素的影响，并确保评判标准的一致性，把握公平原则。如果我们对事物的评判使用双重标准，这种评判就会像噪音一样惹人反感，并给人留下言行不一的虚伪印象。

例如，如果我反复向你强调你对我的重要性，却在许多事情上欺骗你，这就是言行不一，会让你认为我虚伪。

以下是进行“摆脱双标思维”的练习步骤：

1.  **识别关系**：在你与朋友、同学或伴侣的关系中进行思考。
2.  **寻找不一致**：找出你的行为与言论不一致的地方，或者你为自己设定的规则与为他人设定的规则不同的地方。
3.  **分析并改变**：意识到这种不公平性，并尝试调整行为，使标准一致。

**举例**：一位女同事在与男友的关系中，发现自己经常查看对方手机，却反感男友查看自己的手机。经过反思，她意识到这是双重标准。调整行为后，她不再理所当然地查看对方手机，两人的关系因此变得更轻松、坦诚，沟通也更深了。这就是通过评判标准一致化来改善关系的实例。

## 第二部分：运用“沙发式”引导思维，让他人放下防备

上一部分我们探讨了如何通过自我检视来改善关系基础，本节中我们来看看一个积极的沟通工具。

在生活中，有些人能让你在沟通时轻易卸下防备，引导你不断展开话题。这类人很可能运用了“引导思维”。运用这种思维，不仅可以主导沟通节奏，还能根据对方的反应探索事物的内在联系。

要锻炼引导思维，首先需要在沟通前进行风险把控。

**风险把控**指的是，在人际沟通中，对方未能敞开心扉往往是因为担心存在风险，例如耗时过长、你怀有特定目的、或想从他身上获取利益等。因此，在谈话前，需思考对方可能在意什么风险，并提前帮助对方化解。若风险未得到把控，对方会陷入焦虑。

**举例**：在采访行业专家时，对方通常很忙。开场时可以说：“今天我只准备请教您三个问题。” 这能快速让对方放下心理防御，安心交流。至于在三个问题下如何展开，则可根据对方状态灵活调整。

在沟通中，可以反复运用“认同观点 + 发散观点”的模式。

**举例**：当对方说“我今天吃了韩式火锅，超好吃”。
*   **如果你也喜欢**：可以认同并补充哪些菜式更棒，或聊聊对韩国辣酱的看法。
*   **如果你不喜欢**：可以先提及身边亲友很喜欢，或肯定韩式火锅的受欢迎程度，再委婉表达自己可能接受不了某些口味。

回顾我们已学的思维方式，如摆脱从众思维、摆脱双标思维等，大多具有自我反思的特性。而“沙发式”引导思维则不同，它更像一种以他人为中心的沟通技巧。

为何要学习这个思维？它对提升批判性思维有何帮助？自我提升的第一阻碍往往是自我认知不清。此前课程鼓励通过他人评价来认识自己。而本课分享的引导思维工具，旨在帮助大家获取更多、更接近真实的信息，为后续的推理与分析打下坚实基础。这种能力的直接好处是获得他人好感，深层好处则是让自己掌握更真实的信息基础。

## 课程总结

本节课我们一起学习了两个关键的思维工具：

1.  **双重标准像是人际交往中的噪音，容易引发他人强烈的情绪抵触。**
2.  **在检视双标思维的过程中，你会反思并修正自己的许多错误行为。**
3.  **“沙发式”引导思维的表层优势是在沟通中获取他人好感，内在优势是让自己获得更多、更接近真实的信息。**

建议大家在与他人沟通时，多加尝试今天提到的思维工具。下节课，我们将学习“如何读懂他人的心理诉求”。
