# 女神养成系列：01：韩式肌肤的秘密 - 先导课：仅通过家庭护理打造健康皮肤

在本节课中，我们将学习如何仅通过家庭护理来打造健康皮肤。课程将介绍讲师背景，并概述后续十二节课程的核心内容，帮助你建立正确的护肤观念。

大家好，我是来自韩国的皮肤管理师院长Emily。今天我将告诉大家，仅通过家庭护理如何打造健康皮肤的方法。

那么，大家认为健康的肌肤是怎样的呢？皮肤光滑透亮、富有弹力、不干燥，并且不易敏感，这样的皮肤就是健康的皮肤。大家都想要这样的健康皮肤。

从今天开始，请跟我一起为了打造健康的肌肤而努力。

首先，很多人都在问，为什么韩国人的皮肤都那么好？他们到底如何进行护理和医美？从结论来说，韩国人并没有接受特别的护理和医美。他们和你们一样，会去打水光针、做激光。韩国人也是和你们一样去做医美。

但根据我数次往返中韩的经验，让我感受最深的是，韩国人非常注重基本护肤。相反，中国人对于基本护肤想得很简单，不是很重视。其实基本护肤非常重要。我会把基本护肤做得很充分。

所以从今天开始，我将跟大家分享我的基本护肤知识。为了让大家对自己有信心，首先给大家看一下我2016年的皮肤照片。

感觉怎样呢？是不是很难以接受。当时是因为压力大突然产生的炎症青春痘。可是，身为美容院院长的我竟然有这样的皮肤症状，谁还会来美容院向我咨询护肤呢？我也会因此对美容咨询失去信心。

因为到处出差，我无法在美容院接受护理。因此，每次当我工作完，回到酒店之后，都会做基本的护肤。各位现在看到的我的皮肤，大家觉得怎么样呢？现在我的皮肤只涂抹了基础的护肤品，没有做任何的彩妆遮瑕。

是的，现在很多人都在为各种皮肤症状而烦恼。虽然去皮肤科很重要，可是在家不做皮肤护理是没有任何用处的。现在是否认知到在家护肤也是很重要的步骤呢？

现在给大家简单讲述一下，我在教程中将要讲述的护肤基本程序。

以下是课程的核心内容概述：

*   **第一课：根据皮肤类型管理的方法。** 要知道自己的皮肤类型才能准确地进行护理。
*   **第二课：正确的清洁肌肤。** 在2016年时，我因为工作忙碌，连卸妆都没有仔细清洗，因此皮肤因压力和彩妆残留物导致了炎症青春痘。所以，卸妆清洁肌肤是一个很重要的步骤。
*   **第三课：去除角质的方法。** 学习怎样去除角质，并根据个别皮肤差异采用不同的方法。健康的皮肤以四周为基准，形成新角质和脱落角质为循环。外部因素、不良营养和工作压力可能导致角质无法正常脱落，从而堵塞毛孔、降低皮肤免疫力，使皮肤表面粗糙。因此，在一定的周期内去除角质很重要。
*   **第四课：挑选适合自己的护肤品。** 市面上有许多护肤品。在众多产品中，哪些是适合自己的呢？选择的标准又是怎样的呢？这些就是我们要学习的内容。
*   **第五课到第八课：常见皮肤问题护理。** 这些课程将讲述大家关心最多的皮肤问题，包括敏感性皮肤护理、美白护肤、毛孔护理以及防治青春痘。
*   **第九课：韩国女明星的皮肤护理方法。** 学习如何打造“水光皮肤”。
*   **第十课：让皮肤看起来年轻的方法。** 让我们一起学习如何让自己的皮肤看起来年轻十岁。
*   **第十一课：打造小脸蛋的方法。** 学习如何在生活中通过改善习惯来减少浮肿，打造小脸。
*   **第十二课：根据季节护理皮肤。** 我们的皮肤会随着季节和环境而变化。例如，干性皮肤到了夏天可能变成油性。所以，让我们一起来学习怎样根据季节去护理皮肤。

大家感觉怎样呢？听完这十二节护肤课程的介绍之后，是否期待和我一起进行学习？跟我一起学习这十二节课程，让自己的皮肤变得更好。和我一起加油吧。
