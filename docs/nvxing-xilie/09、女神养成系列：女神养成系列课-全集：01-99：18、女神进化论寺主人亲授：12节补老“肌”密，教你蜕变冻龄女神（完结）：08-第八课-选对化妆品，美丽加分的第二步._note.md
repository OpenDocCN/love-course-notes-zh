# 女神养成系列：08：选对化妆品，美丽加分的第二步

在本节课中，我们将学习如何根据个人需求，科学地选择一整套适合自己的化妆品。我们将从常见误区入手，然后分步骤指导你完成首次和二次采购清单，帮助你避免浪费，精准投资。

## 化妆品选购的三大误区

上一节我们介绍了化妆的基础知识，本节中我们来看看选购化妆品时常见的三个误区。

**误区一：认为必须拥有“全套”化妆品才能化好妆。**
*   不存在客观的“全套”化妆品清单。
*   每个人的需求和妆效目标不同，使用的产品组合也完全不同。
*   应根据自身需求选购，而非盲目购买看似完整的套装，以免购入大量无用产品。

**误区二：盲目跟风，别人买什么自己就买什么。**
*   每个人的肤质、肤色和问题各异，适合他人的产品未必适合你。
*   购买前需自问：`产品功能 + 是否解决我的问题 + 能改善何种现状`。
*   例如，在不清楚自身需求时跟风购买妆前乳，可能完全无法解决你的底妆问题。

**误区三：误解价格与品质、安全性的关系。**
*   **公式：产品价格 ≠ 产品安全性 ≠ 妆效好坏**
*   只要经过药监局备案并正规生产，无论价格高低，化妆品都是安全的。
*   过敏与产品具体成分有关，与价格无关。
*   昂贵的产品有其价值，但妆效未必更好。选择**适合自己**的产品，其效果可能远胜于昂贵但不合适的产品。

## 第一步：首次采购核心化妆品

说完了误区，我们来动手为自己定制一套化妆品。以下方法能帮你节省大量不必要的开支。

首次采购的目标是解决素颜时最需要提升的面部问题。以下是选购清单：

**1. 底妆产品**
首先观察素颜皮肤，根据瑕疵程度选择产品：
*   **皮肤极好**（肤色均匀、无瑕疵）：可选择**有色面霜**类产品，如素颜霜、CC霜、有色妆前乳。它们遮盖力弱，但更轻薄自然。
*   **皮肤一般**（略有肤色不均、轻微瑕疵）：选择**轻薄型粉底液**，例如 `MAKE UP FOR EVER 水粉霜`。
*   **皮肤问题较多**（肤色严重不均、瑕疵明显）：需要**遮盖力强的粉底液**，例如 `MAKE UP FOR EVER 高清粉底液`。
*   **关于BB霜**：BB霜与粉底液本质相同。大部分BB霜油脂含量较高，**油皮及油敏肌**需谨慎，可能致痘。
*   **关键**：选择粉底时，务必匹配**肤色（色号）** 和**肤质**（参见上节课内容）。

**2. 眉部产品**
检查眉毛状态，按需购买：
*   **修整杂毛**：需购买修眉刀，新手推荐**贝印最小号修眉刀**。
*   **修剪过长眉毛**：若眉毛过长超出边缘，需购买**圆头修眉剪**，更安全。
*   **梳理眉毛**：需一把**螺旋眉梳**（许多眉笔末端自带）。
*   **填补眉色**：若眉毛过淡或不均匀，需购买**眉粉或眉笔**。新手推荐**伊蒂之屋眉笔**（注意品牌名，非“爱丽小屋”）。
    *   **色号选择**：选择与发色接近的颜色。若不确定，可选**2号灰褐色**，该色系容错率高。

**3. 眼部产品**
眼睛是妆容重点，必备品如下：
*   **睫毛膏**：必备品，能立刻让眼睛有神。新手推荐**悦诗风吟纤巧精细睫毛膏**，妆效自然，不易结块（出现“苍蝇腿”）。
*   **睫毛夹**：推荐**贝印睫毛夹**，便宜好用。
*   **眼线产品**：必备**眼线液笔**（非眼线液、眼线膏或眼线笔）。它线条干净，易于新手操作。品牌推荐**Kiss Me**，顺滑持久。
    *   **备选方案**：若使用眼线液笔仍手抖，可尝试**眼线膏**，它更易上手，但妆感稍重。

**4. 唇部产品**
唇妆是提升气色的关键：
*   **润唇膏**：必备打底产品，使用你日常惯用的即可（例如凡士林润唇膏）。
*   **口红**：选择适合自己肤色的颜色。
    *   **最佳方法**：前往专柜试色，重点观察涂抹后**整体肤色和气色**是否得到提升。
    *   **简便方法**：网购最便宜的**珊瑚色、南瓜色、砖红色**三种口红试色，通常其中必有一款适合你。

## 第二步：检查妆后问题，进行二次采购

购买并学会使用首次采购的化妆品后（具体画法下节课教），需要观察妆后效果，并进行二次针对性采购。

**1. 底妆问题与解决方案**
检查并解决以下常见底妆问题：
*   **毛孔粗大**：需在粉底前使用**遮盖毛孔的妆前乳**。
*   **瑕疵未完全遮盖**：
    *   首先检查粉底是否“少量多次”涂抹足够。
    *   若用量足够仍有瑕疵，需购买**遮瑕膏**。
        *   **裸色遮瑕膏**（与粉底色号一致）：用途广泛，可遮盖色斑、痘印、暗沉等。在底妆前或后使用均可。
        *   **有色遮瑕膏**：针对性更强，但需在底妆前使用。
            *   **公式：绿色遮瑕膏 → 遮盖红色瑕疵（红血丝、红痘印）**
            *   **公式：桃色/橙色遮瑕膏 → 遮盖黑眼圈**
*   **快速脱妆（出油）**：需在粉底后，于易出油部位使用**散粉**定妆，并随身携带**粉饼**方便补妆。
*   **干燥卡粉**：
    *   首先加强护肤保湿。
    *   若仍卡粉，可随身携带乳液，用棉签蘸取卸除卡粉处，再用湿粉饼补妆。
*   **面部缺乏立体感**：可购买比肤色浅一个色号的**遮瑕液/膏**作为高光，塑造轮廓（具体操作下节课讲解）。
*   **面色苍白无气色**：可购买适合的**腮红**。但当下妆容潮流中腮红非必需，可根据个人感觉决定。

**2. 眼妆问题与解决方案**
检查并解决以下常见眼妆问题：
*   **眼部缺乏立体感**：若完成眼线和睫毛后仍觉平淡，可增加**眼影**。新手推荐 **Kate 骨感眼影（色号 BR-2）**，安全易上手。
*   **眼部脱妆/晕染**：
    *   上眼皮晕妆：可在上粉底前，于眼皮使用**妆前乳**打底。
    *   下眼皮晕妆：可在下眼皮轻扫**散粉**。若已晕妆，用化妆棉轻轻弹掉即可。

## 课程总结

本节课中，我们一起学习了如何系统性地选购化妆品：
1.  **避开三大误区**：不追求“全套”、不盲目跟风、正确看待价格。
2.  **分两步进行采购**：
    *   **首次采购**：针对素颜问题，购入**底妆、眉部、眼部、唇部**核心产品。
    *   **二次采购**：针对首次化妆后的妆效问题（如遮瑕、定妆、立体感等），进行针对性补充。
3.  **核心原则**：始终围绕“**我有什么问题？我需要解决什么？**”来选购，才能构建出最适合自己的化妆品组合。
