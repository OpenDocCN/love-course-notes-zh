# 花镇情感课程：01：十秒吸引法则：如何通过话术增进感情

## 概述
在本节课中，我们将学习如何通过有效的话术来增进人际关系，特别是如何避免因“过于诚实”而引发他人反感，并掌握基于人性需求的沟通技巧，从而成为一个更受欢迎的人。

---

上一节我们提到了沟通在人际关系中的重要性，本节中我们来看看一个具体的反面案例，理解什么样的话会让人讨厌。

### 反面案例：过于诚实的代价
我之前有一位学员，他因单位人际关系差而感到痛苦。他举了一个例子：单位会计走到他面前说“我觉得自己长胖了”。他抬头看了看会计，回答说“好像是有点哦”。从此以后，他去会计那里报销，总会拖延很长时间。

这位学员说的是实话吗？是实话。但实话往往会让人讨厌。有一句经典的话叫做“诚实是丑陋的美德”。我们从小被教育要诚实，诚实本身没有对错之分。然而，一旦你过于诚实，在这个复杂的社会中就很容易得罪人。

其实，只要情商在线，就应该知道告诉那位会计：“我觉得你不胖啊，不胖不胖，你一点都不胖。”这就是人性。如果你真的想增加别人对你的好感，加深感情，你必须学会从人性的角度来回应他人。如果你仅仅基于事实和你从小接受的教育来回答，往往可能会让别人反感。

---

理解了反面案例后，我们来看看人性的核心特点是什么。

### 理解人性：接纳与夸赞
人性是什么？人性喜欢别人夸自己，喜欢别人接纳自己。否则怎么会有“忠言逆耳利于行”的说法呢？很多时候，对你有益的话，恰恰是你讨厌听到的。例如，很多女孩子听到男朋友说“你脾气太差了，要改一改，否则咱俩没法走下去”，往往会暴跳如雷。听到对自己好的建议应该接纳，这是一种教养，但它不是人性。人性就是听不得别人对自己的差评。

但事实上，这样的话是对你好。因为两个人只有脾气好才能和谐相处。如果整天吵架，积累的负面情绪多了，感情就容易破裂。

---

那么，我们如何才能既说出符合人性的话，又让话听起来有艺术感、不做作呢？以下是具体的拆解步骤。

### 第一步：判断真实意图
首先，你必须站在人性的角度判断，对方说的话是不是他真正的心里话？他真正想听到什么？

就像一个女孩子跟你说“我胖了”，她是真的希望你来肯定她胖了吗？她难道自己没有称过吗？就像你的领导坐你的车，问你“要不要给你一些油钱”，他是真的希望听到你说让他付钱吗？即使他真的想付钱给你，你也不能说要。人性就是希望少花钱、多赚钱，希望别人给自己花钱。如果你真的拿了领导的钱，对不起，你以后就别想发展了。

真正聪明的人会这样回答：“领导，您坐我的车是我的荣幸，怎么能收您的钱呢？”这样回答，你肯定会得到重用，因为领导会觉得这个人懂事。

我再举个例子。以前我有一位学员，她的老公跟她说：“我觉得自己掉头发好厉害，我可能会秃头，你会不会嫌弃我？”这个时候如果你说嫌弃，那不是找抽吗？你肯定要回答不嫌弃。

如果你能做到第一步，听懂别人的真正意图，而不仅仅是基于事实回答，那么你在话术方面已经及格了，至少你是一个不那么让人讨厌的人。

---

然而，仅仅及格对我们来说是不够的。我们的目标是成为一个让别人对我们有好感的人。那么，第二步该怎么办呢？

### 第二步：真诚且艺术地夸赞
第二步是回答，并且让别人觉得你夸他夸得很真诚、不做作。

我们回到第一个话题，会计说他自己胖了。这时你应该怎么回答呢？很简单。

**公式示例：幽默否定 + 转移焦点**
```
你可以说：“不许造谣，造谣是要坐牢的。”
```
这时，会计的注意力只会放在你的幽默回应上。你满足了他的人性，因为你在夸他瘦，在否定他的胖，这正是他希望听到的答案。而且你的回答比较有趣，他的注意力被你吸引走了，不会纠结你是在讨好他。

我再给你们举个例子。我有一位学员，她的老公（当时是男朋友）小时候比较黑，他很讨厌别人说自己黑，因为亲戚和老师都因此打击他，这在他心里留下了阴影。所以他跟我学员谈恋爱时，就问：“我觉得自己挺黑的，你会不会嫌弃我啊？”

这时，你会怎么回答呢？肯定是说不讨厌，对吧？但不要说得太生硬，比如“不讨厌不讨厌，我喜欢你的黑”，这样别人也不会太开心。这就相当于你问你男朋友“你觉得我胖吗？”，男朋友说“我不介意，我很喜欢你的胖”，你还是会难受，对吧？

那这时怎么回答呢？你可以这么说：

**公式示例：关联正面形象**
```
“怎么会讨厌你呢？你是我的老公啊。而且，你这可是古天乐同款。”
```
这时他就会开心，因为把他跟大帅哥古天乐类比起来了。就像那个说自己要秃头的老公，他老婆可以说：“没事，你这是吴彦祖同款。”他能不开心吗？

这其实就是一种公式，大家可以套用。下次别人再说自己哪里不行的时候，你可以说：“哎，你这是古天乐同款/吴彦祖同款/刘德华同款。”随便讲，反正别人不会难受。但你千万别讲什么赵本山、潘长江同款，那会招来别人的“暴揍”。

这种话术非常简单，只要你学会，在很多场合都能用。因为我们社会中有很多人不自信，对自己的各方面都不自信。

例如，男人跟你说没钱。这时你能告诉他“你没钱我讨厌”吗？不要这么说。你要跟他这么讲：

**公式示例：价值重述 + 引用权威**
```
“没事啊，我跟你在一起是最开心的。而且，你没听马云说吗？钱太多是一种负担。没有太多的钱，反而能让我享受到最纯真的爱情。”
```
那你说他能不开心吗？

---

你们可能会想：“老师，能够真正听懂别人话的意图，我是能够做到的。但是怎么才能讲得比较有艺术呢？”我再给大家举个例子，拆分开来说。

### 进阶技巧：结合多重人性需求
其实还是我的一位学员。之前他跟领导出去吃饭出差，吃完饭领导说：“我们俩要不要AA？”他回了个“好的”，然后把一半钱给了领导。回去之后，领导就给他穿小鞋了。他来求助说怎么办。我说放心，跟领导出去还有机会，下次把握对就可以了。

下一次，领导果真又问：“咱俩要不AA？”我们先问大家，能不能AA？肯定不能AA，这不找抽吗？肯定是你付。因为从人性的角度说，人是不喜欢花自己钱的，都喜欢花别人的钱，对不对？

那么，第二个问题，怎么能把它说得艺术一点呢？当时我是教我的学员这么说的，给大家复述一下：

**公式示例：提供理由 + 表达荣幸**
```
“领导，平时请您的人这么多，我们想请您都没有机会。今天正好有这样一个机会，您不如就让我请您吃饭喽。”
```
你们说说看，领导听你这么说话，能对你没有好感吗？回去之后很多事情都方便多了，领导再也不给他穿小鞋了。

我们来分析一下这里面的技巧：
1.  **人性角度**：没有人希望自己花钱，都希望别人花钱。即使领导是真心想付钱，你不让他花，他也会开心。
2.  **虚伪性**：人都是“虚伪”的。就像你希望他付钱，但你的真正想法不会直接说“你付就你付吧”，这样会显得你很小气。因为传统教育告诉我们“大方”，直接要钱会遭致指责。他希望的是，他说他来付，然后你来说“不用，我来付”。懂了吗？人就是这么“虚伪”。
3.  **提供理由**：人需要别人给自己一个理由，让自己能够心安。作为领导，他也知道让下属请吃饭可能不合规，甚至觉得他年龄大是在欺负你。但是，你给他找了一个合适的理由：“平时请你的人太多，我都没有机会，今天正好给我一个机会。”从这三个角度，人就会觉得很舒服。

---

为什么我要跟你们讲这样一个案例呢？我只是为了告诉你们，真正好的话术都来自于对人性的分析。把人性分析透了，包括我们上一节课讲的“不要把压力点放在自己身上，要把压力点放在别人身上”，这也是人性。因为人一旦纠结于别人是否会接纳、喜欢自己时，压力点就全在自己身上，人就会紧张。不被人接纳、不被人喜欢，是人性最讨厌的东西之一。

所以说，优秀的话术没有一个特定的、可以打遍天下无敌手的公式。但是，一个公式可以用在很多领域。就比如“吴彦祖同款”、“古天乐同款”，你可以在外貌方面用得非常多，但它不能包打天下。所以你要在生活当中，在各种事情中，用人性来分析、来提炼。慢慢地，你懂得了人性，那么你给别人的回话基本上都是让别人开心的，至少你不会让别人讨厌你。

请大家放心，我们的课程前面主要是给大家介绍各种各样的技巧分类，包括相对应的心理和人性。在后期，我们会给大家补充大量的场景应用公式，来帮助你消化，并且跟你补充各种各样的人性思考，方便你对人性的理解更加全面。

---

## 总结
本节课我们一起学习了如何通过话术增进感情。我们首先通过反面案例认识到“过于诚实”可能带来的问题，然后深入探讨了人性的核心是渴望被接纳和夸赞。接着，我们拆解了有效沟通的两步法：第一步是判断对方的真实意图，第二步是真诚且艺术地进行夸赞。最后，我们通过进阶案例分析了如何结合多重人性需求（如避免自己花钱、满足“虚伪”的社交期待、提供心安的理由）来达成更高阶的沟通艺术。记住，优秀的话术源于对人性的深刻理解与灵活运用。
