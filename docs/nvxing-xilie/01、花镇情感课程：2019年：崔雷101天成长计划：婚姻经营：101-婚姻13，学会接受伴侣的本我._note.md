# 花镇情感课程：01：崔雷101天成长计划：婚姻经营：101-婚姻13，学会接受伴侣的本我

在本节课中，我们将要学习一个核心概念：如何通过理解伴侣的真实需求，来建立自信并改善关系。许多关系中的焦虑和自卑，都源于不了解对方想要什么。我们将通过一个具体案例，深入探讨如何运用“男性思维”的知识来应对婚姻危机，并让关系重回正轨。

## 不知道对方要什么，是关系紧张的根源

上一节我们介绍了课程主题，本节中我们来看看问题的核心表现。

你不知道对方要什么，导致的结果就是约会的时候，你不知道该聊什么话题。现在你要给你前任发信息，你不知道你该发什么信息。因为你怕他不回复你，你不知道发什么信息可以投其所好，让他愿意回复你。

愿意跟你沟通。如果是老公的话，你所做的一切行为，在你老公看来，都是为了让他早点回家。所以他很有底气，他知道你要什么东西。而你却不知道你老公在要什么。

你会琢磨他心里有没有你、爱不爱你、到底怎么想的、会不会跟小三在一起、会不会跟她生孩子、会不会去相亲结婚等等。你会想很多，因为你不知道对方要什么。举一个最简单的例子：你面对你喜欢的男神时，想追他，你会很紧张。为什么呢？你不知道他要什么。但是一个条件一般的人来追你的时候，他千般讨好万般讨好，你会很有底气。为什么呢？因为你知道他做的一切都是为了得到你。这意味着你有绝对的自信，是因为你知道对方要什么。

因为太在意一个人，就会看不明白他要什么。其实太在意的不是那个人，是自己。一个人只有眼里全都是自己时，眼睛里是看不见别人的。

所以当你看不见别人的需求的时候，就说明你内心里和脑子里全都是你自己。你说“不对呀，我天天琢磨他喜欢什么呀”，那也意味着你脑子里要么都是你自己，要么都是你自己的付出，没有把对方放在心里去理解。

所谓的太在意一个人，其实在意的还是自己。一个人之所以自信，是因为我知道你要什么东西，而你却不知道我要什么。敌在明，我在暗。

就是这样的一个格局，可以让人自卑自信，忽高忽低。当你不知道的时候，你就发现很紧张。举个例子，聊天的时候不知道该发什么信息。朋友圈他发动态的时候，你不知道该怎么去评论。因为你不知道对方要的东西是什么，所以你担心犯错误。而你很清楚对方要什么的时候，你会觉得很有底气。

所以在一段关系里面，几乎你的所有自卑都来源于你不知道对方要什么东西。到最后你就会发现，你根本就不知道自己要什么东西，也不知道对方要什么东西。所以你做什么都吸引不了对方的注意。

## 获得关系自信的关键：掌握“男性思维”

所以，如果要在关系里获得自信，有一个技能是必须掌握的。有一个学科，你是必须明白的。

我有一个学员，她老公出轨了。她现在很焦虑、很担心。晚上老公不怎么回来，两人分居，她知道老公在小三那里。她现在就想让老公早点回来，但她不知道老公的需求是什么。

但至少我清楚一个点。我清楚的点是：所有男人在出轨期间都会进入一种**观望状态**。

以下是关于“观望状态”的具体表现：

*   **他在观望是选择家里面还是选择外面**。站在他的角度，他是两者都想要的，既想要家里红旗不倒，又想要外面彩旗飘飘。利益最大化是他都想要的。
*   **如果不能两全，他会进入观望状态**。这个时候他就会去评估是小三还是妻子，会站在情感、喜欢、利益的综合角度去评估到底要跟谁在一起。
*   **在这个节骨眼上，谁作谁闹，都会让这个男人离他很远**。只要小三作，老公就会回家。原配作，老公就会跟小三在一起。谁沉不住气，男人就会偏向于另外一方。所以他一直是处在一个观望状态。

我现在给你举一个例子。假设你男朋友出轨了，然后跟你说：“亲爱的，我特别想摆脱那个女人，让我们一起来击退她吧。”你怎么办？如果你跟他一起去击退这个小三，好像这个男人很有诚心。但我要告诉你，这个千万不能帮。这个知识点就是**男性思维**。你要去了解一个男人的想法，男人在出轨期间，他一定是观望、评估的状态。他能不发生冲突就不发生，因为他占着便宜。如果一定要做选择，他会观望，等着这两个女人谁先沉不住气，或者评估哪个利益大。这就是男性思维的一个知识点。

## 运用知识制定策略：一个实战案例

那我们掌握了这个知识点之后，要做什么呢？如果我要帮助我的学员让她的老公回归家庭，我需要怎么做？我是不是应该让我的学员沉住气、稳定情绪的同时，让小三沉不住气。

以下是当时我给学员制定的具体行动方案：

1.  **雇佣第三方介入**：去找一家跑腿公司，或者雇一个女大学生。为什么要雇别人而不是你自己？因为你的情绪可能不受控制。雇一个人，花点钱。
2.  **制造接触与证据**：让这个女孩（我们称她为小A）去加你老公微信，引导你老公给她发52或520元的微信红包，并且**一定要截图**。这个截图就是证据。
3.  **创造信息泄露机会**：要让小A和老公有单独相处的机会见面。中间必须创造一次机会，比如男人上厕所手机放在桌上，或者女方假意说“我玩会你的游戏好不好”。有这么一个动作就行。
4.  **传递信息并撇清关系**：把小三的微信给小A。因为小A并不知道小三的微信是哪个，分不出来，但你一定要给。关键在于，中间小A有玩过男人的手机，男人一旦发现小三被骚扰，就会认为是小A自己去找的小三微信，而不是你给的。
5.  **激化矛盾，隔岸观火**：接下来，让小A用小三的微信，把截图发给小三，并说一句“臭婊子”。这样小A和小三就会争吵起来。之后，小A可以废弃这个微信号，不再联系。小三就会对这个男人的忠诚度产生质疑和怀疑。而当他们两个有矛盾的时候，原配完全没有任何参与和干预。
6.  **保持稳定，展现价值**：这个时候，原配在家里装作什么都不知道，该做饭做饭，该干嘛干嘛。男人会迫于压力，回归到自己的家庭。

当时给他设计了这样的一个方案。这个女生在听到我这样的方案之后，她的情绪就稳定下来了。

## 策略生效的原理：知识与自信的来源

我现在要问你的一个问题是：她为什么从一个自卑的情绪变得稳定了呢？他从一个很焦躁、很不安的情绪，听我讲完，瞬间变得稳定一些了。是什么让他内在现在比较稳定、并且很有自信了呢？

很简单，是因为我了解男人那种特性。我知道她的老公处在观望状态，**谁施加压力，他就会远离谁**。所以他在观望，因此原配必须不能抱怨，要“忍着秋后算账”，然后让小三自乱阵脚。小三以为自己胜券在握的时候，突然被一个陌生人骂“臭婊子勾引我男人”，很容易被激怒。

激怒一个女人的最好办法，就是攻击她的占有感和正当性。但我想告诉你们一个点是：你一定要有这种干练的方法，而这种方法就来源于你对男人的了解。因为知道了男人怕压力、男人在观望、男人会因为某一方压力大而选择另外一方，所以你就让小三自乱阵脚，然后你自己保持稳定。因此他就会回到你身边。

这个知识点就是**男性思维**。而你掌握这个男性思维之后，再去制定计划和方案，就是很简单的事情了。

本节课中我们一起学习了关系紧张的核心在于不了解伴侣需求，并深入探讨了“男性思维”中的一个关键状态——**观望状态**。通过一个让第三者内部产生矛盾的实战案例，我们看到了如何运用对男性心理的了解来制定策略，从而稳定自身情绪，让关系向有利于自己的方向发展。掌握对方的需求和思维模式，是建立关系自信、扭转局面的根本。
