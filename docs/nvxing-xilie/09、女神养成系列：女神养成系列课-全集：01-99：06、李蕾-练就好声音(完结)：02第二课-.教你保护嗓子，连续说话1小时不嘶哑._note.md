## 声音是如何产生的？
我们一生下来就会发出声音，但大部分人并不清楚声音到底是如何产生的。发出声音需要四个步骤：
1.  **肺部呼出气流**。
2.  气流**震动声带**发出基础声音。
3.  声音经过**共鸣腔**放大。
4.  最后通过**口唇部咬字**形成清晰的语音。

上一节我们介绍了声音产生的四个物理步骤，本节中我们来看看“声”与“音”的区别。

## “声”与“音”的区别
“声”和“音”不是一回事，需要拆开来看。
*   **音**：指的是吐字归音，通过控制口唇部肌肉发出，其标准是**准确清晰**。
*   **声**：是通过肺部呼出气流，震动声带并经过共鸣腔而产生。**声的根本在于气息**。气息越足，声音的“本钱”就越好。

因此，我们常说的“练嗓”，如京剧的“吊嗓子”，其实主要是在**练气**。

## 自测：你的气息充足吗？
了解原理后，你可能会想知道自己的气息是否充足。以下是自我检测的方法：
1.  打开录音设备或微信语音。
2.  全身放松。
3.  **男生**持续发“啊——”音，**女生**持续发“咿——”音。
4.  发音时喉咙放松，**用气流推动发声**，尝试持续5秒以上。
5.  回听录音，检查声音是否出现**不平稳、断断续续、忽高忽低、破音**等情况。

如果出现上述状况，说明你的气息不足，说话方式可能不正确，并容易伤害嗓子。

## 保护嗓子的三大纪律
对比自测结果，你就能发现问题所在。接下来，我们讲解如何解决这些问题，核心是遵循保护嗓子的三大纪律。

在讲解纪律前，需要先了解一个关键概念：**喉元音**。

### 认识“喉元音”
喉元音是指你发音时主要用力的部位。测试方法如下：
1.  将手指轻轻放在咽喉部位。
2.  边说话边上下移动手指，找到**震动最剧烈**的点，那就是你习惯用力的发音部位。
3.  手指轻轻搭在该处（不要用力按压），然后尝试在说话时用力按下去，你会感到非常累。
4.  松开手，保持手指轻搭，全身放松，**喉头不用力**，仅用一点点气流轻轻推动。

你会发现，即使不用力，也能发出声音。这说明：**发出声音本身是一件不需要喉咙过分用力来完成的事**。

### 纪律一：节约气息，避免大喊大叫
好声音的第一标准是**节约气息**。不要用喉咙大喊大叫，否则嗓子容易疲倦、充血，引发炎症。
*   **真嗓**：完全用自身声带发出的声音，费力且伤嗓。
*   **小嗓/气声**：通过**气息和肌肉控制**发出的声音，更省力。

优秀的京剧演员或歌唱家都擅长**真假声结合**，这依赖于对气息的自由调动。

### 纪律二：掌握“深吸慢呼”法
普通人如何练气？一个重要方法是**深吸慢呼**。
*   正常人呼吸一次约3-4秒。
*   通过训练，呼气时间可延长至十几秒甚至20秒以上，这考验对气息的控制能力。

以下是深吸慢呼的练习方法：

**吸气要领（深吸）**
目标是吸气“一大片”。练习时，请背靠墙壁站立，保证双肩下沉、背部挺直，让共鸣腔通道畅通。
1.  意念上，想象将周围大片的空气都吸进来。
2.  气息要吸到**丹田**位置（肚脐下方约一巴掌处）。
3.  吸气时感觉两肋张开，丹田鼓起变硬，整个背部紧贴墙壁。
4.  公式表示：**有效吸气 → 丹田鼓起 + 两肋扩张**

**呼气要领（慢呼）**
目标是呼气“一条线”。最忌讳急促“喷气”。
1.  呼气要**平稳、持久、均匀**。
2.  想象面前有落满灰尘的书架，用嘴唇吹出一条细长的气线，轻柔地吹走灰尘。
3.  这条气线越绵长，你的声音就越稳定、传得越远。

**控制练习：数枣子**
这是一个经典的气息控制训练，适用于播音员、主持人等。
练习文本：
> 出东门，过大桥，大桥底下一树枣，拿着竹竿去打枣，青的多，红的少。

练习步骤：
1.  念完上面文本后，**深吸一口气**。
2.  用这一口气连续数：“一个枣、两个枣、三个枣……” **中间不能偷气、换气**。
3.  记录一口气能数多少个枣。
    *   初学者：20个左右为不错。
    *   通过练习，可以逐渐增加数量，目标是稳定进步。

**深吸慢呼的三个重点**
1.  **吸气要深**：贴墙站立，气沉丹田，两肋张开。
2.  **呼气要稳**：平稳持久，如一条线，用腹肌控制气流送出。
3.  **喉咙要松**：呼气时喉头放松，避免产生“扼喉感”，保证从腹腔到喉腔的通道畅通。

### 纪律三：状态不佳时的应对策略
即使掌握了方法，也可能遇到感冒或疲劳等嗓音状态不佳的时候。以下是应对建议：
1.  **少说话**：尽可能节约用嗓。
2.  **节约用嗓**：必须说话时，使用更节省气息的方式。
3.  **改变发音位置**：这是高级技巧，通过调整共鸣焦点来减轻声带负担。
4.  **按摩缓解**：找到舌根与喉骨之间的凹陷处，用指腹轻轻画圈按摩酸痛点。
5.  **日常养护**：
    *   少吃油腻食物。
    *   嗓音疲劳时，避免辛辣、过冷过热食物。
    *   少喝碳酸饮料、高糖饮料，少抽烟。
    *   冬季注意颈部保暖，可戴围巾。
    *   **多喝白开水**，正确的喝法是：小口含在嘴里，慢慢咽下。
    *   润喉糖只能暂时缓解，不可过量依赖。
