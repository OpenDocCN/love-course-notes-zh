# 舒化课程合集：第28节：为什么吃鸡可以收到大红包 🎮🧧

在本节课中，我们将学习如何通过线上游戏互动，吸引异性关注并建立高价值形象，从而引导对方主动进行物质付出。课程将拆解一次具体的游戏社交案例，分析其中的沟通技巧、心理洞察和价值展示策略。

## 背景介绍

上一节课程中，我们现场演示了“撩汉”技巧。许多学员对演示中提到的、通过玩“吃鸡”游戏与异性互动并收到红包的过程感到好奇。本节将详细分享那次互动中的具体沟通方法与策略。

那次互动发生在5月3日。我通过玩“吃鸡”游戏认识了一个男孩，并利用游戏中的交流窗口与他聊天。随后，对方主动发送了10个金额为199元的红包，总计近2000元。金额虽不大，但过程值得分析：如何让一个未曾谋面的男性，在短时间内多次主动联系，并心甘情愿地发送大额红包？

## 第一步：建立互动与观察

首先需要交代背景。许多学员可能不常玩“吃鸡”这类游戏。我玩的时间也不长，可以简单总结：这是一个多人组队竞技游戏。当玩家被攻击时，生命值（血液）会减少，需要通过使用“能量饮料”或“急救包”来补充。

我的游戏技术并不好，动作笨拙，经常找不到方向，也捡不到好的装备和补给品。那天和这个男孩组队时，他总是陪在我身边。每当我的生命值因被攻击而减少时，他都会说：“快点，你过来我身边，你该补点血了。”然后他会将自己身上的能量包或急救包丢给我，让我捡起来使用。

这个举动与之前和其他人玩游戏时不同。他似乎能察觉到我缺乏补给。如果当时不使用这些物品，我的生命值就会耗尽。他总是愿意将自己的资源分享给我。

## 第二步：运用声音魅力与给予赞美

这时，我采取了关键行动：开口夸赞他。我通常会在游戏设置中关闭自己的麦克风，只收听队友语音。但因为他，我特意打开了麦克风。

我说：“谢谢你，你很贴心。”请注意第一点：**声音的吸引力**。过去在私教课中我强调过，与异性交流时，需要注意修饰自己的声音。平时与姐妹交流可用本声，但当你想吸引特定异性时，声音需要有所修饰，变得更为圆润、动听。

声音具有魔力，能直接影响异性对你的初步判断，包括样貌、年龄和性格的想象。一个生动的例子是：我的一位姐妹，平时声音很普通，但一旦和男朋友通话，声音会变得极其温柔甜美，像换了一个人。新认识的男性往往很吃这一套，这类似于“孔雀开屏”，是异性吸引中的一种法则。

因此，在吸引男性的初期，务必注意你的声音。这就是为什么以前我总让大家练习说“喂”这个字的语气和感觉。

## 第三步：进行“冷读”与提供情绪价值

在夸赞之后，我进一步说道：“看样子你经常对你的其他小姐姐小妹妹这么好。其他的小姐姐小妹妹肯定都很愿意跟你玩吃鸡吧？因为你这么体贴，我相信任何人跟你一起玩，都会觉得你很棒。”

对方回应道：“并没有。我每次给别人血包和能量的时候，他们都没有感谢过我。你让我觉得你挺特别的，你很感恩。”

这里运用了“冷读”技巧。通过他喜欢在游戏中保护队友的行为，我“冷读”出他的内心特质：**他具有“大男子英雄主义”，渴望拯救弱小，保护他人**。洞察到对方的心理需求后，就要投其所好。

整个晚上，我们玩了三次游戏。只要我不处于极度危险中，我就会适时示弱并请求保护，同时不断给予赞美。例如，当有敌人靠近时，我会说：“你快来，我不知道怎么开枪，你帮我打死他！”（实际上我会开枪）。然后在他解决敌人后，立刻夸赞：“保护我，救我，你真棒，你就是英雄！”

通过这种方式，我持续为他提供了极高的“情绪价值”。

## 第四步：无意中展示“高价值”

那么，他为什么愿意发送10个红包呢？我夸赞他、满足其英雄主义需求，这些是我“刻意”为之的策略。但他发送大额红包的举动，却包含一些“无意”中促成的因素。

那天晚上较晚的时候，我想吃小龙虾。昆明有一家店不提供外卖，只能通过跑腿服务购买。当跑腿员将小龙虾送到时，我需要支付商品费用（128元）。当时我正在玩游戏，由闺蜜帮忙开门和付钱。

我拥有两个钱包：一个旧的Gucci钱包，装卡和零钱；一个新买的限量版钱包，主要装大额钞票。闺蜜在找零钱时喊道：“你那个有没有2块钱的零钱？……两个Gucci里面都没有！”这句话被游戏语音里的男孩听到了。

他立刻反问：“哎，你也喜欢用这些名牌？”我回答：“是的呀，我是一个很虚荣的女孩。”

我后来反思，他最终愿意发送大额红包，很大程度上是因为这个无意中的对话，**展示了我对名牌的喜好和相应的消费习惯**，即无意中展示了“高价值”。他可能因此认为我是一个在生活方面比较大方、有品位的女孩。

随后，我们简短聊了约3分钟关于奢侈品的话题（例如某款限量版包包），这进一步强化了这种高价值印象。这解释了为什么有时你只收到对方20、50元的小红包时会不高兴——核心问题在于你没有向对方充分展示出自己的高价值。

**价值决定待遇**。如果对方知道你日常用的东西价值一两千，他就不太敢只送你一两百的东西。在他心中，对你的价值有一个衡量和平衡。就像如果你的男友习惯穿300元一条的CK内裤，你不可能买10元三条的内裤送他。道理是相通的。

所以，想让对方给你发大红包，前提是**你自己必须具备并展示出相应的高价值**，让对方高看你。

## 第五步：提出要求与测试对方

那天我用的一个具体话术是：“那你给我发10个红包吧，一个有5.3块就行。” 这句话是**故意说的**，目的是测试他的反应和诚意。他最终每个红包发了199元，这出乎我的意料。

这个结果验证了之前无意展示的高价值起了作用：他觉得不能给我发太小的金额，否则会被我看轻。同时，这个方法也能反向测试出对方的消费水平和档次。

## 第六步：制造悬念与强化吸引

为什么他之后会不停地想加我好友、找我？原因可以总结为以下几点：
1.  **声音展示**：让他对我的初步印象不差。
2.  **品味展示**：通过名牌话题，塑造了有品位的形象。
3.  **情绪满足**：满足了他的大男子主义和被崇拜的需求。

当这些点聚集在一起，吸引力就形成了。游戏结束后，他提出加微信。我同意了，但**没有立刻通过好友申请**。他在游戏里不断邀请我再玩一局，我拒绝了，并说：“要不然我们明天玩吧，刚好明天给你讲个更有意义的事情。”他问是什么事，我回答：“太晚了，我要睡了，明天见。”

这句“明天见”之后，我**有6到7天没有登录游戏，也忘了通过他的好友申请**。这种“不理会”反而激发了他的好奇心、新鲜感、追逐欲和占有欲。他会想：为什么约好了“明天见”，却一直没上线？当他和别的女孩玩游戏时，也会想起我。这就是利用男性的心理特点来强化吸引。

## 第七步：识别与反制对方的“套路”

需要指出的是，这个男生本身也是高段位的。他后来将我设为“星标好友”并截图给我看，这很可能是一种“反套路”，意在展示他对我的特别关注，并观察我的反应。

他发送10个199元的红包，同样可能是一种测试。**如果我全部收下，在他心中，我可能就被标上了“2000元可以搞定”的价码**。但我只收了一个，并说：“剩下的我就不收了。” 这给他传递的信息是：我对2000块钱不屑一顾，我是一个价值更高、更难以搞定的女孩。

这反而进一步刺激了他的征服欲。他会想：到底多少钱才能打动她？是2万，20万，还是更多？这种“得不到”的感觉会让他更想“得到”。

## 核心要点总结与警示

本节课我们一起学习了如何通过一次游戏社交，综合运用多种技巧吸引异性并获得物质付出。我们来总结一下核心要点：

1.  **声音修饰**：与目标异性交流时，有意识地修饰你的声音，提升听觉吸引力。
2.  **观察与冷读**：细心观察对方行为（如保护欲），“冷读”出其内心核心需求（如英雄主义），并投其所好。
3.  **提供情绪价值**：通过赞美、示弱、寻求保护等方式，持续满足对方的心理需求。
4.  **展示高价值**：在互动中，自然、无意地展示你的品味、生活方式或消费层次，让对方感知到你的高价值。
5.  **制造悬念**：运用“延迟满足”、“明天见”等话术，结合暂时的“消失”，激发对方的好奇心与追逐欲。
6.  **反套路思维**：警惕对方可能进行的价值测试（如发送特定金额红包）。不要轻易接受所有馈赠，以免被对方“标价”。你的价值应由你自己定义，而非对方的一次馈赠金额。

**最后的重要警示**：永远不要在自己的社交圈（如朋友圈）展示他人给你的转账记录。这等同于**给自己明码标价**。无论金额大小，都是在降低自己的无限可能性，让自己变得被动。记住，你是无价之宝，你的价值不应由任何一次转账来定义。

请反省一下，在你的生活中，是否曾因为急于求成，而不小心给自己标了一个“小价格”，导致后期陷入被动？是否已经将本节课所讲的关于好奇心、冷读、追逐感等方法运用起来？希望本节的案例分析能给你带来启发。
