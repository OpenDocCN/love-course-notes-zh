# 韩允格（三套合集）：韩允格999（完整版）：１：做个吞金小白兔 🐰💰

在本节课中，我们将要学习如何通过建立个人原则与自我管理，来构建稳固的自我价值体系。课程将分享从个人经历中提炼出的核心生活智慧，帮助你成为一个既柔软又强大、既顺从又坚定的“吞金小白兔”。

## 课程前言

我一直在思考应该分享什么内容。我所认识到的东西，可以分为“道德正确”和“生活正确”两个层面。

我相信你们想听的并非市面上那种教条式的知识点。你们想听的是我的故事。我的人生成长经验中，包含了所有的人生智慧。

你们知道我的情况。我27岁离婚。我前夫有一些不良嗜好。我给了他三次机会，但他没有把握住。我是对自己特别狠的人。

我一直强调：**女人不狠，地位不稳**。这个“狠”指的是对自己的“狠”。它体现在多个方面，例如容貌的保养、身材的管理、情绪上的管理、事业上的规划，以及存钱不乱花钱。这些都属于自我管理。女人必须对自己够狠，才能一步一步接近目标。

当我发现与前夫的日子实在过不下去时，我痛定思痛。他除了长得帅和能提供情绪价值外，我看不到其他希望。离婚时，我经历了自断手腕般的痛苦，但我熬过来了。因为我知道不断不行，日子过不下去了。

27岁那年我离婚了。离婚前，我认为我做对了几件事情。我想要慢慢与你们分享。

总的来说，在这个前言部分，我希望你们能够做到：女人一定要建立属于自己的城堡。只有自己拥有城堡，才能成为女王般的人物。否则，任何时候腰板都挺不起来。

我希望今天所有我的宝贝们，能够做到以下几点。

以下是核心原则列表：
*   **柔软又强大。**
*   **顺从又坚定。**

送你们一个词：我希望你们全部都能做我的**吞金小白兔**。这是我对你们的期待。

为什么是“小白兔”？因为看起来纯洁，让人怜爱。为什么是“吞金”？意思是懂得对自己好一点。当你把所有好东西都给自己时，别人也不会轻视或怠慢你。

这是我们的序言。
