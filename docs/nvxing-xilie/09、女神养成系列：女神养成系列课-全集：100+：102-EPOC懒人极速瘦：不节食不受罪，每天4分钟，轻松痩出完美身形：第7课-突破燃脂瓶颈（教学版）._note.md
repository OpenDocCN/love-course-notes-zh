# 女神养成系列：09：突破燃脂瓶颈（教学版） 🚀

在本节课中，我们将要学习如何突破燃脂与体能的瓶颈。这是Level One阶段的最后一课，我们将通过五个全身性训练动作，特别是被称为“燃脂之王”的波比跳，来挑战极限，为进入更高阶段做好准备。

---

上一节我们介绍了课程的目标，本节中我们来看看今天的神秘礼物——“燃脂之王”波比跳。这个动作能调动全身大部分肌肉群，高效燃烧脂肪，并帮助你突破当前的体能极限。

## 动作一：交替支撑

首先，我们来学习第一个动作：交替支撑。这个动作能有效训练核心肌群、髋部、臀部以及手臂和肩部的力量。

以下是动作要领与步骤：

1.  准备一张垫子，俯卧在垫子上。
2.  选择支撑方式：
    *   **跪姿支撑**：用膝盖支撑身体（适合初学者）。
    *   **直臂支撑**：用脚尖支撑身体（强度更高）。
3.  支撑时，确保头、背、臀、脚在同一直线上。避免腰部塌陷或臀部过高翘起。
4.  双腿并拢，感觉向中间用力夹紧。腹部持续发力，肩膀远离耳朵。
5.  从肘支撑开始，单手交替伸直变为直臂支撑，再换手循环。公式表示为：**肘支撑 -> 单侧直臂支撑 -> 还原肘支撑 -> 换侧重复**。

现在，请跟随示范一起练习这个动作。你可以根据自身情况选择适合的支撑方式。

## 动作二：俯身登山跑（慢速版）

完成了交替支撑，我们激活了核心与肩部。接下来，我们进行第二个动作：俯身登山跑。今天我们将以慢速完成，以启动更多深层肌肉。

以下是动作要领与步骤：

1.  以直臂支撑姿势开始，手臂与地面垂直，肩部主动发力。
2.  收紧腰腹，避免塌腰或弓背。脚尖支撑，双腿并拢夹紧。
3.  保持髋部稳定，不要左右摇晃。
4.  缓慢地将一侧膝盖尽力向上抬起靠近胸部，感受腹部收紧。
5.  缓慢还原后，换另一侧腿重复。配合呼吸：抬腿时呼气，还原时吸气。

这个动作能深层刺激肩部和核心，高效燃烧卡路里。请跟随示范一起慢速练习。

## 动作三：空中脚踏车

前面两个动作主要启动了上身和核心，现在我们来针对下腹部进行训练。第三个动作是空中脚踏车。

以下是动作要领与步骤：

1.  仰卧在垫子上，双手放于身体两侧以保持稳定。
2.  将双腿抬起至“桌边高度”：大腿与地面垂直，小腿与地面平行。
3.  **基础版**：头部可放松置于垫上。
    **加强版**：将肩部卷离地面。
4.  腹部持续发力，保持身体稳定。想象双脚在交替蹬一堵墙。
5.  一腿向前蹬直时，另一腿保持弯曲。交替进行。注意收腿时，大腿与身体的夹角尽量不小于90度。
6.  配合呼吸，每交替一次完成一次呼吸（例如：蹬出时呼气，交换时吸气）。

这个动作能塑造腹部线条并消耗更多热量。请跟随示范一起练习。

## 动作四：开合跳

前三个是垫上动作，后两个需要站立完成。第四个动作是开合跳，它将作为波比跳之前的心肺热身。

以下是动作要领与步骤：

1.  站立，保持头、背、臀、脚在同一直线上的良好姿态。
2.  跳跃时，双脚向两侧打开，同时双手向上在头顶击掌。
3.  再次跳跃，双脚并拢，双手还原至身体两侧。
4.  落地时，用脚尖过渡到脚跟，膝盖微屈以缓冲。
5.  配合呼吸：打开时吸气，并拢时呼气。

现在，请跟随示范一起练习几次，为接下来的核心动作做好准备。

## 动作五：波比跳（基础版）

最后，我们进入今天最重要的环节——燃脂之王“波比跳”。为了打好基础，本节课我们学习其最基础的分解动作。

以下是动作分解与步骤：

1.  **深蹲**：站立，双脚与肩同宽，脚尖微微向外（如钟表11点05分方向）。下蹲，确保膝盖方向与脚尖一致，双手撑于身前地面。
2.  **后跳支撑**：双脚依次向后跳，伸直身体，进入直臂支撑状态。此时身体应成一条直线（平板支撑姿势），避免塌腰或撅臀。
3.  **收腿还原**：双脚依次向前收回，**务必确保每次收回时脚跟站稳地面**。还原至深蹲姿势。
4.  **起身**：从深蹲姿势站起，完成一次动作。

波比跳融合了深蹲、俯卧撑和跳跃三个元素，能同时锻炼腿部、臀部、核心、背部、手臂和胸部，因此燃脂效率极高。由于动作复杂，请仔细观看分解示范并跟随练习。
