# 尧西情感：06：高级课程：如何通过争吵增进感情

在本节课中，我们将学习如何将情侣间的争吵转化为增进感情的契机。我们将分析争吵的本质、核心原则、具体禁忌，并学习一系列实用的话术技巧，帮助你在冲突中保持理性，促进双方的理解与亲密。

## 全面理解争吵

上一节我们介绍了课程主题，本节中我们来看看争吵的本质。

心理咨询中有一个“10分之90原则”。这个原则指出，当下引发你情绪反应的事件，可能只占10%的原因；而90%的反应，实际上源于过往经历和情绪的积累。特别是在两性关系中，争吵往往不是由单一事件引起，而是以往矛盾积累加上当前导火索共同作用的结果。

在人际关系中，存在两种力量。一种是自我安全与信任，带来平和、接受、理解与包容。另一种则是因为自我不够安全、不够自信而产生的防御心理。因为不愿意接受“自己不好”，所以容易在冲突中采取防御姿态。

既然我们要学习“争吵”这门学问，第一步就是从理解的角度全面分析它的概念。很多时候，当下的争吵情形触发了你过往的相似经历。如果一个人人格平和，没有太多挫折经历，经常受到肯定，心态就会比较好，更容易客观看待事情。反之，如果一个人长期被批评、不被信任，当伴侣指出其问题时，他的自动思维会是“我又被批评了”、“他不喜欢我”，其内心真实感受是“我自己不够好”。这实际上是重叠了过去的不良反应经历。

因此，对于冲突的基本认识是：**冲突并不代表你不够好，只是你触碰到了对方的“扳机”**。冲突表明了沟通的重要性。许多冲突都源于沟通不良：你的本意是一种理解，但表达出来给对方的感觉可能是另一种，由此产生误会。如果沟通不畅，就容易引发冲突。所以，你需要尝试理解对方的过去，明白他为何会如此反应。

## 争吵的原则与禁忌

理解了争吵的深层原因后，我们来看看在具体争吵中应遵循的原则和必须避免的禁忌。

以下是情侣争吵时的几大禁忌：

1.  **不上升到“爱不爱”的程度，不许提分手**：争吵时最忌讳突然质问“你是不是不爱我了”，或脱口而出“分手”。这种话非常伤感情。争吵应就事论事，提出自己的意见，不能上升到人身攻击（如“你怎么这么刻薄”）。人身攻击既伤感情，又解决不了问题。
2.  **不回避、不扩大，限定时间与主题**：回避冲突实质上是另一种对抗，显得不自信和无奈。如果情绪激动，可以采取“暂时撤离”的策略，或用幽默化解。可以尝试说：“你说吧，我听你说。”或者约定：“我们先吵20分钟，你先说，我后说。”要搞清楚，引发争吵的是对方让你不爽的态度，还是具体行为。很多时候，态度差异源于看问题的角度不同，彼此换位思考有助于解决问题。
3.  **不冷战**：争吵后不要避而不谈。应尽快给出合理的交代和解决方法，这样双方都能更轻松。吵完架要尽快和好，对方给台阶时要适时下来。如果是自己错了，要真诚道歉。例如可以说：“亲爱的，下次我会注意。谢谢你告诉我。”女生道歉并不丢脸，反而显得善解人意。切忌争吵后删除联系方式或扭头就走，更不要一吵架就回娘家。最好能做到“今日架，今日毕”。
4.  **幽默化解**：在争吵中注入幽默感是高级技巧。例如，可以说：“你生气的样子好帅啊。”或者说：“气死我了，要不是你长得帅，我肯定三天不理你。”用撒娇或调侃的方式化解对方的怒火，能让气氛瞬间缓和。脸皮可以“厚”一点，不要过于敏感。
5.  **不要翻旧账**：牢记“10分之90原则”，不要指责对方过往的错误。陈述自己的感受时，建议使用“我-信息”沟通模式：**“当……（描述具体行为）的时候，我感到……（表达感受），因为……（说明影响）。”** 例如，把“你怎么又不给车加油！”换成：“当我发现汽油快用完时，我觉得很着急，因为这样我明早可能会迟到。” 多说感受，少说指责。争吵升级往往是因为翻旧账和贴标签。
6.  **真诚道歉**：如果你需要道歉，切记态度一定要真诚。不要说“我错了还不行吗？”这类敷衍的话。可以认真地说：“这件事是我不对，对不起。”或者“我仔细想了，今天这件事我做得不好。”
7.  **不要乱发誓**：为了让对方原谅而轻易发誓“我保证再也不这样了”，往往难以做到。特别是女生记性很好，如果下次再犯，矛盾会升级。可以说“我会努力去改”，而不是把话说绝。
8.  **安抚对方情绪**：当对方生气时，想哄他/她，第一句话可以说：“亲爱的，冷静一下，你是个理智的人。”然后看着对方的眼睛，或温和地拥抱他/她。接着问：“你怎么了？有什么想法想和我分享吗？”对于女生，可以这样哄男朋友：“你生气的样子都这么迷人，真拿你没办法。”然后可以说：“我理解你心里不好受。现在说出来感觉好点了吗？我爱你。”这些话术能有效平息争吵。
9.  **学会接受差异**：接受对方与自己不同的观念和方式，是爱情中的必修课。争吵能带来成长，让你学会接纳，让心胸变得更宽广。既然选择了对方，就要在享受其优点的同时，也接纳其缺点。
10. **少用敏感词汇**：争吵最能体现一个人的教养和风度。切记不要说侮辱性或过于敏感的词汇，这有利于双方保持理智、清晰的沟通。在良好的氛围中，才更容易解决问题。

## 争吵中的实用话术技巧

掌握了原则与禁忌后，我们学习一些能让争吵“化险为夷”甚至“增进感情”的具体话术。

以下是四招实用的话术技巧：

1.  **偷梁换柱，巧换概念**：情侣吵架多为小事（如黄瓜横切还是竖切）。过分争执对错容易导致僵局。此时可巧妙绕开问题本身。例如，对方责怪你黄瓜没按他的方式切，你可以说：“我已经按你说的切了，你要不要吃？难道你崇高的人生理想，就因为一根黄瓜放弃了吗？”关键在于发挥发散思维，把小事引申到其他对方在乎的层面（如健康、共同的理想），用幽默化解对立。
2.  **金蝉脱壳，花式甩锅**：如果对方逻辑清晰，你可以优雅地把“责任”转移，同时显得大度。例如，对方指责你撕厕纸没沿虚线，你可以说：“我沿虚线撕了呀，是这卷纸质量不行。都说了别买B牌，要买A牌。你看，无良商家害得我们都吵架了。不过这也不能怪你，错的是这个社会。”这样既指出了问题可能不在自身，又体现了包容，让对方自惭形秽。
3.  **传音入密，震耳发聩**：争吵时，除了讲道理，有时要从气势上引导对方意识到“我们在吵架”。例如，当对方音量升高，你可以说：“你是在吼我吗？你现在是要跟我比谁声音大吗？”这样直接把焦点从具体事情转移到“争吵行为”本身，促使对方冷静下来反思自己的态度。
4.  **口嫌体正直（心口不一）**：这是杀伤力很强的一招，核心是“用甜言蜜语化解攻击”。当对方说：“你别以为你长得漂亮/帅就可以无理取闹！”你可以回应：“我长得漂亮/帅为什么不能无理取闹？而且你都不知道，你无理取闹的样子特别可爱，你自己都没发现你有多可爱。”此话一出，争吵很难继续。

## 如何避免不必要的争吵

最后，我们来看看如何从根源上减少伤害感情的争吵。

虽然理性争吵能增进感情，但能避免则尽量避免。小吵怡情，大吵伤身。

以下是维护感情、避免恶性争吵的几个要点：

1.  **给予尊重，换取爱意**：给对方越多的尊重（尊重他的生活、社交、爱好），他通常会回报越多的爱。即使有不喜欢的点，也不要严厉指责。把他当孩子，告诉他“玩得开心，有事乐意倾听”。给予足够的信任和尊重，他会更愿意对你坦诚。这叫“抓大放小”，不要事事斤斤计较。同时，对他为你做的任何小事都要表示感谢，这是他继续付出的动力。
2.  **不吝啬崇拜与赞美**：他是你的男朋友，就把他当成全世界最好、最帅的人来夸。夸他的外貌、身材、衣品、爱好。告诉他，你有多幸运遇到他，为他感到骄傲。这能极大满足他的存在感和舒适感，让他更愿意呵护这段关系。
3.  **像哥们儿一样平等相处**：除了恋人，也要把对方当作可以平等交流的哥们儿。认真倾听，探讨问题可以持有不同见解，但不要经常指责对方缺点。聊天要轻松愉快，除了情话，最重要的是像朋友一样平等相处，不强迫对方同意自己的观点，不要求对方按自己的方式生活。感情是相互的，在要求对方之前，先看看自己做得如何。

**一个警示案例**：女生生病了，男生因加班无法立刻照顾。女生说：“你不来就是不爱我，12点前不来就分手。”男生最终疲惫地说：“那我们分手吧。”这个故事告诉我们，无底线的“作”和用分手威胁，是消耗感情最快的方式。男人心里有一张“积分卡”，每次恶性争吵都会扣分，分数扣完，关系也就结束了。感性的争吵产生矛盾，理性的争吵（或巧妙化解的争吵）才能增进感情。
