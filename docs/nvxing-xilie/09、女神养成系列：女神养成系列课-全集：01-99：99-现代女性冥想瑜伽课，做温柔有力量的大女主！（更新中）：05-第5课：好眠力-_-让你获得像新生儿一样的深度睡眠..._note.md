## 睡眠的现代困境与改善原理

上一阶段我们梳理了与外界的关系，并进行了放松练习。本节中，我们将焦点转向自身，探讨睡眠问题。

睡觉是每个人日常必需的活动。人的生命约有三分之一时间在睡眠中度过。然而，睡眠在今天已成为一个普遍问题。一份名为《中国互联网用户睡眠环境白皮书》的报告指出，2019年，中国网民中有67.42%的人认为自己存在睡眠问题，表现为白天困倦、夜晚难以入睡。

我曾是失眠者之一。因工作压力、环境及家庭等多方面原因，我每晚难以入睡，有时需依赖安眠药。但药物刺激后，白天状态更差，更加疲惫。通过冥想和瑜伽练习，我的睡眠状态得到了显著改善。现在，我在睡眠上已无障碍。

这背后的原理与我们大脑中的**松果体**有关。松果体能分泌**褪黑素**，这种激素帮助我们进入正常的睡眠状态。幼年时松果体体积较大，能分泌大量褪黑素。随着年龄增长，松果体会逐渐萎缩并钙化。这解释了为何婴儿睡眠多而老年人睡眠少。

美国一项针对冥想对松果体影响的研究表明，经过8周、每天30分钟的冥想练习后，人的失眠症状会有所改善。当然，睡眠不能代替冥想，冥想也不能代替睡眠。但冥想练习能改善我们的睡眠状态，增加深度睡眠的时间。

因此，在今天的声音瑜伽体式中，我将介绍有助于睡眠的体式。同时，在今天的冥想引导中，我会运用呼吸觉知的技巧，引导你进入良好的冥想状态。

## 改善睡眠的瑜伽体式练习

我们今天的练习旨在改善睡眠，主要针对下半身的血液循环。以下是具体的体式引导。

### 体式一：勾脚金刚座

我们今天的体式从金刚座开始。金刚座是一个非常重要的体式。有一个瑜伽老师都知道的秘密：如果吃多了，想加快消化，以金刚座姿势坐20分钟即可见效。

以下是具体步骤：
1.  身体向前，脚趾回勾。
2.  坐骨坐在脚跟上，这是勾脚的金刚座。
3.  如果膝盖或脚部比较敏感，可用抱枕或厚枕头垫在膝盖下方。
4.  调整坐姿，让坐骨坐在脚跟上，脊柱向上延展。
5.  确保骨盆处于中立位置，不前倾也不后倾。
6.  十个脚趾需均匀受力，全部接触地面。
7.  如果某只脚受伤，可将受伤的脚打开放在旁边，用另一只脚练习，但停留时间需根据舒适度自行决定。

调整好坐姿后，在此体式中停留一分钟。

在这个体式中，你需要感受身体内部的变化。感受脚趾的压力是否在承受范围内。手部和脚部是身体的末端和神经末梢，通常难以运动到。若想改善睡眠，需让血液循环更通畅。此体式能对肾经和膀胱经产生一定刺激。

慢慢睁开眼睛，身体向前，将双脚平铺在垫子上，膝盖落下。在此调整两三个呼吸，感受身体的回响。

### 体式二：绷脚金刚座

接下来我们做绷脚的金刚座。它与金刚座类似，但为了给脚背和脚两侧施加更大刺激，会让膝盖稍微抬起。

以下是具体步骤：
1.  取一条毯子或毛巾，卷成卷，垫在膝盖下方，使膝盖微微抬离地面。
2.  依然让坐骨坐在脚跟上。
3.  保持脊柱直立与骨盆稳定。

在此同样停留一分钟。

金刚座是一个看似简单、实则强烈的体式。大家需根据身体承受能力选择合适的时间。如果脚部压力过大导致呼吸急促，可停留10秒、20秒或30秒。随着练习加深，若觉得此体式效果显著，可适当延长停留时间。

现在慢慢退出体式。身体向前，将两个小腿放到一侧，坐下来。放松脚趾，感受身体的回响。

### 体式三：倒箭式

接下来的倒箭式很适合睡前练习。现在建议你找一面墙，或床头有靠背的地方。

以下是具体步骤：
1.  躺下来，双腿伸直搭在墙上或床帮上。
2.  双手自然垂落身体两侧，大拇指朝向地面或床面。
3.  两个肩胛骨尽量向中间收拢，躺下时不要驼背。
4.  腰部后方的骶骨（用手可摸到的平坦处）需完全贴平在床上或地上。

在此体式中可停留较长时间，因为它非常适合站或坐了一天的我们。现在闭上眼睛，慢慢感受身体的变化。

### 体式四：仰卧束角式

接下来进入仰卧束角式。

以下是具体步骤：
1.  弯曲膝盖，向两边打开。
2.  如果在墙边或床边，双脚可放在墙或床上。
3.  吸气，呼气时让膝盖再向两边打开一些。
4.  保持双手自然垂落，后背和骶骨平贴地面或床。
5.  闭上眼睛，自然缓慢地呼吸，倾听身体内部发生的变化。

当你感到非常舒适并想退出体式时，将双手放在大腿内侧以保护膝盖，然后合上双腿。请注意退出方式：将右手举过头顶，身体转向右侧卧。依然闭着眼睛，用左手轻轻推起身体，慢慢坐起。

### 体式五：摊尸式变体

从今天开始，我将介绍一些摊尸式的变体，它们都具有很好的修复功能。即使你今天很累不想做体式，也可以用这些修复性变体来休息。

今天的摊尸式变体需要一个抱枕。如果没有，可用稍大、稍厚的枕头代替。

以下是具体步骤：
1.  躺在垫子或床上，将抱枕或枕头垫在膝窝下方。
2.  双膝打开，比骨盆略宽，双脚踩地。
3.  慢慢躺下，调整姿势。
4.  让两个肩胛骨向中线收拢。
5.  由于将在摊尸式中停留5分钟，建议盖一条毯子。

在这个摊尸式变体中，我们的腹股沟位置能得到完全放松。这个位置平时因走路或久坐很难放松，血液循环较差。在此，双腿获得支撑，能感受到大地的滋养。

现在慢慢收回意识。轻柔地活动一下手指和脚趾，将意识唤回。

退出体式时请注意：将右手举过头顶，然后将膝窝下的抱枕或枕头挪到身后。身体倒向右侧卧。调整几个呼吸后，缓慢地用左手推起身体。这就是我们今天的摊尸式练习。

## 呼吸觉知冥想引导

从这节课开始，我将引导大家练习呼吸的觉知。接下来，我将用呼吸觉知技巧带大家进入今天的冥想练习。

我们用了四周时间寻找身体全然放松的感觉，然后进入静默。大家会发现，要处于静默之中并不容易。我们需要获得全然的觉知。今天，我们就从觉知呼吸开始。

呼吸非常重要，但我们似乎很少关注它。呼吸与整个身体、心理进程相关，同时也与我们的神经系统和大脑相关。有许多技巧可以调节呼吸，引导我们建立对呼吸的觉知，从而唤醒更多觉知。

呼吸觉知技巧非常容易。我们将变得对呼吸有所觉知。不仅你的呼吸能放缓，觉知也会更深。

此刻，你的呼吸有三种模式：快的、慢的、乱的。这三种模式总在发生，不是同时，而是逐一出现。即使有时你觉得舒适，呼吸也会时快时慢，有时会受到干扰。

我们从小听到一种说法：呼吸应该是慢的。每当呼吸变快，我们就开始担心。这是一种束缚，你需要打破。呼吸快一点不是坏事。例如跳舞或运动时，呼吸就是快的。呼吸变快或被打乱，不代表是坏事。

每当你感受呼吸时，如果呼吸快了，你应该继续去感受加快的呼吸；如果呼吸被打乱了，你应该继续去感受被打乱的呼吸。不应该做任何改变。在许多宗教传统中，你持续不断地去知晓：如果呼吸被打乱了，就持续感受，不要尝试改变。你的呼吸会有变化。

现在，让我们以最舒适的方式躺下。只需要观察和感受你的呼吸。

首先，去找到你的呼吸状态属于哪个类型。如果最初你的呼吸是快速的，那么就去感受这种快速的呼吸，不要做任何改变。如果你能正确跟随引导，就会意识到这是非常有帮助的。

如果呼吸模式自动改变了，你应该持续感受这种新的呼吸模式，而不是之前的。让变化去发生。不管变多少次，都不要担心。每当它改变时，请记住去感受新的模式，而不是之前的。

如果你的呼吸是乱的，就去感受这种乱的呼吸。如果它变慢了，就去感受慢的呼吸。倘若它再次变快，就感受快的呼吸。我们没有去改变自己的呼吸模式。如果它自然发生改变，我们就持续去感受这种改变。

你的呼吸可能会变化很多次。你需要做的只是去感受。每当它有改变，不管是哪种模式，只要去感受。允许你的呼吸变化。不管变多少次都不用担心。你需要做的只是去感受，不要主动去改变它。

你逐渐意识到，你的呼吸正以最自然的方式慢慢变化。如果下一秒你的呼吸变快了，那就感受它。快的呼吸也没什么可害怕的。现在，你的呼吸可能又变慢了。即使在这个阶段，呼吸受到了干扰，你就让它乱，不要担心。在这种情况下，我们去感受，持续感受这被打乱的呼吸。

你有一个觉知让你发现呼吸变慢了。这种慢是哪种慢呢？这种慢的呼吸充满了活力，也就是说，它充满了觉知。因此，你的呼吸被觉知包围着。当你呼吸时，尽管呼吸通过大脑起作用，但它自然而然地变得更加深入了。

现在你可以去观察，在整个身体之内发生了什么。有某种感觉在体内。你的呼吸持续是慢的。它可能受到干扰，那么就持续感受这受到干扰的呼吸。这缓慢的呼吸中也有觉知。这缓慢的呼吸在你身体之内创造着一种感觉。这是一种我们感受到但无法用语言描述的感觉。

你要做的就是持续感受身体的内部。或许在体内你感觉到了什么。这种感受、感觉和缓慢呼吸的觉知是相关的。感受内在觉知。这些通过我们的身体发生了。现在你要做的就是持续这一切。

在整个身体之内发生了什么？某种觉知就在那儿。在你的身体之内。在整个身体之内。这种感觉向着你的身体。你无法描述，但是你可以感受得到。继续让这个说不出的感受在你身体之内发生着。随着缓慢的呼吸，你觉知到整个身体的放松感和舒缓感越来越深入。一种无法描述的感受，蔓延到你整个身体。我们持续地去感受它。

现在，静默。只有静默。这是一种非常轻微的感受，在身体内部发生了。在身体深深的内在，有一种感受发生了。我们只是持续地去感受。在内部去感受这种舒适感，并继续去观察这种舒适感。如果你去感受它，这种舒适感会越来越深。那么会来到最深的时间。当这种感受达到最深处时，静默又发生了。这是深深的静默。请你继续留在这儿。如果没有什么特别的发生，那么观察它，持续在静默之中。
