# 女神养成系列：03：体态管理与减压瑜伽随堂练习 🧘‍♀️

在本节课中，我们将学习一套完整的减压瑜伽流程，包含呼吸法、体式练习与冥想。通过交替呼吸、脊柱灵活、核心强化及深度放松等练习，帮助管理体态、释放压力并提升内在气质。

## 概述
本节课的练习旨在通过系统的瑜伽动作与呼吸控制，灵活脊柱、促进能量流动并达到深度放松。我们将从调息开始，逐步进入体式练习，最后以冥想结束。

## 呼吸法练习：左右鼻孔交替呼吸法 🌬️
上一节我们介绍了课程的整体安排，本节中我们来看看如何通过呼吸法来平衡身心。左右鼻孔交替呼吸法，也称为经络调理呼吸法，能帮助平衡神经系统。

**做法如下：**
1.  以至善坐或简易坐姿开始，左手呈智慧手印放于膝盖上。
2.  右手抬起，指尖向上。用右手大拇指按住右鼻翼，关闭右鼻孔。
3.  用左鼻孔吸气，感受气息进入。
4.  吸满后，用右手无名指或小拇指按住左鼻孔，同时松开大拇指。
5.  用右鼻孔将气完全呼出。
6.  继续用右鼻孔吸气。
7.  关闭右鼻孔，松开左鼻孔，用左鼻孔呼气。

**循环模式为：** `左吸 -> 右呼 -> 右吸 -> 左呼`。

以下是练习要点：
*   保持呼吸深长、均匀。
*   若难以稳定，可采用计数法：吸气数4下，呼气数4下。
*   完成最后一次呼吸后，深吸气，屏息并收缩根锁，凝视眉心，然后深呼气放松。

## 体式练习一：苏菲旋转与猫牛式 🐱🐮
掌握了平衡呼吸的方法后，我们进入体式练习部分。首先通过苏菲旋转来灵活脊柱。

**苏菲旋转做法：**
1.  保持坐姿，双手握膝，背部直立，下腹微收。
2.  吸气时，脊柱向前画圈；呼气时，向后画圈。
3.  动作初期应缓慢，保持胸腔打开，避免含胸或低头。
4.  完成一侧后，吸气回正，换反方向练习。
5.  结束时，深吸气并屏息，深呼气后自然放松。

接下来是猫牛式，它能进一步温暖和活动脊柱。

**猫牛式做法：**
1.  来到四足跪姿，双手在肩下，膝盖在髋下。
2.  吸气时，尾骨向上，腰部下沉，胸腔和颈椎抬起（牛式）。
3.  呼气时，尾骨内收，背部拱起，头部放松（猫式）。
4.  可跟随呼吸节奏练习，如：吸气数4下，呼气数4下。
5.  适应后，可逐步加快节奏。
6.  最后一次，深吸气保持，然后深呼气放松，回到坐姿。

## 体式练习二：自我触身与按摩 👐
完成了对脊柱的激活，现在我们将通过一些温和的拉伸和按摩来放松身体。

首先是自我触身体式，它能伸展髋部与背部。

**做法如下：**
1.  从坐姿开始，左腿向前大步迈出，左臀坐于脚跟上。
2.  右腿向后伸展，右髋部下沉。
3.  躯干向前弯曲，前额贴地，双手放于身体两侧，掌心朝上。
4.  保持平衡，自然呼吸，内观眉心，保持约1分钟。
5.  双手放头两侧，推地吸气起身。换另一侧重复。

接下来是针对喉颈和耳朵的按摩。

**按摩方法：**
*   **喉颈按摩：** 单手置于喉部，从脖子底端以打圈方式波浪式按摩至顶端，持续约2分钟，保持呼吸。
*   **耳朵按摩：** 双手用拇指和食指由下至上打圈按摩耳朵。结束时，深吸气，向外轻拉耳朵并屏息10秒，然后深呼气放松。

## 体式练习三：扭转与核心 🌀
经过放松，我们通过扭转和核心练习来强化身体中段能量。

首先是坐姿躯干扭转。

**做法如下：**
1.  保持坐姿，双手在腰后十指交叉，肘部伸直，脊柱挺直。
2.  吸气时，身体向左扭转；呼气时，向右扭转。
3.  以中等偏慢速度进行，头部随躯干转动，持续约2分钟。
4.  结束时，吸气回正，屏息片刻，呼气放松。

接下来是分腿仰卧起坐，锻炼核心。

**做法如下：**
1.  仰卧，双腿尽量分开，双手伸向肩膀上方。
2.  吸气，运用核心力量；呼气时，腹部收紧，躯干抬起，前额找向地面，保持约1分钟，关注眉心。
3.  吸气起身，缓慢落回。再次吸气起身，扭转身体向左腿方向，呼气时腹部靠近左大腿，保持3次呼吸。
4.  吸气回正，呼气落回。换右侧重复。

## 放松与整合练习 😌
在进行了强度练习后，需要通过彻底的放松来整合效果。

**大放松（摊尸式）：**
1.  仰卧，双腿分开与髋同宽，双手放身体两侧，掌心朝上。
2.  依次放松骨盆、脊柱、肩膀、颈部、头部。
3.  自然呼吸，感受身体沉入地面，持续放松约1分钟。

**唤醒练习：**
1.  动动手指、脚趾，转动手腕、脚踝。
2.  深吸气，双手举过头顶伸展，像伸懒腰一样；用嘴呼气放松。重复一次。
3.  右脚踩左膝，双膝倒向左侧，脸转向右侧，感受拉伸。换边重复。
4.  屈膝，脚掌相对摩擦，用搓热的双手捂住双眼按摩。
5.  双手抱膝靠近胸腔，左右摆动按摩背部，然后前后滚动，借惯性坐起。

## 冥想与结束静坐 🕉️
最后，我们通过冥想将意识内收，巩固练习带来的平静。

**冥想准备：**
1.  采用舒适的坐姿（如简易坐），可垫高臀部以保持脊柱长时间挺直。
2.  双手合十呈祈祷式放于胸前。
3.  眼睛微睁，凝视鼻尖，感受脊柱向天空延展。

**冥想引导：**
*   通过均匀的呼吸（例如五吸五呼），释放压力，清理情绪，寻找内在和平。
*   将意识关注在内在与外在的心灵能量感受上。
*   在心中默念积极的肯定语，例如：
    *   我把时间放在能让自己快乐的事情上。
    *   我疼爱我自己。
    *   任何时候我都知道我是最好的。
    *   我做我爱做的事情，财富随之而来。我是美丽的、丰盛的、喜悦的。

**结束：**
1.  深吸气，能量提升，双手合十，屏息凝视眉心。
2.  深呼气，完全放松，双手呈智慧手印落于膝上，保持静默。
3.  再次双手合十于胸前，在深呼吸中感受身体的平衡、开放与内在的深度连接。

## 总结
本节课我们一起学习了完整的减压瑜伽流程。从**左右鼻孔交替呼吸法**（`左吸->右呼->右吸->左呼`）开始，经历了灵活脊柱的**苏菲旋转**与**猫牛式**，进行了深度放松的**自我触身**与**按摩**，并通过**扭转**与**核心**练习强化身体，最后在**大放松**与**冥想静坐**中整合身心，以积极的自我肯定结束。坚持练习有助于管理体态、释放压力并培养宁静气质。
