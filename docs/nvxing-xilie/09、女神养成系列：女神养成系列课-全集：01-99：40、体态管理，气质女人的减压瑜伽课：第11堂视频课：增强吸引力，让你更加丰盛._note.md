## 吸引力法则的原理

上一节我们介绍了课程的主题，本节中我们来看看吸引力产生的根本原因。

为什么有的人一分耕耘，半分收获，有的人却是一分耕耘，多分收获。为什么有的人像向阳花，有的人却像臭百合。为什么有的人在职场中如鱼得水，有的人却孤舟独桨。

我们常将原因归结于外在条件，例如相貌、出生或学历。但吸引力法则指出，我们的内在信念吸引并创造外在的一切。如果你的内心感到贫瘠和匮乏，你会试图用外在的物质来填补，但这无法消除内在的匮乏感。

**吸引力公式**：`内在世界的丰盛 → 吸引外在世界的显化`

吸引力产生于内心的思维和起心动念，形成于独立的人格，并展示在你的观念、信仰、呼吸、欲望和行为举止中。它最终体现在你的外表精神、内在修养和行为道德上。世界上99%的人的吸引力，并非天生，而是通过长期对内心和身体的训练形成的。

## 带来机运与绿色能量的序列练习

了解了原理后，我们进入实践环节。以下是增强吸引力的训练序列，旨在调和内心与身体的能量状态。

### 1. 脊柱弯曲
这个体式能灵活脊柱，促进能量运行。

**练习步骤**：
1.  以延石座（跪坐）姿势坐于脚跟。可在臀腿间垫毯子以确保稳定舒适。
2.  吸气时，脊柱向前弯曲，心中默念“Saa”。
3.  呼气时，脊柱向后弯曲，心中默念“Naa”。
4.  吸气时专注尾骨区域，呼气时专注丹田区域。
5.  持续练习2分钟。
6.  结束时，进行三次“吸-屏-呼-屏”呼吸循环：
    *   深吸气，屏息并收根锁，观想眉心，停留10秒（或力所能及的时间）。
    *   深呼气，屏息并收根锁，停留10秒（或力所能及的时间）。

### 2. 身体下落
此体式能强壮核心力量与臂力，推动能量上行。

**练习步骤**：
1.  向前伸直双腿，双手放于臀部两侧地面。
2.  吸气时，用核心和手臂力量将身体（包括脚跟）提起离地。
3.  呼气时，身体快速下落。
4.  持续练习2分钟。初期若无法抬高，仅做臀部轻微抬离的动作亦有效果。

### 3. 乌鸦式
乌鸦式能强壮下腰背部力量，支持心脏区域。

**练习步骤**：
1.  蹲姿，脚踝有压力可垫毯子。手臂伸直向前，手指交叉，食指伸出指向远方。
2.  目视前方目标，保持脊柱向上延伸，避免塌腰耸肩。
3.  进行深长呼吸或火呼吸，持续2分钟。
4.  结束时深吸气，屏息并从心轮向外投射意图与能量，然后深呼气放松。

### 4. 原地跑步
此练习能快速提升身体能量与活力。

**练习步骤**：
1.  站立，进行原地高抬腿跑步，膝盖尽量抬高。
2.  同时配合向前出拳动作。
3.  可从慢速开始，逐渐加速至快速，持续3分钟。
4.  结束时深吸气，呼气放松。

### 5. 创造潜能莲花式
此体式帮助强壮丹田和下腰背部，控制平衡，促进能量上升。

**练习步骤**：
1.  简易坐姿，膝盖分开。用双手前三指抓住同侧大脚趾。
2.  吸气，用核心力量抬高并伸直双腿，同时背部挺直，双臂伸直。
3.  目视前方，进行深长呼吸或火呼吸。
4.  结束时深吸气，深呼气放松。
    *   **初学者变体**：可先练习单腿抬起，或使用伸展带辅助。

### 6. 扩展心轮式
此体式通过唱诵震动和扩展心轮。

**练习步骤**：
1.  左腿弯曲坐于左脚跟，右脚放于左大腿上（困难者可垫高或腿放下）。双手在肚脐下方呈杯状。
2.  提起横膈膜，收紧腹部。
3.  从心轮区域唱诵曼陀罗：“Ong So Hung”。
4.  持续唱诵约3分钟，感受音流震动。
5.  结束后放松抖动，回到简易坐姿。

### 7. 手臂平举式
此体式增强心轮及上三轮（喉轮、眉心轮、顶轮）的磁场能量。

**练习步骤**：
1.  简易坐姿，双臂向两侧平举，掌心朝上。
2.  集中注意力，观想能量从左手掌流入，经彩虹般的拱形流向右手掌。
3.  进行火呼吸（经期或孕期做深长呼吸），持续2分钟。
4.  结束时深吸气，观想能量拱桥，深呼气放松。

### 8. 向无限顶礼式
此体式刺激脑垂体和松果体，引导能量流向眉心与头顶。

**练习步骤**：
1.  简易坐姿，双手于颈后结智慧手印（手指交叉，女性左手拇指在上）。
2.  呼气时，额头带向地面，默念“Saa”（做不到可垫瑜伽砖）。
3.  吸气时，身体抬起，默念“Naa”。
4.  眼睛始终关注眉心，意念将能量从丹田带往眉心。
5.  持续动态练习2分钟，感受臣服与接纳的心态。

### 9. 手臂升降式
此体式协调身体，提升能量流动速度。

**练习步骤**：
1.  手臂向前伸直，掌心朝下。
2.  配合呼吸，将手臂交替抬升至60度角：吸气上抬，呼气下落。
3.  从慢速开始，逐步加快至接近火呼吸的速度，持续2分钟。注意肩膀放松。
4.  结束时，双臂一起举至眉心高度，观想能量向无限连接，然后呼气落下。

### 10. 与无限连接式
此练习分三步，深度连接更高维度的能量。

**练习步骤**：
*   **第一部分**：双手在头顶上方约10厘米处结智慧手印，掌心朝下。闭眼，眼球向内上方专注顶轮。进行火呼吸2分钟。结束时深吸气，屏息观顶轮，呼气放松。
*   **第二部分**：双手保持原位，改为合十手印，食指伸直向上。继续专注顶轮，进行深长呼吸2分钟。结束时深吸气，屏息，呼气放松。
*   **第三部分**：双手保持原位，指尖相触，掌心张开如帐篷状。继续专注向内向上投射，进行火呼吸2分钟。结束时深吸气，尽可能久地屏息投射，呼气放松。

### 11. 制造绿色能量式
此练习旨在激活心轮的绿色能量。

**练习步骤**：
1.  简易坐姿，手臂在身体两侧弯曲，前臂上抬，掌心朝前。
2.  微微上提丹田，观想心轮的绿色能量。
3.  由心唱诵曼陀罗：“Har Haray Haree Wah Hay Guroo”，持续5分钟。
4.  结束时吸气，呼气，以智慧手印静坐冥想。

## 开发机运的冥想

最后，我们通过一个冥想，破除障碍，创造新机会。

**冥想步骤**：
1.  简易坐姿，手臂放松。双手合十于心轮处，但第二指节略微分开，两大拇指呈45度角向外，不触碰。
2.  眼睛微睁1/10，凝视鼻尖。
3.  以平顺的呼吸唱诵以下曼陀罗，咬字清晰：
    `Aad Such, Jugaad Such, Hai Bhee Such, Naanak Hosee Bhee Such.`
    （意为：真理存在于初始，真理历经时空，真理存在于此刻，真理亦将永存。）
4.  持续唱诵冥想。
5.  结束时，深吸气，深呼气，双手合十于胸前静默，观想内心。
