# 女神养成系列：09：唇颊教程

在本节课中，我们将要学习唇颊妆容的核心知识与技巧，包括口红的分类与颜色选择、唇部结构分析、不同唇形的画法以及腮红的多种打法。课程内容精炼，旨在帮助初学者掌握基础，提升化妆技能。

## 第一部分：口红的分类

上一节我们学习了眼妆，本节中我们来看看唇颊妆容的基础——口红的分类。了解口红的种类有助于我们根据场合和需求选择合适的工具。

以下是常见的口红种类及其作用：

*   **唇膏/润唇膏**：无色，主要作用是滋润唇部，为后续上妆做准备。
*   **唇蜜**：通常用于涂在口红之上，起到提亮和增加光泽感的效果。
*   **润唇啫喱**：作用与润唇膏类似，质地更清爽，用于唇部保湿。
*   **唇膜**：用于夜间或日常护理，深层滋养和修复唇部肌肤。
*   **口红**：质地相对厚重，多为哑光或丝绒质地，色彩饱和度高，是日常使用最广泛的种类。
*   **唇釉**：质地轻薄水润，光泽感强，易于晕染，不易卡唇纹，但饱和度通常不如同色号口红。
*   **唇部遮瑕**：用于遮盖原有唇色，使后续口红的显色更纯粹、更准确。
*   **唇线笔**：用于勾勒清晰唇形，帮助塑造或修正唇部轮廓，尤其适合画轮廓分明的唇妆。

## 第二部分：口红的颜色选择

认识了口红的种类后，我们来看看如何根据肤色选择口红色号。这是打造完美唇妆的关键一步。

口红色系繁多，主要可分为裸色系、红色系、粉色系、橘色系、紫色系和深色系。选择时需考虑肤色冷暖（冷皮/暖皮）和深浅（白皮/黄皮/深色皮）。

以下是不同肤色选择口红的要点：

*   **白皮**
    *   **适合**：明亮鲜艳的颜色，如正红、砖红、烂番茄色等暖调中度色。
    *   **避免**：裸色系和棕色系，否则会显得气色不佳或皮肤暗黄。
*   **黄皮（亚洲人常见）**
    *   **适合**：珊瑚色系、饱和度高的橘红色、暖调红色（如砖红色、烂番茄色）。
    *   **避免**：桃红、紫红等冷色调，这些颜色容易让皮肤显得更黄。
*   **深色皮**
    *   **适合**：裸色系、棕色系、姨妈色、玫紫色等。
    *   **避免**：粉色系和荧光色系，这些颜色在深色皮肤上会显得突兀。

**公式**：选择原则 = **根据肤色冷暖选择色调（暖皮选暖色，冷皮选冷色）** + **根据肤色深浅调整饱和度（浅肤可选鲜艳色，深肤可选沉稳色）**。

## 第三部分：认识唇部结构

在动手画唇妆之前，我们需要先了解唇部的基本结构，这能帮助我们更精准地上妆。

唇部主要结构包括：
*   **唇角**：嘴唇最外侧两端。
*   **唇峰**：上唇中央两个凸起的最高点。
*   **唇珠**：两个唇峰中间凸起的部分。
*   **唇谷**：唇珠两侧的凹陷处。
*   **上唇与下唇**：通常下唇厚度是上唇的1.5倍左右。

画口红时，找准**唇峰**的位置至关重要，它决定了唇形的风格（如欧美妆、咬唇妆等）。

## 第四部分：唇形的分类与画法建议

每个人的唇形都不同，认识自己的唇形有助于选择最适合的画法。以下是几种常见唇形及其特点：

*   **含珠唇**：上唇正中唇珠明显凸起。
    *   **特点**：适合画咬唇妆，能营造楚楚可怜、妆感自然的精致效果。
*   **小翘唇**：自然状态下嘴唇微微上翘，饱满丰润。
    *   **特点**：唇形饱满，适合勾勒清晰唇线，打造性感或可爱的妆容。
*   **嘟嘟唇**：唇峰、唇珠、唇谷分明，上下唇比例标准，唇形标致。
    *   **特点**：属于标准唇形，各种画法都适用。
*   **薄唇**：上下嘴唇均较薄，唇线较长。
    *   **画法建议**：可适当将口红涂出唇线外，以视觉上增加嘴唇厚度，避免使用深色口红勾勒过细的唇线。
*   **厚唇**：上下嘴唇均较厚。
    *   **画法建议**：强调唇线，勾勒出清晰、有型的轮廓，避免模糊边界的画法（如咬唇妆）。

## 第五部分：腮红的打法

腮红和修容一样，都能有效修饰脸型。不同的腮红画法能带来截然不同的视觉效果。以下是几种常见的腮红打法：

以下是不同腮红形状的画法与适用脸型：

*   **椭圆形腮红**
    *   **画法**：微笑找到笑肌，在笑肌上方以打圈方式由外向内晕染。
    *   **特点**：甜美温柔，最日常通用，适合所有脸型。
*   **直线型腮红**
    *   **画法**：从苹果肌开始横向朝太阳穴方向轻扫。
    *   **特点**：能在视觉上缩短脸型长度，适合**长脸型**。
*   **斜线型腮红**
    *   **画法**：从苹果肌（约鼻翼处）开始，斜向上朝太阳穴方向刷扫。
    *   **特点**：能在视觉上拉长、缩窄脸型，适合**圆脸型**。
*   **W型腮红**
    *   **画法**：在鼻梁和两颊间连接画出一个长长的“W”形，两颊颜色可稍重。
    *   **特点**：富有活力，适合特定妆容（如晒伤妆、雀斑妆）。
*   **放射性腮红**
    *   **画法**：从笑肌上方顶点向外放射性延伸，可结合高光。
    *   **特点**：能缩短中庭宽度，放大太阳穴，柔和颧骨，适合**菱形脸**。
*   **爱心型腮红**
    *   **画法**：在笑肌位置以左右来回的方式画出爱心形状的下弧线。
    *   **特点**：可爱甜美，非常有治愈感，适合少女系或特定主题妆容。

**代码**（以椭圆形腮红为例）：
```
1. 工具：腮红刷（建议白色刷毛）。
2. 动作：微笑，找到凸起的笑肌。
3. 手法：刷子蘸取腮红后，在笑肌上方以打圈方式由外向内晕染。
4. 范围：不超过鼻子下方，向上可延伸至眼尾下方。
5. 关键：晕染成“面”，而非“线”。
```

## 第六部分：唇颊实操要点总结

最后，我们通过实操来巩固今天所学的唇颊知识。以下是操作的核心要点总结：

**腮红实操要点**：
1.  **避免画成一条线**：务必以打圈或横扫的方式晕染开，形成自然的红晕面。
2.  **位置不宜过低**：腮红最低点不要超过鼻翼水平线，否则会显得老气、没精神。

**口红实操要点（以全唇妆为例）**：
1.  **注意对称性**：画完后检查上下唇、左右唇角是否对称，可进行微调。
2.  **保证边缘流畅**：可用棉签或遮瑕膏修饰唇线，使轮廓干净清晰。
3.  **确保色彩饱满**：特别是深色或鲜艳色口红，需多次叠加涂抹，直至呈现出饱满、纯正的色调。
4.  **基础步骤**：先涂润唇膏打底 -> 勾勒唇峰和唇线 -> 填充颜色 -> 抿嘴使颜色均匀 -> 检查并修饰。

**其他唇妆**：
*   **咬唇妆**：仅涂抹嘴唇内侧，然后抿嘴或用手指向外晕染开。
*   **果冻唇**：完成口红后，叠加一层透明唇蜜或同色系光泽唇釉。
