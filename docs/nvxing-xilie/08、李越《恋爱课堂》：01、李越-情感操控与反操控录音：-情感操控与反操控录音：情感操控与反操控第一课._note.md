# 恋爱课堂：01：情感操控与反操控概述

在本节课中，我们将要学习情感操控的基本概念、其存在的现实背景，以及学习相关知识的初衷与目的。我们将探讨为何需要了解情感操控，并介绍本课程的整体结构。

---

很多人认识我，因为我从事情感工作多年。我做情感工作至今，有时仍无法相信世界竟是如此模样。

人心竟是如此模样。我早年便知人心软弱，但也难以想象，在21世纪的今天，那些追求独立自主的个体，稍加操作便可被人玩弄于股掌之间。

直播时有人来提问，我觉得问题过于简单。他们常问：被伤害时，如何让伤害者停止伤害，甚至反过来对自己好。

这些人未曾想过，一个人在已是既得利益者且无需付出时，为何要回馈你。我们渴望世界美好，但现实摆在面前，告诉我们世界并不完美。

基于网上常见的想倒利事件、家暴事件或情感操控事件，也基于近期的直播与咨询答疑，我们决定推出这门课程。

这门课程讲述情感操控。我知道课程初衷会被如何理解，也知必定有人跳出来骂我们，指责我们教坏他人、带坏社会风气，不去弘扬正能量反而教唆他人变坏与操控。

我认为这样指责的人可怜又可悲。我举一个例子：世上有种武器叫**核武**。

当世界有核武器后，有两种应对方式。第一种是全世界高声呐喊，呼吁不要核武器、要和平。第二种是：你有核武器，那大家都有，互相牵制制衡威慑。

对比这两种方式，是天天摇旗呐喊“不要核武器”更有效，还是“你有我也有”从而达成威慑更有效？现实世界中，拥有核武器者互相制衡，没有核武器者则是摇旗呐喊的弱者。

我们推出此课程的根本目的正源于此。因为我们深知世间与人心竟是如此，只要一点手段和方法，人心便可被操纵。

在21世纪的今天，情况依然如此。我们深刻认识到此问题。这就像你总喊“人要善良”，但无法保证走在路上不会遇见坏人与匪徒。

你能怎么办？你只能在遇见前，先了解他们的手段与方法。你可以携带防身物品，学习防身术。如果允许，你甚至可以带上一把枪，但我们国家不允许持枪。

然而在情感世界中，你总喊着人心要善良、大家要和睦相处。你将幸福、快乐与对未来的畅想，寄托于他人的道德准绳之上，希望他人讲道德、有文明、善待你。

这在我看来，就是弱者的呐喊。因此我们讲授此课程，更多是希望让大家了解情感中存在的操控，其原理究竟为何。**知己知彼，百战不殆**，这是我们开课的初衷。

我们希望将来面对情感时，你不会不知不觉被人操控，或已被操控得死死的却未曾想过反抗。同时，我们也希望大家理解，情感操控本身有助于你在感情中与伴侣达成更和谐、舒适与安全的关系。

世间任何方式、方法与武器都是双刃剑，关键看你做事的初衷。我当然希望大家学习此课程、了解其逻辑顺序时，是出于善良而非别有用心。

我们常说感情中不论输赢，因为感情不是打仗，不应将伴侣视为对立面。但现实生活中，确实存在别有用心之人。

我们能做的，就是去了解这些人的坏、恶与不良企图。在与他们相处时，才能更游刃有余地处理关系。

毕竟在现实中，情感操控并非仅是所谓渣男或经过专业训练的PUA者才会。许多普通人因在情感世界中感到绝望与无助，也会被迫采用这种方式，将我们卷入情感漩涡。

因此我们需要对此有所了解，不能让情感世界仅陷入奴役与被奴役的局面。这不是我们想看到的事。

我们更希望大家能真正成为自由独立的个体，敢于追逐、反抗，同时也敢于向他人表达。所谓表达，即“我不被你操控”，但在某些情况下，我可对你形成一定的控制与引导。

尤其对女孩子而言，在关系中适度引导男孩子，是一种比较正向的体验。但我们的操控方向，是希望关系正向化，而非将对方奴役成附属品。

目的是让对方更懂你内心的需求与想法，更接受你想做的事，让他更懂你、更爱你。这是我们开设此课程的另一个初衷。

我们将课程总共分为四个部分：
1.  选总
2.  预总
3.  收割
4.  控制

本课程由我和子修共同录制与制作完成。我们希望通过大量实际案例，让大家更了解情感操控的逻辑顺序、前因后果及其所有内涵。

以致大家将来面对此类情况时，不会谈“情感操控”即色变，或已被操控而不自知。又或者，当你面对男友时，不知如何将他培养成你心中满意、渴望的对象。

这是我们录制此课程的最大初衷。当然，很多人会觉得此课程三观不正。我前面已解释过，便不再多说。

接下来，我们将开始一课一课地了解这门“三观极为不正”的课程。
