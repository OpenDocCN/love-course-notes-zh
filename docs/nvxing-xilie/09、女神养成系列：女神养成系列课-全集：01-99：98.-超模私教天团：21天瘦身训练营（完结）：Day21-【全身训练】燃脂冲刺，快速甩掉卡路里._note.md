## 概述

本节是21天训练营的收官之日。我们将进行一次综合性的全身力量训练，通过俯卧撑、仰卧起坐和深蹲三个动作的循环组合，进行高强度冲刺。目标是鼓励大家在30分钟内尽可能完成更多组练习，以达到最佳的瘦身效果。

## 热身运动

在开始正式训练前，充分的热身至关重要。它能激活身体，预防受伤。由于是最后一次训练，本次热身将更为全面。

以下是热身动作列表：

1.  **掌心向上屈臂后震**：进行15次。
2.  **双手向上后震**：进行15次，幅度尽量大。
3.  **屈腿后蹬直**：左右腿各进行16次。
4.  **半蹲**：双脚脚尖向前，宽度约10厘米，进行10次。
5.  **分腿半蹲**：双脚与肩同宽，脚尖外展约30度，进行5次。

热身结束后，准备开始正式训练。

## 正式训练

今天的冲刺训练包含三个核心动作：俯卧撑、仰卧起坐和深蹲。我们将以循环组的方式进行。

**训练方案**：
以 **10次俯卧撑 + 10次仰卧起坐 + 10次深蹲** 为一个小组。三个小组构成一个大组。每组完成后可进行组间休息。目标是在30分钟内，完成尽可能多的大组。前期组间歇应控制在1分钟以内，后期可延长至2分钟左右。整体完成组数应力争超过10组。

以下是每个动作的要领：

*   **俯卧撑**
    *   **动作描述**：身体俯撑，双手略宽于肩。下降时，感受**三头肌**发力，身体不要抬得过高。
    *   **呼吸**：保持均匀呼吸。
    *   **代码/公式表示**：`动作次数 = 10次/组`

*   **仰卧起坐**
    *   **动作描述**：仰卧，屈膝。起身时，躯干贴向大腿正面，目光向上看。
    *   **呼吸**：起身时呼气，下躺时吸气。
    *   **代码/公式表示**：`动作次数 = 10次/组`

*   **徒手深蹲**
    *   **动作描述**：站立，双脚与肩同宽。身体向下蹲，同时屈膝。
    *   **呼吸**：下蹲时吸气，站起时呼气。
    *   **代码/公式表示**：`动作次数 = 10次/组`

**训练流程**：按照“俯卧撑 -> 仰卧起坐 -> 深蹲”的顺序，连续完成三个动作（即一个大组）。重复此循环，直至30分钟训练时间结束。坚持就是胜利！

## 拉伸放松

高强度的训练结束后，必须进行拉伸，这有助于缓解肌肉紧张，促进恢复。

以下是拉伸动作列表：

1.  **双臂后震拉伸**：进行15次，幅度要大。
2.  **双腿分开体前屈**：双脚与肩同宽，吐气向下，保持20秒，拉伸大腿后侧。
3.  **双腿并拢体前屈**：双腿并拢，保持20秒，拉伸大腿后侧及外侧。
4.  **眼镜蛇式拉伸**：双手撑地，腰部挺起，均匀呼吸，保持20秒，拉伸腹部肌群。
5.  **婴儿式放松**：向后坐，放松核心，保持20秒。
6.  **股四头肌拉伸**：单腿后屈，手握脚踝，将腿拉向臀部，保持30秒。换另一侧。

拉伸结束后，还原到起始位置，起身。
