# 婚姻经营课：01：安全感：想离婚的真正原因是什么？

在本节课中，我们将探讨现代婚姻中一个核心且普遍的问题——安全感缺失。我们将分析安全感对婚姻稳定的重要性，并分别从女性和男性的视角，深入剖析安全感缺失的具体表现与深层原因。

随着社会经济快速发展，人们生活水平不断提高，现代社会的离婚率却逐渐攀升。那么，离婚的真正原因是什么？生活水平提升，为何离婚率反而增加？本节课的核心答案只有一个：**安全感**。

## 女性的安全感需求

上一节我们提出了安全感是婚姻问题的核心，本节中我们来看看安全感在女性视角下的具体体现。

我之前讲授婚姻相处课程时，一位读者的评论令我印象深刻。他说：“离婚当饭吃的年代，结婚和不结婚区别有那么大？这年代等男人给安全感想想都可笑。”这个想法相当有代表性。我身边不只单身女性，许多已婚女性也认为婚姻并不幸福，在她们看来，指望男人给安全感是荒谬的。

这部分女性中，有些人曾被深深伤害，不再相信爱情；有些人则是不折不扣的女权主义者，坚持不婚主义，鄙视在婚姻中寻求安全感的行为，认为那等同于无能、软弱和不独立。

是否存在依靠男人获取安全感的女性？当然存在。安全感可分为**物质安全感**与**情感安全感**。或许你会说，有人为了物质安全感，舍弃了情感安全感。但这种寄人篱下的安全感真的安全吗？昔日港姐朱玲玲曾自曝，豪门生活光鲜背后全是心酸，连出门佩戴的珠宝都需向家族借用。一旦离开那个环境，你就什么都不是。当女性在一段关系中连独立人格都失去时，安全感根本无从谈起。

女权主义者常说：“安全感都是自己给的。”这句话说对了一半，但并不绝对。有一种精神上的安全感，是只有爱情才能给予女性的。一个能提供安全感的男人，才能带来期望中的婚姻。更绝对地说，这份安全感应是女性决定步入婚姻的必要条件。

遗憾的是，在经济压力增大、诱惑增多的社会，物质安全感成了必需品，精神安全感却成了可有可无的附加选项。房子、车子被认为是必要的，而爱与呵护，许多女性自己都觉得可以不要，认为“男人应以事业为重”、“不归家不要紧”、“知道回家就好”。这类观点正是婚姻危机的征兆。一段缺乏安全感的婚姻或许不会立刻瓦解，但绝不会牢固。

几个月前，我参加一个饭局，旁边一位女孩问我能否送她回家。路程不远，我便答应了。途中，我听到她给丈夫打电话，才知她已结婚。女孩说：“你就到小区门口接我一下就行。”电话那头的男人不耐烦地回答：“就那么两步路，自己走过来不行吗？怎么那么矫情。”几番纠缠后，女孩放弃了，生气地说：“行吧，你别管了。”放下电话，她叹了口气：“天天打游戏，什么都没他的游戏重要。”一路上，男人再未打电话或发短信询问。我将女孩送到漆黑无灯的小区门口，目送她远去。那一刻，我觉得她是全世界最孤独的人。

这正是部分人拒绝婚姻的原因：他们经历了太多伤害，宁愿像刺猬一样小心翼翼保护自己。但这是遇人不淑，并非婚姻本身的错。婚姻原本应是美好的。

如果说有的婚姻无法给予女性安全感，那么还有一种婚姻，则让你误以为自己拥有安全感。

我曾与一位女孩聊天，她硕士学历，眉清目秀，丈夫是运动员出身，只有中学文化。在我看来，两人简直不是一个世界的人。但女孩谈起丈夫却一脸崇拜。她说丈夫特别有男子气概，并举了许多例子：例如因被门口卖水果的老头坑了几块钱，丈夫便拿棍子砸了水果摊，将对方打住院一周。这些事没有一件是遭遇抢劫强奸等必须出手的情况，全是芝麻绿豆大、动动嘴皮就能解决或其实无所谓的琐事，最终都以丈夫将对方打得“满地找牙”告终。

我终于明白，她爱他，是因为他给了她足够的安全感。而在她眼中，这份安全感就是“被保护欲”。放在《水浒传》的时代，这样的丈夫或许是武松般的英雄；但在法制社会的今天，这算什么？说难听点，这是**违法犯罪**。女性需要安全感是对的，但一个缺乏基本理性和克制、时时将自己置于危险边缘、挑战法律道德的男人，带来的绝非安全感。

那么，真正的安全感是什么？去年有部电视剧《红色》不算太火，但豆瓣评分很高。剧中男女主人公没有轰轰烈烈的爱情，连吻戏都没有。女主角总在不知情中闯祸，男主角每一次都默默在她身后处理，承担一切危险，有几次甚至差点丧命，回来后却一脸云淡风轻。她曾经的未婚夫是国民党要员，却在关键时刻丢下她逃命；而他只是菜市场的一名会计，一个连去西餐厅都只舍得喝免费咖啡的“小男人”，却让她感到满足与感动，愿意与他手牵手买菜做饭，感受前所未有的美好。

一个能给你安全感的男人，永远不会让你担心颠沛流离，永远不会让你在深夜里哭泣，永远不会让你活得小心翼翼，永远不会把你逼成福尔摩斯，更不会让你把离婚当饭吃。女性在婚姻中需要安全感，是天经地义的事，不是矫情，不是没有自我。越是独当一面的女性，回到家，越需要一个能帮她分担的男人。这个世界上，没有人会嫌爱太多。我见过被惯坏的女性，但从未见过哪个女性是被尊重坏、保护坏、体贴坏的。

一段能带来安全感的婚姻，绝不会让你失去自我。恰恰相反，你会变得更自信、更从容。你会相信，无论在外受了多大委屈，家里永远有一个怀抱等着你。幸福的婚姻应让夫妻双方感到彼此爱恋、相互尊重和安全可靠。如果安全感不够，会给健康的关系带来负面影响。

以下是婚姻中缺乏安全感的六个迹象，方便大家了解并对症下药：

1.  **患得患失，怕失去对方**：常担心失去对方，猜测对方是否真的喜欢自己。这种担忧可能源于对方的行为，也可能源于自身不自信。一旦缺乏自信，就会对伴侣的行为缺乏信任，容易导致婚姻失败。
2.  **强烈的嫉妒**：强烈的嫉妒不会把对方绑在身边，反而容易适得其反。除了练习控制情绪，应开诚布公地与伴侣沟通，直接说出哪些行为让自己不舒服。
3.  **查看另一半的电子设备**：如手机、电脑或社交媒体账户。这样做或许是出于挽救婚姻的初衷，但时时被监视可能让对方反感。婚姻中相互信任最为关键，遇事保持心平气和也是一种修养。要知道，我们不可能完全掌握对方的一举一动。
4.  **害怕孤单，不喜欢独处**：总是粘着对方，也是不健康关系的体现。可以多与亲朋好友聊天，培养业余爱好，让自己暂时从婚姻世界中解脱，获得更广阔的视角。学会关爱自己，你会发现有时独处反而更好。
5.  **逃避争执**：婚姻里缺乏安全感时，一方会避免对抗性的话题或举动，因为担心哪怕流露出一点点反对迹象，对方都会拂袖而去。健康的婚姻关系中，坦诚沟通非常关键，这意味着需要将不愉快的话题摆上桌面，交流彼此的观点和想法。
6.  **把“爱不爱我”挂在嘴边**：一有机会就问“我有没有魅力？”、“你爱我吗？”、“你对我忠诚吗？”、“你为什么不喜欢我？”等问题。这都是缺乏安全感的表现，因为你想通过对方不断的肯定答复来获得自我安慰。适度从对方那里得到确认是可以的，但这不应成为彼此话题的主要内容。如果感到压抑且需要不停获得确认，应考虑接受婚姻咨询，以更好地认识并接纳自我。

## 男性的安全感需求

提到“安全感”三个字，我们通常会联想到女性，也因此容易忽略一件非常重要的事：**男人也需要安全感**。在爱情与婚姻里，安全感对男性的重要性从不亚于女性。但男性不像女性那样会主动索要安全感，他们通常不会告诉对方“我觉得你不爱我”、“我觉得你会离开我”、“我觉得你不忠诚”、“我对我们的婚姻没信心”。恰恰相反，他们总是在极力掩饰，因为这些话语会让男性觉得自己软弱、无能。于是他们把一切不安藏在心里，累积到一定程度，潜意识里就会产生判断：“这段感情迟早要完蛋”、“这段婚姻走不到最后”、“我不能相信她”。最可怕的是，他的另一半可能完全没察觉到。等问题显现时，为时已晚，男人可能已在财产上划清界限，或做好了离婚准备。

那么，哪些因素会影响男性在婚姻中的安全感呢？以下是几个关键点：

1.  **你的结婚动机**：对男性而言，结婚动机非常重要。他们会潜意识判断你嫁给他是为什么。如果你是因为年龄大了、父母催促、找不到合适的人才嫁给他，那么他会担心，万一哪天你遇到真正喜欢的人，就会后悔。这样的婚姻对他而言不安全。同样，为了钱、因为孩子、未婚先孕、只因贪图他的好、或嫌弃他女儿而结婚，都可能让男性潜意识里对婚姻感到不安。一旦他认为你的结婚动机不是因为爱情，这种不安就会始终萦绕，除非你能在日后相处中用行动改变他的判断。
2.  **你为前男友付出很多**：如果你有一个深爱过的前男友，并为之付出很多（如同居甚至打胎），这件事很可能成为丈夫的心魔，尤其如果婚前有所隐瞒。他会潜意识里拿自己与你前男友对比。一旦你挑剔或指责他，他就会想：“你为前男友付出那么多，却什么都不愿为我做，你根本不爱我，你爱的是他。如果他回来找你，你肯定会背叛我。”当然，这种判断未必属实，你可能很爱他且早已忘记前男友，但你很难让他真正相信这一点，除非你有高超的表达技巧。如果你始终具备心理优势，并懂得满足对方的“责爱观”，证明自己在爱情上的可靠性，那么长时间后，他的不安可能会慢慢淡化。
3.  **你对这段婚姻不满意**：如果你经常抱怨和挑剔丈夫，并常表达“我怎么会嫁给你这种男人”、“我真后悔嫁给你”等想法，他就会觉得你对婚姻不满意。既然不满意，你就可能离开他，去寻找满意的男人，或给“他戴上绿帽子”。这种不安是潜移默化的，并非偶尔一句抱怨或气话就会让男人失去安全感。但如果你总是以抱怨的方式相处，总说气话，久而久之，他可能分不清哪些是气话、哪些是心里话，从而把所有气话当成你的真心话。
4.  **你把“离婚”挂在嘴边**：很多女性并非真想离婚，但吵架时总把“离婚”当成威胁挂在嘴边，这是非常不好的行为。因为语言存在暗示作用。也许你第一次说“离婚”，他认为是气话；但经常这么说，就会形成心理暗示：“她想跟我离婚，我们的婚姻到不了头，她迟早会离开我。”长此以往，男人可能对婚姻失去信心，做好离婚准备。某天你再说离婚时，他可能不再当回事，或一反常态地答应。

## 男性失去安全感后的表现

如果男性认为一段婚姻无法长久，他们就会在心里对另一半设防。但你通常感觉不到，等你感觉到时，情况往往已比较严重。男性对婚姻真正失去安全感后，会在两个方面对妻子设防：**财产**和**感情**。

因为担心你有一天会离开，并将他的财产分给“小白脸”，出于本能，他会在财产上对你有所防备，不会轻易让你掌握主要资产，或在收入方面对你有所隐瞒。

如果男性觉得你不爱他，他们就不会再对你敞开心扉。在婚姻里出轨的几率会非常高，而且出轨对象通常是他们能表现出真爱的女性。

很多时候，由于两性差异、彼此缺乏了解、缺乏有效的沟通方式，在长期婚姻关系中，双方都不知道如何避开雷区，如何让对方感受到爱。因此，想真正避开婚姻陷阱，需要双方共同努力，加强沟通，避免安全感的缺失。

## 总结

在柴米油盐的生活中，没有绝对正确的婚姻保鲜高招，也没有对所有婚姻都管用的灵丹妙药。婚姻是否有活力、能否保鲜，最关键的因素往往就是一份彼此所需的安全感。

本节课中，我们一起学习了安全感在婚姻中的核心地位。我们从女性视角探讨了安全感缺失的表现与危害，以及真正安全感的模样；接着，我们转换视角，深入分析了男性同样深刻却常被忽视的安全感需求，及其影响因素和后果。希望这些内容能帮助你更好地理解婚姻，经营一段稳固、幸福的关系。
