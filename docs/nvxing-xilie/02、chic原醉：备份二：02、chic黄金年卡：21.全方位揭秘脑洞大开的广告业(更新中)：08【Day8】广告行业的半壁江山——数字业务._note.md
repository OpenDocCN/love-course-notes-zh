# 广告行业职场解密：02：广告行业的半壁江山——数字业务

在本节课中，我们将要学习广告行业中至关重要的数字业务板块。我们将了解数字业务包含哪些具体职位、每个职位所需的技能，以及在这一领域的职业发展路径。

上节课我们分享了策划职业的相关内容。本节中，我们来看看广告行业另一个核心组成部分——数字业务。

## 概述：数字业务的重要性

数字业务，在许多广告公司也被称为数据部门，已成为广告行业发展的关键方向。早年，广告市场投入主要集中在电视媒体。近年来，随着互联网发展，广告投资占比不断被手机端、自媒体等领域瓜分。

通过各类互联网平台，我们可以收集消费者数据。例如，通过京东或天猫可以获取消费者的购物行为数据、年龄、喜好等。因此，数字类业务工作已成为广告行业的重要组成部分。

为了帮助理解这个概念，这里有一个例子。雀巢咖啡通过数据分析发现，其产品在每年5到7月份的销量高于其他时段。分析表明，这是因为5到7月是中考和高考期，主要消费人群是备考的学生。因此，每到这个时期，雀巢咖啡就会针对该人群画像，定制以考试提神为主题的广告内容。总的来说，数字类业务的核心目标是为广告提供精确的方向。

本节课内容将分为三个部分：
1.  数字类业务包含哪些职位。

3.  数字业务类工作的职业发展空间。

## 第一部分：数字类业务的职位构成

广告公司的数字业务部门主要为客户提供报告和分析服务。公司会与不同的第三方数据公司合作，采集各类媒体数据。由于客户需求各异，不同公司的职位设置和工作内容也可能不同。

以下是大多数公司数据部门通常会配备的四种职位：

*   **电商数据分析**：此职位的工作是分析导入电商页面入口的数据。“导入”即引导进入。例如，在淘宝或京东中看到的“有好货”、“猜你喜欢”等广告位。电商数据分析师就是分析这类广告投放所引导进入商品的用户数据。例如，分析哪些广告带来了点击，哪些没有；或消费者点击后是否很快退出而未购买。通过分析这类数据，来发现投放问题并指导调整。
*   **SEM**：即搜索引擎优化。这项工作具有一定的公关效果。当消费者想购买某物但不了解时，可能会去百度等搜索引擎查询。例如，养猫的人可能会搜索“猫吃什么猫粮好”。SEM的工作就是优化搜索结果，使所服务的品牌排名靠前，或在搜索“某品牌猫粮好不好”时，让差评沉到后面。同时，还需分析不同搜索关键词的点击转化数据（“转化”指最终形成购买）。
*   **程序化购买**：这个职位可能较难理解。早期广告投放使用“点位图”工具，它简单显示在哪个时间段、在哪个媒体投放了广告。但在数据时代，广告位数量庞大（如今日头条的广告位和文章数量无法精确计算），难以用点位图规划。因此出现了程序化购买：通过程序，根据品牌需求（如预算5000万、目标人群为一线城市25-30岁用户）定制投放，系统会自动分析并投放到性价比和匹配度最高的广告位。程序化购买职位的工作，就是向品牌方推荐并使用这种投放模式，并为品牌方确定需求并执行投放。
*   **媒体分析**：此职位与电商数据分析的工作内容相似。区别在于，电商数据分析针对电商平台内部入口的数据，而**媒体分析**则是分析各个外部合作平台（如健身APP、社交APP）导入电商页面入口的数据。例如，分析哪个合作平台带来的消费者更多、哪个平台吸引的消费者购买力更强等。

## 第二部分：各类职位的能力配备

在了解了数字业务部门的职位后，我们再来分析一下各类职位需要哪些能力。

首先说明这四种职位的共性需求。由于都是数字类工作，最基础的是要对数字敏感且严谨。其次，必须熟练掌握表格（如Excel）和PPT这两种办公软件。数据工作者的本质是将复杂庞大的数据进行总结分析，分析结果需要通过这两种软件呈现。如果你注意力难以集中或容易马虎，则非常不建议从事此类工作。

以下是各职位的具体能力配备：

*   **电商数据分析**：与其他职位不同，必须对电商平台有足够了解，并熟知电商用户的购买习惯。建议学会使用 **`Hive`** 和 **`Python`** 这两个工具。简单解释：`Hive`是将大量数据转化为数据库表格的工具；`Python`是一种计算机程序设计语言，常用于大型项目开发。大家可以在课下单独了解和学习。
*   **SEM**：需要熟练掌握百度、360、搜狗等搜索引擎的投放模式与账户操作方法。同时，要懂得使用GA（Google Analytics）、百度统计等分析工具。
*   **程序化购买与媒体分析**：
    *   **程序化购买**需要对编程有一定了解，以便操作和优化程序，并能向客户介绍程序的优势。
    *   **媒体分析**类职位，从业者最好本身拥有各大平台的资源，以便拓展投放渠道。

对于非相关专业的同学，掌握这些技能可能有些困难。如果学校专业不涉及这些，可以选择外部靠谱的培训机构进行学习。这类培训机构很多，费用大致在15000元到数万元不等。

## 第三部分：职业发展空间

听完前两部分，相信大家对数字类业务已有一定了解。最后，我们来讲讲数字业务类工作的职业发展空间。

在职位晋升方面，4A广告公司普遍的晋升空间比小公司要小，因为高层职位多为外籍人士。在该体系工作十年能做到总经理（GM）已很了不起。因此，数字类工作的晋升是一个需要耐心积累的过程。但好处在于，4A公司的方法论和数据体系非常专业，接触的客户也是顶级的，有机会得到很好的历练和打磨。

如果有同学想进入广告行业从事数字业务，个人建议是先去4A公司历练，再结合感兴趣的板块，选择垂直领域继续钻研。发展到一定阶段、积累一定资源后，如果想创业，建议抓住以下两个重点：

1.  **痛点提取**：从事数字行业的人，一直用技术协助广告公司和客户达成目标，因此在工作中很容易抓住市场、客户或公司的痛点。可以根据痛点作为创业方向。例如，我认识的一位朋友开了一家小广告公司，主打智能广告业务。因为他过去在做数字广告投放时，发现很多图片尺寸固定，当品牌将同样内容投放到多个平台时，常出现格式不匹配的情况。于是他开发了一个人工智能工具，只要输入所需元素，工具会自动匹配好尺寸。
2.  **独特性**：广告市场相对透明，如果数字工作者创业方向缺乏独特性，很难占据市场。我的一位同事通过一个独特算法创业，帮助品牌了解消费者。他通过收集网络大数据，计算品牌价值的积极度和消极度百分比来评估品牌价值，并制作品牌榜单。许多品牌为了提升排名会选择与他合作。把握客户痛点和创业方向的独特性这两个重点，才能更好地找到创业突破口，也更容易坚持下去。
