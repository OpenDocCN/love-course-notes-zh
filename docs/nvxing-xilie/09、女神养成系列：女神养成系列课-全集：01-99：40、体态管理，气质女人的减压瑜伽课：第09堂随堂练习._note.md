# 女神养成系列：09：体态管理，气质女人的减压瑜伽课：第09堂随堂练习

## 概述
在本节课中，我们将一起进行一堂完整的减压瑜伽随堂练习。课程将包含唱诵、呼吸法、体式序列以及冥想，旨在帮助您放松身心、调整体态并提升内在气质。

## 课程内容

### 唱诵与呼吸准备
选择一个舒适的坐姿，双手呈祈祷式合十放于胸前。

在进入练习之前，我们先进行一个名为 **“Om Namo Gurudev Namo”** 的唱诵。这与之前只唱诵“Om”不同，我们将完整唱诵三次。其含义是：“我召唤我内在的老师，我向更高的智慧顶礼致敬。”这有助于我们连接更高智慧的引领与保护。

以下是唱诵与呼吸的步骤：
1.  首先，请深深吸气。
2.  然后，深深呼气。
3.  接着，吸气并开始唱诵：**Om Namo Gurudev Namo**。
4.  唱诵完毕后，深吸气，屏息并收束会阴，同时观想眉心。
5.  最后，深呼气放松，双手自然落于膝盖，进行自然呼吸。

### 四段式呼吸法
现在，我们开始进行四段式呼吸练习。这个方法是用鼻子分四段吸入空气，然后用鼻子一次性完全呼出。

以下是练习步骤：
1.  首先，用鼻子分四段缓慢吸气。
2.  然后，用鼻子一次性缓慢、彻底地呼出。
3.  重复此过程数次。
4.  最后一次结束时，深深吸气。
5.  屏息，收束会阴，观想眉心。
6.  再多吸入一点空气。
7.  最后，长长地呼气，保持放松。

### 热身与关节放松
完成呼吸练习后，我们进行热身。双手打开，双腿打开，轻轻抖动身体，放松关节、膝盖和脚踝。

接着，平躺进入伸展式。进行两分钟的伸展和呼吸，调整丹田区域，然后开始练习。
1.  吸气，抬起头部和腿部。
2.  稳定后，保持五次呼吸，脚部绷直。
3.  练习过程中，请根据自身情况调整。
4.  最后十秒钟，结束时深深吸气。
5.  然后呼气，摊开放松身体。

### 核心与脊柱练习
接下来，我们进行抱膝式和呼吸练习。
1.  屈膝抱腿，进行抱膝和呼吸，然后起身。
2.  最后十秒钟，结束时深吸气并屏住。
3.  深呼气放松，头部和颈部放松，保持腿部不动。
4.  通过深长的呼吸调整丹田区域的压力，并前后轻轻滚动身体。
5.  然后坐起，来到坐姿。

在坐姿中继续进行呼吸练习。
1.  手臂举高，保持呼吸。
2.  保持整个胸腔微微抬起。
3.  如果感到劳累，可以在保持呼吸的同时调整手臂位置，进行上下调整，但不要停止呼吸。
4.  最后十秒钟，结束时拇指相触，其余手指向上张开。
5.  保持住，深深呼气。呼气完毕后，收束会阴。
6.  再次深深吸气，感受丹田能量向上。
7.  然后深深呼气，打开并放松身体。

### 生命神经伸展
现在，我们进入生命神经伸展练习。首先，喝一小杯温水。

然后，双腿打开，调整腿部内侧和后侧。双脚向内勾。
1.  吸气，双手带动脊柱向上伸直。
2.  呼气，转向左侧。
3.  吸气，延展向上。
4.  呼气，转向右侧。
5.  保持动作清晰，富有节律。重复数次。
6.  最后三次，深吸气回正。
7.  深呼气打开，放松，抖动双膝。

接下来是第二部分的生命神经伸展。
1.  吸气延伸脊柱。
2.  呼气，让手去够能触及的最远距离，抓住脚趾或脚踝，保持脊柱和背部伸展。
3.  转动骨盆，微微向前滚动。
4.  吸气，延展脊柱。
5.  呼气，屈肘向下。可以将瑜伽砖或毯子垫在额头下方。
6.  重复吸气和呼气的配合动作。
7.  最后三次，结束时深深吸气并延展脊柱。
8.  深深呼气，在最远的位置停留，自然呼吸。也可以将手支撑在肩膀下方。
9.  结束时深吸气，推动背部起来。
10. 呼气，抖动放松。双腿收回，呈简易坐姿。

### 脊柱灵活练习
在简易坐姿中，我们进行头颈和脊柱的弹动练习。
1.  吸气，身体向前。
2.  呼气，身体向后。
3.  保持收束会阴和收颌。
4.  重复吸气向前、呼气向后的动作。
5.  动作的快慢和幅度由您自己决定。呼吸快则动作快，呼吸慢则动作慢。
6.  脊柱灵活者可以幅度大一些；脊柱状态不佳者，请减小幅度，以舒适、柔软、灵活和放松为目标。
7.  最后三次，深深吸气，用胸椎带动脊柱延展，下巴微收。
8.  深深呼气，放松身体，回到简易坐姿放松调息。

接着来到跪姿延石座，继续脊柱弹动。
1.  双手落于大腿处，稳定脊柱，收束会阴。
2.  吸气向前，呼气向后。
3.  练习中尽量保持专注，可以闭上眼睛关注眉心。不确定动作时可以睁眼参考，确定后闭眼进入自己的练习节奏。
4.  最后三次，结束时深吸气并屏住。
5.  深呼气放松。

### 肩颈与手臂练习
自然调息后，双手打开来到肩膀高度，保持延石座，进行脊柱扭转。
1.  吸气转向左侧，呼气转向右侧。
2.  以中等偏慢的速度开始，保持脊柱稳定。
3.  最后三次，结束时深吸气回正，打开胸腔和腋下。
4.  深呼气落下放松。

双手再次来到肩膀高度。
1.  吸气，肘部抬起，手腕相触。
2.  呼气，落到水平位置。
3.  保持脊柱稳定，肘部不要完全落下。
4.  最后三次，结束时深深吸气并拢。
5.  保持住，然后深深呼气慢慢落下放松。

接下来是手臂上下运动。
1.  食指交叉，手臂上下运动。
2.  吸气向上，呼气向下。
3.  专注在手臂的运动上，保持脊柱不动。
4.  最后三次，结束时深吸气保持住。
5.  深深呼气放松。

回到简易坐姿，进行交替耸肩。
1.  保持脊柱正直，收束会阴。
2.  吸气，左肩起右肩落；呼气，右肩起左肩落。交替进行约一分钟。
3.  最后三次，结束时深吸气保持住，呼气放松。
4.  交换方向：吸气，右肩起左肩落；呼气，左肩起右肩落。保持脊柱稳定，动作轻微。
5.  结束时深吸气，呼气放松。

然后进行双肩同时耸起落下的练习。
1.  吸气耸肩，呼气落下。
2.  逐步加快速度。
3.  最后几次，深深吸气并屏住。
4.  深深呼气放松，双手打开放于臀部两侧，保持脊柱延展。

### 颈部与最终体式
现在进行颈部扭转练习。
1.  吸气，颈部转向左侧；呼气，转向右侧。进行约一分钟。
2.  最后三次，结束时深深吸气回正。
3.  深深呼气放松。
4.  交换方向：吸气往右，呼气往左。
5.  最后三次，结束时深吸气回正。
6.  深呼气放松。结智慧手印，落于膝盖。

经期的同学请保持放松。其余同学抖动双腿放松后，我们进入最后一个体式——蹲式。
1.  脚跟并拢，脚掌打开，屈膝下蹲。
2.  手落地，脊柱延展，面部朝前保持微笑。
3.  尽可能完成26次，或做到您的最大限度。
4.  准备开始练习。
5.  最后几次，结束时深深吸气保持。
6.  呼气，脚跟落地放松，然后慢慢起身。
7.  保持脊柱弹动，吸气慢慢卷起身体回到直立。
8.  先放松。这个动作一定要慢，防止低血糖或头晕。
9.  抖手抖脚，慢慢进入大放松。

### 放松与冥想
平躺到垫子上，盖好身体。双腿打开，双手放在身体两侧，掌心朝上。
1.  感受练习后全身能量的运行。
2.  调整几次呼吸，让身体贴地的一面深深沉入地面。
3.  放松身体感到紧张的部位。
4.  深深沉入地面，越来越沉，放开对身体的控制。
5.  随着音乐进入内在的空间，好好放松自己。
6.  吸气时腹部膨胀，呼气时腹部放松。
7.  慢慢将意识带回身体，动一动手指和脚趾，转动手腕和脚踝。
8.  深吸气，坐起并伸展腰部。
9.  用嘴巴长长地呼气放松。
10. 转向左侧侧躺，可以屈膝略微弓起背部，停留一会儿。
11. 慢慢起身，回到简易坐姿，调整好坐姿。如果天气较冷，请盖好肩膀、后腰和膝盖。

最后，进入拉雅冥想的唱诵。
1.  调整坐姿，准备深呼吸。
2.  深深吸气，深深呼气。
3.  吸气开始唱诵：**Ek Ong Kar, Sat Nam, Siri Wahe Guru**。
4.  重复唱诵多次，每次收横膈膜向上，推动感明显。
5.  最后三次唱诵后，深深吸气，饱满地吸入，屏息并收束会阴，观想眉心。
6.  深深呼气。
7.  再一次深深吸气，感受能量从底端通过三个半循环来到顶端，保持屏息并收束会阴，观想眉心。
8.  深深呼气放松。
9.  第三次长长地呼气，保持静默。

双手呈祈祷式合十放于胸前。在深呼吸中感受身心的平衡、开放和放松。
1.  手掌继续向上来到眉心，拇指指尖轻抵第三眼的位置。
2.  感受眉心的跳动，聆听直觉的声音，顺从直觉的引领。
3.  将这份专注、深度和热情带入生活当中。
4.  双手回到胸前，深深吸气，深深呼气。

唱诵三次 **“Sat Nam”** 来结束今天的课程。
*   **Sat Nam**：愿真理与我同在。
*   **Peace all, loved all, life all**：愿所有的人都享受和平，享受关爱，享受生命。
