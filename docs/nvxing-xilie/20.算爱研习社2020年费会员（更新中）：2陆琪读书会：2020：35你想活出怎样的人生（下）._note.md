# 算爱研习社2020年费会员（更新中）：2：陆琪读书会：35：你想活出怎样的人生（下）

在本节课中，我们将继续探讨《你想活出怎样的人生》这本书的核心思想，学习如何理解“值得庆幸”的真正含义，如何审视所谓的“伟大”，以及如何面对人生中的过错与悔恨。我们将通过书中的故事和舅舅的教导，来思考我们应当如何生活。

## 理解“值得庆幸”的真正含义

上一节我们介绍了小哥白尼与舅舅的对话，本节中我们来看看“值得庆幸”这个词的深刻内涵。

舅舅对小哥白尼说，希望他现在能够明白，在当下的社会中，能够不受打扰地学习、随心发展才能，是多么值得庆幸的事。

“值得庆幸”这个词，通常用来表示应该感谢的事。但它的原意是指“这种情况很难形成”，即不太常见的事。只有意识到自己的幸福并不常见，我们才能抱有真正的感激之心。因此，“值得庆幸”等同于“应该感谢的”，表达了一种感激的心情。环顾广阔的世界，再回顾一路走来的自己，所拥有的正如话中所说，是值得庆幸的。

这段话非常重要。我们需要思考的不是一句简单的“应该感谢”的鸡汤，而是理解其“不常见”的本质。当你理解原意后，就知道为何要感谢——因为你活在这个世界上本身就是不常见的。

以下是几个关于“不常见”的例子：
*   几亿颗精子中，只有一颗存活并成为了你。
*   你所拥有的一切，比如理想的生活，在这个世界上并不常见。
*   根据社会数据，在中国的人口中，只有约5%的人读过大学。如果你读过大学，你就是那5%的稀有群体。
*   在生活里，当你有足够的时间、能力和闲暇去自由恋爱时，这是值得庆幸的。因为很多人被生活或父母逼迫，无法自由选择。

幸福来自于知道自己所拥有的一切其实很珍贵。这就是所谓的幸福。

## 审视“伟大”：能力与作为

在故事中，小哥白尼和他的朋友们听了水谷姐姐讲述的拿破仑故事，开始崇拜这位英雄。然而，舅舅提出了一个关键的思维方式转变。

舅舅在信中向小哥白尼分析了拿破仑辉煌而悲剧的一生，从他卑微的出身到成为欧洲的霸主，再到最终的失败。但舅舅随后提出了一个深刻的问题：拿破仑拥有非凡的斗志、勇气和活力，但他究竟**成就**了什么？他运用这非凡的能力，到底完成了什么？

这个问题不仅适用于拿破仑，也可以用来审视任何被称为伟人或英雄的人。他们具有高于普通人的能力，能完成普通人无法做到的事。但我们依然可以质疑：他们运用非凡的能力**做了什么**？他们成就的非凡之事又**起到了什么作用**？运用非凡的能力，会不会也做了非凡的恶事？

这打破了我们常有的“权威崇拜”。我们崇拜一个人，不应只看他的能力或地位，而应看他**究竟做了什么**。

以下是审视“伟大”的几个角度：
*   有些科学家终其一生只写对升职称有用的论文，而有些科学家（如袁隆平）则在田间地头做出了改变世界的贡献。关键看他做了什么。
*   如果以人类发展的长河（历史潮流）为背景来审视，我们会发现：
    *   许多人心目中的伟人，不过是这条大河中漂浮的水珠。
    *   脱离这条大河，再非凡的人完成的事也都是脆弱无常的。
    *   只有那些着眼于推动大河朝正确方向流淌，并为此倾尽一生和能力的人，才真正值得尊敬。
    *   在他们非凡的事业中，只有顺应人类进步所做的事，才真正有价值。

这让人想起《了不起的盖茨比》中的话：“我们奋力前行，小舟逆水而上，不断地被浪潮推回到过去。”个人的志向也应与人类进步的潮流相结合。例如，父母的伟大并不在于拥有非凡的能力，而在于他们历经时代波澜，竭尽全力将最好的生活和教育给予下一代，这本身就是顺应潮流、推动家庭乃至社会向前的非凡之举。

## 面对过错、悔恨与成长

故事中，小哥白尼因为胆怯，在朋友北建被高年级学生欺负时没有遵守共同挨打的约定，陷入了深深的痛苦和自责。这映射出我们生活中都会遇到的场景：说错话、做错事、伤害他人而后悔。

当人做错事时，最常见的反应是**在痛苦中逃避**。觉得自己已经很难受了，从而原谅自己。但这其实是自我原谅，而非真正的面对。舅舅严厉地教导小哥白尼，他必须像个男子汉一样，为自己的行为负责到底，真诚地向朋友道歉，无论结果如何。

母亲也给小哥白尼讲了自己年轻时的一个故事：她曾想帮一位爬石阶的老奶奶拿行李，却因犹豫一次次错失机会，最终未能付诸行动。这件事让她后悔多年。但她从中领悟到：人生中每一次邂逅都是唯一的，无法重来。当邂逅发生时，必须切切实实地将内心的美好付诸行动。虽然会为错失而后悔，但也能从中学到重要道理，并在之后的人生中抓住更多美好。

面对过错，正确的态度不是嘴硬地说“我从不后悔”，而是**要后悔，要正确地后悔**。后悔意味着你从错误中学到了东西。例如，因一次未能提醒他人避免损失而后悔，可以让人在未来变得果断，抓住帮助他人的机会。

相反，很多人陷入**自我惩罚**的循环，反复咀嚼痛苦，走不出去，甚至导致抑郁。这并没有真正面对和解决问题。

舅舅在名为《人的烦恼、过错与伟大》的信中，引用了帕斯卡的思想：人之所以伟大，在于能承认自己的不幸和过错。人类独有的痛苦，是意识到自己犯下不可挽回过错时的悔恨。这种痛苦恰恰证明了我们拥有分辨对错、并希望遵从正确道义行动的能力。

以下是这封信的核心观点：
*   我们会感到痛苦，是因为我们知道“原本应该”如何。
*   承认错误是痛苦的，但人正因为会为此痛苦而伟大。
*   我们拥有决定自己言行的力量，因此会犯错；但也正因如此，我们才能从错误中重新站起来。
*   歌德说：“错误同真理的关系，就像睡梦同清醒的关系一样。一个人从错误中醒来，就会以新的力量走向真理。”

因此，从悔恨和失败中汲取前进的力量，才是真正的勇敢。

## 总结：你想活出怎样的人生？

本节课中，我们一起学习了《你想活出怎样的人生》下半部分的核心启示。

这本书虽然是一本面向青少年的教育读物，但其养分适用于任何年龄段的我们。它告诉我们：
*   **人与人紧密相连**：我们都不是孤立地活着，应以善良、平等之心对待他人。
*   **伟大在于贡献**：一个人的伟大不取决于其能力大小，而在于他是否运用能力推动了他人或社会向好的方向发展。
*   **从过错中汲取养分**：正视悔恨，从中学习，才能获得成长和新生。

无论你正处于20岁、30岁、40岁或50岁，无论你正经历离婚、情感挫折还是其他困境，都可以将此刻视为一次新生的开始。听完这本书，不妨问问自己：**在接下来的人生里，你想活出怎样的人生？**
