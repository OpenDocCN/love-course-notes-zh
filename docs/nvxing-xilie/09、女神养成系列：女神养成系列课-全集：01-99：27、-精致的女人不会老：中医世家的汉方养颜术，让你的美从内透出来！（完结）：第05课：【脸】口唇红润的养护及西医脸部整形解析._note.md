# 中医美颜术：05：【脸】口唇红润的养护及西医脸部整形解析

## 概述
在本节课中，我们将要学习如何通过中医方法养护口唇，使其保持红润亮泽，并简要解析现代西医脸部整形方法的原理与注意事项。

---

口唇是整个面部中颜色最鲜亮的部位。美丽的口唇应该是唇型圆润，颜色红润。红润圆润的口唇，能显著为整个面部增色。

红润的口唇给人的感觉是气血调匀，胃气充足。我们常说“珠唇轻启，风情万种”，这会让女性的魅力无限增大。嘴唇的美是女性美一个特别重要的组成部分。

试想，如果一个人眉眼俊秀，顾盼神飞，但嘴唇颜色淡白无华，那么他整个面部的色调就会让人感到有些失望或遗憾。反之，即使眉眼并非特别出众，但如果拥有一张鲜艳或娇艳的珠唇，就能为整个脸蛋增色不少，并增添一种性感的魅力。因此也有“万绿丛中一点红，动人春色不须多”的说法。整张脸上，只要嘴唇是红润的，你就是最动人的。特别是在说话时，珠唇轻启更是风情万种。所以，无论其他部位多美，只要口唇淡白或青紫，这份美就多少有些遗憾。

口唇的红润亮泽，必须依赖气血两旺，才能丰润亮泽。

那么，对于口唇青紫的人群该怎么办呢？这类人群可以尝试用生姜或艾叶泡脚。慢慢地，青紫的口唇就会逐渐褪去青色，恢复血色。

对于口唇淡白的人群又该怎么办？口唇淡白的人，平时可以用红糖配阿胶粉冲服。请注意，月经期间不可服用。在非经期时，可以每天冲服这样一杯。

无论是青紫还是淡白的口唇，都需注意在自然环境好、空气清新的地方，增加运动量。有意识地寻找这样的地方进行健身运动，不仅能提高身体素质，更能促进血液运行，并提高身体的含氧量。因为口唇白和口唇紫的人，身体含氧量通常偏低。因此，运动也是让口唇红润起来的简便易行的操作方法。

那么，有的人因脾燥或胃火过盛导致口唇干裂，该怎么办呢？方法特别简单。比较常见的做法是涂抹蜂蜜，蜂蜜确实具有滋润效果。但还有比蜂蜜更好的东西，那就是家中的香油，即芝麻香油，也就是炸完芝麻酱后浮在最上层的那一层油。经常用香油涂抹，就可以让干裂的口唇很快修复好。

关于保持面部美丽，还有一个很重要的问题，就是均衡的营养配给。这让我想起了白毛女的故事。白毛女之所以变成白毛女，就是因为在山里缺乏食物，饮食非常单一，缺少营养均衡的状态。均衡的食物搭配能为五脏提供滋养。因此，她出现了头发变白和过早衰老的现象。

祖国医学对于美容的观点，绝不认为单靠外在涂抹就能解决问题。根本在于内在的调养。必须养内，使内在的气血调和，脏腑康泰，才能达到外表充盈、青春永驻的效果。所以，大家在日常饮食中要注意合理的营养搭配。要有蔬菜、水果、干的、稀的、荤的、素的、主食、副食等等。一定要注意营养搭配。

近年来，有一些朋友向我提出关于现代医疗美容的相关问题。有人说，中医美颜太慢了，等不及，需要立刻美丽起来。也有人说，现代医疗美容方法一次就能完成，不用天天在家繁琐操作。

关于现代很火的医疗美容，我有几个注意点提供给大家，以便大家做出选择。

一般来讲，现在针对面部皮肤松弛、下垂、褶皱，有哪些调整方法呢？比较简单且现在比较多见的，就是针剂注射。将针剂注射到皮下，让皮肤隆起，从而撑开褶皱。它可以维持一段时间。如果手法好，注射后效果也蛮好看。但这个需要大家自己考虑身体状况，能否抵御排异反应。因为毕竟有异物打进了身体，你的排异反应是否能抵御？此外，还有一种方法是假体植入，使脸部丰满起来。这个假体植入同样更需要考虑排异反应问题。前面我也跟大家介绍过，找到正规医院和正规医生，是任何时候都必须注意的。

还有一种情况是觉得下颌角长得不够美丽，从而去削骨。削下颌角需要注意一个问题：这是我们咀嚼时非常用力的一个部位。下颌角削薄之后，在咀嚼较硬的食物，比如啃骨头时，可能这个位置就不那么给力了。所以，削下颌角这个操作，会让我们骨骼原本能承受的力量明显减低，使骨质骨的功能发生改变。因此，大家要仔细权衡，做出取舍。

还有的做双眼皮手术。治疗性的双眼皮是可以的。什么是治疗性的双眼皮？就是上眼睑下垂特别严重的情况，通过去除一些脂肪，将眼皮提起来。我知道我们国家一位很有名的光学专家，就是因为光学辐射导致眼皮抬不起来。像他这种情况，就是找到微整形医生，将眼皮做了切割，做成了双眼皮。这是治疗性的双眼皮。如果只是为了美丽，其实我觉得双眼皮也没有那么大的必要。因为不管是单眼皮还是双眼皮，各有各的魅力。在我的朋友圈、同学圈里流行一句话：“大眼睛，双眼皮，一看就是讲究人儿。小眼睛，单眼皮，一眨一眨特勾人。”所以，不论什么样的眼皮，都有自己独特的魅力，蕴含着你自己的神韵。

还有人烫睫毛。如果是倒睫，提倡你去烫一下，这也是治疗性的。有的人想拥有漂亮的睫毛，因为睫毛烫后上翘，特别漂亮，能让眼神每一次闪动都撩动人心。眼睫毛是离眼睛最近的毛发，同时也是最娇嫩的毛发。眼睫毛是人体毛发中生命周期最短的一种，平均寿命是3到5个月，而且它会脱落、新生，不断更新。所以，如果是倒睫，烫睫毛还真有必要。

还有一种就是刚才我们说到口唇要红润亮泽，有的人就去漂唇。漂唇之后确实挺漂亮。但这样做不利于我们观察你真正的气血运行状态。这是一方面。另一方面，漂唇使用的也是一种带颜色的漂染剂进入你的身体。因此，还是要看你的排异反应，这很重要。

最后还有一个认识误区要告诉大家。现代医疗美容方法操作之后，无论是恢复期还是保持期，都同样需要我们前面介绍的这些传统中医美颜方法来做后期支持。所以，你到现代医学美容院做了手术后，院方一定会推荐你继续进行中医的一些补充治疗。没有中医美颜方法的支持，恢复期会相对漫长，保持期也会相对短暂，无法持久。

## 总结
本节课中，我们一起学习了如何通过中医方法（如泡脚、饮食调理、外用香油等）养护口唇，使其红润亮泽。同时，我们也简要解析了现代西医脸部整形（如注射、削骨、双眼皮、烫睫毛、漂唇等）的基本原理与需要重点考虑的排异反应、功能影响等问题。最后我们明确了，即便进行现代医疗美容，中医的调理方法在术后恢复与效果维持方面也起着重要的支持作用。
