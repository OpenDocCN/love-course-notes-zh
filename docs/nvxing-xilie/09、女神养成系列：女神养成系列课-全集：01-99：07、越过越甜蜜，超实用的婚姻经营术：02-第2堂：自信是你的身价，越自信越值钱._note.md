# 女神养成系列：07：自信是你的身价，越自信越值钱

在本节课中，我们将要学习如何成为一个有主见的女人。我们将探讨“主见”的含义，识别缺乏主见的具体表现，并深入分析其背后的核心原因——低自尊。最后，我们将学习一个实用的工具“YES的CK手环”，帮助你建立自信，勇敢表达，从而在婚姻中掌握主动权，提升个人魅力与价值。

## 什么是主见？

上一节我们介绍了课程的主题，本节中我们来看看“主见”的具体定义。

主见是主动的，并且有自己的见解。它意味着一个人能自主地做出正确的判断，并在相信自己的判断的基础上做出主动选择。这里所说的主见，不是固执己见或独断专行，而是有头脑、有思想、不盲从。

## 缺乏主见的四种表现

理解了主见的定义后，我们来看看在婚姻生活中，一个没有主见的女人通常会有哪些具体表现。以下是四种常见表现：

1.  **容易后悔**：不敢轻易做决定，做出决定后又反复质疑其正确性。
2.  **多变**：不清楚自己内心的真实需求，因此想法和决定经常变化。
3.  **易受他人情绪影响**：容易被外界事件或他人言论影响自身情绪，例如因他人收到礼物而对自己的伴侣不满。
4.  **不会做选择**：凡事都需要别人来拿主意，但当对方给出建议时又常常否定。

## 场景分析：缺乏主见的互动

为了更直观地理解，我们来分析一个日常场景。在这个场景中，妻子因为结婚纪念日安排与丈夫产生了矛盾。

> **丈夫**：你今天准备做饭？
> **妻子**：你自己想。
> **丈夫**：我不知道啊。
> **妻子**：有人惹你生气了。今天是我们结婚纪念日，你忘了吗？
> **丈夫**：你想吃什么？我们出去吃好不好？
> **妻子**：你决定吧。
> **丈夫**：嗯…那吃西餐。
> **妻子**：太贵了。
> **丈夫**：吃日料。
> **妻子**：太远了。
> **丈夫**：哎，不然我们去吃火锅吧，好不好？
> **妻子**：谁结婚纪念日吃火锅啊？那你说吃什么？
> **丈夫**：你都不知道我怎么知道呢？
> **妻子**：你想吃什么你就直说，要不咱别吃。
> **丈夫**：那就别去了。
> **妻子**：好。

在这个场景中，妻子有两条突出的缺乏主见的表现：

1.  **不会做选择**：凡事要丈夫做决定（“你决定吧”）。
2.  **易受情绪影响**：因对纪念日安排失望而生气，并将情绪发泄在沟通中。

真正破坏纪念日的，并非丈夫的遗忘，而是妻子因缺乏主见而引发的负面互动模式。

## 缺乏主见的根源：低自尊

上一节我们看到了缺乏主见在行为上的表现，本节中我们来深入探讨其根本原因。

缺乏主见的核心原因是**低自尊**。在婚姻需求金字塔中，自尊属于精神需求。低自尊是自信的对立面，低自尊的人不肯定、不认可自己，怀疑自身能力，并且常常不喜欢自己。

许多女性正是带着这种低自尊进入婚姻关系的，而“没主见”正是低自尊在婚姻中的一种常见表现。缺乏主见具体通过三个维度展现：

1.  **经常不满，情绪消极**：容易因伴侣的言行不符合预期而生气、沮丧，且不易释怀，喜欢“翻旧账”。
2.  **不直接表达真实需求**：有两种常见心态：一是希望对方能猜中自己的心思，以此作为“爱”的证明；二是害怕被直接拒绝，这会进一步挫伤本就脆弱的自尊。
3.  **表面顺从，内心积怨**：当矛盾激化时，可能会表面顺从对方的决定（如“那就不去了”），但内心不满会持续累积，导致日后更多的争吵或冷战。

低自尊的信念通常源于童年时期与父母的互动，主要形成原因有两个：

*   **情绪原因**：父母常用发脾气的方式指责孩子，导致孩子学会用情绪（如摆脸色、生闷气）而非语言来表达不满。
*   **认知原因**：认为自己“不行”或“还没长大”，因此觉得自己没有资格或能力去表达和争取需要。

## 解决方案：构建YES的CK手环

认识到问题根源后，我们需要一个系统的方法来改变。作为家庭CEO，你需要将被动心态转为主动心态，从“依赖他人”变为“独立决策”。为此，我们引入一个核心管理心法：**YES的CK手环**。

任何想法都会引发连锁反应。消极想法导致消极行为与结果，积极想法带来积极行为与结果，各自形成一个循环。我们用CK手环来表示这个循环。

*   **C** = Cycle， 意为了解思维行为模式的循环。
*   **K** = Know， 意为了解自己。

一个CK手环由5个要素构成，我们以消极循环（LOW的CK手环）为例：

1.  **触发事件**：引发反应的具体事情（如：丈夫忘记结婚纪念日）。
2.  **内心的负面想法**：对事件的自动化解读（如：“他一定是不关心我了”）。
3.  **出现的情绪**：由想法引发的感受（如：愤怒、沮丧）。
4.  **应对的行为**：基于情绪采取的行动（如：生闷气、否定所有提议）。
5.  **结果**：行为导致的后果（如：纪念日泡汤，关系恶化）。

**CK手环的意义在于**：如果你能从一件事中看到自己的这种思维行为模式，你就会发现自己在其他事情上也沿用着相似的模式。这能帮助你找到自己“总是没主见”的原因，从而有机会扭转局面。

我们的目标，就是将 **LOW的CK手环** 转变为 **YES的CK手环**。

### 第一步：了解优势，建立自信

对应YES手环的起点——建立积极的自我认知。

1.  准备一张A4纸和笔。
2.  在纸上写下至少**10条你自己的优点**（例如：厨艺好、有亲和力、执行力强、好学等）。
3.  每天早晨，**站在镜子前，看着自己的眼睛**，用坚定有力的声音大声读出所有优点。
4.  读完後，告诉自己：**“我就是这样的人。我是这个世界上独一无二、特别有价值的人，我很喜欢我自己。”**

### 第二步：改变想法，改善情绪

对应YES手环中的“内心想法”和“情绪”要素。

你需要有意识地将**消极的负面想法**转变为**积极的正面想法**。

*   **消极想法**：老公不记得结婚纪念日了，一定是不关心我。
    *   **积极转变**：老公可能最近太忙了，我可以主动提醒或一起筹备。
*   **消极想法**：老公连我想吃什么都不知道，真的不在乎我了。
    *   **积极转变**：老公在积极寻找吃饭的地方，他是在考虑我的需要。

当想法变得积极，你的情绪自然会从**愤怒、沮丧、自卑**转向**淡定、平静、喜悦**。学会为自己的情绪负责，是变得有主见的关键一步。

### 第三步：勇敢表达，采取行动

对应YES手环中的“应对行为”要素。

用“勇敢表达句式”直接说出你的需求，取代消极的应对行为（如生闷气）。

*   **消极行为**：生闷气，认为“老公应该为我负责”。
*   **积极行为**：直接表达：“**老公，我想去吃西餐，我们去那家最贵的吧？**”

无论对方最初反应如何，勇敢表达本身就能打破旧有的被动循环。通过清晰、自信的沟通，你更有可能争取到想要的结果，实现双赢。

当你持续运用YES的CK手环，你会形成一个全新的、积极的循环：**积极事件 -> 积极想法 -> 积极情绪 -> 有效行动 -> 双赢结果**。这会让你在婚姻中越来越有主见和魅力。

## 总结与作业

本节课中我们一起学习了如何成为一个有主见的女人。我们了解到，老婆没有主见，长期来看会消耗伴侣，也可能损害婚姻。缺乏主见是低自尊的表现，其核心信念是“我不够好”、“我不值得”、“我不配拥有”。

要成为家庭CEO，你需要主动建构**YES的CK手环**，通过了解优势、改变想法、勇敢表达这三步，从依赖他人决策转变为独立自信的个体。只有这样，你才能更好地把握自己的婚姻，提升个人身价。
