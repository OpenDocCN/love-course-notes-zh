# 朋友圈架构术：02-07：制造“校花效应”，建立你的魅力引力场

在本节课中，我们将学习如何通过朋友圈营造“校花效应”，从而有效提升异性吸引力。我们将深入分析校花魅力的成因，并学习如何通过展现核心价值与怡人性，以及在合照中巧妙站位，来构建自己的魅力磁场。

## 校花魅力的奥秘

上一节我们探讨了朋友圈氛围的营造，本节中我们来看看如何塑造如同“校花”般的个人魅力。年少时期，每所学校都有一位风云人物——校花。她们未必是最美的女孩，但往往多才多艺，笑容甜美，一举一动都牵动人心。

她们不仅能吸引男生的目光追随，也能获得女孩们的喜爱。即使恋爱，她们身边也永远不乏追求者。课桌上总是时不时出现情书和礼物。这种让男友神经紧绷、越发努力宠爱的现象，就是“校花效应”。追求校花的男生越多，她的魅力似乎就越大，她的男友就越是宠爱她。

### 校花为何“不恋战”

我们总是会听到看到一些男生苦苦追求校花的故事，却很少听过校花为了喜欢的男孩低到尘埃里。本质上源于校花们懂得如何愉悦自我，有很强的主场意识。

她们虽然平易近人，却不会放低姿态去讨好他人。她们通过释放魅力的方式吸引其他人来到自己身边。所以她们在爱情里是绝不恋战的长胜将军，不会因为一时一地的损失而难过。如果小男神不喜欢自己，没关系，她们依旧会稳定地输出魅力，等待下一个小男神出现。

### 追求校花的社会认同感

除此之外，追求校花还能赋予男生社会认同感。校花往往是三好学生，或者能歌善舞，是老师的心头肉。当男生追求优秀的校花时，潜意识里希望能够通过和校花交往，超越自身的平庸性。也就是说，当他们在追求校花时，实际上追求的是更好的自己。

所以我们经常能看见一些普通的男生，甚至是老师眼里的坏孩子不遗余力地追求校花。在电影《那些年，我们一起追过的女孩》里，男主角柯景腾就是因为暗恋女主角沈佳宜开始努力学习，最后考上了好大学，成为了更好的自己。

## 打造朋友圈的“校花身份”

聊完了校花的魅力奥秘，接下来我们就通过两个维度来制造自己的朋友圈校花身份。

### 维度一：升维——展现自我核心价值

首先，通过展现自我核心价值来实现魅力升维。你也许会发现朋友圈里有这样的女孩存在，她们的朋友圈充满自己美艳动人的照片，总是出入各种高级餐厅。精致的妆容和高端的场景都体现出她良好的审美和消费水准，但你也会在心里打上一个问号：她到底是做什么工作的呢？

很多女孩没有输在美貌、有趣等能给人带来感官上直接愉悦的方面，而是输在了不懂得在朋友圈体现自我核心价值。

**核心价值**是指那些能帮助你实现自我价值，或者是能为彼此带来长久愉悦度的技能，让人觉得和你长期交往，能够获得和生命的滋养。想要体现核心价值，不仅局限于一板一眼地介绍自己的工作，或者是展示和工作相关的话题。在生活中，我们可以从两个方面来展现。

以下是展现核心价值的两个方面：

1.  **展现自己性感的大脑**
    如何展现呢？现在有个很火的词是“认知”。你如何去了解一件事物，如何去思辨讨论一个问题，往往就反映了你的认知。假如你很喜欢看《奇葩说》，那么看完之后，可以针对每一期的辩题，写出自己的想法。也可以根据你支持的选手观点，写出你为什么会支持他，你们有哪些共同的观点。亦或是你是知乎的深度用户，你可以将你的知乎回答分享到朋友圈里。亦或是你想要安利一部你很喜欢的电视剧，那就细细分析一番服化道的考究、剧情的逻辑线和节奏感。总之就是将自己生活中的娱乐项目，变成输出自我思考能力的途径，同时带有一定的趣味性，能够激发出朋友关注和讨论的欲望。

2.  **展现自己的才艺**
    有些女孩明明会唱会跳，却从来不在朋友圈里显山露水，把自己伪装成一颗平平无奇的小白菜。要么是觉得自己的身材不够好，跳起舞来不够好看。要么是觉得自己的琴技不够好，羞于展示。实际上，当你大胆地释放自己的魅力，你身材上的小缺点、琴技的不完美，都会被你自信的光芒掩盖。如同你看向夜空中一颗闪亮的星星，并不会想着星球表面是如何的凹凸不平，而是单纯地为它在黑夜中闪耀的光芒而愉悦。建议你每个月拍摄一段自己才艺展示的小视频，配上相应的场景，通过场景的氛围来塑造你的形象。举个例子来说，如果你喜欢古琴，那就可以选一个高雅的茶室来拍摄，对应的氛围也能有助于你的发挥，让你能够享受当下，展现出更好的自己。

### 维度二：降维——展现自己的怡人性

完成了升维部分，再来说说如何降维，主要在于如何展现自己的怡人性。从心理学的角度来说，**怡人性**是指一个人在社会上的亲和力与合作水平，也就是让人喜欢的能力。

为什么怡人性如此重要呢？怡人性高的女孩，让人愿意和她长久相处，更有亲近她的欲望。也许你会发现校花们往往不是恃宠而骄的冰山美人，而是平易近人，和男生女生都玩得来，是公认的“好女孩”。

什么是怡人性不高的女孩呢？朋友圈里清一色的美照，以及惯性打卡一般的晒物、晒旅游，展示的标签指向单一。看起来似乎没有问题，却因为失去了真实的情感流露，这份美貌反而会给人距离感，有种时刻紧绷的感觉，美得不够轻松自然。

想要展现自己的怡人性，同样也是从两个方面来体现。

以下是展现怡人性的两个方面：

1.  **展现自己性格中的亲切与自然**
    尝试着展现逗逼与示弱的一面。日本女演员天海佑希是一个典型的御姐型女生，英气十足的她扮演了一个又一个强势又不失成熟女性魅力的角色。但在生活中，她偶尔也会示弱，做出嘟嘴的样子，而且十分幽默，还因为喜欢小黄人而为动画片配音，让大家直呼性格好可爱。大家也可以大大方方地在朋友圈自嘲或者是示弱，例如“人不如猫系列节目第一集：我和猫猫一起生病了。我可以不看病，但是猫不可以”，或者你去滑雪的时候摔倒了，也不妨直接坐在原地，拍一张自己坐在地上的照片，然后配上文字：“今天是越摔越想就地躺下的小吴同学”。这样的朋友圈一定会引来许多好友的关心，顺便还能触发一个和男神约会的机会呢。

2.  **展现自己居家的烟火气**
    一个认真生活的女孩，总是会让人感觉充满积极能量，也有家庭氛围的亲切感。你可以在周末烹饪的时候做一些摆盘，或者是展现自己养的几颗多肉、新买的一束花。即使只是收拾屋子之后的窗明几净，也可以分享在朋友圈。

## 直观制造“校花效应”：合照的C位法则

聊完了如何从两个维度展现校花魅力之后，我们再来教大家如何通过照片直观地展现校花效应，在朋友圈C位出道。

年少时期，我们经常会发现，当一些男生开始明目张胆地追求校花，就会引来更多男生喜欢校花。除了讨论度较高容易引起注意，从而形成了跟风喜欢校花的现象。这背后还潜藏着男性的竞争心理。澳大利亚昆士兰大学心理系教授希普尔发现，男性在吸引女性之前，通常会先和同性进行竞争。同时，这种竞争环境的出现，会促使男性更加积极地追求异性。

我们该如何在朋友圈里制造一个竞争环境呢？在我们的朋友圈里经常会出现一些聚会合照，但是大家往往会忽略自己所占的位置，甚至有些害羞的女孩还会故意站在角落。其实合照中的不同位置，会给人造成不一样的观感，尤其是站在C位的人，会有一种众星捧月的感觉。

**C位**就是合照第一排的正中间。当一个女孩合照的时候站在C位，尤其是当她身边围绕着一群男性，就会产生校花的感觉，激发出男生的雄性竞争本能，从而促使对方更关注你。

所以大家记得，每一次合照的时候，都站在中间的位置，或者是一群男生的中间，制造校花效应。

这里有个地方需要注意。如果你经常发合照，而合照中的男性友人总是变幻莫测，场景常常是灯红酒绿之地，就会引来错误的遐想，让自己的风评下降。所以每次发合照的时候，大家要注意避开这些场所，同时配上明朗的文案，比如“同学聚会”、“再见旧同事”等等。

## 课程总结

本节课中，我们一起学习了如何制造“校花效应”。一方面要**升维**，在朋友圈体现出自己的**核心价值**，包括展现思考能力和个人才艺；另一方面又要**降维**，体现出自己自然亲切的**怡人性**，包括展现性格中的亲切感和居家的烟火气。最后，我们还可以通过合照站在**C位**来制造直观的校花效应，激发男性的竞争心理，帮助你更快吸引心仪男神。
