# 女神养成系列：11：如何打造大小脸，打造黄金分割比 🧘‍♀️

在本节课中，我们将学习如何通过睡眠和自我抚触等自然、温和的方法，调整面部的不对称问题，从而改善大小脸，追求更协调的面部黄金分割比。

## 概述：面部不对称的普遍性与调整原理

我们照镜子时仔细观察，会发现每个人的脸都存在一些不平衡，通常表现为左右脸大小不一。

眼睛、眉毛左右两边的高低可能不同，嘴角的弧度左右两边也可能不一样。除非是天生的明星脸，否则完全对称的脸是少数。即便是明星，也可能需要后天调整。

一位专业化妆师曾分享，从专业人士角度看，一个人是否好看，关键不在于五官是否精致，而在于整张脸是否对称、协调。

即使五官不突出，但只要组合在一起非常对称，让人看着舒服，就会觉得好看。例如徐静蕾和舒淇，她们的五官单独看或许不惊艳，但组合在一起就很美。

这是因为她们的脸部对称，五官比例协调。除了整容、医美，是否有更温和、自然的方法调整五官比例？答案是肯定的，但需要长期坚持。

在介绍具体方法前，我们先了解其原理。这些方法基于客观的人体解剖学理论，可称为科学美容。

## 第一节：理解面部结构的底层逻辑

上一节我们概述了目标，本节中我们来看看调整面部的基础原理。

首先，脸部大小建立在面部骨骼、肌肉和筋膜的排列组合上。因此，真正调整脸部大小，必须从调整骨骼、肌肉和筋膜入手。徒手整形就是基于这个理念。

但我们自己操作整骨不太可能，那该怎么办？这里涉及一个人体工程力学的小秘密。

在解剖学上，骨骼是筋膜的“属下”，肌肉听从筋膜的安排。所以，即使你去整骨，如果对应部位的肌肉力量未跟上，筋膜牵拉力依然不平衡，很快又会恢复原样。

相反，如果我们从筋膜入手去平衡其拉力，反而可能得到意想不到的效果，因为筋膜是主要的操控者。

其次，脸部不平衡或五官大小不一致，其实反映了全身的不平衡。我们的身体覆盖着一张巨大的筋膜网，这张网各部分相互影响。

如果筋膜网的拉力张力不平衡，身体就会向筋膜紧张的部分偏移，长期以往身体变得不对称或倾斜，进而影响面部五官也变得不对称。

身体的不对称可以从面部看出。例如，骨盆对应脸部的蝶骨（位于眼睛后方眼眶内的骨头）。如果骨盆高低不同，间接也会影响眼睛大小不一、眉毛高低不同。

通过触诊大量身体发现，人体基本呈现一致规律：身体哪边的筋膜紧张，身体就会向哪边倾斜。观察一个人的脸，基本可以判断其骨盆是否端正、有无长短腿、脊柱是否侧弯等问题。面部的筋膜同样反映全身筋膜状况。

## 第二节：影响脸型的关键因素——下颌骨与睡眠

说完了蝶骨，我们再来看下颌骨（牙齿两边的咬肌部分）如何影响脸型。

下颌骨的大小除了受遗传因素影响，还与饮食、睡眠及睡眠姿势有关。例如，如果总是用一边牙齿咀嚼，该侧咬肌就会比较紧张、显得壮大，该部位筋膜张力也较大，导致这一侧下颌骨看起来较大。

实际上，这些问题多与肌肉和筋膜有关，与骨头关系不大。但很多人误以为是下颌骨过宽而去削骨，这是错误的。

为什么下颌骨与睡眠姿势有关？这涉及到打呼噜的问题。打呼噜直接影响下巴美观，长期打呼噜的人，下巴会越来越短，越来越往脖子方向缩。

或者睡觉时总是用嘴巴呼吸，没有闭合牙齿用鼻子呼吸，这个人也会有下巴越来越短小的趋势，长期形成难以挽救的短下巴或双下巴。

现在说说睡眠姿势对脸部的影响。如果长期偏向一侧睡觉（例如右侧睡），会导致左侧脸庞较大。长期朝哪边睡，另一侧的脸庞就会较大一些。

因为在睡眠时，全身关节、肌肉、筋膜都非常放松。在这种放松状态下，姿势会无形中进行微调。所以别小看平常的睡眠姿势，它无形中影响着你的面部五官。

## 第三节：四大调整方法详解

大家很关心是否有方法调整。当然是有的，现在就来介绍几个方法。

### 方法一：面部筋膜松解

经常进行面部筋膜松解，有助于更好地平衡面部五官。最好养成睡前或日常松解筋膜的习惯。

以下是具体步骤：
1.  如果你是坐着，双手搭在桌子上放平，以稳定手肘。
2.  双手轻轻贴着你的脸庞，不施加任何外部力量，只是贴着。
3.  停留一会儿，直到感知到脸部左右两侧出现规律性的律动。
4.  双手感知到这个律动后，就跟随它。随着脸部扩张，双手就扩张；随着脸部收缩，双手就往内收缩。
5.  持续跟随律动约2分钟，筋膜松解就完成了。

同样也可以在睡前把双手放在脸上，用同样的方法进行。

### 方法二：调整骨盆平衡

调整骨盆高低，可以间接调整蝶骨，从而影响眼睛眉毛的高低。如果骨盆摆正，眼部也会更加平衡好看。

我们自己如何调整骨盆？方法很简单：
1.  平躺在床上，双手轻轻搭在髂骨上（骨盆两侧高高凸起的骨头）。
2.  等待一会儿，什么也不用做。
3.  如果足够敏感，你会感知到骨盆也有小幅度的律动和浮动。
4.  发现骨盆律动后，双手持续跟随这个律动即可。
5.  同样持续2到3分钟。坚持一个星期，每天睡前进行。

你可能会觉得神奇：双手只是放在那里，骨盆就正了？这是人体的一个规律。不妨尝试一下，平躺并将双手放在髂骨上，感知骨盆左右高低是否一致。坚持几天，再次躺下时，你可能会惊奇地发现骨盆变得更平、更正了。

### 方法三：针对性调整睡眠姿势

通过调整睡眠姿势，可以针对性调整下颌骨和眼部。

以下是调整原则：
*   **调整下颌骨**：睡前观察，如果哪侧下颌骨（咬肌部分）更大更突出，今晚就朝哪侧睡。例如，左侧脸更大，就向左侧睡，或增加向左睡的时间。
*   **调整眼角高低**：照镜子发现哪侧眼角较低，就朝哪侧睡。例如，左侧眼角较低，就睡在左边。
*   **调整骨盆**：骨盆的方向和蝶骨一致。如果骨盆哪边较低，就朝哪边睡。如果习惯朝某个方向睡很久，意味着那一侧的眼角或骨盆可能较高，因此需要调整睡向对侧。睡前先睡向反方向，以增加调整侧的睡眠时长。

### 方法四：睡眠手势摆放提拉面部

第四个方法是结合睡眠姿势和手势摆放，以调整面部下垂。

具体方法是：
1.  当你侧卧时，一只手垫在脸的下方。
2.  有意识地将手拖着脸部，向眼角（太阳穴）方向轻轻往上推一点点。
3.  在这个位置睡觉，有助于面部提拉，让五官更紧致。
4.  注意力度一定要轻，只是手有一个向上推的趋势即可。

## 第四节：彩蛋——预防鱼尾纹与法令纹

最后分享一个彩蛋：如何避免鱼尾纹和法令纹的生长。它们可能与不正确的睡眠姿势有很大关系。如果睡觉不注意，早上起来可能会发现嘴角、眼角皮肤出现淡淡褶皱。

避免鱼尾纹和法令纹，最好的方法是尽量减少睡眠时对面部的挤压。

以下是具体建议：
1.  尽量避免趴着睡，这对面部五官和皮肤影响不好。
2.  尽量以平躺为主。如果侧躺，不要太挤压面部，保持五官轮廓舒展。
3.  平躺时，可以配合前面教的手法：双手放在髂骨上方，校正脸部平衡。
4.  也可以躺着时，双手轻轻放到脸上（不施加力量），感知到脸部律动后，跟随节律，用双手将脸颊轻轻往上提。力道要非常轻柔。

每天晚上坚持，对法令纹和鱼尾纹会有明显改善。坚持三五天，就会发现它们明显减少。效果取决于力道是否足够轻，以及是否跟随脸部律动做轻微上提，而非一味蛮力上推。

这与平常涂抹护肤品时随意上推的效果完全不同。一定要感受到脸部律动后，再做上推动作，效果才会明显。
