# 创业资本论：02：融资流程详解与核心风险规避

## 概述
在本节课中，我们将学习创业融资的完整流程，了解成为明星项目的关键要素，并重点解析融资过程中两个常见的风险点：一票否决权与对赌协议。掌握这些知识，将帮助你更高效、更安全地完成融资。

---

上一节我们介绍了不同阶段的融资技巧，本节中我们来看看融资的具体步骤与核心风险。

融资过程可以类比为一段“联姻”。以下是其关键步骤的拆解：

1.  **初步接触（寻找投资人）**：投资人在众多商业计划书中发现具有潜力的项目。
2.  **初次会议（第一次约会）**：双方进行初步沟通，若意向良好，则进入下一阶段。
3.  **签订投资意向书（确立关系）**：双方签订 **Term Sheet**，即TS，标志着初步投资意向的达成。
4.  **尽职调查（双方家庭了解）**：投资方对项目进行全面的背景、财务、法律调查。
5.  **投资决策委员会通过（征求家族同意）**：投资机构内部进行决策，即“过会”。
6.  **签署正式协议（领取结婚证）**：双方签订正式的 **Share Purchase Agreement**，即SPA。
7.  **项目交割与打款（举行婚礼）**：完成法律手续，投资资金到位。
8.  **投后管理（婚后磨合）**：投资方与创业团队进入合作与磨合阶段。

---

了解了整个过程之后，我们该如何把握融资的节奏呢？关键在于打造一个“明星项目”。以下是成为明星项目需要具备的五个核心要素：

*   **创始团队基因**：创始团队的背景、经验和认知深度，基本决定了公司发展的天花板。例如，大疆无人机的创始团队源于香港科技大学，并拥有平均超过10年的研发经验。
*   **赛道选择**：在正确的时间进入正确的行业至关重要。风口的本质是未被满足的**刚性需求**，在技术、政策等因素推动下爆发。例如，共享经济诞生于人们对出行便利的需求。
*   **巨大想象空间**：早期项目的商业模式可能不成熟，但必须展现出巨大的成长潜力和扩展可能性，不拘泥于当前局面。
*   **独特竞争壁垒**：必须在技术、品牌、产品或商业模式上建立区别于竞争对手的护城河。例如，江小白的互联网营销或今日头条的智能推荐算法。
*   **明星投资人背书**：知名投资机构（如红杉资本、金沙江创投）的加入，能为项目带来强大的品牌增信和资源网络。

为了便于理解，我们可以分析瑞幸咖啡的案例。其创始团队及投资方（如愉悦资本刘二海）深谙资本运作，选择了互联网咖啡外卖这一新兴赛道，并通过巨额补贴和明星代言快速建立品牌，最终迅速上市。其核心公式可概括为：**明星团队 + 新兴赛道 + 资本驱动 + 品牌营销 = 快速规模化**。

---

我们再回到融资过程中。有不少创业者在签订协议时会遇到陷阱，尤其体现在TS和SPA环节。这里我们给同学们划两个重点。

### 一票否决权
一票否决权通常是为了保障投资人对资金使用和公司规范经营的监督。投资人可指派董事，对重大事项拥有否决权。代码逻辑可简单表示为：
```python
if investor_veto == True:
    company_decision_executed = False
```
然而，滥用此权利可能导致公司决策僵局。 ofo共享单车的困境，部分原因就在于多位主要投资方都拥有一票否决权，导致在关键时刻（如与摩拜合并）无法做出有效决策。

### 对赌协议
对赌协议（估值调整机制）是融资中的高风险条款。其核心在于，若创业者未达成约定业绩（如净利润、上市时限），则需向投资方进行补偿。补偿方式通常为：
1.  现金补偿
2.  股权补偿（转让或回购）

其风险逻辑是：**经营不善 -> 触发对赌条款 -> 丧失股权或现金流**。俏江南创始人张兰因对赌协议中关于上市时间的条款未达成，最终失去了公司控制权，这是一个惨痛的教训。

既然对赌协议风险极高，我们该如何应对？以下是四点建议：

1.  **增强自身筹码**：若你拥有投资人看重的核心技术，在谈判中可更强势，拒绝不合理条款。
2.  **优选补偿方式**：如果必须接受对赌，尽量选择以**股权补偿**替代现金补偿，以保护公司现金流。
3.  **凸显无形资产**：利用核心技术、知识产权、品牌价值等作为谈判筹码，争取更宽松的条款。
4.  **充分尽职调查**：在配合投资方尽调时，尽可能透明化，让投资方基于充分信息设定合理的业绩目标，避免不切实际的期望。
