# 恋爱知识星球：07、媚娘：二哥野生小动物园 第八课 《让我们谈谈钱》：第一段

在本节课中，我们将要学习如何在不同场景下看待和处理与钱相关的事务。我们将从工作、朋友关系和男女关系三个主要方面展开讨论，并提供一些实用的原则和建议。

## 工作中如何谈钱

工作中谈钱主要涉及薪资、福利等问题，谈判对象通常是上级或老板。对于职场新人而言，每一份工作的薪资都至关重要，它不仅影响当前收入，更会作为下一份工作薪资的重要参考依据。

以下是找工作或面试时谈薪资的一些提示。

**在面试流程中，当被问及期望薪资时，你需要结合自身实际能力、当前薪资状况、对方公司给出的薪资范围以及市场行情来给出一个数字。**

*   除非你拥有超常的能力和资源，否则最好不要超出用人单位给出的薪资范围。
*   同样，也不要因为求职心切而将期望薪资说得过低，这反而可能让人怀疑你的能力。
*   一个合理的参考是：跳槽时，在现有工资基础上期望10%到25%的涨幅是较为常见的比例。

**确切的工资数字可以等到面试流程后期或结束时再谈。**

在工作这个双向选择的过程中，在你对公司和职位有深入了解，同时对方也对你有一定认可后，再谈具体数字会更加实际。

**用人单位对薪资进行讨价还价是正常现象。**

因此，在找工作前，提前设定好自己的薪资底线，并在此基础上进行协商。

上一节我们介绍了求职时如何谈薪资，本节中我们来看看在职期间如何谈加薪。

谈加薪是一门技术。你需要明白，钱在谁手里，谁就掌握主动权。作为员工，心态要放平，理解加薪可能涉及公司政策、公平性、评估机制等多方面因素。

**针对谈加薪，我总结了一些原则供大家参考。**

*   **先有价值，再有加薪**：你的价值在于能为公司创造多少利润或节省多少成本。无论是提升销售业绩、优化市场投放效益还是提升客户满意度，你必须先证明自己的价值，这是谈加薪的根本基础。
*   **公司政策优先**：如果公司有明确的调薪制度或固定的年度调薪比例，应以公司政策为准，试图成为特例的成功率通常很低。
*   **提加薪与表忠心相结合**：千万不要让老板或上级产生“不给你加薪，你就会消极怠工”的担忧。你应该表达的是寻求可能性，同时表明无论结果如何，都会尊重公司决定并继续保持良好的工作状态。
*   **选择合适时机与方式**：时机最好选在自己有优异表现之后。谈话时，可以从了解老板对自己工作的评价和要求切入，再自然地带出加薪话题。场合尽量选择相对轻松的环境。
*   **给对方留台阶**：这是职场情商的体现。无论结果是否符合预期，都不要把对方逼到没有回旋余地的境地。妥善处理此事，对你个人的评价也有加分作用。
*   **避免横向比较**：千万不要拿其他公司或同事的薪资来作比较。这不仅是幼稚的心态，也是在将老板的军，极易招致反感。你只需要专注于总结自己的工作和进步。

**最后，我个人不太提倡拿着别家公司的录用通知（offer）作为与现公司谈判加薪的筹码。**

除非你已决心离职，而现公司执意挽留，否则这种行为并不可取。

## 合作与投资中如何谈钱

接下来，我们谈谈合作与投资中涉及钱的问题。无论是朋友间合伙做生意，还是一起做投资，这本身是好事，但处理不当容易导致人财两空。

**在与人合作生意或投资时，只要涉及钱，最基础的前提是各方对钱的看法必须一致。**

无论关系多么亲密，只要不是小数额，一定要事先把有关钱的事情说得清清楚楚。

**以下是需要事先明确的关键事项。**

*   盈利了如何分配？何时分配？
*   亏损了如何承担？
*   合作过程中有人想退出怎么办？
*   各自的具体分工和职责是什么？

**务必避免模棱两可，想着“做了再说”。**

事后的纠葛麻烦，远大于事先把话说清楚的麻烦。把话说清楚有两大好处：第一，能避免日后因理解不同产生纠纷，强烈建议将具体细节书面化并签字确认；第二，如果在谈清楚的过程中对方表现出不悦或不配合，这本身就是一个信号，说明对方可能并非理想的合作伙伴。

**请务必分清日常情谊和生意合作，并确保你的合作伙伴也能分清这两者。**

## 朋友间如何谈钱

现在，我们从工作场合转向朋友间如何谈钱。首先绕不开的话题就是“借钱”。

**基于多年的个人与社会经验，我的核心建议是：朋友间，能不谈钱就不谈钱。**

这并非钱本身有问题，而是“人”的问题太不可控。下面我分享几段亲身经历来说明。

**第一段经历发生在我大学刚毕业时。**

一位玩得好的朋友因买房开口借十多万，我出于义气和不好意思拒绝了。结果，只要回了寥寥几千元，这位朋友也从此失联。这段经历教训在于：社会经验不足，感情用事，拉不下面子，更重要的是完全没有评估对方的偿还能力、信誉和“使坏成本”。

**第二段经历大约在五六年前。**

一位有家庭、有稳定工作的朋友多次游说我投资某个项目。我考虑再三，最终决定以借款形式给予一笔钱，约定固定回报。事与愿违，我花了两年时间才拿回本金，回报自然无望。这次我判断对了对方的“使坏成本”较高（有家庭、有工作，赖账代价大），但不足在于没有进一步核实其配偶是否知情，后来发现问题时，其配偶果然不知情。

**第三段经历是关于一位前员工。**

他因急事开口借一笔对他而言数额较大的钱。我判断他短期内无力偿还，但事情听起来紧急，于是缩小了借款金额给他，并做好这钱送他的心理准备。出乎意料的是，过了一段时间，他如数归还并心怀感激。对于这种“不帮不是人”的情景，建议是：在力所能及的范围内提供帮助，平衡好自己的心理价位和良心，做好钱拿不回来的准备。

**第四种情况是与极少数知根知底、关系过硬的朋友之间。**

这种关系下的临时资金周转反而简单直接，双方都会把握分寸，不让对方为难，并且还款非常准时。

**综上所述，如果朋友间不得不涉及借钱，请注意以下几点。**

*   **评估偿还能力**：对方是否有稳定的收入或资产来保证还款。
*   **了解人品信誉**：你对对方的诚信度是否有清晰的了解。
*   **判断“使坏成本”**：对方如果赖账，可能付出的代价（如社会声誉、家庭关系、职业生涯损失）有多大。
*   **明确借钱目的**：遵循“救急不救穷，救穷不救大”的原则。对于有偿还能力的急用可以考虑；对于确实贫困的，借款金额不要超过自己能承受的风险范围。
*   **保持理智，不要感情用事**：如果因为不好意思而借钱，事后就不要懊悔。

**同时，如果你是开口借钱的一方，在有诚意和偿还能力的前提下，请务必做到以下几点。**

*   主动提供还款计划，告知确切的还款时间。
*   说明自己的偿还能力（例如，“我每月有固定收入X元，可以每月还您Y元”）。
*   主动立下完整的字据。
*   最重要的一点：准时还款。如有变动，务必提前沟通。

**关于如何讨债，有一句很现实的话：钱在谁手上，谁就有主动权。**

有时借钱出去，心里就要做好这钱要不回来的准备。因此，核心思路是把风险控制在最开始，理智判断，避免做傻事。

除了借钱，朋友间相处还有其他与钱相关的注意事项。

**第一，注意“人言可畏”，情面上最好互不相欠。**

相处应有来有回，例如今天你请客，明天我买单。再比如，请朋友从国外代购，无论价值高低，最好事先把钱给对方，而不是让对方垫付。这些细节会影响你在朋友圈中的评价。

**第二，尽量坦诚自己的经济情况与经济实力。**

不必刻意“装富”或“装穷”。装富会很累，且容易陷入高消费陷阱；装穷则可能让人误解。在朋友了解你真实情况的前提下，量力而行的慷慨反而更显真诚和可贵。

**第三，别人有钱是别人的事，你不能觉得“理所应当”（take advantage）。**

结交经济条件好的朋友无可厚非，但不能抱有“他有钱，所以他请客、送礼、帮忙都是应该的”这种心态。要懂得感恩和回馈，在自己的能力范围内给予反馈。真正的有钱人往往更看重情谊和人品，而非小恩小惠的价值。

**第四，不得不承认，在某些人脉和感情投资层面，的确需要花钱。**

请客吃饭、送礼、共同出游等，既是证明实力的一种方式，也是表达诚意的途径。因此，要学会把钱投资在值得投资的人身上。

## 本节课总结

本节课中，我们一起学习了在不同场景下如何理性看待和处理与钱相关的问题。

1.  **工作中谈钱**：求职时要基于市场和自身价值设定合理期望；谈加薪要遵循“先有价值、再有加薪”等原则，注意方式方法。
2.  **合作中谈钱**：务必事先厘清所有资金相关的权责利，书面确认，将生意与情谊分开。
3.  **朋友间谈钱**：借钱需极度谨慎，重点评估对方偿还能力、人品和使坏成本；日常相处要互不相欠、坦诚相待、懂得感恩，并理性进行必要的人际投资。

希望这些经验和原则能帮助大家在涉及“钱”的事务上更加清晰、理智，保护好自己也维护好关系。
