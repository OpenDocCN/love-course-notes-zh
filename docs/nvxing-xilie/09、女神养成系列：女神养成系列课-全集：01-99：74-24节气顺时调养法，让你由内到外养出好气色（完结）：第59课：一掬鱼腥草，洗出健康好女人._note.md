# 女神养成系列：59：一掬鱼腥草，洗出健康好女人

在本节课中，我们将学习妇科疾病的日常护理，特别是如何正确地进行外阴清洗。我们将探讨常见的误区，并介绍一种既经济又有效的中草药保健方法。

## 课程内容

上一节我们讨论了妇科护理的一些基本观念，本节中我们来看看具体的清洁方法与注意事项。

我遇到过一位女性患者。她因冬天泡温泉而感染了妇科病，经医院输液治疗才痊愈。她讲述病情时仍心有余悸。她认为这样就能传染妇科病，以后不敢再去公共场合。她自称非常注意个人卫生。和上节课提到的夫妻患者一样，她家中备有多种洗液。她每天清洗，有时一天两三次，近乎洁癖。她的理论是越干净就越不该生病。但这次泡温泉经历让她发现，越是讲卫生，反而越容易感染和患上妇科病。她带着疑问问我：到底该洗还是不洗？

毫无疑问，需要清洗。但关键在于方法，不能无节制地乱洗。过度清洗不仅无法防病，反而会降低阴道免疫力，更易生病。清洗分为两种：保健性清洗和治疗性清洗。

保健性清洗指在一般情况下，仅使用温开水清洁外阴，不添加任何药物。在没有外阴瘙痒、白带异常或阴道炎的情况下，只需使用蒸馏水清洗。蒸馏水是妇科清洗的最佳选择。

说到这里，有人会问：阴道内有菌不就是生病吗？并非如此。阴道本身就是一个有菌环境。这些生理性菌群如同清洁工，能抵御外来有害菌群的入侵。此时若使用抗菌杀虫的中西药清洗，会清除所有菌群，打乱阴道内菌群平衡，破坏其自我调节能力。严重来说，这种盲目清洗是一种暴力干预，反而给细菌病毒可乘之机。前述泡温泉的女性正是如此，过度洁癖反而害了她。

第二种是治疗性清洗，即使用西药类消毒液或中成药煎剂清洗外阴。西药旨在抗菌消毒，用于治疗滴虫、霉菌等引起的阴道病。中药如蛇床子、苦参、黄柏等，能清热燥湿、解毒杀虫。当出现白带异常、发黄、恶臭、呈豆腐渣状，或外阴瘙痒难忍，被诊断为阴道炎、外阴炎时，无论是滴虫还是霉菌感染，都应立即使用洗液清除病菌。此时使用西药消毒很有必要。

操作方法是先用碘伏局部消毒外阴，再用中成药洗液先熏后洗。洗后约一小时，用蒸馏水冲洗干净，这样效果最佳。

使用外洗液时需注意一点：不要认为药物浓度越大，抗菌杀虫效果越好，这是错误的。外阴和阴道在西医解剖学上是粘膜层，非常脆弱，容易因洗液药物浓度过高而受伤。曾有一位女性治病心切，使用洗液浓度是平时的两倍，结果灼伤阴道粘膜形成溃疡，外敷云南白药半个月才痊愈。因此，女性使用外洗液时必须严格按比例配制。这个配比浓度经过临床实践验证，只有这样才能达到治病不伤身的目的。

此外，有一种很好的外洗保健方，既简单省钱，又能防病治病，不会像西药产生耐药性，也不会扰乱阴道内菌群平衡。这就是鱼腥草。去药房购买半斤鱼腥草，可用半年。每次取3克，煎煮2000毫升水。每周清洗三次，可隔日一次或连续清洗三天。

我们生活在这个大家庭中，除非不食人间烟火，否则难免接触公共场合。在游泳、泡温泉、出差住旅馆、使用公共浴室或性生活后，都可以用鱼腥草清洗。鱼腥草能清热解毒、利水消肿，可治疗泌尿系统感染、妇科白带异常、湿疹、疥癣等。最关键的是，它是一种可以煎汤外洗的中药。并非所有中草药都适合外洗，只有少数草叶类中草药才具有治疗效果。中医常用鱼腥草治疗妇科病，西医同样将其广泛应用于临床，用以消炎抗病毒、治疗妇科疾病，甚至对咳嗽、气管炎、肠炎等也有很好疗效。

清洗方法很简单：用3克鱼腥草煎汤，这是保健外洗的用量，而非治疗量。洗后同样需要等待约一小时，再用蒸馏水冲洗干净。鱼腥草在带来经济实惠的同时，也能助你预防妇科疾病。但怀孕后需注意，保健外洗量也有药性，过度清洗可能动胎气，应停止使用。

最后，强调一下女性湿热带下需特别注意的方面。我们常说湿热，湿热从何而来？凡事有因有果。吃辣可开胃，喝酒可释放压抑情绪，但吃辣过多会上火，喝酒过量会口干口渴。当你感觉小便发黄、有灼烧感时，就代表湿热已至。女性阴道本是津液濡润的环境，这些坏习惯损耗体内阴液，导致阴道粘膜充血水肿，分泌物气味加重、白带增多，例如呈豆腐渣样。由此可见，许多阴道炎、带下病都是自身不良习惯造成的。
