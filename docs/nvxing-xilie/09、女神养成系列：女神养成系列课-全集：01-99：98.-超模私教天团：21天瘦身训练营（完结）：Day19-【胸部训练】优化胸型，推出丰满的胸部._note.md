# 女神养成系列：09：98. 超模私教天团：21天瘦身训练营（完结）：Day19 【胸部训练】优化胸型，推出丰满的胸部 💪

在本节课中，我们将要学习第19天的训练内容，即“推日”训练。本节课程以推类动作为核心，旨在锻炼胸部、肩部和手臂后侧的三头肌，帮助优化胸型，塑造挺拔丰满的胸部线条，同时紧致手臂。

## 课程概述与目标 🎯

推日训练的主要目标肌肉群是胸大肌、肩部以及手臂后侧的三头肌。对于女性而言，进行胸部训练有助于提升胸部线条，使胸部看起来更挺拔。同时，作为辅助肌群的三头肌得到锻炼，也能让手臂线条更加紧致漂亮。训练开始前，必须进行充分的热身。

## 热身环节 🔥

以下是三个热身动作，每个动作重复15次，旨在激活目标肌群，预防受伤。

**动作一：掌心向上屈肘**
*   掌心向上，屈肘活动肩关节与肘关节。

**动作二：俯身小臂后伸**
*   俯身，腰背收紧，肘部抬高，屈小臂向后伸展。

**动作三：侧平举**
*   肩胛骨收紧下沉，挺胸抬头，进行侧平举动作。

热身完成后，我们正式进入训练环节。

## 第一大组训练 💦

上一节我们完成了热身，本节中我们来看看第一个大组的训练内容。本大组包含三个动作，需循环完成三遍，每遍之间休息90秒。

以下是本大组的动作列表：

**动作一：俯撑**
*   **要领**：肘关节收紧，与身体保持约75度夹角，腰背挺直。感受三头肌发力。
*   **公式/代码**：`动作次数 = 15次`

**动作二：健步蹲**
*   **要领**：身体不要前趴，前腿膝盖不超过脚尖，后腿膝盖弯曲小于90度。
*   **公式/代码**：`每侧动作次数 = 15次`

**动作三：俯卧登山**
*   **要领**：重心不要靠后，臀部不要抬起或弓背。吸气时勾起脚尖。一边一次算一次。
*   **公式/代码**：`动作次数（左右合计）= 15次`

完成三遍循环后，休息2分钟，进入第二大组。

## 第二大组训练 🏋️‍♀️

在完成了以自重训练为主的第一大组后，第二大组我们将加入哑铃，进行推举和划船训练，并锻炼核心。

以下是本大组的动作列表：

**动作一：站姿哑铃推举**
*   **要领**：肘关节不要内扣或外撇，不要含胸或过度仰身。推举时哑铃呈A字型轨迹。
*   **公式/代码**：`动作次数 = 15次`

**动作二：俯身哑铃划船**
*   **要领**：划船轨迹朝向髋关节，而非肋骨。背部与肩胛骨收紧，尽量减少身体借力。
*   **公式/代码**：`动作次数 = 15次`

**动作三：仰卧起坐**
*   **要领**：区别于卷腹，需让躯干贴向大腿正面，锻炼整个腹肌群。
*   **公式/代码**：`动作次数 = 15次`

同样完成三遍循环后，休息2分钟，进入最后的第三大组。

## 第三大组训练 ⚡

这是今天的最后一个大组，包含高强度的复合动作，旨在提升心肺功能并强化核心。

以下是本大组的动作列表：

**动作一：移动俯卧撑**
*   **要领**：在标准俯卧撑基础上进行左右移动，过程中保持核心收紧，感受胸部和三头肌发力。
*   **公式/代码**：`动作次数 = 15次`

**动作二：波比跳**
*   **要领**：蹬腿时不要塌腰，不要耸肩低头，背部保持挺直收紧。
*   **公式/代码**：`动作次数 = 15次`

**动作三：仰卧抬腿**
*   **要领**：腿部伸直，全程不要接触地面，保持悬空。抬腿时吐气，下放时吸气。
*   **公式/代码**：`动作次数 = 15次`

完成三遍循环后，所有训练动作结束。

## 拉伸与冷却 🧘‍♀️

训练结束后，必须进行拉伸，帮助肌肉恢复，缓解酸痛。

**主要拉伸部位与方法**：
1.  **胸大肌拉伸**：手臂后伸，维持15秒。
2.  **三头肌拉伸**：手臂过头弯曲，另一手向后拉肘部，维持20秒。
3.  **肩部与背部拉伸**：手臂横过胸前拉伸，维持15秒。
4.  **股四头肌拉伸**：俯身后用手拉住脚背，维持20秒。
