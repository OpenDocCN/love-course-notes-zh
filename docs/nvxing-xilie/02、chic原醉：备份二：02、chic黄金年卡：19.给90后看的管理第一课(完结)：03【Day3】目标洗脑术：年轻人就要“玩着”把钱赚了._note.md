# 团队管理术：02：目标洗脑术：年轻人就要“玩着”把钱赚了

## 概述
在本节课中，我们将要学习团队管理的核心技能之一：目标激励与管理。我们将探讨如何通过科学的方法设定目标，如何进行有效的考评反馈，以及如何处理考评中常见的挑战，从而让团队更高效、更快乐地达成业务目标。

---

## 第一部分：搭建目标激励模型

上一节我们探讨了领导责任与业务责任的区别，明确了管理的最终目的是确保业务目标的达成。因此，目标管理是团队管理工作的重中之重。本节中，我们来看看如何搭建一个有效的目标激励模型。

许多管理者都有过这样的经历：制定了目标和计划，最终却因各种原因未能实现。其中一个不易察觉的关键原因，就是目标本身存在问题。团队目标管理一旦出现问题，不仅会阻碍公司整体业务推进，还会引发下属不满，使管理能力受到质疑。

那么，什么样的目标激励模型能轻松激励下属，使其乐于向业务目标奋进呢？

这个模型就是 **SMART模型**。

以下是SMART模型的具体含义：
*   **S** 代表 **具体** (Specific)：目标必须清晰明确。
*   **M** 代表 **可度量** (Measurable)：目标必须可以量化衡量。
*   **A** 代表 **可实现** (Achievable)：目标必须是团队付出努力后能够达成的。
*   **R** 代表 **相关性** (Relevant)：目标必须与团队及公司的整体业务高度相关。
*   **T** 代表 **时间限度** (Time-bound)：目标必须有明确的完成时限。

当你为团队设立的业务目标符合以上原则时，这样的目标最易激励团队成员接受并乐于追求。

例如，你带领一个客服团队，当前业务问题是“客户意识不强”。如果直接将团队目标定为“增强客户意识”，这显然不符合SMART原则。每个下属会依据自己的理解执行，缺乏统一、可量化的标准，最终必然导致考核冲突。

此时，运用SMART模型来设立目标就能很好地帮助你。你可以将“增强客户意识”拆解为两个具体目标：
1.  **减少客户投诉**：统计过去客户投诉率为3%，目标是在一个月内将其降低到1.5%。
2.  **提升服务速度**：原来解决一个客户问题平均需要7天，目标是将答复时间缩短至4天内。

这样，目标变得具体、可度量、有时限，团队成员执行起来方向明确，考核标准也一目了然，能有效避免不必要的管理冲突。在现实管理中，智慧的管理者善于设定清晰的KPI并合理分配利益，而非空谈口号。

## 第二部分：针对目标管理的考评反馈技巧

掌握了如何设定目标后，接下来我们需要学习如何对目标的完成情况进行考评与反馈。作为团队领导者，考评反馈是必不可少的环节。

考评反馈主要有两种方式：团队反馈和一对一反馈。团队反馈相对容易，可以借助业务复盘会公开透明地进行，还能起到激励竞争的效果。然而，让许多新晋管理者感到棘手的，反而是与下属的一对一面谈反馈。很多管理者最初进行面谈时，难以把握谈话的深度与内容，常常使面谈变成自己的“独角戏”，效果不佳。

那么，与下属进行一对一面谈反馈时，该怎么做才好呢？建议你把握住一个原则和一个法则。

即 **提问原则** 和 **28法则**。

这意味着，在面谈中，你将80%的时间留给员工发言，自己只占用20%的时间。而在自己这20%的时间里，你可以用80%的时间来提问，只用20%的时间来给予指导或建议。因为下属通常比你更清楚自己工作中存在的问题，你只需要通过提问引导他们自己思考和解决，让他们自我评价工作进展，而不是一味讲道理或发号施令。

理解了一个原则和一个法则后，如何调动下属，让他主动利用那80%的面谈时间呢？这里分享一个四步面谈策略：

以下是具体步骤：
1.  **指出错误行为**：具体、客观地描述下属工作中出现的问题。
2.  **说明行为后果**：清晰阐述该错误行为对团队、业务或公司造成的负面影响。
3.  **征求改进措施**：询问下属如何看待这个问题，并引导他提出自我改进的方案。
4.  **共同展望未来**：与下属探讨未来的发展机会，给予激励。

例如，人力资源部门助理M在发放录用通知书时频繁出错。使用上述面谈策略，你可以这样与M沟通：
*   **第一步**：“3月7号，你发放的录用通知书又出现了错误，候选人对应的期权数值填错了。这已经是第二次出现同类错误。”
*   **第二步**：“由于这个工作失误，我们在与候选人沟通入职时遇到了很大困难，并且增加了其他同事的工作量来补救。”
*   **第三步**：“你怎么看待这个问题？你认为我们可以采取哪些措施来避免它再次发生？”（此时是M发挥的时间，你要通过延展性、深入性的提问引导M充分表达。）
*   **第四步**：“很好，我同意你的改进意见。希望在接下来的工作中可以看到你更出色的表现。团队即将启动一个重要项目，我希望把这个机会留给你。”

这就是考评反馈中非常好用的一个原则、一个法则和一个面谈策略。如果你也对一对一面谈感到有压力，建议在面谈前参照这个技巧进行预演。

## 第三部分：破解绩效考评中的常见问题

掌握了目标设定与反馈技巧后，我们来看看新晋管理者在执行绩效考评时容易遇到的两个典型问题及其解决方法。

第一个问题是 **下属对考评结果的不满与异议**。例如，下属看到与工资挂钩的绩效考核结果后非常愤怒，抱怨“这不公平”、“我知道你不喜欢我”、“这不合理”。面对这样的情绪爆发，否认或争辩都不是最佳策略。

最好的办法是先认同下属的情绪，并尽量让其释放，而非急于压制。

你可以这样说：“我能看到你对这个结果非常难过，我很想听听你的意见。” 当下属充分表达后，他的情绪会得到释放。此时，你再重申具体事例，并解释自己的考评观点，下属就比较容易接受了。

之所以采用这种策略，是因为处理管理冲突时，最重要的是管理对方的情绪。管理的本质之一就是管理情绪。让情绪先得到释放，理性沟通才能成为可能。

说过了情绪激动的情况，我们再看看完全相反的另一种情况：**面对考评非常“佛系”，考评对他们起不到激励作用的下属**，该怎么办？

这里教你三个方法：聊承诺、聊后果以及赋予责任。

例如，下属S对任何考评结果都反应平淡，总是说“还好吧”、“我觉得还行”。要调动S，你可以尝试这样沟通：首先，让S思考自己的发展领域，确认他能否为改进业务目标做出具体承诺，并将承诺结果作为下一次的考核目标。如果他拒绝做出承诺，就需要和他聊聊可能产生的后果，例如可能会被调离核心岗位等。这既能稍微敲醒S，也能为你后续的用人决策铺路。

此外，面对佛系下属，还有一种方法是赋予他更大的责任。例如，你可以为S安排一个“团队观察员”的角色，让他观察并反馈团队中出现的问题以及相应的解决办法。这能迫使S的思维脱离舒适区，重新活跃起来。

## 总结
本节课中我们一起学习了目标激励与管理的核心知识。

第一，能够轻松激励下属的目标管理模型是 **SMART模型**。设立团队业务目标时，需遵循清晰具体、可执行、可量化、高相关性及可实现性原则。

第二，进行一对一的考评反馈时，可运用三点技巧：**提问原则**、**28法则**以及一个四步**面谈策略**。即将80%的时间留给下属发言，自己用20%的时间，其中80%用于提问，20%用于指导。

第三，面对下属对考评的不满，最重要的是先让其释放负面情绪，再探讨解决方法。而面对提不起精神的佛系下属，考评反馈应遵循聊承诺、聊后果并赋予责任的路线。
