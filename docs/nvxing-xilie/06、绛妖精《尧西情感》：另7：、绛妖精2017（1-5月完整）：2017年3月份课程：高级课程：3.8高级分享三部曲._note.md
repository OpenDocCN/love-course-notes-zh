## 性在婚姻中的核心地位

正常的夫妻之间一定需要必要的夫妻生活来维系。千万不要低估了性生活的力量。当然，如果你觉得柏拉图式的恋爱也可以天长地久，那说明你们之间的无性生活在彼此的接受范围之内。

## 后续课程预告与答疑

我的第二部曲是**手把手教你营造和谐性爱的氛围**。一个和谐的性爱氛围对性生活质量非常重要。

女性性冷淡或无性高潮也多与心理因素、前戏不足或沟通不畅有关，也是可以治疗的，通常需要双方共同参与性治疗。
