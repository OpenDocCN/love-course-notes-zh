# 产后修复瑜伽：02：腹直肌修复与核心强化 🧘‍♀️

在本节课中，我们将学习如何通过特定的瑜伽练习来修复产后腹直肌分离，并强化核心肌群。课程将详细讲解腹直肌分离的原理、自我检测方法，并提供一系列安全有效的修复动作。

## 腹直肌分离概述

上一节我们介绍了产后修复的基础知识，本节中我们来看看腹直肌分离的具体问题。腹直肌位于腹部前侧，是构成“马甲线”或“腹肌”的主要肌肉。在怀孕期间或过度肥胖的情况下，腹直肌可能被撑开并向两侧分离，导致腹部松弛、腰围变宽。

盲目进行卷腹等传统腹部练习，可能会加重腹直肌的分离，造成二次伤害。因此，修复的关键在于通过锻炼腹外斜肌等侧腹肌肉，将分离的腹直肌“包裹”并拉回中线位置。

## 腹直肌分离自测方法

在开始练习前，你需要了解自己腹直肌的分离程度。以下是自我检测的步骤：

1.  仰卧在垫子上，双膝弯曲，双脚踩在臀部前方。
2.  双手抱头，将腹部露出。
3.  将手指放在肚脐下方的腹中线上。
4.  呼气时，卷腹收腹，用手指左右触摸。
5.  你会摸到两条较硬的肌肉，测量它们中间的缝隙宽度。

**判断标准**：
*   缝隙宽度小于两指：属于正常范围，但最好修复到一指以内以获得更紧实的腹部。
*   缝隙宽度大于两指：需要进行修复练习。若分离严重（如三指以上），建议咨询医生。

## 练习准备与辅助工具

本课程会使用一些辅助工具来帮助初学者更安全、有效地找到发力感并加强练习效果。

以下是推荐的辅助工具：
*   **波速球（或小波速）**：用于制造不稳定的支撑面，迫使核心肌群自动收紧。
*   **普拉提小球**：用于夹在双腿或脚踝间，帮助启动大腿内侧肌群，防止膝盖外翻。

## 核心修复练习序列

### 热身：波速球肘板支撑

首先，我们用一个不稳定的支撑动作来激活核心肌群。俯卧，将手肘放在波速球上，进入简易的肘板支撑。

**动作要点**：
*   肩胛骨向后拉，脊柱向前延展。
*   脚跟向后蹬，大腿向上提，保持背部平直。
*   结合盆底肌收缩：吸气时拉长脊柱，呼气时收紧腹部和盆底肌。
*   保持30秒到1分钟。

### 侧腹强化系列

接下来是一系列针对腹外斜肌的练习，这是修复腹直肌的关键。

**动作一：侧板支撑（波速球辅助）**
侧卧，下方手肘支撑在小波速球中心。髋部向上提起，上方手臂伸直。

**动作要点**：
*   肩膀远离耳朵，身体呈一条直线。
*   髋部提得越高，侧腹收缩越强烈。
*   每侧保持30秒到1分钟。

**动作二：波速球交替提膝**
呈肘板支撑，双手肘分别放在两个波速球上。呼气时，交替将膝盖向前靠近同侧手肘。

**动作要点**：
*   保持背部平直，避免塌腰。
*   收紧腹部和盆底肌。
*   每侧完成15次为一组。

**动作三：侧板髋部升降**
在侧板支撑的基础上，吸气时髋部缓慢下落，呼气时用力将髋部向上提起。

**动作要点**：
*   动作缓慢且有控制。
*   下方手掌可轻触地面以保持平衡。
*   每侧完成10-15次。

**动作四：侧卧卷腹**
侧卧，下方手臂前伸，头枕于其上。双腿屈膝，上方手抱头。呼气时，上方手肘与臀部同时向上抬起，向中间挤压侧腰。

**动作要点**：
*   发力点是侧腹，而非用腿抬起。
*   肩膀不要向前送，保持身体正位。
*   每侧完成15次。

### 仰卧核心整合

**动作五：仰卧双腿扭转**
仰卧，双腿抬起至90度，脚踝夹住波速球。手臂体侧平举。呼气时，收紧盆底肌和腹部，将双腿有控制地倒向一侧，保持对侧臀部不离地。

**动作要点**：
*   肩膀尽量贴地。
*   双腿大腿做外旋。
*   左右交替为一次，完成10次。

**动作六：仰卧卷腹（小球辅助）**
仰卧，双膝弯曲，双脚踩地，双腿间夹住普拉提小球。双手抱头。呼气时，卷腹起身，同时做转体，让对侧手肘寻找对侧膝盖。

**动作要点**：
*   起身时下巴微收，肩膀远离耳朵。
*   夹球的目的是防止膝盖外翻，帮助稳定。
*   可在起身后保持一个呼吸，再缓慢落下，增加肌肉持续发力时间。
*   左右交替为一次，完成10次。

**动作七：仰卧举腿点地**
仰卧，双手放于体侧。双腿抬起至90度，脚踝夹球。呼气时，收紧核心，将脚跟轻轻点地，保持3秒，吸气时抬起。

**动作要点**：
*   动作要慢，想象有阻力阻止你下落和抬起。
*   全程保持盆底肌和腹部的收紧。
*   完成15-20次。

### 挑战：波速球平衡坐姿

坐在小波速球（凸点面朝上）上，屈膝。稳定后，尝试将双脚抬离地面，双手向前伸直或放在体后支撑。

**动作要点**：
*   坐骨坐稳，脊柱向上延展。
*   腹部向内收紧，盆底肌上提。
*   保持平衡，挑战30秒。

## 练习注意事项与总结

本节课中我们一起学习了针对腹直肌分离的修复性练习。核心要点总结如下：

1.  **安全第一**：剖腹产妈妈需伤口完全愈合（通常4个月）并经医生许可后再练习。顺产妈妈出月子后可开始。
2.  **重在侧腹**：修复期应避免传统卷腹，重点强化腹外斜肌，从两侧将分离的腹直肌拉回。
3.  **结合盆底肌**：几乎所有动作都应在呼气时同步收缩盆底肌和腹部，实现协同修复。
4.  **质量优于数量**：动作应缓慢、有控制，感受肌肉的发力与收紧，避免追求速度和次数。
5.  **循序渐进**：根据自身体能选择练习时长和组数，每周逐步增加，给身体适应的时间。
