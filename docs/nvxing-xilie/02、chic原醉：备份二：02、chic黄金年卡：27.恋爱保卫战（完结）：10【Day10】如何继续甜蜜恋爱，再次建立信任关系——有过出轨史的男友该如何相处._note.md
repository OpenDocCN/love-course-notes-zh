# 恋爱保卫战：02：如何重建信任关系——与有过出轨史的男友相处指南

## 概述

在本节课中，我们将学习如何在与有过出轨史的男友复合后，重建被破坏的信任关系，并让恋情重新走向稳定与甜蜜。我们将探讨信任的本质、重建信任的具体方法，以及如何设定并维护个人底线。

---

信任是亲密关系的基石。当信任因背叛而破裂后，即使双方决定继续在一起，关系也常常会陷入尴尬与不安。上一节我们探讨了信任破裂后的普遍困境，本节中我们来看看如何系统地重建信任账户。

人与人之间信任最贫瘠的时刻并非相识之初，而是信任被打破之后。相识之初，只要对方未做出令人失望的行为，信任就能像往账户里存钱一样逐渐积累。如果伴侣身边充满诱惑并不断做出令人失望的行为，就如同不断从账户中取钱，使信用额度越来越小。而伴侣的背叛，则像账户被盗，导致信任账户瞬间归零。

因此，决定继续恋情后，首要任务不是急于恢复之前的恋爱状态，而是耐心地恢复信任账户的额度。

你可能会认为，重建信任意味着男友必须做到细心温柔、消息秒回、随时报备。起初，怀有愧疚感的男友或许会尽力满足这些要求。然而，长期如此，他会开始质疑自己的努力为何换不回信任。面对你时，他心中充满的将不再是爱意，而是沉重的愧疚感，最终可能导致其逃避甚至关系再次破裂。

那么，在重建信任的过程中，最关键的问题是：**如何展现你对男友的接纳，让他获得被信任的感觉？**

以下是帮助你达成此目标的两个核心方法。

### 方法一：润物无声——运用重复信息法卸下愧疚包袱

通过在日常对话中不断重复表达对男友的信任，可以帮助他逐渐从愧疚感中脱离，让恋情重回正轨。

具体做法是，用一万句“花式”甜言蜜语，不断向男友传递一个核心信息：**“我相信你。”**

其中，最关键的是在敏感问题上展现出坦然的信任。例如，当男友报备今晚要与朋友聚会并发送地址时，你可以大方回应：

```
“好好去玩吧。这么点小事以后不用给我发定位了，记得给我打包一份宵夜哦。”
```

在修复信任的初期阶段，暂时不要提出想一同参加聚会等请求。这类平时无关紧要的要求，在敏感时期可能被男友误读为不信任的表现。

当你感觉双方关系不再过分拘谨后，便可以引入第二个方法。

### 方法二：运用皮格马利翁效应——用赞美引导男友成为理想伴侣

当男生犯错后回归，常会在心里给自己贴上“坏男友”的标签，准备接受质疑和考验。这个过程过于痛苦，可能导致许多男生最终逃离。

为了帮助男友度过这个时期，你需要给予正确引导，帮助他成为你理想中的伴侣。这时，“皮格马利翁效应”（又称“期待效应”）便能发挥作用。

美国心理学家罗森塔尔和雅各布森提出的**皮格马利翁效应**指出：当一个人获得另一个人的信任与赞美时，他会感觉获得了社会支持，从而增强自我价值。为了持续获得这种支持，他会尽力达到对方的期待。

也就是说，如果你表达出对男友的肯定，并展现出享受被爱的感觉，他会为了继续获得你的肯定，而努力按照你的期待来宠爱你。

当男友为你吹头发、做饭或带你出去玩时，你都可以告诉他：

```
“我觉得好幸福啊，你真的是一个好男友。”
```

你甚至可以表现得惊讶一点：

```
“哇，你怎么这么爱我，你真是一个好男友。”
```

你还可以高调地在朋友圈展示幸福感，分享男友送的礼物或为你做的小事，配上甜蜜的文案。

你可能会问：难道只需要我主动示好，男友就不需要做任何事来重建信任吗？接下来，我们就谈谈男生需要承担的责任。

---

当男友重新回到你身边，他需要自发地对自身行为进行约束，即对你做出有效的承诺。

罗伯特·西奥迪尼在《影响力》一书中提出的**承诺与一致原理**指出：人一旦做出承诺，其后续行动都会倾向于与承诺保持一致。这是因为个人内部和外部的压力会迫使我们言行一致。

你可能会说，男友已经口头承诺不再与“莺莺燕燕”来往。但这并不足以让承诺发挥效用。要让承诺产生约束力，必须为男友的承诺制造外在压力，驱使他做出符合承诺的行为。

如果只是单纯的口头承诺，缺乏外在压力，男友即使违背诺言，也不会产生强大的心理负担。因此，要让承诺生效，需要满足以下三点：

以下是承诺生效的三个必要条件：

1.  **主动承诺**：男生需发自内心、主动地做出承诺。这就是为什么首先要帮助男友卸下愧疚感——让他在没有压力的情况下做出承诺，确保这是出于他真实的想法，而非为了暂时安抚你。
2.  **书面承诺**：将口头承诺转化为白纸黑字。书面承诺在潜意识里更具正式感，能驱使人们更认真地对待。同时，书面形式也使承诺内容更明确，为公开承诺做准备。
3.  **公开承诺**：承诺需要被公开。当我们对周围有影响力的人（如朋友、家人、同事）做出承诺后，为了维持在他们心中的既定形象，我们会迫使自己按照承诺行动。

例如，许多人在朋友圈打卡学习或成立学习小组，就是利用公开承诺和他人监督来激励自己。

这里的公开承诺，并非要求男生大声向亲友宣布“我再也不会出轨”，而是通过与你一同出席家庭聚会、公司活动等亲密行为，间接向他的社交圈宣告你是他认定的伴侣。

你可以将书面承诺与公开承诺结合，例如让男友在朋友圈持续“秀恩爱”，同时经常参与他的家庭聚会、公司活动。在他最在意的亲朋好友和同事领导面前建立其“完美男友”的人设，进而促使他自发约束自身行为。

---

即使男友做出了承诺，你可能仍会担忧：如何让他不敢再犯，乖乖遵守底线？

你必须明确自己的底线，并让男友清楚知道：一旦他再次越过底线，将不可能得到原谅。许多女孩会遇到男友“旧病复发”，就像戒烟者无数次被抓到偷偷吸烟却无可奈何。对于与异性的关系也是如此，即使说好不再联系，但再次被发现时，你可能依然费心解决却选择留下。

因此，你需要用自己的行为告诉男友：你有很强的底线意识，一旦被触碰，绝不会心软。

什么样的人在生活中看起来特别有底线意识？答案是：**非常自律、说到做到、绝不打破既定规矩的人**。

例如，你下定决心戒糖后，绝不会再点奶茶；决定减肥后，每天下班都会去健身房。相反，如果你总是无法完成对自己的约定，男友就会潜意识里认为，你设立的规矩连自己都无法遵守，那么他也可以轻易突破你的底线。

想要男友尊重你的底线，就要先学会遵守自己的底线。你可以定期为自己设定小目标并不断完成，以此强化你在男友心中的影响力。

例如，你决定践行“断舍离”，**三个月不买一件新衣服**。这对于习惯“剁手”的你而言是挑战，但若能坚持下来，必定让男友感到讶异。或者，你平时很难读完一本书，但若能**连续读完几本大部头著作**，定能让男友体会到何为“士别三日，当刮目相待”。

---

## 总结

本节课中，我们一起学习了如何与有过出轨史的男友重建信任关系：

1.  **表达信任与积极引导**：首先，通过日常小事表达对男友的信任，再运用皮格马利翁效应，肯定他对你的好，正面促进他的行为。
2.  **获取有效承诺**：其次，引导男友对你做出书面承诺和公开承诺，利用外部压力促使他对自我行为进行约束。
3.  **树立并坚守底线**：最后，通过不断达成个人目标来展现并强化自己的底线意识，让男友学会尊重你的底线。

希望本课内容能帮助你建立一段忠诚度与甜蜜度并存的亲密关系。
