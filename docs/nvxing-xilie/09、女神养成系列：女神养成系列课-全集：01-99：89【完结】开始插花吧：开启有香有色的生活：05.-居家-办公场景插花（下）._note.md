# 花艺教程：05：居家与办公场景插花（下）

## 概述
在本节课中，我们将学习如何为办公场景设计和制作一束清新淡雅的花艺作品。我们将从花材选择、色彩搭配到具体的插花步骤，一步步完成一个能营造轻松、舒心氛围的办公桌花束。

上一节课程我们分享了家居花艺的要领，包括颜色与氛围的衬托和搭配。本节中，我们来看看办公场景的花艺应如何设计。

通常情况下，办公场景的颜色偏向淡雅。整个氛围处于忙碌状态中。我们需要用颜色营造出氛围中比较轻松的感觉。在工作时，也要让自己有放松的时间。其次，颜色的整体组织要尽可能看起来清新和淡雅。这样能让自己和同事在看到花束时感到舒服和舒心。

接下来介绍选择今天这些花材的原因。主要的颜色搭配以白绿色系为主，点缀一点点黄色。在办公室氛围里，我们想要营造一种安静和舒心的感受。这些颜色看起来清新淡雅且安静。下面具体介绍每一种花材。

以下是今天使用的花材列表：
*   **橘叶**：一种叶材，用于构建基础结构。
*   **黄金球**：花材，外形是小而圆的黄色球体。
*   **芬德拉玫瑰**：单头玫瑰品种，特点是白色中透出一点点肉色，让白色调看起来更温暖。
*   **绿精灵小雏菊**：花材，外形圆圆小小，显得非常活泼。
*   **洋甘菊**：进口花材，香气清新，有助放松。
*   **浅绿色洋桔梗**：多头花材，花型与玫瑰相似但无刺，感觉美好。

接下来演示如何打造这样一束花。上一次课程以打螺旋的方式教学。今天则以插花的方式，将花一枝枝插入花瓶。大家可以根据自己的喜好选择不同的插花方式。这完全可以凭个人个性发挥。

## 插花步骤演示

**第一步：构建基础结构（打龙骨）**
首先选择放置叶子。放置叶子是一个打龙骨的过程。只有把结构做好，整束花的效果才会美好。因此先从叶子开始。叶子已修剪完毕。接下来量取高度，决定修剪程度。

第一枝量好长度，直接插入。
第二枝希望高度稍矮一些，可以修剪到合适长度。这个完全取决于个人，无所谓对错。这样有高有低，整体感觉更有变化性和丰富感。
第三枝希望高度能稍高一些。比好高度后进行修剪。

三枝龙骨已经打好。插花时要注意让花枝互相交叉。整个制作过程就像打螺旋一样，让方向顺势而为。

**第二步：增强结构**
龙骨打好后，需要放入一些能帮助增强结构的部分，选择放入小菊。和刚才的叶子一样，要先比一下高度。比好高度后，胸有成竹地剪下，一定记得是斜剪45度。

如果一开始插花觉得东倒西歪，没关系。可以在最后花材越来越多时进行调整。
第二枝小菊，希望它的高度能再矮一点点。根据实际高度进行修剪。如果按照希望的高度，需要剪掉很长的茎杆。剪长茎杆时有一个小诀窍：可以分段剪。这样垃圾桶能更好地容纳垃圾，更环保。将长茎杆分成3块剪掉。一定记得按照方向顺势而为。

现在，这束花已经有了一个大的框架。

**第三步：放入主花**
在放完龙骨花材后，接下来放入一些主花。首先放入芬德拉玫瑰。
第一支芬德拉玫瑰，希望它稍微高挑一点。比好之后给它一个参照。这么长的茎杆，仍然可以分段剪。45度斜剪。因为结构打得比较好，玫瑰可以支撑在里面。
再来另一支。这一支希望它的高度稍稍矮一点点。比好高度后进行修剪。

接下来放入洋桔梗。洋桔梗上的小花头虽然小，但体现的感觉非常可爱。一般情况下，如果不是必须，不会剪掉这种花头。但有时花头非常脆弱，在运输过程中可能脱落。如果脱落了，可以轻轻剪掉。或者如果花头是从某处折断，更没问题。可以把折断的花头留出来，稍后做小花束。

在插花过程中，高度和位置完全取决于自己。可以按照喜好高低不同地穿插。在整个穿插过程中，可以时不时转动一下，查看什么地方还比较空缺，接下来应该补什么位置。

**第四步：加入点缀与丰满造型**
放完几支洋桔梗后，放入最喜欢的洋甘菊。洋甘菊花头非常多，看起来活泼。希望它能在花束中稍高一点，起到点缀作用。插这枝花时要注意，因为花头非常蓬松，插入后会呈现收拢状态。没关系，稍微拨弄一下，让它从其他花中探出来一点。动作要轻柔，避免因用力拽拉使花头脱落。

花束的雏形基本已经出来，接下来继续丰满它。这束花插得高低参差不齐，这完全出于个人喜好，希望整体感觉非常自然。插入花后，如果发现有多余的叶子，放轻松揪掉即可。

再来调整一下位置，看看哪里不太满意。调整位置。到了现在这种状况，基本上想要的感觉已经出来了。但如果是放在办公桌前作为视线焦点，会觉得某块区域稍微有点空。因此准备在这个地方补一点不一样的东西。现在有小菊、洋桔梗和玫瑰。考虑补什么比较合适，觉得还是补一支玫瑰比较合适。可以让它的高度比其他几支稍高一点，但又不至于挡住后面出挑的部分。因此给它定一个这样的高度。

基本上，这束花的大形状已经完成。想在这个地方再加一点点稍高的东西。从现在的感觉上讲，是加入黄金球的最佳时机了。拿出刚才修剪好的黄金球。黄金球的茎杆很长，要记住它的作用是为了画龙点睛。画龙点睛的同时，也要让整体氛围感比较好。因此也要让它高低有些不同的造型，能给这束花提更多气。

第一支黄金球想要在某个位置盛放，同时让它出挑出来。按照这个想法定一个高度。塞不进去时，一定不要硬塞，要找对角度，让它能轻轻放进去。
最后一支黄金球打算放在另一个位置。

当整束花达到自己喜欢的状态时，可以再微微调整不同花材的位置，让它看起来更加丰满。办公桌前的这束花就完成了。

## 设计灵感总结
今天的办公室花艺，颜色灵感主要取决于大家在办公时压力大、很辛苦。因此希望有一些白色但透着温暖颜色的花，让大家在放空时也能感觉到一丝温暖和高兴。其次，大家都喜欢野外，希望把自然的颜色融入花束。同时还想要一点点小点缀的颜色，以及淡淡的清香味，让大家精神上更愉悦。所以这束花被打造成了白里透粉，同时有绿色和黄色点缀的作品。

## 总结
本节课我们一起学习了为办公场景设计花艺。我们选择了白绿色系搭配黄色点缀的花材，以营造清新、安静的氛围。通过先构建叶材龙骨、再添加主花与点缀的步骤，完成了一束高低错落、自然生动的桌花。关键在于根据个人喜好调整结构，并时刻观察整体造型进行微调，最终创造出一个能带来轻松与愉悦感的办公桌伴侣。
