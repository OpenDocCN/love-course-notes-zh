# 机器学习入门：01：线性回归基础

在本节课中，我们将要学习机器学习中最基础且重要的算法之一——线性回归。我们将了解它的核心思想、数学原理以及一个简单的实现示例。

## 概述

线性回归用于寻找自变量（X）与因变量（y）之间的线性关系。其目标是找到一条最佳拟合直线，使得所有数据点到这条直线的距离之和最小。

上一节我们介绍了课程概述，本节中我们来看看线性回归的具体定义。

## 核心概念与模型

线性回归模型试图用一条直线来拟合数据。其数学表达式是一个线性方程。

其核心公式为：

**y = wX + b**

其中：
*   **y** 是我们要预测的值（因变量）。
*   **X** 是输入特征（自变量）。
*   **w** 是权重（或斜率），代表特征对预测结果的影响程度。
*   **b** 是偏置（或截距），代表当所有特征为0时的基准预测值。

我们的目标是找到最合适的 **w** 和 **b**，让预测值尽可能接近真实值。

理解了模型形式后，我们需要一个标准来衡量模型的好坏，这就是损失函数。

## 损失函数：均方误差

为了找到最佳的 **w** 和 **b**，我们需要一个量化预测误差的方法。最常用的损失函数是均方误差。

均方误差的公式为：

**MSE = (1/n) * Σ(y_i - ŷ_i)²**

其中：
*   **n** 是样本数量。
*   **y_i** 是第 i 个样本的真实值。
*   **ŷ_i** 是第 i 个样本的预测值（即 **wX_i + b**）。

MSE计算的是所有样本预测值与真实值之差的平方的平均值。我们的目标就是最小化这个MSE值。

知道了如何衡量误差，接下来我们需要一个方法来调整参数以减小误差，这就是优化过程。

## 参数优化：梯度下降

梯度下降是一种迭代优化算法，用于寻找损失函数的最小值。它通过计算损失函数关于参数的梯度（导数），并沿梯度反方向更新参数。

参数更新遵循以下规则：

**w = w - α * (∂MSE/∂w)**
**b = b - α * (∂MSE/∂b)**

其中：
*   **α** 是学习率，控制每次更新的步长。
*   **∂MSE/∂w** 和 **∂MSE/∂b** 分别是损失函数MSE关于 **w** 和 **b** 的偏导数。

以下是梯度下降算法的简单步骤描述：

1.  随机初始化权重 **w** 和偏置 **b**。
2.  计算当前参数下的预测值和损失（MSE）。
3.  计算损失函数关于 **w** 和 **b** 的梯度。
4.  按照更新规则调整 **w** 和 **b**。
5.  重复步骤2-4，直到损失收敛到最小值或达到预设的迭代次数。

掌握了理论基础后，我们通过一个代码示例来直观理解整个过程。

## 代码示例

以下是一个使用Python和NumPy实现简单线性回归的示例：

```python
import numpy as np

# 生成模拟数据
np.random.seed(42)
X = 2 * np.random.rand(100, 1)
y = 4 + 3 * X + np.random.randn(100, 1)

# 初始化参数
w = np.random.randn(1)
b = 0

# 设置超参数
learning_rate = 0.01
n_iterations = 1000

# 梯度下降
for iteration in range(n_iterations):
    # 计算预测值
    y_pred = w * X + b
    # 计算损失（MSE）
    loss = np.mean((y_pred - y) ** 2)
    # 计算梯度
    dw = (2 / len(X)) * np.dot(X.T, (y_pred - y))
    db = (2 / len(X)) * np.sum(y_pred - y)
    # 更新参数
    w = w - learning_rate * dw
    b = b - learning_rate * db
    # 每100次迭代打印一次损失
    if iteration % 100 == 0:
        print(f“迭代 {iteration}， 损失 {loss:.4f}”)

print(f“训练完成。最终参数：w = {w[0]:.3f}, b = {b:.3f}”)
```

## 总结

本节课中我们一起学习了线性回归的基础知识。我们首先了解了线性回归模型 **y = wX + b** 的基本形式。然后，我们引入了均方误差作为衡量模型预测好坏的损失函数。接着，我们学习了如何使用梯度下降算法来迭代优化模型参数 **w** 和 **b**，以最小化损失函数。最后，通过一个简单的代码示例，我们将这些理论概念付诸实践。线性回归是理解更复杂机器学习模型的基石，掌握其原理至关重要。
