# 花镇情感课程：01：职场经济：巧用投资三法则，搞定复杂人际关系

## 概述

在本节课中，我们将学习如何将经济学中的投资法则应用于职场人际关系管理。人际关系并非简单的社交活动，而是一种需要精心经营和投资的“情感账户”。通过理解并运用投资的三大核心法则——收益性、安全性和流通性，我们可以更有效地构建和维护有价值的职场人脉网络，从而在复杂的职场环境中游刃有余。

---

## 人际关系的收益性：打造你的“情感账户”

上一节我们概述了将投资思维引入人际关系管理的理念。本节中，我们来看看第一个核心法则：收益性。

每个人都有一个银行账户，用于存储和支取金钱。当账户余额不足时，我们会感到恐慌。同样，作为一个社会人，我们还有一个“情感账户”。这个账户存储着我们与他人之间的信任、情感连接和善意。想要获得良好的人际关系，经营这个情感账户至关重要。

许多人只在情感账户中支取，却很少存储或投资。这是人际关系无法获得回报的重要原因。金钱需要先存储才能使用，否则只能透支。人际关系同样如此。如果没有长期的情感积累，在需要时要么无资源可用，要么使用就是透支关系。只知道索取容易导致对立、紧张和冲突。如果没有后续的投入来维持平衡，关系会持续恶化直至破裂。

相反，一个充盈的情感账户能带来足够的信任。即使你偶尔犯错或冒犯对方，也更容易获得原谅。情感账户富足的人，生活更开心快乐，仿佛处处有贵人相助。这其实都是他长期积累的结果。

人际关系收益性并非追求短期回报，而是通过持续积累获得。打开收益大门的关键，在于打造你自己的核心竞争力。

原因很简单：如果自身没有核心竞争力，不能为他人提供价值，那么优秀的人为何愿意与你交朋友？优秀者的时间非常宝贵。要想让他们为你付出时间，你必须先提供一个理由。这时，你需要展示价值并进行交换。**创造价值，你才是有价值的人。给予价值，你的价值资源才会取之不尽。**

如果你理解这个能量交换的原理，就需要执行人际关系中的规则：展示价值，付出交换。因此，你必须打造属于自己的核心竞争力。无论是写作能力突出、投资知识深厚，还是市场经验丰富，都能在特定环境下成为你的独特优势。关键在于，你要主动将其展示出来。这样才能吸引更多有才华的人与你交往，从而扩大你的交际圈。

人们正是通过给予，使自己的稀有技能得以继续发展和成长。通过给予和投资他人，你的关系网络将会遍布各地，相应的，赚钱也会变得更容易。你的关系网络将成为你的净资产。

## 人际关系的安全性：择优而交，识别“李斯特式”老板

理解了如何通过提升自我价值来增加人际关系的“收益性”后，我们来看看如何保障人际关系的“安全性”。什么样的状态才算人际关系安全？如何保持交往中的安全度？

一个最简单的方法是“择优而交”。对于职场中形形色色的人，你需要选择理性交往，以保证自己的人际关系安全性。

《杜拉拉升职记》是一部早年很火的职场小说。其中有一位领导叫李斯特。李斯特是外国人，被派到中国区担任人力资源总监，向总裁汇报。李斯特快退休了，他在职场上的唯一追求就是安全退休。因此，该做决定时他犹豫，该承担责任时他授权。

小说带有一些美化色彩。现实中往往是这样：你辛辛苦苦工作，大部分业绩是你产出的，大部分工作也是你做的。当你拿着微薄的薪水去找老板要求升职加薪时，你以为老板至少会许下一个承诺，或者至少会说考虑一下。但相反，他可能会批评你，指出你有很多需要提高的地方，比如情商不足、上班迟到、沟通方式生硬。他会说对你期望很高，但你还有很多需要改进。

这还只是小情况。还有更严重的情况：你为你的部门做事，与别的部门或个人产生了冲突。你认为自己这方合理，双方可以共赢，而对方有点欺负人。于是你将事情层级上报，希望你的老板出面协调。可是老板不但不出面，反而严肃地告诉你，心胸要开阔一些，不要总是去争，应该让大家开心，有功劳大家分享。而你的内心却在想：明明是别人在欺负我。

这就是套路。这样的老板实质上都是“李斯特式”老板，普遍存在于各类组织中。他们通常满足以下几个特点：

以下是“李斯特式”老板的典型特征：

1.  **安于现状**：这份工作是他们能找到的最好工作，他们得到这份工作往往有特殊原因。离开这家公司，他们很难找到同等水平的工作。
2.  **回报丰厚**：相对于他们的实际能力，他们获得的回报很丰厚。
3.  **追求稳定**：他们的核心目的是在这个位置上待得越久越好。
4.  **漠视公司利益**：公司利益与他们个人毫无关系，他们并不关心。
5.  **漠视下属利益**：你的利益与他们更没有什么关系。如果有举手之劳的小恩小惠，或者动动嘴皮子就能做到的鼓励，他们可能会给。但绝对不会在关键时刻、遇到冲突时帮你争取根本利益。
6.  **寻找替罪羊**：在需要的时候，你就会成为背锅侠，甚至会被抛弃。

李斯特是一种比较糟糕的老板类型，但他们非常普遍。作为职业人，你要清楚地知道：在李斯特们的手下，需要非常幸运才能与之一起飞黄腾达。他们因为能力和上进心有限，会使你的事业处于一种凝固和损耗的状态。

那么，遇到这种情况该怎么办？你一定要做的就是“邀功”。

首先，要确保你的利益不受损。我们大多数人都身处职场江湖，每个人都使尽浑身解数以获得一席之地。即使你没有称霸的野心，信奉“酒香不怕巷子深”，认为埋头苦干总会被慧眼识珠的老板看见，也要留心：说不定老板在巷口闻见酒香时，功劳已经被别人截胡了。对于李斯特式领导，更要主动邀功。你的一切功劳，如果你不提，他也不会提。你做一件事就和他汇报一次，所以有功劳一定要争取。

其次，你还要找到一个能共同承受、共同进步的“良性共同体”，以脱离那种凝固损耗的状态。对于李斯特式老板，切记不要深交。这种老板对你的职业发展不会有太大帮助，他们会不断让你干活，却不会给你明确的实际性奖励。这时，你就要找到你的良性共同体，帮助你走出老板的不作为。

在职场中，交友不慎可能会毁掉你所有的努力。你一定要拒绝与以下几类人交往：

以下是职场中需要警惕并远离的四类人：

1.  **远离满口承诺却不行动的人**。职场中最忌讳那些满嘴跑火车、吹牛皮却不兑现、不做实事的人。这种人专门挑别人爱听的说，轻易许下承诺，凡事都是一副“好办，包在我身上”的模样，但事实上往往不能兑现，或者兑现的结果总差强人意。你可能难免会碰上这种人，吃一次亏可以，但绝不该上第二次当。
2.  **远离好逸恶劳、终日混迹之人**。俗话说“万恶懒为首”。职场中不怕你有能力、有才华，怕的是你既没能力、没才华，还不勤快。职场如同逆水行舟，不进则退。
3.  **远离工于心计之人**。善于玩心机的人往往不善于做事，善于做事的人往往不善于玩心机。俗话说“防人之心不可无”，初入职场的新人一定要特别注意这类人。平常工作交流时可以面带微笑，但千万不要浅交言深，避免“说者无心，听者有意”。
4.  **远离永不学习、毫无上进心之人**。很多职场人对学习的认知可能还停留在“毕业了还谈什么学习”的阶段。职场遵循物竞天择、适者生存的法则。总在原地打转、只懂一项技能的人，迟早会被取代。因此，在职场人际交往中，要和爱学习的人交往才会进步，你的职业预期才会有所提高。

和什么样的人交往，会决定你成为什么样的人。而你是什么样的人，又会让你在职业人际关系中左右逢源，获得更多资源。

## 人际关系的流通性：创造交集，盘活资源网络

在学会了如何保障人际关系“安全性”之后，我们来看最后一个法则：流通性。什么是人际关系的流通性？

流通性是指，你可以在有限的朋友圈中获得无限的资源。这里的资源既包括人才资源，也包括信息资源。重点不在于你交了多少朋友，而在于你能通过这些朋友获得多少有用的资源。信息的流通才是最重要的。

良好的人际关系能够促进人们之间的信息交流和信息共享。现代社会知识量激增，信息爆炸，如果不建立一个和谐的人际关系网络进行交往，就不可能尽快获得所需信息。有人曾说，现代社会80%的信息都是通过人际交往获得的。掌握了信息，就等于增加了成功的砝码。

那么，如何把握好人际关系流通性原则？怎样才能获取有效资源？

首先，也是最重要的，就是**创造交集**。人与人之间如果没有交集，就没有关系。例如，两人之间兴趣不同、饮食不同、认知不同、地域不同、信仰不同、利益不同、文化不同等等，没有任何交集，这样是不可能建立关系的。

人与人之间能够相互认识、相互交往，本质上是因为对方与自己存在价值关系。创造交集从本质上来说有两个方向：你可以为他创造价值，或者让对方为你创造价值，同时你提供相应的回报。

## 总结

本节课我们一起学习了如何运用经济投资的三法则来管理复杂的职场人际关系。

1.  **收益性**：在人际交往中，你首先要投资自己，提升核心竞争力。当你的价值提高后，自然会吸引到有价值的朋友，这些朋友本身就是一种财富。
2.  **安全性**：你需要“择优而交”。对于职场中形形色色的人际关系网络，你要能够辨识哪些人值得深交、能给你提点，而哪些人可能会成为你职业生涯的绊脚石，尤其是要警惕“李斯特式”老板。
3.  **流通性**：你要善于通过现有的关系网络去扩展朋友圈，并能够从中获取有价值的信息和资源，为你的交际圈增砖添瓦，打下坚实基础。
