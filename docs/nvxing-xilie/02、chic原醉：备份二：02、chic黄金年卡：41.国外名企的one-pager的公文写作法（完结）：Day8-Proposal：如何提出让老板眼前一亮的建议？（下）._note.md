# 公文写作法：02：如何提出让老板眼前一亮的建议？（下）

## 概述
在本节课中，我们将继续学习商业建议书（Proposal）的撰写方法。上节课我们介绍了背景分析，本节我们将深入探讨方案分析、财务模型构建以及预期效果阐述这三个核心部分，帮助你完成一份结构完整、说服力强的商业建议。

## 方案分析：阐明“做什么”与“为什么做”
上一节我们介绍了如何为老板清晰阐述项目背景。在读者对背景有了清晰了解之后，接下来就需要进入最核心的方案分析部分。这部分主要回答两个问题：**我想做什么（What）** 以及 **我为什么想做（Why）**。

以下是梳理“What”与“Why”的具体方法：

### 关于“What”：明确建议内容
在描述“想做什么事情”时，建议用最简明扼要的语言直接给出建议的名称和核心意义。

*   **示例**：建议研发新的业务平台，并通过铺设线下渠道，实现线上线下的联动。
*   **多条建议的处理**：如果建议不止一条，应逐条标号并**加黑加粗**，使结构更加清晰。
*   **关键技巧**：尽量使用具体的动词代替虚拟的动词。
    *   **虚拟动词示例**：“拓展某某领域的市场”。
    *   **具体动词示例**：“制定某某领域的拓展计划”。
*   **原因**：“拓展”是一个虚拟动词，具体如何操作仍不明确。而“制定计划”则让人明确后续行动。因此，在阐述“What”时，务必通过这种方式将任务**具象化**和**可操作化**。

### 关于“Why”：阐述行动理由
阐述“为什么想做这件事”时，可以从正向和负向两个层面进行综合分析。

1.  **正向因素（积极意义）**：阐述做这件事的主要正面意义。这部分思路可参考背景分析，从外部环境、内部条件进行综合论述，并可增加时间维度的考量。
    *   **示例阐述**：当前外部数据显示某领域流量潜力大，资本关注度高，且公司基因足以支撑进入该细分领域，目前正处在最佳介入时机。

2.  **负向因素（潜在风险）**：阐明可能存在的负面影响或风险。主动提及负面因素并非不利，而是为了帮老板进行预筛选，展示你已全面权衡利弊，避免给人“报喜不报忧”的印象。这部分篇幅可以精简。
    *   **示例阐述**：过渡期间可能造成的影响是损失原有市场约10%的用户。
    *   **注意**：在阐述负向因素后，需提供估算的依据。

在正向和负向因素都阐释完毕后，需要用一两句话进行总结，向老板说明：尽管存在负向因素，但其影响整体上会被正向因素的优势所覆盖，综合来看**利大于弊**。这样的阐述会使整个建议更具说服力。

## 财务模型：量化“如何做”与“值不值”
“Why”的部分完结之后，接着就要进入“How”的部分，即给出具体的财务模型。这需要说明公司需投入何种资源、性价比如何，以及是否值得执行。

财务模型的核心是**投入**与**产出**的量化分析。
*   **投入**：公司需要额外花费的财力、物力、人力成本。
*   **产出**：预期会带来的收入、现金流等。

通过两者的差额和比值，可以反映关键信息：
*   **投入产出比（ROI）**：公式为 `ROI = (收益 - 成本) / 成本`。比值越高，证明资金使用效率越高，这笔投入越值得。
*   **利润率**：公式为 `利润率 = 利润 / 收益`。利润率越高，表明项目自身的盈利（造血）能力越强。

可能你会疑惑，项目尚未执行，如何预测这些数据？这依赖于你对市场的判断、过往工作经验，或通过查询资料、咨询有经验的前辈来获取信息。

在财务模型部分的阐述中，通过列举和分析以下四个方面的数据，就能清晰地说明问题：

1.  **人力物力成本**：列出主要成本项，并简要说明为何这些成本是必须的。
    *   **示例**：需投入50万元用于渠道铺设，因为公司当前在该领域缺少相关渠道资源。
2.  **预期收益**：估算投入后能带来的具体转化或收入。
    *   **示例**：投入50万元广告费后，预期能带来200万元的销售额。
3.  **投入产出比（ROI）**：计算成本与预期收益的比值，评估资金使用效率。
4.  **毛利率或利润率**：根据预期成本和收益估算得出，用于评估项目对公司的利润贡献程度。

## 预期效果：描绘宏观价值蓝图
财务模型完成后，便进入最后一个部分：预期效果分析。有人可能会问，上一步不是已经有预期收益了吗？没错，但预期收益主要关注财务层面，而**预期效果是一个更加宏观的概念**，涵盖了该建议能为公司创造的各维度价值。

这些价值可能包括：预计完成的市场份额、达到的流量增量、产生的品牌推广效果等。其中，部分效果可以量化，部分则难以量化。因此，在表述时需要分开处理。

*   **可量化的效果**：先陈述有具体数额的效果，再说明推算依据。
    *   **示例**：根据该渠道以往的数据统计，预计能带来20万的独立访客流量增量。
*   **难以量化的效果**：描述可能产生的效果，并具体说明如何衡量该效果。
    *   **示例**：此举预计将提升品牌在目标人群中的影响力。效果可通过后续市场调研中品牌提及率的增长来间接衡量。

## 总结
本节课中，我们一起学习了商业建议书（Proposal）的后三个核心组成部分：**方案分析**、**财务模型**和**预期效果分析**。结合上节课的**背景分析**，至此，一份结构完整、论证清晰的商业建议书才算完成。希望你能运用这些方法，未来提出更多对公司和团队有价值的商业建议。
