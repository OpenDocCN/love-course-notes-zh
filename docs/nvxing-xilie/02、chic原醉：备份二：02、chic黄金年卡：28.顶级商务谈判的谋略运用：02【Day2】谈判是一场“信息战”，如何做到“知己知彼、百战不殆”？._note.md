# 谈判哲学：02：顶级商务谈判的谋略运用

## 章节编号：02【Day2】谈判是一场“信息战”，如何做到“知己知彼、百战不殆”？

### 概述

在本节课中，我们将要学习谈判前信息准备的核心方法。上节课我们提到，实现双赢谈判的前提是在认知层面清晰地了解自己和对方。本节课我们将深入探讨，具体应从哪些维度搜集信息，以及通过哪些渠道获取这些信息。

### 信息不足导致谈判困境

在开始具体讲解前，让我们设想一个场景：你想在一线城市购买一套房产，但你对房地产行业、政策和市场都不了解。此时，你不会直接开始讨价还价，因为你知道，在毫无准备的情况下谈判，成功率很低，甚至可能被对方反制。

我们分析一下这背后的主要原因：

以下是导致谈判困境的三个核心原因：

1.  **对外部市场信息掌握太少**：这导致我们在谈判中缺乏参照依据。不清楚市场价格与走势，会让我们缺少底气和回旋筹码，容易被对方掌控节奏。
2.  **对自身诉求和实力不够明确**：过于单一（例如只追求低价）的诉求，往往导致谈判结果走向极端——要么失败，要么得到不理想的结果。
3.  **对谈判对手不够了解**：不知道对方的优势、劣势和可接受的底线。当我们一味争取自身利益而不小心突破对方底线时，容易导致谈判“硬着陆”并陷入僵局。

综合以上三点，问题的症结在于三个字：**信息量**。正所谓“知己知彼，百战不殆”，只有掌握充足信息，才能在谈判中从容应对。

### 信息收集的三个关键维度

在信息储备过程中，我们会接触到大量零散信息。但总体来看，对我们最关键的信息只有三个维度：除了“知己”（我方）和“知彼”（对手方）的信息外，还有一类经常被忽略的信息——**宏观市场环境以及双方在其中的相对位置**。

谈判是双方势能的较量。市场中还存在其他行业和竞争者，我们和对方各自都可能拥有其他选择。因此，在信息收集整理时，我们需要同时关注**对手企业**、**我们自身**以及**整个行业与社会环境**的信息。

### 判断双方势能的四大指标

以下是帮助我们判断双方势能的四个关键指标：

#### 1. 企业的营业收入与现金流情况

这个指标能直接反映企业的日常经营与资金状况。通常，资金实力越强、营业收入越高的企业，其整体竞争力和行业地位也越强。

对于上市公司，我们可以从其官网下载近三年的财务报告，重点关注：
*   **利润表**中的 **`营业收入`**
*   **现金流量表**中的 **`经营活动现金流`** 与 **`融资活动现金流`**

**`经营活动现金流`** 反映企业依靠主营业务“造血”的能力，即正常业务带来多少现金流入。**`融资活动现金流`** 则反映企业依靠外部“输血”的情况，例如在二级市场发行股票、债券等。

如果合作方是非上市公司，可以通过查询其披露的融资信息及相关新闻来判断。

#### 2. 企业的行业排名与竞争对手情况

这个指标同时适用于我方和对手方。

对于谈判对手，我们通过其行业排名和市场竞争情况来判断两件事：一是其提供的服务是否具备足够竞争力，我们是否有更好的选择；二是其品牌影响力与资源掌控力如何。通常，排名越靠前的公司，影响力和资源背书能力越强。

> 就像创业者选择投资人时，在相同估值和金额下，他们更倾向于知名机构，因为后者能提供资金以外的潜在资源整合力。

同样，我们也需要清楚自己在行业中的排名、竞争对手情况，明确自身的竞争优势、侧重点以及有待加强的环节。我们可以通过行业新闻、IT桔子、天眼查等信息公示网站查询行业和竞品信息。

#### 3. 企业在谈判过程中的复合需求

“复合需求”指的是谈判主体所抱有的诉求往往不是单一的，而是由两个以上因素综合而成。

以买房为例，诉求可能包含价格、位置、户型、社区环境、交通等多个因素，这些可被综合为“性价比”这个概念。性价比就是一个典型的复合需求，包含了“性能”和“价格”两个要素。

单纯追求低价可能得到品质不佳的产品；单纯追求性能则可能远超预算。因此，谈判双方抱有复合需求是正常的。但正如上节课所说，双方能完全达成一致的往往只是其中一个方面，其他因素可能存在分歧。谈判的目的就是尽量协调这些分歧，达成双方都满意的结果，促成合作。

在需求挖掘方面，我们首先需明确自己的复合需求包含哪些点，以及它们的优先级关系。例如，在价格、服务深度、资源、品牌影响力等因素中，我们自己最看重哪几项？同时，可以通过预沟通大致了解对方在意的因素，完成需求层面的信息收集。

> 例如，我们可以询问对方：“您对本次合作最大的期待和诉求是什么？”以此开启对对方复合需求的探寻。

#### 4. 底线与置换方案

“底线”是谈判中的最后一道关卡，也是我们需要通过一定对抗和博弈去探寻对方的最重要信息。只有了解对方底线，才能更好地把握局面。同时，尽可能保护自己的底线不过早暴露，也是对谈判功力的考验。

底线往往与“复合需求”相辅相成，通常出现在我们复合需求的最高优先级上。例如，如果在各种需求因素中最在意价格，那么底线往往也体现在价格上（例如必须守住某个价位）。

因此，我们会发现，底线往往也出现在对方最在意的点上。与此同时，我们必须提前想好“置换方案”。即万一不小心触及对方底线，为了避免“硬着陆”，如何采用缓冲方式让双方都不至于难堪。

> 例如，如果对方最在意价格，而我们的预算远低于其接受范围，此时可以考虑用哪些其他资源或利益进行置换。

可能有同学会问：试探对方底线是否与“双赢谈判”背道而驰？难道不是双方都得到想要的东西才最理想吗？

其实，只要谈判结果在双方底线之上，得到的就是双赢结果。早些了解对方底线，能为谈判增加回旋空间；提前准备置换方案，也能有效避免真正触及底线时产生的尴尬。

### 总结

本节课中，我们一起学习了在谈判开始前，如何从信息层面做准备。

在进行一场谈判的准备时，首先应侧重三个维度的信息：
1.  对手方的相关信息
2.  我们自身的信息
3.  整个市场或行业的信息，包括双方在其中的相对位置

由此延伸出，在具体信息收集过程中需要参考的四大指标：
1.  企业的营业收入与现金流情况
2.  行业排名与竞争对手情况
3.  企业的复合诉求
4.  双方的底线与置换方案
