# 情感课程：02：如何培养男友的浪漫细胞：09【Day9】长期关系中的浪漫

在本节课中，我们将要学习如何在长期关系中维持和培养浪漫感。许多人认为浪漫会随着关系进入长期阶段而消退，但通过一些具体的方法和技巧，我们可以有效地让浪漫持续存在，甚至成为加固关系的纽带。

## 长期关系中的浪漫挑战

人们常常认为浪漫的消亡是伴随长期关系开始的。我一个早早结婚的朋友对此深信不疑。她说，假如一个结婚多年的男人，有一天回家突然送你一束花，那就有两种可能，要么花店大减价，要么他在外面做了亏心事。假如还有第三种可能，那就是老公被掉包了。

虽然这样的说法听起来没有要给老公活路的意思，不过也说明了大多数人所遇到的问题，就是长期关系消退了激情，磨灭了浪漫。我们这节课就来和大家讨论一下怎么样才能维持浪漫，使长期关系更加坚固。

## 维持长期浪漫的两个核心技巧

首先，维持长期浪漫有两个稳固的技巧，它们适用于长期关系中的任意阶段。

### 技巧一：用浪漫引导模仿

有一种心理学现象是，我们会无意识的模仿别人的行为、表情和怪癖。比如男朋友喜欢在家“葛优躺”，即使你从前坐姿端正，也不能保证有一天不会懒懒的躺下来，成为女版“葛优”。再比如你说话常带语调欢快的语气词，回应别人的时候会说“是的呢”，几个月之后，画风严肃的男朋友也会用上这句轻巧灵动的回应。

这种模仿行为在心理学被称为**变色龙效应**，即我们会像变色龙一样，根据环境改变一定的行为表情。人们常说的夫妻相也是源于这个道理。

我们也可以利用这种变色龙效应，使另一半无意识的模仿出浪漫行为。模仿的焦点可以是情话的语调、神情，也可以是制造浪漫的时机和方式。例如，一起走在昏黄的路灯下，你可以靠得更紧的说：“超喜欢和你走在路灯下。”也许下一次他就掌握了亲密接触的浪漫方法，把你搂得更紧，说出类似的情话。这种习惯性的模仿，对于情感起伏平稳的长期关系来说是非常有帮助的。

### 技巧二：突出对方的人格魅力，延长浪漫感受

根据心理学家阿伦的研究，发现自己被一个有魅力的人喜欢，你会感受到浪漫。其结论可以理解为：我们的浪漫体验，可以通过对方的魅力来获得。

假如你爱上了一个作家，你的浪漫体验可能就来自于他的写作才能。你发现他的文笔很好，心思细腻而敏感。但是交往十年后，他失去了创作的欲望，转而在游戏中获得成就感。你还觉得有任何浪漫可言吗？可见，我们对对方优势长处的欣赏，能形成一种稳定的浪漫崇拜。

让对方维持自己的魅力和优势，就是让他保持对这件事长久的激情，而激情则来自于成功。所以你可以用“鼓励加赞扬”的方式，让他发挥优势。比如你欣赏另一半超强的自制力，能保持健身习惯，你就可以鼓励他在知乎、微博分享自己是如何做到坚持的，又有哪些有趣的健身经历。当这些分享得到转发点赞的时候，就夸赞他，使他获得收获的喜悦和新的激情。

## 维持长线浪漫的两个具体方向

上一节我们介绍了两个核心技巧，本节中我们来看看可以从哪些具体的方面着手来维持长线浪漫。以下是两个可以努力的方向。

### 方向一：通过合情合法的调情，维持长期关系的暧昧和激情

能够保持日常的调情习惯，浪漫感往往也会更强烈。这一点，我们通过情侣的交往时长就能区分出来。交往已久的情侣在看电影的时候，会和黄金单身汉抢最佳观影区，而热恋中的情侣，则会抢占后排无人区。即便片源是《喜羊羊与灰太狼》，也不妨碍他们分出一半的心思谈情说爱。

也就是说，延续“你侬我侬”的情人状态，有助于长期浪漫的维持。你可以参照一位女画家的做法，她和丈夫已经步入中年，依然在“琴瑟文化”上保持着不变的习惯。有一天，身为作家的丈夫从笔会回来，进门后没发现妻子，却在饭桌上找到一张纸条，上面写着：“酒在柜子里放着，菜在蒸笼里热着，你的夫人在被窝里睡着。”

当然，有些人会觉得热恋期早已离自己很遥远，要重新拾起湿漉漉的情话，不免有些生疏和尴尬。说不准，男生还会以为你“无事献殷勤”。这样的话，你就可以从一些简单的话切入，比如先称赞对方在外形或说话方式上有哪些柔和的变化，让女友产生了一些初次相识的感觉。当男生从你的身上重新获取了成就感，他在看你的时候，也会同时萌生爱意，对你更深层次的挑逗也就更容易接受了。

### 方向二：抛物线魅力值的修炼

“抛物线魅力值”就是让自己的魅力以抛物线式逐渐提升。可能你会有疑问，不提升魅力就没有长期浪漫可言吗？难道男朋友变得大腹便便，说起情话来就变成油腻的聊骚；我不修边幅的调情就变成了耍流氓？这似乎和我们所追求的精神型浪漫相违背啊？

的确，浪漫和颜值、物质都不应该直接挂上等号，而我们要传达的是保持个人魅力所需要的坚持和耐力。就像我们在课程开头强调的，当你发现自己被一个有魅力的人喜欢时，你就能感受到浪漫。这时，男友保持身材、修炼谈吐的强劲势头，就是浪漫的起源。相反，我们同时作为提供魅力的一方，也能让男友产生浪漫的感受。

你不妨鼓励男友去健身，改善自己的弱势。虽然这并不是一件容易的事，但是你可以通过自身的有效实践去砥砺他。比如你把“抛物线魅力值”的横坐标设置为自信、独立、体重、风格等，然后再通过一个个小目标去量化自己的成果。通常来说，这种行为激励比简单的语言激励更卓有成效。就好比你的好朋友曾不厌其烦的让你尝试某种方法去减重，你不置可否。但是一旦对方用同样的方法改头换面出现在你眼前时，你的改变动力就会冲到顶点。因此，利用实打实的结果也能刺激男友产生行动。

## 如何安排长期浪漫的频率和轻重起伏

强调浪漫的频次和轻重，一是因为我们无法让生活时刻萦绕在浪漫氛围里，二是因为即便可以，泛滥成灾的浪漫也起不到升华感情的加成作用。这和吃龙虾有些类似，我们不会每天都吃到，就算可以，吃龙虾也不会再被认为是多么特殊而有意义的事情。所以我们应该遵循感情走向稳定期的规律，在必要时刻制造高低起伏的浪漫。

制造浪漫的时机，可以是你们聊天频次降低的时候，或者是你们解决完矛盾争端之后。这样长期关系中的浪漫就会凸显其效果，即给你们增加更多话题度，或给出现问题的感情发挥修复力。

“轻重起伏”，说的则是穿插小浪漫和大浪漫。今天说情话来个小小的调情，半个月后的纪念日再制造一出浪漫大戏，这样就能保证浪漫维持在一个刚刚好的节奏里。假如说男朋友把握的不太到位，一个月里有二十几天都在秀情话，或者更多的是小浪漫匮乏，只在最重要的节日里绞尽脑汁策划浪漫大事，你就可以在他节奏失控的时候，把浪漫的气氛重新调动起来。

## 课程总结

本节课中我们一起学习了如何在长期关系中培养和维持浪漫。

首先，我们提到了两个可以在长期关系中使用的核心技巧：一是利用**变色龙效应**，先示范出浪漫的行为，从而引导他模仿出同样的浪漫；二是发掘对方的魅力之处，通过“鼓励加赞扬”的方法，让男朋友不断凸显自己的优势，而这种优势本身就是一种能被感知的浪漫。

然后，我们还列举了两个可以具体着手的方面：第一就是掌握好“情色文化”，把它贯穿在长期关系里；第二就是提升自身魅力值，并以此刺激男友提高自身条件。这种做法旨在凸显我们及男友自我提升背后的毅力和态度，当这些特质被对方意识到时，浪漫的感觉就会自然而然地弥漫出来了。

最后，我们探讨了要按照适当的频次和轻重来调节浪漫的节奏，以确保其效果最大化。
