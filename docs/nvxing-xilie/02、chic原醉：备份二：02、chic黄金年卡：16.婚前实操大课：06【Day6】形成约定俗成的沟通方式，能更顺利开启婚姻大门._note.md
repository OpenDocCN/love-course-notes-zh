# 恋爱到婚姻的关系转变：02：形成约定俗成的沟通方式，开启婚姻大门

在本节课中，我们将学习如何与伴侣建立有效的沟通模式。良好的沟通是婚姻关系顺利开启和长久维持的基石。我们将探讨如何区分意见不合与三观不合、如何避免无效争吵、如何建立专属的沟通信号，以及在冲突发生时如何有效平息争端。

## 意见不合不等于三观不合

上一节我们介绍了沟通在关系中的重要性，本节中我们来看看如何正确看待沟通中的分歧。沟通中意见不合，是否意味着三观不合，未来会产生很多困难？我们可以带入这样的场景。

男朋友提议去成都旅行。你很高兴，因为这是你们很早之前就提到过的事，现在时间预算都有了，只差一个周详的计划。但在讨论交通工具时，你们产生了分歧。你喜欢坐火车，认为只有沿路欣赏轨道边的风景，旅行才有了意义。而男朋友希望减少路程上的消耗，把更多时间放在当地游玩上。

这能说明三观不合吗？当然不能。我们在旅行上的爱好对加深感情有所帮助。假如男朋友说：“坐火车有什么好的，嘈杂混乱，风景又有什么可看的？”你就会因为喜好不被尊重，与之产生难以调和的分歧。

**三观不合**，应该是我们对对方的想法和喜好加以贬低和嫌弃，让沟通产生无力感。但是大多数时候我们目标一致，只是方法不同。例如前面这个例子，旅行的目标一致，只是在选择交通工具上有不同建议。

最好的办法是尊重彼此的爱好及观念。例如，去程选择飞机缩短时间，回程选择火车给旅行一个美好的完结。所以，意见不合不能与三观不合划等号。关键是我们能不能做到“君子和而不同”，找到最好的解决方案。

以下是两个具体建议：

*   **肯定对方的想法**：真心诚意地认为自己和对方的观点都值得被尊重。
*   **把实践对方的观点当成是逃离舒适圈的机会**：我们习惯了用自己的逻辑做事，但思维定式会加重自身的局限性。所以，成长的一种表现就是思考别人的处事逻辑，加以借鉴。

## 沟通四部曲：避免无谓争吵

关于沟通方式，我们可能还有一个顾虑：为什么沟通时会没来由地吵起来？一个原因是某个情绪或某句话让一方感到被攻击，于是被攻击的一方开启防御机制予以回击，这就带来了程度逐渐加重的循环。

我们可以用“沟通四部曲”来避开雷区。

### 第一步：澄清意图

想一想你在被对方的话刺激到时，是不是会先揣测对方的意图？而麻烦之处在于，我们的揣测通常更偏向于糟糕的情况。

在《爸爸去哪儿》第三季中，林永健带着儿子大俊住进了新房子。大俊很兴奋，走到自己房间时看到了桌上摆放的QQ星，于是特别高兴地把QQ星装进了书包里。这一幕被爸爸撞见了。爸爸先是厉声制止他，然后说：“占便宜可不好啊，我们干嘛要占便宜呢？”爸爸还没有问清楚缘由，就先揣测儿子拿饮料的意图是贪小便宜。但是儿子对他说，自己只是想把QQ星分享给其他小朋友。

从这个例子可以看出，沟通中产生的误解，其实来自于真实意图没有得到澄清。所以我们在交流的时候，要表明自己的想法和动机。例如，你可以说：“我提这件事，其实不是想评价你，而是我希望通过讨论加深我们两个人的理解。”

### 第二步：说明事实

说明事实，就是要把讨论的事情客观呈现出来。我们先对比一下这两句话：
1.  你总不把我的话放在心里。
2.  你一边听我说话，一边玩手机，我们没有办法很好地沟通。

你觉得哪一句更客观？显然是第二句。第一句不仅带有责怪的语气，还用了敏感词“总”，对方一听就会据理力争。所以，要做到客观陈述，就要避免使用绝对词，直接描述事情本身即可。

### 第三步：暴露心情

柴静在《看见》这本书里说：“理解的基础是感受。人能感受别人的时候，心就变软了。软不是脆弱，是韧性。”也就是说，如果你希望得到他人的理解，一定不要羞涩于把自己的感受表达出来。

如果你觉得难过，就说：“这件事（或这个行为）让我不高兴。”即使是程度比较深的愤怒情绪，你也可以说：“我现在真的非常生气。”但语气要相对平和，不上升到对伴侣的批评。

### 第四步：展示需求

很多时候吵架会有一个奇怪的现象：吵到最后，我们对为什么吵架已经不在乎了，甚至忘了是怎么吵起来的，但为什么还会接着吵？原因就是需求没有被满足。

女生的需求可能只是希望男生多陪陪自己，但男生没有理会到，他更倾向于理清逻辑、分析利弊。只有你直接地展示需求，才能够让沟通更简单高效。

## 建立专属的沟通信号

然后我们来说一说如何建立两个人专属的沟通信号。根据美国伊利诺伊大学的研究发现，很多幸福美满的夫妻都有一套“秘密语言”，就像暗号，只有他们两个人懂，外人不知道。有时甚至不用说话，一个表情就能心领神会。

有一套自己的沟通信号，就像手握遥控器，在沟通的时候暂停、快进、换台，都是很容易办到的事。同时，对于发展一段长期关系来说，它能够给你们的生活制造浪漫回忆，升华情感。

根据相处模式的不同，沟通信号也是多种多样的。以下是几个案例：

*   一对夫妻说，他们只要吵架时喊“123”，就会同时说“对不起”，这表示了各退一步的诚意。
*   还有一对夫妻沟通不下去的时候也吵不起来，因为只要四目相对，他们就会情不自禁地放声大笑，这帮助他们绕开了很多不必要的争吵。
*   结婚65年的一对老夫妻更是将早餐当做了破冰的信号。即使他们前一晚还互不搭理，但起床碰面就会自然地打声招呼，坐在餐桌上就算作和好如初的第一个约会了。

你可以和男朋友讨论，当你们想保持安静的时候，或者想休战的时候，都有哪些表现？如果还没有形成特殊的沟通习惯，你们可以根据前面的案例改良出更适合自己的沟通信号。

## 从争执到平息：切换沟通状态

假如沟通不慎变成了争吵，怎么才能让事件风平浪静呢？在精神病专家艾瑞克·伯恩的著作《人间游戏》里，我们能了解到关于“交往分析”的理论。这种理论指出，我们在与人交往沟通的过程中，会呈现出三种状态。

**第一种是“父母”状态**。也可以理解为传统的中国父母对待子女的状态，比较严肃、理性。处于“父母”状态的人会说：“你看我是不是提醒过你控制一下情绪？”或者“快把你的东西收拾起来，床铺这么乱，怎么休息呢？”“父母”状态的对话其实很容易刺激对方，使火药味越来越重。

**第二种状态是“儿童”状态**。相比之下，此状态的人更注重自己的感受。他们像孩子一样希望自己的喜怒哀乐得到关注和宣泄。我们常常能够听到这样一句话：“你无情无耻无理取闹。”这就是“儿童”状态的话语，其更深的含义是说：“你伤害我的感情了，你怎么可以这样？”

**第三种沟通状态是“成人”状态**。他们在交流的过程中不断接收信息，会去思考怎么说、怎么做，能够让对方更容易接受。虽然我们在不同的场景中会不断切换自己的角色，但是当争论产生时，“成人”状态是最能有效平息战火的沟通状态。

我们要自由切换到“成人”状态，可以掌握善用此类方法的人有哪些特征：

*   **练习自我控制**：只把对方吵架当做情绪受忽视，而不是专门针对你的攻击。这样的好处是暗示自己和对方仍是同队队友，都希望感情和睦，只是因为情绪被忽视，暂时画起了“三八线”。
*   **懂得使用肯定话语**：例如，他们会说：“我知道我回来太晚，让你担心了。今天工作很忙，我应该提前打电话告诉你的。”

希望这些方法能够帮助你建立一个和谐温馨的家庭环境。

## 课程总结

本节课中我们一起学习了如何建立有效的伴侣沟通模式。

首先，我们明确了在沟通中产生的意见不合，并不能说明三观不合。要做到和平共处，一是要尊重彼此的差异，不坚信自己的意见比对方的高明多少；二是要转换思维，通过学习对方的思维方式，跳出自己的舒适圈。

其次，我们对如何良好沟通提供了四个步骤，分别是：**澄清意图**、**说明事实**、**暴露心情**以及**展示需求**。

接着，我们探讨了建立专属沟通信号的重要性，它能像秘密语言一样增进默契，化解矛盾。

最后，我们学习了如果在沟通中发生争执，要注意转换沟通角色，让自己从“父母”状态或“儿童”状态转换到“成人”状态，也就是说学会思考控场，用肯定的话语平息争端。

掌握这些沟通技巧，能帮助你们更顺利地开启并经营好婚姻关系。
