# 形象管理：05：领导范儿：兼顾领导力与亲和力

在本节课中，我们将学习如何通过形象管理塑造兼具领导力与亲和力的“领导范儿”，从而在职场中脱颖而出，为晋升管理层打下坚实基础。

上一节我们探讨了形象对职业发展的基础作用，本节中我们来看看如何具体打造权威与亲和并存的领导者形象。

## 一、评估个人形象影响力

想要提升形象，首先需要客观评估自己当前的形象影响力。美国形象专家Judy提出了“个人影响力循环图”：

**公式：积极职业形象 → 自信 + 他人积极印象 → 促进工作关系 → 业绩增长**

这个循环可以是良性的，也可以是恶性的。一个得体、积极的职业形象能开启良性循环，反之则可能陷入恶性循环。

以下是评估自我形象是否得体的三个具体步骤：

1.  **每日拍照自检**：每天出门前，为自己的整体形象拍摄一张全身照。通过照片观察自己往往比照镜子更客观。
2.  **对照目标打分**：根据当日的形象目标（例如：专业、权威）给自己打分（例如：80分）。
3.  **收集外部反馈**：收集他人对你当日形象的反馈，并尝试量化（例如：70分）。对比自评分与反馈分，找出差距并改进。

万事开头难，建议你坚持**21天**进行上述刻意练习，完成第一阶段的自我督导。3个月后，可启动第二个21天的评估阶段。通过多次循环，你的形象管理能力将得到质的飞跃。若条件允许，寻求专业形象顾问的督导会更高效。

当你的形象变得笃定自信，便会像自带光环一样散发出强大能量场，吸引更多机会。

## 二、掌握TPORM场合着装密码

上一节我们介绍了自我评估的方法，本节中我们来学习在不同场合精准着装的核心原则——TPORM。

TPO原则是着装的基础，即考虑**Time**（时间）、**Place**（地点）、**Occasion**（场合）。扩展后的TPORM原则包含五大要素：

*   **T (Time)**：时间（白天/晚上）。
*   **P (Place)**：地点（室内/室外）。
*   **O (Occasion)**：场合（会议/庆典等）。
*   **R (Role)**：角色（参与者/主讲人）。
*   **M (Message)**：信息（希望传递权威感还是亲和力）。

在出席任何重要场合前，都应先思考这五大要素，再挑选服饰，以确保形象传递的信息准确无误。在商务场合，得体永远排在第一位。

根据正式程度，商务场合可大致分为以下几类：

**第一类：偏正式的商务场合**
*   **正式度**：1-3度。
*   **建议着装**：正式商务装或经典商闲装。
*   **传递信息**：专业、权威、保守、可靠。
*   **常见场合**：正式会议、商务谈判、接待外宾、媒体上镜。
*   **着装方式**：深色套裙/套裤，或拆套搭配。色彩以深蓝、碳灰、黑、棕、米色为主。

**第二类：偏轻松的商务场合**
*   **正式度**：4-6度。
*   **建议着装**：品味商闲装。
*   **传递信息**：有品味、亲和、有能力。
*   **常见场合**：商务社交、客户用餐、培训、日常办公。
*   **着装方式**：外套（挺括或针织面料）搭配连衣裙、衬衫或内搭。柔性面料（如雪纺、针织）更显亲和。白天可搭配精美饰品或胸针。

**第三类：偏休闲的商务场合**
*   **正式度**：7-9度。
*   **建议着装**：轻便商闲装。
*   **传递信息**：创意、效率、轻松。
*   **常见场合**：陪同客户郊游、周末加班、公司团建。
*   **着装方式**：最低要求有领有袖。上身宜两件式、颜色清爽，下身宜卡其裤、收腿裤等利落款式。避免工装裤、短裤、破洞牛仔。

**特别说明：商务社交场合**
*   **白天**：参照“品味商闲装”。
*   **晚上**：根据正式程度，可选择小礼服，或在白天着装基础上做“加减法”（如减去外套，增加华丽配饰和晚装手包）。

## 三、运用服饰搭配整合技巧

了解了场合着装密码后，我们还需要掌握让着装更显高级、和谐的搭配技巧，即服饰的“整合”搭配。

整合搭配的核心在于创造视觉上的呼应和关联，达到“你中有我，我中有你”的和谐效果。最常用的有以下三种呼应法则：

1.  **服装细节间的呼应**：例如，上衣颜色取自半裙花色中的一种，形成色彩渗透。
    ```python
    # 示例：色彩关联
    半裙颜色 = [“红色”, “蓝色”, “白色”]
    上衣颜色 = 半裙颜色[0]  # 选择红色作为上衣
    ```

2.  **配饰与服装的呼应**：配饰颜色与服装中的某种颜色一致。例如，穿着有红色元素的服装，搭配红色手包或鞋子。

3.  **多样配件间的呼应**：当服装本身关联性不强时，通过配饰间颜色一致来链接整体。例如，腰带、鞋子、手包采用同色系。
