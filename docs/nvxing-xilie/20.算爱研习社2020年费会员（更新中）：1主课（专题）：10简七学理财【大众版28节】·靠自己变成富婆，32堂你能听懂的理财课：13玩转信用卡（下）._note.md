# 简七学理财：13：玩转信用卡（下）

在本节课中，我们将深入学习信用卡使用中的核心概念、关键日期、相关费用以及逾期处理，帮助你更聪明、更安全地使用信用卡。

上一节我们介绍了如何选择一张适合自己的信用卡。本节中，我们来看看信用卡使用中会遇到的各种专业术语和规则。

## 一、 理解信用卡额度相关名词

在使用信用卡时，你常常会看到一些专业名词。我们先来了解一下与信用卡额度有关的名词。

以下是三种常见的授信额度类型：

*   **授信额度**：指你的信用卡可用额度，即这张卡最多能消费的金额。公式表示为：`可用额度 = 授信总额度 - 已使用额度`。
*   **临时授信额度**：银行在特定时期（如节假日前）为你临时调高的额度。
*   **不免息的授信额度**：部分信用卡的额度可能超过标称额度，但超额部分从消费当天起计息，不享受免息期。
*   **综合授信额度**：在同一家银行办理的多张信用卡，其额度通常是共享的。

## 二、 认识最低还款额

理解了额度，我们再来看看还款时的一个关键概念——最低还款额。

顾名思义，最低还款额是你每期账单最少需要偿还的金额，通常为当期账单金额的10%左右。它主要为暂时还款困难的人提供缓冲。只要偿还了最低还款额，就不算逾期违约，但剩余未还部分将不再享受免息待遇，并从消费当天开始计收利息。

例如，本月账单为5000元，最低还款额为500元。若只还500元，则剩余的4500元将从消费入账日起，按日利率**万分之五**（即0.05%）计息。

## 三、 掌握关键日期：账单日、还款日与免息期

了解了信用卡额度，我们再来看看使用过程中接触到的几个关键日期，它们直接决定了你的消费能否免息。

银行会记录两个核心时点：

1.  **账单日**：银行统计你上一个账单周期内所有消费并生成账单的日期。
2.  **还款日**：你必须在此日期前偿还账单，以免逾期。

**免息期** 是指从消费当天到还款日之间的时间段。在此期间全额还款，则消费享受免息待遇。

例如，账单日为每月1日，还款日为每月20日。你在1月2日消费1万元，这笔消费会计入2月1日的账单，还款日为2月20日，因此你享受了约50天的免息期。

## 四、 巧用账单日的两个技巧

现在你对账单日和还款日有了更深的了解。这里补充两个巧用账单日延长资金使用时间的方法。

以下是两个实用技巧：

*   **技巧一：充分利用免息周期**。如果你有多张信用卡，在大额消费时，尽量选择刚过完账单日的那张卡，这样离最终还款日最远，资金利用效率最高。
    *   *示例*：你有A卡（账单日10日）和B卡（账单日20日）。在15日有大额消费，应选择A卡，因为它的新账期刚开始。
*   **技巧二：调整账单日**。许多银行支持每年修改一次账单日。若短期资金紧张，可在账单日前修改账单日至最远的日期，以延长还款周期。此方法每年仅一次，建议谨慎使用。

## 五、 了解信用卡相关费用

在日常用卡过程中，别忽略了卡片的各类费用，它们都是你的使用成本。

以下是四类主要费用：

1.  **信用卡年费**：银行每年收取的卡片使用费。普通卡/金卡通常可通过消费次数或金额减免；高端卡可能需用积分抵扣或为刚性收费。
2.  **罚息**：未及时全额还款或信用卡取现时，银行对全部透支额征收的利息。日利率通常为 **万分之五**。
3.  **提现手续费**：使用信用卡提取现金时收取的费用，一般为提现金额的1%-3%。叠加罚息后，综合成本年化利率可能超过 **20%**。
4.  **分期手续费**：账单分期还款时收取的费用，换算成年化利率通常也在 **10%** 以上。

整体而言，按时全额还款，信用卡使用成本很低甚至免费。但若未能养成良好的还款习惯，利息成本会迅速累积。

## 六、 逾期的影响与处理方法

最后，我们来谈谈另一个关键问题：如果信用卡忘记还款，会对信用产生影响吗？

银行通常设有 **1到3天** 的还款宽限期。在宽限期内还清欠款，一般不会产生逾期利息，也不会影响征信记录。但频繁使用宽限期可能影响银行对你的信用评估。

不同银行的宽限期规则（如截止时间）可能不同，建议致电信用卡中心确认。

如果不幸在宽限期内仍未还款，将构成逾期。此时应：
1.  立即还清全部欠款。
2.  尝试联系银行，说明情况，看能否开具 **非恶意欠款证明** 并申请删除不良记录。
3.  若不良记录已上传至征信系统，根据《征信业管理条例》，该记录将在还清欠款后保留 **5年**，5年后自动消除。
