# 负面情绪管理课程：02：正确认识你的悲伤情绪

在本节课中，我们将全面学习悲伤情绪。我们将了解悲伤的定义、来源、具体表现以及它发展的不同阶段。掌握这些知识，是有效管理情绪的第一步。

## 什么是悲伤？

悲伤是一种负性基本情绪，通常指由分离、丧失和失败引起的情绪反应。它包含沮丧、失望、气馁、意志消沉、孤独和孤立等情绪体验。

悲伤的程度取决于失去事物的重要性和价值大小，同时也依赖于主体的意识倾向和个体特征。

**重要性与价值**：例如，丢失一枚有纪念意义的情侣戒指所带来的悲伤，远比丢失一个早餐煎饼果子要强烈。

**主体意识倾向**：如果你对一段感情已心灰意冷，那么戒指的丢失可能带来解脱感；而那个一口没吃的煎饼果子带来的悲伤反而会更强烈。

**个体特征**：失去亲人会让人悲伤，但悲伤的程度和表达方式因年龄和经历而异。例如，重病富人的子女与远房亲戚的悲伤感受是不同的，这是亲密关系中本能的情感差异。

根据程度不同，悲伤可细分为：**遗憾、失望、难过、悲伤、极度悲痛**。这些情绪常伴随哭泣，有助于释放紧张、缓解心理压力。它是一种消极的减力情绪，也是一种心理保护措施。

然而，过强的悲伤对身心健康有害。持续的悲伤会使人感到孤独、无助、绝望，甚至引发临床抑郁。它也会削弱身体免疫功能，可能导致消化系统疾病、心血管疾病、肿瘤等。严重的悲伤甚至可能影响生理机能，导致猝死。因此，悲伤情绪需要酌情控制。

## 悲伤从何而来？

上一节我们定义了悲伤，本节中我们来看看悲伤情绪的源头。

人类的悲伤通常源于经历上的失败和挫折，以及对真实事物的失去。总的来说，都是因为既定事实对自己产生了负面冲击，从而表现出的情绪。

例如：大学毕业与好友各奔东西、离开家人去陌生城市打拼、与恋人分手、与朋友吵架、亲人离世等。

## 悲伤情绪的五个表现方面

悲伤的表现因人而异，没有固定模式。尽管如此，我们可以确定悲伤反应有五个宽泛的方面。

以下是悲伤在五个不同维度的具体表现：

1.  **情感方面**：可能包括忧伤、愤怒、负罪感、羞愧、解脱、无力、无助等感觉。
2.  **认知方面**：思想发生改变，可能包括对失去对象的强迫性思念、注意力分散、无法集中、爱发呆、感到迷失方向、表现得冷漠淡然。在面对生死离别时，可能反复深思死亡或离别场景。
3.  **躯体方面**：表现为躯体化症状，如头疼、肌肉酸痛、腹痛、胸闷、恶心、对噪音不耐受、紧张、食欲不振和睡眠紊乱。强烈悲痛还会削弱免疫系统功能，升高压力激素水平，并使人忽略健康的饮食和锻炼习惯。
4.  **行为方面**：最集中的表现是止不住地哭泣，并伴随焦躁不安的行为，如害怕社交、吸烟饮酒量增加，或频繁就医怀疑自己患病。
5.  **对人类存在的判断方面**：当生活中确定无疑的事情被中断，会推动我们去探索各种事情的意义。例如，亲人离世会让你思考死亡的意义；恋情失败会让你质疑自己的精神信念和价值观，通常导致对核心信念进行重新评估。

重要的是，悲伤是一个动态和反复的过程。以宠物离世为例：初期会痛苦不已并自责；一段时间后可能感到生活冷清、失去意义；随时间推移，虽然仍感悲伤，但情感不再难以承受，你会希望回归正常生活和社交，重新体验快乐。

## 悲伤的阶段与自我检测

悲伤情绪可以自愈。例如，告白被拒后，你会难过、闷闷不乐、对事物失去兴趣，但经过一段时间自我调整后，情绪会得到缓解。这种失望、不安、心塞的感觉可能持续一两天到四五个月不等，随后会随时间推移而恢复。

那么，这个过程如何划分？我们又该如何判断自己正处于哪个阶段？

美国精神科医生库伯勒-罗斯在《论临终与死亡》一书中，描述了人们面对不幸和灾难时要经历的五个独立阶段，被称为**库伯勒-罗斯模型**，也称作**悲伤的五阶段模型**。

你可以根据以下描述判断自己正处于哪个部分。

以下是悲伤情绪发展的五个典型阶段：

1.  **否认**：例如，当恋人突然提出分手，另一方的第一反应是不愿相信，认为这是个玩笑或试探。这种否认是一种防卫机制，我们选择性地隐藏事实以避免面对。
2.  **愤怒**：当意识到事情即将成为既定事实时，会感到非常恼火。可能会想：“我哪里做得不好？你凭什么分手？”并可能产生质问或报复的念头。
3.  **讨价还价**：试图寻找一切办法挽留即将逝去的关系。可能会说：“只要不分手，我什么都愿意改。”“我们还能做朋友吗？”通过一系列言行讨好、挽留对方。
4.  **消沉**：这是悲伤情绪的高峰。觉得一切无法挽回，努力也无用，从而陷入痛苦、无助、沮丧和自我怜悯中。整个人被悲伤压倒，觉得希望、梦想和未来计划都破灭了。
5.  **接受**：经过一段时间后，开始接受无法改变的现实。例如，接受分手事实，开始努力工作、接触新人、发展新恋情。当再次看到前任信息时，能坦然评价，不再抱有“我好难过”的情绪，新生活就此开始。

我们可以这样理解：情感思维是低级思维，逻辑思维是高级思维。大脑处于“省电模式”时，逻辑思维启动会滞后。因此遇到刺激时，第一反应往往是先爆发情绪（如遇危险先感到恐惧而逃跑）。悲伤的这五个阶段也是一个完整的心理防御过程。

**否认阶段**是本能地回避精神痛苦。如果一直走不出来，可能会沉沦，例如借酒消愁。
**愤怒阶段**是用愤怒情绪来振奋精神，缓冲痛苦。如果一直停留在此，人生可能只剩下恨和不信任。
**讨价还价阶段**是试图寻找回到安全、熟悉轨迹的方法。
**消沉阶段**是在所有尝试失败后的绝望，会让人放弃挣扎，从而有机会寻找新的解决之道。但长期消沉可能导致抑郁。
**接受阶段**是在经历所有步骤后，对痛苦的恐惧减弱，开始尝试拥抱痛苦、接纳悲伤，甚至将其视为新天赋（例如，地震后失去双腿的女孩安装假肢后想学习舞蹈）。

这五个阶段是必须的，无法跳过，因为前一个过程是在为后一个过程做准备、积蓄力量。面临失去和分离后的痛苦是正常的，这证明我们还活着。这些悲伤的回忆会随时间流逝和自我成长而减弱甚至消失，能让当事者缓解心情，重新适应环境，这也是悲伤情绪的最大益处。

## 总结

本节课中，我们一起学习了如何正确认识悲伤情绪。

*   **什么是悲伤？** 悲伤是一种由分离、丧失和失败引起的负性基本情绪，包含沮丧、失望、孤独等多种体验。
*   **悲伤从何而来？** 通常源于经历上的挫折失败和对真实事物的失去，是既定事实产生负面冲击后表现出的情绪。
*   **悲伤的五个表现方面**：情感、认知、躯体、行为以及对人类存在的判断。
*   **悲伤可以自愈**：经过一段时间（一两天到四五个月）的自我调整，情绪能够得到缓解和恢复。
*   **悲伤的五阶段模型（库伯勒-罗斯模型）**：
    1.  **否认**：防卫机制，选择性隐藏事实。
    2.  **愤怒**：对事实感到恼火与不平。
    3.  **讨价还价**：试图挽留或改变结果。
    4.  **消沉**：陷入痛苦、无助和绝望。
    5.  **接受**：坦然面对现实，开始新生活。
    这五个阶段是一个完整且必要的心理防御过程，无法跳过。
