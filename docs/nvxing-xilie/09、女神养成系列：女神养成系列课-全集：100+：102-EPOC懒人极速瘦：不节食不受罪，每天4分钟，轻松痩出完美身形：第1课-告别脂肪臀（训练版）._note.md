# 女神养成系列：09：告别脂肪臀（训练版）💪

在本节课中，我们将一起完成一套针对臀部的HIIT（高强度间歇训练）训练，并学习训练后至关重要的拉伸放松方法。整个过程无需器械，每天仅需4分钟，旨在帮助你有效塑形。

## 训练环节 🏃‍♀️

现在，我们进入今天的陪练环节。请准备好，跟随指令一起完成训练。

以下是第一组训练动作，共四个动作，我们将进行多轮循环。

1.  **臀桥**
    *   仰卧，屈膝，双脚平放于地面。
    *   吸气准备，吐气时臀部发力向上顶起，直至身体呈一条直线。
    *   在顶端收紧腹部和臀部，然后缓慢下放。

2.  **侧卧抬膝（棒式变式）**
    *   侧身，用肘部和脚部支撑身体，呈侧平板姿势。
    *   保持髋部稳定，将上侧的膝盖尽量向上方抬高。
    *   感受臀部侧面的发力感。

3.  **深蹲**
    *   站立，双脚与肩同宽。
    *   屈髋屈膝向下蹲，保持背部挺直，膝盖与脚尖方向一致。
    *   在起身时，刻意感受臀部发力夹紧，并向前顶髋。

4.  **开合跳**
    *   站立，跳跃时双脚向外打开，同时双手向上击掌。
    *   再次跳跃时，双脚并拢，双手放回身体两侧。
    *   如果感到疲劳，可以减慢速度，或改为原地深蹲。

第一组训练完成后，你可能已经感到腿部酸疼。但我们尚未结束，接下来将立即开始第二组训练，中间没有休息。

第二组训练我们重复上述四个动作，请在30秒内尽可能多地完成每个动作。过程中保持腹部收紧，在臀桥动作中尽量将臀部顶高。

## 拉伸环节 🧘‍♀️

训练结束后，不要马上喝水、坐下或躺下。就像我们热身一样，训练后的拉伸是让肌肉更好恢复和生长的关键环节。现在，有请安迪老师带领我们进行放松拉伸。

以下是全身各部位的拉伸动作，每个动作保持15-30秒，保持均匀呼吸。

*   **颈部拉伸**
    *   **侧屈拉伸**：将食指和中指放在头部外侧，轻轻向另一侧拉动头部。保持身体中立，强度不宜过大。
    *   **旋转拉伸**：头部微微向斜前方旋转，同侧手放在头部后侧轻轻下压，眼睛看向斜前方，拉伸颈部后侧。

*   **肩部与背部拉伸**
    *   **肩部后侧拉伸**：一只手臂伸直向前，另一只手扣住其肘关节外侧，向身体另一侧拉伸。
    *   **肩部前侧拉伸**：双手在背后十指相扣，手掌向外旋转，肩胛骨下压，向后伸展，打开胸腔。
    *   **背部拉伸**：双脚分开站立，一脚后移。一手向斜上方延伸，另一手扶髋，保持腰椎中立，感受背部的牵引。

*   **下肢拉伸**
    *   **髂腰肌拉伸**：单膝跪地，前侧腿弓步，双手向上延伸，收腹顶髋，身体微微前倾。
    *   **股四头肌拉伸**：站立，一手扶墙保持平衡，另一只手抓住同侧脚踝，将脚跟拉向臀部，感受大腿前侧的拉伸。
    *   **臀部拉伸**：坐姿，将一侧脚踝放到对侧膝盖上方，身体缓慢前倾。或采用四足跪姿，将一侧脚踝放到对侧膝盖前方，身体下压。
    *   **大腿后侧拉伸**：坐姿，一侧腿伸直勾脚尖，另一侧腿弯曲，身体缓慢向伸直腿的方向前倾。

拉伸环节到此结束。你是否感觉像泡了一次温泉，做了一次按摩，全身血液流通更顺畅了呢？这非常舒适。
