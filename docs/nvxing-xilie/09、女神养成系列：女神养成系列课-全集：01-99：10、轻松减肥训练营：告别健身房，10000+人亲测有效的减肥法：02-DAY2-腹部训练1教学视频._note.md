# 轻松减肥训练营：02：腹部训练1教学视频

在本节课中，我们将学习一套高效的腹部训练动作。这套训练无需器械，在家即可完成，核心在于掌握正确的呼吸方法并保持腹部持续发力。我们将通过几个循环的动作，帮助你有效锻炼腹部核心肌群。

## 🎯 概述：训练核心与呼吸

上一节我们介绍了训练的整体理念，本节中我们来看看具体的腹部训练动作。在整个训练过程中，请务必使用我们之前强调的**腹式呼吸法**，即在吸气时腹部鼓起，呼气时腹部收紧内收。这能确保你的腹部肌群深度参与每一个动作。

## 🧘 第一部分：仰卧腿部练习

首先，我们进行仰卧位的腿部练习，重点激活下腹部。

1.  请跟随指令仰卧在垫子上。
2.  将左腿向上尽量伸直。若无法完全伸直，可微屈膝盖。双手扶在左腿后侧作为支撑。
3.  右腿向下伸直，准备开始动作。
4.  采用腹式呼吸。**吸气准备，呼气时将右腿向上抬起**。
5.  将腿抬至垂直于地面的高度，然后**吸气，缓慢下放**。
6.  此动作重复进行6次。
7.  完成规定次数后，换另一侧腿进行相同练习。

以下是动作要点列表：
*   抬起腿时呼气，放下时吸气。
*   动作过程中，保持上半身稳定，腹部收紧。
*   感受腹部发力，而非用腿部惯性完成动作。

## ✨ 第二部分：百次拍击

完成腿部练习后，我们进入经典的腹部训练动作——百次拍击，以锻炼腹部耐力。

1.  仰卧，将双腿抬高至与地面水平。
2.  将上半身慢慢卷起，确保腰部紧贴垫面。
3.  双臂伸直置于身体两侧，上下小幅度摆动（拍击）。
4.  保持肩膀下沉，呼吸节奏稳定。
5.  持续进行此动作100次。

以下是动作调整建议：
*   若颈部感到不适，可将双手扶在头后作为支撑。
*   过程中保持呼吸，不要憋气。
*   当感到疲劳时，可稍作停顿，但尽量坚持完成。

## 💪 第三部分：平板支撑

接下来，我们通过平板支撑来强化核心整体稳定性。

1.  俯卧，用双肘和脚尖支撑身体。
2.  双手可十指交扣，双肘位于肩膀正下方。
3.  收紧腹部和臀部，使身体呈一条直线。
4.  保持这个姿势5秒钟。
5.  之后休息，进行两次深呼吸。
6.  将此支撑练习重复5次。

以下是动作要领：
*   **身体从头顶到脚跟应保持一条直线**。
*   呼气时，刻意感受腹部进一步收紧。
*   避免臀部过高或塌腰。

## 🚶 第四部分：站姿抬腿

最后，我们以站姿动作结束训练，在动态中巩固腹部控制力。

1.  保持站姿，双臂向前平举。
2.  在保持上半身完全不动的前提下，交替向前抬腿。
3.  利用腹部收紧的力量将腿抬起。
4.  抬腿时呼气。

以下是关键提示：
*   动作的重点是**用腹部发力**，而不是摆动腿部。
*   保持身体平衡，核心收紧。
*   动作速度放缓，感受每一次收缩。

## 📝 总结与循环

本节课中我们一起学习了四个核心的腹部训练动作：仰卧抬腿、百次拍击、平板支撑和站姿抬腿。记住，所有动作的质量远比数量重要，确保使用腹式呼吸，并时刻感受腹部肌群的发力。

你可以将以上所有动作视为一个训练循环。根据自身能力，重复进行2-3个循环，组间可进行短暂休息。坚持练习，你将感受到腹部力量的提升。
