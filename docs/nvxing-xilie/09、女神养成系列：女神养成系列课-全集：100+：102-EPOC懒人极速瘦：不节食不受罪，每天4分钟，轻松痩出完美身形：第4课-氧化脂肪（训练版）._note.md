# 女神养成系列：第4课：氧化脂肪（训练版）

在本节课中，我们将学习一套高强度间歇训练（HIIT）动作，旨在通过短时高效的训练促进脂肪氧化，并配合完整的拉伸流程，帮助身体恢复。课程总时长约4分钟，无需节食，适合初学者跟练。

## 🏃‍♀️ 第一部分：高强度间歇训练

上一节我们介绍了训练原理，本节中我们来看看具体的动作流程。训练共包含四个动作，每个动作持续30秒，动作间无休息，连续完成两组。

以下是第一个动作的要点：
*   **动作名称**：马步蹲跳
*   **起始姿势**：马步下蹲，膝盖弯曲。
*   **动作要领**：利用大腿发力进行上下蹲跳，保持均匀呼吸，不要憋气。
*   **注意事项**：感觉疲劳时，身体可略微抬起，但膝盖需保持弯曲。后期可加快速度。

以下是第二个动作的要点：
*   **动作名称**：开合跳
*   **起始姿势**：自然站立。
*   **动作要领**：跳跃时双脚打开，双手在头顶击掌；回落时双脚并拢，双手放于身体两侧。
*   **注意事项**：落地要轻，注意保护膝盖。动作幅度可尽量大一些。

以下是第三个动作的要点：
*   **动作名称**：原地登山跑
*   **起始姿势**：平板支撑姿势，身体呈一条直线。
*   **动作要领**：交替将膝盖向胸部方向快速提拉，尽量让膝盖靠近地面。
*   **注意事项**：身体保持平板姿势，核心收紧，避免腰部下沉或拱起。

以下是第四个动作的要点：
*   **动作名称**：弓步提膝
*   **起始姿势**：从右侧或左侧弓步开始。
*   **动作要领**：从弓步姿势发力向上，将前腿膝盖向上提起至高位，然后换另一侧进行。
*   **注意事项**：此动作旨在冲击心肺，请尽力加快速度，保持努力。

完成第一组四个动作为一轮，紧接着不休息，立即开始第二轮相同的四个动作。

## 🧘‍♀️ 第二部分：训练后冷身

完成高强度训练后，心脏处于高速跳动状态，不宜立即坐下或躺下。正确的冷身有助于心率平缓恢复。

以下是冷身建议：
*   在空间内缓慢走动。
*   或采取舒适站姿，双手扶膝，臀部后推，挺胸打开胸腔，进行深呼吸。
*   **呼吸模式**：用鼻子深吸气，用嘴巴缓慢呼气。

## 🤸‍♀️ 第三部分：全身拉伸放松

拉伸是训练中至关重要的环节，能有效缓解肌肉紧张，促进恢复。请跟随指导依次进行。

以下是颈部拉伸动作：
1.  **侧颈拉伸**：用食指和中指放在头部一侧，轻轻向对侧拉伸，保持15秒，换边。
2.  **颈部后侧拉伸**：头部向右前方旋转，右手置于头后轻轻下压，保持15秒，换边。
3.  **颈部前侧拉伸**：十指相扣，大拇指顶住下巴，轻轻向上抬头，保持15秒。
4.  **胸锁乳突肌拉伸**：头部向一侧旋转，感受颈部侧前方拉伸，保持15秒，换边。

以下是肩部与背部拉伸动作：
1.  **肩部后侧拉伸**：一手臂伸直向前，另一只手扣住其肘关节向外拉伸，保持15秒，换边。
2.  **肩部前侧拉伸**：双手在背后十指相扣，肩胛骨下压，向后拉伸，打开胸腔，保持15秒。
3.  **背部拉伸**：双脚分开，左脚放于右脚后，左手向右上方伸展，右手扶腹，感受背部拉伸，保持15秒，换边。

以下是下肢拉伸动作：
1.  **髂腰肌拉伸**：单膝跪地，前腿弓步，双手上举，收腹顶髋，身体向前平移，保持15秒，换边。
2.  **股四头肌拉伸**：站立，一手抓住同侧脚踝，将脚跟拉向臀部，身体可前倾，保持15秒，换边。
3.  **臀部拉伸**：坐姿，右腿弯曲跨过左腿，左手环抱右膝向身体拉近，保持15秒，换边。
4.  **大腿后侧拉伸**：坐姿，一腿伸直勾脚尖，身体向前俯身，保持15秒，换边。

## 📝 总结

本节课中我们一起学习了由四个动作组成的高强度间歇训练流程，理解了训练后冷身的重要性，并完成了一套系统的全身拉伸序列。记住，**训练强度决定收获**，坚持练习，配合正确的呼吸与拉伸，就能在高效燃脂的同时保护身体，获得舒适的训练体验。坚持下去，你一定能看到改变。
