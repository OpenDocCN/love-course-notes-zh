# 《富贵兵团》：第七期：富二代的困局 🏰

在本节课中，我们将通过分析豪门媳妇徐子淇的经历，探讨富二代在家族财富与地位传承中所面临的复杂局面。我们将看到，即便身处泼天富贵之中，个人也需不断调整策略、明确自身价值，才能在激烈的内部竞争中稳固地位。

上一节我们介绍了徐子淇嫁入豪门后的初期境遇，本节中我们来看看她在遭遇重大挑战后，如何反思并调整自己的生存策略。

徐子淇在为李氏家族生下两个孙女后，其地位受到严重冲击。李兆基的大儿子李兆杰通过代孕生下三个男孙，此事对徐子淇是一次沉重的打击。此后，徐子淇在社交场合基本销声匿迹。一方面，徐子淇希望尽快与丈夫李嘉诚生育男孩。另一方面，徐子琪通过这次事件，开始深度反思自己在超级豪门中的核心价值所在。

我们可以假设，如果徐子淇第一胎就是男孩，她可能被捧上神坛，在李氏家族立下头等功。但事情的发展往往出人意料。徐子淇在大伯子生下三个男孙后，经过一年多尝试各种方法，最终也生下了一个男孩。但此时，徐子淇在李氏家族的地位已大不如前。整个情势发生了巨大变化。

徐子淇最初认为富贵来得轻松，因而掉以轻心。她忽视了周围环境一直暗潮汹涌。原来嫁入豪门并非徐子淇想象中那样，并非每日豪车豪宅、购物消费、高调张扬。所有东西都需要代价。徐子淇刚嫁入时非常高调。在她生下两个女孩后，李兆杰与李兆基高调地将代孕所生的三个男孙照片公布给媒体。这或许是李兆基对徐子淇高调作风的一种敲打。老人家心底可能早已对这个媳妇不满。

李氏家族所有人的核心就是李兆基。无论是大儿子、小儿子李嘉诚，还是三个女儿，全部都要围绕李兆基来转。这个家族的强人只有老爷子一人。无论是徐子淇嫁给的超级富二代李嘉诚，还是她的大伯子李兆杰，他们都是标准的富二代。他们的存在都基于其父亲、真正的创富一代——李兆基。

在这种顶级豪门，尤其子女众多的家庭中，兄弟姐妹从小在富贵氛围中生活，耳濡目染皆是名利。他们学到的核心技能之一，便是对财富与资源的争夺。

以下是媒体常见场景背后的逻辑：
大家看到媒体报道李嘉诚出席活动时，经常看到他的大儿子、二儿子、女儿、孙女、徐子淇及其父母一同参加。这种全家出动的场面，看似其乐融融。但媒体上父慈子孝、团结友爱的场面，往往只是一瞬间的定格。背后转脸，笑容可能就消失了。

如此众多的子女与亲家围绕在李兆基身边，目的何在？有一句老话叫“见面三分情”，核心就是要刷存在感。整个家族财富的核心是李兆基，必须让他看见自己、记住自己。因为李兆基有两个儿子、三个女儿，他偏爱谁，就可以给谁更多资源。不喜欢、不常见到的人，自然所得就少。这关系到未来的财富分配。所以大家争相围绕李兆基，一是刷存在感，二是获取情报。李兆基是整个家族的决策核心，他的动向与投资方向是至关重要的情报。

真正与李兆基走得近的，起初是大儿子，然后是大女儿李佩玲。李佩玲是个特别聪明的人。他们都知道，抓住老头子，才能把家族资源抓在手里。因此，顶级豪门那些其乐融融的照片极具迷惑性。

徐子淇的丈夫李嘉诚，是李兆基的儿子。他最初在家族中存在感并不强。因为李嘉诚本身是一个资质平庸的富二代，学习与工作均不突出，只是命好。反而在娶了徐子淇之后，因徐子淇高调生子，不断帮助李嘉诚刷存在感。

例如，李兆基家族在深圳湾一号买房时，给每个子女都买了一套。其实全家都跟去了售楼部。换句话说，买房直接买给子女即可，为何兴师动众都跟着？还是那句话：见面三分情，让老头子记得自己。徐子淇及其母亲深谙此道，所有李兆基在场的场合，她们一定要跟上，绝不能落后于大哥和大姐。徐子淇的目标就是把丈夫李嘉诚推上去，让他在父亲面前不断刷存在感。

李兆基的大女儿李佩玲聪明至极，李兆基到哪，她就跟到哪。李佩玲自己有三个女儿和一个儿子，都把李兆基哄得团团转。她心里清楚，所有东西都是父亲的，只要父亲喜欢自己，就什么都有了。只要牢牢抓住李兆基，她和孩子以后就无需发愁。

在这种超级豪门之家，兄弟姐妹之间存在竞争关系。**竞争的本质公式可以概括为：`谁离核心资源（李兆基）更近 ≈ 谁获得更多财富分配`**。谁离老头子更近，谁就离财富更近。老头子最喜欢谁，就会把更多财富分配给谁。

徐子淇是什么人？她在被大伯子李兆杰代孕三个男孩打脸之后，彻底调整了自己在家族中的战略。徐子淇明白，她所有的靠山就是丈夫李嘉诚，而李嘉诚的靠山是李兆基。她必须将李嘉诚托举到李兆基面前，让老头子重视起这个小儿子。如此，徐子淇才能巩固自己在家族中的地位。

徐子淇14岁就混迹娱乐圈做模特赚钱。她十几岁就出来混社会，此时必须发挥自己长袖善舞的社交天赋。徐子淇的老本行就是做模特、作秀，把最好最完美的一面展示给大家。她的核心技能就是经营形象。现在，她只是将工作平台转换到豪门的上流社交舞台。这对徐子淇来说易如反掌。

但术业有专攻，专业的事情需要专业的团队来做。此时，徐子淇找到了一个对其豪门形象转变至关重要的团队——天机公关公司及其CEO郑绍康。天机公关是赌王女儿何超琼创立的公司。时间线显示，在徐子淇生下第三个男孩后，她便整装待发，全心打造自己在豪门中的重要IP。郑绍康的团队为她量身定制了后期规划。

香港真正的上流名媛圈，以前徐子淇难以进入。这不仅是钱的问题，更关乎传统。此时，天机公关帮助徐子淇做到了。徐子琪在这方面估计花费不菲。上流社会的慈善拍卖会等社交场合，座位排位很有讲究。谁是大咖、二咖、三咖，背后都有一系列公关策划，需要付费。并非想做C位就能安排到C位，而是明码标价、对号入座。

大家需要转换一个思维：参加这种名媛社交场合，并非只是吃穿玩乐。社交是一件非常耗精力的事情。出席前需考虑穿什么衣服、是否新款、如何应对记者提问、如何公关不利问题等。这非常辛苦，远不如在家看电视刷手机轻松。所以，社交是一项辛苦的工作。

后来，经过何超琼旗下天机公关的打造，一个全新的徐子淇重返社交场合。经过约一年的深耕，徐子淇完全打入了真正的上流社会名媛圈，如鱼得水。她与林恬儿等Old Money家族的名媛都保持了非常好的关系。并且，她把丈夫李嘉诚也带入了核心的豪门社交圈，让丈夫的存在感不断增强，这也是徐子淇的最终目的。

这里补充一点：李嘉诚这个富二代资质平平。从他出席社交场合的眼神可以看出，基本上是妻子徐子淇在控场。他的眼神显得木讷呆滞，最初不知如何与记者打招呼交流。但徐子淇14岁出道，在与记者的较量中完全掌握主动权。可以说，是徐子淇带着李嘉诚成长。在社交场合深耕一两年后，徐子淇真正把李嘉诚带出来了。后来李嘉诚面对记者刁钻提问，也能从容应答。

徐子淇另一个聪明之处在于，她和李嘉诚结婚时，李兆基送了他们两套打通的500平房子，但他们一直未住，而是与老爷子李兆基住在一起。李兆基是做房地产出身，整个家族住在麦当劳道的豪宅“惠苑”中。徐子淇和李嘉诚一直跟老爷子住在一起，这也能看出儿媳妇的聪明。必须每天在老爷子面前出现，才能让他更疼自己、更注意到自己。

另外，令人佩服的是徐子淇的身材管理。怀第一胎、第二胎女儿时，她不太注意身材，保持婴儿肥。但到生第三胎、第四胎时，她越生越瘦，并一直保持白瘦幼的身材。可见她后期非常自律，对自己够狠。她的衣着品味也从刚嫁入时的夸张爆发户风格，慢慢收敛。最终，徐子淇找到了最适合自己的定位：为李家站台做慈善，担任“智行基金会慈善大使”的身份。这个公开身份非常适合她，捐衣服、捐款、看望难民，对她来说易如反掌。

此时徐子淇的生活已是天家富贵。李兆基最看重的其实是“名”。徐子淇非常会揣摩老爷子的心思，她就来做李家的“名”，做慈善。她每次穿得漂漂亮亮（品牌方赞助），捐点东西和钱（反正不是自己赚的钱），顺便介绍李氏家族的智行基金。最终，徐子淇名利双收。

在嫁入李氏家族的十几年里，她不断调整、变通、摸索，最终找到了最适合自己的定位。后期观察徐子淇的状态，她是变化的。相信她在真正做慈善后，见识到世界上活得非常辛苦的人，也被自己“慈善大使”的身份所滋润和感动。因此，徐子淇近年的状态日趋平和，很难再看到她刚嫁入时像火鸡一样夸张、用力过猛的打扮。现在露出的照片多是与朋友吃饭喝茶的正常生活照。这说明徐子淇的心态变得平和、正常。

说到这里，我们想问：像徐子淇嫁给李嘉诚这十几年，一方面帮助李嘉诚刷新了家族存在感，稳固了双方地位，到底谁更幸运？是徐子淇更幸运，还是她老公李嘉诚更幸运？其实两个人都很幸运，因为他们彼此成就，彼此帮助。

有一个小段子：美国前总统克林顿带妻子希拉里去加油站，发现站长是希拉里的前男友。已是总统的克林顿说：“要不是嫁给我，你现在就是加油站站长的老婆了。”希拉里反驳道：“那你应该庆幸娶了我，不然的话，这个加油站的站长就是美国总统了。”这只是一个段子，但说明一个道理：真正强大的女人嫁给什么样的男人，其实是那个男人的福气。所以李嘉诚和徐子淇都是幸运的。

以上是我们用三期课程讲述的超级豪门阔太徐子淇的故事。大家最后看徐子淇是阔太，其实她一直在摸索自己的身份，不断变通、调整、改变，最终达到了比较适合自己的状态。

由此，我想延伸谈论一个关于“全职太太”的话题。上周微博有个热点：全职太太离婚只补偿5万元。要不要做全职家庭女性？我来说一点观点。我们富贵兵团直播间前后几期穿插讲了董花花、徐子淇这类故事。我已经把全职家庭女性讲透了。我想问大家一句：富贵如董花花、徐子淇这种段位的超级阔太，在人生某个阶段都会遇到危机，都会知道自己不事生产、不懂变通。如果只知道花钱购物、喝下午茶，都可能让自己处于危机状态。请问普通如我们，请问你们的老公有董花花和徐子淇的老公有钱吗？如果不是，你们哪来的底气去做一个真的什么都不操心、不事生产、又不帮助老公的全职家庭女性呢？在这个问题上，我不表态，只问大家这一句，大家自己去领悟。

今天我们的主题是以徐子淇为线索，讲到了富二代的困局。徐子淇帮助丈夫李嘉诚在兄弟姐妹众多的家族中刷新了存在感。最终，李嘉诚在前两年成功巩固了自己在家族的地位，获得了恒基兆业集团在香港的全部业务，而他的哥哥李兆杰则得到了内地的全部业务。徐子淇在这里面多多少少是有功劳的。

以上所有观点，是我从一个女人的角度看徐子淇的困局，包括她丈夫作为富二代的困局。接下来，我们把时间交给老李校长。他接触的人更多，将从男人的角度，谈谈他接触的富二代中有哪些真正的困局。

---

老李校长分享：

今天王阿姨让我讲富二代困局，也就是“富不过三代”怎么理解。我给大家讲讲我的看法。“富不过三代”这句话，在穷人眼里和富人眼里的理解完全不同。

我先说穷人角度怎么理解。作为穷人，“富不过三代”一定带有酸的成分，有嫉妒成分。他并不能真正理解为什么富不过三代。他可能认为，因为生的孩子一代不如一代，吃喝玩乐，能力越来越差，把家败完了。穷人的理解，想象力仅限于此。

从我的角度，我是白手起家、屌丝逆袭，能理解赚钱的辛苦。所以我理解的“富不过三代”和屌丝时不一样。我屌丝时肯定也那么想：每天吃喝玩乐，肯定差，还不是靠你爸。我理解的富二代困局，第一有以下几个原因。

首先，与他父亲的商业模式有关。如果像李嘉诚那种商业模式，例如掌握电力系统、码头系统、自来水等垄断性公司，说难听点，你放条狗管都不一定比人管得差，因为这是垄断行业，不需要激烈竞争。商业模式决定传承难度。如果诞生于这样的家庭，富二代可以富好多代，因为有法律保护财产，竞争也不激烈。

但如果这个富二代的家族生意，是一个高度内卷、高度竞争的模式，那就真的不好说了。也许不是说你去英国美国读个MBA，从一线做起就能解决。你把事情想得太简单、太容易了。江浙那边有一些上市公司，儿子死活不接班。为什么不接班？因为他知道公司靠借新还旧、靠贷款运作为生，这在传统重资产行业太普遍了。开加油站等生意特别辛苦。所以这样的富二代，不是他不努力，而是他无法努力。人不能与天斗。随着竞争激烈、新对手不断进攻，一代代往下传真的很难。

我在北京认识几个江浙沪的富二代，英国美国留学回来，死活不接班。有去未来汽车上班的，有去Uber上班的，就是没人回去接班。因为他爸的生意太苦了。

还有一种商业模式，是关系型商业模式，例如现在靠拿工程、拿项目的。如果这种富二代接班，也非常非常难。如果富二代是女孩，第一，她爸的生意她没法接，因为全是喝酒、赔笑、中标、组织工人进场施工这些事。这只能父辈干，富二代没法干。如果他是男富二代，我要告诉大家：关系是没办法继承的，钱可以继承，但人与人之间的交情无法继承。你想想，你跟你同事关系好，你会买他小孩的账吗？生意上的人脉关系需要重新去巩固。所以作为富二代，往下传承，跟生意模式也有关。

当然，如果你做家族信托。在美国，家族信托是1000万美金起，用于合理避税。否则，光英国、美国的继承税就非常重。我认识的富二代，没有一个移民美、英、加、澳、新的，为什么？税太重了。他们移民都去希腊，或通过加勒比岛国做跳板，或者去新加坡，因为税低。新加坡税很低。否则，光遗产税，你在几代以后就没有了。

这是我今天准备的，关于“富不过三代”，不同人有不同坎。还有，人要有自知之明。很多富起来的人，他知道这事真难。他知道如果再来一次，得经历多少苦、遭多少罪，需要何等的毅力、决心和手段。所以他有时也放弃了让孩子像他一样勇敢果决的想法。因为人的能力不是通过读书就能培养出来的。高度竞争的商业环境下，人的能力来自于挫折、苦难和事情的折磨。但富二代那个财富级别的家长，会不会让孩子受这个苦？这都是个问号。就如同我认识的富二代，剑桥毕业的，不去继承家里整天要拿项目的苦生意，反而去未来汽车上班了。因为与后者相比，前者更苦，上班反而轻松。

好的，老李从一个男人的角度分享了他对富二代目前困局的看法。无论是我也好，老李校长也好，伯爵也好，我们周围接触过形形色色的富二代，各行各业，高的、胖的、瘦的、懒的、勤奋的、成功的、平庸的、败家的都有。这个群体到底是怎样的？我们上次跟老李聊选题时，也准备在下一期真正把“富二代”这个话题聊透。每个人对富二代的看法，包括怎么破局，我们都给他聊透了。下一期可能还会邀请外企的老福总裁，让他从他的角度来看看富二代的现状和怎么破局。

今天的徐子淇话题，我们通过三期课程基本就结束了。我们下一期再见。

本节课中我们一起学习了富二代在豪门家族中面临的复杂困局。通过徐子淇的案例，我们看到个人价值定位、持续的战略调整、专业的形象打造以及深刻理解家族权力核心的重要性。同时，也从更宏观的视角探讨了“富不过三代”与商业模式、竞争环境及个人能力传承的深层关联。真正的稳固，来自于对自身价值的不断挖掘与对环境的持续适应。
