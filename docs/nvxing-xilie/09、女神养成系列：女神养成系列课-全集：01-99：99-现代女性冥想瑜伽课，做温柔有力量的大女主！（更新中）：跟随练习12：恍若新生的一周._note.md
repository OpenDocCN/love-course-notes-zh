## 无感质感法：收摄感官

任何时候，你都可以选择仰卧下来。现在，我们开始进行**无感质感法**的练习。

我们拥有不同的感官。面部有四种，全身一共有五种感官，分别是：**嗅觉**、**味觉**、**视觉**、**听觉**、**触觉**。

我们的感官总是在头脑的驱使下去追逐外界的感官对象。头脑是一个媒介，借由这个媒介，所有的感官都向外趋向感官对象。

如果我们能通过**无感质感法**的练习，那么我们的头脑就无法再驱赶感官向外寻找。

你的感官与身体的某些特定部位会建立关联。例如：
*   关照**小舌头**可以改善视力。
*   关照**舌根**可以改善听力。
*   关照**鼻尖**可以改善嗅觉。
*   **舌尖**与味觉相关。
*   **舌头的中间**与触觉相关。

我们通过关照身体内部的这些关键部位，来进行无感质感练习。在关照每个部位时，保持4到7轮的呼吸（一吸一呼为一轮）。我们用这种积极的方式来控制感官，控制一些不必要的感官享受。

现在，保持自然的呼吸。在感受刚才呼吸觉知技巧三带来的效果的同时，开始以下练习：

以下是具体的练习步骤：
1.  **关照鼻尖**：保持7轮呼吸。在吸气时观察发生了什么，在呼气时观察发生了什么。鼻尖与我们的嗅觉相关，在关照鼻尖的同时，让我们的嗅觉收回来。
2.  **关照舌尖**：保持7轮呼吸。在吸气时观察发生了什么，在呼气时观察发生了什么。舌尖与我们的味觉相关。
3.  **关照舌头中间**：保持7轮呼吸。在吸气时感受舌头中间发生了什么，在呼气时感受这个部位发生了什么。舌头的中间与我们的触觉相关。
4.  **关照舌根**：保持7轮呼吸。在吸气时感受舌根部位发生了什么，在呼气时感受舌根部位发生了什么。舌根与我们的听力相关。
5.  **关照小舌头**：保持7轮呼吸。在吸气时感受这个部位发生了什么，在呼气时感受这个部位发生了什么。小舌头与我们的视力相关。

以上是无感质感法的练习。在冥想、运动和平时的日常生活中，我们都可以运用无感质感法来收摄心神。

## 昆达里尼总持法：引导能量

现在，我们来练习一个简单但非常有效的总持方法，称为**昆达里尼总持法**。

吸气时，感受气息来到**根轮**（会阴区域）。我们的呼吸保持正常。
呼气时，让你的气息去触碰并依次经过不同的脉轮，自下而上：从**根轮** -> **腹轮** -> **脐轮** -> **心轮** -> **喉轮** -> **眉心轮** -> **顶轮**，最后超越顶轮到**明点轮**，再一直向上直到**梵天轮**。

这种方法虽然简单，但十分有效。通过正确的练习，可以帮助我们在冥想中找到清晰的能量通道。

现在，开始总持的练习：
1.  **吸气**：气息进入鼻尖，向下引导至**根轮**。
2.  **呼气**：气息从根轮依次向上，经过**腹轮** -> **脐轮** -> **心轮** -> **喉轮** -> **眉心轮** -> **顶轮**，并一直向上。这样为一轮。

请你自己独立练习15轮。

感受呼吸去触及到的每一个脉轮，在那里产生了什么样的效果？

在完成了15轮以后，去感受身体内发生的整体效果。
