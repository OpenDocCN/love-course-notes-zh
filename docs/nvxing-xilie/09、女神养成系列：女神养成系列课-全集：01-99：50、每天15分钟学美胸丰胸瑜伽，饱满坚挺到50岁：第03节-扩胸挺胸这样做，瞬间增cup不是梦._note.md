# 女神养成系列：03：扩胸挺胸这样做，瞬间增cup不是梦 💃

在本节课中，我们将学习如何通过特定的瑜伽动作和呼吸法来打开胸部形态，塑造挺拔饱满的胸型。课程将包含四个核心练习，并会使用书本和椅子作为辅助工具。

上一节我们学习了如何塑造背部曲线，本节我们将专注于胸部的塑造。

## 美胸呼吸法 🧘‍♀️

正确的呼吸是改善体态的基础。美胸呼吸法能有效帮助胸部挺拔和扩张。

**核心要点**：无论在吸气还是呼气时，胸部都应保持向外扩张和上提的感觉。

### 练习一：仰卧书本辅助呼吸

首先，我们需要准备几本书作为道具。

**动作步骤**：
1.  仰卧，将书本垫在肩胛骨区域下方。
2.  屈膝，脚掌踩地。书本的高度应使头部向后仰时，刚好能触碰到地面。
3.  双手可放松置于身体两侧，或一手放在胸部上方以感受胸部的起伏。
4.  开始呼吸练习：
    *   **吸气**：胸部向外扩张，向上顶起。
    *   **呼气**：腹部收紧，但胸部保持上提，不要下落。同时，肩胛骨微微向后夹紧。

这个动作不仅能纠正背部曲线，配合胸式呼吸更能有效塑造胸部线条。对于有驼背习惯或久坐的办公族尤为有益。

### 练习二：跪姿支撑呼吸

如果找不到胸部发力的感觉，可以尝试这个动作。

**动作步骤**：
1.  跪立，双手在臀部后方约20厘米处撑地，五指大大张开，指尖朝向臀部。
2.  手肘可微屈或伸直，但需向内收紧，不要向外打开。
3.  挺起胸部，开始美胸呼吸：
    *   **吸气**：胸部向上、向外顶起。
    *   **呼气**：肩胛骨向内、向后收紧，胸部继续保持上提。

这个动作能更好地将呼吸引导至胸腔，让胸部感觉更灵活和丰满，同时也能有效活动肩胛骨。

## 强化胸部曲线练习 💪

掌握了基础的呼吸法后，我们通过以下动作进一步强化效果。

### 练习三：仰卧顶胸式

这个动作能同时锻炼胸部、手臂和头部，益处颇多。

**动作步骤**：
1.  仰卧于垫上，双手伸直放于臀部旁侧，肩膀放松下沉。
2.  **吸气**，腹部隆起。
3.  **呼气**，收腹，弯曲手肘，用头顶和手肘向下推地，同时将胸部向上顶起。
4.  保持姿势约12秒，期间配合美胸呼吸（吸气时胸部上顶，呼气时肩胛骨内夹）。
5.  起身时，用手肘撑地，将头部慢慢抬起还原。

**动作益处**：
*   **丰胸**：直接刺激胸部区域。
*   **提升记忆力**：头部后仰按摩百会穴，促进脑部血液循环。
*   **改善睡眠与美容**：刺激下丘脑和垂体，有助于调节激素（如**雌性激素**）分泌，使肤色红润。

### 练习四：婴儿式变体（及椅子辅助）

这是课程的最后一套动作，能深度拉伸并扩张胸部。

**地面练习步骤**：
1.  跪立于垫上，大腿垂直于地面。
2.  呼气，身体前趴，臀部抬高。额头点地，双手向远方伸直。
3.  保持大腿垂直，配合呼吸：吸气时胸部打开，呼气时让胸部更好地贴近地面。
4.  建议早晚空腹时练习，每次保持12秒，做3组。

**椅子辅助步骤（适合身体僵硬者）**：
1.  面向椅子跪立，将手掌及手腕以上部位放在椅面上。
2.  双脚打开与髋同宽，脚趾回勾。
3.  呼气时，胸部向下寻找地面，手指向前延长。
4.  始终保持大腿垂直于地面。

这个变体同样要求根基稳定（膝盖与脚趾扎根），上半身尽力延展。坚持练习，背部与胸部曲线会越来越明显。
