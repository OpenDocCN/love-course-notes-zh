# 理财入门课：01：金钱的秘密 💰

在本节课中，我们将要学习金钱的三个核心秘密。理解这些基础但至关重要的视角，将帮助我们建立全新的财富观，从而更有效地掌控和管理自己的财务。

## 金钱是一种度量衡 📏

上一节我们介绍了课程概述，本节中我们来看看审视金钱的第一个视角。金钱本质上是一种度量衡。追溯历史，秦始皇统一了中国称量东西的器具，即度量衡。与此类似，金钱是我们用来换取各种其他东西的度量衡。在货币诞生之前，人们进行以物换物。后来金钱出现，充当了交易的中介，提升了交易效率。

然而，钱作为中介，其兑换标准时时都在变化。例如，过去40年来，一斤鸡蛋的价格从几毛钱涨到了七八块钱。鸡蛋本身的使用价值并未变化，这反映的是钱的兑换标准在动态变化。同时，我们的收入（即劳动力的价格）也在增长。当工资水平和物价水平同比例变化时，度量衡的变化尺度是一致的，我们的购买力不会受太大影响。

所以，第一个概念是：**钱是一种度量衡，本身的使用价值并不高**。纸币既不如餐巾纸柔软，也不能用来写字，其价值基本由它能交换的东西决定。因此，“有钱”是一个相对概念，通常指我们所拥有的财富对应的交换价值。

由此，我们可以得出一个结论：**钱本身没有使用价值，其背后代表的是交换价值**。这种思维意味着，我们关注金钱不应只关注具体数字，而应关注我们用什么交换了金钱，以及希望用金钱去交换什么。这些决策与生活的方方面面紧密相连，这也是“理财就是理生活”的原因。

## 金钱的价值在不断变化 📉

在理解了金钱作为度量衡的角色后，我们来看看第二个视角：钱的交换价值无时无刻不在发生变化。我们需要关注这个变化，因为当我们的财富增长速度小于购买商品服务所需金钱的增长速度时，财富就会缩水。这种现象每时每刻都在发生，它就是**通货膨胀**。

什么是通货膨胀？为什么它影响巨大？我们可以通过一组数据来感受：2003年到2013年间，国内大多数主要城市的房价年均增长10%以上。如果度量衡能在收入和支出两端保持同比例变化，即收入与物价上涨比例相同，购买力不会受影响。但事实并非如此。

以北京房价为例，在房价翻7倍的前提下，人均可支配收入只翻了3倍。这意味着房子价格比劳动力价格增长快一倍多，我们的财富被贬值了。整体上，收入增长的速度赶不上房价增长的速度。这解释了为何许多人选择买房，除了居住需求，也因觉得作为度量衡的钱“不太靠得住”，持有实物资产更踏实，从而形成了“越买越涨，越涨越买”的循环。

历史上通货膨胀极端时，人们会陷入抢购日用品的疯狂状态，例如津巴布韦的恶性通货膨胀。目前我们面临的多是缓慢的、不易察觉的通货膨胀。从定义上说，通货膨胀指的是流通中的钱增多了，但商品和服务的实际数量并未相应增多，导致购买同样东西需要花更多钱。常见原因是货币发行量高于实际流通所需。其他因素如收入增加、企业利润上涨等也会间接加强通胀效应。

除了直观感受，我们如何衡量通货膨胀？以下是核心的衡量指标：

*   **CPI（消费者物价指数）**：根据日常生活必需品的综合价格衡量物价增长。统计局每月发布数据。例如，“CPI同比上涨1.9%”意味着相比一年前，同样金额的钱购买力缩水了1.9%。
*   **其他参考指标**：CPI不包括房价，因此实际感受的贬值速度可能更快。评估房价可参考“70个大中城市住宅销售价格统计数据”，工业方面可参考“PPI（生产价格指数）”。

通货膨胀是对财富悄无声息的打劫。因此，学习理财的重要目标之一就是战胜通货膨胀。战胜它的原理很简单：**让财富增长速度比通货膨胀速度快**。一方面，投资自己以提升收入；另一方面，不能让钱闲置在银行账户中，否则会持续被通胀侵蚀，并错过金钱的另一个重要机会——复利效应。

## 利用复利效应滚雪球 🌊

认识到通货膨胀对财富的侵蚀后，我们自然要寻找对抗它的武器。这就是金钱的第三个秘密：**复利效应**，它能让我们的金钱像滚雪球一样增长。

什么是复利效应？我们先讲一个古印度寓言：一位国王与大臣下棋，大臣获胜后要求用麦粒填满棋盘64格，数量按2的倍数递增。国王欣然答应，事后才发现这是一个天文数字。最后一个格子需放入 **2^63** 粒麦子，按当时生产力，全世界需生产约500年。这个增长遵循的就是复利原则。

所谓复利，就是“利滚利”，从本金中获得的利息会立刻加入本金中继续产生收益。为了更好地理解复利的效果，我们可以分析其三大要素：**本金、回报率和时间**。

请看以下四种投资情况的对比（假设从各起始年初投资至2018年初）：
1.  1998年投资5万元，年复利2%。
2.  1998年投资1万元，年复利2%。
3.  1998年投资5万元，年复利6%。
4.  2003年投资5万元，年复利2%。

通过计算比较可以发现：
*   本金越大，最终积累越多。
*   回报率越高，增长越显著。
*   开始时间越早（时间越久），复利效果越惊人。

复利效应适用于大多数人的财富积累。即使是巴菲特，其财富也是在长期复利积累后，于59岁后才迎来爆发式增长。因此，在复利效应前提下，我们应做到以下三点：

以下是践行复利效应的三个关键行动：
1.  **尽早开始财务规划**：时间是最强大的复利因子。
2.  **搭建投资体系，提高回报率**：通过学习和实践，寻求合理的投资回报。
3.  **积累第一桶金，扩大本金**：本金是复利增长的起点。

对于本金不多的初学者，可以采取“积少成多”的策略，例如每月进行基金定投，持之以恒。同时，**切勿忘记投资自己**，这通常是回报率最高的投资，并且在个人成长、专业能力等方面，复利效应同样适用。

英国自行车队Sky的故事印证了这一点。领队Dave Brailsford通过在所有环节（如睡眠、按摩恢复）追求“1%的改进”，利用微小的、持续的进步（类似复利），最终带领车队多次赢得环法自行车赛冠军。

复利不仅是一个财务概念，更是一种通用的思考方式。专业能力、学习成长、经验积累等都符合复利规律，并会直接作用于我们的财务状况，大大影响财富积累速度。

## 课程总结 🎯

本节课中我们一起学习了关于金钱的三个核心秘密。

首先，**金钱是一种度量衡**，其价值由它能交换的东西决定，而非其本身。其次，**金钱的价值并非一成不变**，由于通货膨胀等原因，其购买力会不断缓慢减少。为了保卫财富，我们必须以战胜通胀为基本目标。

最后，**复利效应**是帮助我们战胜通胀、积累财富的强大工具。影响复利效果的三个主要因素是**本金、时间和回报率**。本金越大、时间越久、回报率越高，复利效应就越明显，财富积累成效越可观。
