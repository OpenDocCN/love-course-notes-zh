# 灵彤彤学院：011：失恋挽回攻略，让前任回心转意：第4课-被自己作没的爱情要怎么挽回

在本节课中，我们将要学习如何挽回因自身不当行为（俗称“作”）而失去的爱情。我们将首先分析几种典型的“作死”行为，然后提供一套具体的挽回策略，帮助你理性地修复关系。

## 概述

很多女性难以承受分手带来的痛苦，会不惜代价去挽回爱情。但大部分挽回是盲目的，她们会做出更多错误行为，结果将想要挽回的男友越推越远。你应该先冷静下来，扪心自问是否真的想挽回这段感情。如果答案是肯定的，就必须了解并避免以下行为，才能增加成功率。

## 第一部分：需要避免的“作死”行为

上一节我们概述了盲目挽回的危害，本节中我们来看看具体有哪些行为会损害关系。以下是五种需要警惕的“作死”行为。

### 1. 盲目任性“作”

很多女人仗着对方的宠爱，只顾表达自我情绪和需求，忽略了对方对自己行为的看法。换位思考一下，如果你的伴侣无论事情大小都盲目任性，一次两次或许可以接受，但次数多了任何人都会感到疲惫，从而选择离开。对方从你屡次不改的行为中，会认定你不会改变，即使还爱着你，也可能选择离开。

有些行为例如连喝水都要对方倒好、吃饭要一口一口喂，这并非证明爱情的必要方式。爱情需要体贴，但过度的要求会变质。另一种情况是，在追求阶段，明明喜欢对方却因高傲而一直吊着对方。你需要反思，对方的离开究竟是因为不爱你，还是你自己的原因造成的。不分场合地任性，例如在公开场合让对方难堪，更是会严重伤害感情。爱情是相互的，不能一边试探一边又指望对方永不离开。

### 2. 决定草率

无论是在说分手、骂人还是争吵时，请务必保持理智。不要在生气时做任何决定或说过激的话。不要天真地认为他会因为你在气头上就理解那些伤害他的话。你的一句狠话可能浇灭他心中最后的希望。

偶尔发泄情绪可以，但请不要说出最伤人的话，否则可能无法补救。这一点不仅适用于挽回爱情，也适用于生活中的许多事。你需要清楚自己决定或说话的后果，以及是否能承受。最好的方式是在那一刻保持冷静，等双方都平静后再讨论争吵的原因。

特别需要注意的是，不要在夜晚做重要决定。心理学家研究表明，晚上是人的情绪波动最大的时候。即使情绪不稳定，也千万不要在此时做决定。等第二天清醒后，再深思是否需要以及如何挽回。

### 3. 暴力情绪发泄

情绪不稳定的人容易用暴力发泄情绪。这类女性可能偏向边缘性人格，情绪起伏极大，主要表现为易怒、暴躁和暴力。暴力行为分为对他人暴力和对自己暴力（如自残、自杀）。

无论是哪种暴力，都不利于挽回爱情。对他人使用暴力会让对方害怕并想逃离你；对自己使用暴力则是不自爱、不顾及他人感受的表现，最终结果仍是让对方想离开。

### 4. 不懂如何沟通

很多感情的悲剧，最初都源于沟通不当或根本不懂沟通。当双方情绪积累到一定程度，可能因一件小事而分手，做出让自己后悔的决定。

无论是在挽回爱情还是婚姻中，沟通永远是第一位的解决方法。在有情绪时，不要带着情绪去沟通。你应该去发现对方让你产生情绪的原因，分析双方需求的差异，思考为何如此生气，以及是否可以通过沟通解决。不要一味只想满足自己的需求、发泄自己的情绪，这种“作死”行为只会导致挽回失败。

当你真正做到有效沟通时，你会发现你们的感情越来越好，对方也更认定你就是对的人，从而不舍得离开你。

### 5. 端着所谓的“自尊心”

这种行为主要分为以下几种情况：
*   **类型一：骄傲心理**。只要对方不对、没主动认错，就不理对方。这无形中给对方巨大压力，让他觉得和你在一起非常累。
*   **类型二：考验对方**。这是最愚蠢的方式之一。考验对方有多爱你，往往是无理取闹的表现。
*   **类型三：死不认错**。自己错了，但碍于面子不主动道歉。这会导致你逐渐失去对方。
*   **类型四：得理不饶人**。本来只是小争吵，但不管对方如何安慰道歉，就是不肯原谅，使劲地“作”。每个人的耐心都有限度，过度发脾气会让对方不愿一直忍受。

第一、三、四种情况与前面提到的“盲目任性作”道理相似，都是只顾自己需求，不懂解决问题。第二类情况也与“盲目任性作”类似，本质上是“作”的一种表现形式。切记不要考验人性，因为在你想要考验的那一刻起，就注定了失败。

## 第二部分：挽回“作没”爱情的具体步骤

以上我们介绍了导致爱情结束的几种“作死”行为。如果你在关系结束后意识到了错误并想挽回，那么需要一定的方法。不要着急，我们接着往下看具体的挽回步骤。

### 第一步：管理好自身形象

能“作”并且能让男人忍受甚至结婚的女人，通常有自己的优点，尤其是外貌上。请保持这个优势，好好打理自己的形象。没有人愿意忍受一个又“作”又邋遢的女人。

### 第二步：进行关键性沟通

和你老公（现在应称前夫）进行一次心平气和的聊天，这次聊天非常关键。

**如果对方已拉黑你或不想联系：**
不必过度担心。你们毕竟在一起过，有共同的亲友。可以找一个中间人帮忙缓和，创造一次见面聊天的机会。对方曾爱过你，否则也不会忍受你之前的各种“作”，所以成功邀约的几率很大。

**见面时的注意事项：**
服装搭配可以好看，但一定不要化浓妆。因为你们刚分开不久，打扮得花枝招展会让男人心里不舒服。请记住，你们曾亲密无间，你美或丑他非常清楚。

**聊天内容的核心思路：**
不要讨论他的变心或错误，他已经受够你了，不会在意是否伤害过你。此时的聊天内容应围绕你的错误展开。

**注意语气和姿态：**
在感情中习惯处于高位或被宠惯的人，此时要收起高高在上的态度，放低姿态，让对方看到你的悔过。但也不要卑微到尘埃里，那样会让你的挽回贬值。

**建议的沟通话术思路：**
你应该谈论自己在这段关系中犯的错，同时巧妙地指出，错误不全在你，对方也有错——错在太过宠你，让你以为这份宠爱会永恒，所以你才放心地“作”。“作”是因为有人爱、有人宠才会有的行为。你可以这样说：“因为你太好，而我习惯了你的好，变成这样真的不是我想要的。” 这种说法能极大满足男人被依赖的心理，减少他对你的反感。

接着，告诉他你不想结束这段婚姻，因为你可能再也遇不到像他那样对你好的人了。但你也理解他因你过去的“作”而想离开的想法。问他是否可以给你一段时间冷静，因为你害怕继续谈下去会情绪崩溃，再次陷入争吵。约定等你冷静下来再联系他，坐下来谈论未来。这为下一次邀约埋下了伏笔。

### 第三步：进行第二次深度沟通

第二次邀约不能太快也不能太慢。太快显得敷衍，太慢则像有意拖延。

**第二次谈话的主要思路：**
告诉对方，你不想结束这段关系，并愿意为修复关系做出努力。毕竟双方都不是年轻人了，能在一起很不容易。请他不用着急答复，可以先观察你的努力和改变，再视情况做决定。

### 第四步：自我提升与改变

在这之后的操作，就和其他挽回操作没有太大区别了：努力提升并改变自己的缺点。以下是提升自我、让对方更爱你的具体建议：

**1. 提升内涵与气质**
女人的气场不仅在于穿衣打扮，更在于内在涵养。涵养是女人最好的化妆品，能让你光彩夺目，吸引成功、财富与赞美。
*   **方法**：多读书（`富有诗书气自华`），多运动保持身材。内外结合，便能成为一个有气质的女人。

**2. 培养良好的情绪管理能力**
聪明的女人懂得掌控情绪。负面情绪让人容易激动、发脾气、产生不切实际的幻想。
*   **方法**：当不良情绪产生时，尝试暂时避开，将注意力和精力投入到其他事情上，以减少情绪的冲击。

**3. 培养人格之美**
真正的美丽不仅在于外表，更在于人格魅力。要学会淡定，淡定是一种智慧，能帮助你在复杂生活中控制情绪，主动追求想要的生活，让你更加自信美丽。
*   **核心**：接受无法改变的条件（如容貌、出身），耕耘好自己的“小园地”，奏响自己生命的乐章。

## 总结

本节课中，我们一起学习了如何挽回因“作”而失去的爱情。我们首先分析了五种典型的“作死”行为：盲目任性、决定草率、暴力发泄、不懂沟通和端着无谓的自尊心。接着，我们介绍了一套四步挽回策略：管理形象、关键沟通、深度交流以及持续的自我提升与改变。

最后需要强调的是，要挽回一段爱情其实不难，难的是如何将它长久地维系下去。好的婚姻需要两个人共同经营，爱的旅途很漫长，唯有相互扶持，才能走到老。希望每位听众都能拥有白头到老的爱情。
