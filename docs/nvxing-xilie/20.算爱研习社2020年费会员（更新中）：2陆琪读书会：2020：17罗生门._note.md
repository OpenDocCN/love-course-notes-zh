# 算爱研习社2020年费会员（更新中）：2：陆琪读书会：17：罗生门

## 概述
在本节课中，我们将要学习日本作家芥川龙之介的短篇小说《竹林中》。这部作品通过多个角色对同一事件的不同叙述，探讨了“真相”的本质。我们将拆解故事中每个角色的视角，理解观察者如何影响对事实的阐述，并思考这对我们看待现实世界有何启示。

---

## 课程内容

### 作者与作品背景
上一节我们介绍了课程概述，本节中我们来看看作者与作品的背景。

芥川龙之介是日本著名作家，出生于明治时期，主要活跃于大正年间，于1927年去世。他是一位倾向于无产主义的作家。

《罗生门》是其著名的短篇小说集。然而，电影《罗生门》并非改编自小说《罗生门》，而是改编自同一作品集中的另一篇短篇小说《竹林中》（也有译作《密林中》）。本节课讲解的核心内容正是《竹林中》这篇小说。

### 故事中的观察者们
上一节我们了解了作品背景，本节中我们来看看故事中出场的各位观察者。小说以一名男子死亡的事件展开，通过不同角色的供词呈现故事。

以下是案件的相关陈述者及其视角：

*   **樵夫**：一名砍柴人，第一个发现尸体。他详细描述了尸体的外观、衣着和现场环境，例如“穿着蓝色衣服”、“只挨了一刀”、“周围染红了”。他代表**好奇的围观者**，只关注表面细节，没有情感卷入。

*   **行脚僧**：一位和尚，声称在前一天见过死者与一名女子同行。他清晰地记得女子“绛红色的衣裙”，但对男子只记得“身上有刀有剑”。他代表**压抑着欲望的观察者**，其观察焦点受到了自身欲望的影响。

*   **捕快**：逮捕了盗贼多襄丸的差役。他因多襄丸是臭名昭著的惯犯（“有名的盗贼”、“好色之徒”），便认定他是此案凶手，呼吁“务必弄个水落石出”。他代表**基于正义进行评判的观察者**，其“正义感”可能先入为主地代替了审判。

*   **老妇人**：死者的岳母。她关心的是自己女儿的下落（“求你……查明我女儿的去向”），对已死的女婿则说“女婿不算”。她代表**利益相关方**，其立场和诉求完全围绕自身利益展开。

### 三位当事人的叙述
上一节我们介绍了外围的观察者，本节中我们进入核心，听听三位直接当事人的说法。他们的叙述充满了矛盾，每个人都从利己的角度重构了事件。

以下是三位当事人的供词概要：

1.  **盗贼多襄丸的供词**：
    *   **核心主张**：承认杀了武士，但强调是公平决斗；声称女子后来不知所踪。
    *   **自我开脱策略**：
        *   **推卸责任**：“不想杀那个男人”，是女子要求“是你死还是我丈夫死”。
        *   **美化行为**：解开了武士的束缚，进行“公平决斗”（“我给他机会决斗”）。
        *   **诡辩**：指责世人“用权力、金钱杀人”，试图模糊自身罪责的独特性。

2.  **女子真砂的忏悔**：
    *   **核心主张**：承认自己杀了丈夫，原因是无法忍受丈夫“鄙视我的冷光”。
    *   **自我开脱策略**：
        *   **归咎于受害者**：将杀人动机归因于丈夫的眼神（“鄙视”、“憎恶”）。
        *   **强调贞洁**：声称本想与丈夫一同殉情（“请把命给我吧，我立刻奉陪”），只是自己未能死成。
        *   **塑造受害者形象**：强调自己被强盗“玷污”，是双重受害者。

3.  **武士亡灵（借巫婆之口）的陈述**：
    *   **核心主张**：声称自己是羞愤自杀。
    *   **自我开脱策略**：
        *   **归咎于妻子**：描绘妻子不仅同意跟随强盗（“请把我带走吧”），还恶毒地要求强盗“杀了他”。
        *   **维护荣誉**：通过描述“自杀”来保全作为武士的尊严。
        *   **宽恕强盗**：“赦免了强盗的罪恶”，将全部罪责指向妻子的背叛。

### 真相与阐述的哲学
上一节我们看到了三个截然不同的故事版本，本节中我们来探讨这背后的核心问题：真相是什么？

故事没有提供唯一的答案。它揭示了一个核心观点：**真相是被阐述的**。我们所知的“事实”，总是经过观察者或当事人基于自身立场、情感、利益过滤和重塑后的产物。

在现实生活中，这种现象普遍存在。例如，公众人物某句有争议的言论，可能被无限放大，用“唯一的坏”来覆盖其所有的好，从而被定性为“恶”。这种评判往往源于我们内心**对“善”与“恶”的情感需求**，而非纯粹的事实本身。

我们执着于追问“为什么”（因）和“结果是什么”（果），就像在感情中不断追问“你为什么不爱我了”或“你是不是有别人了”。然而，对方给出的“因”可能是经过防御机制修饰的，承诺的“果”也未必可靠。

### 方法论：跳出因果，专注行动
上一节我们讨论了真相的不可靠性，本节中我们探讨一个可能的应对方法。

与其陷入对“因”和“果”（即所谓“真相”）的无尽追寻与痛苦中，不如尝试**跳出因果**，专注于 **“怎么做”**。

公式可以表达为：
```
关注点 = 当前处境 + 应对行动
```
即，当一件事发生时，接受它已发生的状态，不再执着于“为什么会这样”或“这会导致什么”，而是思考“基于现状，我该如何行动才能让事情向好的一面发展”。

这要求我们放弃部分对“绝对真相”的执念，因为那常常混合了我们自身的善恶需求与情绪投射。将精力从“追寻”转向“建设”，或许是更务实的态度。

## 总结
本节课中我们一起学习了芥川龙之介的《竹林中》。通过分析樵夫、行脚僧、捕快、老妇人等旁观者，以及盗贼、女子、武士三位当事人的不同叙述，我们认识到：**真相并非客观静止的，而是被每个叙述者基于自身角度重新阐述的结果**。这启示我们，在现实生活中，应警惕被单一叙述或强烈情绪所裹挟，对所谓的“事实”保持审慎。同时，可以尝试将注意力从对因果（真相）的执着追问，转向对当下行动的关注，从而更有效地处理问题。
