## 潜意识吸引：让吸引力在无形中发生

上一节我们介绍了课程的整体框架，本节中我们来看看什么是潜意识吸引。潜意识吸引是指通过一系列不经意的行为，让对方在不知不觉中被你吸引。这是一种深植于人类本能的吸引力。

雪晨和东东是青梅竹马，关系一直很好，但多年来从未产生“化学反应”。直到大三暑假后的一次见面，东东突然对雪晨产生了强烈的心动感。雪晨的外表并无巨大改变，只是做了一些“轻微的调整”，东东就不再把她当“哥们儿”了。相比之下，另一位姑娘小双虽然条件不错，却因为无法做出类似的调整，最终与心仪的男生渐行渐远。

### 什么是性吸引力？

心理学家弗洛伊德认为，人对性的渴望是潜意识层面的，这种本能被称为**性本能**。从自身角度是性本能，从对方角度就是**性吸引力**。

吸引力在恋爱初期的作用，是激发对方的**兴趣**。爱因斯坦说，兴趣是最好的老师。只有产生了兴趣，对方才会愿意花时间了解你的内在。根据马斯洛的需求层次理论，人既有生理需要，也有情感和归属的需要。性吸引力能同时满足对方的动物性需要（生理）和对爱的需要（情感）。

## BHR技巧：引诱-上钩-推开的艺术

上一节我们学习了如何通过潜意识行为建立吸引，本节中我们来看看如何通过有意识的互动技巧，进一步激发对方的情绪投入。这就是BHR技巧。

我有一位非常优秀的朋友，身边追求者众多，但恋情总是无疾而终。经过分析，我发现她的问题在于太过古板，聊天像“挤牙膏”，让男人觉得无趣且累，从而知难而退。我教给她BHR技巧后，情况立刻得到了改善。

### 什么是BHR？

**BHR**是**引诱（Bait）- 上钩（Hook）- 推开（Release）**的缩写，可以简单理解为**推拉**。这是一套连续的动作，能有效引导对方追求你。

这个技巧适用于**暧昧期或恋爱初期**，在稳定期也可酌情使用。其核心是**不暴露过多需求感**。如果在关系初期就暴露强烈需求，对方会觉得你已经“被搞定”，从而不再愿意付出更多心思。然而，喜欢一个人时，我们又难免会流露出需求感。BHR技巧就是在自我暴露与保持神秘之间找到平衡。

从社会心理学看，人际关系发展分为四个阶段：定向、情感探索、情感交流、稳定交往。**BHR属于情感探索阶段最浅层次的自我暴露**。通过适当的自我暴露（引诱）吸引对方（上钩），然后适时后撤（推开），让对方猜不透你的心思。男人是天生的猎手，越是猜不透，就越想探索，从而对你投入更多时间、精力和情感。

### 如何建立运用BHR的框架？

要想运用好BHR，首先必须建立自己的**强框架**。框架就是你传递给对方的核心信息，是你的原则、底线和气场。

*   **弱框架示例**：给对方发完微信后，就焦灼地等待回复，不断猜测对方心思，情绪完全被对方牵动。
*   **强框架示例**：发完微信后，继续自己的生活。你的生活丰富多彩，无需为他是否回复而焦虑。他爱回不回，你只根据自己的心情和话题有趣程度来决定是否回复。这种“不在乎”的姿态，反而能最大程度激发男人的征服欲。

强框架是运用BHR的前提。BHR的本质就是制造**情趣**和**情绪波动**。让对方和你在一起时，心跳加速，情绪起伏，从而产生爱的感觉。

从心理学角度，当你做出某个动作时，对方会根据经验对你的下一步行为产生预期。如果你的行为符合预期，他不会产生波动；如果你的行为出乎意料（比如引诱后推开），他就会产生好奇、疑惑等情绪波动，从而开始关注你、研究你，在不知不觉中进行了情感投资。

### BHR实战案例

以下是几个BHR的实际应用例子，帮助大家更好地理解：

1.  **约会中的贴近与抽离**：

    *   **推开**：（停顿一下后）“幸好我的室友回来了，充了电卡。”**中间的停顿至关重要**，没有停顿就没有上钩的过程。
3.  **言语上的调侃**：

4.  **喂食的挑逗**：
    *   **引诱**：用筷子夹起食物，假装要喂到他嘴里。
    *   **上钩**：他张嘴准备吃。
    *   **推开**：突然把手撤回来，把食物放进自己嘴里，并调皮地看着他。

方式A通过推开制造了不确定性，更能激发对方的征服欲和兴趣。方式B则更直接热情。在恋爱初期，建议使用方式A。

### 自我暴露的层次

BHR属于浅层次的自我暴露。随着关系深入，自我暴露的程度会加深，大致分为四个层次：

1.  **第一层：兴趣与经历**（情感探索阶段）。陈述事实，如爱好、经历。BHR就在此层运作。
2.  **第二层：态度与看法**（情感交流阶段）。开始透露对事物的观点和态度，涉及部分三观。例如，讨论社会新闻时表达自己的看法，并说：“所以有个喜欢的人在身边保护我该多好。”（需求感开始明显）
3.  **第三层：人际关系与自我概念**。倾诉私人感受、脆弱和期望。例如，向对方诉说职场委屈和孤独：“每当这个时候我就特别想给你打个电话...只要你在我身边，我就特别安心。”（需求感强烈）
4.  **第四层：个人隐私**（稳定交往阶段）。透露最深层的隐私，如原生家庭创伤、最深层的恐惧等。

了解这些层次，有助于你在恋爱不同阶段，更好地掌控自我暴露的深度和节奏，平衡需求感的表达。
