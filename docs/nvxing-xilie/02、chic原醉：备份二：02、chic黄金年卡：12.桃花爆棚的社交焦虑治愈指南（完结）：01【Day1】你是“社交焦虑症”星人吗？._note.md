# 社交焦虑治愈指南：01：你是“社交焦虑症”星人吗？

在本节课中，我们将从认知层面了解社交焦虑。我们将对自己进行一次“X光式”的自我扫描，帮助你识别自己可能属于哪一种社交焦虑类型，并深入探讨焦虑背后的核心原因。

## 概述：当代年轻人的社交困境

说到社交恐惧，这是一种什么样的体验？在芬兰有一个著名的笑话：一个内向的芬兰人和你说话时，会看着自己的鞋子；而一个外向的芬兰人和你说话时，他会看着你的鞋子。如果说芬兰是对社交感到羞涩的国度，那么日本对社交的感觉则更倾向于冷漠。在日本，大多数人小心翼翼地保持着距离，表面的彬彬有礼，实际上是对深度交往的疏离。

近年来，不少中国年轻人也开始自称“社恐”。各种颓丧、佛系的情绪引领着社交网络上的主流心态，催生了一批批宅男宅女。他们形容自己是“间歇性渴望社交，持续性自我封闭”。难道我们也得了社交恐惧症吗？

其实，不同于芬兰的羞涩和日本的冷漠，中国年轻人所说的“社恐”更多带有一种**焦虑**的意味。例如：害怕应酬，逃避各种饭局；害怕春节时与亲戚尬聊；害怕在公共场合说话，担心冷场没面子；害怕在路上看见熟人，纠结是否要打招呼。这些“中国式社恐”并非医学上的严肃诊断，更多是消费文化下的一个自嘲标签。但它真实地揭示了当下一种焦灼的精神困境：**人们渴望社交，同时也害怕社交**。这种不健康的社交状态，在心理学上被称为 **社交焦虑**。

根据调查显示，70%拥有社交焦虑的人，长期下来会不再渴望建立社交关系和亲密关系，形成自我孤立。那么，我们应该如何改变现状，重新建立社交自信呢？让我们先从了解社交焦虑的类型开始。

## 中国式社交焦虑的三种典型类型

以下是三种最常见的社交焦虑表现类型，你可以对照看看自己属于哪一种。

**第一类：假面式合群**

这一类社恐大多是工作中长期压抑真实情绪的都市白领。他们通常迫于职业要求，拥有许多社交活动。表面上看，他们的社交生活十分充实，给人八面玲珑、左右逢源的印象，但内心十分恐惧这种虚假的社交，极其渴望独处。最经典的习惯是下班后喜欢躲在车厢里休息，在此期间拒绝任何打扰，享受一个人的时光。

在日剧《无法成为野兽的我们》中，新垣结衣饰演的深海晶就是这样一个“假面式合群”的社恐女孩。每天踏进公司的那一刻，她就自动开启“职业假笑”模式，迎合上司，顺从同事，忙碌于各种商务应酬，最终成为了一个“假面人”。唯一可以做回真我的地方只有一家深夜酒馆。

对于这类社恐，他们焦虑的核心是：**在社交中无法做真实的自己**。英国心理学家温尼克特指出，如果真实自我和虚假自我之间的矛盾过大，自我认同感就会十分脆弱。改变这种状态，必须学会在社交中平衡“真我”和“假我”。

**第二类：肥宅式孤僻**

这一类社恐大多是缺乏社交技能的宅男宅女。他们平日能不出门就不出门，线上可以放飞自我，但线下与人面对面交流时，却不敢对视、脸红害羞、语无伦次。如此恶性循环，以至于越来越畏惧社交。

在日剧《最终幻想女孩》中，松冈茉优饰演的良香就是一个不折不扣的社恐宅女。表面上她性格孤僻，沉浸在自己的幻想世界里，实际上却十分渴望在现实生活中与人真诚交流。她鼓起勇气向暗恋多年的男神表白，但表达方式十分笨拙，最终发现对方连她的名字也没记住，感到十分挫败。

对于这类社恐，她们的焦虑源于**社交技能的缺乏**。主观上受“懒宅”心理影响，幻想多而实践少；但更多是客观原因——一直以来接受的教育并未教会他们如何正确处理人际关系。社交教育的缺失，让他们不知道面对新同事该说什么、与朋友的恋人该如何相处、面对合作伙伴应保持怎样的沟通频率等。改变这种状态，需要系统地学习社交技巧。

**第三类：佛系式交友**

如今我们大部分人的社交状态都受这种“佛系”情绪影响。网上有一个笑话：有人问，“你们这么佛系是怎么交朋友的？”回答说，“我们佛系的人一般不主动交朋友，我们都等有缘人来主动领养我们。”

说白了，这其实是心理学上的一种**自我封闭的“鸵鸟心理”**。因为在社交中遇到过太多困难，所以干脆破罐破摔，拒绝主动社交。在日剧《逃避虽可耻但有用》中，星野源饰演的津崎平匡就是这样一个看起来看破红尘的佛系IT男。他单身35年的原因，就是打从心眼里认为自己生活中不需要异性，抗拒认识朋友。对于这类社恐，更需要从根源上摆正对社交的心态，换回对社交的信心。

## 社交焦虑的三大核心来源

了解完三种典型的社恐类型后，我们进一步分析：当我们在说社交焦虑时，我们到底在焦虑什么？

**1. 对控制感丧失的焦虑**

演员高圆圆曾在微电影《我，39》中自白，她有很长一段时间拥有人际障碍，不想接触陌生人，最害怕的事情就是“不能掌控自己”。相比社交，独自在家打游戏、看电影等单人活动，可控制的部分更多。但面对陌生人或环境，即便是和熟人聚会，也无法预测对方会说什么、会对自己作何反应。更别提社交中可能遇到的尴尬窘况：如果话不投机怎么办？如果冷场怎么办？如果产生矛盾怎么办？这一系列不能完全由自己控制的问题，便是焦虑的来源。

**2. 对社交模式变化的焦虑**

这与每个人的成长经历有关。在学生时代，与同学的交往模式是结伴学习、课后玩耍，社交是一种**相互支持和陪伴**的模式。到了大学，每个人课程、作息、爱好不同，先前的“支持和陪伴”模式不再完全适用，更适合的是以**精神交流**为主的模式，例如与社团成员聊兴趣，与求职同伴聊经验。等到进入社会，模式又转变为基于**合作**，与同事共同完成工作目标。

没有一种社交模式可以贯穿始终。面对不同的人、不同的对象，我们都需要采用适宜的方式。有的人在社交模式转变中感到不适应和挫败，正是因为以往熟悉的方法不再起作用。因此，掌握多元化的社交技能，学会切换人际交往模式尤为重要。

**3. 对未知负面评价的焦虑**

如果说前两种还是相对个性化的焦虑来源，那么**担心自己被评价**可以说是更普遍的压力源。例如：你担心别人说你不大方，所以你抢着买单；你担心别人说你无趣，所以你努力讲段子；你担心别人觉得你负能量，所以你不敢吐苦水。长期处在这种试图表现出不属于自己特质的状态下，任何人都会感到焦虑和疲惫。

## 社交焦虑带来的负面影响

不管是哪种原因导致的社交焦虑，这种不健康的社交状态都会对日常生活造成极大困扰。

一方面，它会直接削弱你的社交关系和情感连接。例如，在韩剧《未生》中，主人公张格莱因为社恐心理，不懂团队协作、羞于向前辈请教，导致在职场中屡屡受挫。在美剧《生活大爆炸》中，天文学家拉杰什也因为严重的社交焦虑，无法正常与异性交往。他第一次见到女生时手抖得说不出话，最后让女生误以为他对自己反感。

另一方面，从人体生物学上来说，社交焦虑不利于身心健康的发展。国际顶级学术期刊《细胞》曾刊登一项研究显示，社交焦虑状态会慢慢改变大脑分泌的神经化学物质，长期下来会让你的思维模式倾向于**负面的认知**。

## 总结与课程预告

总结以上内容，相信大家都明白，其实每个人的内心都是一个矛盾体：一方面渴望融入群体，一方面希望和他人保持边界感。天平的两端如果失衡，就会导致社交焦虑的出现，不利于我们各方面的成长。因此，我们应该积极调整这种消极的社交状态，而不是任由其“佛系”发展或自暴自弃。

现在让我们回顾一下本节课的重点：
1.  我们从宏观上分析了当代年轻人的社交状态。所谓的“中国式社恐”，并非医学上的严肃判断，但它揭示了如今一种普遍的精神困境：人们渴望社交，同时也害怕社交。
2.  我们总结了三种最典型的社交焦虑类型：**假面式合群**、**肥宅式孤僻**以及**佛系式交友**。
3.  造成社交焦虑的主要原因有三点：**控制感的丧失**、**社交模式的多变**，以及**对未知评价的恐惧**。
4.  我们得出结论：社交焦虑是一种不健康的自我孤立状态，它不仅会影响思维模式的发展，还会阻碍职场人脉和亲密关系的建立。
