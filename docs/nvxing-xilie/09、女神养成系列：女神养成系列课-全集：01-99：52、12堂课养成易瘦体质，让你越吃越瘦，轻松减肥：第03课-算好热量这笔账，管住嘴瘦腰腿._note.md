# 女神养成系列：03：算好热量这笔账，管住嘴瘦腰腿

在本节课中，我们将学习如何将抽象的卡路里数字，转化为具体可执行的食物份量。通过理解中国居民膳食指南的食物分类，你将学会如何搭配一日三餐，轻松管理热量摄入，实现健康减脂。

上一节我们介绍了如何制定科学的减肥计划，明确了每日应摄入的卡路里目标。本节中，我们来看看如何将这些卡路里目标，落实到具体的食物选择和份量控制上。

## 减脂人群的食物结构比例

减脂人群的三大营养素摄入比例要求为 **5:3:2**，即 **碳水化合物:蛋白质:脂肪**。脂肪和蛋白质的比例按 **2.5:2.5** 也是可以的。这与正常健康人群（碳水化合物占60%-70%）不同。

具体分配到一日三餐，减脂人群的要求是早餐、午餐、晚餐按 **5:3:2** 或 **4:4:2** 的比例摄入。早餐应尽量吃好吃饱，晚餐需减少摄入但不能不吃，以避免因晚睡而产生夜间进食的冲动。

## 中国居民膳食指南与食物份概念

具体到每一样食物，我们需要通过计算卡路里以及每种食物的碳水化合物、蛋白质和脂肪含量来确定摄入量。但这过程繁琐。因此，我们引入“食物份”的概念来简化测量。你只需知道每天每类食物应摄入多少“份”即可。

首先，我们依据《中国居民膳食指南（2016）》的膳食宝塔，将日常食物分为以下九类。

以下是九类食物的详细介绍及其对应的“份”的概念。

### 1. 油脂类
膳食宝塔要求每日食盐摄入量小于6克，约一个啤酒瓶盖铺平的量。减脂人群要求减半，每日不超过3克（低血压人群除外）。

油脂每日推荐摄入25-30克。一个小的茶匙油（约10克）定义为 **一份油脂**。健康人群每日摄入2-3份，减脂人群每日不超过2份。推荐交替使用橄榄油、大豆油、芥花籽油、菜籽油。

### 2. 奶制品类
膳食宝塔要求每日摄入300克奶制品。一盒普通牛奶（约250ml）算 **一份**，一小盒酸奶（100ml）算 **0.5份**。

减脂人群要求摄入脱脂或部分脱脂奶，每日摄入2-3份奶制品。乳糖不耐受人群可采取少量多次、搭配主食、饮用酸奶或舒化奶等方式摄入。

### 3. 豆仁类
此类包括大豆制品和坚果，是优质蛋白质和脂肪的来源。减脂人群每日必须摄入。

一份坚果对应量为：核桃1.5个、碧根果1.5个、花生8粒、开心果15颗、杏仁8粒、腰果5粒。黑芝麻、葵花籽、南瓜子等一茶匙为一份。推荐选择开心果和杏仁。

一份豆制品对应量为：一大杯豆浆（300ml）、一掌腐竹（约4-5条）、半块北方豆腐或3/4块南方豆腐。减脂人群应喝无糖豆浆，避免油炸豆制品。

减脂人群每日需摄入1-2份豆仁类，建议一份豆制品加一份坚果。

### 4. 肉禽蛋类
减肥必须摄入肉类以获取优质蛋白。选择顺序为：鱼虾海鲜类 > 鸡鸭鹅禽类 > 牛羊肉类 > 猪肉（减脂期间避免）。

一份肉禽蛋类对应量为：半手掌大小的鱼或瘦牛肉、8只正常大小的虾、半块鸡胸肉（约50-60克）、10个贝壳类海鲜、半只去皮的鸡腿、1个鸡蛋、半手掌三文鱼。

避免食用香肠、火腿、腊肉等烟熏腌制肉类。减脂人群每日摄入量需根据总卡路里调整，可能多于健康人群。鸡蛋蛋白每日可摄入3-4个，蛋黄每日不超过1个。烹饪前需去除肉类可见脂肪。

### 5. 蔬菜类
蔬菜是“绿灯食物”，减脂期间可不限量多吃。减脂人群每日至少应摄入5-6份蔬菜。

一份蔬菜对应量为：一个中等大小的西红柿/黄瓜/茄子/白萝卜（需煮熟），或半根胡萝卜。生菜则是一手掌轻轻捧起的量。煮熟后的蔬菜，小半碗为一份。

建议每日摄入至少5种颜色的蔬菜（绿、红、黄、白、黑），以确保营养均衡。上班族可每日补充一粒复合维生素片。

根茎类蔬菜（如土豆、红薯、藕）归类为主食（谷薯类），是优质主食来源。

### 6. 水果类
水果属于“黄灯食物”，需限量食用。一份水果的热量与一份主食相当。

一份水果对应量为：一个中小号苹果/梨/桃（拳头大小）、半根香蕉、1个猕猴桃、20颗小樱桃、10颗草莓、半个橙子、两瓣柚子、半个小木瓜、1个小芒果、1/3个牛油果（每日不超过半个）。

水果建议作为加餐，在两餐之间食用，饭前1小时吃最佳。应选择新鲜、当季、种类多样的水果，避免果汁、水果罐头、果脯。

### 7. 谷薯类（主食）
这是日常所说的主食。推荐用粗粮代替精细米面。

一份谷薯类对应量为：小半碗米饭/面条、半个馒头、两大勺干燕麦、一碗杂粮粥、一片全麦面包、一根中等玉米、一个拳头大小的红薯/土豆/藕。

建议制作杂粮饭、杂粮粥或混合面食，搭配多种谷物和豆类。

### 8. 饮用水
人体感到口渴时已处于缺水状态。平时应养成主动喝水的习惯。

以下情况更需多喝水：感冒发烧时、便秘时（应大口快速吞咽）、减肥期间（每日至少2-3升）、呕吐腹泻时。

### 9. 零食类
此类为加工食品，非人体必需，减脂期间严禁摄入。

一份零食（约150大卡）对应量为：1/8个披萨、10片薯片、2个小球冰淇淋、1个甜筒、半份巧克力圣代、1块香辣鸡翅、小半碗烧仙草、3块鸭脖、5片饼干。酒类、蛋糕、巧克力等也应避免。

## 不同热量方案的饮食举例

以下是针对不同每日总热量需求的四套食物份数方案（1000, 1200, 1500, 1800大卡）。你需要根据自己计算出的每日摄入目标来对应选择。

我们以1200大卡方案为例，展示一日餐单如何搭配（按早餐:午餐:晚餐=5:3:2的比例）：

*   **早餐（5份比例）**：包含谷薯类1份、奶类1份、肉禽蛋类1份、蔬菜类1份。例如：两大勺燕麦片（谷薯类）+ 一盒脱脂牛奶（奶类）+ 1个鸡蛋（肉蛋类）+ 1个西红柿（蔬菜类）。
*   **上午加餐**：开心果15颗（坚果1份） + 水。
*   **午餐（3份比例）**：包含谷薯类1份、豆仁类1份、肉禽蛋类2份、蔬菜类2份。例如：1根玉米（谷薯类）+ 半块豆腐（豆仁类）+ 8只水煮虾（肉禽蛋类）+ 两份绿色蔬菜。
*   **下午加餐**：一份水果（如1个苹果）或一份酸奶。
*   **晚餐（2份比例）**：包含谷薯类1份、蔬菜类2份、肉禽蛋类1份。

掌握“食物份”概念后，同类食物中的不同种类可以互换，让你的饮食更加丰富。

## 减脂期饮食技巧总结

本节课我们一起学习了如何将热量目标转化为具体的食物份量。最后，总结五个能立刻见效的饮食技巧：

1.  **主食替换**：用粗粮代替精细米面。
2.  **少食多餐**：每天吃5-6餐，随身携带健康加餐（如黄瓜、西红柿、酸奶）。
3.  **保证必需摄入**：确保每日摄入奶类、豆类、坚果和足量蔬菜。
4.  **戒除零食饮料**：不吃零食，不喝含糖饮料和酒。
5.  **控制晚餐时间**：晚上8点后不再进食。

只要能做到以上几点，你就能看到体重的变化。
