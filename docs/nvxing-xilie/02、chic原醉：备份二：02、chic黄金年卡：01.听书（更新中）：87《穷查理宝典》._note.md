# 《穷查理宝典》解读：02：查理·芒格的智慧与投资哲学

在本节课中，我们将一起学习《穷查理宝典》这本书，深入探索查理·芒格的生平、其核心的投资策略与生活智慧，以及他所倡导的多元思维模型。通过了解这位低调投资大师的思想，我们可以汲取宝贵的普世智慧。

## 02：01：查理·芒格如何从普通男孩成长为金融大亨

上一节我们概述了本书的价值，本节中我们来看看查理·芒格的成长轨迹，了解他非凡人生的起点。

1924年1月1日，查理·芒格出生在美国奥马哈市。巧合的是，沃伦·巴菲特也出生在这座城市。早年，芒格曾在巴菲特祖父的杂货店打工，这段艰苦的经历让他和后来的巴菲特都受益匪浅。

打工和上学之余，芒格最喜爱的事情就是阅读。他深受本杰明·富兰克林的影响，后者以笔名“穷查理”出版的《穷查理年鉴》，正是本书书名的灵感来源。富兰克林所推崇的节俭、负责、勤劳等美德，被芒格终身践行。

对科学的兴趣也早早在他心中萌芽。通过翻阅邻居医生的医学书刊，芒格对自然科学产生了浓厚兴趣。在密歇根大学，他选择了数学专业，并迷上了物理学的逻辑与广阔。二战爆发后，他应征入伍，期间学习了工程学、热力学等多门学科。

退伍后，芒格进入哈佛法学院，并以优异成绩毕业。广泛的跨学科学习为他日后形成独特的思维模型奠定了基础。他坚信阅读的力量，曾说过：“我这辈子遇到的聪明人，没有不每天阅读的。”

毕业后，芒格成为了一名律师。然而，家庭的增长让他感到经济压力，律师的按小时计费收入已无法让他满意。于是，他开始涉足投资，并在1960年代初的房地产投资中获得了第一桶金。

1959年，因父亲去世回到奥马哈的芒格，在一次晚宴上遇到了29岁的沃伦·巴菲特。两人一见如故，在商业、金融和历史等领域无所不谈，建立了深厚的信任关系。这种关系并非简单的合作，更像是两位高手相互切磋，共同成长。

在巴菲特的影响下，芒格于1965年离开律师事务所，开始专职投资。他与合伙人创立的公司取得了优异的投资记录，最终公司被伯克希尔·哈撒韦收购，芒格与巴菲特正式联手，共同创造了投资史上的奇迹。

## 02：02：查理·芒格的投资策略与生活智慧

了解了芒格的成长历程后，本节中我们将重点剖析他取得惊人投资业绩的核心策略，以及他为人处世的生活智慧。

查理·芒格的投资成功，并非仅仅源于博览群书，更在于他严格的投资纪律与清晰的评估框架。

以下是他的核心投资策略：

*   **聚焦能力圈**：他首先进行全面的筛选，将投资局限在“简单且好理解”的项目内。他常说：“我只想知道我将来会死在什么地方，这样我就可以永远不去那里了。”这体现了坚守能力圈的重要性。
*   **评估企业价值**：他将潜在投资分为三类：可以投资、不能投资、太难理解。对于“可以投资”的项目，他会全面评估企业内外部因素及行业状况，计算其内在价值。他尤其强调要购买“股价公道的伟大企业”，而不仅仅是“股价超低的普通企业”。
*   **耐心等待，集中下注**：他耐心等待最佳的买入时机，一旦确定，便会大手笔集中投资，并长期持有。他认为，好的投资机会很少，所以一旦出现就要牢牢把握。他的投资风格是集中投资少数几家优秀公司。

除了投资策略，芒格的生活智慧同样发人深省。

以下是他的几条核心生活原则：

*   **做事公平周到**：他曾教育孩子，借别人的车，归还时一定要加满油。这种处处为他人着想的品格，体现在他生活的方方面面。
*   **运用逆向思维**：他教导孩子遇事要先用逆向思维思考，即首先想清楚要避免什么。在投资上，就是先思考哪些公司不能投。
*   **坚守诚实正直**：在他的职业生涯中，当面临诚实与利益的抉择时，他总是毫不犹豫地选择诚实。他认为，做诚实的事能为个人品格和事业带来长远好处。
*   **保持耐心与专注**：他相信复利的力量，也相信每日进步的积累。他说：“每天起床的时候，争取变得比以前更聪明一点。”这种长期主义的耐心，是投资者最宝贵的品质之一。

## 02：03：查理·芒格的多元思维模型

掌握了芒格具体的投资与生活方法后，本节我们来看看他赖以做出决策的底层思维框架——多元思维模型。

如今已年过九旬的查理·芒格提倡学习各个学科的核心思想，并将它们融会贯通，形成一个用于分析和解决问题的“多元思维模型”。这个模型是他应对商业和投资中复杂问题的利器。

多元思维模型是一个收集、处理信息并依此行动的思维框架。它完美地糅合了历史学、心理学、数学、工程学、生物学、物理学、统计学等传统学科的分析工具与方法。

芒格认为，掌握八九十个模型就差不多能让你拥有普世智慧，而其中几个尤为重要。

以下是几个关键的思维模型举例：

*   **数学模型**：如**复利模型** `A = P(1 + r/n)^(nt)`。芒格认为不需要太复杂的数学，掌握复利、排列组合等基础概念足矣。
*   **心理学模型**：他特别重视“误判心理学”，总结了24种常见的心理倾向，如：
    *   **奖励/惩罚超级反应倾向**：激励机制能极大改变行为。
    *   **避免不一致性倾向**：人们倾向于维持现状，不愿改变既有习惯或认知。
    *   **被剥夺超级反应倾向**：失去造成的痛苦远大于等量获得带来的快乐。
*   **微观经济学模型**：如**规模优势**概念。生产或销售规模越大，单位成本可能越低，竞争优势越明显。这种优势可能来自信息（如广告）、社会认同或垄断地位。
*   **生物学思维**：用生态竞争的观点看待商业世界，像沃尔玛这样通过竞争在特定生态位（如小城镇）建立起优势的企业，就具有强大的生命力。

芒格将各个学科的知识产生链接，形成一个不断更新完善的系统。这个系统能产生类似“核爆炸”的合力，他称之为“lollapalooza效应”。掌握多元思维模型，就是为了避免“手里拿着锤子，看什么都像钉子”的单一思维局限，从而更全面、更理性地应对现实世界的挑战。

## 总结

本节课中，我们一起学习了查理·芒格的智慧体系。

首先，我们回顾了他如何从一个热爱阅读的奥马哈男孩，通过跨学科学习、法律实践，最终与巴菲特相遇并携手成为投资巨擘的历程。

其次，我们深入探讨了他严谨的投资策略，包括聚焦能力圈、深度评估企业价值、耐心等待并集中下注，以及他倡导的公平周到、诚实正直、逆向思维等生活智慧。

最后，我们解析了他强大的思想武器——多元思维模型。这个模型强调跨学科学习，将数学、心理学、经济学、生物学等学科的核心原理融会贯通，用以做出更理性的决策。

查理·芒格的成功，不仅是投资哲学的成功，更是其小心谋划、专注行事的生活方式的副产品。他的思想、品格与思维方式，为我们提供了一幅通往理性与智慧的宝贵地图。
