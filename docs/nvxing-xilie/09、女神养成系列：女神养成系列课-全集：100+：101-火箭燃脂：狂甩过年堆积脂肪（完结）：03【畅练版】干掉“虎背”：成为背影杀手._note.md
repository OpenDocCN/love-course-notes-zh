# 女神养成系列：03：畅练版-干掉“虎背”

在本节课中，我们将学习一套针对背部、臀部和核心肌群的训练动作，旨在改善体态，塑造紧致的背部线条，告别“虎背熊腰”。这套动作结合了动态与静态练习，能有效激活相关肌群。

## 🎯 动作详解

上一节我们介绍了训练目标，本节中我们来看看具体的动作流程。以下是分解动作说明。

**1. 起始与手臂动作**
*   身体俯卧，双臂向前伸直。
*   吸气准备，呼气时同时向上抬起双臂和双腿，手臂向两侧打开后收回。
*   重复此动作，过程中保持下巴微收，眼睛看向地板。

**2. 臀腿激活**
*   双脚依次向上抬起，再依次落地。
*   随后双手向身体两侧打开，像飞机摆动机翼一样上下摆动双臂。

**3. 核心与臀部强化**
*   双脚同时向上抬起，膝盖伸直。
*   夹紧臀部，转身变为手臂支撑姿势。
*   单侧髋部向上顶起，同时保持臀部收紧，腹部绷紧。

**4. 臀桥及其变式**
*   坐立后仰卧，进行臀桥练习：双脚踩地，臀部向上顶起，使整个后背离开垫子。
*   脚跟蹬住地板，夹紧臀部。
*   在臀桥顶端，进行脚尖交替点地，双腿保持并拢。

**5. 背部伸展与划圈**
*   转身回到俯卧姿势。
*   双臂侧平举，保持胸口抬离地板。
*   双臂进行小幅度的画圆动作，下巴保持微收。

**6. 动态拍打与超人式**
*   身体放松，双脚快速上下拍打，模仿游泳打腿动作。
*   进行“超人式”：双臂向前伸直，与双腿同时上下摆动两次。
*   手摸臀部，尝试将上背部再抬高一点，全程夹紧臀部并绷紧腹部。

**7. 支撑与伸展**
*   手脚一起抬起，转身变为单手支撑。
*   非支撑侧的脚向前伸展，同侧髋部向上顶起，身体与地面保持平行。

**8. 坐姿与仰卧收尾**
*   坐立，双腿伸直，进行上下交替摆腿。
*   最后仰卧，脚跟立起，再次进行臀桥，重点感受臀部收紧和大腿后侧的燃烧感。
*   可进行双腿伸直的连续顶髋动作，过程中绷紧下腹部。

本节课中我们一起学习了针对背部、臀部和核心的系列训练动作。坚持练习可以有效改善上半身体态，强化后背肌群，让你逐渐拥有挺拔迷人的“背影杀手”气质。
