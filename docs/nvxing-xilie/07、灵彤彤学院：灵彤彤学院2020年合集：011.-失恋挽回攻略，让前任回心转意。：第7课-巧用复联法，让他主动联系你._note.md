# 失恋挽回攻略：011：第7课-巧用复联法，让他主动联系你

## 概述
在本节课中，我们将学习如何在分手后，通过巧妙的复联方法，让对方主动联系你。我们将探讨如何避免引起反感，并逐步重建联系，为挽回感情奠定基础。

---

上一节我们讨论了分手后的心态调整，本节中我们来看看如何迈出重新联系的第一步。分手后，很多人会后悔并想挽回对方，但关键在于如何在不引起对方反感的前提下，让对方回心转意。

很多人会压抑自己主动联系的冲动，这种感觉令人无助。尤其在深夜情绪脆弱时，会特别想联系对方，但往往因为害怕而放弃。虽然强忍着不联系，但内心的想念是真实的。每次想发送问候时，都会想起最后决绝的对话，于是作罢。反复编辑又删除消息的滋味很难熬。

我们害怕联系对方，主要是害怕被拒绝，害怕显得自己价值低。但需要明白，难熬的不止你一个人。爱情是两个人的事，在你伤心的同时，对方可能也在默默伤心。双方都在等待对方先联系，却都不敢迈出第一步。

作为一个成年人，应该清楚自己想要什么。既然念念不忘，就不应坐以待毙，要愿意迈出主动联系的第一步。

首先要记住一句话：稳中求胜，让你们的联系变得有意义。

那么，如何做到稳中求胜呢？这里要介绍一个秘诀：**镜子法则**。

每个人都有照镜子的经历。当你精神焕发时，镜子会展示精神饱满的你；当你萎靡不振时，镜子只会展现颓废的你。在联系对方的过程中，我们可以应用这个法则。

以下是应用镜子法则的两个核心要点：

**1. 聊天状态的时间间隔同步**
在微信或QQ聊天时，如果对方回复很慢，你绝不能秒回。即使他的回复让你很开心，也要忍住。秒回会给对方造成压力，让他觉得你一直在等待。你应该使用镜子法则，与对方保持同步。例如，对方隔10分钟回复，你也应隔10分钟或更久回复。这能减少对方的压迫感，也让你能冷静、理性地回复。

**2. 感情状态同步**
如果你的热情总是换来对方的冷漠，那么请收起热情，耐心等待。如何判断对方的回应是热情还是冷漠呢？如果对方回复字数少、内容简短客观、缺乏个人感受，就属于冷漠回复。反之，如果回复内容较长、间隔时间短，则属于热情回复。

例如，你告诉对方：“我今天在学校西门的烧烤店吃了一份烧烤，老板弄的烤鱿鱼比以前好吃多了。”如果对方热情地回应：“早就变得很好吃了呀！我上次和同学去吃过，真心很赞。”这说明对方加入了自我感受，属于热情回应，为后续交流做了良好铺垫。如果对方只是冷淡地回复：“嗯嗯，是的。”这表明对方除了礼貌性回应外，不愿多说，态度冷漠。此时，你可以试探性地进一步确认，但如果对方依旧冷漠，你就应回归冷淡，与对方保持同步。

介绍完联系中的重要秘诀后，接下来我们具体讲解如何逐步进行感情回温，重新建立联系。

**第一步：分手后进行适当冷冻**
任何分手都伴随着高矛盾点，因此挽回前进行冷冻是必要的。分手后纠缠不休甚至以死相逼，只会让对方更厌恶你，贬低你。你的行为在别人眼里只是作践自己，最终感动不了对方，反而可能让对方彻底断绝联系。提出分手的一方，此时对你多是负面印象。冷冻期也是让自己冷静反省恋爱期间所犯错误的最好时期。例如，是否在对方需要你时没有给予回应反而数落对方？是否试图强迫改造对方？是否在对方需要安静时喋喋不休？许多错误可能已被对方默默记下，最终导致分手。请先好好反省，不要急于纠缠对方。

**第二步：在冷冻期内努力改变自己**
当你解释或哀求对方回到你身边，却只得到不耐烦的回应时，就应该选择冷冻这段关系。注意，是冷冻情感，而非完全切断联系。停止你对他的情感、金钱和时间投入，但如果他不反感，可以保持联系。在冷冻期间，你应该做两件事：

*   **提升外在形象**：外在改变是最快捷有效的改变负面形象的方法。无论是在职场还是情场，外在形象都非常关键。如果你是一个强势的女人，而对方已厌倦你的强势，可以尝试更温柔的打扮，比如穿暖色系连衣裙，搭配同色系高跟鞋，打造浪漫的发型和淡妆。当你以新形象出现时，可能会让对方感到惊艳。如果觉得自己不够时尚，可以多阅读时尚杂志，慢慢找到适合自己的风格。
*   **提升内在修养**：在情感中，内在提升与外在提升同等重要。对一个人的爱慕往往始于颜值，忠于人品。踏踏实实提升内在，才能让魅力经久不衰。作为女性，应具备一定的居家能力，能处理好家庭琐事，体谅另一半，懂得打扮自己，时尚而不风骚。同时，也需要有自己的朋友圈，让生活丰富多彩。

**第三步：通过微信尝试主动联系**
微信是分手后常见的挽回途径。在联系时，需要注意以下小技巧：

*   **重建联系**：如果分手时删除了微信，可以尝试重新添加。不要使用小号，那样显得偷偷摸摸。可以先将微信绑定的手机号解绑再重新绑定，如果对方通讯录仍有你的号码，系统可能会自动提示对方添加你。在此之前，你应该更换头像并建设好朋友圈，展示你的积极转变，以引起他的好奇，增加重新添加的几率。即使对方没有马上添加，也可能在空闲时查看你的朋友圈。如果微信加不上，还可以制造偶遇，精心打扮并预设好场景。只要见面时你没有表现出过度需求，形象也有积极改变，对方很可能不会拒绝与你交流，此时提出加回微信，对方通常难以拒绝。
*   **编辑消息的技巧**：如果双方仍保留联系方式，可以直接联系，但消息编辑至关重要。错误的消息可能弄巧成拙，甚至导致被拉黑。编辑消息时需注意三点：
    1.  **避免高需求感**：不要在消息中表现出“没有你不行”等高需求、低价值的内容。这只会增加对方压力。应从无需求联系开始，例如分享“最近我发现了一家很有趣的餐厅”这类话题，引起对方兴趣但不暴露需求感。
    2.  **避免对立观点**：不要在消息中与对方争执分手时他指出的你的缺点。即使你不同意他的观点，争执赢了也可能失去对方。应建立共同点，表示你认同他的某些观点，并意识到自己某些方面做得不对，这有助于降低矛盾。
    3.  **建立二次吸引**：如果对方不回复，可能是因为看不到你的改变。朋友圈可以成为联系的桥梁。不要在朋友圈散发负能量或装作可怜。应积极进行自身建设，展示丰富的生活和改正缺点的行动。例如，如果对方曾嫌弃你生活懒散，你可以在朋友圈分享早上工作的照片或睡前喝牛奶的照片，这会引起对方的好奇，觉得你确有改变，从而可能主动了解你的近况。通过沟通技巧改变对方对你的印象，重新建立兴趣，达到二次吸引的目的。

很多姐妹想挽回感情，却害怕主动联系，害怕得不到回应或惹对方反感。事实上，只要在适宜的时间用正确的方法复联，并不困难。

## 总结
本节课我们一起学习了如何巧用复联法让对方主动联系你。我们介绍了“稳中求胜”的核心原则，详细讲解了“镜子法则”及其在聊天间隔和感情同步中的应用。接着，我们分三步走：先进行适当的情感冷冻与自我反省；然后在冷冻期内着力提升外在形象与内在修养；最后，通过微信或偶遇，运用无需求联系、避免对立和展示积极改变等技巧，逐步重建联系并建立二次吸引。请记住，挽回需要耐心和正确的方法，不要着急，一步步来。相信通过今天的学习，你已经明确了挽回的大方向，并知道了如何更好地联系对方。
