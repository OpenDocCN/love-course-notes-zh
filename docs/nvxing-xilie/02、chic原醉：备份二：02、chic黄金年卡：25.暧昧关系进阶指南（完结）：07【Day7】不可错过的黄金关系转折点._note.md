## 第一种：钢铁直男的特殊关怀
有些男生的关心可能藏在种种看似奇葩的举动之中。例如，网上有一个段子：女生对男生说“好冷啊”，男生回答道“蹦蹦来像我一样蹦一蹦就不冷了”。

从这个案例中，我们不难看出，男生是在用自己的专业知识或擅长的技能来表达关心。一方面，他的潜意识中期待被女生肯定，所以故意展现自己的特长和能力。另一方面，他付出的时间成本，可能比一些单纯买花送礼物的男生更高。

### 如何回应：自尊按摩法
此时，我们可以对他进行“自尊按摩”。即对意中人进行全盘的考察和理解，抓住他的炫耀心，不断地去肯定他、赞美他，让他的自尊心得到满足。

例如，上面例子中的女生就很善于运用自尊按摩。她在知道男生擅长运动后，就抓住了运动这个点。首先，她听从男生的安排一起跑步，领会了男生想要关怀她的心意。接着，她顺着运动的话题向男生请教经验和方法，进一步肯定了男生的骄傲点。最后，她带着运动后红扑扑的脸蛋对男生说：“谢谢你让我第一次体会到，运动也这么有魅力，你真是会想办法让我开心。” 这既让男生的自尊得到了极大的肯定，也让感情得到了升华。

**核心技巧公式**：`肯定行为 + 请教细节 + 表达独特感受 = 有效自尊按摩`

## 第二种：闷骚文艺的小惊喜
日剧《初恋那一天所读的故事》中，女主顺子一直感叹自己找不到好男人。其实，东大出身的商界精英雅致一直就对她情有独钟，暗恋了她20年，她却毫无察觉。剧中的诸多小细节，早就暴露了雅致的心意。顺子的家里有一面柜子，放满了各种奇怪的挂饰、吊坠、涂鸦、面具等等，那都是雅致每次出差送给她的礼物，也是他没有开口说出的爱意。

送特别的小惊喜和小礼物，隐含着想要引起对方兴趣的想法。心理学研究发现，人的大脑会对特别和非常规的信息更加敏感。所以，当对方表现出特别的一面、送你特别礼物的时候，就是关系更进一步的时机。

这时候，一定要放大你收到礼物时的惊喜感，给对方更多肯定，让对方觉得自己的心意被珍视了。这样有助于对方表现更多浪漫行为，认定自己是你的准男友。

### 如何放大惊喜感
以下是两个有效的方法：

**1. 言语肯定与身份暗示**
不单是说“谢谢，我很高兴，我很喜欢”，还要将他的行为和“男朋友”的身份挂钩。
例如，当你发现有些苗头时，可以对他说：“太高兴收到这个了！为什么送我这个礼物呀？只送我一个人吗？” 强调“只送一个人”，就是暗示他在履行男友义务，没有给其他女生送特殊礼物，从而拉近两个人的心理距离。韩剧《我的ID是江南美人》中，美莱低着头拿着朋友送的香水，羞涩地问：“是真的是送给我的吗？只给我一个人吗？” 这句话就很好地拉近了两人的距离。

**2. 行动展示**
收到男生的礼物之后，可以晒在朋友圈，并配上一段表达惊喜和开心的文字，让男生有机会看到。如果是衣服、小饰品之类的礼物，就在和男生约会见面的时候穿戴上，让他直观地感受到你对他的惊喜和浪漫有多么的喜爱和肯定。

**核心行动代码**：
```plaintext
if (收到特别礼物) {
    表达言语肯定 + 进行身份暗示；
    在社交平台或约会中展示礼物；
```

## 第三种：怦然心动的亲密距离
坐电梯的时候，如果里面已经有一个陌生人，那么下一位乘客一定会站在离他最远的一个角落里。这个对角线的距离就是人际交往中的心理安全距离。

心理学家曾做过实验，请80对陌生人在一个仅有两个人的空间里相处，然后测试他们之间的舒适距离。实验发现，任何一个人都无法忍受一个陌生人和自己坐得太靠近，这个最近的距离大约在15厘米左右。

例如，双方靠得特别近地走在街上，你可以借一个转身的机会将距离直接缩短为0。或者，在他凑到你面前和你说话之后，你又借机贴近他问他问题。保持这种亲密距离的互动，能使关系升级迈进一大步。
