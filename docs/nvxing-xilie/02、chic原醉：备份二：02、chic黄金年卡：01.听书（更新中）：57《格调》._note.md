# 格调解读：02：社会阶层的划分与识别

在本节课中，我们将一起学习《格调》这本书的核心内容。本书深刻剖析了美国社会的阶级问题，并指出文化品味和生活方式是划分社会等级的关键因素，其重要性甚至超过财产本身。我们将分三个部分来探讨：美国社会的阶层划分、从衣食住行判断阶层的方法，以及从精神层面识别阶层的视角。

## 第一部分：美国社会的阶层是怎样划分的

上一节我们概述了本书的核心观点，本节中我们来看看作者是如何具体划分美国社会阶层的。

美国政府虽宣扬平等，但其行为却揭示了社会的等级化，例如将公务员分为18个等级。在作者福塞尔看来，社会阶层应结合资产与文化因素来划分。他首先根据传统资产观念，将社会分为**上层阶级**、**中层阶级**和**下层阶级**。进而，他又依据品位格调等标准，将这三大阶级细分为九个具体阶层。

以下是上层阶级的三个细分阶层：

*   **看不见的顶层**：全部资产来自继承，极度注重隐私，很少公开露面。例如洛克菲勒家族。
*   **上层**：资产部分继承、部分自赚，常出现在公众视野，热衷马术等贵族活动。
*   **中上层**：资产主要依靠自己赚取，视继承为耻，是大多数美国人羡慕的阶层。

接着，我们来看中层阶级。这个阶级人数最多，流动性也最大，包含了四个阶层：

*   **中产阶层**：即白领阶层，人数众多，重视体面与外界看法，渴望归属感，常加入各种俱乐部。
*   **上层平民**：主要是技术工人（如印刷工、木匠），为自己的专业技能自豪。
*   **中层平民**：从事技术含量较低的工作（如公交司机），常担心失业。
*   **下层平民**：纯粹的体力劳动者（如码头工人），工作压抑，举止可能较粗野。

最后是下层阶级的两个阶层：

*   **赤贫阶层**：可见的底层穷人，如乞丐、流浪汉。
*   **看不见的底层**：生活在福利机构或管教所中，处境更为绝望。

## 第二部分：从衣食住行入手，快速判断社会阶层

了解了阶层的划分后，本节我们来看看如何通过日常生活的细节来快速判断一个人的社会阶层。这些外在表现往往是内在品味无意识的流露。

首先从“衣”，即外貌和服饰说起。

*   **外貌与体重**：美丽是通往上流社会的入场券之一。社会阶层越高，肥胖率越低，因为他们更注重身材管理。
*   **服饰风格**：上层阶级穿着合体、多层，偏爱羊毛、丝绸等天然面料，颜色喜藏青或暗淡柔和之色。他们与流行保持距离，追求简约高级感。
*   **平民阶层特征**：常穿合成纤维衣物，喜欢带有明显文字或logo的服装，紫色是其代表性颜色。过分整洁的着装有时反暴露了对地位的不安。

接下来是“食”，即饮食偏好与习惯。

*   **饮品**：阶层越高，越偏好低糖饮品（如白葡萄酒、伏特加）；平民阶层则喜爱高糖的啤酒、可乐。
*   **用餐**：晚餐时间越晚，阶层往往越高（上层约8点，平民约6点）。平民吃饭速度快，很少交流。
*   **口味与社交**：层级越高，口味越清淡；经常与亲戚聚餐是平民阶层的特点。

然后是“住”，即居住环境与房屋特点。

*   **上层阶级住宅**：位置隐蔽，客厅挑高，铺硬木地板和旧手工地毯，装饰偏好欧洲风格。
*   **中产阶级住宅**：追求整齐、对称，偏爱带有英国风情的名字。
*   **平民阶层住宅**：环境可能较杂乱，房屋较小，混凝土板建筑是典型标志。

最后是“行”，主要观察汽车与车道。

*   **汽车**：上层人士对车不感兴趣，开车慢且随意；中产阶级喜爱奔驰、宝马等名车；平民喜欢装饰车辆。
*   **车道**：社会阶层越高，车道往往越长、越曲折，这象征着地位而非实用价值。

## 第三部分：从精神层面判断社会阶层

除了物质生活，精神世界同样是判断社会阶层的重要维度。本节我们将从教育背景和阅读习惯两方面进行探讨。

首先，教育背景，特别是大学教育，至关重要。

名校教育不仅能提供知识和环境，更能带来人脉和圈层的改变，对维持或提升社会地位有决定性作用。例如，通过个人努力考入顶尖学府，是实现阶层跨越的重要途径。这解释了为何人们对名校如此执着。

其次，阅读习惯是更深层的阶级烙印。

*   **上层阶级**：阅读随性，追求惊喜，爱读管理类书籍、侦探小说，不在乎他人评价。
*   **中产阶层**：阅读焦虑，重视书籍的“正统”与权威性，喜欢指导生活的实用书籍。
*   **平民阶层**：偏好猎奇、丑闻和揭秘上流社会私生活的读物。

一个有趣的现象是，每个阶层似乎都不安于现状，底层想伪装成中层，中层想成为上层，而上层则想低调如“看不见的底层”。

然而，书中还提出了一个超越传统阶层的群体——**“另类人物”**。他们思想自由、行为独立，不盲从主流成功标准，按自己的喜好生活和工作。例如我国作家木心，无论在何种境遇下都保持着精神的自由与生活的精致。他们是这个时代的“精神贵族”。

对于我们当代人的启示在于：虽然社会存在阶层，但通过保持好奇心、强化行动力、坚持终身学习，并抓住时代变革的机遇（如互联网浪潮），个人依然有可能实现阶层的提升或活出自由的“另类”人生。

## 总结

本节课中我们一起学习了《格调》的精华内容。

首先，我们了解到社会阶层的划分远不止于财产，还包括财富来源、职业类型、生活心态和文化品味，作者将其细分为三个阶级、九个阶层。

其次，我们学习了如何从“衣食住行”等外在细节快速判断一个人的社会阶层，这些细节包括服饰面料、饮食口味、居住环境和出行方式等。

最后，我们探讨了从教育背景和阅读习惯等精神层面识别阶层的方法，并认识了超越传统框架、追求精神自由的“另类人物”。

本书如同一面镜子，提醒我们关注生活中不易察觉的品味与格调，同时也鼓励我们，在认清现实的基础上，可以通过个人奋斗与坚持学习，寻求属于自己的人生突破与精神自由。
