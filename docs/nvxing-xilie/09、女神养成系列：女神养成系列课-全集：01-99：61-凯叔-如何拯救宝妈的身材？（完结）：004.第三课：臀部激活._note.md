# 女神养成系列：01-99：61 第三课：臀部激活 💪

在本节课中，我们将学习如何通过一系列普拉提动作，针对性地激活和锻炼臀部肌肉，帮助改善产后常见的臀部下垂、外扩及“假胯宽”等问题。课程内容简单易学，适合在家使用瑜伽垫和简易道具进行练习。

## 概述

本节课将专注于臀部塑形。对于许多产后妈妈而言，骨盆和臀部的变化是常见的困扰。我们将通过科学的动作顺序，学习如何正确募集臀部肌肉，避免错误发力导致大腿变粗，从而有效提升臀线、改善臀型。

## 第一节：俯撑抬腿（鸟狗式）

上一节我们概述了课程目标，本节中我们来看看第一个激活全身及臀部的准备动作——俯撑抬腿，也称为鸟狗式。这个动作能锻炼核心稳定性，并预先激活臀肌。

首先，转身俯撑于垫面之上。双手食指张开，指尖抓地以减少手腕压力。双膝分开与髋同宽，脚背压实垫面。调整骨盆至微微前倾（轻微塌腰翘臀）。肩膀下沉远离双耳，肩胛骨保持平贴。头部后仰顶向天花板，微收下颌。

**动作步骤：**
1.  吸气准备，运用横向呼吸法。
2.  呼气时，将左腿沿垫面向后延长，核心收紧以保持骨盆稳定。
3.  再次吸气准备，呼气时将左腿与右手同时向上平举，与身体成一条直线。脚趾回勾，脚跟用力后蹬。
4.  吸气时缓慢落下手脚，收回膝盖。
5.  换另一侧重复：吸气准备，呼气时右腿向后延长，再次呼气时抬高右腿与左手。

**进阶练习：**
在手脚抬高的姿势下，呼气时尝试用右手肘去触碰左膝，然后还原。动态练习中需特别注意收紧腹部核心。

**动作要点：**
在将腿向上抬高前，必须先向后延长腿部，并预先夹紧臀部肌肉，感受臀肌向内收缩，再将腿抬起。这样能将训练力量集中于臀部。

## 第二节：俯身单腿上提

完成了基础的稳定性练习后，本节我们将进行一个针对臀大肌的孤立训练动作——俯身单腿上提。这个动作能有效矫正臀型，激活臀大肌力量。

俯卧于垫面，双手上下交叠置于前额下方。肩膀下沉，头部保持在脊柱延长线上。调整骨盆位置，使骨盆前侧最高点（髂前上棘）压实地面。双腿分开与髋同宽，脚背压实地面。

**基础练习步骤：**
1.  吸气准备，呼气时收紧腹部和盆底肌，保持腹部与地面有空隙。
2.  预先用力夹紧臀部肌肉。
3.  呼气时，先将左腿向远处延长，然后向上抬起约30度。
4.  吸气时缓慢落下。换右腿重复。
5.  单侧重复5到8组。

**进阶练习（画字与画圈）：**
1.  抬腿至适当高度后保持不动。
2.  **画“一”字：** 呼气时，想象脚尖在对侧墙上横向画“一”字，然后还原。
3.  **画“1”字：** 接着想象脚尖竖向画阿拉伯数字“1”。
4.  **画圈：** 最后用脚尖在对侧墙上画小圆圈。
5.  每个小动作重复3到5次，然后换腿。

**动作要点：**
始终收紧腹部核心，肩膀下沉。抬腿前务必预先夹紧臀部，确保发力点正确。

## 第三节：骨盆卷动

上一节我们学习了针对臀大肌的练习，本节我们引入一个经典动作——骨盆卷动。它不仅能强化臀部，还能锻炼脊柱灵活性和骨盆底肌，对产后恢复尤为重要。

仰卧于垫面，屈双膝，双脚踩地。确保膝盖与第二脚趾方向一致。骨盆调整至中立位（腰下可放入一个手掌）。肩膀放松贴地，头颈延长。双手掌心向下放于身体两侧。

**动作步骤：**
1.  吸气，呼气时收紧盆底肌及腹部，同时夹紧臀部，使骨盆做轻微后倾（腰部更贴近地面）。
2.  在夹紧臀部的状态下，想象脊柱是一串珍珠，用骨盆的力量带动脊柱从腰椎开始，一节一节向上卷离地面，直至身体呈斜板状（肩胛骨下缘仍接触地面）。
3.  在顶端保持5-10秒，收紧臀部。
4.  呼气时，想象胸口有冰块融化，从胸椎开始，一节一节缓慢下落，最后放松臀部。

**动作要点：**
全程保持腹部收紧，颈部放松。上升和下落的过程要有控制，专注于脊柱的逐节运动。预先收紧臀部是发力关键。

## 第四节：侧卧抬腿系列（激活臀中小肌）

接下来，我们将注意力转向臀中小肌。这部分肌肉的无力会导致骨盆不稳定和“假胯宽”。本节通过侧卧抬腿系列动作进行强化。

侧卧，下侧手肘弯曲支撑头部，上侧手在胸前支撑以保持平衡。身体呈一条直线，想象骶骨、后背和后脑勺贴靠墙面。收紧核心，避免腰部塌陷。

**动作一：侧卧直腿上抬**
1.  上侧腿向前伸直，与身体呈120-160度夹角。双脚呈“V”字位（普拉提站姿）。
2.  吸气，预先夹紧上侧臀部，呼气时将上侧腿向上抬起。
3.  吸气下落。重复5-8次。
4.  可将脚位转为平行（脚尖向前），重复练习。
5.  **动态变化：** 勾脚向前摆动，绷脚向后摆动，如同画“一”字。保持骨盆稳定。
6.  **画圈：** 绷脚尖，在空中画小圆圈。

**动作二：侧卧蚌式开合**
1.  上侧腿屈膝，脚掌踩于下侧腿膝盖前方。
2.  吸气，预先收紧臀部，呼气时将下侧腿向上抬起。
3.  吸气下落。重复3-5次。
4.  **画圈：** 抬腿后，用脚尖画小圆圈。
5.  **组合练习：** 伸直上侧腿，呼气时下侧腿向上并拢，吸气时分开。

**动作三：侧卧棒式**
1.  双腿屈膝并拢，身体保持直线。
2.  吸气，双脚并拢，呼气时将上侧膝盖向上打开（如同贝壳打开），感受臀部外侧收缩。
3.  吸气还原。重复5-8组。
4.  **角度变化：** 将大腿与身体夹角调整为90度，再进行膝盖上抬练习，此时能更强烈地刺激臀大肌上部。

## 第五节：哑铃屈腿硬拉

最后，我们学习一个能有效提升臀线和臀围的复合动作——哑铃屈腿硬拉。此动作主要激活臀大肌和腘绳肌。

站立，双脚与髋同宽。双手各握一个哑铃（或装满水的矿泉水瓶），掌心朝向身体，置于腹股沟位置。微屈膝，膝盖不超过脚尖。

**动作步骤：**
1.  呼气，将坐骨向后推，上半身前倾，哑铃沿大腿前侧下滑。保持背部挺直，核心收紧，腹部与大腿间应有空隙。
2.  吸气时，预先用力夹紧臀部，利用臀肌发力将身体推回起始站姿。
3.  重复5到8组。

**动作要点：**
动作过程中脊柱保持延展，重心在脚跟。上升阶段的主导发力点必须是臀部肌肉。

## 课程总结与拉伸

本节课我们一起学习了从臀部激活到强化的系列动作。关键在于掌握正确的肌肉募集顺序：在任何一个抬腿或伸展臀部的动作前，都先有意识地收紧臀部肌肉。

练习后若感到臀部酸痛，可进行简单拉伸：
1.  **仰卧抱膝：** 仰卧，将一侧膝盖拉向胸部。
2.  **仰卧4字拉伸：** 仰卧，将一侧脚踝放到对侧膝盖上，双手抱住下方大腿拉向胸部。

坚持练习，你将逐渐掌握臀部发力的技巧，有效改善臀型，收获更健康、更优美的身体线条。
