# 职场进阶：02：资本寒冬下的职场胜出秘籍：11【Day11】寒冬中如何成功获得升职加薪？

在本节课中，我们将要学习在资本寒冬的背景下，如何通过有效的策略和沟通技巧，在原公司内成功实现升职加薪。我们将探讨其背后的逻辑、具体的准备方法以及谈判的关键步骤。

上一节我们介绍了通过跳槽提升职业价值的高难度操作。本节中我们来看看一个相对更稳妥的路径：如何在公司内部争取升职加薪。

## 为何寒冬中升职加薪更难？

诚然，在资本寒冬期间，升职加薪的难度变得比平时要高得多，主要原因有两个。

以下是两个核心原因：

1.  **成本控制**：在资本寒冬时期，公司控制成本花的每一分钱都要经过深思熟虑。所以升职加薪的预算自然而然就会减少。可以被纳入升职加薪计划的人数，自然而然也会减少。
2.  **竞争加剧**：资本寒冬的这种状态下，人人自危，不想跳槽的大家一定会想方设法留在公司。为了能够留在公司，也能够让自己的职业生涯得到发展，大部分人都会选择抢占升职加薪的坑位，这就导致升职加薪的竞争也变大了。

## 升职加薪的具体方法

那么是不是在资本寒冬之中想要升职加薪就一点办法也没有呢？其实也不是。我们升职加薪的具体方法可以参考以下几个方面。

### 从老板的视角出发

虽然对比于平时，资本寒冬中的升职加薪是难了点儿，但是本质上却还是老板对于员工未来价值的一种投资。但是由于资本寒冬中，企业用的每一分钱都要十分小心，所以老板更希望有综合性的人才出现。

什么叫做综合性的人才呢？通俗来说，就是一个人可以干好几种活。

在老板的眼中，薪酬体系更像是撬动员工潜力的一根杠杆。老板们往往希望用升职加薪来挖掘员工更大的潜力。这个所谓更大潜力的意思就是说为公司创造更多的价值。

也就是说，想要升职加薪，你就必须在公司承担更多的责任和业务。这是因为你的薪水衡量标准是你以前做的那些事。而你想要拿到更多的回报，就意味着你要在这个基础之上做更多的事情。

**公式化理解**：
> 当前薪水 ≈ f(历史贡献价值)
> 目标薪水 ≈ f(历史贡献价值 + 未来预期价值增量)

比如说你的薪水是按照你做的事情给公司带来了80万收益的标准给的。那么如果你想要更上一层楼，你就必须要创造100万的收益。

### 如何拓展业务与承担责任

那么怎么样才能找到机会去为自己拓展、承担更多的业务呢？其实方法有两种。

以下是两种主要途径：

1.  **内部整合，承担上下游工作**：在自己的上下游之中承担更多，通过为公司节省人力成本来创造收益。
    *   **示例计算**：一个公司有3个部门，每个部门的人力成本是100万。如果你的部门可以承担其他两个部门的工作职责，那么就可以把其他两个部门精简掉，然后给你部门的人工资翻倍。这样计算下来，公司就会从人力精简中获得100万的营收，也给核心的骨干部门集体涨了工资，稳定了军心。
2.  **主动开拓，寻找新增长点**：自己寻找新的业务给老板。因为企业没有新项目，对于上下游的工作也相对稳定，想要额外承担，就需要我们自己为公司寻找增长点来做。这种做法不但能突出你的老板思维，而且也能够展现你对公司的忠诚度。

在现实的职场当中，我们要怎么做才能够让老板给我们更多的业务呢？

我们要首先确保之前被交到自己手中的工作完成度和还原度是百分之百的。因为只有有了之前项目的背书之后，老板才会想要给我们更多尝试的机会。你可以试想一下，如果你都完成不好老板之前交在你手中的任务，还想在这个基础之上拿到更多的业务展现自己，这怎么可能。

在这个基础之上，你需要勇敢地毛遂自荐。告诉老板，你想要承担更多的业务。在资本寒冬之中，每个公司一定会想要找到新的增长点，这就是一个绝佳的展现机会。这种项目就是我们之前课程当中所说的，老板期待不高的项目。这种项目做好了，升职加薪；做不好呢，也不能完全怪你。所以只要注意好每一个自己能把控的细节，把最大的努力呈现给老板就好了。

关于如何寻找新增长点，比较常见的方法就是竞品比对。看看市面上同类型的产品都做了哪些我们没有做的业务。分析一下，怎么样可以用最小的成本在公司里推动这件事情。做一个比较完整的方案交给老板，然后在新项目中展现自己的职场肌肉。如果通过这种方法找不到新业务，可以试着从行业研究的新方向去寻找一些突破口，找寻行业的新方向，一般就是通过专业性的网站、公众号、行业研究论文等等。

### 判断自身定位

可是在资本寒冬之中，老板并不是想对任何的员工都进行鼓励。在老板的心目当中会对员工进行分类：可以升职加薪的、暂且在考察的、以及需要被裁员的。我们就需要明白在老板心目中，我们自己是属于哪一个档位的。

那么如何判断自己在老板心目中属不属于被升职加薪的目标呢？判断我们属于哪个档位，可以从三个维度出发。

以下是三个关键判断维度：

1.  **工作任务完成情况**：这个不用多说，就是你过往实力的证明。这个部分一般分为基本完成、全部完成和超额完成三种情况。超额完成这种情况，就是更容易获得升职加薪的机会。
2.  **你的职能重要性**：也就是你的工作在公司中起什么作用。这是因为在资本寒冬之中，老板往往会砍掉一些非核心部门的员工来节省成本。所以这个时候不是核心部门的你去谈涨薪，很容易被老板认为你没有眼力劲，对你产生不好的印象。
3.  **和老板的关系紧密度**：我们前面说过，职场中比较少存在有经常和老板交流、老板用得十分顺手的人被老板裁掉的情况发生。所以老板如果经常和你进行交流，那么大概率上你也不会在备裁的名单里。

如果三条判断之后，你发现你工作超额完成，又是核心部门的员工，还经常能和老板交流，那么大概率上就可以准备和老板谈升职加薪这件事了。

## 如何与老板谈判升职加薪

接下来我们就来一起学习，如何和老板谈升职加薪这件事吧。

### 谈判心态：平等交换

首先是心态上，是谈判态度平等交换。在我的咨询案例中，有数量非常之多的涨薪失败的案例，我发现这其中有非常大的一部分人是抱着“我为你贡献了这么多，创造了这么多，我来找你要回报”的这种心态去和老板谈升职加薪。这其实是由于我们和老板对于贡献认定的视角差决定的。

更好和老板谈升职加薪的心态，应该是不卑不亢、平等交换。

### 谈判核心：谈现在、过去与未来

说完心态建设，我们再来说说实际行动中应该如何一步一步的操作。所谓升职加薪，其实就是你想要借着今天的成绩，让老板对你的过去进行肯定，而老板呢想要从今天的成绩出发，对你的未来进行鼓励。我们在谈升职加薪这件事儿的时候，主旨就是和老板谈现在、谈过去、以及谈未来。

以下是谈判的三个核心环节：

1.  **谈现在（选择时机）**：在升职加薪谈判中，什么时候说非常重要。首先我们要知道，升职加薪必须要遵循公司整体收益趋势和人事决策。比如说如果最近公司账目上颗粒无收，疯狂裁员，而你在这个时候想要去提升职加薪，不是自己往枪口上撞吗？但升职加薪最好的时期，就是一个项目完结且有一定成果的时候。因为在这个时候，老板对你的未来预期是最大的，也是最想要投资你的时候。所以这个时候呢，往往提升职加薪这件事儿的成功几率就会变高。
2.  **谈过去（展示成绩）**：时间节点确定之后，我们就要开始给老板寻找一些“非我们不可”的投资理由了。这个投资理由就得从我们曾经做出的业绩之中去寻找，即老板的心态试探以及自己的成绩总结。这一步最好是可以形成一个书面报告来向老板展示。这样不仅可以清晰地将自己为公司创造的业绩成长呈现给老板，还可以给老板展示自己的汇报沟通能力。

以下是整理书面报告的具体方式：

*   **倒序列出工作成就清单**：这个清单的时间顺序，最好按照最近到之前的顺序。
*   **数据化佐证成绩**：每个项目最好找出一些数据化的东西来佐证自己的成绩，比如卖了多少东西、涨了多少粉丝、吸引了多少流量、或者为品牌进行了多少次的曝光等等。然后将这些数据做成可视化图表的样子，清晰直接地给老板展示你的价值。
*   **区分本职与超额工作**：报告内容中，最好还可以梳理出你自己完成的本职工作部分和你超额完成的工作。这样可以有理有据，多方位的为老板呈现你的价值。
*   **展示技能与成长**：在最后你还可以列出一个关于自己现阶段的技能清单，包括硬实力和软实力。然后对比一下自己入职前后的能力变化，来体现你的学习能力，为下一步谈未来做准备。

这个部分建议大家在准备提升职加薪之前，提前做好报告，并且反复练习如何给老板做这份成绩汇报。因为在这种面对面的谈判中，哪怕一个不美好的小细节，都可能毁了最后的结果。

3.  **谈未来（规划与承诺）**：现在和过去都说完了，升职加薪谈判就剩下了老板最关心的未来环节了。在这个环节中，我们要牢牢记住一点，那就是“我想与公司共同成长”。在过去和现在的环节，你描述了你的能力水平和你的成长速度。而在未来环节中就是展现你的未来职业规划。

关于你的职业未来规划，你可以从“如果老板满足你的需求，你还会从哪些方面改进自己的现有工作”以及“你还有能力，还能承担多少工作”两个方面去论述。这个“多承担多少工作”呢，可以从和自己同类型的工作中去寻找，也可以从自己的产业上下游之中去寻找。

**举例说明**：比如说你作为一个即将被老板升职加薪的初级产品设计，你以前只需要负责一个产品的一个部分。那么你在做未来规划的时候，就应该从整个产品的整体出发来做一个优化计划。
