# 女神养成系列：02：左右肩交替前摩术

在本节课中，我们将学习“摩肩驻颜术”的第二个核心动作——左右肩交替前摩。这个动作是在上一节“双肩同时前摩”基础上的一个变化，通过两肩交替画圆，能更精细地活动肩颈区域，促进气血循环。

上一节我们介绍了双肩同步前摩的动作，本节中我们来看看如何让两肩交替运动，以达成更均衡的锻炼效果。

## 🧘‍♀️ 动作分解与演示

起始姿势与之前保持一致：
1.  **双脚**与肩同宽，自然站立。
2.  **头部**保持正直，目视前方。
3.  **身体**放松而挺拔，不僵硬。
4.  **双臂**自然垂放于身体两侧。

在以上姿势基础上，开始进行交替前摩。以下是动作的核心描述：

*   **动作模式**：双肩**轮番、交替**地向前画圆。
*   **动作顺序**：例如，先让左肩向前、向上、向后、向下画一个完整的圆；当左肩画到后半程时，右肩开始接续同样的向前画圆动作。如此循环往复。
*   **动作要求**：在个人关节活动度允许的范围内，**尽量将画圆的幅度做大**，以充分拉伸和活动肩部肌群。

## ✨ 核心要领与提示

为了确保练习效果，请关注以下要点：

*   **交替节奏**：动作的关键在于“交替”，即一肩接一肩，形成连续流畅的波浪式运动。可心中默数“1、2、3、4”以帮助建立节奏。
*   **幅度优先**：在保证动作正确、身体无痛感的前提下，应追求最大的画圆幅度，而非单纯追求速度。
*   **保持放松**：画圆时，手臂始终自然下垂，无需用力。动力应来自于肩关节本身的旋转，身体其他部分保持稳定放松。
*   **呼吸配合**：保持自然、深长的呼吸，无需刻意屏气。

本节课中，我们一起学习了“左右肩交替前摩术”。这个动作通过双肩交替画圆，能有效缓解肩颈僵硬，促进局部血液循环。请记住，练习时务必专注于动作的幅度与交替的节奏，并持之以恒。

坚持练习，你将感受到肩颈日益舒展与轻盈。下一节我们将继续深入，介绍新的变化动作。
