# 理财入门：07：做个体检：你的财务健康吗？💡

在本节课中，我们将学习如何利用前两节课梳理出的财务数据，为自己的财务状况进行一次全面的“体检”。通过检查一系列关键指标，你可以提前发现财务隐患，并朝着更健康、更自由的财富状态迈进。

上一节我们梳理了收入与支出，本节中我们来看看如何综合评估财务健康状况。

现在许多公司会组织员工体检，我们也会带家人做健康检查。通过体检指标，可以预先发现健康隐患。对你的财务而言，道理是类似的。

之前我们提到财富中潜藏着四类隐患：意外、疾病、无规划的支出和混乱的投资系统。这些隐患都可以通过今天的财务体检提前进行预防和准备。

通过财务体检的各项指标检查，你可以查漏补缺，避免意外和疾病带来的财务冲击，还能改善储蓄情况，合理规划收支，并观察自己是否通过投资让财富持续增值。

下面我们就一起手把手给财务做下体检。

## 财务体检的三大方面 📊

身体检查有各项衡量指标，财务体检也是类似的。总的来说，你可以关注以下三大方面：**安全指标**、**成长指标**和**财务自由程度**。

这三类指标可以构成一个三角形，类似于心理学中的马斯洛需求层次理论。最基础的是保障基本生活需求，对应**财务安全指标**。上一层用来检测财务保值增值的**成长能力**。最顶层是对**财务自由**的需求，让我们更坦然选择想要的生活。

实际中有一句话：“仓廪实而知礼节，衣食足而知荣辱。”只有先解决了生存问题，才能一步步实现自由的状态。

让我们一个个来看。

### 一、安全指标：你的财务“保护垫”够厚吗？🛡️

这个指标用来检查保障我们财务安全的基本“保护垫”是否足够厚实。

以下是几种常见的不安全财务状态，你可以自查：
*   我是月光族，收入增加却存不下钱。
*   有时钱不够花，会不知不觉用到信用卡循环借贷。
*   追求高收益，手上的钱都投出去了，没留多少流动资金。
*   家人几乎没买过保险，遇到大风险需要东拼西凑。

如果你或多或少有这几类问题，很可能意味着你的财富安全指标有问题。

财务安全指标分为三层：**应急**、**偿债**和**保障**。这三层保护可以一层比一层更牢固地守护我们的财富。

#### 1. 应急能力

**概念**：指如果有突发事件发生，短期内暂时没有家庭收入，也能维持现有生活水平的能力。例如突然生病住院需交大笔押金，或跳槽换工作时暂时没有收入。

**检测方法**：
`应急能力 = 流动资产 / 每月生活支出`

**公式解读**：
*   **流动资产**：指能够迅速变现的钱（活钱）。
*   **每月生活支出**：包括日常开销及必须偿还的房贷、车贷等债务。

**健康标准**：计算结果在 **3到6** 之间为达标。这表示如果遇到突发事件，你手上可迅速变现的钱能够维持3到6个月的生活。

动手算一算你的应急能力能满足几个月的生活开支。

#### 2. 偿债能力

**概念**：检验我们是否有陷入债务危机的风险。

**检测分为短期和长期**：
*   **短期偿债能力**：
    `短期偿债能力 = 流动资产 / 短期负债`
    如果结果 **小于2**，需要格外注意，及时备足偿债的流动资产。
*   **长期负债健康度**：
    `资产负债率 = 总负债 / 总资产`
    总负债**绝不能超过总资产的60%**，最好**不要超过50%**。

**额外提醒**：
*   **还款压力**：每月还贷金额最好不要超过月收入的**40%**，否则压力会威胁生活品质。
*   **借款利率**：尤其需留意信用卡、花呗、借呗等小额消费贷款的实际年化利率，可能高达**10%**以上，超过普通人的理财收益水平。

正所谓“借钱如山倒，还钱如抽丝”，生活中借贷时一定要留意还款压力和成本。

#### 3. 保险保障能力

**概念**：重在测试当我们遇到难以承受的重大风险（如意外、重疾）后，生活水平是否能够基本维持。

**核心**：配置合适的保险。我们会在后面的保险课程详细介绍。

有了这层保护，我们就可以抵御“变穷”方法中的前两个因素：**意外**和**疾病**。

### 二、成长指标：你的钱在好好赚钱吗？📈

这个指标检查你的财富是否在有效增值。即，你的钱有没有在好好地为你“钱生钱”。

以下是几个需要自查的问题：
*   你大部分的钱“躺”在哪里？
*   你是否大部分甚至全部投资都在高风险领域？
*   你是否了解自己的风险承受能力和投资水平，应该追求怎样的投资回报率？

我们可以从三个维度测试财富成长度：**储蓄能力**、**生息资产比例**和**投资回报率**。

想让财富越来越多，主要有两个方法：一是“**人赚钱**”（通过储蓄积累财富），二是“**钱生钱**”（通过投资利滚利）。以下三个维度就是在解释如何让财富不断增值。

#### 1. 储蓄能力

**检测方法**：查看收入支出表。
`储蓄率 = 每月储蓄额 / 每月收入`

**健康标准**：如果每月储蓄额超过收入的**30%**，恭喜你是储蓄小达人。

**注意**：有房贷的朋友，每月还款中的**本金部分**已以房产形式为你“储蓄”，计算时可考虑进去。

如果暂时储蓄比例不达标，不用担心，后面会有专门课程帮你轻松养成储蓄习惯。

#### 2. 生息资产比例

**概念**：查看你的钱中有多少比例在帮你“钱生钱”。

**检测方法**：打开资产负债表。
`生息资产比例 = 投资资产 / 总资产`

**健康标准**：如果这个数值**没有超过总资产的50%**，会大大阻碍财富增值速度，需要留意。

**注意**：对于刚买完首套自住房的年轻人，房产在总资产中占比可能很高。计算时，可考虑在总资产中**扣除自住房产的价值**，再来计算投资资产占比会更合理。

#### 3. 平均投资回报率

**概念**：衡量投资效率的核心指标。

**检测方法**：
`平均投资回报率 = 年投资收益 / 总投资资产`

**公式解读**：
*   **年投资收益**：在收入支出表中查找。
*   **总投资资产**：在资产负债表中查找。

同样投一笔钱，收益率越高，回报就越高，代表财富增值越快。

**参考标准**：能够达到**5%**左右的回报就是一个好的起点。巴菲特的长期年化回报率也在20%左右，所以不要相信30%-40%的收益神话，普通人应追求适合自己的回报率。

**如何寻找适合自己的投资回报率？**
建议在课程资料包中寻找**风险承受能力测试表**。通过测试，你可以找到比较适合你的投资回报率参考。

**额外提醒**：投资回报还和当前市场形势相关。有些年份市场整体回报率高，但近些年有下降趋势，切勿盲目追求高收益。与投资有关的内容会在后面课程详细展开。

### 三、财富自由度：你离财务自由还有多远？🎯

这个指标关乎我们的终极目标——实现财务自由。

**计算方法**：
`财富自由度 = 被动收入 / 日常支出`

**公式解读**：
*   **被动收入**：指不需要你再额外付出时间和劳动就会获得的收入（在收入支出表中查找）。除了投资收益，版税、租金等也是被动收入。
*   **日常支出**：在收入支出表中查找。

**指标意义**：
*   如果数值**超过100%**，说明你不用工作，只靠“钱生钱”就能满足日常生活。理论上，这就是**财务自由**的状态。
*   如果现在这个比例已经**超过15%**，那么你的财富自由度已经是很不错的情况。

**重要理解**：财务自由并非一个刻板的比率。一方面，直接把它作为目标压力很大；另一方面，由于通货膨胀，理财是我们一直需要考虑的事。它更是一种让我们能更自由选择生活状态的能力。

## 总结 📝

本节课中我们一起学习了如何进行财务体检，它包含三项核心指标：

1.  **安全指标**：分三层（应急、偿债、保障），安全级别逐步提升，是财务健康的基础。
2.  **成长指标**：由三部分组成（储蓄能力、生息资产比例、投资回报率），评估我们的财富是否在不断累积和“钱生钱”。
3.  **财富自由度**：当超过100%时，被动收入就能完全覆盖日常生活，让你能更自由地选择想做的事。

通过定期进行这样的财务体检，你可以像关注身体健康一样，持续关注并改善自己的财务健康，稳步走向更富足、更自由的人生。
