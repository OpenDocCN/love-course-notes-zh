# 理财入门：10：学会记账，做个心中有数的女人 💰

在本节课中，我们将要学习一种名为“资产记账法”的财务管理方法。它将帮助你从财富增长的角度，而非琐碎的流水账，来管理日常收支，让你对自己的财务状况更加清晰。

上一节我们介绍了储蓄的方法和财富水池模型。本节中我们来看看如何通过记账来具体管理每日的资金流动。

很多人对记账的第一反应是麻烦且无效。吃一次外卖、喝一杯奶茶都要记录，感觉越过越小气。一些记过账的朋友也表示，记了几个月的账，既繁琐，钱又没变多，最终沮丧放弃。

别担心，我们今天要聊的“资产记账法”将首先抛弃记流水账的旧观念。我们从把握三个新原则开始，告别繁琐和无用功。

## 资产记账法的三个核心原则

以下是资产记账法需要遵循的三个原则，它们将改变你对记账的看法。

### 原则一：重计资产，轻计收支

你可能已经注意到，我们的方法叫做“资产记账法”。带上“资产”二字，重点就不同了。还记得资产负债表和收支表之间的关系吗？公式可以表示为：

**收入 - 支出 = 结余（储蓄）**

而这些储蓄会增加你资产负债表中的**净资产**。资产记账法就是把关注重点，从记录每一笔收入支出，转移到关心最终目标——**净资产的增减**，也就是财富值的增加或减少上。我们不仅要记录挣钱和花钱，更要关心“钱袋子”有没有变鼓。因此，记账的侧重会转移到资产负债上，而简化日常收支的记录。具体方法稍后介绍。

### 原则二：重视定期分析

记账不在于把日常每一笔收支都记得精确无误，而在于能否从过往数据中，发掘自己的“吸金点”和“财务黑洞”。所以，记账只是中间环节，重点在于有意识地分析收支，以及财务增长是否符合你的预期。我们不仅要关心钱袋子的变化，还要思考如何让它更好地变鼓。

### 原则三：学会制定预算

这意味着要大概知道自己每个月花多少钱，并严格控制在这个总额内。把记账和制定预算结合起来，说明我们不光是在记录收支，而是在**管理收支**。我们不仅要找到钱袋子变鼓的原因，还要以此为指导，实践让钱袋子更鼓的方法。

了解了核心原则，接下来我们看看如何具体实践资产记账法。你可以边听边动手，通过以下四步开启你的资产记账之旅。

## 四步实践资产记账法

以下是实践资产记账法的四个具体步骤。

### 第一步：准备工作

俗话说，工欲善其事，必先利其器。为了更高效地记账，你需要先做一些准备：挑选一个合适的记账软件，并准备好你已有的银行卡、电子钱包账户、投资账户等基础信息，方便记录。

挑选记账软件时，你可能会发现有的功能多，有的界面精美，有的设计简单。建议重点把握两个方面：第一是自己用起来方便顺手；第二是能方便地与资产记账法结合，例如便于记录资产、带有分析统计功能等。

### 第二步：从资产负债开始，完成账本初始设置

这一步很简单，就是将你各项资产和负债都记录在记账软件上。相信梳理过资产负债表的你能轻松完成。

我们以网易有钱APP为例。其记账功能主要分为“资产”和“账单”两部分。“资产”一栏用于记录我们资产负债表的各种科目。

点击“添加资产”，你可以参考软件的分类，记录信用卡欠款、银行卡余额、支付宝/微信余额、基金、P2P等各类投资，甚至房子、车子等资产负债信息。

对于投资账户清点麻烦的问题，这里有一个小技巧：善用记账软件的**同步功能**。现在很多软件支持同步主流银行卡、支付宝、微信的账单，以及基金、股票软件甚至网贷平台的信息。这样记录非常方便，相当于拥有了一张动态的资产负债表。

当然，如果你担心个人信息安全，挑选有安全保障的记账软件就很重要。除了查看软件的安全说明，也可以直接咨询官方客服，例如询问平台服务器是否会储存密码、个人信息是否经过技术加密等。如果你实在不放心，所有记账软件也都支持手动录入，你可以选择不进行同步。

### 第三步：巧记日常收支

梳理了资产和负债后，我们来看如何记录日常收支。日常收支繁多，如何记录呢？除了利用账户同步功能直接同步零散消费，这里分享一个轻巧的技巧：**专卡专用，分账管理**。

结合财富水池模型分析，日常资金流转大致可分为三类：
1.  **日常开支**：如衣食住行，每笔金额不大，但很频繁。
2.  **大额目标支出**：如年底旅游费、孩子学费，金额大、次数少、时间可控。
3.  **投资资金进出**：如定投基金、投资到期回款等。

如果只用一张银行卡管理，账单会很凌乱。但根据用途挂钩不同的银行卡和投资账户，不仅方便记录，还能提前做好预算和规划。

例如，用户娜娜使用3张不同的银行卡：
*   **第一张：零钱卡**。绑定支付宝/微信，所有日常消费都经此卡。每月工资发下后，立即转入5000元作为生活费。日常是否超支，看此卡余额便一目了然。
*   **第二张：目标储蓄卡**。存放短期目标资金，如孩子学费、换车钱。并为每个目标匹配期限、风险和收益适当的理财产品（如两年期银行理财）。
*   **第三张：金额卡**。挂钩基金股票账户，用于长期投资。除非万不得已，不动用此卡资金，这是坚持长期投资的基础。

你会发现，分账管理的技巧与财富水池的思路非常相似。万变不离其宗，从需求出发，更容易找到适合自己的方法。

具体到“记”这一步就很简单了。一般利用软件的同步功能上传账单。如有遗漏，手动添加即可。此外，如果夫妻共同记账，不少软件支持建立家庭账本，同步两人账单并区分消费人，有助于更好的家庭财务分析。

### 第四步：分析消费投资结构

记录只是中间环节，分析收支、评估财务情况才是重点。具体如何实践呢？

一方面，你可以结合之前课程介绍的财务体检指标来分析。目前大多数记账软件都提供统计功能和可视化图表（如饼图），帮助你快速分析。例如，生成收支和投资的饼图，能更清晰展示消费和投资的去向。曾有朋友通过查看支出饼图，发现每月电话费过高，才想起未关闭出差资费包，及时进行了修改。

另一方面，要回归资产记账法的重点：将资产负债情况与财富水池法结合。
*   除了通过预算管理日常现金池，我们还应定期分析**目标池**中各项资产是否能按预期完成目标。
*   关注**金额池**中，各类投资的比例是否符合你的预期配置，财富滚雪球的速度是否符合预期等。

## 总结与复习

本节课中，我们一起学习了“资产记账法”。内容较多，以下汇总了一个表格，方便你复习查看：

| 核心原则 | 具体方法 |
| :--- | :--- |
| **重计资产，轻计收支** | 关注净资产增减，而非每笔流水。 |
| **重视定期分析** | 从数据中发掘“吸金点”与“财务黑洞”。 |
| **学会制定预算** | 管理收支，而不仅是记录收支。 |

记账这件事，实质重于形式。想要记账更省力，不妨从上述三项原则出发。具体方法很简单：先将各项资产记录其中，每月的收支变动最终都会反映在净资产的增减上。多使用账户同步功能，并采用分账户打理的方式，就能轻松将财务信息记录清楚。
