# 安全感修炼手册：02：03【Day3】谈恋爱不如养条狗啊——为什么有些人对爱情“态度冷漠”？

在本节课中，我们将要学习亲密关系中“态度冷漠”现象背后的心理原因，即“亲密关系恐惧”与“假性独立”。我们将探讨其根源，并提供具体的改善方法。

---

很多人宣称自己是独身主义者，他们认为爱情无法解决温饱问题，与其沉迷不如逃离。更直白地说，他们不相信爱情。

然而，他们的内心世界可能搭建得非常丰富多彩，设想着有一天，真命天子会驾着七彩祥云从天而降。

这种伪装的独身主义者抛弃信仰、开始一段感情后，表现又是怎样的呢？他们会躲避矛盾，想到结婚就头疼，并且总与伴侣保持着一种过于客气的距离感。

在心理学中，这属于缺乏安全感的表现。他们害怕爱情给自己带来伤害，所以总是以冷冰冰、硬邦邦的态度来回应伴侣。

上一节我们介绍了缺乏安全感的一种表现，本节中我们来看看第二个表现：**亲密关系恐惧**。

电视剧《欢乐颂》中的安迪就有亲密关系恐惧症。她不喜欢和人勾肩搭背，闺蜜给她拥抱时她会觉得别扭，男朋友要亲吻她，她会下意识地把对方推开。她的亲密关系恐惧来源于成长经历中的双亲缺位。

她的母亲患有精神病，父亲远走他乡，因此没有人能教会在这种家庭环境下长大的孩子如何与他人进行情感互动。

我们在前面的课程里已经知道，母亲的贴心照顾对于建立安全型依恋关系很重要。这里我们想说一说父亲对不安全感造成的影响。父亲承担的作用，是给孩子树立未来伴侣的良好初印象。

假如一个父亲长期酗酒、暴躁，你去问他的女儿“希望找一个什么样的男朋友？”，她大概率会说：“不要像我爸那样的。”因为女生对异性的认知最初就来自父亲。父亲品行不端或者暴力乖张，女儿就很可能认为男生都是如此，进而对男生保持不安全的心理距离。

除此之外，惨痛的失恋经历同样会使人缺乏对异性的信任。

重点是，原因我们都知道，但成长背景与过往经历就像一道难以逾越的坎，要完全跨过去，难度堪比让李莫愁变成含情脉脉的小白兔。因为李莫愁正是在被陆展元抛弃之后，才立誓要把天下的负心汉全部杀掉。

心理学里有一种**强化理论**，即连续不断出现的某种刺激能强化某个行为。用零食训练猫狗坐下，就是一种反复的强化。而陷入痛苦的回忆，强化的则是“我是一个可怜人”、“我活着太不容易了”的观念。

虽然回忆的事情并不美好，但我们能从自怜中获得快感。这种自怜快感，又会诱导我们陷入下一次痛苦的回忆，所以我们时常无法自拔。根本原因就是使用了错误的强化刺激。

如果不暗示自己可怜，不暗示自己身世悲惨，就可以很大程度上弱化过往经历给自己造成的负面伤害。

当你再次回想起被前任伤害、被父母忽视的时候，不要急于同情自己，而是去强化良性的观念。例如：
*   “我的成长背景很特别，但这完全不影响我向阳生长。”
*   “弱者身在温室，强者长在山崖，而我明显是后者，能在生活中得到更多历练。”

正如个体心理学创始人阿德勒所说：“我们的烦恼和痛苦都不是因为事情的本身，而是因为我们加在这些事情上的观念。”也就是说，我们把自怨自艾的观念抛弃掉，用良性的观念去强化自己，就可以更顺利地把事情往下一步推进。

接受了过去的成因之后，我们来说一说，对于已经成型的冷漠行为，应该采取哪些措施去改善。

心理学又把这种冷漠行为称为**假性独立**。假性独立的人像蜗牛，外表有坚硬的独立外壳，内心却柔软脆弱。例如，他们可以从容自然地去医院挂号打点滴，但一静下来又不可避免地感受到孤独。

再比如，他们和伴侣分开时，客气得像陌生人，联系对方的积极性和次数都很少。大多数时候，他们都不愿意承认自己需要伴侣，还把自己包装成独立自主的样子。

那么，我们就可以根据内外两个方面，构建假性独立的破解方程，一个是“拿”，一个是“给”。

以下是关于“拿”的具体说明。“拿”不是索取，而是寻求情感或行为上的帮助。它能帮助假性独立者建立真正独立的内心核心。

《武林外传》的佟掌柜就有这个明显特征。外敌入侵时，他会对老白大呼“救命”；老白的心思都放在免罪金牌上时，他又会撒娇、倾吐不满。

绝大多数时候，佟掌柜都不会掩饰自己需要老白。更重要的是，这并没有影响他展示自己的独立品质，因为他始终是一个能掌控大局的人，是个非常讨喜的角色。所以，不强装独立，袒露自己需要帮助，是假性独立者需要迈出的第一步。

我们举一些常见的例子帮助大家操作：
*   男生要上班或出差之前，你可以说：“怎么办？我会很想你的。”
*   当你们两地分隔又想念对方时，可以发送相关的表情包，或者带有映射思念含义的新闻、趣事给对方。
*   再比如，男朋友嘴笨，说了某些伤人的话，你也不用假装无所谓，或者转头向闺蜜大吐苦水，而是找一个合适的时机说：“你说了那些话，我听了心里有点难受，希望你能体谅。”

然后我们再说第二点：“给”。

有的人说，假性独立其实就是“爱无能”。所以，“给”就是做出爱的行为。心理学家弗洛姆在《爱的艺术》一书里说道：“爱是一种积极活动，不是消极的情感；是主动的投入的活动，而不是盲目的沉迷的情感；是给予，而不是接受。”

这是否还有点抽象？爱好像是一种说不清道不明的情感，要怎么去表现呢？因此，弗洛姆给出了四种具体可操作的行为：**关心、责任感、尊重和了解**。

**关心**很简单，就是“你吃了吗？”、“冷不冷？”、“昨晚睡得好不好？”。对于安全感强的人来说，语言上的关心看似微不足道，但对假性独立者却很重要。

因为他们对自己独立的要求，导致他们对伴侣的情绪感知很迟钝。说得更具体一些，他们觉得自理能力是必备的（冷了多穿衣，饿了多吃饭），这是没有必要强调的，因而他们也缺乏对伴侣这方面的感知力。在两个人的交流体系中，更是很少出现这些看似无关紧要的“废话”。

除此之外，假性独立者还要把语言关心升级为行为关心。例如，询问对方“冷不冷”之后，回房间拿出一件外套给对方披上，即不止把关心停留在“说得好听”的层面。

再说**责任感**，就是有改善双方关系的主动性。从另一方面看，假性独立者的冷漠是太专注自我的表现，他们倾向于专注自己现在是什么心情、有什么感受，而不是伴侣怎么想。

有一个网友说到，他每次收到伴侣的礼物，内心都毫无波澜，并不觉得对方表达心意有什么实质性作用。有责任感的做法其实是，不管礼物美丑，都能感谢对方的付出，也能做出一些贴心的回应。关键是不时提醒自己：你对感情是要负起责任的，需要采取某些行动来加深感情。

然后我们来说第三个爱的行为：**尊重**。伏尔泰有一句至理名言：“尊重，不一定是接受。”这暗示了你在接受对方的思想、行为习惯、隐私要求之外，还要主动付出某种行动。

例如，你的男朋友也是一个缺乏安全感的人，他常常追问你在哪儿、又和谁在一起。尊重就要求你不仅接受他是没有安全感的人，还要减轻其他事情的负荷，匀出时间给对方回应，并致力于建立双方的信任。

最后一个行为则是**了解**。我们前面说到，假性独立者的情感走向其实很模糊。他们既希望进入更亲密的情感阶段，又害怕负责任。这种感情的不确定性，也包括对伴侣的不确定性。而加深对伴侣的了解，能极大地帮助他们增强对未来的信心。

所以，你可以从以下几个方面入手，充分了解对方：
1.  他的早年生长环境是什么样的？与你所见的现在的他有什么因果联系。
2.  你和他目前处于什么样的恋爱状态？他有没有和你讨论过安全感的问题，并给予你支持。
3.  他的人际交往情况如何？有哪些得体或不合时宜的地方？
4.  他规划了怎样的生命进程？对未来有什么安排？

双方在这些地方有深入了解，或对改善安全感有一致目标，那你们就能得到对方的强大助力。

当你在关心、责任感、尊重和了解四方面都做出一些改变后，你就能很大程度上改善冷漠和假性独立。当然，希望你从追本溯源开始，接受自己不那么顺利的成长背景，改善亲密恐惧症，然后通过坦诚情绪、请求安抚，以及慢慢表现出爱的行为，来让自己真正做到“真独立”。

---

本节课中，我们一起学习了“亲密关系恐惧”和“假性独立”这两种因缺乏安全感而表现出的冷漠态度。我们探讨了其源于成长经历或过往创伤的成因，并介绍了通过“接受过去”、“学会‘拿’（寻求帮助）”和“练习‘给’（践行关心、责任感、尊重与了解）”来逐步改善的方法。改变始于自我觉察与接纳，进而通过具体行动，在关系中建立起真正的安全感与独立。
