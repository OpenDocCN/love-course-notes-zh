# 女神养成系列：第19课：你必须了解的2018年时尚搭配风标

## 概述
在本节课中，我们将要学习2018年的核心时尚趋势。我们将从流行色、关键单品和搭配技巧三个方面进行分解，帮助你理解流行趋势的来源，并学会如何根据自身条件进行选择和搭配，从而穿出优势。

## 流行趋势的来源与选择

上一节我们概述了课程内容，本节中我们来看看流行趋势是如何产生并传播的。

流行趋势的主要源头是法国巴黎的流行色中心。它向全球发布色彩趋势后，各大服装公司和面料商便开始开发相应的流行面料与款式，继而产生时装秀。明星艺人受邀出席秀场并进行穿搭，再通过杂志和时尚达人的传播，最终将流行趋势蔓延至大众。

很多人的困扰在于，发布的流行款式（如2018年宽大的款式）可能不适合自己的身材（例如矮胖体型），导致感觉买不到能穿的衣服。这涉及到流行选择的问题。

大家总是一个困扰接着一个困扰。我们首先应该了解流行趋势到底流行什么。然后，根据自己的体型、样貌、职业场合等特征，进行有选择性的采购和购买。不是所有的流行都适合你。

## 2018年三大流行趋势解读

了解了趋势的来源与选择原则后，我们正式进入2018年时尚风向标的解读。我们将从色彩、单品和搭配三个方面进行讲解。

### 一、 流行色
色彩是时尚最直观的表达。了解年度流行色是把握趋势的第一步。

国际流行色由巴黎的流行色中心发布。这一体系始于上世纪50年代，旨在规范服装行业，避免制造业浪费。发布流行色后，面料商和服装公司会据此开发产品，使得市场色彩呈现统一性。

以下是2018年重要的流行色：

1.  **脂粉色**
    这是一种带有科技感和未来感的电子粉，色调偏冷，有种“脏脏”的感觉。它不属于胭脂红色系，而是空灵色调，会出现在唇膏、服装、发色等多个领域。

2.  **海岸中蓝色**
    这种蓝色提取自夏季海岸线，是金黄色过渡到单宁色的略深蓝色。它对亚洲人的偏橙黄色肤色非常友好，因为蓝色是橙黄色的补色，驾驭度高，预计会成为2018年的主力蓝色。

3.  **紫外光紫**
    这是一种接近蓝色的紫色，而非粉紫或葡萄紫。它传达出独特精巧的感觉，能很好地衬托肤色，显白皙。但需注意，紫色对款式和面料的高级感要求较高。

4.  **番茄红**
    红色是时尚圈的常驻色彩。2018年番茄红的关键在于穿得简单利落，造型越简约，高级感越强。肤色暗黄者需谨慎选择，皮肤白皙者则能很好驾驭。

**肤色搭配建议**：肤色暗黄的同学，建议多选择**蓝色**和**紫外光紫**；皮肤白皙的同学，以上色彩均可尝试。

### 二、 流行单品
色彩是抽象的，而单品是具体的服装载体。以下是2018年春夏值得关注的关键单品。

1.  **截短上装**
    短款上装对腰腹部线条（马甲线、腹肌）要求较高，与全球健身热潮相关。它有利于调整身材比例（通常搭配长款下装），充满叛逆和运动气息。
    *   **款式特点**：多为宽松版型，非紧身。可能有立领、衬衫款、夹克款甚至西装款等多种设计。
    *   **搭配建议**：胸围丰满的女生应选择大领口，避免高领，以防显壮；身材扁平、腰腹线条优美者非常适合。

2.  **马球衫/裙**
    运动风持续盛行，马球衫（Polo衫）以新面貌回归。它不再死板，而是与风衣、连衣裙等款式进行混搭设计，兼顾运动感与高级感。
    *   **搭配趋势**：可与彩色袜子、运动凉鞋搭配，或本身设计成连衣裙样式，适应从年轻到成熟的不同人群。

3.  **工装风**
    工装风格在2018年非常盛行，特点是多口袋、廓形宽松，并融入了刺绣、徽章、明线等装饰，更加前卫时髦。
    *   **搭配建议**：想避免过于男性化（“汉子气”），可搭配性感高跟鞋、女性化手袋和长发造型，形成风格混搭。
    *   **重要提醒**：个子矮或腿不长的人，要特别注意工装腰线的位置，**选择高腰线款式**以优化比例，避免选择低腰哈伦裤型。

### 三、 流行搭配
衣服买得好，不如搭得巧。搭配是组合的艺术，至关重要。

核心原则是：**造型简约，强调高级感**。例如穿着番茄红时，应追求简单利落的剪裁，避免过多细碎元素。

混搭是精髓：例如将**工装**搭配**高跟鞋**，将**运动风马球衫**与**优雅连衣裙**结合。通过不同风格单品的碰撞，创造出既时尚又个性化的造型。

接着，我们解读了四大关键流行色：**脂粉色**、**海岸中蓝色**、**紫外光紫**和**番茄红**，并给出了肤色搭配建议。

然后，我们分析了三个核心流行单品：对身材有要求的**截短上装**、焕新回归的**马球衫/裙**以及风格硬朗的**工装风**，并分别提供了选购和搭配要点。

最后，我们强调了**搭配**的重要性，指出简约造型和风格混搭是打造时尚感的关键。
