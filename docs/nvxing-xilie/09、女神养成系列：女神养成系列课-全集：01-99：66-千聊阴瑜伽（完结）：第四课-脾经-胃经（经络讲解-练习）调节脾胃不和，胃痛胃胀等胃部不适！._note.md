# 阴瑜伽：04：脾经与胃经（经络讲解与练习）🧘‍♀️

在本节课中，我们将学习如何通过阴瑜伽练习来调节脾经与胃经，以改善脾胃不和、胃痛胃胀等不适。课程将结合经络知识的讲解与舒缓的体式练习，帮助初学者理解并实践。

## 课程概述 📖

本节课的核心是理解脾经与胃经在中医中的作用，并通过特定的阴瑜伽体式来疏通这两条经络。我们将从放松冥想开始，逐步进入体式练习，并在过程中穿插讲解经络知识。

---

## 一、冥想与呼吸放松 🧠

首先，我们通过冥想和呼吸让身心平静下来，为后续的体式练习做好准备。

请舒适地盘坐，双手轻轻覆盖在膝盖上。翻转掌心向上，让大拇指与视线方向一致。

伴随吸气，感受脊柱向上延展。收紧腹部，闭上眼睛，调整呼吸。

先进行几组自由的呼吸，让身体彻底放松。

去感受你的身体，让它完全放松下来。

想象我们来到一片碧绿的湖面上。湖面波光粼粼，微风轻拂过我们的面庞。

感受身体无比的放松。将你的嘴角微微上扬，在心底给自己一个淡淡的微笑。

深深地吸气，吸入新鲜的氧气，将这清新的气息吸入心底。感受春风拂面，感受生命的能量随着呼吸进入身体。

随着氧气慢慢进入身体，并随着血液扩散到每一个角落。

缓缓地吐气，将体内所有的浊气、废气缓缓地吐出体外。

感受身体变得越来越清澈。

再次深深地吸气，吸入新鲜的氧气到心底。感受生命的能量在体内慢慢绽放，如同开出朵朵烟花。

缓缓地吐气，释放体内所有的悲伤与不安。

放下所有的不安，感受当下身心的放松。

伴随着这样的深呼吸，感受身体逐步向上飘升。

感受你的身体，感受生命的能量。

## 二、体式练习：半英雄前屈式（Ardha Virasana）🦵

上一节我们通过呼吸放松了身心，本节我们开始体式练习，首先来拉伸大腿前侧的胃经。

将身体侧过来，坐在垫子的一侧。先将臀部靠在垫子的左侧。
1.  左腿伸直靠在垫子左侧，屈右膝，将右脚跟放在臀部外侧，膝盖向外打开。
2.  大小腿尽量呈90度角，向后提拉一下臀部肌肉。
3.  双手放在体前。吸气时，背部向前延展；呼气时，手臂带动身体缓慢向前折叠。
4.  动作需轻柔缓慢，可以用多个呼吸配合完成。
5.  注意打开的角度不要大于90度，以免增加膝盖压力。
6.  身体下方可以垫毯子，以缓解身体的紧张。

我们今天的主题是脾经和胃经。中医常说“脾胃不和”。我们的脾胃究竟是如何“不和”的？

首先了解胃经，它位于大腿的前侧。在做这个半英雄式时，屈膝的那条腿正好拉伸到大腿前侧的胃经。所以请尽量将脊柱向前延展。

保持在这里呼吸。你可以伴随着轻柔的音乐，闭上眼睛，感受身体被伸展的部位，让身心放松下来。

吸气，将身体缓慢推起。然后双腿向前伸直，放松。

## 三、胃经讲解与单腿背部伸展式 🤲

上一节我们拉伸了胃经，现在让我们更深入地了解胃的功能，并进入下一个体式。

现在将双腿并拢，坐在垫子上。
1.  吸气时，手指向天空延展，放松身体。
2.  呼气时，身体缓慢向前折叠。
3.  不需要一次性折叠到极限，可以逐步深入。
4.  身体僵硬的练习者，可以把手放在地板上，无需完全俯身。
5.  始终注意保护身体安全，将意识关注在伸展部位。

胃的主要功能是消化食物，将有用的养分输送给脾，不纯净的成分送到小肠进一步过滤。食物在胃里会停留约4小时才被分解。胃为我们供应生理、精神乃至心灵层面的能量。当你饥饿或过饱时，情绪都会受到影响，这就是精神层面的体现。

因此，我们一定要摄取无污染且适合自身体质的食物和水。饮食对胃气的干扰非常大。建议少喝饮料和含添加剂的水。选择任何功能性食品时，一定要根据身体状况。

不易消化的食物（如泡面、部分海鲜）会在胃中停留更久，增加负担。如果你的胃消化功能不好，应选择易消化、刺激性小的食物。

对于需要应酬喝酒的人，喝酒伤胃也伤肝。可以在喝酒前喝一杯牛奶或酸奶，它们能附着在胃壁上保护胃黏膜，减缓酒精吸收。

足阳明胃经的运行时间是早上7点到9点。因此要在9点前吃完早餐，让胃有食物消化，避免“空转”磨损胃壁。

吸气，缓慢将身体推起，避免激活过多的阳性能量。然后将双腿向前伸直。

## 四、体式串联：下犬式与低弓步 🐕

了解了胃经的知识后，我们通过流动的体式进一步激活身体能量。

屈双膝跪立，双手落在垫子上，来到四脚板凳式。
1.  踮起双脚脚跟，膝盖离地，进入下犬式。
2.  在这里保持稳定，不要抖动，让能量沉淀。
3.  保持几个呼吸后，吸气抬头，将右脚向前一步来到双手之间。
4.  呼气，后脚脚背铺平，保持在这个低弓步。
5.  髋关节向下沉。双脚距离不够可以调整后脚，让它更向后一些。
6.  缓慢地向下压髋，不要提髋。手掌可以落地，放松身体即可。

胃经的运行方向是从上向下。它起于眼球正下方的眼眶缘，沿身体前侧、大腿前侧、小腿前侧，一直运行到第二脚趾外侧的厉兑穴。胃经属胃络脾，与脾的配属关系最为密切。

人体依赖脾和胃的运化来受纳、腐熟水谷（食物），才能将食物消化吸收，化生气血津液，使全身脏腑组织得到营养，维持生命。所以说脾胃是“气血生化之源”。

中医认为，脾胃不和，则气血不足。肾为先天之本，脾胃为后天之本。如果脾胃运化不好，吸收不佳，吃再多补品也无法吸收，身体毒素反而会增多。

吸气，身体慢慢向上抬起，双手轻放在前侧膝盖上。髋关节依然保持下沉。你可以将双手交叠，轻松地保持在这个位置。后侧膝盖疼可以在膝盖下方垫毯子。注意脚踝在膝盖正下方。

人体每餐所需的营养成分是有限的。如果摄入的营养远超过吸收能力，多余的部分就会在体内变成毒素和垃圾。所以吃东西要量力而行。

春夏季节适合排毒、出汗，帮助深层清理。秋冬季节则适合进补。夏天清理垃圾后，冬天进补才能更好吸收。如果体内垃圾很多，可能连基础的营养都吸收不了。

养生需注意：春夏不需大补，可轻补；秋天清补；进入隆冬后，可进行一些大补。

呼气，双手落回脚两侧垫子上。将重心向后移，前腿伸直，脚尖回勾。
1.  吸气，背部向前延展。
2.  呼气，身体放松，缓慢向前向下折叠，保持在这里。

胃经在早上7点到9点运行，此时按摩胃经、吃早饭非常好。如果不吃早饭，胃空转摩擦胃壁，难以修复。脾经运行时（上午9-11点），若腹中空空，脾就会吸收体内的垃圾毒素，对身体不利。所以按时吃饭非常重要。

胃经上有一个非常重要的穴位——**足三里**。它位于膝盖骨外侧凹陷下方约四指宽（三寸）的位置。**足三里**是人体第一保健大穴，刺激它可以增强胃肠蠕动、提高消化酶活力、增进食欲、帮助消化，并能改善心脏功能、提高机体防御能力。

吸气抬头，重心向前移，回到低弓步。双手不动，保持呼吸。

## 五、体式加深：半蛙式与脾经讲解 🐸

现在我们将体式加深，并开始讲解与胃经相辅相成的脾经。

屈后膝，用手抓住后侧脚背。身体可以向后扭转，把髋关节向下压，收紧腹部。可以将掌根落地保持。
1.  抓不住脚背可以使用瑜伽伸展带。
2.  注意髋关节向下压，前膝保持在脚踝正下方。
3.  确保整个身体和大腿呈一条斜线，臀部没有上翘。

呼气，解开手和脚，重心向后移。伸直前腿，脚尖回勾。
1.  吸气，背部向前延展，肩胛骨向后。
2.  呼气，将前脚跟向前蹬。
3.  注意将髋摆正：左髋向前，右髋向后。
4.  下不去不要着急，可以把髋抬起来，手撑在垫子两侧调整髋部。
5.  只是髋在动，将后侧大腿做内旋，让膝盖朝向地面。
6.  能够下来的练习者可以俯身，手抓脚尖。但务必保持髋部摆正。
7.  一定要用手支撑身体，保护安全。手够不到地可以用瑜伽砖垫高。

呼气，将髋向上提，踮起后脚脚尖，膝盖离地，前脚向后收回，回到下犬式。然后抖动双腿，屈膝放松一下。

吸气，将左脚向前来到双手之间，进入另一侧的低弓步。保持呼吸，放松身体，感受伸展。膝盖疼可在膝下垫毯子。

我们之前讲了从上向下运行的足阳明胃经。现在来讲**足太阴脾经**。脾经的运行方向与胃经相反，是**从下向上**运行的。

脾主运化，但它有“四怕”：怕湿、怕冷、怕腻、怕撑。所以我们不要吃生冷、太黏腻、太撑的食物。

古人云：“冰冷绝人种，油炸毁容颜。”冰冷食物吃多影响生殖系统；油炸食物吃多损害容颜，易长痘、皮肤油腻。

“早起用姜做茶送，熬粥火汤面。”早上起来可以先喝杯姜茶。早餐最好喝粥或吃汤面，这样能润滑和保护胃，也让脾有丰富的营养可吸收。

很多人不知道脾脏的功效。脾脏是重要的消化器官和储血器官，大小与心脏相当，位于胃的后方。它负责回收红血球，也是白血球对抗感染时的补给部位。脾脏萃取食物饮料的养分，将其转化为血液和“气”（谷气）输送到全身，再上输到肺。脾脏的气不平衡，整个系统都会不协调。

吸气，身体慢慢向上，双手轻放双膝，髋关节向下沉。

脾经又是一条治疗妇科病的首选经络，主要运行在上午9点到11点。此时人体阳气上升，疏通脾经能很好地平衡阴阳。

脾经有21个穴位，从大脚趾末端的隐白穴开始，沿下肢内侧、内踝前缘向上，经过小腿内侧、大腿内侧，与肝经相交后继续上行，直达胸腹部。

呼气，双手落到垫子上，重心向后移。吸气，脊柱向前延展；呼气，身体向下沉。前腿伸不直可以微屈膝，脚尖回勾即可。

脾主运化，掌管食物的消化、吸收和排泄，将精华转化为气血津液，输送到全身。脾也统血，脾虚可引起血液循环问题，导致各种出血症状或贫血。脾虚还可能引起便秘、咳嗽。健脾的最佳时间也是上午9点到11点。

中医认为“气为血之帅”，血要在脉管中规规矩矩运行，需要“气”的约束，这个气主要就是脾气。如果脾气虚弱，约束不力，就会造成各种内脏出血，非常危险。

脾还有生血功能。贫血常伴有营养不良，与饮食有关。所以一定要按时吃饭，营养均衡。不要为了减肥节食，这会让整个循环系统受损，得不偿失。同样，也不要暴饮暴食。保证每餐所需营养足量但不过量。

## 六、体式结束序列与脾胃关系总结 🔄

在课程的最后，我们通过扭转和放松体式平衡身体，并总结脾胃不和的关键。

吸气抬头，重心向前移，髋关节向下压。屈后膝，左手从背后抓右脚的脚背（抓不住可放腿两侧或让脚跟找臀部）。注意髋关节向下压，膝盖与脚尖方向一致。

保持这个姿势可能会感到疼痛，请尽量放松。胃不好、胃经淤堵的人，这里会感觉特别疼。

脾主四肢肌肉，开窍于口，其华在唇。脾功能正常，则肌肉健壮、四肢有力；反之则食欲不振、肌肉消瘦、四肢无力。唇色淡白或萎黄，是脾虚、贫血的症状。

松开手脚，重心向后移。伸直前腿，吸气延展脊背，呼气将前脚跟慢慢向前蹬。注意髋要摆正。在做这个类似一字马的体式时，不要着急。先把髋摆正（左髋向后，右髋向前，右大腿内旋），让髋关节向下扣。摆正了才能安全地下沉，下不去就支撑住，身体前倾即可。

吸气，将髋关节向上抬，踮起后脚脚尖，膝盖离地，前脚向后收回到下犬式。抖动双腿，屈膝放松。

呼气，双膝跪地，脚背铺平，重心后移，来到婴儿式放松休息。也可以双手放身体两侧，转头向两侧，彻底放松。

了解了脾和胃各自的功效后，我们讲一下为什么“脾胃不和”。

脾是总调度，胃是加工车间。脾主升，胃主降。胃气降，食物才能向下便于消化；脾气升，才能把营养输送给肺，再由肺输送给其他脏腑。所谓的“脾胃不和”，就是胃气不降、脾气不升，没有交合，身体消化吸收就不会好。

脾喜燥恶湿，脾湿就会大便溏泄、经常拉肚子。胃喜润恶燥，吃太多干燥、上火、燥热的食物，就会胃火上炎，导致眼睛赤痛、牙痛。如果胃气不降，就会出现呃逆、呕吐、胃痛等症状。如果脾气不升反而下陷，就会出现少气懒言、久泻、脱肛、胃下垂、子宫脱垂等内脏下垂的问题。

所以，该升的不升，该降的不降，才会有脾胃的一系列问题。一定要注意饮食规律。

## 七、放松与休息术 😴

课程主体部分已经结束，现在让我们彻底放松，进入休息术。

吸气，伸直手臂，身体缓慢向前，来到四脚板凳式。
1.  呼气，双膝向身体两侧打开，双脚大脚趾相对。
2.  重心向后移，可以俯身下来，拉伸大腿内侧的脾经。放松身体即可。

吸气，将身体推起。双脚并拢，双腿向前伸直。

将双手向两侧平举。吸气，将双腿向上抬起90度。
1.  呼气时，双膝倒向右侧，头转向左侧。闭上眼睛，让肩膀贴向地面。
2.  吸气，双腿向上，身体还原。
3.  呼气，双腿倒向左侧，头转向右侧。
4.  吸气，双腿向上。
5.  呼气，蹬直双膝，仰卧下来，进入休息术。

进入休息术前，请盖好毯子，注意保暖。
仰卧在垫子上后，将双肩向外缓慢旋转，轻微扭动身体，找到最舒适的位置，然后完全静止下来。

## 课程总结 📝

本节课中，我们一起学习了：
1.  **胃经**（足阳明胃经）的运行时间、走向和功能，了解了保养胃部的重要性和方法（如按时早餐、按摩足三里）。
2.  **脾经**（足太阴脾经）的运行方向、功能及其与气血运化的密切关系，知道了脾的“四怕”和健脾的最佳时间。
3.  通过**半英雄前屈式**、**单腿背部伸展式**、**低弓步**及其变体等阴瑜伽体式，有针对性地拉伸了脾经和胃经。
4.  理解了“**脾胃不和**”的中医内涵，即胃气不降、脾气不升所导致的一系列消化吸收问题，并认识到规律饮食、选择合适食物的重要性。

希望你能将这些知识融入日常，通过持续的练习和良好的生活习惯，呵护好自己的脾胃健康。
