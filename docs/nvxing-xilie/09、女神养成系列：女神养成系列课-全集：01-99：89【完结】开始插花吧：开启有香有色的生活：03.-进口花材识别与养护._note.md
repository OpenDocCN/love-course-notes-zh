# 女神养成系列：03：进口花材识别与养护 🌸

在本节课中，我们将学习如何识别和养护进口花材，并通过实践一种名为“空气感插花”的新风格，来创作一束独特的花艺作品。我们将了解进口花材的来源、分类方法以及插花的基本形态，最后还会简要回顾欧式插花的历史发展。

---

## 进口花材的定义与来源

上一节我们介绍了基础的花材知识，本节中我们来看看什么是进口花材。

进口花材是指从国外进口的花卉。这些花材主要产自一条特定的纬线区域，这条纬线大致从非洲的肯尼亚、埃塞俄比亚，穿过南美洲的哥伦比亚和厄瓜多尔。这些地区海拔较高，昼夜温差大，非常适合花卉生长，因此能培育出根茎粗壮、品质优良的鲜花。

相比之下，中国最适合种花的地区是云南。但由于全球气候和品种的差异，进口花材在形态和种类上常常与我们常见的花卉不同。

---

## 实践：空气感插花

认识花材最好的方式就是动手实践。本节我们将尝试一种目前在北欧流行的插花风格——空气感插花。这种风格不同于传统饱满的螺旋式花束，它更强调随意性和自然感，每种花材可能只用一支，整体显得轻松而富有呼吸感。

### 插花步骤与使用花材

以下是本次插花练习所使用的核心花材及步骤：

1.  **选择主花材**：我们选用一支绣球作为视觉焦点。虽然绣球不是进口花材，但其饱满的形态非常适合奠定花束基础。
    ```text
    花材：绣球 (非进口)
    作用：团状主花材，奠定基础形态。
    ```

2.  **添加形态花材**：加入一支大飞燕。这种进口花材形态修长、质感高级，能为花束增加线条感和造型感。
    ```text
    花材：大飞燕 (进口)
    作用：线状花材，提升造型与写意感。
    ```

3.  **搭配色彩与层次**：插入一支进口白色芍药，与蓝色系花材形成柔和对比。进口芍药的优势在于全年供应。
    ```text
    花材：进口芍药
    作用：团状花材，丰富色彩与层次。
    ```

4.  **点缀点状花材**：加入洋甘菊。这种一簇簇如小星星的进口花材，能像满天星一样起到点缀作用。
    ```text
    花材：洋甘菊 (进口)
    作用：点状花材，增添灵动与细节。
    ```

5.  **融入叶材与果实**：插入网红叶材“冠纵蕨”增加自然感，并用金丝桃（果实类）进行搭配。
    ```text
    花材：冠纵蕨 (叶材)， 金丝桃 (果实类)
    作用：叶材增强自然写意，果实丰富质感。
    ```

6.  **丰富色彩与调整**：加入进口多头玫瑰（花园玫瑰）和刺芹，进一步丰富色彩。插花时保持花茎较高，留出足够“空气”，并随时调整位置以达到随意、平衡的效果。
    ```text
    花材：进口多头玫瑰， 刺芹
    作用：多头玫瑰丰富花束主体，刺芹增加独特蓝色调。
    ```

空气感插花的秘诀在于**追求随性**，让花束中有足够的空间和呼吸感。

## 花材的分类方式

在插花过程中，我们从不同角度认识了花材。根据特性，花材通常分为以下四类：

*   **花卉类**：即观赏其花朵的部分，常作为主花材，如花园玫瑰。
*   **叶材类**：植物的叶片，如冠纵蕨。它能增加花束的自然感和写意风格。
*   **果实类**：植物的果实或果序，如金丝桃、松果，能增添独特的质感和趣味。
*   **干花类**：干燥后制成的花材，如帝王花。它能长久保存，象征永恒。

此外，根据形态，花材又可分为四种类型，这有助于我们在插花时安排它们的位置：

*   **团状类**：花形饱满，第一时间吸引视觉焦点，如绣球、芍药。
*   **点状类**：花形细小分散，用于点缀和填充，如洋甘菊。
*   **线状类**：形态修长，勾勒线条和造型，如大飞燕。
*   **不规则类**：形态独特，既可作为主花也可作为配花，如红掌、木百合。

## 欧式插花简史

为什么鲜花能让人愉悦？这可以从插花历史中找到线索。欧式插花最早源于古希腊和古埃及文明，最初用于婚礼等场合的装饰。

*   **14-15世纪**：鲜花装饰风格趋向隆重、华丽，常见于大型花园和宫廷。
*   **17-18世纪**：随着资产阶级家庭生活兴起，适合多角度观赏的**圆形插花**开始流行并进入千家万户。
*   **19-20世纪至今**：插花风格出现自然主义倾向。近年来，在北欧等地流行的“空气感插花”，正是在自然主义基础上，更加强调随意、放松和与居室环境的融合。这或许反映了在现代都市生活中，人们愈发渴望与自然生命建立随性、紧密的联系。
