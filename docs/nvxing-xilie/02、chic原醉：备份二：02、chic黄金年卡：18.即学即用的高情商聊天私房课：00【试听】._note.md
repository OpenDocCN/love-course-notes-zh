# 聊天艺术：02：即学即用的高情商聊天私房课

## 概述
在本节课中，我们将要学习高情商聊天术的核心价值与它能带来的具体改变。掌握这项技能，能帮助你在人际交往中更自信、更有效地沟通。

---

一提到聊天，人们常会想起被不善言辞者冒犯的经历。例如，有人在聚会时当众取笑你的穿着。相识不久，便追问你的工资或年龄等隐私问题。遇到这类不擅长聊天的对象，人们往往想避而远之。

同时，许多人自身也缺乏聊天口才，在关键场合无法有效表达。例如，遇到心仪对象时，不知如何上前搭讪，从而错失相识机会。与久未谋面的朋友聚会时，找不到话题，导致场面冷清。

有时，一句无心之言也可能引起他人误解。因此，无论在何种场合，扮演何种角色，会聊天都是一项至关重要的技能。本课程《聊天的艺术》旨在帮助大家掌握这项人际交往中的必备技能。

---

## 高质量聊天术带来的五大改变
首先，我们来看看掌握高质量的聊天术能为你的生活带来哪些积极变化。以下是五种主要改变。

### 1. 开拓异性资源
许多女性抱怨与异性交流困难，不知如何开启对话或互动。这除了害羞，更多是聊天能力不足的表现。掌握优秀的聊天技能后，向心仪对象搭讪、寻找共同话题将变得轻而易举。

例如，在日剧《蜕变研究室》中，讲述了擅长聊天的秘诀。女主角新公司的两位女同事不仅打扮出众，更擅长聊天。她们深受同性欢迎，在联谊会上也总能获得异性的好感。

她们最擅长展现天真好奇的一面。同时，她们总能从他人的话语中找到可赞美之处。这正是异性乐于与她们交谈的原因。

与异性聊天时，除了展示好奇心和赞美对方以获取好感，还可以从对话信息中挖掘对方感兴趣或专业的话题，以此增强沟通动力。

初次见面时，通过训练眼神、动作、语气，并选择恰当话题，有助于打造受欢迎的个人魅力，给对方留下深刻的第一印象。

**核心技巧**：`挖掘话题 + 真诚赞美 + 展现好奇`
这些技巧能帮助你把握各种异性资源，在社交场合中成为吸引异性的焦点。

---

### 2. 结交更多人脉资源
许多人都自认情商不低，说话不易得罪人。但你是否遇到过以下困扰？

有时，出于照顾对方情绪的考虑，你无法大胆提出建议。例如：你的合租室友早晨占用洗手间时间过长；朋友的某些习惯困扰着你；对接的同事工作进度缓慢，拖累了你的工作等。

此时，高情商似乎告诉你要忍耐，以免破坏关系平衡。然而，运用恰当的聊天技巧，你可以合理地提出建议，同时避免让对方难堪。

**核心方法**：`非暴力沟通公式`：陈述事实 + 表达感受 + 提出具体请求
