# 创业资本论：02：成功撩到投资人的秘诀（下）- 讲好故事

## 概述

在本节课中，我们将学习如何向投资人有效地讲述你的创业故事。上一节我们介绍了商业计划书的写作技巧，本节我们将进入实操路演阶段，重点探讨“讲故事”这一核心技能。一个精彩的故事能让你在众多项目中脱颖而出，快速获得投资人的青睐。

## 讲故事的重要性

我们在之前的课程中提到，早期项目的投资人在做决策时，会非常看重创始人的个人能力。讲故事是让投资人更快速、更形象地了解你的创业项目和个人能力的重要方式。

一个生动的例子是昆仑决的创始人姜华。他在一次饭局上讲述了“在北京建立一个世界搏击中心”的故事。饭局结束后，弘泰基金的创始人之一盛希泰便要了他的账户，不到24小时，昆仑决就获得了数百万的天使轮融资。

## 讲好故事的硬核技巧

以下是向投资人讲故事的几个核心步骤。

### 第一步：明确目标听众

在讲故事之前，首先需要确定你要“撩”谁。这样做的主要目的是做到知己知彼，了解对方的投资偏好、投资风格、对项目的把控程度，以及已投项目创始人的反馈。这些信息能帮助你缩小故事听众的范围，使你的讲述更具针对性。

正如投资人在融资前会对创业者进行尽调，我们在接触投资人前，也有必要对目标投资机构进行一轮调查。

**例如**，如果你是一个新能源汽车项目的创始团队，想向该领域的权威投资机构“未来资本”融资。通过调查，你了解到对方偏好投资新能源技术、自动驾驶、智能制造等科技领域，且倾向于技术较成熟的公司，如首汽约车、嘀嗒拼车等。明确了这一点，你讲故事的方向就会更加清晰。

### 第二步：聚焦核心，清晰表达

与人交谈是一门艺术。与投资人沟通时，内容应相对聚焦，用简短的语言表达出产品的独特价值。你需要描述清楚市场现状、获客渠道、与竞品的差异化，以及自身的商业护城河如何构建。

上节课我们提到了阿里巴巴的例子。马云是一位讲故事的高手，事实上，几乎所有商界领袖都深谙此道。那么，如何在面对投资人时，把自己的故事讲得清新脱俗呢？

投资人每天要看几十个项目。想要迅速抓住他们的眼球，从众多项目中脱颖而出，就不能像小学生讲故事那样枯燥乏味。即使你的项目再好，如果表达出来的实力只有三分之一，也是得不偿失的。

### 第三步：科学讲故事指南

我们为同学们总结了一套科学的讲故事指南。

#### 1. 在大环境基础上发挥独特优势

假设你现在想做一个奶茶产品。我们先来看目标市场的现状。行业的发展规律普遍是：从出现“一枝独秀”，到“百花齐放”的竞争，再到更细分领域的进一步角逐。

奶茶行业已然如此：从号称“销量绕地球三圈”的香飘飘，到请周杰伦、桂纶镁代言的优乐美等传统速溶奶茶；再到以统一阿萨姆奶茶、康师傅经典奶茶为代表的瓶装奶茶；接着发展到街头奶茶品牌，如一点点、喜茶、CoCo、鲜果时间等。整个市场越来越热闹，也越来越细分。例如，主打女性市场的奈雪的茶，以文案走红的答案茶等，都找到了自己的独特之路。

因此，所谓“在大环境基础上发挥优势”，就是要求你学会在市场蓬勃发展的背景下，在众多竞争者中，解释自身存在的合理性，并通过这种方式，让自己在投资人眼中更具辨识度，比如独特的品牌特色、主打的垂直赛道等。

在基于一份靠谱的商业计划书前提下，给投资人讲故事时，通过对大环境的解读，也可以映射出你的创业动机和个人魅力。

以奶茶行业为例，虽然该领域还未出现上市公司，但近年来资本追逐的脚步却从未停下。近两年，我国茶饮行业出现了前所未有的融资高峰。我们耳熟能详的网红奶茶，如喜茶、奈雪的茶、答案茶以及鹿角巷，都获得了少则千万、多则上亿的融资。

究其原因，根据美团点评发布的饮品发展报告估算，我国现制奶茶行业的潜在规模已接近1000亿元，基本与中国现磨咖啡市场规模相当。因此，投身奶茶行业是一个很有前景的选择。

#### 2. 阐明做好此事的复杂性

接下来，你需要向投资人解释清楚：既然这么有前景，那么做好这件事的复杂性在哪里？

例如，奶茶行业的入门门槛较低，商业模式也不过于复杂，大致分为运营型和策略型。前者重销量，在提升销量上，有的商家打价格战（如买一送一、第二杯半价），或通过新奇文案和造型吸引顾客。后者则通过线下活动策划吸引流量来提升品牌知名度，类似于星巴克的“第三空间”打法。奈雪的茶和喜茶的店面构造就可圈可点，着重考虑店面平效。喜茶初期以小店形式扩张，但后来意识到环境和品牌对树立消费者忠诚度的重要性，于是开始开设100平米以上的大店及多种主题店（如简约标准店、黑金店、LAB店、PINK店）。奈雪的茶则是第一个将营业面积扩展到200平米以上，配以精美装修，将价格抬到30元以上的品牌。

虽然看起来门槛低、易操作，但想要做好一家奶茶品牌，仍然极具复杂性。例如，产品单一、同质化严重一直是行业痛点。上述商业模式虽简单，但也有发展瓶颈。再者，因门槛低导致的管理者经验匮乏和专业知识缺失，极易造成失败。最后，该行业缺少规则制定者，食品安全和健康问题也是埋在行业中的一颗定时炸弹。

在讲故事时，你应该从以上几点进行自我理解的阐述。

#### 3. 给出解决复杂性问题的方案

这时可以适当借鉴前人的经验。

**例如**，针对产品单一、同质化严重的痛点，喜茶的策略是不停探索研发新品。根据其公众号信息，2018年喜茶共发布48款新品，平均每月两款，包括新产品类型、季节限定款、旧产品升级、口味改良、原材料升级等。强大的新品推出能力，让消费者总能发现新目标，使喜茶保持了长久的新鲜感和吸引力。

此外，喜茶将“饥饿营销”发挥到了极致，几乎一夜爆火。严格来说，限流、限时、限量才是饥饿营销。通过限流效应激发用户的“饥饿”状态，加上产品本身特殊的口味，让喜茶迅速成为有口碑的网红奶茶。

喜茶的走红已成为现象级。那么，在这个行业继续创业，是否就没有机会了？并非如此。

奶茶行业创业门槛低，但也因此导致市场同质化严重、品牌杂乱。“创新”是想要脱颖而出的关键词。喜茶主要做口味创新，而后出现的奈雪的茶则主打20-35岁的白领女性群体，以果茶和鲜萃茶为主，区别于喜茶主打的芝士茶。

相信谈了以上三点后，投资人已对你的专业性和行业认知有了一定认可。接下来，你需要给出进一步的解决方案。

#### 4. 明确未来进场策略

面对已趋于饱和的市场，后来者该如何进场？

这就要求你对自己创业领域的未来发展有明确判断。我国茶饮行业在经历了粉末时代和街头时代后，正向着第三阶段迈进，以喜茶、奈雪的茶为代表的中国新式奶茶将成为市场主导。奶茶饮品市场更新迭代速度很快，单一的健康奶茶饮品已无法满足消费者需求。多元化、复合式的经营方式是未来主流。

就像马云跨界做直播，OPPO跨界联合小黄人做推广，农夫山泉跨界网易云音乐打造乐评瓶一样。至于奶茶行业如何结合，同学们可以参考我们之前的跨界营销相关课程。

## 总结

本节课，我们主要从三个大方向——**大环境分析**、**复杂性阐述**、**解决方案**——拆解了向投资人讲故事的步骤。我们学习了如何定位听众、聚焦表达，并通过科学的故事结构，清晰展示项目的优势、面临的挑战以及你的应对之策。
