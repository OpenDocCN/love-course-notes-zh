# 女神养成系列：01：粉底选择与打底基础

在本节课中，我们将学习如何根据肤质选择正确的粉底类型，并掌握两种核心的打底手法：适用于日常自然光环境的轻薄打底法，以及适用于有辅助光源环境的立体修容打底法。

## 肤质自测与粉底选择

粉底液的选择不应在手背或胳膊上测试，而应在脸上测试，因为粉底最终是用于面部的。我们的皮肤大致分为几种类型：**混油性皮肤**、**混干性皮肤**、**干性皮肤**、**油性皮肤**和**中性皮肤**。

以下是针对不同肤质的粉底类型选择建议：
*   **油性皮肤**：建议选择遮盖力较强的**粉底膏**。
*   **干性皮肤**：建议选择滋润度高的**粉底霜**（包括BB霜、CC霜等）。
*   **混合性及中性皮肤**：建议选择质地轻薄的**粉底液**。

如果你不确定自己的肤质，可以前往专业的化妆品连锁店进行皮肤测试，以获得准确的建议。确定肤质后，选择化妆品会容易得多。生活中80%以上的普通人，包括正在学习的你，最常使用的粉底产品就是粉底液。

## 上妆工具的选择

日常上粉底的工具主要有几种。专业化妆师可能会使用粉底刷，但这并不完全适合日常生活。我个人习惯使用**打底海绵**，它能非常轻薄且均匀地推开粉底。

**核心建议**：如果你的粉底液价格昂贵，最节省用量的方法是**直接用手**。使用海绵或刷子等工具会多消耗约三分之一的粉底液。因此，生活中最好学习用手指打底。

## 立体打底的核心原理

很多朋友打粉底时，会先全脸涂抹，然后在三角区提亮，再在颧骨下方加深，这相当于涂了三层粉底，妆感厚重。

正确的方法是使用两种颜色的粉底：**提亮色**和**均匀色**。将两者混合后涂抹在三角区。这样，在自然肤色（暗度）与三角区提亮色的对比下，我们没有上粉底的部位——即从**颧弓下线**往后至耳垂的区域——就会形成一个自然的阴影。

让我们明确一下位置：
*   **提亮区**：面部三角区。
*   **过渡区**：提亮区与阴影区之间的衔接部分。
*   **自然阴影区**：从颞线（太阳穴附近）开始，经过颧弓下线向后延伸的区域。这部分保留皮肤原本的肤色或使用暗影色。

上一节我们介绍了立体打底的基本分区，本节中我们来看看在不同光源环境下如何具体操作。

## 日光源下的轻薄打底法

日光源（如自然日光、室内顶光）下，粉底非常容易被看出痕迹，因此我们需要追求轻薄、透亮的妆效，而非一味追求白净。

以下是日光源下的打底步骤：
1.  **准备**：取约一粒黄豆大小的提亮色粉底液于手背。
2.  **上妆**：使用打底海绵或手指，以**按压**的手法，从鼻翼、法令纹等瑕疵最明显的位置开始上妆。
3.  **重点**：将粉底**用力按压**进皮肤，而不是擦拭，这样能更好地遮盖瑕疵并贴合肌肤。
4.  **范围控制**：粉底仅覆盖三角区及需要均匀肤色的区域，**颧弓下线**往后的区域完全不留粉底，保留皮肤自然质感。
5.  **效果**：这样打造的皮肤质感通透，瑕疵被柔和遮盖，在日光下看起来非常自然。

## 辅助光源下的立体修容打底法

在辅助光源（如灯光、舞台光）下，我们可以通过明暗对比来塑造更立体的面部轮廓。关键在于**先打暗影，再上均匀底妆**。很多朋友把打阴影放在最后一步是错误的，那样会难以衔接。

以下是辅助光源下的打底步骤：
1.  **第一步：上暗影色**。在以下位置使用暗影色粉底，并采用**拍擦**的手法（轻轻拍打并向外晕染）：
    *   从颞线到颧弓下线的“3”字形区域。
    *   眼窝。
    *   鼻翼两侧。
    *   嘴角两侧（避免嘴角肌肉下垂的视觉感）。
    *   下巴凹陷处（突出下巴尖的受光面）。
2.  **第二步：上均匀色/提亮色**。使用接近肤色的粉底液，以**按压**的手法，覆盖暗影区以外的部分（主要是三角区），并向暗影区自然过渡晕染。
3.  **手法区别**：
    *   打提亮区：手法是 **`按压`**。
    *   打暗影区：手法是 **`拍擦`**。
4.  **效果**：先深后浅的顺序，即使衔接不那么完美，肉眼也难以察觉。最终呈现的效果是面部中央提亮、轮廓自然收缩的立体小脸效果。

## 课程总结

本节课中我们一起学习了：
1.  **根据肤质选粉底**：油皮选粉底膏，干皮选粉底霜，混合/中性皮选粉底液。
2.  **工具选择**：手指最省粉底，海绵适合追求均匀妆效。
3.  **两种核心打底法**：
    *   **日光源轻薄法**：重在轻薄、按压上妆，保留颧弓下线后的自然肤色作为阴影。
    *   **辅助光源修容法**：重在立体，遵循“先暗影，后均匀”的顺序，运用“拍擦”与“按压”不同手法塑造明暗对比。

记住，在自然光下，**笑容是最好的化妆**；在需要精致妆容的场合，巧用明暗对比能让你的妆容更具高级感。
