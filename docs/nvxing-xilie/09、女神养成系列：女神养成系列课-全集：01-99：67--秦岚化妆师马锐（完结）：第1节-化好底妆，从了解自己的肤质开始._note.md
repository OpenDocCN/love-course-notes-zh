# 女神养成系列：01：了解肤质与选择底妆

## 概述
在本节课中，我们将要学习如何通过了解自己的肤质来选择最适合的底妆产品。这是打造完美妆容的第一步，也是最重要的一步。

## 了解你的肤质
上一节我们介绍了课程的目标，本节中我们来看看肤质的重要性。选择保养品或底妆产品时，了解自己的肤质是最重要的特点。底妆产品的作用是帮助我们了解皮肤类型并选择合适的产品。

肌肤类型主要分为几种：干性肌肤、油性肌肤、混合性肌肤以及最完美的中性肌肤。每种肌肤都有其特点。

以下是油性肌肤的特点总结：
*   **油**：皮肤经常出油。
*   **糙**：皮肤粗糙。
*   **黑**：皮肤黝黑、暗黄。
*   **米**：皮肤容易过敏。
*   **痘**：肌肤容易长痘痘。

油性肌肤在选择底妆产品时，重点是如何让底妆看起来清爽、干净、净透。油性肌肤女生画完底妆后容易出现“大油田”和满脸油光的效果，这会让皮肤看起来不好看。

以下是干性肌肤的特点总结：
*   **老**：皮肤易显老。
*   **干**：皮肤干燥。
*   **糙**：皮肤粗糙。
*   **米**：皮肤容易过敏。
*   **痘**：肌肤容易长痘痘。

干性肌肤和油性肌肤都可能出现过敏和痘痘问题，这与雾霾天气、身体反应、卸妆不干净或皮肤代谢物堵塞毛孔有关。干性肌肤化底妆时通常会出现卡粉、干燥、浮粉以及纹路明显的问题，让皮肤看起来不平整光滑。因此，干性肌肤的底妆重点在于打造光滑、水润、轻薄和清透的效果。

以下是混合性肌肤的特点总结：
*   **甘**：部分区域干燥。
*   **油**：T区等部位出油。
*   **腻**：皮肤油腻。
*   **糙**：皮肤粗糙。
*   **痘**：肌肤容易长痘痘。

混合型肌肤通常是T区很油，而脸颊等区域很干燥。T区容易长痘，其他区域则干燥并出现纹路。对于混合性肌肤，保养和底妆选择需要特别技巧，不同部位需采用不同的上妆方式才能达到良好的皮肤质感。

以下是敏感性肌肤的特点总结：
*   **薄**：皮肤薄。
*   **米**：皮肤容易过敏。
*   **红**：有红血丝。
*   **养**：皮肤易发痒。
*   **痘**：肌肤容易长痘痘。

敏感性肌肤是目前很多女生面临的困扰，可能伴有红血丝、长痘、皮肤薄、底妆遮瑕不干净等问题。敏感性肌肤在选择底妆时，反而要更强调遮瑕效果、产品安全性以及使用后皮肤的真实感。底妆需要具备遮瑕力，但不能过于厚重。

这就是我们分享的皮肤类型。听完后，你需要分析自己的皮肤属于油性肌、干性肌、混合性肌肤还是敏感型肌肤。无论你属于哪种类型，在本节课中，你都能学会如何选择最适合自己的底妆产品。

## 底妆产品的类型与选择
上一节我们了解了不同肤质的特点，本节中我们来看看市面上有哪些底妆产品以及如何根据肤质进行选择。

底妆分为很多类型。首先，在上底妆之前，必须使用妆前隔离产品。妆前隔离起到承上启下的作用，可以让皮肤有很好的隔离效果，并使底妆更持久。

以下是妆前隔离的选择方法：
*   如果有红血丝，可以选择**绿色**隔离来中和红血丝，调整肤色。
*   如果皮肤暗沉或暗黄，可以选择**紫色**隔离来恢复和调整皮肤状态。

因此，底妆的第一步是使用隔离，这包括带颜色的隔离。此外，市面上还有无色隔离。无色隔离涂在皮肤上是隐形和透明的，不需要颜色遮瑕，但可以帮助皮肤更润泽。这是在化妆前需要使用妆前隔离的部分。

接下来分享市面上常见的底妆产品及其适合的肌肤类型。

首先是**粉底液**。粉底液的特点是薄和透，比较适合皮肤本身较好或干性肌肤的女生，因为它涂上后非常水润、有光泽感。粉底液也比较适合混合型肌肤。

第二种是**粉底霜**。粉底霜介于粉底膏和粉底液之间，有一定遮瑕度，可以让皮肤有非常好的均匀肤色和匀透效果。混合型肌肤、油性肌肤或过敏型肌肤，粉底霜是非常好的选择。

第三种是**粉底膏**。粉底膏质地非常厚重、浓重，比较适合过敏性肌肤和痘痘肌（即油性肌肤），因为它可以让妆容更持久，遮瑕度更好。

第四种是**气垫BB**。气垫的特点是轻薄、多效合一，可以有防晒、定妆和均匀肤色的效果。气垫有几种用法：如果需要打造轻薄底妆，只需用气垫轻轻遮一层；如果希望皮肤呈现光泽感和匀透造型，可以先使用粉底液或粉底霜打底，再用气垫做局部定妆，这样妆容看起来非常自然和清透。

第五种是**遮瑕膏**。很多女生只有某些部位的皮肤不好，比如有痘印或痘痘。这时可以选择轻薄的粉底液加上遮瑕膏。粉底的作用是均匀肤色，而遮瑕膏是帮助局部修整和调整。因此，可以选择轻薄底妆加上局部遮瑕，让皮肤变得非常干净和浸透。

第六种是**蜜粉**。每个女生家里都应该有蜜粉。蜜粉的效果是帮助底妆更持久，持续8到12个小时。在秋冬季节，一定要使用蜜粉，因为它可以保持皮肤干燥，形成自然屏障，防止风沙浮尘粘附在含有油脂的底妆上。无论任何肤质，秋冬季节蜜粉是必须的。蜜粉也属于底妆的一部分。

第七种是**干湿两用粉饼**。干湿两用粉饼的意思是：如果是油性肌肤，可以干着使用；如果是混合型肌肤或敏感性肌肤，可以蘸一些纯净水喷湿后使用。这样打造出来的底妆质感不同：对油性肌肤主要是控油，对干性肌肤主要是保证肌肤有一些光泽感，让皮肤更平缓。

第八种是**高光和提亮**产品。随着直播和网红时代的到来，底妆不仅要让皮肤看起来干净，更要突出立体感，通过底妆达到微整形的效果。市面上有白色或浅色的产品，可以让突出的位置更加立体；暗色的产品可以帮助修容皮肤外侧，让下轮廓感更明显、更明确。这就是双修的部分。

## 总结与上妆方法
本节课中我们一起学习了如何根据肤质选择底妆产品。现在我们来总结一下如何选择和使用这些产品。

以下是不同肤质的底妆选择建议：
*   **干性肌肤**：可以选择**粉底液**来保持皮肤的清爽属性，达到滋润的效果。
*   **油性肌肤**：要选择**粉底霜**或**粉底膏**，再加上**蜜粉**，以达到控油和持久的效果。
*   **混合性肌肤**：可以选择**粉底霜**、**粉底液**，然后加上**气垫BB**和一些**局部遮瑕**来使用。
*   **敏感性肌肤**：同样可以选择有一定遮瑕度的**粉底膏**或**粉底霜**等底妆产品。

对于上妆方法，也有很多选择。以下是几种常见的上妆工具：
*   **美妆蛋**：通过轻拍的方式上妆。
*   **粉底刷（汤勺型）**：一种非常厉害的上妆工具。
*   **扁平型粉底刷**：另一种可以达到上妆效果的工具。
*   **气垫**：也可以作为上妆工具使用。
