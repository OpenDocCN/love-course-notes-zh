## 呼吸法：左右鼻孔交替呼吸法
上一节我们提到了内在平衡的重要性，本节中我们来看看一种能有效平衡身心的呼吸法。

今天要分享的呼吸法叫做左右鼻孔交替呼吸法，也称为经络调理呼吸法。顾名思义，这种呼吸法通过左鼻孔吸气、右鼻孔呼气，然后右鼻孔吸气、左鼻孔呼气的方式进行。它能够帮助我们平衡左右大脑半球，调节身体的左脉和右脉，从而使整个身体协同工作，带来更深沉的健康与和谐感。

以下是具体的练习步骤：
1.  以简易坐姿坐好，可以交叉盘坐，保持轻松。若采用至善坐，则是右脚跟抵住会阴，左脚跟叠放在上，使身体处于平衡状态。可以在臀部后方垫一块瑜伽砖，抬起骨盆，这样下腰背部会更有力量。
2.  保持坐姿，挺直脊柱向上伸展，打开肩膀并放松下沉。
3.  左手结成智慧手印放在左膝上。右手指尖向上立起，用大拇指按住右鼻翼。
4.  先用嘴巴将气息完全呼出。
5.  然后用左鼻孔吸气。
6.  吸满之后，用右手的小指和无名指按住左鼻孔，同时松开大拇指，用右鼻孔呼气。
7.  呼干净之后，继续用右鼻孔吸气。
8.  然后交换方向，用左鼻孔呼气。

以上就是一次完整的左右鼻孔交替呼吸：**左吸右呼，右吸左呼**。

建议练习3分钟，并逐渐延长时间。在这个练习中，一个常见的错误是呼吸不均匀、不等长。建议通过数数的方式来帮助稳定节奏，例如：吸气时数1、2、3、4，呼气时也数1、2、3、4。通过前几次的调整，稳定到一个固定的速率，然后循环进行练习。

结束练习时，放下手，深深地吸气，将气息吸满至胸腔锁骨。然后屏息（不吸也不呼），收住下腹部，眼球由内向上翻转，凝视眉心。保持这个状态片刻，然后呼气放松。这就是该呼吸法的完整过程。

## 体式序列练习
接下来这套体式序列，对男性和女性都非常有益。因为它针对整个骨盆内部、生殖区域以及身体的性能量循环点，能有效刺激脑垂体，平衡和调节全身的荷尔蒙与内分泌。它的动作编排与传统常见的伸展、扭转体式有很大不同，是一套非常科学的序列。

### 苏菲旋转
首先介绍的体式是苏菲旋转。这个体式能够通过旋转下腹部和骨盆区域，带动能量流动，从而很好地放松腹部，并灵活柔软腰椎。

练习步骤如下：
1.  坐姿，双手打开握住膝盖，肘部伸直，整个背部脊柱处于伸展状态。
2.  进行苏菲旋转：吸气时，躯干带动脊柱向前画圈；呼气时，向后画圈。
3.  吸气向前，呼气向后。注意坐骨要稳定扎根，不要偏离。头部保持平衡，不要随着旋转而甩动。应由胸椎带动整个脊柱进行画圈。
4.  如果感觉身体比较僵硬，记住幅度和范围可以小一点。等身体更加灵活柔软后，再慢慢加大画圈的角度。
5.  做完左侧后，一定要做右侧，这样系统才会平衡。做右侧时，方向相反即可。
6.  结束练习时，深深地吸气，将身体拉回正直的坐姿，挺拔向上。吸气后屏息，收束并凝视眉心。然后呼气放松下来，自然调息。

### 猫牛式
接下来介绍猫牛式，这是一个非常经典且备受喜爱的体式。它能够从脊柱底端运动到顶端，活化整条脊椎和背部，带来非常好的放松效果。

练习步骤如下：
1.  从金刚坐（跪坐）进入。双手放在大腿上，保持脊柱伸直。身体不是前倾或后倾，而是正面直立向上，收腹，下巴微收。
2.  进入四角板凳式：双手在肩膀正下方，双膝分开与髋部同宽。五指张开，压实地面。
3.  吸气时，尾骨向上翘起，腰椎下沉，胸椎、颈椎逐节向上抬起，眼睛看向上方。
4.  呼气时，转动骨盆，尾骨内收，背部向上拱起，头部向下放松。
5.  可以直观地感受到整条脊柱的二十六节椎骨从底端到顶端的运动。腰部不适的朋友在做这个动作时，不需要做到很大的幅度。可以在腰部有损伤的位置慢慢移动，感受一下最大舒适强度在哪里。如果继续向下有不舒服的感觉，就往回收一些。这是与身体对话并做出实际调整的方式。
6.  注意头部始终是在最后才抬起，最后才落下。手臂和腿部保持稳定，只有脊柱在运动。
7.  结束练习时，深深地吸气，保持在猫式（拱背）中。呼气时，重心向后移动，臀部坐向脚跟，前额贴地，双手收回身体两侧，掌心朝上。放松背部，自然呼吸调整。起身时，重心前移，重新回到金刚坐。

### 自我放松的冥想（刺激脑垂体）
下面这个练习是一种自我放松的冥想，能够刺激脑垂体、放松卵巢，同时对眼睛有极佳的益处。由于没有具体名称，我们暂且这样定义它。

练习步骤如下：
1.  从侧面进入，保持一个跪坐的姿势。
2.  左腿向前折叠，左臀坐在左脚跟上。右腿向后伸直，膝盖和脚背贴地。身体正面朝向前方。
3.  如果觉得难以保持平衡，可以用毛巾或毯子垫在臀部下方。双手在身体两侧支撑。
4.  身体重心向下，前额贴地。双手放在身体两侧，掌心朝上。让肩部、背部及全身肌肉放松。
5.  保持自然的呼吸。
6.  眼睛仍然要关注眉心，尽量闭眼去刺激眉心这个点（即脑垂体区域）。脑垂体是全身荷尔蒙的总司令，闭眼从内部刺激这个点，能够重新调节身体内分泌的平衡。
7.  在练习过程中，右侧的髋部要有意识地下压，不要向外翻，保持一定的平衡力量。
8.  做完一侧后，换另一侧练习。

### 喉部与颈部按摩
这个体式有助于整个喉部区域的放松和平衡。

练习步骤如下：
1.  从金刚坐进入。如果脚踝疼痛，可以将毛巾或毯子折成方块，垫在脚踝下方，以减小角度，缓解不适。
2.  保持脊柱伸直。用单只手从喉咙底端开始，有节律地、有节奏地按摩到下颚顶端。
3.  也可以换手进行按摩。
4.  在这个简单的过程中，请闭上眼睛，很好地去感受呼吸，感受被按摩的区域。

### 耳朵按摩
大家都知道耳朵对应了全身的各个穴位，按摩耳朵时，也在对全身部位进行按摩和刺激。同时，它还能刺激整个淋巴系统的循环。

练习步骤如下：
1.  保持金刚坐。如果感到脚痛或腿麻，可以打开双腿抖动放松，缓解压力后再调整回来。在课堂练习中，要注意根据身体感受调整动作，偶尔暂停也没关系。
2.  进行耳朵按摩：五指并拢，用掌根贴在耳朵下部。手掌与耳朵形成一个角度，不要完全贴合。用手掌由下至上打圈，进行上下按摩。
3.  保持呼吸。
4.  结束按摩时，深深地吸气，双手向外向下轻轻拉动耳朵，然后屏息10秒钟（可心中默数3、2、1）。最后深深地呼气放松。

### 躯干扭转
这个体式主要针对下腰区和骨盆的放松，并对排泄系统非常有益处。

练习步骤如下：
1.  十指交叉握拳，置于身体后部。保持手肘伸直，背部直立向上。
2.  吸气时，身体向左扭转；呼气时，向右扭转。
3.  注意头部是随着躯干转动，而不是躯干不动只转头。速度应保持中等偏慢，进行完整、彻底的扭转，不要过快，以免对腰部造成损伤。
4.  结束练习时，深深地吸气，身体回正。屏息并凝视眉心，然后呼气放松下来。

### 分腿仰卧起坐
仰卧起坐大家都很熟悉，需要躺平后运用腹部力量起身。这里的变式是腿部分开的，并且包含一连串的动作。

这个体式的益处在于，它能刺激坐骨神经及其末梢的循环，针对下腹部，并能消除焦虑引起的紧张。

练习步骤如下：
1.  仰卧在垫子上，将双腿尽量分开（无需强迫）。躺平。
2.  双手来到肩膀后方，拇指在肩胛骨附近，四指在前。
3.  先不着急做动作，放松呼吸，感受身体的每一块肌肉。
4.  起身时，运用意念和全身的力量。然后身体前屈，让前额贴地（或靠近地面）。在这个体式中保持一分钟的呼吸。
5.  再次吸气起身，呼气躺回垫子上。
6.  再次起身，这次将躯干转向左腿的方向，保持三次呼吸。然后起身，躺回垫子。
7.  再次起身，将躯干转向右侧，低头向下，保持三次呼吸。然后起身，躺回垫子。
8.  最后，双腿收回，双手掌心朝上，调整放松。

以下是常见的错误点及调整方法：
*   腿部分得越开，核心力量越难收紧。需要在过程中找到适合自己发力的分腿位置。
*   起身主要依靠核心力量。如果实在无法起身，可以借助手部力量辅助。
*   如果腰部不适或脊柱僵硬，在前屈时无法让前额贴地，可以保持在自己能承受的幅度，或使用瑜伽砖、毯子来支撑前额。
*   转向左侧或右侧时，要确保整个躯干有扭转过来，向下时同样可以使用辅具帮忙。
*   同样的体式，不同的人可以做到不同的程度。必须了解自己身体当前的能力，并在此基础上练习，无需一步到位，可以逐渐进步。

### 手肘和膝盖行走
这个体式充满趣味性，名为手肘和膝盖行走。

练习步骤如下：
1.  从金刚坐开始，身体前倾，让手肘和膝盖落地。双手可以放在肩膀附近。
2.  双脚尽量靠近臀部，如果做不到也没关系，保持核心收紧的状态。
3.  面部朝前，眼睛向前看。
4.  开始“行走”：用手肘和膝盖配合，交替向前移动。
5.  这个动作有助于循环系统和心脏系统，并能平衡体内的微量元素。
6.  回来时，可以回到婴儿式放松调息。

### 莲花布（行走）
瑜伽坐姿中有一个姿势叫莲花盘，即双盘。接下来这个动作与双盘有关，叫做莲花布（行走）。

双盘坐姿如下：
1.  双腿向前伸直。先弯曲一侧腿，翻转脚踝，尽量让脚底朝上，然后将脚放在对侧大腿的根部。
2.  弯曲另一侧腿，同样翻转脚踝，放在另一条大腿的根部。
3.  调整身体，保持平衡。这就是标准的双盘。

如果无法做到双盘，可以保持自然坐姿，或尝试单盘（只完成一侧的盘腿）。如果盘腿时膝盖高高翘起，建议尽量不要这样，因为会使盆腔区域的能量和血液循环受阻。应尽量让膝盖低于髋部。

在双盘（或简易盘坐）之后：
1.  双手放在膝盖前方的地面，确保能很好地发力。
2.  身体前倾。吸气，呼气时收腹，利用核心力量向前轻微甩动身体，有一种前后弹动的感觉。
3.  莲花布的行走能很好地帮助释放丹田能量，缓解头痛，同时放松盆腔内部的性器官。即使无法完全双盘，在简易盘坐中也可以尝试练习。
4.  关键是骨盆坐骨向下扎根的力量，这有助于调整坐骨神经和盆腔的律动。

### 背部平板行走
平板支撑大家都很熟悉。背部平板则是双手放在臀部后方撑地。

练习步骤如下：
1.  双手放在臀部后方撑地。
2.  吸气时，收腹夹臀，抬起身体，使身体从侧面看呈一条直线。眼睛朝前看。
3.  呼气时，身体下落。
4.  很多练习者可能会觉得困难，只能抬起一点，并且容易耸肩。因此要运用腰腹核心力量。如果力量不够，就接纳自己目前能做到的程度，例如吸气时只能抬到一定高度，那就保持在这个高度，但肩部要下沉，远离耳朵。
5.  呼气落下。这就是静态的背部平板。

“行走”则是在此基础上加入脚步动作：
1.  在平板姿势中，配合呼吸，弯曲膝盖，交替向前移动脚步。
2.  在这个过程中，核心力量与身体的配合更加重要，身体不能塌陷。
3.  可以尝试进入这个动态练习。无需有受挫情绪，这需要时间和练习。

## 冥想：释放情绪压力
今天分享的冥想能够释放过去的情绪压力，带来内在平和，非常有助于释放压力。

练习步骤如下：
1.  手印：双手五指指尖相触，但五指张开呈伞状，置于胸腔前方，手不接触身体。保持肩部放松下沉。
2.  眼睛：睁开十分之一，凝视鼻尖。也可以先闭上眼睛内观眉心，或者感觉透过闭上的眼睛专注在鼻尖。如果觉得难，简单地闭上眼睛即可。
3.  呼吸法：采用**5秒吸气、5秒屏息、5秒呼气**的节奏。屏息即不吸也不呼。最重要的是保持吸气、屏息、呼气的时间比例为 **1:1:1**。
4.  开始练习：准备好手印和眼神。先用嘴巴吐气，将腹部清空。然后闭上嘴巴，用鼻子吸气，心中默数1、2、3、4、5。屏息，默数1、2、3、4、5。呼气，默数1、2、3、4、5。如此循环。
5.  结束练习时，深深地吸气，提升气息能量。屏息，收束并凝视眉心。去感受手指尖的按压感，以及手心中的能量。然后呼气，长长地呼出。
6.  不要着急，保持静默一分钟，静静地与自己在一起。去感受呼吸、感受能量、感受心脏周围的磁场状态。
