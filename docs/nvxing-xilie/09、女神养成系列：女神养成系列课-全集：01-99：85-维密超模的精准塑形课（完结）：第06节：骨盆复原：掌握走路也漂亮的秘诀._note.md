# 女神养成系列：06：骨盆复原：掌握走路也漂亮的秘诀 🚶‍♀️💫

在本节课中，我们将学习一系列针对骨盆区域的瑜伽体式。骨盆周围的肌肉无力会导致臀部松垮、下肢线条不佳。通过增强臀部、腿部及核心肌群的力量，我们可以帮助骨盆回到正确位置，从而塑造更挺翘的臀部和更显修长的腿部线条。

## 动作一：桥式肩倒立 🌉

桥式肩倒立能有效改善腹部和臀部力量。首先，我们通过一系列过渡体式来帮助身体找到正确的发力感。

以下是过渡体式“臀桥”的练习步骤：

1.  仰卧，双手手掌压向地面，肩膀摆正，双脚踩稳地面。
2.  吸气准备，吐气时臀部发力向上推高身体。
3.  抬升幅度不需特别高，使肩膀、臀部、膝盖形成一条斜直线。
4.  肋骨向内回收，通过脚后跟下压来激活臀部向上发力。
5.  吸气时身体向下放落，吐气时再次向上推起。注意下放时，肋骨与臀部同时下落，腰部保持稳定，核心收紧。

找到发力感后，可以尝试在臀桥位置进行踏步练习：

1.  在臀桥姿势中保持稳定，吐气时微微抬起一侧脚，离地约一个A4纸的厚度。
2.  吐气时将脚踩下的同时，将臀部进一步推高。
3.  双脚依次交替踏步，可缓慢加快速度。
4.  每次脚后跟踩下时，都感受臀部更多的向上发力，同时踏步幅度越来越小。

这个练习能激活腹部内外斜肌以稳定骨盆，并增强臀部发力感。

完成过渡练习后，正式进入桥式肩倒立：

1.  自然仰卧在垫子上，确保身体呈一条直线。
2.  屈腿，尝试让脚后跟尽可能靠近臀部。可以伸直手臂丈量，使指尖接近或快要接近脚后跟。
3.  肩膀两侧贴实地面，眼睛始终看向天空方向，头部不左右摆动。
4.  吸气稳定，吐气时臀部微微抬起。
5.  找到足够空间后，将肩膀两侧向中间移动，感觉后背肩胛骨相互靠近。
6.  尝试手臂伸直，掌心相对，然后交叉握在一起。
7.  当肩膀、头部和足底都能稳定支撑后，根据自身柔韧性调整幅度。吸气，吐气时仍可继续上抬臀部。
8.  保持骨盆微微回收，肋骨内收，感觉腹部启动。
9.  吸气稳定，吐气时打开双手，臀部落回地板。

## 动作二：棒式小组合 🦵

上一节我们学习了激活身体正面的桥式，本节我们来看看侧面的练习。棒式小组合能更好地激活腿外展功能，激活臀部上侧肌肉，对维持骨盆稳定非常有益。

以下是基础侧卧棒式的练习步骤：

1.  侧卧，用下侧手臂枕于头下，找到舒适姿势。
2.  确保整个身体在侧面呈一条直线，腰下侧保持悬空。这种悬空感能帮助腰腹收紧，脊柱处于中立位。
3.  双腿膝盖微微弯曲，确保脚后跟与坐骨形成一条直线。想象身体和脚后跟靠在后方一堵墙上。
4.  吸气稳定，吐气时加强收紧腰腹部，同时慢慢打开膝盖向外侧。此时应感觉到臀部上侧发力带动腿部。
5.  脚尖勾回，脚后跟保持对齐并微微悬空呈小八字。这能使脚后跟与臀部处在连接的直线上，发力更精准。
6.  吸气时膝盖下落，吐气时再次推高。上侧手臂可轻扶地面以保持平衡。

接下来是改良棒式，通过增加动作幅度来提高强度：

1.  轻轻抬高双腿。
2.  吸气时，用膝盖轻轻触碰在一起，腰侧仍保持悬空，臀部稳定。
3.  吐气时，脚后跟触碰在一起，膝盖分开。
4.  整个过程想象后背和臀部靠在后方墙上，没有过分向后翻转，以确保锻炼精准。

练习完成后，进入侧卧拉伸体式：

1.  下侧膝盖保持弯曲，上侧腿慢慢伸直并抬高。
2.  用同侧手的食指和中指抓住大脚趾。
3.  吸气稳定身体，吐气时慢慢伸直腿部，将脚后跟蹬向天空方向，以此拉伸整个腿部内侧。
4.  稳定3到5个呼吸后，松手，弯曲上侧腿部，舒服地倒向地面。保持脊柱在一条直线上，头部摆正。
5.  旋转身体向后侧扭转，以放松臀部上侧刚刚锻炼到的位置。
6.  3到5个呼吸后，换另一侧进行相同练习。

## 动作三：虎式 🐯

接下来我们进入一个模仿动物姿态的体式。虎式能有效锻炼臀部和核心的稳定性。

以下是虎式的练习步骤：

1.  双手支撑地面，双膝位于臀部正下方，双手位于肩膀正下方，脊柱自然弯曲。
2.  一侧腿向后伸展，前脚掌点地。
3.  吐气，收紧腰腹以稳定身体，臀部发力将腿向上拉起。注意脚后跟不需特别高，微微高于臀部即可，整个腰腹仍需维持稳定。
4.  开始动态练习：吸气时弓背低头，尝试用膝盖去接近下颚、鼻尖或额头；吐气时回到刚才的位置。

如果在此难以找到臀部发力感和核心稳定感，可通过“三角猫”过渡体式来建立：

1.  一侧脚尖勾回。
2.  吸气准备，吐气时腰腹轻轻往回收紧，维持脊柱稳定，同时臀部发力抬起同侧膝盖向上，直到找到臀部发力感。
3.  吸气下落，吐气抬起。重复练习。
4.  完成后，收回臀部向后，以婴儿式放松休息。

## 动作四：幻椅式与高弓步 🪑🏹

现在我们从地面练习过渡到站立练习。幻椅式能培养下肢力线，帮助臀腿同时回到稳定状态，从而稳定骨盆。

以下是幻椅式的练习步骤：

1.  身体向前，回到站立姿势，保持山式站立稳定。
2.  吐气，臀部向后坐，双手经前向上指向头顶方向。感受臀腿收紧，肋骨向内关闭。
3.  如果肩膀打开感到不适，可稍稍放低手臂至舒适位置，选择合适幅度。
4.  找到标准幻椅式后，依次抬起一侧脚后跟，保持头顶位置没有任何移动，然后交换脚后跟。通过此方式感受足底力量。
5.  如果能力允许，可以尝试双腿同时抬起，这对核心挑战更大。可以一点一点慢慢过渡：吸气下落，吐气抬起，也可尝试在此停留几秒钟。
6.  舒服地放松落下后，站立起身，调息休息。

完成幻椅式后，我们进入力量感更强的高弓步。高弓步能显著增加臀腿力量，对稳定骨盆非常有益。

以下是高弓步的练习步骤：

1.  找到标准高弓步的站姿：双腿自然分开，与臀部同宽。前腿膝盖位于脚踝正上方。
2.  稳定后，上半身轻轻立起，双手放在身体两侧，使脚后跟、臀部和头顶形成一条斜直线。感受前腿的臀部和大腿在稳定身体。
3.  吸气，双手经前向上举起，进入高弓步。整个过程保持骨盆稳定，没有翻转，腰腹收紧，肋骨内收。
4.  吸气，再次向下，回到刚才手扶地的位置。
5.  配合呼吸重复动作：吐气下蹲，吸气抬起。重复3到5次后放松落下。
6.  后腿膝盖放松，小腿和脚面铺平地板，进入低弓步进行拉伸。
7.  前侧腿可轻松向外移动，直到肩膀和手臂位于腿部内侧，选择合适幅度对臀腿部进行拉伸放松。
8.  换另一侧进行相同练习。注意在整个过程中，身体脊柱需维持中正，转换动作时尽量让骨盆保持在中立位，不发生偏转。
9.  最后以婴儿式放松。
