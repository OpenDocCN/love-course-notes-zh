# 明星美妆术：09：短发造型打造教程

## 概述
在本节课中，我们将学习如何为短发打造两款时尚造型。短发造型的关键在于掌握轮廓、层次与支撑度，我们将通过具体的步骤和产品使用，帮助你从“不会打理”变为“造型高手”。

---

## 当前流行短发趋势分析
上一节我们介绍了课程主题，本节中我们来看看当前流行的短发趋势。了解趋势有助于我们选择并打造适合自己的造型。

以下是几种流行的短发款式：

1.  **欧美超短发**：风格超帅气，带有雌雄莫辨的中性美感。代表明星有焦俊艳等。
2.  **齐刘海妹妹头**：这是减龄必备发型。发尾略带卷度并打造出空气感，可以避免死板。加入些许层次设计还能修饰脸型。
3.  **直线微卷发尾**：发尾剪成一条直线并带有微卷，很有女人味。袁姗姗是这种发型的代表。
4.  **狗啃式刘海**：本季度非常流行，带有二次元风格，时髦又减龄。吴昕、郭采洁常以此造型亮相。
5.  **丸子头**：持续流行的发型，能有效拉长脸型。本季度趋势是短发也能梳出好看的丸子头，具有逆龄效果。

## 造型前分析：模特的发型问题
了解了流行趋势后，我们以一位模特为例，分析其发型存在的问题，以便对症下药进行改造。

模特发型现状：
*   **发质**：偏软、偏黄。
*   **问题**：整体没有型，侧面看年龄感重。缺乏层次感，头发下垂，后部发量偏多，前部扁平，导致整体显得厚重、没有精神。刘海也参差不齐。

许多女生都有类似的发型，虽然洗头方便，但难以打造出帅气或酷感的造型。接下来，我们将为她打造两款造型。

## 造型一：凌乱自然感短发
这款造型的目标是打造英姿飒爽又减龄的效果，带有一点狗啃式齐刘海的随性感。

### 核心原理与准备
打造短发造型，必须理解一个核心结构逻辑：**上高下低，两边收缩**。即头顶要蓬松有高度，下半部分及两侧头发要收敛，这样才能让发型立体、有层次，并修饰脸型（尤其是圆脸）。

准备工作：你需要准备**蓬松水**、**发泥**和**发胶**。

### 分步打造教程
以下是具体的操作步骤：

1.  **使用蓬松水**：选择一款蓬松水（冷蓬松或热蓬松均可）。将其喷在发根处，并用手指按摩头皮，使其融合。然后用吹风机热风轻吹，增加头发支撑度与摩擦度。公式表示为：`蓬松水 + 按摩 + 热风 = 基础支撑度`。
2.  **涂抹发泥**：取适量发泥在掌心搓匀。**关键手法**：用手指由内向外（从发根到发梢）涂抹，确保每根头发，尤其是发根都沾到发泥。这能为头发提供持久的哑光支撑，避免油腻感。代码式描述手法：`for hair in all_hairs: apply_mud_from_root_to_tip()`。
3.  **搓揉发根**：将沾有发泥的手指插入发根，轻轻搓揉。这一步能进一步增强头皮与头发间的摩擦与支撑力。
4.  **塑造轮廓与形状**：
    *   **处理刘海**：采用“三七分”或“八字刘海”来拉长脸部比例并增加空气感。手法是边拧转边按压，然后自然垂下。
    *   **处理两侧**：将两侧头发轻轻压扁收紧，避免让脸型变宽。
    *   **处理头顶**：轻轻揉搓头顶发根，将其提起，创造饱满的头顶轮廓。
    *   **处理后脑勺**：通过揉搓和梳理，让后脑勺的线条饱满清晰。
5.  **定型**：使用发胶进行最终定型。喷洒时，**由下往上**喷，同时可用于轻轻捏住发束帮助定型。这样能保持造型的持久性与空气感。

完成以上步骤后，你会得到一个轮廓清晰、头顶蓬松、两侧收敛、带有自然凌乱感的帅气短发造型。

## 造型二：干练利落背头
如果你不擅长复杂造型，这款两分钟就能完成的背头造型非常适合你，它能让任何短发女生迅速变得酷感十足。

### 核心原理与准备
这款造型的核心是将头发全部向后梳理，打造紧致、光洁的线条感，突出面部轮廓的立体与干练。

准备工作：你需要准备**尖尾梳**、**啫喱膏**、**强力定型啫喱（如钢丝啫喱）**、**发夹**和**发胶**。

### 分步打造教程
让我们开始打造这款利落背头：

1.  **分区**：用尖尾梳划分发缝。对于圆脸，建议采用斜上方的“三七分”，能使轮廓更清晰。代码表示：`part_hair(angle="diagonal_37")`。
2.  **涂抹啫喱膏**：将啫喱膏均匀涂抹在全部头发上，使头发呈现湿润的质感，便于向后梳理。
3.  **使用强力定型啫喱**：将钢丝啫喱**只涂抹在头发表层**，目的是将头发牢牢固定，压出光洁感。切记不要涂到头皮深层。
4.  **梳理造型**：用鬃毛梳（内长外短型）将头发紧密地向后梳理，打造出清晰、利落的线条感。
5.  **初步固定**：用发夹将侧边及后部的头发暂时固定住。
6.  **处理刘海与头顶**：同样用啫喱和梳子将前额刘海及头顶头发梳理出支撑度和立体感，并向后固定。
7.  **强力定型**：在已经定位好的部位大量喷洒发胶。然后用吹风机的**冷风**档，由前向后吹干，进行最终加固。

完成后的造型非常帅气、干练，耳后清晰的线条和整体的紧致感能极大地提升气质。

## 总结
本节课中我们一起学习了两款短发造型的打造方法：
1.  **凌乱自然感短发**：关键在于使用蓬松水和发泥，遵循“上高下低，两边收缩”的结构，通过搓揉发根和塑造轮廓来打造空气感与立体感。
2.  **干练利落背头**：核心在于使用啫喱产品将头发全部向后梳理固定，通过清晰的线条和光洁的表面展现强势与利落。

掌握这两招，你就能轻松应对日常与特殊场合的短发造型需求。如果你不是短发，这些技巧同样适用于为男性打造发型。
