# 恋爱知识星球：07：媚娘野生小动物园 第四课《蓄水期应该做什么》：蓄水期3

在本节课中，我们将要学习蓄水期阶段，不同背景的大学生应如何规划自己的精力与行动。我们将重点探讨“985/211”学生与“野生”学生的不同发展路径，并深入讲解情商培养、社交选择、朋友筛选以及如何应对原生家庭影响等核心课题。课程旨在帮助你建立清晰的自我认知和行动策略，为未来的人生打下坚实基础。

## 蓄水期不同人群的精力分配

上一节我们介绍了蓄水期的核心概念，本节中我们来看看不同背景的学生在蓄水期应如何分配精力。

对于“985/211”高校的学生，你们的核心任务是**专注于学业**。你们的“主要经营项目”是学习专业知识、准备升学（考研或读博）。不建议在大学期间花费大量时间和精力去尝试做生意。你们的“财富”积累将在后续阶段通过专业能力实现。

对于普通大学（“野生”）的学生，情况则有所不同。如果学历在就业市场上不占优势，则需要有“第一桶金”的意识，可以考虑在大学期间从小生意起步。这不是因为羡慕别人，而是现实所迫，是“逼出来”的路径。你们的“钱在后面”，但需要更早开始行动。

## 大学期间可尝试的小生意思路

以下是几位同学在大学期间成功实践的小生意案例，可以为大家提供一些思路。

*   **美甲创业路径**：一位在省会读大学的女生，在寝室为同学提供美甲和手足护理服务。她通过Instagram等平台关注最新款式，价格比校外便宜，生意很好。她通过给室友送水果、小礼物维护了良好的人际关系。毕业后，她利用支付宝小额贷款在省会开了一家美甲店，初期客户来自同学，成功转型为店主，月收入远超普通打工。
*   **技能变现还债**：一位美术院校的学生被同学欺骗背负网贷。她通过学习文眉技术，以低于市场的价格（如1500元/人）为同学服务，并为介绍人提供返佣（如300元/人），两个月内还清了债务并有盈余。
*   **人肉代购起步**：一位同学从事韩国人肉代购，通过提前收款、购买廉价机票、住廉价床位的方式控制成本。积累了客户信任后，她开设了淘宝店，并最终发展成一家小公司。
*   **数码服务变现**：一位同学在大学期间通过售卖二手苹果手机、提供贴膜和越狱服务赚钱。他提到，当时学校勤工俭学8元/小时，而贴膜一次可赚50元，越狱一次可赚100元（与柜台老板分成后）。他在获得入党名额后便停止了勤工俭学。

**重要提醒**：大学期间**不要**进行风险无法预估的大投资，尤其**不要借钱创业**。股票、期货、P2P等投机类项目**一定不要碰**，否则极易血本无归。

## 理解并接受原生家庭

你已经成年，必须理解并接受关于原生家庭的现实。

如果你的家庭资源优越，**不要产生“躺靠”的惰性**。家庭的现金流并不完全由你或父母掌控，它受社会行业和政策影响。你的努力是为了让自己飞得更高更远，并确保在家庭可能出现变故时，你依然能凭借真本事立足。公式可以理解为：**个人能力 = 抵御风险的基石**。

如果你的家庭无法提供资源，**最不应该做的就是自怨自艾或埋怨**。埋怨改变不了现状，也不会引来“伯乐”。所有能被伯乐赏识的人，自身都已具备一定能力。你能完全掌控的，是**为自己设定目标，并坚定执行直到产生成效**。请按照之前课程教授的方法，列出有序计划并认真执行。

关于地域差异，例如小地方的同学羡慕上海同学的环境。上海同学可能因“面子”问题不愿从事基层工作（如贴膜、卖烤串），起步薪资可能不高。而上海本地那些厉害的“野生”同学，往往有精明能干的父母作为榜样。因此，无需盲目羡慕，关键在于认清自身路径并坚决行动。

## 情商的定义与练习方法

很多人不知道情商是什么以及如何练习。我们可以将其分为初级、中级、高级来理解。

*   **初级情商**：目标是让身边人与你相处时感到开心。核心是**避免说或做让他人不舒服的话和事**。
*   **中级情商**：目标是让身边的人愿意主动围绕在你身边，使你成为话题领袖（KOL）。
*   **高级情商**：目标是能控制人心，你说的话大家愿意立刻听从并执行。这常见于优秀的团队领导者，他们能识别并组合团队成员的优缺点。

在恋爱关系中运用情商，要注意：**不要反复向伴侣强调自身的短板**（如“我不好看”）。这不会让对方降低预期，反而会强化其负面印象。正确的做法是**私下迅速改进短板**，在交往中展现自信。

## 蓄水期需避免的低情商行为

在大学及恋爱中，有些低情商行为一定要避免。

*   **不要做挑拨离间、勾心斗角等无意义的事**，这纯粹是浪费时间。
*   **恋爱中避免“下头”行为**：“下头”指让人瞬间失去兴致。例如，在新恋情中详细诉说前任因出轨而分手，这毫无益处且可能破坏现有关系。代码逻辑类似于：`if (提及负面旧事): 当前关系.兴致 -= 1`。
*   **不要向伴侣索要空洞的保证**（如“保证每年生日都陪我”），这也很“下头”。
*   **被欺负时不要做软柿子**。要敢于反击，一次退让可能招致持续的欺负。
*   **注意场合着装与言行**。例如，工作场合应穿着职业装，以示专业态度。
*   **面对流言蜚语，最高境界是“不理”**。将精力用于自我提升（学习、护肤、健身），而非与流言纠缠。记住：**不要让低层次的人影响你的步伐**。
*   **做人要适当低调**。优秀时不要嘚瑟，避免不必要的攻击。推荐的做法是：**平时低调，关键时刻适当高调展示，然后继续回归低调**。

## 区分有意义的社交与无意义的社交

人脉不是万能的，核心还是**自身的专业本领**。如果事情没做好，再强的人脉也无用。

社交活动必须带有目的性，区分如下：

*   **无意义的社交**：没有明确目标，随波逐流地参加各种聚会，只为“好玩”或“有空”，让时间变得毫无价值。
*   **有意义的社交**：带有学习目的。例如，向局中某位事业成功的姐姐请教经验，或观察学习他人的优点。公式是：**社交价值 = 是否获得学习或成长**。

有目标的人对时间的使用非常吝啬和谨慎。他们的目的是**主动构建对自己有益的朋友圈**，而非被动地被各种局构建。选择朋友的标准是：**彼此能否促进对方积极向上**。

## 朋友的选择与断舍离

在18-25岁的蓄水期，要对朋友进行筛选。

*   **可以删除的朋友**：但凡让你感到不适、浪费你精力、你从TA身上学不到任何东西的朋友。例如：总是哭诉但不解决问题、永远沉溺于失败感情、总是拉你进行无意义娱乐、总是免费消耗你的精力解决问题。
*   **对付“心机”朋友**：一旦发现对方有心机迹象，哪怕是对别人，也应远离。因为当你的利益与TA冲突时，心机很可能会用在你身上。
*   **高情商处理删除后的尴尬**：如果见面被问起，可以推说“系统问题”或“不知道”，并当场加回，过段时间再删。核心是**表面保持和谐，内心坚定远离**。
*   **接受朋友的自然分流**：随着个人成长和阶级变化，朋友间因共同话题、消费水平的差异而疏远是自然规律。无论是你被抛下，还是你抛下别人，都应管理好情绪，不必过度纠结。**阻碍你发展的人，就像埋在身边的地雷**。
*   **关于借钱**：**不要借钱给任何人**，也不要轻易让别人知道你有钱。可以自己进行高消费，但别人问起，可以轻描淡写地带过。
