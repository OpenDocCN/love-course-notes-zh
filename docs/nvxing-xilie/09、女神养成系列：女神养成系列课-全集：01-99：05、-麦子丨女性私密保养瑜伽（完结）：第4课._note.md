## 矫正不良站姿与常见问题
上一节我们概述了课程目标，本节中我们来看看具体的不良体态及其成因。

许多人在站立时存在以下问题：
*   双腿并非自然伸直，而是小腿外翻，形成类似O型腿的形态。
*   脚掌外翻，导致鞋外侧磨损更严重，大腿外侧肌肉异常紧张。
*   整个髋关节区域活动受限，仿佛被“锁住”。

此外，日常生活中常见的姿态问题包括：
*   含胸、圆肩、颈椎前引（即脖子向前伸）。
*   长时间低头看手机，导致颈后部肌肉紧张。

对于宝妈而言，抱孩子时因手臂和肩部力量不足，常会不自觉地拱背、耸肩。当孩子体重增加时，还可能通过挺肚子、骨盆前倾的方式来借力，这会对身体造成二次伤害。

## 问题改善思路与辅助工具使用
要改善上述问题，核心是加强薄弱肌群的力量。我们需要：
1.  练习**腹直肌**（后续课程会涉及）。
2.  锻炼**肩后侧**的力量。

本节课将重点加强肩颈腰背的练习，改善驼背和脊柱弯曲的状态。我们还会使用辅助工具来放松紧张的筋膜。

### 使用花生球放松脊柱
以下是使用花生球进行放松的步骤：

**放松颈椎与上背部（靠墙进行）**
1.  背对墙站立，将花生球置于颈椎位置。
2.  屈膝，利用身体上下移动来滚动球体，按摩颈后肌肉。
3.  头部尽量向后靠，下巴微收。这个动作能缓解颈椎前引带来的肌肉紧张。

**放松胸椎与中背部（仰卧进行）**
1.  仰卧在垫子上，将花生球置于上背部（胸椎区域）。
2.  双手环抱胸前，屈膝，臀部抬离地面，前后滚动身体，让球按摩脊柱两侧。
3.  如果遇到特别紧张或疼痛的点，可以停留在该处，保持片刻，配合深呼吸。

**注意事项**
*   花生球主要用于放松脊柱两侧的肌肉和筋膜。
*   单独的筋膜球则用于按摩肩胛骨周围的空隙。
*   如果存在**颈椎或腰椎间盘突出**，请避免直接使用花生球挤压脊柱，建议改用瑜伽轮进行更安全的伸展。

## 使用瑜伽轮进行安全伸展
对于有椎间盘突出隐患的练习者，瑜伽轮是更安全的选择。它可以提供支撑，避免脊柱受到直接挤压。

**瑜伽轮后弯练习（打开胸腔）**
1.  坐在瑜伽轮前，缓慢向后躺下，让瑜伽轮支撑上背部。
2.  躺下前，先将双肩向外旋转，将颈后肌肉向下捋顺，避免肌肉堆积在颈部。
3.  头部放松落在轮上或垫子上，下巴微抬。
4.  根据自身情况，可以逐步将轮子向下移动，以加深后弯幅度，但始终以舒适为准。

**前屈与中立体式**
*   **前屈**：身体向前折叠的体式，如坐立前屈。练习时需保持脊柱延展，从髋部折叠。
*   **后弯**：身体向后仰的体式，如瑜伽轮上的练习。
*   **中立**：脊柱保持垂直伸展的体式，如直角坐姿。这类体式对腰椎颈椎问题较为安全。

**关键原则**
*   椎间盘突出的练习者，应多做**脊柱延展**的动作，避免在屈曲或后弯时给椎间盘施加压力。
*   例如，在猫式中，拱背（脊柱屈曲）比塌腰（脊柱后弯）更安全。

## 骨盆调整与收髋练习
骨盆宽大，尤其是产后骨盆未能很好恢复，是常见问题。以下练习有助于收紧髋部。

**战士一式变体**
1.  山式站立，右脚向后撤一大步。
2.  吸气，双臂上举；呼气，屈左膝，下沉髋部。
3.  将左髋向后拉，右髋向前推，使骨盆摆正。
4.  感受双腿向中线夹紧的力量，这有助于收紧髋部。保持5个呼吸后换边。

**加强侧伸展式**
1.  双脚大大分开，脚尖朝前。
2.  从髋部折叠身体，双手落地或扶砖。
3.  呼气，右手带动身体向右侧扭转，将右手放在右脚内侧。
4.  吸气延展脊柱，呼气加深扭转，将胸腔转向右侧。保持呼吸，感受髋部内侧的收紧。换边练习。

**螺旋式**
1.  从加强侧伸展式进入，右手撑地。
2.  吸气，左臂向上打开，带动身体向右螺旋式扭转。
3.  这是一个强烈的收髋体式，在此保持15个呼吸。换边练习。

## 坐姿收髋与开肩序列
完成站姿练习后，我们进入坐姿序列，进一步巩固收髋效果并放松肩颈。

**坐姿扭转前屈**
1.  简易坐姿，将右脚放在左臀外侧。
2.  吸气延展脊柱，呼气身体向前折叠。
3.  腰椎间盘突出者，保持脊柱延展即可，无需过度下压。保持15个呼吸，换边。

**牛面式手臂（开肩练习）**
1.  简易坐姿，臀部下方可垫砖以保持脊柱直立。
2.  吸气，双臂侧平举；呼气，右臂在上，左臂在下，于背后相扣。
3.  保持肩膀下沉，胸腔打开。停留10-15个呼吸。
4.  可加入身体向对侧倾斜的变体，加深拉伸。换边练习。

## 核心强化与骨盆稳定训练
骨盆前倾常与腹部核心无力相关。以下方法能安全有效地强化核心，稳定骨盆。

**波速球平衡训练**
1.  坐在波速球中央，双手抱膝，脚尖点地。
2.  收紧腹部，脊柱向上延展，寻找平衡。
3.  此过程会自动激活核心肌群来稳定身体，并能帮助你感知骨盆是否水平。
4.  能力提升后，可尝试将脚尖离地，增加难度。

**普拉提球卷腹**
1.  仰卧，将小球夹在脚踝间。
2.  双臂上举，肩膀微抬离地面。
3.  呼气时，收紧腹部，双手和双腿同时向上抬起，手去够球。
4.  吸气时，有控制地落下，但腿不触地。
5.  重复20-25次为一组，完成3组。注意发力时呼气，肩膀始终抬起。

**燃脂搏击组合（针对腹斜肌与核心）**
1.  **基础旋转**：双脚打开，双手握拳置于下颌两侧。核心收紧，转动髋部带动身体向左右旋转。左右各25次为一组。
2.  **侧向步结合出拳**：在基础旋转上加入侧向移动步伐，同时配合左右直拳。锻炼核心与腿部力量。
3.  **深蹲组合**：侧移一步后接一个深蹲，站起后换边。全程保持核心收紧，上半身稳定。
