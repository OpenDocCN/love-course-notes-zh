# 瑜伽与产后修复：006：子宫保养

## 概述
在本节课中，我们将学习如何通过特定的瑜伽练习来保养子宫，加强骨盆区域的血液循环，并预防产后可能出现的相关问题。课程将包含坐姿、仰卧及站立体式，适合产后妈妈在安全的前提下进行练习。

---

## 第一部分：准备与吉祥坐姿

在开始练习前，我们需要准备一些辅具：一把带靠背的椅子、一条毛毯、两块瑜伽砖以及一个高度约为8至11厘米的枕头（可用两个枕头叠放代替）。

首先，我们从一个简单的盘坐姿势开始，这有助于伸展髋部并加速骨盆区域的血液循环。

以下是吉祥坐姿的练习步骤：
1.  选择舒适的盘坐姿势，可将一侧脚踝置于另一侧小腿中部。
2.  若感觉背部无法挺直或骨盆后倾，可将折叠的毛毯垫于臀部下方。
3.  调整呼吸，吸气时腹部收紧，胸口上提；呼气时气体从鼻子均匀呼出。
4.  双手食指交叉，掌心向前推出，同时肩胛骨向后收，感受背部的伸展。
5.  保持脊柱向上延展，进行5至8次均匀呼吸。若感觉重心不稳，可靠墙练习。

这个坐姿有助于放松身心，为后续练习做好准备。

---

## 第二部分：仰卧束角式放松

上一节我们介绍了帮助放松髋部的坐姿，本节中我们来看看一个能充分放松骨盆和子宫的仰卧体式。

以下是仰卧束角式的练习步骤：
1.  坐立，双脚脚心相对，双膝自然打开。若膝盖无法下沉，可在膝下各垫一块瑜伽砖。
2.  将准备好的高枕置于身后（靠近骶骨位置，避免腰部架空），缓慢向后仰卧，让头、背完全得到支撑。
3.  双臂自然放于身体两侧，掌心向上。保持下巴微抬，不要过度内收。
4.  在这个体式中完全放松，感受骨盆区域的舒展。保持5至8次均匀呼吸。
5.  结束时，双膝并拢，双脚踩地，身体转向右侧卧，再用手推地起身。

这个体式能有效放松骨盆和横膈膜，对缓解泌尿系统问题也有帮助。

## 第三部分：站立体式——树式

通过之前的放松练习，我们为骨盆区域创造了空间。现在，我们将通过站立体式来学习如何稳定骨盆，这是子宫保养的核心。

首先练习树式，它能帮助建立平衡感，并找到会阴上提的感觉。

以下是树式的练习步骤：
1.  山式站立，双脚与髋同宽。可将椅子放于身旁作为支撑。
2.  重心移向左脚，吸气时屈右膝，将右脚掌放于左大腿内侧、膝盖或小腿内侧。
3.  保持支撑腿脚趾有上提感，左腿微微内旋，右髋外展。感受从脚底到头顶的向上延展力。
4.  视线凝视前方固定点以保持平衡。收紧腹部，避免塌腰翘臀，重点感受会阴区域的上提。
5.  保持3至5次均匀呼吸后，呼气，右脚缓慢落地。换另一侧重复练习。

这个练习能加强核心力量，对于纠正因孕期重心前移导致的骨盆前倾有积极作用。

---

## 第四部分：站立体式——战士二式及其变体

掌握了基础的平衡体式后，我们将进行更有挑战性的战士二式练习，它能更强效地加强骨盆稳定性。

以下是战士二式及其变体的练习步骤：
**战士二式：**
1.  双脚大大分开，距离大于一条腿长。右脚尖转向右侧，身体和肚脐仍朝前。
2.  收紧臀部，呼气时屈右膝，使右大腿与地面平行，小腿垂直地面（膝盖不超过脚尖）。可手扶椅背辅助。
3.  双臂可向两侧平举或扶住椅子。后方腿伸直，膝盖骨上提，脚掌内外缘均匀压地。
4.  保持会阴上提，胸腔展开，进行3至5次均匀呼吸。然后伸直右腿，双脚内外八字收回。

**侧角伸展式（战士二式变体）：**
1.  从战士二式进入，右手落于右脚内侧地面（或瑜伽砖上），左臂向上伸展。
2.  可扶住前方的椅背以加深伸展并保持平衡。同样保持会阴上提，脊柱延展。
3.  保持3至5次呼吸后，吸气，用手推地，带动身体回正。随后换另一侧重复以上两个体式。

练习中需注意腿部力线的正确，避免膝盖内扣或外翻，以确保练习安全有效。

## 第五部分：摊尸式放松

在完成一系列站立体式后，身体需要充分的休息与整合。摊尸式放松能让身体在支撑下彻底放松，这对于产后恢复至关重要。

以下是摊尸式放松的练习步骤：
1.  将高枕纵向放于垫面，缓慢侧卧，然后仰卧，让整个背部、头部完全枕在靠枕上。
2.  双脚自然分开，脚尖微微外八即可，无需过度打开。可在颈下垫毛巾增加舒适度。
3.  双臂放于身体两侧，掌心向上。放松全身肌肉，尤其放松腰部，让其完全贴合靠枕。
4.  保持深长而均匀的呼吸，让身体进入深度放松状态。剖腹产妈妈可在此保持15-20分钟。
5.  结束时，先屈膝，脚踩地，身体转向右侧卧，再缓慢用手推地坐起。

这个体式能有效缓解紧张情绪，促进精力恢复，特别适合产后恶露排净前的阶段练习。

## 总结
本节课我们一起学习了子宫保养的系列瑜伽练习。我们通过吉祥坐、仰卧束角式等开髋体式促进了骨盆区域的血液循环；通过树式、战士二式等站立体式，学习了如何建立力量并上提会阴，从根本上稳定骨盆、养护子宫。课程最后以摊尸式放松结束，确保身体得到充分修复。请根据自身产后阶段（如42天内以放松为主，42天后可加强力量练习）选择合适的体式规律练习。
