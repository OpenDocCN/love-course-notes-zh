# 《两性幸福》课程：第22集：爱的回报与共同成长

在本节课中，我们将探讨关于付出、等待回报以及伴侣共同成长的核心课题。我们将通过一个寓意深刻的故事，理解爱的不同境界，并学习如何在两性关系中实现协调与共同进步。

## 一则关于付出的寓言

生活总会给你答案，但不会马上把一切都告诉你。

一个旅行者在一条大河旁看到一位婆婆，正在为渡水而发愁。已经精疲力竭的旅行者用尽浑身气力，帮助婆婆渡过了河。结果过河之后，婆婆什么也没说，就匆匆地走了。

旅行者很懊恼，她觉得是否很不值得耗尽气力去帮助婆婆，因为她连“谢谢”两个字都没有得到。哪知道几个小时后，就在她累得寸步难行的时候，一个年轻人追上了她。年轻人说：“谢谢你帮了我的祖母。祖母嘱咐我带些东西来，说你用得着。”说完，年轻人拿出了干粮，并把胯下的马也送给了她。

不必急着要生活给予你所有的答案。有时候，你要拿出耐心等等。即便你向天空或大海呼喊，也要等一会儿才会听见那绵长的回音。也就是说，生活总会给你答案，但不会马上把一切都告诉你。

## 人生的三重境界

上一节我们看到了耐心等待回报的故事，本节中我们来看看付出与回报在人生中的不同层次。

当我们的人生有三重境界：
*   **第一层是“要求就会得到”**：那是**索取的爱**。
*   **第二层是“付出才会有回报”**：这个叫做**交换的爱**。
*   **第三层境界是“付出不求回报”**：反而会得到真正重大的回报，这叫做**无条件的爱**。

当我们所付出的，只是我们认为应该做的，也许一切都会变得不一样。世界上唯一不贫穷的就是爱。爱可以解决我们人生当中所遇到的一切问题，爱是这个世界上最伟大的力量。

我们用尽人生当中的每一天，用爱去做连接，连接生命的美好。把爱给予自己，给予身边的家人、朋友，给予世界，给予万事万物。也许，一切都会变得不同。

## 伴侣的共同成长：两人三足跑

理解了爱的境界后，我们来看看如何将这种爱应用于两性关系中，实现共同成长。

男人和女人都需要成长。伴侣的共同成长，最初就像“两人三足跑”，两个人的腿绑在一起。刚开始有点别扭，互相配合、配合，就协调一致了。如果没有磨合的耐心，随意解开了两个人的连接，结果就是：要么一个人跑太快了，再也连不上了；要么跑太快的人被落在后面的各种“妖蛾子”（麻烦）给拽了回去，再也跑不了了。

裁判说，单个跑不符合规定，跑得快也拿不到金牌。成长有很多种方式。以前，男人通常在艰苦创业中成长，女人常因负责家庭事务的原因无法共同成长。断开了链接的女人，在家几年后，落后的不是一星半点，男人和女人的维度就不一样了，和老婆总是话不投机，严重时便向往红颜知己。

其实男人也可以选择更直接的成长路径，不是在艰苦创业中反复经历失败（“成功之母”），时间宝贵，要做的事情很多，争取直接迎接成功本身。女人更加需要成长，你的维度低，孩子的维度就低；你不尊敬伴侣，孩子就鄙视甚至敌视父亲；你没有成就，孩子就没有榜样；你心理不平和，家庭气氛就紧张，然后家里所有人都容易生病，亲人之间关系都不好，儿女婚姻也不好。看，多重要。

男人女人都需要成长，从“两人三足跑”开始，好好沟通，一二一地迈步，慢慢地彼此协调了，两人跑也可以变得分外轻松。当你聚精会神，上天就会成人之所愿。
