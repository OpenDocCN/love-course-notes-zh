# 冥想瑜伽入门：01：课程概述与准备

在本节课中，我们将要学习现代女性冥想瑜伽的基本概念，并了解开始练习前需要做的准备工作。冥想瑜伽结合了冥想的心智训练与瑜伽的身体练习，旨在帮助练习者达到身心平衡。

## 课程核心概念

冥想瑜伽的核心在于**联结呼吸、身体与意识**。其基础公式可概括为：

**平静的内心 = 专注的呼吸 + 放松的身体姿态**

## 课前准备事项

在开始正式练习前，你需要做好以下准备。以下是创建一个适宜练习环境的关键步骤：

1.  **选择安静空间**：找一个不易被打扰的安静角落。
2.  **准备瑜伽垫**：使用专业的瑜伽垫或柔软的毯子。
3.  **穿着舒适衣物**：穿着宽松、透气、有弹性的衣服。
4.  **设定时间**：建议每次练习持续15至30分钟。
5.  **关闭电子设备**：将手机调至静音或勿扰模式，减少干扰。

## 基础呼吸法

上一节我们介绍了环境准备，本节中我们来看看练习的基石——呼吸。正确的呼吸是连接身体与意识的桥梁。

以下是初学者应掌握的一种基础呼吸法：

*   **腹式呼吸**：吸气时，感受腹部自然隆起；呼气时，感受腹部缓缓内收。尝试让呼吸深长而均匀。

## 首个冥想姿势

了解了呼吸方法后，我们可以尝试结合简单的身体姿势。这是一个适合所有人的基础冥想坐姿。

*   **简易坐**：双腿自然交叉盘坐于垫上，背部挺直，双手轻放于膝盖或大腿上。

## 课程总结

本节课中我们一起学习了冥想瑜伽的入门知识。我们明确了其核心是身心联结，完成了练习前的环境与物品准备，掌握了基础的腹式呼吸法，并尝试了简易坐姿。请记住，初学阶段的关键在于感受过程，而非追求完美姿态。

在接下来的课程中，我们将逐步深入，学习更多的瑜伽体式与冥想技巧。
