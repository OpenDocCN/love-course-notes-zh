# 商业管理：02：基业长青：缔造长盛不衰的企业

在本节课中，我们将要学习《基业长青》这本书的核心思想。我们将探讨什么样的公司能被称为“基业长青”，它们为何能历经风雨而屹立不倒，以及如何借鉴其经验来缔造或管理一家卓越的组织。

## 概述

《基业长青》是一本经典的商业著作，它研究了那些经受住时间考验、长期保持卓越业绩的公司。本书旨在揭示这些“高瞻远瞩公司”的共同特质，为企业家和管理者提供打造持久成功组织的蓝图。

---

## 第一重点：何为基业长青的公司？

上一节我们介绍了课程概述，本节中我们来看看什么样的公司能被称为“基业长青”，它们与普通公司有何本质区别。

所谓“基业长青”的公司，是指那些**长时间保持业绩优秀**的公司。根据统计数据，中国企业的平均寿命仅约3.9年，而基业长青的公司经营历史往往接近甚至超过百年。它们不仅跨越了艰难的初创期，更能历经多个经济周期，在繁荣时发展壮大，在衰退时保全自身。

这些公司不仅是“老字号”，更是“常胜军”。在漫长的经营史中，它们始终保持卓越的经营业绩，在行业内乃至全球范围内拥有广泛影响力，其管理理念和方法常被奉为圭臬。

如果把经营企业比作联赛，这些公司就是历史悠久且实力强劲的“老牌劲旅”。作者的团队历时6年，从顶级商业杂志的排行榜中，也只筛选出了18家这样的公司，如通用电气、惠普、波音、默克制药等。

一个有趣的反常识发现是：多数基业长青的公司**没有普遍意义上的“魅力型领导”**。与大众印象中那些光芒四射、充满感召力的企业家不同，这些公司的领袖大多谦逊、低调、专业，甚至不为公众所知。

例如，世界500强企业3M公司，其产品遍布生活各个角落。然而，从1902年成立到本书写作的1994年，其历任CEO中很少有人被商业媒体广泛报道。其中任职时间最长的威廉·麦克奈特，在公司史中仅被描述为“轻声细语、彬彬有礼”。

相比之下，许多我们耳熟能详的明星企业家，其公司成立时间相对较短。例如，阿里巴巴成立于1999年，小米在2019年才进入世界500强榜单。这揭示了一个核心观点：**伟大企业的核心创造，往往是公司本身——其制度、文化与价值追求**。神奇的产品和长期的繁荣，只是这个伟大组织的副产品。

以惠普公司为例，其创始人在创业初期就坚持打造鼓励创新的工程文化，而非盲目追求利润。这种“工程师乐园”的文化，最终孕育了打印机、电脑等划时代的产品，其理念也成为许多创新公司的模板。

**总结而言**，基业长青的公司不仅是活得久、赚钱多，更在于它们创造并传承了一套能适应不同时代、甚至能改变行业规则的组织体系。

## 第二重点：基业长青的驱动力是什么？

上一节我们探讨了基业长青公司的定义与特点，本节中我们来看看它们能够长盛不衰的根本原因是什么。

很多人认为，企业能长期发展，核心原因就是**盈利能力强**。这固然重要，但《基业长青》的研究指出，更深层的原因是这些公司拥有**比利润更高的追求**。以利益为驱动或许能创造一个好公司，但很难成就一个伟大的、持久的企业。

利益导向的企业倾向于追逐短期热点以实现利润最大化，但这往往会限制长远视野和创新可能。而基业长青的企业，往往由一个**超越利润的核心理念**所驱动。在这个理念下，它们敢于进行长期投入，甚至从事暂时不盈利但符合理念的事业。

例如，默克制药曾投入巨资研发治疗“河盲症”的特效药，却发现目标市场的政府和民众根本无力购买。最终，默克决定免费提供药品。当被问及原因时，高管的回答很简单：“默克的事业是保存和改善生命。” 这源于公司创立之初就确立的传统：“药品旨在治病救人，利润随之而来。”

**核心理念公式可以概括为：`核心理念驱动 -> 长期坚持与创新 -> 伟大产品与声誉 -> 利润（副产品）`**

当然，拥有崇高追求并不意味着忽视收入。利润如同空气、食物和水，是生存的必需品。大部分基业长青的企业在初创期都曾为生存而忙碌，但它们将利润视为**实现目标的手段，而非目标本身**。

惠普公司在创立初期，为了支付电费和维持运转，也曾生产过保龄球道指示器、马桶自动冲水器等与后来高科技定位“不搭边”的产品。正是这些“妥协”支撑公司度过了最艰难的时期，直到接到迪士尼的大订单，才逐步迈向高科技产业。

需要注意的是，**并不存在一个“绝对正确”的核心理念**。有的公司（如惠普）核心是“关心员工”，有的（如3M）是“创新”，有的（如索尼）是“大胆冒险”。它们的共同点在于：**都笃信自己的理念并一以贯之**。关键在于，这个理念是否能有效引导和激励每一位成员。

**总结而言**，基业长青的秘诀在于拥有一个超越利润的、坚定的核心理念，并将利润作为实现理念的工具，而非终极目标。

## 第三重点：如何缔造基业长青的公司？

上一节我们分析了基业长青的内在驱动力，本节中我们将具体探讨，纵观历史，要如何实践才能缔造一家这样的公司。

通过对多家卓越公司的研究，作者总结出四个关键方法：

以下是缔造基业长青公司的四个核心方法：

1.  **制定胆大包天的目标**
    一个伟大的公司需要一个能激发全员斗志的宏伟目标。所谓“胆大包天的目标”，是**远超公司当前能力和规模**，甚至看似不可思议的挑战。例如，20世纪50年代，主营军用飞机的波音公司，毅然决定进军民用喷气客机市场，挑战当时主流的螺旋桨飞机。他们投入巨资，最终研发出波音707，一举引领民航进入喷气时代，并成为行业领导者。这样的目标如同灯塔，能统一思想，集中力量，推动公司实现跨越式增长。

2.  **自行培养经理人**
    我们之前提到，基业长青公司多由内部培养的领导者掌舵。在18家公司总计1700年的历史中，仅有4位CEO来自外部。内部培养的领导者高度认同企业文化，能保证核心理念的传承，且拥有良好的人际网络，能减少管理阻力。例如，通用电气的传奇CEO杰克·韦尔奇，就是从公司内部历经长达数年的严格筛选和培养后脱颖而出的。他的成功，是几代领导者理念传承与延续的结果。

3.  **塑造教派般的文化**
    这里“教派”意指**高度一致且深入人心的文化**。这种文化能将核心理念转化为每位员工的思维和行为准则。例如，迪士尼要求所有员工（“演员”）以“表演”的心态工作，将顾客视为“贵宾”。通过严格的筛选和培训，迪士尼塑造了独一无二的欢乐氛围。这种强烈的文化认同感能形成精英意识，大幅降低沟通与执行成本，让企业在各个层面达成共识。

4.  **不断进化**
    市场环境瞬息万变，一套方法无法永远适用。基业长青的公司懂得围绕不变的核心理念，**灵活调整经营策略、方向甚至产品线**。例如，3M公司最初是一家失败的矿业公司，在濒临破产时转型研制砂纸，并意外发明了防水砂纸获得成功。此后，公司确立了“鼓励创新、快速试错”的理念，不断进入新领域，最终成长为横跨多个行业的多元化巨头。进化思维要求公司勇于试错，快速迭代，保留适应市场的经验。

**代码式总结这四大方法：**
```plaintext
function buildEnduringCompany() {
    const coreIdeology = getCoreIdeology(); // 确立核心理念
    setBigHairyAudaciousGoal(); // 设定胆大包天目标
    cultivateLeadersInternally(); // 内部培养领导者
    buildCultLikeCulture(); // 塑造教派般文化
    while (companyExists) {
        evolveAndAdapt(); // 持续进化与适应
    }
}
```

## 总结

本节课中我们一起学习了《基业长青》的核心内容。

首先，我们明确了**基业长青公司的定义**：它们是那些能长期经营并始终保持卓越业绩的组织，其最伟大的创造往往是公司本身及其文化制度，而非某个具体产品或魅力领袖。

其次，我们探讨了其**长盛不衰的根本原因**：它们拥有超越利润的核心理念，并将利润视为实现理念的手段。关键在于对理念的笃信与坚持，而非理念本身的内容。

最后，我们学习了**缔造此类公司的四个实践方法**：设定胆大包天的目标、内部培养经理人、塑造高度一致的企业文化，以及具备不断进化的能力。

虽然本书主要面向企业经营者，但其思想对我们每个人都有借鉴意义。无论是在日常工作中，还是在未来可能走上管理岗位时，我们都可以尝试用“基业长青”的视角去思考：如何建立持久的价值？如何凝聚团队？如何在不确性中灵活应变？避免短视，追求长期价值，这或许是本书带给每位读者最宝贵的启示。
