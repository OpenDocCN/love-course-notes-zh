# 撒娇魅惑术：01：暧昧期识别与约会准备

在本节课中，我们将学习如何识别与男神是否处于暧昧期，以及如何为一次成功的约会晚餐做好前期准备。这是施展撒娇技巧、提升关系亲密度的重要基础。

## 识别暧昧期信号

上一节我们介绍了课程的整体目标，本节中我们来看看如何判断你和他是否处于可以施展技巧的暧昧期。以下是判断暧昧期的四个关键信号：

1.  **时常偷看与留意**：当男生想和你发展关系时，他会时常偷看你，留意你的小举动。他在你面前可能显得沉默或逞强，但在你朋友面前却会神气活现，目的是让你的朋友在你面前提起他。

2.  **放大自身存在感**：当你在现场时，即使他在和别人说话，也会故意提高声量、放大动作，企图引起你的注意。但他不会直接对你多说，可能只是展示一个帅气的侧面。

3.  **使用暗示性言语**：处于暧昧中的男人，如果想更进一步，会使用半赏识、半承诺、半甜蜜的暗示性言语。例如：“其实你也挺不错的，就是太矮了一点，如果你找不到男朋友，我也可以勉强接受的哦。”或“放假了，你不用太想我哦。”

4.  **进行试探性肢体接触**：他会尝试做一些情侣间常见的小动作进行试探，例如轻轻摸你的头、弹你的额头、走路时借机贴着你的肩膀、搂腰、牵手、捏脸颊等。通常，人们对暧昧对象的肢体接触容许度会更高。

根据以上表现，如果你确定和他处在暧昧期，就可以使用接下来提到的技巧。

## 约会地点与时间的选择

确定了暧昧关系后，选择合适的约会地点和时间至关重要，它们本身就能传递出不同的信号。

*   **地点选择**：地点本身带有暧昧属性。晚餐时间比午餐时间更暧昧，中式餐馆比西式餐馆更暧昧，包房比卡座更暧昧。因此，如果你想推进关系，应选择**中餐厅的包房共进晚餐**；如果你想保持庄重，则应选择**咖啡厅的卡座共进午餐**。

*   **赴约时间策略**：赴约时间也是一门学问。对于直男或喜欢姐姐型的男生，可以稍许迟到，因为他们对女生有较高的包容度。但对于其他类型的男生，建议比约定时间提前15到30分钟到达。

提前到达后，在约定地点等待对方。当对方准时或迟到到达时，你便拥有了撒娇拉近距离的机会。

**示例一：对方准时**
> 男生看到你先到，可能会惊讶或愧疚地问：“哎，怎么来那么早啊？”
> 你可以一边用手捋一下头发，一边微笑着歪头，轻声说：“因为我不想让你等太久啊。”说的时候要自然，不经意却又认真地看着他的眼睛。

**示例二：天气寒冷时**
> 你提前到达后，先在室内等候。估摸他快到了，主动联系他，确认后到室外等他。当他赶来时，你可以缩缩脖子，俏皮地说：“哎呦，好冷啊。”
> 他会表示抱歉，并问：“等了很久了吗？”
> 无论实际等了多久，你都统一回答：“十来分钟而已，也没有等很久。”
> 听到这里，他通常会再次道歉。此时，你可以笑嘻嘻地看着他说：“你帮我买一杯热咖啡，让我热热手就最好了。”

这里的核心技巧是：将等待时间模糊为“十来分钟”。这个时长既不会短到无法撒娇，也不会长到让对方负罪感过重，恰到好处。

本节课中我们一起学习了如何识别暧昧期的信号，以及如何通过选择合适的地点、策略性地把握赴约时间来为约会营造良好开端。下一节，我们将进入约会见面后的具体撒娇互动环节。
