# 女神养成系列：03：吸引力 _ 在柴米油盐中，找回相爱的初心

## 概述
在本节课中，我们将要学习冥想如何帮助我们调节情绪，特别是通过影响大脑中的杏仁核，从而改善亲密关系。课程将包含一套完整的阴瑜伽体式序列，为冥想练习做好准备，最后引导大家进行一次深度放松的冥想。

在上周的课程中，我们介绍了冥想能为身体带来的好处。在高强度的压力生活下，人体会产生一种名为**去甲肾上腺素**的脑内毒素，它会加速身体老化并引发各种疾病。而冥想练习能促使身体产生另一种激素——**内啡肽**，它能让人产生愉悦感，并有效对抗去甲肾上腺素的负面影响。

## 杏仁核：情绪的关键开关
本节中我们来看看冥想对大脑情绪中心的具体影响。杏仁核是大脑中负责产生、识别和调节情绪的组织。**杏仁核激活水平高**时，人会变得消极、易怒、共情能力低。在高噪音、高污染、高压力的生活环境下，杏仁核的激活水平就会升高。

科学家们对如何控制杏仁核水平进行了大量研究。美国一所知名大学的研究者发现，冥想训练可以有效降低杏仁核的激活水平。研究显示，无论是长期还是短期的冥想者，其杏仁核激活水平都有所降低，其中长期冥想者的效果尤为显著。

**杏仁核激活水平低**时，人会处于直觉敏锐的状态，不易被消极情绪控制，且共情能力高。这在现实生活中至关重要，尤其是在处理与伴侣的关系时。试想，当双方都处于高压环境下，很容易在不知不觉中对彼此发脾气，长此以往，关系会变得迟钝。

我的朋友妙妙就是一个例子。她与丈夫结婚十年，从最初的恩爱到如今相对无言。在跟我练习瑜伽冥想两个月后，她告诉我最大的改变是，自己不那么容易发脾气，也不那么爱对丈夫吹毛求疵了，似乎找回了当初恩爱的感觉。

冥想并非解决亲密关系的灵丹妙药，它无法让一个不爱自己的人爱上自己。但它能帮助我们不被负面情绪淹没，为改善关系创造内在的平静空间。

## 阴瑜伽体式准备
那么，就让我们开始今天的习练吧。首先，我们从阴瑜伽的体式开始，为冥想做好准备。今天的练习会用到抱枕、瑜伽砖和毛毯，请将这些辅具放在垫子周围随手可及的地方。

### 体式一：束脚式
以下是束脚式的练习步骤：
1.  双腿依次向前伸展。
2.  用右手扶住右膝窝，拉起右膝，将右脚放在左大腿内侧。
3.  用左手扶住左膝窝，使两脚掌心相对。
4.  双手放于臀部后侧，调整坐姿。
5.  若大腿内侧拉伸感过强，可将两脚掌向前挪动约半个脚掌的距离。
6.  调整好后，双手在臀部两侧推地，伸直脊柱向上，感受头顶有绳向上牵引。
7.  呼气，身体向前折叠，双肘依次落于前方垫面，低垂头颈。
8.  在此体式停留一分钟，找到最舒适的范围。
9.  若感觉强烈，可稍微抬起身体，取一块瑜伽砖垫于额头下方。
10. 若仍感强烈，可移开瑜伽砖，取抱枕或硬枕头抵在胸口，双手放松，低垂头颈停留。
11. 一分钟后，慢慢睁开眼睛，抬起额头，移开辅具。
12. 双手掌心扶住两膝外侧，轻轻将双膝合向中线，脚掌前移。
13. 双手抱住小腿胫骨，低垂头颈，休息五个呼吸。
14. 睁开眼睛，抬头，双腿向前伸展。

### 体式二：半鞋带式到鞋带式
接下来我们进行第二个体式序列。

**右侧半鞋带式：**
1.  用右手扶住右膝内侧，保护膝盖，将右脚拉向左臀外侧。
2.  双手手掌推臀部后侧地面，推直脊柱向上，感受坐骨稳定扎根。
3.  呼气，身体轻轻向前向下折叠，双手放松置于左腿两侧。
4.  左脚保持放松。
5.  在此停留十个呼吸。
6.  慢慢撑起身体，卷背向上，一节节拎高脊柱。

**左侧半鞋带式：**
1.  现在将左脚拉向右臀外侧。
2.  双手撑于臀后，推直身体。
3.  感受坐骨稳定扎根后，身体向前向下俯身，双肘落于前方垫面。
4.  若颈部后侧有压力，可取瑜伽砖垫于额头下方。
5.  在此体式停留一分钟，感受大腿外侧、臀部外侧的拉伸，观察腹股沟是否紧张，呼吸是否因拉伸而变得急促。

**鞋带式侧伸展（右侧）：**
1.  抬起身体，将瑜伽砖放到旁边。
2.  双手放于臀部两侧，先推直脊柱。
3.  落下右肘于身体右侧垫面。
4.  抬起左臂伸展至头部上方。
5.  若感觉不轻松，可取瑜伽砖垫于右肘下方。
6.  感受身体左侧的拉伸，将呼吸带到左侧，头部自然低垂。
7.  保持左臀牢牢扎根地板。
8.  在此保持十个呼吸，可闭上眼睛。
9.  睁开眼睛，稍微推起身体，左手在前方轻轻画圈，身体回正。

**放松与换边准备：**
在做另一侧之前，我们先放松双腿。
1.  双腿伸直向前，双手放于臀部后侧约20公分处，指尖朝向臀部。
2.  推起身体，抬起胸腔，放松双腿。
3.  在此保持五个呼吸，可轻松活动双腿。
4.  身体回到正中。

**左侧半鞋带式：**
现在开始另一侧的练习。
1.  用左手扶住左膝内侧，将左脚拉向右臀外侧。
2.  双手手掌推臀部后侧地面，轻轻推直身体，感受头顶向上牵引。
3.  坐骨稳定扎根后，呼气，身体轻轻向前向下折叠。
4.  双手依次放于右腿两侧。
5.  若感觉有压力，可取瑜伽砖垫于额头下方。
6.  右腿与右脚保持放松。
7.  在此保留十个呼吸。
8.  轻轻抬头，推起身体，移开瑜伽砖。

**完全鞋带式：**
1.  弯曲右膝，将右脚拉向左臀外侧，来到完全鞋带式。
2.  双手撑于臀后方，感受脊柱直立向上。
3.  坐骨稳定扎根后，身体向前向下，双肘落于垫子前方。
4.  可取一块瑜伽砖垫在额头下方。
5.  在此停留一分钟。

**鞋带式侧伸展（左侧）：**
1.  抬起额头，轻轻推起身体，移开辅具。
2.  抬起左臂，左肘落在左边垫面上。
3.  抬起右手臂伸展至头部上方。
4.  若感觉右侧腰腹拉伸强烈，可取瑜伽砖垫于左肘下方。
5.  将注意力和呼吸带到右侧腰腹部，保持右臀落于地板，头部自然低垂。
6.  在此保留十个呼吸。
7.  轻轻推起一点身体，右手在前画圈，回到身体两侧。

**再次放松：**
现在稍微休息一下，伸直双腿释放压力。
1.  转身，双腿伸直，双手放于臀后约20公分处，指尖朝向臀部。
2.  推起身体，胸腔向上，腿部放松。
3.  保持五个呼吸。
4.  回到正中。

### 体式三：睡天鹅式
接下来我们学习今天非常重要的一个体式。睡天鹅式在阴瑜伽中非常重要，经常练习受益良多。它能锻炼到臀部外侧、大腿内外侧、腹股沟以及整个后背部的肌肉。从经络穴位角度看，这些部位是身体肝胆经和膀胱经经过的区域。

以下是睡天鹅式的练习步骤，需要用到毛毯和抱枕。

**右侧睡天鹅式：**
1.  身体转向垫子短边，双腿盘坐。
2.  抬起臀部，将左腿向后伸展。
3.  若感觉右臀拉伸强烈，可将右小腿向后调整，右脚跟拉向耻骨方向。
4.  若想加深体式，可将右小腿向前放，尽量与垫子短边平行。
5.  俯身向下。
6.  若右臀外侧离地很高，取毛毯垫在右臀下方。
7.  将瑜伽抱枕放于身体前方，轻轻趴在上面，放松手臂和面部。
8.  在此体式停留两分钟。

**放松与换边：**
1.  先将头回正，轻轻推起身体，移开抱枕和毛毯。
2.  直接收回左腿，放松双腿。
3.  双手放于臀后约20公分处，推起胸腔，放松腿部，保持五个呼吸。
4.  睁开眼睛，推起身体。

**左侧睡天鹅式：**
1.  将两个小腿倒向身体右侧，抬起臀部，右腿向后伸展。
2.  根据自身活动范围调整：想加深拉伸，将左小腿拉向与垫子短边平行；想更轻柔，则将左脚拉向耻骨方向。
3.  轻轻趴下，用毛毯垫在左臀下方，瑜伽抱枕放于身体前侧。
4.  慢慢趴下放松，停留两分钟。

**最终放松：**
1.  慢慢将头回正，依次用双手撑地抬起身体。
2.  移开辅具，臀部坐在左侧，收回双腿放松，停留五个呼吸。
3.  睁开眼睛，推起身体。
4.  现在进入摊尸式大休息，可盖好毯子，躺下休息。
5.  结束时，轻轻活动手指脚趾，将意识收回。身体慢慢转向右侧，以右臂为枕，用左手推地坐起。

## 冥想练习引导
以上是今天的阴瑜伽体式练习，接下来让我们一起进入冥想练习。

1.  轻轻仰卧躺下，头部不要垫太高，地面不宜太硬，以舒适为准。若感觉不适，可先坐起。
2.  现在以舒适的坐姿坐好，保持下背部微微向前一点。
3.  只是舒适地注视前方，可以让视线向下，选择感觉舒适的方向，允许眨眼正常进行。
4.  同时，你应当由内而外地感受身体，以一种柔和、不刻意的方式。
5.  在眨眼继续的同时，很自然、柔和地由内而外感受身体，你会感觉到身体在放松。
6.  这种放松从眼睛内部开始蔓延，然后你的眼睛可能会开始自动闭合。让眼睛慢慢闭上，不要强迫。
7.  你可能还在眨眼，继续由内而外地感受身体。慢慢地，眼睛会闭合。一旦闭合，就不再睁开。过程中不要着急。
8.  眼睛闭上后，你会感受到眼睛内部非常平静。若仍有干扰，可微微睁眼一点，继续眨眼等待其闭合。
9.  过一会儿，你会注意到眼睛越来越重，自动闭合。去感受眼睛内部的放松感，身体其他部位也有，但眼睛的更明显。
10. 现在，只是持续去感受眼睛的放松，观察会发生什么。
11. 当你持续感受时，这种放松感可能在任何时刻蔓延。一旦整个面部感受到放松，就持续感受这种放松。
12. 持续感受面部的放松，观察其效果。就是这么简单。
13. 慢慢地，自然而然地，这种放松感会在整个身体蔓延。不要去想象。
14. 现在，继续去感受放松感在身体中蔓延。此刻，继续感受整个身体的放松。
15. 如果你现在想仰卧，那么可以躺下了。躺下后，不要去想象，要去感受身体发生了什么。
16. 你会发现躺下后，整个身体的放松感发生了。参与感受整个身体的放松，同时感受每一个部位的放松：双腿、双脚、双手、双臂、头部、胸部、腹部、背部都在放松。
17. 放松正在沉向身体，并且持续。你要继续享受这种放松。
18. 如果走神了，没关系，把它拉回来。慢慢地但坚决地，让放松感继续在身体内。
19. 身体内还发生其他的吗？你能够感知到身体在空间中的位置：双腿、双手、头部、胸部、腹部、背部都在这个空间当中。
20. 如果你继续，会感受到身体中升起了某种觉知。你周身都有放松的感觉，同时也有觉知。换句话说，你既有放松感，也对整个身体有觉知。
21. 现在，感受身体内放松与觉知的共存。我将给予一分钟没有引导语的时间。
22. （一分钟静默）
23. 现在问自己：身体之内发生了什么？是整体地去感受。通过身体去寻找，周身好像发生了什么？你无法描述，但能够感觉到。请记住，无需描述，只是去感受。
24. 不管身体发生了什么，继续去感受它在身体之内到底发生了什么。在你的双手、双脚、双臂、双腿、头部、胸部、腹部、背部，整个身体去感受。
25. 如果你没有去感受整个身体，效果也不会呈现。一定要感受全身。
26. 在继续感受时，你可能走神，可能有念头出现，以至于丢失了与当下感受的连接。不要担心，可以把它带回来。
27. 你只需要做一件事：同时去感受你的双脚、双手、双腿、双臂，参与其中，由内而外地感受，并继续。如果你这样做，那种无法言表的感觉又会出现。
28. 现在我们要做的就是持续地感受，这处于“总持”阶段。首先要有总持，让这说不清的感觉持续发生，尽可能久。
29. 当然，一旦它丢失，就再次通过同时感受双手、双脚、双腿，让这种感觉慢慢回来。所以，继续让这种说不出的感觉在体内发生。
30. 当你继续这样做的时候，感受它的效果。是的，它产生了一种非常舒适的、好的效果。你可以继续观察、感受。
31. 这种舒适的感觉在身体最深处发生了什么？感受它的状态，感受这种放松。任何时刻，它都可能呈现一种深深的静默。
32. 留在静默之中继续感受。如果有任何特殊情况发生，你都可以去观察。只是静默，只有静默。
33. 现在，简单地去感受你腹部的中段下方，肚脐之下。完整全然地感受这个部位。
34. 现在去感受头部的中心。只是去感受。你是否能完全感受到头部的中心？
35. 同时感受腹部的中段和头部的中心。继续感受这两点。
36. 如果你持续不断地感受腹部中心、头部中心，那么你就能感受到最头顶的顶轮。
37. 现在你就停留在头顶，并且去感受它。要做的只是停留在头顶，没有其他。
38. 任何时刻，若感觉到头顶之外有一定的距离也发生着什么，那么持续去感受。倘若头顶之外没有任何感受，也没有问题，只需要感受顶轮（头顶）。
39. 在任何情况下，都不需要去想象。现在忘掉一切。此刻，你只是要停留在静默中。只有静默，没有其他。
40. （较长时间静默）
41. 当你感觉可以了，随时可以坐起来，根据自己的情况来决定。
42. 继续闭着眼睛。在你坐着的时候，简单地让时间流淌一下：一秒、两秒、三秒。任何时候，这种舒适感都可能再度出现。
43. 继续在你的头脑之内观察它，留在静默之中，只有静默。闭上眼睛几秒钟。
44. 当你感觉可以了，就允许自己的眼睛慢慢地睁开。

## 总结
今天的冥想练习就到这里，谢谢大家。本节课中，我们一起学习了冥想通过降低杏仁核激活水平来帮助管理情绪、提升共情能力的原理。接着，我们通过束脚式、鞋带式、睡天鹅式等阴瑜伽体式为身心做好了放松准备。最后，在深度的冥想引导中，我们练习了由内而外地感受身体、培养觉知与放松共存的状态，以期在柴米油盐的日常中，找回内心的平静与相爱的初心。
