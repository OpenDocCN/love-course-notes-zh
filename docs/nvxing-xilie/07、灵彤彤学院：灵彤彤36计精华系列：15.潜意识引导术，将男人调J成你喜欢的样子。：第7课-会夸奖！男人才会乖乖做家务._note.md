# 灵彤彤学院：灵彤彤36计精华系列：第7课：会夸奖！男人才会乖乖做家务

## 概述
在本节课中，我们将学习如何通过有效的沟通与引导，鼓励伴侣主动分担家务。课程将分析男人不愿做家务的常见原因，并提供一系列具体、可操作的策略，核心在于运用“夸奖”和“潜意识引导”来培养伴侣参与家庭事务的习惯。

---

许多女性在婚后会发现，自己仿佛多了一个需要照顾的“儿子”。常见场景包括伴侣不断询问：“我的袜子放哪儿了？”“我饿了，你怎么还不做饭？”“我的裤子怎么还没洗？”这些情况你是否也感到熟悉？

婚后，特别是生育后，女性往往需要承担家务、育儿，同时还要照顾伴侣，感到不堪重负。这究竟是伴侣的问题，还是女性自身过于要强？事实上，许多女性在婚前相当独立，能自己处理各种事务，不习惯求助他人。长期如此，导致委屈和辛苦独自承受，变得愈发独立坚强。

一些女性婚后成为家庭主妇，心理上可能处于弱势，认为自己没有收入就应承担更多家务，让“辛苦养家”的伴侣多休息。这种模式持续一两年尚可，但五六年积累的委屈爆发时，对方可能仍感到茫然不解。

男人不做家务是许多已婚女性的困扰，也是家庭矛盾的来源之一。不少男性认为不做家务是天经地义的事。然而，家务看似不累，实则繁琐耗时，若全部由一人承担，十分辛苦。家庭事务本应是双方共同的责任。

## 影响男人分担家务的因素
以下是决定男人是否愿意分担家务的几个关键因素。

1.  **原生家庭模式**：他从小是否被培养做家务，是否有过独立生活的经历。现状是，许多独生子女从小到大基本不参与家务，婚后这些习惯自然会显现。
2.  **夫妻感情基础**：感情良好的夫妻，在家务分工上通常更容易协商，不会过分计较。
3.  **个人修养与处境**：个人修养好的人，即使感情一般，也不会做得太过分。有时，男人若在外界受挫，也可能更倾向于在家中寻求掌控感。
4.  **客观条件与妻子引导方式**：客观条件是否允许，以及妻子对待此事的处理方式至关重要。
5.  **经济收入观念**：由于男性普遍收入较高，一些人会自然地认为“我赚钱多，就不用做家务”，这是一种普遍存在的大男子主义思维。

对于受原生家庭影响（因素一）的男性，需要理解这种模式的根深蒂固。他们可能真的没有意识到问题。因此，最好找一个机会，清晰、平和地提出你的要求，避免带情绪或指责，并先肯定他做得好的地方。当然，谈话前可以先询问他自己的想法。

如果问题源于感情基础（因素二），那么重点应放在增进夫妻感情上。感情正常的夫妻，家务通常不会分得太细，谁有空谁做，做得多的一方也不会在意，因为做家务也是表达爱的方式。

## 如何引导老公做家务
从上述第四点（引导方式）出发，以下是如何有效引导伴侣参与家务的具体方法。

### 1. 把老公“捧上神坛”
多在家人、朋友面前夸奖他。例如，一位妈妈常在家长群里夸赞自己的丈夫是“模范丈夫”，结果丈夫越发愿意在大家面前表现，主动为孩子买雪糕、为妈妈买咖啡。这种公开的欣赏，能激发对方的积极性。

### 2. 用积极态度创造参与机会
不要抱怨，而是带动。例如，大扫除时，不要自己埋头苦干，而是兴致勃勃地为丈夫和孩子分配区域，全家一起动手。孩子都在干活，丈夫通常也不好意思玩游戏。完成后，可以全家一起外出简单就餐，作为奖励。你还可以通过“家庭会议”教育孩子要做家务，并顺势进行分工：“爸爸最擅长这个，所以这项由爸爸负责。”

### 3. 多夸赞，多示弱
如果他做了某件家务，一定要给予正面肯定。平时多示弱，多请他帮忙：“这个我真的不会修，还是你厉害。”让他感受到被需要和价值感。帮助一旦开始，就可能成为习惯。

### 4. 让他体验你的感受（针对严重情况）
如果情况比较严重，可以尝试让他体验你的角色。例如，周末找借口带孩子回娘家，记得不要赌气离开，而是多打电话关心他，让他自己处理饮食起居。这给了他学习成长的机会。

### 5. 保持经济与人格独立
你必须拥有一份自己的工作，即使收入不高。经济独立是人格独立的基础，能让你拥有自己的社交圈，思维不落伍。有人会问：“我上班孩子谁带？”解决方案可以是请婆婆帮忙或聘请保姆。**关键公式是：经济独立 = 谈判资本**。有了工作，你就可以和丈夫平等协商家务分工，例如“一三五我做，二四六换岗，周日休息”。拥有独立人格，你才有“讲价”的资本。

引导男人做家务，核心在于你的方法。然而，许多女性常会使用错误的方式。

## 引导过程中的注意事项
以下是引导伴侣做家务时需要避免的误区。

1.  **不要嫌弃他做得不完美**：只要他肯动手，就及时给予赞扬。例如，丈夫第一次洗碗可能只洗了碗，锅和灶台都没清理。聪明的妻子会先肯定他“碗洗得真干净，还把地拖了”，然后自己再悄悄收拾残局。男人也需要鼓励，苛刻的指责只会打击他的积极性。
2.  **相信他能做好，甚至比你更好**：家务既是细致活，也是体力活。不要先入为主地认为男人做不好。有时，当你放手让他负责一项任务（如清理油烟机），结果可能比你做得还干净。问题往往在于男人觉得这不是“分内事”，而非能力不足。
3.  **允许犯错，多些耐心**：在学做家务这件事上，男人的“智商”可能不超过三岁。会出现晾了衣服忘记收、脏袜子和内衣一起洗、炒完菜才发现没煮饭等情况。这时，妻子要聪明一点，不要本能地嫌弃，而是耐心教导。把这些小错误看作生活情趣。
4.  **不要反复抱怨和咒骂**：反复的负面指责会破坏对方的心情和积极性，可能引发争吵，让他更不愿意动手。
5.  **在他做家务时，不要喋喋不休**：很多女性喜欢在伴侣做事时指挥和吐槽：“桌子没擦干净！”“地上还有东西！”这会引起反感。试想，如果你在做事时旁边有人不停指点，你也会不舒服。做饭是项繁重的脑力+体力劳动，如果他能负责工作日做饭，已是不小的贡献。不妨角色互换体验一下。

## 总结
本节课我们一起学习了如何通过夸奖和引导，让伴侣愿意主动分担家务。核心要点包括：理解其行为背后的原因（原生家庭、感情、观念等），通过公开赞扬、创造参与机会、及时肯定、适当示弱等积极方式进行引导。同时，务必保持自身的经济与人格独立，这是平等协商的基础。在引导过程中，要避免嫌弃、指责、喋喋不休等错误方式，多给予耐心和鼓励。

一个家庭的维系需要两个人的共同努力。对于认为“工作价值大于家务”的丈夫，妻子需要让他明白，为家庭投入与付出，其意义与工作成就不同，家庭才是人生最重要的投资。
