在本节课中，我们将要学习第二阶段腹肌耐力训练的四个核心动作。这些动作旨在持续刺激腹部肌群，特别是下腹和腹斜肌，以提升肌肉的耐力和控制力。

上一节我们介绍了腹肌耐力练习的基础，本节中我们来看看四个更具挑战性的动作组合。

## 🎯 动作一：海星卷腹 (Starfish Crunch)

第一个动作是海星卷腹。这个动作主要针对下腹的上部，通过交替抬腿来卷动身体。

**动作要领**：
*   身体平躺，双手向两侧张开，形似海星。
*   交替抬起一条腿，同时用下腹力量带动身体微微卷起。
*   保持动作的连贯和控制。

## 🔄 动作二：反向卷腹 (Reverse Crunch)

接下来是反向卷腹。这个动作专注于孤立刺激下腹部。

**动作要领**：
*   平躺于垫上，双手放于身体两侧或臀部下方。
*   双腿屈膝抬起至90度。
*   用下腹力量将骨盆向胸口方向卷起，带动双腿向上移动。
*   关键点在于保持大腿与上身的角度不变，避免用惯性甩腿。公式可理解为：**动作幅度 = 骨盆后倾 + 下腹收缩**。
*   下落时需有控制地缓慢放回。

## 💥 动作三：肘击膝 (Elbow to Knee)

第三个动作是肘击膝。这是一个锻炼腹斜肌的经典动作。

**动作要领**：
*   采用仰卧姿势，双手置于耳侧，双腿屈膝抬起。
*   用一侧肘关节去触碰对侧膝盖，同时呼气并收缩腹斜肌。
*   注意保持肘关节稳定，避免用手臂摆动借力，应通过躯干的旋转完成动作。代码逻辑可描述为：
    ```python
    while 训练进行中:
        左侧腹斜肌收缩 -> 右肘触碰左膝
        右侧腹斜肌收缩 -> 左肘触碰右膝
    ```

## ✨ 动作四：Y型转体 (Y-Twist)

最后是Y型转体。这个动作能综合锻炼腹直肌和腹斜肌。

**动作要领**：
*   起始动作为仰卧，双臂向上伸直，与身体呈Y字形。
*   用腹部力量将肩部抬离地面，同时向一侧转体。
*   感受腹斜肌的收缩，它主要负责身体的旋转功能。
