# 公文写作法：02：商业回顾（下）—— 行动计划与目标设定

在本节课中，我们将继续学习商业回顾（Business Review）的写作方法。上节课我们介绍了如何呈现结果性数字以及分析原因的工具。本节中，我们将重点探讨如何基于分析结果制定具体的行动计划，并为下一阶段设定合理的目标。

## 第三部分：行动计划与改进措施

基于前面对客观成果和原因的分析，接下来需要明确具体的行动方案。这部分被称为反馈步骤，旨在总结上一阶段并为提升未来表现制定计划。因此，商业回顾的第三部分通常是“行动计划”或“改进措施”。

以下是构成一个完整行动计划的三个核心部分：

1.  **行动步骤**：基于前面分析出的原因，提出具体的解决措施。
2.  **完成期限**：为每个行动步骤设定明确的时间节点。
3.  **投入产出模型**：评估行动所需的成本和预期收益。

### 行动步骤的制定

行动步骤应直接对应原因分析的结果。我们通过一个例子来说明：

假设通过“三元素法”分析，发现上周点击量下滑的原因是：
*   **直接原因**：标题不够吸引人，内容互动性不足。
*   **主要原因**：市场同质化内容过多，缺乏特色。
*   **根本原因**：缺乏对用户喜好和阅读兴趣的准确洞察。

针对不同层次的原因，可以制定相应的行动步骤：

*   **针对直接原因（立竿见影）**：
    *   立刻寻找并分析10个以上10万+阅读量公众号的高点击标题，总结规律并进行模仿。
*   **针对主要原因（深入调整）**：
    *   深入思考自身相较于其他KOL的独特优势和不可替代性，明确未来的市场定位和发展方向。
*   **针对根本原因（根源解决）**：
    *   一方面，学习消费心理学、市场规律等理论知识。
    *   另一方面，通过实践验证理论。例如，运用“权威心理”理论，将自身定位塑造为某个领域的权威，以吸引固定受众。

### 完成期限的设置

为行动步骤设定期限时，需考虑其实现性和可操作性：

*   **立即执行**：如竞品分析、风格筛选。
*   **短期目标（约1周）**：完成初步的风格调整或方案测试。
*   **长期目标（如3个月）**：进行深度的风格打磨和理论储备，并为此设定阶段性的细分目标。

### 投入产出模型的设计

这一步至关重要却常被忽略。它帮助决策者（如老板）评估计划的可行性和成本效益。在每项行动计划后，应简要分析其所需的投入和预期产出。

**示例**：
> 计划：为提升团队业务水平，聘请外部资深讲师进行培训。
> *   **投入**：讲师课酬 `X` 元，内部协调与准备时间成本约 `Y` 小时。
> *   **预期产出**：团队关键技能掌握度提升 `Z%`，预计可间接促进下季度业绩增长 `A%`。

## 第四部分：设定下一阶段目标

行动计划完成后，还需为新的工作周期设立明确的目标。检验计划执行效果的核心指标，就是目标与实际结果的相关性。以下是三个设定目标的小技巧：

### 技巧一：用区间代替具体数值

由于结果受可控与不可控因素共同影响，设定具体数字目标风险较高。使用区间目标可以预留缓冲空间。

**公式**：
> 将 `目标 = 80万` 改为 `目标 ∈ [70万, 90万]`

### 技巧二：按时间梯度设置上涨比率

在职场中，保持稳定（匀速运动）可能显得缺乏进取心。随着经验增长，工作效率应有所提升，因此目标应体现出适当的“加速度”。设定一个比上一阶段有所增长的目标是合理的。

**建议**：
*   **常规增长率**：`20% ~ 30%`。这是一个相对稳妥、能体现“稳中有升”态势的区间。
*   **特殊情况**：若任务难度极大或期限极短，可酌情将增长率调整至 `10%` 左右。

### 技巧三：增加限制性条款（线性因素）

为应对战略调整、突发任务或不可抗力等意外情况，增加限制条款能使目标设定更严谨。

**示例代码/条款**：
```plaintext
阶段性目标设定基于当前已知条件和战略方向。如发生重大计划调整或不可抗力事件，目标将根据实际情况重新评估与修订。
```

## 课程总结

本节课中，我们一起学习了商业回顾（Business Review）的最后两个核心部分：**行动计划**与**目标设定**。

一个完整的商业回顾写作思路如下：
1.  **呈现结果**：运用绝对数字、纵向相对数字和横向相对数字，动态与静态结合地展示成果。
2.  **分析原因**：借助分析工具（如三元素法）拆解成果或问题背后的原因。
3.  **制定计划**：基于原因分析，制定包含行动步骤、完成期限和投入产出模型的行动计划。
4.  **设定新目标**：为下一阶段设定合理、有挑战性且包含缓冲空间的目标。

商业回顾不仅是日常汇报的工具，更是个人进行工作复盘、持续优化的重要方法。掌握这一页纸的写作法，能帮助你更清晰、更有说服力地呈现工作价值与规划。
