# 花镇情感课程：01：第3课 男人最喜欢女人怎么撩他？

在本节课中，我们将要学习如何运用“撒娇”这一技巧，以符合男性思维的方式增进两性关系。我们将探讨撒娇的不同层次、适用场合以及核心要点，帮助你掌握分寸，让感情更加甜蜜。

## 概述：撒娇的分寸与场合

上一节我们介绍了与男性沟通的基本思维，本节中我们来看看“撒娇”这一具体技巧。撒娇对部分女性是与生俱来的天赋，对大多数女性则是一项需要学习的技巧。在两性关系中，撒娇如同蜜糖，适量则甜，过量则腻。若分寸把握不当，可能用力过猛，令人反感。

例如，在一次同学聚会中，两位男性带了女友。A女友在公开场合过度撒娇，如“你喂我嘛”、“讨厌哼”，令在场众人尴尬，其男友也感到不适。将私密对话置于公共场合，有失大体。B女友则表现为温柔乖巧，挽着男友、微笑倾听、握手表达疼爱，举止得体。

**核心公式：合适场合 + 恰当行为 = 有效撒娇**

上得厅堂，下得厨房，意指女性在不同场合应呈现不同姿态。穿围裙上厅堂或穿晚礼服下厨房均不合适。撒娇同理，需考虑环境。二人世界内，可说“你好坏”等话，令男性心动。公共场合则不宜。

撒娇初衷是为男性带去心动，激发其怜惜与疼爱。若仅为吸引注意，则与抽对方一巴掌无异。不分场合的撒娇非但不能促动感情，反而可能引发反感。

因此，把握撒娇的分寸感至关重要。女性学会适度撒娇，能使爱情更加甜蜜。接下来，我们将深入探讨让撒娇更进一步的学问。撒娇可分为高、中、低三个等级。首先从最低等级开始。

## 初级撒娇：言语撒娇

如果你留意网络上的恋爱技巧内容，会发现一件有意思的事。大多数人对撒娇的定义停留在简单的说话方式上，即我们今天要讲的初级撒娇技能——言语撒娇。

在电影《撒娇女人最好命》中，隋棠饰演的台湾软妹“贝贝”在展现其柔软一面时，刻意使用了大量叠词，如“兔兔喝水水”、“睡觉觉”，加之嗲嗲的声音，让剧中见惯女汉子的龚智强招架不住。

生活中，除了恰当使用ABB式叠词，还可改变一些词语的使用方式。

以下是具体方法列表：
*   将句子中的“我”改为“人家”，例如“我要吃饭”改为“人家要吃饭”。
*   加入口语化语气词，如“真的呀”、“好呀”、“可不可以呀？”。
*   放慢语速，拉长尾音，让声音听起来更温柔。

如法炮制，多加练习，便能熟练掌握撒娇的初级技能。

当然，说到这里，可能有很多朋友要开始反驳：孩子都已长大，作为母亲难道还要对身边人说“吃肉肉”、“上班班”吗？这太肉麻了。或作为天生女汉子，突然开始嗲声嗲气，伴侣可能以为精神出了问题。

没错，对于十几岁少女或天生自带萌系的女性，这种方法可能十分奏效。但对于年龄稍长的女性，这种言语撒娇可能不那么百分百合适。所以，接下来我们要谈谈撒娇的更高段位。

## 高级撒娇：形体与仪态

刚提到“形体撒娇”这个词，大家可能有点迷糊。我来举个简单例子，大家就明白了。

在著名美国电影《闻香识女人》中，阿尔·帕西诺饰演一名双目失明的上校。他阅历丰富，爱女人胜过一切。即使失明，他在与女性交往时，也能凭借对方身上的香水或肥皂味，精准判断其个性。

电影中有一幕经典镜头：在纽约高级餐厅，这位失明上校被邻桌一位女士吸引，并绅士地邀请对方跳了一支即兴探戈。音乐响起，被邀女士一袭长裙，肌肤白皙，耳畔珍珠耳环光彩夺目。两人翩翩起舞，空气中弥漫着阿根廷乐曲《一步之差》以及女士身上淡淡的少女香皂味。此情此景不仅赢得了在场所有人的倾心，更成为电影史上最香艳的镜头之一。

说到这里，聪明的你一定明白了我的意思。当语言褪色、声调消失，你依然可以凭借以下方面，无需一言一语，完成更高级的撒娇：
*   **随时随地干净整洁的装扮**
*   **进退得体的举止**
*   **温柔娴静的仪容**

## 核心本质：永葆少女心

说到这里，想必你已经大概掌握了撒娇的一些基本技能，例如说话更柔软、举止更娇媚、仪表更精致等。但有些女孩会嘀咕：减肥美容非一日之功，可男神就在眼前，该如何在最短时间内抓住他？

确实，罗马非一日建成。若以上软萌技能、举止技能、外形技能统统缺失，是否就注定与撒娇无缘？答案当然是否定的。世界上的叶子尚无两片完全相同，更何况形形色色的女性。

但即便如此，有一句话套用在这里非常合适：**不可爱的女孩都一样，可爱的女孩却各有各的可爱。**

香港电影中的女明星众多，每个人的说话方式、外表、举止都不相同，但却同样惹人怜爱。例如：
*   《秋天的童话》中，钟楚红扮演的“十三妹”文静知性，加上淳朴天然的美丽，成为当时众多男性心中的梦中情人。
*   《花样年华》中，张曼玉身材曼妙，言语寥寥，但少妇的韵味俘获了超多男人的心。
*   《新龙门客栈》中，张曼玉扮演的老板“金镶玉”泼辣却不失妩媚，风骚却不失善良，风情万种的表演成为经典。
*   《大话西游》中的紫霞仙子，一袭红衣，语笑嫣然，时而欢快，时而落寞，甘愿为爱飞蛾扑火。

这些角色性格、故事各异，却同样让男性无比怜爱。其中的奥秘是什么？仔细思考这些女孩的共性，答案便迎刃而解。她们都有一个显而易见的共同点，即当下非常热门的词——**少女心**。

所谓少女心，核心在于其包含的“少女内核”。年少女孩追求感情时，往往不在乎外在条件，如对方家庭、财产、未来，重要的是眼前这个人本身。

前段时间，46岁的俞飞鸿在一段长达数小时的采访中谈到其感情观。她表示，即便现在，在感情中依然注重情感本身，而非依附于外在条件。无论如何，“合拍”依然是她在感情中追求的根本。

如此看来，这便是我们所谓的标准少女心了：**有梦就去做，有爱就去追，不瞻前顾后，尽情享受当下，享受眼前人带来的微小感动，并以最温柔的方式去呵护和回应。**

如此，无论年龄几何，都将在两人的感情世界中永葆娇媚，永葆少女心。

## 实践提醒：撒娇的敌人——抱怨

很多姑娘婚前善于撒娇，婚后却渐渐丢掉这个技能。撒娇有个敌人叫做“抱怨”。如果你经常对一个男人抱怨之后再去撒娇，可能就不奏效了。因为他总是看到你强势和情绪化的一面，就会对你娇柔的那一面产生抵抗力甚至怀疑——你是否在演戏？

**核心要点：先处理情绪，再运用技巧**

因此，如果你在婚姻中是一个强势且喜欢抱怨的女人，首先要做的不是学习撒娇技能，而是先让自己变得温柔平和。我们希望被男性温柔相待，男性也同样希望被我们温柔相待。夫妻之间许多事，撒撒娇便可解决，无需剑拔弩张、拼命讲道理。

微微就领教了这一点。和田昊结婚不久，她每天回家切菜做饭，累得半死，田昊却坐在沙发上玩手机等开饭。她终于忍不住大声抱怨：“你就知道吃，跟死人一样，就不能过来帮帮忙吗？”结果田昊十分不情愿地走进厨房，饭后两人一直黑着脸，此后田昊长期处于“不喊不动”的状态。

后来，在闺蜜指导下，微微改换战略，将抱怨模式改为撒娇模式。“老公，过来帮我洗个青菜嘛。”一开始田昊还端着：“我为什么要洗青菜？你给我个理由先。”微微继续撒娇：“因为我们快点吃完饭，就可以早点开始激情之夜了。”田昊一听，立即以百米冲刺的速度冲进厨房。

你看，属于小两口之间的撒娇，不仅能达到目的，还可以增进感情，因为男性“吃这一套”。
