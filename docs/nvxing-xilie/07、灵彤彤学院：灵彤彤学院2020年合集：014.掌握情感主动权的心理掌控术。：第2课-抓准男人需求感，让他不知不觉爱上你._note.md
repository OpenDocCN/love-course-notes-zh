# 灵彤彤学院：014：掌握情感主动权的心理掌控术：第2课-抓准男人需求感，让他不知不觉爱上你

## 概述
在本节课中，我们将学习如何通过理解和满足男性的核心心理需求，来建立更稳定、更吸引他的亲密关系。核心在于，男性在感情中需要被肯定、被理解、被尊重，当这些需求得到满足时，他会自然而然地被吸引。

---

每一个女人都希望拥有幸福的爱情，希望自己在感情当中用心付出，能得到对方的回应。但是有一些女人却经常碰壁，无论为男人付出多少，男人都无动于衷，甚至对他越来越冷淡。这让女人感到非常痛苦。

你想过你给她的真的是他想要的吗？他不想要，即使你付出再多，都是徒劳。男人喜欢一个女人多半会不顾一切的奋起直追，而女人却总是不紧不慢，向喜欢的男人不断暗示引导。

那么有没有这样一种情况，男生直接对女生说，是的，我就是爱上你了呢？这个问题对于不懂的男人来说，或许会有不一样答案。但相信所有的男人，答案都离不开：当他的心理需求能够被女人理解和满足时，他无疑就会爱上她。

那么女人应该如何才能抓准男人的需求呢？上一节我们探讨了吸引力的基础，本节中我们来看看具体满足哪些需求能让他不知不觉爱上你。

以下是九个关键的心理需求点：

**第一，用心听男人的夸夸其谈。**  
男人在女人面前吹牛，往往是一种缺乏自信心的表现。女人如果不能倾听，男人的自信心难以建立就会崩溃。要知道，当男人谈论自己的事情时，他们觉得认真倾听的人非常有魅力。  
**核心策略**：`倾听时间占比 = 80%`，`自我表达时间占比 = 20%`。  
用20%的时间谈谈你自己或热点话题，然后让他们控制80%的谈话时间。男人最需要的是一种被肯定、被崇拜、被需要的感觉。女生需要了解男人深入的内在，去懂得满足男人的心理需要，才能让感情更加稳定。

**第二，允许男人沉迷一些没有意义的小事。**  
比如男人把打火机拆来拆去，或者一股劲儿地打游戏。其实，男人往往透过这些癖好来达到心理缓冲。当男人得到女人对小癖好的允许时，他们会感到被接纳。  
**核心策略**：`表示兴趣 = 即使讨厌，也尝试了解和讨论`。  
男生希望女生对他们感兴趣的东西感兴趣。所以，就算你讨厌他最喜欢的运动，也可以跟他一起看比赛、讨论，告诉他你对他有兴趣的东西也有兴趣。

**第三，能让男人和朋友们消磨时光。**  
不要让他舍弃和朋友一起的时间来陪你，他会感觉自己被困住了。如果他真的喜欢你，他会为你找时间的。在男人身上背负着工作和家庭的双重压力，他们缓解压力的方法之一就是和朋友一起消磨时光，这从心理上是少年时逃避母亲过分关爱的心理再现。因此，女人要学会理解他。

**第四，为他做一点小事，表达关注与支持。**  
让他感觉到你已经注意到他对你的感觉，并且愿意回应。但是不要做得太过分，让他觉得不好意思，点到为止即可。  
**具体做法包括**：在他无聊时主动陪伴；帮他分担一点工作或家务；给他带一杯咖啡或零食；在他心情沮丧、身体不适或压力大时，支持并安慰他。

**第五，给男人面子。**  
男人非常需要别人的尊重和感激。不管他心里怎么想，你的尊重可以加强他本身的信心。  
**具体做法**：以轻松诙谐的态度面对事情；在社交场合，如果他不在行，你可以大方地站出来介绍、寒暄，为他解围。他会十分感激你。

**第六，给男人表现的机会。**  
保护女人、展示强大魅力是男人天性中的一部分。因此，女人应该时不时让男人看到自己柔弱的一面，以激发他想照顾你和与你更加亲密的念头。  
**具体做法**：在一些事情的决定上让他做主，最后感激他；找他帮你修理东西；征求他的意见，比如旅游该穿什么衣服。

**第七，允许男人做自己。**  
男人在和女人交往期间，总希望做原来的自己。如果你坦白地跟他说，不希望他因为你而强迫自己改变，他会觉得你了解他，而不是想极力改造他。于是，他会更加全心全意地投入到爱当中。

**第八，尊重男人的隐私。**  
男人非常注重自己的隐私。不要以为你是他的女朋友或妻子就可以随意翻他的抽屉、查看他的手机。男生都喜欢值得他信赖的人。  
**核心原则**：`保密 = 信任`。  
说话前需谨慎，不要把你们的私密谈话细节告诉闺蜜，这样男生会更信任和尊重你。

**第九，让男生看到自己的价值与独特面，并用温柔体贴感化他。**  
男人好面子，有虚荣心。当他看到自己的女人优秀、有魅力时，他也会觉得脸上有光。  
**具体做法**：展示你独特有价值的一面，如厨艺、才艺；注重个人形象与卫生；保持真诚的微笑。  
同时，男人需要女人的体贴和温柔。在生活当中不妨展示自己温柔女性化的一面，可以帮他购买衣物护肤品，弥补他粗枝大叶的一面。
