# 产后修复瑜伽：01：盆底肌修复基础 🧘‍♀️

在本节课中，我们将要学习产后修复的核心内容——盆底肌修复。我们将了解盆底肌的位置、功能，以及它为何在孕期和产后会变得松弛。课程将包含寻找盆底肌感觉的方法、适合产后初期练习的温和动作，以及一些重要的注意事项，帮助初学者安全有效地开启修复之旅。

## 盆底肌是什么？

上一节我们介绍了课程主题，本节中我们来看看盆底肌的具体定义。

盆底肌是位于骨盆底部下端的一组肌群。它也可以被看作一整块软组织，形状类似一张“吊床”。我们的骨盆有前后左右四个端口，盆底肌就位于下方，在子宫最下方、阴道稍上方的位置。它像吊床一样，承托着盆腔内的所有内脏器官，并由四个角支撑在骨盆下缘。

## 为何需要修复盆底肌？

了解了盆底肌的构造后，我们来看看它为何在产后需要特别关注和修复。

在怀孕期间，随着胎儿成长，腹腔压力逐渐增大，盆底肌会因此被向下压迫，慢慢变得松弛。分娩时，无论是顺产还是剖腹产，盆底肌都会受到损伤。顺产时，阴道会扩张数百倍以让胎儿通过；而剖腹产虽不经过产道，但孕期持续增大的腹压同样会导致盆底肌松弛。因此，所有产后女性都需要进行盆底肌修复。

盆底肌松弛可能带来以下问题：
*   **阴道松弛**：影响生活质量。
*   **恶露排出不畅**：产后子宫内的残留物可能滞留。
*   **妇科感染风险增加**：松弛形成的褶皱易藏匿细菌、毒素。
*   **压力性尿失禁**：表现为大笑、咳嗽时漏尿。
*   **骨盆难以复原**：导致髋部变宽、腹部突出。

## 谁需要练习盆底肌？

盆底肌修复并非产后妈妈的专利，其必要性广泛存在于各个阶段的女性，甚至男性。

**核心原因在于**：不正确的姿势（如刻意翘臀）、提重物、年龄增长导致的胶原蛋白流失、以及妇科检查等，都可能损伤或弱化盆底肌。对于男性，盆底肌无力也可能导致尿频、尿急或性生活方面的问题。因此，这是一项终身受益的练习。

## 如何找到盆底肌的感觉？

理论部分已经明确，现在我们来实践如何启动这块肌肉。

寻找盆底肌收缩感觉是修复的第一步。请在练习前排空膀胱。

**方法一：中断排尿法**
在小便时，尝试中途憋住尿液，中断尿流。此时所用到的肌肉就是盆底肌。**注意**：此法仅用于寻找感觉，切勿作为日常练习，以免导致排尿问题。

**方法二：仰卧收缩法**
如果无需排尿，可采用以下步骤：
1.  仰卧，双脚踩地，分开与髋同宽，脚尖朝前，脚跟距臀部约一个脚掌距离。
2.  呼气时，轻轻收缩肛门和阴道周围的肌肉（会阴部），感觉像将它们向上向内提起。同时，微微卷动尾骨朝向肚脐，臀部可能轻微抬离地面。
3.  保持收缩3秒钟。
4.  吸气时，先将身体放松落回地面，再完全放松盆底肌。
5.  重复5-10次，专注于收缩和放松的感觉。

**辅助方法**
如果感觉微弱，可以：
*   在私密空间，将干净的手指放入阴道，感受阴道壁轻轻夹紧手指的感觉。
*   仰卧时，在双膝之间轻夹一个瑜伽小球或砖块，帮助大腿内侧微微内收，以辅助感知盆底肌发力，但注意避免大腿过度用力。

## 产后初期修复练习

找到了盆底肌的感觉后，我们学习产后初期（特别是月子期间）可以安全进行的练习。

**重要前提**：顺产有侧切的妈妈，需根据伤口愈合情况决定开始练习的时间，通常建议产后第二周开始。剖腹产妈妈在伤口疼痛期，可只做盆底肌收缩练习，暂不配合卷尾骨等腹部动作。

**练习一：仰卧腹式呼吸**
此练习可在产后第二天开始，旨在温和地恢复元气、刺激腹直肌。
1.  仰卧，双腿伸直，双手轻放于小腹。
2.  吸气时，感受腹部自然微微隆起。
3.  呼气时，腹部缓慢内收，双手随之下压，将气息慢慢排出。
4.  保持呼吸均匀绵长，剖腹产妈妈尤其需轻柔。随时可练习。

**练习二：盆底肌电梯练习**
在腹式呼吸熟练后，可加入盆底肌收缩。
1.  吸气，腹部隆起，完全放松盆底肌（电梯在1楼）。
2.  呼气，收缩盆底肌，想象它像电梯一样匀速上升。初期可升至小腹（3楼），保持3秒。
3.  吸气，将盆底肌如电梯般匀速放松下降回1楼。
4.  随能力提升，逐步延长保持时间（如4秒），并尝试将“电梯”升得更高（如肚脐、胸口）。

**练习三：喉式呼吸（乌加依呼吸）**
此呼吸法能帮助在收缩盆底肌时保持呼吸，避免憋气，同时恢复心肺功能。
1.  双手交叉环抱肋骨两侧。
2.  吸气时，感受肋骨向两侧扩张，腹部保持不动。气息经过喉咙时，会发出轻柔的“嘶”声。
3.  呼气时，肋骨内收，同样保持喉咙的轻微声响，气息摩擦上颚和喉咙后部。
4.  熟练后，可在盆底肌收缩练习中配合此呼吸。

## 收腹带的正确使用

一个常见的误区是收腹带的使用方法，错误使用会加重盆底肌压力。

**正确绑法**：收腹带应主要包裹住髋关节和骨盆区域，从会阴部下方开始向上托持，目的是承托因胎儿娩出而突然失去支撑的内脏，稳定骨盆和髋部。

**错误绑法**：将收腹带绑在腰腹部（胸廓下方）。这样会强行勒回腹部，反而增加下腹压力，导致盆底肌被向下压迫，造成二次伤害。

**注意**：收腹带主要在产后初期使用以缓解内脏下垂，出月子后通常不再需要。非产后人群无需使用。

## 基础盆底肌强化动作

当身体逐渐恢复（如出月子后），可以开始以下强化练习，这些动作也适用于所有需要锻炼盆底肌的人群。

以下是四个基础动作：

**动作一：站立提踵**
1.  山式站立，双脚与髋同宽，脚尖平行向前。
2.  双手十指交扣，翻转掌心向上推，沉肩。
3.  呼气，收盆底肌，同时缓慢提起脚跟。
4.  吸气，脚跟落地，然后放松盆底肌。重复10-15次。

**动作二：猫式变体**
1.  四角跪姿，双膝间夹砖或小球。
2.  吸气，抬头塌腰，延展脊柱。
3.  呼气，收盆底肌和腹部，拱背向上，眼睛看肚脐，双膝轻轻内夹。重复10次。

**动作三：下犬式变体**
1.  从四角跪姿进入下犬式，双膝间夹球，手脚距离可稍宽。
2.  吸气，伸展脊柱。
3.  呼气，微屈膝，脚跟抬起，臀部向上提，同时收缩盆底肌。保持3秒。
4.  吸气，脚跟回落。重复10次。

**动作四：动态桥式**
1.  仰卧，屈膝踩地，双膝间夹球，双脚与髋同宽。
2.  呼气，收缩盆底肌和腹部，将臀部抬离地面，进入桥式。
3.  吸气，臀部有控制地落下。重复10次。**注意**：恶露未排尽时不宜练习此动作。

## 深度激活与放松

完成主动练习后，可以通过一些对抗练习进一步激活盆底肌，并以放松体式结束。

**动作：仰卧手膝对抗**
1.  仰卧，将右脚踝放在左膝上方。
2.  身体转向左侧，左手推住右膝外侧，右膝则对抗左手向天花板方向用力。
3.  呼气时，对抗发力，同时强烈收缩盆底肌。
4.  吸气放松。每侧完成5个呼吸后换边。

**最终放松**
练习结束后，以仰卧束角式或简单的仰卧放松姿势结束，让身体完全休息，感受盆底肌的轻微酸胀感。

## 总结与要点回顾

本节课中我们一起学习了产后盆底肌修复的全套基础方法。我们明确了盆底肌的位置与功能，理解了修复的必要性，并掌握了从感知、初期呼吸练习到基础强化动作的一系列安全步骤。

**核心要点回顾**：
1.  **人人需要**：盆底肌健康关乎所有年龄段和性别人群的生活质量。
2.  **安全第一**：产后练习需根据分娩方式和伤口愈合情况循序渐进，切忌冒进。
3.  **感知为先**：正确找到并孤立收缩盆底肌是有效练习的前提。
4.  **呼吸配合**：避免憋气，学会在收缩时保持呼吸（如喉式呼吸）。
5.  **工具善用**：正确使用收腹带，避免绑错位置。
6.  **持之以恒**：盆底肌练习可融入日常生活，随时进行，贵在坚持。
