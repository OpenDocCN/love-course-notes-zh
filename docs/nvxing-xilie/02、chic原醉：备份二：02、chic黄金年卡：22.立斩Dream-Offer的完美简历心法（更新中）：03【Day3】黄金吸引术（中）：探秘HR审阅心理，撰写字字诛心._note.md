## 自我评价：简历的“摘要”与“序言”

在HR明确了应聘者的岗位意向后，他们的第三个习惯性动作是查看“自我评价”部分。某知名互联网公司招聘负责人Q先生对此的解释是：自我评价对于简历的作用，好比一篇论文的摘要或一本书的序言，是集合简历精华于一身的部分。HR能借此快速预判应聘者与岗位的契合度。

然而，许多应聘者未能理解其重要性。在自制Word简历时，常出现遗漏此项，或将其置于简历末尾的情况，这属于本末倒置。

**自我评价的核心作用**是起到简历内容概览与先知的作用，便于HR快速建立对应聘者综合素养的画像。

以下是撰写自我评价时常见的误区与正确方法：

*   **常见误区**：将此处当作生活化自我介绍平台，填写与求职关联度低或过于宏观的信息。
    *   例如：“爱好爬山、旅行”、“性格外向、有团队协作精神”。
*   **正确方法**：自我评价应聚焦于职场专业素养，需包含三个核心要素：
    1.  **专业**：体现你的知识底蕴与基本功。
    2.  **经验**：证明你的实操能力、问题解决能力与岗位价值。
    3.  **优势**：展示你区别于其他竞争者的核心竞争力。

这三个要素组合在一起，便能清晰描绘出你的岗位能力画像。

**示例**：一位应聘UI设计师的求职者，其自我评价可撰写为：
> 毕业于广告设计专业，拥有4年UI设计经验，负责过小红书、潮汐等App的移动端与PC端产品设计。熟悉产品开发流程，能基于用户理解制定解决方案。具备完备的产品UI/交互规范作品集，美术基础扎实，对设计趋势嗅觉敏锐，作品视觉冲击力强。

这样的描述，系统呈现了专业度、实操能力与个人优势。

## 工作经验：HR最关心的“重头戏”

在审阅自我评价后，HR的第四个脚印会迈向“工作经验”。超过98%的HR表示，这是他们最为关心、审阅耗时最长的部分。企业期望通过丰富的工作经验，降低新人的磨合成本。

工作经验可根据个人情况细分展示，以更清晰地呈现你的能力图谱：

*   对于有一年左右经验者，可组合为：**工作经历 + 实习经历**。
*   对于常接触项目者，可组合为：**工作经历 + 项目经历**。

撰写工作经验时，有四个必不可少的要素，以下是具体解析：

1.  **时间段**：HR据此初步判断你的工作稳定性与对岗位的熟悉度。通常，每份工作持续3-5年会被视为稳定性较高。
2.  **公司名称**：有助于HR判断你在行业中的位置与职业选择。名企经历可为你的能力背书，但非名企经历也无需隐藏。例如，创业公司经历所体现的“从0到1”的全盘思维与执行力，可能成为独特优势。在职跳槽若需保密，建议使用招聘网站的屏蔽功能，而非在简历中隐藏公司名。
3.  **具体岗位**：清晰定义你在企业中的角色（如“活动策划专员”与“文案策划主管”工作重点不同），避免让HR猜测，提升审阅效率。
4.  **核心工作内容**：这是整个简历的重中之重，详细展示你的职责与成就。我们将在下节课深入探讨其撰写技巧。

本节课中我们一起学习了HR审阅简历心理地图的第三与第四步。我们明确了“自我评价”应作为专业能力的概括总结，需包含**专业、经验、优势**三要素；同时了解了“工作经验”部分的构成与撰写要点，其四要素为：**时间段、公司名称、具体岗位、核心工作内容**。
