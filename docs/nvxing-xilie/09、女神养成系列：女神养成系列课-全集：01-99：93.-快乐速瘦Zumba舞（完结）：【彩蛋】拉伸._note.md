# 快乐速瘦Zumba舞：93：【彩蛋】拉伸 🧘‍♀️

在本节课中，我们将学习Zumba舞蹈后的拉伸放松动作。拉伸有助于缓解肌肉紧张，促进恢复，让身体从高强度运动中平稳过渡。

上一节我们完成了Zumba舞蹈的主体部分，本节中我们来看看如何进行有效的拉伸放松。

以下是腿部拉伸的步骤：
*   双脚并拢站立。
*   将一只脚脚背搭在另一条腿弯曲的膝盖上。
*   用手抓住抬起脚的脚踝，保持身体平衡。

拉伸时，保持呼吸平稳，感受小腿后侧的伸展。

接下来是手臂与上半身的拉伸。
*   将一只手臂伸直，横过胸前。
*   用另一只手轻轻按压伸直手臂的肘部后方。
*   换另一侧手臂重复上述动作。

这个动作有助于放松肩部和上背部肌肉。

在拉伸过程中，请保持动作舒缓，每个姿势维持15-30秒，避免弹震式拉伸。

本节课中我们一起学习了Zumba舞后的关键拉伸动作，包括小腿拉伸和手臂交叉拉伸。规律的拉伸能有效提升柔韧性，减少运动后的肌肉酸痛，是健身过程中不可或缺的环节。
