# 算爱研习社2020年费会员（更新中）：1主课（专题）：07脱单【22节】·十五天脱单攻略：03如何克服“恐男”魔咒 🧙‍♀️

在本节课中，我们将要学习如何克服“恐男”心理，特别是如何鼓起勇气进行搭讪。我们将深入分析阻碍你行动的心理与社会原因，并提供一套循序渐进的训练方法来提升你的“齿商”，让你能够自信地迈出认识心仪对象的第一步。

上一节我们介绍了无需搭讪也能认识男人的方法。本节中，我们来看看如何通过主动搭讪来结识异性。

说到搭讪，市面上有不少所谓的搭讪心理学。毫无疑问，这些内容大多是教男性如何追求女性的，仿佛搭讪是男性的专属行为。然而，我们前段时期的调查发现，女性想要主动搭讪他人的概率是48%，而男性仅为30%左右。这是因为在中国，男性的社交活动常常围绕异性展开，而女性的社交则更多围绕同性进行。

这件事可能很多人没理解。举个例子，男性从大学开始，他们的许多聚会、联谊、社团活动，甚至毕业后的工作和夜生活，都倾向于去有女性的场合，并经常互相介绍女性朋友。而女性聚会时虽然也常谈论男性，但真正去结识新异性的次数却很少。男性社交是围绕异性的，而女性社交更像是一群女性的内部活动。

对于女性而言，想要认识心仪的男人，搭讪可能占据了一半的机会。这意味着，如果你放弃搭讪，就等于放弃了一半找到心爱之人的可能性。而你一生中，能让你心动的人可能寥寥无几。

我曾经有一位助理，长相很好看，但也会有感情空窗期。有一次录影结束后，我们去杭州一家我很喜欢的餐馆吃饭。我正吃得高兴，一抬头发现助理满脸通红，双目含春。我回头一看，果然身后一桌有一位小帅哥在吃饭，他高高瘦瘦、白白净净，明眸善睐。我看他那桌吃得差不多了，就催促助理赶紧去搭讪，但她不敢。后来我提议我们一桌人一起帮她去搭讪，她还是不敢。就这样错过了。之后很久，她还经常向我提起那次邂逅，可惜再也没能遇见。

人的一生中，能让你面红耳赤的人可能只有几个。错过任何一次机会，都相当于错过了一段可能的人生。我后来思考，是什么阻碍了我的助理去搭讪？是自卑吗？她长得好看。是没勇气吗？她是白羊座，本应最敢冲锋。我后来领悟到，抗拒搭讪有着深层次的心理和社会文化缘由。正因如此，我们更要克服它。这并非因为你未来的丈夫一定来自搭讪，而是为了确保自己不错过幸福。我们错过的幸福，远比得到的更多。

那么，具体来看，为什么你们鼓不起勇气去搭讪呢？

以下是阻碍你搭讪的四个主要原因：
1.  害羞，脸皮太薄。
2.  不自信，自卑。
3.  高冷。
4.  要求太高。

这四个原因往往是阻拦你去搭讪的障碍。你可以先思考自己属于哪一种。作为一个诚意满满的课程，我肯定会把所有原因都分析透彻。如果不把原因解析明白，你们就不会知道自己出了什么问题。那样的话，即使我教给你搭讪的渠道和方法，你学会了却不敢用，还不如不学。

好，首先我们来看第一点：害羞。

在厚脸皮这件事上，我承认男性通常更厉害一些。我读书时，隔壁寝室有个兄弟很厉害。有一次他遇到我们学校的校花（传媒院校的校花特别好看），但想不到搭讪方法。最后他跑到路边捡了块砖头，一个箭步上前说：“同学，这是你掉的吧？”

而女性遇到让她脸红心跳的人时，第一反应往往不是“怎么要到联系方式”，而是开始害羞。对方帅气地出现，没看你也没撩你，你就开始害羞了，这到底是什么原因？

害羞可能是人类史上最严重的、让人裹足不前的心理现象。你明知道应该去做的事，因为害羞而不敢跨出那一步；你明知道应该去认识的人，因为害羞而不敢开口。

那么，害羞是如何形成的？它的心理成因是什么？有的学者说害羞是一种基因，来自遗传。社会心理学者说害羞是一种标签，是自我认同和他人认同的结果。我认为这些说法都不准确。

我告诉你，害羞由先天和后天两部分组成。

所谓的先天害羞，也就是天生害羞。其心理学成因是潜意识之下，内心激烈冲突而表现于外在。简单来说，就是你天生容易想太多。当你看到一个男生想去搭讪时，你会想：万一他不理我怎么办？万一他有女朋友怎么办？万一他嘲笑我怎么办？万一被熟人看到怎么办？万一他是个骗子怎么办？你只是看了他一眼，心里却已经上演了无数个故事，预想了许多会丢脸、伤心、难过的结果。这些念头一次次诞生，让你纠结、痛苦，自己和自己拧巴。一方面想去认识人，另一方面又被自己想象的场面吓到，觉得一定会丢死人。这就是许多人先天害羞的原因：想太多，自己吓自己。

另一方面，是后天的害羞。心理学上的描述叫“对社会事件恐惧感的病态表现”。解释一下，就是曾经被吓到过，现在再也不敢了。这种后天对社会事件的恐惧感也分几个层面。

有的人是以前搭讪时受过伤害，甚至被羞辱过，从而对主动表达、主动搭讪产生了终身的恐惧。这种情况很常见。很多时候，童年或读书时期的一件小事就会带来心理创伤。例如，有个女孩告诉我，她读书时特别喜欢班里的一个男生。有一次班里搞交换礼物的活动，她特别开心，精心准备了一个礼物打算送给那个男生。你们知道青春期的男生情商可能比较低。她送出礼物后，旁边有人起哄说：“哎呀，你怎么那么倒霉，收到她送的礼物？她长那么丑。”从此以后，那个女孩就特别自卑。这就是后天害羞的一个症状。

心理学上还有一个概念叫“心锚”。曾经发生过的事情带给你不可磨灭的记忆和伤痛，那么一些相似的场景和情绪会把那些伤痛重新带回来，使你放弃想做的事情。关于“心锚”，我们在未来关于恋爱、分手、失恋的课程中会详细讲。

这里着重讲一下后天害羞的第二个层面：社会文化对你的恐吓。也就是说，你对搭讪充满恐惧感，是因为社会文化对你进行了恐吓。这不关你的事。

我们中国的传统社会文化，对于“性”这件事带有强烈的羞耻成分。因为在传统文化里，认为“圣人无欲”，有欲望是件坏事。身体是私有的，给别人看是罪恶的。生殖和交配应该偷偷进行，光明正大地谈论则是道德败坏。在这样的社会文化下，男女之间的事情，我们还在讲“发乎情，止乎礼”。这些话出自孔子删定的《诗经》，他把“关关雎鸠，在河之洲。窈窕淑女，君子好逑”这样的诗句删掉了。孔子认为男女之情顺理成章地产生，但不要这么直接露骨，还要受到礼法的约束。

以前，男人和女人之间不能直接传递东西，必须有一个中介人过一手才行，否则就叫“私相授受”。男女之间也不能直接讲话，吃饭都要分席。一年里只有元宵节相当于相亲大会，这一天大家才可以抛头露面上街玩耍。所以，这是中国有史以来最早的、对于男女（尤其是女性）性道德的约束。男追女尚且如此，更不用说女追男了。你可以想，但不能真动手。

我们现在看这个问题，觉得这是封建思想。但这个思想潜移默化地约束了中国女性上千年，让我们始终处于“男人可以撩妹子，但女人不可以撩男人”的状况下。这让你觉得，一个大姑娘家去搭讪男人是有羞耻感的。这种羞耻感不是事情本身带来的，而是刻在你脑子里的价值观、社会文化和家庭教育恐吓你，让你觉得这件事是羞耻的。

但女人搭讪男人这件事，真的羞耻吗？好，我们来讲一下这个问题。

孔子说“乐而不淫”，意思是喜欢一个人可不能真下手，情感得有节制，“止乎礼”，要合乎礼法，让男人主动上门来提亲。但这就像我特别喜欢吃一个东西，还不能真去吃，得等这东西主动送到我嘴里来。我不付钱，人家能给我吃吗？

现在是什么时代？但凡好男人里有点好货色，都被人一阵疯抢。同学们，我告诉你这个时代有多残酷。我是79年出生的，我读大学的时候，毕业季哀嚎一片，除了单身狗都去找工作，教室里三三两两的情侣相对双泪流，那时的毕业季叫“分手季”。可现在新一代的小朋友厉害了，进大学前就被父母教育“千万不要当剩女”，还有我这样的人整天说“赶紧谈恋爱去吧”。现在的毕业季再也不是分手季，而是结婚季。你知道吗？现在姑娘一毕业就抓着大学里的男朋友去登记。想考研？可以啊，先结婚嘛，反正考研连孩子都能生。你还在想“发乎情，止乎礼”，人家连孩子都生了。

曾经的道德观到现在已经不太适用，因为时代变化太快。10年前，“约炮”还是一个贬义词，大家都不能说。现在呢？孩子们都可以公开讨论了。没错，谈恋爱就是寻找一个固定的伴侣，结婚就是长达一辈子的承诺。既然如此，为什么你连去搭讪一下都会觉得害羞呢？

我们再来讲一下第二点：脸皮薄，怕丢脸。

你知道人为什么会脸皮薄、怕丢脸吗？很多心理分析说，脸皮薄是一种社交障碍，是因为人缺乏自信。这也是胡扯，大错特错。很多脸皮薄的人非常有自信，在同性甚至异性面前都自信满满。但如果让他去搭讪某个人，或者在某个他喜欢的人面前做事，瞬间就怕丢脸了。有些心理分析师喜欢把这个问题归结于“你没自信”，都是胡扯。

脸皮薄、怕丢脸的真正原因是什么？不是没自信，而是因为你的**过度自我**。当一个人总期盼着自己在别人面前保持一个完美的形象时，他才会时刻注意不让自己丢脸。“丢脸”、“出丑”这样的词汇描绘的都是一个意思：你害怕自己做了某件事情之后，你的形象就不再完美了，会让别人对你有看法。这种看法可能是奚落、嘲笑、鄙视等等。

那什么样的人受不了别人这样的看法呢？过度自我的人。简单一句话来说：你把自己看得太好了，或者你希望自己能在别人面前维持住一个非常好的形象。

弗洛伊德讲“本我”、“自我”、“超我”。当“本我”产生一个强烈的欲望时，比如你很喜欢那个小帅哥，想去撩他，这是强烈欲望。在这个时候，你的“自我”就出现了。“自我”在那咆哮：千万别去，去了会丢脸，去了丢死人了。这就是“自我”对于“本我”的保护。你的自我意识牢牢压抑住了你的欲望，束缚住了欲望，使得你永远都不可能做出让自己有任何丢脸风险的事情。因为强大的自我意识把这个人牢固地保护了起来，像一座城堡。外面的任何风吹草动，自我就会觉得特别危险，千万不能放进来；而里面的欲望，自我就会觉得太丢脸，千万不能放出去。所以，自我意识已经成为了你最大的敌人。

害羞和怕丢脸连在一起，就是羞耻观。当然，很多姑娘会说：有羞耻观是好事情嘛，每个人都应该有羞耻观啊。但你的问题是**齿商太低**。

“齿商”这个词是我造的，出自二次元的“尺度”这个词。“尺度”是“羞耻的尺，温度的度”，本来是“尺度”和“羞耻”的合体版本，意思是让人感觉到羞耻的程度。“智商”是你智力高低的程度，“情商”是你情绪管理的能力，而“齿商”就是字面意思：你能够忍耐羞耻的能力。

现在很多姑娘“尺度”也许很高，但“齿商”非常低。她们可以看特别“污”的节目，可以说很“污”的事情，但要让他们做一点点羞耻的事情，她们就完全受不了了。这就是尺度高而齿商低。

齿商高是怎样的呢？最典型的代表是**袁湘琴**。你们不要以为袁湘琴只是书里写的，其实作者多田薰的真实人生经历就是如此。她自己长相普通，但她老公当年是一个玩乐队的小鲜肉，特别帅，有很多追求者。经过一段艰苦卓绝的“女追男”时光，两人才走到一起。所以，如果你去看80年代日本版的《恶作剧之吻》，你会发现直树这个角色特别冷酷，因为这就是作者当年对自己老公的印象。而她就是袁湘琴，她就可以为了喜欢的人，不害怕一次次的失败和丢脸，最终收获了甜蜜的爱情。

不过，也有姑娘会说：想维系别人眼中的完美形象有错吗？我来跟你分析一下。

首先，你一直在苦苦维系所有人眼中的完美形象，这种行为在别人眼里本来就是很傻的行为。只有一种人是永远不丢脸的，那就是硬撑的人。你以为别人看不出你硬撑吗？看不出你死活要让自己显得特别没问题吗？大家都能看出来，不仅能看出来，还能看到你的咬牙切齿，看出你对于欲望的抗拒。大家都看出来了，你还硬撑着，不是显得很可笑吗？

然后，谁又告诉你完美形象才是对的呢？心理学里有一个叫“仰八脚效应”，说的是完全没有缺点的人，反而不会让人喜欢，因为别人感觉不到你和他有共同点。有一个心理学实验讲这个效应：让几个人一起上台做演讲，三个人水平差不多，但第三个演讲者打翻了桌上的水，并向全场道歉。结果，所有听众的评分都给了第三个演讲者。这也是为什么失败者反而能得到更多同情的原因。

我在长期对星座做分类研究时，提出了一个叫“双鱼座效应”。双鱼座效应是什么呢？就是越是可怜兮兮、装无辜的人，得到的就会越多。在群体里处于最弱势位置的人，会成为群体里获取资源最多的那个人。这个效应最初是从研究喜欢装可怜、装无辜、喜欢“卖惨”的双鱼座身上得来。

我给你打个比方：你们闺蜜群里相互介绍男孩子，这时候有一个姑娘出来悠悠地说：“哎，我就不要了吧，反正也不会有男人喜欢我的。”你猜会怎么样？大家会把最优质的男人都介绍给她。

所以，你要那么完美的形象干嘛呢？《最好的我们》这部电视剧大家都看过吧？里面的耿耿就是典型的双鱼座效应。她因为成绩不好遭到同学嫌弃和嘲讽，没人愿意跟她坐一起。表面上好像出了丑，但更加激发起了余淮的保护欲，愿意跟她坐一块。然后，好不容易有一次发挥特长的机会——给日本使团拍照片，又搞砸了。因此，又是路星河给她“擦屁股”、收拾残局。你看，这就是越弱势的人就得到越多。

我之前说过一句话：在现代社会，人们追求成功比拼的已经不再是道德和能力，而是在比拼谁的底线更低一点。“厚黑学”到今天还被当做瑰宝，“皮厚心黑”是多么简单的攻略教程。我们被那些让厚黑学洗礼过的人，抢走了一切：更好的工作、更高的职务、更赚钱的事业、更好的男人。然后呢？看着他们风光无限地全球飞，你还是单身，被压在社会最底层吗？

我要求你们做的事情很简单，就是要你们提高自己的齿商，让你们不要那么容易感觉到羞耻。和陌生男人聊天、参加陌生人组织的活动都是很正常的事。你觉得做不到，不是因为别人有问题，而是你有问题：你的齿商太低了。

所以，从今天起，我要求你们去做一系列功课来提升你们的齿商。当然有姑娘会说了：提升齿商，那不就是变相的不要脸吗？你现在有男朋友了，你出轨了，那叫不要脸。可你现在连个男朋友都没有，让你去搭讪个把人，你都觉得是不要脸了。你这么难，怎么不穿越回古代去呢？别说《甄嬛传》了，就算《芈月传》，我估计你也活不过三集。

羞耻感是人类的本性。所谓“江山易改，本性难移”，本性是非常难改变的。别看我今天跟你说了半天，你好像听懂了也听进去了，甚至好像被我说服了，可你明天敢出去搭讪吗？恐怕不敢。

想要提高齿商，改变你的本性，真正需要的不是听课，而是用一系列的小训练来改变和强化。佛罗里达州立大学的心理学专家提出过一个概念叫“刻意练习”。这套理论的核心概念是：专家级的水准可以通过一系列的方法慢慢训练出来。这套理论后来被畅销书作家拿去，就成了一个叫“一万小时成天才”的伪理论——就是你花一万个小时在某件事情上就可以成为大师了。

但实际上，刻意练习的真正精髓，不是花一万个小时在某件事情上。我打个比方：做饭，你做番茄炒蛋特别好吃，也特别方便，所以你就天天做番茄炒蛋。可就算你做一万个小时的番茄炒蛋，顶多也就是一个番茄炒蛋的熟练工而已，并不能成为顶尖厨师。同样，你不搞懂核心理论，就算你用一万个小时在说话聊天上，最多也是个话痨，不会是演说家。

刻意练习的核心理念是什么？心理学家把人们的知识和技能分成了相互嵌套的三个圆形区域。

第一个区域叫**舒适区**，这里面是我们已经掌握了的知识和技能。你有没有发现，让你去做一件事情，你特别爱做那些自己会的、做得特别好的？你每天宅在家里不出门，这都是舒适区。

最外面那个区域叫**恐慌区**，是你目前完全无法学会的知识和技能。因为很多知识要循序渐进，要先学这个再学那个。直接给你最难的知识，你学不到，就会觉得恐慌、低落，信心受挫。

两者之间还有个区域叫**学习区**。这个区域里指的是你还没有掌握，但是通过努力却能学会的知识和技能。

刻意练习，不是让你去重复舒适区的内容，也不是让你去挑战恐慌区的内容，而是让你在学习区中不断学习，一步步掌握学习区的内容，并且将学习区向外拓展。

还拿做饭来比喻：番茄炒蛋是你的舒适区，你特别会。法式大餐是你的恐慌区，怎么也学不会。很简单，我们先从番茄炒蛋开始，然后进阶到糖醋排骨，下一步做粉蒸肉，再接着做虾饺等等，一步一步最后变成大厨。

我不知道你的齿商到底在哪个标准上，所以我们会把齿商的训练分成10个标准。当你可以完全脸不红、心不跳地做到某个标准之后，我们就接着去挑战下一个标准。

好，我来讲一下关于齿商训练的10个标准。

以下是齿商训练的十个阶段：

**齿商1.0：童话世界**
训练方法：盯着自己最爱偶像的半裸照看5分钟。如果脸红心跳，那第二天就要继续训练。如果盯着看5分钟一点事都没有，就算通过。

**齿商2.0：虚幻世界**
训练方法：独自看一部情色片（这里说的不是A片。日本那种没什么情节的叫A片；情色片是有故事情节的）。如果看的时候脸红了就没过关，第二天继续看，一直要到你能够正常面对为止。

**齿商3.0：友情世界**
训练方法：把你刚才看的那部情色片分享给朋友，至少分享三部以上。如果从中感到不好意思，觉得让人知道你看情色片不好意思，那就从头再来。

**齿商4.0：路人世界**
训练方法：去外面的市场或商场买东西，并且和商贩讨价还价。讨价还价的过程必须超过5分钟。如果不到，就找下一家商贩继续，直至可以自如地讨价还价，毫无心理负担，就算过关。

**齿商5.0：偶遇世界**
训练方法：叫一辆网约车（如滴滴、优步等），然后主动开始和司机聊天，一直聊到目的地为止。这个阶段必须挑战5次以上，尝试遇到各种不同类型和脾气的司机。过程中如果有聊不下去的状况，就从头再来。

**齿商6.0：乱入世界**
训练方法：乱入你朋友和别人（相对你来说是陌生人）的对话，至少要完成三次。三次中任何一次让人觉得有尴尬的感觉，就从头再来。

**齿商7.0：开放世界**
训练方法：先确定你能接受的最大尺度的衣服，然后去买比这个尺度还要“大一个境界”的衣服（比如更短的裙子或裤子，更低的领口，露更多的皮肤）。然后穿出去到公共场合，直至有异性来搭讪，并毫无违和感地拒绝对方。如果失败，就一直继续下去。如果成功，则再放大一个境界的着装尺度，重新再来一次。

**齿商8.0：主动世界**
训练方法：主动去搭讪一个你有好感的男人（有好感很重要），并且要到他的联系方式。重复三次，确定自己可以毫无尴尬感就可以终止。只要过程中有尴尬或脸红，就从头再来。

**齿商9.0：拒绝世界**
训练方法：向你一直暗恋的对象表白，并且等待被拒绝。一般而言，对暗恋很久的人表白都会被拒绝。你要坦然接受这样的拒绝。如果你去表白的时候成功了，那就恭喜你了。

**齿商10.0：公众世界**
这个尺度太大了，我们就不要学了。

以上9个阶段，一般你能做到齿商8.0，就可以过得很好了。所以我们今天布置课后作业：首先，你给自己定个级，然后从自己这个级别开始，慢慢地往上做。到齿商8.0之后，就可以选择继续或者放弃。
