# 快消行业职场课：02：灵活的响应机制：供应链

在本节课中，我们将要学习快消行业供应链系统的核心构成与运作机制。供应链远不止物流，它是一个确保产品从生产到消费高效流转的复杂体系。

上一节我们介绍了快消企业的后端制造与供应部门。本节中，我们来看看构成供应链的三个关键职能：需求预测、计划与物流。

## 需求预测：市场的翻译官

一个工业化生产的公司，在生产和销售之间需要衔接环节。早期快消公司以产定销，即工厂生产多少，销售就卖多少。在物资紧缺时期，特色产品易出现供不应求，因此无需需求预测。

随着经济发展，物资日益丰沛，机器生产效率提高，市场普遍供大于求。产品有保质期，仓储也产生费用，公司不能无限制生产库存。因此，生产运营面临核心问题：本月应生产多少产品？这需要倒推预测市场的总需求量。

消费者的需求看似难以预测，这是一个贴近市场但颇具挑战的职能。从实际运作看，它完全是为满足运营需求而存在的后端环节。

了解了这个职能的存在意义，接下来我们看看其具体操作方法。

需求预测最核心的能力是建模。模型的输出是公司每月的产品需求量。输入则包含三方面因素：
*   **历史数据**：公司过去的生产销售数据，是预测的重要基础。
*   **时间节庆**：例如春节等销售旺季，由于日期每年变动，是需要重点考量的输入。
*   **市场投入**：如促销活动、广告冠名等市场行为，都会对终端销售产生积极影响，是重要的影响因子。

不同的影响因子叠加，共同构成需求预测的整体输入。

那么，如何衡量需求预测工作的成效呢？自然是看预测准确度。销售发生前进行预测，销售完成后即可验证。通常，需求预测的准确度需达到**80%**以上。若误差过大，基于错误预测的原料储备和生产准备将严重影响运营效率。

## 计划：工厂的指挥官

了解了市场的“翻译官”——需求预测之后，我们接下来看看工厂的“指挥官”——计划部门。

计划是承接需求预测的下游部门。需求预测做整体市场判断，计划则需将整体需求细化到每个工厂、每条生产线。这并非简单地将需求平均分配。

以下是计划部门需要考虑的具体事项：
*   **产能限制**：工厂产能不会无限扩大。在销售旺季，产能常小于需求。计划需将旺季的部分需求提前分配至前几个月生产。
*   **工厂平衡**：需考虑不同工厂的地理位置与产能。例如，会优先用广东的工厂供应南方需求，而非河北工厂。
*   **生产线状态**：需考虑每条生产线的具体情况，如是否处于检修停产期，是否有工程改造项目影响生产等。

因此，计划是一个既需宏观统筹，又需微观周到的职能。

需求预测和计划部门对人才的考量类似，都要求极强的数据建模与处理能力。这两个部门的人才在培养中也经常相互流动。对于高阶人才，通常需要掌握这两个模块。

## 物流：产品的供应管道

最后我们来说一说物流。很多人认为物流就是运输，包括陆运、空运和海运。其实，物流除运输外，还包括库房仓储、客户服务等模块。

先来看物流工作的骨架，即一个快消品公司的工厂与大仓设置。工厂所在地决定了产品大致的辐射能力。全国覆盖能力越强的公司，工厂设置越密集。例如，红牛在北京怀柔、广东佛山、湖北咸宁、海南海口、江苏宜兴等地均设有工厂。

红牛的产品是饮料，运输成本高，一旦市场成熟，便会相应增设工厂。即便是糖果或干果等较轻产品的公司，也至少会在南北方分别设厂。

工厂是快消品公司的生产基地，大仓则是产品通往消费者路上的重要关卡。

物流的运输模型通常是：**工厂 -> 大仓 -> 各省市**。例如，一家公司在北京、浙江设有工厂，通常会在北京、沈阳、湖北、四川、浙江、广东等地设置大仓，作为辐射各大区域的基础。产品到大仓后，再分送至各个省份和城市。工厂到大仓的运输称为**干线运输**，大仓到省市的运输称为**支线运输**。产品就这样层层分配，输送到每一个门店。

物流职业人士需要掌握多方面技能：
*   **懂运输**：不同产品对运输条件要求不同。生鲜和低温产品（如酸奶、冰淇淋）要求尤其高。
*   **懂仓储**：懂得在何时需要新建大仓，如何实现仓库内货物的最快周转，如何设置合理的货物与人员动线。
*   **懂客服**：懂得处理产品运送不及时等投诉，与客户建立良好关系，在长期合作中提升双方效率与满意度。

物流职能通常要求专业对口，并有较长的基层培养期。只有通过长期基层历练，才能掌握该职能的不同模块。从能力模型看，物流人员需要较好的逻辑能力，擅长现场管理，并能与各类人员有效沟通。

## 职能前景与重要性

从薪资待遇看，这三个职能的入门水平相似。普通岗位入门年薪通常在**10万元**以内，职业生涯中位数大约在**20万到30万元**之间。但高级人才非常稀缺，大型公司高管年薪百万是常态。

很多人认为进入快消企业一定要去市场或销售部门，这才是核心。然而，随着新零售的发展，供应链的重要性日益凸显。

例如，生鲜公司产品保质期短，一旦需求预测出错或库存管理出现问题，将极大影响运营成本。因此，供应链职能至关重要。不止生鲜公司，现磨咖啡外送的瑞幸、中央厨房统一制作配送的Wagas餐厅等，其供应链都扮演着关键角色。营销固然重要，供应链同样是制胜点。可以说，现代零售业正将供应链从幕后推向台前，这个原本低调的职能部门正逐渐走到风口上。

## 总结

本节课中，我们一起学习了供应链的三个核心职能：**需求预测**是市场的翻译官，**计划**是工厂的指挥官，**物流**则是产品的供应管道。供应链职能在现代零售发展中的重要性日益凸显，其职业起点虽不极高，但发展空间巨大，在许多生鲜及外送公司中已成为举足轻重的部门。
