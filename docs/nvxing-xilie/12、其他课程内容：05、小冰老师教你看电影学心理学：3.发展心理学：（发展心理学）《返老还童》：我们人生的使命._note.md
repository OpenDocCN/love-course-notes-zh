# 发展心理学：05：我们人生的使命

## 概述
在本节课中，我们将通过电影《返老还童》的独特视角，结合心理学家艾里克森的发展阶段理论，来探讨人一生的核心使命与成长任务。我们将学习艾里克森将人生划分为八个阶段的理论，并理解每个阶段需要解决的关键矛盾。

## 电影《返老还童》简介
这部电影由大卫·芬奇执导，由布拉德·皮特和凯特·布兰切特主演。影片讲述了一个名叫本杰明·巴顿的男子的奇幻人生。

男主角本杰明·巴顿出生时，母亲因难产去世。他因一出生便呈现老年人的模样而被亲生父亲视为怪胎并抛弃。一位好心的黑人妇女收养了他，并在随后的岁月里抚养他、教育他成长。

本杰明的身体越长越像老年人，这使他常被同龄人嘲笑。但他的养母始终对他不离不弃，给予他爱与关怀，让他得以生存下去。在他11岁时，他遇到了生命中第二个重要的女性——当时年仅6岁、纯真可爱的小姑娘黛西。黛西彻底征服了巴顿的心，而巴顿同样真诚清澈的心灵也感动了黛西，两人萌生了爱慕之情。

两人很快分开。十几年后，二战爆发。本杰明和许多美国人一样，乘船从美国前往英国参军。二战结束后，他重返美国。此时他已不再是儿时那副老年人的模样，而是变成了一个帅气、魅力十足的中年人。他在纽约与黛西命中注定般地再次相遇。此时，他们的年龄和外表非常匹配。

他们在一起度过了幸福美好的几年，并结婚生子。但随着时间推移，事情变得不对劲。本杰明继续变得越来越年轻。在黛西生下孩子后，他害怕自己越长越年轻会对孩子的成长不利，便选择离家出走。

多年以后，他回到养老院度过余生，此时他的外表已是一个小孩子。黛西知道后，便经常去养老院陪伴他。最终，本杰明在变成一个婴儿后，在黛西的怀中去世，离开了这个世界。

这部电影一反导演往日的风格，被认为是他对生命与生死的思考。它像倒带一样，促使我们反过来思考自己的人生。

## 艾里克森的发展阶段理论
这部电影让我联想到心理学家艾里克森的理论。他的理论提供了一个独特的视角，如同从高空俯瞰，重新审视人生命的意义。

艾里克森将人的一生分为八个阶段。他认为，在每一个阶段，人都像打游戏一样，有一个固定的任务，即一个核心矛盾。这个任务和矛盾解决了，人格才能顺利发展到下一个阶段。如果任务没有解决，人格就可能存在一些不完善之处。

接下来，我们将详细讲解这八个阶段。

### 第一阶段：婴儿期（出生至18个月左右）
这个阶段的基本任务是**获得信任感，克服不信任感**。

所谓信任感，是指婴儿在受到母亲或其他照料者抚育时，感觉到自己所处的环境是安全的，周围的人是值得信赖的。这种对父母的信任会扩展到对一般人的信任。

**核心公式**：**信任感 = 安全的环境 + 可靠的照料者**

如果婴儿得不到周围人细心的照料和关注，就会对外界，特别是周围的人产生怀疑和害怕的心理。这会影响到下一阶段的发展。这个理论与之前提到的依恋理论有相似之处。

### 第二阶段：童年期（18个月至3岁左右）
上一节我们介绍了婴儿期建立信任感的重要性，本节中我们来看看童年期的任务。

这个阶段的人生使命是**获得自主感，避免怀疑感和羞耻感**。

到了这个阶段，儿童开始有了独立的要求，例如想自己穿衣服、吃饭、走路、玩玩具。他们开始认识到自己与父母是两个独立的个体，并乐于积极探索周围世界。

以下是培养孩子自主感的关键点：
*   如果父母允许孩子独立做一些力所能及的事情，并在他们完成后给予表扬，就能有效培养孩子的意志力，让他们获得自主感和控制感。
*   相反，如果家长对孩子过于爱护，事事包办，或者过分严厉，对孩子稍有差错就粗暴呵斥甚至体罚，孩子就会产生强烈的挫败感，进而形成自我怀疑和羞耻感。

### 第三阶段：学前期（4岁至5岁）
在建立了基本的自主性后，儿童进入了学前期。

这个阶段的人生使命是**获得主动感，克服内疚感**。

通常情况下，这个阶段的孩子开始上幼儿园，活动范围扩展到家庭之外。他们对周围环境充满好奇心，开始意识到性别差异，常常问这问那。

以下是发展孩子主动性的方法：
*   如果成人对孩子的探索不加以阻挠，而是鼓励他们参与各种活动，耐心解答问题，而不是嘲笑、禁止或指责，孩子的主动性就会得到进一步发展，表现出更大的积极性和进取心。
*   反之，如果父母对孩子特别苛刻，孩子就会认为自己的行为不好，提出的问题很笨，自己在父母面前总是不讨人喜欢的，从而产生强烈的愧疚感和失败感，影响今后的发展。

### 第四阶段：学龄初期（6岁至12岁）
随着孩子进入小学，他们迎来了学龄初期。

这个阶段的使命是**获得勤奋感，避免自卑感**。

这是人的智力，特别是逻辑思维能力得到显著发展的阶段。孩子提出的问题变得更广泛、更有深度，能力也得到快速发展。此时，对他们影响最大的不再是父母，而是同伴、邻居，尤其是老师。他们开始关心物体的构造、用途和性质，对各种技术产生兴趣。

以下是帮助孩子建立勤奋感的方式：
*   如果孩子的勤奋能得到成人的支持、鼓励和表扬，就能加强他们的勤奋感。
*   很多时候，父母可能将孩子的勤奋探索视为捣乱。这种想法很容易让孩子形成自卑感，觉得自己不如别人。

艾里克森认为，这个阶段的孩子非常需要鼓励。父母应该鼓励孩子努力完成任务、获得成功，激发他们的勤奋感和竞争心，鼓励他们尽最大努力进行社会交往，使他们相信自己是有能力的、聪明的，任何事情都可以通过努力做好，从而怀有成就感。

### 第五阶段：青春期（12岁至18岁左右）
度过以学习为主的童年，人便进入了充满变化的青春期。

艾里克森认为这个阶段至关重要，其核心问题是**形成自我同一性，避免角色混乱**。

这个概念在艾里克森的理论中非常重要，含义广泛。你可以将其理解为：个人需要与社会要求相统一，行为不能过于出格，需与社会规范保持一致；同时，也要保持自我认知（主体）与外界评价（客体）的统一，即清楚自己是什么样的人，扮演着什么角色。

这个阶段的人经常思考“我到底是怎样一个人”，会特别关注自己。例如，青春期的我们如果脸上长了一颗痘痘，都可能觉得天要塌下来，这是因为我们将关注点从外在回归到了自身。我们会从他人的态度、自己所扮演的各种社会角色中逐渐认清自己。

我们甚至会疏远父母，努力从对父母的依赖中解脱出来，更积极地与同伴建立友谊，从而进一步认识自己。

如果能保持这种同一性，顺利了解自己及自己与周围人事物的关系，就能成功进入成年期。否则，就会出现角色混乱，经常自我怀疑，做事马虎，因为理不清自己的努力与成就之间的关系。这些都是青春期遗留的问题。

### 第六阶段：成年早期（18岁至30岁左右）
在确立了自我同一性后，人便进入了成年早期。

这个阶段，艾里克森认为我们的使命是**获得亲密感，避免孤独感**。

简单来说，就是寻找一个人建立亲密关系，这种关系既包括友情，也包括爱情。

**亲密感**意味着你可以与他人同甘共苦、互相关怀，甚至共同承担义务。它是在共同完成某项任务中建立起来的联系，例如谈恋爱本身就是建立一个共同任务。

艾里克森认为，如果一个人不能与他人分享痛苦与快乐，不能进行思想情感的交流，就通常会陷入孤独、寂寞、苦恼的情境中。这与古人所说的“三十而立”很相似。

### 第七阶段：成年中期（30岁至60岁左右）
成功度过成年早期后，我们便进入了漫长而稳定的成年中期。

这个阶段的使命是**获得繁殖感（创造力感），避免停滞感（自我专注）**。

这个阶段的人有两种发展可能性：
*   一种朝向积极方向发展：个人开始关怀家庭成员、养育子女，并将关怀扩展到社会上其他人，从事公益活动，关心下一代及子孙后代的幸福。在工作上也勇于创造，追求事业成功。
*   另一种可能性是只顾自己和自己家庭的幸福，不顾他人的困难和痛苦。这类人很容易产生中年危机。

### 第八阶段：老年期（60岁以后）
最后，我们来到了人生的最后一个阶段——老年期。

这个阶段的人生任务是**获得完善感，避免失望感**。

如果在前七个阶段中，积极的成分多于消极的成分，那么在老年期就会汇聚成一种完善感。回顾一生时，会觉得过得很有价值、很有意义。

相反，如果消极因素多于积极成分，就会产生失望感，觉得自己一生失去了很多机会，走错了方向，想要重新开始又觉得为时已晚，因此痛不欲生，产生绝望感，精神萎靡，马马虎虎地混日子。

现在很多成年子女不知如何与父母相处。其实，最好的相处方式就是让父母察觉到他们这一生是有价值的，所得远大于所失。如果他们意识到了这一点，就可以很好地安享晚年。这绝不仅仅是给点钱、买点东西那么简单。

## 总结
本节课中，我们一起学习了一部电影《返老还童》，并深入探讨了艾里克森提出的自我发展八个阶段理论。他的理论具有相对的合理性，在整个心理学界影响很大。这个理论从生命全程的视角，为我们理解每个年龄段的成长任务和人生使命提供了清晰的框架。艾里克森的理论启发我们，人生的每个阶段都有其独特的价值和需要完成的课题，顺利度过这些阶段，才能获得完满的人生。
