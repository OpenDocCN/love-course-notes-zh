# 女神养成系列：05：色彩搭配魔咒：身上真的不能超过三色吗？ 🎨

在本节课中，我们将要学习色彩搭配的核心法则，破除“身上不能超过三种颜色”的流行迷思。我们将探讨如何通过专业方法，实现多色彩和谐搭配，让你衣橱里的所有颜色都能被自信地穿在身上。

很多人对色彩搭配既感到兴奋又感到担忧。兴奋源于色彩本身的美感。担忧则来自实际搭配时感受到的难度。今天要讨论的色彩魔咒是：身上真的不能超过三个颜色吗？

这个概念一直在流传。但我并不完全认同。观察近年的流行趋势和艺术形态，一套服装完全可以超过三种颜色。像Gucci等国际一线品牌，也经常演绎多色彩搭配。

那么，为什么在日常观念中，一套服装不宜超过三个颜色？主要原因是我们不擅长将不同色彩有效地组合在一起。

以红配绿为例。很多人认为红配绿很难看。但也有很多人认为红配绿很美。是红色不美，还是绿色不美？仔细想想，红色和绿色都很美。为什么红绿搭配有时美，有时不美？

主要原因在于我们没有掌握好的组合搭配方式。我们知道，不是色彩本身不美，而是缺乏优质的搭配手段和方法。从红配绿的例子可知，红色和绿色本身都美。搭配效果的好坏，主要取决于色彩之间最重要的比例关系，包括面积大小和视觉强弱。

所以说，没有不美的色彩，只有拙劣的搭配手段。今天我将为大家带来色彩搭配的方法。我们如何玩转衣橱里众多的色彩？例如今年流行的焦糖色等美丽色彩，我们喜欢，但如何穿到身上？今年流行的各种冰色系，如何搭配才能显得肤质更好、气质更佳？

接下来我们看两组搭配。每一套服装都超过了三种颜色。

在这两套超过三种颜色的搭配中，左边的搭配感觉更舒服。右边的搭配则让人觉得色彩有些混乱。左边的搭配中，我们能明显感受到有些色彩比较强烈，有些色彩则有后退的效果。例如，裙子的亮蓝色很前进，外套的蓝色处在其次，淡淡的草绿色则在最后面。

我们发现，色彩原来有前进和后退的秩序感。如何解释这个问题？玩过专业相机的人都知道“景深”这个概念。当我们拍摄具体人物时，人物是清晰的，背景景物则是模糊虚化的。配色中也要找到类似的方法，即我们常说的前进、后退、虚与实。

例如“万绿丛中一点红”，红是“实”，绿色作为“虚”的背景。这样搭配就会有主有次、有强有弱、有虚有实、有前进有后退。

再看第二个造型。玫红色的服装和蓝色的帽子都非常抢眼，同时跳入我们眼帘。因此，第二套配色缺乏虚实的视觉效果。

请大家慢慢感受一下。

再看后面这套搭配。我们发现，红配绿也能这么美，身上超过这么多颜色，效果依然不错。左边这套红配绿，其美感并非来自虚实，而是来自大小面积：大面积的绿，小面积的红。红色在视觉上略强于绿色。

在这套搭配中，运用了大小面积的比例法。绿色面积最大，其次是红色，最后还有蓝色的鞋和眼镜。这些色彩加起来达到或超过了三种，但我们依然能在搭配中找到某种秩序，获得非常舒服的感觉。

再看右边这套搭配。上下装呈现出1:1的色彩搭配关系。唯一值得赞赏的是其红色面积控制得还算小。在日常搭配中，上红下绿会带来1:1、没有主次之分的搭配效果。这种效果会大大降低搭配的视觉美感。

通过以上两组搭配，相信大家已经感受到，配色不仅仅关乎“红配绿还是红配黄”，更关乎一种“配色关系”。这种关系有强有弱、有虚有实。我们将这种关系总结为一种“秩序”。秩序和层次才能真正带来美感。

这里引出一个问题：美到底是什么？无论是配色，还是服装与配饰的搭配，秩序都至关重要。只要在配色中把握住“秩序”这个概念，无论有多少种颜色，无论是什么颜色相配，我们都能实现和谐搭配。

接下来，我为大家总结两种具体方法。第一种叫**面积法**。第二种叫**虚实法**。今天如果能掌握这两个方法，相信大家打开衣橱后，就能立刻对整套搭配做出美感判断。掌握第一步，就说明你已经开始慢慢找到搭配的美感了。

让我们一起来看一下吧。

## 方法一：面积法 🎯

通过刚才对红配绿的分析，我们发现红色和绿色要想搭配得美，可以设想：如果是一件大面积的红色风衣，搭配小面积的绿色耳钉，会不会觉得很美？在我们民族的着装中，红色外套搭配绿色的滚边和盘扣，是不是可以接受？在这个过程中，面积的搭配非常重要。面积的大小主次关系，在色彩搭配中能达到高度的美感和谐。

因此，当多种颜色出现时，运用这种面积大小、主次排列的方法至关重要。

大家记住了吗？好，我们来看这套搭配。

这位街拍女生的外套是橙色系，手包是黄色，裤装是蓝色，内搭是白色，还包括黑色的眼镜和鞋。在这套搭配中，色彩绝不止三种，但其面积的调整让我们看到：蓝色的面积最大，其次是外套的驼色面积。同时，包包和内搭的面积几乎相同。这就形成了主要的大面积色、其次的辅助色，以及最后点缀的最小面积色。我们称之为**色彩搭配的面积法**。

在这套搭配中，我们看到黄色的外套和黄色的包包，同时内搭是蓝色，鞋又出现了粉色和蓝色。整身搭配出现了三种以上的颜色，以黄色作为大面积色，蓝色和粉色作为小面积色，形成了面积法的效果。

**面积法**指的是：多色彩搭配时，一定要区分出大小面积的主次关系。这样，人的视觉在感知色彩美感时，会形成大小主次的分离，从而产生层次感，而层次感就会带来美感。

## 方法二：虚实法 🔍

上一节我们介绍了利用面积主次创造美感的方法，本节中我们来看看如何利用视觉的虚实关系来搭配色彩。

**虚实法**指的是色彩的前进与后退。在相机拍摄中，我们知道有“景深”：有的景物较远，呈现虚焦效果；有的景物靠前，呈现清晰实景效果。一清晰，一模糊。虚实法指的就是这种方法。

在色彩搭配中，我们可以利用这种虚实法来制造视觉的前进与后退关系。

我们来看一套搭配。

在这套搭配中，同时出现了黄色、白色、蓝色和黑色。很明显，黄色和白色作为“实色”在前进，而蓝色和黑色作为“虚色”在后退。也就是说，蓝黑色是陪衬色。因此，在我们的视觉中，就呈现出了前进与后退的主次关系效果。

在第二套搭配中，出现的颜色更多，如柠檬黄、玫红色、灰色、绿色、黑色等多个颜色。这里我们很快能找到“实色”：帽子的玫红色和外套的柠檬黄作为前进色，而灰色、绿色和黑色作为后退色。因此整套搭配的视觉层次相当好。这肯定是高手搭配出来的。

通过以上介绍的**面积主次搭配法则**和**虚实前进后退搭配法则**，我们发现，当身上出现超过三种颜色时，就需要利用这些美感的搭配方法来实现。

在搭配中，我们的身上不仅可以出现双色、三色乃至更多的颜色，最重要的还得看我们是否运用了专业的搭配方法来实现美感的协调原则。

## 总结 📝

本节课中我们一起学习了：
1.  **破除迷思**：“身上不能超过三色”并非铁律，多色彩搭配完全可以实现和谐美感。
2.  **核心概念**：搭配的美感源于**秩序**，即色彩之间的主次、强弱、虚实关系。
3.  **两大方法**：
    *   **面积法**：在多色彩搭配中，区分出**大面积主色**、**中面积辅助色**和**小面积点缀色**，形成视觉层次。
    *   **虚实法**：利用色彩的**前进感（实）** 与**后退感（虚）**，营造出类似摄影景深的主次视觉效果。

掌握这两个方法，你就能更自信地驾驭衣橱里的各种颜色，创造出既有层次又和谐的个人风格。
