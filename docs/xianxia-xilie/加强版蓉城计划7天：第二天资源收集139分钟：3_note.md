# 蓉城计划7天：第二天：资源收集与窗口管理

在本节课中，我们将学习资源收集的策略以及如何通过有效管理社交窗口来推进关系。核心内容包括模糊邀约、资源搜索技巧以及窗口识别与备注方法。

上一节我们介绍了资源收集的初步概念，本节中我们来看看当初步资源耗尽时，应如何继续推进。

## 模糊邀约的后续跟进

当通过常规渠道（如两个句号或两个加号代表的平台）询问后，对方均表示没有时间，我们仍需尝试邀约。此时，我们可以基于之前已经发出的模糊邀约进行再次沟通。

以下是具体操作步骤：

*   **触发对话**：发送诸如“说好的出来约会呢？”或“说好的出来做一做呢？”这类信息，重新激活对话。
*   **提出新提议**：可以尝试使用“出来溜达溜达”等更随意的说法进行测试。
*   **持续尝试**：关键在于持续管理对话，保持联系。

通过以上方式，我们可以将之前处于“模糊邀约”状态的对话，转化为更明确的约会契机。

## 窗口识别与备注管理

上一部分我们探讨了如何推进对话，接下来我们学习如何有效识别和管理对方的兴趣窗口，并做好备注。

窗口大小反映了对方的兴趣程度。我们根据反应强烈程度进行标注。

以下是窗口分类与备注标准：

*   **大窗口**：对方反应积极，兴趣浓厚。备注方式为 **`大窗口`**。
*   **小窗口**：对方有反应，但兴趣一般。备注方式为 **`小窗口`**。
*   **无窗口**：对方无明显兴趣。无需特别标注。

对于有窗口的联系人，还需在备注中记录关键信息，格式建议为：**`[城市][年龄]`**。例如：**`A6汉`** 表示来自武汉的26岁女性。

本节课中我们一起学习了资源耗尽的应对策略，即通过模糊邀约的后续话术重新激活对话。同时，我们也掌握了根据对方反应识别兴趣窗口（大窗口、小窗口）的方法，并学会了规范化的备注格式（如`A6汉`），以便高效管理社交资源。
