# 加强版蓉城计划7天：第二天：资源收集与形象管理

在本节课中，我们将学习如何高效地进行资源收集，并掌握使用手机软件进行个人形象管理的基础操作。核心内容包括朋友圈内容策略、必备软件介绍及其核心功能，以及如何拍摄合格的个人展示照片。

## 朋友圈内容策略

上一节我们介绍了课程的整体安排，本节中我们来看看朋友圈的管理原则。朋友圈是展示个人形象的重要窗口，内容需要精心设计。

以下是发布朋友圈内容时的几个关键原则：
*   发布的照片数量应尽量少。
*   避免直接转发他人或网络上的图片。因为不同手机客户端（如苹果与安卓）处理照片的方式存在差异，转发可能导致图片被压缩或带上平台水印，与他人的照片风格不一致，容易被识别出来。
*   整体朋友圈内容应保持风格统一。

## 必备软件清单

了解了基本原则后，接下来我们需要准备工具。以下是进行个人形象管理必须安装的四个手机软件，请确保你的手机中已备好。

以下是四个核心软件的名称：

3.  Instagram
4.  Snapseed

这些软件在苹果和安卓手机上的功能基本一致，主要差异在于不同社交平台客户端的显示效果。请大家先花几分钟时间下载并安装。

> **提示**：如果找不到或不会使用某个软件，可以随时私聊询问。Instagram的图标是白色相机，中间有彩虹色条纹；Snapseed的图标是一片树叶。

## 各软件核心功能解析

软件准备好后，我们来看看每个软件的核心用途。每个软件只需掌握其最擅长的1-2个功能即可，无需复杂操作。

以下是各软件的核心功能定位：
*   **美颜相机**：主要用于**自拍**。我们只使用它的拍摄功能。
*   **美图秀秀**：核心功能是**人像美容**。我们主要使用它来优化面部细节。
*   **Instagram**：主要作用是**添加白色边框**。使用其边框功能可以让照片排版更美观。
*   **Snapseed**：核心功能是**精细修图**。这款工具功能强大，需要一定的技巧，例如使用其“头部姿势”功能调整面部立体感，或修饰眉毛、唇色等细节。

## 合格自拍的拍摄要领

最后，我们来学习如何使用美颜相机拍摄一张合格的自拍照片。这是展示形象的第一步，需要掌握正确的姿势和角度。

以下是拍摄时的具体操作步骤：
1.  打开美颜相机的**自拍**功能。
2.  使用**前置摄像头**，确保视线与镜头平行，然后将手机镜头**稍微向下压一点**。
3.  人物在画面中的最佳位置是：从**肚脐往上**的**上半身**。
4.  头部顶端与画面上边缘应留有**一定空间**。
5.  人物最好位于**画面中央**。
6.  **切忌逆光**拍摄，应让光线从正面或侧面照来。
7.  拍摄时，按住快门**连拍一两张**。
8.  关键技巧：**多角度拍摄**。尝试左侧面、正面、右侧面等多个角度，每个角度都多拍几张，从中选出自己最满意、最上镜的照片。
