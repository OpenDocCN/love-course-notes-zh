## 应对初次抗拒：冷冻与抽离 ❄️

**核心操作公式：抗拒 → 立即抽离 → 保持距离 → 观察反应**

具体步骤如下：

1.  **立即停止**：停止一切肢体接触。
2.  **物理抽离**：起身离开，坐到房间的另一端或较远的位置。
3.  **转移注意力**：开始做自己的事情，例如看手机或电视，完全不理睬对方。
4.  **保持时间**：这个“冷冻”过程至少需要持续5分钟。

这样做的目的是通过你的行为，而非语言，向她传递一个明确信息：你对她刚才的抗拒行为感到不悦。由于之前你们已经建立了暧昧舒适的互动氛围，你的突然抽离会制造一种强烈的落差感。

这种“冷冻”策略之所以有效，是基于一个共同的心理机制。

冷冻大约5分钟后，你需要重新测试她的态度。通常有两种积极信号：

以下是不同约会结果的应对策略：
