# 浪迹导师培训计划：2：待遇、行业与职业发展 🎯

在本节课中，我们将要学习浪迹教育导师培训计划的核心内容，包括公司的现状、导师的职责、职业发展路径以及相关的待遇和合同问题。我们将以简单直白的方式，确保初学者能够理解每一个要点。

## 概述

目前，我们的团队虽然取得了一些成绩，但仅仅是起步阶段。随着知名度的提升，现有的100名导师已无法满足日益增长的需求。因此，我们需要更多志同道合的兄弟加入，共同发展这项事业。

## 导师职责与成长路径

上一节我们介绍了公司的发展需求，本节中我们来看看导师的具体工作与个人成长。

助教的核心工作是服务学员并解答疑问。如果回答错误，可能导致严重后果。因此，我们选拔助教的标准是踏实可靠，并且在情感交往领域具备足够能力。

以下是助教工作的关键原则：
*   必须确保回答的准确性。随意解答学员问题可能导致被开除。
*   遇到不确定的问题，应当向上级导师（如海科或我本人）请教。请教过程本身也是重要的学习机会。
*   通过每日处理大量学员案例，助教可以快速积累经验，深化对两性关系的理解。

加入公司只是一个开始，是迈入行业的第一步。职业成长道路漫长，并非经过7天培训就能立刻胜任所有工作。进步是在实际工作中逐步实现的，切忌好高骛远。

我个人用了7年时间才达到现在的水平。前两年非常辛苦，但通过我们的系统培训，新加入的兄弟可以快速掌握核心知识，避免走我当年的弯路。然而，某些更深层次的经验积累无法跳过，仍需亲身经历。

## 招聘计划与薪资保障

接下来，我们来谈谈具体的招聘计划和薪资待遇。

我们计划总共招聘20名助教，分为两个方向各10人。只要通过考核且表现合格，都有机会留下来。我们对人数没有严格限制。

关于薪资，我们有足够的能力支付。即便招聘20人，每月总薪资约6万元，我们只需招收少量学员即可覆盖这笔支出。因此，薪资发放是有保障的。

我们招募团队的目的不是为了从你们身上盈利。公司目前月营业额约900万，主要收入来源于私教服务等业务。招募大家是为了更好地服务学员、扩充团队、培养未来的核心力量。

## 未来规划与线下拓展

现在，让我们展望一下公司的未来规划。

我们计划在下半年于全国多个城市进行巡回演讲，目的是探索各地市场机会，并寻找合适的合作伙伴。长远目标是希望在各地建立分公司或俱乐部实体。

线下实体存在非常重要，它能消除网络的虚拟感，让学员觉得我们是一个可靠、有实力的团队。学员可以到线下场所交流、展示，这能极大提升信任度。

关于分公司模式，我们尚在探索中。理想情况不是在每个城市都从总部派遣资深导师，而是将当地优秀的兄弟培养成类似“老吴”这样的核心负责人，像种子一样在当地发展。

初期，我们的资深导师会到各地巡回演讲并开拓市场。参与其中的兄弟将有机会周游全国，快速增长见识和阅历，这是一种极佳的培养方式。

## 业务定位与服务体系

在明确了发展蓝图后，我们来看看公司具体的业务定位。

“浪迹天团”是一个私教服务团体，它弱化了单个导师的个人品牌（IP），强调以服务为核心。对于新关注我们的粉丝，他们可能不认识具体的导师，因此销售会推荐一个性价比高（定价12000元）、服务全面的私教套餐。

这个套餐提供全面的情感问题解答和课程指导，相当于一个有服务的在线网络课程。与过去2000元无服务的课程相比，现在的模式更细致，学员满意度更高。

“浪迹天团”定位偏向把妹和炫酷生活方式，而“饺子”团队则更偏向传统的挽回、狙击等业务，两者有区别。其他资深导师则继续运营其个人品牌的私教业务。

这种模式将过去分散的服务整合成专业的团队服务，让学员体验更好，也更利于口碑传播。正因为我们需要大量服务人员，且无法从路边随意招聘或挖角，所以才启动本次培训计划来培养新人。

## 合同、税务与社保问题

了解了业务模式，我们还需要明确一些重要的实务问题，例如合同与待遇。

实习期会签订实习合同。转正后，如需购买社保，可以由公司代缴，但费用需从个人薪资中扣除。

这样安排的原因是，培训行业目前存在一些不规范之处。例如，我们不向学员提供发票，是为了避免税费，从而能将更多费用分配给导师，提供更优质的服务。同样，对于导师，我们也希望你们能拿到尽可能多的实得收入。

我们通过专门的公关团队与税务部门维护良好关系，公司统一处理税务问题。因此，导师无需缴纳高额的个人所得税（例如月入5万需交30%的税）。薪资会以真金白银的形式全额发放。

签订合同主要是为了保障双方权益，明确如每月3000元底薪等条款。公司目前处于健康发展阶段，从未拖欠过任何员工的工资。

## 合作模式与个人发展路径

最后，我们来探讨一下与公司的合作模式以及个人的发展道路。

对于因故无法全职担任助教的兄弟，完成培训并通过考核后，可以获得公司认证的“PUA上岗证”。凭此认证可回到自己所在城市发展，但流量需要自行开拓。

如果想以“老吴模式”合作，主要有三种方式：
1.  **线下课程合作**：公司提供生源并补贴部分场地费，收入按比例分成。
2.  **私教服务合作**：公司分配私教学员，导师提供服务并获取提成（例如12000元课程提成32%）。
3.  **网络课程合作**：公司负责制作、宣传并销售导师的网络课程，导师获得销售额的提成（例如25%）。

“老吴”本人经历了约两年的成长周期：先作为学员实战并自发分享一年，证明能力后获得公司支持录制课程，最终转型为全职导师。你们的助教期（三个月）大大缩短了这一过程。

如果选择回到地方发展，可以在公司群内分享经验和成功案例，为公司宣传。当公司认可你的能力后，可能会为你推出课程并由公司负责招生。但请注意，如果利用公司资源为自己私下招生，则无法继续合作。

公司教授大家知识体系是毫无保留的。你们学成后，完全可以凭借自己的能力去开班教学。公司初期可以协助宣传，为你们在当地积累初始名气。但后续的流量和发展需要依靠你们自己裂变。

我们鼓励大家成为优秀的PUA导师。你们已经支付了3万学费，获得了相应的知识。之后凭借自身能力去赚钱，公司不会额外收费，但除了初期的宣传助力外，也无法提供持续的免费流量支持，因为公司的流量获取成本也非常高昂。

## 总结

本节课中我们一起学习了浪迹教育导师计划的多个核心方面：从导师的职责与成长，到公司的招聘与薪资保障；从未来的线下拓展蓝图，到“浪迹天团”的服务定位；最后明确了合同、税务问题以及多种合作与个人发展路径。关键在于脚踏实地，在服务学员中积累真才实学，并与公司共同成长。
