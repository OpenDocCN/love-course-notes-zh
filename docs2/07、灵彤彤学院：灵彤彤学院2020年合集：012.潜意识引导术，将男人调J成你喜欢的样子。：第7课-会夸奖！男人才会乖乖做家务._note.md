# 灵彤彤学院：012：第7课-会夸奖！男人才会乖乖做家务 🏠✨

在本节课中，我们将学习如何通过有效的沟通与引导，鼓励伴侣共同承担家务，从而建立更和谐、平等的家庭关系。

---

很多女性在婚后会发现，自己仿佛多了一个需要照顾的“儿子”。常见的情景包括：伴侣不断询问物品位置、催促做饭或抱怨衣物未洗。这些场景对许多已婚女性而言并不陌生。

婚后，特别是生育后，女性往往需要独自承担家务、育儿，甚至额外照顾伴侣，感到不堪重负。这引发了一个思考：是伴侣不够尽责，还是女性自身过于要强？

事实上，许多女性在婚前相当独立，能处理各种事务，不轻易求助他人。这种模式持续到婚后，尤其是成为家庭主妇后，容易因心理上的弱势而包揽所有家务，认为这是自己“分内之事”。短期内或许可行，但长期积累的委屈与辛劳终会爆发，而对方可能对此毫无察觉。

男性不做家务是许多家庭的矛盾来源。部分男性认为不做家务是天经地义的，但事实并非如此。家务看似简单，实则繁琐耗时，若由一人承担，十分辛苦。家庭事务本应是双方共同的责任。

上一节我们探讨了问题的普遍性，本节中我们来看看影响男性是否分担家务的几个关键因素。

决定男性是否能分担家务，主要与以下几个方面有关：

1.  **原生家庭模式**：他从小是否被培养做家务，是否有独立生活的经历。当前独生子女普遍缺乏家务经验，这种习惯会延续到婚姻中。
2.  **夫妻感情基础**：感情深厚的夫妻，在家务分工上通常更灵活、更不计较。
3.  **个人修养与处境**：个人修养好的人，即使感情一般，也不会过于推脱。有时，男性若在外界受挫，也可能试图在家中寻求掌控感。
4.  **客观条件与妻子引导方式**：包括工作时间等客观因素，以及妻子对待此事的沟通和处理方法。
5.  **经济收入观念**：部分男性因收入较高，或受大男子主义影响，认为赚钱多就不用做家务。

对于受原生家庭影响（因素一）的男性，需要理解其模式根深蒂固。他们可能真的意识不到问题。最佳方式是寻找合适时机，**清晰、平和、非指责地**提出你的需求，并先询问他自己的想法。

如果问题源于感情基础（因素二），那么重点应放在增进夫妻感情上。感情好的夫妻，家务分工通常更自然，做家务也成为表达爱意的方式。

那么，从妻子引导方式（因素四）的角度，我们具体该如何鼓励丈夫做家务呢？以下是几个核心方法：

**首先，公开赞扬，树立形象。**
在家人和朋友面前多夸奖丈夫的付出。例如，一位妈妈常在家长群夸赞丈夫是“模范丈夫”，这促使丈夫更愿意在大家面前表现，主动为家庭做些小事。这种公开的认可能激发他的积极性。

**其次，创造机会，全家参与。**
不要抱怨，而是以带动的方式邀请他加入。例如，大扫除时，可以兴致勃勃地给丈夫和孩子分配不同区域，全家一起动手。完成后，可以一起外出简单就餐，作为奖励。也可以通过家庭会议，在教育孩子分担家务时，自然地将任务分配给“最擅长”的爸爸。

**再次，及时肯定，遇事示弱。**
当他完成某项家务后，给予正面肯定。平时多示弱，请他帮忙，例如：“这个我真不会修，还是你厉害。”让他感受到被需要和价值感，帮助行为就可能从第一次、第二次逐渐成为习惯。

**另外，体验感受，促进理解。**
对于比较严重的情况，可以尝试角色互换，让他体验你的日常。或者找个周末，你带孩子回娘家，但记得要平和地离开，并电话关心他，给他机会独自处理家庭事务，学习“长大”。

**最后，保持经济与人格独立。**
拥有一份自己的工作至关重要，即使收入不高。这能帮助你保持独立人格、社交圈和时代思维。经济独立是与丈夫平等协商家务分工的重要基础。例如，可以商定“一三五我做，二四六换岗，周日休息”的模式。

引导男性做家务需要技巧，但很多女性常会使用错误的方式。以下是一些关键的注意事项：

**首先，多鼓励，少挑剔。**
只要他肯动手，就及时赞扬，不要嫌弃他做得不够完美。例如，丈夫第一次洗碗可能只洗了碗，锅和灶台都没清理。聪明的妻子会先肯定他“碗洗得很干净”，再悄悄收拾残局。**公式：行为 + 正面反馈 = 更高积极性**。相反，苛责会打击他的动力。

**其次，相信他的能力。**
要相信男性可以做好家务，甚至做得更好。家务既是细致活，也是体力活。给予信任和空间，他可能给你惊喜。男女在家务能力上并无本质区别，区别往往在于意愿和周围人的信任。

**再者，允许犯错，保持耐心。**
每个人都会犯错，男性刚开始做家务时，可能犯各种“低级错误”，例如晾了衣服忘记收、买菜需要清单等。此时，妻子需要多一些耐心和宽容，把这看作生活情趣，并温和地指导他如何做得更好。

**第四，避免反复抱怨与咒骂。**
反复的负面指责会破坏对方的心情和积极性，可能引发争吵，让他更不愿意参与家务。

**第五，在他做事时，不要喋喋不休。**
很多女性喜欢在丈夫做家务时指挥和吐槽。这就像在别人工作时不断打扰，会让人非常反感。将心比心，如果角色互换，你也会感到压力。

总之，家庭的维持需要双方共同努力。部分男性认为自己的工作能创造更大价值，回家理应休息。妻子需要让丈夫明白，为家庭付出与工作创造价值同样重要，且意义不同。家庭是最重要的投资，伴侣是人生最重要的支柱。

本节课中我们一起学习了通过公开赞扬、创造参与机会、及时肯定、保持自身独立以及注意沟通方式等方法，引导伴侣共同分担家务，从而构建更平衡、幸福的家庭关系。

---
**总结**：有效引导伴侣分担家务的核心在于**正面激励、信任尊重、平等沟通**。通过夸奖代替指责，用邀请代替命令，并保持自身的独立性，才能共同经营好家庭这项事业。
