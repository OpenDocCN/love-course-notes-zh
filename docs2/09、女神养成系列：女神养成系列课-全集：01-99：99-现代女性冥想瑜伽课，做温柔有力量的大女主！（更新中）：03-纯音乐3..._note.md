# 冥想瑜伽入门：03：呼吸与专注

在本节课中，我们将学习冥想瑜伽的基础——呼吸与专注。这是连接身体与意识、进入冥想状态的关键第一步。

上一节我们介绍了冥想的基本概念，本节中我们来看看如何通过呼吸来建立专注。

## 核心概念：腹式呼吸

冥想的核心在于将散乱的注意力收回到当下，而呼吸是最佳的锚点。腹式呼吸是一种深层的呼吸方式，其公式可以简化为：

**吸气 → 腹部鼓起；呼气 → 腹部收缩**

以下是进行腹式呼吸的三个步骤：

1.  找到一个舒适的坐姿，背部挺直，双手自然放在膝盖上。
2.  闭上眼睛，将注意力轻轻放在鼻腔或腹部。
3.  用鼻子缓慢吸气，感受腹部像气球一样微微鼓起；然后用嘴巴缓慢呼气，感受腹部自然内收。

掌握了基础的呼吸方法后，我们需要一个具体的方法来练习专注。

## 练习方法：计数呼吸

计数呼吸是训练专注力最经典的方法之一。其代码逻辑可以描述为：

```
while 练习中:
    if 注意力分散:
        温和地将注意力拉回呼吸
    吸气，心中默数“1”
    呼气，心中默数“2”
    ... 一直数到“10”后，再从“1”开始循环
```

以下是练习计数呼吸的要点列表：

*   数数只在呼气或吸气时进行，选择一种并保持。
*   如果中途忘记数到哪里或思绪飘走，只需从“1”重新开始。
*   不必追求完美的过程，觉察到走神并返回，本身就是最重要的练习。

在练习中，你可能会遇到一些常见问题。

## 常见问题与建议

初学冥想时，思绪纷飞、身体不适是正常现象。关键在于如何应对。

*   **思绪太多怎么办？** 不必抗拒或批评想法。只需在意识到走神时，像对待一个迷路的孩子一样，温和地将注意力带回到呼吸和数数上。
*   **身体坐不住怎么办？** 可以在臀部下方垫一个枕头，让骨盆略高于膝盖。也可以选择坐在椅子上练习，确保双脚平放地面。

本节课中我们一起学习了冥想瑜伽的基石——腹式呼吸与计数呼吸法。你掌握了通过呼吸锚定注意力的方法，并了解了如何应对初期的挑战。请记住，冥想是一场关于觉察与回归的练习，而非思维的战斗。每天坚持5-10分钟，你便能逐渐感受到内心的平静与力量。
