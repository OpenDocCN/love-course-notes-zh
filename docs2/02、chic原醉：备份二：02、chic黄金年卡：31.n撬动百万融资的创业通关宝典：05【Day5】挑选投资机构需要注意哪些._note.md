# 创业资本论：02：挑选投资机构注意事项

在本节课中，我们将学习如何挑选合适的投资机构。我们将从投资机构的品牌、背景、专业性和过往业绩等多个维度进行分析，并提供具体的筛选方法和建议。

上一节我们介绍了融资的整体流程，本节中我们来看看如何从众多投资机构中做出明智的选择。

## 品牌与声誉的考量

投资机构也存在品牌效应。明星投资机构或投资人（如徐小平）自带光环，但他们的溢价能力强，可能导致创业者被迫出让过多股份或获得较少投资。同时，明星投资人事务繁忙，难以对单个项目提供充分指导，其资源也难以向众多项目开放。

对于创业者而言，不必强求寻找明星投资人，除非对方主动接洽且谈判结果符合预期。

## 巨头战略投资的利弊

在国内市场，互联网等行业巨头（如BAT）的市场占有率已相对固定。它们的业务呈网状布局，常通过收购具有独特创新能力的中小初创公司来扩张业务版图。

因此，创业者常面临一个选择：是否接受巨头公司的投资？职业投资人主要在资本和资源上提供助力，通常不会强势干预公司经营。而巨头公司的投资，除了带来行业资源，也可能导致创始人丧失对公司的大部分主动权，使公司成为巨头战略棋盘上的一颗棋子。

## 如何判断投资机构是否靠谱

我们可以从以下四个方面对投资机构进行初步筛选。

### 1. 名气和声誉

这里并非一味追求知名机构。正如前文分析，知名机构竞争力和溢价能力强，除了品牌背书，可能无法提供足够的支持与关怀。

风投机构为保持良好声誉，会注重品牌形象包装。因此，创业者应多方面考察，重点不是机构的名气，而是它们对创业项目的具体看法。

最佳的考察时机是在拿到投资意向书（Term Sheet）后、决定是否接受的阶段。可以重点关注那些正在积极宣传的机构，它们可能刚试水投资业务，需要打响知名度，投资门槛相对较低。

对于投资机构的排名，可查询权威榜单，例如投中、清科定期发布的私募股权投资机构排行榜。

### 2. 基金背景（LP构成）

基金的背景很大程度上决定了其投资方向和风格。主要背景类型包括：
*   **美元基金**：更看重企业成长性，退出期较长。接受其投资的公司通常需搭建**VIE架构**（Variable Interest Entity，可变利益实体），即在境外注册实体进行融资操作。该过程耗时较长，费用由公司承担。美元基金通常对培育“独角兽”公司有较高偏好。
*   **人民币基金**：更关注企业的收入和利润，追求通过被投公司上市或被收购，赚取一二级市场差价。这主要是因为国内A股上市对企业利润有严格规定，因此人民币基金投资的项目需有较强的变现属性。
*   **产业基金/上市公司战略投资**：更关注投资标的与自身业务的协同效应。接受此类投资，可能意味着创始人丧失大部分主动权，因为大公司通常出于商业版图考虑进行收购，而非单纯财务投资。

选择人民币还是美元基金，首先要看公司计划的终极上市地点（如纳斯达克/纽交所，或国内A股），这决定了应接受何种币种的投资。中途转换币种（如搭建或拆除VIE结构）将产生高昂的时间和金钱成本。

### 3. 专业性判断

通过与投资人沟通交流，可以判断其专业性。投资人主要分为两类：
*   **职业投资人**：擅长从资本角度思考，对财务数据及市场联动反应敏锐。
*   **企业家投资人**：更聚焦企业本身成长，关注创始人行业背景、技术痛点解决等。

创业者应根据自身需求（是需要资本助力还是行业资源）来选择。在交谈中，可评估对方的真诚度、视野广度、商业见解逻辑性以及所提建议的建设性。同时，判断投资人是否对你所在的领域有深入洞察，好的投资人会对意向行业进行深入研究。

### 4. 过往投资成绩

考察投资机构的过往业绩，特别是同类项目的投资经验，具有重要参考价值。主要关注以下几点：
*   **过往投资案例**：了解其历史投资项目的标准和逻辑，并通过这些项目的当前发展，窥探其行业资源与洞察力。**注意**：对于管理规模在1亿元以内的中小型基金，若已投资过类似项目，则需谨慎接触，因为它们通常在同一赛道只布局一家企业，你的商业信息可能被透露给其已投企业。而像红杉、IDG等大型基金则可能在同一赛道布局多家。
*   **近期投资案例（一年内）**：从其近期投资项目的质量和数量，判断该基金当前的资金充裕程度。
*   **人员流动率**：如果基金的投资经理离职率很高，或近期没有投出项目，可能意味着其处于资金短缺阶段。这类信息虽难从公开渠道获得，但常在创投圈内流传，多与圈内朋友交流可获得。

## 如何寻找靠谱的投资机构

基于以上判断排除不匹配的机构后，以下是寻找靠谱机构的几点建议：

以下是寻找靠谱投资机构的具体步骤：

1.  **选择深耕领域的投资人**：优先选择长期关注并投资你所在领域的投资人。他们能提供同行成长经验、行业最新情报，帮助你减少试错成本。
2.  **关注资金到账时效**：对于处于早期发展窗口期的公司，资金及时到账至关重要。天使阶段问题不大，但到VC阶段，需了解基金内部决策流程及受LP限制的情况，评估时间成本。在接受条款前，应特别关注基金打款前的内部流程。
3.  **寻找重视项目的投资人**：热情是投资人对感兴趣项目的应有态度，但创业者需保持冷静。警惕空头承诺，真正好的投资人会不遗余力地直接开始提供帮助。
4.  **与被投企业交流**：与该投资机构已投资的企业创始人聊一聊，能快速了解其投后管理服务的真实情况，例如是否存在过度干预、是否能在公司低谷时陪伴支持等。
5.  **慎签排他协议**：尽量避免在融资早期签署“期间不得接触其他投资机构”等排他性协议，除非你已确定找到了最佳投资方，否则这会让你错失其他机会。
