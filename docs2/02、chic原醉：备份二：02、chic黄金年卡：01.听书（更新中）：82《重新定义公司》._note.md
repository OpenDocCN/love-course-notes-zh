# 重新定义公司：02：谷歌运营之道解析

## 概述

在本节课中，我们将学习《重新定义公司》一书的核心内容。本书由谷歌高管撰写，详细解析了谷歌在人才战略和管理方面的独特做法，揭示了互联网时代公司真正需要的管理模式。我们将通过三个重点内容，了解谷歌如何运营、如何吸引人才以及如何保持持续创新。

---

## 第一个重点内容：谷歌与其他公司最大的区别在哪里？

要想学习谷歌先进的管理模式，首先需要清晰地了解谷歌与其他公司的区别。这种区别主要体现在四个方面：违反常识的企业文化、以技术为主的战略方针、依靠数据的决策机制，以及以创意精英为主的人才需求。

### 1. 违反常识的企业文化

大多数公司的企业文化严谨而全面，通常涵盖客户、员工和股东的利益。例如，雷曼公司的企业文化是：“通过企业成员的知识、创意及努力，与客户建立坚韧无比的合作关系，为客户带来无可匹敌的价值，为股东创造客观的利益。”

然而，谷歌的企业文化核心是：**持续不断地为用户提供优质的服务，同时为伙伴提供最好的成长环境**。他们看重长期利润，不考虑短期利益，也不关注公司股票的变现价值。这在许多管理者看来是违反常识的。

谷歌文化中的特殊现象包括：
*   喜欢拥挤的办公室，认为更有利于协作沟通。
*   鼓励新员工发表观点，屏蔽经验丰富者的固有意见。
*   废除阶级制管理，让组织结构扁平化，减少管理层监督，赋予员工更多自由。
*   在工作中制造快乐，甚至可以拿高层管理人员来取乐。

### 2. 以技术为主的战略方针

大部分公司的战略方针会明确产值或市值目标，例如：“5年内产值达到千万”。这些目标以盈利为最终目的。

谷歌制定战略方针的基础是技术。他们的方针是：**用基于技术洞见的创新方式解决重大难题，不断优化规模，而不是提高收入**。他们让优秀的产品带动市场前进。因此，谷歌的战略中很少出现具体的利润增长数据，更多的是关注用户体验的不断提升。

以谷歌搜索引擎为例，创始人深信它会带来盈利，但从未设定具体的盈利数据目标，而是不断依靠技术解决用户搜索中的难题，优化用户体验。

### 3. 依靠数据的决策机制

大多数公司的决策依据是高层讨论或个人主观判断。

在谷歌，决策的主要依据是数据。谷歌的会议室配有两台投影仪，一台用于会议，另一台专门投射数据。所有人员开会交流时都遵循一个习惯：**一切发言以数据为依据**，而不是凭借经验或猜测。“我认为”这类言论在谷歌很少出现。

### 4. 以创意精英为主的人才需求

所谓“创意精英”，是指那些不仅拥有过硬的专业知识，还具备实战经验的精英人士。他们可以是工程师、设计师、医生、数学家等。他们是真正的行家，不仅会设计理念，还会建造模型并进行商业分析。

**公式：创意精英 = 专业知识 + 实战经验 + 多领域精通**

谷歌需要的正是这类懂得如何将产品推向市场的多领域人才。

---

上一节我们介绍了谷歌区别于传统公司的四大特点。很多人会问，企业都希望拥有创意精英，但如何吸引并留住他们呢？接下来，我们来看看本书的第二个重点。

## 第二个重点内容：谷歌为什么能吸引那么多创意精英？

这个问题涉及谷歌的人才管理机制。他们首先拥有独特的招聘机制，其次有完善的人才管理方法，最后在公司内部营造了良好的沟通氛围。

### 1. 独特的招聘机制

谷歌认为招聘是最重要的工作。他们招聘的人才必须具备三个条件：
*   **有激情**：做事持之以恒，能从失败中重新站起。谷歌看重擅长运动的人。
*   **是学习型人才**：拥有成长型思维，相信自己可以通过学习不断成长。面试官常会要求应聘者剖析自己曾犯的一个错误，以考察其总结和成长能力。
*   **要有趣**：指待人亲切、值得信赖、处事周全、性格平和。为了考察这一点，招聘人员会尝试在生活中接触应聘者，例如共进晚餐，观察其对待服务员的态度。

谷歌在招聘上秉承宁缺毋滥的原则，面试问题深入且出乎意料，例如：“当你遭遇危机，同时又要面对重大决策时，你如何应对？”以此判断应聘者的独立性和协作能力。

### 2. 独特的人才管理机制

招聘到人才后，谷歌通过以下方式留住他们：
*   **给予超出常规的回报**：但前提是员工必须为公司做出突出贡献。
*   **保持工作的挑战度和趣味性**：避免让精英感到安逸。例如，让感到平淡的工程师参与管理会议，从新角度激发其工作热情。
*   **严格执行接班人计划**：要求每位高管必须培养出自己十年后的接班人，确保公司不会因关键人物离开而受影响。
*   **慎重对待离职员工**：针对不同情况的离职员工（如年轻员工、创业者、被高薪挖角者），采取个性化的挽留或祝福策略。

### 3. 营造良好的内部沟通氛围

人才的去留有时与沟通有关。谷歌的内部沟通氛围有三个特点：开放、真诚和高效。
*   **开放**：大部分内部沟通完全开放，甚至董事会文件和高管会议纪要都向全公司分享。员工需将工作计划和研究成果共享在内部平台上。
*   **真诚**：管理者创造机会让员工说真话。例如，在每周员工大会上设置“多利”匿名问答环节，员工可匿名提问，由高管直接回答。
*   **高效**：推行“一对一清单制”，沟通双方提前列出清单，按清单逐一讨论，确保沟通效率。

---

上面两个重点内容揭示了谷歌在管理和人才制度上的独特之处。然而，许多公司学习后却发现，最难模仿的是谷歌持续创新的能力。接下来，我们进入第三个重点。

## 第三个重点内容：谷歌是如何保持持续不断的创新的？

创新是每个公司的追求，但能持续创新的公司极少。谷歌在创新方面一直领先。我们先了解谷歌对创新的定义，再分析其保持创新的原因。

谷歌认为，真正的创新包含三层含义：**新颖、出人意料和非常实用**。创新往往不是线性达成的，而是多次量变引起的质变。例如，谷歌搜索引擎每年进行500次改进，每次改进可能都新颖但不够“非常实用”，但所有改进叠加起来，就实现了真正的、实用的创新。

埃里克在书中分析了谷歌能够持续创新的五大原因：

### 1. 有利于创新的环境
谷歌独特的企业文化、战略决策和管理运营模式，共同构成了最佳的创新环境。

### 2. 公司高层的领导
谷歌的首席执行官必须兼任首席创新官，并直接领导创新团队，关注进展。这与许多公司建立官僚机构来“管理”创新的做法截然不同。

### 3. 永远聚焦用户
对谷歌而言，“用户”（广大互联网使用者）和“客户”（广告商）通常不一致。谷歌坚信，企业的成功之道在于**始终为终端用户提供最好的服务**，利润便会水到渠成。他们有时会收购短期内不盈利但能极大改善用户体验的技术公司。

### 4. 目标足够远大
创新目标必须足够远大，甚至令人感觉匪夷所思。远大的目标能激发创意精英的最大潜力，同时，巨大的失败风险所带来的压力，往往更能促使团队成功。

### 5. 合理配置资源
资源配置分为公司层面和个人层面：
*   **公司层面**：采用 **70/20/10法则**。
    *   70%资源投入核心业务。
    *   20%资源投入新兴产品。
    *   10%资源投入全新的大胆构想。
*   **个人层面**：推行 **20%时间制**。
    *   员工用80%时间完成主要工作。
    *   用20%时间研究自己感兴趣的项目。例如，Gmail（谷歌邮箱）最初就是一名工程师用20%自由时间开发出来的。

**代码/公式表示资源配置：**
```
公司资源分配：核心业务(70%) + 新兴产品(20%) + 全新构想(10%)
员工时间分配：主要工作(80%) + 自主项目(20%)
```

## 总结

本节课我们一起学习了《重新定义公司》的核心内容。本书通过解析谷歌的运营之道，为我们指明了互联网时代公司的发展方向。

首先，我们了解了谷歌与传统公司的四大区别：反常识的文化、技术驱动的战略、数据决策的机制以及对创意精英的渴求。

其次，我们分析了谷歌吸引和留住人才的秘诀：独特的招聘标准、注重贡献与挑战的管理方式，以及开放、真诚、高效的内部沟通氛围。

最后，我们探讨了谷歌保持持续创新的五大支柱：创新的环境、高层的直接领导、始终聚焦用户、设定远大目标，以及公司与个人层面的资源合理配置。

对于创业者，可以学习谷歌领导人的开放与包容；对于企业高管，可以借鉴其对创意精英的管理模式；对于职场人士，则应明确未来需要的是“创意精英”型人才——既要精通专业，也要有激情和有趣的灵魂。愿大家都能以此标准要求自己，拓宽自己的职业道路。
