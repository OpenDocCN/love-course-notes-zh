# 02、chic原醉：备份二：02、chic黄金年卡：39.高薪高福利集聚的快消业揭秘：【Day6】体验王牌特工任务：销售.

嗯。🎼嗨大家好，欢迎来到快消行业职场课。上节课我们为大家揭晓了快销的超级大脑品牌和市场职能部门。今天继续为大家介绍前端的重要部门销售销售人员没有品牌市场，那么光鲜亮丽，但是他们活跃在真正的市场第一线。

市面上发生了什么，他们比品牌人更快之导，市场部制定的策略需要销售来落地。市场部向大脑销售部向奔跑的双脚，更重要的是，对于公司来说，每一分收入都是销售人员拿回来的，所以，销售可谓是公司的生命之源。

销售的重要性不言而喻。今天的课程我们会围绕销售的两个重点分支为大家展开。一个是大客户销售，一个是区域销售。😊，🎼先来说说大客户销售，有的同学可能会奇怪，快消品的消费者不是广大群众吗？

为什么还需要大客户销售？😊，🎼的确，我们最终的消费者散布在全国各地，但是为了能触达这么多的消费者，我们需要渠道的协助。这里的大客户就是渠道中非常重要的组成部分。🎼快消品的渠道是什么？

我们可以想象每个快消品公司是一个水源地，那么渠道就是把这个水源地的水输送到各个地方的工具。所以对于快消品来说，渠道就是我们经常看到每天都会接触的超市。🎼渠道的生态有很多种。

比如一种是一站式购买的沃尔玛、家乐福、大润发。这些卖场超市的典型特点是里面什么都有，价格一般比较优惠。但是这些超市通常不会离家很近，我们经常是要攒了一堆的采购清单才会专门去一趟。😊。

🎼再比如我们在繁华街边经常看到的sevenel全家喜事多等等。这些便利店的典型特点是门店分布很广，非常方便。但是里面的东西不会很全，一般是日常最高频的食品、饮料和日用品，而且产品价格也偏高。

除了这些装修漂亮的连锁超市之外，我们在社区附近也会看到很多小的食杂店，一般只有店主一个人收银装修很普通，甚至商品的摆放也没有那么整齐，东西也是最日常需要的价格适中。在我们平时逛街的时候还会看到一种超市。

他们一般在大型综合体的B一层，里面有很多进口的品牌，购物环境非常好，服务也很周到，比如BHGOLE等，这些都属于精品超市的范畴。在这里购物可以找到平时其他超市里很难看到的一些。😊，🎼国外食材。

很多外国友人也偏爱在这里买东西。以上四种，卖场超市、便利店、食杂店和精品超市就基本涵盖了渠道的最主要生态。而大客户销售就是来点对点，突破这些生态里的大客户的。😊，🎼当保洁、雀巢等快销公司需要上新的产品。

他们首先就要谈下沃尔玛、家乐福之类的大客户。那大客户销售平时的工作需要做什么呢？涵盖了三个方面，首先就是最直接的拿到订单，为公司创造收入。大客户经理会根据公司的策略和客户需求。

把一整年的需求拆分到每个月，并且在每个月的时期落实订单。如果没有订单，公司没有收入，消费者也不会在终端看到产品。😊，🎼第二，大客户经理需要作为品牌方的执行下游，把品牌发起的很多活动落实到渠道中。

🎼比如我们在逛超市的时候，不难发现，很多品牌都会有一些很漂亮的产品造型地堆，每个品牌都会有自己的活动，有的是买赠，有的是满减，有的会请你品尝新产品等等。对于渠道来说，超市的地盘寸土寸金。

怎么在有限的资源里获得最大的产出，就是客户的要求，而客户经理就要配合品牌让超市相信我们做的活动可以带来更好的收益。最后，在客户服务层面，客户经理也要配合渠道，一方面进行老旧货品的处理，直销商品的退换等。

另一方面协助渠道做好整个品类的发展等等。😊，🎼所以，大客户销售的工作从策略层面看，是作为快消品公司和渠道方的链接点，帮助双方更好的发展品类，促进产品在终端的单位产出。从执行层面看，则有很多流程性的工作。

比如落实订单、意境新产品，维护条码，监控库存等等。大客户销售的职业发展路径也非常的清晰。一开始可能负责一些相对比较小的客户，或者作为某一个大客户服务团队的一员。这个阶段统称为客户经理。

在经历和资历足够之后，一般会升任高级经理，乃至客户群经理。从客户经理升任到高级经理，最重要的是要展开出来对渠道的掌控，和双方共赢的策略性思维。越向上发展，对策略性的偏重越大。

最终成为客户总监负责全部的大客户。🎼聊了大客户销售之后，我们再来看看区域销售。😊，🎼大家可能首先会疑问，为什么有了大客户销售之后，还需要区域销售呢？

其实大客户销售是快销公司厂商和大的渠道商总部层面的合作。要说到最终销售的呈现，还是要靠区域销售。😊，🎼在一个大的快消品公司，大客户销售可能在总部几十号人，但是区域销售。

全国各地的团队加起来超过千人是非常常见的。🎼区域销售要负责的事情非常多，小到要落实每一家门店产品的陈列活动的进行，库存的周转，大到一些总部顾及不到的客户，也全部都在区域进行谈判与合作。😊。

🎼区域销售非常锻炼人。快销界有句话说，如果你无法做到一个公司的CEO，那么你还可以做一个分公司经理。一个分公司经理要负责的事情和承担的责任就是一个小型公司的CEO。他需要负责手下二三十号人的团队。

要负责公司的收入尾款，给总部交数，也要负责自己区域内所有客户产品的及时送达。库存的精细化管理和整体满意度。😊，🎼那在区域销售发展的职业道路如何呢？🎼区域销售职业的开端是一名普通的销售代表。

这个阶段你负责的可能是随机分配来的三5家门店。你的工作内容就是每天去这些门店拜访，和店主、采购负责人、现场的促销员姐姐等等打交道。该陈列的品相是否陈列到位，面积是不是有保证促销活动有没有执行。

有没有缺货、断货的等等。😊，🎼在这个阶段，送货库存盘点也会是你的工作内容之一。这样的工作支撑起了快消品销售终端真正的顺畅。😊，🎼销售代表之上是销售主任，主任带领两三名销售代表负责更大的业务内容。

在这个阶段，不是所有事情都要你亲自做，但是你需要带领团队，让他们落实一线的执行。有的销售代表搞不清的事情，也需要你去解决。比如某个门店的活动档期和别的品牌冲突了。比如定好的促销员突然到不了了等等。

有很多突发事件都需要你调动资源来解决。😊，🎼销售主任之后是销售经理，然后再是城市经理大区总监，道路越往上走，承担的责任就越重。区域销售的职业发展当然也需要响应和执行公司的策略，但是在晋升之路中。

最重要的是对终端强有力的掌控和业务结果的保证。一言以蔽之，执行力排在第一位。如果你掌管的门店区域持续完美执行，业绩达成稳定百分百，那么晋升就指日可待了。😊，🎼每个销售人员每天一睁开眼。

就是等待完成的任务和要解决的问题。当然，与责任匹配的也是超强的接力和成就感。销售是个多劳多得的岗位，带领团队完成业绩的骄傲，也是其他职能都无法匹敌的。😊。

🎼区域销售和大客户销售两者的职业路径基本上是分开发展的。如果你能够被公司给予机会跨领域工作，那就等同于进入了职场发展的快速路。🎼同样，如果你具备了两个领域的经验，在销售方面的高级职位上会非常有竞争力。

😊，🎼整体来看，销售职能的进入门槛相对较低，起薪可能4000到5000不等。无论是大客户销售还是区域销售，到了销售经理这个级别，中位数大约是30万左右的年薪，做到渠道总监或者称霸一方的区域总经理。

年薪百万也并不鲜见。最后不可不提的是，经统计，最后能走到快消品公司的CEO职位的，有60%都是做销售出身。😊，🎼什么样的人适合做销售呢？有强烈的进取心，非常自我驱动，喜欢并擅长和人打交道。

还有一点就是shk经常提到的，要有一点邮政结合的气质，比如你既可以和客户总部的人，在会议室里正经微做的谈论策略，也要可以在门店和理货的大哥大姐们打成一片，让他们给你的产品更早摆上货架，货架打理的更美观。

😊，🎼好了，今天的课程内容基本到了尾声，我们来简单回顾一下今天的重点内容。😊，🎼快消品的销售有大客户销售和区域销售两个分支，大客户销售更偏向总部层面的对接，而区域销售则更多在于落地执行。

销售是一个进入门槛并不高，后期成长空间非常大的职能，有强烈进取心，擅长和人打交道，有着邮政结合气质的同学，不妨从销售开始你的快销职业生涯。😊，🎼下节课我们开始为大家揭晓后端的职能部门，咱们下期见，拜拜。
