# 花镇情感课程：01：反搭讪奥义——这样的搭讪你该拒绝

## 概述

在本节课中，我们将学习如何识别并拒绝那些不怀好意或缺乏诚意的搭讪。搭讪是认识新朋友的一种方式，但并非所有搭讪都值得回应。我们将了解男性搭讪的常见动机与套路，并掌握三个关键原则，帮助你有效过滤不良搭讪，保护自己。

## 认识搭讪与反搭讪

搭讪能帮助我们认识生活圈以外、感兴趣并想深入了解的朋友。反搭讪则帮助我们过滤掉恶意的骚扰。反搭讪并非否定所有搭讪行为。有诚意、有礼貌的搭讪是值得鼓励的。但以锻炼口才、刷个人数据或改变内向性格为目的的搭讪，则应予以拒绝。

上一节我们介绍了搭讪与反搭讪的基本概念，本节中我们来看看男性进行搭讪的根本动机。

## 男性搭讪的本质动机

作为女性，首先要认识男性的本质。他们的本能是繁衍后代。面对心仪的女性，他们会本能地产生冲动。他们将这种邂逅称为“一见钟情”。但事实上，这通常只是“一见钟性”。萍水相逢难以产生真正的爱情。可以确定的是，此刻搭讪你的男性正散发着强烈的本能冲动，试图用雄性荷尔蒙吸引你上钩。至于最终目的是发展短期关系还是长期关系，多数人打算先建立短期关系，得手后再做打算。

俗话说，知己知彼，百战不殆。我们只有掌握了男性搭讪的常用套路和技巧，才能真正理解反搭讪的精髓。那么，男性通常如何搭讪呢？

## 男性搭讪的三种属性与策略

男性在搭讪时通常展现出三种属性：绅士、流氓和AV男优。这三种属性往往合为一体。具体使用哪一种，取决于周围环境的人流密度和搭讪对象。如果第一次使用绅士搭讪法成功，在后续交流中他也会依次使用其他属性。

以下是其常见的策略步骤：

首先，他们会进行伪装。为了在人群中脱颖而出，他们善于塑造健康、乐观、幽默、健谈、温文尔雅甚至多金的虚假形象。面对在夏雨屋檐下没打伞的御姐，他们多伪装成多金善良的温柔大叔。面对图书馆的萝莉，则伪装成文艺男神。

其次，成功搭讪后，他们会开始第二步：保持绅士风度与距离。例如，下雨时为你打伞，宁可自己淋湿也要保持风度和距离，照顾好你并微笑注视。

最后，他们会制造话题。此目的是成功引起你的好奇心。接下来，他们就可以引导对话的走向。

了解了男性的搭讪策略后，我们来看看如何识别并拒绝不良搭讪。当你遇到以下几种情况的男性时，切勿被其外表或外在条件迷惑。

## 应予以拒绝的三种搭讪者

### 1. 爱炫耀的男人

很多男人一上来搭讪，就故意露出手上的名贵手表、车钥匙等物品。他们会故意大声谈论房地产、投资、金融证券等话题，让自己看起来很有钱、很繁忙。这种男人往往是伪装的，他们并没有钱，却故意想贴上“有钱人”的标签。这种男人的搭讪一定要拒绝。

作家亦舒曾写道：“真正有气质的淑女，从不炫耀她所拥有的一切，她不告诉人她读过什么书，去过什么地方，有多少件衣服，买过什么珠宝，因为她没有自卑感。” 这句话对男人同样适用。

**核心概念**：炫耀是内心需要被关注和肯定的表现。一个缺乏自信的人，希望通过炫耀他人不易拥有的东西，来获得外界的羡慕，从而建立自信。

```
炫耀行为 ≈ 缺乏自信 (甚至自卑)
```

我曾听说一个案例：某房地产公司的销售总监，30岁左右，全身包裹各种名牌。同事形容他手上带钻的手表“在他面前最好戴墨镜，免得晃眼”。作为总监，收入固然不错，但可持续购买一线大牌的能力在同事看来存疑。同事私下的评论多带鄙视和不屑。

男人一旦自卑，心态就会失衡。即使你们在一起，他也会害怕你离开，但又怕被你看穿，因此情绪容易大起大落，甚至做出过激行为。时间长了，必然影响感情发展。因为自卑，他可能产生强烈的占有欲和控制欲，让你感到束缚，从而想逃离。总而言之，应远离自卑的男人。

### 2. 油嘴滑舌的男人

有的男人一上来就夸赞你的美貌和装束。这种直奔外表、性暗示明显的开场，表明他是“流氓属性”的搭讪者。油嘴滑舌的男人通常有以下几个表现，交往需谨慎。

一般来说，老实的男人和女孩子聊天时可能拘谨，甚至找不到话题。但油嘴滑舌的男人则不同，他们胆大心细，什么都敢说敢做，经常把女生撩得云里雾里，几个回合就能博得好感。

女人一定要小心那些一上来就跟你谈天说地的男人。坏男人先用甜言蜜语勾住你的心，然后不断给出小承诺、大承诺。他们承诺起来比吃饭还轻松。这种男人一定要看清，得到你之后就可能不再理睬，当初的热情也会消失。

女人不能完全相信男人的承诺，特别是那些很会说话的男人。他们一般会承诺一些基本不会做到、也根本不会去做的事情。除了那些连给你一个承诺都需要考虑很久的好男人，其他的基本都不要相信。

有些男人嘴巴很会说，但相貌并不出众，于是选择“广撒网”。他们先编辑好一段打动人心的话，被你拒绝后立刻寻找下一个目标。总会有女人回复并被他们撩倒。这种男人饥不择食，只要是女性都可以。反正吹牛不犯法，他们很会利用女人的虚荣心给出承诺。等他得到你时，可能已经开始寻觅下一个目标了。和油嘴滑舌的男人在一起，你会缺乏安全感，总担心他把搭讪你的套路用在别人身上。所以说，这样的男人一定要拒绝。

### 3. 不知分寸的男人

分寸是对问题恰如其分的回应，多一分则腻，少一分则冷。具备分寸感的男人通常有品位、情商高。他们不会在你工作或忙碌时没完没了地搭讪，也不会硬劝女生喝酒或做不喜欢的事。

我见过周游世界的大叔不顾众人疲倦，侃侃而谈建筑如何精妙、音乐如何美妙。也见过青年才俊在饭桌上对社会问题指点江山，搞得大家吃饭都不顺心。我一位做奢侈品的朋友很喜欢评价别人的手表和包，常让对方尴尬。我认为这就是不知分寸。

凡是在不恰当的场合或不恰当的对象面前，盲目展现自己，不懂得克制优越感和资源优势的，往往很难称之为有品位。

**核心概念**：分寸感体现在对时间、场合和他人感受的敏锐把握。

```
有分寸 ≠ 在不合时宜的场合过度自我展示
```

知道分寸的男人，往往不是以自我为中心的大男子主义者。一个知道分寸的男人，肯定会事事为别人着想，常常考虑他人感受，不会让你感到尴尬，甚至会在遇到突发事件时礼貌地主动为你解围。

## 反搭讪的实用技巧

男人们会各种伪装，你们也要学会以其人之道还治其人之身。当男人伪装成绅士时，你们可以装傻，假装听不懂他的搭讪。他几次三番暗示，你巍然不动，装聋作哑，他自然会知难而退。

当然，拒绝的方式一定要礼貌，不卑不亢，不过于高冷，但态度要坚定明确。例如，可以说：“不好意思，我没有兴趣认识你。” 此言一出，大多数搭讪者都会知难而退。

但也会有极品的搭讪者不轻易放弃。这时你必须使出另一个绝招：用其他行为转移他的注意力。例如，说“不好意思，我朋友来了”，然后立刻跑开。不要不好意思，成功的人生从拒绝渣男开始。

很多女孩面对搭讪没有及时拒绝，也是出于自己对一场艳遇的期待。而一旦从中受伤，又把一切责任归结于年轻的冲动和对情感的无知。这也促使一些猥琐的搭讪者成功狩猎到目标。对于这样的女性，我只能表示遗憾。

所以，女神们，当你们为同胞的遭遇感到悲哀和愤慨时，不如先管理好自己。记住，你们的男神不会轻易出现。要做高冷女神，学会拒绝，克制好自己的好奇心。

本节课我们一起学习了反搭讪的奥义。我们首先了解了男性搭讪的本质动机和常见策略。接着，我们重点学习了三种应坚决拒绝的搭讪者类型：**爱炫耀的男人**（本质是自卑）、**油嘴滑舌的男人**（缺乏真诚，惯于欺骗）和**不知分寸的男人**（情商低，以自我为中心）。最后，我们掌握了一些实用的反搭讪技巧，如礼貌而坚定地拒绝，以及必要时转移注意力并离开。记住，学会识别并拒绝不良搭讪，是保护自己、迈向更健康社交关系的第一步。
