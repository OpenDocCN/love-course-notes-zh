# 社会心理学：02：《乌合之众》大众心理学研究解读

在本节课中，我们将学习法国社会心理学家古斯塔夫·勒庞的经典著作《乌合之众》。我们将探讨群体心理与个人心理的核心差异，分析如何对群体施加持久影响力，并揭示个体在群体裹挟下可能面临的结局。本书揭示了群体行为的非理性特征，其观点至今仍具启发意义。

## 一、 群体意识与个人意识的差异

上一部分我们介绍了本书的基本背景，本节中我们来看看群体意识与个人意识的核心差别。勒庞在书中总结了群体意识的三个主要特征。

以下是群体意识的三个特征：

1.  **群体意识是低智商的。**
    个人意识可分为**表层意识**（支配日常理性行为，如遵守公司规章）和**深层无意识**（受本能支配，如饥饿时想进食）。当个人融入群体后，其理性思维往往被削弱，无意识的本能占据上风，导致行为趋于非理性和低智化。例如，一些针对老年人的“讲座”诈骗，利用群体氛围使原本高知的个体也做出非理性消费决策。

2.  **群体更容易受到暗示和轻信。**
    群体常处于一种期待被引导的状态。置身群体中，人们容易放弃独立思考，接受他人暗示，并让情绪在群体中迅速传染。例如，福岛核泄漏事件后，关于“碘盐防辐射”的谣言引发抢购风潮，这正是群体轻信与情绪传染的典型表现。个人则更可能用理智阻拦此类暗示。

3.  **群体更容易产生极端的情感和道德感。**
    群体因“人多势众”及“法不责众”的观念，其行为往往比个人更极端。例如“中国式过马路”，个体在临时组成的群体中，会因人数而忽视交通规则。同时，群体也更容易表现出极端的英雄主义或崇高的献身精神。

## 二、 对群体施加持久影响力的手段

了解了群体意识的特征后，我们自然会问：如何能对群体施加长久的影响力？勒庞在书中提出了四种关键手段。

以下是施加影响力的四种手段：

1.  **制造鲜明的形象。**
    群体缺乏推理和批判能力，但对形象化的刺激非常敏感。他们依据看到的形象而非理性讨论来做判断。例如，时尚杂志通过精美的模特画报，塑造令人向往的形象，从而引导消费潮流。

2.  **传播简单的观念。**
    简短、绝对、口号式的观念最容易对群体形成影响力。例如，原始社会的图腾崇拜，或现代网络中的“震惊体”标题（如“为什么年轻人的性生活越来越少了？”），都利用了观念的简单性和绝对性来吸引注意。

3.  **利用合适的套话。**
    鲜明的形象和简单的观念，需要借助特定群体的“套话”才能有效传播并激发共鸣。例如，在区块链领域，“共识”、“去中心化”是套话；在创业圈，“梦想”、“商业模式”是套话。掌握这些语言是施加影响的前提。

4.  **构建共同的幻觉。**
    勒庞认为，文明社会中的群体始终生活在各种幻觉（如宗教、主义、明星人设）之中。擅长制造幻觉的人往往能获得广泛拥护。例如，媒体塑造的明星形象（如玛丽莲·梦露的“性感无脑”人设）一旦被公众接受，便成为一种强大的集体幻觉。当现实与幻觉冲突时，便会出现“人设崩塌”。

综合运用以上手段，并通过**不断重复**，领袖便能建立起巨大的影响力，即“名望”。名望与真实地位无关，而是一种能让人盲目信服的气质，如拿破仑即使兵败被流放，仍能令缉拿他的军队倒戈。同样，希特勒通过演讲将信念植入民众心中，引发了集体的病态狂热。

## 三、 群体裹挟下个体的结局

在探讨了如何影响群体之后，本节我们来看看，一旦个体被卷入群体浪潮，最终可能奔向何种结局。勒庞分析了不同类型群体的命运。

以下是群体中个体可能面临的几种结局：

1.  **意志群体的必然分裂。**
    由不同民族、意识形态个体组成的“意志群体”（如初创团队），其聚集动力仅是共同利益。最终会因内在分歧而走向决裂，难以善终。

2.  **选民群体的易被操纵。**
    选民群体想法各异，但仍具备群体特征。高明的政客通过做出宏大却经不起推敲的许诺，就能轻易赢得其支持。

3.  **犯罪群体的责任豁免感。**
    在群体中作恶时，个体常不认为自己是在犯罪，而是在履行“群体责任”。例如，法国大革命中攻占巴士底狱的暴动者残忍杀害典狱长，便是例证。

4.  **议会群体的决策简单化与暴力倾向。**
    议会本身也是一种意志群体，其决策往往带有群体的非理性特征，变得简单而抽象，甚至可能导向暴力。法国大革命期间，雅各宾派执政后实施的“恐怖统治”，将无数政见不合者送上断头台，正是议会群体失控的体现，最终使革命沦为一场血腥闹剧。

在群体的疯狂裹挟下，个人意识几乎无法左右任何事情。群体的思想可能退化，变得混乱无序，最终使一群原本可能富有才华的个体，退化为“乌合之众”。

## 总结与争议

最后，我们需要客观看待本书。由于成书于一百多年前的法国，作者将群体完全推向非理性一面，观点有时过于武断，且书中存在当时时代局限下的种族与性别歧视观点。然而，勒庞在缺乏现代大数据支持的情况下，能洞察并总结出至今仍适用的社会心理学规律，足以证明其敏锐的观察力。
