# 幸福感提升的秘诀：02.07：让性爱回春的两大法宝——调情术与刺激源的适度运用

## 概述
在本节课中，我们将要学习如何通过调情术与适度更换刺激源，来改善长期关系中的性生活，从而提升双方的幸福感。

热恋时，我们很少担心性问题，感觉两人是天作之合。但随着相处时间增长，性可能成为引发冲突的主要原因之一。俗话说，性和谐时，它只占生活的10%；性不和谐时，它却占据了生活的90%。

## 性爱对长期关系的重要性
大家都知道，性爱对长期关系的保鲜有很大好处。它能反复刺激身体分泌性激素，提高免疫力，带来生理满足。俗话说的“床头吵架床尾和”，也体现了它对修复亲密关系、为平淡生活注入激情、提升幸福感的作用。因此，拯救性爱激情对长期关系的“回春”至关重要。

美国的家庭治疗师大卫·施奈奇研究发现，伴侣的平均性频率比人们想象的要低得多。只有**26%**的伴侣性频率达到一周一次。**67%**的伴侣性频率是一个月1到2次，一年几次，甚至整年没有性生活。只有**7%**的伴侣性频率达到一周几次。

但有一个问题：拥有更多的性，就会拥有更多的快乐吗？其实不一定。我们应该关注当前的性频率是否满足彼此的需求。从这个方向出发，我们就能找到让性生活和谐、彼此幸福的方法。

## 识别问题类型
首先，我们将问题分为两种类型。

第一种是当前的性频率没有满足你们的需求，包括一方满足而另一方不满足，或双方都未得到满足。

第二种是当前的性频率已经满足了两人的需求。

接下来，我们将按照问题的不同，分别讲述解决办法。

## 问题一：性关系不和谐与需求不对等
第一个问题是，你们当前的性关系不和谐，存在需求不对等，或彼此对性生活都不满意。

很多女性在面临各种“求而不得”的情况时，不擅长开口询问或索取。她们担心被拒绝，并且普遍认为是因为自己缺乏魅力，另一半才会对自己失去欲望。这其实是对性爱缺少探索的表现，也是一种“爱能”的匮乏——既不知道如何爱自己，也不懂得另一半到底需要怎样的性爱。

接下来，我们将帮助大家从学习调情术和更换刺激源两个方向入手，抓住性爱的主动权，找回安全感。

### 方向二：更换刺激源
接下来，我们再聊一聊改善性生活的第二个方向：更换刺激源。

**3. 改变亲热地点**
改变亲热地点也是一个很棒的法宝。客厅、浴室、夜晚的阳台等等，都可以尝试一下。不过前提是一定要确保安全。

这些刺激源的改变，都能吸引对方的注意力，帮助你们增加激情，从而建立和谐的性关系，获得更多幸福感。

## 警惕“激情怪圈”与掌握“配菜原则”
不过，如果我们在长期关系中总是寻求刺激感，就会陷入“激情怪圈”。就像热恋一样，这种上瘾的刺激感会影响你的判断，比如导致过度宽容、毫无底线。

身边的一个小姐妹为了和丈夫寻求刺激感，答应了很多无理的要求。激情退去之后，一面负罪感很强，一面感到特别茫然。

想要适度地享受激情，就需要掌握 **“配菜原则”** 。在西餐设计中，一般奉行“一主六配”的套餐规则。虽然配菜很多，但每一样都不同，也绝不会抢主菜的风头，而且都是为了给主菜开胃解腻的。

在长期关系中，性爱激情也应该像配菜一样：**丰富、适度、有自己的界限**，而不是毫无节制。这样才能提升幸福感，让彼此的感情稳定而健康。

本节课中，我们一起学习了如何通过两大法宝来改善长期关系中的性生活。

我们可以从**学习闺房调情术**和**更换刺激源**入手，调起对方的兴趣，让性关系更和谐，增加感情的幸福因子。

此外，两性关系的甜蜜并不一定靠充实的性爱来获得。**不给彼此压力、充满童趣、保持适度的性爱激情**，才能给爱情锦上添花，让你们越来越幸福。
