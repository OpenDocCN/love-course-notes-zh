# 聊天艺术：02：打造个人品牌的聊天术

## 概述
在本节课中，我们将学习如何在聊天中塑造独特的个人品牌，从而在获得他人好感的同时，留下深刻印象。我们将探讨成功个人品牌的三个核心特征，并通过具体方法帮助你建立属于自己的语言风格。

---

上一节我们探讨了如何选择聊天话题，本节中我们来看看如何让我们的聊天内容更具个人特色。

前苏联作家高尔基曾说，一个人应在灵魂深处树立标杆，将个性中与众不同的东西汇集在周围，显示出鲜明的特点。这不仅适用于性格，也适用于说话方式。用现代词汇解释，就是拥有属于自己的个人品牌。性格的表现形式有很多种，语言是其中之一。

许多人为了迎合他人，会弱化自己在语言中的性格特色。例如，有人觉得说话太幽默会显得不够正经，于是开始慢条斯理；有人觉得太直爽不够世故，于是开始假装圆滑。

我们知道猫咪喵喵叫，狗狗汪汪叫。如果有一天所有动物都用同一种叫声，我们就不会觉得它们的声音独特可爱了。人的语言也是如此，带有性格特色的语言才会让人印象更深刻，例如朴树的慵懒、郭采洁的可爱、董卿的知性、金星的犀利。

试想一下，如果我们通过前面的课程掌握了倾听、肢体语言和选择话题，虽然能轻易获得听众好感，却可能因毫无特色的和善而很难被记住。在聊天这门艺术的学习上，就只能浅尝辄止。

例如在聚会中，若你只负责倾听和引出话题，聚会结束后，大家对你的印象可能只是“很好相处”、“很爱笑”或“很安静”，而不会给出更高评价。这种浅显的印象不利于发展人脉。

今天的课程将告诉大家如何在赢取好感的同时，塑造个人品牌，给别人留下深刻印象。

下面我们先来具体介绍成功的个人品牌应拥有哪些特征。

成功的个人品牌应具有三个基本特征。

### 第一种基本特征：独特性
独特性是指说话时拥有与别人不一样的观点或视角。他们往往像乱石堆中的美玉或沙滩上的金子，散发夺目光彩。例如在头脑风暴时，其观点剑走偏锋，让人耳目一新；写的文案文风犀利，让人拍案叫绝。

当然，这里提到的个人品牌是受人欢迎的积极品牌，而非无价值、无营养、无内涵的“三无”品牌。想要拥有独特的个人品牌，可以从两方面发力：**具有一项优秀的品质**，或**在某项领域有发言权**。

优秀的品质有很多，但要让独特性更醒目，就需要吸引别人的注意力。

以下是两种优秀品质的例子：

1.  **坚强**：当大家面对同一困难垂头丧气时，你能鼓舞士气，说出振奋人心的话。
2.  **幽默**：列宁曾说，幽默是一种优美的、健康的品质。让人发笑是让对方喜欢你的信号。幽默也会让听者产生优越感，例如“自黑”。但要注意，自黑不应变成习惯性讨好，而应建立在自信基础上。即平时的你积极向上，偶尔通过自黑调侃自己，让别人看见你的内心世界。

例如，老师有一位能力很强但人际关系紧张的朋友。一次聚餐时他自黑道：“其实我一点也不擅长处理人际关系。在8岁时，我的社交状态就已达到人生巅峰。”大家听后哈哈大笑，突然与他亲近了许多。可见，即便是自黑，也应结合当下处境点到为止，而非一味讨好众人，淹没自己本来的个性。

在某一领域有发言权，指的是对某些事项比较专业或擅长。无论是学识类还是日常生活中的小能力，都可以成为你的专业领域。

例如，你通晓香水品牌、对漫画有研究、口才很棒等，都可以作为你的特点。当你向人们展示时，就能刷新他们对你的认识。若你在这些领域比周围的人强一点，他们就容易记住你。

你还可以另辟蹊径，研究一些特别小众的领域，例如各国超小众香水、日本恐怖漫画或世界顶级摄影家等。只要说起这些，你就能滔滔不绝地发表看法，勾起听众兴趣，让大家对你刮目相看。

不过，谈论自己擅长的事时，需注意不要占据聊天中心，让别人无话可说。建议大家按照前面课程提到的**交通灯法则**来约束自己的表达，偶尔将话题引向对方，或在说的过程中停顿并询问对方意见。

**交通灯法则伪代码示例**：
```
if 我正在谈论自己擅长的话题:
    暂停一下
    询问对方：“你对这个怎么看？”
    倾听对方的回答
```

你还可以通过对比与赞美来赢得好感。例如，当你讲完自己的专业后，可以询问聊天者对哪些事感兴趣。如果恰好是你不擅长的，你可以说：“这个领域我真的不了解，可以给我说说吗？”等对方聊完后，再说：“这方面你还蛮在行的，比我强多了。”通过赞美获得好感。

### 第二种基本特征：相关性
相关性是指能够与他人认为重要的东西联系起来，展现你的高价值。高价值展示的核心在于，你能实实在在地帮助他人解决麻烦。

这并非指在聊天时，别人向你讨教经验，你只说“用心就没问题”。这就像看见有人摔倒，你只弯腰对他说：“我以前也摔过，但我很快就站起来了。”然后拂袖而去。

想要打造成功的个人品牌，就要保持相关性。你可以分享你的资源、经验和见解，替对方实质性地解决问题。这是因为人们对具有**利他性**品德的人会格外有好感。利他性是指经常自愿做对别人有好处，而对自己无任何明显益处行为。

只要你善于发挥利他性的作用，就能轻松赢得对方喜欢，他也会对你印象深刻。

例如，你的朋友正苦恼找不到新工作。聊天时，你可以先倾听对方的具体情况，帮助他分析原因：是否需要优化简历、拍一张更美观的登记照、复盘面试流程，或者你是否有认识的HR可以帮忙推荐简历。这样做才是实实在在地帮助对方走出困境，是利他性的表现。对聊天对象来说，你向他输出的价值很高，这样你充满热情的个人品牌就打出来了。

### 第三种基本特征：一致性
一致性是指人们在你身上观察到的行为具有某种连贯性。也就是说，你总是保持着这种特色，不轻易打破自己的规则。人们对你的这种特色形成思维习惯，产生了符号认知。

例如，一提到相声演员岳云鹏，我们脑海里就冒出“搞笑”二字；一听说主持人蔡康永，我们立马会说“高情商”；再提到林志玲，我们的印象就是“知性美丽”。这是因为在荧屏前，他们展示的样子或口才都具有一致性。在数百场表演、数千次主持和采访中，他们始终如一地保持着自己的特色，这就是他们的符号，也可称之为“人设”。

说到这里大家应该明白，想要打造个人品牌，就需要始终如一地保持特色，才能让大家对你的印象统一起来。

---

## 总结
本节课我们一起学习了如何在聊天时一边获得好感，一边打造个人品牌。这需要保持三个基本原则：**独特性**、**相关性**和**一致性**。

*   **独特性**要求你拥有吸引人注意的品质，并在某一领域具有一定的发言权。
*   **相关性**要求你向聊天对象展示你的高价值和利他性。
*   **一致性**则要求你建立并坚守自己的人设规则。

这样，你的个人品牌就能慢慢塑造成功。下节课我们将学习如何用高超的聊天技巧缓和与谈话者的冲突。
