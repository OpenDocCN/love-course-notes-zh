## 第一部分：瑜伽体式练习

### 准备辅具与坐姿调整
我们今天的体式均针对开肩与抗衰。练习需要两种辅具：两块瑜伽砖和一条伸展带（可用毛巾代替）。

首先，以金刚坐姿坐在垫子上。将一块瑜伽砖横向放置于坐骨下方，双脚脚心夹住砖块，坐骨坐在砖上。另一块瑜伽砖竖放在臀部正后方。

调整坐姿：吸气时脊柱向上延展，胸腔上提，同时打开锁骨。保持均匀缓慢的呼吸。

### 坐姿扭转（激活脊柱）
以下是坐姿扭转的步骤，旨在找到脊柱挺拔直立的感觉。

1.  吸气，脊柱延展。
2.  呼气，身体转向右侧，用右手推身后的瑜伽砖以加深延展。
3.  呼气，身体回正。
4.  吸气，脊柱上提延展。
5.  呼气，身体转向左侧，左手推瑜伽砖，让胸腔上提，加深扭转。
6.  呼气，回到中心。

重复此序列。在扭转过程中，请记住胸腔上提、锁骨打开、脊柱挺拔的感觉。完成后，移开瑜伽砖，准备进入下一个体式。将毛巾放在随手可及的地方。

### 侧卧扭转体式（深化挺拔感）
身体趴下，先来到人面狮身式：双肘落地，双手平行，手肘位于肩膀正下方。

**进行右侧练习：**
1.  左手肘撑地，右手推身体向右侧转动。注意，若左手无法直接向前，可打开一个倾斜的角度。
2.  在此体式中，需找到坐姿时脊柱的延展感，并感受左侧腰部的拉伸。
3.  伸直右腿向前，用右手抓住左脚内侧（左脚弯曲）。若抓不到，可用毛巾套在左脚背上，抓住毛巾。
4.  将左脚压向地面，身体微微扭转，收下巴。保持3到5个呼吸。
5.  松开手，收回腿，回到人面狮身式调整。

**进行左侧练习：**
1.  右手肘撑地并微微向内打开角度，左手推起身体，感受右侧腰部的拉伸。
2.  将左腿向前伸出约45至60度。
3.  用左手抓住右脚，将右脚和右小腿落向地面。若无法做到，可用毛巾辅助。
4.  保持右侧肩部撑起身体，找到挺拔感，扭转头部看向左肩延长线。
5.  保持后，慢慢松开双手，收回腿，在人面狮身式中休息。

这不是一个轻松的体式，但能帮助你非常好地找到挺拔感。

### 开肩体式
拿一块瑜伽砖，放在垫子右前方。身体完全趴下。

**进行右侧练习：**
1.  右手侧平举，使右手大拇指与耳朵在一条延长线上，将右侧脸部贴在瑜伽砖上。
2.  推起身体，弯曲左膝，将左脚放在右腿后方。
3.  在这个开肩体式中停留10个呼吸。
4.  慢慢收回左腿，身体转回。

**进行左侧练习：**
1.  将瑜伽砖放在垫子靠左位置，左手侧平举，身体趴下，左侧脸部贴在砖上。
2.  轻轻推身体转向右侧，同时将右脚放在左脚后侧。
3.  右手可放在身体上或背后。
4.  在此开肩体式中停留10个呼吸，可闭上眼睛。
5.  慢慢睁开眼睛，收回右腿，身体转回垫子，面朝下。移开瑜伽砖。

### 休息体式（鳄鱼式）
由于今天进行了较多开肩练习，结束体式以鳄鱼式代替。此体式适合冥想前进行。

趴在垫子上，双手交叠垫于额头下方。双腿打开与髋同宽，双脚脚尖自然朝外打开。调整好姿势后，在鳄鱼式中进行摊尸式的休息。慢慢活动双腿双脚，给自己时间放松，然后推起身体。

以上是针对女性状态提升的瑜伽体式练习。

## 第二部分：冥想引导

### 冥想准备
在前三节课中，我们学习了呼吸觉知的技巧一、技巧二和技巧三。在今天的冥想中，我们将综合运用这些技巧。

今天我们学习用清理经络呼吸法进入冥想。选择舒适的坐姿，可在臀部下方垫高。背部微微挺直，保持头顶、颈部、脊柱、骨盆在一条直线上。双腿可以舒适地盘起。

### 清理经络呼吸法教学
坐好后，举起右手，掌心面向面部。弯曲右手的食指和中指，让无名指和小指并拢。

**手势与呼吸节奏如下：**
1.  用大拇指轻轻闭合右鼻孔。
2.  从左鼻孔开始自然吸气，完全吸满。
3.  吸满后，用无名指按住左鼻孔外侧，关闭左鼻孔。
4.  打开右鼻孔，从右鼻孔自然呼气，完全呼出。
5.  从右鼻孔吸气，完全吸满。
6.  用大拇指关闭右鼻孔，松开无名指，打开左鼻孔，从左鼻孔呼气，完全呼出。

以上过程为一轮。公式可简化为：**左吸 -> 右呼 -> 右吸 -> 左呼（一轮）**。

再重复一遍：关闭右鼻孔，左鼻吸；关闭左鼻孔，右鼻呼；右鼻吸；关闭右鼻孔，左鼻呼。

### 呼吸法的要点与原则
在呼吸过程中，感受腹部在吸气时微微隆起，呼气时微微凹陷。观察呼气时间是否比吸气时间略长。初期不要刻意追求1:2的呼吸比，练习一两天后，呼气时间会自然变长。

呼吸的进入方式是：吸气时，胸腔微微向外，腹腔向外幅度更明显；呼气时，腹部先向内，然后是胸腔。这既非单纯的胸式呼吸，也非腹式呼吸，横膈膜始终参与其中。

同时需要微微收缩根锁（肛门稍上方，不是直接提肛，要轻柔）。根锁启动时，肌肉受到自然控制：吸气时腹部不会过度膨胀，呼气时也不会过度凹陷，保持弹性。吸气和呼气时都需微微收根锁。

### 正式练习与深度放松
现在保持坐姿，自然呼吸。我们练习十轮清理经络呼吸法。

**跟随以下节奏：**
1.  关闭右鼻孔，左鼻吸。
2.  关闭左鼻孔，打开右鼻孔，右鼻呼。
3.  右鼻吸。
4.  关闭右鼻孔，打开左鼻孔，左鼻呼。（一轮）
5.  左鼻吸，右鼻呼，右鼻吸，左鼻呼。（两轮）
继续完成剩余轮数。

把握三个原则：缓慢自然的呼吸；吸入与呼出足够的气；在呼吸过程中感受身体最敏感的部位。

呼吸法结束后，保持眼睛闭合，只是去等待和感受此法带来的功效。你只能等待。如果想躺下，随时可以仰卧。

### 身体觉知与静默
如果躺下，只需静静等待放松感的到来。接下来，感受身体如何占据空间：感受双手、双脚、双腿、双臂、腹部、胸部、头部等所有部位。

现在，整个身体都在放松，同时也保有觉知。你应当同时感受到：全身的放松、全身的觉知、以及经络清洁呼吸法的效果。继续这样做。

如果你能同时感受到这三者，是可能的。如果只能感受到一种或一点点，也可以，只需用最自然、最温和的方式去感受。

现在，只是平静下来，忘掉其他。在平静的当下，只有等待。通过身体，等待说不出的感觉发生。你要做的是等待它的到来。

### 处理杂念与深入静默
如果脑海中出现很多念头或受到任何打扰，只需用最简单的方法：停止冥想。保持眼睛闭上，告诉自己：“我现在不冥想了，我只是躺在这儿，呆着。”让时间流淌一会儿（一秒、两秒、五秒、十秒）。随时，舒适感都可能发生。

这种舒适感可能会变得越来越深，越来越明显。随着更深入的舒适感发生，静默出现了。在静默中待一会儿。

如果有任何念头或画面出现，你可以去观察它，或说“我不需要你”。如果都没有，就继续留在静默之中。如果受到干扰，再次运用同样的方法：停止一切，甚至停止冥想，什么也不做，只是让时间流过。当我们允许时间流淌时，我们就处于“零”的状态，舒适感就会发生。

### 能量感受与结束
现在，去感受腹部的中心，在此停留一会儿。然后，来到头部的中心，停留一会儿。接下来，同时感受腹部中心和头部中心，观察发生了什么。你可能会感受到有能量向上流动。倘若没有，不要想象，只需继续感受。

现在，放下一切，只有静默。在静默中，我们停留一会儿。

如果你感觉好了，随时可以慢慢坐起来。如果还没准备好，可以继续仰卧。在坐姿中再静默一会儿。准备好了，慢慢地允许眼睛睁开。

这是我们今天的冥想引导。
