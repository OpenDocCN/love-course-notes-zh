我们常听到一种说法：平时注意保持身材，但在某些特殊时期，即使大吃大喝也不会胖。不信？请听下面这些话是否熟悉：
*   大病初愈时，身体虚弱，多吃可以补元气，不会胖。
*   孕期哺乳期，只有多吃才能给宝宝提供足够营养，不会胖。
*   经期失血多，多吃是必须的，当然不会胖。

这些所谓的“真理”听起来似乎有道理，但如果照做，就错了。这些看似正确的观点，其实都是想当然。下面我们来逐一分析。

首先说说大病初愈。此时身体确实虚弱，但进补不能过猛，必须循序渐进。因为脾胃功能还较弱，一下子无法负担过多营养。饮食应由少到多、由稀到稠，从流质慢慢过渡到正常食物。**公式：饮食恢复 = 流质 -> 半流质 -> 软食 -> 普食**。如果盲目进补，不仅不利于保持身材，更会加重肠胃负担，危害健康。

接下来谈谈孕期及哺乳期。很多妈妈认为这是可以放纵饮食的时期，可以无限制地大吃大喝。这样做的后果往往是身材严重走样，且可能难以恢复，给妈妈带来身心双重伤害。实际上，除非是非常瘦弱的妈妈，通常不需要特别进补。日常均衡饮食摄入的营养，已足够供给宝宝。因为宝宝从母体汲取的营养量并非想象中那么多。所以，千万不能借机大吃大喝，要为身材考虑。

最后说说大家关心的经期。经期时代谢速度确实会有所加快，但这个幅度远不及普通运动的消耗。**核心概念：经期代谢增幅 < 运动消耗**。因此，经期要做的不是放纵饮食，而是合理进食，多摄入富含铁的食物，以补充因失血造成的营养流失。

认清这些误区后，我们进入今天的食谱环节。今天将为大家带来可以吃出白嫩肌肤的三餐减脂餐。

**早餐：黑芝麻鸡蛋软饼 + 低脂牛奶 + 菠萝一片**

我们热爱清晨的阳光，但炎炎夏日也不要忘记防晒。上班前吃一顿美肌餐，开启一天好气色。

大家都知道多吃黑芝麻对头发好，其实它也是护肤好食材，能为肌肤补充健康油分。传说慈禧太后为保持肌肤美丽就酷爱吃芝麻。现代营养学认为，芝麻富含亚油酸及维生素E，可使肌肤柔软有光泽。

以下是**黑芝麻鸡蛋软饼**所需材料与做法：
*   **材料**：全麦面粉30克，鸡蛋1个，黑芝麻适量，葱花少许，盐少许。
*   **做法**：
    1.  将面粉放入碗中，加入少许盐。
    2.  打入鸡蛋，加少许水，充分搅拌成粘稠但可流动的面糊。
    3.  锅烧热（非不粘锅需刷少许底油），舀入面糊摊匀。
    4.  撒上葱花和黑芝麻。
    5.  待饼边缘微卷时翻面，煎约半分钟即可。

**午餐：茄汁白萝卜炖牛腩 + 银耳莲子莲藕粥**

别人在夏日午后昏昏欲睡，你会因一顿元气午餐而能量满满。

*   **番茄**：美白和瘦身效果俱佳，能保护维生素C，帮助吸收维C和抗氧化物质。想做白皙美人，多吃番茄准没错。
*   **白萝卜**：富含天然抗氧化剂维生素，能抑制黑色素合成，阻止脂肪氧化，防止色素沉积。常吃可使皮肤白净细腻。
*   **银耳**：有滋阴效果，常吃对祛除脸部雀斑有帮助。富含胶原蛋白，能快速为肌肤补充水分和养分，并排出有害物质，嫩肤效果极好。
*   **莲藕**：富含维生素C及矿物质，能促进新陈代谢，防止皮肤粗糙。与莲子同用，效果更佳。

以下是**茄汁白萝卜炖牛腩**的做法：
*   **材料**：牛腩150克，番茄1个，白萝卜1/3个，干辣椒2个，葱段、姜片（3片）、花椒、八角（1个）少许，料酒、生抽、盐少许。
*   **做法**：
    1.  牛腩切块。白萝卜洗净切滚刀块。番茄去皮切块。
    2.  牛肉焯水，捞出沥干。
    3.  锅内加油烧热，放入番茄炒至出汁，盛出备用。
    4.  另起锅加少许油，烧热后放入葱段、姜片、八角和花椒爆香，闻到香味后捞出花椒。
    5.  放入牛腩和干辣椒，加少许生抽翻炒约1分钟。
    6.  放入白萝卜炒匀，倒入炒好的番茄。
    7.  加适量热水和盐，大火烧开后转小火炖至牛肉软熟。

以下是**银耳莲子莲藕粥**的做法：
*   **材料**：泡发的银耳1朵，莲子8个，莲藕1小段，枸杞少许。
*   **做法**：
    1.  银耳撕小朵，莲藕切丁。
    2.  全部材料放入锅中，加足量水。
    3.  大火煮开后转小火，煮至银耳软糯即可。

**晚餐：木耳黄瓜炒肉片 + 瑶柱烧冬瓜 + 燕麦粥**

今天的晚餐，我们继续以美肌为目的，做一顿既美味又能吃出美丽的餐食。

*   **木耳**：富含食物纤维，有助于改善因排泄不畅导致的皮肤粗糙。
*   **黄瓜**：含大量维生素、游离氨基酸和果酸，能清洁美白肌肤，消除晒伤和雀斑，缓解皮肤过敏，是传统养颜圣品。

以下是**木耳黄瓜炒肉片**的做法：
*   **材料**：泡发的木耳8朵，猪肉100克，黄瓜半根，葱花、蒜末、盐、生抽少许。
*   **做法**：
    1.  猪肉和黄瓜分别切片。
    2.  锅中加油烧热，放入蒜末爆香。
    3.  放入肉片，加少许生抽炒匀。
    4.  放入木耳翻炒约1分钟。
    5.  最后放入黄瓜片和盐炒匀，撒葱花出锅。

*   **冬瓜**：含微量元素锌和镁。锌促进生长发育，镁使人精神饱满、面色红润、皮肤白净。

以下是**瑶柱烧冬瓜**的做法：
*   **材料**：冬瓜1块，提前泡发的瑶柱8粒，葱花、盐少许。
*   **做法**：
    1.  冬瓜切小块。瑶柱撕碎。
    2.  锅中倒少量油烧热，放入冬瓜翻炒约1分钟。
    3.  加入泡软的瑶柱和水。
    4.  盖上锅盖焖煮，让味道融合。
    5.  冬瓜软化后，加葱花和少许盐调味即可出锅。

**晒食记小技巧：点缀法**

又到晒食记环节。今天分享的是“点缀法”，大家可以参考我们的黑芝麻鸡蛋软饼。平淡无奇的一张饼，最后点缀上葱花和黑芝麻，瞬间颜值大涨。同理，除了葱花，小辣椒碎、白芝麻、葱姜末等都可以用作点缀，为原本平淡的菜式增色。这样的小心机最适合晒朋友圈了。
