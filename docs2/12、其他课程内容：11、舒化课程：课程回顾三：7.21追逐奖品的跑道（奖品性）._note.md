# 舒化课程：11：课程回顾三：7.21 追逐奖品的跑道（奖品性）🎯

在本节课中，我们将要学习一个核心概念——**奖品性**。我们将通过一个故事引入，详细解释奖品性的含义、重要性，以及如何通过具体的方法提升自己的奖品性，从而在感情中占据高位，引导对方进行有价值的追逐。

## 概述

奖品性，是指一个人像奖品一样，具有被追逐、被争取的特质。它并非与生俱来，而是可以通过思维和行为的调整来塑造和提升。掌握奖品性，意味着你掌握了关系的主动权，能够有效地筛选出真正适合你的伴侣。

## 从一个小故事说起

我们先来看一个招工的故事。我的朋友在人事部工作，老板要求招100个人，但一周只招到10个。他很沮丧。我帮他修改了招工广告，内容变为：“由于本公司业务不断扩大，前一段时间已招满18到24岁女工200名。在此基础上，面向社会招聘100名生产工人。”广告贴出一周后，公司招够了100名年富力强的青壮年。

这个故事说明了什么？它揭示了人性中的一个特点：人们天生追逐稀缺和被争夺的事物。女孩子天生具有“奖品”的特性，而男生天生喜欢“追逐”。

## 什么是奖品性？🎁

奖品，大家都不陌生。幼儿园的小红花、运动会的奖杯，都是奖品。但奖品是对能力的证明，并非人人可得。只有按照规则表现最好的人才能获得。

**公式：奖品 = 高价值 + 稀缺性 + 获取规则**

把这个概念放到两性关系中：奖品性高的女性，就像校花一样，自身优秀，被众多男性追逐。谁最终追到她，会被认为是一种能力的证明。我们要做的，就是把自己打造成这样一个“奖品”。

## 为什么要有奖品性？🤔

上一节我们介绍了奖品性的概念，本节中我们来看看为什么它在感情中如此重要。

男人和女人在感情中的模式不同。男人做的是减法，初始热情高，但会随时间递减。女人则容易在做加法的过程中，因对方的持续付出而感动、习惯并喜欢上对方。

如果你轻易答应对方的追求，就失去了让他“闯关”和“付出”的过程。没有付出成本，他就不会珍惜。男人的爱情需要心理波动（一会儿觉得你喜欢他，一会儿觉得你不喜欢他）和持续付出（时间、精力、金钱）。只有让他像闯关一样付出足够多，他才会觉得你珍贵，从而在长期关系中珍惜你。

**核心逻辑：你的“价格”取决于他付出的“成本”。成本越高，你在他心中的价值越高。**

## 如何提升奖品性？🔝

理解了奖品性的重要性后，我们来看看具体如何操作。提升奖品性是一个系统工程，涉及思维、自身建设和关系互动。

以下是提升奖品性的四个核心步骤：

### 1. 植入“我是奖品”的思维💡

首先，你必须从心底认同“我是奖品，需要别人来追逐和适应我”。记住，是对方来适应你的规则。

*   **具体做法**：明确你喜欢大方、愿意付出的男生。如果对方小气，直接筛选掉。不要不好意思提要求，因为他在追逐你，要么想让你做女朋友，要么有其它目的。你的选择权非常有价值。
*   **关键心态**：接受对方的红包或礼物时，不要有负担。这是他对你表达好感的方式。你可以大方接受并给予正面反馈（感谢、夸奖），这既维护了他的面子，也肯定了他的付出。

### 2. 成为一个“美好”的奖品✨

奖品本身必须是美好的、值得追逐的。这分为外在和内在两个方面。

*   **外在建设**：不能懒惰。保持漂亮和好身材需要运动、保养。这是吸引力的基础。
*   **内在建设**：多学习、多看书，成为一个有趣、有见识的人。这样与人交流时，对方会感到舒适并被吸引。
*   **核心是自信**：自信是最耀眼的光。接受自己的不完美，用积极心态面对。例如，身材不好就找到适合自己的穿搭风格；经历过失败，就把它看作成长的学费。每天完成小目标、保持好奇心、注意体态，都能积累自信。

### 3. 学会情绪控制与引入竞争机制⚖️

奖品性高的女孩必须能控制情绪，不因对方的行为而过度焦虑。

*   **情绪控制**：当等待对方信息感到烦躁时，立刻转移注意力。发展自己的兴趣爱好，或者与其他人交流。
*   **引入竞争机制**：拥有“备胎”或展示其他追求者的存在，对男人非常有效。竞争会激发他的征服欲和投入度。就像校花，追的人越多，优秀的挑战者就越想加入。

### 4. 制定并坚守你的规则📜

这是奖品性的核心。你必须有自己的高标准和硬框架，并以此筛选追求者。

*   **建立标准**：例如，有教养、有公德心、会做饭、幽默、大方、专一等。列出你最看重的5-6条核心标准。
*   **筛选与测试**：不要直接告知对方你的标准，而是通过观察和测试来验证。例如，想知道他是否大方，可以在节日或他惹你生气时，看他是否有送礼或发红包的意愿。对于符合标准的行为，给予夸奖（“副格”），引导他朝你喜欢的类型发展。
*   **框架要硬**：你的框架决定了别人能“欺负”你的程度。对于不符合核心标准或不愿在你框架内追逐的人，果断筛选掉，不要犹豫。不要怕推得太远，只要吸引力还在，未被征服，他很可能继续追逐。

## 奖品性的视觉化模型🏃‍♀️

我们可以将追逐过程想象成一个跑道（你的框架），你是终点高位的女神。追求者需要闯过一道道代表你标准的关卡，才能接近你。

```
      女神（高位）
        /|\
       / | \
      /  |  \
标准1 标准2 ... 标准N （台阶/关卡）
     |        |
    [跑道内：按规则追逐的追求者]     [跑道外：不追逐或未达标的追求者]
```

*   **跑道内**：愿意按照你的规则（标准）一步步付出和追逐的人，给予机会。
*   **跑道外**：即使部分条件优秀，但不主动追逐或达不到核心标准的人，不予考虑。

**注意**：规则不宜过于苛刻。如果对方能满足大部分核心标准（例如5条中符合4条），就可以给予机会深入接触。

## 引导追逐：心态与技术的结合🎭

明确了规则，如何具体引导对方追逐呢？这需要心态和技术的结合。

*   **心态**：坚信自己是奖品，稳坐高位，不焦虑，不主动扑上去。
*   **技术**：包括话术、节奏掌握、推拉、引入竞争、后撤等。这些技术需要通过“练手”来磨练。
*   **练手的重要性**：与不同的人接触、实践，能让你在遇到真正喜欢的人时，即使心态波动，也能下意识地运用正确方法应对，避免犯错。

引导付出的具体操作是：展示高价值 -> 释放一点点可得性 -> 在他行动时提出具体要求（你的规则）-> 根据他的付出程度决定关系推进。

## 总结

本节课中，我们一起学习了“奖品性”这一核心概念。

1.  **奖品性是什么**：像奖品一样具有高价值、稀缺性和获取规则，值得被追逐的特质。
2.  **为什么需要奖品性**：为了在关系中占据高位，通过引导对方付出（成本）来提升自身价值，筛选出真正适合且愿意投入的长期伴侣。
3.  **如何提升奖品性**：
    *   从思维上认同“我是奖品”。
    *   内外兼修，成为一个美好且自信的人。
    *   学会控制情绪，并善用竞争机制。
    *   制定清晰的个人标准与规则，并坚决执行。
4.  **核心行动**：在互动中，通过“练手”不断磨合心态与技术，引导对方在你的框架内进行有价值的追逐。

记住，**只有当你拥有高的奖品性，稳稳地站在高位时，男生才会来追逐你，而你才能从容地选择最适合的那一个。**
