# 公文写作法：02：Issue Sheet：如何理性汇报一个问题？（下）

在本节课中，我们将继续学习Issue Sheet的写作方法，重点探讨如何为汇报的问题提出有效的解决方案。我们将通过一个三步思维模型，帮助你理清思路，从发现问题过渡到解决问题。

上一节我们介绍了Issue Sheet中如何清晰说明问题和分析原因。本节中，我们来看看如何在此基础上，提出有建设性的解决方案。

## 带着解决方案去汇报

一份有效的汇报，不能仅仅停留在指出问题。在紧急情况下，老板更希望下属是带着解决方案而来，而非单纯地抛出问题。那么，解决方案应如何提出呢？我们先从一个故事开始。

我曾接到筹办线下训练课程的任务。作为线上课程作者，我对此毫无经验，且临近春节，场地选择面临诸多困难：面积、规格、档期、预算均难以协调。这导致其他工作无法展开。

此时，我使用了Issue Sheet。我打开一份A4纸大小的Word文档，梳理了所有备选场地，对比其性价比、地理位置、特色和档期。然后，我列举出最推荐的三个选项，并附上推荐理由。这份One-Pager的制作时间不超过2小时，汇报仅用了半小时，便迅速确定了场地，后续工作得以有序展开。

Issue Sheet不仅是汇报工具，更是帮助我们理清思路、培养解决问题思维的方法。

## 解决问题的三步思维模型

从问题到解决方案看似困难，但只需遵循以下三步模型，就能养成有效的思维习惯。

以下是解决问题的三个核心步骤：

1.  **锁定目标，解决主要可控因素**
    在分析问题成因时，我们曾区分主次因素和可控性。**主要因素中的可控因素**，是我们能立刻行动、最快速推进问题解决的部分。
    *   **公式**：`首要行动 = 主要因素 ∩ 可控因素`
    *   例如，在危机公关案例中，对手散播谣言是主要但不可控因素；我方暂时无法证伪是主要且可控因素。因此，首要行动建议是“立即携带产品前往权威部门进行检测”。

2.  **确定核心利益取向**
    在紧急情况下，完美解决方案往往难以实现，通常需要在核心利益与其他利益间做出取舍。尽快确定核心利益取向至关重要，它能指导决策并帮助梳理方案。
    *   例如，选择活动场地时，考量因素包括性价比、地理位置、档期、特色。若核心利益是“性价比”，则筛选方案时以此为主要依据。
    *   如何确定老板的核心利益取向？这需要你**前期观察**、**深入理解项目**，并与老板**直接沟通确认**。例如，公司处于品牌建设期，则声誉可能为首要；若公司资金紧张，则性价比成为关键。

3.  **给出总结性推荐反馈**
    结合整体目标和公司利益取向锁定几种方案后，需要站在老板立场进行初步判断，推荐更优方案。同时，必须预见到任何方案都可能存在弊端，并提前思考应对措施。
    *   在这部分，应从**优势**、**弊病**和**应对措施**三个方面进行阐述。
    *   **示例结构**：
        > **推荐方案**：C方案
        > **优势**：性价比最高，地理位置相对合适。
        > **弊病**：场地周边配套设施一般，缺少购物场所。
        > **应对措施**：可通过统一预订外卖或提前采购零食饮料等方式弥补。

本节课中，我们一起学习了Issue Sheet中提出解决方案的核心方法。我们通过两节课完整阐述了Issue Sheet的写法：汇报问题时要结果先行，清晰阐述原因并聚焦于主要可控因素，最终通过“锁定目标、确定核心利益、给出推荐反馈”这三步来提出解决方案。
