# 尧西情感：06：高级分享课修炼职场高情商

在本节课中，我们将要学习职场高情商的修炼方法。课程将从情商的核心定义入手，澄清常见误区，并重点分析职场中常见的低情商表现。最后，我们将介绍一个实用的自我认知工具，帮助你了解自身性格特点，从而在职场上扬长避短。

## 情商核心复习

上一节我们介绍了情商的基本概念，本节中我们再来明确一下情商的根本定义。

情商并非指让他人感到舒服的能力，其真正含义是**控制自己情绪的能力**。所有情商技巧的核心都建立在“自控”这一点上。让他人感到舒适，是良好自控带来的结果，而非根本目的。

情商可以分解为九大能力，以下是这九大能力的列表：

1.  直觉自知能力
2.  理解平衡能力
3.  控制冲动能力
4.  自我激励能力
5.  挫折承受能力
6.  揣摩他人内心情感的能力
7.  人际沟通能力
8.  人际关系调试能力
9.  发现人际内在联系的能力

你可以对照这九点进行自我剖析，找出需要加强的方面。提高情商的具体技巧包括写情绪日记、遇事先冷静思考30秒等。关键在于，**只有在你内心真正平静、能控制情绪时，你才有选择地表达情绪**。

## 职场情商的特殊性

理解了通用情商后，我们来看看职场情商的独特之处。

职场情商与恋爱中的长期相处情商有所不同。职场互动通常是特定场景下的、片段式的交流，人们往往只展现自己的一面。因此，职场情商的提升通常比长期亲密关系中的情商提升见效更快，也更容易实践。

职场情商为何重要？许多人存在认知偏差，认为情商高就是“会拍马屁”。实际上，从管理学角度看，情商高的人是高效益的保证。现代企业，尤其是非体力劳动行业，员工的心理状态直接影响工作效率。因此，管理者倾向于提拔那些能维持良好团队氛围、懂得换位思考、能知人善任的人。技术能力是职场发展的必要条件，但情商是让能力充分发挥的催化剂。

## 职场低情商的十一种表现

认识问题才能解决问题。以下是职场中常见的十一种低情商表现，你可以进行对照自查。

**第一种：投机取巧，逃避责任**
这种人自认为聪明，善于避开脏活累活。但长期如此，会损害同事利益，让人感觉缺乏担当和责任感，难以成为团队领导者。

**第二种：只知埋头苦干**
这种人踏实肯干，任劳任怨。但若仅限于执行指令，缺乏主动协调和解决问题的能力，则容易被取代，因为其价值并非不可替代。

**第三种：情绪不稳定，易被煽动**
这种人过于感性，容易受外界事件或煽动性言论影响，情绪波动大。这与情商核心——情绪控制背道而驰。需要事后冷静思考，多角度分析问题。

**第四种：轻视上司**
总觉得上司能力不如自己，对其权威不服。这是人性的弱点，容易放大别人的缺点。但必须明白，领导的存在即有原因。在无法改变现状前，请在内心中接受，因为你的不满很容易在言行中流露。

**第五种：过度默默无闻**
甘当绿叶，功劳被抢占也忍气吞声。这种情怀高尚，但在现实职场中过于单纯。你需要清楚什么是自己应得的，并在表现谦让与维护权益之间找到平衡。

**第六种：排斥人际关系**
认为社交是浪费时间和金钱，不如直接给领导送礼。这种想法是错误的。职场离不开人情，领导更看重日常的了解与信任，而非一次性的礼物。建议强迫自己定期进行社交，例如每月拿出一定比例收入请人吃饭。

**第七种：把公司当聊天室**
在工作场所过度分享个人生活。这些分享大多可归为两类：**炫耀优越**和**抒发郁闷**。炫耀易招致嫉妒，而频繁抒发郁闷则会散播负能量，让人敬而远之。应选择分享无伤大雅、能引起共鸣的话题。

**第八种：无条件习惯性忍让**
厌恶争斗，一味退让。结果往往是脏活累活全包，好事却没份。职场中需要有框架和底线，让别人知道你的原则和界限，这样才便于协作。**借题发挥**需要在冷静、能掌控局势时进行，用以明确底线，而非情绪失控的发泄。

**第九种：锋芒毕露**
在所有事情上都急于显示自己的优越感。适当的展示能让人记住你，但过度则易招人嫉妒。应将有限的“展示机会”用在最关键的地方，平时保持低调。

**第十种：盲目崇拜上司**
对上司的话全盘接受、无条件服从。这会让上司觉得你缺乏分析能力和主见，难当大任；同时也会让同事认为你是马屁精。

**第十一种：吝于赞美他人**
不愿表扬别人，觉得那是谄媚。请记住，**在工作中的赞美与感谢永远不过时**。尤其是在需要跨部门协作时，及时的认可能激励对方以更好的态度和质量配合你的工作。

## 提升职场情商的三步骤

分析了问题表现后，我们来看看提升职场情商的具体路径。可以将其类比为“把大象从冰箱拿出来”的三个步骤：**自我认知 -> 正确定位 -> 扬长避短**。职场情商的核心永远是：第一，把握自己（自控）；第二，理解他人。

### 第一步：自我认知
自我认知指了解你在工作中的角色，以及你在同事眼中的形象。一个科学的方法是进行职业性格测试。

目前流行的测试有MBTI十六型人格、九型人格等。这里重点介绍相对易懂的PDP职业性格测试，它用动物比喻将人分为五种类型：

*   **老虎型**：支配者，竞争力强，果断，有担当。适合开拓性、行动性强的工作。压力下可能忽略细节和他人感受。
    *   `优势 = 行动力 + 掌控力`
*   **孔雀型**：表达者，乐观，善于沟通和感染他人，有表现欲。适合宣传、人际导向型工作。思维跳跃，难以专注细节，易情绪化。
    *   `优势 = 表达能力 + 社交魅力`
*   **考拉型**：耐心者，随和，稳定，有耐力。适合安定、专精型工作，如特定技术岗位或内部管理。可能缺乏冲劲。
    *   `优势 = 耐心 + 稳定性`
*   **猫头鹰型**：精确者，分析能力强，重规则，追求完美。适合财务、架构稳定的工作。过于注重细节和流程，可能缺乏灵活性，不重人情。
    *   `优势 = 分析能力 + 精准度`
*   **变色龙型**：整合者，中庸，适应力强，善于沟通协调。适合对外交涉、资源整合。可能缺乏鲜明立场和专精领域。
    *   `优势 = 适应力 + 协调性`

绝大多数人是前四种的复合型。通过测试，你可以了解自己的主导性格，判断当前工作是否适合自己。

### 第二步与第三步：正确定位与扬长避短
基于自我认知，你需要调整发展方向，使其与自身性格优势相匹配。例如，让一个猫头鹰型的人去做开拓型销售，或将一个孔雀型的人放在需要极度严谨的会计岗位，都很痛苦且低效。

在职场中，策略应是**扬长避短**，而非像在长期亲密关系中那样“扬长补短”。集中精力发挥你的优势，并将你性格中的短板可能带来的负面影响降到最低或避开相关场景。
