# 谈判哲学：02：顶级商务谈判的谋略运用：05【Day5】“劣势谈判”的化解—如何与心仪公司达成合作（下）🔍

在本节课中，我们将继续学习“劣势谈判”的化解策略。上节课我们介绍了如何运用“思想家”和“恋人”角色来明确谈判形势。本节中，我们来看看如何运用“梦想家”和“勇士”角色，将洞察转化为具体的行动方案，并最终促成合作。

## 从洞察到行动：引入“梦想家”与“勇士”

通过“思想家”和“恋人”的分析，我们已明确了双方的核心诉求与底线，将对立局面转化为统一目标。但这距离成功谈判仍有距离，因为我们尚未展现出完美执行合作项目的能力与决心。

这时，就需要“梦想家”与“勇士”这对角色登场。在之前的课程中我们提到，“梦想家”负责战略方向与目标设定，如同大脑董事会中的CEO。“勇士”则负责执行战略蓝图，如同董事会中的COO首席运营官。

那么，在谈判中，他们应如何正确发挥作用呢？

## 梦想家的战略：开天辟地，重塑合作蓝图

在我们通过理性与感性分析，明确双方复合需求与底线后，合作目标的雏形已基本清晰，核心是“各取所需”。然而，实践中谈判常因各种原因陷入僵局。

例如，对方提出的利益分配条件严重损害我方利益，远超我方底线。若直接针锋相对，谈判可能破裂，前功尽弃。

此时，需及时请出“梦想家”来开天辟地。你可以礼貌地向对方表达：“我理解贵公司设置此条款的初衷，但这会极大损伤我方利益，进而影响我们的长期合作关系。为表示合作诚意，我们可在现有服务基础上，增加价值1万元的附加服务，您看是否可行？”

请注意这段话的关键点：
*   首先，避免使用攻击性词汇，而是通过强调长期合作的影响，将对方从对立面拉向己方。这是梦想家的第一重作用：**缓和冲突，引导共识**。
*   其次，为打破僵局，提出了新的合作策略。这基于前期对对方需求的洞察。若对方在意性价比，而我方在价格上有底线，则可通过提供**增值服务**作为置换条件来达成目标。

## 勇士的执行：制定方案，灵活调整

如何设置置换条件，并将双方目标落到实处，就需要“勇士”出场。

勇士的作用主要有两点：
1.  **拿出具体的合作方案**。
2.  **根据谈判进展灵活调整方案，即设置置换条件**。

许多公司未能与心仪对手合作，很大原因在于执行层没有拿出靠谱可行的计划。最可靠的做法是在谈判前充分考虑各种可能性，并草拟初步合作细则。

需要注意的是，在重要且我方处于劣势的谈判中，只准备一套方案是危险的。因为谈判中总可能出现意外情况。

因此，需在基本方案外，准备附加的置换条件来应对对方的要求。置换条款的设置逻辑，应围绕我们上节课讲到的“复合需求与底线”。

这样做并非无条件退让，而是预先想好对策，为自己留有余地，避免在谈判中陷入被动。

例如，前述通过提供价值1万元服务进行置换的做法，我方的余地就体现在服务的定价权在于我们，实际未必需要付出等额成本。

## 应对劣势谈判的三步策略

综合来看，应对劣势谈判的步骤可概括为以下三步：

**第一步：见招拆招，摆平顾虑**

在谈判桌上，对方即使有意合作，也可能提出许多附加需求。我们的做法是：逐一记录，并进行分类。

以下是处理不同类型需求的方法：
*   **无伤大雅的需求**：对核心诉求和底线影响不大，可尽量满足，以提高对方合作意愿。
*   **不合理或损害利益的需求**：尝试用置换条款来满足。

**第二步：守住底线，强化共识**

面对触及底线的需求，首先不能轻易让步。其次，拿出预先准备好的置换条件（如提供额外宣传资源），不断强化对方对这场合作的期待值。

随着期待值攀升，对方在价格等条款上的立场也可能松动。

**第三步：把选择权留给对方**

对于我们提出的合作条件与置换条款，最终选择权应交由对方。这能充分体现对合作对手的尊重，是共情力的重要体现。

例如，可以说：“在我们提出的几种方案中，您认为哪一种最合适？” 在此过程中，可站在对方立场适当引导，推荐真正适合他的方案。

## 课程总结

本节课中，我们一起学习了“劣势谈判”化解策略的后半部分。我们探讨了“梦想家”如何制定战略、重塑合作蓝图，以及“勇士”如何制定具体方案并灵活执行。

对于这种谈判格局，我们总结出三条核心策略：
1.  **见招拆招，摆平顾虑**：分析对方需求，对不影响核心利益的要求尽量满足。
2.  **守住底线，强化共识**：对难以接受的条款不轻易退让，并通过置换条件增强对方对我方价值的认同。
3.  **把选择权留给对方**：体现尊重与共情，引导对方做出选择。
