# 11、恋爱知识星球：10、护肤攻略：丁香妈妈护肤攻略：3.3防晒霜的涂抹方式和用量：3.3.

🎼你好，我是曾相如，欢迎收听我在丁香妈妈的五福攻略。上节课和你聊了如何选择防晒霜，这节课就来说说防晒霜的用量和涂抹方法。也许你天天都在涂防晒，但好像还是会变黑，这是为什么？大量调查发现。

大部分人涂抹防晒霜的量只达到了所需量的4分之1不到一半。量涂不够的结果就是你以为你涂了SPF50的防晒霜。而实际防晒效果其实只有SPF7，只要没涂够，防晒效果就是大打折扣的。那么涂多少才算够呢？

最最最精确的量是至少2毫克每平方厘米，每个人脸的大小都不一样，取个平均数大约总共需要1。25克的防晒霜才算涂够了。不过我想可能没有多少人会去量自己的脸有多少个平方厘米。

我们更经常用的比喻是挤道手心的防晒霜能有1块钱硬币大小，这个量和脸上所需的量1。25毫升或者1。25克，答案是相等。脖子需不需要涂，这也是你很容易忽略的问题。其实你想特别是平时坚持给脸涂防晒的话。

你们的脸和脖子的衰老程度一样吗？很多人坚持给脸防晒，看起来倒是比同龄人年轻，但颈纹依旧非常突出，其实就是这个道理。颈部需要多少防晒霜呢，至少和面部一样，也应该是和一个1块钱硬币差不多大小的量。

虽然明确了脸和脖子分别应该涂多少，但我也知道谁也不可能每天用的那么精准。那么有没有更简单直接的判断方法呢？其实你只要看自己买的防晒霜能用多长时间就够了。以资生堂的安耐晒小金瓶为例，小金瓶是每瓶60毫升。

假设你是一个办公室工作的小仙女，不需要大量补图。早上出门前15分钟，脸上涂1块钱硬币，脖子1块钱硬币。到了中午外出午餐前补涂一次，脸和脖子各1块钱硬币，并且假设下班的时候，太阳落山了。

那么一天就需要4个1块钱硬币，1块钱硬币大小大约是1。25毫升，也就是一天就需要4乘1。25等于5毫升。每天5毫升的话，60毫升一瓶的安耐晒，只能够涂12天，就算你周末都是宅在家。

一个小金瓶也就刚好涂够两周。所以如果两个星期都没用完一瓶小金瓶，你肯定就没涂够量。那如果你说你放弃了对颈部皮肤的保护，随便以后颈纹怎么长，防晒霜只涂面部，那么一瓶安耐晒也只够涂一个月。

说完了脸和脖子还得说说身上。比如说去海边玩耍，穿的是比基尼，这个时候需要全身涂，应该涂多少呢？刚刚我说了，至少要涂到2毫克每平方厘米。不过每个人的高矮胖瘦不一样，我也取一个平均值一。73平方米。

假设在海边除去泳衣和泳裤遮盖的面积。按照2毫克每平方厘米的量来涂的话，我们大概需要涂30毫升。也就是说，安耐晒小金瓶一次30毫升，一瓶安耐晒60毫升，也就够涂两次。如果按照两个小时补涂一次的节奏。

如果到海边玩一天，那么你一个人就需要带上至少2到3瓶安耐晒，做好防晒，真的挺烧钱的。听到这里，你可能要问了，防晒涂够亮了，脸白的像易妓怎么办呢？第一种方法，在肤质适合的情况下，选择化学防晒霜。

避开物理防晒霜。这种情况可以很好的缓解泛白。好的化学防晒霜在抹开大约5至10分钟后，白色会逐渐消散。物理防晒，尤其是纯物理防晒产品，泛白是避免不了的。不过我之前推荐的物理哑光防晒也可以很好的避免这一点。

第二种方法，在涂抹防晒霜的时候，几1块钱硬币大小的防晒霜出来，分成2至3层涂抹，一层一层一层抹匀，等防晒霜吸收，不泛白之后再开始涂抹第二层，直到涂抹够亮。但有一点要说明的是，完全不泛白。

涂了跟没涂一样的防晒霜是不存在的。如果一款防晒霜在涂抹足量的情况下，既轻薄又不泛白，你就有理由怀疑这款防晒霜的实际防晒效果是不达标的。因为现阶段的化妆品工艺还做不到这一点。再说了，为了不晒黑不变老。

泛一点白又算什么呢？但有一点要说明的是，如果你想要更好的防晒效果，就应该在接受日晒前的15分钟就涂好。因为防晒霜里的化学防晒剂大概需要15分钟，才能和皮肤充分反应结合，进而起到防晒的效果。

如果是纯物理防晒产品时间上的要求会低一点，它可以及时发挥效果。不管是早上还是中午外出吃饭，都要在出门15分钟之前，就把防晒霜涂好。另一个问题就是补涂了。正常生活中，如果持续待在户外。

你最好每2小时补涂一次。就像上节课说的，理论上，防晒霜的防护时间应该很长。但这仅仅就是理论上，由于摩擦、出汗等原因，防晒时长会大打折扣。尤其是你待在海边泳池或者处在户外旅行、运动登山。

这些需要大量出汗的情况下的时候，补涂的频次还要增加，基本上需要提高到90分钟一次，哪怕是防水型的防晒霜仍然要补涂到这种程度，才能很好的避免被晒黑晒伤，毕竟防水型防晒霜的最长防水效果也只能维持80分钟。

很多人出去玩会抱怨说涂了防晒霜仍然会被晒黑了。最后一问，怎么涂的，基本就是早上出门前涂了一次，然后在外玩了一天。老实说，这样的防晒霜涂抹方法，完全不补涂，晒伤晒黑，光老化，都是在所难免的。还有的人说。

如果我就是早上开车上班晒一会儿，中午出公司吃个饭，下午开车回家，太阳都落山了。那么这种通勤时日晒的情况，推荐早上出门前涂一次，中午吃饭前再涂一次也是可以的。尽管有各种文章说需要这样抹。

那样抹才能达到最好的防晒效果。但实际上，不管是谁都很难在日常涂抹中做到完美。其实你只要记住一点就够了。避免用力揉搓，轻轻的涂抹均匀就好，这样就能很好的把防晒霜盖在皮肤上起到保护作用。同时。轻轻涂抹。
