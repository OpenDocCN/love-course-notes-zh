# 腹肌马甲线训练：08：第四阶段——腹肌力量进阶练习B 💪

在本节课中，我们将学习第四阶段的腹肌力量进阶训练。课程包含四个高难度动作，旨在强化上腹和下腹的肌肉力量。请根据自身情况调整训练强度，确保安全。

上一节我们介绍了力量进阶的初步练习，本节中我们来看看更具挑战性的动作组合。

## 动作详解

以下是本节课的四个核心训练动作。

**1. 直腿反向卷腹**
*   平躺于垫上，双腿伸直。
*   脚尖勾起，利用下腹力量将双腿向上顶起，指向天空。
*   缓慢下放双腿，回到起始位置。
*   公式：`起始姿势（平躺，腿伸直） -> 收缩下腹，直腿顶起 -> 缓慢下放`

**2. 抬腿剪刀**
*   平躺，双腿伸直并抬起至与地面垂直。
*   双腿像剪刀一样向两侧打开，然后并拢。
*   将双腿缓慢下放至接近地面，再次抬起并重复开合动作。
*   代码逻辑：`while(训练中) { 腿抬起 -> 打开 -> 合拢 -> 下放 }`

**3. 交替摸膝触脚**
*   平躺，屈膝，双脚踩地。
*   交替用对侧手触摸膝盖，同时锻炼上腹与下腹。
*   保持呼吸节奏，动作稳定。

**4. 起身停留**
*   平躺，执行类似“木乃伊起身”的动作。
*   在起身至约45度角时，刻意停留2秒钟。
*   核心要领是控制肌肉缓慢下放，而非利用惯性回落。
*   关键点：`起身 -> 在中间点停留(约2秒) -> 缓慢下放`

## 训练流程与要点

现在，我们将这些动作组合起来进行计时训练。

*   **准备**：调整呼吸，集中注意力。
*   **执行**：按顺序完成每个动作，注意动作质量而非速度。
*   **强度调整**：若感到吃力，可在组间短暂休息。确保动作全程由腹肌主导发力，特别是“起身停留”动作，应避免用髋部力量代偿。
*   **结束**：训练完成后，进行简单的腹部拉伸与放松。
