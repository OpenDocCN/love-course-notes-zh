# 瑜伽与专注力提升：09：清晰头脑，提升专注，变身效率达人

在本节课中，我们将学习如何通过特定的瑜伽练习来提升专注力、清晰头脑，从而成为更有效率的人。课程将涵盖专注力下降的原因分析，以及通过呼吸法、体式和冥想来进行系统性训练的方法。

## 概述：专注力的重要性

专注力下降是普遍存在的现象。它表现为工作效率低下、学习时注意力不集中、陪伴家人时心不在焉。专注力下降会导致生活工作缺乏焦点、自信不足、决策犹豫、容易出错以及压力感不明。

专注力如同穿越草原的道路。若不加打理，杂草会掩盖去路；一旦打通，便能走向更远的远方。

上一节我们概述了专注力的重要性，本节中我们来看看导致专注力下降的具体原因。

## 专注力下降的原因

专注力下降主要有两大原因：外界干扰和自身因素。

### 外界原因：信息过载

我们处于信息过剩的时代。庞杂的信息分割了我们的时间与精力。大量无用信息占据头脑，即使未刻意关注，那些信息也会在大脑中形成印记。信息无法有效处理，导致大脑神经系统负荷过重，专注力明显下降。

### 自身原因

以下是三个主要的自身原因。

**第一，睡眠不足与质量低下。**
晚睡对身心伤害很大。睡眠质量低导致身心修复能力差。大脑处于混沌状态却需迅速进入工作，许多人依靠咖啡或茶提神。长此以往，神经系统疲惫不堪，无法放松，形成恶性循环。

**第二，负面情绪与能量堆积。**
日常压力、焦虑等负面情绪和能量若未及时疏通释放，长期堆积会对中枢神经系统、脑垂体及大小脑产生抑制，容易导致失衡，从而使专注力和记忆力显著下降。

**第三，缺乏专注力训练。**
我们的大脑如同热闹的城市广场，充满无尽想法，我们常希望同时完成所有事。但时间、精力和情感是有限的。未经专注力训练，会导致无觉知、心神散乱、缺乏定性。

执行任务时，大脑需要热身才能进入状态。进入状态后，后脑部相关区域被激活。若流畅运转的大脑被干扰，进程会被终止。重新启动需要再次热身，缓慢进入专注状态。这导致专注力与工作效率下降，心灵空间堵塞，从而产生疲累、混沌和压抑。

通过瑜伽和冥想练习，我们可以经历从外向内收的过程。这能在一定时间内保持与身心的连接，保持专注。我们会捕捉到更多信息，获得深入的体验和感受。只要身心经过训练，我们的觉知力和专注力就能专注于当下，力量便会大大增强。

上一节我们分析了专注力涣散的原因，本节中我们来看看如何在日常生活中进行改善。

## 日常专注力养成攻略

日常生活中，可以通过定制计划来锻炼和提高专注力。

以下是具体方法。

*   **从短时练习开始**：从每天5分钟的瑜伽或冥想开始，逐步增加到15分钟、30分钟甚至一小时。专注于瑜伽练习和身心感受。此方法已被许多学员实践验证。
*   **静坐放松**：通过观察呼吸、凝视、阅读或默写等方式，培养专注于当下的能力。

接下来，我们将从瑜伽技术上，学习如何通过呼吸法、体式练习和冥想来提升专注。

## 瑜伽技术练习

本节将介绍四段呼吸法、一系列体式以及拉雅冥想，帮助您观察自身模式，学会成为自己大脑的主人。

### 四段呼吸法

四段呼吸法能在身体内形成有节律的脉冲，强化身体能量，提升觉知力。

**练习步骤：**
1.  采用舒适稳定的坐姿（如简易坐或金刚坐），脊柱挺直，微收下巴。
2.  闭上眼睛，凝视眉心。
3.  双手结智慧手印放于膝盖上。
4.  用鼻子将吸气均分为四段：`吸-吸-吸-吸`。
5.  吸满后稍作停留。
6.  用鼻子一次性缓慢呼尽：`呼————`。
7.  重复此模式（四吸一呼）练习3分钟。

**要点**：确保每次分段吸入均等，以形成节律性脉冲。呼气时，想象类似轻轻吸鼻涕的感觉。

### 体式练习序列

以下体式旨在调整丹田能量中心，增强核心与背部力量，进而提升整体的专注与稳定。

#### 1. 伸展式

此体式旨在调整和强化丹田（肚脐下约四横指深处）能量中心，并训练下腰背部力量。难度较高。

**做法：**
1.  仰卧，双脚并拢，双手放于身体两侧。
2.  吸气，同时将头、颈、肩部和双腿抬起。
3.  双手在臀部上方伸展，掌心相对，相距约20厘米。
4.  眼睛看向绷直的脚趾远方。
5.  在此姿势中保持深长呼吸，或进行火呼吸（Kapalabhati）。
6.  结束时，深吸气并屏住，然后深呼气放松。

**要点与调整：**
*   主要难点在于下腰背部和核心力量不足，可能导致压力集中在颈肩。
*   **调整方法**：若感到压力过大，可重新调整下腰背部落地的感觉；或保持单腿弯曲；甚至可将单腿放下以增强支撑；感到非常累时，可完全平躺放松，待有力气时再收紧核心进入体式。
*   **目标**：在伸展式中坚持1到3分钟（本节课为2分钟）。
*   **禁忌**：孕期、经期不宜练习。身体虚弱者应循序渐进，从1分钟开始逐步延长时间。

#### 2. 抱膝火呼吸式

此体式效应与伸展式类似，但额外有益于消化系统和排泄系统。

**做法：**
1.  完成伸展式放松后，屈膝。
2.  双手抱住膝盖。
3.  收紧下腹部核心，感觉力量向下扎根。
4.  利用反作用力，让颈肩抬起，尝试用鼻子去找膝盖。
5.  保持深长呼吸或火呼吸。
6.  放松时，保持抱膝状态，进行深入的吸气和呼气。

**要点**：关键在于尾骨向下扎根。若力量不足，上半身抬起时会非常费力，导致颈肩紧张。解决方法是核心收紧，尾骨内收，使力量向下，从而让上半身更轻松地抬起。感觉疲劳时，可保持火呼吸并放松颈肩。

**建议**：此体式需花费时间深入体验。建议持续练习三周以上，以逐步感受力量的增强。

#### 3. 滚动放松与坐姿火呼吸

**滚动放松：**
抱膝火呼吸式结束后，若感腰部酸，可抱住膝盖左右轻轻晃动，并前后滚动（吸气向后，呼气屈腿坐起）两到三次。

**坐姿火呼吸：**
1.  坐直，可垫高臀部。
2.  双臂高举，与地面呈60度角，指尖用力推指根，拇指张开相对。
3.  保持胸腔上提，肋骨内收，核心收紧，根基稳定。
4.  进行火呼吸：深吸气，深呼气，然后开始快速的节律性呼吸（吸=呼）。
5.  结束时，深吸气，拇指并拢，其余手指向上张开并延伸；然后长呼气，感受胸腔、肋骨、腹部依次向内收缩。
6.  重复数次后，放松身体，喝些水。

**火呼吸要点**：火呼吸是快速、均等、节律性的呼吸。重点在于丹田（太阳神经丛）区域的收放，应如蝴蝶振翅般轻微自然，而非强烈的风箱式呼吸。若不清晰，请回顾相关基础课程。

#### 4. 生命神经伸展（分腿式）

此体式伸展从脚趾到眉心的整条生命神经（主要沿膀胱经和脊柱）。

**第一部分：动态伸展**
1.  坐立，双腿分开（角度依柔韧性而定），骨盆摆正，双脚勾回。
2.  吸气，双臂向前向上伸展。
3.  呼气，转向左侧，腹部找大腿，胸腔向下。
4.  双手尝试抓大脚趾。
5.  吸气，拉直身体回正。
6.  呼气，转向右侧落下。
7.  配合呼吸重复：吸气默念“Sat”，呼气默念“Nam”。
8.  结束时，深吸气延展，深呼气放松，收腿抖动。

**调整方法**：
*   若腿后侧或内侧过紧，可在臀部或膝盖窝下垫毯子。
*   常见错误：身体转向一侧时，对侧臀部抬起。应保持双臀落地。
*   初学者可减小双腿角度，手抓小腿或脚踝即可。

**第二部分：静态保持**
1.  保持分腿姿势。
2.  双手尽量抓住脚趾。
3.  吸气，脊柱延展向上。
4.  呼气，屈肘，腹部找地面。
5.  在此姿势中保持2分钟，持续伸展腿内侧和生命神经。
6.  结束时，吸气并屏息，关注眉心；然后呼气放松。

**要点**：确保身体顺位（尾骨、胸腔、头部三位一体），避免为抓脚趾而弓背挤压心脏。可使用伸展带辅助，套住脚掌，双手抓住带子以适应伸展幅度。动作应为目的（伸展生命神经）服务。

#### 5. 脊柱弹动式（简易坐）

此体式激活丹田能量，使其上行，并保持脊柱灵活。

**做法：**
1.  简易坐，双手抓住小腿胫骨或脚掌外缘，肘部伸直，背部挺直。
2.  吸气，胸椎带动脊柱向前。
3.  呼气，脊柱向后回落。
4.  保持骨盆和头部稳定，仅让骨盆以上的脊椎活动。
5.  眼睛始终凝视眉心。
6.  配合呼吸重复。
7.  结束时，深吸气提升能量，屏息并关注眉心；然后深呼气放松。

**调整**：若腰椎不适，可减小动作幅度，并让呼吸引领动作。

#### 6. 脊柱弹动式（金刚坐）

此变体在跪姿中能产生更大动作幅度，更有效地推动能量向胸腔以上运行。

**做法：**
1.  金刚坐，双手放于大腿，膝盖并拢，根基收紧，脊柱延展，下巴内收。
2.  吸气，脊柱向前；呼气，向后。
3.  保持根锁和颈锁稳定，避免晃动。
4.  练习2分钟。
5.  结束时，深吸气，深呼气放松。

#### 7. 坐姿脊柱扭转

此体式增强脊柱灵活性，促进能量运行和淋巴排毒。

**做法：**
1.  金刚坐（不适可改用简易坐），双臂侧平举，屈肘，手抓肩膀（拇指在后）。
2.  肘部向两侧打开，与肩同高，肩膀放松。
3.  吸气，脊柱向左扭转；呼气，向右扭转。
4.  以中慢速开始，后可逐步加速。
5.  结束时，深吸气回正，深呼气，手臂落下放松。

**要点**：扭转时，肘部始终保持水平位置，不落下。

#### 8. 肩部画圈

此体式灵活肩关节、肩胛骨区域，疏通堵塞，促进能量流动，缓解压力。

**做法：**
1.  保持跪姿（不适可改用简易坐），双手搭肩，肘部向两侧打开。
2.  吸气，双肘向上抬起，尝试在后脑勺相触。
3.  呼气，肘部落回水平位置。
4.  配合呼吸重复，后可加快速度。
5.  练习2分钟。
6.  结束时，深吸气，深呼气放松。

**要点**：动作应由呼吸带动，而非惯性。肘部落下时保持水平。

#### 9. 合十上举式

此体式继续灵活肩周，并从能量层面净化磁场。

**做法：**
1.  十指交叉相扣（女性左手拇指在上，男性相反），结成“金星锁”。
2.  脊柱稳定，身体呈稳定的“L”型。
3.  吸气，双臂经前向上举过头顶。
4.  呼气，双臂落下至大腿前。
5.  配合呼吸重复。
6.  结束时，深吸气保持，深呼气打开双手放松。

**要点**：避免利用惯性甩动双臂，保持根锁和颈锁，确保只有手臂在动，脊柱稳定。

#### 10. 交替耸肩与双肩耸落

**交替耸肩：**
1.  简易坐，双手放膝。
2.  吸气，耸左肩；呼气，左肩落，耸右肩。
3.  重复1分钟。
4.  深吸气，深呼气放松。
5.  换方向重复1分钟（吸气耸右肩，呼气右肩落、耸左肩）。

**要点**：保持脊柱稳定，避免头部随肩膀晃动。

**双肩耸落：**
1.  保持坐姿，根锁、颈锁收紧，面部放松，闭眼凝视眉心。
2.  深吸气，双肩极力向上耸起找耳垂，保持。
3.  呼气，有力地将肩膀放下，如泄洪般释放压力。
4.  开始可稍慢，适应后加快力度和速度。
5.  重复数次后，深吸气，深呼气放松。

#### 11. 颈部左右扭转

此体式轻柔打开喉部，促进喉轮能量平衡，有益于甲状腺。

**做法：**
1.  坐姿，双手放身体两侧，指尖触地。
2.  深吸气，颈部向左扭转，感受右侧从耳下到手指的伸展。
3.  呼气，缓慢转向右侧，感受左侧伸展。
4.  保持肩膀以下不动，身体不歪斜。
5.  重复1分钟。
6.  深吸气回正，呼气放松。
7.  进行反方向（吸气右转，呼气左转）1分钟。
8.  结束时，吸气回正，呼气放松，双手转成智慧手印。

#### 12. 蛙式

此体式挑战核心和腿部力量。孕期、经期不宜练习。

**做法：**
1.  蹲姿，脚跟并拢，脚趾打开，臀部坐于脚跟上。
2.  双手向前，指尖张开触地。
3.  面部朝前，脊柱延展，胸腔打开。
4.  吸气，伸直双腿，臀部向上推向天空，头部放松。
5.  呼气，屈膝下蹲，回到起始姿势。
6.  配合呼吸重复。建议完成26次，可渐增至54次。

**降阶与调整：**
*   在手下方垫瑜伽砖或任何垫高物，缓解手腕压力。
*   起身时腿可微屈，不必完全伸直。
*   若膝盖有伤或吃力，可让脚跟落地练习。
*   从5次或10次开始，循序渐进，尊重身体感受。

**完成**：26次后，深吸气起身，呼气时脚跟落地，手放松，轻微弹动身体。然后深吸气，慢慢卷起背部，旋转肩膀放松。

### 拉雅冥想

拉雅（Laya）意为“超凡脱俗”。这是一个具有创造力的冥想，帮助我们将注意力集中在生命中的优先事项和更高觉知上，摆脱散乱、依附的状态，获得清晰直觉和深度专注。

**准备：**
1.  保持坐姿，脊柱挺直，下巴内收。
2.  双手可合十于胸前，或结智慧手印放于膝盖。

**曼陀罗（咒语）：**
`Ek Ong Kar Sat Nam Siri Wahe Guru`
（意为：唯一的宇宙创造之神，真实是它的名字，其伟大难以言表。）

**练习方法：**
1.  唱诵此曼陀罗时，需配合丹田有力的收缩和横膈膜的上提。
2.  声音应从丹田发力，经心轮，从喉轮发出，嗓音靠后，而非只用嗓子。
3.  重点感受唱诵时丹田的运动和横膈膜三次有力的上提。
4.  跟随课程视频进行11分钟的唱诵冥想练习。
