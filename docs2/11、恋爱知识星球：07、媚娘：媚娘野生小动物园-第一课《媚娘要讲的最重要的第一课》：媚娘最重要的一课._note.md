# 恋爱知识星球：07、媚娘：媚娘野生小动物园 第一课《媚娘要讲的最重要的第一课》

## 概述
在本节课中，我们将要学习媚娘认为最重要的第一课。课程将帮助你梳理思路，提升格局与认知，了解外部世界，并审视自身状况。我们将通过11个核心要点，探讨如何准确自我定位、把握市场行情，从而在感情、职场等各方面提高胜率，节约时间，避免不必要的失败。

## 第一章：加分项与减分项
上一节我们介绍了课程的整体目标，本节中我们来看看如何评估自身的“价值”。我们将人的价值分为**加分项**和**减分项**，这有助于你进行准确的自我定位。

以下是主要加分项：
*   **身材与颜值**：外在形象。
*   **变现能力**：即赚钱和工作能力。
*   **家庭环境**：指家庭是否和睦恩爱，而非单纯指经济条件。
*   **年龄**：在婚恋等市场中，年龄是一个客观因素。

以下是次要加分项：
*   **不能直接变现的技能**：如乐器、骑马、小语种等，用于增加个人魅力。

以下是常见的减分项：
*   **失态行为**：如重要场合醉酒失仪、说不着调的话。
*   **讨好型人格**：通过迎合他人来维持关系，这会导致关系不平等。
*   **爱计较爱比较**：自身格局不高却看不起他人。
*   **盲目膨胀**：取得一点小成绩就自大，让自己看起来像个笑话。

**核心技巧：换位思考**
如果你不知道如何判断，请学会换位思考。想象十年后的自己如何看待现在的你。例如，假设你23岁，在男友的生日派对上喝醉失态，形象狼狈。请客观地思考，别人看到这样的你会作何感想？如果像个笑话，那就不要做。

**关于讨好型人格的特别提醒**
讨好型人格是重大减分项，尤其在主要加分项都不错的情况下，它会让你非常吃亏，容易吸引渣男。任何健康稳定的关系都不是靠单方面讨好建立的，这首先意味着不平等。人都有“贱”性，你越讨好，对方越觉得你不值钱。感情中应尽量保持公平交流的状态。

**主次关系**
请务必分清主次。如果主要加分项（身材、颜值、变现能力、家庭、年龄）分数不够，次要加分项再多也用处不大。必须把心思放在提升主要加分项上。同时，若主要加分项不错，但有一个雷区减分项（如过于计较的性格或非常无趣），也会很吃亏。

## 第二章：你到底有没有自我？
梳理了自身的价值项后，我们需要探讨一个更根本的问题：你是否拥有独立的“自我”？我发现很多人缺乏主心骨，做决定时总是依赖“我妈妈说”、“我朋友说”，而非基于自身的真实感受。任何旁人的话都可能让他们动摇，甚至因此毁掉一段原本不错的感情。

**他人意见的影响**
有些人可能自己过得不好，便以“关心”为名质疑你的另一半，希望你的感情也出问题。有些长辈则可能用落后于时代的人生经验，对你的事情上纲上线。面对这种情况，你需要有自己的判断。

**核心原则：听从内心**
别人的意见可以参考，但最终判断必须听从自己内心的感受。作为当事人，一段感情里是开心多还是伤心多，只有你自己最清楚。如果开心多，就继续；伤心多，就离开。担心离开后遇不到更好的？那恰恰说明你自己不够好。当你足够好时，便无需担心。

**关于父母意见**
如果父母是成功人士，他们的“野生家学”是宝贵财富，应该多听。如果父母受教育少、与时代脱节，你可以选择不听，但反抗就要反抗到底，或者表面屈服、实际按自己的想法做。等你做出成绩，他们自然会闭嘴。

**学经验要找对人**
学习经验一定要跟有经验的人学。你的时间和精力非常宝贵，年轻时光转瞬即逝。人生下来就是独立的个体，你必须掌握自己生活的节奏。如果连自己的人生都控制不了，只会被周遭的意见粉碎，最后一事无成。

**在感情中如何把握“自我”**
在感情中，不要过分强调自尊。自尊自爱是对自己的内在要求，让自己保持在周围人认可的状态，而不是挂在嘴上指责“你不尊重我”。别人如何对你，往往是你给了对方这样对待你的机会。如果对方不尊重你，你要反思自己为何让这样的人有机会靠近你，并学会判断和提早远离。

**不要浪费精力**
不要为不值得的人和事浪费精力。比如，在低消费场所遇到没素质的人，这是常态，不必大惊小怪或四处声讨。有这精力不如用来提升自己。同样，如果亲戚看不起你，也无需计较，未来如何尚未可知。

**恋爱中的具体建议**
在恋爱中，不要让人觉得很烦。如果对方不回信息，不要连环追问“你在干嘛”、“为什么不理我”。这很掉价。没有人喜欢被细节框住。
**恋爱初期建议公式**：
```
见面时：热情似火
分开时：冷若冰霜
回短信：延迟半小时
回电话：隔一小时再拨，并诚恳道歉（“刚才在忙工作”）
```
切忌电话秒接、信息秒回。你要有自己的世界，对方不理你时，你也有自己的事情要做。

**关于被欺负**
如果老受欺负，先看看欺负你的人是否比你强很多。如果是，你可能只能暂时忍耐，同时反省自己为何显得好欺负（是收入低、娘家不硬，还是性格太软）。通常，当场拒绝和反抗是最好的解决办法。无意义的忍让不会让人记得你的好，反而会让人觉得你好惹。

**关于借钱**
不要为了投资自己（如整容）去借钱，特别是网贷。网贷合同复杂，复利惊人，风险极高。真有急用小额资金（1-2万），可使用支付宝等正规渠道的小额贷款，并尽快还清。不要轻易向朋友借钱，消耗人情。

**初期期限问题**
恋爱初期“端架子”的期限不一定是三个月，取决于关系发展程度。前两个月可以保持一定距离感，第三个月逐渐放松，要循序渐进。

## 第三章：社会的真面目
在探讨如何建立“自我”之前，我们需要先认清我们所处的环境。我们必须认清社会现实，理解人性本恶，并学会面对、理解乃至利用人性的弱点。

**世界观**
要认清，人类的本性有其丑陋的一面，这是必须接受的现实。并非美好不存在，但在大多数情况下，无权无势的生活可能一团糟。我们要学会巧妙地避开不必要的麻烦，节省精力和时间。不要遇到点事就想改变社会，实惠的做法是先提升自己，保护好家人。

**为何是人性本恶？**
观察现实，很多人连陪伴宠物到老都做不到，更不用说赡养父母了。这就是人性，很正常。接受这个真实的世界，你反而能发现更多机会，因为人都有弱点。

**不要自我设限**
同时，也不要过度悲观，不要事情还没做就先把自己判了死刑。很多人因为没见过和自己条件差不多的人是如何成功的，所以缺乏信心。如果你看到同学开淘宝店从亏钱到年入百万，你自然会想“他能行，我也能行”。关键在于看到可能性。

## 第四章：你到底想做一个怎样的人？
认清了社会的残酷面目后，我们来思考一个核心问题：你究竟想成为什么样的人？以下是我个人认为性价比最高的做法。

**核心准则**
所有卑劣的手段和做人方式，终究没有好下场。时代给每个人都留有机会。卑劣包括故意损害他人名声、造谣、挑事、开不恰当的玩笑还反怪别人开不起玩笑等。对于这种人，请不要争吵，直接拉黑远离，并提醒共同朋友不要在他面前提及你。

**谨慎交友**
例如，30岁以上却得不到原生家庭和伴侣关爱的人，负能量通常很重，建议远离。人的一生（20岁到70岁）只有大约18250天，请用心经营，尊重自己唯一的一辈子，不要被负能量的人消耗。

**确定目标，不要攀比**
想清楚你想要什么样的生活状态：是小富即安，家庭和睦？还是物质更丰裕？想好后，就心甘情愿地去经营，不要后悔。最怕的是定了目标，看到别人过得更好又心生嫉妒。嫉妒使人后退，浪费你打造自己的宝贵精力。

**远离不同层次的人**
对于生活中比你差很多的人，也要远离，拉黑删除，不让他看到你的生活。你的自信不应来自于穷人的羡慕。如果因为没保持距离，对方因嫉妒而攻击你，那后果只能自己承担。

**明确目标**
有人可能目标很直接：就是想做一个能让男人愿意为她花钱的女人。这没有问题。关键在于，这样的女人能给予男人想要的东西，而有的女人给不了。具体给什么，我们后面会讲。首先，你要成为一个对自己有要求的人。

## 第五章：做一个对自己有要求的女人
现在，假设你已经决定要成为一个对自己有要求的人。例如，你的要求是拥有奔驰宝马和大平层的生活（更顶级的要求我们另开课讨论）。那么，具体该如何做呢？

**设立崇拜与学习对象**
人无完人，你需要在不同领域设立学习目标：
*   **美妆美容**：找一个学习对象。
*   **身材管理**：找一个目标（不一定是健美教练）。
*   **工作事业**：找一个努力追赶的对象。
*   **家庭经营**：找一个参照物。
目标可以是身边的人，也可以是某个你欣赏的公众人物。一定要有参照物进行对比。

**案例：专注工作提升价值**
我有个小姐妹，条件很好但感情不顺。她把心思全放到工作上，非常欣赏一位女领导，拼命学习、加班。两年后获得晋升，薪水翻番。这时再去谈婚论嫁，对方家庭非常重视，不敢怠慢。这就是“磨亮了剑再上战场”，当你的主要加分项（如变现能力）很强时，底气自然足。

**分领域设定目标**
*   **外貌**：向皮肤好的美妆博主学习，只吸收养分，不做道德评判。
*   **身材**：找靠谱的教练。这涉及开支，所以年轻时要努力积累财富和经验，30岁左右才能做到“有闲有钱”去打造自己。
*   **工作**：不断提升，这是最快捷的人生自由之路。设定实际的目标（如先挣100万），拼命靠拢。
*   **经营家庭**：能把家庭和男人照顾得很好，让他离不开你，会很有成就感。这需要你提前积累好工作和财富，才能在需要分散精力时从容不迫。
*   **原生家庭**：父母能给是福气，给不了也不必埋怨。你要想清楚自己要成为什么样的父母。原生家庭给得太多，有时反而让孩子失去奋斗动力。

**本章重点**
找到正确的努力目标，越早思考并执行越好。优秀的人无非是以最快速度对自己执行最高标准，用努力和狠劲甩开别人。他们后来的雍容华贵是理所应当的。早点决定目标，奋力前进，不要怕苦。

**自我审视**
最后问问自己：之前做事是否全力以赴了？很多人只是敷衍了事。要知道，如果你每次都做到120分，长期积累下来，这种思维模式一定会给你带来回报。天道酬勤，努力的方向很重要，但全力以赴的态度本身就有价值。

## 第六章：你的时间究竟有多宝贵？
明确了目标，我们来看看实现目标最重要的资源：时间。我常提到几个关键阶段。

**18-25岁：七年蓄水期**
大学时期就要开始学会“交易”和“变现”，哪怕只是小生意或卖技能，积累的是经验。切记不要盲目投资或捞偏门。可以谈恋爱，但只图开心和情绪价值就好，**切记不要让男人阻碍你前进的步伐**（如为他放弃出国、留在不喜欢的城市）。关于大学赚钱，我会在微博分享更多案例。有条件的可让父母帮忙在大城市买房，没条件的别乱买商铺，最重要的是积累社会经验和个人能力。

**25岁：女人的分水岭**
此时，你应该通过之前的积累，知道自己有能力赚钱，从而拥有自信。女人的自信来自口袋里的钱和周围人的重视。如果25岁还浑浑噩噩，这节课后请好好反思，还来得及，只是会更辛苦。

**25-35岁：黄金十年**
这十年，你必须靠自己的双手，拥有车子、存款、好身材和颜值。样样都要花钱。**不要把时间过多浪费在无法变现或增加魅力值的事情上**，如追剧、追星。要合理分配精力。
**重要提醒**：次要加分项（如弹钢琴）不能和主要加分项（身材、外貌、变现能力）本末倒置。弹钢琴只是在男人面前展现魅力的一小点，不能当饭吃。核心还是提升主要加分项。

## 第七章：赚钱要从什么时候开始？
答案很明确：从大学开始。你必须要有变现的意识。大学不是混日子拿学分的地方。当你的同学开始做淘宝年入几十万时，你要思考“为什么他行我不行”。这种意识比赚钱多少更重要，重在尝试和积累经验。

**选择专业**
对于即将高考的人，尽量选择容易变现的专业，避开那些毕业后不知如何变现的专业。

**风险控制**
切记投机和捞偏门。生意可以从小做起。在没有较大把握时，不要做高风险生意（如借钱囤货）。学生时期抗风险能力弱。

**案例：江小姐的经商思维**
我闺蜜江小姐小学时，用100块雇班长帮她做完全部暑假作业，自己则在家设计美少女战士图画。开学后复印几百份，以5-20元的价格卖给同学，甚至卖到其他年级，赚了几千块。她从小就有这种经商思维。虽然被老师批评，但这种意识很宝贵。

**大学赚钱案例**
大学里赚钱的方式很多：做淘宝、代购韩国衣服、利用信息差赚差价等等。这个话题很大，以后可以单独开课分享。

## 第八章：你的价值在哪里？
22岁走出校园，成为社会人。社会是残酷的，不再是同学间的“哈哈哈”。你拿什么和别人交换？你的利用价值是什么？

**可变现的价值**
例如：剪辑视频、PS、编程、文案写作、心思细腻等。如果你拥有别人需要到外面购买的价值，而你能提供，你就为自己创造了机会。

**分清价值类型**
弹钢琴等不能变现的技能，除非你想做钢琴老师，否则不必过度深造。这些属于次要加分项，是在男女关系初期用来吸引对方的“魅力值”，但不是核心吃饭的本事。你需要有一两项，以免生活无聊，但更要紧的是打造可变现的核心价值。

## 第九章：人脉真的是万能的吗？
很多人遇到事情喜欢靠关系：靠父母、靠朋友。但人脉真的是万能的吗？不是。

**核心是自身能力**
要做成事，除了人脉，你自己必须有本事。别人把路铺好，你做砸了，以后就没人再给你用人脉。例如，父亲托关系给你找了工作，但你做不好，公司也不会白养你（除非亲戚是大客户）。如果你花几年时间混圈子，终于得到一个项目，却因能力不足做黄了，就再难有合作机会。
**主次要分清**：人脉重要，但自身能力更重要。好比有一个亿广告预算，但产品不好，也是白搭。

**永远保持谦逊**
要多向大佬请教，永远认识到自己不够好。不要只在自己小圈子里觉得高，要放到社会中和更优秀的人比较，看到差距，努力追赶。

**什么是真朋友？**
朋友讲义气，雪中送炭才是真朋友。平时吃吃喝喝看不出什么，需要经历事情（往往需要两三年）的考验。真朋友会明面上支持你，有错私下告诉你。

**交友原则**
*   **远离负能量**：远离那些生活各方面都一团糟的人，他们负能量太足，且容易嫉妒你。
*   **避免层次相差太大**：身边尽量不要有等级相差太多的朋友，以免引起不必要的麻烦。无情是最省力的。
*   **利益见真情**：一个人的义气体现在能为你牺牲多少利润。例如，开饭店的朋友让你永远免费吃。
*   **谨慎对待穷人**：不要和穷人谈信誉，他们容易被利益诱惑。我宁愿吃亏退让，也要赶紧远离，以免被赖上。
*   **接受友谊的变化**：年少时的朋友，如果成长速度差异持续扩大，关系会自然淡化。这是常态。
*   **不说坏话**：不要因为你和A关系不好，就要求好朋友也必须和A绝交。社会不是非黑即白，每个人都有自己的选择和利益考量。
*   **不做软柿子**：软柿子派不上用场，没有真朋友。
*   **女性友谊**：女人之间99%没有真正的友谊，这很难。能经历大事小事、彼此信任5年以上的，才勉强算是。要随时祈祷朋友越混越好，以免生妒。
*   **观察期**：对新结交的社会朋友，切记不要一开始就掏心掏肺，要有一个观察期。保护好自己的隐私。
*   **职场无朋友**：职场中更要小心谨慎，没有真正的朋友。不要讲同事和领导的坏话，别人对你的好基于你的工作能力，而非你带的奶茶。

## 第十章：保养大纲
很多人关心保养。这里先讲一个大纲，以后开单课细讲产品和使用细节。

**1. 颜值**
*   **25岁后**：精华液至少2瓶以上（如保湿+美白/修复）。
*   **每周**：去一次角质（敏感肌选温和产品）。
*   **日常护肤**：25岁后建议步骤：洁面 -> 水 -> （美容）油 -> 精华（2-3种） -> 眼霜 -> 乳/霜。
*   **每周**：至少敷3次面膜。建议选择欧美系（效果立竿见影）或韩系（更天然）。护肤品尽量用同一系列。
*   **挽面**：可让妆更服帖，修掉面部绒毛。
*   **脱妆根源**：通常是保湿、去角质没做好，或定妆不好。日常备粉饼补妆。
*   **关键**：皮肤底子好是关键。做好护肤，即使平价粉底液效果也不错。明星皮肤好，所以粉底打得薄。

**2. 周期医美**
最立竿见影的是激光美容，建议每月进行。**切记不要去小作坊用垃圾机器**，最好选择以色列进口的机器。国内技术和医生水平有待观察，很多人选择飞韩国做。

**3. 头发**
*   **洗发水**：不用开架货，选择更好的产品。
*   **吹风机**：**戴森吹风机必买**，不伤发。
*   **护理步骤**：洗发后使用发乳、发油。头发少/贴头皮可用丰盈喷雾（如Oribe）。
*   **脱发**：头发少可能气虚，需早睡和食补。

**4. 补充剂**
我觉得胶原蛋白补充剂用处不大。目前觉得有效的是一款抗糖丸（POLA），睡前吃，皮肤会变好。

**5. 指甲与睫毛**
*   **指甲**：定期修剪，保持干净。日常建议透明或裸色、深红色等沉稳单色，避免花哨。职场中尤其忌讳。
*   **睫毛**：可嫁接，使人更精神。但不建议常做（伤自身睫毛），可三个月一次。懒人可选。

**6. 身体保养**
*   **泡澡/泡脚**：对女性极好，促进血液循环，增强免疫力。
*   **拨筋**：每周进行面部和背部拨筋，可疏通经络、提升紧致、淡化纹路。推荐台湾手法的拨筋。
*   **去角质**：每天或定期全身去角质（推荐产品：Soap & Glory, Lush）。
*   **身体乳**：洗澡后必须涂抹全身，特别注重脚底、脚踝、手肘、膝盖、臀部等易干燥部位。
*   **除毛**：腋毛必须处理干净。比基尼区毛发也需修剪整洁。有条件可做激光永久脱毛，或购买家用脱毛仪（如SILKPRO）。
*   **防晒**：**每天出门必须涂高倍数防晒（SPF50+）**，抵御紫外线侵蚀。

**7. 坚持与时间利用**
认真完成一套保养流程可能需要两小时。可以利用这段时间听音频或规划思考。坚持半年，你会有质的飞跃。

## 总结
本节课我们一起学习了媚娘认为最重要的11个要点，从认识自身的加分减分项，到建立独立的自我，认清社会现实，明确人生目标，珍惜宝贵时间，及早培养赚钱意识，厘清自身价值与人脉关系，并了解了全面的自我保养大纲。核心在于，你要成为一个对自己有要求、有目标、能独立思考、并愿意为之全力以赴的人。请课后认真思考并执行，你的生活将会发生积极的变化。
