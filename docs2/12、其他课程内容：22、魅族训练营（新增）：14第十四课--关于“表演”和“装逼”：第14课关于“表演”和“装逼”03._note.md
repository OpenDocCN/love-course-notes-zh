# 魅族训练营：14：关于“表演”与“装逼”的核心概念与实践

在本节课中，我们将要学习“表演”与“装逼”在魅力塑造中的核心作用，理解其背后的心理机制，并探讨如何在人际交往中恰当运用这些策略。

## 概述：理解“装逼”的本质

“装逼”本质上是一种自我包装与魅力展示的策略。它并非贬义，而是一种通过适度修饰与表演来提升个人吸引力的普遍技巧。

## 面对非议：有所得必有所担

上一节我们介绍了“装逼”的本质，本节中我们来看看如何面对随之而来的非议。

从某种意义上来讲，这个世界是公平的。你想要有所得到，就要有所付出。既然你选择做“装逼”这件事，就不能害怕被别人说闲话。你不要太在乎别人的眼光。

对于极端自我的人也是同样的道理。比较自我的人是不太在意别人对自己的看法的，就爱做自己的事情。所以你会发现那些很有魅力的女生，她们会经常做一些你所认为很装逼的动作。但是你又会发现可能会有一些女生暗地里说她坏话，但是她完全不在乎。

这是一个很大的性格要点。既然你要获得魅力、制造魅力，你要做“装逼”这个动作，那你就不会在意别人对你的看法。你就要勇于承担可能会有人说你坏话的后果。我们会在最后一课讲关于担当的问题。你没有办法避免说，我做了这个行为就能防止别人不说我坏话，这是不可能的。只要你做这件事，就可能有些人会看不顺眼。

你看到这个世界上，现在主流的价值观已经往“绿茶婊”这个角度来转了。有很多现在主流的媒体开始宣扬，你就是要做一个“绿茶婊”，因为这样男人才会比较爱。所以，还是那句话，有多大收获就有多大付出。你要有所担当。你既然做了这个事情，你就应该有所承担。

## 关于类型偏好：优秀胜于类型

关于是否所有女人都喜欢“绿茶婊”类型，我们以前关于男人喜欢什么样的类型的女人都回答过。

其实我的答案就是：男人喜欢各种类型的女人。无论你是清纯一点的女人，还是浪荡一点的女人，男人都喜欢。如果一个女人选择当“绿茶婊”类型的女人，那么她就是选择了这个类型。但是你不要认为男人不喜欢“绿茶婊”，他也会喜欢，所有类型他都喜欢。关键在于，只要你在这个类型中表现优秀就可以了。

所以说，那些所谓的当了“绿茶婊”的女人，是选择了这种类型。可能她们认为这种类型更能够吸引男人，而且也比较称得上自己，可能更加虚荣。相比其他类型的妹子，可能这类女人的虚荣心和自我存在感、自恋的感觉更强一点。不同的类型各有优缺点，是这样。并不是说一定要你成为“绿茶婊”，大家就是要自己权衡。

## 澄清误解：“装逼”是普遍技巧

这里可能有一个误解。我们说“装逼”这个概念，可能让大家联想到了“绿茶婊”，其实不是。

可能有100个人都在“装逼”，其中有10个人成为了“绿茶婊”，还有其他90人成为各种类型。就像我刚才讲的，就比如说我要邀约一个女生，结果这个女生说什么她跟姐妹在逛街所以不愿意出来，从而拒绝了我，这也叫一种“装逼”。这是一种自我的包装策略，就是展示魅力的方式，并不是说一定要像所谓的“绿茶婊”那样虚荣。

可能你们认为所谓的“装逼”就是“绿茶婊”的特征，并不是所有人都会“装逼”。只不过有的人在这个概念上不强，没有我们所谓的“造魅”的概念。我希望给大家讲清楚，大家头脑中要有这个概念：人是需要包装的。

什么叫包装？就像我以前也讲过，我讨厌做人要谦虚的观点，我认为人就应该骄傲，就应该虚荣。如果你有什么优点，有什么成就，你就应该拿出来说。那么我们拿出来说的时候，也可以稍微包装一下，或者稍微吹牛一下。

**公式示例：自我包装的适度夸大**
`实际成就值 = 8`
`对外宣称成就值 = 实际成就值 + 适度夸大值 (例如：8 + 1 = 9)`

比如说你在某某比赛中得了第二名，你完全可以吹牛说是第一名。这是为了包装的需要，为了“造魅”的需要。你得让别人感觉你很厉害。那么男人对女人也有很正常的这种心理需求，可能要让女人觉得自己厉害，那女人的很多魅力点、价值点也要通过包装去体现。

我不知道大家有没有人是设计师，比如平面设计。其实平面设计最大的一个作用，就是把一些平淡无奇的东西修饰得美好一点。也就是说这个东西可能平淡无奇，比如摄影。可能你去看那个日出很平淡无奇，但是摄影师能把日出拍得非常美，这其实也是一个美化，也是一个包装的作用。你可能是一个普通的人，那么我们把你进行包装一下，你也会比普通的你显得更有魅力一点。实际上“装”的意义在这里，而不是让大家和“绿茶婊”有过多的联想。不要认为你“装逼”就一定吸引来那些关注虚荣的女人，并不是这个意思。它是一个普世的价值，是一个普遍的技巧，并不是只针对某一类型。

## 核心误区：对喜欢的人更需“装”

这是一个最大的误区。你认为对喜欢的人就“装”不起来，这是你犯了一个传统的误区。你就认为我喜欢一个人，那我就应该理所当然对他好。其实你这个观点是很有问题的。

实际上，最原始的观点是，你在跟你喜欢的人相处时，满脑子里都是爱，从所谓推拉角度来讲，你都是“拉”的动作，几乎没有“推”的动作。你认为我喜欢一个男人，我为什么要做“推”的动作？甚至你脑中都没有博弈的概念，这就是我们需要给你补充的。

对于喜欢的人，哪怕你喜欢这个男人，你也需要去对他“装”，而不是说你喜欢一个男人，你就应该对他坦荡、坦然相处。时间长了，你也会丧失魅力。你需要保留，需要包装自己，也需要有一些博弈的动作。这就是我们一直要强调的。

传统的这种思想最毒害的就是：你只要喜欢一个人，你就应该对他好。这种观点是最害人的。我们一定要有博弈的概念。不能说喜欢跟一个男人谈恋爱了，就什么都不管了，跟他好就行了，不能这样。

## 技巧探寻：源于观察与贯通

这是一个非常宏大的话题，你需要从生活中去学，并不是我们今天两句话就能讲完的。我今天只是初步讲一下这个概念。当然具体怎么去做，你可能要有更多的观察、更多的思考、更多的方法。

当然我也简单提了一两个方法。比如我刚才讲，跟陌生人刚开始接触的时候，对方第一次邀约你，尽可能想各种方法拒绝掉，这也是一个技巧。我认为你要从各个方面去学习。其实我们每堂课几乎都有相关的恋爱技巧去学，只不过你不要认为它是一个固定的技巧，它是会贯通的。

比如我们讲“推拉”，“推拉”其实也有“装”的意识在里面。比如说实际上你很喜欢这个男生，但是从恋爱吸引力角度来讲，你又不得不做一些“推”的动作。就是你明明喜欢这个男生，你完全可以跟他一直在“拉”近距离，但是为了保持距离、保持神秘感、保持新鲜感等等，你又不得不去做一些“推”的动作，这些都是恋爱技巧。

## 实战建议：朋友圈的高效学习法

对于在朋友圈中女生发什么内容比较容易提高“逼格”这件事，讲太多话都是没用的。

最好的方法就是找一个高手，观察他的方式就行。也就是说，你可以在你身边或者在你的朋友圈中，找一个你认为优秀的女生，然后你看她发什么内容就行了。这是一种最好的学习方法。这样比我讲各种空话都有用。

**代码示例：高效学习路径**
```
1. 定位你欣赏的“高手”（身边或朋友圈中的优秀女性）。
2. 系统观察并记录她发布的内容类型、风格、频率。
3. 分析其内容背后的目的与效果。
4. 内化并尝试模仿、调整，形成自己的风格。
```
