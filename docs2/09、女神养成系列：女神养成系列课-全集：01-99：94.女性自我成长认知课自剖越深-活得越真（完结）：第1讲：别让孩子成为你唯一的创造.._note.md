# 女性自我成长认知课：01：别让孩子成为你唯一的创造

## 概述
在本节课中，我们将要学习一个常见的、可能困住女性的心理剧情：即渴望通过他人（如伴侣或孩子）来获得稳定与价值，而非直面自我成长的责任与挑战。我们将剖析这种心态背后的原因，并探讨如何走向真正的精神独立。

---

直接活在人生的真相中会带来痛苦，但能获得真正的自由。你是否敢于做出这样的选择？欢迎来到女性自我成长认知课。

最常见的困住女性的剧情，是渴望由一个男人带给自己稳定的归宿。当一个穷男人对女人说“我养你啊”，这会让很多女人动容。当一个事业有成的男人这样许诺未来时，对于内心住着无助小女孩的女人来说，这简直是致命诱惑。

我们都太渴望跳过直面生活、挑战自恋的艰难过程，而一脚踏入收获期的平稳与丰盛。著名的存在主义女哲学家、《第二性》的作者波伏娃曾说：男人的幸运在于，从一开始就没有被告知有一条容易的路。而女人则被鼓励去走那条看似容易的路。

没有退路地直面生活，需要一次次挑战自恋、承受自恋受挫的精神折磨。这个过程可谓扒皮抽筋般的重塑。一边是铺满鲜花的伊甸园之路，另一边是未知的丛林。你会选择哪一边？

即便智慧如波伏娃这样的女性也会指出：鲜花之路不一定通往天堂。无论男女，都必须走自己的路，完成自己的英雄之旅，才能抵达真正永恒的伊甸园。

女人可以创造孩子，但别让孩子成为你唯一的创造物。你的创造力与激情，并不少于任何男人。你不需要任何人的支持与许可，可以直接去创造属于你的世界。

---

上一节我们讨论了女性被鼓励走“容易之路”的社会现象，本节中我们来看看个人经历如何印证这一点。

以我自己为例。我是独生女，并未感受到父母重男轻女，因此从小认为男人能做的事女人一样能做，甚至曾狂妄地觉得没有哪个男生比我聪明，并乐于与男性竞争。即便有这样的成长背景，当我中年回望，才懂得二十多岁的我一样深陷“女人是第二性”的集体无意识催眠中。

陷入这种剧情的根本原因，在于虚弱的**自体**。

**公式：陷入依赖剧情 ≈ 虚弱的自体**

因为自体太虚弱，所以会幻想一个强大的男人可以拯救自己。这个幻想中的强大客体，有时会投射为我自己膨胀的自恋，觉得自己无所不能；而在自恋受挫时，我又会幻想有一个像好爸爸一样的男人为我遮风挡雨。

可是，一个男人凭什么为你遮风挡雨呢？尤其是那些未被父母宠爱过的女孩，更难以相信自己配得到爱。于是，各种情感导师会教女人如何讨好男性、留住男人的心。我看不上这种“跪舔”方式，但我内心的逻辑与之是一致的，只是我的方法显得更高冷——我孤傲地显摆自己的聪明才智，内心其实害怕自己不被看见。我妄想自己足够优秀，就能得到一个男人的庇护。

网络上有一句话：“你想找个男人遮风挡雨，没想到风雨都是这个男人带来的。”我想，对于大多数自体虚弱、希望让男人扮演强大客体的女人来说，结果往往是遭遇给自己带来凄风苦雨的男人。

这并不难理解。一个人在觉醒前，总是无法避免照着父母的样子寻找伴侣，哪怕外在看起来不同，但内核是一致的。或者说，找到的总是与自己心理发育水平相近的男人。或许对方外在能力很强，然而在情感能力上一样很虚弱，自恋脆弱。

因此，我直到中年才开始停止逃避自己的虚弱和恐惧，决心冲上自己生命的一线，百分百地负起责任来。我不再期待男人能为我的痛苦做些什么，也不再期待男人能给我怜悯和支持。我经历一次又一次因自恋受挫导致的自体破碎感，在生与死的较量中，看看自己到底能不能从“第二性”的梦中醒来。

---

认识到问题后，我们需要具体分析它是如何影响个人发展的。以下是我的个人经历剖析。

大家都知道，我的前任男友武志红名气比我大很多。我自认为在心理学的热爱和洞察力上，我与他相差不算太多，但显然他的名气、被认可的程度和收益都是我的很多倍。这是什么原因呢？

因为我曾经深陷在“第二性”的催眠中，认为关系是最重要的。我觉得“我跟武志红不分你我，帮助他红了，就是我的成功”。所以在那段关系中，我大部分的经营意识都用在了他身上，很少有为自我发展而努力的想法。那七年，我甘愿不给自己留后路，发展的步伐整整落后于武志红。

那么，我为何会深陷“第二性”的催眠呢？部分原因是对与母亲共生的渴望。但是很多男人也缺乏母婴链接，他们为何没有陷入共生的妄想？我觉察到自己内心有一种恐惧。

**核心概念：恐惧直面人生**
```
if (恐惧直面风险 && 恐惧自恋受损) {
    选择躲在他人背后；
}
```

我不想直面自己的人生，我恐惧冲上前线可能遭遇的风险。我害怕挫折导致自恋受损。所以我很“鸡贼”地躲在武志红背后做一个谋臣。他成功，我沾光；他不成功，那也不等于我不行。我既享受了他的荣光，又避免了自恋受挫的痛苦。

我的经历正印证了波伏娃的话：女人最大的不幸，是误以为有一条更容易的路等着自己走。简而言之，就是因为我是个懦夫。这世界上其实没有容易的路，男人女人都如此。男女的差别只在于，女人通常更晚领悟这一点。

---

我的故事并非一个完美的励志典范。以下是对“独立”的真正解读。

可能会让各位女性失望，我并没有创造什么价值数亿的奇迹，我只是独立地活下来了。这个“独立”，对于长期关注我的粉丝而言，不仅指物质独立——这对我并不难——难的是精神上的独自存活，即百分百为自己所有的痛苦负起责任来。

诗人鲁米曾说：“万物生而有益。”无论你是那个不被期待出生的女儿，还是被精神阉割的男人，或是饱受父母折磨的虚弱受害者，都可以记住：万物生而有益，而且没有人可以折断它。

## 总结
本节课中，我们一起学习了女性可能陷入的心理依赖剧情，其根源在于虚弱的自体与对直面人生的恐惧。我们通过个人经历的分析看到，逃避自我责任、将价值寄托于他人，最终会阻碍个人成长。真正的自由与力量，源于停止幻想“容易的路”，并勇敢地为自己的生命负起全部责任，完成属于自己的英雄之旅。
