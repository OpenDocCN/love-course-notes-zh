# One Pager公文写作法：02：如何使用附件与排版提升专业度？

在本节课中，我们将学习如何通过合理使用附件和优化排版，来提升One Pager汇报材料的专业度与可读性。这是本系列课程的最后一节。

## 附件的使用场景

上一节我们介绍了One Pager的核心内容构建，本节中我们来看看如何通过附件来补充信息。附件并非One Pager的必需部分，但在特定情况下能发挥重要作用。

以下是几种常见的附件使用情形：

1.  **占篇幅较大的表格、图表或详细展开的信息**。当正文中提及“某某明细或清单见附件”时，意味着这些过于细节的内容被移至附件，既节省了正文篇幅，又方便查阅。
2.  **内容或数据的原始文件**。例如合同、协议、收付款凭证、测算表等。将数据估算依据或确切来源作为附件，能为汇报内容的可靠性提供支持。
3.  **相关的深度分析、操作手册、参考案例等补充资料**。当汇报内容需要额外批注或说明时，可采用附件形式呈现。

## 整理附件的注意事项

说完了几种附件存在的常见方式，接下来我们主要聊一聊整理附件的一些重要注意事项。

以下是整理附件时需要遵循的几个关键点：

*   **为附件编号并标明页码**：当附件不止一份时，需在正文中注明引用的具体附件序号及页码，例如“详见附件1，第3页”。
*   **按出现顺序排列附件**：多个附件的排列次序最好与正文中提及的顺序一致，便于查找。
*   **注意附件间的关联关系**：确保相关联附件（如数据分析与基于此的测算表）之间的数据准确、逻辑一致，以维持文档的可靠性。
*   **控制附件数量**：附件数量建议控制在**三个以内**。过多的附件会喧宾夺主，影响One Pager简洁高效的阅读体验。许多优秀的One Pager即使没有附件，也能清晰地阐述问题。

## 排版的五大原则

有关附件的话题我们就先聊到这里。接下去我们讲讲排版。优秀的排版能直观地传递专业感和美感，提升阅读兴趣。

以下是One Pager版面设计的五大核心原则：

1.  **整体色调统一**：整篇材料最好选用同一色系，通过深浅区分层次。使用公司Logo的主色调是稳妥的选择。在非彩色环境下，**黑、白、灰**的搭配是经典的高级选择。
2.  **调试行间距与页边距**：页边距建议遵循“横宽纵窄”的原则，例如左右各**3.18厘米**，上下各**1.27厘米**，以营造专业、克制的视觉感受。行间距推荐使用**1.2倍**行距，避免单倍行距带来的压抑感。
3.  **字体统一**：全文应贯穿使用一种字体。重点内容通过**加粗**等方式凸显，而非使用彩色字体，以保持整体风格一致。
4.  **适度高亮（Highlight）**：对关键标题、核心数据、结论性语句进行标注。但重点标注的总篇幅不宜超过整页的**1/4**，否则会失去重点。
5.  **图表与文字错落布局**：当页面同时包含文字和多个图表时，应采用错落有致的布局。例如，三个图表可按“左-右-左”的方式排列，使页面结构更稳定、舒适。
