# 机器学习入门：第一章：线性回归基础

在本章中，我们将学习机器学习中最基础且重要的算法之一——线性回归。我们将了解它的核心思想、数学原理以及如何用代码实现一个简单的模型。

## 概述

线性回归用于寻找自变量（X）和因变量（y）之间的线性关系。其目标是找到一条最佳拟合直线，使得所有数据点到这条直线的距离之和最小。

## 核心概念与模型

线性回归模型试图用一条直线来拟合数据。其数学表达式是一个线性方程。

其公式表示为：

**y = wX + b**

其中：
*   **y** 是我们要预测的目标值（因变量）。
*   **X** 是输入特征（自变量）。
*   **w** 是权重（或斜率），代表特征对目标值的影响程度。
*   **b** 是偏置（或截距），代表当所有特征为0时的基准值。

## 损失函数：衡量误差

为了找到最佳的 **w** 和 **b**，我们需要一个标准来衡量预测值与真实值之间的差距。这个标准就是损失函数。

在线性回归中，最常用的损失函数是均方误差（MSE）。它的计算方法是求所有预测误差平方的平均值。

其公式为：

**MSE = (1/n) * Σ(y_i - ŷ_i)²**

其中：
*   **n** 是样本数量。
*   **y_i** 是第 i 个样本的真实值。
*   **ŷ_i** 是第 i 个样本的预测值（即 **wX_i + b**）。

## 优化方法：梯度下降

现在我们知道要最小化MSE，但如何找到使MSE最小的 **w** 和 **b** 呢？最常用的方法是梯度下降法。

梯度下降的核心思想是：通过迭代的方式，沿着损失函数下降最快的方向（梯度负方向）更新参数，逐步逼近最小值。

以下是参数更新的核心步骤：

1.  **初始化参数**：随机为 **w** 和 **b** 赋予初始值（例如0）。
2.  **计算梯度**：计算损失函数关于 **w** 和 **b** 的偏导数。
3.  **更新参数**：用梯度乘以一个很小的数（学习率），然后从当前参数值中减去它。
4.  **重复迭代**：重复步骤2和3，直到损失函数的值不再显著下降或达到预设的迭代次数。

参数更新公式为：

**w = w - α * (∂MSE/∂w)**
**b = b - α * (∂MSE/∂b)**

其中 **α** 是学习率，控制每次更新的步长。

## 代码实现示例

理解了原理后，我们来看一个简单的Python实现。这里我们使用`numpy`进行数值计算。

```python
import numpy as np

# 生成模拟数据
np.random.seed(42)
X = 2 * np.random.rand(100, 1)
y = 4 + 3 * X + np.random.randn(100, 1) # 真实关系为 y = 4 + 3x + 噪声

# 初始化参数
w = np.random.randn(1)
b = np.random.randn(1)

# 设置超参数
learning_rate = 0.01
n_iterations = 1000

# 梯度下降
for iteration in range(n_iterations):
    # 计算预测值
    y_pred = w * X + b
    
    # 计算梯度
    dw = -(2/len(X)) * np.sum(X * (y - y_pred))
    db = -(2/len(X)) * np.sum(y - y_pred)
    
    # 更新参数
    w = w - learning_rate * dw
    b = b - learning_rate * db

print(f"训练得到的权重 w: {w[0]:.4f}")
print(f"训练得到的偏置 b: {b[0]:.4f}")
```

## 总结

在本章中，我们一起学习了线性回归的基础知识。我们首先了解了线性回归模型用 **y = wX + b** 来描述特征与目标之间的关系。接着，我们引入了均方误差（MSE）作为衡量模型预测好坏的损失函数。为了最小化这个损失函数，我们学习了梯度下降法的原理和步骤，即通过计算梯度并迭代更新参数 **w** 和 **b**。最后，我们通过一个简单的代码示例演示了如何从零开始实现线性回归的梯度下降过程。掌握这些内容是理解更复杂机器学习模型的坚实基础。
