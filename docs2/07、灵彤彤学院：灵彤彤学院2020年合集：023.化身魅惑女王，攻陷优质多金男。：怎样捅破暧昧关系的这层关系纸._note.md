# 灵彤彤学院：07：怎样捅破暧昧关系的这层关系纸

## 概述
在本节课中，我们将学习如何判断暧昧关系是否成熟，以及如何巧妙地引导关系升级，而不是通过直接表白来推进。我们将探讨四个核心策略，帮助你从暧昧阶段平稳过渡到恋爱关系。

## 第一点：千万不要表白
上一节我们概述了课程目标，本节中我们来看看第一个核心原则：避免主动表白。

首先，一定不能把主动表白当作关系升级的方法。在我大量的案例库中，大部分女生主动表白，都会被发好人卡，结局以阵亡收场。

这里有一个很奇妙的现象。我问过我身边的大叔型男人们，如果一个非常优秀的女孩子站在你面前，主动向你表白，表示愿意做你女朋友，你会马上同意吗？我本以为男生们都会愿意，但没想到80%的男生都回复不愿意。

女神们都有这么高的失败率，更别说是大部分普通的我们了。这到底是为什么呢？我们来分析一下男生的心态。

女生主动表白，其实会激发他们内心的防御机制，他们会不断猜忌。面对这种表白，他们第一感觉是疑惑对方的动机。他们会想：为什么一个条件那么好的姑娘会直接跟我告白？她周围应该会有很多优秀男性才对，她想清楚了吗？是不是有什么目的？

我以前给我一个女性朋友介绍一个对象。但是当时她正心有所属，所以对此没有表现出太大热情。对方找她聊天的时候，她也是有一搭没一搭的随便处理。聊了一段时间后，男生知难而退，事情也就告此一段落了。虽然以后没有聊天，但两个人仍保留着对方的微信，所以可以看到彼此朋友圈。而通过男生发的各种动态，她发现自己竟然对男生慢慢有了兴趣。也开始主动从朋友那里获得男方信息。再后来她发现自己越来越喜欢对方，这种感觉甚至超过了自己曾经暗恋的对象。于是她鼓足勇气又主动找对方聊天，但对方很明显没有了曾经的热情，不仅不再主动，而且感觉还有点生疏。但她还是感觉很喜欢对方，毫不犹豫地主动表白了。结果可想而知，对方再也没有给她联系的机会了。

大叔款男人和小男生不一样。对小男生来说，被一个女人表白是一件非常自豪的事。但是对大叔型男人来说，被表白是一种压力，因为对他们来说，这是对自己生活节奏和工作节奏的一种破坏。表白这种事会给他们造成很多无形的社交压力。到了这个年纪，他们更喜欢稳定、踏实、循序渐进，并且让自己感觉节奏舒服的爱情。而轻易的表白，只会让他们觉得很唐突，产生对这段感情的抗拒心理。

不管你们的暧昧程度有多深，感情有多好，主动表白的一方也不应该是由女孩子来做。当我们的关系到了可以进一步发展的时候，我们需要做的不是表白，而是引导，给男孩子创造机会，让他们来升级关系。如果你进行了正确引导，也创造了机会，但是对方没有继续推进，就只能说明这段感情还没有到火候，你们完全可以稳下来继续相处，给这段感情增温的机会，而不是直接表白。一旦表白，很可能你们的关系就回到了原点。

**核心公式**：`关系升级 ≠ 主动表白`，`关系升级 = 引导 + 创造机会`

表白不应该是发起进攻的号角，更应该是胜利的标志。

## 第二点：保持一致性
理解了避免表白的重要性后，我们来看看第二个关键点：在相处过程中保持情绪和态度的一致性。

所谓的保持一致性，就是你和对方相处过程中的情绪和态度要保持一致，推进关系要循序渐进。

曾经我的来访者问我，为什么之前男生明明很主动？但奇怪的是，慢慢的我喜欢上他之后，当我主动的时候，对方突然变得非常的冷淡。这就是你失去了一致性的表现。

在早期的时候，一般都是男生处于主动方。很多时候女孩子是非常被动的。后来，女生慢慢感动了，感情就主动了起来。这种反馈的反差，如果一夜之间比较大，就很容易让人产生警惕心理。还是以前讲过的，男生心里也会犯嘀咕：你为什么会突然这么主动？是不是有什么事儿，或者只是单纯为了我而来迎合我。这就解释了为什么很多时候我们跟男生处于你进我退的状态。

以下是保持一致性需要注意的方面：
*   **避免态度突变**：不要从极度被动突然转为极度主动。
*   **循序渐进**：感情的投入和反馈应随着了解的加深而逐步增加。
*   **保持自然**：你的行为和关心应基于当前关系阶段，显得自然而不刻意。

打个比方，比如说你的一个同学，高中毕业以后你们就不怎么联系了，基本上一个月说那么两三句话。有一天，他突然跑过来找你非常热情的聊天，这个时候你是不是也会心里犯嘀咕？是不是也会想，他今天好奇怪，为什么这么热情？你是不是要掂量掂量他为什么要这么做呢？如果你以前和他感情就很一般，你会不会考虑他来找你是出于什么目的？

我们对其他人会有一种既定的判断。这种判断会形成一种意识。当对方打破我们认知的时候，会让我们从心底产生怀疑跟恐惧。这种怀疑跟恐惧就会成为无形的压力，影响两个人的相处模式以及定位关系。这就是为什么很多时候女生越是想升级关系，却往往事与愿违。

所以说，如果你想要跟对方升级关系，循序渐进才是正确方法。朋友们和我在酒后聊天打趣中说过这样一段话：如果一个女生突然对你很热情，那么她一定是电脑有问题，或者写论文遇到困难。要么就是让你改改简历。总之你是一个好人，妹子给你卖个萌，说句晚安，你就感觉他对你跟对别人不一样了。活该备胎，打图机器，免费司机，免费摄影，免费后勤，免费食物快递员。人生三大错觉，手机震动，有人敲门，还有他喜欢你。看到了吧？这就是男人心中最真实的想法。

所以如果你想和男人升级关系的话，一定不要着急，稳扎稳打，循序渐进才是正确做法。

## 第三点：在什么阶段就做什么阶段的事
在学会了保持一致性后，我们需要学会准确判断关系所处的阶段，并采取相应的行动。

当你想推进关系的时候，我们要做的第一件事情不是应该如何推进，而是对现在的关系进行一个判断。你们的感情现阶段处于什么位置？你们的聊天深度是否足够，你们的相处是否舒服，这些问题都需要我们去考虑。可惜的是，大部分女生对于感情阶段的判断是没有把握的。这就是为什么很多女生在火候未到的时候贸然出击，结果死得很惨的原因。

以下是常见的阶段判断错误：
*   **认识初期过度关心**：比方说双方还在认识不久的时候，就过度关心，总是问对方在做什么，跟谁在一起。
*   **过早代入角色**：在对方还没有进入恋爱状态的时候，就过早的把自己放在女朋友的位置上，提出很多限制和要求。

这些做法会让男人在心里产生一些不适应，从而有抗拒心理，把一段本来可能性非常高的感情越推越远。

那么，如何判断你们现在的感情是否适合推进关系呢？这里就要教给大家一个技巧了。这个技巧叫做**服从性测试**。

做法非常简单，就是让一个男生做一件事，或者是给他一个指令去服从，通过他的服从度以及服从的积极性来判断现阶段你在他心中的定位。

**核心代码（行为逻辑）**：
```python
if 对方服从度低:
    关系深度不足，继续培养感情
elif 对方服从度高:
    有好感，可进一步观察
    if 对方主动深入话题或表现出焦虑关心:
        好感度很高，可以考虑引导升级
```

打个比方，比如说你让男生帮你拿个快递或者取一份文件。总之要占用他的一定时间和精力来为你做一件事情。如果他表现得很不愿意，甚至反过来对你说“凭什么”，那么就说明了你们的感情还是不够深，这个时候要做的就是继续加深感情。如果他很痛快地答应了你帮你做事，说明他对你还是有好感的，但是不能完全说明他很喜欢你。如果他继续对这个话题深入下去，比如“天天买买买是不是给男朋友买的呀？你男朋友真好”，或者是非常焦虑地关心你，那么很有可能对你有了很高的好感，说明你在他的心中已经有了很高的地位比重。

并且服从性测试还具有亲密关系提升的作用。比方说最开始你让他帮你拿快递，然后他帮你带份早餐，最后你累了，躺在他身上休息一下，从小到大慢慢升级。所以说，服从性测试可以说是一举三得，一方面可以增加男生付出，一方面可以测试现阶段身份定位，另一方面还可以亲密升级。

服从性测验其实也可以用在挽回中。举个例子说明一下吧，之前有个学员分手前是和男方同居的。那个时候，他们共同养了一条狗。分手以后，男方搬走了，小狗留给了他。我的战略是让她对他进行服从性测试，对感情进行搭建。铺排了一个月左右，服从性测试就有了明显的效果，就差对方提出复合了。

具体操作是，我让她告诉他，既然分手了，这小狗她不想养了，让他来带走，让他来养。最后，男方说，以后他每天都会来喂它、遛它，并且负责它的生活护理、预防针接种，所有关于小狗的事都由他办，说服她留下小狗。这一测试非常成功，因为对方已经等于完全表达出自己想要保持联系的意愿，保持稳定的状态，离他开口提复合就差临门一脚了。

## 第四点：给男人创造机会，让他主动升级关系
通过服从性测试确认对方有好感后，最后一步就是巧妙地创造机会，引导对方主动推进关系。

如果说前面几点你都做到了，并且根据服从性测试了解到男生对你有很高的好感度，那么接下来要进行最后的一步就是升级关系。

我的很多学员都会在深夜着急忙慌地给我打个电话过来：“老师老师，我现在在和一个男生聊天，话题已经很深入了，他应该对我的好感度很高啊，为什么还是不对我关系升级呢？难道是我判断出错，好感度还不够吗？”

其实很多时候男生不推进关系，不是因为好感不够，而是因为你没有给男生足够的机会。即使男生向女生表白，也是需要很大的勇气的。如果你们之前谈论的话题都是吃喝玩的衣食住行，甚至你们都没见过面，联系都是在网络上，出来约会只是看看电影，你们之间还没有牵手拥抱，这更像是一种普通朋友的相处模式。男生心里是根本没有底的。男生又不会读心术，不懂你真实想法，尤其是对于30岁的成功男人来说，没那么多时间去揣摩你心思。

所以呀这个时候我们要给男人一定的暗示与引导，给他一定的场合和机会，让他走出最后一步。对于一个连手都没有牵过的人，你让他直接当你男朋友。而你的目标对象是一个深谙世故的大叔，你认为可能吗？

肢体触碰是最能够增加彼此亲密度的，并且给他明显的暗示。以下是创造机会的方法：
*   **制造合理的肢体接触**：比方说让男生帮你扎头发。在过马路的时候，抓住对方手臂，这些小动作都可以增加你跟他的肢体碰撞频率。
*   **选择利于互动的场景**：你也可以让他带着你去游乐场玩。在这样的特定情况下，会有更多跟他亲密接触的机会。越是人多的场景进行肢体升级，反而不会给他造成紧张感，因为他真的会认为你是出于需要帮助才做出这样请求的。

当你们的肢体接触频率升高，就更容易暧昧了。这个时候你们就会进入亲密区了。频繁地进入亲密区，才能够更快地达到恋爱状态。进入恋爱状态后，只需要建立更多的约会，有更多私密相处的机会，再通过更多的肢体暗示，等待男生舒适而自然地迈出这一步。

大家记住，所有的恋爱都是建立在你们互相有好感的情况下，只要有好感，再加上适度的肢体关系，把控节奏，升级关系就很容易了。我们需要做的就是引导男人，让他们对你表白。
