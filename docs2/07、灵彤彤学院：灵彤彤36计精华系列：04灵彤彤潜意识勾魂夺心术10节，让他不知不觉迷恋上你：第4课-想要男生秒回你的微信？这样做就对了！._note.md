# 灵彤彤潜意识勾魂夺心术：04：如何让男生秒回微信

## 概述
在本节课中，我们将要学习如何应对男生不秒回信息的情况，并掌握一系列方法和理论，引导对方更积极、及时地回应你的消息，从而提升你在关系中的吸引力和主动权。

上一节我们探讨了潜意识吸引的基础，本节中我们来看看沟通中的具体实践。当你对一个人动心时，往往会变得敏感，容易因小事争吵。对方与异性多说几句话，你可能会吃醋甚至胡闹。对方的任何举动都容易挑起你的情绪，他的任何变化都会让你胡思乱想。因此，你可能会不自觉地要求对方秒回信息。当男生开始不秒回你的信息时，你需要明白以下四点原因。

**以下是男生不秒回信息的四种可能原因：**

1.  **他只是忙，并不是不爱你。**
    男生不秒回信息，可能确实在忙。爱你的人也不可能无时无刻盯着手机等待你的消息。他在忙时，无法立刻秒回。但一个爱你的男人即使很忙，无法秒回，也会事后向你说明情况，并用心回复你的信息，将你说的话记在心上。所以，并非所有不秒回信息的男人都不爱你。

2.  **他可能没有把你放在心上。**
    一个不秒回你信息的男人，可能真的没有把你放在心上。因为心里不在乎，即使收到消息，也可能拖延回复，最终忘记。当你问他为何不回复时，他总会说“没看到”或“忘记了”。这样的男人是真的不在乎你。

3.  **他厌烦你，不想回复。**
    如果你总是唠叨，或一件事重复许多遍，他可能感到厌烦，看见你的信息也不知道如何回复。因此，男生没有秒回信息，你也应反省自己是否说了一些他不爱听的话，让他不知如何回应。

4.  **他可能变心了。**
    用心爱你的人，看到你的信息会第一时间回复。因为他不愿错过你的每条信息，不愿浪费和你说话的每一秒。当一个男人不秒回你了，也可能是真的不爱你了，因为心里已经没有你的位置，所以对你的任何信息都开始不感兴趣。

很多不了解以上原因的女生会有疑惑：男朋友刚追我时很热情，信息秒回，天天早安晚安，有说不完的话。但现在他越来越不主动，不主动发信息打电话，我发信息他也不回，是不是不爱我了？最近相亲的男生，开始挺热情，天天找我聊天，但最近好几天没理我，他是不是不喜欢我了？有个追我的男同事，开始天天聊天，但最近几天没联系，他是不是喜欢上别人了？

确实，很多女生都会遇到男人不主动甚至不回复信息的时候。那么问题来了，究竟怎样做，才能让男人喜欢给你发信息、打电话呢？

**以下是引导男生主动联系的六招：**

1.  **认真倾听并给予鼓励。**
    在双方交流时，你要认真倾听，关注点放在他身上。在恰当的时期给予小小的鼓励、一个赞美、一个眼神甚至一个“嗯”，都能让他觉得与你交流很愉快、舒适。

2.  **适当恭维对方。**
    比如他想要逗你开心、给你讲笑话，哪怕不是很好笑，但态度是好的。即使礼貌性地，你也应该附和几下。虽然可能有些违心，但每个人的笑点不同，他的初衷是好的，需要你的鼓励和反馈。下次他必然会更加用心。因为对于男人来说，他的付出收到你的反馈，就是一种认可。每个人都渴望获得别人的认可。

3.  **保持愉悦温和的语气。**
    如果你的语言文字中透露负面情绪，谁都不会愿意和你聊天。保持正能量才能更愉快地交流，毕竟谁也不是谁的情感垃圾桶。

4.  **有时不回复也是一种态度。**
    不要每次聊天都由你来收尾。对于那些可回可不回的信息，那就断网睡觉。保持自身的神秘感，也是一种撩人的手段。说不定对方会一直想着你，隔一会儿就看下手机。

5.  **保持矜持，但也要适时主动。**
    矜持是必须的，但也不能完全不主动。该主动找对方时也要主动一些，不然他会以为你对他没意思。时间久了，他可能转移目标，那时你就后悔莫及了。

6.  **在聊得兴起时主动结束。**
    当你们聊到正兴起时，要学会主动掐断话题或挂电话。理由可以有很多，比如：聊着聊着睡着了、困了明天再聊、断网了、手机没电关机了、突然有紧急事务要处理。他必然会有失落感，但心里会一直想着你，并且会很乐意再主动来找你。**“意犹未尽”**就是这个道理。日子还长，多增加他对你的时间和精力投资，他会越来越爱你。

除了以上六招，如果你想让男人对你有一个及时的反馈，就不可避免地会涉及到以下两个核心理论。

上一节我们介绍了具体的行为技巧，本节中我们来看看背后的心理学理论。掌握这些理论，能让你更游刃有余。

**理论一：激发他们的欲望。**

卡内基在《人性的弱点》中说过：**“人类每一个行为都源于他们心底根本的欲望。”** 因此，你要改变男人的态度，让他们及时回复信息，就要学着去激发男人的欲望。这包括两个方面：他展示自我的欲望，以及他想要了解你的欲望。

在和男人交谈过程中，不要将重心一直放在自己身上。当你谈论的都和男人无关，他自然就没欲望聊下去。多聊聊他熟悉的领域、他的社交圈子、他的工作和爱好。当聊到他感兴趣的话题时，他肯定会迫不及待地展示自己的能力和成就，这是男人的天性。

在这个过程中：
*   如果他谈到的是你不熟悉的内容，那就安静地听他说，时不时表达一下肯定和赞美。
*   如果他谈及的是你有所了解的，那就适当表达自己的观点。例如，他说喜欢旅游，刚去了杭州，你可以说：“杭州我觉得还行，我更喜欢苏州。” 他心里自然会想：你是不是也喜欢旅游？为什么喜欢苏州？等等。自然就会和你聊个不停。

兴趣是两性关系中最曼妙的催化剂，产物就是你们之间的**情绪共鸣**。当有了共鸣以后，他就会认为你们是一类人，自然会想摸清你的底。那么接下来，即使你不回复他，他也会没话找话地找你聊天。只有这样才能满足他想要了解你的欲望。

**理论二：掌握聊天的节奏。**

讲道理，为什么男人会不秒回你的信息呢？有可能是他在套路你，还有可能是你自己没能掌控你们聊天的节奏。

*   **情况一：他在套路你。**
    他们想故意用疏远的套路来打乱你的思考和节奏。但你说再多也是无谓的。应对方法就是将计就计，结束话题。回头发个朋友圈（去逛街、吃饭、散步等），告诉他们：老娘正在享受自己的人生，你撩不撩我都一样精彩。要知道，对付套路最好的方法就是**反套路**。当他们知道你不吃那一套之后，就会收敛很多。
*   **情况二：你未能掌控节奏。**
    应对方法有很多，最有效的就是利用他们的欲望，制造神秘感来让他们更有情绪投入。例如，当你们聊得很嗨的时候，马上找一个借口结束话题，会给男人一种不尽兴的感觉，吊足他们的胃口。在你们结束聊天之后，可以去给他朋友圈里最新的动态点个赞。注意，不是评论，也不是继续聊天，就是单纯的点个赞。

当你这样不按套路出牌之后，他心里一定会嘀咕：“刚刚才说有事，现在又给我点赞，是不是对我的表现不满意？” 那么在下一次和你聊天时，他们就会去试探你的态度，并注意自己的表现。这样一来，聊天的节奏就已经完全掌控在你的手上了。男人秒回信息，就是水到渠成的事情。

一般情况下，如果你能学会以上两个理论，让男人秒回信息对你来说已经不是一件难事。如果你觉得自己火候还不太够，别担心，这里还有三个实用的小技巧。

**以下是三个让男生秒回的小技巧：**

1.  **巧用“撤回”功能。**
    例如，你可以发一张他没见过的自拍，然后掐着时间撤回。如果他看见了肯定会追问，你别解释，就和他说“发错人了”就行。这时男人心里肯定会记挂你的自拍（没看清楚），又可能猜测你是不是在和别的男人聊天，因而吃醋，事后就会更加主动。如果他没看到你撤回了什么，半天之后跑来问你发了啥，那更好，你就故左右而言他，就是不告诉他发了什么。被这样“折腾”过的男人，积极性都不会太差。下次你找他的时候，他肯定会第一时间打开手机，生怕再次错过。

2.  **别忘了使用语音消息。**
    如果你想保持男人对你的热情，让他们时刻保持对你的渴望，那么一定要给到他合适的情绪反馈。单纯的文字冷冰冰，看不出情绪。用语音就行。例如，男人和你谈到他的成长经历时，若直接发一句“你很厉害哦”，不仅不能让他感受到你的认可，还可能让他产生误会。如果你发的是语音，他感受到的则是你真真切切的认可，给到他的就是正面反馈。那么在你们以后的互动中，他就会更加上心，以获得你的情绪价值。

3.  **欲擒故纵。**
    有时候遇到男人真有事而不能及时回你信息时，千万不要着急抱怨。如果你太在意男人是否及时回复信息，也可能会将你的需求感过多地暴露出来。**女人魅力最大的敌人恰恰就是需求感。** 所以，当他们因为忙而没有时间回你时，别太在意。有空的话就去逛逛街，吃些好吃的，发个美美的自拍，让自己的知性和魅力自然散发。另外，对于那些因为玩游戏或和朋友出去玩而不回复你的男人，建议“以毒攻毒”：约上几个朋友（一定要有男性）出去走走，然后把和男性朋友的合照发到朋友圈。不用多久，男人会主动找你的。

其实，无论是微信聊天还是其他方式，相较其他沟通方式，都有其独特的文化和技巧。情商高的女人不会成为聊天终结者，而是用话语展示自己的高情商，让男人看到你的消息就想秒回。

那么，高情商的女人在聊天中有什么要诀呢？

**以下是高情商聊天的三个要诀：**

**要诀一：选择合适的话题。**

不同的背景、文化、职业，说话聊天方式也不同。
*   和学生聊天，切记不要总是提到“考试考得怎么样”，你的问题可能一开始就引起对方反感，对方大多敷衍应对，“不怎么样”、“还好吧”成为他们终结聊天的法宝。
*   和单身人群聊天，他们最讨厌提到“有对象了吗？”“年龄也不小了，不要那么挑。”所有单身男女大多享受单身时光，最讨厌在微信中聊这类话题。
*   和已婚人士聊天，他们最讨厌提到“什么时候生孩子”。对于那种生育受限或不想生孩子的人来说，这样的问题会让他们内心很烦躁。
*   和另一半聊天时，不要总是追问“你在哪里？”“和谁在一起？”“什么时候回来？”。你的夺命连环Call只会把他越推越远。

**要诀二：注意用词。**

“呵呵”、“哦”、“在吗？”这样的词能不用就不用。
*   **“呵呵”**：如果给聊天终结词汇排个榜，“呵呵”在相当长的时间里占据榜首。当对方输入一段文字等待你回应时，你却“呵呵”一下，这样回答会让对方语塞，导致以后的交谈受阻，双方感情变淡。
*   **“哦”**：含义很多，在微信中大多让对方摸不清你的意思。本来对方打一大段文字，在体力上已经“输”给了对方，还让对方猜谜语似的猜不懂你的含义，两个人的交流很容易产生矛盾。所以以后说“哦”，还要把后面的句子带出来。
*   **“在吗？”**：通常在微信中，以“在吗？”开始的聊天，有八九成的可能是借钱或让你帮忙，这样的开头总是让人心惊胆战。其实有事情不妨直说比较好，这么小心翼翼、一副委屈的样子说“在吗？”，会搞得大家都很紧张，很有负担。

**要诀三：不要成为聊天终结者。**

我们应该怎么做呢？以下是几点建议：
1.  **多问开放性问题。** 不要局限于一点，多从常见的问题开始，这样聊天局限性会减少，双方的沟通会更到位。
2.  **寻找共同点。** 在聊天中尽可能寻找对方和你身边的共同点，也能很好地摆脱聊天中的尴尬。
3.  **照顾对方感受。** 聊天过程中，不要总是以“我”开头，一味宣扬自己的想法，总是打断别人，或反驳别人的观点。这些太自我的行为在聊天中是不受欢迎的。
4.  **善用辅助表达手段。** 人与人沟通中，除了语言和文字，表情包和一些辅助手段也很重要。在微信聊天中穿插一些搞笑的表情包能起到很好的润滑作用。除此之外，在聊天陷入僵局时，适当发几个小红包也能很好地调节气氛。

很多人说不喜欢情商高的人，说他们虚伪。可是真正的情商高，不是虚伪，而是**你的自信和让别人感到舒服的能力**。

越爱就会越在意，所以想让你爱的男人秒回你的消息无可厚非。但是一定要注意方法和技巧。
