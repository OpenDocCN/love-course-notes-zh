# 恋爱知识星球：10、护肤攻略：丁香妈妈护肤攻略：2.2保湿：挑选面膜的方法：2.2-2

在本节课中，我们将学习关于皮肤保湿的核心原理，并了解如何根据自身肤质选择合适的保湿产品。我们将澄清一些常见的护肤误区，并掌握正确的产品使用顺序。

上一节我们介绍了保湿的基本概念，本节中我们来看看如何挑选和使用保湿产品，特别是针对不同肤质的注意事项。

皮肤里的水分并不是从外面来的，而是靠饮食的吸收利用。没有任何产品能直接补水，不管是保湿水、乳液霜膏还是面膜，起到的都是**保湿**作用，也就是尽可能保住皮肤水分。

🎼 第二点，化妆水、爽肤水等水类保湿产品不会增加皮肤的含水量。也就是说，在保湿时，水类产品可有可无。你只需根据个人习惯来决定要不要用保湿水。

第三点，市面上常见的保湿产品可以分为三大类。以下是这三类产品的简要说明：

*   **乳液类**：质地轻薄，含水量较高。
*   **膏类**：质地居中，滋润度较好。
*   **霜类**：质地厚重，油性成分多，封闭性强。

不同肤质适合的保湿产品不一样。以下是针对不同肤质的建议：

*   如果你是**干性皮肤**，建议你选择油性重、保湿力强的面霜。
*   如果你是**油性或混合性皮肤**，建议你选择质地轻薄水润的乳液或精华。
*   如果你的皮肤**很敏感**，建议你尽可能选择成分简单的保湿产品。

第四点，保湿产品的使用顺序是**水 → 精华 → 乳液/霜**。这个顺序有助于产品更好地被皮肤吸收。

最后，有一个重要的特别提醒：如果你是痘痘皮肤，最好不要用贴片式面膜，不管是补水面膜还是其他功效性的面膜，否则有可能加重痘痘。

最后我来总结一下这节课的内容。本节课一共讲了4个要点。

1.  皮肤水分源自体内，护肤品起**保湿**作用。
2.  化妆水等水类产品不增加含水量，可按习惯使用。
3.  按肤质选择产品：干皮用**霜**，油/混皮用**乳液/精华**，敏感肌选成分简单的产品。
4.  产品使用顺序为：**水 → 精华 → 乳液/霜**。
