## 认识面部黄金比例
上一节我们介绍了不同肌肤适合的底妆产品。本节中，我们来看看打造立体底妆的核心理论基础——面部的黄金比例。

完美的面部轮廓遵循“三庭五眼”和“四高三低”的法则。

### 三庭五眼
以下是“三庭五眼”的具体定义：
*   **三庭**：指脸的长度比例。从**发际线**到**眉头**的距离，应等于从**眉头**到**鼻尖**的距离，也应等于从**鼻尖**到**下巴**的距离。这三个部分长度相等即为完美比例。
*   **五眼**：指脸的宽度比例。以一只眼睛的长度为单位，从一侧**发际线外侧**到**眼尾**的距离、**眼长**、**两眼间距**、另一只**眼长**，以及从另一侧**眼尾**到**发际线外侧**的距离，这五个部分应大致相等。

**公式**：完美比例 = 三庭相等 && 五眼相等

### 四高三低
除了“三庭五眼”，面部立体感还取决于“四高三低”。
以下是“四高”的位置：
1.  **额部最高点**
2.  **鼻尖**
3.  **唇珠**
4.  **下巴尖**

以下是“三低”的位置：
1.  **人中凹陷**（唇珠上方）
2.  **下唇下方凹陷**
3.  **鼻翼外侧凹陷**（鼻翼与脸颊交界处）

**公式**：立体感 = 四高突出 + 三低凹陷

掌握这些法则后，你可以自行测量面部比例，明确需要修饰的区域。例如，上庭偏长可在发际线涂暗影收缩，下庭偏短可在下巴涂高光提亮。

## 3D立体底妆实战步骤
理论是基础，实践是关键。接下来，我们将一步步演示如何为一位混合性肌肤的模特打造3D立体底妆。

### 第一步：妆前准备与控油
模特属于T区油、其他部位干的混合性肌肤。第一步是为皮肤打好基础。

我们首先使用**隔离乳**，以轻拍方式均匀涂抹全脸，为肌肤提供滋润，使皮肤呈现自然光泽。

**核心技巧**：水润不等于油腻。对于偏油肌肤，在粉底前有一个关键步骤——使用**散粉刷**蘸取**定妆散粉**，轻柔地扫过T区等易出油部位。这个动作能吸附多余油分，防止后续底妆因油脂融合而变得过度油亮。

```text
# 妆前步骤代码化
if 皮肤类型 == “混合性/油性”:
    涂抹(隔离乳)
    使用(散粉刷)轻扫(定妆散粉，区域=T区)
```

### 第二步：塑造面部轮廓（暗影处理）
底妆的立体感始于明暗对比。我们先处理需要“收进去”的暗影部分。

使用暗影产品，在以下位置进行修饰：
1.  **鼻翼外侧**及**眼窝**：制造鼻部立体感与眼窝深邃感。
2.  **发际线外侧**：如果上庭偏长，从此处向内轻扫，视觉上缩短额头。
3.  **颧骨下侧至下颌线**：从颧骨下方的凹陷处开始，由外向内、向下轻扫，一直连接到下颌线，塑造清晰的面部轮廓线。

### 第三步：突出视觉焦点（高光处理）
让该饱满的地方“凸出来”，是打造立体感的另一面。

在以下“四高”区域使用高光产品：
1.  **T区**（额头、鼻梁）
2.  **眼睛下方的三角区**（泪沟、苹果肌上方）
3.  **唇珠**
4.  **下巴尖**

这些区域适用于大多数脸型，能有效提升面部饱满度。

### 第四步：均匀肤色与融合底妆
现在，我们将肤色统一并融合刚才的明暗修饰。

选择适合肤色的**粉底液**，用**美妆蛋**以轻拍的方式，将粉底均匀覆盖全脸。美妆蛋能快速实现均匀、贴合的妆效。
*   **额头**：由中心向发际线轻拍。
*   **脸颊**：由内向外轻拍。
*   **下颌骨**：特别要注意，应由外向内轻拍，使底妆与脖子肤色自然过渡，避免色差断层。

完成半边脸后，对比效果：未上妆侧扁平无光，已上妆侧则自然、均匀且富有立体感。

### 第五步：加强立体感与定妆
为使立体感更自然持久，我们使用一个进阶技巧。

选用一个**大号粉扑**，蘸取少量**暗影粉**，在手背揉匀后，以**轻轻按压**的方式，叠加在之前画过暗影的区域（如颧骨下方、下颌线）。这种方式上妆，粉质与皮肤融合度更高，妆效更自然，也更持久。

最后，用同样手法完成另一侧脸部的上妆。
