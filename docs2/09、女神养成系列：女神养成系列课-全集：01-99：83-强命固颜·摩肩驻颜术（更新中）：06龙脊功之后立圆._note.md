# 女神养成系列：06：龙脊功之后立圆

在本节课中，我们将学习龙脊功的第二个动作——后立圆。这个动作旨在通过特定的轨迹活动脊椎，帮助强健体魄、改善体态。

上一节我们介绍了龙脊功的第一个动作，本节中我们来看看如何完成后立圆动作。

## 预备姿势
预备姿势与之前相同。两脚开立，与肩同宽，身体保持正直、放松。

## 动作分解
以下是后立圆动作的详细步骤。

1.  右手掌心贴在肚脐位置。
2.  左手抬起，掌心朝前，在空中固定位置，作为动作的参照物。
3.  动作的关键在于，**不是手在主动画圈，而是身体（主要是脊椎）带动贴在腹部的右手进行运动**。
4.  身体带动右手，向后、向上、再向前、向下，画一个垂直的立圆。
    *   **轨迹公式**：`后 -> 上 -> 前 -> 下`
5.  为了更清晰地展示侧面动作轨迹，可以观看侧面演示。身体带动右手完成“向前、向上、向后、向下”的循环。
6.  此动作通常以**4个8拍**为一组进行练习。为节省时间，此处仅作演示。

## 练习要点与辅助方法
如果练习时感觉腰、胯、膝盖的动作幅度过大，可以借助工具来辅助。

*   **辅助练习法**：坐在一个稳定的凳子上进行练习。凳子可以限制臀部的辅助运动，从而迫使练习者更专注于用脊椎带动动作，达到直接锻炼脊椎的目的。

本节课中，我们一起学习了龙脊功的第二个动作“后立圆”。我们掌握了以身体带动手臂画立圆的要领，并了解了通过坐姿练习来精准锻炼脊椎的辅助方法。坚持练习，有助于增强脊柱灵活性与核心力量。
