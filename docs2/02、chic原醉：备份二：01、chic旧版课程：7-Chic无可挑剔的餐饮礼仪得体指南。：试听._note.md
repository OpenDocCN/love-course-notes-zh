# 餐饮礼仪得体指南：02：优雅享用日式料理

在本节课中，我们将要学习如何优雅地享用日式料理，避免常见的失礼行为。我们将从理解餐饮礼仪的本质开始，逐步深入到具体的日料用餐规则，特别是寿司的正确食用方法。

## 理解餐饮礼仪的本质

在正式开讲之前，我们首先要理解在日常饮食中礼仪的本质是什么。它绝不是做作假客套的代名词，而是能让餐厅里所有人都觉得舒服的言谈举止。厨师会觉得你值得被更好的对待，你也能真正地品尝出食物的美味。

对此，有人认为，礼仪无非就是要保持安静，动作轻慢。的确如此，但这些只是基本原则，仅仅让人看着不粗鲁而已。要给人留下有家教、得体、优雅的印象，期间还有一大段距离。

就像我们今天要说的日式料理，很多人都吃过，可对其具体的饮食礼仪却少有了解。

## 常见的日料礼仪错误

上一节我们介绍了礼仪的本质，本节中我们来看看在日料店用餐时，大多数人常犯且容易忽视的礼仪错误。以下是四个典型的错误示例：

1.  **错误混合蘸料**：吃寿司时把酱油和芥末放在同一个碟子里，自行搅拌后，用底部的寿司米蘸取混合物食用。这犯了两个错误：混合蘸料会破坏寿司原本的味道层次；用寿司米蘸取，会使它失去该有的温度，甚至散开解体。
    *   **正确做法**：将酱油单独倒在一个碟子里。用寿司的上层食材部分蘸取酱油。芥末可加可不加，因为寿司师傅通常已在制作时放入适量。若需额外添加，也应抹于食材之上。高级寿司一般无需额外蘸取。

2.  **不当用餐姿势**：因日料店的吧台或桌子通常较低，很多人习惯弓着背，几乎把脸趴在桌面上吃东西。这被称作“猫背犬食”，意指只有猫狗才会这样不得体地进食。
    *   **正确做法**：吃饭时应直起后背，把碗碟端起来，慢慢享用。但吃拉面时不必端碗，应用筷子夹起拉面放入勺中，再将勺子送到嘴边食用。

3.  **用筷子扎取食物**：遇到很难夹起的圆滑食物时，许多人下意识会用筷子扎着吃。这会显得很粗鲁。
    *   **正确做法**：坚持用筷子夹取。如果实在不行，使用勺子是可以接受的。

4.  **喷洒香水用餐**：喷香水去日料店用餐是绝对禁忌。它会阻碍你品尝食物最原始的味道，同时也会干扰料理师傅和其他客人。
    *   **正确做法**：身上没有气味就是最好的。

以上这些用餐礼仪在国内的日料店应留心遵守，在日本则必须特别注意。因为日本人通常不把喜怒挂在脸上，即使你行为失当，他们也可能不会当面指出，只会在心里给出差评。

## 寿司的专用用餐礼仪

了解了普遍的错误后，本节我们聚焦于日式料理中最常见的寿司，看看有哪些重要的专用礼仪。以下是需要注意的六个要点：

1.  **尊重寿司的完整性**：对待面前摆好的寿司，万万不要掀开上层的食材。这个受好奇心驱使的行为对很多寿司师傅来说是极大的不尊重。

2.  **选择正确的取用工具**：一般来说，坐在吧台位时，用手拿起寿司食用。而在4人或6人的餐桌，建议用筷子比较妥当。

3.  **一口吃完**：寿司的正确吃法是一口解决。寿司师傅在制作时已将其大小控制在一口吃完的标准，并精心平衡食材与米的比例。不要咬一口再放回去，这会影响口感，是一种不讲究的做法。

4.  **正确理解姜的作用**：寿司店提供的姜片不是小配菜，而是用来重置味觉的。应在吃完一个寿司后，吃一片姜，再吃下一个，以防止不同寿司的味道互相干扰。把姜当菜吃是会闹笑话的。

5.  **回转寿司的取用原则**：在回转寿司店用餐时，应该吃一盘拿一盘。这样既能保持寿司的温度，也不会给人留下浪费食物的印象。

6.  **区分不同寿司的蘸酱方法**：军舰卷（用海苔裹成椭圆，上放鲑鱼籽等食材的寿司）与普通握寿司的蘸酱油方法不同。
    *   **正确做法**：军舰卷应用海苔的一侧蘸取酱油。这里有一个小技巧：如果店里有提供姜片，可以用姜片蘸上酱油，再涂抹在军舰卷上，这会显得你很懂行。
