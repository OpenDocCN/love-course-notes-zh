# 女神养成系列：07：直角衣架肩：6个练肩动作打造立体感

## 概述
在本节课中，我们将学习如何通过六个简单的瑜伽动作，针对性地塑造肩部线条，打造流行的“直角肩”或“衣架肩”。课程将结合柔韧性与力量训练，帮助你打开胸腔、增强上背部力量，从而改善体态，让肩部轮廓更显立体。

## 直角肩的必备要素
上一节我们介绍了全身整合练习的重要性，本节中我们来看看塑造直角肩需要具备的两个关键条件。

以下是塑造直角肩的两个核心要素：
1.  **平坦的肩膀与正确的肩胛骨位置**：这需要你的上背部中间具备足够的力量。
2.  **笔直的手臂线条**：圆肩、含胸驼背等不良体态会影响手臂线条。若存在这些问题，建议回顾第一周的拉伸与第二周的加强练习。

今天，我们将针对这些要素进行强化练习。

## 动作详解
如果你已准备好，我们来看看具体的动作怎么做。以下是六个核心练习动作及其细节。

### 动作一：弓式
弓式能有效打开胸腔，调整肩颈，是打造直角肩的基础动作。

**标准做法：**
1.  俯卧在垫子上。
2.  依次弯曲左右腿。
3.  吸气，抬头挺胸，目光微向前看。
4.  呼气，左右手依次向后抓住同侧脚踝。
5.  再次吸气，感觉小腿微微向上提；呼气时，手脚对抗，向后伸展，像一张弓一样保持姿势。
6.  在整个过程中，需在呼气时收紧腰腹，肚脐内收上提，以保护腰椎。

**动作要领：**
*   保持姿势时，感受胸腔的打开与锁骨向两侧伸展。
*   吸气，呼气，然后舒适地落回地面，双手交叠，前额置于手背上放松。

**过渡与变体：**
若无法完成标准弓式，可通过以下方式逐步练习：

**单侧弓式：**
1.  俯卧，双手向前伸直。
2.  吸气，微微抬头挺胸。
3.  弯曲一侧腿，同侧手向后抓住脚踝。
4.  吸气呼气，进行单侧练习。可选择将对侧手脚压向地板以保持稳定，或让双手双脚均悬空。
5.  整个过程中，保持骨盆中立，髂前上棘始终接触地面，呼气时收紧腰腹。

**弓式简易式（针对腰部不适）：**
1.  俯卧，双腿弯曲，双手向后抓住脚踝。
2.  开始时，前额贴地。
3.  吸气，呼气，仅将小腿和膝盖抬离地面。
4.  此方式有助于培养臀部发力感，并伸展大腿前侧。
5.  也可尝试吸气稳定，呼气时膝盖落回地板；再次吸气稳定后，呼气缓慢抬起上半身。
6.  保持膝盖压向地面的同时，重点加强胸椎伸展与胸腔打开。

完成以上两个方向的练习后，再尝试标准弓式会更容易。

### 动作二：蛙式
蛙式有助于强化上背部、臀部与大腿后侧，同时确保腰部无压力。

**做法：**
1.  俯卧，双手放于胸部两侧。
2.  吸气，微微抬头挺胸，确保肋骨舒适地贴近地面。
3.  呼气，弯曲双膝，勾脚尖，脚后跟朝向天空。
4.  再次吸气；呼气时，收紧腰腹，肚脐上提，同时膝盖缓慢抬离地面，胸口上抬，背部发力，挺胸，目视前方。
5.  动作重点不在于抬升幅度，而在于腰部无压力，且能感受到上背、臀部及大腿后侧的发力感。
6.  可尝试微微降低膝盖高度，同时更深地内收肚脐，感受后背向内夹紧、胸口向前推出的感觉。
7.  吸气稳定，呼气舒适地落回地板。

### 动作三：鸽子式（仰卧变体）
此动作用于放松深层臀部肌群，缓解之前练习的紧张。

**做法：**
1.  仰卧，双腿弯曲，脚踩地面。
2.  将一侧脚踝置于对侧大腿上。
3.  吸气稳定；呼气，双手穿过下方腿的膝窝，抱住大腿。
4.  上方腿的膝盖缓慢向旁侧推开，以伸展臀部深层肌群。
5.  保持3到5个呼吸后，换另一侧重复。
6.  想象上方腿的膝盖向外轻轻推开，找到适合自己的幅度。

### 动作四：螃蟹式
螃蟹式能充分伸展身体前侧链，打开胸腔，收紧臀部与背部。

**做法：**
1.  从仰卧姿势，双手抱小腿，向前滚动坐起。
2.  坐姿，双腿自然分开与臀同宽，双手在身体后侧支撑，指尖朝向侧方或后方（根据肩部柔韧性决定）。
3.  吸气稳定；呼气，臀部推高。
4.  一侧手臂经面前划过头顶，指向远方。
5.  可再次缓慢推高胸腔和臀部，让手臂超越头顶，进一步打开胸腔，启动背部力量。
6.  吸气还原，呼气换另一侧。
7.  动作幅度根据自身柔韧性调整，注意保持骨盆稳定并推向天空，臀部收紧发力，腰腹保持收紧，后背收紧以推动胸口向上。

重复此动作3到5组后，简易盘坐，吸气延展脊柱，呼气前弯放松。

### 动作五：反台式
反台式能巩固加强身体后侧链，提升后背力量，使胸腔更开阔。

**做法：**
1.  坐姿，双手于体后支撑，指尖朝前或朝侧，双腿踩稳地面。
2.  吸气，呼气时臀部发力向上推高，直至膝盖、臀部、肩膀呈一条直线。
3.  稳定后，再次吸气；呼气时，感受后背中间夹紧，推动胸口进一步向上，整个腹部保持平稳收紧。
4.  眼睛可看向膝盖或天空，根据自身能力决定。
5.  保持3到5个呼吸后，吸气再次上抬臀部，呼气落回地板。
6.  转为盘坐，吸气延展背部，呼气前弯放松。

通过螃蟹式伸展前链、稍激活后链后，再用反台式进行巩固加强，能有效改善体态，让“衣架肩”离你更近。

### 放松与休息
1.  仰卧放松，自然伸直双腿。
2.  双臂放松置于身体两侧，略微空出腋窝。
3.  调整头和脊柱至正中位置。
4.  闭合双眼，进行深长的呼吸，缓和今日的所有练习。

## 总结
本节课我们一起学习了六个针对肩部塑形的瑜伽动作。通过弓式、蛙式、鸽子式、螃蟹式和反台式的序列练习，我们重点打开了胸腔、增强了上背部力量，并放松了相关肌群。坚持每周练习3到5次，你将感受到肩颈更加舒展舒适，背部更有力量，肩部线条也会逐渐显现出清晰的立体感。现在，请跟上跟练视频开始练习吧。
