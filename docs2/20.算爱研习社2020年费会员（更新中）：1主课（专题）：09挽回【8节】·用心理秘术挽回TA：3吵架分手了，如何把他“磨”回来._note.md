# 算爱研习社2020年费会员（更新中）：1主课（专题）：09挽回【8节】·用心理秘术挽回TA：3吵架分手了，如何把他“磨”回来

在本节课中，我们将要学习如何应对因吵架而导致的分手，并探讨如何科学地挽回对方。我们会分析吵架分手的本质、不同类型吵架的成因，并给出具体的复合策略与节奏。

上一堂课我们探讨了男人出轨后的复合方法。本节中我们来看看另一种常见情况：因吵架而分手，应如何挽回。

超过三分之一的分手源于吵架。而在吵架分手后，超过50%的人会产生复合的念头。这可能是最容易让人后悔的一种分手状态。但吵架分手后，不要急于复合。你需要先弄清楚，这次分手是否真的仅仅因为吵架。

在许多案例中，吵架只是表象，本质另有原因。在采取复合行动前，你必须先明确：你们是因吵架而分手，还是因想分手而吵架。

我之前讲过人的心理防御机制，其中很重要的一条是“转嫁投射”。男人在想分手时尤其喜欢使用这招。例如，一个男人出轨并爱上了别人，非常想分手。他通常不会直接承认，因为承认劈腿会产生极大的羞愧感。

此时，心理防御机制就会启动。它像一个无微不至的母亲，试图避免儿子丢脸。于是，它启动了“转嫁投射”。具体做法是：男人开始与女友冷战，处处看不惯女友，对女友做的任何事情都抱怨和挑错。

这种情况下，女友必然会吵架。吵过几次后分手，男人便可以说：“看吧，是你作，是你跟我吵，我只能分手了。”言下之意是：错都在你，他就不用愧疚了。

超过30%的吵架分手属于这种情况：他劈腿了、他父母反对了、他厌倦你了，但他会通过找茬吵架，把分手的过错转嫁到你头上，逼你提分手。这不是“因吵架而分手”，而是“因想分手而吵架”。

因此，当你决定复合时，请先检查这次分手属于哪种性质。以下是三个检查指标。

以下是判断分手本质的三个关键指标：

1.  **判断对方是否有出轨可能性**：男人出轨时，常使用冷暴力让女人受不了而主动提分手。你可以参考本社课程《如何第一时间知道他是否出轨》进行判断。等你回过神来，他可能早已另结新欢。这种分手并非吵架分手，而是“甩锅分手”。
2.  **观察他是否在无理取闹地求吵架**：如果对方本就打算分手，他会通过吵架逼你离开。当你表现很好时，他就会开始无理取闹、主动挑起争端。例如，挑剔你做的饭菜，无论你怎么调整他都不满意。记住，当爱人开始对你横挑鼻子竖挑眼时，往往意味着爱已消逝。因为爱会让你全身是优点，而不爱时则只会看到缺点。
3.  **观察他在谈及分手时是否保持理性**：吵架分手是冲动型分手，意味着情绪失控。如果双方情绪都失控，才可能突然分手。但如果吵架时你情绪失控，他却保持理性，且没有哄你而是顺势答应分手，这就很明显是处心积虑、预谋已久，就等着你在狂怒下提分手。

请回想分手时，是否出现以上指标。如果确有出现，就需要思考男人预谋分手的真正原因，不要简单归咎于吵架。

假定以上三个指标都未出现，你们的确是因为吵架而分手。在解决问题前，我们需要先了解吵架的原理，因为不同原因的吵架会导致不同的结果，其复合策略也不同。

以下是四种常见的吵架原因及其特点：

1.  **沟通失控**：两个人在沟通时，容易发生情绪堆积。每一句带负面情绪的对话都会增加一层负能量，如同温水煮青蛙，最终情绪彻底爆发。吵架结束后，负面情绪宣泄掉，理智回归，往往会后悔。其心理成因是“情绪堆积”。这是最需要复合的分手模式，就怕双方都不懂如何宣泄负能量，无法冷静下来。
2.  **磨合期宣泄**：两个独立的个体建立亲密关系，如同两个充满气的皮球相互碰撞，必然产生摩擦和矛盾。吵架成为宣泄火气的渠道。这种吵架的本质是交换负能量，关键在于不要刻意避免冲突或单方面隐忍，要做到对事不对人，并且不翻旧账、不戳底线。由此导致的分手常是冲动型分手。
3.  **单向情绪溢出**：这是一方长期忍受，另一方不断释放负面情绪，导致承受方负能量积累到极限后突然爆发的情况。例如，女方太“作”，男方一直忍耐直至崩溃。这种分手破坏了亲密关系，往往需要更多手段才能复合。
4.  **价值观不同**：心理学上称为“自我价值保护逆反”。双方因理念差异而争吵，本质是对彼此的不认同，并会攻击对方的自我价值。例如，关于家庭角色分工的争论。这种争吵如同互相“喷黑漆”，时间长了会相互鄙视。由此导致的分手最难复合，也不建议复合，因为根本矛盾难以解决。

说完了吵架的原因，我们现在讲讲复合。很多姑娘在复合时容易犯一个错误：太着急，不懂节奏。前一刻还在冲动分手，后一刻就要哭着复合，结果气没消，嘴上又吵起来，这成了“以复合的名义继续吵架”。

你要明白，吵架是一时的情绪波动和宣泄，所说的话是非理智的。当这波情绪还未完全宣泄时，双方仍处于吵架余波中，此时去复合，任何一句重话都可能再次引发争吵，极大增加复合难度。

复合的节奏至关重要。为此，我专门提出“一期规则”：吵架后想要进行第一次有效沟通，最快必须在**一天之后**，最慢也必须在**七天之内**。

因为人类情绪的堆积和宣泄都有其时间窗口。吵架后第一个小时，情绪可能更加愤怒。一小时后，若无新导火索，情绪会慢慢平缓，但负能量并未消散。三小时后，情绪开始逐步递减。此时心理防御机制仍在工作，你只会觉得对方不好。

直至**24小时之后**，或经过睡眠，心理防御机制逐渐关闭，人才开始审视自己。所以，当男人说“要冷静几天”时，他可能并非真想分手，而是需要时间宣泄情绪。如果你此时逼他立刻表态，相当于在求他说分手。

那为什么冷静期不能超过**七天**呢？因为一个人的习惯养成期大约是七天。恋爱中的亲密模式是一种习惯，只要习惯还在，他就没有真正离开你。但七天之后，他会逐渐习惯没有你的生活，那时冷静期就可能变成真分手了。

因此，**一天之后，七天之内**，是你们复合的最佳节奏。当然，如果你已错过这个时机，后面的技巧仍有用，只是成功率会略有降低。

在七天之内应该如何复合呢？首先，切忌两种极端做法：一是声泪俱下地恳求，二是完全高冷无视。很多时候，你越用力，失去的越多。

复合这件事，千万不要一下子出大招，要分阶段进行。

**第一阶段：给出台阶，引导对方来哄你**

很多吵架分手只是一时之气，并无原则问题。但如果姑娘太高冷，反而会把男人推远。此时，你只需巧妙地给出一个台阶，就能四两拨千斤地让男人回来认错。

例如，情侣吵架后陷入僵局，姑娘可以打电话给男人，用平淡的语气问一件具体的小事，比如：“上次你说很好吃的面馆叫什么名字？” 这只是在询问信息，并非求复合，所以不丢脸。男人回答后，姑娘可以冰冷地说：“哦，那你吃不吃？” 如果男人答应，两人就有了见面的机会。

记住，见面时姑娘仍要装作生气。这并非真生气，而是表明你尚未原谅他，这次见面只是给他的一个台阶。男人看到姑娘不开心，自然会顺水推舟地活跃气氛、哄哄姑娘。整个案例中，最关键的是姑娘问出的“那你吃不吃？”，这就是“台阶”。

台阶是可上可下的。男人顺着台阶过来哄你，便是“上”，你们可顺利复合。如果男人不接台阶，说明他确有分手之意，你们便进入了“假性分手”阶段。

有姑娘会问：吵架本来就应该男人来哄，为什么我要先给台阶？这里涉及一个心理学原理：**负向关系循环**。

当你们吵架分手后，男朋友害怕自己的复合行为会遭到你的攻击和指责，所以他反而会选择“冷漠远离”。这不是因为不爱你，而是害怕被你攻击。**你给人台阶下，对方才知道你允许他靠近**。给台阶是告诉他：现在可以来哄我了。

给台阶的技巧很简单：如果还住一起，就用平淡语气问他一点事，目的是引起他注意。如果不住一起，就简单地创造一个他能看见你的机会。你的态度要把握住：还在生气，但已不激动。

**第二阶段：假性分手下的中介物吸引**

如果男人不接你的台阶，你们便正式进入“假性分手”阶段。此时你们已分手，但他尚未完全切断与你的亲密模式。请注意，你们已处于真分手的边缘，千万不要再推一把。**控制情绪**是关键。

在假性分手状态下，男人趋向于分手，而非趋向于你，所以他会逃避你。如果长时间无法见面，假性分手很容易变成真性分手。此时最好的方法是通过“中介物”来吸引对方。

这就是为什么我常建议情侣共同养宠物。此时可以“晒猫晒狗”，并配文：“爸爸你去哪了？宝宝很想你。” 如果是离婚夫妻，有孩子则更好，让孩子卖萌。例如，妻子可以发孩子吃饭不开心的照片，说：“怎么都调不对（爸爸做的饭）的味道。” 丈夫看到孩子这样，很容易心软回家。

大多数不算特别渣的男人都有责任感，在共同拥有的中介物面前，很难完全硬起心肠。但请注意，这招的目的**并非直接让男人低头认错**，而是**创造见面机会，给出二次台阶**。

见面之后，给他一个“二次印象”至关重要。心理学上有“近因效应”，即人们更容易记住最近一次见面的印象。你们最后一次见面是在吵架，所以他脑海里你一直是凶神恶煞的样子。

这次见面，你要塑造的二次印象是：你已经洗心革面、有所不同了。多说说自己最近看了很多书、电影，吸收了新知识，想通了很多事情。总之，要表现出你脱胎换骨了，之前吵架的问题已不复存在，你甚至变得更好了。

如果二次印象成功，男人分手的理由就不存在了。再加上你们是假性分手，亲密模式未完全切断，复合的可能性就大大增加。以此为由多见几次面，再提出复合，巩固美好形象，最后一举成功。

如果没有宠物或孩子，你们最爱的餐厅、想看的电影、一起买过的东西，都可以作为中介物。

**第三阶段：真性分手下的长信技巧**

如果男人连中介物都不管，打定主意不见你，那就进入了“真性分手”阶段。对方已决心分手，并开始切断亲密关系，甚至开始新的恋情。此时他对你抱有极大的负面情绪，你做任何事都可能让他厌烦。

这时，我们只能尽量使用“爱情三角”原理来尽人事地挽回。爱情由**激情、亲密、承诺**三个因素组成。对方因对你不满，已对你失去这些情愫。你唯一的优势是你们曾拥有过这样的爱情。

此时可以使用“**长信技巧**”来唤醒他对过往的回忆。此技巧**只能使用一次**，且信必须**非常长**。

切忌用微信或短信分段发送。因为唤醒回忆的初次情绪波动只有一次。分段发送会让他的感动迅速淡化，甚至觉得你在套路他。邮件或手写信则属于非互动式私密阅读，他能一次性沉浸其中。

长信可以承载更多的回忆和情感。写5000字、1万字都可以，要写到你自己痛哭流涕为止。把你们过往所有的温暖、美好和承诺都写进去。

男人看到这封长信，会产生巨大的情绪波动，即“**带入感**”。他会把自己带入到信中描述的那个过去深爱着你的男人角色中，那些温暖的点点滴滴会在这一刻复苏。

只要他给你任何回应，无论是回信、回电，哪怕是拒绝，都意味着他恢复了与你的接触。这就是机会。随后，你可以尝试约他见面，给他二次印象。如果能约出来，恭喜你，你们就从真性分手退行到了假性分手，可以继续努力复合。

如果以上所有攻略都未能让你挽回这个男人，你也可以尝试我们上一课讲的心理控制术方法。但请注意，挽回任何男人都只应尝试两次，不要给彼此更多无谓的机会。

有些人会来，有些人会走，找的理由各不相同，但原因都一样。来是因为爱，走是因为不爱。爱你时能忍受你的刁蛮任性，不爱你时，连一句话说错都忍不了。世上最温暖的是爱，最凉薄的是不爱。

3.  **尝试长信技巧**：如果已进入真性分手，尝试写一封真挚的长信。
