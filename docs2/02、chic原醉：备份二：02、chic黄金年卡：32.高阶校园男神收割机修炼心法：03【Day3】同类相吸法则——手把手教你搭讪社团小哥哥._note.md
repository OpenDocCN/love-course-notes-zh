# 校园搭讪全场景攻略：02：高阶校园男神收割机修炼心法：03【Day3】同类相吸法则——手把手教你搭讪社团小哥哥

在本节课中，我们将要学习如何在大学社团里，与那些充满潜力的男生进行有效搭讪。社团是结识优秀同龄人的绝佳场所，掌握正确的方法，能让你更自然地开启一段美好的关系。

上一节我们介绍了在图书馆中如何向学霸小哥哥搭讪。本节中，我们来看看如何在社团活动中，与未来的潜能选手擦出爱情的火花。

## 🎯 社团小哥哥：值得关注的潜力股

说到参加校园社团的男生，他们通常充满激情，拥有阳光灿烂的脸庞。他们是入校第一天接待新生的师兄学长，也是在校园内外各种比赛中崭露头角的先锋人物，更是在图书馆中默默无闻整理书柜的志愿者。相比于下课后赖在宿舍打游戏的男生，他们往往更有计划地学习和生活。

麦克斯公司多年研究表明，在大学里参加过社团的学生，通常拥有更高的个人素养，未来的就业前景也更广阔。这是因为大学社团工作让他们提前预备了一定的工作经验和处事能力，培养了自己的责任感和使命感。就业时，他们的适应能力就比没有参加过社团的学生要更强，在职场中晋升也会更快。因此，社团小哥哥极有可能是具备管理潜能的人才，也可能是值得依靠、并和你一起并肩前进的绝佳伴侣。

## 🗺️ 三大场景与对应策略

下面我们将社团里见到心动小哥哥的场景分为如下三类，分别是：第一次例会上、在社团联欢会上、在社团工作的对接中。按照场景的不同，我们会给出相应的搭讪策略。

### 场景一：第一次例会——运用吸引力法则

首先是在社团的第一次例会上，见到让我们心动的男生时，可以利用吸引力法则来和对方产生共鸣。

在心理学上，吸引力法则是指思想集中在某一领域的时候，跟这个领域相关的人、事、物都会被他吸引而来。俗话说“物以类聚，人以群分”，说的也是这个道理。人总是倾向和自己的同类待在一起，这样就能获得认同感。比如遇见心动男生时，共同的社团爱好就会命运般地让我们聚到一起。这就是你们因为本身的兴趣产生的同类相吸规律。

第一次在例会上见面时，可能你们就已经有了一定的亲近感。想要寒暄认识一下，其实很简单。

以下是开启对话的步骤：

1.  **开启话题**：你可以尝试对他说：“你为什么要报名舞蹈社呀？”并等待他的回应。
2.  **延展故事**：等他回应之后，你就可以延展对方的故事，让他得到倾诉的快感。例如对方说道：“我练舞蹈好多年了呢，想在大学继续提升舞蹈技能。”
3.  **产生共鸣**：面对一个对方感兴趣的话题，不要戛然而止地开启新话题。你要同时表示你对舞蹈的热爱或者曾经为练舞受过的苦，和对方产生共鸣。你可以先表示崇拜：“哇，那你跳舞一定很厉害啊！”然后说：“你会不会觉得练舞很辛苦啊？有一次我为了练某个动作，膝盖都磕破皮了。”
4.  **核心要点**：重点是间接表示，你们是同类人，有某种类似的经历。

有一点要强调的是，搭讪时一定不要暴露自己的目的。比如直接说出“小哥哥可以加个微信吗？”或者“同学我觉得你好帅啊，可以认识一下吗？”这样的话。即使有些男生不会拒绝这样的直白要求，你也不可避免地将自己的需求感表露无遗。如果以后感情得以顺利开展，你也可能长期处于被动状态之中。

所以你在搭讪时要尽可能地带入自己当前的角色，即社团成员之一。在交流时要以与社团相关的话题开场，不让人产生突兀的感觉。他也会因为共同的相似点对你产生认同感和好感。

### 场景二：社团联欢会——运用即时邂逅法

除了在第一次社团例会中抓住短暂的交流时间，我们还可以在社团联欢会上命运般地邂逅意中人。

在校园里，我们更喜欢将社团联欢会视为社团联谊会，因为大家对这样的集体活动的本质心知肚明，那就是交际至上。很多原本没有机会认识的人，都可以通过这次团建产生交集。小到春游，大到比赛和校庆联欢会，这些都是社交的好时刻，大家一定要把握好机会。

在社团联欢会上搭讪时，就要运用即时邂逅法来制造认识的机会。比如，假设心动的男孩子是参加比赛的选手。首先你要记下他的姓名和班级，在比赛散场的时候，及时上前搭讪，让对方感觉受到了你的关注。这样他会对你更容易产生好感，因为人在开心的时候，很难拒绝喜欢自己的人。

这里所说的即时邂逅的意思是，在对方情绪还处在兴奋中时制造见面机会，你的搭讪不仅不会让对方反感，还会促进你们之间更深的交流。

我大学的时候，室友是一个搭讪高手。有一次在校庆晚会上，她对台上表演小提琴独奏的男生一见倾心。比赛结束后，她甚至尾随男生及其同学到了他们就餐的餐厅。她埋伏在隔壁桌，等到这个男生中途离席的时候，就跟上去发起了进攻。她佯装不确定的样子说：“你是刚才表演小提琴的某某吗？”男生点了点头。她接着说：“哇，你刚才真的好厉害啊！我对小提琴也很感兴趣，可以认识一下吗？”说完，男生当即十分欣喜地给了联系方式，后来还招呼女生和他们一桌吃饭。没多久，他们就成了恋人。

我的舍友就是及时抓住了认识的机会，让男生还沉浸在情绪高涨的时候发起进攻，给他们的感情奠定了初遇时的美好印象。

我们再来总结一下女孩子的搭讪句式，大家可以参考一下：

以下是标准搭讪流程：

1.  **疑问句开场**：“你是刚才参加比赛的谁吗？”表示自己是因为比赛和活动认识到对方的，这样能容易卸下对方的防备。
2.  **表达赞美**：配合崇拜的眼神，表示自己对他的关注。“你在台上的表现太棒了，好厉害！”这样更能增加他对你的好感。
3.  **制造共同点**：作为进一步认识对方的理由，对他说：“我也很喜欢（演讲/辩论等），可以认识一下吗？”这样你们之间的粘合关系就逐渐形成了。

### 场景三：工作对接——运用请求帮助法（富兰克林效应）

最后，我们还可能在社团工作对接中见到让我们喜欢的男生。这时候除了可以使用吸引力法则，还可以用请求帮助法来拉近彼此的关系。

在心理学上，我们把这种通过向对方寻求帮助来赢得好感的现象，称之为**富兰克林效应**。既然你是社团成员，负责管理或者协助执行各种事务，就免不了要对接其他社团的同学。只要有工作对接，你就有了向对方寻求帮助的理由。

具体来说，假如你是校团委的成员之一，按学校安排要写篇新闻稿去广播台播告。这时候你需要交接的同学就有播音专业的、负责调控广播设备的，还有管理播音室的志愿者等等。见面后，当然是以工作为第一话题，让对方觉得你为人认真负责。工作完成后，你可以跟他说：“谢谢你啊，少了你的帮忙，还不知道怎么应付呢。下次有麻烦的时候，还可以拜托你吗？”

这样会让对方有被需要的成就感，然后你就能自然而然地要到对方的微信号码，方便以后的交流了。你还可以为了表示感谢，请对方喝个下午茶，相信对方一定不会拒绝你的好意。这样下次独处就只差确认具体日期了。
