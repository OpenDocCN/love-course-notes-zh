# 爵士舞零基础教程：08：韩流爵士7：步伐教学（一）

在本节课中，我们将要学习爵士舞中最基础的两类步伐：韩式风格与欧美风格。我们将从简单的脚步动作开始，逐步加入身体、手部和头部的协调练习，帮助你掌握不同风格的核心感觉。

## 概述

爵士舞的步伐是舞蹈的基石。本节课将重点分解两种不同风格的步伐：韩式步伐偏向可爱、轻盈；欧美步伐则更强调力量与帅气。我们将从最基础的脚步移动学起，并逐步加入上半身的配合。

## 韩式风格步伐练习

上一节我们概述了课程内容，本节中我们来看看韩式风格的具体步伐。其核心在于脚步的轻盈与身体的协调。

**核心动作公式**：`踢脚 -> 向下坐 -> 快速送出`

以下是韩式基础步伐的分解步骤：

1.  **踢脚与下坐**：右脚做踢脚动作后迅速向下踩，同时跨部放松并向外送出。
2.  **保持轻盈**：脚步落地要轻，体现轻盈感。
3.  **节奏控制**：动作节奏为“快-慢”，即快速出脚，缓慢收回或转换。
4.  **上半身姿态**：上半身保持挺直。踢脚时，肩膀可配合向后微转。

掌握了基础脚步后，我们可以加入手部动作来增加协调性。

以下是手部配合的要点：

*   手部动作通常跟随身体自然摆动。
*   动作方向与脚步方向协调，例如脚步向后时，手部也可向后划动。
*   让手部动作“大”起来，增加舞蹈的表现力。

现在，让我们结合脚步与手部进行完整练习。节奏可以数拍子：`1, 2, 3, 4, 5, 6, 7, 8`。

为了增加舞蹈的灵动感，我们还可以加入头部的动作。

以下是头部动作的要点：

*   头部可以轻轻向旁边甩动。
*   头部的方向与踩脚的方向相同。
*   在转身或移动时，头部随之自然转动。

**练习组合示例**：`脚步移动 + 手部划动 + 头部轻甩`，边走边转圈。

## 欧美风格步伐练习

上一节我们学习了轻盈的韩式步伐，本节中我们来看看力量感更强的欧美风格步伐。其核心在于跨部的运用和身体的延展。

**核心动作公式**：`向前快速踏步 -> 重心前移，后脚虚点 -> 顶胯送出`

以下是欧美向前步伐的分解步骤：

1.  **向前踏步**：一脚快速向前踏出。
2.  **重心转移**：身体重心移到前脚（主力腿），后脚放松，脚尖点地。
3.  **顶胯**：跨部向前顶送出去。
4.  **换脚重复**：交替进行，左右脚练习。

同样，我们可以为这个步伐加入手部动作，增强整体线条感。

以下是手部动作的要点：

*   手部贴着身体移动。
*   可以沿着身体从下向上抚摸式滑动，与脚步节奏配合。

欧美步伐不仅限于向前，向后移动也同样重要。

以下是向后步伐的分解：

*   快速将跨部向后收缩送出，前脚顺势向后拖动。
*   保持身体协调，手部可以自由搭配。

**练习组合**：将向前与向后的步伐连接起来，连续改变方向。记住向前时后跨下压，向后时前跨收缩。

## 滑步专项练习

除了基础移动，滑步也是爵士舞中常见的技巧。下面我们来学习向旁的滑步。

**核心动作概念**：以跨部为中心，带动脚部滑动。

以下是向旁滑步的分解步骤：

1.  **出跨迈脚**：向旁迈出右腿时，跨部同时向同侧推出。
2.  **缓慢收脚**：另一只脚贴着地面，慢慢拖回至中心。
3.  **关键要点**：收脚的速度一定要慢，感觉脚与地面的摩擦力。
4.  **加入推手**：滑步时，手可以向相反方向推出，形成对抗力。

滑步可以应用于多个方向。

以下是四个方向的滑步概念：

*   **向前滑步**：身体如弓箭般向前冲，跨部挤出，胸腰打开。
*   **向后滑步**：臀部向后送，前脚贴地回收。
*   **向旁滑步**：如上所述，跨部向旁推，脚贴地收回。
*   **要点总结**：无论向哪个方向，收回来的脚都必须贴着地面移动。

## 核心要点总结与练习建议

本节课中我们一起学习了韩式与欧美两种风格的爵士舞基础步伐，以及滑步技巧。

以下是本节课的总结与课后练习建议：

*   **风格区别**：韩式风格偏可爱、轻盈；欧美风格偏帅气、有力。
*   **力量要求**：所有步伐都要有力，动作要放大。
*   **滑步关键**：用跨部带动脚步，收脚一定要慢，贴地回收。
*   **动作感觉**：向前步伐呈“弓箭步”姿态；向后步伐要“臀部后坐”。
*   **舞蹈灵魂**：在掌握动作的基础上，加入自己的感觉，自信地跳舞。
*   **课后练习**：多观看韩舞和欧美爵士舞视频，模仿并感受不同风格。
