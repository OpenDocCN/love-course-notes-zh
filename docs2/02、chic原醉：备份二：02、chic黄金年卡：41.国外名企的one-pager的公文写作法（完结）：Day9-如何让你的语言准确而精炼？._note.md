# 公文写作法：02：如何让你的语言准确而精炼？

在本节课中，我们将学习如何让公文写作的语言更加准确与精炼。在掌握了基本的逻辑架构后，精炼的表达能极大提升信息密度与阅读体验。

上一节我们介绍了汪培角公文的三种主要形式，本节中我们来看看如何优化语言层面。

## 精炼语言的价值

对比以下两句话，哪句信息传递更高效？

**第一句：**
> 我司在过去的一个月内完成了200单销售业绩，达到了前所未有的销量，销售额超额增幅比上月同期大幅增长了25%。

**第二句：**
> 上个月，我司产品销量和销售额均创历年新高，销量达200单，销售总额达5000万，同比增长25%。

通过对比，我们可以发现精炼表达的几个优势。

首先，第二句话篇幅更短，字符数从51缩减至36。在表达同等甚至更多信息时，更短的篇幅意味着**更高的信息密度**。对于一页纸的汇报材料，高信息密度能带来更好的阅读体验。

其次，第二句话减少了如“前所未有”、“超额”、“大幅”等**主观修饰词**。这类词语虽能留下印象，但容易使表述拖沓，并因主观色彩影响信息接收的客观性。客观陈述事实更令人信服。

再次，第二句话体现了**清晰的结构化思维**。其逻辑层次如下：
1.  中心论点：销量与销售额创历年新高。
2.  分论点一：销量的绝对值（200单）。
3.  分论点二：销售额的绝对值（5000万）。
4.  分论点三：增长的相对值（同比增长25%）。

短短三四句话，通过三层结构，让读者对业务状况形成清晰认知。

## 核心技巧：缩句练习法

为了让语言精炼准确，我们介绍第一个核心技巧：**缩句练习**。

具体做法是：写下初稿后，不断对句子进行缩减和整合，只保留基本框架。接着梳理逻辑，最后再填充必要信息。

以下是具体步骤：

### 步骤一：缩减与整合逻辑

我们通过一个例子来练习。请尝试缩句并整合下面这段话：
> 面对当前中美贸易战导致的经济下行压力，对外贸易强力缩水的局面，建议我司提前对国内市场进行布局，及早将海外市场的人员与资源重新配置，使之向国内市场倾斜，以便从容应对风险。

这段话含义明确，但逻辑稍显混乱，语言拖沓。我们需要对其进行整合。

首先，用结构化思维梳理其语意，它主要包含四层意思：
1.  **背景**：中美贸易战导致经济下行。
2.  **动因**：为应对对外贸易强力缩水。
3.  **方案**：
    *   尽早布局国内市场。
    *   调整海外市场的人员与资源配置。
4.  **目的**：从容应对风险。

通过整合，我们发现“动因”（第2点）和“目的”（第4点）可以合并。同时，“方案”中的两点存在逻辑关系：**调整资源配置是方式，向国内市场倾斜是结果**。

经过整理，语序可简化为：
> 经济下行压力下，对外贸易缩水，为应对风险，建议调整资源配置，向国内市场倾斜。

逻辑更通顺，语义也更简明。

### 步骤二：填充必要信息

完成精简后，需检查是否遗漏关键信息。以下是必要的筛选与填充过程：

以下是需要补充的关键信息点：
*   **背景层**：“中美贸易战”是关键宏观背景，需补充。
*   **动因层**：“强力”一词能体现事态的紧迫性，需保留；“的局面”三字则属冗余，可删除。
*   **方案层**：“海外市场”是资源配置的具体对象，必须明确。

### 最终成果

运用缩句法精简，再用结构化思维整合，最后填充必要信息，我们得到最终表述：
> **中美贸易战引发的经济下行压力下，对外贸易强烈缩水，为应对风险，建议调整海内外资源配置比例向国内市场倾斜。**

对比原文，新表述在保留原意的同时：
*   逻辑结构更清晰易懂。
*   篇幅字数缩减了约**40%**。

缩句法不仅是写作技巧，更是训练思维精准性与逻辑清晰度的有效方式。这种练习可融入日常表达，通过缩减句子结构看清逻辑本质，再重组语意以实现高效传达。

## 本节课总结

本节课中，我们一起学习了在撰写汪培角公文时，让语言更加准确和精炼的方法。我们详细讲解了**缩句法**的三个核心步骤：
1.  **精简语言**：去除多余的修饰用语。
2.  **整合结构**：利用结构化思维梳理语义与逻辑。
3.  **填充信息**：将缺失的必要关键信息补充回去。

掌握此法，你便能在一页纸的有限空间内，承载更大信息量，并提升汇报的专业性与说服力。
