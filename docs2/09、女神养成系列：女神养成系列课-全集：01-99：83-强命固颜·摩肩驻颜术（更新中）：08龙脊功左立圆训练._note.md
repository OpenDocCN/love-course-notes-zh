# 女神养成系列：08：强命固颜·摩肩驻颜术之龙脊功左立圆训练

在本节课中，我们将学习龙脊功中的左立圆训练方法。这是一种通过脊椎的旋转运动来锻炼身体、促进气血流通的练习。

上一节我们介绍了龙脊功的基本概念，本节中我们来看看具体的左立圆训练动作。

## 动作要领

以下是进行左立圆训练的核心步骤与技术要求。

1.  **起始姿势**：将你的右手手掌放置在肚脐位置。同时，将左手伸于体前，作为动作轨迹的参照物。
2.  **运动轨迹**：以脊椎为轴，带动身体和右手，开始描绘一个立体的圆形轨迹。这个轨迹的顺序是：**向右 -> 向上 -> 向左 -> 向下**。
3.  **动作核心**：整个画圆的过程，应依赖于**脊椎的自主旋转**来驱动，并以此带动放置在腹部的右手完成圆周运动。
4.  **练习节奏**：按照上述轨迹，规律、缓慢地进行画圆练习。例如，可以按“一、二、三、四……”的节奏计数，确保每个方向的动作到位。
5.  **侧面观察**：从侧面视角观察，动作同样应呈现出清晰的圆形路径，遵循向右、向上、向左、向下的循环。

本节课中我们一起学习了龙脊功左立圆训练的具体方法。关键在于通过脊椎的旋转来带动肢体，完成一个标准的立圆轨迹。坚持练习有助于活络脊柱，促进周身气血循环。
