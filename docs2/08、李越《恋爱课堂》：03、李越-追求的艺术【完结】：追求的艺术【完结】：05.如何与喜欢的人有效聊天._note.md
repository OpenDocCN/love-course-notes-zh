# 恋爱课堂：03：追求的艺术【完结】：05：如何与喜欢的人有效聊天

在本节课中，我们将学习如何与喜欢的人进行有效聊天。我们将探讨聊天的真正目的、不同阶段的聊天策略，并学习一个核心的沟通公式，帮助你实现真诚、走心的交流。

上一节我们介绍了邀约与第一次约会。在现实生活中，如果第一次约会后对方对你感觉不错，通常会在约会后主动给你发消息。这个阶段才是两个人真正开始沟通交流的时候，也是本节课的重点。真正的聊天沟通，往往发生在见面约会之后，尤其是一次正式的约会之后。

很多人其实并不擅长聊天。有时查看他们的聊天记录，就能深刻感受到内容的空洞。因此，我们今天要探讨的内容就是关于聊天。我们将审视我们对聊天的认知，与正常情况下的聊天究竟有多大区别。

## 为什么要和喜欢的人聊天？

在讲解如何聊天之前，我们首先需要思考一个根本问题：我们为什么要和喜欢的人聊天？

很多人一旦喜欢上一个人，就会想和对方聊天、说话、发消息。但他们可能从未深入思考过“为什么一定要和对方聊天”这个问题。他们给出的答案往往非常肤浅。很多人即使被劝告不要频繁聊天，也听不进去，最终导致关系恶化，甚至被对方拉黑。

以下是现实生活中，人们喜欢找心仪对象聊天的几个常见原因：

1.  **暗示对方“我喜欢你”**：很多人，尤其是很多女生，喜欢通过频繁聊天的方式向对方暗示自己的好感。她们认为，经常主动聊天就足以让对方明白自己的心意。
2.  **试探对方是否喜欢自己**：这与第一点高度相关。他们的逻辑是：我主动找你聊天，暗示我喜欢你；你回复我，就代表你也喜欢我；你不回复，就代表你不喜欢我。因此，他们并不在乎聊天内容，只在乎对方是否回复。
3.  **认为聊天可以增进感情**：很多人相信“日久生情”，认为只要聊天时间够长，陪伴够久，感情就会自然产生。然而，聊天本身只是一个载体，而非培养感情的工具。感情的增进更多依赖于面对面的相处和真实的吸引力。
4.  **因为无聊、空虚、寂寞**：许多人的生活枯燥，空闲时间多。一闲下来，他们就感到无聊，想找人聊天。在聊天中，他们通过幻想未来的发展来获得愉悦感，用聊天来排解时间。

以上四点，是现实生活中很多人去聊天的目的和原因。然而，这些都属于不正确的目的。正是因为这些不正确的目的，很多人的聊天无法达到预期效果。

## 聊天的真正目的

那么，聊天的真正目的是什么呢？很多人并没有想过这个问题。

聊天的目的需要根据具体情况来区分，这是一个被很多人忽视的问题。很多人正是因为分不清情况而盲目聊天，导致失败。

聊天主要分为三种情况：
1.  见面约会前的聊天
2.  见面约会时的聊天
3.  见面约会后的聊天

接下来，我们将逐一分析。

### 见面约会前的聊天

在见面约会前，聊天的**唯一目的就是约对方出来见面**。没有见过面的追求都不算真正的追求，同样，没有见过面的聊天也基本属于无效沟通。

因此，在这个阶段，应避免闲聊。核心操作是：表达欣赏喜欢，并进行模糊邀约。

为了保证能够成功邀约对方，你需要做到以下三件事：

1.  **真诚**：就像第三课“打开方式”中讲的一样，实话实说。例如，你可以说：“你照片很好看，加个好友吧，有空一起吃个饭。” 对方通常没有理由拒绝。加上好友后（建议使用微信），可以发一个小红包以示友好，然后就不必急于频繁聊天了。
    *   **给女生的提醒**：网络上很多男性目的不纯。个人不建议女生完全依赖网络结识终身伴侣。更推崇在现实生活中结识异性朋友，这样相对更靠谱。
2.  **安全**：这主要是男性需要注意的问题。女性在与陌生男性接触时，普遍缺乏安全感。为了帮助男性理解，这里讲一个故事：
    > 想象你是一个瘦弱的亚洲男性，被关进了美国监狱，里面全是高大强壮、有纹身的壮汉。这时，一个这样的壮汉过来和你搭讪，问东问西，还逐渐靠近你，把手搭在你肩上。你会是什么感受？是否会感到害怕？这个监狱里的感受，就类似于现实生活中漂亮女性对世界的感受——极度缺乏安全感。
    因此，不要给对方压力，不要急功近利，不要天天追问。安全感可以通过时间来建立。在初步联系后，可以隔一段时间再发消息，例如：“好久没聊了，你好像又变美/帅了。” 然后自然地提出模糊邀约：“改天有空一起吃个饭。”
    *   **如果是朋友介绍的异性**：加好友后，请主动做自我介绍。不要装酷或逗比。清晰地说明你是谁、谁介绍的、为什么介绍，并表达不要给对方压力，最后发出模糊邀约。例如：“我是XX的朋友，他挺热心，觉得我们可能合得来，就把你推荐给我了。这样加你有点唐突，希望你别有压力。大家都是年轻人，认识一下交个朋友也挺好的，不一定非要谈恋爱。有空可以一起出来吃个饭、聊聊天。”
3.  **对方对你要感觉不错**：在未见面的情况下，想让对方对你有好印象，你的头像、照片、朋友圈内容至关重要。它们应该能反映出你良好的外形、社会地位或匹配价值。很多人约不出来或聊不好，根本原因往往不是技巧问题，而是对方根本不想和你聊。一旦聊天陷入“一问一答”的模式，基本就说明对方兴趣不大。真正的关键在于提升自身建设，让对方愿意了解你。

**核心技巧：消除戒备心——斩断可能性**

这是一个非常重要的方法论，在后续邀约、表白等环节都会用到。它的核心是**消除对方的戒备，让对方感觉没有压力**。

回想上面自我介绍的例句：“大家都是年轻人，认识一下交个朋友也挺好的，不一定非要谈恋爱。” 这段话就是在“斩断可能性”——斩断“我会追求你”的可能性。意思是，我们只是认识一下，你不要觉得我想追你，从而放下防备。

对于自身条件有短板的人，这个方法尤其适用。例如，一个体型较胖的朋友可以这样说：“我知道我配不上你，所以我绝对不会对你有非分之想，你可以放心跟我做朋友。” 这样先降低对方预期，解除戒备，再通过后续接触展现其他魅力。

**关键点**：喜欢和在一起是两回事。喜欢一个人没有成本，但要求在一起就意味着对方需要做出选择，而选择意味着放弃其他可能性。因此，一开始就“斩断可能性”，只表达喜欢而不强求结果，给对方充足的时间去权衡和思考，反而能制造一种“唾手可得”的感觉，让对方更容易接受你。

**重要提醒**：说完“斩断可能性”的话之后，行为上必须配套。不要立刻又开始天天找对方聊天，那样会显得很矛盾，毫无安全感。应该按照正常的、低频的朋友节奏来互动。

如果你能做到以上三点（真诚、安全、感觉不错），那么最终的聊天都会导向一次线下的约会。

### 见面约会时的聊天

关于约会时的聊天技巧，我们在上一堂课已经有所涉及，并且在后续关于多次约会的课程中会详细展开，因此这里不再赘述。本节课的重点在于见面约会后的聊天。

### 见面约会后的聊天

如果你和喜欢的人见过面、约过会，并且你已经表达过欣赏和喜欢，那么回家后，如果对方还主动发消息给你，或者你发消息时对方愿意回复，这才是你们真正开始聊天、拉近关系的时候。

## 聊天到底聊什么？

很多人对聊天沟通并不了解。他们聊天时，总是喜欢聊一些**客观事实**，比如家乡、工作、日常琐事等。

**问题在于**：客观事实有其局限性，很容易聊完。对于生活没有交集的两个人，这些内容可能无法引起共鸣，对方也可能不感兴趣，难以达到共情的效果。

**聊天的本质，应该是聊情绪和感受**。情绪感受（如开心、难过、向往、渴望）是人人都有的，而且每个人的体会是相通的，每天都在发生，因此聊不完。聊情绪感受，就是展示我们的内心世界，表达真实想法，这才是“走心”的沟通。

## 沟通的三个层次

人与人之间的交流沟通分为三种境界：

1.  **脑与脑的交流**：这是很多人常用的方式。聊天时总是在思考：我该怎么回复？对方会怎么想？怎么说才显得有魅力？这种方式非常累，且不走心，容易产生隔阂感。
2.  **心与心的交流**：这是我们真正需要学习的“走心”沟通。它真诚、直接，关注彼此的感受和内心。
3.  **眼与眼的交流**：这需要面对面才能实现，通过眼神传递情感，难度较高，需要专门训练。

本节课，我们重点学习**心与心的交流**。

## 核心沟通公式

如果你觉得自己不会聊天，可以遵循以下公式进行反思和练习。这个公式的背后是**真诚**。当你没有渊博学识和巧妙技巧时，真诚是最好的聊天方式。

**公式：欲望 + 恐惧 + 情绪**

**注意**：最好在见面约会并表达过欣赏喜欢之后，再使用这个公式。否则，在关系未建立时使用，可能会让对方觉得唐突。

### 1. 表达欲望

欲望就是你内心真实的想法和想要得到的东西。例如：我想你、我想和你聊天、我希望你喜欢我、我想亲你等等。

**但问题**：如果直接表达欲望，会显得具有攻击性和压迫感，容易让对方不舒服或抗拒。

### 2. 表达恐惧

恐惧就是你不敢直接表达欲望的原因，或者你害怕得到的不好结果。例如：害怕你不喜欢我、害怕被你讨厌、害怕被你拉黑、害怕自己配不上你等等。

**解决方案**：在表达欲望的同时，把你的恐惧也一起说出来。这样可以极大地减少欲望表达带来的压迫感。

**举例对比**：
*   **只想说“我想你”** -> 变成问“在吗？”（因为害怕被觉得轻浮）
*   **用公式表达**：“其实我挺想你的，但又怕你觉得我太浮夸，所以只敢问一句‘在吗？’”
*   **只想问“你有没有想我？”** -> 变成“最近过得怎么样？”
*   **用公式表达**：“我特别想问你有没有想我？不过想想我这样子，估计你也不会想我。”
*   **只想说“我想关心你，让你喜欢我”** -> 变成“多喝热水”
*   **用公式表达**：“我想关心你一下，毕竟我喜欢你嘛，万一你将来喜欢上我了呢？不过我也不知道怎么关心人，所以…你还是多喝点热水吧。”

### 3. 表达情绪

当欲望和恐惧产生矛盾时，就会引发各种情绪感受。把这些情绪感受也表达出来。

**完整公式示例**：
“我其实特别想跟你聊天（**欲望**），但又怕老是找你，让你有压力，觉得我想泡你，然后你就不理我了（**恐惧**）。唉，人有时候就是这样，明明很想做一件事，却因为害怕而束手束脚，心里还挺难受的（**情绪**）。”

### 公式的延伸应用

这个公式引出的对话，可以自然地导向分享你自己的故事、经历、看法和价值观。

例如，在表达了“想问你喜不喜欢我但又害怕”之后，你可以接着说：“这让我想起我以前喜欢一个人的时候，也是这种纠结的心情……” 然后就可以讲述一个相关的个人故事，分享你的人生观、感情观。

**重要提醒**：
1.  **自我认知是关键**：如果你对自己内心的欲望、恐惧和情绪都缺乏认知，那么使用这个公式会显得生硬，甚至可能被对方认为是神经病。你需要先了解自己。
2.  **适可而止**：不要一聊起来就停不下来，聊到深夜。一次好的、走心的聊天，在情绪和感受达到一个不错的高度时，就应该主动终止。你可以说：“今天聊得很开心，感觉真好。改天我们见面再聊吧/空了打电话给你。” 这样为下一次邀约埋下伏笔。
3.  **推动关系**：整个流程应该是：见面表达喜欢 -> 回家后通过走心聊天加深联系 -> 再次邀约。通过这种方式，关系才能得到有效推进。

## 总结

本节课中，我们一起学习了如何与喜欢的人进行有效聊天。

我们首先分析了人们聊天的错误目的，然后明确了聊天应根据“约会前、约会中、约会后”不同阶段，设定不同的目标（尤其是约会前应以邀约为唯一目的）。我们深入探讨了实现成功邀约需要做到的三点：真诚、安全、让对方感觉不错，并学习了“斩断可能性”这一重要技巧来消除对方戒备。

最后，我们掌握了心与心交流的核心公式：**欲望 + 恐惧 + 情绪**。这个公式帮助我们在表达真实想法的同时，照顾对方的感受，实现真诚、走心的沟通，并能自然引出个人故事的分享，从而深化连接。

记住，这个世界上没有绝对的“聊不来”，只有“是否愿意聊”。如果对方根本不想搭理，再多的技巧也无用。因此，提升自身吸引力永远是第一位的。
