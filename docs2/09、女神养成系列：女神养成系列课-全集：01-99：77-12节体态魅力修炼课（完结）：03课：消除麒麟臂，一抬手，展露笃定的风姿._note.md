## 旗袍流派简介：京派与海派

上一节我们介绍了旗袍的整体魅力，本节中我们来看看旗袍的主要流派。从主流上划分，旗袍分为京派和海派。

我本人尤其偏爱京派风格，这可能与我生活在北方有关。京派旗袍特别能够体现女性的高贵与大气。

例如，在反映大宅门内生活的电视剧中，人物的服饰就具有鲜明的京派旗袍特征。虽然我今天的着装并非完全如此，但也具备一些相关特点。

以下是京派旗袍的几个核心特征：

*   **领型较高**：京派旗袍的领型比海派更高。高领自然会对穿着者的体态产生一种约束，要求必须挺直脖颈。正如前面课程所讲，挺拔的脖颈能体现女性的高贵与知性，也反映出穿着者的内在品格。
*   **镶边较宽**：海派旗袍的镶边讲究细致精巧。而京派旗袍的镶边通常比较宽，有时甚至不止一道边，可能是两道或三道，这反映了不同的审美感觉。
*   **面料厚实**：面料特点可能与京派源于北方有关，因此更厚实，穿上后更显笔挺，例如常用的织锦缎。
*   **绣花讲究**：京派旗袍在绣花部分特别讲究，这与满族服饰的传统有关。满族服饰在滚边和绣花方面历来非常考究，其中包含许多学问和说法。我们会在后续的视频节目中，为大家详细介绍不同绣花图案的内涵。这就是所谓的旗袍文化，它体现在每一个细节之中，让我们能更深入地理解旗袍。

## 手臂之美与旗袍穿着

现在很多女性喜欢穿无袖旗袍，但手臂的形态，尤其是所谓的“拜拜肉”，很容易暴露年龄感。

确实，手臂非常容易出卖我们的年龄。那么穿旗袍时，应如何注意手臂姿态呢？最关键的就是要解决“拜拜肉”的问题。下面我将通过具体动作来帮助大家进行改善。

## 消除“麒麟臂”练习

以下是针对手臂的练习，旨在增强手臂力量，紧致线条。

**第一个练习：侧平举伸展**
双腿打开，比肩略宽。双手向身体两侧无限延伸，手臂需用力，与肩膀保持一条水平线。
*   **起始姿势**：手心朝下。
*   **动作过程**：跟随口令进行小幅度的上下摆动练习。口令示例：`5, 4, 3, 2, 1, 2, 3, 4, 2, 2, 3, 4, 3, 2, 3, 4, 4, 2, 3, 4, 5, 2, 3, 4, 6, 2, 3, 4, 7, 2, 3, 4, 8, 2, 3`（停顿）。注意摆动弧度可稍大。
*   **要点**：手臂始终保持力量感。

**第二个练习：侧平举翻掌**
接下来进行手心朝上的练习。
*   **起始姿势**：在侧平举基础上，将手心翻转朝上。
*   **动作过程**：同样跟随口令进行上下摆动。口令示例同上。
*   **要点**：记住手臂一定要用力，特别是在向后扩展时，要感受到一种对抗的力量感。

完成这两个动作后，您可能会感觉手臂酸胀发热，这表明练习是有效的。当然，长期坚持训练至关重要。

## 手腕灵动性练习

女性的手是体现优雅的关键部位，而手部之美也源于手腕的灵动。女性的举手投足之美，离不开手腕的灵活与优美。这非常重要。

以下是两个专门训练手腕的动作：

**第一个练习：十指交扣转腕**
*   **起始姿势**：双手十指相扣。
*   **动作过程**：尽量活动手腕，让其滚动起来。跟随口令练习，例如：`5, 4, 3, 2, 1, 2, 3, 4, 2, 2`（滚动起来），`3, 2, 3, 4, 4`（保持微笑），`2, 3, 4, 5, 2, 3, 4, 6, 2`（再美一点），`3, 4, 7, 2, 3, 4, 8, 2, 3`（停顿）。

**第二个练习：兰花指腕部练习**
*   **起始姿势**：双手平行，与手臂呈90度，手心朝上。中指向前微微顶出，大拇指轻抵在中指的第一指节处，呈现一点兰花指的形状。
*   **动作过程**：进行手腕的压、提、转掌练习。跟随口令：`压，提，转掌，压，提，转掌，压，提，转掌，1, 2, 3, 4, 2, 2`（滚动起来），`3, 2`（再美一点），`4, 2, 3, 4, 5, 2, 3, 4, 6, 2, 3, 4, 7, 2, 3, 4, 8, 2, 3`（停顿）。

美丽的女性，其举手投足之美，一颦一笑之姿，都离不开手臂形态的美化。因此，我们既要消除“拜拜肉”，也要让手腕更加优美灵动。这份举手投足间的美感，源于我们对每一个细节的关注与练习。世上没有丑女人，只有懒女人。
