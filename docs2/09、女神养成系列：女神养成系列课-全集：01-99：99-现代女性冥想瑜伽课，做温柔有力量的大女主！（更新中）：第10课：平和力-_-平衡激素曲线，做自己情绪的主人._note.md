# 女神养成系列：10：平和力 _ 平衡激素曲线，做自己情绪的主人

## 概述

在本节课中，我们将学习如何通过瑜伽体式和冥想技巧来培养“平和力”，从而更好地管理情绪，成为自己情绪的主人。课程将分为三个主要部分：理解情绪与平和力的关系、学习三个放松身心的瑜伽体式，以及掌握一种名为“十六指呼吸法”的古老冥想技巧。

## 情绪与平和力

上一节我们探讨了如何唤醒内在觉知。本节中，我们来看看一个更具体的话题：平和力。平和力与情绪密切相关。情绪非常重要。

情绪能够影响心念的波动。心念的波动又会影响生命能量的波动。当生命能量波动涣散时，我们很难处于完全平静和平和的状态，甚至难以进入冥想。

大部分人曾错误地认为情绪是不好的，只有负面作用。实际上，情绪除了负面方向，也有正面方向。例如，开心、高兴都是情绪正向引导的表现。过去，我们容易受情绪控制。

也就是说，成为情绪的奴隶。当一种情绪影响我们时，我们容易投入其中。这时，我们会认为自己与情绪是一体的。因此，我们会想：情绪就是我，我就是情绪。

那么，什么是正确态度？

当情绪产生时，我们要做旁观者，从旁观角度观察情绪。这样，我们才能将自己与情绪剥离。情绪便不会控制我们，我们也能成为情绪的主人。举个例子。

家中有老年人，例如太爷爷、太奶奶。当太爷爷、太奶奶坐在沙发或椅子上，看小朋友们打闹玩耍时，他们会一直笑眯眯地看着，不会制止、阻止或评判孩子们。我们对待情绪也应采取这种态度。

当情绪升起时，我们不要判断它，不要先用主观想法投入分析。我们只需站在旁观者角度观察即可。

这样，我们就可以做到不被情绪控制。冥想不仅能唤醒内在觉知，也能让我们带着这种觉知，回到生活中更好地处理与情绪的关系。接下来，让我们开始今天的练习。

## 瑜伽体式练习

大家好，我们现在开始今天的体式。今天，我们将学习三个非常轻柔且易做的体式，它们对身体的放松效果非常显著。

以下是三个体式的练习步骤：

**1. 悬挂式**

我们从站立姿势进入。站立时，双脚与髋部同宽，脚趾指向正前方。从髋部开始，身体折叠向下。

```
动作要领：
1. 双脚与髋同宽站立。
2. 从髋部折叠，上半身向下。
3. 双手放松搭在地面。
4. 头部、肩部、颈部放松，感觉头悬挂在肩膀上。
```

如果大腿或小腿后侧紧张，可微微弯曲双膝。弯曲双膝后，身体向下的空间会更大。在此体式中，保持上半身放松，可以互抱手肘。停留一分钟。

> **注意**：高血压者可不做此体式。低血压者在结束体式时，应卷着背慢慢站立。正常练习者可直接从悬挂式进入下一个体式。

**2. 蹲式**

从悬挂式直接落下臀部蹲下。保持双膝与髋同宽，双手合十。

```
动作要领：
1. 从悬挂式直接蹲下。
2. 双膝与髋同宽。
3. 双手合十于胸前。
4. 骨盆放松，但盆底肌微微上提。
```

此体式可放松和强化脚踝关节。若无法完全蹲下，可将毛巾卷放在脚后跟下。在蹲式中放松一分钟。然后慢慢睁开眼睛，双手撑地，直接坐在垫子上，伸直双腿，释放压力。

**3. 鹿式**

用双手帮助，先弯曲左膝，将左小腿拉向左大腿外侧。然后弯曲右膝，将右小腿放在身体前侧。

```
动作要领：
1. 弯曲左膝，左小腿置于左大腿外侧。
2. 弯曲右膝，右小腿置于身体前侧。
3. 左大腿内旋，右大腿外旋。
4. 确保两侧坐骨均落地。
```

若任何一侧膝盖有压力，可将该侧小腿拉近大腿。小腿与大腿距离越短，体式越易；距离越大，体式越难。此体式能很好放松腿部紧张，尤其适合孕期腿部浮肿的宝妈，但孕晚期者可不做。双手自然撑在臀部两侧，保持十个呼吸。然后换另一侧练习。

## 十六指呼吸冥想法

体式部分结束后，我们学习新的冥想技巧。今天，我们将学习一种非常古老的呼吸技巧，也是一种很好的冥想方法，它更偏向于意识的感受，称为“十六指呼吸法”。

首先，找到十六指的距离。从鼻尖开始向下测量：四指、八指、十二指、十六指。我们先找到十六指的位置。

**练习方法如下：**

1.  **十六指**：感受气息从鼻尖到达十六指位置，然后从十六指回到鼻尖。气息之后去向何处，不管。一呼一吸为一轮，做四轮。
2.  **十二指**：感受气息从鼻尖到十二指，再回到鼻尖。做四轮。
3.  **八指**：同样方式，做四轮。
4.  **四指**：感受从鼻尖到四指距离的呼吸，做四轮。
5.  **两指**：感受呼吸带到两指距离，做四轮。
6.  **一指**：做四轮。
7.  **零指（鼻尖）**：意识将呼吸留在鼻尖，做四轮。
8.  **眉心（两眉中间）**：此位置不能呼吸，但用意识感受呼吸带到这里，做四轮。
9.  **眉心轮（头部正中心）**：用意识将呼吸带到眉心轮，再呼出到鼻尖，做四轮。
10. **喉轮**：感受气息从喉轮进出，做四轮。
11. **两肺**：感受气息直接从肺部进出宇宙，做四轮。
12. **心轮**：感受气息从心进出，做四轮。

每一轮可做4到7次呼吸，初学者可从四轮开始。此呼吸法锻炼的是觉知，引导意识去往不同地方，而非实际气息的长短。

现在，我们开始练习十六指呼吸冥想法。

请按照上述步骤练习，每一步做四轮呼吸。注意，我们并非真的把气吹到那些位置，只是用意识去感受更深层的呼吸。呼吸本身应保持自然。

当你的呼吸来到心的位置时，可以做十轮。吸气时感受从心开始，呼气也从心开始。练习时，可以体会这个方法产生的效果。

如果你已完成整个过程，观察一下你的呼吸发生了什么改变。它充满了觉知，同时极其缓慢。这就是此呼吸法带来的好效果。

接下来的任何时间，你都可以躺下。在仰卧时，感受之前发生的变化仍在身体里呈现。去感受这些舒适的感受，它们正是呼吸技巧产生的效果。

如果你感觉到意识深处有一种深深的平静，也去感受它。感受已经发生的一切。

在接下来的静默中，再次观察呼吸法带来的效果。呼吸非常缓慢，它产生了作用。感受整个身体发生的变化。有一种说不出的感受正在发生。

非常轻柔地，让这种说不出的感受持续蔓延。现在，再看看缓慢呼吸产生了什么效果。能否将“说不出的感受”与“缓慢呼吸的效果”结合起来？

结合的意思是同时感受这两种功效。如果你能同时感受两者，那么你已经把它们结合起来了。在内心继续观察这种结合的作用，你就会感受到冥想发生了。

现在，静默出现了。深深的静默。一旦你处于静默之中，就去继续感受这种静默。

现在，感受脉轮。感受腹部的中心，头部的中心。不要想象下一步，只要留在当下的感受中。

继续感受这两个脉轮。首先，在腹部（肚脐以下中间区域）停留一段时间。然后，去感受头部的中心并停留。接下来，同时感受这两个区域。

或许现在，你已经感受到头顶（顶轮）发生了一些改变。但你不应当去想象，只是留在这里，简单地等待。等待这一切的发生。

现在，忘掉一切，脉轮也忘掉，只是静默。如果你走神了，没关系，只需要回到当下，再一次等待。在等待中，静默就会来到。

这时，只有静默。当你感觉可以时，可以坐起来（保持眼睛闭着），也可以继续仰卧。在坐姿中，我们保持2到3分钟，让静默再持续一会儿。

当你准备好了，可以允许自己的眼睛睁开。这是今天的冥想练习。

## 总结

本节课中，我们一起学习了如何通过理解情绪来培养平和力，练习了三个放松身心的瑜伽体式（悬挂式、蹲式、鹿式），并掌握了一种引导意识的古老冥想技巧——十六指呼吸法。通过持续的练习，我们可以更好地观察情绪，不被其控制，从而成为自己情绪的主人，让内心获得更深的平静与力量。
