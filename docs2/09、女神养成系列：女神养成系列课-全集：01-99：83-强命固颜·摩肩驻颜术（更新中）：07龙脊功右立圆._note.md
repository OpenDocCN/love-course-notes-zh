# 女神养成系列：07：龙脊功右立圆

在本节课中，我们将要学习龙脊功的第三个动作——右立圆。这个动作的核心在于通过身体的转动，带动右手在身体右侧画一个垂直的立圆，同时左手悬空作为参照，以强化脊柱的灵活性与身体的协调性。

上一节我们介绍了龙脊功的基础姿势与呼吸配合，本节中我们来看看具体的右立圆动作如何执行。

## 🌀 动作分解与要领

首先，将你的右手手掌轻轻贴放在右侧大腿根部（即“肚底”位置附近）。左手则自然抬起，掌心向下，稳定地悬于身体左前方的空中。整个过程中，左手的位置应保持固定不动，它将作为我们身体转动的空间参照物。

以下是右立圆的完整运动轨迹描述，请跟随步骤练习：

1.  **起始**：右手掌贴于右大腿根部。
2.  **向左**：以腰胯为轴，身体向左微微转动，带动右手沿弧线向左移动。
3.  **向上**：身体继续转动并稍向后仰，带动右手从左侧向上画弧，举至头部左上方。
4.  **向右**：身体开始向右转动，带动右手从左上方向右侧画弧移动。
5.  **向下**：身体转正并稍向前俯，带动右手从右侧向下画弧，最终回到起始的右大腿根部位置。

如此，右手便完成了一个 **“向左 → 向上 → 向右 → 向下”** 的垂直圆形轨迹。

## 👁️ 关键细节与侧面演示

请注意，这个动作的驱动力来源于身体的旋转，而不是单纯地用手臂画圈。你的右手应始终轻贴身体，仿佛被身体的转动“推”着走。

从侧面观察时，你能更清晰地看到：右手紧贴身体侧面，由身体的转动带动其画圆。与此同时，左手始终悬停在初始位置，作为一个静止的参照点，这有助于你感知和控制身体转动的幅度与平面。

本节课中我们一起学习了龙脊功的右立圆动作。我们掌握了以身体带动右手画立圆的轨迹，并理解了左手作为空间参照物的作用。请务必记住，动作的流畅性源于腰胯的转动，多加练习将帮助你更好地感知脊柱的运动，达到强健体魄、提升气韵的效果。
