## 呼吸法：口哨式呼吸（鸟嘴呼吸） 🐦

上一节我们介绍了课程目标，本节中我们来看看第一个练习：口哨式呼吸法。这种呼吸法能改变全身血液循环，激活喉部腺体（如甲状腺），增强肺活量并释放压力。

**练习步骤如下：**

1.  选择一个舒适的坐姿（如半莲花盘、至善坐或简易坐），保持脊柱挺直。
2.  撅起嘴唇，模仿鸟嘴或吹口哨的形状。
3.  闭上眼睛，将注意力内观于眉心。
4.  通过撅起的嘴唇吸气，发出高音的口哨声。
5.  呼气时闭上嘴，用鼻子缓缓呼气。

**公式/代码描述核心动作：**
```
吸气：嘴唇撅起 -> 发出口哨音
呼气：嘴唇闭合 -> 鼻子呼气
```

如果在练习中无法发出声音，只需保持“嘴吸鼻呼”的模式即可。建议跟随课程一起练习3分钟。

## 热身体式

在进入主体练习前，我们需要充分热身。以下是三个针对肩颈和喉部区域的热身体式。

### 1. 虎式背伸展

这个体式能有效活动下腰背部和肩部，增加其灵活性，特别有助于改善久坐带来的酸痛，并能促进盆腔区域的能量流动。

**练习步骤如下：**

1.  四角板凳式准备，双手与双膝落地。
2.  吸气时，右腿向后向上伸展，腰椎下沉，颈部延展，抬头。
3.  呼气时，屈膝收腿，弓起背部，低头。
4.  重复8-12次后，换另一侧练习。

**常见问题与调整：**

*   **重心不稳**：抬起腿时，收紧核心，将同侧髋部向下压，避免胯部翻转。
*   **手腕压力大**：避免耸肩，将肩膀远离耳朵。如果压力过大，可在手腕或膝盖下方垫上毯子以缓冲。
*   **膝盖疼痛**：在膝盖下方垫毯子，减轻直接压力。

聪明的同学可能已经发现，虎式背伸展很像猫牛式，它正是在猫牛式的基础上加入了腿部的伸展，使盆腔区域的活动更彻底。

### 2. 打开肩膀与喉部的练习

接下来，我们进行一个专门针对肩膀、上胸腔及喉部的梳理练习。它能有效减压，并刺激喉部经络的能量流动。

**练习步骤如下：**

1.  保持舒适坐姿，脊柱挺直。
2.  手印：拇指抵住小指根部，其余四指包住拇指握拳。
3.  双拳相对，屈肘置于胸前。
4.  吸气，屈肘向两侧打开，肩胛骨靠拢。
5.  呼气，伸直手臂向两侧打开。
6.  吸气，回到屈肘打开姿势。
7.  呼气，感觉像将一盆水从下向上轻洒向自己，手臂上举。

这是一个四步组合动作，配合呼吸进行：吸气（开）-呼气（伸）-吸气（合）-呼气（举）。

**协调技巧：**
为了帮助呼吸与动作协调，可以在心中默念曼陀罗（Mantra）“Sa Wa He Gru”。其中“Sa”代表真我，“Wa He”表达喜悦与赞叹，“Gru”代表从黑暗走向光明的智慧。通过心念振动这个频率，可以更好地保持身、心、意的连接。

### 3. 颈部全方位练习

由于本节课主题与喉部相关，颈部的舒展至关重要。以下是一套360度无死角的颈部练习。

**练习步骤如下：**

*   **前后伸展**：
    *   吸气，从胸腔开始延展，带动颈部前侧线条，头部微微后仰。
    *   呼气，脊柱拉直回正，然后低头，感受后侧伸展。
*   **左右侧弯**：
    *   吸气，头部向左肩倾斜，感受右侧颈部拉伸，下巴微收。
    *   呼气，头部回正，再向右肩倾斜。
*   **左右扭转**：
    *   吸气，面部带动颈部向左扭转。
    *   呼气，向右扭转。
*   **颈部轮转**：
    *   呼气低头，吸气头部向左、向上缓慢画圈，到顶部后屏息片刻，呼气向右、向下完成旋转。反方向亦然。

**练习要点：**
动作务必放慢、放柔，重点感受每个角度的拉伸，而非追求幅度。如果在某个位置感到卡顿，可在此稍作停留并轻微活动以缓解。若练习时头晕，可尝试微睁眼睛注视前方固定点，并确保呼吸顺畅。

## 主体序列：Wa 体系练习 🌀

热身结束后，我们进入今天的主体部分——Wa 体系练习。这套序列针对胸腔、肩颈、甲状腺、脑垂体及松果体，能激活腺体，调节内分泌，平衡喉部能量。

### 1. 椅姿式

这个体式结合了姿势、呼吸与唱诵，能有效调动腹部、胸腔和喉部的能量。

**练习步骤如下：**

1.  站立，双脚分开比髋宽，屈膝下蹲，背部尽量平行于地面。
2.  双手从膝盖内侧绕到外侧，搭在脚背上，脊柱伸展，面部朝下。
3.  吸气时，身体重心向左移动，唱诵“Wa He”。
4.  呼气时，身体重心向右移动，唱诵“Gru”。
5.  持续练习2分钟。

**要点：**
保持胸腔开放，脊柱延展。唱诵时，身体会自动调节呼吸，无需刻意控制。重点是找到放松、顺畅的感觉。

### 2. 站立后仰

这个体式能进一步伸展脊柱，打开胸腔。

**练习步骤如下：**

1.  站立，双脚微分开，双手扶于后腰。
2.  吸气，脊柱整体向后伸展，微微后仰，打开胸腔，肩膀下沉。
3.  头部向左转时唱“Wa He”，向右转时唱“Gru”。
4.  持续练习后，吸气回正，呼气放松。

**要点：**
避免过度后弯压迫腰部，感受脊柱的延伸。膝盖可微屈以保持稳定。唱诵时，配合腹部的自然收放：“Wa”时注意力在腹腔，“He”时在胸腔，“Gru”时在撅起的嘴部。

### 3. 站立前屈

从后仰体式自然过渡到前屈，持续作用于甲状腺区域。

**练习步骤如下：**

1.  从站立姿势，微屈膝，双手放于膝盖上，保持脊柱伸直，胸腔打开。
2.  身体向左移动时唱“Wa He”，向右移动时唱“Gru”。
3.  练习结束后，呼气放松背部，吸气慢慢卷动脊柱起身。

**要点：**
避免耸肩，肩部向下拉以延长脊柱。保持膝盖舒适的角度。

### 4. 直立伸展

这个体式能提升能量，锻炼平衡。

**练习步骤如下：**

1.  站立，双脚并拢或分开（根据平衡能力），收紧核心。
2.  吸气，双手由前向上伸直。
3.  脚掌踩地时唱“Wa He”，踮起脚尖时唱“Gru”。
4.  练习2分钟后，吸气延伸，呼气放松。

**要点：**
核心收紧以保持骨盆稳定和身体平衡。练习中允许身体有轻微晃动，重点是快乐地完成，而非追求完美。

### 5. 狮身人面式

这个跪姿体式能温和地刺激喉部与眉心区域。

**练习步骤如下：**

1.  跪姿准备（膝盖不适可垫毯子），双手五指分开置于膝前地板，手推地，脊柱延展。
2.  吸气延伸脊柱时唱“Wa He”。
3.  呼气屈肘，额头尝试靠近地面时唱“Gru”。
4.  练习2分钟后，吸气伸直手肘，抬起上身，呼气放松。

**要点：**
如果额头无法点地，可在前方垫瑜伽砖或垫子。闭上眼睛，眼球向内向上看眉心。保持动作与呼吸的节奏。

### 6. 萨塔拿妈（Sat Nam）唱诵与脊柱波动

这是序列的收尾部分，通过唱诵连接宇宙的无限循环。

**练习步骤如下：**

*   **静坐唱诵**：盘坐，先轻声持续唱诵“Sat Nam”2分钟，再大声唱诵2分钟。
*   **脊柱波动**：转为至善坐。吸气脊柱向前弯曲时唱“Sat”，呼气脊柱向后弯曲时唱“Nam”。重复2分钟。

**要点：**
唱诵时，头部随脊柱自然运动，不要单独甩动。收住会阴，保持下盘稳定。“Sat”代表真理、无限，“Nam”代表名字、身份，整体意为“真理即我的身份”。

## 放松与冥想：萨塔拿妈冥想 🧘‍♀️

完成主体练习后，进行大放松和冥想，让身心得到深度修复。

**大放松（摊尸式）：**
平躺，双腿分开比髋宽，双手放于体侧，掌心朝上。依次放松脚踝、膝盖、髋部、肩膀、肘部、手腕和颈部，保持自然呼吸，休息片刻。

**萨塔拿妈冥想：**
这是昆达里尼瑜伽中一个经典冥想，能平复心绪，深化自我连接。

**练习方法：**

1.  **手印**：大拇指依次与食指、中指、无名指、小指用力相触。分别代表智慧、自律、能量、沟通与假我的结合与循环。
2.  **观想**：想象一道光束从头顶进入，从眉心呈L型射出，形成一个能量通道。
3.  **唱诵**：眼球向内向上看眉心。按照“大声唱（2分钟）-> 轻声唱（2分钟）-> 默唱（2分钟）-> 轻声唱（2分钟）-> 大声唱（2分钟）”的顺序循环唱诵“Sat Nam”。
4.  **意义**：这个过程象征着从对外表达，到亲密低语，再到内在对话，最后回归外界的完整能量旅程。
