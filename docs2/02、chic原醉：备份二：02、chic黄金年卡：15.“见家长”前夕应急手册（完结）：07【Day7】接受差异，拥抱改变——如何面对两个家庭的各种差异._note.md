# 见家长攻略：02：接受差异，拥抱改变——如何面对两个家庭的各种差异

在本节课中，我们将要学习如何处理两个家庭之间可能存在的各种差异。这些差异包括经济条件、地域文化以及生活方式等方面。我们将探讨这些差异如何影响关系，并提供具体的应对策略，帮助你在见家长时能够大方得体，有效化解潜在矛盾。

上一节课中，我们主要提及了原生家庭中父母的亲密关系对你们的影响。今天的这节课，我们继续讲一讲两组家庭之间的各种差异，会导致你们产生怎样的矛盾以及应对的方法。

当人们听说一对情侣的家庭条件、文化背景十分契合，就会评价他们“真是天生一对”。这种评价很大程度上是因为传统认知中“门当户对”的概念依然存在影响。人们认为两个人的生活方式、经济条件、文化背景一致，会让婚姻更加和谐，而巨大的差异则可能引发各种矛盾，让未来的路异常崎岖。

在过去，“门当户对”是长辈们约定俗成的相亲规则。为了保证这个规则顺利实施，“父母之命，媒妁之言”成为了年轻男女必须遵守的铁律。“门当”指的是挣钱的门道相当，说的是男婚女嫁时，两家人财富地位差不多才能够成亲。在《红楼梦》中，贾母将女儿贾敏嫁给林如海，就是想成就一门最门当户对的亲事。不过如今，我们对“门当户对”的要求已经越来越弱，越来越宽容。但面对经济条件、文化水平、生活方式等各种家庭差异时，要做到泰然处之，依然不太简单。

下面我们就具体聊一聊，在第一次见家长时，面对各种家庭差异，如何通过恰当的处理方法，让矛盾由大变小，由小变无。

以下是面对不同家庭差异时的具体应对方法。

**第一种：家庭条件的差异**
这种差异主要是指经济条件、社会地位上的差异，表现在双方家庭储存的社会财富不一样，父母所从事的职业有区别。面对这些差异时，我们要怎样表现才能够显得大方又得体呢？

在心理学上有一个概念叫**定式效应**，是指有准备的心理状态，能够影响后继活动的趋向、程度以及方式。参考这一效应，我们就可以在见家长时给自己先预设一个良好的心理状态。暗示自己“我和对方在精神世界是平等的，我们要互相尊重”，从而让自己保持不卑不亢的态度，这样交流起来就会顺利很多。

*   **如果你的家庭条件更强**：就要保持谦逊有礼的态度，时刻照顾伴侣的感受和对方家人的情绪，让你们之间平等交流，从而获得好感。
    *   **例如送礼物**：以实用型为主，不要送昂贵又不实用的东西，造成对方家人的心理负担。
    *   **例如见面时**：要表现出晚辈对长辈的尊敬，不知道怎么称呼，就统一用敬语“您”。行为举止要大方自然，更要懂得礼让，让他们亲近你，认可你。

陈小春与应采儿这对明星夫妻，彼此原生家庭的差距很大。应采儿第一次去陈小春家里时，打扮十分朴素，始终面带笑容，表现得平易近人，丝毫没有明星架子，一下子就赢得了陈小春家里老人的好感。

*   **如果你的家庭条件较弱**：就要真诚交流，诚实地回答自家情况，并且表明两个人的亲密关系是建立在真情实意之上，从而获得对方家庭的信任。
    *   **例如在韩剧《我叫金三顺》中**：当被问及家世时，女主三顺直接告知实情。虽然最初让对方产生了猜疑，但几次交流之后，她的真诚就打动了对方。所以，处于弱势时，不要产生自卑情绪或用撒谎伪装自己，实事求是地说出真实情况，其实更能够得到理解和认可。

在这里还有一个小技巧，就是**制造与对方家庭的隐形边界**，表明自己与伴侣建立“二人组”的立场来打消他们的提防。

*   **例如**：当对方妈妈问三顺为什么和她儿子在一起时，三顺回答：“我希望能把自己做得最可口的蛋糕，第一个给真宪品尝。”这句话说明了她有自己的事业，也对未来二人生活充满了期许，给对方妈妈营造出“我只是看上你儿子，而不是看上你们家”的感觉，也就是画出了一条隐形的界限，获得了对方的信任。

**第二种：地域文化的差异**
这种差异主要是由于生长的地域环境和民族属性不同所导致的饮食习惯、风俗习惯、生活习惯上的差异。例如，广东人不喜欢娶外地媳妇，怕语言不通；南方人和北方人结合，会担忧饮食习惯不合导致冲突。

其实，这种地域文化的差异是由于生活经历不同，你不知道这些可能性，但这不一定是你们的差距，就像南方人没见过北方的暴雪和暖气，北方人没吃过各种精致小菜一样。

当我们第一次见家长就面临这种差异时，有一组操作公式可以很好地应对，那就是：**好奇 + 赞美 + 尊重**。

这是指要对一切差异保持礼貌的好奇心，适时表示出自己的赞美，还能做到彼此尊重。简单来说就是接受差异，拥抱改变。

*   **例如**：你第一次去男友家吃到地道的北方饭菜，就可以说：“这是我第一次吃到这么正宗的锅包肉”，或者“这饺子怎么做的这么好吃？阿姨可以教教我吗？”展现出自己对男友家乡美食的赞美和好奇，从而获得家人的认可。
*   **如果你不能适应**：可以委婉地说“我吃的不太习惯”，而不是流露出厌恶的表情或直接说难吃，这样只会让对方难堪。
*   **当然，如果你可以克服一下自己的固有习惯**，大胆尝试新口味，就能够让自己获得更多的好感。

在面对地域文化的差异时，想做到入乡随俗，或者真正去接受新东西，并没有想象中那么难。回忆一下，我们小时候不管是升学还是换新班级，都会很紧张，但每次都还是顺利地融入了集体。现在我们已经是成年人，对事物的接受能力更强，去吃从未吃过的食物，向从未见过面的人嘘寒问暖，根本不是难事。而且努力尝试他们的饮食，适应当地的风俗习惯，还可以让你和对方家长产生新的链接，从而使关系更加亲密。

**第三种：生活方式的差异**
这种差异往往由文化水平的不同导致，表现为做事方法、兴趣爱好和审美情趣的不同。如果两个人或两组家庭的生活方式区别比较大，就会造成各种矛盾的出现，例如家庭计划不一致、生活习惯不同以及聊天话题比较少。

*   **例如**：在美剧《生活大爆炸》中，莱纳德全家都是高学历，但他的女友佩尼学历较低，因此他妈妈觉得聊不到一起去。这就是两个家庭文化水平差异导致的沟通矛盾。

面对这样的差异，我们该如何与对方父母交流呢？物理学上有个规律叫**同频共振**。用在人际交往中，就是人与人之间如果能够主动寻找共鸣点，使自己的“固有频率”与别人的“固有频率”相一致，就能够增进情感，结成亲密关系。

要让你和伴侣的家庭产生同频共振，你可以尝试以下两种方法：

*   **技巧一：同频尝试**。这是指靠近对方父母的世界，做他们喜欢的事。
    *   例如：对方父母知识水平高，可以尝试送精装书籍或文玩字画；他们喜欢看电视剧，可以聊聊最近的新剧；对方父母喜欢旅游，一套舒适的冲锋衣可以作为礼物；对方妈妈喜欢打麻将，那就上桌陪着打几圈。这些方法可以加深了解，让他们对你的初始印象更加良好。
*   **技巧二：创造共振**。这是指一起去做大家都没有做过的事情，让你们之间产生全新的链接。
    *   例如：将见面地点选在都没去过的旅游胜地；一起做手工；一起尝试做从来没做过的菜。通过共同的新鲜体验和交流，让思维产生共鸣和碰撞，从而让感情的基础更加稳固。

好了，今天的课到这里就结束了，我们来总结一下吧。

本节课我们一起学习了面对两个家庭差异的三个主要内容：
1.  面对**家庭条件的差异**时，用**定式效应**给自己预设良好心态，展示不卑不亢的大方形象。
2.  面对**地域文化差异**时，向家长表示出**好奇、赞美和尊重**，让他们更加喜欢你。
3.  面对**生活方式的差异**，要通过和家长产生**同频共振**，寻找共同话题，他们就会和你更加亲近。
