# 女神养成系列：09：夏日晒伤妆

在本节课中，我们将学习如何打造一款充满夏日气息的“晒伤妆”。这款妆容的重点在于运用独特的W型腮红，营造出仿佛被阳光亲吻过的自然可爱感。课程将分为理论认识和实际操作两部分，确保你能掌握其核心技巧。

## 第一部分：夏日晒伤妆的认识

上一节我们介绍了妆容的整体概念，本节中我们来看看夏日晒伤妆的具体特点。这款妆容源自时尚潮流，其最显著的特征是腮红的画法。

以下是夏日晒伤妆的几个核心特点：
*   **底妆**：要求妆感自然且持妆久，以适应夏季炎热易出汗的环境。
*   **眉毛**：眉形自然，颜色与发色协调，遵循“前虚后实”的原则。
*   **眼妆**：通常使用橘色系眼影，眼线需精致自然，睫毛要卷翘。
*   **唇颊**：这是妆容的灵魂。采用**橙色系的W型腮红**，并可在脸颊点缀少许小雀斑，以增加俏皮感。口红颜色需与腮红相呼应。

这款妆容并非追求刻板的精致，而是突出一种健康、自然、不做作的少女感。其重点公式可以概括为：**常规眼眉画法 + W型腮红 + 同色系唇妆**。

## 第二部分：妆容实操演示

了解了理论知识后，我们现在通过为模特改妆，来实际演示如何一步步完成夏日晒伤妆。本次实操将在模特原有妆容基础上进行调整和强化。

以下是具体的操作步骤：
1.  **强化眼影层次**：模特原有眼影颜色较浅。使用更深的红棕色眼影，从睫毛根部向上小范围晕染，并加深下眼影，使眼睛更有神。
2.  **调整眉形**：用眉刷将眉头的颜色向前自然晕染，避免线条过于刻板，打造圆润的眉头，使眉毛更柔和。
3.  **处理睫毛**：夹翘睫毛后，以“Z”字形从睫毛根部向上刷睫毛膏，打造根根分明的效果。
4.  **补充眼线**：沿睫毛根部补充外眼线，确保线条流畅顺滑。
5.  **绘制W型腮红（核心步骤）**：使用腮红刷，以打圈晕染的方式，从一侧颧骨开始，向下经过鼻梁上方，再连接到另一侧颧骨，形成一个“W”形状。注意颜色要自然，不宜过重。可先用浅色打底，再用偏橘的哑光腮红叠加，增加层次感。
6.  **面部提亮**：在鼻头、山根、眉骨下方、眼下及下巴等部位涂抹高光，提升面部立体感。眼下高光需与腮红自然融合。
7.  **修饰鼻侧影**：从鼻头两侧向山根方向晕染阴影，使鼻梁更显立体，并让面部妆容的衔接更整体。
8.  **整体修容与定妆**：轻扫修容粉修饰面部轮廓。最后使用散粉进行全脸定妆，确保妆容持久。如果腮红过重，可用海绵蛋沾取少量粉底轻轻遮盖，再定妆，能营造出腮红从肌肤透出的自然效果。
