# 简历优化课程：02：黄金吸引术（上）：拆解500强企业偏爱的简历架构

在本节课中，我们将学习一个关键概念——**体验地图**，并运用它来理解HR审阅简历的心理路径。我们将重点拆解HR在审阅简历时最初关注的两个核心部分：**基本信息**与**求职意向**。

上一节我们介绍了如何通过自我认知模型挖掘个人优势。本节中，我们来看看如何从企业HR的视角出发，设计一份能迅速吸引注意力的简历。

## 理解“体验地图”

**体验地图**是商业设计中的一个重要概念。它描绘了用户进入一个场景或使用一个产品的典型路径，以及在各路径节点上的行为与期待。

例如，你去星巴克买咖啡的体验地图可能是：发现店面、推门而入、感受环境、去吧台排队、浏览菜单、对比选择、购买付款、选择座位、饮用、离开。在这条路径的每一步，你都有特定期待并做出相应反应。

再例如，使用一款陌生人社交软件时，你通常希望先通过照片、个人资料等信息初步认识陌生人。如果判断可能合得来，就会想进一步了解甚至发起对话。软件的设计，如首页展示、切换按钮、详情页内容，都旨在满足这条体验路径上的需求。

从以上例子可以看出，体验地图的核心思维是：**你必须站在需求方的视角去设计产品**。

那么，HR审阅简历的体验地图是怎样的呢？

## HR审阅简历的体验地图

经过对大量企业HR的访谈，我们绘制了HR审阅简历的动态地图。这条地图共有5个关键步骤。本节课，我们先探讨前两个步骤。

### 第一步：了解应聘者基本信息

这与陌生人社交的初始心理行为一致。HR需要通过一些基本信息快速了解陌生的应聘者。

以下是HR在此步骤中通常会关注的信息点：

*   **照片**：几乎所有HR都会第一眼关注求职照。这并非因为HR是“颜控”，而是当文字与图像同时出现时，人的视觉会本能地优先处理图像。简历上的照片能为HR留下立体化的个人印象。因此，简历中应当放置照片。
    *   **公式**：`简历吸引力 = 文字信息 + 视觉印象（照片）`
*   **关键文字信息**：文字形式的基本信息作用直接。一是加速初步筛选（如匹配学历、工作地要求），二是获取联系方式以便后续沟通。

在基本信息中，必须包含以下元素：

1.  姓名
2.  性别
3.  最高学历
4.  专业
5.  工作年限
6.  现居住地
7.  联系电话
8.  邮箱

**关于简历照片的注意事项**：应使用本人近期的免冠职业证件照。避免使用宠物照、卡通图、风景照、旅拍照、过度美颜的自拍照，这些都无法有效展示职业素养。

### 第二步：明确求职意向

大多数HR反馈，他们希望打开简历后能立刻、清晰地知道求职者应聘的是哪个岗位，而不是去猜测。

你可以从以下两个位置体现求职意向：

1.  **简历文件命名**：通过招聘网站投递时，简历文件名常默认为邮件主题。在文件名中加入意向岗位名称，能让HR在打开前就清楚你的目标。
2.  **简历内部显要位置**：在简历正文靠前部分明确写出求职意向。

撰写求职意向时，需要掌握两个技巧：

*   **单一性**：一份简历只针对一个明确的岗位方向。避免写“销售或编辑”这类模糊表述，这会让HR认为你目标不清、稳定性可能较差。如果尝试不同方向，应准备多份简历。
*   **精准度**：意向岗位应具体到级别和细分领域。
    *   **不佳示例**：`求职意向：运营` 或 `求职意向：设计`
    *   **推荐示例**：`求职意向：用户运营主管` 或 `求职意向：UI设计师`

## 本节课总结

本节课中，我们一起学习了以下核心知识点：

1.  **体验地图**是商业设计中的重要概念，指用户进入场景或体验产品的惯性路径。按照体验地图设计，能提升需求方的兴趣和使用深度。
2.  HR审阅简历的体验地图包含5个重要步骤，本节课介绍了前两步：**应聘者基本信息**与**求职意向**。
3.  基本信息中必须包含：照片、姓名、性别、最高学历、专业、工作年限、现居住地、联系电话和邮箱。简历照片应使用本人近期免冠职业照。
4.  求职意向是HR打开简历后想确认的第一件事。撰写时需确保意向岗位的**单一性**和**精准度**。
