# 瑜伽理疗入门：01：告别含胸圆肩的肩颈练习 🧘‍♀️

在本节课中，我们将学习一套针对肩颈问题的理疗瑜伽序列。这套练习旨在通过循序渐进的温和动作，帮助改善肩周炎、圆肩驼背和颈椎前引等问题，逐步建立身体的稳定性并打开关节。

## 课程概述

肩颈理疗练习分为三个阶段，旨在逐步改善肩周炎与颈椎问题。常见的肩颈问题包括肩周炎、圆肩驼背、颈椎前引以及椎间盘突出。通过循序渐进的练习，可以慢慢建立身体稳定性，打开关节，从而缓解不适。

## 第一部分：肩部激活与放松

首先，我们通过一个简单的活动来学习如何放松肩膀。许多人的肩膀长期处于耸肩圆肩的紧张状态。

以下是肩部环绕练习的步骤：

1.  采取舒适的坐姿，保持身体放松。
2.  双手轻搭在双肩上。
3.  吸气时，手肘向上抬起，尝试让手背在颈后相碰。
4.  呼气时，手肘向下落，尽量靠近肋骨两侧。
5.  按照自己的呼吸节奏，缓慢重复此动作10到15次。

**注意要点**：身体保持垂直于地板，骨盆摆正，避免向前倾或塌腰。肋骨内收，肩膀放松。对于肩周炎患者，手臂抬不到标准位置也没关系，能抬到哪就抬到哪，但需注意手肘尽量向外展开，不要向前扣。

完成上述练习后，我们进行下一个肩部开合动作。

以下是肩部开合练习的步骤：

1.  将双手手肘在胸前相碰。
2.  吸气时，手肘向上并向两侧展开。
3.  呼气时，手肘向下收回，同时拱背低头。
4.  按照呼吸节奏，重复此向内合拢、向外打开的动作15次。

**注意要点**：每次呼气打开胸腔，每次呼气时身体向前拱背。保持动作缓慢，确保练习效果。

完成15次后，在最低点保持手肘相碰，然后开始进行肩部环绕。

以下是肩部环绕练习的步骤：

1.  吸气时，手肘向后、向上环绕，尝试让手背在颈后相碰。
2.  呼气时，手肘向前、向下环绕。
3.  缓慢地、有控制地重复此环绕动作15次。

**核心要领**：动作越慢，对肩膀的激活和放松效果越明显。避免快速、无效的晃动。

## 第二部分：脊柱扭转与侧伸展

上一节我们活动了肩关节，本节中我们通过扭转来伸展脊柱和侧腰。

首先进行坐姿扭转。

以下是坐姿扭转的步骤：

1.  手臂向两侧平举。
2.  呼气时，右手带动身体向右侧伸展，左手放在右膝上。
3.  吸气，身体向上延展；呼气，收腹，身体向后扭转。
4.  保持双肩平行于地板，不要耸肩。坐骨稳定向下扎根。
5.  吸气，后侧手臂向上；呼气，身体向左侧伸展，保持身体呈一个平面。
6.  感受侧腰的伸展，保持顺畅呼吸。
7.  吸气身体回正，换反方向重复。

**关键点**：扭转时，下侧肩膀向前推，上侧肩膀向后打开，以加深扭转。保持臀部均匀坐于垫面。

接下来，我们进行一个简单的伸展来整合上半身。

以下是手臂上举伸展的步骤：

1.  双手十指交扣，翻转掌心向上。
2.  吸气时，手臂带动身体向上延展。
3.  呼气时，双肩主动向下沉。
4.  保持肋骨内收，肩膀放松。肩周炎患者可根据自身情况调整手臂高度，但务必避免耸肩。

## 第三部分：灵活胸椎与猫牛式

圆肩驼背往往伴随胸椎僵硬。下面这个练习能有效灵活胸椎。

以下是后仰支撑的步骤：

1.  双手放在身体后侧，指尖朝前，掌根压地。
2.  身体微微后倾，膝盖指向脚尖方向。
3.  吸气时，胸椎向上提，抬头，下巴微收。
4.  感受胸腔打开，肩膀向后展开，颈椎后侧有舒缓感。
5.  注意抬头时眼睛看斜上方即可，不要过度后仰给颈椎造成压力。

现在，我们进入经典的猫牛式，进一步灵活整条脊柱。

以下是猫牛式的步骤：

1.  四足跪姿准备，手腕在肩膀正下方，膝盖在髋部正下方。
2.  脚背平行朝后，向下压地。
3.  **吸气**：抬头，胸椎向前推，尾骨向后向上，肩胛骨向后拉，伸展脊柱前侧。
4.  **呼气**：低头，拱起上背部，眼睛看肚脐，腹部内收。
5.  按照自己的呼吸节奏，缓慢流畅地重复15次。

**正位要点**：双手肘窝相对，避免肘关节超伸。重心均匀分布在四肢，避免手腕疼痛。动作的重点在于胸椎的充分屈伸，而非腰椎的过度挤压。

## 第四部分：颈部针对性拉伸

最后，我们针对紧张的颈部肌肉进行拉伸。颈椎前引会导致颈前侧肌肉紧张、后侧肌肉缩短。

首先拉伸颈侧肌肉（斜方肌上束）。

以下是颈侧拉伸的步骤：

1.  身体坐直，右手放在左耳外侧。
2.  吸气延展脊柱，呼气时手臂轻轻带动头部向右肩方向倾斜。
3.  保持下巴微收，不要前探。感受左侧颈部的拉伸。
4.  保持30秒后，换另一侧。

接下来拉伸胸锁乳突肌。

以下是胸锁乳突肌拉伸的步骤：

1.  右手手指按住左侧锁骨上方的两根筋（胸锁乳突肌下端）。
2.  吸气准备，呼气时头部向斜后方缓慢转动并后仰。
3.  感受颈部前侧的拉伸感。保持30秒后换边。

最后，放松非常容易紧张的枕下肌群。

以下是枕下肌群放松的步骤：

1.  双手大拇指找到后脑勺下方头骨边缘的两个凹陷处。
2.  拇指按住该点，吸气时身体向上。
3.  呼气时，微收下巴，想象头顶向前方水平延展，感受拇指按压点深处的拉伸。
4.  保持呼吸，放松双肩。

## 课程总结

本节课中，我们一起学习了从肩部激活、脊柱扭转、胸椎灵活到颈部拉伸的一系列理疗动作。虽然动作简单，但关键在于练习的质量：动作要缓慢到位，呼吸要配合流畅，并且需要持之以恒地练习。对于初期感到疼痛的部位，应量力而行，在自身可承受的范围内逐步加深练习。坚持正确的练习，才能让身体真正受益，逐步告别含胸圆肩的困扰。
