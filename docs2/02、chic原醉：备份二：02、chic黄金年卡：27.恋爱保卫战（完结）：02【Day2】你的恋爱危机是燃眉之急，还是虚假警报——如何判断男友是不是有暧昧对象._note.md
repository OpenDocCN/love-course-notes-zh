上一节我们介绍了课程的整体框架，本节中我们来看看影响我们对恋情判断的两个主要心理成因。

## 没有安全感的根源：成长经历的影响

新精神分析派心理学家艾里克森认为，人在0至2岁的阶段会经历基本信任与不信任的心理冲突。如果婴儿时期的基本需求没有得到父母及时、耐心的回应与满足，成年后就容易缺乏安全感，难以信任他人。

在电视剧《都挺好》中，苏明玉的角色是典型代表。由于童年缺乏关爱，她难以建立对亲密关系的信任。她的台词“钱比男朋友更能给我带来安全感”触动了许多有相似经历的人。

人们常通过两种方式应对这种缺失：
*   一些人会寻找替代品来掩盖真实需求，例如像苏明玉一样成为工作狂。
*   另一些人则会在伴侣身上寻求补偿，期待对方扮演“完美父母”的角色，从而经常陷入“他是不是不爱我”的焦虑中。

**纠正方法**是认识到伴侣无法为你的幸福负全责。你需要学会自我关爱，正视内心需求。可以尝试与“内心的那个小孩”对话，允许情绪自由表达，做自己的支持者。

了解了安全感缺失的成因后，我们再来看看另一个常见的认知误区。

## 认知的偏差：行动者与观察者效应

当我们对社会事件或他人行为进行解释时，很容易犯**归因偏差**。这背后是**行动者与观察者效应**在起作用。

**公式**：对自身行为，我们倾向于做**外部归因**（如：我迟到是因为堵车）；对他人行为，我们倾向于做**内部归因**（如：他迟到是因为不上心）。

例如，如果你拖延制定旅行计划，你可能会归因于工作太忙（外部归因）。但如果是男友拖延，你可能会认为他对旅行不上心（内部归因）。

**避免方法**是在下判断前尽可能多地收集信息。在亲密关系中，这意味着要加强沟通。当发现男友行为异常时，可以尝试询问：“最近是不是工作太忙？需要我帮忙吗？” 这能避免武断判断，并增进感情。

## 识别暧昧：超越表面行为的本质洞察

网络上流传着各种识别男友出轨的“经验帖”，例如手机不离身、突然爱发朋友圈等。然而，高明的男生会进行伪装，例如修改备注、删除记录，就像电影《前任3》中的余飞那样。

更本质的表现是：**男生将大量注意力从你身上转移到了其他女生身上**。当他对某个女生产生暧昧兴趣时，会像沉迷新游戏一样投入时间和情感。由于人的情感分配有限，投注给你的部分自然会相对减弱。

从科学角度看，女性的**情感洞察能力**通常比男性更强，能察觉细微的情绪变化。乔布斯的妻子劳伦娜曾形象地描述这种感觉：当他的关注点转移时，“你就会觉得人生非常黑暗”。

**关键判断**：当你持续感觉到男友投注给你的情感在减少，往往意味着出现了情感裂缝。此时应提高警惕，观察他身边是否出现了新的异性，并冷静收集信息。

## 暧昧期的男生心理：理性与感性的权衡

根据经济学中的**理性经纪人**假设，人在做决定时会趋利避害，追求利益最大化。男生身处一段稳定关系中并对其他女生产生好感时，通常不会立即背叛，而是会进行理性权衡。

**代码**表示其决策逻辑：
```python
if (道德压力 + 现有情感链接) > (新好感 + 新关系潜在收益):
    维持现状或停留在暧昧
else:
    可能选择行动（如出轨或分手）
```

简单来说，男生停留在暧昧层面，是因为尚未决定是否要为一时好感牺牲现有稳定关系。**这个阶段是拉回男友的最佳时机**。

一旦暧昧对象给予明确回应，男生可能进入非理性状态，挽回难度会**指数级增加**。因此，早期发现并遏制萌芽，是成本最低的做法。

## 你的优势：情绪价值的不可替代性

男生对女友和暧昧对象的评价标准往往不同：
*   对**女友**，更看重**软性价值**，尤其是**情绪价值**（如理解、支持、舒适感）。
*   对**暧昧对象**，更看重**硬性社会价值**（如学历、外貌、视野）。

只有当暧昧对象的硬性价值明显超越你，**并且**能提供足够情绪价值时，男生才会真正动摇。这意味着，相比暧昧对象，你在提供**稳定情绪价值**方面拥有天然优势。只要你能持续输出稳定的积极情绪，就能牢牢掌握关系主动权。
