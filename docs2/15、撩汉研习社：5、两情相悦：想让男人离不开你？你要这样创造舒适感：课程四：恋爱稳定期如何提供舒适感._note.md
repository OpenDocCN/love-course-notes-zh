# 撩汉研习社：5：两情相悦：想让男人离不开你？你要这样创造舒适感

## 课程四：恋爱稳定期如何提供舒适感 💑

在本节课中，我们将要学习当感情进入稳定期后，如何通过创造舒适感来维系长久的关系。舒适感是决定关系能否持续的关键因素。

上一节我们讨论了亲密关系后的特别期，本节中我们来看看关系进入稳定模式后，如何系统地提供舒适感。

当两个人的感情进入稳定期，相处能否长久，就是一个保持舒适感的大命题了。这是第四个阶段。你们的生活处于一种模式，已经有一段时间了，好比电影《爱情呼叫转移》开始的那段自白：从现在开始往后每一分钟，我都知道会发生什么事情，他正在做一样的炸酱面，看一样无聊的电视剧。

这就是过了激情期以后的恋爱稳定期。这个时候需要提供给对方相当多的舒适感。今天的课程我们就来从以下四点教你稳定期创造舒适感的方法。

以下是创造舒适感的四个核心方面：
1.  创造精神上的舒适感。
2.  创造生活习惯上的舒适感。
3.  创造消费习惯上的舒适感。
4.  学会处理矛盾和建立信任感。

下面我们就来逐一看看这四点内容。

### 一、精神上的舒适感 🧠

有些话，男人只能和你说，只有你听得懂，他会很幸福，怎么会离开你呢？

在《琅琊榜》里面，静妃娘娘一直超然于物外，结果皇上却越来越喜欢她，在她这里能够安心的睡一会儿，得到最好的舒适感。有一次，皇帝处理完太子在宫中饮酒作乐的事情，已有废太子的意思，但不忍心杀他，只是幽闭。皇帝头很痛，跟静妃闲聊：“朕真的老了，是不是？”静妃悠悠的说：“皇上想想太皇太后的高寿，就知道自己不老，何必胡思乱想呢。”

皇上接着说：“今日之事若发生在朕壮年之时，处置起来，当不会是这个局面。可见人老了，心就软了。”而静妃则继续说：“臣妾不知道太子做错了什么，但觉得陛下未动雷霆之怒，当属为父之慈，与年龄无关。”

听听这话，又贴心，又让男人有满足感：自己不老，还是关爱儿子，值得颂扬，特别有面子有里子。

再比如我们日常生活里遇到的场景：男朋友和你约好了去共进晚餐，临赴约前突然他哥们儿杀出来，男人一般都不好意思取消女朋友的约会，又想和哥们儿一起打台球，他害怕女朋友不高兴，很纠结。不善解人意的姑娘会说：“这有什么好纠结的，是你兄弟重要还是我重要？我不管你答应陪我的，你今天必须得陪我。”

而善解人意的姑娘会给他个台阶下，大方地说：“亲爱的，要不然你和哥们去打台球，我在旁边找个地方按摩，等你打完过来找我，咱们去看电影啊。”

善解人意且有心机的姑娘会说：“亲爱的，不如我陪你们一起去打台球吧。你不用管我，我就在你身边陪着你就好了。”

总之，要做到不过分粘人、没有压力、频率一致、有精神共鸣。

女生谈恋爱之后，通常比男生更有依赖感，可能会比较粘人，但一定要控制这个度，过分粘人会让男人喘不过气。**轻松是舒适感的重要组成部分**。想让男人跟你在一起的时候感觉很轻松，就不能给他压力。

而**频率一致**，是指当他想看书的时候，你就安静的拿起一本书陪伴，而不是粘着他跟你看电影，也不是在一旁拖地板、转动洗衣机制造声响。当他想要看喜剧电影的时候，你就抱着一桶爆米花跟他一起开怀大笑。这样就会让他觉得和你特别合拍，精神上很有舒适感。

如果他想看书的时候你想唱K，他想看电影的时候你想出去逛街，他想工作的时候你想轰趴，你为了舒适感总是要迁就男人的节奏，要么你们就各自做各自的事情，等你们都想做一件事的时候再在一起，要么你们就应该分手了。两个频道如此不同的人在一起，只能是互相折磨。

**频道一致**，是他想看书的时候你也想看，你周末想逛街的时候他也想出门散步。频道一致的人才能够保持长久的舒适感。**千万不要为了满足男人的舒适感，就忽略了你自己的舒适感**。恋爱是让你更开心、更舒服，而不是找一个人让他舒服。

### 二、生活习惯的舒适感 🏠

男人的爱情观和女人不一样，女人是“遇到对的人，所以我想谈恋爱了”，而男人是“我想安定下来了，然后找一个他喜欢的合适的女人”。所以为男人营造家的感觉，特别重要。

俗话说的好，抓住一个男人的心首先要抓住男人的胃。比如学习做几道家常风味，学习做几道甜品。不会做饭的妹子怎么办？选择简单又容易做好的菜式。

有一个女学员跑过来跟我说：“我做西红柿炒鸡蛋怎么样？”你要知道，中国菜式里想做的好吃，最难的就是西红柿炒鸡蛋。为什么有的人喜欢吃甜的西红柿炒鸡蛋，有的人喜欢咸的？主要是这个菜还体现不出你的水平。

所以聪明的做法就是选择最容易做的好吃的，那是什么呢？**炖鱼**。鱼总不可能是甜的吧？如果是清蒸鱼呢，有些人会觉得太淡了不好吃，所以炖鱼是最好的选择。

**炖鱼简易食谱**：
1.  鱼要足够新鲜，买的时候在菜市场让商贩给处理好。
2.  锅里放上水，里面放上葱、姜、花椒、大料、酱油，一点白糖。
3.  水开之后，把鱼放进锅里，水要没过鱼。
4.  中火炖20分钟。

炖鱼总不会有甜口的吧，只要你按步骤做就不会太难吃。如果还是觉得难，那就做牛排拌沙拉或者是披萨。超市都有半成品的食材，把香肠、菜、奶酪往披萨饼底上一摆，烤箱烤20分钟就好了。淘宝上简易烤箱才150块就能搞定。

早餐给他做点蛋挞，淘宝上有卖蛋挞皮、蛋挞液的，你只需要把蛋挞液倒到蛋挞皮里，烤20分钟就可以了。

而且中国男性普遍认为会做西餐的姑娘比较“高大上”，就跟我们认为银行、律师等行业的男人高大上是一样的，没有什么太具体的理由，就是觉得高大上。这么高大上的女朋友，男人舍得请你离开吗？

而且当你们俩的关系进入稳定期的时候，你的喜好可以循序渐进渗入生活当中。比如我老公平时会买很多关于赵本山的碟，也搜集了很多相关的小品，只要他休息，都会把时间安排来好好的欣赏一出二人转。这样我就知道他喜欢看赵本山、喜欢看二人转了。

我老公在空闲的时候经常会泡一杯茶嗑瓜子儿，而且是那种小小的特别难嗑的瓜子儿，每次我看他嗑的都可开心了。那么我就知道他喜欢喝茶、嗑小小的瓜子儿。我平时会在她快吃完瓜子儿的时候给他“囤货”。过年期间，我在深圳度假，老公给我打来电话说要来，我立刻就入手一堆她喜欢的瓜子，给做好迎接她的准备。

我们的相处模式，就是我们要相互知道对方的喜好，并且用对方喜欢的模式去相处。这是为什么有的人会和老公感情一直这么好。

当然，接下来可能你老公也会在车子限行的时候，在车窗上给你贴一个小提示，让你别开这个车；也会买好你喜欢的乐团演出票，并且当天还会发微信提醒你；也会跋山涉水的为你准备好你喜欢吃的东西。这就是舒适感，其实也是默契，尤其是在婚姻当中特别需要。

### 三、消费习惯的舒适感 💰

花钱上统一步调达成默契，给他一个会过日子的你。

既然是奔着成家立业去的，你得让他知道你会过日子。当然，这个时候需要你们已经确定关系，并且见过家长了，你才可以这样做。

你可以带他去逛超市，在他面前展现你勤俭持家的一面。比如你买一包巧克力，你得对比一下同牌子散装的、有包装的，哪种牌子的哪种净含量高、哪种划算。当然你的男朋友可能会说“哎呀，这个不用算”，但是你要做给他看，这就是勤俭持家。

又比如约会看电影吃饭，提前买团购券，一般打折都蛮厉害的。尤其是吃饭点菜的时候，一般男人都比较不拘小节，容易点多了。这个时候你就该提醒着说：“菜差不多够了，别点太多，吃不了，该浪费了。”

就这么简单的一句话，男人在心里会给你下定论：你是个会过日子的女人。试想一下，一个月5000的生活费活出2万的质量，有哪个男人会不想和这样的女人过日子呢？

### 四、处理矛盾和信任感 🤝

当感情稳定，面临结婚的时候，往往会遇到彩礼的问题。我就听过一个姑娘讲述快结婚的时候，因为彩礼自己的父母和男朋友的父母闹翻了。一怒之下，她说了分手，等到男朋友重新去相亲，她又接受不了了，哭着要复合。这样分分合合，感情几经折腾，她特别的苦闷。

我用一句很俗的歌词来诠释这个事情：**爱情不是你想买，想买就能买**。他家的大门，也不是美容会所，随时向你敞开。你对他有算计，他对你就会有防备。人都不傻，在你精打细算的时候，他也在衡量你的真心。你先颠覆了男人在你身上的爱情幻想，你让他看到你并不是愿意与他甘苦与共的那一个，你随时可能因为现实问题离开他，这是特别典型的不懂得与男人相处的舒适感的表现。

而面对争论和吵架的时候，怎么做才是释放舒适感呢？

我有一个朋友，她跟她的老公吵架有一个原则。她老公的嘴特别硬，每回都会说“我怎么可能错呢？”他老婆就跟他讲：“对对对，你每回都对，但我就是怎么怎么不开心，你就要怎么怎么哄我。”先把情绪摊开了，大家表达出来。然后两个人冷静的分析怎么处理，理性的拿出一个方案来。这样不管是大事小情都能够相互信任，把矛盾化于无形。

舒适感是让姑娘们在男人心里霸占一个不可取代的位置，让他离不开你。当他想安定下来成家立室的时候，成为他老婆的第一个人选，就是你。舒适感不是没有自尊，不是讨好他，是在感情里面占据主导的位置。别以为男人活得很糙，再糙的男人也会选择一个让她舒服的女人来共度一生。学会不同阶段的舒适感营造的技巧，真诚沟通，彼此滋养，用这些小心机，让他离不开你吧。

## 课程总结 📝

好了，到这里，我们整个系列课程就已经接近尾声了。下面我们来对本期课程的主要内容做一个总结。

**第一节课：如何在认识初期给男人制造舒适感。**
这节课我们一共讲到了4点：
1.  认识初期制造舒适感的6个注意事项。
2.  给男人一个善解人意的你，在外面给足男人面子，让男人觉得自己捡到了宝。
3.  给男人传递正能量，给他描绘出你们未来的蓝图，让他向往跟你在一起之后的生活。
4.  绝对不能“作”。在认识初期太“作”了，只能让男人对你敬而远之。

**第二节课：热恋期如何制造舒适感。**
这节课我们一共讲到了两点：
1.  让男人感受到你浓浓的爱意，学会跟男人说情话，在有机会的时候就多夸对方。还有就是千万不能太“作”。
2.  给他一个会安慰人的你。当男人出现不良情绪的时候，引导他找到情绪的出口。在他事业的低谷期给他需要的支持和陪伴。

**第三节课：发生亲密关系之后的特别期，如何制造舒适感。**
我们一共讲到了4点：
1.  **对爱的表达**：发生关系之后，不要急于表达出要男人为你负责的态度，不要为他套上道德枷锁。用更委婉和温柔的态度，间接表达你想跟他在一起的想法，会给男人更多的舒适感。
2.  **对性的表达**：可以从准备一场浪漫的、充满激情的约会开始。
3.  **处理不安全感**：这个阶段，女人的不安全感会增多。很多女生由于缺乏安全感，都会有太多明显的和隐形的“作”。比如问让对方陷入两难的问题，经常由一件小事的吵架都能够上升到“对方爱不爱你”，这些都是万万不可取的。女人的安全感是从自我价值的提升和学习让男人离不开你的恋爱技巧当中得到的。
4.  **对婚姻的态度**：这个时候对婚姻和承诺应该是不紧逼的态度，给男人创造一些美好的画面感，让他去憧憬和你在一起之后的生活，而不是一味的给他压力和逼婚。

**第四节课：恋爱稳定期如何提供舒适感（本节内容）。**
舒适感的创造可以从以下4点入手：
1.  **精神上的舒适感**：做贴心、善解人意的另一半，不要过分的粘人，双方保持频率一致，减少给他的压力，达到和他精神上的共鸣。
2.  **给他营造家的感觉**：比如你的好厨艺，了解对方的喜好，并且用对方的模式去相处。
3.  **消费习惯的舒适感**：花钱上统一步调达成默契，给他一个会过日子的你。
4.  **处理矛盾和信任感**：遇到矛盾和冲突的时候，学会冷静分析和处理，把信任作为根基，把矛盾化于无形。

爱是本能，爱也是后天可以学习的一种能力。恋爱很简单，只是你没有学习，不会而已。

本节课中，我们一起学习了在恋爱稳定期从精神、生活习惯、消费习惯以及矛盾处理四个方面创造舒适感的方法。掌握这些技巧，能帮助你在长期关系中占据主动，让彼此的关系更加稳固和幸福。
