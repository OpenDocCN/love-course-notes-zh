# 中医美容：48：调养气血，实现“白里透红”的食疗古方

## 概述
在本节课中，我们将学习如何利用冬季时节调养气血，并介绍一个源自药王孙思邈《千金方》的食疗古方，帮助不同肤质的朋友由内而外地养出健康、红润的好气色。

## 健康肤色的标准
上一节我们提到了调养气血的时机，本节中我们来看看什么是健康的肤色。中国人的基本肤色是黄色，其中存在偏白或偏棕的个体差异。然而，如果偏白的皮肤显得惨白，或偏棕的皮肤显得阴暗发乌，这通常表明气血亏虚，身体不健康。真正气血充盈、健康的表现是 **白里透红** 的肤色，这是一种自然的健康美。

## 一个真实的案例
我曾有一位年近40岁的女性朋友前来门诊。她起初希望我能通融，为她注射一些药物。这些药物包括维生素、保肝药、血管扩张剂以及能降低皮肤毛细血管血流量的药物——即所谓的“美白针”。我婉拒了她的请求，因为滥用药物，尤其是没有医生指导的情况下，可能对身体造成伤害，导致皮肤呈现不健康的惨白，并伴随头晕、身体虚弱等副作用。

## 安全有效的古方介绍
鉴于药物美白的风险，我向她推荐了中医的方法。药王孙思邈在《备急千金要方》中记载了一个方子，能安全有效地改善肤色。

以下是该方子的基本组成与配比：
```
橘皮（或陈皮）: 200克
桃花（阴干）: 100克
白瓜子仁（即炒熟的南瓜子仁）: 50克
```

## 方剂的制作与使用方法
我的这位朋友决定尝试这个古方。以下是具体的制作与服用步骤：

1.  **准备材料**：将橘皮、阴干的桃花和炒熟的白瓜子仁准备好。
2.  **加工成粉**：将上述三味材料混合在一起，加工成细粉。
3.  **分装储存**：将加工好的细粉分装成每包约6克的小包。每次服用1包，每日两次。也可以一次性配置500克左右，供一个月服用，注意用保鲜膜包好冷藏保存。
4.  **服用调整**：
    *   若追求美白效果，可每次额外增加5克南瓜子仁。
    *   若有便秘问题，可适当增加2-3克桃花粉的量。

## 方剂原理与成分解析
服用一个月后，我的朋友对比照片发现肤色明显改善，显得健康且有活力。这个方子的原理如下：

*   **橘皮/陈皮**：主要作用是**理气**。中医认为气统帅全身脏腑，气机通畅，脏腑功能才能正常运行。
*   **桃花**：并非因为其颜色而让人面色红润，而是因为它具有**通泻、调理胃肠**的功效。中医讲究“六腑以通为用”，代谢通畅才能排出旧废物，吸收新营养。
*   **白瓜子仁（南瓜子）**：富含蛋白质、维生素E、微量元素等。炒熟的南瓜子有美容功效，能**调补脾胃**，同时兼具杀虫、解毒等辅助作用。

## 注意事项与体质调整
部分女性服用后可能出现腹泻，这通常与桃花通泻的药性有关，也和个人体质密切相关。

以下是针对不同体质的调整建议：
*   **易腹泻者**：多为阴虚体瘦、脾胃功能较弱的人群。可将配方中桃花粉的量从100克减少至50克或更少。
*   **便秘者**：多为体胖、常有便秘困扰的人群。可按原方或适当增加桃花粉量，正中下怀。

**重要提示**：现代人体质与古人不同，药材质量也需考量。若服药后出现腹泻并伴有发烧，应立即停药并就医。切勿为了美容而损害健康。

## 总结
本节课我们一起学习了药王孙思邈的一个美容妙方。通过将**橘皮（陈皮）、桃花、南瓜子仁**按比例碾粉服用，每日两次，长期坚持可以温和地调理气血，改善肤色，达到“白里透红”的效果。记住，美丽应建立在健康的基础之上。
