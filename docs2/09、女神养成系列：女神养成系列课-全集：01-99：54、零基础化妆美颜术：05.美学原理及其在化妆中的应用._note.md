# 零基础化妆美颜术：05：美学原理及其在化妆中的应用

## 概述

## 色彩原理

上一节我们介绍了面部形状分析，本节中我们来看看色彩的基础知识。色彩是化妆中塑造氛围和调整肤色的关键工具。

自然界中看似无穷的颜色，都源于最基础的**三原色：红、黄、蓝**。将三原色两两混合，可以得到**三间色：橙、绿、紫**。

```
红 + 黄 = 橙
黄 + 蓝 = 绿
红 + 蓝 = 紫
```

将这六种颜色继续两两混合，可以得到更多复合色。将这些颜色按顺序排列，就形成了**色相环**。色相环的外圈颜色鲜艳，内圈颜色柔和，这是因为在内圈颜色中加入了白色，改变了颜色的**明度**。

在色相环上，颜色之间的距离决定了它们的关系。距离近的颜色称为邻近色，距离远的（如对角线上的）称为互补色。了解这些关系，有助于我们进行色彩搭配。

## 色彩在化妆中的应用

理解了色彩的基本原理后，我们来看看如何在化妆中具体运用这些颜色。

### 同色系/邻近色的运用

以下是眼影配色中，处理同色系或邻近色的基本步骤：

1.  **找出最深色**：在眼影盘中，找出颜色最深的一块。这块颜色通常用作**眼线色**，紧贴睫毛根部使用，范围不宜过大。
2.  **找出最浅色**：找出颜色最浅的一块。这块颜色通常用作**打底色**（哑光）或**提亮色**（珠光），用于眼窝大面积打底或局部提亮。
3.  **确定过渡色**：选择一个介于最深和最浅之间的颜色作为**过渡色**，用于连接眼线色和打底色，形成自然渐变。

通过“打底色 -> 过渡色 -> 眼线色”的深浅过渡，可以起到放大眼睛或调整眼型的作用。对于多色眼影盘，你只需任意挑选出三个深浅不同的邻近色，即可组成一套基本配色。

### 对比色与互补色的运用

在色相环上间隔约120度的颜色称为**对比色**（如红、黄、蓝），视觉冲击力强，多用于舞台或个性妆容。

在色相环上呈180度对角的颜色称为**互补色**（如黄与紫、蓝与橙、绿与红）。互补色在化妆中应用广泛，原理是相互中和。

*   **校正肤色**：使用紫色妆前乳可以中和泛黄的肤色；使用绿色妆前乳可以中和泛红的肤色。
*   **遮盖血管型黑眼圈**：对于因皮下血管透出形成的蓝紫色黑眼圈，应使用橙色的遮瑕膏进行中和，再叠加浅色提亮。

### 冷暖色的区分与应用

以黄绿色和紫红色为分界线，色相环可分为**暖色调**和**冷色调**。暖调让人感觉温暖、活力（联想阳光、大地）；冷调让人感觉冷静、洁净（联想天空、海水）。

在妆容中，冷暖色的应用至关重要，主要体现在以下两点：

1.  **粉底色号选择**：需要根据自身皮肤的冷暖倾向选择相应色调的粉底。肤色冷暖是相对的，可以通过与身边人对比或在自然光下对比来判断。
2.  **妆容色调一致性**：眼影、腮红、唇膏的色彩应尽量保持冷暖调性一致。例如，冷调的粉色唇膏搭配暖调的橘色腮红会显得不和谐。

选购时，可以观察产品的整体色号图进行对比。例如，在红色系中，偏橘的是暖调，偏粉紫的是冷调。选择同色调但不同明度的产品，可以打造出有层次感又和谐的妆面。

## 明暗表现（光影手法）

除了色彩，塑造面部立体感的另一个重要技巧是**明暗表现**。亚洲人面部轮廓相对扁平，可以通过高光和阴影来增强立体度。

简化记忆方法是参照面部的“四高三低”结构：**凸出的部位用高光，凹陷的部位用阴影**。

以下是具体的高光与阴影位置及注意事项：

*   **高光位置**：
    *   **额头**：从眉心向发际线方向晕染。
    *   **鼻梁**：使用哑光膏状高光或浅色遮瑕，避免使用珠光高光粉。
    *   **唇珠**：提亮上唇中间凸起部位。
    *   **下巴尖**：下庭偏长者可省略或向上提亮。
*   **阴影位置**：
    *   **眉额交界、人中沟、下唇下方**：本身不凹陷者可略过。
    *   **颧骨下方、下颌骨**：用于增加骨感，塑造瘦削脸型。甜美风格妆容中需谨慎使用。

需要明确的是，**高光和阴影是相对于自身肤色而言的**。比肤色亮的颜色可作为高光，比肤色暗的颜色可作为阴影。因此，买错色号的粉底液可以变废为宝，用作修容产品。

明暗手法不仅用于修容，也用于处理特定问题。例如，对于因皮肤凹陷产生阴影的**结构型黑眼圈（眼袋）**，可以在凹陷处使用明亮的高光色进行视觉填充，但改善效果有限。

## 审美总结：追求和谐

在这两节审美课中，贯穿始终的核心观点是**和谐**。无论是面部比例还是色彩搭配，我们都强调平衡与一致性。

不同的妆容风格（如欧美妆、日系妆、韩式妆、中式妆）都是一整套美学元素的和谐呈现。学习原理是为了“先做到不出错”，而“不出错”的基础，正是基于对自己面部特征和色彩属性的清晰认知。

分析自己的特征时，请保持中立态度。特征不是绝对的优点或缺点，你可以选择接受它，也可以运用所学方法调整它。最终如何向外界展示自己，取决于你内心的真实选择。

## 附录：黑眼圈处理总结

课程中提到了几种黑眼圈的处理方法，现总结如下。判断方法：用手指轻轻向上推黑眼圈下方皮肤，观察其变化。

1.  **血管型黑眼圈**：
    *   **特征**：呈现蓝紫色，因眼下血液循环不畅导致。
    *   **改善**：调整作息，热敷按摩。
    *   **遮盖**：运用互补色原理，使用橘粉色或橙色遮瑕膏中和。
2.  **色素型黑眼圈**：
    *   **特征**：棕色或咖啡色，因色素沉淀导致。
    *   **改善**：加强防晒和彻底卸妆。
    *   **遮盖**：先用深色遮瑕膏均匀肤色，再用浅色遮瑕膏提亮。
3.  **结构型黑眼圈（眼袋）**：
    *   **特征**：因眼下皮肤松弛、脂肪凸起或凹陷形成阴影。
    *   **改善**：注重眼部防晒与抗老护理（如佩戴太阳镜）。
    *   **遮盖**：在凹陷的阴影处使用高光色提亮，但化妆改善效果有限。
