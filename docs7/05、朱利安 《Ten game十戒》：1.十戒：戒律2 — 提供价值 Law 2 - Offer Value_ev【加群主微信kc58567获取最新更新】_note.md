# 朱利安《Ten game十戒》：05：戒律2 — 提供价值 🎁

在本节课中，我们将要学习“十戒”中的第二条戒律：**提供价值**。这是克服内心障碍、停止执着于结果、并最终在互动中感到轻松自如的最简单方法。

---

上一节我们介绍了戒律的核心思想，本节中我们来看看如何具体实践“提供价值”。

**提供价值**，意味着将你的注意力从“我能得到什么”转向“我能给予什么”。每当你感到紧张、焦虑或遇到阻碍时，通常是因为你正专注于“索取”。

以下是帮助你转换思维的关键方法：

*   **自我提问**：当你感觉状态不佳时，问自己：“**我此刻想索取什么？**”这可能很微妙，比如寻求认可、维护自我形象或满足自尊心。
*   **转换焦点**：接着问自己：“**我能给予什么？**”将你的意图从索取转向给予。

例如，想象你走向某人，心里想着：“嘿，去要100块钱。别搞砸了。”这会让你压力巨大。但如果你想着：“嘿，去给那个人100块钱。”整个过程就会顺畅、轻松得多，因为你不再是自己前进道路上的障碍。

---

理解了“索取”与“给予”的基本区别后，我们通过一个练习来亲身体验。

以下是体验“给予”心态的练习步骤：

1.  尝试独自尽可能大声地喊叫。
2.  注意你喊叫前的心理活动。你可能会想：“我该喊什么？”“喊这个会不会很酷？”“我看起来会怎样？”这些想法都属于“索取”心态——你在试图获取一个特定的形象或反应。
3.  现在，以“给予”的心态再做一次。例如，和一群人一起喊：“我能做到！”当你成为群体的一部分，心态就从“竞争”（我要打动他们）转变为“协作”（我们是一起的）。这时，你会发现自己能更轻松、更响亮地喊出来，因为你不再试图从外界索取认可。

这个练习揭示了社交互动中的一个核心原则：**人们不是你的敌人**。无论是公开演讲还是与女孩交谈，如果你认为“我必须赢得他们”，你就会把自己放在对立面，感到紧张。相反，如果你认为“我们是一个团队，一起来学习和享受”，互动就会变得轻松。

---

基于“给予”的心态，我们可以将其应用到具体的社交开场中。

一个强大的技巧是：**最好的开场就是跳过开场**。想象你已经认识对方，并据此行动。

*   **公式**：`行为 = 假设你们已经相识`
*   **代码示例（心态模拟）**：
    ```python
    # 传统“索取”心态：紧张地思考如何开场
    if approaching_stranger:
        anxiety_level = high
        think("我该说什么才能让她喜欢我？")

    # “给予”心态：假设已认识，直接互动
    if seeing_potential_friend:
        vibe = relaxed
        say("嘿，过来看看这个！")  # 如同对朋友说话
    ```

你可以直接走过去，像对朋友一样说：“嘿，来看看这个。”通过假设熟悉感，你跳过了充满试探和压力的“开场白”阶段，直接进入了“给予”和连接的氛围。

---

为了持续实践“提供价值”，你需要明确知道自己能提供什么。以下是具体方法。

首先，**制作一份你的价值清单**。问自己：一个女孩和你在一起（或发生关系）能获得哪些好处？

以下是你可以考虑的价值的例子：

*   **良好的情绪**：乐趣、兴奋、无忧无虑的感觉。
*   **真实感**：在当今世界，真实不做作非常稀有且珍贵。
*   **很棒的性体验**。
*   **与一个很酷的人相处**。
*   **个人成长**：如果你专注于自我提升，这也会正面影响她。
*   **其他独特优势**：例如，如果你经济宽裕，她可能会享受到美好的晚餐等。

列出这份清单至关重要。当你清楚自己能给予巨大价值时，你就不会感到内疚或猥琐。你会觉得引领对方获得这些好处是你的责任，而不是在“骗取”什么。这在销售和人际交往中是相通的：如果你确信自己的产品（或你本人）能给对方带来巨大利益，你就能自信、真诚地去“推销”。

---

除了列清单，还有两个任何人都可以立即提供的核心价值。

任何人都能提供的两种核心价值是：

1.  **良好的情绪**：成为那个能带来快乐、兴奋和轻松氛围的人。
2.  **真实感**：敢于做真实的自己，不伪装。

在当今社会，这两者都非常稀缺，因此极具吸引力。如果你能提供这些，你自然会吸引他人。

---

最后，我们可以通过一个简单的测试来检验你的心态是“给予”还是“索取”。

**无声开场测试**：走向一个女孩，但不要说话，只是保持沉默，观察她的反应。

*   如果你的能量场是“索取”的（紧张、期待、试图获取反应），她很可能会感到不适并离开。
*   如果你的能量场是“给予”的（平静、自信、分享你的存在感），即使不说话，她也可能被吸引并留下来。

这个测试说明，**“给予”始终是一种氛围和能量，而不仅仅是语言**。即使你问一个问题，比如“最近怎么样？”，也可以有两种方式：
*   **索取式**：急于得到她的回答，以此评估自己的表现。
*   **给予式**：不关心具体答案，只是通过问候传递友好、关心的情绪。语言只是传递良好情绪的载体。

---

本节课中我们一起学习了“提供价值”这条戒律。核心在于将思维从 **“我能得到什么？”** 转变为 **“我能给予什么？”** 。通过制作个人价值清单、专注于提供良好情绪与真实感，并实践“跳过开场”等技巧，你可以走出自己的障碍，在互动中变得放松、有吸引力。记住，当你提供足够多的价值时，对方会为了感谢你而想和你在一起。
