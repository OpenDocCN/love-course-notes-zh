# 满分游戏：06：练习7 — 保持理智 🧠

在本节课中，我们将要学习一项核心技能：无论外界发生什么，都能保持镇定与专注。这项技能需要通过练习来培养。

## 概述

我们将通过一个具体的练习来训练“保持理智”的能力。练习的核心是，当你在表达时，需要完全无视他人的干扰，专注于自己的思路。

## 练习说明

以下是练习的具体步骤。

1.  两人一组进行练习。
2.  其中一人用30秒时间，逐一讲述今天学到的课程要点。
3.  在讲述过程中，另一人的任务是进行各种干扰。
4.  讲述者的目标是**不被干扰分心**，不要陷入自己的思绪中，要像干扰者不存在一样，继续讲述课程要点。

## 干扰的原理与示例

上一节我们介绍了练习的规则，本节中我们来看看干扰是如何起作用的。干扰的核心是**让对方产生自我意识**或**制造突发意外**，从而分散其注意力。

以下是两种主要的干扰方式及其原理。

*   **引发自我意识**：例如模仿对方的笑声。这会突然将对方的注意力从正在说的话，转移到自己的行为上，导致其思路分裂。
*   **制造突发意外**：例如突然将对方拉近。这种出乎意料的举动会让对方的大脑瞬间聚焦于这个事件，而非正在讲述的课程。

干扰者的目标就是尽可能多地分散对方的注意力。从逻辑上看，这个练习很简单，就是不停地说“我学到了这个，我学到了那个”。但当干扰者将对方的注意力分裂成2份、3份、4份，并设定了干扰的节奏时，保持专注就变得极具挑战。

## 练习的目的与场景应用

请注意，这个练习主要是为了训练你自己。目的是让他人在干扰你时，你仍能保持镇定。

因为当你进入酒吧、俱乐部等嘈杂环境时，各种状况都会发生。正如之前所说，环境有它的“频率”。你需要保持在自己的频率上。

这个练习正是如此。干扰者施加了一个非常强烈的“频率”，而你则需要像什么都没发生一样，继续说“我学到了这个，我学到了那个”。通过这样的训练，你将能保持镇定。未来即使身处高级餐厅，面对所有人的注视和压力，你也能从容应对。

## 开始练习

这个练习对保持专注的一方来说很有挑战，但对进行干扰的一方来说也很有趣，可以尝试各种方法让对方产生自我意识。

现在，请两人一组，用大约30秒的时间进行练习。一人讲述课程要点，另一人尽力干扰。开始。
