# 赢家游戏：3.3.4：挑战4—人群中的目标

在本节中，我们将学习如何克服社交压力，在周围有其他人的情况下，自信地接近你感兴趣的目标。这是许多人在实战中面临的一大挑战。

上一节我们讨论了在相对私密环境下的互动，本节中我们来看看如何在公开场合采取行动。

挑战四对许多人来说是个大难题。没有人希望在众目睽睽之下进行搭讪。

很多时候，如果一个女孩独自站着或坐着，你会过去和她交谈。但假设有几个坐在离她两三英尺远的地方，你就会想：这太尴尬了，太奇怪了，我不想这么做，等等。

**核心逻辑是：这可能是你唯一的机会。你必须行动。**

例如，你在地铁上，地铁里还有其他人。你可能再也见不到那个女孩了。

在这些情况下，你需要做点什么。我建议尽可能表现得愉快、积极和善于社交。如果你担心尴尬，也许一开始不要那么明显地表现出追求意图。即使周围有其他人，你甚至可以在一定程度上把他们当作她同伴的一部分来互动。

但浪费这个机会是绝对愚蠢的。我向你保证，没有人会因为你主动、积极和友好而对你做出负面评价。

**核心心态是：如果你积极友好，谁会因此对你做出负面评价呢？**

任何因此刁难你或嘲笑你的人，很可能是因为他们嫉妒，他们希望是自己去接近她。

关键是，女孩们经常出现在公共场所，你需要学会在有人围观的情况下接近。

所以，鼓起勇气，行动起来。顺便提一下，适合进行此类挑战的好地方包括：公交车站、火车站、机场这类场所；公园里人群聚集的地方；或者办公楼里的公共区域。这些地方通常会有女孩，并且周围有人。

如果你在寻找这类情境，另一个绝佳的机会其实是在健身房。女孩在健身房里可能不是独自一人，附近的其他器械上还有别人。尽管如此，依然要去接近，不要担心别人怎么想。

---

本节课中，我们一起学习了如何在有他人在场的公开场合克服心理障碍，进行自信的搭讪。关键在于转变心态，将关注点放在积极友好的互动本身，而非他人的眼光，并抓住那些可能稍纵即逝的机会。
