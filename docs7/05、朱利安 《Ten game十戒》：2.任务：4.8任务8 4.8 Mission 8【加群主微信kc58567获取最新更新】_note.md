# 朱利安《Ten game十戒》：05：任务8

## 概述
在本节课中，我们将学习任务8的具体内容。这个任务包含四个核心练习，旨在巩固你的社交习惯、提升带领与连接人群的能力，并克服外在形象带来的不安全感。这是连续外出社交的最后一周，之后节奏可以适当放缓。

## 任务核心内容

上一节我们强调了持续行动的重要性，本节中我们来看看任务8的具体四项挑战。

### 1. 持续外出
在接下来的七天里，你仍需每天外出社交20至30分钟。这是连续高强度社交的最后一周，之后你可以适当放松。经过前七周的练习，每日外出对你来说不应再是难事，你应该已经对此感到习惯和轻松。

### 2. 合并社交组合
在上一项任务中，我们进行了“大规模挑战”。本周，你需要练习合并社交组合，目标是**连续合并至少5组**。

以下是合并组合的具体步骤：
1.  首先，开启第一个组合。例如，对第一个女孩说：`“嗨，最近怎么样？”`
2.  与她进行简短交流后，不要结束互动，而是对她说：`“嘿，过来见见我的朋友。”`
3.  带领她走向第二个组合，开启互动并介绍她们认识：`“嗨，这是苏西。”`
4.  此时，你可以选择留在新组合，或回到第一个女孩身边，但关键规则是：**你必须带着当前互动的女孩，才能去开启下一个组合**。
5.  重复此过程，像人猿泰山在树枝间荡跃一样，从一个女孩到另一个女孩，直到完成至少5次连续的组合合并。

为了让过程更有趣，你可以加入一些角色扮演，例如：`“嘿，来见见我的女朋友。”`关键在于保持带领姿态和微笑。常见的错误是介绍完后便沉默，导致冷场。你需要像教练一样引导：`“来，握个手，保持眼神交流。”`即使互动短暂或失败，你也可以在组合间切换，重要的是积累带领和介绍的体验。

### 3. 带领女孩移动
这项练习与合并组合相辅相成。很多人在带领女孩移动时感到困难，因此本周你需要专门练习。

以下是需要完成的挑战：
*   在合并组合时，自然地带女孩移动位置。
*   **额外挑战**：在本周至少一个晚上，尝试带领3个女孩走出酒吧、俱乐部或当前场所。

移动时，请遵循“小步推进”原则，逐步获取对方的顺从。不要一开始就提出很高的要求。例如：
*   代码示例（渐进带领）：
    ```
    1. “能握个手吗？”
    2. “站起来一下好吗？”
    3. “过来这边看看这个。”
    4. “我们去喝点水吧。”
    5. “我们出去透透气。”
    ```
通过渐进的方式，你成功带领对方移动的几率会大大增加。即使只是以微笑和“过来一下”为引导，你也能轻松地将对方带到门口。

### 4. 克服外在的不安全感
最后一项挑战是针对表面上的不安全感。你需要选择一天，故意以让你感到尴尬或不适的造型外出社交。

具体做法如下：
*   穿上你最觉得尴尬的旧衣服，或任何让你觉得不自在、显眼的服饰。
*   改变你的发型。如果你平时用发胶，那天就别用；如果平时不用，那天就多用一些，做一个让你不习惯的造型。
*   目的就是**主动触及所有关于外表的顾虑**，然后你会发现这些根本无关紧要。这只是一个帮助你摆脱那些愚蠢的小顾虑的挑战。

## 总结
本节课中我们一起学习了任务8的四个核心练习：连续七天每日外出、连续合并至少五个社交组合、练习带领女孩移动（包括将三人带出场所），以及通过改变造型来克服外在的不安全感。坚持完成这些练习，将极大增强你的社交带领力和心态。祝你好运。
