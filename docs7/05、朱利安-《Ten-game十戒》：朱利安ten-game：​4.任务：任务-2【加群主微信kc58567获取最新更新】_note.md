# 朱利安《Ten Game十戒》：05：任务二

## 概述
在本节课中，我们将学习朱利安《Ten Game十戒》的第二个任务。此任务旨在巩固第一周的成果，并引入新的挑战，以帮助你提升社交互动技巧，培养积极心态，并掌握关键的对话逻辑。

## 任务二详解
上一节我们完成了连续七天、每天至少30分钟外出的基础任务。本节中，我们将在延续此要求的基础上，增加几项关键调整，以深化你的练习。

### 混合日间与夜间游戏
以下是本周的第一个调整：你必须混合进行日间游戏和夜间游戏。我们总是倾向于回到自己最习惯的模式。这个调整旨在让你成为一个无论在白天还是夜晚，都能在各种环境中轻松自如地与女性开启互动的人。

*   **日间游戏者**：本周必须加入一些夜间游戏。
*   **夜间游戏者**：本周必须加入一些日间游戏。

请在未来七天及以后，尝试混合安排。

### 禁止自我否定
另一个挑战是：在未来七天内，完全不要否定自己。这是保持正确心态的关键。无论发生什么——被拒绝、没拿到号码，或者行动不如预期——都不要否定自己。

这意味着你必须寻找积极面，发现自己的优点和成功之处，并且只能认可自己。因此，在接下来七天的报告里，不允许出现任何负面、抱怨或受害者心态的内容，只允许积极反馈和自我鼓励。

这并非要你压抑或假装拒绝没发生，而是要实事求是，同时聚焦于你做得好的地方。例如，被拒绝后，可以想：“**我有勇气去搭讪。这太棒了！**” 我们习惯于自我否定，但这容易滑向消极和放弃的深渊。每次外出结束后，你都应该祝贺自己。

### 关注后勤问题
此外，我们需要开始运用“悖论游戏”策略，进入互动的逻辑层面，即关注后勤信息。

你的挑战是：在未来七天，每天至少在一次互动中，获取以下五个后勤问题的答案。这需要你平衡情感互动与逻辑询问的技巧。

以下是五个核心后勤问题：
1.  你和谁一起来的？（`who_are_you_with`）
2.  你住在哪个区域？（`what_area_do_you_live_in`）
3.  你怎么回家？（`how_are_you_going_home`）
4.  你有室友吗？（`do_you_have_roommates`）
5.  你今晚/今天晚些时候/明天有什么安排？（`what_are_you_doing_later`）

> **注意**：请重温“悖论游戏”第九条法则，学习如何自然地融入这些问题，避免像审问一样逐个提问。

### 使用特定开场白
最后一项新增内容是：本周你必须使用“**Daddy‘s home**”（爸爸回家了）这个开场白开启三次对话。

这个开场白要求你无法隐藏意图，必须带着充分的自信和一定的自嘲精神去执行。假设这个角色，同时对自己一笑置之，这对成功至关重要。

操作方式是：走向女孩，说“Daddy‘s home”，然后**自然地继续对话**。不要说完就停住等待反应，也不要试图为这个开场白寻求认可。

## 总结
本节课我们一起学习了任务二的全部内容。我们来回顾一下未来七天的要求：
1.  每天外出至少30分钟，连续七天（总计将达14天）。
2.  混合进行日间游戏与夜间游戏。
3.  在报告和自我对话中，只关注积极面，禁止自我否定。
4.  每天至少在一次互动中，获取五个后勤问题的答案。
5.  本周使用“Daddy‘s home”开场白三次。

祝你好运。
