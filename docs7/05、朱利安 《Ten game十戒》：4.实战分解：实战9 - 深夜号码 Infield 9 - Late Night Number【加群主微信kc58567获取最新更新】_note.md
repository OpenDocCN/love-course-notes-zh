# 搭讪艺术：05：实战分解 - 深夜号码

在本节课中，我们将通过分析一个深夜街头获取号码的实战案例，来学习如何在实际互动中运用核心技巧。我们将重点关注互动中的能量、自我娱乐以及临场应变。

## 概述

这是一个深夜获取联系方式的案例。虽然由于客观条件限制，最终只停留在了交换号码的阶段，但其中的互动动态和核心原则依然具有很高的学习价值。本节将拆解这次互动，特别关注**短互动动态**和**自我娱乐**的氛围营造。

## 互动动态与能量差异

上一节我们介绍了互动的基本框架，本节中我们来看看如何在实际对话中建立吸引。对话始于一个关于名字的轻松话题。

以下是对话的初始互动：
```
我：你叫Brittany？像Brittany Murphy（布兰妮·墨菲）那个演员吗？
她：嗯，类似吧。
我：我懂，就像Brittany Donovan， Brittany Cooper。所有80年代出生的“布兰妮”们，我们的妈妈们都以为这名字是她们独创的。
```
在这个开场中，需要注意我和同伴之间的能量差异。他全程的状态是“好吧……”，似乎在寻求对方的认可，无法在得不到回应时保持自在。他仿佛置身事外，说“嘿，就像那样”，但女孩的注意力却被吸引到我这里。

原因在于能量模式：我在“给予”，而他在“索取”。我的状态是“像布兰妮·墨菲，这样，那样，这样”，而他的状态则不同。我能穿透这种氛围，甚至加入一点挑战，比如直接说“顺便说下，很高兴认识你”，然后转向中间的女孩。

## 自我娱乐与内在玩笑

建立舒适感后，关键在于保持自我娱乐。我不需要从对方那里获取或索取任何东西。我完全沉浸在和自己的内部玩笑中。

例如，我穿着粉色衬衫，我会说：
```
“看这件衬衫，我为了看起来更帅才穿的。这不是我的衣服。”
```
这是一种高价值的开启方式。接着，我分享了一个故事：
```
“今天有个服务生跟我搭讪。体验太糟糕了。我当时在吃饭，我……我很困惑，完全懵了。”
```
当对方问“你来自新泽西？”，我回答：
```
“我来自泽西海岸。” （同时手在背后做波浪动作，暗示电视节目《泽西海岸》）
“我可不是那儿的人。那可能是因为……顺便说一下，是……”
```
这里，我创造了一个只有自己懂的幽默点，并乐在其中。

## 营造吸引与故事叙述

当环境嘈杂时，需要进行轻微的“隔离”——不是让她完全离开朋友，而是让她的朋友们暂时不在视野内，创造只有你和她的小空间。

此时可以加入一点挑战来测试兴趣：
```
“你和男人说话会紧张吗？我都还没跟你讲那个服务生的故事呢。”
```
同时，让自己感觉良好至关重要。你会注意到我经常被自己说的话逗笑。你也应该这样做，让自己像小孩子一样开心。讲述故事时要带着真实的激情。

例如，我这样描述服务生的故事：
```
“那个家伙整晚都在问‘你想吃什么？’，还试图使眼色。那种激情……你怎么能不被这个故事吸引？他一直在做这个，做那个。”
```
当对方反应平淡时，我自信地回应：
```
“你听起来对我没太大兴趣？我看起来怎么样？这是我第一次……我知道我有点帅。”
```
这是一种自信的、男性化的表达。我甚至自嘲说：
```
“我其实很害羞，不太会和女人说话。这是个问题，像种病。骗你的啦。我清醒的时候更糟。”
```
然后继续推进故事：
```
“总之，那个服务生看着我。后来我去洗手间，他紧跟着我进来，就站在我旁边，但一言不发。我猜因为他在工作，他在等我主动。然后我做了。”
```
用幽默和夸张收尾：
```
“之后发生了一些事，情况有点失控。出了很多汗。这就是为什么我……你知道的。”
```

## 展示激情与脆弱面

一个好的故事能展示你的激情和个性。当对方说“你是个很棒的讲故事的人”，我回答：
```
“是的，这就是我的方式。我是做什么的？我演讲，关于个人自助，帮助人们成为更好的自己。因为我就是这样一个优秀的人。”
```
并加入幽默的自夸与自嘲：
```
“而且我非常谦虚。非常非常谦虚。”
```
为了增加深度和联系感，可以适当展示脆弱面，但要用一种超越了的、带有幽默感的方式：
```
“我小时候是个艺术家。真的，我爸在我小时候经常喝酒，每晚回来就开始打我，一直打。最糟糕的是，我从来不知道原因。他从不告诉我，就让我一个人困惑地躲在角落哭。后来我有只小猫会来安慰我。但有一天，他抓住了那只猫……（停顿）他没那么极端，他只是慢慢地抚摸它，然后……（耸肩）就这样。这就是我的生活。我的生活像部电影。”
```
这种分享建立了情感连接。

## 风格对比与个性化

我的风格核心是**好玩、无忧无虑、充满激情**。我完全投入自己所说的话，乐在其中，非常放松。我什么都不需要，对方会不由自主地被吸引。

这与一些其他风格形成对比。例如，有些人的风格非常技术流，因为那种精确性让他们感觉良好。而对我来说，过于技术化会扼杀感觉。我追求的是自由。

再比如，我从不和女孩跳舞，但有些人在舞池中会感觉非常好。这些都是个性化的体现。

找到让你自己感觉良好的主题，比记住特定的台词或动作更重要。可能性是无限的。你外出实践越多，就越能打磨出自己独特的风格，同时它也会不断进化。

我个人喜欢“拉伸”情境或制造尴尬。例如：
```
她：你多大了？
我：38岁。（面无表情）
她：我18岁。
我：（继续面无表情）嗯，我38岁。我女儿在酒吧，她今天21岁生日。
她：真的吗？
我：是啊，怎么了？这是肉毒杆菌的功劳。等你老了就懂了。
```
这种离谱但一本正经的说法，游走在“他是不是在耍我”的边缘，我觉得这非常有趣。

## 持续练习与心态调整

关键在于找到你行为背后的主题。随着时间的推移，如果你每天都练习，可能会对“组合”感到麻木。

建议是改变这“30分钟”对你的意义。不要把它看成“必须完成的任务”，而是将其分散到一天中：在杂货店花5分钟打个招呼，晚些时候再花5分钟，然后把约会也算进去。理想情况下，让它成为你生活的一部分——你就是一个会和女孩互动的男人。这应该是一种自我表达，是让你感觉良好的事。

如果你感到平淡，缺乏戏弄和推拉，你需要有意识地投入努力。一个有效的方法是**把要点写在手臂上**作为提醒。例如，写下“不要平淡”。你需要不断的提醒。时间和觉察会解决这个问题。无论你的主要障碍是什么，整晚都需要某种提醒。写在手臂上能大大缩短你的学习曲线。

## 实战收尾

在互动中，90%是表达真我、享受乐趣，10%是收集信息。所以，在保持趣味的同时，也要穿插了解基本信息：
```
“那么，你是做什么的？”
“你们是最好的朋友吗？一起工作？”
```
在最后阶段，直接提出要求：
```
“你们现在准备干嘛？……过来Burger Lou（某个地方）吃点东西吧。”
```
如果对方拒绝或犹豫，可以加入幽默的推拉：
```
“对我这么凶？我做了什么？我把你引上了正道，你却不跟进。”
```
或者展示一点独特性：
```
“我其实来自瑞士，三年前才搬过来。（用上口音）”
```
当交换联系方式时，可以加入好玩的服从性测试：
```
“来，输入你的号码……输完后把手机还给我，要说‘给，主人’。”
“开玩笑的。你太让我紧张了。我只是个穿粉色衬衫的失败者。”
```
最后，友好结束：
```
“很高兴认识你。其实并没有。好了，就这样。”
```
核心流程很简单：营造乐趣 -> 处理实际 logistics（可惜本次未实现）-> 获取号码。就是：乐趣，转场，达成目标。

## 总结

本节课中我们一起分析了一个深夜获取号码的完整互动。我们学习了如何通过**自我娱乐**和**内在玩笑**来建立吸引力，如何用**充满激情的故事**和**适当的脆弱面**来加深连接，以及如何通过**个性化风格**和**持续的心态调整**来保持互动动力。记住，关键在于享受自我表达的过程，让互动变得有趣，同时不忘实际目标。
