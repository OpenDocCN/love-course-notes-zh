# 《PIMP 核心笔记》：3. 第三部分：外部技术模块：3. 上钩：上钩模块 - 视频示范：39.重要的是社交强度，而非能量级别。💃

在本节课中，我们将要学习如何处理夜店舞池中的组合，并理解一个核心概念：**社交强度**与**能量级别**的区别。这对于成功开启互动至关重要。

如果你在舞池中遇到一个组合，尤其是当女孩跳舞非常投入和疯狂时，你该如何进行下一步？处理舞池组合的方法，无论她是与朋友在一起还是独自一人，其核心原则是相同的。

但是，你必须拥有与女孩同等甚至略高的**社交强度**。这里需要明确一个关键点：**社交强度不等于能量级别**。

在过去的教学方法中，通常会强调你需要提高自己的“能量级别”。例如，他们会教你：你需要去跳舞，你需要释放更高的能量，你必须提供价值。他们建议你以同等或更高的“能量级别”去切入互动。

**这种方法是完全错误的。重要的是社交强度，而非能量级别。**

那么，什么是社交强度？它与能量级别有何不同？举例来说，当一个女孩在舞池中疯狂跳舞时，她展现出了很强的社交强度。此时，我完全可以走到她面前。

我可以带着**低能量**，但同时保持着**很高的社交强度**。这是一种什么样的感觉？它意味着你的存在感、自信和互动意图非常强烈，但你的外在表现（如动作幅度、语速）可以是冷静和沉稳的。
