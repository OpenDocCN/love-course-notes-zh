# 朱利安《Ten Game十戒》：05：练习4 - 提升表达力 🎭

在本节课中，我们将通过一个无语言的表演练习，来探索和扩展你的非语言表达能力。我们将专注于面部表情和肢体语言，这是沟通中强大但常被忽视的层面。

上一节我们介绍了沟通的基础，本节中我们来看看如何通过夸张的表演来强化你的表达范围。

## 练习说明

我们将进行一个小组练习，每组四人。练习规则如下：

*   你不能说话，只能使用面部表情和肢体动作。
*   我会给出一个具体的情境。
*   你需要用10秒钟时间，向组内其他三人表演并传达这个情境。
*   每轮结束后，我会喊出下一个情境，表演者轮换。

这个练习的关键在于展示你表达力的范围。沟通有两个主要渠道：语言和**非语言信号**（如表情、肢体）。我的很多技巧都聚焦于后者。通过这个练习，你能清楚地看到自己目前的表达范围，并努力去扩展它。

## 练习情境列表

以下是需要你表演的具体情境。请全力以赴，投入情感，让你的表演充满张力和趣味性，吸引你的同伴。

1.  你走在街上，一位老妇人朝你吐了口水。
2.  你在机场错过了航班，伤心哭泣。你再也见不到你的女朋友了。
3.  你刚刚发现自己失去了所有的钱。
4.  你刚刚和杰西卡·阿尔芭发生了关系。
5.  你正在玩一个非常酷的电子游戏，并且完全沉浸其中，像个十足的发烧友。
6.  你正在锻炼，并且完全投入，这是你人生中最激烈的一次训练。
7.  你正在坠入爱河，爱的感觉此刻淹没了你。
8.  （最后一轮）你自己想一个情境，让其他三个人猜出你在表演什么。

## 核心要点回顾

本节课中我们一起学习了如何通过刻意练习来扩展非语言表达的范围。这个练习旨在让你关注沟通中的**底层信号**。当我与女孩交谈时，我同时在语言和非语言两个层面上操作。将丰富的非语言表达与语言结合，效果会非常出色。

记住，关键在于**投入激情和情感**，让你的整个身体都参与进来，而不仅仅是做几个简单的动作。夸张一些，让它变得有趣，让你的同伴被你的表演吸引。
