# 《PIMP 核心笔记》：3.3：上钩模块 - 视频示范：28. 搭讪时，朋友打岔的处理方法 🛡️

在本节中，我们将学习一种常见的社交挑战：当你与目标对象开始交谈时，她的朋友突然介入。我们将探讨如何快速应对，以赢得朋友信任，并最终将注意力转回你的主要目标。

## 概述

在搭讪过程中，目标对象的朋友突然介入是一种常见情况。处理得当，可以化障碍为助力；处理不当，则可能导致整个互动失败。本节将详细讲解应对此情况的步骤与核心心法。

上一节我们介绍了如何开启与目标对象的对话，本节中我们来看看当对话被第三方打断时，应如何应对。

## 核心情境与应对策略

具体情境是：当你向目标对象打招呼并开始交谈后，她的朋友突然介入并提出质疑，例如“你谁啊？”。

此时，你的目标是快速稳定局面，赢得这位朋友的信任，而非与她对抗。核心策略是“为调情而调情”，即通过友善、轻松的互动化解紧张气氛。

以下是处理此情况的具体步骤：

1.  **快速承接话题**：立即以友好、开放的态度回应介入的朋友。例如，可以说：“嘿，你们是闺蜜吗？”
2.  **给予真诚赞美**：迅速对朋友或她们的共同点给予一个真诚的赞美。例如，接着说：“我觉得你们今天穿的闺蜜装挺可爱的。”
3.  **转移注意力**：在简短地与朋友互动、建立初步融洽关系后，自然地将主要注意力重新放回你的初始目标女孩身上。

**核心心法公式**可总结为：**化解敌意 → 建立同盟 → 回归主线**。

首先，通过友善回应化解朋友的防御心态。接着，通过赞美将其转化为短暂同盟，而非障碍。最终，在氛围缓和后，重新聚焦于最初的目标互动。

## 总结

本节课中我们一起学习了搭讪时应对朋友打岔的方法。关键点在于保持镇定，通过快速赞美介入者来赢得其信任，为后续与目标对象的互动创造良好环境。记住，目标是扫清障碍，而非开辟新战场，因此最终务必优雅地将注意力回归主线。

掌握这一技巧，能让你在复杂的社交场景中更加从容不迫。
