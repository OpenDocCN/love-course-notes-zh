# 《PIMP 核心笔记》：3. 第三部分：外部技术模块：4. 创造气氛、情绪共振：视频示范：27.语言结构的五句话 🗣️

在本节课中，我们将要学习一个用于开启对话的、结构化的五句话语言框架。这个框架旨在帮助你自然地接近他人并展开初步交流。

## 概述

一个清晰的开场结构能有效降低社交焦虑，并引导对话走向。本节将详细介绍这个由五句话组成的固定语言结构，你需要牢记并熟练运用它。

## 五句话结构详解

以下是构成该开场白的五个核心问句，它们按顺序设计，以逐步建立初步联系。

1.  **第一句话：引起注意**
    “嘿，等一下，我想认识你。” 这句话的目的是让对方停下并注意到你。

2.  **第二句话：询问背景（地域）**
    “你是本地人吗？” 这是一个简单、安全的话题，可以根据你所在城市进行具体化。例如，在沈阳就问“你是沈阳本地姑娘吗？”，在长沙则问“你是长沙姑娘吗？”。

3.  **第三句话：了解职业**
    “你做什么工作的？” 此问题有助于了解对方的日常生活和社会角色。

4.  **第四句话：询问年龄**
    “你多大了？” 这个问题可以进一步加深对对方的了解。

5.  **第五句话：探询后续计划**
    “你接下来要去干嘛？” 或 “你一会儿要去干嘛？” 这个问题为对话的延续或即时邀约创造了可能性。

## 记忆与应用要点

上一节我们分解了每一句话的含义，本节中我们来看看如何掌握它。

为了确保你能在需要时流畅使用，请遵循以下步骤：

*   用笔和纸将这五句话写下来。
*   反复背诵，直到滚瓜烂熟。

## 总结

本节课中我们一起学习了一个简洁的五句话开场结构。通过“引起注意 -> 询问地域 -> 了解职业 -> 询问年龄 -> 探询计划”这个流程，你可以更有信心地开启一段陌生对话。请务必通过书写和背诵来熟练掌握这个基础框架。
