# 《PIMP 核心笔记》：3. 第三部分：外部技术模块：3. 上钩：上钩模块 - 视频示范：27.和二人组开场的示范 🎬

在本节课中，我们将要学习如何单独面对一个二人组合进行开场搭讪。这种情况需要你同时与目标女孩及其朋友互动，不能忽视任何一方。

上一节我们介绍了搭讪的基本概念，本节中我们来看看面对二人组时的具体操作方法。

## 核心场景与策略

此场景的核心在于：**你独自一人接近，且目标对象并非单独一人，而是有朋友在场**。这意味着你无法依赖同伴去分散其朋友的注意力，必须自己同时与两人建立初步联系。

以下是处理此情况的关键策略列表：
*   必须同时与两人展开对话，不可忽视目标女孩的朋友。
*   开场应直接面向你的主要目标（即你感兴趣的女孩）。
*   在初步接触后，需迅速将对话范围扩大到她的朋友，将两人视为一个组合进行互动。

## 示范对话解析

下面是一个具体的开场白示范，它展示了如何应用上述策略。

> “哎，等一下，我觉得你其实挺不错的。所以说想认识你。嘿哎，你们两个人都是沈阳姑娘吗？😡”

这段示范对话可以拆解为三个步骤：
1.  **直接开场**：以“哎，等一下”引起注意，然后直接对目标女孩表达兴趣（“我觉得你其实挺不错的”），并表明意图（“想认识你”）。
2.  **扩大范围**：使用“嘿哎”或类似的语气词（注：此处为示范内容，实际应用时可简化为“你们”），将话头转向她的朋友，把两人都纳入对话场。
3.  **提出共同话题**：通过一个简单的问题（“都是沈阳姑娘吗？”）向两人同时提问，从而开启一个轻松的、包含双方的对话。后面的表情符号😡在示范中可能用于传递轻松、玩笑的语气。

本节课中我们一起学习了如何应对二人组搭讪场景。关键在于**独自应对、兼顾双方、由点及面**，即从对目标个人的直接开场，迅速过渡到将两人作为一个整体进行互动，从而平稳地开启三人对话。
