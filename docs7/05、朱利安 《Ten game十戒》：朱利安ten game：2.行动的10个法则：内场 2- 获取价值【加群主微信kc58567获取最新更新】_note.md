# 朱利安《Ten Game十戒》：05：内场 2 - 获取价值

在本节课中，我们将要学习一个在社交互动中极为普遍的现象——“获取价值”模式。我们将通过分析一个具体的互动案例，来理解这种心态如何影响你的行为、能量以及最终的结果。你会发现，即使你的言行与成功时相似，但内在的驱动力不同，结果也会截然不同。

## 互动案例分析：获取价值模式

上一节我们介绍了“提供价值”的心态，本节中我们来看看它的反面——“获取价值”。这是一种绝大多数男性在互动中不自觉陷入的模式。

以下是一个典型的“获取价值”式互动的文字记录。请注意，尽管对话在表面上可能包含有趣的提问或陈述，但其底层能量是索取而非给予。

```
- 你好。我今天看了那部电影《珍珠猎人》，是西蒙纽尔启发我的。我在找“巨魔”。我是个巨魔怪物。
- 你叫什么名字？我叫朱利安。
- 你真的来自奥佐吗？再说一次？你来自布里根还是奥斯陆？
- 你喜欢还是讨厌它？喜欢还是讨厌？
- 所以你是瑞典人？
- 所以你是搞价值科学的，来吧。
- 我懂一点点。好的，我懂。嗨，让我知道你是否理解，我的口音怎么样。
- 但是…那个…是好是坏？
- 是更好还是“黄油”？
- B，既然照片是维克托…我喜欢它。你去过索昂吗？
- 也许另外两个地方。如果让你选，包括瑞典或挪威，你会选哪个？
- 你去过美国旅行吗？
- 你去过斯堪的纳维亚以外的地方旅行吗？非洲？
- 你是亚历克斯·瓦斯科的粉丝吗？
- 你是演员亚历山大·萨斯孔的粉丝吗？
- 你多大了？
```

## 模式的核心特征

即使行为相似，获取价值模式与提供价值模式在本质上存在关键差异。以下是获取价值互动的主要特征：

*   **能量索取**：你的问题或陈述背后，隐藏着希望对方给予你良好反应、认可或验证的渴望。你感觉自己是“稀缺”的，试图从对方那里“获取”价值来提升自我感觉。
*   **对结果的依附**：你的情绪状态与互动结果紧密绑定。如果对方离开或反应不佳，你会感到失落（“感觉糟透了”）。这源于你“期望得到某些东西却没有得到”。
*   **操纵性而非表达性**：你可能会使用那些在状态好时（“心流”中）会说的台词，但目的不再是纯粹的自我表达，而是将其作为一种“技巧”来操纵对方，以获取你想要的反应。
*   **紧张与不自然**：身体语言会显得更僵硬、更紧张、更易受外界影响，缺乏从容和 grounded（脚踏实地）的感觉。整体能量是“失衡的”、“有点强迫的”。
*   **投资失衡**：你通过不断提问，试图让对方在对话中投入比你更多。虽然你可能说了很多，但本质是你在“索取”对方的注意力、信息和情绪投入。

## 为何这种互动难以推进？

这种模式解释了生活中大量“无疾而终”的社交互动。女孩可能不会立即拒绝你，甚至会容忍你15分钟，但互动不会升温。原因在于：

1.  **你不是在搞砸**：你的表现并非糟糕到让人立刻想离开。
2.  **缺乏更好选择**：对方可能没有更佳选择（例如，她的朋友正在和别人聊天，她不想当电灯泡）。
3.  **但不足以吸引**：你的能量和模式不足以让她产生“我想和这个人在一起”的强烈吸引力。

这导致了典型的“社交地狱”：你投入了大量努力维持对话，但最终当你索要号码时，只会得到“我要去洗手间”、“我有男朋友了”或“我没带手机”等回应。

## 如何自我觉察？

判断自己是否处于“获取价值”模式，有一个非常简单的内在标准：**如果你感觉到情绪波动**。

*   **公式**：`感到失落/沮丧 = 对结果有依附 = 获取价值模式`
*   **原理**：如果你“期望得到某些东西却没有得到”，情绪就会下跌。如果你内心已经“拥有一切”，那么无论对方如何反应，对你都“没关系”。

## 关于“高手”的迷思与现实

一个阻碍我们的常见迷思是，认为社交高手永远都不会“陷入头脑”（过度思考）或感到紧张。一旦我们自己陷入这种状态，就会感到挫败，觉得自己很糟糕。

现实是：
*   即使是高手，也会时不时地“陷入头脑”。
*   即使是高手，也经常会被拒绝。
*   **如果你在真实地表达自己，不是每个女孩都会与你产生共鸣，这很正常。** 试图让每个人都喜欢你的唯一方法，就是戴上面具、妥协真实的自我，而这最终会让你感觉更糟。

目标是追求进步，而非完美的幻想。搞砸一次互动、说些无聊的话，这都很正常，甚至让整个过程更人性化、更有趣。女孩通常会从整体来看待互动。

## 案例背后的根本原因

在这个具体案例中，阻碍我的根本原因是：**对结果的依附**。

这种依附通常源于：
*   **内在的“需要”**：你真正“需要”从这次互动中得到某些东西（如验证、号码、约会）。
*   **情境压力**：在这个案例中，工作坊即将结束，我想在实拍中完成一次“收尾”（成功带女孩离开）来让今晚“圆满”。这种“如果…我会感觉更好”的想法，立刻将心态从“提供价值”切换到了“获取价值”。

本节课中我们一起学习了“获取价值”这一普遍社交模式。我们分析了它的外在表现、内在驱动力（如能量索取、对结果的依附），以及它为何导致互动停滞不前。关键在于学会自我觉察，通过感受自己的情绪是否因外界反应而波动，来判断自己处于哪种模式。请记住，进步比完美更重要，真实且有起伏的互动过程本身就是人性化和有趣的。
