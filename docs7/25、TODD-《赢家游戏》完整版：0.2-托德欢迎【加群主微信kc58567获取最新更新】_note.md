# 赢家游戏：0.2：托德的欢迎致辞 🎬

在本节课中，我们将学习托德·V的《赢家游戏》课程介绍。本节内容将阐述课程的核心理念、结构以及如何高效利用本课程以获得最佳效果。

## 课程起源与核心理念

欢迎来到《赢家游戏》。我很高兴你来到这里。我确信你会喜欢这个课程的内容。在你深入学习材料之前，我想让你了解《赢家游戏》的起源、主要内容，以及如何最好地利用这个课程以获得最佳效果。

首先，我要说一些看似矛盾的话。一方面，整个课程的前提是，我在生活的某些领域——例如游戏、商业、各种体育项目和学术——取得过成功，这使我在某种程度上具备教导你一些事情的能力。另一方面，我不希望你把我当作权威导师。我希望你独立思考，因为《赢家游戏》的一个主要前提是，你在生活中所做的一切都应该是**可执行的**，并且基于你自己的经验。

因此，当你观看各个模块的课程时，我希望你秉持这样的想法：这是一个成功人士在分享经验，所以很可能其中有值得借鉴之处。然而，我不希望你将其视为绝对真理。对于我教给你的每一点，我希望你做的是：理解这个想法，将其与你自己的生活经验联系起来，然后去实践这个想法，并观察它在你的生活中产生的效果。因为这些想法基于我所经历的真理，我非常确信它们会帮助你，并且会与你将在现实世界中看到的情况很好地契合。但是，我希望你信任这些想法，是因为它们在现实世界中对你有效，而不是因为它们出自我之口。这一点非常重要，尤其是在一个我会谈论那些被某些人奉为圭臬的、不真实言论的课程中。

所以，我要非常明确地指出：我是你的指导者，是你的老师、导师，但不要把我当作权威导师，也不要认为我无所不知。我知道很多事情，知道很多行之有效的方法，但我并非全知全能。你决定在生活中，尤其是在人生哲学方面做出的任何重大决定，最终的裁决者应该是你自己，并且需要与你自己的生活经验产生共鸣。这是我要说的第一点。

## 课程的个人性与结构

我要说的第二点是，这可能是我发布过的最具个人色彩的课程。过去看过我作品的人都知道，我并不经常大量谈论“内在游戏”。这有几个原因。一个主要原因，正如你将在本课程中看到的，是过去所教授的许多“内在游戏”内容我并不认同，我认为它们有些胡扯。因此，当我不同意那些观点时，很难去教授与之相符的“内在游戏”。

不过，老实说，我过去没有教授“内在游戏”的第二个原因是，“内在游戏”非常个人化。为了告诉别人你如何看待世界、你的哲学是什么、你的个人心理学是什么、你经历过哪些个人挣扎，这需要很大的开放性。教授一个搭讪技巧，它是一个事实，你去用它，它就有效。教授“内在游戏”，你必须分享你自己的个人故事，你必须敞开心扉，这是我过去没有做过的事情。因此，这对我来说是一个非常个人化的课程。从这个意义上说，它是我发布过的课程中，我绝对最引以为豪的课程之一。因此，我希望我在课程中的经验和人生历程能够传递给你，并以这种方式帮助你。

现在，课程具体包含什么以及如何使用它？课程基本上有四个模块。

以下是四个核心模块的简要介绍：

1.  **第一模块：谎言与真相**：探讨搭讪和自助社区中的谎言与不实之词。
2.  **第二模块：赢家游戏**：这是我个人对“内在游戏”的看法，我称之为“不糟糕的内在游戏”或“非失败者的内在游戏”，因此命名为“赢家游戏”。它涵盖了我从游戏、竞技扑克、高水平足球比赛、高层商业活动以及学术生涯中观察到的，那些超越显而易见的技能和天赋，区分赢家与输家的因素。是什么让一个人在条件相同甚至处于劣势时能够获胜？我们稍后会深入探讨。
3.  **第三模块：搭讪的内在游戏与行动课程**：在这个模块中，我将逐一剖析阻碍人们进行搭讪和投入练习以精通游戏的所有借口和障碍。随后附有一个行动课程，我将一步步指导你，从完全无法进行搭讪，到完成即使游戏多年的老手通常也会回避的最高难度搭讪。
4.  **第四模块：实战录像**：包含未经剪辑的实战录像。正如你将在“关于实战录像的真相”模块中看到的，实战录像应该达到某些标准。我已经确保本课程中的实战录像达到了这些标准，使其成为我所能制作的最具教育意义和最诚实的实战录像。

你可以按任意顺序学习这些模块。但我建议，在观看实战录像之前，先观看“关于实战录像的真相”模块，因为它会为你提供如何使用这些录像的背景信息。另外，如果你在搭讪社区待了很长时间，我强烈建议你将“真相”系列视频多看几遍，以理解其中的细微差别。这些观点非常微妙，并非简单地断言“外表和金钱重要”或“外表和金钱不重要”，而是非常细致、非常情境化的。或者说，实战录像好或不好？不，我不这么说。我说实战录像在这方面是好的，但它可能有那些缺点。所以多看几遍，理解其中的细微差别，尤其是在搭讪社区或自助研讨会社区待得越久，你可能越需要“重新布线”，可能需要多复习几遍。

## 学习建议与行动号召

同样，我强烈建议你反复观看“赢家游戏”视频。其中有很多深度和内涵。你会发现，在你的学习进程中的不同阶段反复回顾它们，你将发现更新、更深、更好的经验教训。任何关于“赢”的事情都像是剥洋葱皮，你可能会回过头来看，发现内层看起来很像外层，只是更具深度、更微妙、更复杂。所以我绝对推荐反复观看。

我想我不必告诉你，为了真正理解我使用的台词、调侃以及完整的分析，你需要尽可能多地观看实战录像。但最重要的是，我想总体告诉你的是：本课程包含一个行动课程，这是有原因的。那是因为游戏是在现场进行的，没有行动，这一切都毫无意义。你也会在“赢家游戏”模块中看到，我多次谈到它必须是**可执行的**，它需要产生结果。如果不可执行，它就无法带你前进。

因此，我强烈鼓励每个人都完成本课程中的行动课程。但即使你不这样做，你也必须去尝试这些方法。我在“关于实战录像的真相”模块中提出的另一点是：除非你亲身实践，有东西可以比较，否则你无法真正判断实战录像的好坏。这对课程中的所有经验教训都是如此。没有尝试、使用它们，没有获得现实世界的经验，你就无法充分评估它们。所以，如果你是那种只想坐着观看这个课程、仅仅把它当作娱乐的人，你会有所收获，会学到一些东西，但不会获得全部价值。只有当你走出去应用这些经验教训，然后不断回顾课程时，你才能获得本课程的全部价值。其中有很多深度，我鼓励你从中获得绝对最大的收益。

综上所述，再次感谢你的加入。享受所有材料。这些确实是我发自内心的分享，是我在搭讪社区中看到最多人做错和最需要的东西。如果你愿意接受，它将以一种非常积极的方式彻底改变你的思维模式。
