# 朱利安《Ten Game十戒》：05：法则七 - 变得更好，而非更苦（第二部分）

在本节课中，我们将深入探讨十大法则中的第七条：“变得更好，而非更苦”。我们将重点分析在搭讪和社交互动中遇到挑战时，应如何调整心态，将逆境转化为成长的动力。

上一节我们介绍了法则七的核心是态度选择，本节中我们来看看如何具体应用这一法则。

正如之前提到的，搭讪过程可能混乱不堪。你会遇到大量逆境。你外出时，会有良好的互动，也会有糟糕的互动。你会经历愉快的夜晚，也会经历糟糕的夜晚。会有很多麻烦事抛向你。

我们通常称这些为“一致性测试”。女孩们会测试你。她们会向你抛出很多难题。这并非坏事。

一方面，你需要预料到女孩会有此行为。她需要这样做来了解你是哪类人。她正在通过行为线索，而不仅仅是肤浅的外表，来判断你的吸引力。她可能会想：“我喜欢这家伙。”但她需要确认你确实是她认为的那种人。任何人都可以伪装一小段时间。她会测试你：“当我给他出难题时，他作何反应？他能保持冷静吗？”如果不能，她会庆幸自己及早发现了。所以她会向你抛出难题。

而你有一个选择。你可以选择变得更好，也可以变得愤怒、消极，陷入“去他妈的”这种情绪中。或者，你可以拥抱这些挑战，变得更好。你可以灵活应对，发挥创意，就像我们之前谈到的，将其与你掌握的众多技巧联系起来。

同时也要接受一个事实：你承受的考验越多，你对它的敏感度就越低，而你也会变得越强大。这就是你的成长方式。擅长此事的关键并非避免逆境。拒绝也是如此。

大多数男人试图避免被拒绝，他们对拒绝有恐惧症。这里的关键不是学会如何不被拒绝，而是**降低你对这种恐惧的敏感度**。你要降低自己对那些废话、评判和社会压力的敏感度。重点不在于避免它，而在于更好地应对它。

还要明白，随着你自身价值的增长，你会受到更多的测试。将会有更多的评判，更大的社会压力。这是身处高位的必然代价。如果你想要成为主角，就要习惯它。

所以，不要回避这些情况，而是要投身其中。你面临一个选择：要么变得更好、更强，拥抱挑战；要么陷入消极的受害者心态，怨天尤人。面对拒绝时也是如此：选择变得更好，还是变得消极。这是你应该时刻保持的心态选择。

以下是关于应用此法则的几个关键点：

*   **拥抱挑战**：抛向你的难题越多越好。你处理的逆境越多，成长就越快。
*   **保持心态**：时刻采取这种心态。拥抱一切，穿越烈火。
*   **理解本质**：大量的搭讪技巧就是穿越烈火的过程。当你外出实践时，你正是在穿越那混乱的烈火。

本节课中我们一起学习了法则七的第二部分：如何在实践中将逆境视为测试和成长的机会，通过降低对拒绝和压力的敏感度来变得更强，并始终选择“变得更好”而非“变得更苦”的积极心态。记住，成长源于穿越烈火，而非避开它。
