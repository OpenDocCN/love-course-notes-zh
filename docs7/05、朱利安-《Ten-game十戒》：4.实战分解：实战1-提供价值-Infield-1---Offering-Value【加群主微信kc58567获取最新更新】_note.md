# 朱利安《Ten game十戒》：05：4.实战分解：实战1-提供价值 🎯

在本节课中，我们将通过分析一段实战互动视频，深入理解“提供价值”这一核心概念。我们将重点关注互动中的氛围、潜沟通以及心态，而非表面的具体行为。

上一节我们介绍了“提供价值”与“索取价值”的区别，本节中我们来看看一个具体的实战案例。这段互动清晰地展示了什么是“提供价值”的心态和行为模式。

**核心氛围描述：** 不在乎结果、有趣、好玩、无忧无虑、非反应性、自娱自乐、主导性强。

关键在于，游戏在很大程度上是**前语言**的。你无法完全用语言向没看过视频的朋友描述这种氛围。你只能说“很好玩”、“不在乎结果”，但这只是逻辑上的理解。你必须从情感上识别它，女孩也能立刻感知到这种状态。这就像你坐在一个非常沮丧或压力很大的人旁边，你也能感觉到那种情绪一样。女孩能立刻感觉到你是否感觉良好、是否真实、是在提供价值还是索取价值、是源于富足心态还是稀缺心态。

这就是**状态传递定律**。不仅她能感觉到，她的朋友能感觉到，就连你看这段实战视频时也能感觉到。通过下一个对比视频（索取价值），你会更明显地感受到观看体验的不同。

以下是互动中的关键对话片段，请注意其中的氛围和潜沟通：

> “我昨天生日，第一次来夜店，好兴奋。”
> “你多大了？为什么你这么小？”
> “你父母也矮吗？也许你妈妈和邮差出轨了？”
> “你知道电影《霍比特人》吗？我要叫你弗罗多。你能帮我个忙吗？看着我，像甘道夫那样说‘操我’。”

现在，让我们根据法则来分解这段互动。

首先，描述整体氛围：不在乎结果、有趣、无忧无虑、非反应性、自娱自乐、主导性强、好玩。

更重要的是，要意识到游戏是**前语言**的。你无法完全用语言向没看过视频的朋友描述这种氛围。你只能说“很好玩”、“不在乎结果”，但这只是逻辑上的理解。你必须从情感上识别它，女孩也能立刻感知到这种状态。

这就是**状态传递定律**。不仅她能感觉到，她的朋友能感觉到，就连你看这段实战视频时也能感觉到。通过下一个对比视频（索取价值），你会更明显地感受到观看体验的不同。

关于具体内容（如经期血液笑话），需要明白：**这与我个人是 congruent（一致）的**，虽然有点出格，但这就是我的幽默风格，这就是真实的我。但这绝不意味着你们需要去说或做同样的事情。

这是一个常见问题：如果只关注“做什么”，你会想：“哦，他说了这个，做了那个，所以吸引了女孩。”这是错误的。因为当你去说或做同样的事情时，你**感觉**不到我在说或做那些事时的感觉。这才是关键。正如之前提到的，女孩看的不是行为，而是**氛围**。

所以，你应该问的问题是：**氛围是什么？以及，我能说什么或做什么，来拥有这种氛围？** 从而让我感觉和朱利安谈论经期血液时一样好。

但同时，我也保持着对女孩和互动的**社交直觉**。请注意，我不是一上来就直接讲经期血液的笑话。我可能会先说“操我，弗罗多”之类的话。如果我说“嘿，说‘操我，我是弗罗多’”时，她没有像视频中这个女孩那样露出灿烂的笑容，而是表现出犹豫，那我就**不会**接着讲经期血液的笑话。

这是一个信号：任何更出格的内容都可能让她无法产生共鸣。我仍然会保持真实，但“真实”并不意味着只有一件事可说可做。有很多事情可以是真实、一致并能让我感觉良好的。我不会执着于“我必须讲那个经期血液的笑话”。

所以，我会选择另一条行动路线。这就像我在玩我的游戏，享受乐趣。整个过程是：我先玩一点，看看她是否还在参与；再玩一点，再看看。你通常可以通过她的紧张程度、身体语言或眼神来判断。如果她身体僵硬、眼神躲闪、紧张地傻笑，或者说“你喝醉了吗？”“你嗑药了吗？”“你怎么回事？”之类的话，这意味着你需要慢下来。你正在接近那个“因过于出格而无法产生共鸣”的极限。

这并不意味着互动结束了，只是你收到了一些警告信号。这对于理解“拒绝”也很重要。男人们倾向于认为女孩要么爱你，要么恨你，并且她能瞬间从爱你切换到恨你。正是这种想法阻碍了你。你会想：“如果我搞砸了一点点，她就会恨我。”这是非黑即白的思维。

不，拒绝是**渐进**的。如果一个女孩对你有好感，她不会突然恨你。她会事先给你那些信号：压力太大了，压力太大了，压力太大了。如果你只是盲目地继续做你的事，那么是的，你可能会得到一句“好吧，祝你今晚愉快”。你会想：“这拒绝来得太突然了。”不，之前有所有的信号，只是你没看到。同样，如果你有点太无聊了，她也会给出信号：太无聊了，太无聊了，太无聊了。然后“好吧，再见”。你会觉得拒绝来得很突然，但其实事先总有信号。

一旦你培养了这种能力——即在表达自我的同时仍然关注女孩——你就不应该得到所谓的“糟糕的拒绝”。我们之后会看一个拒绝合集，那些其实都不算糟糕。大多数拒绝都是“很高兴认识你”。有人会问：“那些疯狂的拒绝呢，比如她把饮料泼你脸上？”那根本不应该发生，只要你保持这种社交直觉。

**社交直觉非常重要，它能传递大量价值。** 你能意识到并能改变行动路线，并且你不会因此“失去状态”。男人们也常会卡在一件事上：“我必须讲那个经期血液的笑话。如果我不讲，我会感觉很糟，或者觉得我妥协了，觉得自己很弱。”例如，讲了弗罗多之后，如果她反应冷淡，他们就想：“糟了，我不能讲经期血液笑话了。”然后他们就失去了状态。这不是在执行计划，而是把你的身份或自我价值投资在了这一件事上。你必须**灵活**。女孩会看到这一点：“哦，他很灵活，而且有社交直觉。”

反之，如果你缺乏社交直觉，只是一味地做那件事，而她又无法与你产生共鸣，你就会失去大量价值。她会想：“这家伙都看不出我感觉不舒服，看不出他做得太过了。既然我无法与他产生共鸣，我怎么能信任他？如果以后我们有孩子，我怎么能指望他在这个世界上养活我们？如果他连这点社交直觉都没有，他出去可能会把自己害死。我完全不能依赖他，所有事都得我来做。”没有女孩想和这样的男人在一起，这**价值极低**。

关于如何识别信号：你会从整体氛围中感觉到，但它会通过她的身体语言、说话内容和方式这些**潜沟通**传递出来。比如紧张、身体僵硬、尴尬的笑、说“你没事吧？”等等，这意味着“慢一点”。如果她无聊、走神，这意味着“活跃一点”。目前，我只想让你们先意识到那种氛围，那是你们需要瞄准的目标。

再说一次，重点不是我**做了什么**。现在应该让你们思考的问题是：**你他妈到底要做什么？你能说什么来让自己那么兴奋？** 我可以保证，在那个环境里，没有其他人能提供那么多价值。听起来可能有点怪，但即使是我在讲经期血液那些事的时候，也没有人比我感觉更好、提供更多乐趣、更无忧无虑、更充满激情和欢乐。这就是她会被吸引的原因。其他男人走过去，试图装酷或索取价值。而我则是：“嘿，价值在这里，看看这个。”没人这么做。
