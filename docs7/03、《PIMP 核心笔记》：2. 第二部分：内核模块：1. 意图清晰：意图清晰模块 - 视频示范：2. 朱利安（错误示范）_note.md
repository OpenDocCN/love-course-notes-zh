# 《PIMP 核心笔记》：2. 第二部分：内核模块：1. 意图清晰：意图清晰模块 - 视频示范：2. 朱利安（错误示范）😅

在本节课中，我们将通过分析一个名为“朱利安”的错误示范视频，来学习在沟通中意图不清晰会带来什么问题。我们将拆解视频中的关键片段，理解其含义，并总结出需要避免的沟通陷阱。

上一节我们介绍了意图清晰模块的基本概念，本节中我们来看看一个具体的反面案例。

视频中展示了一个意图不清晰的沟通片段。说话者的表达含糊且不完整，使用了“It‘s like this little cr”这样的短语，句子没有说完，意思不明确。这种表达方式会让听者感到困惑，无法准确理解说话者想传递的信息或描述的对象。

紧接着，说话者又补充了一个同样不完整的问句“like， do you like that”。这个问句缺乏明确的指代对象，听者无法确定“that”具体指的是什么，导致沟通效率低下。

以下是该错误示范中暴露出的几个核心问题：

*   **句子不完整**：使用了“cr”等未完成的词汇，导致信息缺失。
*   **指代不明**：问句中的“that”没有明确的前指对象。
*   **结构松散**：以“like”等填充词开头，显得犹豫且不专业。

通过这个简单的例子 `不完整的句子 + 模糊的指代 = 困惑的听众`，我们可以看到，缺乏清晰意图的沟通无法有效传递信息。

本节课中我们一起学习了“朱利安”错误示范所揭示的沟通问题。核心在于，支离破碎的语言和模糊的指代会严重阻碍信息传递。在后续课程中，我们将学习如何避免这些错误，使自己的表达意图清晰、直接有效。
