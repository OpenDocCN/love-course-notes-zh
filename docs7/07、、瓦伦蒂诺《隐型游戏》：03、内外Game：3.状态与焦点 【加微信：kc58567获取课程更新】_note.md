# 隐型游戏：07：内外Game：状态与焦点 🎯

在本节课中，我们将要学习如何通过调整你的**思维焦点**来改变情绪状态。上一节我们介绍了通过身体改变状态的方法，本节中我们来看看如何运用你的心智，特别是你关注的内容，来快速有效地调整你的感受。

## 思维如何改变状态 💭

改变情绪状态的第一种方法是你的身体，第二种方法则是你的心智。

如何使用你的心智，尤其是你关注什么？

因此，与其告诉自己“要开心，要开心，要开心”——这通常无效——你的思维中存在一个“后门”。通过这个后门，你有时甚至能比通过身体更快地改变情绪状态，尤其是在你多次练习之后。

## 自我对话与焦点 🔍

当我们关注事物时，我们的焦点由我们的**自我对话**决定。而我们的自我对话，基本上就是在向自己提问。

如果你曾真正倾听过内心的声音，你会发现你几乎总是在向自己提问。而你问自己的问题，甚至能决定你生活的质量，这一点我们稍后会探讨。

但现在，我只想告诉你：**自我对话就是向自己提问，而这些问题的答案，就是你关注的内容，进而使你产生某种特定的感受。**

想象一下，如果你总是在问自己消极的问题：
*   为什么这行不通？
*   为什么我不够好？
*   为什么这次又不会成功？

然后你走出去，假设你看到一个女孩。这时你问自己：“为什么这行不通？为什么这次又不行？为什么那个女孩不会对我有积极的反应？”接着你开始寻找理由——因为她旁边有个男生，或者她看起来很忙——你开始关注这些，你就不会感到充满力量，反而会觉得：“哦，我不该这么做。”

相反，一个拥有积极关注点的人，他会问自己积极的问题，即使面对同样的情况，他也会获得完全不同的体验。

## 大脑的过滤机制 🧠

其运作方式是这样的：即使当我们去酒吧或参加派对时，我们的注意力也只能接收当下正在发生的全部信息中**极小的一部分**。

就像此刻你在听我说话，你可能并没有关注到你右臂或右手血液流动的脉搏。😊 但当我提到它时，你可能会想：“哦，真的有脉搏吗？”——你向自己提了一个问题，然后你就可能感觉到右手之前未曾注意到的感觉。你其实一直能感觉到它，但那是在潜意识层面，你没有把意识焦点放在上面。

在这个房间里，我周围有所有这些灯光，有摄像师确保拍摄完美，我们在进行整个录制，检查音频，我接收着所有这些颜色，周围有很多你们甚至看不到的东西。😊 你们看着我，看着我的啤酒、头发、眼睛，你们可能在自己的卧室里，能看到各种不同的东西——所有这些都在被刺激，但你们**删除了其中大部分**，因为它们不相关，或者信息量太大，大脑没有足够的“内存”来全部处理。因为现实发生的一切实在太多了。

所以我们的大脑只能聚焦于某些事物，并**字面意义上地删除其余一切**。这意味着我们并非在体验完整的现实，而只是体验了其中的一小部分。因此，即使一个情境中可能存在积极因素，如果你不关注它，你就**根本不会看到，也根本感觉不到**。

## 焦点决定感受 🎬

让我们回到酒吧的例子。你可能一生中去过酒吧一次，或者上百次。而你关注什么，将直接决定你的感受。

你可以看到所有人。例如，有些人可能在打架，两个男人在争执。你可能会想：“哇，这太激烈了。这个派对，这里非常危险。” 你不仅只关注这一件事，还可能把它放大，就像用摄像机变焦一样。你放大它，让它显得比实际情况更严重。然后当你看着它时，它在你的脑海中变得异常清晰和生动，这使它感觉非常强烈。😊

与此同时，可能还有人在变得亲密、彼此相爱，另一些人则玩得非常开心。但因为你可能在问自己：“这里有什么危险？这里有什么危险？”你可能只看到了打架。然后你会觉得：“哇，这就是这里的全部现实。” 你甚至可能再也不想出去了，因为你觉得：“哇，太危险了。” 因为你删除了其他一切，只盯着那一件事。然后，这种积极或消极的关注会让你产生不同的感受。

## Reticular Activating System (RAS) 📡

我们的注意力通过一种叫做**网状激活系统**的东西运作，我们简称它为**RAS**。它主要关注以下三类事物中的一类，并直接过滤掉其他所有信息：

以下是RAS主要关注的三类事物：
1.  **这能杀死我吗？**（威胁生存）
2.  **我能与此交配吗？**（繁衍机会）
3.  **我能吃这个吗？**（获取资源）

😊 听起来很简单，但归根结底，我们实际关注的事物，通常都与这三件事之一有关，或是为了让这三件事变得更好。后两者（吃、繁衍）偏向积极，而第一项（生存威胁）则偏向消极。

例如，你现在正在看这个视频。希望你不会经历，但我生活中经历过地震。假设我们正在这里拍摄，突然一切开始摇晃，屋顶开始坍塌。我会想：“哇，我得赶紧跑！” 我不会关心这里的任何其他东西，因为我要确保自己不死。所以，我的注意力瞬间聚焦到那件可能杀死我的事情上。

对你来说也一样。想象你正在看我的视频，你觉得：“嗯，内容不错，我很喜欢看，很有道理。我知道学了这些并实际应用后，我的生活会好很多。” 然后想象屋顶开始坍塌，东西开始往下掉。你会想：“哇，糟了，我得离开！” 你不会再关心这个视频了，因为你会想：“我首先要确保自己不死。” 所以你的焦点会转移到那件可能杀死你、你必须避免的事情上，然后你才能准备好接收信息。

但现在你在家，这里没什么能杀死你（或可能性极低）。所以你可以关注另外两类事物，对吧？我们通常更受痛苦驱动，而非快乐驱动。所以我们总是想先确认：“没有什么能杀死我吗？”或者“我确定没有东西能杀死我吗？”——这甚至包括我们脑海中臆想出来的、完全不真实的事情。一旦你确认安全，你才会想：“好吧，也许这个能帮我更好地找到伴侣，也许这个能帮我获得更好的食物，确保我生存下去。”

我希望所有这些概念现在开始逐渐串联起来。

## 总结 📝

本节课中我们一起学习了如何通过心智调整情绪状态。核心在于理解，我们的**自我对话**（即向自己提出的问题）决定了我们的**关注焦点**，而焦点直接塑造了我们的感受。大脑的**RAS系统**会本能地将注意力优先导向生存、繁衍和资源获取相关的事物。通过有意识地向自己提出积极的问题，我们可以引导焦点，从而改变情绪体验，即使在相同的外部环境下，也能获得截然不同的内心感受。关键在于练习，让这种积极的“思维后门”成为你的习惯。
