# 即兴表演十戒：05：练习5 - 单词联想 🧠

在本节课中，我们将要学习一个经典的即兴表演练习——单词联想。这个练习旨在训练思维的流畅性和快速反应能力，帮助你在大脑放松、无压力的状态下，建立想法之间的自然连接。

上一节我们介绍了即兴表演中“接受”与“推进”的重要性，本节中我们来看看如何通过具体的练习来提升这种思维的流畅度。

## 练习概述

这是一个两人一组的练习，每人进行30秒到1分钟的连续表达。练习的核心规则是：从给定的一个单词开始，构建一个句子，然后从这个句子中选取一个单词（不能是最后一个词），再围绕这个新单词构建下一个句子，如此循环往复。

## 核心规则与要点

以下是进行单词联想练习时需要遵循的具体规则和技巧：

1.  **句子长度适中**：句子不应过长或过短。过长会偏离练习目的，过短则无法提供足够的词汇选择。
    *   **错误示例（过长）**：“我手里拿着一个水瓶，瓶身是蓝色的，上面还有个瓶盖，我不知道这是什么牌子的……”
    *   **错误示例（过短）**：“瓶子很酷。”（几乎没有可选的词）

2.  **禁止选取句末词**：构建新句子时，不能选择前一个句子的最后一个词作为新起点。这迫使你的思维进行跳跃。

3.  **追求流畅与加速**：初始目标是保持思维的连贯性。熟练后，应尝试加快联想和表达的速度。

## 练习流程演示

为了更好地理解，我们来看一个具体的演示流程。假设起始词是 **`bottle`**（瓶子）。

1.  起始词：**`bottle`**
    *   造句：`我手里拿着一个水瓶。`
2.  从上一句选取词：**`拿着`** (holding)
    *   造句：`拿着以字母H开头。`
3.  从上一句选取词：**`字母`** (letter)
    *   造句：`字母也由纸制成。`
4.  从上一句选取词：**`制成`** (made)
    *   造句：`《Made You》是埃米纳姆的一首歌。`

整个过程应像思维自然流淌一样：`瓶子 -> 拿着 -> 字母 -> 制成 -> 歌曲……`

## 练习示范与要点

现在，让我们通过一个更快的现场示范来感受这个练习的节奏。请注意演示者如何快速选取词汇并推进。

（示范开始）
起始词：**`shoe`**（鞋子）。
*   `每个人都买新鞋子。` -> 选取 **`每个人`**
*   `每个人都喜欢新东西。` -> 选取 **`东西`**
*   `人们买东西。` -> 选取 **`人们`**
*   `人们试图在造句时思考事物。` -> 选取 **`思考`**
*   `思考意味着什么？` -> 选取 **`意味着`**
*   `这意味着一切。` -> （尝试加速）`一切皆一，一二三，选择，选一还是选三？哪个？哪个吸血鬼？`
（示范结束）

正如你所见，关键在于**持续流动**。即使感觉困难或“糟糕”，也要强迫自己基于刚刚说出的句子，选取一个新词继续下去。练习的目标是训练这种不间断的联想能力。

## 总结

本节课中我们一起学习了**单词联想**练习。这个练习通过强制性的、快速的词汇跳跃与句子构建，有效打磨了即兴思维所必需的流畅性。记住核心：从给定词出发，造句，选词（非句末词），再造句，并努力使这个过程如行云流水般顺畅。多加练习，你会在即兴表演和日常沟通中感受到思维的解放。
