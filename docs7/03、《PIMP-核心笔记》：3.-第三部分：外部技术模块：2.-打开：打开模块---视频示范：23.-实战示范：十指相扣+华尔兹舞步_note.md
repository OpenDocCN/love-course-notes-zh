## 动作分解与执行步骤

以下是“十指相扣+华尔兹舞步”组合动作的详细分解步骤。

1.  **建立初步连接**：首先，以自然的方式接近目标，伸出右手发出邀请。
2.  **完成十指相扣**：在对方回应后，掌心相对，手指自然交叉相握，形成稳固连接。
3.  **引导华尔兹舞步**：以十指相扣的手为引导点，自身向后小退半步，引导对方向前。
4.  **执行旋转动作**：通过手臂的轻微带动，引导对方以你为中心完成一个简单的华尔兹旋转步。
5.  **恢复面对面位置**：旋转结束后，通过手势将对方轻柔带回初始的面对面位置。

## 核心要点与公式

这个动作的核心在于**引导**而非**拉动**。力量传递应遵循以下原则：

**引导力 = 意图 (明确) + 力度 (轻柔) + 时机 (准确)**

在代码逻辑上，可以理解为：
```python
if 对方手部回应 == True:
    执行 十指相扣()
    while 进行舞蹈引导:
        施加 轻柔的导向力()
        检查 对方跟随状态()
    结束旋转()
    恢复初始位置()
```

## 常见误区与注意事项

在执行此组合动作时，需要注意以下几个关键点。

*   **力度控制**：避免用力过猛拉扯对方，应以手腕和指尖的轻微带动为主。
*   **眼神交流**：在整个过程中保持自然的目光接触，增强连接的亲密感。
*   **节奏把握**：华尔兹步的节奏应是“强-弱-弱”，三步完成，确保动作流畅不仓促。
*   **空间意识**：留意周围环境，确保有足够的空间完成旋转，避免碰撞。
