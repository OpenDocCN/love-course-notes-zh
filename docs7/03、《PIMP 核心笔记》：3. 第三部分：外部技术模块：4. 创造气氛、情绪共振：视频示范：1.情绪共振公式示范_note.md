# 《PIMP 核心笔记》：3. 第三部分：外部技术模块：4. 创造气氛、情绪共振：视频示范：1.情绪共振公式示范 💡

在本节课中，我们将要学习一个核心的互动技巧——情绪共振。我们将通过一个具体的对话示范，来理解如何通过节奏性的互动，在对话中创造吸引力和张力。

上一节我们介绍了创造气氛的基本概念，本节中我们来看看如何通过具体的对话模式来实现情绪共振。

男人通常擅长将注意力持续集中在一件事上，并进行自我娱乐。在实际的现场互动中，当我走到女孩面前开始聊天时，我会在对话中刻意加入一些轻微的“推”，即带有轻微挑战或调侃意味的语句。

以下是一个典型的互动循环过程：

1.  我走过去开始与女孩聊天。
2.  在聊天中，我会加入一个小小的“推”。例如，我说：“😡（表示一个略带调侃或玩笑的指责）。”
3.  这时女孩的反应可能是：“哦，天哪，你怎么可以这么说我？”
4.  我随即进行“拉”，缓和气氛：“啊，没有没有，只是开玩笑，只是开玩笑。”
5.  然后我们继续轻松地聊天。
6.  聊一段时间后，我再次加入一个“推”：“哦，天哪，你怎么这么说我呀？”
7.  我再次进行“拉”来平衡：“哦，没有没有，开玩笑开玩笑。”
8.  这个“聊天 -> 推 -> 拉 -> 继续聊天”的过程会一直反复。

所以，情绪共振的核心公式可以总结为：

**聊聊聊聊聊 -> 情绪高点（推/拉） -> 聊聊聊聊聊 -> 情绪高点（推/拉）**

这个公式的目标是维持一种动态平衡。关键在于让女孩保持在一种情绪被轻微调动、类似“上瘾”的状态。这本质上是引导对方持续进行情绪投资。

我会持续重复这个“施压（推）与减压（拉）”的循环。随着时间推移，正因为这种持续的、有节奏的情绪张力调节，女孩的心理状态会逐渐发生变化，最终导向“我已经准备好和这个男人亲密”的结论。😡。

本节课中我们一起学习了情绪共振的实践公式与循环过程。其精髓在于通过“推拉”制造情绪波动，并引导对方持续投资，从而逐步建立吸引。
