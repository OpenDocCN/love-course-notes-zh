# 朱利安《Ten Game十戒》：05：任务六详解 🎯

在本节课中，我们将详细解读朱利安《Ten Game》的第六个任务。这个任务包含四个核心练习，旨在通过持续外出、延长互动、应对社交压力以及进行非语言开场，来系统性地提升你的社交能力与自信。

## 核心任务一：连续七天外出 🌅

上一节我们介绍了任务框架，本节中我们来看看第一个具体练习。你需要连续七天外出，每天至少20到30分钟。此举的目的是让你稍作休整，为本周高强度的实战练习做好准备。同时，你需要在Facebook群组中每日打卡以督促自己。

**核心要求**：`连续外出天数 >= 7` 且 `每日时长 >= 20分钟`

## 核心任务二：进行夜间长互动 🌃

在掌握了日间互动后，本节我们将挑战在夜间进行。本周你需要完成一次夜间互动，持续时间至少20分钟。

以下是执行要点：
*   **时间**：最好在午夜12:30之后，建议选择周五或周六。
*   **地点**：酒吧等夜间场所。
*   **目标**：互动时长不少于20分钟，并努力使其持续更久，甚至实现关系的进一步突破。

## 核心任务三：社交压力练习 😰

为了巩固你对社交压力的适应能力，本周的练习将升级。你需要在白天，选择一个城市的拥挤公共场所。

具体步骤如下：
1.  走入人群密集的区域（如公园喷泉旁）。
2.  站在人群中央。
3.  计时一分钟，大声谈论你学习并应用《Ten Game》以来的最大收获。

例如，你可以大声说：“自从学习《Ten Game》，我学到的第一课就是如何突破自我设限。我以前非常内向……” 这个练习旨在让你习惯被注视和评判，从而在压力下保持从容。

## 核心任务四：完成三次无声开场 🤐

最后，我们来练习非语言沟通。本周你需要完成三次“无声开场”，可以在白天或夜间进行。

“无声开场”指的是：走向目标，在互动的前10秒内不说一个字。这将迫使你运用沟通的其他层面。

你可以尝试以下方式：
*   **动作**：轻拍对方肩膀，或直接伸出手握手。
*   **表情**：通过丰富的面部表情进行交流，例如挑眉、微笑、做出疑惑或欣赏的表情。
*   **流程**：在10秒无声互动后，再开始正常对话。例如，握手后，用表情示意“怎么样？”，然后再开口说话。

**代码逻辑示意**：
```python
if interaction.time_elapsed < 10:
    communicate(through=“facial_expression”, “body_language”)
else:
    communicate(through=“verbal”)
```

## 总结 📝

本节课中我们一起学习了任务六的四个核心练习：
1.  **持续行动**：连续七天，每日外出。
2.  **深化互动**：在夜间进行至少20分钟的长互动。
3.  **压力适应**：在公共场所大声演讲一分钟，以锻炼抗压能力。
4.  **多元沟通**：完成三次前10秒无声的开场，练习非语言交流。

请严格执行这些任务，它们环环相扣，是构建强大社交自信的基石。祝你好运。
