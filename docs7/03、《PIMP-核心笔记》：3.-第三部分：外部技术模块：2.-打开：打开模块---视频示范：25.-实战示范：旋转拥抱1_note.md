# 《PIMP 核心笔记》：3. 第三部分：外部技术模块：2. 打开：打开模块 - 视频示范：25. 实战示范：旋转拥抱1 🌀🤗

在本节课中，我们将学习一个名为“旋转拥抱1”的实战舞蹈动作示范。这个动作结合了旋转与拥抱的姿态，是外部技术模块中“打开”环节的重要练习。

上一节我们介绍了打开模块的基本概念，本节中我们来看看如何通过具体的舞蹈动作来实践它。

以下是“旋转拥抱1”动作的分解步骤：

1.  起始姿势为舞蹈站立姿态。
2.  执行一个旋转动作。
3.  在旋转过程中或结束时，做出一个模拟拥抱的姿态。
4.  动作结束时恢复稳定站姿。

在练习时，需要注意动作的流畅性与情感表达的结合。舞者曾自问“我能留住她吗？”，但随即意识到“不，我不能，我知道我不能”。这种内心独白有助于为动作注入故事性和情感深度，使舞蹈不仅仅是技术展示。

本节课中我们一起学习了“旋转拥抱1”舞蹈动作的分解步骤与情感表达要点。掌握这个动作有助于更好地理解如何将技术模块与个人表达相结合。
