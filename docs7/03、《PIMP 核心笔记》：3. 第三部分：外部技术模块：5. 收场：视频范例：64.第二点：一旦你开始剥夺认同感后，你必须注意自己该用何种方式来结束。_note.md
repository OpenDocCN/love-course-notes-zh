# 《PIMP 核心笔记》：03：外部技术模块：5. 收场：视频范例：64. 第二点：如何正确结束认同感剥夺

在本节中，我们将探讨一个关键技巧：在运用“剥夺认同感”技术后，如何以正确的方式收场。错误的结束方式会破坏已建立的吸引力，而正确的方式则能导向最终目标。

上一节我们介绍了剥夺认同感的基本概念，本节中我们来看看如何为这个过程画上圆满的句号。

## 核心问题：直接赋予资格的陷阱

一旦你开始剥夺对方的认同感，你必须谨慎选择结束的方式。例如，你通过一系列互动不断叠加挑战感，使对方开始被你吸引并追逐你的认同。此时，如果你突然直接对她说：“我喜欢你。”

**错误示范**：`“我喜欢你。”`

如果你以这种直接的方式赋予对方资格，她会立刻得到她一直追逐的认同感。结果会怎样？她会想：“我得到了想要的认同感，我得到了想要的东西。” 于是，她的兴趣会迅速消退。随之而来的性张力会不断下降，对方也会慢慢离你而去。

## 正确策略：以“推”的方式赋予资格

那么正确的做法是什么？如果你想更极端地运用剥夺认同感技术，你需要强调是因为自己“太喜欢”对方，才会做出这些行为，并且将责任“归咎”于她。

以下是实施此策略的步骤：

在将挑战感叠加到一定程度后，你可以采用类似以下的表达方式：

*   “天哪，我都不知道为什么自己会喜欢上你，你对我做了什么？”
*   “你给我下药了是吧？（营造‘都怪你’的感觉）”
*   “我烦死你了，我都不知道为什么自己现在还喜欢你。”
*   “你就是个小坏蛋，你太坏了。”

这种方式的核心在于，**用一种看似“推”开对方或抱怨的方式，来间接地“拉”近关系并赋予资格**。直到关系发生实质性进展（如发生性爱）。如果你想更极端地使用剥夺认同感，就应始终以这种心态来操作，最终才给予对方她一直寻求的终极认同感。

**核心公式**：`叠加剥夺感 -> 以“抱怨/责怪”的方式表达喜欢 -> 达成最终认同（性爱）`

本节课中我们一起学习了在剥夺认同感后如何正确收场。关键在于避免直接给予认同，转而使用一种“推拉结合”、归咎于对方的方式来表达情感，从而维持张力并导向最终目标。记住，终极的认同感往往体现在关系的实质性进展之中。
