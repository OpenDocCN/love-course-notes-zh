# 老板课程：11：服务标准与奖金设定 🎯

在本节课中，我们将学习如何通过设定个人服务标准来建立边界，并练习在现实生活中应用这一标准，特别是在涉及小费或奖金的场景中。核心在于学会区分并回应不同质量的服务。

上一节我们探讨了个人成长与边界设定。本节中，我们来看看如何将这一理念具体应用到日常的消费与服务互动中。

个人成长始于边界的建立。最初，你可能没有明确的个人标准和边界，容易被人轻视。随着发展，你开始形成自己的标准和边界，成为一个更成熟、稳重的人。

这个过程体现在你与世界互动方式的变化上。过去，当他人越界时，你可能会妥协，那条“界线”总是偏向他人。现在，通过设定价值观和标准，这条界线会向你期望的位置移动，变得更加平衡，例如从过去的80%偏向他人变为更接近50/50的平衡状态。

建立这种平衡有时需要你同意某些事，有时则需要你拒绝。那么，如何在生活中实践这一点呢？

以下是实践这一理念的具体任务，即本周的挑战：

*   **任务内容**：本周外出时，例如去俱乐部、餐厅或乘坐出租车，如果遇到差劲的服务（低于三星标准或二星服务），请不要支付小费。
*   **执行方式**：当服务不佳时，不付小费，并向服务提供者解释原因。例如，你可以说：“抱歉，我认为这次服务没有达到我的预期。”或者说：“希望下次服务能更好。”
*   **核心目的**：奖励优质服务，而非将小费视为理所当然。你需要决定自己对于“合格服务”的界线在哪里，并在每一次涉及金钱的互动中执行它。

以网约车服务为例，其评分体系可以作为一个参考。服务标准可以用一个简单的条件判断来描述：
```python
if service_quality >= 3:  # 三星或以上服务
    give_tip()
else:                     # 低于三星服务
    explain_no_tip(reason="服务未达预期")
```

这可能会引发一些略显对峙的尴尬时刻。你需要学会停留在这种对峙感中，不要逃避。不要攻击对方，而是体会这种感受，并借此激励对方在未来提供更好的服务。

需要注意的是，小费文化在全球各地有所不同。在某些地方，给小费可能被视为不礼貌，而在另一些地方（如北美），不给小费可能被看作失礼。你需要根据所处环境做出判断，但核心原则不变：只奖励真正优质的服务。

总而言之，本周的任务就是：外出消费两到三次，当服务不佳时，拒绝支付小费并礼貌地说明原因。完成后，请将你的经历和报告分享到课程指定的社群或应用中。

本节课中，我们一起学习了如何通过设定清晰的服务标准来强化个人边界，并通过“有条件支付小费”这一具体行动进行练习，从而在人际互动中建立更健康、更平衡的关系。
