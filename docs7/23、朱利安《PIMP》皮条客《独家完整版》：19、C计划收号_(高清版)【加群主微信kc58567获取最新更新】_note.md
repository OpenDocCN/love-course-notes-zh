# 约会教程：19：C计划 - 获取联系方式 📞

在本节课中，我们将学习当A计划（直接发生关系）和B计划（转场后发生关系）因客观条件限制而无法实施时，如何执行C计划——获取对方的联系方式。我们将重点讲解获取号码的时机、方法以及如何为未来的再次见面铺平道路。

上一节我们介绍了A计划和B计划，本节中我们来看看当这些计划因客观条件受阻时，我们该如何行动。

有时，客观条件会阻止任何实质性进展的发生。例如，你遇到一个女孩，她的室友喝醉了，她可能会说：“我不能跟你走，我的室友喝醉了，我得照顾她，我是指定的司机。” 同时，B计划也无法实施，因为车里没有空间，她的室友已经在后座吐了。在这种情况下，无论你给出多少逻辑上的借口，强行推进只会适得其反，将关系彻底破坏。

这时，你应该选择C计划：获取她的电话号码。**你总是可以拿到她的电话号码**。这样，你可以在未来某个客观条件更好的时间再次见到她，并促成一些事情。

**核心公式：**
```
主要目标：A计划 (直接) > B计划 (转场) > C计划 (获取号码)
C计划 = 获取电话号码 (作为备用方案的备用)
```

## 电话号码的本质与定位

首先，必须明确电话号码不是你的主要目标。请始终记住，要优先尝试A计划和B计划。如果这些行不通，电话号码才是备用方案的备用。你不应将其视为通往发生关系的“垫脚石”。有些男人会认为，对于某些“更矜持”的女孩，需要先拿到号码，再通过约会达成目的。这种想法是错误的。**你应始终以发生关系为目标，电话号码只是最后的备用选择。**

有趣的是，当你努力尝试A或B计划时，有时最好不要提前索要号码。因为如果你先拿到了号码，然后说“我们去After Party吧”，女孩跟你走的动力或“筹码”就减少了。她可能会想：“我可以去After Party和这家伙发生关系，或者不去，反正我们已经有号码了，以后还能见。” 如果你不给她号码，她的想法则是：“我可以去After Party和这家伙发生关系，但如果我不去，我可能就再也见不到他了。” 所以，号码从来不是主要目标。

其次，电话号码不应被视为一种小小的“认可奖章”。不要觉得“哦，我拿到电话号码了，太好了”。电话号码本质上是一座桥梁，它将一次完整的互动分割成了两部分。

**核心概念：**
```
完整互动 = [第一部分互动] --> (电话号码作为桥梁) --> [第二部分互动 (未来，更好的客观条件下)]
```
当你无法一次性达成最终目标时，就用电话号码将互动分割成两段。第一部分到获取号码为止，号码是桥梁；第二部分则是你未来在更好的客观条件下计划完成的最终步骤。

## 获取号码的最佳时机

你发现自己最常需要执行C计划（获取号码）的场景，通常是在夜晚的前半段。请记住，在这个时间段，女孩的“情绪温度”或“兴致”通常还比较低。她刚来到现场，还没准备好发生关系。她会想：“我才刚到这儿，不能就这么离开，我想先体验一下情绪上的铺垫。” 因此，A计划和B计划很可能被拒绝。这时你该怎么办？C计划。

当然，偶尔你也会在此时遇到一个“兴致”很高的女孩，可以直接实施A计划。但大多数时候，你只是在执行C计划。所以，在夜晚的前半段，**我总是在获取大量的电话号码**，我称之为“建立潜在关系线”。

以下是这样做的好处：

*   **积累资源**：与其在夜晚前半段只专注于玩乐和积累吸引力，不如同时开始向你交谈的每个女孩索要电话号码，建立这些“关系线”。
*   **创造机会**：到了夜晚后半段，当“兴致”更高时，你可以专注于某个女孩，尝试A或B计划。如果这些计划都没成功，你心想“真倒霉”，但此时你拥有所有之前拿到号码的女孩，你可以给她们发信息、打电话，尝试创造机会。
*   **避免浪费**：你没有让那些互动完全浪费掉。当然，没有互动是真正浪费的，每次行动你都会学到东西。但拿到号码，你就有机会在未来促成一些事情。第二天你仍然可以联系她们。

不要因为她们当时的“兴致”低，就以此为借口不去尝试。如果你真的喜欢这个女孩，只是当时无法带走她，那就拿到号码，机会之门依然敞开着。

**所以，在夜晚的前半段建立“关系线”，是你获取最多电话号码的时候。**

当然，在夜晚后半段也可能发生。有时你正在和一个女孩聊天，但可能你并不想对她投入太多。你会想：“这个女孩的吸引力，不值得我去克服那些糟糕的客观条件。” 例如，一个非常迷人的女孩，但客观条件极差（比如有一群朋友，她是司机）。你可能会想：“这里可能有其他同样迷人但客观条件更简单的女孩值得我投入。” 在这种情况下，你也可以选择C计划。但总的来说，你获取的大部分号码将来自夜晚的前半段。

## 执行C计划的核心步骤

### 1. 明确意图

当你索要电话号码时，首先要确保你的意图非常清晰。你拿号码的目的当然是为了再次见到她。你必须确保她也清楚地知道这一点。因为对女孩来说，给出电话号码并不是什么大事，她们几乎可以给任何人。很多时候，她们给出号码是为了摆脱对方、作为一种奖励，或者只是出于礼貌。

你经常看到这种情况：一个男人走过去开始搭讪，甚至像个醉汉一样对每个人大声讲故事，他就是不离开。她怎么摆脱他？她会说：“给，这是我的号码。” 那个男人说：“太好了，谢谢！” 然后走开，而她再也不会见到那个人。或者，如果那个男人非常努力，她可能会想：“这家伙真好，真贴心，像只努力的小狗。我不想让他难过，不想显得刻薄。” 于是作为奖励，给了他电话号码。

**你不想成为这样。** 你应该传达的意思是：“我现在因为客观条件无法执行A计划或B计划（无法和你发生关系），所以我拿你的电话号码，是为了再次见到你，以便在未来执行A计划或B计划。” 确保她知道这一点，并且要完全坦然，毫不歉意。直接问：“你的电话号码是多少？” 不要用“可以给我你的电话号码吗？”这种语气。

如果你一开始这样做有困难（因为这可能有点吓人），可以养成一个习惯：**规定自己，如果一次互动持续超过2到3分钟，离开前就必须索要电话号码。** 这并不意味着在互动2-3分钟时就立刻索要，而是尽量推进，但如果互动时间超过了这个长度，就养成索要号码的习惯，即使你认为可能拿不到。有时你永远不知道结果，直到你开始向很多女孩索要号码，并开始掌握其中的微妙之处。

最终，你希望这成为一种习惯，不再需要思考。这只是你行为的一部分，是你这个人的一部分。每次和女孩交谈后，很自然地说：“总之，你的号码是多少？” 这非常随意，不是什么大事。

**附带建议**：养成在手机里整理号码的习惯。如果你每次互动超过两三分钟都要号码，你会拿到很多。一个常用的方法是，在她们名字前加一个字母，比如“A Chelsea”、“A Julie”，这样你可以在夜晚“兴致”更高时或第二天轻松找到她们。回家后，再更详细地整理。

如果你忘记了女孩的名字（这很常见），一个简单的方法是反过来问她：“顺便问一下，我叫什么名字？你还记得吗？” 她很可能说“不太记得”，因为大家都会忘记名字。这时你可以说：“什么意思？我也忘了，你叫什么来着？” 她就会告诉你。如果她记得你的名字并反问，不要慌张，轻松应对：“我记得关于你的一切，除了名字。抱歉，我记名字很烂，对每个人都这样。认识一个人时，我最不关心的就是名字，我更在意感受对方的氛围。” 你甚至可以把这变成一个小挑战。

### 2. 预设未来互动 logistics

记住，当你拿到号码，就把互动分成了两部分。它不再是直达发生关系的完整过程，而是：互动 -> 号码（桥梁）-> 第二部分（发生关系）。**你在当面做的事情越多，需要在电话里做的事情就越少**，而当面进行要容易得多。你最终要尽量减少在电话上需要做的工作。

所以，当我要号码时，我会说类似这样的话：“我们什么时候见面？”、“今晚晚点和我见面吧？”、“明天和我见面吧？”。我会在当下就谈论互动的第二部分，以及我们最终如何实现它。这样我就不用在电话里再做这些了。

如果是在夜晚前半段遇到的女孩，下一个可用的见面机会是什么时候？是当晚晚些时候，当“兴致”更高时。所以，当我拿到号码时，我会说：“听着，你可能得回你朋友那儿，我也得回我朋友那儿。我们都在营造各自的情绪氛围，不想接下来四小时只和对方聊天，那多无聊。我们晚点见吧？你的号码是多少？” 或者“晚点一起喝一杯？你的号码是多少？” 我会在当面就种下当晚晚些时候见面的种子，然后讨论客观条件是否允许。

如果她当晚不行，或者你是在夜晚后半段和她交谈，那么下一个可用的机会就是第二天。你可以说：“那明天和我见面吧？你明天做什么？” 我个人常用的方式是：“明天和我喝杯咖啡吧？你明天做什么？” 正如我们之前讨论过的，引入“第二天喝咖啡”这个概念，有助于让你在她现实中更真实、更接地气。这传达的信息是：“我不是只在娱乐场所出现的有趣情绪，我是一个真实的人。” 这也强化了你拿号码是为了再次见她的意图。即使我第二天很忙，我也会先这样提议，之后可以再改。

一旦你提出这个建议，她可能会说“我明天可能很忙”或者“好的”。**你们讨论的细节越多越好。** 比如：“我们什么时候见？你能下午4点吗？5点呢？” 细节越多，它就越成为一个现实的选择，而不是一句客套的“好啊”。如果她第二天不能喝咖啡，那也没关系，但要弄清楚客观条件：“你明天晚上做什么？后天呢？你什么时候有空？” **当面做得越多越好。**

### 3. 尽快安排见面

必须强调，在为互动的第二部分设定和确定客观条件时，**尽快安排见面至关重要**。一旦你决定用电话号码作为桥梁将互动分割，中间间隔的时间越长，她放你鸽子的可能性就越大，你也越不可能再见到她。**你要让这座桥尽可能短，让事情尽快发生。**

如果我和一个女孩交谈，决定引入第二部分并要号码，那么我问自己：下一个可用的见面机会是什么？如果是夜晚早些时候，就是当晚晚些时候；如果是夜晚晚些时候，就是第二天。如果她第二天不行，那就尽快找下一个时间。不要拖延，也不要计划一周以后的事情。

你必须意识到，女孩们**时时刻刻都在被搭讪**。尤其是如果她们非常迷人，会有很多男人上前搭讪、要号码。你希望趁你还在她脑海里时，尽快促成一些事情。等待的时间越长，你在互动中建立的所有投资、所有“资格认证”、所有的共鸣感都会消失。所以，尽可能快地安排。

### 4. 增加投资与资格认证

第三件要做的事，除了尽可能缩短桥梁间隔外，就是**增加她的投资和强化你对她的“资格认证”**。这能极大地降低她放你鸽子或不再见你的可能性。

例如，当我要号码时，我可能会说：“你明天做什么？我们明天喝咖啡吧，见面聊聊。” 她可能说：“好啊，听起来不错。” 为了增加投资，我可能会创造一个“投资窗口”：“你是真的想，还是只是说说？” 确保你的语气不是不安全感，而是一种带着自我娱乐感的挑战性语气，就像在说：“你不想的话可以不用，我们也可以再也不见，我无所谓。” 她可能会说：“不，我真的想。”

你还可以在拿到号码时这样说：“说‘给我打电话’。” 她说：“给我打电话。” 你说：“再说一次，要 convincing（令人信服）一点。” 这就是在增加投资。

此外，更多地“资格认证”她。比如：“说真的，我其实挺喜欢你的。” 或者“我现在对你很好奇。” 如果她问为什么，你可以说：“等我见到你时告诉你。” 或者说：“很高兴认识你，很期待和你见面。我知道一个很棒的咖啡店。” 通过这种方式，增加投资，强化资格认证。

### 5. 保持 discreet（谨慎）

最后，**索要电话号码时要保持 discreet（谨慎）**。这类似于克服“最后一刻抵抗”时，不要突然掏出避孕套让她觉得自己像个荡妇。同样，不要在要号码时突然掏出手机。

即使她的朋友喜欢你，如果你掏出手机，所有的目光都会集中在这个女孩身上。大家会在心里“评判”她：“哦，他要号码了，看她会怎么做。她会更随便一点吗？会给号码吗？” 她会感觉到这种压力，有时仅仅因为不想被评判而拒绝你。或者，她的社交圈里可能有个喜欢她但她不喜欢的男人，她为了维持在那个人眼中的价值，也可能拒绝你。

所以，**要 discreet（谨慎）地获取号码**。即使你成功拿到了，有时你一离开，她的朋友可能会说：“哦，你给那家伙号码了？” 这会带来社交压力。因此，要 discreet（谨慎）地进行。
