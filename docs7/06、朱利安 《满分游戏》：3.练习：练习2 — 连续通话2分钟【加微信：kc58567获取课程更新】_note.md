# 满分游戏：06：练习2 — 连续通话2分钟 🎤

在本节课中，我们将学习一个名为“两分钟连续通话”的练习。这个练习旨在帮助你克服内心障碍，提升表达的流畅度和速度。通过强制性的不间断说话，你将学会如何关闭自我评判，让想法自然流出。

---

## 练习概述

这个练习需要两人一组进行。它的核心规则非常简单：在整整两分钟内，你必须背对同伴，不间断地快速说话。你不能有任何一秒的停顿。

## 练习规则详解

以下是练习的具体规则和步骤：

1.  **分组**：两人组成一队。
2.  **姿势**：其中一人转身，背对同伴。
3.  **任务**：背对同伴的人开始说话，持续两分钟。
4.  **核心要求**：说话必须**不间断**。你可以讲故事、提问，但绝不能停下。
5.  **监督**：聆听的同伴负责监督，确保说话者没有停顿。

**关键代码规则**可以概括为：
```plaintext
while (timer < 2分钟) {
    说话();
    // 不允许出现 break; 或 pause;
}
```

## 练习目的与技巧

上一节我们明确了规则，本节中我们来看看这个练习的目的和完成技巧。

这个练习极具挑战性。两分钟听起来很短，但在持续快速表达时会感觉非常漫长。完成它的唯一方法是彻底关闭你的大脑。

如果你在过程中有任何自我评判，比如思考“我这样说合适吗？”或“这听起来蠢吗？”，你将无法达到高速。你的话语应该像水流一样自然涌出，无需逻辑分析。

## 如何提升与日常应用

完成基础练习后，你可以通过以下方式提升难度并将其融入日常训练：

*   **增加时长**：当你能够轻松完成2分钟后，尝试5分钟、10分钟甚至15分钟。
*   **每日练习**：建议每天早晨进行。你可以对着镜子中的自己练习，进行不间断的独白。
*   **效果**：如果你能达到15分钟不间断高速表达，你在社交场合的表现和结果将得到巨大提升。

## 总结

本节课中，我们一起学习了“两分钟连续通话”练习。我们了解了它的**规则**（两人一组、背对说话、绝对不间断），明确了它的**核心目的**（关闭自我评判，实现思维与表达的流畅同步），并探讨了**提升方法**（增加时长、每日对着镜子练习）。坚持这个练习，能有效打破内心障碍，让你在任何场合都能更自信、流畅地表达。
