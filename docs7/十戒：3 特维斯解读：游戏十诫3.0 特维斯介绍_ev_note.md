## 概述

我是特维斯，是德里克（全球最受欢迎的现场约会教练）的主要训练营导师和学徒。我于2017年亲身结识德里克，在聆听他的分享后，我意识到当学生准备好时，导师自会出现。尽管在首次参加训练营前，我仅有两个月出入酒吧和夜店的经验，并且当时正考虑婚前守贞，我仍决定加入训练营。

截至本次录制时，我以客户身份参加了三次训练营，以助教身份参与了70次，在两年内与超过60位女性建立亲密关系并成功收尾，并一直在全球范围内教授他人掌握相同技能。

本解读凝聚了2000多条详细的社会互动细节笔记，以及我与德里克共同参与的超过73次训练营的经验（此前的助教无人达到此数量）。

我的背景是：我以班级第一的成绩毕业，拥有常春藤联盟的认知神经科学教育背景。我花费十年时间学习一门乐器并达到专业水平，曾师从该领域的一些顶尖大师。我曾为多家财富500强公司担任分析师。我的成长过程也经历了一些非常严峻的困难，其中一些甚至引起了国际关注。此外，我研习自我提升领域已超过十年。

我制作本解读的目标，是引导一个完全的新手达到高级水平，正如我自己在极短时间内做到的那样。在决定参加德里克的三次训练营之前，我仅有两个月的实战经验。而在最后两次训练营中，我每晚都能成功建立亲密关系并收尾。

你将在本解读中听到一些在这个社群中前所未闻的内容，它们将挑战你的思维方式。但我鼓励你实践这些内容，因为我已经在自己和我们的客户身上测试过一切，它们确实有效。我见过德里克的每一位前助教，包括肖恩，他们都对我的德里克核心原则解读感到特别兴奋，并很高兴得知我正在添加解读，因为我是唯一一个从近乎新手起步，却能迅速达到高级水平的人。

这些都是非常高级的概念，我建议你慢慢消化。尽管如此，我将一步步引导你，告诉你从开场到收尾应该做什么，同时避免过于琐碎的细节，以免妨碍你的成功。

这门学问难以教授，因为我们不希望冒险变得过于机械化。因此，我们必须间接地教学。我会尽可能具体地说明你应该做什么，但如果我再具体下去，反而会破坏课程效果。所以，请尽你所能遵循指示。除了我在这些戒律中提供的具体指示外，没有其他需要知道的信息。

我的目标是为你提供一个系统，让你能够以最小的阻力持续外出，继续你的生活，找到心仪的女孩，并以一种让性成为自然结果的方式与她沟通。你做得越多，你的生活改善就越多。
