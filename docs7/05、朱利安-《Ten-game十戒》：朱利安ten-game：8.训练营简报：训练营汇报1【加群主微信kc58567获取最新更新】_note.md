# 吸引力法则：05：训练营简报与核心心态调整

在本节课中，我们将深入探讨在社交互动中至关重要的核心心态。我们将学习如何摆脱对结果的执着，专注于自我愉悦，并通过调整行为模式来建立真正的吸引力。关键在于将整个夜晚视为一个整体，而非一系列孤立的搭讪。

## 核心心态：放弃追求议程

上一节我们介绍了社交互动的基本框架，本节中我们来看看如何从根本上调整心态。关键在于不带着“必须达成某个目标”的议程去互动。

即使面对女孩时，心态也应该是完全无所谓。可以和她一起回家，也可以独自回家。这需要逐步培养。记住我们在夜晚外出后讨论的内容：在训练营结束后，进行几个晚上的“无目标”练习——不设定收号或约会目标。

这很大程度上也是一种“断开连接”，因为你可能听说过“社会 conditioning”。社会 conditioning 有好有坏。同样，在摆脱社会 conditioning、用搭讪技巧重新 conditioning 自己时，也可能陷入一些误区。其中一个就是**为了寻求认可和结果而行动**，本质上变成了“终极讨好者”。

很多男人进入这个领域，并不是真的想要那个女孩，甚至不是想在当晚有所成长。他们只是希望每个人都喜欢自己，永远不想搞砸任何一次互动。他们宁愿每个女孩都喜欢自己，而不是真正发生什么。这又是严重的讨好型人格。

一个重要的心态转变，并且能极大减轻压力、最终产生更多结果的，是真正理解并接受：**你不必对每个女孩都成功**。你不需要，也不应该，而且现实中也做不到。

每个人都有那种需求：和女孩说话时，想以高光时刻结束，想感觉“我做得很好”。但事实是，你不必让每个人都喜欢你，也不是每个人都会喜欢你。一旦接受了这一点，压力就会消失很多。那种“随时随地搞定任何女孩”的想法是个神话。有些女孩就是不适合你，有些女孩你深入了解后甚至可能会讨厌。

## 关键行为调整

以下是基于核心心态需要调整的具体行为：

*   **不害怕直接离开**：你不必总是以“我做得不错”的感觉离开。只要不是把它当作逃避所有互动的借口，你可以直接放手。你的夜晚应该持续进行，感觉不到任何低落。
*   **将整晚视为一个大组合**：不要把每次搭讪看成独立的。整个夜晚就是一个大的互动。你在组合之间的行为和你在组合内的行为同等重要。你所做的不是只在和女孩说话时才“开启”状态，而是整晚都保持一种体验。
*   **重新定义“开场”**：你最初被女孩的外表吸引，但你上前时说“嘿，我来提供乐趣”，你还没有完全“卖”给她。你还不确定是否会喜欢她。所以不是去讨好她，而是去发现她是什么样的人。心态要从“努力让所有女孩都接受我”转变为“我把自己展现给所有女孩，然后筛选掉不合适的，找到有化学反应的那一个”。
*   **关注自身感受**：判断你表现好坏的唯一标准是：你是否比之前更开心、更无忧无虑、更投入？而不是女孩是否喜欢我，或者我是否得到了号码。我们的目标是像给一个气球打气一样，让自己在夜晚结束时达到最佳状态。当然，在这个过程中你自然会获得一些线索和号码。
*   **关注表达方式而非内容**：如果女孩对谈话内容不感兴趣，不要只看内容，要看你的**表达方式**。一个很好的例子是乔·罗根的播客。人们喜欢它，很多时候并不是因为讨论了多么新颖的内容，而是因为他表达的方式和带来的感觉。女孩也一样，她不在乎你具体在谈论勺子还是播客，她在乎的是当你说话时她感受到的情绪。这就是你的独特性和价值。只有你能以那种方式说出来，并让她产生那种感觉。词语只是传递你个人特质的载体。
*   **自我愉悦，激发自身情绪**：当你说话时，把它变成她听过的最棒的“嘿，你好吗”，加上你独特的印记。当我提到“刺激女孩情绪”时，我实际上是在刺激**我自己的情绪**。如果感觉太无聊，我会说点什么来打破它，比如突然问“你早餐吃了什么？”，这打乱了我自己的思维，从而为互动注入了火花。
*   **摆脱过滤器，让互动自然发生**：一旦你不再试图搞定每个女孩，就会帮助摆脱所有过滤器，你也不必担心推拉等技巧，因为它们会自然发生。你自然会进行一些 playful 的争论。在外出时，要“ dumb it down”（放轻松，别想太多）。虽然你仍然有聪明、理智的思维，但要放大你内心“傻小孩”的那一面。没有任务，但微妙之处在于：你的任务就是没有任务。你通过放手和允许事情发生来积极地追求它。这就像把鱼吓跑 vs. 让鱼自己游过来。

## 培养个人幽默感与自我表达

现在，让我们深入探讨如何培养你独特的幽默感和自我表达方式，这是吸引力的核心。

你需要思考并探索：我觉得什么东西有趣？我幽默背后的模式是什么？你可以在“ sticky points”（记录点）做这件事：写下你说过的三件有趣的事。坚持一个月，你会积累很多，然后按主题分类。这样你就知道该按哪个“按钮”来调动情绪。这因人而异。

另一个层面是回顾你的生活，看看你小时候觉得什么有趣，你独特的特质是什么。如果你不太熟悉，可以看一些喜剧，学习如何真正地大笑，不只是微笑，而是开怀大笑。

女孩不在乎笑话或幽默的内容，她想感受到**你说那话时让你自己感受到的情绪**。所以对我来说，我喜欢拿不同的刻板印象开玩笑，或者制造一点尴尬，带着一本正经的表情说些话，或者开一些让人摸不着头脑的玩笑。这些是我觉得有趣的方式。关键是那种情绪，那个“按钮”，我可以把它用在任何情境中。

当你的焦点甚至不在女孩身上，而在你说话的过程本身时，这也是放开结果的关键。当你如此专注于自我愉悦时，你甚至没有时间去注意结果。如果你在问自己“我是否在意结果”，那你已经在在意了。

## 实用技巧与现场调整

以下是更实用的一面需要牢记的事项：

*   **夸大你的表现力范围**：当你笑的时候，夸张一点。保持自己更多地处于运动状态，这是最大的秘诀之一。即使站着不动，也要保持一些微小的动作。
*   **组合之间，别忘了问自己“下一步做什么”**：不要给自己时间在现场分析刚才的互动。扼杀那个逻辑思维，带出那个有趣的小孩。可以开一个新组合，可以来找我，可以互相闹着玩。我们应该建立那种好玩的氛围。离开一个组合时击掌，来找我时带着大大的微笑击掌。
*   **练习处理后勤和收号**：表现得好像每次互动后都有“10分”要打。进行后勤安排和收号，并养成习惯。最好在互动结束前、在她走开之前要号码。你需要感知互动的节奏。
*   **玩得更开心**：一开始，真的去和女孩闹着玩，更离谱一点。不要害怕搞砸，只要不太过分。也不要害怕和没有吸引力的女孩玩，学会和她们 genuinely 地玩得开心。你会发现，不好看的女孩往往有最好的 verbal game。
*   **融入与带动**：当你和女孩说话时，应该总是想着“我怎么能更有趣？怎么能让更多人参与进来？” 把她的兴趣引向其他人，一开始没有特定目标。
*   **处理后勤和带走**：我们将更多地思考如何处理后勤和带走。我可能会把你们叫到一边，就各种情况提问，比如：一旦有了 logistics，如何转场？如果她说这个怎么办？如果发生那个怎么办？
*   **接近方式**：永远不要直接冲进去。我通常轻拍两下。你可以感受一下群体的“温度”，然后选择相应的行动方案。从一开始就握手，养成习惯，身体前倾和后仰。这很难，但女孩需要这个。所以前倾，然后开始让她们来追逐你。

## 针对特定问题的建议

针对一些常见问题，以下是具体建议：

*   **关于长互动**：很少有长互动（45分钟到1.5小时）能在一个小环境中完成，那太无聊了。你必须经常带着组合移动。
*   **当她说“我得去找朋友了”**：这没什么不好。你可以尝试说“嘿，你知道的，待会短信你”等等。但一旦感觉到她仍然非常担心朋友，我会说“我帮你找他们”，把找朋友变成共同的任务。从她的角度看，如果她的目标是找朋友，而你成了她达成目标的障碍，即使你很酷，也不会合拍。微观层面也是如此。
*   **关于行走的组合**：你可以尝试让他们停下来，但如果你感觉到他们还是要继续走，或者停了一分钟但仍然要去别处，那就和他们一起走。永远不要把自己置于他们目标的障碍位置。理想情况是找到让双方目标融合的方法。
*   **关于过早退出**：意识到“退出”是一种习惯，也是一种自我设限的成功屏障。你在做你习惯的事。你想让自己对“停留更久”脱敏，建立这方面的“肌肉”。一个简单的纠正习惯的方法是每次有离开的冲动时，用另一个习惯替换，比如稍微移动一下位置。即使只是移动几英尺，新的环境也能让互动焕然一新。
*   **关于表达想法**：学会将你的想法、感受用语言表达出来。你的脑海里总有声音，总有情绪闪现。我们倾向于评判它们，然后担心“如果我说了这个别人会怎么想”。现实是，摆脱这种想法，把它们都说出来。一旦掌握了这个，你永远不会没话说。当你感到“有资格”时，你就不那么担心别人对你的话的看法，从而允许自己全部释放出来。
*   **关于不把事情个人化**：这不只是不把事情个人化，更是意识到这是自然的方式。如果你完全展现自己，每个人都喜欢你才是不自然的。所以要接受这一点，就像接受自然法则一样。当然我会被拒绝。甚至不要把它看作坏事，就想着“当然会这样”，然后进行下一个。
*   **关于记录有趣的事**：当我说“有趣的事”时，不要想着必须达到单口喜剧笑话的水平。可以是让你比平时笑得更开心的事。你需要在每次互动中找到三件这样的事，可以是一句话或一个行为。降低标准，你会发现更多。女孩不看内容，她看内容引发的情绪。这必须是你个人独特的。
*   **关于眼神接触**：注意我的面部表情，因为我的游戏中80%都在面部表情上。当我与女孩交谈时，我会用眼睛和脸做很多细微的动作，通过它们投射不同的情绪。注意那种“激光眼神接触”，它不仅仅是直视，而是一种稳定的、带着意味的注视。即使你添加一点点，你说的话也会带有完全不同的含义和情绪。
*   **关于站位和能量**：不要过多地站在边线上，直接跳进行动的中心，把注意力吸引到你身上，不要回避注意力。当你站在她面前时，确保你在前倾和后仰。为了让女孩了解你，她需要体验你。你离某人越近，她体验你就越强烈。靠近时，也要确保后仰，因为她体验你更强烈，但也意味着可能太快达到那个程度。要意识到这一点。我总是在进进出出，从来不是静止的。
*   **关于身体语言和能量**：投入你的身体，不要玩“雕像游戏”。要动起来，放松。你们俩在组合之间要多闹着玩，真正建立一个有趣的氛围。
*   **关于挑战自我**：主要是学会承受更多压力。更多压力意味着当你想离开时坚持住，意味着把人们的注意力吸引到你身上。建立那方面的肌肉。拥抱那种张力。

## 从“男孩”到“男人”的转变

最后，我们来谈谈一个重要的转变：从“男孩”到“男人”。这很重要，要对任何带有“男孩”特质的东西感到厌恶。

有一个关于小公牛和老公牛的故事。它们站在山上，看着下面所有的母牛。小公牛很兴奋，说：“嘿，我们跑下去搞定一头母牛吧！” 老公牛看着下面说：“不，儿子，我们走下去搞定它们所有。” 这就是你的目标。

有很多方式来形容这种转变。这关乎控制力。虽然保持兴奋是好的，但有两件事：一是这源于一种“需求感” vs. 只是“想做”；二是对女孩来说，这显得太努力、太强烈了。这带走了踏实感，也让她没有机会追逐你，并且难以产生共鸣。更强大的方式是做得更少，更加踏实。

另一种看待方式是《教父》中的马龙·白兰度。那就是你应该有的举止。你可以是 intimidating 的，但也可以是温和而坚定的。那甚至更有威慑力。这就是我想要的：不是极度 aggressive，而是温和而坚定。你仍然可以有这些爆发的乐趣，但这源于不同的内在位置。这是些细微的调整。

所以，我会一层层分解：首先，注意你的行为，开始厌恶它，让它安静下来，将你的注意力导向它，任何过度反应、过度兴奋的，都慢下来。然后，尝试找到用更少的行动达到同样效果的方法。再然后，尝试变得“无聊”。这对你来说将是一个挑战。你现在习惯做的外向、疯狂的事情已经不是挑战了。真正的挑战，也是更难的地方，实际上是控制住它，不按照自动模式行事，放慢速度，让它更冷静，甚至更无聊，同时仍然能吸引女孩。这就是你必须去的地方。尝试控制它，尝试放慢动作，尝试降低兴奋度，尝试找到那个“刚好足够”的底线，那个仍然能得到女孩的最无聊的底线。看看那条线在哪里。这就是挑战。

你可能会发现自己都觉得无聊，困难的部分在于如何在放慢速度的同时仍然保持投入。不要把它看作“浪费了时间建立起来”。你必须先建立起来才能达到这一点。如果你是个害羞的人，你不能直接去看你能变得多无聊。你必须先学会激发，但然后你不能迷失在激发中。现在要回到完整的循环，这样你就两者兼备了，那就完美了。

这会很难，但记住这一点会打开一个全新的世界。一旦你习惯了，一旦它烙印在你的脑海中：“这就是‘足够好’的底线”，从那里，你可以选择重新将激发融入其中，但正如我提到的，这不是因为你必须这样做，而是因为你想这样做。

我和你的一个细微差别是：我们都可以走到一个女孩面前，我可以说些恶心的话，但我的方式更轻松、更随意，因为我没有必要这样做。对你来说，更像是“游戏时间到了”。对女孩来说，感觉是不同的。有两种存在方式：一种是你总是做做做，因为你认为你需要；另一种是更偏向于控制，更像是“不让你的思维干扰你，在陌生的环境中感到自在”，而不是强迫某事。所以这更像是抵抗那些冲动，更像是控制 vs. 行动。这就是肌肉、焦点和注意力所在的地方。

当我外出时，我当然在行动，但我的焦点更多是“让我们保持在这种环境中自在，对结果自在”。更像是保持踏实和自在，而不是试图强行推动什么。当行动中少了一些“做做做”时，氛围就不那么 aggressive 了，更像是 centered，像大猩猩而不是吠叫的狗。人们不会感觉到那种对话。

我仍然会做一些小爆发。这就像给她一个“预览”。如果我感觉需要快速得到一个好反应，或者需要去别的地方，我会控制住自己。所以你的焦点应该放在保持那种轻松感上。

成为男人或保持踏实、有吸引力的第二部分是控制你的情绪。所以，即使你确实感到兴奋，这也是一种肌肉，你的焦点应该放在：慢下来，不要陷入其中。如果你是一个容易感到兴奋并过于投入的人，女孩会觉得你缺乏情绪控制力。这很合理。因此要踏实和控制。缺乏控制并不意味着压抑情绪，你仍然感受它们，但你应该能够采取视角，并以一种校准过的、主动的方式行动，而不是疯狂地对它们做出反应，让它们压倒你。

这是以一种方式与她连接，是共情的一面。你们学过惯例，从惯例中得到的一个技能是：你展示惯例，然后看女孩的反应，然后根据她的反应选择下一个。有趣的是，这也是我希望你更多关注的技能。现在没有那么多惯例了，你是完全做自己并表达一些东西，但这不只是不顾女孩感受地做自己，否则你会变成那个喝醉的疯子。他是在做自己，但没人能产生共鸣。如果你太疯狂，也可能陷入那种状态。所以更多是：我做我的有趣的事，但我总是回头看，确保她仍然参与其中。所以就像：有趣 -> 回头看 -> 她有点紧张 -> 我会做一些仍然一致但更冷静的事，让她能产生共鸣 -> 如果太无聊她开始看别处 -> 我会做一些仍然一致但更出格的事来让她重新投入。总是：我做 -> 我看 -> 我做 -> 我看。但你必须“看”并读懂她的感受和想法。

我可能会就此考考你。我会来找你，问：“这个女孩现在感觉如何？她在想什么？” 让你以这种方式思考，设身处地为她着想，看看你的解读是否有任何出入。

找到在环境中感到自在的方法，找到减压的方法，找到缓解紧张的方法。你能拥有的最糟糕的氛围就是“酒吧电池”，就像被电击一样。这种氛围可能会暂时有效，但不可持续。女孩最初可能喜欢，但如果你刺激过度，要么难以产生共鸣，要么她们会觉得“这家伙太过了，他慢不下来，这源于一种需求感和不安全感”，然后她们就会离开。

另外，如我所提到的，尝试降低兴奋度，但要以一种仍然一致的方式进行。尝试进行更多真正的对话，而不只是说有趣的话。给女孩空间说话，暂停，后仰引导。也要享受其中，让她参与进来。因为你总是在做做做，但她也需要参与。如果你想让她追逐，那么即使我前倾说了些什么，然后我会后仰，让她来“品尝”。理想情况下，你本身已经足够有料，以至于你可以很无聊但仍然能得到那个女孩，你甚至不必疯狂，因为你本身就足够酷。
