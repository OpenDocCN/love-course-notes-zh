# 麦克斯《女朋友游戏》：3.1：一夫一妻制或与其他女孩发生关系 🧠

在本节中，我们将探讨一个极具争议性的话题：一夫一妻制与多偶关系的对比。我们将从生物学、心理学和社会叙事的角度进行分析，旨在拓宽你的思维，让你以更开放的心态思考亲密关系的可能性。

## 核心争议与开放心态

本节内容可能是整个课程中最具争议的部分。它旨在提供一种不同于主流叙事的思考方式，作为一种平衡。首先，请保持开放的心态。

## 一夫一妻制的生物学视角

上一节我们介绍了本节的主题，本节中我们来看看生物学基础。我将提出一个大胆的观点：**一夫一妻制是一个谎言**，是主流叙事灌输给我们的观念。

从生物学角度看，一夫一妻制并非我们灵长类祖先的常态。在我们的基因中，并不存在专一的一夫一妻制倾向。

*   **非一夫一妻的灵长类**：与我们亲缘关系接近的黑猩猩、倭黑猩猩、猩猩都实行多偶制。
*   **唯一的一夫一妻制灵长类**：长臂猿是唯一实行一夫一妻制的灵长类动物，但它们在约2200万年前就与我们的进化路径分离了，亲缘关系甚远。

如果你相信进化论，这是一个生物学事实。如果你信仰宗教，认为亚当夏娃是一夫一妻制，那也很好。但本节旨在从生物学和现实角度进行探讨。建议阅读克里斯托弗·瑞安和塞西莉亚·吉哈的著作《性在黎明》，以更开放的心态了解相关研究。

## 对个人欲望的诚实审视

现在，让我们放下自我，诚实地思考一夫一妻制和个人欲望。

请诚实地问自己以下问题：

*   你是否真的只对一位女性产生过欲望？
*   当你看到其他女性时，是否从未感到过 arousal（性唤起）？
*   你能想象在未来数年里只与一位女性发生性关系，并且对你的性生活感到100%满足，同时看到其他女性时毫无感觉吗？

## 性行为与爱的分离

让我们深入探讨一个核心问题：与其他女性发生性关系，是否意味着你减少了对伴侣的爱？

思考以下场景：你与一位你深爱的女孩处于关系中。你与另一位对你毫无意义的女孩发生了关系。这是否真的削弱了你与伴侣的情感连接？

对比一下**自慰**的行为。当你观看色情内容时，你显然被其中的女性所唤起。这是否意味着你减少了对你女朋友的爱？可能不会。那么，自慰和与另一位女性发生关系，除了涉及 ego（自我）和社会叙事驱动的“背叛”概念外，在本质上有什么区别？

与其他女性发生性关系，是否自动意味着你爱你女朋友更少？这是否意味着你不会为她付出一切、保护她、爱她？

## 案例假设：保罗与珍娜

让我们通过一个假设案例来具体分析。假设一个名叫保罗的男人。

保罗深爱并珍视他的女朋友珍娜。他愿意为她做任何事，每天想着她，带她享受美食，与她拥有热烈的性生活，并向所有朋友夸赞她。他崇拜她，愿意为她牺牲。

但偶尔，当保罗遇到一个有吸引力的女性时，他会去接近她。当珍娜不在身边时，他可能会与这位女性发生关系。他使用了安全套，度过了一段愉快时光，然后精神焕发地回到珍娜身边，心中对她的爱丝毫未减。他们的关系继续顺利发展。这真的那么糟糕吗？

## 关键前提与沟通

现在，珍娜会对此有何感受？她的价值作为伴侣是否被贬低了？她是否遭到了背叛和欺骗？保罗是否应该对她诚实？

让我们在三个关键前提下继续思考：

1.  **保罗始终对珍娜非常好，是完美的伴侣。**
2.  **珍娜拥有高自尊，清楚知道自己在关系中的价值。**
3.  **保罗对此并不感到抱歉，他认为性是美好而自然的，无需羞耻，并且他真心认为珍娜是最适合他的人。**

我们甚至可以进一步假设：他们事先就此进行过沟通，就像在“设定边界”章节中讨论的那样。保罗坦诚地说：“嘿珍娜，上周我遇到了一个很有魅力的女孩，我们约会并发生了关系。这没什么严重的，只是两个人享受了几个小时的快乐。你依然是我的女王，那个女孩对我毫无意义。世界上没有任何事能动摇你作为我第一优先的地位。事实上，我现在感到精力充沛，焕然一新，我觉得我更爱你了。”

## 多偶关系对男性的潜在影响

在上述假设下，保罗感觉如何？通常，与一位有吸引力的女性发生关系后，男性可能会感觉：

*   更像一个男人
*   精神焕发、充满活力、受到鼓舞
*   准备好征服世界
*   感到自身价值更高、更自信

**公式：** `性经历` -> `积极情绪与自信提升` -> `在伴侣眼中更具吸引力`

这些难道不是非常优秀的特质吗？这难道不会让你在女朋友眼中表现得更加出色和性感吗？

## 对比场景：压抑的后果

让我们想象相反的情况。假设保罗因为珍娜自尊低，且他不了解“四个B”（课程中的其他概念），多年来一直处于性压抑和“beta男性”状态，在关系中非常不快乐。

这样的保罗会感到上述的精力充沛、自信、高价值吗？他还能吸引珍娜吗？还是珍娜也会因为他的 beta 男性行为而失去吸引力，两人仅仅因为舒适感而勉强在一起？

## 多偶关系的“共赢”视角

让我们审视两种情形对珍娜的影响：

**情形A（多偶）**：保罗与另一位女性发生关系。他感到自信、焕然一新，使用了安全套，没有人受到伤害。每个人都对此诚实。这是一场共赢：
*   随机遇到的女性获得了一段与高价值男性的愉快经历。
*   保罗（如果诚实，这就不是“欺骗”）变得敏锐、自信、准备好征服世界，并准备给予女友更多的爱。
*   珍娜获益，因为她拥有一个高质量的男性伴侣。

**代码逻辑：**
```plaintext
if (诚实沟通 + 安全措施 + 高自尊双方) {
    结果 = “多方共赢”;
} else if (欺骗 + 不安全 + 低自尊) {
    结果 = “伤害与问题”;
}
```

唯一受损的是被 ego 卷入其中的人。主流叙事中的许多说法可能并不合理。

## 对等思考：伴侣的多偶权利

那么，保罗是否也应该允许珍娜与其他男性发生关系？这是一个很好的问题。

首先，想到你的女孩与其他男性发生关系，你感觉如何？你是否感到愤怒、好斗、嫉妒？你为什么要求她不能这样做？这背后隐藏着什么不安全感？你是否真的拥有高自尊？

让我们做一个对等的假设。假设你的女朋友疯狂地爱着你，向所有朋友夸赞你，每天早晨为你口交，为你做饭，认为你是最聪明、最强壮、最性感的男人。同时，你自我感觉也极佳，生活各方面都很成功，感到强大和满足。

此时，你的女朋友外出与一位健身房遇到的肌肉男发生了关系。他可能长得帅，但你知道他永远不如你对她那样有吸引力。假设这只是一时冲动，持续了很短时间，并使用了安全套。她回家后说：“宝贝，今天在健身房我和一个肌肉男发生了关系，他很帅，但他远不如你，你才能真正满足我。你永远是我的国王。不管怎样，我现在要继续给你做晚餐了。”

这个例子有些夸张，但它说明了问题。在这种情况下，你真的会感到被背叛吗？在拥有如此多优势、如此良好沟通的前提下，关系真的就结束了吗？或许，你内心是否有一小部分会为你的女朋友从别处获得了一些快乐情绪而感到高兴？也许她的性需求很高，你有时难以完全满足，你甚至会为有人“分担”了这项工作而感到一丝轻松？

## 重新审视“性”的意义

坦率地说，性并不是什么天大的事。每个人都在做。你的父母、祖父母都做过。通过你把性看得如此特别和重大，你是否在给人们施加压力？这是否导致了人们对性产生不健康的不安全感？例如，电视上可以播放枪击新闻，却不能展示乳头。这是否导致许多青少年对性产生了扭曲的看法？

性难道不是生活中美好、自然、有趣的一部分吗？它难道不能是一场各方共赢的游戏吗？阻碍你接受这一点的，是否仅仅是你的 ego（自我）？而 ego 又曾为你带来过什么好处呢？

## 本节总结与实践建议

本节课中我们一起探讨了一夫一妻制与多偶关系的对比。本节的目的主要是开阔你的思维，让你思考如何在当前或未来的关系中处理这个问题。

我并非贬低一夫一妻制的关系。但我见过太多一夫一妻制失败的例子，也感觉太多人在从未尝试或认真思考的情况下就排斥多偶关系。

**实践建议如下：**

*   **尝试思考**：至少进行一次思想实验，尝试思考多偶关系的可能性，不要立刻否定。
*   **真实尝试**：如果你在现实中尝试多偶关系但失败了，问自己原因。是否源于你自身的不安全感？这是一个很好的信号，提示你需要更多地进行自我建设，提升自信，学习课程中的“四个B”等概念。
*   **伴侣质量**：如果失败源于伴侣的不安全感，那么你需要学习如何吸引更高自尊、更高质量的伴侣。
*   **自主选择**：如果你经过思考，确实对多偶关系毫无兴趣，那么完全可以进入一段排他性的关系。只要你们双方都100%快乐、关系健康，并且是出于真正的渴望而非社会压力或稀缺心态，这完全没问题。

总之，给多偶关系一个机会，至少是思考上的机会。让你的 ego 暂时退场，真诚地思考一下。否则，如果你和伴侣都真心想要并完全接受一夫一妻制，那就去做吧。
