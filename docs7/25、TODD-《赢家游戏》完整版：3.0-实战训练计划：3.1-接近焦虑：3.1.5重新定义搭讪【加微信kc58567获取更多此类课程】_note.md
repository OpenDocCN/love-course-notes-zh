# 赢家游戏：3.0：实战训练计划：3.1：接近焦虑：3.1.5：重新定义搭讪 🧩

在本节中，我们将学习一种克服接近焦虑的核心技术：重新定义“搭讪”这个行为本身。我们将简化你对搭讪的认知，降低行动门槛，从而让你能够轻松地迈出第一步。

上一节我们讨论了焦虑的根源，本节中我们来看看如何通过改变思维来直接化解焦虑。许多人感到焦虑，并非因为走过去说“嗨”这个动作本身，而是因为担心之后需要发生的一切。他们焦虑的是：“天哪，我得维持半小时的对话”、“我得想办法应对”、“我必须表现得机智、有创意、有趣、幽默，还得编故事”。这让人不堪重负，因为你思考得太远了。

在实际行动前，试图在脑海中预演一小时的搭讪过程是不可能的。但很多人却希望如此，他们希望拥有所有答案，希望每一步都按计划进行，而这根本做不到。即使你有一个计划，30秒内这个计划也会失效，因为实际发生的情况总会与你预期的不同。

因此，我现在要告诉你的是：不要将搭讪想象成一件复杂、困难、需要耗时数小时的大事。相反，你应该把它看作一件非常简单的事。

**我希望你将搭讪重新定义为：走过去，说点能引起她注意的话，声音足够大即可。就这样。**

这就是你的全部任务。因为，再次强调，思考未来三小时要做的一切令人难以承受，但思考“走过去打个招呼”则不会。我希望你允许自己，即使只做到这一步，也完全没问题。

以下是重新定义搭讪的具体要点：

*   **成功标准**：如果你走过去说了“嗨”，然后因为紧张忘了怎么说话——成功。如果你走过去说了“嗨”，那一刻害怕到恐慌性地吐了女孩一身——恭喜，成功。我关心的只是你完成了“说嗨”这个动作。
*   **简化步骤**：你不需要完成1000步、100步甚至10步。你只需要完成搭讪中那最初的一小步。有趣的是，一旦你完成了那一步，一旦你身处那个情境，你自然就知道接下来该怎么办。
*   **渐进学习**：作为人类，我们是渐进式学习的。如果你习惯了走过去说点什么，并且不再为此紧张，那么接下来你就能多说一点。当这也不再是难事时，你就能说得更多。这就像学走路，婴儿从几乎不能动，到学会翻身、爬行、站立、迈步，最后学会跑和运动，都是分阶段学习的。
*   **接纳当下**：无论你处于哪个阶段，都要真正接纳那个水平。如果你现阶段觉得走过去打个招呼都很可怕，那么就去说“嗨”，然后为自己做到了而感到自豪。不要给自己施加压力，要求必须做出惊人的表现，这不是必须的。唯一的要求是你在行动、在尝试、在进步。

我喜欢用学习中的一个例子和一句话来总结这个理念：

**“如果你每天在某件事上进步1%，一年后你会进步多少？”**

很多人回答是365%，但这并不正确。实际上是**约3800%**，因为这1%是复利增长的。学习搭讪技巧、克服接近焦虑也是如此。如果你每天只是进步1%，到年底时，你会变得好到让人认不出来。

关键在于，不要在你和那个行动之间设置障碍。只要你**行动**，你就会成长，你就能获得那每天的1%，并取得惊人的进步。所以，对你的进步保持耐心，但务必确保你在行动。

最后，再次强调，降低你对于“一次成功的搭讪”的标准。一次成功的搭讪，就是你当时需要克服的困难本身，仅此而已，无需更多。不要担心接下来的100步。

本节课中，我们一起学习了如何通过**重新定义搭讪**来克服接近焦虑。核心是将复杂的预期简化为一个可执行的微小动作：**走过去并说点什么**。我们强调了降低成功门槛、接纳当前水平、以及相信渐进式学习的力量。记住，关键在于**行动**本身，而非一次完美的表现。从最小的步骤开始，你就能持续进步。
