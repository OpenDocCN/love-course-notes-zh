# 朱利安《Ten Game十戒》：05：任务一：建立日常一致性

在本节课中，我们将学习如何将“游戏”融入你的日常生活，使其成为你身份的一部分，而非仅在周末开启的技能。核心在于通过每日简短的实践，建立一致性和动力。

---

在我遇见你之前，我就已将目光投向你。我是朱利安，欢迎来到第一个实战任务。

现在，你已经学习了“十诫”法则，因此你从逻辑上理解了什么是“游戏”。你也完成了互动练习，从情感上体验并认可了它。现在是时候走出去，将其付诸实践了。

我希望你将接下来的实战任务视为你的每周“游戏”锻炼计划。这启发了我设计这个课程。你去健身房时，不会毫无计划。你不会随便走进健身房，心想“我听说要举重，那就随便做做吧”。你会坚持你的锻炼计划，并随时间调整。这才是快速进步的方法。我们对待“游戏”也要如此。

在接下来的十周里，我会给你每周的“游戏”锻炼计划。你需要坚持执行，并随时间进行调整。

好的，第一周，你的第一个任务，我们将致力于让“游戏”成为你身份的一部分，融入你的日常生活。我们将专注于**一致性**。这不是你只在周末外出时才开启的东西。这应该是你日复一日的生活方式。

你的任务是：在接下来的七天里，**每天外出至少30分钟**，与女性互动。

听到这个，别慌张。别想“天啊，我的生活已经这么忙了，怎么可能每天出去”。不，只是30分钟。我相信无论你多忙，每天都能找到30分钟。最坏的情况，就从睡眠里挤30分钟。你认为自己能在接下来七天少睡30分钟吗？答案是肯定的。

你甚至可以分散时间。比如上班路上2分钟，午休时5分钟，下班后2分钟。你可以去一个普通的酒吧，哪怕是在周一或周二，只要离家几个街区。这都没关系。只要总计每天有30分钟，你在外面与女孩互动就行。**约会也算在内**。

我基本上只是希望你不要把这看作是周末才开启的事情。这不应该是一件大事。而让它成为日常生活的一部分，就能实现这一点。

好吧，现在，如果你告诉我：“嘿，每天出去30分钟和女性互动。”我会说：“哦，天啊，30分钟？”但对我来说，这当然不是大事。这很正常。这不是“把妹时间”，这只是“哦，我只是在和女孩聊天”。你在星巴克排队时，可以聊上5分钟。在杂货店，又是5分钟。下班后走在街上，与其回家看电视，不如和某人聊聊天。还有那些营业到很晚的酒吧、餐厅露台，随便聊聊，20分钟就进去了。真的不难，也不应该那么吓人。

这真的就是“游戏”：你越不把它当回事，效果就越好。甚至你与女孩互动时，如果你不觉得那是大事，互动也会更随意。你会不再自我设限。别再认为性是件大事。这又不是我开启的“重大把妹时间”。

所以，你必须真正让这成为日常生活的一部分。这是一种存在方式：在接下来的七天里，每天都要出去。

现在你可能也注意到了，如果你已经应用了目前所学，我相信你度过了一个很棒的夜晚。我一直在浏览“十诫游戏”的Facebook群组，有些故事简直太精彩了。然而，你可能意识到，你出去应用了这些，度过了美好时光，回家后，第二天，旧的思维模式又占据了上风，你又回到了那种压抑、受限的心理状态。

你必须明白，现在这个压抑的你，是你多年来养成的。有很多自动驾驶的思维模式会接管一切。如果你想让你在周末那个能拥有精彩经历、史诗体验的你，成为你身份的一部分，你就必须每天努力。你必须开始放下所有这些阻碍你的、自动驾驶的思维模式，比如保持压抑、保持条件反射、不表达自我。这就是你，那个安静内向的你。你必须获得更多经验，并开始建立新的思维模式，这些新模式基本上能促进相反的行为。

如果你只是每周做一次，比如在周末，那么很难让这成为真正的你。很难做到一周7天、一天24小时都不自我设限。所以你必须每天都做。

你还会意识到，你将建立起一种**动力**。这在我刚开始学习“游戏”时真的帮了我大忙。我在瑞士起步。我记得一开始，我只在周五、周六出去。我发现周五总是很痛苦。从周一到周五都没出去，我就想：“天啊，我有点陷在自己的头脑里了。”周末来了，我在周五晚上拼命努力，通常要花一整晚时间，到深夜才开始进入那种良好的状态，真正走出自我，表达自己。那感觉相当棒。

然后我回家，周六到了。周六，我没有从周五结束时的低点开始，而是从高点起步。我想：“哦，我可以从上次停下的地方继续。”并把事情推得更远。然后，直到下个周五我才再次出去。而我意识到，每个周五，我总是从零开始，浪费每个周五来回到上个周末的状态。

所以，后来我在瑞士改变了做法：除了周五、周六，中间的每一天也出去。这样我就能**保持那个动力**。现在，30分钟里可能真的不会发生什么。不过，你永远不知道。但假设你周五出去，建立了状态，周六从停下的地方继续，然后每天出去30分钟，保持在高位。当下个周五来临时，你就能从上次停下的地方继续，而不是浪费每个周五来回到原点。

这也保持了那种我之前提到的感觉：不被压抑、感觉良好、走出自我设限的状态，让它保持活力，并开始设定“这就是新常态”。

所以，这很关键。你必须做到。重申一遍，这就是任务，是挑战：在接下来的七天里，**每天至少外出30分钟**。

如前所述，约会也算。如果你在某个小城市，该怎么办？就在周四、周五、周六出去，试着要一些号码，然后安排一些约会。如果没有，就去一个普通的酒吧，即使没有辣妹，也要和女性互动。这就是现在的你：你正在走出去，把自己置于各种情境中。这不是“我是压抑的我，周末到了我才爆发”。

为了让你自己负起责任，我希望你在“十诫游戏”的Facebook群组里创建一个帖子。帖子标题写“任务一”，然后在这个帖子下，**每天**简单地记录你30分钟的总结。我会浏览这些帖子，虽然不会一一回复，但我会看谁真正在做，谁没有。以此来让你自己负责。努力在接下来的七天里，每天都出去做这件事。

你会意识到，过一段时间后，这会变得非常容易、顺畅。一开始可能有点困难，但你会意识到，虽然内心有抗拒，但这真的不是什么天大的事。

所以，**七天挑战，每天30分钟**。这就是你的任务一。我们将在此基础上继续，下周进行实战任务二。

在“十诫游戏”Facebook群组里创建那个帖子，并在接下来的七天里每天更新。我们下周见。
