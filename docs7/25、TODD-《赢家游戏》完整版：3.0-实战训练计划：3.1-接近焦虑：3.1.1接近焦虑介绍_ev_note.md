# 赢家游戏：3.0：实战训练计划：3.1：接近焦虑：3.1.1 接近焦虑介绍 🎯

在本节课中，我们将要学习一个核心概念：**接近焦虑**。这是许多人在尝试与心仪对象交谈时，内心产生的一种强烈恐惧或犹豫感。我们将探讨它的本质、来源以及为什么它既是真实的，又是可以被克服的。

---

让我们来谈谈接近焦虑。作为一名有15年搭讪教学经验的人，接近焦虑是一个有趣的矛盾体。

一方面，它非常、非常真实。另一方面，它又完全是胡扯。

在我学习搭讪的成长过程中，我有很长一段时间都深受严重的接近焦虑困扰。直到18岁，我想和无数女孩交谈，但从未付诸行动。我总是给自己找借口，比如“她可能很忙”、“我不知道该说什么”或者“她不够有吸引力”——尽管她常常显然很有魅力。我会找理由不去行动。这可能是阻碍我提升与女性交往能力的最大障碍，是我在自我设限。从这个意义上说，它非常真实。

从另一个角度看，接近焦虑又非常不真实，或者说很大程度上是胡扯。我敢这么说，是因为在我的教学经历中，每一个有接近焦虑的学员，只要他们有足够的**欲望**、**动力**，并施加一定的**社会压力**，他们都能够克服接近焦虑。它从来不是一个无法逾越的障碍。所以，它显然是你头脑中的产物，并非真实存在于外部世界。但它确实是一个非常真实的困境。

公平地说，这种焦虑有其真实的来源。接近焦虑源于我们是在小部落中进化而来的。我们一生都围绕着同一群人成长和生活。做出愚蠢或尴尬的社交行为，可能会给我们带来非常负面的后果。

想象一下，你生活在一个小部落里，部落里最漂亮的女孩是酋长的妻子。如果你去接近她，尤其是以糟糕的方式，但即使是任何形式的接近，都可能导致酋长杀死你，或将你驱逐出部落，流放到沙漠或森林中，让你自生自灭。因此，这种强烈的恐惧确实有其进化根源。

然而，在21世纪的今天，在一个非常安全的世界里，这种恐惧已经不合时宜，可以说是胡扯了。

所以，我希望你理解，接近焦虑确实没有道理。你有这种焦虑，并不代表你是个坏人，也不代表你不够男人。它有进化的原因。但是，你必须明白，它是非常**非理性**的，在当下并不真正合理。

我想说，它就像任何恐惧一样。任何恐惧都有其自然的根源，但在某些情况下可能是没有根据的。我记得小时候第一次走上高台跳水板，我实际上是“跳”下去的，因为“跳”比“跳水”感觉容易些。我走到跳板尽头往下看，感到极度恐惧，结果又走回来，爬下楼梯，当时整个泳池的人都在看着我。但当我最终说服自己跳下去之后，我发现乐趣无穷，想一次又一次地尝试。

接近焦虑对你来说也应该是同样的道理。它是恐惧，但这种恐惧伴随着肾上腺素带来的兴奋感，这也可以转化为积极的事物。
