# 《PIMP 核心笔记》：3. 第三部分：外部技术模块：2. 打开：打开模块 - 视频示范：29. 实战示范：华尔兹舞步2 🎼

在本节课中，我们将要学习华尔兹舞步的第二个实战示范，重点在于掌握其节奏与动作的协调配合。

上一节我们介绍了华尔兹舞步的基本概念，本节中我们来看看如何将理论应用于实际动作。

华尔兹是一种三拍子的舞蹈。其核心节奏模式可以用以下公式表示：
**节奏 = 强拍 + 弱拍 + 弱拍**
在音乐中，这通常对应着“嘭-嚓-嚓”的节拍。

为了完成这个舞步，你需要协调身体的不同部分。以下是动作分解步骤：

首先，从准备姿势开始。
1.  双脚并拢，身体保持直立。
2.  重心均匀分布在双脚上。

接着，开始第一个三拍循环。
1.  第一拍（强拍）：左脚向前方迈出一大步，身体重心前移。
2.  第二拍（弱拍）：右脚向侧方滑动，与左脚并拢但不着重。
3.  第三拍（弱拍）：左脚在原地轻轻点地，完成闭合。

然后，进行反向的三拍循环。
1.  第一拍（强拍）：右脚向后方退一大步，身体重心后移。
2.  第二拍（弱拍）：左脚向侧方滑动，与右脚并拢。
3.  第三拍（弱拍）：右脚在原地轻轻点地，完成闭合。

练习时，请始终将注意力集中在节奏上。动作的流畅性比速度更重要。确保每一步都清晰地落在对应的拍子上。

本节课中我们一起学习了华尔兹舞步2的完整动作循环，包括前进与后退的基本步法。掌握这个节奏模式是跳好华尔兹的关键。
