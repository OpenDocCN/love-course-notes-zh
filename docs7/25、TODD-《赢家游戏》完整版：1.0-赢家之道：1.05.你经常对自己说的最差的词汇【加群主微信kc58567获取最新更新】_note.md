# 赢家游戏：1.0：赢家之道：1.05. 你经常对自己说的最差的词汇 🚫

在本节课中，我们将要学习一个需要从你的词汇表中彻底清除的“坏词”。这个词并非粗言秽语，而是“应该”。我们将探讨这个词如何制造不必要的压力、阻碍你的成长，并学习如何用更积极、更现实的思维方式来取代它。

## 概述：为什么“应该”是危险的词

大多数人使用“应该”这个词的方式，会将自己变成受害者，并在生活中制造大量不必要的压力。例如，“我应该得到那次晋升”或“她不应该拒绝我”。每当你使用“应该”时，你实际上是在说世界“应该”是某种样子，而现实并非如此。这等于你将自己置于现实的对立面。

上一节我们探讨了面对现实的重要性，本节中我们来看看“应该”这个词如何让我们与现实的对抗具体化。

## “应该”如何制造压力与受害者心态

当你认为现实“应该”不同时，你就在与现实对抗。这必然导致巨大的压力，因为现实是一回事，你认为它“应该”的样子是另一回事，两者之间的差距会积累大量被浪费的紧张情绪和情感。最终，这会让你感觉像一个受害者。

拥有一个完美的“受害者故事”非常诱人，尤其是如果你想在生活中失败或寻找一个简单的借口。例如，想着“我应该得到晋升，但我没有”，这让你可以告诉自己你什么都没做错，而不是去正视并承担自己在现实中的行动责任。

## 赢家思维 vs 受害者思维

在现实中，你“应该”没有得到那次晋升。即使你比得到晋升的人更合格一点，你在现实中“应该”做的，是成为一个如此压倒性的候选人，以至于你的老板绝无可能选择其他人。或者，你在现实中“应该”如此优秀，以至于即使错过了这次晋升，你也能在其他地方获得更好的工作机会，因为你已经建立了出色的履历。

赢家以这种方式思考。他们不会寻找用“应该”来开脱并扮演受害者的微小途径。他们会审视现实：世界就是它本来的样子。是的，存在运气因素，但有多少是我能控制的？我本可以做什么？我现在能做什么？无论从概率上讲什么“应该”或“不应该”发生，事实就是事实。

以下是赢家思维的核心公式：

**赢家思维 = 接受现实 + 分析可控因素 + 采取下一步正确行动**

任何你花费在试图假装现实应该不同，或编织一个满足自我而非直面现实的故事上的时间、精力和情感，都是一种浪费，并且会阻碍你未来的成功。

## 在情感关系中的具体应用

让我们看看一些男性在情感关系中的误解。许多男性持有这样的观念：他们“应该”天生就擅长与女性相处。因此，他们从不费心去学习技巧。

*   **错误观念**：我“应该”在生活中自然而然地擅长与女性相处，所以我为什么要学习技巧？
*   **现实**：如果你没有付出努力、没有进行实践、没有发展技能，那么在现实中，你“应该”不擅长与女性相处，并且你得到任何负面回应都是应得的。

关键在于接受你目前“应该”不擅长这个现实，但同时认识到，存在一条通过行动可以非常可预测地带来更好结果的路径。不要与现实对抗。

以下是基于现实采取行动的例子：

1.  **情境**：一个女孩被她的朋友拉走了，没有跟你回家。
2.  **受害者思维**：她“应该”跟我回家的。
3.  **赢家思维**：在现实中，她“不应该”跟我回家。因为在现实中，我“应该”更好地处理她的朋友，我“应该”更好地管理那个情境以防止它发生。下次我知道该怎么做了。

同样，如果你接近一个有男友的女孩，她“应该”有男友，因为她是个有魅力的女孩，这就是现实。你需要做的是学会处理现实，绝对不要与之对抗。

## 实践指南：消除“应该”的步骤

如果你想消除生活中大约90%的压力，只需摆脱“应该”和“不应该”这两个词。每当你发现自己想说“应该”或“不应该”时，请改为说“在现实中……”。

以下是具体的实践步骤：

1.  **识别**：注意到自己使用了“应该”或“不应该”。
2.  **转换**：将句子改为以“在现实中……”开头，并说出相反的情况。
3.  **分析**：列出在现实中事情为何如此的**原因**。
4.  **行动**：制定基于现实，你现在要采取的**行动步骤**。

例如：
*   **原句**：“我应该得到那次晋升。”
*   **转换与分析**：“在现实中，我不应该得到那次晋升。原因可能是我的项目影响力展示不足，或者与团队沟通不够主动。”
*   **行动步骤**：“我现在要做的就是：1. 主动承担一个能显著体现价值的核心任务；2. 每周与主管进行一次简短的工作进展汇报；3. 报名参加一个提升演讲能力的课程。”

最重要的部分是：接受现实，并基于现实采取下一个正确的行动。不要将你的时间、精力和情感浪费在与现实对抗上。不要用“应该”来困扰自己。

## 总结

本节课中我们一起学习了“应该”这个词的危害。它通过让我们与现实对抗，制造压力、滋养受害者心态，从而阻碍我们的成长和成功。赢家的思维方式是直面现实，接受它本来的样子，从中分析自己可以控制的部分，并果断采取建设性的行动。记住这个核心公式：**接受现实，聚焦可控，立即行动**。从今天起，练习将每一个“应该”转化为对现实的分析和具体的行动方案，这将是你迈向赢家之路的关键一步。
