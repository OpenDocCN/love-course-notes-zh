# 国外自然流大师Todd全套：2.理论部分：08.收尾

在本节课中，我们将要学习“收尾”这一环节。收尾是游戏过程中至关重要的一步，没有收尾，之前的所有互动都将失去意义。我们将探讨收尾的重要性，并学习一个高效的“收尾序列”结构，以帮助你更顺利地完成互动。

## 收尾的重要性

上一节我们介绍了互动中的各个阶段，本节中我们来看看收尾。收尾是整个互动中可能最重要的部分。如果没有收尾，其他环节的意义何在？销售领域有句话：“光说不卖”。同样地，在搭讪中，“光聊不收尾”也不行。

你可以和女孩进行很多对话，但这并不意味着你在进行真正的“游戏”。如果你不主动要号码、不邀约、不尝试带她回家、不将互动向亲密关系推进，那么你实际上并没有在进行“游戏”。因此，**如果没有收尾，就没有游戏**。这凸显了收尾阶段的重要性。

我曾说过，如果互动只能保留两个阶段，那么“收尾”必定是其中之一。但这不仅仅是提出要求那么简单，有很多方法可以显著提升收尾的成功率。

## 收尾序列

第一种提升方法是，不仅要有收尾动作，更要有一个“收尾序列”。事实上，你可以将整个搭讪过程视为一个宏大的收尾序列。你从接近开始，就知道最终目标是要么拿到号码，要么成功邀约，要么带她回家。这三件事之一必定发生在互动的结尾。因此，整个互动都是为了促成这个目标而构建的。从这个意义上说，整个互动就是一个收尾序列。

但在这里，当我们谈论“收尾序列”时，特指紧接在收尾动作之前的那一小段互动。很多人的做法是典型的糟糕游戏：进行半小时漫无目的的闲聊，没有任何意图，然后突然说：“嘿，你号码是多少？”这样问号码几乎会让女孩感到意外。

即使对话中带有一些意图和互动，如果你没有建立前提、没有对她进行评估、没有构建某种叙事，那么收尾可能就显得不合逻辑。这些其实都是收尾序列的一部分。然而，即使你已经完成了这些铺垫，在即将分别的时刻，围绕收尾本身再创造一点内容也会很有帮助。公平地说，如果你已经完成了评估和叙事构建，你或许可以直接说“嘿，你号码是多少？”并且成功。

但让我们来看一个更简短的互动场景。假设你只有一次非常快速的互动，比如你在公交站遇到一个女孩，你走过去和她交谈，然后她的车来了，你需要在那一刻立即收尾。最直接的做法当然是问号码。但假设你们只聊了15秒，你有多大可能拿到号码？即使拿到，这个号码很可能也很不靠谱。

那么，你能做些什么来让收尾更稳固呢？如何将这个本应是整个互动的漫长收尾序列，在收尾前快速地压缩完成，从而提高成功率呢？方法如下：

**收尾序列的核心结构是：铺垫 -> 评估 -> 收尾。**

## 收尾序列详解

以下是收尾序列的具体步骤：

1.  **铺垫**：在提出收尾前，先植入一个关于未来可能发生之事的想法或前提。
2.  **评估**：对她进行一个快速的、积极的评估，让她“通过”一个小测试或表示认同。
3.  **引导**：在完成铺垫和评估后，直接、自信地提出收尾要求。

例如，在一个快速互动中，你可以这样说：“**其实，你看起来挺……（铺垫：提出一个特质，如‘爱冒险’）**。你爱冒险吗？（**评估：让她回答‘是’**）” 如果她回答“是”，你可以接着说：“**好的，酷。那样的话，我想我们可以一起玩玩。你号码是多少？（引导：直接收尾）**”

你在这里所做的是：为收尾创造了一个前提，为收尾提供了一个合理的理由。你基本上**缩短了整个评估过程**。记住，我们讨论过评估期，其中一个重点是：你必须表明你喜欢她是基于真实的理由，否则收尾就显得没有道理。我刚才的做法就是表明我喜欢她是基于一个真实的理由，让收尾变得合理。同时，我也让她对我进行了一点点“资格认证”（让她说“是”）。这实际上是一种典型的销售技巧：先让对方说“是”，然后再提出要求。

其他在快速互动中可用的例子：
*   “你住这附近吗？（评估） 好的，酷。那样的话，我们可以做朋友。你号码是多少？（引导）”
*   “你经常出去玩吗？（评估） 好的，酷。给我你的号码，下次我和朋友出去玩时叫你。（引导）”

你正在**提供一个前提，通过让她说“是”来评估她，然后进行收尾**。这个收尾序列能将许多原本不太理想的互动变得非常好。即使你在前期已经做好了所有铺垫工作，使用这个序列也能让效果更好，锦上添花。

很多时候，我甚至会以否定的方式植入收尾想法：“我挺想去做某件事……不过，嗯，我好像还不够了解你。（铺垫/评估）” 然后，如果她反应积极，我可以说：“好吧，我想我们可以试试。不过，也许还是算了？……不，还是算了。（故意推拉）” 然后再说：“好吧，那样的话，你号码是多少？（引导）” 实际上，我是在引导她来“说服”我完成收尾，而不是直接索要。这就是采用了收尾序列。

## 完美结构总结

因此，你希望拥有的完美收尾结构是：**铺垫 -> 构建框架 -> 评估 -> 引导**。

*   **铺垫**：提出收尾所涉及的内容（例如，一起做某件事的想法）。
*   **构建框架**：将收尾构建成对她有利的局面（例如，这是一次有趣的经历）。
*   **评估**：这是评估阶段的一部分，让她投入、承诺，或者即使没时间做这些，至少也要表明你对她有积极的评价（例如，“你看起来挺酷的”）。
*   **引导**：直接行动。例如，要号码时说“**你号码是多少？**”并直接把手机递过去，而不是问“**我能要你的号码吗？**”（这是一个容易被拒绝的是非题）。引导意味着，一旦你做好了基础工作，并且确信自己值得获得收尾，就**假设一切已成定局，然后直接引导**。

**收尾序列的通用流程是：铺垫 -> 构建框架 -> 评估 -> 引导。** 在你的收尾过程中，尽可能地实施这个结构，你将会得到比凭空直接收尾好得多的结果。

本节课中我们一起学习了收尾的核心重要性，并掌握了一个高效的“收尾序列”框架。记住，没有收尾就没有真正的游戏。通过使用“铺垫-评估-引导”这个序列，你可以为收尾创造合理性、提高对方接受度，并最终更自信、更顺利地完成互动目标。请在实践中尝试运用这个结构，它将显著提升你的收尾成功率。
