# 价值大师班：05：价值的内在博弈：3 反馈回路：伪造与被伪造 🔄

在本节课中，我们将探讨个人成长中“内在博弈”与“外在博弈”的关系，并深入理解“反馈回路”这一核心机制如何驱动我们的进步。

---

上一节我们介绍了内在博弈与外在博弈的基本概念。本节中，我们来看看连接这两者的关键：反馈回路。

存在内在博弈与外在博弈的区分。你应该学习技巧，还是应该学习内在哲学与内在博弈？我明天会详细讨论内在博弈，现在先简要提及。我一直主要是一个外在博弈者，原因就在于这个反馈回路。

你可以将内在博弈视为你的**内在状态**（如信念、心态）。你可以将外在博弈视为你的**外在行为**（即在现实世界中展现的行动）。因此，你在现实世界中展现的行为将最直接地决定你的结果。你的行为将导向结果。

然而，有趣的是，你的结果会反过来改变你的内在博弈。如果你获得了好结果，这将提升你的内在博弈。通过提升内在博弈，它会让下一次采取良好行为变得更容易，或者让你已有的良好行为在下一次表现得更好。

以下是这个过程的运作方式：

1.  **行为**：你采取一个外在行动。
2.  **结果**：该行动产生一个现实结果。
3.  **内在影响**：这个结果会改变你的内在状态（信念、信心等）。
4.  **强化行为**：改变后的内在状态促使你下一次采取更好或更自信的行为。

这就形成了一个不断变得更好的正向反馈回路。这就是为什么我建议采用诸如“**假装成功，直到真正成功**”这类方法。走出去，进行实践，获取结果，然后以正确的方式归因。

以下是实现正向反馈回路的关键步骤：

*   **正确归因成功**：将成功归因于自己的努力和有效策略。
*   **正确看待失败**：将失败视为学习机会和调整方向的反馈。

正因为如此，你将获得这个持续的正向反馈回路。由于它是自我强化的，无论你从哪里开始都无关紧要。即使你从一个行为水平很低、内在博弈也很低的起点开始，只要你能让这个反馈回路对你有利，你就能以可预测且相对快速的方式变得优秀。

---

本节课中，我们一起学习了反馈回路如何连接内在博弈与外在博弈。关键在于通过行动获取结果，并用正确的方式解读这些结果，从而启动一个自我强化的正向循环，驱动持续成长。无论起点高低，这一机制都同样有效。
