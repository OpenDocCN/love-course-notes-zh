# 《PIMP 核心笔记》：3. 第三部分：外部技术模块：5. 收场：视频范例：46.示范：用情绪状态带动女人

在本节课中，我们将要学习一个具体的互动技巧：如何通过表达自身的情绪状态来影响对方的感受，从而营造更具吸引力和顺从性的氛围。这个技巧的核心在于，通过语言传递你即时的、积极的情绪反馈，来带动对方的情绪投入。

上一节我们介绍了收场阶段的各种情境，本节中我们来看看一个具体的语言应用范例。

这个技巧的关键在于，在互动即将升级或进入更亲密阶段时，主动用语言描述你当前的情绪和感受，并对对方做出积极评价。例如，在即将进行身体接触（如“搜身”）的时刻，你可以这样说：“嘿，我们这样子不算是在做爱哦。天哪，这样我会觉得你更性感。”

紧接着，你需要进一步强化这种情绪反馈。以下是表达这种积极情绪状态的具体话语示例：

*   “天哪，宝贝，你太性感了哦。”
*   “你太辣了哦。”
*   “天哪哦，宝贝儿我很享受。”

通过持续输出这种带有强烈情绪色彩的正向描述，你的对话伙伴会感知到你对当前时刻的投入和享受。这种直接的、充满感染力的情绪表达，能够有效地将你的积极状态传递给对方。

于是，对方很可能被这种情绪所带动，从而更顺从地融入当下时刻的氛围与互动中。

本节课中我们一起学习了如何通过主动表达“享受”、“性感”等积极情绪状态，来影响互动对象的感受，为关系升级创造自然、充满吸引力的情绪场。关键在于自然、真诚地传递你即时的正面反馈。
