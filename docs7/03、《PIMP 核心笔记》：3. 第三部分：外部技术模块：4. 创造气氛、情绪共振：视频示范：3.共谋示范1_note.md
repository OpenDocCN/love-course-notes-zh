# 《PIMP 核心笔记》：03：第三部分：外部技术模块：4. 创造气氛、情绪共振：视频示范：3. 共谋示范1

在本节中，我们将学习一种被称为“共谋”的情绪共振技巧。这种方法的核心在于创造一种你与目标对象之间独特的联结感，让你们感觉像是一个团队，共同面对外部世界。

上一节我们介绍了情绪共振的基本概念，本节中我们来看看如何通过“共谋”来具体实现它。共谋理论是指通过创造一种动态的社交互动，让对方感觉你们两人是“一伙的”，从而产生一种你们共同对抗全世界的亲密感。其关键在于建立只有你们两人知晓的小秘密或内部笑话。

以下是实现“共谋”的几个具体方法：

*   **创造内部笑话**：发展一些只有你们两人能理解的玩笑或“梗”。例如，你可以把一个女孩拉到陌生人面前，然后说：“**这是我的亲妹妹。**”接着亲吻她，再对陌生人补充说：“**嗯，没错，我们从小就很亲密。**”
*   **扮演共同角色**：对面的陌生人并不知道你在撒谎，但你和女孩共享这个小秘密，并会各自扮演“哥哥”和“妹妹”的角色。这种方法能强化你们二人一体的感觉。
*   **扩展共谋场景**：当你们成为“一伙的”之后，这种技巧可以扩展到其他社交场合。例如，在见女孩的朋友时，你也可以玩这样的共谋。比如可以说：“**啊，我们已经认识好多年了，我们在高中就是同桌，他老打我。**”或者“**我们还一起解剖过青蛙。**”
*   **虚构共同经历**：你可以随意编造一件你们一起做过的事情，而她的朋友并不知道你们两人根本没有做过这件事。

本节课中，我们一起学习了“共谋”这一情绪共振技巧。其核心是通过创造共享的秘密或虚构的共同经历，在两人之间建立起一种排他的、团队般的亲密联结，从而有效拉近关系。
