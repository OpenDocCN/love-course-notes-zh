## 动作分解

以下是“握手转圈圈2”动作的完整步骤分解。

1.  启动程序，初始化交互环境。
2.  检测到目标对象后，发送握手请求信号。
3.  建立连接后，执行同步旋转指令。
4.  在旋转过程中，持续进行平衡校准。
5.  完成预定圈数后，缓慢停止动作。
6.  发送连接终止信号，结束交互。

## 核心逻辑

该动作的核心在于握手后的协同旋转控制。其逻辑可以用一个简单的条件循环来描述。

```python
while 连接状态 == 已建立 and 当前圈数 < 目标圈数:
    执行同步旋转()
    实施平衡校准()
    当前圈数 += 1
缓慢停止()
结束连接()
```

## 关键要点

在实施过程中，需要特别注意以下几个关键点。

*   **信号同步**：握手请求与接受的信号必须完全匹配，否则连接会失败。
*   **平衡参数**：旋转时的校准参数 `CalibrationFactor` 需要根据实时反馈动态调整。
*   **安全停止**：停止过程必须是渐进的，公式为 `减速梯度 = 当前速度 / 停止帧数`。
