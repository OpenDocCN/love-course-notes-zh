# 朱利安《Ten Game十戒》：05：互动练习3 - 消除所有过滤器 🧠

在本节课中，我们将要学习一个核心的互动练习，旨在帮助你消除内心的评判声音，实现更真实、更流畅的自我表达。这个练习将训练你停止过滤自己的想法，并直接将其表达出来。

上一节我们介绍了建立互动的基础，本节中我们来看看如何通过一个具体的练习来打破内心的障碍。

## 练习核心：消除过滤器

这个练习建立在之前内容的基础上。我们将放慢节奏，专注于消除那个不断评判你所说每一句话的“小声音”。当你听到“保持真实”时，尤其是在与女孩互动时，这意味着不伪装，说出你内心真正想说的话。但在所有情境中，包括现在，我们内心总有一个声音在问：这够好吗？别人会怎么看我？我说这个合适吗？这符合我的形象吗？请放下这些顾虑。

以下是练习的具体步骤：

你将进行两人一组的练习，每人轮流进行30秒到1分钟。你需要尽可能地清空大脑，等待有东西“蹦”出来。这个练习听起来可能有点奇怪，但它非常关键，对我个人帮助很大。

*   **清空与等待**：尽你所能清空大脑，等待有东西“蹦”出来。
*   **即刻表达**：无论“蹦”出来的是**一个想法、一种感觉、一幅图像、一种身体感觉、一个物体，或任何东西**，一旦出现，立刻用一个词把它说出来。
*   **重复过程**：然后再次清空大脑，等待下一个东西出现，再把它说出来。

你会注意到，当有东西出现时，你可能会想去分析它，比如“这说出来够酷吗？”。不要这样。一旦它出现，就说出来。这个练习的目的就是**润滑从“思维”到“表达”的过程**，直接释放它。

这个练习也向你证明，你永远不缺话可说。你的脑子里总是有东西会“蹦”出来，但我们通常会对它们进行筛选，就像有一个审核标准：这个够好，那个不行。女孩越有魅力，这个标准就越高。本练习的目标就是**消除这个标准**。

## 示例演示

让我来举个例子。清空大脑，无论出现什么，都说出来。

```
兔子。
```
（再次清空大脑）
```
桌子。
```
让我再做几秒演示（在实际中是单人轮流进行）：
```
顺滑。
塑料。
尖的。
疣。
牛排。
老鼠的牙齿。
```
我就停在这里。你看，就是这些最随机的东西。我要你把它说出来。这有点难，因为你要说的正是你脑子里冒出来的东西，在某种程度上，这很个人化。但我们通常会把大部分憋回去，担心别人怎么看我。**说出来**。无论是什么，说出来，去掉伪装，别再试图表现得很酷。

## 额外挑战与规则

这个练习还有一个额外的挑战：当你在说你的词时，另一个人会看着你，评判你。他不能说话，但他会表现出“真奇怪，你为什么说这个？”的样子。

进行练习时，请遵守以下规则：

*   **不要依赖环境**：不要根据周围环境来找词。
*   **不要关联前词**：不要根据前一个词联想出下一个词。你说完一个，清空大脑，等待全新的东西出现，再说出来。
*   **不要跳过任何想法**：例如，如果你想到了“老鼠的牙齿”，然后觉得“这个太怪了，我等下一个”，那你就做错了。想到什么就说什么，所见即所得，直接释放它。
*   **保持眼神交流**：可以进行眼神交流，也可以看向别处，怎么舒服怎么来。

每人进行30秒到1分钟，我会说“换人”。你们决定谁先开始。

这个练习更放松，你的状态就是放松，然后让想法自然流出并说出来。内容越个人化越好。

## 核心要点与总结

你的脑子里总是有东西。要养成**直接表达它、说出它**的习惯。不要憋着，不要评判它。这样，你的夜晚会更有趣，互动也会更有趣。如果你评判自己说的每句话，就像走在蛋壳上，压力很大。**释放它**。

本节课中我们一起学习了“消除所有过滤器”练习。我们了解了其目的是打破内心的评判机制，通过即刻表达随机涌现的想法来训练真实表达。记住，关键在于清空等待、即刻 verbalize、并克服对他人评判的恐惧。坚持练习，你将能更轻松、更自信地进行互动。
