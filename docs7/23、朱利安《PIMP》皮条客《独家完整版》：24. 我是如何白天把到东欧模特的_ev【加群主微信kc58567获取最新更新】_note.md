# 搭讪艺术：24：日间搭讪心态与热身技巧 🎯

在本节课中，我们将学习日间搭讪的核心心态与一套行之有效的热身技巧。这套方法的核心在于通过特定行为快速进入自信、放松的“状态”，从而克服社交压力，提升搭讪成功率。

上一节我们探讨了搭讪的基本概念，本节中我们来看看如何在实际操作中调整心态并快速进入状态。

## 心态框架：与所有人交谈 🗣️

你们中一些人提到，日间进入状态时应具备的心态是：**和所有人交谈**。

你不应该像扫描环境一样只寻找那些漂亮女孩。如果这样做，找到足够多的女孩交谈会变得更困难。在瑞典也许不难，但如果你四处张望，心里想着“下一个在哪里”，有时需要两三分钟，你的头脑就会陷入混乱。你会在分析之上，寻找那些容易接近的情况。一旦你的大脑开始寻找漂亮女孩，你也会开始寻找容易接近的目标。你会想：“那个女孩走得太快了”、“那个有朋友陪着”、“那个看起来有点凶”、“她看起来匆匆忙忙要去某个地方”。然后你会寻找像“独行侠”或无所事事的人。这很糟糕。

因此，我秉持的心态是：**和所有人交谈**。当我白天外出时，首先，在进入状态之前，我并不专注于“带走”或“得到”那个女孩。我只想让自己进入这种不受束缚的状态，建立乐趣。然后，再去接近我想要的女孩。记住，你可以通过与不漂亮的女孩、普通女孩、老太太、任何人交谈来进入这种状态。不一定非得是普通女孩。

以下是我们在训练营做日间搭讪时的一个练习：我让所有学员沿着街道走，对遇到的每一个人说“Nice...”。比如“Nice shoes”、“Nice shirt”、“Nice face”。对每个人都说。无论是老太太、带着孩子的年轻母亲（说“Nice baby”），还是一对情侣。你基本上就是在喊出来。你想养成将个性展现出来的习惯，并再次收集“什么都不会发生”的证据。

你做得越多，如果你觉得“这有点吓人”，因为在白天有很大的社会压力，这似乎确实有点可怕。但当你喊出“Nice face”，所有人都盯着你看，然后你意识到“我还活着”。这就像你有了后盾，现在可以更大声一点了。比如“Nice baby”——我还活着。“Nice girlfriend”——你上过吗？因为我上过。好吧，别这么说。但你可以那样大声。然后你意识到，“哦，什么都不会发生”。接着当你走向那个女孩时，砰，你就进入状态了。

这就是我在街上走时会做的事。当然，我的默认起点要高得多，因为我沉迷于处于状态之中。但当我走过时，我会说“哦，Nice shoes”、“Nice shirt”。对女孩说“Hey”。就像那样。对店员说“Hey”。就像大多数男人做的那样。和所有人交谈。焦点在你身上。

## 理解环境关联性 🌍

现在理解这一点：收集“什么都不会发生”的证据。记住，这与环境密切相关。

你可能在夜店或酒吧有过这种体验：你在那里和一群人交谈。同样，越是什么都没发生，你就越感到轻松自在，你的假设会从消极转向积极。

但如果你在夜店感觉很好，然后走到街上呢？你会想：在街上还一样好吗？这种状态非常依赖于环境。这适用于夜店或街道，但也适用于夜店的不同区域或街道的不同地段。

在这个视频中，你将看到的是一次日间搭讪的开始，它发生在一个星巴克。我的整个思路是：我会选择星巴克，为了推动自己进入状态，我就在这个星巴克里进行搭讪。因为环境很小，如果我在这里和一群人交谈，我能比在白天开始时四处游荡、零星搭讪更快地进入状态。这个小环境将成为我的“主场”。

## 热身实践：从评论开始 ☕

我走进星巴克，我们正在架设设备。我和摄像团队、几个朋友在一起，Alex也在。即使在架设设备时，我也在对人们发表评论。我记得走进去时，有个女孩坐在我朋友旁边，我直接走过去对她说：“哦，他最近单身了，你知道的，只是告诉你一声。”女孩一脸疑惑。当我说这话时，所有人都开始看过来。然后我转向另一个金发女孩，对她喊道：“你是密歇根人吗？”她又一脸疑惑。所有人又都盯着看。我想要什么？我想要人们看过来。一开始有点吓人，但他们看得越多，越是什么都没发生。我就想：“好的，我是这里的老大。”我和所有这些人都说过话了，没有回头路了。一旦你做了这么出格的事，就好了。就像跳进冷水池，你会犹豫，但一旦你在池子里了，就没事了。

所以，只是一些小评论，比如：“你是密歇根人吗？”、“别打我朋友主意”、“Nice shirt”、“Nice shoes”。稍微大声一点，引起注意。“顺便问下，你是荷兰人吗？”等等。

当你这样做时，如果你打算只在星巴克里和人交谈，不要做得太极端，以至于让人们觉得你是个古怪、疯狂、无家可归的精神病。别一进去就做些疯狂的事，比如脱裤子。别那么做。聪明一点，只是一些小评论，稍微大声一点的评论。慢慢地、逐步地让自己对那种社会压力脱敏。这就是我的全部焦点。

## 进入状态的最佳途径：笑声 😄

进入状态的最佳方式之一是通过笑声。这就是为什么第四条法则是：**我怎样才能让自己笑出来**。

所以一开始，我会想：好吧，这里那里说几句评论。我走到外面的小露台区，看到第一个女孩，她是瑞典人。我走向她，我并没有想“哦，我要做什么？我怎么带走她？”，我只是想：“我怎样才能让自己笑出来？”这是我唯一的焦点。当我跟她说话时，就是笑声。我开始说话，开始进入状态。每次我笑或者说些有点傻的话，我就会更放松一点。所以一开始的焦点就是这个。这也应该是你的焦点。笑，笑，笑。

并且接受“不流畅”。这也很重要。如果你刚出门，脑子里还在想事情，或者你刚睡醒，白天的时候，你不会像那样滔滔不绝。你的思绪不会完全集中。你会有点磕磕绊绊，你需要唤醒你的思维。这有点像……然后你就进入那个该死的状态了。所以我接受这一点，而不是试图完美，允许自己磕磕绊绊。没关系。只要在后面加上幽默感就行。所以你会看到我走过去，有点……嗯，不是那么顺畅，但我只专注于找乐子。我甚至不在乎和这个女孩能不能成，只要……我放松了一点。

## 建立主场优势：与朋友互动 👥

我和这个女孩聊天，玩得很开心。然后我的朋友进来了。同样，我们正在建立我们的“派对”。我们想要主导这个环境。如果你被白天的环境所吞噬，那是最糟糕的，甚至比夜店环境还糟，因为至少在夜店，能量还比较高，虽然吓人，但不会像白天那样让你感到窒息。如果你在白天陷入自己的思绪，你会觉得每个人都在盯着你，每个人都在评判你。这是最糟糕的感觉。你在那个星巴克里，感觉“我不属于这里，即使我这样做，每个人都会评判我的举动。天啊，如果我说‘嗨’，太可怕了”。你只想主导，想成为那个更大的“派对”。就像，是的，我们很大声，我们在这里，每个人都在看，他们只是被吸引进来。

所以我的朋友进来，我们开开玩笑，互相即兴发挥。比如我说：“你见过同性恋男人吗？”他说：“你一次见过两个男人吗？”我说：“你一次有过两个男人吗？”哦，你是瑞典人，当然有过。就这样建立乐趣。

## 合并社交圈：扩大影响力 🔗

我发现她有个朋友在里面点咖啡。所以我想，哦，再建立一点。我说：“你朋友在里面吗？去跟朋友打个招呼。我要做朋友。她叫什么名字？”我会尽量表现得很好，她还不知道我是个混蛋，但我其实不是。我会尽量超级友好，你知道的，不，嘿，嘿，我会说，嘿，嘿，她可能会……我表示怀疑。

现在，当我走向那个朋友时，我已经差不多进入状态了，因为我和之前的女孩聊过。所以当我走向这个朋友时，我想，好吧，和这个女孩一起，稍微放慢一点，平衡一下。就像巨大的乐趣和磕绊，现在乐趣少一点，但更放松流畅。所以我走向她，说：“哦，怎么样？”我再次专注于放松下来。始终监控自己的状态。如果你很紧张，就告诉自己：放松，开始变得非常非常随意，甚至有点无聊。同样，白天你不需要那么多。这不是动作大片，是独立电影。甚至是一些无聊的话，比如“哦，你知道，我做了这个”。她笑得像疯了一样。所以我开始和她慢慢聊，我知道她会回到之前的女孩那里。一切都好。放松一点。

然后，在某个时刻，他们组里第三个人进来了，是个男的。她说：“哦，这是我朋友，Andrea。”那个男的和我握手。现在，大多数男人当一个男的进入组合时会怎么想？天啊，摧毁他。不，不要有那种心态。记住，世界是友好的。你没事。如果这个男的是她的朋友，你不能敌视他。如果他不喜欢你，你就完了。她更在乎他对她的看法，而不是你的。她认识他更久。所以我的看法是：哦，她就像我们共同的朋友。我们有个共同的朋友，像对待朋友一样对待他。想象一下，就像你的社交圈，你和一个女孩是朋友，她说“这是我另一个朋友”，你会说“哦，嘿，哥们，怎么样？”和他打个招呼。在那里多待一会儿，然后决定跳回之前的女孩那里。

## 主导环境：成为焦点 👑

进去和她说话，然后我差不多更进入状态了，就像两者结合，把她们拉回之前那里。我拿着她的饮料，说：“哦，拿到你的咖啡了，感觉怎么样？”“完全抢了你的饮料，感觉怎么样？”“这是你的饮料。”“这是我的饮料。”“那也是我的女朋友。”然后另一个女孩出来了。我同时拥有她们两个。我所有的朋友都走了，除了那个男的。我想，太棒了，就像，再次，这个环境是我的家。星巴克是我的家。我走进去，有两个我开场时架设摄像设备的女孩，这里有两个瑞典女孩，还有一个男的在那里点饮料。当然，你会感觉更容易。所有这些社交联盟都在支持你，所有“什么都不会发生”的证据。

然后我想，好的，我现在可以更实际一点了，现在我可以慢慢开始专注于“带走”。所以我把Alex拉进来，介绍给女孩们。然后我看到另一个女孩走过，我想，天啊，那是我女朋友，跑过去，抓住她。那是我的另一个女朋友，顺便说下。是真的，不好意思。过来，我现在的女朋友。和她握手。别让我难堪。相信我，她是瑞典人。告诉她我们认识好多年了。嗨，这是我的女朋友。现在和她握手。这又是合并组合。就像第一个瑞典女孩，第二个瑞典女孩，合并，下一个女孩，抓住她，也合并进来。

合并组合很酷的一点是，一方面，你可以做更多出格的事。比如这个女孩走过，我说“哦，那是我女朋友”，跑过去抓住她，告诉她“过来和我的女朋友们打个招呼”。这更容易不执着于结果。不像“过来和我打招呼”，而是“过来和我那边的女朋友打个招呼”，她看过去，那边有两个该死的辣妹和Alex在等她过去打招呼。有更多的社会压力，就像“你最好他妈的照做，因为你不仅会让我失望，也会让她们失望”。所以压力更大一点。而且因为你不执着于结果，你可以做更多事，比如直接抓住她。同时，在那一刻我是言行一致的，因为我为此做了铺垫。如果你为你的朋友开场一个女孩，你可能有过这种体验，你说“嘿，过来，我朋友”，你可以做更多事。

## 做出选择并行动 🚀

把她拉进来。然后我进入状态了。我想，砰，这太他妈棒了。那个男的进来了。我开始把每个人介绍给每个人。就像，你是那里的老大。然后我想，好吧，这个女孩，最后一个，我选择她。我想，好吧，你现在在做什么？她说：“我要去H&M。”来吧，一起走。弹开。我是个强奸犯。我是个强奸犯。我有艾滋病。我强奸小婴儿。就像那样每天，感觉怎么样，你接近崩溃了吗？这是我亲爱的朋友们，我们刚疯狂地回来。太糟糕了。但是，你现在要去哪里？H&M？带上我。再见，问一下。他是的。现在跟我走，女人，现在。

真的从那个女孩的角度看，她会想：这他妈是谁？她看到的不是一个像独行侠一样在街上游荡的人，而是看到我和两个辣妹在一起，玩得很嗨。把她拉开一点。然后有趣的是，之前那个组合里的那个男的跟着我，用一种非常直接、非同性恋的方式要了我的号码。连他都想：这他妈是谁，跟所有人说话。那个女孩说什么？哦，天啊，被那个男的要了号码。什么鬼，哦，天啊。而你他妈的正处于最佳状态。当然，我们会的。

## 总结与核心要点 📝

本节课中我们一起学习了进入日间搭讪状态的方法。

以下是进入状态的方法之一：开始和所有人交谈，开始合并组合。合并组合时也要让它有趣。比如“嘿，过来见我女朋友”、“嘿，这是我女朋友”，砰。有趣的是就这样。建立乐趣，然后你就会感到自在。你认识所有这些人，就像一个友好的地方。不是每个人都在评判你，每个人可能还很羡慕。我告诉自己：看看周围，大多数人在星巴克里等着他们的饮料，每个人也很压抑，你只是……没关系。如果这很荒谬，就嘲笑自己，因为他妈的嘲笑也有助于你放松。

**核心公式**：
```
进入状态 = 与所有人交谈 + 收集“无事发生”证据 + 通过笑声放松 + 主导小环境 + 合并社交组合
```

记住，关键在于将焦点从“结果”转移到“过程”和“自身状态”上。通过简单的互动建立自信和主场感，从而为高质量的目标搭讪铺平道路。
