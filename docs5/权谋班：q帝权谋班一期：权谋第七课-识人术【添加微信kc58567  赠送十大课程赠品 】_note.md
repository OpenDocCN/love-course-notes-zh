# 权谋班：第七课：识人术 👁️

## 概述
在本节课中，我们将要学习识人术的核心原理与实践方法。课程将围绕如何观察、测试并最终有效控制和使用人才展开，摒弃基于个人好恶的情感判断，转而建立基于利益和可控性的理性用人体系。我们将探讨依赖性与可替代性的构建、不同层级人才的特性，以及如何通过系统性方法识别并驾驭各类人员。

---

## 课程核心：从情感判断到利益控制

上一节我们探讨了个人成长与资源积累，本节中我们来看看如何识人与用人。普通人用人往往基于感情，如个人喜好或厌恶，但这容易导致最终伤害感情。理性的用人标准应基于两点：**感情**与**利益**。

基于感情的判断包含三点：
1.  个人好恶。
2.  以情谊为重，不论是非。
3.  强调服从指令。

基于利益的判断也包含三点：
1.  是否服从命令。
2.  工作能力是否胜任。
3.  是否忠诚，即以你的利益为中心。

我们识人用人的困惑常源于用感情出发点去做利益判断的事。正确的做法是：**在用人过程中不断剔除感情成分，聚焦于对方是否听话、有能力、且忠诚**。

## 控制人的核心：恐惧与欲望

要控制一个人，关键在于掌握其**恐惧**与**欲望**。核心法则是：**优点可用，缺点可控**。意思是，一个人的长处你可以利用，但他的短处你必须能够控制得住。如果某个缺点你无法控制，那么最好不要使用这个人。

**公式：控制力 = 利用恐惧 + 利用欲望**

这引出了控制人的两大支柱：**依赖性**与**可替代性**。

---

## 支柱一：建立依赖性

让人依赖于你，他就难以离开。依赖性主要体现在三个方面：

1.  **经济依赖**：提供高于市场水平的薪酬，并鼓励其进行高消费（如买房、买车），使其生活方式依赖于这份高收入。
2.  **技术/结构依赖**：只让他负责整个业务流程中的一环，使其技能单一化，离开现有平台就无法独立完成工作。
3.  **精神依赖**：营造愉悦的团队氛围和归属感，使其在精神上获得满足，不愿离开。

**代码示例（依赖性的形成）：**
```plaintext
IF 提供(高薪 & 高消费引导)
THEN 形成(经济依赖)
IF 安排(单一环节工作)
THEN 形成(技术依赖)
IF 营造(积极团队文化)
THEN 形成(精神依赖)
最终结果：人员稳定性提高
```

---

## 支柱二：确保可替代性

当依赖性不足或失效时，可替代性就是你的安全网。主要有三种替代策略：

以下是三种关键的替代策略：
1.  **备胎式替代**：培养具备相同技能的备用人员，平时不委以重任，但随时可顶替关键岗位。
2.  **副手式替代**：让副手深度参与核心业务，一旦主管离开，副手可立即无缝接替。
3.  **平行式替代**：设立两个或多个团队做同样或类似业务，形成内部竞争，互相制衡，也互为备份。

**自然过渡**：理解了如何让人依赖，以及如何准备替代方案后，我们还需要知道，有些人既难以依赖，又难以替代。这时，就需要第三重保障：**制衡**。

## 特殊手段：利用与制衡“小人”

在团队中，“小人”并非毫无用处，关键在于如何控制和使用。小人有三个可用之处：

以下是“小人”的三个可控用途：
1.  **制衡**：利用小人去制衡那些能力强但不服管束的“能人”。
2.  **刺探**：小人喜欢打小报告，这恰好可以作为你了解团队真实情况的耳目。
3.  **背黑锅**：在需要平息众怒或转移矛盾时，小人可以成为理想的“替罪羊”。

**原则**：小人可用，但不可重用，更不可提拔。要将其作用限制在特定范围内。

## 识人的方法：观察与测试

识人不能靠感觉，而要靠系统的观察和测试。测试主要从四个方面入手：

以下是四种核心的测试方法：
1.  **以利诱之**：用金钱或利益去试探，观察他对财富的态度和处事原则。
2.  **以事处之**：通过共同处理具体事务，观察他的为人、协作能力和是否顾全大局。
3.  **以谋问之**：提出实际问题或困境，询问他的看法和策略，考察其思维能力和见识。
4.  **以事临之**：观察他在突发危机或压力事件下的反应和应对能力。

这个过程分为三个阶段：
1.  **描述性阶段**：客观收集并描述一个人的行为模式样本。
2.  **预测性阶段**：根据已知模式，预测他在特定情境下会如何行动。
3.  **解释性阶段**：能合理解释他所有行为背后的动机，此时意味着你已基本掌控此人。

## 分层用人：不同层级的不同需求

不同层级的人，需求和驱动力完全不同，用人策略也需区分：

1.  **高层（务虚）**：他们需要的是**位置**和**尊重**，而非具体权力或金钱。他们思考战略，把握方向。应多用思维活跃、见识广博的“文科生”类型。
2.  **中层（物理）**：他们掌握**具体权力**，需要“礼”（规则、道德）来约束。应选择技术专精、守规矩、家庭负担较重的“老实人”，他们更珍惜职位，不易冒险。
3.  **底层（务实）**：他们最需要**实在的利益**（金钱）。只需赏罚分明即可。适合用野心不大、执行力强的普通员工。

**公式：企业三大权利 = 位置（高层） + 权力（中层） + 利益（底层）**

---

## 实践心法：用人如器

“用人如器”是根本心法，意指像使用工具一样使用人才：

以下是“用人如器”的三个要点：
1.  **了解性能**：充分了解每个人的长处和短处（性格、能力）。
2.  **搭配互补**：不要追求全才，而是让不同特长的人互补，形成团队合力。
3.  **根据需要开发**：优先使用像白纸一样的应届生或可塑性强的人，按需培养，而非强行改变成熟员工的固有模式。

对于不同类型的人，策略如下：
*   **能人**：用“紧箍咒”（依赖性与可替代性）控制，并确保其有事可做，避免生事。
*   **庸人**：赏罚分明，满足其基本需求即可。
*   **小人**：按上述方法控制使用，发挥其特殊作用。

## 总结

本节课我们一起学习了系统性的识人术。核心在于转变思维：**从情感化的“识人”转向理性化的“测人”和“控人”**。我们掌握了通过**依赖性**和**可替代性**来控制人员，学会了通过**利、事、谋、临**四种方法来测试人，并理解了针对**高层、中层、底层**的不同用人策略。最终，牢记 **“优点可用，缺点可控”** 和 **“用人如器”** 的原则，就能在复杂的的人际互动中，有效地识别、使用并控制人才，为实现目标服务。

记住，识人的终极目的不是为了评判，而是为了**有效地使用和控制**。
