# 调情兵法2.0：008：拉黑21点（上）-B

在本节课中，我们将要学习“超越阶段类”问题。这类问题指的是男性做出超出当前关系阶段应有行为的情况。我们将详细分析几种常见表现，并探讨其背后的核心逻辑，帮助你避免因行为越界而被对方“拉黑”。

上一节我们介绍了第一印象类问题，本节中我们来看看第二大类——超越阶段类问题。

所谓超越阶段类问题，概括来说，是指男性做出了一些超出当前关系应有阶段的行为。

例如，尚未成为情侣关系的人，却做了只有情侣关系才会做的事。这种情况非常普遍。

以下是几个具体的例子：

**1. 身体接触（“奴”）过于急躁**
这是女性朋友反映最常见的问题之一。有些男性在初次约会时就急于牵手，或者在仅见面一两次、双方关系尚未明确时，就做出搂抱等过于亲密的举动。这种行为力度过大，过火了。
从技术层面讲，核心在于他们错误估计了关系的“水温”。任何亲密动作都应该与你们两人关系所处的阶段相匹配。
**公式：** 亲密行为时机 ≠ 你的欲望，而 = 当前关系阶段（水温）
即使你们是夫妻，如果正在冷战，你突然想发生关系也可能被拒绝。在任何情况下，无论名义上的关系是什么，两人相处时中间实际的“水温”才是最实在的。身体接触过急有多种表现。

**2. 过早进行性话题（Sex Talk）**
另一大类问题是男性太快谈论性话题。这与我们上一集在朋友圈所讲的内容本质相同。

我不理解为何国内很多人有这种“处女情结”。这或许是一种心理障碍或精神洁癖。如果你有精神洁癖，我可以理解。但反过来，如果你对对方有这样的要求，对方也可能对你有同样的要求，这就很难回答了。我通常的回答是：“那么明显的问题，还需要问吗？” 这样回答，对方就知道你不是处男了。

讲完这些过于急躁的情况，接下来我们看看一些更常见的问题。

**3. 过度的关心**
什么是过度的关心？通常指在双方只是一对一关系（并非稳定情侣）的情况下，男性就经常询问：“你在做什么？”、“你在哪里？”、“和谁在一起？” 这类问题。
在女性看来，你需要审视自己的身份是否适合问这些问题。
大家记得，我们在本节开头就提到，很多男性在做不符合自己身份或高估了关系水温的事。你问这些情况，即使是老公问老婆，我自己都不会问。她该回来自然会回来，你问她做什么？特别像我老婆是女强人，工作很忙，你问她反而打扰她工作。
从女性角度看，她会想：“你是我谁啊？问我这些，我为什么要回答你？” 如果这个女性同时在和几个人交往，她更没必要回答你。在女性眼中，她们会认为：“谁准备了‘方法’，谁就会问这些。” 这是无法避免的。
如果过度关心进一步恶化，就会进入第二阶段。

**4. 提出限制（第二阶段）**
第二阶段，男性会开始对女性提出一些限制。例如：“不准你和某某来往”、“你必须几点回家”、“有些地方你不能去”。

其次，夜场有很多种。你和一个女孩在一起，不是要占据她个人的全部，你不能要求她完全没有自己的生活圈子。

但这件事是双向的。我也见过一些女性，交了男朋友后就不准他和以前的女性朋友来往。这种情况也很多。我不太认同这种做法，因为为了得到一个人而失去一个圈子，不值得。更严重的问题是，你会不会像那首歌里唱的：“得到你的人，却得不到你的心”。
你因为某种吸引力或当时合拍而得到了这个人，但这只是这段关系的开始。在一段长期关系中，你不断做这些“垃圾”事，只会自己降低自己的分数，何必呢？
我比较认同且自己也一直在实践的做法是：尝试进入对方的圈子，去了解对方的世界。如果双方能互相融入对方的世界，起码是一件好事。因为你融入她世界的同时，也可以更多地了解她的朋友。当你清楚她身边是些什么人时，你甚至会放心一些。例如，你可以建议：“和姐妹出去玩可以，但喝酒要节制，两点钟就回家吧。” 这样，哪怕她那班朋友不是传统意义上的“好人”，你也能通过你的影响力去影响她。人总是需要放松的，在有限制和节制的情况下，没有问题。

讲完地点和人物的限制，还有时间。老实说，如果你懂得欣赏她，心态就会不同。
我的心态是：如果你和你的女朋友（或疑似情人）根本不理睬她这些事，并且你已经懂得欣赏她，她反而会更珍惜你。如果她珍惜你，她会自己注意时间。第一，她可能不会玩通宵；如果你们同居，她也会觉得“我要回去交人”，到了某个时间就会走。

讲完第二阶段，还有第三阶段。

**5. 提出各种不合理的要求（第三阶段）**
例如，很多男性会提出：“不如今晚去开房。” 这是尚未发生关系时最急切的要求之一。

这又回到了“包容性”的问题。

讲完“包容性”这个主题，接下来要讲两种非常常见、只有“屌丝”才会做的事。我才会这么说。
