# 情狼微信畅聊私密课程：微信畅聊-微信聊天的八种思维

在本节课中，我们将要学习微信聊天的八种核心思维。这些思维贯穿整个与异性交流的流程，掌握它们能让你在聊天中游刃有余，建立源源不断的话题和吸引力。

## 第一种思维：发散思维

上一节我们介绍了课程概述，本节中我们来看看第一种核心思维——发散思维。发散思维也被称为多项思维或辐射思维。它指以一个目标为中心，思维向四面八方无限扩散，并与时间、空间及万物产生联想。

**公式：** `核心点 -> 联想 -> 多方向话题`

在聊天应用中，主体通常由你、聊天对象以及情感话题构成。为了更直观地理解，我们可以通过拆解聊天目标并加入联想来应用这种思维。

以下是应用发散思维的具体方法：

*   **拆解目标：** 将聊天对象的特征拆解，例如五官、服饰、配饰等。
*   **组合联想：** 将拆解出的元素重新组合，并加入你的想象，形成聊天话题。

例如，从目标的五官切入：

*   **眉毛：** 你的眉毛像一轮弯月。/ 这难道是传说中的柳叶眉吗？
*   **眼睛：** 你有一双水汪汪的大眼睛。/ 我发现你长了一双会说话的眼睛。
*   **鼻子：** 你的鼻子很秀气。/ 喜欢你这种微微上翘的鼻子。
*   **嘴巴：** 看你的小嘴，口才一定很好。/ 你的嘴唇生得很有福气。
*   **耳朵：** 我发现你有一个很厚的耳垂。/ 你耳朵的轮廓像贝壳一样精致。

同样，也可以从其他部分切入：

*   **手：** 你的手又细又长，很适合弹钢琴。
*   **腿：** 你的腿又长又直。
*   **服饰：** 你好像很喜欢红色的衣服。/ 我发现你蛮喜欢某个品牌。

这种发散思维在聊天初期尤其好用，能快速找到话题切入点。

## 第二种思维：引领话题

学会了发散思维开启话题后，我们需要掌握如何引导对话方向。这就是第二种思维：引领话题。其核心是不断引导和带领聊天方向，在时间、空间及万物中自由切换话题，让对话保持新鲜感。

在实际聊天中，八种思维常会交织使用。例如，引领话题时常常会结合发散思维。

以下是引领话题的一个对话示例，展示了如何多次切换话题：

1.  **（开启）** 你有一双水汪汪的大眼睛。（*发散思维*）
2.  **（切换一）** 怎么谢我？请我看电影吗？
3.  **（切换二）** 这个时间你应该在上班吧。
4.  **（切换三）** 看来我很有魅力，让你上班还能偷偷跟我聊天。（*末尾加偷笑表情*）
5.  **（切换四）** 只顾跟你说话，不小心把杯子碰倒了，刚写的东西全毁了。（*末尾加抓狂表情*）

通过不断切换话题，避免在一个话题上纠缠到底，能让对方觉得你思维活跃、与众不同，从而激发她更多的交流兴趣。

## 第三种思维：浅尝辄止

上一节我们学习了如何引领话题，本节中我们来看看如何控制话题的节奏，这就是第三种思维：浅尝辄止。很多人在聊天时习惯将一个话题聊到无话可说，这很容易让对话变得乏味。

正确的做法是，在对方对当前话题兴趣正浓时，主动切断它。这样做能制造神秘感和期待感，让对方对你产生好奇，想要了解更多。

浅尝辄止的思维与引领话题紧密相关。保持神秘感能让女生更被吸引。

**原则：** 一个话题最多不要超过两个回合，尤其在聊天初期，应多切换话题。

以下是浅尝辄止的应用示例：

*   **你：** 你是做什么的？
*   **对方：** 我是平面模特。
*   **你：** 哦。（*平淡反应，展示高价值姿态*）难怪你的腿又直又长。（*结合发散思维*）
*   **对方：** 还好吧。
*   **你：** （*切断话题*）我有点饿了，这几天忙死了，一直加班。（*引入新话题*）
*   **对方：** 你做什么的？
*   **你：** 我在外贸工作。（*第一段*）【配合一张有逼格的公司照片】（*情感画面植入*）
*   **对方：** 你在哪个城市？
*   **你：** （*延迟回复，浅尝辄止*）稍后告诉你。（*制造悬念*）这个时间你应该很忙吧，怎么还有空聊天？（*再次引入新话题*）

掌握浅尝辄止，能让你的聊天张弛有度，始终吸引对方的注意力。

## 第四种思维：轻度调情

让女生觉得你与其他男人不同的一个法宝是适度的情趣，这就是第四种思维：轻度调情。女人不喜欢赤裸裸的调戏或骚扰，她们喜欢的是有分寸、能引发遐想的调情。

调情的重点在于“轻度”。一个懂分寸的男人懂得通过轻度调情让女人去猜测“他是不是喜欢我？”，这种不确定性更有情趣。

在聊天中，可以运用比喻、拟人、夸张等手法进行轻度调情。

以下是轻度调情的几个例子：

*   **例1：**
    *   **你：** 你多高？
    *   **对方：** 168。
    *   **你：** 踮起脚尖，刚好能亲到我。（*后加害羞表情*）
*   **例2：**
    *   **你：** 在干嘛？
    *   **对方：** 在上班。
    *   **你：** 真可惜……（*第一段*）都不能和我去浪漫的约会了。（*第二段，后加害羞表情*）
*   **例3：**
    *   **对方：** 在干嘛呢？
    *   **你：** 我在想你啊。（*后加害羞表情*）

注意保持“轻度”，让对话好玩、有趣，而非具有压迫感。

## 第五种思维：多聊女人

心理学指出，人最关心的是自己，女人也不例外。因此，第五种思维是：多聊女人。在聊天初期，话题应多从女人身上开启（结合发散思维），聊与她自身相关的话题，她才会更感兴趣。

聊天高手更擅长将自己和女人联系在一起。

以下是多聊女人并关联自己的例子：

*   **例1：**
    *   **你：** 你的眼睛又大又圆。
    *   **对方：** 有吗？
    *   **你：** 对啊，你看我的眼睛就没你的大，我都有点嫉妒你了。（*关联自己*）
    *   **对方：** 你是男人还嫉妒我？
    *   **你：** （*切断，应用引领话题*）你在干嘛？（*引入新话题*）
*   **例2：**
    *   **你：** 在做什么？
    *   **对方：** 在上班。
    *   **你：** 我也在上班，突然想你了。（*后加害羞表情*）
    *   **对方：** 你嘴真甜。
    *   **你：** （*结合轻度调情*）你怎么知道我嘴甜？难道昨天在我梦里偷亲我的是你？（*后加害羞表情*）

学会将话题引向对方，并巧妙关联自己，是维持对话热度的关键。

## 第六种思维：展示自己

展示自己是让女生产生好感的关键，但方法不当会显得刻意和讨好。第六种思维就是：恰当地展示自己。你的网络展示面、头像、朋友圈都是在侧面展示自己。

**最佳展示时机是当对方主动问你问题的时候。** 如果对方没问，你就主动炫耀，会惹人反感。

以下是展示自己的正确示例：

*   **例1（展示厨艺）：**
    *   **对方：** 你会做饭啊？（*看到朋友圈后提问*）
    *   **你：** 哦，对呀。
    *   **对方：** 看不出嘛。
    *   **你：** 刚毕业那会儿正好和一个厨师朋友住在一起，就顺便学了一些。（*展示技能和学习经历，隐含价值*）
*   **例2（展示孝心）：**
    *   **对方：** 你还没睡啊？
    *   **你：** 在帮我爸爸挑生日礼物。
    *   **对方：** 你爸爸要过生日了？
    *   **你：** 对啊。……（*简短交流后切断*）我在外地工作太忙回不去，就给老爸挑件礼物寄回去吧，老人辛苦一辈子，现在该我们尽孝了。（*展示重视家人的价值观*）
*   **例3（展示家庭观）：**
    *   **对方：** 你干嘛呢？
    *   **你：** 带我小侄女来动物园玩。
    *   **对方：** 你好有童心啊。
    *   **你：** 对呀，我希望自己以后也有一个这么可爱的女儿。（*展示喜欢女儿的不同观念*）

在对方提问时自然展示价值，并配合相关照片，效果会事半功倍。

## 第七种思维：自大风趣

单纯的“自大”显得浮夸，单纯的“风趣”可能沦为小丑。第七种思维是：将自大与风趣完美结合，形成一种有格调或低调的“装逼”，营造轻松好玩、无压力的氛围。

以下是自大风趣的应用例子：

*   **例1：**
    *   **你：** 刚看到你朋友圈照片，你的左脸很精致啊。
    *   **对方：** 那我右脸就不精致了？
    *   **你：** 抱歉，你那张照片没有右脸啊。（*加偷笑表情*）
*   **例2：**
    *   **你：** 我从你眼神里看到一个别人很少发现的优点。
    *   **对方：** 是什么？
    *   **你：** 你夸我一句，我就告诉你。
    *   **对方：** 你真帅。
    *   **你：** 我发现你很诚实嘛。（*加坏笑表情*）
*   **例3：**
    *   **对方：** 你应该喜欢旅游吧？
    *   **你：** 你怎么看出来的？
    *   **对方：** 感觉到的。
    *   **你：** 是不是感觉我这个男人特高端大气接地气？（*加坏笑表情*）
    *   **对方：** 你太自恋了。
    *   **你：** 这么隐秘的优点都被你发现了，注意我多久了？

自大风趣能让女生觉得你自信、有趣，与众不同。

## 第八种思维：双线理论

最后一种核心思维是双线理论。它要求你在聊天中学会在“正经”与“逗逼”两种状态间自由切换。一直正经显得无趣刻板，一直逗逼则像小丑缺乏价值。

一个优质的男人能将这两种特质完美结合：工作中一丝不苟，生活中懂情趣有品味。

**关键：** 该正经时正经，该逗逼时逗逼。切勿在该表示关心时逗逼（如对方说肚子疼），或在轻松场合过分严肃。

以下是双线切换的对话示例：

*   **你：** 你电到我了。（*逗逼，轻度调情*）
*   **对方：** 真的吗？
*   **你：** 对啊，我开会都不能专心了，被手下人笑话。（*切换正经，并隐性展示管理职位价值*）
*   **对方：** 你在公司啊？（*正经话题*）
*   **你：** 对啊。【发开会照片】我在努力工作，赚奶粉钱。（*先正经后逗逼*）
*   **对方：** 你有孩子了？
*   **你：** 应该快了……没看见我正和孩子他妈在沟通感情嘛。（*逗逼，轻度调情，加坏笑表情*）
*   **对方：** 你又没正型了。（*加流汗表情*）
*   **你：** 怎么才算有正型？努力工作吗？（*切换正经*）
*   **对方：** 那必须的。（*加得意表情*）
*   **你：** 好……遵命，老婆大人。（*切换逗逼，加害羞表情*）

通过不断在正经与逗逼间切换，能让对话充满张力，让对方觉得你生动有趣，欲罢不能。
