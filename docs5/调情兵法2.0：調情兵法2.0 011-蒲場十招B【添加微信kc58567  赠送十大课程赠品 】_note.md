# 调情兵法2.0：011：夜场十招详解

在本节课中，我们将详细解析在夜场环境中常用的一些策略与技巧。这些方法涉及自我保护、氛围营造和互动节奏控制，旨在帮助初学者理解夜场社交的潜在规则。

## 偷鸡技巧

上一节我们介绍了课程概述，本节中我们来看看具体的技巧。“偷鸡”在这里指代各种减少饮酒量或避免醉酒的小方法。

以下是几种常见的“偷鸡”方式：

1.  **使用纸巾**：这是最基本的一步。可以将酒倒在纸巾上，或者用纸巾擦拭杯口时顺势倒掉一些酒。在古代，人们喝酒时也会用类似方法。
2.  **注意酒类选择**：红酒或其他有色酒类并非最佳选择。除非灯光非常昏暗，否则容易被发现。国内许多夜场使用紫色灯光，这会让很多东西看起来发红，但仍需注意。
3.  **警惕混合饮料**：纯烈酒或许容易应对，但混合了酸性汽水的饮料更容易引起胃部不适导致呕吐。这是因为汽水中的气体所致，而非酒精本身。
4.  **利用水果**：在酒杯中放入如小番茄、菠萝块等水果，酒量就会减少。例如，放入一块西瓜后，杯中的酒可能只剩半杯不到。这样既减少了饮酒量，外观上也不易察觉。

如果没有特定目标，可以大方地制作一杯“水果宾治”，自己少喝的同时，也能陪他人慢慢饮用。如果有目标，则需视情况决定是否使用此招，以保持公平。可以在游戏进行到一定程度、需要调整节奏时再使用。

关于将自己杯中的水果放入对方杯中以示关心，这种做法因人而异。从男性视角看，女性这样做可能让对方感觉更舒服。另一种更含蓄的方法是吃掉自己杯中的水果。将东西放入对方杯中是一种更明显的示好行为。但需注意，果盘中的水果可能已沾染酒渍或烟灰，因此更稳妥的做法是处理自己杯中的水果。

“偷鸡”还有其他方式，例如频繁去洗手间，以吐出酒液，但这招不能常用。另一种方法是假装呕吐去洗手间催吐，以减少体内酒精，此方法可在饮酒过量时使用。

打电话也是一招。可以假装有电话进来，借此离开卡座。女性也常用“接朋友”这招，声称朋友在门口但始终未到，实际上可能已转去其他卡座。

## 呼叫支援（Call马）

讲完扮醉，就要讲“Call马”（呼叫支援），这招极其常用。

如果你出去玩，几乎不可能单独一人。我从未见过单独去夜场的女性。我见过最少的是两人同行，两人站在吧台点一杯长岛冰茶，这摆明了是“待勾搭”的状态。这不仅是为了玩，就算是为了工作，也很少女性会独自落座，这确实危险。

第二个原因是，即使你酒量再好，也架不住对方车轮战。被围攻是常有的事。在国内夜场，可能几个男性开一张卡座，消费平摊下来每人没多少，但他们可以轮流上来敬酒。

“Call马”最常用的情形是，当你和两三个朋友一起去但不在同一卡座，而你在自己卡座被灌得不行时，你就会呼叫支援。最快的方式是联系现场的朋友或熟人。

如果是我的话，我会叫一些级别较高的“观音兵”过来一起玩，然后把注意力引到他们身上，营造出他是来救你的气氛，这样可以杜绝现场其他有不良企图的人。

这让我想起之前的一件事：我和朋友出去玩，有个女孩不知为何被隔壁卡座的一群人围着灌酒。我们想过去救她却做不到，我们自己过去也可能被灌醉甚至被请出场。最后我建议她叫个男性朋友来救她，女孩自己叫不来。后来她叫了一个男性朋友，结果那个男性也被灌醉了。据我所知，最后那个男的回不了家，那个女孩也不知怎么去了别人家。这是“战斗力”的问题。

## 后发制人

讲完“Call马”，要讲另一招非常常见但比较隐晦的技巧：“后发制人”。

为什么这么说？因为有些女孩一开始会忍耐。她们在刚被带到卡座或游戏开始时，会不断推脱，说“我不行”、“我不会”，以此保存自己的战斗力。这通常是经验比较丰富的人才会做的。

因为她们知道自己常驻夜场，需要保存实力。我自己就常用这招。不一定是为了保持战斗力，而是当我感觉当前没有目标时，我就会先不玩，保存实力。同时，这也是一种保持矜持的技巧。老实说，即使在夜场工作的女孩，一开始也要矜持，通过推脱来告诉对方：你需要花点心思才能让我跟你玩。

我个人有时做得比较绝：就算别人用激将法撩我玩，我也完全不理，或者直接告诉对方“你别来搞我，把我搞火了大家都不好看”。在夜场里，别人不知道你的底细。所以，各位男士如果遇到这样的女性，你就要看和她对话的前十几秒。我们常说“秒决定生死”。如果女孩一开始就有这种防御姿态，那么全场那么多目标，不一定非要是她。

## 扮鹌鹑（装柔弱）

讲完“后发制人”，最后讲一点：“扮鹌鹑”（装柔弱）。这招与后发制人有些相反，因为它需要在一开始就使用。

“扮鹌鹑”这招在“邓场”（指跳舞、气氛热烈的场子）要慎用，几乎是绝杀招。因为“邓场”给人的感觉是能喝、会玩、放得开。如果你一进去就扮鹌鹑，别人会觉得你很例外，像刚“下海”一样。不过也要看客人，如果你面对的是常泡吧的老手，你用这招可能没用，对方会直接拆穿你。这招适合对付那些看起来比较“斯文”或“有身份”的客人。

为什么这招在“邓场”要慎用？因为我见过的“邓场”女孩不够多，但我以前见过的小姐用这招就很合适。你知道在东莞那些地方，很多女孩会编造类似“家庭困难”的故事，有三千个版本，千万别信，大家听的故事都差不多。扮鹌鹑在“邓场”也可以用，适合那些刚入职不久的新人，或者你自己和朋友出去玩时，第一次见的女孩也可以用这招。
