# 挽回爱情-蜕变高级音频课程：23：心理准备 🧠

在本节课中，我们将学习在尝试联系前任前，如何进行充分的心理准备。这是确保后续行动有效、避免关系恶化的关键一步。

上一节我们介绍了断联的作用，本节中我们来看看如何评估自己是否已经做好了重新联系的心理准备。

理想的情况是，通过一段时间的断联，你能够意识到过去关系中的错误做法。同时，对方可能也愿意与你进行一些联系，或者你们的关系已经有所缓和。

然而，有时这种情况并不会发生。如果断联后关系仍未改善，通常意味着你的前任目前认为，与你在一起无法解决你们之间存在的任何问题。你的前任可能宁愿选择与别人在一起，或者已经遇到了他认为更合适的人。另一种可能是，你的前任对你已经失去了基本的信任。

一旦断联没有起到预期的作用，你就可以考虑采取直接或间接的方式去联系对方，并按照课程中的方法来重新建立联系。**我们并非只能被动等待对方主动联系**。在当前的阶段，主动发起直接或间接的联系是完全可行的选择。

但是，在采取行动之前，你必须做好充分的心理准备。

以下是几种你必须准备好面对的情况：
*   你可能会看到你的前男友与其他异性在一起。
*   你可能会看到他与他的亲人或朋友在一起。

当你们无意中碰见，或者你通过微博等其他途径得知这类信息时，**千万不要惊慌失措、感到痛苦或产生嫉妒**。这些情绪反应对于修复你们的关系毫无帮助，这一点至关重要。除非你已为此做好准备，否则在联系层面，你应当暂时停止。

如果你目前仍然感到身心疲惫，无法理解或接受另一个男人或女人出现在你前任身边，那么你可能还需要继续断联。断联最主要的目的，是让你自己做好准备。**主要是在你仍然妒火中烧的情况下，采取任何联系方式都不会有太大效果**。因为对方可以从你的语气、文字甚至表情中，察觉到你仍然非常在乎他。

你的情绪化往往会导致一切努力回到原点，甚至使关系比之前更糟。

因此，我今天的建议是，大家需要做一个自我测验。通过这个测验，来评估自己是否能在上述情况下保持情绪稳定。

以下是自我测验的核心评估标准：
*   **通过测验**：代表你已经准备好了，可以尝试接触。
*   **未通过测验**：代表你暂时还不适合接触你的前任。

本节课中我们一起学习了进行心理准备的重要性，以及通过自我测验来评估是否具备重新联系前任的情绪稳定性。这是挽回过程中保护自己、稳步推进的关键一步。
