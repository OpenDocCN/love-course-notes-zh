## 线上速约的核心方法

上一节我们概述了课程目标，本节中我们来看看如何高效地进行线上邀约。我的方法强调“速约”，即用最少的聊天将女生约出来。

### 速约的基本逻辑
女生答应见面，通常需要解决两个核心问题：
1.  **我为什么要见你？** (提供合理的见面理由，即“台阶”)
2.  **我们出来做什么？** (明确约会内容)

### 速约的标准流程
以下是执行速约的具体步骤：

1.  **建立基础融洽感**：前期聊天仅需5句左右，目的是建立初步的友好氛围，并观察女生反应。
2.  **判断兴趣指标**：如果女生反应良好（例如主动提问、回复内容多、使用表情追问），即可视为兴趣指标，可以直接邀约。
3.  **发出邀约**：邀约信息需同时包含“见面理由”和“约会内容”。
    *   **公式**：`邀约信息 = 见面理由 + 约会内容`
    *   **示例代码（模拟话术）**：
        ```python
        见面理由 = "我明天要离开北京了，走之前很想见见你。"  # 或“觉得你挺有眼缘的”
        约会内容 = "我们去附近的小酒吧喝一杯吧。"
        邀约信息 = 见面理由 + " " + 约会内容
        发送(邀约信息)
        ```
4.  **应对废物测试**：速约几乎100%会遭遇女生的拒绝或质疑（如“我们不熟”、“太远了”）。此时关键在于态度强硬、框架坚定。
    *   **核心策略**：不解释，不恳求，而是用强势态度反转局面。
    *   **关键话术**：“你不想见我的话，我来不来都不重要。” 或 “机会是留给勇敢的人的。”

### 邀约后的注意事项
即使女生答应了邀约，也可能在见面前临时变卦。如果被放鸽子，正确的应对方式是：
*   **表达态度**：展现出“你让我失望了，我受委屈了”的姿态，而非忍气吞声。
*   **再次邀约**：紧接着提出补偿要求，为下一次邀约铺垫。
