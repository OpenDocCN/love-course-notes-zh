# 新必牵手挽回课程：7：第七课：挽回的间接联系技术 🎯

在本节课中，我们将要学习挽回过程中的一项核心技术——间接联系技术。当你与前任已经度过冷处理阶段，并开始恢复初步联系时，这项技术能帮助你稳定、低风险地推进关系，为最终复合打下坚实基础。

上一节我们介绍了冷处理与断联的重要性，本节中我们来看看如何巧妙地重新建立联系。

## 间接联系技术的作用与时机

间接联系技术能有效且稳定地加快双方复合的进程。其风险较低，能循序渐进地推进挽回。配合下一课的直接联系技术，将更高效地帮助你挽回前任。

使用此技术的前提是：冷处理时间已足够。你送出的复合礼物与所做的挽回努力已产生积极效果。部分学员甚至已被移出黑名单，或前任开始主动联系、在社交平台留言。

我知道此时你迫切希望复合，但必须百分百明确：**现在不能直接联系对方，更不能提及任何感情话题**。即使对方约你见面，也不能谈论复合，否则会让挽回变得更加困难。

你可能会问，为何前任已主动示好，却不能提复合？请先冷静，我们来分析前任当前的心理状态。

## 前任的心理状态分析

恢复联系初期，前任的心理通常由两种因素构成：
1.  对你个人改变的好奇，以及残留的感情（约占50%）。
2.  过去积累的不满、未通过的情感测试或负面情绪（约占50%）。

人性趋利避害。一段时间不见，你的高价值回归与积极改变，会让他重新产生兴趣，但又不敢贸然接近。你们正处于一个“复合试水”阶段。若把握得当，他很可能会逐渐对你开放机会，甚至主动提出复合。

那么，具体该如何操作呢？以下是间接联系技术的具体方法和技巧。

## 核心操作方法与技巧

首先，**保证100%不主动联系对方**，除非他主动联系你。若他主动联系，可以像哄小女生一样回应，或传递正能量内容，但切记不可谈论感情。

同时，你需要持续在朋友圈或QQ空间，有针对性地发布展示你生活提升、变得成熟稳重的动态。记住要**散发正能量，图文并茂更佳**，目的是吸引他主动联系或留言。

当他开始在你的动态下留言或主动联系时，你就可以借此机会与他进行间接的、轻松的聊天。

**补充说明**：若在挽回过程中遇到节日，可假装群发一条简短的祝福短信。例如：
```
平安夜快乐。
```
或
```
元旦快乐。
```
发送后观察其反应。若有回复，可聊聊近况，但**依然不能提及任何感情话题**。

你可以分享近期的工作与生活，内容需与你之前展示的改变一致，是更新鲜、更有意义、更正能量的事情。如果他的情绪被你感染，则证明间接联系技巧效果显著，你的挽回成功迈进了一大步。

也有学员因是同事或同学，在断联后，对方主动找你聊新鲜事物并被你感染，这同样是挽回成功的重大进展。

接下来，我将教你间接联系技巧中威力最大、最有效的方法。

## 威力巨大的进阶技巧：旧情人出口，好人帮你出手

此方法的核心是：**借助你与前任的共同好友或他的闺蜜来为你说好话**。

首先，你需要能“收买”或调动他身边的人来帮忙。因为有些朋友虽想帮忙，但不会说话，反而可能弄巧成拙。我们的学员中已有不少此类反馈。

因此，你需要让帮忙的“好人”熟记你精心设计的话术模板，再去沟通。最好事先请他们吃饭、送些礼物以建立良好关系。

以下是一个话术模板示例（以“汪峰想挽回章子怡”为例）：

> 子怡呀，最近汪峰整天找我，但是都没说你们之间的感情事。我见他人开朗了很多，精神状态和脸色都很好。
> 
> 你之前不是整天说他那里不好吗？之前你们在一起的时候，我也觉得他缺点很多。不过你们分手这么久，我最近见他工作越来越努力，好像奋发图强了。
> 
> 而且你之前不是说他不成熟吗？之前我和一些朋友去他家吃饭，看见他书房多了很多创业和两性心理学的书。我问他都看过吗？他说都看过两三遍，还说不懂爱情，现在懂了。
> 
> 我觉得他现在变了很多，变得成熟稳重，还会做家常菜，像个居家好男人了。我真的很惊讶失恋的力量这么大。
> 
> 我还听说他家里给他介绍了条件很好的女孩，但他好像不怎么感兴趣。又听朋友说，别人给他介绍了两个美女，那些美女都挺喜欢他的，但他说最近以事业为重，都没谈。
> 
> 对呀，汪峰和你分手之后，好像变得越来越好了。你不好奇吗？不去看看他是不是真的变得这么好吗？如果真的变好了，就顺手捡回来呀。要是你不捡回来，可能就便宜其他女人了。

如果她身边的好友都能这样为你说话，可以保证他很快就会主动来找你。

当他主动找你时，继续使用前述技巧，慢慢缓和关系。如果你感觉到他对你的兴趣指标越来越高，就可以准备运用下一课将教的直接联系技术，促使他回到你身边。
