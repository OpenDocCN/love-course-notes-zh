# 权谋班：5：政变术

## 概述
在本节课中，我们将学习权谋流程中的关键转折点——政变术。政变术是从积累能量、夺取权力到维护权力的核心环节。我们将探讨政变的本质、适用人群、时机、手段与具体步骤，并结合职场实例，帮助初学者理解如何在实际环境中应用这些原则。

## 课程结构回顾与本节定位
上一节我们介绍了权力积累的过程，本节中我们来看看如何实现从量变到质变的飞跃——政变。

我们整个十节课的权谋学习是一个完整的流程：前四课属于夺权的过程，需要大家在人生中逐步练习；第五课“政变术”是关键的转折点，标志着从积累到夺取的实质性转变；而接下来的五课则将重点转向如何管理和维护已获得的权力。从本节课开始，内容将更多地涉及大家可能缺乏直接经验的领域，需要通过案例观摩来学习。这些知识对于未来步入管理岗位或经营自己的事业至关重要。

## 学习权谋的三种核心思维
在学习具体的政变术之前，必须掌握三种基础的思维方式，这是理解并运用后续所有权谋技术的前提。

以下是三种核心思维方式：
1.  **从（领导的）利益角度出发**：一切行动的出发点应是维护领导的个人利益，而非个人或集体利益。领导的感觉和信任是关键。
    *   **公式**：`你的行动准则 = 最大化领导的个人利益`
2.  **从领导（大局）角度出发**：学会换位思考，站在领导的位置考虑问题。这能让你知己知彼，至少立于不败之地。
    *   **核心原则**：先求不败，而后求胜。保存现有成果，避免犯错，再图发展。
3.  **以发展的眼光看问题**：不要局限于眼前，要识别和投资未来的“潜力股”。编织关系网应着眼于未来十年能成为社会中坚力量的人。
    *   **策略**：选择未来有潜力的对象进行长期投资，而非巴结当前得势者。

## 职场实践：如何向上级要求加薪或权力
在深入政变术之前，我们先探讨一个直接的职场应用：如何有效地向上级提出加薪或获取更多权力的要求。这本身就是一种微型的“权力谈判”。

### 谈判前的自我评估：四个筹码
在向领导提出任何要求前，你必须扪心自问，是否具备以下四个筹码：

以下是评估自身价值的四个关键问题：
1.  **自我价值**：你个人具备什么样的能力和价值？
2.  **已创造的价值**：你过去为公司/领导具体做出了哪些贡献？
3.  **忠诚度**：领导是否绝对相信你的忠诚？你是否毫无私心？
4.  **潜在价值**：除了本职工作，你还能为领导/公司创造哪些额外价值？

### 提出要求时的四个禁忌
如果你决定提出要求，务必避免以下四种错误方式：

以下是提出要求时必须避免的四种行为：
1.  **不要在关键时刻或领导安排工作时提**：这形同要挟，即使领导暂时屈服，也会对你产生负面看法。
2.  **不要以“未来会更努力”作为交换条件**：这暗示你现在不够努力，是一种要挟。
3.  **不要在公开场合提出**：这将领导置于两难境地，容易导致关系僵化，迫使你离职。
4.  **不要忽视领导的风险意识**：领导会考虑给你加薪后，你是否会得寸进尺，或引发团队不平衡。你需要持续展示忠诚度以降低其风险感。

### 理解领导的思维：四个考量
同时，你需要理解领导在面对你的要求时，是如何思考的：

以下是领导考虑是否同意要求时的四个核心考量：
1.  **工作绩效**：你实际干了多少活？是否只是在混日子？
2.  **员工需要**：你是否真的有迫切需求（如结婚、家人生病）？
3.  **团队平衡**：给你加薪后，如何平衡其他员工的感受？是否会引发连锁反应？
4.  **额外风险**：这次妥协是否会纵容你或其他人未来继续要挟？

**最佳策略**：先做出卓越贡献，通过不着痕迹的暗示（如提及生活压力）让领导主动意识到你的价值。聪明的领导自然会给予激励，因为这符合他留住核心人才的利益。

## 政变术的核心内涵
现在，我们正式进入“政变术”的主题。政变术并非只适用于宫廷斗争，在企业管理和职场晋升中，它体现为一种能量的爆发式转换。

### 政变的三重意义
1.  **能量积累的爆点**：当个人或派系积累的能量（资源、人脉、威望）达到临界点时，必然发生质变。例如赵匡胤“黄袍加身”，下属需要他更进一步以获取自身利益。
2.  **能量转换的转折点**：此时不进取，就会成为他人的威胁而被除掉。例如司马懿、燕王朱棣，政变是自保的必要手段。`公式：能量临界点 → (主动政变 ∪ 被清除)`
3.  **高风险性**：政变成败关乎生死存亡，不存在维持现状的选项。但不行动的风险是平庸一生。

### 具备政变潜力的人群
历史上，三类人最易发动政变：皇族成员、天子近臣、朝廷权臣。他们的共同点是**最贴近权力核心资源**。
在现代职场和企业中，对应的是以下三类人：

以下是现代组织中具备“政变”潜力的三类人：
1.  **业务人员**：掌握核心销售渠道和客户关系。他们最容易“叛逃”并带走资源，或自立门户。
2.  **核心技术人员**：掌握公司的技术秘密、专利或核心配方。这是公司立身之本，一旦被窃取，竞争力将大打折扣。
3.  **人脉中心人物**：并非指官员，而是各种信息的联络枢纽，能整合行业资源。他们利用平台结识的关系网，在时机成熟时可以变现。

## 政变的时机、手段与步骤
### 政变时机
1.  **内部导向完成时**：当你手下的人员（尤其是业务骨干）大部分都倾向于你，唯你马首是瞻。
2.  **组织困难时期**：当公司出现资金链断裂、内部混乱等危机时，是出手的良机。有时甚至需要主动“敲打”或制造混乱，以创造机会。
3.  **关联交易准备就绪时**：通过设立外部公司进行关联交易，逐步掏空原公司资源，为最终接管或另起炉灶做准备。

### 政变手段（现代企业版）
1.  **送入监狱（现代“暗杀”）**：收集对手的黑材料，通过合法手段使其入狱，从而接管其势力或公司。关键是要确保自己能接收成果。
2.  **拉走队伍（“兵变”）**：凭借个人魅力和威望，在关键时刻煽动核心团队集体出走，另立山头。
3.  **制造坏账/混乱（“伪造遗诏”）**：利用职权签订有漏洞的合同，制造无法收回的坏账，或诱导领导做出错误的投资决策，导致公司陷入困境，再由你来收拾残局。

### 政变五步法
以下是发动一次成功“政变”的五个关键步骤：
1.  **舆论准备**：在行动前，通过舆论（如水军、流言）塑造己方正义、对方无能的形象，为行动提供合理性。例如抹黑对手，宣扬自身能力。
2.  **忠诚测试**：在关键节点测试团队的忠诚度，确保核心人员与你站在一起。历史上的“指鹿为马”即是经典案例。
3.  **接收考评**：评估政变成功后，胜利果实是否能稳稳落入你手中，是否会为他人作嫁衣裳。必须确保自己有实力控制局面。
4.  **寻找/制造爆点**：主动制造或利用混乱（如公司危机、舆论事件），在“浑水”中实现“摸鱼”。太平无事则难有机会。
5.  **粉饰栽赃**：成功夺权后，需要粉饰太平，将政变行为合理化，并将责任栽赃给前任或对手，将自己塑造成“救世主”或道德领袖，以稳固权力。

本节课我们一起深入学习了权谋中的高级技术——政变术。我们从职场谈判的微观应用切入，理解了基于价值与筹码的博弈思维。进而，我们探讨了政变的本质是能量积累到临界点的必然爆发，识别了业务、技术、人脉三类具备政变潜力的人群。最后，我们系统梳理了政变的时机选择、三种手段以及从“舆论准备”到“粉饰栽赃”的五步操作流程。

政变术是权谋从积累转向夺取的实战环节，其核心在于对时机的精准把握、对资源的有效控制以及对人心向背的透彻理解。记住，所有的行动都应建立在深刻的换位思考和对利益格局的清醒认知之上。下节课，我们将进入权力维护阶段，学习如何巩固胜利果实。
