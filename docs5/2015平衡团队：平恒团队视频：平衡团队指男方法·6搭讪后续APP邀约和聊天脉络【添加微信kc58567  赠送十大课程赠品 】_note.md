# 平衡团队指男方法：6：搭讪后续APP邀约和聊天脉络 🗣️💬

在本节课中，我们将学习如何通过社交软件（如微信、QQ）进行搭讪后的后续联系。我们将重点介绍一种独创的“双管齐下法则”，并详细解析其核心组成部分——聊天脉络，帮助你高效地营造吸引力和舒适感，从而顺利推进关系并完成邀约。

上一节课我们介绍了电话后续。但在很多情况下，在街头搭讪时，更多人倾向于交换微信或QQ等社交软件的联系方式。

所以，本节课将专注于APP后续，并分享我们团队独创的“双管齐下法则”。顾名思义，这个法则要求同时营造**吸引力**和**舒适感**。

如果单独先营造吸引力，再去营造舒适感，这个过程会非常漫长。可能各需要一小时，无疑会浪费大量精力和资源。

如果同时营造吸引力和舒适感，可能仅需半小时即可完成。

“双管齐下法则”分为四大块：
1.  聊天脉络
2.  基本信息交换方法：快速交换法
3.  邀约的时机
4.  约会邀约的话术以及“真程测试”

“真程测试”对大家来说可能是一个陌生的概念。

聊天脉络是基于“多重对话脉络”系统升级创造而来。它与“多重对话脉络”最大的区别在于更经典、更实在。

大家记住一句话：聊天脉络是“多重对话脉络”的**升级版**，而非进化版。

在我们平时的聊天中，分为**主脉络**和**副脉络**。通常的聊天只有一条主脉络。

例如，你问女孩子：“你在干什么？” 这是一句话，也是一个主脉络。同时，它也可以被视为一个副脉络。

这里大家可能会疑惑：主脉络和副脉络的区别是什么？

主脉络是指你发出的一段完整的话。例如：“艾伦喜欢游泳，因为他觉得这有益身体健康，同时可以提升性能力，当然平衡导师也打算学一下游泳。” 这是一条主脉络。

这条主脉络中包含4条副脉络：
1.  艾伦喜欢游泳
2.  因为这有益身体健康
3.  可以提升性能力
4.  平衡导师也打算学一下

当我们和女孩子聊天时，她可以针对这些副脉络的内容进行回应，并将话题延续下去。每一条副脉络都可以发展成一条新的主脉络，而每一条主脉络也同时包含好几个或一个副脉络。这个概念必须理解清楚，因为它贯穿后续的所有步骤。

在普通聊天中，我们通常只有一条主脉络。例如：
-   男：你在干什么？
-   女：我在洗澡。
-   男：哦，洗澡呢。我也在洗澡。

这就是只有一条主脉络的对话。

技术好一点的兄弟，懂得使用一个以上的副脉络来引导女孩子说更多内容。例如：
-   男：你在干嘛呢？
-   女：我在洗澡。
-   男：洗澡呢？看来你是个蛮爱干净的姑娘。那你一周洗几次澡？
-   女：一周洗两次。
-   男：哦，我们平时都是洗三次的。

但要注意，即使副脉络再多，也不可能在不同的思维模式间随意转换。你不能正在用理性思维聊天时，突然插入一个感性的副脉络。在一条完整的主脉络内，只能有一种思维模式：要么理性，要么感性。

如果我们有两条主脉络同时进行聊天，一条感性，一条理性，效果会如何？

我给大家举个例子。有一次在微信上，我跟一个女孩子聊天。
开场我发了一个笑脸过去。这是一个很经典的开场。搭讪后打招呼，发一个笑脸即可，不要多说其他。

她回了我一个笑脸。
接着，我开启了一条主脉络（定义为感性）：“你还记得我叫什么吗？”
她回复：“杜子豪。”
我说：“你答对了。我看你的朋友圈，感觉你是一个特别喜欢读书的女孩子。”（这里我回应了她，并同时开启了另一条主脉络，定义为理性。）

此时，我们有了两条主脉络：一条感性（关于名字的记忆），一条理性（关于读书的爱好）。

她说：“对，平时没事喜欢读书。”
这时我要从理性切换回感性。我说：“那你把我名字里‘杜’的木字旁去掉，再把‘子’去掉，你看看是什么。”
这是一个感性的互动。

我不停地在两个话题间切换聊天，这是只有熟人之间才会做的事。她会觉得这样聊天很舒服，因为永远不用担心接下来聊什么，永远不会出现因不知如何回复而长时间不回消息的情况。

她回复：“土豪。”
这时她已经能感受到这种理性的互动了。当她以为我会继续时，我的第三句话又是一句理性的：“今天不是说我也挺喜欢读书的。我觉得读书就是一种旅行。”
这是一种很理性的说法。

我说完这个，她说：“对呀，不是有句话叫‘要么读书，要么旅行，身体和灵魂总有一个在路上’。” 她很主动地回应，这是一种理性的交谈。

我的第四句话是：“恭喜你认识这个土豪。” 或者说：“恭喜你正在和一个土豪聊天。”
这时她的反应会是笑。我们在用理性展示价值的同时，也用感性展示了幽默和与她的合拍。

理性、感性、理性、感性……就这样一直聊下去。可能不用多少句，十几句话后，我就可以说：“周日怎么安排？”

这就是双重主脉络的意义：一重理性，一重感性。它能让她更快地熟悉我们，感觉我们是一个可靠的男人，更快地了解和吸引她。

就像看一部连续剧，里面有开心、有悲伤、有放松、有枯燥、有高潮。如果全是高潮，看一会儿就觉得没劲；如果全是枯燥无聊，根本不会看。但如果是高潮之后有个瓶颈期，当你觉得无聊时突然又来一个高潮，你就会觉得“很爽”。我们要的就是这种感觉，这就是两条主脉络的作用。

接下来我们讲副脉络。副脉络的目的是在吸引她的同时，去营造舒适感。

就像平时兄弟们聊天，你问：“你在干什么？”（主脉络）
女人说：“我在吃饭。”

这里说一个题外话，也是一个小技巧：女人回复我们的内容有不同的含义。
-   如果她只回“在吃饭”，证明兴趣不大。
-   如果她说“在吃饭，吃的米饭”，态度相对普通。
-   如果她说“我在吃饭啊，在一家很不错的地方吃着米线，这家很好吃”，她说了具体事情和感受，这时你就知道可以筛选并邀约了。

这是回答的三个层次，题外话简单了解即可。重点是今天讲的内容。

女人回“我在吃饭”后，普通兄弟会延伸这个主脉络：“哦，在吃饭啊，吃了什么？” 女人可能回“米饭”。然后你问：“自己做的吗？” 女人回“是或不是”。这个过程毫无意义，只是浪费时间。

我们来看使用副脉络聊天，在主脉络中加入多个副脉络的效果。

还是刚才的对话：
-   男：你在干什么？
-   女：我在吃饭。

我们开始在主脉络中加入副脉络。我们原本的主脉络是想问“你吃了什么”。但加入副脉络后，我可以这样说：“哦，我也该吃饭了。/我刚吃完。朋友给我送的他亲手捏的饺子，很好吃。对了，你吃的什么？”

我们为什么要这样说？这里面有几个副脉络：
1.  我刚吃完饭。
2.  饭是朋友送的。
3.  他亲手做的。
4.  很好吃。
5.  你吃的什么？（保险问题）

这可以让女人从多个话题点回应。她可能回复：
-   针对1：“你吃饭挺早。”
-   针对2：“你朋友真好。”
-   针对3：“她漂亮吗？是不是在追你？”
-   针对4：“是吗？下次有机会尝尝，我包的也很好吃。”
-   针对5：“我吃的是米饭，我妈做的也很好吃。”（这是最容易回复的保险问题）

即使她对你不感兴趣，我们也可以用最后一个副脉络（保险问题）让她回复。这等于装了一个“肯定回复”的保险。

同时，这些副脉络展示了价值：朋友（而且是女性）亲手送饭，证明我们是一个不错的男人，能得到贤惠女孩的青睐。“很好吃”展示了我们尊重朋友。最后一个则是保险。

我们在展示价值的同时，用普通朋友聊天的方式（就像熟人之间随口分享日常），在营造舒适感的同时营造了吸引力。

这时，女人可能会回复更多内容。因为你提供了多个价值点，根据平衡导师心态课所讲，你以什么态度对人，别人也会以什么态度对你。同样，我们开启了多个副脉络，她也会回复2到4个副脉络。这样，我就可以根据她的副脉络扩展更多话题。

这样，我不但话题源源不断，而且不断在吸引她，不断增进熟悉感。她会感觉聊天越来越放松，同时觉得我是一个擅长社交、很不错的男人。这时，我们就可以推进关系了。

这就是我们整体的聊天脉络：主脉络分为两条（一理性，一感性），副脉络则服务于两个目的（吸引和舒适感）。

现在给兄弟们说一下具体如何操作，如何扩展主脉络，如何区分它们。

很多兄弟明白了概念区别，但在具体聊天中还是分不清，把副脉络也当主脉络发，或者反过来。

错误示例：问“在干什么”，女人回“在吃饭”。然后他连续发三条：
1.  “哦，我也刚吃完。”（发送）
2.  “朋友给我送的。”（发送）
3.  “很好吃。”（发送）

这是不对的，这相当于发了三条主脉络。

正确的副脉络是：在主脉络里一次性说完想说的话再发送。例如：“我刚吃完，朋友给我送的他亲手捏的饺子，很好吃。对了，你吃的什么？” 这是一条主脉络，里面包含多个副脉络。

第二条主脉络什么时候发？等你发完上一条，她回复后，你再根据她的回复开启下一条主脉络。

**只要是一次性说完发送的，就是一条主脉络，里面可能包含多个副脉络。** 兄弟们千万不要搞混。

这就是主脉络和副脉络的区别，以及理性和感性的区别。

很多兄弟会问：我听懂了主副脉络，但怎么才能做到理性和感性同时存在呢？

其实你们只是陷入了思维死角。我们还用刚才的例子。
我们问：“你在干什么？” 她回：“在吃饭。”
这时我们可以说：“哎，我今天发生了一些奇怪的事情。”（**理性**主脉络开启）
她可能会回：“什么事？”（针对理性回应）
然后你说：“对，我也刚吃完饭，你们吃的够晚的啊。”（**感性**地延伸刚才“吃饭”的话题）
不管她回什么，我们下一句可以说：“就说今天我在万达买衣服的时候，撞了一个女的一下，她非说我非礼她。”（**感性**主脉络）

用切换话题的方式去区分理性和感性。理性很简单，就是问一些关于她的日常问题。感性则可以通过讲故事、分享趣事来达到。

最简单的方式就是，别管那么多，从我们的案例库里随便挑一个有趣的故事去“泡”就行。
