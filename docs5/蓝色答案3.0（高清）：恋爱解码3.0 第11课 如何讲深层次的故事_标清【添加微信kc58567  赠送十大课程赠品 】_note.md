# 恋爱解码3.0：第11课：如何讲述深层次的故事 🧠

在本节课中，我们将要学习如何讲述关于自己的深层次故事。这类故事能够分享你内心的情感和情绪，是建立深度情感连接的关键技巧。我们将探讨其核心概念、讲述时机、重要性以及具体的讲述方法。

## 概述：什么是深层次的故事

深层次的故事是指能够分享你内心情感和情绪的个人经历。这类故事的题材通常包括你的梦想、童年经历、家庭亲情或情感经历等。这些题材能有效地带出你当时的情感和情绪。

学会讲述深层次的故事，可能产生一个很好的效果：你们能够产生情感共鸣，对方会更理解你、更懂你。同时，你营造出的这种氛围，更容易让对方也向你倾诉，从而使你们能够很好地走入彼此的内心世界。因此，这个部分非常重要。

## 讲述深层次故事的时机 ⏰

上一节我们介绍了深层次故事的概念，本节中我们来看看讲述它的正确时机。

在生活中，有人可能会犯一个错误：当他想拉近与对方的心理距离，并且觉得对方反应也不错时，他会主动去讲自己的事情。这个习惯很好，但时机可能不对。时机不对，效果往往会非常糟糕。

**核心原因在于关系阶段**：当你还没有能力去升高关系，没有能力给对方提供性的刺激时，去讲很多关于自己的事情、去讲深层次的故事或经历是不合适的。尤其是篇幅很长的故事，对方不可能保持注意力。你讲着讲着，对方会觉得无聊，因为故事与他没有直接关系。

通常，一个深层次的故事短则5分钟，长则10分钟，时间比较长。如果对方注意力一直集中在你身上，但过程又缺乏乐趣，他是不会给你持续注意力的。

**有效的时机公式**：
```
有能力升高关系 + 有能力提供恰当的性吸引力 -> 讲述深层次故事有效
```
除非你有能力去升高你们之间的关系，有能力以正确的态度给对方提供性的刺激。你非常勇敢、确定，善于眼神接触和表达欣赏。当具备这个能力之后，再去讲述深层次的故事才是有效的。你再去讲自己内心的一些情感和情绪才是真正有效的。**一定是要在升高关系之后，或者说在能够给对方性的刺激之后**，不然是没有效果的。

因为当你升高关系后，你们已经是潜在的恋爱关系。你是他今后可能的恋爱对象，他当然会很关心，会把注意力给你。哪怕听很长且不是很有趣的内容，他也会非常关心并沉浸其中，因为这个人是他所关心的。所以这个人的经历他愿意去听。

但是，如果你没有这样的能力，那他就不关心。这对他来说根本不重要，而且还没有意思，还很无聊、很长。所以这就是天壤之别。

**关键点总结**：一定要在你有能力升高关系、有能力给对方提供性的刺激之后，你再去讲深层次的故事，再去分享自己内心的情感和情绪，这才是有效的。这是第一要点。

## 为何要讲述深层次的故事？💡

上一节我们明确了讲述的时机，本节我们来探讨为何需要讲述深层次的故事。

如果问一个问题：与对方推进关系、发生亲密关系，是否一定需要讲深层次的故事？答案可能是不一定。但是，如果要让对方懂你、理解你，要让你们真正地彼此走入内心，可以说这是必须的。你必须把内心深层次的情感去跟对方分享，这样你才能够得到对方的理解和支持。这是必须的。

这可能源于我们以往的生活经历。从小到大的成长经历，让我们的内心就像一扇大门一样关闭着。我们不习惯或者害怕去分享内心的一些情感和情绪，这其实是一种过度的自我保护。你不去说，别人不会有评价、轻视或嘲笑。但是，你过度保护了自己，对方也无法真正去知道你内心经历过什么、有过怎样的体验。对方无法真正去理解你。

所以，你过度保护自己，把这扇心门关上了，对于你与对方真正的情感融合、走入彼此内心，是有很大问题的。

因此，如果我们不学会去分享自己内心的情感和情绪，就会出现这样一个问题：即使你跟一个人关系推进了，甚至有了亲密的接触，甚至是发生了关系，但是你总会觉得跟这个人有隔膜、有隔阂。这种感觉很不好，就是你跟一个人身体可以很亲近，但是你的心跟他的心永远有一层隔膜。这个感觉是很糟糕的。所以我们一定要学会去跟对方分享我们内心的情感和情绪。

## 如何讲好一个深层次的故事 🎬

我们了解了时机和重要性，那么具体该如何操作呢？我们如何去讲好一个深层次的故事？我们一定要注意几个很重要的部分。

以下是讲述深层次故事的四个核心步骤：

**1. 交代清楚故事要素**
在故事的一开始，一定要交代好时间、地点、人物等基本要素。
*   `2007年是我第一次去西藏。`
*   `我的初恋是18岁的夏天。`
一个清晰的开头很重要，它交代清楚了故事的一些基本要素。

**2. 抓住故事的主干**
你一定要搞清楚你要讲的核心是什么。你所有的语言组织都要围绕着这个中心点。你要去分享你的情感和情绪，那最能体现或表达你当时状态的部分，就是故事的主干。你一定要抓住故事的主干，而不是纠缠于不重要的细节。
我们可能要讲一个很长时间的故事，如果抓不住主干，东讲一点，西讲一点，即使对方对你很感兴趣，他的注意力也是有限的。他注意力一分散，一判断这故事好长、好无聊，虽然他表面上在听，但注意力就不集中了。所以你一定要把故事的主干抓住，理清先讲什么后讲什么，很清晰，不要涉及很多无关的部分。

**3. 添加合适的细节**
你要根据故事的主干，来添加合适的细节。添加一些让故事变得更生动，或者能让对方更沉浸其中的细节。
例如，讲第一次被女生从背后环抱的故事，可以添加这样的细节：
> “我当时起身去买东西……忽然感觉有一双手环到了我的腰上，然后他整个人贴上来。我感觉一瞬间定住了，愣住。我感觉有一股温暖包围了我，好像有一股电流，从我的脊椎一直窜到头顶。一瞬间，我头脑一片空白。回过神来，我想到，原来这就是幸福的感觉，有一瞬间的晕眩。”

这些就是细节，是能够让对方身临其境的细节。“电流从脊椎一直上升到头顶”这样的描述，是能够很打动人心的。
再例如，讲一个去表白的故事，可以添加环境细节：
> “那是一年的秋天……我走在学校的小路上，路上都是落叶。我拿着礼物，踩着落叶，发出沙沙的声音。”

这也是细节，添加关于环境的细节，能让对方脑海当中更有画面感。
当然要注意，不要非常啰嗦，不是每个地方都有很多细节。可能一个故事只有一两个或两三个地方有细节，其他地方可以带过。要合适地添加细节，不要显得啰嗦。

**4. 沉浸在自己的世界**
你一定要学会沉浸在自己的世界里。去讲一个深层次的故事，其实是在你自己的世界里去把自己的情感表达出来。所以你一定是要沉浸其中的，而不要太过于在意对方的反应。
你有时候甚至不需要像平常说一些话题时那样，刻意去进行眼神接触。但是当你去说这些深层次话题的时候，你根本不需要去眼神接触。你可能是沉浸在自己的世界里这样一个状态。

## 故事示范与核心要点 🎭

接下来我可以示范一个深层次的故事。因为是在拍摄，可能跟我现场讲会有一些区别，但大家也可以稍微去揣摩或学习一下。

（模拟情境，对“对方”讲述）
> “其实，我是一个初恋很晚的人。我18岁才开始初恋。当时，我是一个在班里面很内向、平时也不怎么讲话的男生。然后当时，我们班里来了一个转学的女生。那是一天下午，我出教室门口，我当时习惯低头走，然后我就看到一双红色的鞋子，还有很白的皮肤。然后我就往上看，我就看到他。当时阳光很好，我看到他，那个画面就好像定格了一样。之后我就想去追他，想让他做我的女朋友。这是我第一次产生这样的念头，就是我想要找一个女朋友。那其实当时我的状态是什么？就是我跟他说句话，我都会紧张，我都会喘气，喘不过来，我都会脸红。当时是这样的一个状况。所以其实他跟我在一起，就是他跟我接触，其实是别扭的一个状态。快那时候快要中秋节了，中秋节之前呢，我就想想送他一个东西，然后想来想去呢，我就决定说想送他一个冰激凌的月饼。当时其实冰淇淋的月饼还蛮少见。然后买月饼倒还挺容易的，关键是没有地方保存。然后我需要去校外，学校里面不太行，我要去校外去找一个能够存放的地方，找个冰箱。然后好不容易去托朋友去帮我搞定了这个冰箱，然后我把这个月饼保存好。等中秋节那天，下晚自习的时候，我就去把月饼取出来，然后下晚自习的时候，我就叫住他，然后跟他说‘祝你中秋节快乐’，然后就把月饼给他。当时他没有看过，他好像有点高兴？可能是他会觉得说人来人往，然后把他叫住，他会觉得有点尴尬，他就不高兴。然后他也不看我，然后他就说了一句说‘啊，我不想要月饼’。当时我就想其实还蛮没面子的嘛。然后我就想，好不容易买了月饼，然后找到冰箱，我就很希望他收下，然后我就说‘反正买都买了，你就收下吧’，然后就把月饼放到他手里嘛。然后之后他做了一件事情，就是，他转过身走向了旁边的一个垃圾箱，然后他就把月饼扔进了垃圾箱里。当时那一刻，我真的很想找一个地缝钻下去。因为我觉得非常非常的丢脸。因为下晚自习嘛，人来人往，我就感觉好像每个人都在嘲笑我，觉得非常的难看。我当时耳朵肯定很红。然后我都不知道我怎么回到家里，回到家，我就直接把自己锁在了房间里。其实我哭了。我在想，为什么会受到这种对待。我想以后如果，我交了女朋友，或者说如果有女生喜欢我，我肯定会让他受到非常好的对待。”

大家注意，在这个故事讲完后，有一个核心要点：**在表达情感或情绪时，一定不要表达负面的东西**。
比如说我刚刚讲的故事，其实是一个我在感情上被伤害的故事。但是，重点并不是说我被伤害了，然后我要怎么样报复他，这是错的。这并不是一个正确的方向。
正确的方向应该是：**你已经释怀了**。这件事情你已经释怀了，然后你是用一个积极的态度再去面对，而不是可能你本身还处在一个负面情绪当中，你非常的愤恨、非常的愤怒，有很多负面情绪。不是这样，你已经走出来了，而且你是带着积极的态度去看待这件事情，或者说至少是带着平静的心态去看待。
而且不能是负面的解读，应该去正面的解读。正面的解读是什么？比如说刚刚那个故事：我被很冷漠地对待，但是我希望我喜欢的人、或者说我的女朋友，我能很好地对待他们。因为我之前的经历告诉我，被粗暴对待会很痛苦，所以我希望我很好地去对待他们。这是一个正面的看法、正面的想法。
所以这一点很重要，大家去讲一个深层次的故事的时候，最后情绪表达完之后，一定是要用积极的态度去面对，而不是出现一些负面情绪、憎恨或嫉妒。不要这样，一定是去正面地解读，带着正能量，这个非常非常重要。
